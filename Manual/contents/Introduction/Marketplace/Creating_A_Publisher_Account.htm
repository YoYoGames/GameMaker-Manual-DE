Ein Publisher-Konto erstellen

Bevor Sie den GameMaker-Marktplatz nutzen können, um Assets zu kaufen oder zu verkaufen, müssen Sie ein YoYo Konto erstellen. Wenn Sie Asset-Pakete verkaufen möchten, müssen Sie sich außerdem für ein Marketplace Publisher-Konto registrieren.

Um sich als Verleger zu registrieren, müssen Sie die folgenden Schritte ausführen:

  • Melden Sie sich bei Ihrem YoYo Konto an.
  • Sobald Sie sich in Ihrem Konto-Dashboard befinden, besuchen Sie die Seite Publisher-Registrierung. Diese finden Sie in Ihrem YoYo Konto-Dashboard im Bereich Publisher, wo Sie auf den Bereich Sell Your Assets klicken müssen:
  • Klicken Sie auf der nächsten Seite auf die Schaltfläche Registrieren und geben Sie dann einige grundlegende Informationen zu Ihrer Publisher-Identität ein, z. B. Ihren Publisher-Namen, Ihre E-Mail-Adresse und die Arten von Assets, die Sie hochladen möchten.

Sie müssen die Marketplace Publisher-Vereinbarung lesen und akzeptieren, bevor Sie die Registrierung Ihres Publisher-Kontos abschließen können. Wenn Sie dies getan haben, ist Ihre Anmeldung vollständig, aber beachten Sie, dass alle Registrierungen von YoYo Games überprüft werden müssen, so dass Ihr neuer Publisher nicht sofort für Sie verfügbar ist. Wenn Ihre Registrierung überprüft wurde, werden Sie über die von Ihnen angegebene E-Mail-Adresse benachrichtigt (bitte beachten Sie, dass wir zu den üblichen Geschäftszeiten im Vereinigten Königreich arbeiten, so dass es bei Anmeldungen am Wochenende usw. zu einer Verzögerung bei der Genehmigung kommen kann).

Sobald Sie die Registrierung abgeschickt haben, werden Sie zu Ihrem Publisher-Konto weitergeleitet, wo Sie die bereits gemachten Angaben einsehen und bearbeiten sowie einen Avatar hochladen können:

Gehen Sie nun zum Abschnitt Zahlungsdetails und füllen Sie dort das Formular aus:

Bitte beachten Sie dies:

  • Derzeit bietet Marketplace nur Zahlungen über PayPal an. Wenn Sie also ein Publisher-Konto anlegen, müssen Sie eine gültige E-Mail-Adresse für ein PayPal-Konto angeben, da Sie sonst nicht bezahlt werden können.
  • Für Nutzer aus dem Vereinigten Königreich/EU sollte eine gültige Umsatzsteuer-Identifikationsnummer angegeben werden, wenn dies auf Ihre Organisation zutrifft (für Nutzer außerhalb des Vereinigten Königreichs sollten Sie Ihrer Umsatzsteuer-Identifikationsnummer die zweistellige Länderkennung code für Ihr Land voranstellen).

Wenn Sie vorhaben, nur kostenlose Assets auf Marketplace hochzuladen, können Sie die Felder PayPal und Mehrwertsteuer ignorieren. Klicken Sie dann auf die Schaltfläche Senden, um fortzufahren.

Sobald Ihre Registrierung überprüft wurde, können Sie die übrigen Optionen der Publisher-Konsole nutzen. Dies ist das "Zuhause" all Ihrer Asset-Publishing-Vorgänge, und Sie sehen ein Menü mit verschiedenen Elementen auf der linken Seite. Jedes dieser Elemente zeigt Ihnen verschiedene Details zu Ihren Veröffentlichungsaktivitäten:

  • Assets: Auf dieser Seite erhalten Sie einen schnellen Überblick über Ihre Assets (sprites, Audio, Projekte usw.) und können deren Details einsehen und bearbeiten, Ihre Verkaufszahlen und Download-Statistiken überprüfen und auch eine neue Version Ihres Assets veröffentlichen. Sie können auch neue Assets, die Sie erstellt haben, zur Veröffentlichung hinzufügen. Weitere Informationen zur Veröffentlichung von Assets finden Sie auf der Seite: Auflisten eines Assets.
  • Dienstleistungen: Hier können Sie die Dienste sehen, die Sie anderen Nutzern von Marketplace anbieten, sowie deren Details bearbeiten und neue Dienste aktualisieren oder hinzufügen. Weitere Informationen über Dienste finden Sie auf der Seite: Einen Dienst auflisten.
  • Kontodetails: Aktualisieren Sie das öffentliche Profil Ihres Verlegers im Shop und fügen Sie die Zahlungsdaten über die Kontodetails hinzu (weitere Informationen zum Einrichten eines Zahlungskontos finden Sie oben).
  • Benutzerzugang: Wenn Sie in einem Team arbeiten, können Sie mehrere Benutzer zu Ihrem Publisher-Konto hinzufügen und festlegen, auf welche Teile der Publisher-Konsole sie Zugriff haben. Das Konto, das das Publisher-Konto registriert, ist der Eigentümer und hat vollen Zugriff auf alle Teile der Developer Console. Der Besitzer kann dann Benutzerkonten hinzufügen und verwalten, auf welche Teile der Developer Console sie Zugriff haben. Geben Sie einfach die E-Mail-Adresse der Person an, die Sie hinzufügen möchten, und markieren Sie dann die Teile der Publisher-Identität, auf die sie Zugriff haben soll.
  • Analytik: Über die Entwicklerkonsole haben Sie Zugriff auf tägliche Verkaufs- und Download-Berichte und Diagramme, mit denen Sie die Leistung Ihrer Assets verfolgen können. Sie können Ihre Berichte nach mehreren Dimensionen filtern, darunter Bezahlt oder Kostenlos, Kategorie, Land und Datum.
  • Käufe: Hier sehen Sie eine Liste aller Käufe, die von Ihren Assets getätigt wurden (kostenlose Käufe werden angezeigt, aber mit einem Preis von $0,00)
  • Autorenbedingungen: Hier finden Sie die aktuellen Autorenbedingungen für die Veröffentlichung auf Marketplace. Sie müssen diese akzeptieren, bevor Sie etwas veröffentlichen können, auch kostenlose Assets.

Beachten Sie, dass Sie nach der Einrichtung und Verifizierung eines Verlegerkontos zu Ihrem YoYo Konto-Dashboard zurückkehren und bei Bedarf weitere Konten erstellen können. Auf diese Weise könnten Sie zwei getrennte Konten einrichten, von denen eines Kunst-Assets und das andere code verkauft (zum Beispiel).
