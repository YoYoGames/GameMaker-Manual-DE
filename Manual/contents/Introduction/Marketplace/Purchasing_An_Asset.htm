Kauf einer Anlage

Um ein Asset auf Marketplace zu kaufen, müssen Sie zunächst in Ihrem GameMaker-Konto eingeloggt sein. Sobald Sie Ihr Konto erstellt haben und bei Marketplace angemeldet sind, können Sie die verfügbaren Asset-Pakete durchsuchen. Sie können offizielle Erweiterungen oder von anderen Community-Mitgliedern erstellte Inhalte herunterladen, die entweder kostenpflichtig oder kostenlos sind.

Wenn Sie ein Asset-Paket für GameMaker kaufen möchten, wählen Sie es einfach aus den Marktangeboten aus und klicken Sie auf die Schaltfläche "Kaufen für [Preis]". Dadurch wird das Produkt zu Ihrem Warenkorb hinzugefügt und Sie können entweder zur Kasse gehen oder die Assets weiter durchsuchen und weitere Einkäufe tätigen.

HINWEIS: Kostenlose Assets werden nicht in Ihren Warenkorb gelegt, sondern Sie werden lediglich aufgefordert, den Bedingungen der EULA zuzustimmen. Sobald Sie diese akzeptiert haben, wird das Asset zu Ihrer Bibliothek hinzugefügt.

Wenn Sie bereit sind, mit der Zahlung fortzufahren, können Sie die Option "Warenkorb" oben rechts auf dem Bildschirm auswählen und Sie werden zu einer Übersicht der Produkte in Ihrem Warenkorb geführt. Hier können Sie per Kreditkarte oder über PayPal bezahlen oder Ihre Einkäufe ändern:

Wenn Sie die Zahlung abgeschlossen haben, wird Ihnen der Bildschirm "Kauf abgeschlossen" angezeigt, wo Sie die Quittung ausdrucken können. Sie erhalten außerdem eine E-Mail mit den Details der Zahlungsbestätigung. Sobald Sie einen Kauf abgeschlossen haben (entweder ein kostenloses oder ein kostenpflichtiges Asset), können Sie zum GameMaker Marketplace Menü zurückkehren und auf den Menüpunkt "Meine Bibliothek" klicken, wo Sie die gekauften Assets herunterladen und installieren können. Auf der Seite über die Bibliothek finden Sie weitere Details zur Installation von Assets.
