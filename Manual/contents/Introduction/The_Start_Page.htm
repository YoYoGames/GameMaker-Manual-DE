Die Startseite

Wenn Sie GameMaker zum ersten Mal öffnen, werden Sie aufgefordert, sich bei Ihrem YoYo oder Opera-Konto anzumelden (oder sich für ein Konto zu registrieren, falls Sie dies noch nicht getan haben - siehe hier). Sobald Sie registriert und eingeloggt sind, öffnet sich die Haupt-IDE von GameMaker auf der Startseite:

Wenn du GameMaker zum ersten Mal startest, kannst du eine Beschreibung zu jedem Startbildschirm-Element sehen, indem du den Mauszeiger darüber hältst.

Hier können Sie auf die Schaltfläche "SKIP to Setup Wizard" klicken, um Ihr erstes Projekt mit Hilfe des Setup-Assistenten zu erstellen, der Sie durch die notwendigen Schritte zur Erstellung Ihres ersten Spiels in GameMaker führt.

Sie können auch auf "Schließen" klicken, um den Assistenten zu schließen und GameMaker manuell weiter zu verwenden, und optional das Kontrollkästchen "Nicht mehr anzeigen" aktivieren, wenn Sie den Assistenten beim nächsten Start von GameMaker nicht mehr sehen möchten.

Elemente der Startseite

Die Startseite enthält die folgenden Elemente:

  • IDE-Menüleiste: Hier werden die Menüs angezeigt, die Sie in der IDE verwenden können; weitere Informationen finden Sie in einem Abschnitt weiter unten.
  • Projekt-Optionen: Im Abschnitt Projekte können Sie Projekte erstellen, öffnen oder importieren. Dies wird weiter unten auf dieser Seite ausführlicher erklärt.
  • Letzte Projekte: Hier sehen Sie eine Liste der früheren Projekte, die Sie öffnen können. Sie können auch den vollständigen Projektnamen und -pfad sehen, und wenn Sie mit der linken Maustaste auf klicken, wird das Projekt geöffnet. Sie können auch switch die Ansicht dieses Bereichs zwischen dem Kachel- und dem Listenmodus wechseln, indem Sie auf die Schaltflächen in der oberen rechten Ecke drücken.
  • Versionsangaben: In diesem Abschnitt finden Sie Angaben zur aktuell verwendeten Version von IDE sowie zur aktuellen Runtime-Version. In diesem Abschnitt werden Sie auch über alle Änderungen informiert, die entweder für IDE oder Runtime verfügbar sind.
  • Benutzerdetails: Hier sehen Sie den Benutzernamen für Ihr YoYo Konto und das letzte Mal, als Sie sich angemeldet haben.
  • Nützliche Ressourcen: Hier werden Ihnen verschiedene Kacheln angezeigt, auf die Sie klicken können, um auf offizielle Tutorial-Ressourcen für GameMaker zuzugreifen.

IDE-Menüs

Oben finden Sie die allgemeinen Menüs IDE, die in den folgenden Abschnitten des Handbuchs erläutert werden:

  • Das Menü Datei
  • Das Menü Bearbeiten
  • Das Menü Bauen
  • Das Menü Windows
  • Das Menü Werkzeuge
  • Das Menü Marketplace
  • Das Menü Layouts
  • Das Hilfe-Menü

Beachten Sie, dass in der oberen Menüleiste auch kontextspezifische Menüoptionen erscheinen, je nachdem, auf welches Fenster Sie gerade den Fokus gelegt haben. Wenn Sie z. B. das Bildbearbeitungsfenster im Fokus haben, finden Sie hier zusätzliche Menüpunkte für "Bilder", "Ansicht" und "Effekte". Diese zusätzlichen Menüpunkte werden in den entsprechenden Abschnitten des Handbuchs für das workspace oder das Fenster, das sie erzeugt hat, erklärt.

Ein neues Projekt beginnen

Sie können auf die Schaltfläche Neu klicken, um ein neues Projekt zu erstellen, auf die Schaltfläche Öffnen , um ein bestehendes Projekt zu öffnen, oder auf die Schaltfläche Importieren , um eine komprimierte YYZ GameMaker-Projektdatei oder eine Legacy GameMaker: Studio 1.4 GMX Projektdatei zu öffnen. Sowohl Öffnen als auch Importieren öffnen den Datei-Explorer, in dem Sie die gewünschte Projektdatei suchen können.

HINWEIS: GameMaker ist nicht vollständig abwärtskompatibel mit GameMaker: Studio 1.4-Projekten, aber importierte 1.4-Projekte sollten trotzdem laufen, da veraltete Funktionen mit Hilfe von Kompatibilitätsscripts automatisch wiederhergestellt wurden. Ausführliche Informationen zu den möglichen Problemen und den an GML vorgenommenen Änderungen finden Sie im Hilfe-Center-Artikel Portieren eines GMS 1.4-Projekts nach GameMaker sowie im Abschnitt des Handbuchs über veraltete Funktionen.

Um ein neues Projekt zu erstellen, klicken Sie einfach auf die Schaltfläche Neu, wodurch sich das Menü Vorlage öffnet:

Wählen Sie eine Vorlage (oder beginnen Sie mit einem leeren Projekt), wählen Sie einen Namen und einen Speicherort für Ihr Projekt und klicken Sie dann auf "Los geht's!".
