Zimmer

Alle Spiele, die Sie in GameMaker erstellen, benötigen mindestens einen Raum (es können aber auch viele, viele mehr sein), und ein Raum ist einfach ein Raum, in dem Sie Instanzen der Objekte, aus denen Ihr Spiel besteht, sowie Kacheln und andere grafische Ressourcen platzieren. Wenn du zum ersten Mal einen Raum im Asset-Browser erstellst, wird dir ein neues Raum-Editor workspace mit dem du seine Eigenschaften bearbeiten kannst, etwa so:

Standardmäßig befinden sich die Raumwerkzeuge auf der linken Seite, mit drei Hauptbereichen:

  • Ebenen-Editor: Alles in einem Raum wird auf Ebenen platziert, und es gibt verschiedene Ebenentypen, aus denen Sie wählen können (mehr dazu später). Die Ebenen sind nach Tiefe geordnet, und diese Tiefe bestimmt die Reihenfolge, in der die Inhalte der Ebenen auf dem Bildschirm dargestellt werden, wenn das Spiel läuft. Die Reihenfolge der Tiefe ist von der höchsten zur niedrigsten, d.h. je niedriger die Tiefe, desto näher an der "Kamera" und je höher die Tiefe, desto weiter entfernt. Zum Beispiel wird eine Ebene mit einer Tiefe von -300 über einer Ebene mit einer Tiefe von -100 gerendert, und eine Ebene mit einer Tiefe von 1000 wird unter allem gerendert, was eine geringere Tiefe als diese hat. Beachten Sie, dass GameMaker die Ebenentiefen standardmäßig nach der Position Sie können die Ebenen mit der Maus verschieben, um sie neu anzuordnen, und GameMaker ordnet die Tiefen entsprechend an.
  • Ebeneneigenschaften: Jede Ebene, die Sie dem Raum hinzufügen, hat ihre eigenen Eigenschaften, und diese Eigenschaften ändern sich je nach Ebenentyp. In diesem Fenster können Sie diese Eigenschaften ändern und festlegen, wie die Ebene gerendert werden soll. gerendert wird.
  • Raumeigenschaften: Auch Räume haben eine Reihe von Eigenschaften, die hier eingestellt werden können. Hier können Dinge wie die Raumgröße, die aktiven Kameraperspektiven und einige andere Dinge eingestellt werden.

Wir haben bereits erwähnt, dass es verschiedene Ebenentypen gibt, also gehen wir kurz darauf ein, was sie sind und wie sie bei der Erstellung Ihrer Spiele verwendet werden können:

Hintergrundschicht Die Hintergrundebene ist eine Ebene, die mit einer einzigen Farbe oder einem einzigen Bild gefüllt werden kann. Sie kann innerhalb des Raumes verschoben und positioniert werden, und Sie können mehrere Hintergrundebenen haben. In der Regel wird sie, wie der Name schon sagt, verwendet, um einen konstanten Hintergrund für alle anderen Ebenen im Raum zu erzeugen. Standardmäßig enthält ein neuer Raum immer eine Hintergrundebene, aber Sie können sie entfernen, wenn Sie sie nicht brauchen.
Instanzschicht Auf der Instanzebene platzieren Sie alle Instanzen der Objekte, die Sie für das Spiel benötigen. Um eine Instanz zu einer Ebene hinzuzufügen, klicke einfach auf auf das Objekt im Asset-Browser und ziehen Sie es dann in den Raumeditor workspace. Wenn du die Maustaste loslässt, wird dem Raum eine Instanz des Objekts hinzugefügt. Standardmäßig enthält ein neuer Raum immer eine Instanz-Ebene, aber Sie können sie entfernen wenn Sie sie nicht brauchen.
Kachel Kartenebene Nachdem Sie ein Fliesenset erstellt haben, müssen Sie die Fliesen zu Ihrem Raum hinzufügen, indem Sie eine Fliesenplanebene erstellen. Eine Kachelzuordnungsebene ist eine Ebene, die es Ihnen ermöglicht, Kacheln aus allen von Ihnen erstellten Kachelsatz-Assets hinzuzufügen, und die automatisch so eingerichtet wird, dass sie ein Raster in der Größe der Zellen des Kachelsatzes verwendet. Sie können jeder Rasterzelle nur eine einzige Kachel hinzufügen. Wenn Sie also mehrere Kacheln auf demselben Platz benötigen, sollten Sie mehrere Kachelzuordnungsebenen verwenden.
Pfad Ebene Die Pfadebene ist die einzige Ebene, die nicht gerendert wird, wenn Sie Ihr Spiel ausführen. Diese Ebene ist eher eine "Bequemlichkeits-Ebene", die Ihnen beim Erstellen oder Bearbeiten von Pfad-Assets hilft, da sie Ihnen erlaubt, Pfade innerhalb des eigentlichen Raums hinzuzufügen oder zu bearbeiten.
Asset-Ebene Manchmal möchte man einen schönen grafischen Effekt, aber man möchte nicht den Overhead einer Instanz haben oder braucht sie nicht, um etwas anderes zu tun als sich selbst zu zeichnen. Sie könnten eine Kachelkarte verwenden, aber da Kachelkarten auf die Verwendung eines Rasters und nur eines Bildes pro ein einzelnes Bild pro Rasterzelle zu verwenden, können sie etwas restriktiv sein. In diesem Fall würden Sie die Vermögensebene verwenden. Die Asset-Ebene nimmt einfach eine Ressource sprite und zeichnet sie mit den Parametern, die Sie beim Hinzufügen zum Raum festlegen. Sprites und sequences können dieser Ebene auf die gleiche Weise hinzugefügt werden, wie Instanzen von Objekten der Instanzenebene hinzugefügt werden, d. h. Sie klicken auf auf sprite im Anlagenbrowser und ziehen dann ziehen Sie es in den Raumeditor workspace.

Um dem Raum-Editor eine Instanz hinzuzufügen, müssen Sie zunächst eine Instanz-Ebene aus dem Ebenen-Editor auf der linken Seite auswählen (oder eine erstellen, wenn keine vorhanden ist). Sie können dann zum Asset-Browser gehen und einfach auf klicken und ein gewünschtes Objekt von dort in den Raum an die gewünschte Position ziehen, wodurch eine einzelne Instanz dieses Objekts zu Ihrem Spiel hinzugefügt wird:

Wenn Sie mehrere Instanzen im Raum platzieren möchten, können Sie ein Objekt aus dem Ressourcenbaum auswählen und dann drücken und gedrückt halten, um eine Vorschau der Ressource anzuzeigen, und wenn Sie zusätzlich die linke Maustaste drücken, können Sie es in die Raumebene "malen".

Beachten Sie, dass Assets eine Position erhalten, wenn sie in den Raumeditor eingefügt werden, basierend auf ihrer X- und Y-Achsenposition relativ zum Raumursprung. Der Raumursprung ist die Position (0, 0), die in GameMaker die obere linke Ecke ist, und nach rechts ist +X und nach unten ist +Y, z. B:

Im obigen Beispiel befindet sich die Instanz in der Mitte des Raumes an der Position (X:512, Y:384). Es ist auch möglich, außerhalb der Raumes zu gehen und Positionen zu haben, die größer als die Raumachsenlängen oder sogar kleiner als der Raumursprung sind - in diesem Fall werden die X/Y-Koordinaten negative Werte sein.
