Sequenzen

Das letzte Asset, das wir in diesem Handbuch besprechen, bevor wir uns damit befassen, wie alles im Raumeditor zusammenkommt, ist das Asset " Sequenzen". Ein sequence ist im Wesentlichen eine Sammlung von anderen Assets, die an einem Ort gesammelt und dann zusammen animiert werden. Die endgültige Animation (sequence) kann dann ähnlich wie Instanzen zu Räumen in Ihrem Spiel hinzugefügt werden.

Sequenzen werden über den Editor Sequenz-Editor der immer dann geöffnet wird, wenn Sie ein neues sequence Asset im Asset-Browser erstellen:

Um eine einfache sequence zu erstellen, fügen Sie entweder einen Sound, eine sprite oder eine Instanz in die Sequenzleinwand ein und positionieren sie dann an der Stelle, an der sie am Anfang der sequence stehen soll. Dadurch wird eine Asset-Spur im Track Panel und fügt außerdem ein Schlüsselbild in das Dope Sheet ein, um die Dauer anzuzeigen, für die das Asset sichtbar sein wird.

Die Zeit für sequence wird in Frames angegeben (was dasselbe ist wie Schritte für Instanzen, d.h. ein Frame ist dasselbe wie ein Spielschritt), und Sie können sehen, was bei einem bestimmten Frame passieren wird, indem Sie die Sequenz Abspielkopf in der dope sheet. Die Position des Abspielkopfes wird auch verwendet, um Keyframes hinzuzufügen, d.h. die Frames, bei denen eine Zustandsänderung beginnen oder enden soll. Die Parameterschlüssel werden auf Parameter Spuren für das jeweilige Asset gespeichert.

Sie würden also das Asset positionieren und seine Skalierung, Farbe und andere Eigenschaften bei Frame 0 des sequence einstellen. Dann zeichnen Sie die Parametertasten für das Asset auf und bewegen den Abspielkopf zu dem Frame, in dem Sie den nächsten Asset-Status (Position, Farbe usw. ....) einstellen möchten. In diesem Frame ändern Sie erneut die verschiedenen Parameter und zeichnen die Tasten für sie auf. Diesen Vorgang wiederholen Sie dann nach Bedarf für die gesamte Länge des sequence. Wenn Sie auf sequence auf Play drücken, wird das Asset animiert, wobei sequence zwischen den verschiedenen Parameterzuständen interpoliert, so dass das Endergebnis etwa so aussieht:

Wir werden hier nicht näher auf die Funktionsweise von sequences eingehen, aber es genügt zu sagen, dass sie ein unglaublich leistungsfähiges Werkzeug für alles sind, von Zwischensequenzen über Menüs bis hin zu Charakterzuständen und vieles mehr.
