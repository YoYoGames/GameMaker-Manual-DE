Wählen Sie

Mit dieser Aktion können Sie eine Reihe verschiedener Optionen angeben (dies kann eine echte Zahl, eine Zeichenkette, eine Ressourcen-ID oder eine Konstante sein), von denen die Aktion eine zur Rückgabe auswählt. Sie können weitere Optionen hinzufügen, indem Sie auf das Plus-Symbol neben neben der Aktion klicken und einen weiteren Wert angeben, der zurückgegeben werden kann. Der zurückgegebene Wert wird dann in der Zielvariablen gespeichert, die zuvor erstellt worden sein kann oder eine neue temporäre lokale Variable sein kann (wenn Sie das Kontrollkästchen "Temp" aktivieren).

Aktion Syntax:

Argumente:

Argument Beschreibung
Option Die Option, die durch die Aktion choose zurückgegeben werden kann.

Beispiel:

Der obige Aktionsblock code wählt eine von vier Raumressourcen aus und speichert die zurückgegebene ID in einer temporären lokalen Variable. Diese wird dann verwendet, um den Raum zu wechseln.
