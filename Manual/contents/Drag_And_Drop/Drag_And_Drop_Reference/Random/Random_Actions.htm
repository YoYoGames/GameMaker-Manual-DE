Bibliothek für Zufallsaktionen

Die Zufallszahlen Bibliothek verfügt über die folgenden Aktionen für die Erzeugung von Zufallszahlen und Zufallsoperationen:

Zufallszahl abrufen
Randomisieren Sie
Wählen Sie

Beachten Sie, dass GameMaker bei jedem Start des Spiels von IDE aus denselben Seed für die Generierung von Zufallszahlen verwendet. Dies dient der Vereinfachung von debugging und bedeutet, dass Sie bei der Generierung von Zufallswerten dieselben Anfangsergebnisse erhalten, was jedoch nicht mehr der Fall ist, wenn Sie das Spiel zu einer ausführbaren Datei kompiliert haben (bei jedem Start wird ein neuer Seed generiert). Wenn Sie dieses Verhalten beim Testen nicht wünschen, sollten Sie die Aktion Randomise einmal zu Beginn des Spiels aufrufen, bevor Sie Zufallswerte erzeugen.
