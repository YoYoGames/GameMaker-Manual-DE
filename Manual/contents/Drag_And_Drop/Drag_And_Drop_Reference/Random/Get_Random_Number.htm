Zufallszahl abrufen

Mit dieser Aktion wird eine Zufallszahl innerhalb der Grenzen einer Reihe von Mindest- und Höchstwerten erzeugt. Sie können den Typ der zu erzeugenden Zahl wählen (entweder eine Dezimal- oder eine Ganzzahl) und dann den Bereich festlegen, innerhalb dessen die Zahl erzeugt werden soll, indem Sie einen Minimal- und Maximalwert. Der erzeugte Zufallswert wird an die Zielvariable zurückgegeben (die als temporäre lokale Variable gekennzeichnet werden kann).

HINWEIS: Bei der Generierung einer zufälligen Ganzzahl gibt die Aktion einen ganzzahligen Wert zwischen dem Minimal- und dem Maximalwert zurück, d. h. ein Bereich von 5 bis 10 kann sowohl 5 oder 10 als auch eine beliebige Ganzzahl dazwischen ergeben. Das Erzeugen einer zufälligen Dezimalzahl wird jedoch nie den Maximalwert zurückgeben, so dass der Bereich 5 bis 10 zwar 5, aber nicht 10 zurückgeben kann (obwohl er 9,999999999999 zurückgeben könnte).

Aktion Syntax:

Argumente:

Argument Beschreibung
Type Der Typ der zu verwendenden Zahlen (Ganzzahl oder Dezimalzahl)
Minimum Die Mindestzahl für den Zufallsbereich
Maximum Die Höchstzahl für den Zufallsbereich
Target Die Zielvariable, in der der zurückgegebene Wert gespeichert wird

Beispiel:

Der obige Aktionsblock code erzeugt einen Zufallswert und weist ihn einer temporären lokale Variable zu. Dieser Wert wird dann in einer Wiederholung loop verwendet, in der weitere Zufallsaufrufe getätigt werden, um Positionen innerhalb des Raums zu erzeugen. Diese Positionen werden dann verwendet, um eine Anzahl von Instanzen zu erzeugen.
