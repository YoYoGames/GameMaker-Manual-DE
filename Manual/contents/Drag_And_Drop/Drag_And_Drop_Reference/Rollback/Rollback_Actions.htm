Rollback Multiplayer-Aktionen

Rollback-Aktionen werden für die Erstellung von Multiplayer-Spielen mit dem Rollback-System von GameMaker verwendet.

Für eine Einführung, siehe: Rollback Multiplayer

Für eine Liste der Variablen, die mit dem Rollback-System verwendet werden, siehe: Rollback-Variablen

Die folgenden Aktionen sind für Rollback Multiplayer vorgesehen:

  • Spiel erstellen
  • Spiel beitreten
  • Spiel starten
  • Spiel verlassen
  • Eingabe definieren
  • Eingangsrahmenverzögerung definieren
  • Mock-Eingabe definieren
  • Spieler definieren
  • Ereignisse anzeigen
  • Infos erhalten
  • Eingabe erhalten
  • Wenn Taste unten (Rollback)
  • Wenn die Taste gedrückt wird (Rollback)
  • Wenn Taste oben (Rollback)
  • Wenn das Rollback-Ereignis ausgelöst wurde
  • Sync On Frame
  • Zufällige Eingabe verwenden
  • Manuellen Start verwenden
  • Nachricht senden
