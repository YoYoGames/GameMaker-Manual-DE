Eingabe holen (Rollback)

Diese Aktion gibt eine Struktur zurück, die die Eingabewerte für einen Spieler enthält. Sie nimmt ein optionales argument an, das die ID des Spielers angibt, für den die Eingaben zurückgegeben werden sollen.

Es wird empfohlen, If Key Down (Rollback), If Key Pressed (Rollback) und If Key Up (Rollback ) zur Überprüfung von Eingaben im Multiplayer zu verwenden.

Bei der Verwendung in einer Player-Instanz, die mit Player definieren (Rollback) erstellt wurde, müssen Sie keine Player-ID angeben, sondern können einfach player_id angeben.

Bei Verwendung in anderen Instanzen oder manuell erstellten Player-Instanzen müssen Sie eine Player-ID angeben.

Informationen darüber, welche Eingaben zurückgegeben werden und wie sie geändert werden können, finden Sie unter Definition von Eingaben.

Aktion Syntax:

Argumente:

Argument Beschreibung
Player ID Die ID des Spielers, für den Eingaben abgerufen werden sollen
Target Die Variable, in der die zurückgegebene Struktur gespeichert werden soll
