Rollback-Spiel beitreten

Mit dieser Aktion wird versucht, einem Spiel beizutreten. Sie müssen sich auf einer URL befinden, die die ID des virtuellen Raums enthält, dem Sie beitreten möchten.

Wenn ein Raum verbunden wurde, werden alle Aktionen, die mit dieser Aktion verbunden sind, ausgeführt.

Wenn ein neuer Spieler einem Spiel beitritt, in dem Sie bereits vertreten sind, wird das Ereignis Rollback Event ausgelöst. Weitere Informationen finden Sie unter Rollback-Ereignisse.

Not Wenn Sie diese Option aktivieren, werden alle angehängten Aktionen ausgeführt, wenn ein Raum nicht verbunden wurde. Sie können dies verwenden, um einen neuen Raum zu erstellen, wenn kein Raum gefunden wurde.
