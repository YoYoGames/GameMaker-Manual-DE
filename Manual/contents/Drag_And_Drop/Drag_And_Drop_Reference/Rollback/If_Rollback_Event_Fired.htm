Wenn das Rollback-Ereignis ausgelöst wurde

Diese Aktion wird im Ereignis Rollback Event verwendet, um zu prüfen, welche Art von Ereignis ausgelöst wurde.

Damit können Sie auf der rechten Seite Aktionen anhängen, die nur ausgeführt werden, wenn das angegebene Ereignis ausgelöst wurde.

Aktion Syntax:

Argumente:

Argument Beschreibung
Event Das zu prüfende Rollback-Ereignis
Not Aktivieren Sie diese Option, um zu prüfen, ob das angegebene Ereignis nicht ausgelöst wurde (was bedeutet, dass die Bedingung für alle anderen Ereignisse, die ausgelöst werden, erfüllt ist)
