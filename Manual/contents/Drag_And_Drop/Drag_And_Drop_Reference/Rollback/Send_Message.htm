Nachricht senden (Rollback)

Diese Aktion wird verwendet, um eine Nachricht an andere Spieler zu senden.

Sie geben eine zu sendende Zeichenfolge mit einer maximalen Größe von 128 Byte an.

Wenn das zweite argument nicht angegeben ist, wird die Nachricht an alle Spieler, einschließlich des Absenders, gesendet.

Sie können in der zweiten argument eine Spieler-ID angeben, die der Spieler ist, der die Nachricht erhalten soll.

Wie verwenden?

Diese Aktion ist für die "lokale" Verwendung gedacht, d. h. sie wird durch lokale Eingaben und nicht durch Rollback-Eingaben ausgelöst.

Siehe das Anwendungsbeispiel am Ende. Warten Sie auf empfangene Nachrichten im Ereignis Rollback Event.

Aktion Syntax:

Argumente:

Argument Beschreibung
Message Die zu sendende Nachricht in Form einer Zeichenkette.
To OPTIONAL Die ID des Spielers, an den die Nachricht gesendet werden soll. Wird diese argument nicht angegeben oder -1, wird die Nachricht an alle Spieler gesendet.

Beispiel:

Wenn die Eingabetaste lokal gedrückt wird, wird der keyboard_string Wert an alle Spieler gesendet. Diese Zeichenfolge wird dann gelöscht, damit eine neue Nachricht eingegeben werden kann.

Dadurch wird das Ereignis " Rollback Event " ausgelöst, wenn die Nachricht von allen Clients (einschließlich dem Absender) empfangen wird.
