Mock-Eingabe definieren (Rollback)

Diese Aktion kann während des Sync-Tests verwendet werden, um dem angegebenen Player eine temporäre "Schein"-Eingabe zuzuweisen. Dadurch werden die für den Player definierten Steuerelemente und alle zufälligen Eingaben von Sync Test außer Kraft gesetzt.

Aktion Syntax:

Argumente:

Argument Beschreibung
Player ID Die Nummer des Spielers, der diese Spiegeleingabe erhält
Input Der Name des zu definierenden Eingangs
Value Die zuzuweisende Eingangskonstante (siehe Benutzerdefinierte Steuerelemente)

Beispiel:

Die obige Aktion definiert die Schein-Eingänge Feuer, links, rechts, oben und unten für Spieler 1.
