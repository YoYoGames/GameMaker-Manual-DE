Eingabe definieren (Rollback)

Diese Funktion wird verwendet, um benutzerdefinierte Eingaben für alle Spieler zu definieren. Sie nimmt ein Input/Value-Paar auf, wobei Input der Name des Inputs und Value die ihm zugewiesene Input-Konstante ist.

Wie Sie benutzerdefinierte Eingänge zuweisen können, erfahren Sie unter Benutzerdefinierte Steuerelemente.

Wenn diese Funktion verwendet wird, muss sie vor der Join/Create-Aktion ausgeführt werden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Input Der Name des zu definierenden Eingangs
Value Die zuzuweisende Eingangskonstante (siehe Benutzerdefinierte Steuerelemente)

Beispiel:

Die obige Aktion definiert die Eingaben Feuer, links, rechts, oben und unten für alle Spieler.
