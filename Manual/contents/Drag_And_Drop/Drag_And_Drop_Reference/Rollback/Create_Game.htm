Rollback-Spiel erstellen

Mit dieser Aktion wird ein neues Multiplayer-Spiel gestartet. Sie gibt die Anzahl der Spieler an, die verbunden sein müssen, um das Spiel zu starten, und ermöglicht es Ihnen optional, den Sync-Test zu deaktivieren.

Wenn der Sync-Test deaktiviert ist, wartet das System auf die angegebene Anzahl von Spielern, bevor es das Spiel startet. Stellen Sie sicher, dass Sie keine verwalteten Instanzen erstellen oder ändern, bevor das Spiel beginnt.

TIPP: Sie können den Spielbeginn erzwingen, bevor alle Spieler beigetreten sind, indem Sie Spiel starten ausführen.

Derzeit können nur bis zu 4 Spieler an einem Spiel teilnehmen.

WARNUNG Es wird nicht empfohlen, bei dieser Aktion eine Standardregion festzulegen, es sei denn, Sie erlauben dem Spieler, diese über ein Menü auszuwählen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Count of players Die Anzahl der Spieler, die verbunden sein müssen, damit das Spiel beginnen kann. Wenn "Define Player " verwendet wird, ist dies die Anzahl der Instanzen, die automatisch erstellt werden.
Sync test Legt fest, ob der Synchrontest aktiviert werden soll, und ist standardmäßig aktiviert. Wenn sie deaktiviert ist, ist die Online-Funktionalität aktiviert.
Region Gibt die Region an, in der der virtuelle Raum erstellt werden soll. Für Informationen über den Erhalt von Regionen, siehe rollback_create_game().
