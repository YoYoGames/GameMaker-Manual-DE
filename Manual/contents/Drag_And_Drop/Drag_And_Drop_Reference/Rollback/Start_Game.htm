Rollback-Spiel starten

Diese Aktion erzwingt den Start des aktuellen Multiplayer-Spiels, bevor alle Spieler beigetreten sind. Sie müssen zuerst ein Mehrspielerspiel mit Rollback-Spiel erstellen erstellen.

Normalerweise wartet das System, bis alle Spieler beigetreten sind, bevor es das Spiel automatisch startet. Diese Aktion kann verwendet werden, um das Spiel frühzeitig zu starten, bevor alle Spieler beigetreten sind.

Sie können Manuellen Start verwenden aufrufen, damit Rollback auf diesen Aufruf wartet, auch wenn alle Spieler beigetreten sind.

Die Ausführung dieser Aktion löst das Ereignis Rollback Start aus.

Aktion Syntax:

Zurück: Rollback Multiplayer-Aktionen
Nächste: Rollback-Spiel verlassen
