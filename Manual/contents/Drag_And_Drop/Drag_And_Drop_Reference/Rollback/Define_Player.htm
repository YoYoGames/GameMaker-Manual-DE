Spieler definieren (Rollback)

Diese Aktion teilt dem Rollback-System das Objekt mit, das für Spieler verwendet werden soll. Dies ist optional, und wenn es verwendet wird, erstellt das System automatisch Instanzen für Spieler, die sich verbinden, und zerstört Instanzen für Spieler, die die Verbindung abbrechen.

Sie können die Ebene angeben, auf der die Player-Instanzen erstellt werden sollen. Wenn keine Ebene angegeben wird, werden die Spieler auf der Ebene "Instanzen" erstellt, die im Raum vorhanden sein muss.

Informationen zur Erstellung eines Spiels mit oder ohne diese Funktion finden Sie unter Definieren eines Spielerobjekts.

Falls verwendet, muss diese Funktion vor der Join/Start-Aktion ausgeführt werden.

Jede mit dieser Funktion erstellte Instanz erhält die auf dieser Seite aufgeführten Instanzvariablen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Object Das für Player-Instanzen zu verwendende Objekt
Layer OPTIONAL Der Name der Ebene, auf der die Player-Instanzen erstellt werden. Standardmäßig "Instances" (wenn kein Wert angegeben wird).
