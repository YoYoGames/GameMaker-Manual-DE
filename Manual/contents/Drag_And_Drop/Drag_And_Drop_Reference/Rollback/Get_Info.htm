Rollback-Informationen abrufen

Diese Funktion gibt eine Struktur mit Informationen über einen Spieler zurück. Diese Funktion sollte nur verwendet werden, wenn Spielerinstanzen manuell verwaltet werden. Wenn Sie Define Player verwenden, erhält jede Player-Instanz diese Informationen automatisch als Variablen.

Für Informationen über den Inhalt der zurückgegebenen Struktur siehe: rollback_get_info

Aktion Syntax:

Argumente:

Argument Beschreibung
Player ID Die ID des Spielers, für den Informationen abgerufen werden sollen
Target Die Variable, in der die zurückgegebene Struktur gespeichert werden soll
