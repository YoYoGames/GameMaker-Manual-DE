Wenn Taste unten

Mit dieser Aktion können Sie die Tastatur abfragen, um festzustellen, ob eine Taste gerade gedrückt wird. Die Aktion gibt bei jedem Schritt, bei dem die Taste gedrückt wird, true zurück, andernfalls false. Wenn Sie jedoch auf den Modifikator not klicken, können Sie prüfen, ob die Taste nicht gedrückt wird, d.h. die Aktion gibt true zurück, wenn keine Taste gedrückt wird, und false, wenn eine Taste gedrückt wird. Wenn Sie nur einen einzelnen Tastendruck überprüfen wollen, verwenden Sie die Aktion If Key Pressed.

Beachten Sie, dass Aktionen, die Sie in den "if"-Block einfügen möchten, an der Seite der Aktion abgelegt werden müssen, wie in der folgenden Abbildung gezeigt:

Alle Aktionen, die an anderer Stelle entfallen, werden nach dem "if"-Block ausgeführt Block ausgeführt.

Aktion Syntax:

Argumente:

Argument Beschreibung
Key Der Name der zu prüfenden Tastaturtaste.
Not Negieren der Prüfung (aus wahr wird falsch und umgekehrt)

Der obige Aktionsblock code fragt bei jedem Schritt den Tastaturstatus ab und prüft, ob eine Taste Wenn eine Taste gedrückt wird, prüft er, ob die Taste zum ersten Mal gedrückt wird. Lautet die Prüfung true auf den ersten Tastendruck, wird ein Ton abgespielt, und während die Taste gedrückt gehalten wird, wird die Überblendung sprite auf rot geändert. Wenn die Taste nicht gedrückt gehalten wird, wird eine Prüfung beim Loslassen der Taste eine Überprüfung durchgeführt, um die Mischfarbe auf Weiß zurückzusetzen.
