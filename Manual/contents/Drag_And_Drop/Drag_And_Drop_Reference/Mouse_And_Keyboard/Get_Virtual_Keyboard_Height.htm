Höhe der virtuellen Tastatur abrufen

Diese Aktion gibt die aktuelle Höhe der virtuellen Tastatur in Pixeln zurück, basierend auf der Größe des Displays. Wenn die Tastatur nicht sichtbar ist, wird 0 zurückgegeben.

Der obige Aktionsblock code prüft, ob die virtuelle Tastatur des Betriebssystems sichtbar ist, und wenn ja, ermittelt er die Höhe der Tastatur (in Bildschirm-Pixeln) und zeichnet dann einen Text darüber mit der Tastatureingabe.
