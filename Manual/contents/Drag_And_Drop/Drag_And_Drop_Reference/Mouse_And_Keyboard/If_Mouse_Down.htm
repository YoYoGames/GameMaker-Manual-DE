Wenn Maus nach unten

Mit dieser Aktion können Sie die Maus abfragen, um zu sehen, ob die ausgewählte Taste gerade gedrückt wird. Die Aktion gibt bei jedem Schritt, bei dem die Taste gedrückt wird, true zurück, andernfalls false. Wenn Sie jedoch auf den Modifikator not klicken, können Sie prüfen, ob die Taste nicht gedrückt wird, d. h. die Aktion gibt true zurück, wenn keine Taste gedrückt ist, und false, wenn eine Taste gedrückt ist. Wenn Sie nur auf einen einzelnen Tastendruck prüfen wollen, verwenden Sie die Aktion If Mouse Pressed.

Beachten Sie, dass Aktionen, die in den "if"-Block eingefügt werden sollen, an der Seite der Aktion abgelegt werden müssen, wie in der folgenden Abbildung gezeigt:

Diese Aktionen werden nun ausgeführt, wenn der "if"-Block den Wert true ergibt, während alle an anderer Stelle ausgelassenen Aktionen nach dem "if"-Block ausgeführt werden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Mouse Button Der Name der zu prüfenden Maustaste (siehe diese Seite für Informationen über Maustasten-Konstanten).
Not Negieren der Prüfung (aus wahr wird falsch und umgekehrt)

Beispiel:

Der obige Aktionsblock code fragt bei jedem Schritt den Zustand der Maustaste ab und prüft, ob die Taste gedrückt gehalten wird, wenn sie zum ersten Mal gedrückt wird. Wenn die Prüfung true beim ersten Drücken der Maustaste erfolgt, wird die Geschwindigkeit der Instanz eingestellt, und dann, während die Taste gedrückt ist, wird die Überblendung sprite auf rot geändert. Wenn die Maus nicht gedrückt gehalten wird, wird beim Loslassen der Maus geprüft, ob die Überblendungsfarbe auf Weiß zurückgesetzt und die Geschwindigkeit auf 0 gesetzt wird.
