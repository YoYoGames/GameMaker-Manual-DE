Wenn Maus freigelassen

Diese Aktion prüft, wann eine Maustaste losgelassen wurde. Sie gibt true bei dem einzigen Spielschritt (Frame) zurück, bei dem die Maustaste losgelassen wurde, und false zu allen anderen Zeiten. Wenn Sie den Modifikator not aktivieren, wird diese Aktion umgekehrt und prüft, ob die Maustaste nicht losgelassen wurde, und gibt false bei dem Spielschritt (Frame) zurück, bei dem sie losgelassen wurde, und true zu allen anderen Zeiten.

Beachten Sie, dass Aktionen, die Sie in den "if"-Block einfügen möchten, an der Seite der Aktion abgelegt werden müssen, wie in der folgenden Abbildung gezeigt:

Diese Aktionen werden nun ausgeführt, wenn der "if"-Block den Wert true ergibt, während alle an anderer Stelle ausgelassenen Aktionen nach dem "if"-Block ausgeführt werden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Mouse Button Der Name der zu prüfenden Maustaste (siehe diese Seite für Informationen über Maustasten-Konstanten).
Not Negieren der Prüfung (aus wahr wird falsch und umgekehrt)

Beispiel:

Der obige Aktionsblock code fragt den Zustand der Maustaste bei jedem Schritt ab und prüft, ob die Taste gedrückt gehalten wird, wenn sie zum ersten Mal gedrückt wird. Wenn die Prüfung true beim ersten Drücken der Maustaste erfolgt, wird die Geschwindigkeit der Instanz eingestellt, und dann, während die Taste gedrückt ist, wird die Überblendung sprite auf rot geändert. Wenn die Maus nicht gedrückt gehalten wird, wird beim Loslassen der Maus geprüft, ob die Überblendungsfarbe auf Weiß zurückgesetzt und die Geschwindigkeit auf 0 gesetzt wird.
