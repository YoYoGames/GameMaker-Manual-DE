Wenn die virtuelle Tastatur angezeigt wird

Mit dieser Aktion kann überprüft werden, ob die virtuelle Tastatur des Betriebssystems angezeigt wird oder nicht. Die Aktion wird als true ausgewertet, wenn die Tastatur offen oder geöffnet ist, oder false, wenn die virtuelle Tastatur geschlossen oder geschlossen ist. Wenn Sie den Modifikator not aktivieren, wird diese Aktion umgekehrt und Sie können den Rückgabewert true / false in einer Zielvariablen speichern, die als temp (lokal) gekennzeichnet werden kann.

Beachten Sie, dass Aktionen, die in den "if"-Block eingefügt werden sollen, an der Seite der Aktion abgelegt werden müssen, wie in der folgenden Abbildung gezeigt:

Diese Aktionen werden nun ausgeführt, wenn der "if"-Block zu true ausgewertet wird, während alle an anderer Stelle ausgelassenen Aktionen nach dem "if"-Block ausgeführt werden.
