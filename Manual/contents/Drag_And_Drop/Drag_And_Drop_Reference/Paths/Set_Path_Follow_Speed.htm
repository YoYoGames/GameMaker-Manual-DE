Pfadfolgegeschwindigkeit einstellen

Diese Aktion legt die Geschwindigkeit in Pixeln pro Spielframe fest, mit der eine Instanz einem Pfad folgen soll (beachte, dass du die Aktion Start Following Path aufgerufen haben musst, um der Instanz einen Pfad zuzuweisen, damit diese Aktion Wirkung hat). Du kannst negative Werte verwenden, um zu signalisieren, dass die Instanz dem Pfad in umgekehrter Richtung folgen soll, und es ist erwähnenswert, dass die Geschwindigkeit, die du hier einstellst, durch den Geschwindigkeitsmodifikator modifiziert werden kann, der im Pfad-Editor für jeden Pfadpunkt eingestellt ist, wenn du also Wenn du also eine Pfadgeschwindigkeit von 2 einstellst und einer der Punkte im Pfad einen Geschwindigkeitsmodifikator von 200% hat, wird die tatsächliche Geschwindigkeit der Pixel pro Spielframe an diesem Punkt 4 sein.

Aktion Syntax:

Argumente:

Argument Beschreibung
Speed Die Geschwindigkeit (in Pixel pro Spielframe), mit der die Instanz dem Pfad folgen soll

Beispiel:

Der obige Aktionsblock code startet eine Instanz, die einem Pfad folgt, und erzeugt dann einen Zufallswert Wert zwischen 2 und 4, bevor er diesen Wert verwendet, um die Geschwindigkeit festzulegen, mit der sich die Instanz entlang des Pfades bewegt.
