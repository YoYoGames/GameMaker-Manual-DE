Pfadfolgegeschwindigkeit abrufen

Diese Aktion ermittelt die Geschwindigkeit in Pixeln pro Spielframe, mit der eine Instanz einem Pfad folgt (beachte, dass du die Aktion Start Following Path aufgerufen haben musst, um der Instanz einen Pfad zuzuweisen, damit diese Aktion einen anderen Wert als 0 zurückgibt). Der zurückgegebene Wert wird in der von dir angegebenen Zielvariablen gespeichert, oder du kannst die Variable als temporäre lokale Variable kennzeichnen, um sie zu erstellen und den Wert zuzuweisen.
