Position entlang des Pfades festlegen

Mit dieser Aktion wird die Position einer Instanz entlang eines Pfades festgelegt. Der Wert ist normalisiert von 0 - 1, wenn du ihn also z.B. auf 0,5 setzt, wird die Instanz genau in die Mitte des Pfades verschoben. Beachten Sie, dass Sie die Aktion "Start Following Path " aufgerufen haben müssen, um der Instanz einen Pfad zuzuweisen, damit diese Aktion eine Wirkung hat.

Aktion Syntax:

Argumente:

Argument Beschreibung
Position Die Position (von 0 - 1) entlang des Pfades, an der die Instanz platziert werden soll

Beispiel:

Der obige Aktionsblock code startet die Instanz entlang eines Pfades und erzeugt dann einen zufälligen Dezimalwert zwischen 0 und 1. Dieser Wert wird in einer temporären lokalen Variablen gespeichert und verwendet, um die Position der Instanz entlang des Pfades festzulegen.
