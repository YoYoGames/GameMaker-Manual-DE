Spiel speichern

Diese Aktion führt eine einfache Speicherung des Spiels in einer bestimmten Datei durch. Du gibst einen Dateinamen an (als String und einschließlich der Erweiterung) und die Aktion führt eine begrenzte Speicherung des aktuellen Spielstandes durch. Es ist wichtig zu verstehen, dass die Aktion als "Schnappschuss" des aktuellen Spielzustands gedacht ist und eher als Checkpoint-System denn als universelles Speichersystem verwendet wird, da sie keine dynamischen Ressourcen (wie Datenstrukturen oder Partikel) speichert. Daher sollte die Aktion sehr vorsichtig verwendet werden, um Fehler oder Speicherlecks zu vermeiden. Ein umfassenderes Konzept zum Speichern Ihrer Spieldaten finden Sie in den Datei-Aktionen.
