Spiel neu starten

Mit dieser Aktion können Sie das Spiel neu starten. Dies ist vergleichbar mit dem erstmaligen Starten des Spiels, so dass das Ereignis "Other - Game Start " und das Ereignis "Other - Game End" ausgelöst werden. Es sollte jedoch beachtet werden, dass globale Variablen nicht neu initialisiert werden, wenn sie nicht explizit als solche kodiert sind - was bedeutet, dass sie ihre vorherigen Werte beibehalten, wenn sie nicht im allerersten Raum des Spiels definiert und gesetzt werden - und dass Datenstrukturen und Partikel werden auch nicht freigegeben, so dass dies vor dem Aufruf der Aktion "Spiel neu starten" oder im Spielende-Ereignis geschehen sollte. Beachte auch, dass das Spiel nicht in dem Moment neu gestartet wird, in dem diese Aktion aufgerufen wird, sondern am Ende des script oder des Ereignisses, in dem die Aktion platziert wurde. Wenn also irgendwelche Aktionen nach dieser Aktion platziert werden, werden sie trotzdem aufgerufen und das Spiel am Ende des Blocks neu gestartet.
