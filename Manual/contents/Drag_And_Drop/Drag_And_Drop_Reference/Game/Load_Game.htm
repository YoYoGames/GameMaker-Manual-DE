Spiel laden

Mit dieser Aktion wird ein Spiel geladen, das zuvor mit der Aktion Spiel speichern gespeichert wurde. Sie geben den Dateinamen der zu ladenden Datei an (als String, einschließlich der Erweiterung) und das Spiel wird geladen und läuft weiter von dem gespeicherten Punkt aus. Beachten Sie, dass dies kein universelles Speicher-/Ladesystem ist und dass der Versuch, ein Spiel zu laden, das in einem früheren Durchlauf des Projekts gespeichert wurde, zu Fehlern führen kann (vor allem, wenn Sie Dinge wie Datenstrukturen oder Partikel verwendet haben), und dass Sie daher nur versuchen sollten, Spiele zu laden, die während des Durchlaufs eines einzelnen Spiels gespeichert wurden (für Dinge wie Checkpoints). Beachten Sie auch, dass die Datei erst nach dem Ende des aktuellen Ereignisses oder script geladen wird, so dass alle Aktionen nach dem Aufruf der Ladeaktion noch ausgeführt werden. Ein umfassenderes Konzept für das Laden gespeicherter Spieldaten finden Sie unter Datei-Aktionen.
