Exit-Spiel

Mit dieser Aktion können Sie das Spiel beenden, indem Sie das Ereignis Sonstiges - Spielende auslösen. Beachten Sie, dass dies nicht in dem Moment geschieht, in dem die Aktion aufgerufen wird, sondern erst am Ende des aktuellen Ereignisses oder script, so dass alle Aktionen, die du nach dem Aufruf dieser Aktion durchführst, weiterlaufen, bis das Ende des Ereignisses/Skripts erreicht ist.

Bitte beachten Sie, dass für diese Aktion die folgenden Einschränkungen gelten:

  • Es meldet einen Fehler auf dem iOS-Ziel, da es gegen die Bedingungen des Marktplatzes verstößt.
  • Es hat keine Auswirkungen, wenn es auf den HTML5- und Windows UWP-Zielen aufgerufen wird.
  • Es funktioniert unter Windows (einschließlich der für Steam entwickelten Spiele), Android, Ubuntu (Linux) und Mac.
