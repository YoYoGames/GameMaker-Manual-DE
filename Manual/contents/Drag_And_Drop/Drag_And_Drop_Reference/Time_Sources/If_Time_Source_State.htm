Wenn [Zeitquelle] Status

Mit dieser Aktion können Sie andere Aktionen auf der Grundlage des Status der angegebenen Zeitquelle steuern.

Sie prüft, ob der in Zeitquellenstatus Konstante der in "Zustand" angegebene Zustand der in "Zeitquelle" angegebenen Zeitquelle ist (oder nicht).

Aktion Syntax:

Argumente:

Argument Beschreibung
Time Source Die Zeitquellen-ID zur Überprüfung des Status von
Not Aktivieren Sie dies, um die Bedingung umzukehren
State Die Zeitquelle Zustandskonstante zur Prüfung gegen

Beispiel:


Dieser code -Block sollte idealerweise bei einem Tastendruck-Ereignis ausgeführt werden.

Sie prüft, ob der Status einer Zeitquelle "aktiv" ist, und wenn das der Fall ist, hält sie die Zeitquelle an.

Ist der Zustand jedoch "pausiert", wird die Zeitquelle erneut gestartet.
