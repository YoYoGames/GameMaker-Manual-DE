Zeitquelle zerstören

Durch diese Aktion wird die angegebene Zeitquelle vernichtet.

Sie können eine Zeitquelle nicht zerstören, wenn sie noch Kinder hat. Zerstören Sie zuerst die untergeordneten Zeitquellen, bevor Sie eine übergeordnete Zeitquelle zerstören(Beispiel).

Aktion Syntax:

Argumente:

Argument Beschreibung
Time Source Die Zeitquellen-ID zu zerstören

Beispiel:


Es wird geprüft, ob eine Zeitquelle existiert, und wenn ja, wird sie zerstört.
