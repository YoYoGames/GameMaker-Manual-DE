Verbleibende Zeit abrufen

Diese Aktion ermittelt die verbleibende Zeit bis zum Ablauf der angegebenen Zeitquelle. Dieser Wert wird in den für die Zeitquelle verwendeten Einheiten ausgedrückt.

Aktion Syntax:

Argumente:

Argument Beschreibung
Time Source Die Zeitquellen-ID zum Abrufen der verbleibenden Zeit für
Target Die Variable zum Speichern des zurückgegebenen Wertes
Temp ob die Variable als temporäre Variable angelegt werden soll
