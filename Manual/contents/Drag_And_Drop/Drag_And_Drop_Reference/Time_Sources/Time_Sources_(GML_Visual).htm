Zeitquellen (GML Visuell)

Diese Seite listet die GML Visual Aktionen auf, die für Zeitquellen verfügbar sind. Einen Überblick über Zeitquellen finden Sie hier.

Die folgenden Aktionen sind für Zeitquellen in GML Visual verfügbar:

  • Zeitquelle erstellen
  • Zeitquelle zerstören
  • Startzeit Quelle
  • Stoppzeit Quelle
  • Pause Zeit Quelle
  • Lebenslauf Zeitquelle
  • Zeitquelle rekonfigurieren
  • Zeitquelle zurücksetzen
  • Kinder bekommen
  • Elternteil werden
  • Zeitraum erhalten
  • Erledigte Wiederholungen
  • Verbleibende Wiederholungen erhalten
  • Zustand erhalten
  • Verbleibende Zeit abrufen
  • Einheiten erhalten
  • Wenn die Zeitquelle vorhanden ist
  • Wenn Zeitquelle Elternteil
  • Wenn Zeitquellenstatus
  • Wenn Zeitquelle Einheiten

Die folgenden Aktionen sind für die Umwandlung von Zeiträumen vorgesehen:

  • Sekunden bis BPM
  • BPM in Sekunden

Die folgenden Konstanten werden bei Zeitquellenaktionen verwendet:

  • Eingebaute Zeitquellen
  • Zeitquelle Einheiten
  • Zeitquelle Verfallsarten
  • Zeitquellenstaaten
