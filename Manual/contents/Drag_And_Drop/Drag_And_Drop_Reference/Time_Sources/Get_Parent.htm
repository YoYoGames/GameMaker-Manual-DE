Elternteil werden

Diese Aktion ruft die übergeordnete Zeitquelle für die angegebene Zeitquelle ab.

Der zurückgegebene Wert kann die ID einer benutzerdefinierten Zeitquelle oder einer der eingebauten Zeitquellen sein, je nachdem, wie die gegebene Zeitquelle erstellt wurde.

Aktion Syntax:

Argumente:

Argument Beschreibung
Time Source Die Zeitquellen-ID um das Elternteil von
Target Die Variable zum Speichern der übergeordneten Zeitquelle
Temp ob die Variable als temporäre Variable angelegt werden soll
