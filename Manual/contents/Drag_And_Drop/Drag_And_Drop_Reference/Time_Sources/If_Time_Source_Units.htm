Wenn [Zeitquelle] Einheiten

Mit dieser Aktion können Sie andere Aktionen basierend auf dem Einheitentyp der angegebenen Zeitquelle steuern.

Sie prüft, ob die in Zeitquelle Einheit Konstante die in "Units" angegeben ist, dem Einheitentyp der in "Time Source" angegebenen Zeitquelle entspricht (oder nicht).

Aktion Syntax:

Argumente:

Argument Beschreibung
Time Source Die Zeitquellen-ID zur Überprüfung des Status von
Not Aktivieren Sie dies, um die Bedingung umzukehren
Units Die Zeitquelle Einheit Konstante zur Prüfung gegen
