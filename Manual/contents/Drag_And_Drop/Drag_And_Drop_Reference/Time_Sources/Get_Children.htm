Kinder bekommen

Diese Aktion gibt ein Array mit allen untergeordneten Zeitquellenfür die angegebene Zeitquelle zurück.

Das resultierende Array wird in der Variablen Target gespeichert.

Aktion Syntax:

Argumente:

Argument Beschreibung
Time Source Die Zeitquellen-ID um die Kinder von
Target Die Variable zum Speichern des Arrays
Temp ob die Variable als temporäre Variable angelegt werden soll

Beispiel:


Diese Funktion ruft die Kinder einer Zeitquelle ab und loops durchsucht das Array, um jede untergeordnete Zeitquelle zu zerstören.

Am Ende zerstört sie die übergeordnete Zeitquelle.
