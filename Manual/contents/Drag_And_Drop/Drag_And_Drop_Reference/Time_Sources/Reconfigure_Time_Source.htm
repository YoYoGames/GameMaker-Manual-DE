Zeitquelle rekonfigurieren

Mit dieser Aktion können Sie die Einstellungen für eine bestehende Zeitquelle, die im Feld "Zeitquelle" angegeben ist, ändern.

Die hier vorgenommenen Einstellungen sind die gleichen wie bei Zeitquelle erstellen, mit Ausnahme des Elternteils.

Dadurch wird die Zeitquelle zurückgesetzt und deaktiviert, so dass Sie sie neu starten müssen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Time Source Die Zeitquellen-ID zum Rekonfigurieren
Period Die Länge des Zeitraums der Zeitquelle, wie lange es dauert, bis sie abläuft
Units Die Einheiten, in denen der Zeitraum ausgedrückt wird (Sekunden oder Bilder)
Callback Die method wird aufgerufen, wenn die Zeitquelle abläuft
Arguments OPTIONAL Ein Array, das die arguments enthält, die an die Callback-Funktion übergeben wird
Repetitions OPTIONAL Wie oft die Zeitquelle insgesamt ausgeführt werden soll, oder -1 für unbestimmte Wiederholungen
Expiry Type OPTIONAL Der Verfallstyp für die Zeitquelle
