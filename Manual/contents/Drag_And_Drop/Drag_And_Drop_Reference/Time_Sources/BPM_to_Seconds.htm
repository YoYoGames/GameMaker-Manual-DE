BPM in Sekunden

Diese Aktion nimmt einen Wert für "Schläge pro Minute" und gibt die Länge jedes Schlags in Sekunden zurück.

Dies kann beim Erstellen einer Zeitquelle verwendet werden, um einen BPM-Wert für die Zeitquelle zu verwenden. Es ist wichtig, dass eine solche Zeitquelle Sekunden als Einheit verwendet.

Aktion Syntax:

Argumente:

Argument Beschreibung
Value Der Wert für die Schläge pro Minute
Target Die Variable zum Speichern des zurückgegebenen Sekunden-pro-Schlag-Werts
Temp ob die Variable als temporäre Variable angelegt werden soll
