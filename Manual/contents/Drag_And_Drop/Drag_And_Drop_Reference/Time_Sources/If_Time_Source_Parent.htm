Wenn [Zeitquelle] Elternteil

Mit dieser Aktion können Sie andere Aktionen auf der Grundlage der übergeordneten Zeitquelle steuern.

Es wird geprüft, ob die in "Parent" angegebene Zeitquelle die übergeordnete Quelle der in "Time Source" angegebenen Zeitquelle ist (oder nicht).

Aktion Syntax:

Argumente:

Argument Beschreibung
Time Source Die Zeitquellen-ID zur Überprüfung der übergeordneten Quelle von
Not Aktivieren Sie dies, um die Bedingung umzukehren
Parent Die Zeitquellen-ID wird mit der übergeordneten Zeitquelle der zuvor angegebenen Zeitquelleverglichen
