Wenn die Zeitquelle vorhanden ist

Mit dieser Aktion können Sie andere Aktionen davon abhängig machen, ob die angegebene Zeitquelle existiert oder nicht.

Aktion Syntax:

Argumente:

Argument Beschreibung
Time Source Die Zeitquellen-ID zu prüfen
Not Aktivieren Sie diese Option, um verbundene Aktionen auszuführen, wenn die Zeitquelle nicht vorhanden ist.

Beispiel:


Es wird geprüft, ob eine Zeitquelle existiert, und wenn ja, wird sie zerstört.
