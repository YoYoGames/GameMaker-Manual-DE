Zeitquelle erstellen

Diese Aktion erstellt eine neue Zeitquelle und speichert sie in der Variable Ziel. Lesen Sie Zeitquellen für einen Überblick.

Dies basiert auf der time_source_create() Funktion. Lesen Sie die Seite für detaillierte Informationen über die arguments.

Sie müssen Zeitquelle starten verwenden, um eine Zeitquelle zu aktivieren, nachdem sie erstellt wurde.

Sie müssen eine Zeitquelle mit Zeitquelle zerstören zerstören, wenn Sie sie nicht mehr benötigen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Target Die Variable, in der die neue Zeitquelle gespeichert werden soll
Parent Die übergeordnete Zeitquelle: entweder eine integrierte Zeitquelle oder eine benutzerdefinierte Zeitquelle
Period Die Länge des Zeitraums der Zeitquelle, wie lange es dauert, bis sie abläuft
Units Die Einheiten, in denen der Zeitraum ausgedrückt wird (Sekunden oder Bilder)
Callback Die method oder Skriptfunktion, die aufgerufen wird, wenn die Zeitquelle abläuft
Arguments OPTIONAL Ein Array, das die arguments enthält, die an die Callback-Funktion übergeben wird
Repetitions OPTIONAL Wie oft die Zeitquelle insgesamt ausgeführt werden soll, oder -1 für unbestimmte Wiederholungen
Expiry Type OPTIONAL Der Verfallstyp für die Zeitquelle

Beispiel:

Dieser Block code hat folgende Aufgaben:

  • Sie erstellt eine neue Funktion, bei der die Option "Temp" aktiviert ist. Dadurch wird eine lokale Methodenvariable erstellt.
    • Diese Methode nimmt eine argument, msg, die sie in das Ausgabeprotokoll ausgibt.
  • Anschließend wird eine neue Zeitquelle mit einer Periodenlänge von 1 Sekunde erstellt.
    • Sie gibt die Variable callback_method als "Callback" und ein Array mit einer argument für die Funktion an.
    • Diese Zeitquelle wird so eingestellt, dass sie sich unbegrenzt wiederholt, da -1 in "Wiederholungen" angegeben ist.
  • Die Zeitquelle wird dann gestartet.

Diese Zeitquelle führt ihren Callback jede Sekunde aus und druckt "A second has passed!" in das Ausgabeprotokoll.
