Erledigte Wiederholungen

Mit dieser Aktion wird die Anzahl der Wiederholungen ermittelt, die die angegebene Zeitquelle seit ihrer letzten Rücksetzung durchgeführt hat.

Aktion Syntax:

Argumente:

Argument Beschreibung
Time Source Die Zeitquellen-ID um die abgeschlossenen Wiederholungen für
Target Die Variable zum Speichern des zurückgegebenen Wertes
Temp ob die Variable als temporäre Variable angelegt werden soll
