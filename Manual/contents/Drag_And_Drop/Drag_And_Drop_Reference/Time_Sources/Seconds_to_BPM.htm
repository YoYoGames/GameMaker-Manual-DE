Sekunden bis BPM

Diese Aktion nimmt die Länge eines Beats in Sekunden und gibt einen Wert für die Beats pro Minute zurück.

Aktion Syntax:

Argumente:

Argument Beschreibung
Value Der Sekunden-pro-Schlag-Wert
Target Die Variable, in der der zurückgegebene Wert für die Schläge pro Minute gespeichert wird
Temp ob die Variable als temporäre Variable angelegt werden soll
