Instanz Alpha setzen

Dieser Aktionsblock setzt den Wert image_alpha für die Instanz. Das Bild-Alpha ist der "Transparenz"-Wert für das Zeichnen aller sprite, die der Instanz zugewiesen sind, und hat einen Wert zwischen 0 und 1, wobei 0 vollständig transparent und 1 vollständig undurchsichtig wäre. Beachte, dass, wenn sprite transparente oder halbtransparente Pixel hat, ihre Transparenz um den Wert skaliert wird. Wenn du also einige 50% transparente Pixel in sprite hast und das Instanz-Alpha auf 0,5 setzt, würde das endgültige Alpha für die transparenten Pixel 25% sein. Sie können diesen Wert auf einen relativen Wert einstellen

Bitte beachten Sie, dass die Instanz entweder kein Draw-Ereignis haben sollte (und GameMaker daher standardmäßig die sprite zeichnet) oder mit der Aktion Draw Self gezeichnet werden sollte, damit Änderungen in dieser Variablen sichtbar werden.

Argumente:

Argument Beschreibung
