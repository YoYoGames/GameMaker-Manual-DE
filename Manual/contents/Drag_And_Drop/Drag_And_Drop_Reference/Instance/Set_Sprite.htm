Sprite setzen

Dieser Aktionsblock setzt die sprite_index der Instanz auf eine andere sprite aus dem Asset-Browser. Sie wählen die sprite aus, zu der Sie wechseln möchten, und geben dann das Bild (Animationsframe) an, das angezeigt werden soll. Wenn Sie nicht möchten, dass sprite nach dem Setzen des Rahmens animiert wird Wenn du nicht willst, dass image_index nach dem Setzen des Rahmens animiert wird, musst du die Aktion " Animationsgeschwindigkeit festlegen " verwenden und den Wert auf 0 setzen, oder wenn du willst, dass sprite weiter animiert wird, musst du den Wert so setzen, dass er die eingebaute Variable stattdessen.

Wenn Sie die sprite ändern, wird auch die Kollisionsmaske der Instanz geändert, es sei denn, Sie haben im Objekteditor auch eine separate mask_index angegeben. Bitte beachten Sie, dass Änderungen in dieser Aktion sichtbar sind, sollte die Instanz entweder kein Draw-Ereignis haben (so dass GameMaker standardmäßig die sprite zeichnet) oder mit der Aktion Draw Self gezeichnet werden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Sprite Die neu zu verwendende sprite (-1, wenn Sie sprite entfernen möchten).
Frame Der Animationsrahmen, der anfangs angezeigt werden soll (wenn keine zusätzlichen Rahmen verwendet werden, verwenden Sie 0).
