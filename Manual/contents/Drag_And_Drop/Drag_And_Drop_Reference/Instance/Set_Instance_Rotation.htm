Instanz Rotation einstellen

Dieser Aktionsblock legt die image_angle (Drehung) der Instanz fest. Der Winkel wird in Grad gemessen, wobei rechts 0° ist, oben 90°, links 180° und unten 270°. Setzen Sie diese Variable auf 0, um die Instanz zurückzusetzen, was bedeutet, dass sie Das bedeutet, dass sie jede sprite so zeichnet, wie sie im sprite Editor definiert wurde. Bitte beachte, dass die Instanz, damit Änderungen in dieser Aktion sichtbar werden, entweder kein Draw-Ereignis haben sollte (so dass GameMaker standardmäßig die sprite zeichnet) oder mit der Aktion Draw Self gezeichnet werden.
