Objektinstanz erstellen

Mit dieser Aktion können Sie eine Instanz eines beliebigen Objekts aus dem Asset-Browser erstellen. Sie geben den Namen des Objekts und eine x/y-Position innerhalb des Raums an, in dem es erstellt werden soll, sowie den Namen der Ebene, auf der die Instanz erstellt werden soll (es muss eine Instanz-Ebene sein). Du kannst das Häkchen bei "relativ" flags setzen, um die Instanz relativ zu der Instanz zu positionieren, die die Aktion code blockiert. Wenn du eine Instanz im Spiel erstellst, erhält sie einen eindeutigen ID-Wert, mit dem du die Instanz dann Diese Aktion erlaubt es dir daher, eine "Ziel"-Variable zu definieren, die diesen ID-Wert enthält (wie im Beispiel unten gezeigt). Beachten Sie, dass das Ziel als temporäre (lokale) Variable gekennzeichnet werden kann, die für die Aktion erstellt wird und in allen nachfolgenden Aktionen verwendet werden kann.

Aktion Syntax:

Argumente:

Argument Beschreibung
Object Der Name des Objekts, von dem eine Instanz erstellt werden soll.
x Die x-Position im Raum, um die Instanz zu erstellen (kann relativ zur aufrufenden Instanz sein).
y Die y-Position im Raum, um die Instanz zu erstellen (kann relativ zur aufrufenden Instanz sein).
Layer Der Name (eine Zeichenkette) der Instanzebene, auf der die Instanz erstellt werden soll.
Target Die Variable, die als Ziel für den Rückgabewert dienen soll.

Beispiel:

Der obige Aktionsblock code erstellt eine neue Instanz und speichert ihren eindeutigen ID-Wert in einer temporären Variable. Diese Variable wird dann in der Aktion " Gilt für..." verwendet, um einige Eigenschaften der erstellten Instanz festzulegen.
