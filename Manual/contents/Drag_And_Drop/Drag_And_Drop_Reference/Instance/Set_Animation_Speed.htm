Animationsgeschwindigkeit einstellen

Dieser Aktionsblock setzt die image_speed (Animationsgeschwindigkeit) für die sprite, die der Instanz zugewiesen ist. Du stellst die idealen Bilder pro Sekunde im Sprite-Editor ein und kannst sie dann mit diesem Wert um ihn zu ändern. Die Standard-Animationsgeschwindigkeit ist auf 1 eingestellt, was bedeutet, dass das Spiel versuchen wird, die Anzahl der Bilder pro Sekunde beizubehalten, die du für die sprite definiert hast, oder Dezimalwerte, um langsamer zu animieren (die Bilder werden in mehreren Schritten angezeigt), oder sogar negative Werte, um die Geschwindigkeit der Animation umzukehren.
