Elternveranstaltung aufrufen

Dieser Aktionsblock führt das Ereignis des übergeordneten Objekts aus, von dem die Instanz ein Kind ist, bevor die Aktionen in der eigentlichen Instanz ausgeführt werden. Wenn Sie einer Instanz ein übergeordnetes Objekt zuweisen, werden alle Ereignisse dieses Objekts Kind "geerbt". Wenn Sie dem Kindobjekt jedoch ein Ereignis hinzufügen, überschreibt es das übergeordnete Ereignis. Mit diesem Aktionsblock können Sie ein untergeordnetes Objekt zwingen, sowohl das übergeordnete Ereignis als auch das Ereignis in der Instanz selbst auszuführen. Weitere Informationen über Elternobjekte finden Sie im Abschnitt Objekteditor.
