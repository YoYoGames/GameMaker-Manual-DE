Benutzerereignis aufrufen

Mit dieser Aktion können Sie jedes der 12 verfügbaren Other - User Events aufrufen. Ein Benutzerereignis ist ein Ereignis, dem Sie scripts oder die Aktion code hinzufügen können und das Sie dann jederzeit mit dieser Aktion aufrufen können, was sie im Wesentlichen zu benutzerdefinierten Objektereignissen macht.

Aktion Syntax:

Argumente:

Argument Beschreibung
Event Die Nummer des aufzurufenden Benutzerereignisses (0 - 11).

Der obige Aktionsblock code prüft eine Variable und ruft dann entweder das Benutzerereignis [0] oder das Benutzerereignis [1] auf, je nachdem, ob die Prüfung den Wert true oder false ergibt.
