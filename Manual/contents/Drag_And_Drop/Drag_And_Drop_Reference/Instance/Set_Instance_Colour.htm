Instanzfarbe einstellen

Dieser Aktionsblock legt die Farbe image_blend für das "Blending" mit der Instanz sprite fest. Der Standardwert ist der hexadezimale Wert code $FFFFFFFF (der die Farbe Weiß im ARGB-Format darstellt) und dies zeichnet die sprite ohne Überblendung und mit vollem Alpha. Jeder andere Wert (einschließlich interner Farbkonstanten wie c_red, oder c_aqua) wird die angegebene Farbe mit der ursprünglichen sprite überblenden und die image_alpha Komponente für die Instanz, aber nur, wenn das Kontrollkästchen Alpha verwenden aktiviert ist. Dies überschreibt jeden Alphawert, der zuvor mit der Aktion Instanz-Alpha setzen gesetzt wurde.

Damit Änderungen in dieser Aktion sichtbar werden, sollte die Instanz entweder kein Draw-Ereignis haben (so dass GameMaker standardmäßig die sprite zeichnet) oder mit der Aktion Draw Self gezeichnet werden. Es ist Es ist auch wichtig zu beachten, dass Sie versuchen sollten, die Überblendung auf HTML5 platform zu begrenzen (es sei denn, Sie verwenden WebGL), da jede überblendete sprite separat zwischengespeichert werden muss und daher viele überblendete sprites die Leistung beeinträchtigen können.
