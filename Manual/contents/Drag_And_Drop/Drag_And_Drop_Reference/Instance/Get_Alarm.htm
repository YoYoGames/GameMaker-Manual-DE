Alarm erhalten

Dieser Aktionsblock ruft den Wert eines Alarm-Countdowns ab und speichert ihn in der angegebenen Zielvariablen. Wenn Sie die Zielvariable als "Temp" kennzeichnen, wird eine neue temporäre (lokale) Variable erstellt, die den Rückgabewert für Sie speichert, wie im folgenden Beispiel.

Aktion Syntax:

Argumente:

Argument Beschreibung
Alarm Der Alarm zum Mitnehmen.
Target Die Zielvariable, die zum Speichern des Alarmwertes verwendet werden soll.

Beispiel:

Der obige Aktionsblock code erstellt eine lokale (temporäre) Variable und speichert den aktuellen Wert von des Alarms[0]. Dieser Wert wird dann überprüft, und wenn er kleiner als 0 ist, wird der Alarm auf 60 gesetzt.
