Skalierung der Instanz festlegen

Dieser Aktionsblock setzt die Werte image_xscale und image_yscale für die Instanz. Die Eingabewerte hier sind Modifikatoren, die auf die sprite angewendet werden, die der Instanz zugewiesen ist, wobei eine Skalierung von 1 (der Standardwert) bedeutet, dass keine Skalierung (1:1), kleinere Werte werden nach unten skaliert (0,5 zum Beispiel halbiert die Breite von sprite), und größere Werte werden nach oben skaliert. Sie können negative Werte verwenden, um die sprite zu spiegeln und zu skalieren, es sei denn, der verwendete Wert ist genau -1 (in diesem Fall wird die sprite nur gespiegelt oder um seinen Ursprung gedreht, ohne Skalierung). Sie können die horizontale oder vertikale Skalierung relativ zu den aktuellen Werten festlegen. In diesem Fall addieren oder subtrahieren Sie den neuen Wert von den aktuellen Skalenwerten.

Beachten Sie, dass eine Änderung dieser Werte auch die Art und Weise ändert, wie die Instanz Kollisionen erkennt, es sei denn, Sie haben im Objekteditor eine separate mask_index (Kollisionsmaske) angegeben. Bitte beachte, dass damit Änderungen in dieser Variablen sichtbar werden, sollte die Instanz entweder kein Draw-Ereignis haben (so dass GameMaker standardmäßig die sprite zeichnet) oder mit der Aktion Draw Self gezeichnet werden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Horizontal Der zu verwendende vertikale Skalierungsfaktor (Standard ist 1).
Vertical Der zu verwendende horizontale Skalierungsfaktor (Standard ist 1).
