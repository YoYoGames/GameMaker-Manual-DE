Objektinstanz ändern

Mit dieser Aktion können Sie eine Instanz in eine andere Instanz umwandeln. Sie müssen das Objekt angeben, das die Instanz in eine andere Instanz verwandeln soll, und die Aktion ändert die Instanz, indem sie das Zerstörungsereignis und das Aufräumereignis der der ursprünglichen Instanz und das Erstellungsereignis der neuen Instanz. Beachte, dass, wenn du keine Richtung oder Geschwindigkeit etc. angibst, die neue Instanz diese Werte "erben" wird. Wenn zum Beispiel die ursprüngliche Instanz eine image_angle (Rotation) hat von 180, bewegt sich mit 2px pro Spieltick und hat eine image_speed (Animationsgeschwindigkeit) von 0,5, dann wird die Instanz, in die sie wechselt, diese Werte beibehalten, es sei denn, sie wird explizit im Create-Event der neuen Instanz festgelegt.

Es ist erwähnenswert, dass das Ändern der Instanz bedeutet, dass du bis zum nächsten Spieltick keine weiteren Aktionen mit dieser Instanz durchführen kannst, insbesondere der Versuch, auf Variablen etc. zuzugreifen, da dies einen Fehler verursacht. Grundsätzlich gilt, dass du die Instanz, aber sie ist bis zum Ende des aktuellen Schritts nicht verfügbar, so dass ein direkter Zugriff auf eine der darin enthaltenen Variablen (z. B. durch den Aufruf von inst_changed.x) nicht funktioniert.
