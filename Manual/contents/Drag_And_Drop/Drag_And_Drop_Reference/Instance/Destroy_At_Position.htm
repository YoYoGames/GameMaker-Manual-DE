Zerstören an Position

Dieser Aktionsblock zerstört alle Instanzen, die an dem angegebenen Punkt gefunden wurden, indem er das Destroy-Ereignis und dann das Clean-Up-Ereignis für alle gefundenen Instanzen aufruft. Beachte, dass diese Aktion auf der Kollisionsmaske der Instanzen basiert, und wenn irgendein Teil der Maske mit dem Zielpunkt überlappt, dann wird die Funktion diese Instanz zerstören (wenn die Instanzen keine sprite oder Maske angewendet haben, dann werden sie nicht zerstört).

Aktion Syntax:

Argumente:

Argument Beschreibung
x Die zu verwendende x-Position im Raum.
y Die zu verwendende y-Position im Raum.

Beispiel:

Der obige Aktionsblock code prüft, ob eine Taste (die Leertaste) gedrückt wurde, und wenn dies der Fall ist wird er alle Instanzen zerstören, die sich an der relativen Position der Instanz befinden.
