Anzahl der Instanzen abrufen

Mit dieser Aktion können Sie die Anzahl der Instanzen eines bestimmten Objekts ermitteln, das derzeit im Raum aktiv ist. Sie können ein beliebiges Objekt aus der Ressourcenliste auswählen oder Sie können das spezielle Schlüsselwort all verwenden, um um die Anzahl aller aktiven Instanzen zu ermitteln. Sie müssen auch eine Zielvariable angeben, an die der Wert zurückgegeben wird, damit Sie ihn in weiteren Aktionen verwenden können. Wenn Sie die Zielvariable als "Temp" kennzeichnen, wird eine neue temporäre (lokale) Variable, die den Rückgabewert für Sie speichert, wie im folgenden Beispiel gezeigt.

Aktion Syntax:

Argumente:

Argument Beschreibung
Object Der Name des Objekts, auf das geprüft werden soll.
Target Die Variable, die als Ziel für den Rückgabewert dienen soll.

Beispiel:

Der obige Aktionsblock code erstellt eine lokale (temporäre) Variable und weist ihr dann die Anzahl der Instanzen des Objekts "obj_Eichhörnchen" zu. Dieser Wert wird dann überprüft und die Instanz wird zerstört, wenn die Anzahl der Instanzen größer als 100 ist.
