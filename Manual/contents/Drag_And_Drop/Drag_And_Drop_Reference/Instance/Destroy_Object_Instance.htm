Objektinstanz zerstören

Sie können diese Aktion immer dann aufrufen, wenn Sie eine Instanz "zerstören" wollen, wodurch sie aus dem Raum entfernt wird. Wenn du eine Instanz zerstörst, wird ihr Zerstörungsereignis unmittelbar nach der Aktion, die die Zerstörung aufruft, aufgerufen. Beachte auch, dass obwohl das Zerstörungsereignis in dem Moment ausgeführt wird, in dem die Aktion aufgerufen wird, die Instanz nicht sofort aus dem Spiel entfernt wird, sondern weiterhin Die Instanz führt die code, die im aktuellen Ereignis enthalten ist, bis zum Ende aus. Erst wenn das aktuelle Ereignis zu Ende ist, wird die Instanz aus dem Spiel entfernt, woraufhin das Ereignis Auf räumen aufgerufen wird.
