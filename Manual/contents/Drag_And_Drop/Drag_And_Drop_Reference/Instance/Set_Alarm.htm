Alarm-Countdown einstellen

Dieser Aktionsblock setzt einen bestimmten Alarm auf einen Countdown, und wenn der Alarm Null erreicht, wird das zugehörige Alarmereignis ausgelöst. Sie geben die Alarmnummer von 0 bis 11 und dann den Wert, auf den der Alarm gesetzt werden soll. Der verwendete Wert ist die Anzahl der "Spielschritte", die vergehen sollen, bevor das Alarmereignis ausgelöst wird, und Sie können diesen Wert relativ zum aktuellen Alarmwert einstellen (addierend oder subtrahierend zum aktuellen Wert). Beachten Sie, dass der Wert eine ganze Zahl sein muss, und Sie können ihn auf -1 setzen, um den Alarm zu stoppen. Beachten Sie, dass bei einem abwärtszählenden Alarm das Ereignis ausgeführt wird, wenn es 0 erreicht, der Alarm dann aber auf -1 herunterzählt, um anzuzeigen, dass er beendet ist. Das Setzen eines Alarm auf 0 zu setzen, führt nicht zur Ausführung des Ereignisses, da er im nächsten Schritt auf -1 herunterzählt. Um einen Alarm sofort auszulösen, sollte er also auf 1 gesetzt werden. Beachten Sie, dass Sie weitere Alarme in derselben Aktion einstellen können, indem Sie auf das Plus-Symbol neben der Aktion klicken.

Aktion Syntax:

Argumente:

Argument Beschreibung
Alarm Der zu setzende Alarm (das entsprechende Alarmereignis wird ausgelöst, wenn der Wert 0 erreicht).
Countdown Der Countdown-Wert in Spielschritten.
