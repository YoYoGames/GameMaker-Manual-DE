Datei löschen

Mit dieser Aktion können Sie eine Datei löschen, die zuvor erstellt wurde. Sie geben den Dateinamen (als String) der zu löschenden Datei an. Beachten Sie, dass Sie keine Dateien löschen können, die im Spiel-Bundle enthalten sind, sondern nur solche, die die mit den Aktionen Ini schließen oder Puffer speichern erstellt wurden (siehe den Abschnitt Das Dateisystem für weitere Informationen).

Aktion Syntax:

Argumente:

Argument Beschreibung
Filename Der Name (als String) der zu löschenden Datei

Der obige Aktionsblock code prüft, ob die Datei "checkpoint.sav" existiert, und wenn dann wird geprüft, ob die Datei "checkpoint_OLD.sav" vorhanden ist. Wenn diese Datei ebenfalls existiert, wird sie gelöscht, und die ursprüngliche Datei "checkpoint.sav" wird in "checkpoint_OLD.sav" umbenannt. Schließlich wird ein Puffer als "checkpoint.sav" gespeichert (im Wesentlichen wird bei jedem Speichern eine Datei buffer gesichert, so dass es immer eine "aktuelle" und eine "alte" Speicherdatei gibt).
