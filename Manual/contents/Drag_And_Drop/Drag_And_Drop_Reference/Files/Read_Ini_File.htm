Ini-Datei lesen

Mit dieser Aktion können Sie Daten aus einer Ini-Datei lesen. Sie müssen die Ini-Datei zuvor mit der Aktion Ini-Datei öffnen zum Lesen/Schreiben geöffnet haben, und dann geben Sie mit dieser Aktion den Typ der zu lesenden Daten (entweder eine reelle Zahl oder eine in "" eingeschlossene Zeichenkette) sowie die Abschnittsüberschrift und den Schlüssel innerhalb dieses Abschnitts, von dem die Daten abgerufen werden sollen. Sie können auch einen Standardwert angeben, der entweder eine reelle Zahl oder eine Zeichenkette sein kann, und der Typ sollte mit dem in der Aktion eingestellten "Typ" übereinstimmen. Diese Standardaktion wird von der Aktion zurückgegeben, wenn einer der folgenden Fälle eintritt:

  • Die Ini-Datei, die Sie geöffnet haben, existiert nicht
  • Die Abschnittsüberschrift, auf die Sie verweisen, existiert nicht
  • Der Schlüssel, von dem Sie versuchen, Daten abzurufen, existiert nicht

In jedem der oben genannten Fälle erhalten Sie den Standardwert zurück und nicht einen Wert aus der Ini-Datei. Der zurückgegebene Wert wird in der von Ihnen angegebenen Zielvariablen gespeichert, die auch in diesem Moment erstellt werden kann, wenn sie als als temporäre lokale Variable gekennzeichnet ist.

WICHTIG! Der Versuch, Daten aus einer Ini-Datei zu lesen, die zuvor nicht mit der Aktion Ini-Datei öffnen geöffnet wurde, führt zu einem schwerwiegenden Fehler und zum Absturz des Spiels.

Aktion Syntax:

Argumente:

Argument Beschreibung
Type Der Typ der zu lesenden Daten, entweder ein string (in Anführungszeichen ""), oder ein real number
Section Der Abschnitt der Ini-Datei, aus dem gelesen werden soll (als String)
Key Der Schlüssel innerhalb des Abschnitts, aus dem gelesen werden soll (als String)
Default Der Standardwert, der zurückgegeben wird, wenn entweder die Datei, der Abschnitt oder der Schlüssel nicht existiert
Target Die Zielvariable, die den Rückgabewert der Aktion enthält

Beispiel:

Der obige Aktionsblock code öffnet eine Ini-Datei zum Lesen und verwendet dann die daraus gelesenen Daten zum Setzen von 3 verschiedene globale Variablen, bevor die Datei wieder geschlossen wird.
