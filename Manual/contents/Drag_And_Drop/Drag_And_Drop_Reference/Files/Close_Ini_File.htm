Ini-Datei schließen

Mit dieser Aktion wird die aktuell geöffnete Ini-Datei geschlossen. Diese Aktion sollte aufgerufen werden, sobald Sie eine geöffnete Ini-Datei gelesen oder beschrieben haben (Ini-Dateien werden mit der Aktion Ini-Datei öffnen geöffnet). Wenn Sie diese Aktion nicht verwenden, nachdem Sie die Aktion In Ini-Datei schreiben verwendet haben, wird nichts auf die Festplatte geschrieben, da die Dateiinformationen im Speicher gehalten werden, bis diese Aktion aufgerufen wird, woraufhin sie aus dem Speicher und in die Datei auf der Festplatte geschrieben. Beachten Sie auch, dass Sie einen Fehler erhalten, wenn Sie versuchen, eine Ini-Datei zu öffnen, ohne zuvor eine andere (oder dieselbe) geschlossen zu haben.
