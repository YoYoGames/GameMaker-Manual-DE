Datei kopieren

Mit dieser Aktion können Sie eine beliebige Datei kopieren, die von Ihrem Spiel gespeichert wurde oder als Teil der enthaltenen Dateien gebündelt wurde. Sie geben den Namen der zu kopierenden Datei (als String) und dann den Namen der Datei an, die als Kopie der Originaldatei erstellt werden soll (ebenfalls als Zeichenkette und einschließlich der Dateierweiterung). Die angegebene Datei wird dupliziert und erhält den neuen Namen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Filename Der Name (als String) der zu kopierenden Datei
Copy To Der Name (als String) der Datei, die als Kopie erstellt werden soll

Beispiel:

Der obige Aktionsblock code prüft, ob eine ini-Datei nicht vorhanden ist, und wenn dies nicht der Fall ist dann wird sie durch Kopieren einer anderen Datei erstellt.
