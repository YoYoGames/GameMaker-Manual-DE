Puffer laden

Mit dieser Aktion können Sie eine buffer laden, die zuvor mit der Aktion Puffer speichern gespeichert wurde. Die Aktion erstellt eine neue buffer, lädt die Datendatei in diese und gibt den eindeutigen ID-Wert zur Identifizierung der buffer an die Variable zurück. Sie geben den Namen der zu ladenden Datei (als String) und eine Variable an, in der die zurückgegebene buffer ID gespeichert werden soll. Beachten Sie, dass Sie, wenn Sie die Variable als temporäre lokale Variable kennzeichnen, sicherstellen müssen, dass Sie die von dieser Funktion erstellte buffer löschen, bevor vor dem Ende der Aktion script oder des Ereignisses löschen, da sonst der Verweis auf die Variable verloren geht und ein Speicherleck entsteht, wenn die Variable verworfen wird.

Aktion Syntax:

Argumente:

Argument Beschreibung
Filename Der Name (als String) der zu öffnenden Datei
Target Die Zielvariable, die den eindeutigen ID-Wert für die beim Laden erstellte buffer enthält.

Beispiel:

Der obige Aktionsblock code erstellt eine Variable, die die Daten von buffer enthält, und prüft dann, ob eine buffer Datei bereits gespeichert worden ist. Existiert die Datei, wird sie in die von uns erstellte Variable buffer geladen. Existiert sie jedoch nicht, wird sie erstellt, beschrieben und dann gespeichert, um beim nächsten Mal geladen zu werden.
