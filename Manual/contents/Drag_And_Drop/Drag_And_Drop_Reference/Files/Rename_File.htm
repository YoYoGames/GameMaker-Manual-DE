Datei umbenennen

Mit dieser Aktion können Sie jede Datei umbenennen, die von Ihrem Spiel gespeichert wurde. Sie geben den Namen der zu ändernden Datei (als Zeichenkette) und dann den neuen Namen (ebenfalls als Zeichenkette und einschließlich der Dateierweiterung) an, und die Datei wird umbenannt. Beachten Sie, dass Sie keine Dateien umbenennen können, die als mitgelieferte Datei in Ihrem Spiel enthalten sind (weitere Informationen finden Sie im Abschnitt über das Dateisystem ).

Aktion Syntax:

Argumente:

Argument Beschreibung
Filename Der Name (als String) der umzubenennenden Datei
New Name Der neue Name (als String) für die Datei

Beispiel:

Der obige Aktionsblock code prüft, ob die Datei "checkpoint.sav" existiert, und wenn dann wird geprüft, ob die Datei "checkpoint_OLD.sav" vorhanden ist. Wenn diese Datei ebenfalls existiert, wird sie gelöscht, und die ursprüngliche Datei "checkpoint.sav" wird in "checkpoint_OLD.sav" umbenannt. Schließlich wird ein Puffer als "checkpoint.sav" gespeichert (im Wesentlichen wird bei jedem Speichern eine Datei buffer gesichert, so dass es immer eine "aktuelle" und eine "alte" Speicherdatei gibt).
