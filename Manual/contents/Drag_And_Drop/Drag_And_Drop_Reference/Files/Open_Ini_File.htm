Ini-Datei öffnen

Mit dieser Aktion können Sie eine Ini-Datei zum Lesen und/oder Schreiben öffnen. Wenn die Ini-Datei an dem Ort, den du prüfst, nicht existiert, kann GameMaker eine erstellen, aber nur, wenn du Daten in sie schreibst.
