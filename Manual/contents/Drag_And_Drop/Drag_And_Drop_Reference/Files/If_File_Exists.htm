Wenn Datei vorhanden ist

Mit dieser Aktion können Sie prüfen, ob eine Datei mit dem angegebenen Namen existiert. Sie wählen die zu prüfende Datei aus (als Sting und einschließlich der Erweiterung), und die Funktion gibt true zurück, wenn sie existiert, und false, wenn sie nicht existiert. Beachten Sie, dass die Funktion true zurückgibt, wenn die Datei im Spiel-Bundle als Included File vorhanden ist oder wenn die Datei während des Spiels ausgelagert wurde (siehe den Abschnitt Das Dateisystem für weitere Informationen).

Beachten Sie, dass Aktionen, die Sie in den "if"-Block einfügen möchten, an der Seite der Aktion abgelegt werden müssen, wie in der folgenden Abbildung gezeigt:

Beispiel:

Der obige Aktionsblock code erstellt eine Variable für die Daten von buffer und prüft dann, ob eine buffer Speicherdatei bereits erstellt wurde. Existiert die Datei, wird sie in die von uns erstellte Variable buffer geladen. Existiert sie jedoch nicht, wird sie erstellt, beschrieben und dann gespeichert, um beim nächsten Mal geladen zu werden.
