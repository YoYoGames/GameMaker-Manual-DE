Puffer speichern

Mit dieser Aktion können Sie eine Datei speichern, die die Daten eines buffer enthält. Sie geben die eindeutige buffer ID an (wie sie beim Erstellen des buffer mit Create Buffer zurückgegeben wurde) und geben einen Namen für die Datei an (als String, mit der Erweiterung Ihrer Wahl) und die buffer wird in diese Datei geschrieben, die dann mit Load Buffer wieder geladen werden kann.

Aktion Syntax:

Argumente:

Argument Beschreibung
Buffer Der eindeutige buffer ID-Wert der zu speichernden buffer
Filename Der Name (als String) der zu speichernden Datei

Beispiel:

Der obige Aktionsblock code erstellt eine Variable, die die Daten von buffer enthält, und prüft dann, ob eine buffer Datei bereits gespeichert worden ist. Existiert die Datei, wird sie in die von uns erstellte Variable buffer geladen. Existiert sie jedoch nicht, wird sie erstellt, beschrieben und dann gespeichert, um beim nächsten Mal geladen zu werden.
