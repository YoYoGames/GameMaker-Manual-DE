Für

Die For-Aktion wird verwendet, um loops auszuführen, die nur beendet wird, wenn ein Wert die angegebene Bedingung erreicht. Sie geben einen Anfangswert als Variable an (die als temporäre lokale Variable gekennzeichnet werden kann - das ist normalerweise das, was Sie wollen), dann geben Sie die Bedingung, gegen die die Variable geprüft werden soll (z. B. ob sie unter einem bestimmten Wert liegt), und schließlich den Betrag, um den die Variable bei jeder Iteration der loop erhöht werden soll. Der allgemeine Ablauf ist also wie folgt:

  • Sie initialisieren eine Variable mit einem Wert
  • Sie legen die Bedingung fest, mit der diese Variable geprüft werden soll
  • Sie legen fest, um wie viel die Variable bei jeder Iteration der Schleife inkrementiert werden soll
  • Sie fügen dann Aktionen in die loop ein, die bei jeder Iteration ausgeführt werden, bis die Bedingung false zurückkehrt oder eine Break-Aktion auftritt.

Beim Hinzufügen von Aktionen, die innerhalb von loop ausgeführt werden sollen, werden diese ähnlich wie bei einem "Wenn" hinzugefügt, d. h., sie werden neben der Aktion und nicht unter ihr eingefügt:

Alle zur Seite hinzugefügten Aktionen werden in das Für aufgenommen und ausgeführt, bevor das Ereignis oder script fortgesetzt wird mit den darunter hinzugefügten Aktionen. Beachten Sie, dass Sie jede loop vorzeitig beenden können, wenn Sie die Aktion Break aufrufen, und wenn Sie For loops verwenden, müssen Sie nicht inkrementieren und können auch dekrementieren, solange die Bedingung die Sie verwenden, angemessen ist. Setzen Sie z. B. die Ausgangsvariable auf 10, setzen Sie die Bedingung so, dass auf weniger als 0 geprüft wird, und setzen Sie dann den Inkrementwert auf -1, und loop wird 10-mal rückwärts von 10 bis 0 zählen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Initialise Initialisierung der Variablen, die für die Zählung verwendet wird, auf einen Wert (kann als temporäre lokale Variable gekennzeichnet werden)
Condition Die zu prüfende Bedingung, um die Schleife zu unterbrechen
Increment Der Betrag, um den die anfängliche Zählvariable bei jeder Iteration der Schleife erhöht (oder erniedrigt) wird
