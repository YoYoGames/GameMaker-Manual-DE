Schleife

Die Loop-Aktion loop läuft unendlich lange, bis eine Break-Aktion aufgerufen wird. Bei dieser Aktion sollte darauf geachtet werden, dass unabhängig von den Aktionen, die innerhalb von loop platziert sind, immer ein Break aufgerufen wird, da sonst andernfalls wird das Spiel einen "not responding"-Fehler auf dem Zielbetriebssystem verursachen und nicht mehr funktionieren. Sie fügen diese Aktion in die Aktion code workspace ein und fügen dann die auszuführenden Aktionen innerhalb der loop hinzu, ähnlich wie Sie eine Aktion zu einem "Wenn" hinzufügen würden, d.h.
