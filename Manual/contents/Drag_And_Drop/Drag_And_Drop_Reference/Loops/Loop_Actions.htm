Schleifen-Aktionsbibliothek

Die Loop-Aktionsbibliothek enthält die Aktionen, die für die Erstellung von loops in Ihrer Spielaktion erforderlich sind. code. Eine loop ist einfach eine Möglichkeit, dieselbe Reihe von Aktionen 1 oder mehrere Male in Abhängigkeit von bestimmten Bedingungen zu wiederholen und ist in jeder Situation nützlich, in der Sie dieselben Aktionen immer wieder wiederholen müssen.

Die Loop-Bibliothek verfügt über die folgenden Aktionen:

Schleife
Wiederholen Sie
Während
Für
Pause
