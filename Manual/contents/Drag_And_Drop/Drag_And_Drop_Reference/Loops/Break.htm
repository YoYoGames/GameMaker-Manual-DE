Pause

Diese Aktion kann nur innerhalb einer loop Aktion (z.B. Repeat oder While) verwendet werden, und der Aufruf dieser Aktion bricht in diesem Moment aus loop aus. Normalerweise läuft eine loop entweder eine feste Anzahl von Malen oder bis eine bestimmte Bedingung erfüllt ist, aber Sie können die Aktion Pause verwenden, um eine loop jederzeit vorzeitig zu beenden (Aktionen innerhalb der loop nach der Pause werden nicht ausgeführt).
