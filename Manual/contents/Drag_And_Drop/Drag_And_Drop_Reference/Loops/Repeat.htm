Wiederholen Sie

Diese Aktion wiederholt alle ihr hinzugefügten Aktionen in der angegebenen Häufigkeit. Sie geben an, wie oft die Aktionen wiederholt werden sollen, und fügen dann die Aktion zu der Wiederholung loop hinzu, ähnlich wie Sie eine Aktion zu einem "Wenn" hinzufügen würden, d. h., Sie fügen neben der Aktion und nicht unter ihr:

Alle zur Seite hinzugefügten Aktionen werden in die Wiederholung einbezogen und ausgeführt, bevor das Ereignis oder Skript mit den darunter hinzugefügten Aktionen fortfährt (wie im folgenden Beispiel gezeigt). Beachten Sie, dass Sie jede loop vorzeitig beenden können, wenn Sie die Aktion " Pause " aufrufen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Times Die Anzahl der Wiederholungen (ein ganzzahliger Wert)

Beispiel:

Der obige Aktionsblock code ermittelt die Anzahl der Einträge in einer Liste und speichert den Wert in einer temporären lokalen Variablen. Diese Variable wird dann verwendet, um die Anzahl der Wiederholungen von loop festzulegen. In der Wiederholung loop werden die Einträge aus der Liste in eine andere temporäre Variable gelesen, die dann verwendet wird, um die Instanzen, auf die sie verweisen, zu zerstören. Nachdem die Schleife beendet ist, wird die Datenstruktur freigegeben.
