Set Sequence Kopf

Mit dieser Aktion können Sie die Abspielposition des angegebenen sequence Elements im Raum auf ein bestimmtes Bild setzen. Sie geben die sequence Element-ID (wie von der Aktion Sequenz erstellen zurückgegeben) und die neue Position des Abspielkopfes in Frames. Informationen über den Abspielkopf finden Sie unter Verwendung des Dope Sheets.

Beachten Sie, dass die Position in Frames angegeben wird. Wenn Sie einen Wert eingeben, der größer als die Gesamtzahl der Frames (oder kleiner als 0) ist, hängt die tatsächliche Endposition des Abspielkopfes von der Art der sequence Wiedergabe ab, die ausgewählt wurde, nach diesen Regeln:

  • Keine Schleifenbildung: Der Abspielkopf wird am Ende von loop (oder am Anfang, wenn der angegebene Wert negativ ist) festgehalten.
  • Schleifen: Der Abspielkopf springt zurück zum Anfang von sequence plus die zusätzlichen Frames, z. B.: Wenn Sie den Abspielkopf auf einem sequence mit 30 Frames auf eine Position von Frame 45 setzen, liegt die tatsächliche Abspielkopfposition bei 15. Wenn Sie einen einen negativen Wert angeben, wird der sequence Abspielkopf in umgekehrter Richtung umlaufen.
  • Ping-Pong-Schleife: Der Abspielkopf bewegt sich bis zum Ende vor und geht dann um die zusätzliche Anzahl von Bildern zurück, die über die gesamte Bildlänge hinausgehen, wobei die Wiedergaberichtung umgekehrt wird, z. B.: Wenn die sequence 30 Frames hat und Sie den Abspielkopf auf auf 40 setzen, ist die endgültige Abspielkopfposition 20 und die Richtung wird umgekehrt (nach unten bis 0). Beachten Sie, dass Sie niemals einen negativen Wert für die Ping-Pong-Wiedergabe eines sequence angeben sollten, da dies zu undefinierten Ergebnisse haben kann.

Wenn Sie die Kopfposition auf diese Weise einstellen, wird die Wiedergabe nicht gestoppt, sondern sequence wird an der neuen Abspielkopfposition fortgesetzt, es sei denn, die Wiedergabe wird angehalten.

Aktion Syntax:

Argumente:

Argument Beschreibung
Variable Die ID des Elements sequence, dessen Kopfposition geändert werden soll.
Head (frames) Die neue Kopfposition (in Frames) für das Element sequence.
