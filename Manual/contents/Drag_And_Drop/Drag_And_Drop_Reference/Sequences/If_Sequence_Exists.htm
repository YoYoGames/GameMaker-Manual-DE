Wenn Sequence vorhanden ist

Mit dieser Aktion kann geprüft werden, ob das angegebene Element sequence im Spielraum vorhanden ist. Sie geben die Element-ID an, die von der Aktion Sequenz erstellen zurückgegeben wird, und wenn das Element sequence im Raum existiert, dann gibt die Aktion true zurück, andernfalls false.

Wenn Sie das Feld "Not" argument markieren, prüft die Aktion, ob es kein solches sequence Element gibt, und wenn keines gefunden wird, wird true zurückgegeben.

Beachten Sie, dass Aktionen, die Sie in den "if" Block einfügen möchten, an der Seite der Aktion abgelegt werden müssen, wie in der folgenden Abbildung gezeigt:

Diese Aktionen werden nun ausgeführt, wenn der "if" zu true ausgewertet wird, während alle Aktionen, die an anderer Stelle ausgelassen werden, nach dem Block "if" ausgeführt werden.
