Sequence Kopf bekommen

Mit dieser Aktion können Sie die aktuelle Position des Abspielkopfes des angegebenen sequence Elements im Raum abrufen. Sie geben die sequence Element-ID an (wie von der Aktion Sequenz erstellen zurückgegeben), und diese Aktion gibt das aktuelle Bild, auf dem sich der Abspielkopf der Sequenz befindet, in der Zielvariablen zurück. Informationen über den Abspielkopf finden Sie unter Verwendung des Dope Sheets.

Beachten Sie, dass die Zielvariable als temporär (lokal) gekennzeichnet werden kann. Sie wird dann für die Aktion erstellt und kann in allen nachfolgenden Aktionen für das Ereignis, in dem die Aktion stattfindet, verwendet werden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Variable Die ID des Elements sequence, dessen Kopfposition ermittelt werden soll.
Target Die Zielvariable zum Halten der Kopfposition.

Beispiel:

Der obige Aktionsblock ruft die aktuelle Kopfposition der Element-ID von sequence ab, die in der der Variablen "seq_id" gespeichert ist. Dieser Wert wird dann verwendet, um die Kopfposition desselben sequence -Elements zu setzen und um 4 Frames zu verschieben.
