Sequence Position einstellen

Mit dieser Aktion können Sie die Position für das angegebene sequence Element im Raum festlegen. Sie geben die sequence Element-ID (wie von der Aktion Create Sequence zurückgegeben) und dann die neue X- und Y-Position an. Wenn Sie das Kontrollkästchen "relativ" ankreuzen, werden die angegebenen Werte für X und Y zur aktuellen sequence Position addiert, andernfalls handelt es sich um eine absolute Position.

Aktion Syntax:

Argumente:

Argument Beschreibung
Variable Die ID des Elements sequence, dessen Position festgelegt werden soll.
X Die neue X-Position im Raum für die sequence.
Y Die neue y-Position im Raum für die sequence.

Beispiel:

Der obige Aktionsblock code ruft die aktuelle Position des Elements sequence ab ID, die in der Variablen "my_seq" gespeichert ist. Diese Werte werden dann verwendet, um die Position von sequence um 10 Pixel nach rechts zu verschieben.
