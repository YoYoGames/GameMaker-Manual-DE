Sequenz erstellen

Mit dieser Aktion können Sie ein sequence Element eines beliebigen sequence aus dem Asset-Browser erstellen. Sie geben den Namen des sequence und eine x/y-Position innerhalb des Raums an, in dem es erstellt werden soll, sowie die Ebene, auf der es erstellt werden soll (diese Ebene muss eine Asset-Ebene sein). Sie können das Häkchen bei "relativ" flags setzen, um die sequence relativ zu der Instanz zu positionieren, die die code Aktionsblöcke ausführt. Wenn du ein sequence im Spiel auf diese Weise erstellst, erhält es einen eindeutigen Element-ID-Wert, mit dem du das sequence in weiteren Aktionen identifizieren kannst. Daher erlaubt dir diese Aktion, eine "Ziel"-Variable zu definieren, die diesen ID-Wert enthält (wie im Beispiel unten gezeigt). Beachten Sie, dass das Ziel als temporäre (lokale) Variable gekennzeichnet werden kann, die für die Aktion erstellt wird und in allen nachfolgenden Aktionen für das Ereignis, in dem sich die Aktion befindet, verwendet werden kann.

Aktion Syntax:

Argumente:

Argument Beschreibung
Sequence Der Name des sequence, von dem ein Element erstellt werden soll.
x Die x-Position im Raum, um die sequence zu erstellen (kann relativ zur aufrufenden Instanz sein).
y Die y-Position im Raum, um die sequence zu erstellen (kann relativ zur aufrufenden Instanz sein).
layer Der Name (eine Zeichenkette) der Asset-Ebene, auf der die sequence erstellt werden soll.
Target Die Variable, die als Ziel für den Rückgabewert dienen soll.

Beispiel:

Der obige Aktionsblock code erstellt ein neues sequence und speichert dessen eindeutigen Element-ID-Wert in einer Variablen. Diese Variable wird dann verwendet, um die sequence mit der Aktion "Pause Sequence " anzuhalten.
