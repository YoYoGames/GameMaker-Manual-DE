Sequence Position abrufen

Mit dieser Aktion können Sie die aktuelle Position für das angegebene sequence Element im Raum abrufen. Sie geben die sequence Element-ID an (wie von der Aktion Sequenz erstellen zurückgegeben), und diese Aktion gibt seine X- und Y-Position in Zielvariablen zurück. Beachten Sie, dass die Zielvariablen als temporäre (lokale) Variablen gekennzeichnet werden können, die für die Aktion erstellt werden und in allen nachfolgenden Aktionen für das Ereignis, in dem sich die Aktion befindet, verwendet werden können.

Aktion Syntax:

Argumente:

Argument Beschreibung
Variable Die ID des Elements sequence, dessen Position ermittelt werden soll.
Target X Die Zielvariable zur Aufnahme der X-Position.
Target Y Die Zielvariable zur Aufnahme der y-Position.

Beispiel:

Der obige Aktionsblock code ruft die aktuelle Position des Elements sequence ab ID, die in der Variablen "my_seq" gespeichert ist. Diese Werte werden dann verwendet, um die Position von sequence um 10 Pixel nach rechts zu verschieben.
