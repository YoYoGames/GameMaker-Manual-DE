Audio-Lautstärke einstellen

Mit dieser Aktion können Sie die Lautstärke eines bestimmten Tons einstellen. Sie wählen den Sound aus dem Asset-Explorer aus und stellen dann den Lautstärkewert ein. Die Lautstärke kann zwischen 0 (stumm) und 1 (volle Lautstärke) liegen und die Skala ist linear, so dass ein Wert von 0,5 der halben Lautstärke entsprechen würde. Beachten Sie, dass sich diese Aktion auf alle Instanzen des Sounds auswirkt, die derzeit im Raum abgespielt werden, und auch auf alle zukünftigen Instanzen des Sounds.

Das Feld "Zeit" bestimmt, wie lange es dauert, bis die Lautstärkeänderung wirksam wird. Ein Wert von 0 führt zu einer sofortigen Änderung, während ein größerer Wert die Lautstärke schrittweise über die angegebene Anzahl von Millisekunden ändert.

Aktion Syntax:

Argumente:

Argument Beschreibung
Sound Die Klangressource zur Einstellung der Lautstärke von
Volume Die neue Lautstärke für den Ton (von 0 bis 1, Standardwert 1)
Time Die Zeitspanne, über die die Lautstärke ausgeblendet werden soll (in Millisekunden).

Beispiel:

Der obige Aktionsblock code prüft eine globale Variable, um festzustellen, ob sie true oder false ist. Wenn sie true ist, wird die Lautstärke des angegebenen Tons abgerufen und in einer temporären lokalen Variablen gespeichert. Anschließend wird geprüft, ob der Wert ungleich 0,5 ist. Ist dies nicht der Fall, wird die Lautstärke des Geräuschs in 1000 Millisekunden (1 Sekunde) auf 0,5 heruntergefahren. Wenn die globale Variable den Wert false hat, wird die Lautstärke des Tons auf 1 gesetzt. Da der Wert für die Zeit hier auf 0 gesetzt ist, erfolgt die Änderung der Lautstärke sofort.
