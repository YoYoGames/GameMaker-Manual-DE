Lebenslauf Audio

Diese Aktion kann verwendet werden, um einen zuvor angehaltenen Sound wieder aufzunehmen. Sie kann für jeden Sound verwendet werden, der mit "Audio anhalten" oder "Alle Audios anhalten" angehalten wurde, und hat keine Auswirkung auf die Sounds die zuvor nicht angehalten wurden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Sound Die Klangressource wird wieder abgespielt

Beispiel:

Der obige Aktionsblock code prüft eine globale Variable, und wenn sie den Wert true hat, wird wird der angegebene Ton angehalten, andernfalls wird er fortgesetzt.
