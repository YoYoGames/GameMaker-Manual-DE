Master-Lautstärke einstellen

Mit dieser Aktion können Sie die Lautstärke für das gesamte Soundsystem einstellen. Der Lautstärkewert kann zwischen 0 (stumm) und 1 (volle Lautstärke) liegen. Die Skala ist linear, so dass ein Wert von 0,5 der halben Lautstärke entspricht. Beachten Sie, dass sich diese Aktion auf alle Instanzen aller Klänge, die gerade im Raum gespielt werden, und alle zukünftigen Instanzen aller gespielten Klänge beeinflussen. Diese Aktion ist auch unabhängig von der eingestellten Lautstärke für jeden einzelnen Sound. Wenn Sie also einen Sound auf eine Lautstärke von 0,5 eingestellt haben eingestellt haben und dann die Master-Lautstärke auf 0,5 setzen, wird der Sound mit einer Lautstärke von 0,25 abgespielt.

Argument Beschreibung

Beispiel:

Der obige Aktionsblock code prüft eine globale Variable, um festzustellen, ob sie true ist oder falsch ist. Wenn sie true ist, wird die Hauptlautstärke für das Soundsystem abgerufen und in einer temporären lokalen Variablen gespeichert. Anschließend wird geprüft, ob der Wert ungleich 0,5 ist, und wenn dies nicht der Fall ist, wird die Grundlautstärke des Soundsystems auf 0,5 gesetzt. Wenn die globale Variable den Wert false ergibt, wird die Gesamtlautstärke auf 1 gesetzt.
