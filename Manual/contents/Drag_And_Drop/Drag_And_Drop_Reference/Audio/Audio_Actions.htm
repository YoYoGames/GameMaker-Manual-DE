Audio-Aktionsbibliothek

GameMaker verfügt über eine komplette Audio-Engine, die auf den Formaten *.ogg, *.mp3 und *.wav Soundformaten basiert. Klänge dieser Formate, die zu IDE hinzugefügt werden, können dann in Ihrem Spiel mit den unten gezeigten grundlegenden Audioaktionen verwendet werden. Für Dinge, die komplexer sind als einfache Soundeffekte oder das Abspielen eines einzelnen können Sie die erweiterten Audiofunktionen nutzen, die sich mit dem Streaming von Audio, der Positionierung von Audio - um 3D-Sound zu erzeugen - und der Gruppierung von Audio befassen.

WICHTIG! Wenn Sie Audio auf dem HTML5-Ziel verwenden, sollten Sie sich bewusst sein, dass nicht alle Browser Web Audio unterstützen und daher möglicherweise keinen Ton für Ihr Projekt abspielen, wenn es ausgeführt wird. Eine allgemeine Vorstellung von der Web Audio-Unterstützung erhalten Sie unter folgendem Link: Kann ich Web Audio verwenden?
