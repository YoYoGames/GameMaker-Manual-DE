Audiolänge abrufen

Diese Aktion kann verwendet werden, um die Länge eines Tons in Ihrem Spiel zu ermitteln. Sie geben die Sound-Ressource an, deren Länge ermittelt werden soll, und eine Zielvariable, die den Wert zurückgibt (oder Sie können das Ziel als temporäre Variable kennzeichnen, damit die Aktion die Variable für die Verwendung bis zum Ende des Ereignisses oder script). Die Länge wird an die Zielvariable (die temporär sein kann) in Sekunden zurückgegeben und kann ein Dezimalwert sein.

Aktion Syntax:

Argumente:

Argument Beschreibung
Sound Die Klangressource zur Ermittlung der Länge von
Target Die Zielvariable zum Speichern des Rückgabewerts

Beispiel:

Der obige Aktionsblock code ermittelt die Abspieldauer der angegebenen Klangressource und speichert sie in einer temporären lokalen Variablen. Diese wird dann verwendet, um einen Zufallswert zwischen 0 und der Länge des Tons zu erzeugen, der dann in einer anderen temporären lokalen Variablen gespeichert wird. Dieser neue Zufallswert wird dann verwendet, um die Startposition für den Sound festzulegen und der Sound wird dann abgespielt.
