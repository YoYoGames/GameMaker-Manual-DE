Audio abspielen

Mit dieser Aktion wird der angegebene Sound abgespielt. Sie wählen den Sound aus dem Asset-Explorer aus und legen dann fest, ob der Sound in einer Schleife abgespielt werden soll oder nicht (ein geloopter Sound wird immer wieder abgespielt, bis er gestoppt wird), indem Sie das Flag "Loop" setzen.

Zusätzlich können Sie die Verstärkung, den Offset (die Position, an der das Abspielen beginnt) und die Tonhöhe einstellen, mit der der Sound abgespielt werden soll.

Der endgültige Wert für Verstärkung und Tonhöhe des abgespielten Tons ist das Produkt aus den für das Tonelement eingestellten Werten für Verstärkung und Tonhöhe. Der Offset-Wert ist der im Feld "Offset" angegebene Wert oder der Offset-Wert des abgespielten Tondokuments, wenn das Feld leer bleibt (siehe Beispiel 2).

Aktion Syntax:

Argumente:

Argument Beschreibung
Sound Das abzuspielende Tongut
Loop Ob der Ton loop sein soll oder nicht (standardmäßig ausgeschaltet)
Target Die Zielvariable zum Speichern des Rückgabewerts
Gain Wert für den Verstärkungsmultiplikator (Standardwert 1). Der hier eingestellte Verstärkungsmultiplikator wird mit dem Verstärkungsmultiplikator des Sound Assets multipliziert
Offset Die Zeit (in Sekunden), auf die der Startpunkt gesetzt werden soll (Standardwert 0). Werte, die länger als die Länge des angegebenen Tons sind, werden ignoriert.
Wird dieses Feld leer gelassen, wird der Offset (Spurposition) des Tondokuments verwendet.
Pitch Der Tonhöhenmultiplikator (Standardwert 1). Der hier eingestellte Verstärkungsmultiplikator wird mit dem Verstärkungsmultiplikator des Sound Assets multipliziert

Beispiel 1:

Der obige Aktionsblock code prüft, ob eine Maustaste gedrückt wurde, und wenn dies der Fall ist, wird eine globale Variable daraufhin überprüft, ob sie nicht true lautet. Ist dies nicht der Fall true, wird ein Ton abgespielt (und in einer Schleife wiederholt) und die globale Variable auf true gesetzt, andernfalls wird der Ton gestoppt und die globale Variable auf false gesetzt.
Der Ton wird mit den Standardeinstellungen abgespielt: Verstärkung und Tonhöhenmultiplikator von 1 und ein Offset von 0 (der Standard-Offset des Sound-Assets).

Beispiel 2:

Der obige Aktionsblock code setzt zunächst die Spurposition des Sound-Assets "snd_MusicBoss" auf 20 Sekunden. Dann prüft er, ob die Instanzvariable "skip_intro" auf true gesetzt ist. Ist dies der Fall, beginnt er mit der Wiedergabe von "snd_MusicBoss" bei 20 Sekunden, dem zuvor eingestellten Wert für die Spurposition (da das Feld "Offset" leer bleibt, wird der Asset-Offset verwendet). Die ersten 20 Sekunden des Audiomaterials werden übersprungen.
Wenn die Instanzvariable "skip_intro" auf false gesetzt wird, beginnt auch die Wiedergabe von "snd_MusicBoss", aber da der "Offset" auf 0 gesetzt wurde, wird der Ton von Anfang an abgespielt. Die Spurposition von 20 Sekunden, die zuvor für das Sound-Asset definiert wurde, wird ignoriert.
