Audio-Lautstärke abrufen

Diese Aktion gibt die Lautstärke des angegebenen Tons zurück. Sie wählen den Sound aus dem Asset-Explorer und geben eine Zielvariable an, an die der Lautstärkewert zurückgegeben werden soll (Sie können diese als temporäre lokale Variable kennzeichnen). Der zurückgegebene Wert liegt zwischen 0 und 1, wobei 0 stumm und 1 die volle Lautstärke bedeutet.

Aktion Syntax:

Argumente:

Argument Beschreibung
Sound Die Klangressource zum Abrufen der Lautstärke von
Target Die Zielvariable zum Speichern des zurückgegebenen Volumens

Beispiel:

Der obige Aktionsblock code prüft eine globale Variable, um festzustellen, ob sie true oder false ist. Wenn sie true ist, wird die Lautstärke des angegebenen Tons abgerufen und in einer temporären lokalen Variablen gespeichert. Anschließend wird geprüft, ob der Wert ungleich 0,5 ist. Ist dies nicht der Fall, wird die Lautstärke des Tons auf 0,5 gesetzt. Wenn die globale Variable den Wert false, dann wird die Lautstärke des Tons auf 1 gesetzt.
