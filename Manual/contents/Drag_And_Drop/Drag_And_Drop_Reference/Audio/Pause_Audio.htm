Audio anhalten

Diese Aktion kann verwendet werden, um den gewählten Sound anzuhalten, wenn er gerade abgespielt wird. Wenn Sie mehrere Instanzen desselben Tons abgespielt haben, werden alle angehalten, und es werden keine Töne angehalten, die nach dem Aufruf der Aktion aufgerufen wurde. Sie können einen angehaltenen Sound mit der Aktion Audio fortsetzen wieder abspielen lassen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Sound Die Klangquelle zum Anhalten

Beispiel:

Der obige Aktionsblock code prüft eine globale Variable, und wenn sie den Wert true hat, wird wird der angegebene Ton angehalten, andernfalls wird er fortgesetzt.
