Master-Lautstärke abrufen

Diese Aktion gibt die Gesamtlautstärke für das Soundsystem zurück. Sie geben eine Zielvariable an, an die der Lautstärkewert zurückgegeben werden soll (Sie können diese als temporäre lokale Variable kennzeichnen). Der zurückgegebene Wert liegt zwischen 0 und 1, wobei 0 für Stille und 1 für volle Lautstärke.

Aktion Syntax:

Argumente:

Argument Beschreibung
Target Die Zielvariable zum Speichern des zurückgegebenen Volumens

Beispiel:

Der obige Aktionsblock code prüft eine globale Variable, um festzustellen, ob sie true ist oder falsch ist. Wenn sie true ist, wird die Hauptlautstärke für das Soundsystem abgerufen und in einer temporären lokalen Variablen gespeichert. Anschließend wird geprüft, ob der Wert ungleich 0,5 ist, und wenn dies nicht der Fall ist, wird die Grundlautstärke des Soundsystems auf 0,5 gesetzt. Wenn die globale Variable den Wert false ergibt, wird die Gesamtlautstärke auf 1 gesetzt.
