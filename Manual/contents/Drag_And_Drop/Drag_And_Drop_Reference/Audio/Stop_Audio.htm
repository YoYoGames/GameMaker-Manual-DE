Audio anhalten

Diese Aktion kann verwendet werden, um die Wiedergabe eines Tons zu stoppen. Sie wählen den Sound aus dem Asset-Explorer aus und alle Instanzen dieses Sounds werden gestoppt (wenn Sie also denselben Sound in drei verschiedenen Ereignissen abspielen lassen, werden alle gestoppt).

Aktion Syntax:

Argumente:

Argument Beschreibung
Sound Anhalten der Klangquelle

Beispiel:

Der obige Aktionsblock code prüft, ob eine Maustaste gedrückt wurde, und wenn dies der Fall ist, wird eine globale Variable, um festzustellen, ob sie nicht wahr ist. Wenn dies nicht der Fall ist, wird ein Ton abgespielt (und in einer Schleife wiederholt) und die globale Variable auf true gesetzt. Andernfalls wird der Ton gestoppt und die globale Variable auf false gesetzt.
