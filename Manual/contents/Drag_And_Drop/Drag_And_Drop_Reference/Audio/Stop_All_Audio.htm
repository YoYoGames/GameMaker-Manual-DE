Alle Audios anhalten

Diese Aktion kann verwendet werden, um die gesamte Audiowiedergabe zu stoppen (beachten Sie, dass selbst wenn Audio als pausiert gekennzeichnet ist, es mit dieser Funktion gestoppt wird und Sie es später nicht wieder aufnehmen können).

Beispiel:

Der obige Aktionsblock code prüft, ob eine Maustaste gedrückt wurde, und wenn dies der Fall ist, prüft er eine globale Variable, um festzustellen, ob sie nicht wahr ist. Wenn dies nicht der Fall ist, wird ein Ton abgespielt und die globale Variable auf true gesetzt, andernfalls wird der Ton gestoppt und die globale Variable auf false gesetzt.
