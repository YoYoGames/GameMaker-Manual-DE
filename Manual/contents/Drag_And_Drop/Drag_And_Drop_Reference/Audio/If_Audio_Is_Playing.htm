Wenn Audio abgespielt wird

Diese Aktion kann verwendet werden, um zu überprüfen, ob eine Instanz einer beliebigen Sound-Ressource gerade in Ihrem Spiel abgespielt wird. Sie geben die zu prüfende Sound-Ressource aus dem Asset-Explorer an und die Funktion gibt true if it is playing and false zurück, wenn dies nicht der Fall ist. Wenn Sie jedoch das "not"-Flag markieren, wird dies umgekehrt und die Funktion gibt true zurück, wenn der Sound nicht abgespielt wird, und ansonsten false.

Beachten Sie, dass Aktionen, die Sie in den "if"-Block einfügen möchten, an der Seite der Aktion abgelegt werden müssen, wie in der folgenden Abbildung gezeigt:

Diese Aktionen werden nun ausgeführt, wenn der "if"-Block den Wert true ergibt, während alle Aktionen, die an anderer Stelle ausgelassen werden, werden nach dem "if"-Block ausgeführt.
