Rasterwert abrufen

Mit dieser Aktion können Sie einer bestimmten Zelle innerhalb der Gitterdatenstruktur einen Wert zuweisen. Sie geben die Variable an, die den Gitterindex enthält (wie von der Aktion Gitter erstellen zurückgegeben), dann die x- und y-Position (die Spalte und Zeile) für die zu setzende Zelle und schließlich den Wert für diese Zelle. Sie müssen auch eine Zielvariable setzen (die als temporäre lokale Variable gekennzeichnet werden kann), um den zurückgegebenen Zellenwert zu speichern, und der Wert kann jeder gültige Datentyp sein. Beachten Sie, dass die x- und y-Position für die Zelle ganze Zahlen innerhalb der Grenzen des Gitters sein müssen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Grid Der (in einer Variablen gespeicherte) Index des abzurufenden Gitters
x Die Position der Zelle auf der X-Achse (die Spalte), die erhalten werden soll
y Die Position der Zelle auf der y-Achse, die erhalten werden soll (die Zeile)
Target Die Zielvariable, die zum Speichern des zurückgegebenen Wertes verwendet wird

Beispiel:

Der obige Aktionsblock code ruft den Wert im Raster ab, der in einer globalen Variablen gespeichert ist, die die der Position der Instanz im Raum entspricht. Dieser Wert wird dann überprüft, und wenn es sich um einen Instanz-ID-Wert handelt, wird diese Instanz zerstört und die Gitterzelle auf das Schlüsselwort noone gesetzt.
