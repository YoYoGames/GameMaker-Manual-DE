Karteneintrag entfernen

Mit dieser Aktion können Sie einen Schlüssel (und seinen entsprechenden Wert) aus einer Map-Datenstruktur entfernen. Sie geben die Variable an, die den Map-Index enthält (wie von der Aktion Create Map zurückgegeben), und geben dann einen "Schlüssel" an (der ist der Bezeichner innerhalb der Map für einen Wert), und die Aktion wird diesen Schlüssel aus der Map entfernen. Beachten Sie, dass diese Aktion nichts zurückgibt, und wenn Sie den Wert für den angegebenen Schlüssel benötigen, sollten Sie die Aktion Get Map Value verwenden, bevor Sie ihn entfernen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Map Der (in einer Variablen gespeicherte) Index der Karte, aus der der Wert entfernt werden soll
Key Der zu verwendende Schlüssel (real oder string)

Beispiel:

Der obige Aktionsblock code ruft den Wert eines bestimmten Schlüssels aus der Map-Datenstruktur ab. Struktur ab und prüft dann, ob der Wert kleiner oder gleich 0 ist. Ist dies der Fall, wird der Schlüssel aus der Map entfernt.
