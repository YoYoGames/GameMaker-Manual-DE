Zur Liste hinzufügen

Mit dieser Aktion können Sie der Liste einen neuen Wert (eines beliebigen Datentyps) hinzufügen, der dann am Ende angefügt wird. Sie geben die Variable an, die den Listenindex speichert (wie von der Aktion Liste erstellen zurückgegeben) und den Wert der gespeichert werden soll.

Aktion Syntax:

Argumente:

Argument Beschreibung
List Der (in einer Variablen gespeicherte) Index der Liste, der hinzugefügt werden soll
Value Der Wert, der in die Liste aufgenommen werden soll

Beispiel:

Der obige Aktionsblock code erstellt eine globale Bereichsvariable und erstellt dann eine neue Listendatenstruktur, deren Indexwert der globalen Variablen zugewiesen wird. Der Bereich wird dann so geändert, dass alle Instanzen des Objekts "obj_Enemy_Parent" ihren eindeutigen Instanz-ID-Wert in die Liste eintragen.
