Index des Listenelements abrufen

Mit dieser Aktion können Sie die Indexposition eines Elements innerhalb der angegebenen Liste ermitteln. Sie geben die Variable an, die den Listenindex speichert (wie von der Aktion Liste erstellen zurückgegeben) und den Wert, dessen Indexposition Sie sowie eine Zielvariable, in der die zurückgegebene Indexposition für das Element gespeichert wird (die als temporäre lokale Variable gekennzeichnet werden kann, die bis zum Ende des script oder des Ereignisses verwendet wird). Beachten Sie, dass, wenn der gesuchte Wert nicht in der Liste vorhanden ist, gibt es keine Indexposition in der Liste, die zurückgegeben werden kann, und der Rückgabewert ist dann -1.

Aktion Syntax:

Argumente:

Argument Beschreibung
List Der (in einer Variablen gespeicherte) Index der Liste, aus der entfernt werden soll
Value Der Wert, auf den die Liste geprüft werden soll
Target Die Zielvariable, in der der zurückgegebene Index gespeichert wird

Beispiel:

Der obige Aktionsblock code prüft auf eine Kollision an der Instanzposition und wenn eine gefunden wird, wird der eindeutige ID-Wert für die Instanz in einer temporären Variablen gespeichert und dann geprüft, ob er in der Listendatenstruktur existiert. Wenn dies der Fall ist, wird das Element aus der Liste entfernt und die Instanz zerstört.
