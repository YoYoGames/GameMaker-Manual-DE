Listenelement erhalten bei

Mit dieser Aktion können Sie die Indexposition eines Elements innerhalb der angegebenen Liste wiederherstellen. Sie geben die Variable an, die den Listenindex speichert (wie von der Aktion Liste erstellen zurückgegeben) und die Indexposition innerhalb der Liste, von der Sie geben die Variable an, die den Listenindex speichert (wie von der Aktion Liste erstellen zurückgegeben), und die Indexposition innerhalb der Liste, von der der Wert des Elements abgerufen werden soll, sowie eine Zielvariable, die den zurückgegebenen Wert des Elements speichert (die als temporäre lokale Variable gekennzeichnet werden kann, die bis zum Ende des script oder des Ereignisses verwendet wird).

HINWEIS: Wenn Sie eine Position angeben, die außerhalb der vorgegebenen Listengröße liegt (z. B. Position 11 in einer Liste mit 10 Werten), gibt die Aktion undefined zurück. Sie können dies mit der Aktion If Undefined überprüfen.

Aktion Syntax:

Argumente:

Argument Beschreibung
List Der (in einer Variablen gespeicherte) Index der Liste, aus der entfernt werden soll
Index Die Indexposition innerhalb der Liste, von der aus das Element abgerufen werden soll
Target Die Zielvariable, in der der zurückgegebene Wert gespeichert wird

Beispiel:

Der obige Aktionsblock code ermittelt die Größe der in der globalen Variable referenzierten Liste und verwendet dann eine for-Schleife durch die Liste loop und zerstört die Instanzen mit den in der Liste gespeicherten IDs. Nachdem loop fertig ist, wird die Liste aus dem Speicher freigegeben.
