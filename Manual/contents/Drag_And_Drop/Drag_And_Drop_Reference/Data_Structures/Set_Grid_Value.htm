Rasterwert einstellen

Mit dieser Aktion können Sie einer bestimmten Zelle innerhalb der Gitterdatenstruktur einen Wert zuweisen. Sie geben die Variable an, die den Gitterindex enthält (wie von der Aktion Gitter erstellen zurückgegeben), dann die x- und y-Position (die Spalte und Zeile) für die zu setzende Zelle und schließlich den Wert für diese Zelle. Der Wert kann ein beliebiger gültiger Datentyp sein, aber die x- und y-Position müssen ganze Zahlen innerhalb der Grenzen des Rasters sein.

Aktion Syntax:

Argumente:

Argument Beschreibung
Grid Der (in einer Variablen gespeicherte) Index des zu setzenden Gitters
x Die Position der Zelle auf der X-Achse, die festgelegt werden soll (die Spalte)
y Die Position der Zelle auf der y-Achse, die festgelegt werden soll (die Zeile)
Value Der Wert, auf den die Gitterzelle gesetzt werden soll

Beispiel:

Der obige Aktionsblock code erstellt eine neue Grid-Datenstruktur und speichert ihren Indexwert in einer globalen Variablen. Er durchläuft dann alle Instanzen im Raum und fügt ihren eindeutigen ID-Wert in die Zelle ein, die ihrer Position entspricht.
