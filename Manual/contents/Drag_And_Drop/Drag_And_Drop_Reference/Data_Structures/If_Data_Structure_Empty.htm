Wenn die Datenstruktur leer ist

Mit dieser Aktion können Sie prüfen, ob die angegebene Datenstruktur leer ist oder nicht. Sie müssen zuerst den Typ der zu prüfenden Datenstruktur auswählen (entweder eine Liste, eine Karte oder ein Stapel - keine Gitter, da diese nicht einmal leer sein können) und dann geben Sie die Variable, die den Index der Datenstruktur enthält (wie von den Aktionen Liste erstellen, Karte erstellen und Stapel erstellen zurückgegeben). Die Aktion wird true zurückgeben, wenn die Datenstruktur leer ist, und false, wenn sie es nicht ist. Beachten Sie, dass Sie "not" in der Aktion markieren können, um zu prüfen, ob eine Datenstruktur Daten enthält , indem Sie prüfen, ob sie nicht leer ist.
