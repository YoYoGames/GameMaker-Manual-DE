Wenn Datenstruktur vorhanden ist

Mit dieser Aktion können Sie eine Variable daraufhin überprüfen, ob sie eine Datenstruktur indiziert oder nicht. Sie müssen zunächst den Typ der zu prüfenden Datenstruktur auswählen (entweder eine Liste, eine Karte, ein Raster oder ein Stapel) und dann die Variable angeben, die die Datenstruktur enthält Index (wie von den Aktionen Liste erstellen, Karte erstellen, Gitter erstellen und Stapel erstellen zurückgegeben). Die Aktion gibt true zurück, wenn die Datenstruktur existiert, und false, wenn sie nicht existiert. Beachten Sie, dass Sie in der Aktion "not" angeben können, um zu prüfen, ob eine Variable keinen Datenstrukturindex enthält.

Diese Aktionen werden nun ausgeführt, wenn der "if"-Block den Wert true ergibt, während alle Aktionen, die an anderer Stelle ausgelassen werden, werden nach dem "if"-Block ausgeführt.

Aktion Syntax:

Argumente:

Argument Beschreibung
Structure Der Typ der Datenstruktur, auf die geprüft werden soll (Liste, Karte, Gitter oder Stapel)
Variable Die Variable, die den Index der Datenstruktur enthält
