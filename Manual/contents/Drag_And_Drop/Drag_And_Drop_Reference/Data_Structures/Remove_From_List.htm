Aus Liste entfernen

Diese Aktion kann verwendet werden, um ein Indexelement aus der angegebenen Liste zu entfernen. Sie geben die Variable an, die den Listenindex speichert (wie von der Aktion Liste erstellen zurückgegeben) und die Indexposition innerhalb der zu entfernenden Liste, wobei die Indexposition zwischen 1 und (Listenlänge -1) liegt. Diese Aktion gibt nichts zurück. Wenn Sie also den Wert an der Indexposition benötigen, sollten Sie Get List Item At verwenden, bevor Sie den Index entfernen.

Aktion Syntax:

Argumente:

Argument Beschreibung
List Der (in einer Variablen gespeicherte) Index der Liste, aus der entfernt werden soll
Index Der zu entfernende Index innerhalb der Liste

Beispiel:

Der obige Aktionsblock code prüft auf eine Kollision an der Instanzposition, und wenn wird der eindeutige ID-Wert für die Instanz in einer temporären Variablen gespeichert und dann geprüft, ob er in der Listendatenstruktur existiert. Wenn dies der Fall ist, wird das Element aus der Liste entfernt und die Instanz zerstört.
