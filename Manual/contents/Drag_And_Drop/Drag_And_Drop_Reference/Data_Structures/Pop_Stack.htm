Pop Stack

Mit dieser Aktion können Sie einen Wert von der Spitze einer Stack-Datenstruktur entfernen. Sie geben die Variable an, die den Stack-Index enthält (wie von der Aktion Create Stack zurückgegeben), und geben dann die Zielvariable an, die die die zurückgegebene Variable enthalten soll (die als temporäre lokale Variable gekennzeichnet werden kann).

Aktion Syntax:

Argumente:

Argument Beschreibung
Stack Der (in einer Variablen gespeicherte) Index des zu verwendenden Stapels
Target Die Zielvariable, die den zurückgegebenen Wert enthält

Beispiel:

Der obige Aktionsblock code prüft, ob der angegebene Stapel leer ist, und wenn nicht, wird den obersten Wert ab und speichert ihn in einer temporären Variablen. Diese wird dann verwendet, um einen Alarm auszulösen. Ist der Stapel leer, werden 10 zufällige Werte auf den Stapel gelegt.
