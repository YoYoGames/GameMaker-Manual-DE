Kartenwert festlegen

Mit dieser Aktion können Sie ein Schlüssel/Wert-Paar in die Map-Datenstruktur einfügen. Sie geben die Variable an, die den Map-Index enthält (wie von der Aktion " Map erstellen" zurückgegeben), und geben dann einen "Schlüssel" an (der der Bezeichner innerhalb der Map für einen Wert), und dann den "Wert", der mit diesem Schlüssel verbunden werden soll. Beide Schlüssel können entweder Integer-Werte (wie 0, 200, 36712, etc...) oder Strings (wie "Name", "Shield" etc...) sein, aber der Wert kann jeder gültiger Datentyp sein.

Aktion Syntax:

Argumente:

Argument Beschreibung
Map Der (in einer Variablen gespeicherte) Index der zu setzenden Karte
Key Der zu setzende Schlüssel (Integer oder String)
Value Der Wert, der mit dem angegebenen Schlüssel gespeichert werden soll
