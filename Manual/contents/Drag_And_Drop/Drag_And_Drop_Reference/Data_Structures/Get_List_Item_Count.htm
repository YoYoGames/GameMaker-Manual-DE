Anzahl der Listenelemente abrufen

Mit dieser Aktion können Sie herausfinden, wie viele Einträge eine Liste enthält. Sie geben die Variable an, die den Listenindex enthält (wie von der Aktion Liste erstellen zurückgegeben) und die Zielvariable, die die zurückgegebene Anzahl der Einträge in der Liste (die als temporäre lokale Variable gekennzeichnet werden kann, die bis zum Ende des script oder des Ereignisses verwendet wird).

Aktion Syntax:

Argumente:

Argument Beschreibung
List Der (in einer Variablen gespeicherte) Index, von dem aus die Anzahl der Artikel ermittelt wird
Target Die Zielvariable, in der die Anzahl der zurückgegebenen Artikel gespeichert wird

Beispiel:

Der obige Aktionsblock code ermittelt die Größe der Liste, auf die in der globalen referenziert wird, und verwendet dann ein for loop, um die Liste loop zu durchlaufen und die Instanzen mit den in der Liste gespeicherten IDs zu zerstören. Nachdem loop fertig ist, wird die Liste aus dem Speicher freigegeben.
