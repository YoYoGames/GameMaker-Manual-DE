Kartenwert abrufen

Mit dieser Aktion können Sie den Wert abrufen, der mit einem bestimmten Schlüssel in einer Map-Datenstruktur verbunden ist. Sie geben die Variable an, die den Map-Index enthält (wie von der Aktion Create Map zurückgegeben), und geben dann einen "Schlüssel" an (der ist der Bezeichner innerhalb der Map für einen Wert) und eine Zielvariable, die den zurückgegebenen Wert enthält (die als temporäre lokale Variable gekennzeichnet werden kann).

HINWEIS: Wenn Sie einen Schlüssel angeben, der in der Map nicht vorhanden ist, gibt die Aktion undefined zurück. Sie können dies mit der Aktion If Undefined überprüfen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Map Der (in einer Variablen gespeicherte) Index der Karte, aus der der Wert abgerufen werden soll
Key Der zu ermittelnde Schlüssel (real oder string)
Target Die Zielvariable, die den zurückgegebenen Map-Wert enthält

Beispiel:

Der obige Aktionsblock code prüft auf eine Kollision an der Instanzposition, und wenn eine gefunden wird, wird in einer Karte nachgeschaut, um einen Wert für den Schlüssel "damage" zu erhalten, und dieser Wert wird dann verwendet, um die aufrufende Instanzvariable "hp" zu ändern.
