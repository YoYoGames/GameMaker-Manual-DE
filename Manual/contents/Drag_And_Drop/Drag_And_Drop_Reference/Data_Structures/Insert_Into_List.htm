In Liste einfügen

Mit dieser Aktion können Sie die Indexposition eines Elements innerhalb der angegebenen Liste ermitteln. Sie geben die Variable an, die den Listenindex speichert (wie von der Aktion Liste erstellen zurückgegeben) und den Wert, dessen Indexposition Sie sowie eine Zielvariable, in der die zurückgegebene Indexposition für das Element gespeichert wird (die als temporäre lokale Variable gekennzeichnet werden kann, die bis zum Ende des script oder des Ereignisses verwendet wird). Wenn Sie einen Wert in eine Liste einfügen, werden alle Indizes Wenn Sie also eine Liste mit 4 Einträgen haben (Indexposition 0 bis 3) und einen neuen Eintrag an Indexposition 2 einfügen, wird der vorherige Eintrag 2 zu Eintrag 3 und der vorherige Eintrag 3 zu Eintrag 4. Wenn der gesuchte Wert in der Liste nicht vorhanden ist, gibt es keine Indexposition in der Liste, die zurückgegeben werden kann, so dass der Rückgabewert -1 ist.

Aktion Syntax:

Argumente:

Argument Beschreibung
List Der (in einer Variablen gespeicherte) Index der Liste, die eingefügt werden soll
Index Die Indexposition innerhalb der Liste zum Einfügen eines Wertes
Value Der zu speichernde Wert an der Position
