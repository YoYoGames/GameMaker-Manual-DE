Auf den Stapel schieben

Mit dieser Aktion können Sie einen Wert auf eine Stack-Datenstruktur schieben. Sie geben die Variable an, die den Stack-Index enthält (wie von der Aktion Stack erstellen zurückgegeben), und geben dann den Wert an, der nach oben geschoben werden soll. Bei Stack-Datenstrukturen Strukturen werden Werte immer an der Spitze des Stapels hinzugefügt und nie am Ende oder in der Mitte (wenn Sie diese Funktionalität benötigen, verwenden Sie eine Listendatenstruktur).

Aktion Syntax:

Argumente:

Argument Beschreibung
Stack Der (in einer Variablen gespeicherte) Index des zu verwendenden Stapels
Value Der Wert, der auf dem Stapel abgelegt werden soll
