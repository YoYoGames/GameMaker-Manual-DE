Raster löschen

Mit dieser Aktion können Sie eine Gitterdatenstruktur auf einen einzigen Wert zurücksetzen (jede Zelle innerhalb des Gitters wird auf den angegebenen Wert gesetzt). Sie geben die Variable an, die den Gitterindex enthält (wie von der Aktion Gitter erstellen zurückgegeben) zurückgegeben wird, und geben dann den Wert an, auf den das Gitter zurückgesetzt werden soll.

Aktion Syntax:

Argumente:

Argument Beschreibung
Grid Der (in einer Variablen gespeicherte) Index des zu löschenden Gitters
Value Der Wert, auf den das Gitter zurückgesetzt werden soll
