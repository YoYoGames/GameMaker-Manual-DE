Zeitleiste Aktionsbibliothek

Zeitleisten sind ein leistungsfähiger Mechanismus zur Steuerung von Ereignissen in Ihrem Spiel und basieren auf "Momenten", wobei ein "Moment" einfach ein einzelner Spielschritt ist. Diese Zeitleisten werden in der Regel über den Asset-Browser auf ähnliche Weise wie ein Objekt erstellt, d.h. Sie erstellen eine Zeitleiste im Zeitleisten-Editor, In jedem Moment, in dem eine Aktion ausgeführt werden soll, fügen Sie eine code oder eine GML Visual Action hinzu.

Sobald eine Timeline-Ressource erstellt wurde, können Sie sie mit den folgenden Aktionen anhalten, starten und manipulieren:

Zeitleiste der Instanz festlegen
Zeitleiste einstellen Moment
Geschwindigkeit der Zeitleiste einstellen
Zeitleistenstatus festlegen
