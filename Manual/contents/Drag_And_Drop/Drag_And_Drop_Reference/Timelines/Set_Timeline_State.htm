Zeitleistenstatus festlegen

Diese Aktion kann verwendet werden, um eine Zeitleiste zu starten, zu stoppen, anzuhalten oder fortzusetzen. Die Zeitleiste muss zuerst für die Instanz mit der Aktion Instanzzeitleiste setzen gesetzt worden sein, und dann können Sie diese Aktion jederzeit verwenden, um den Status der Instanz aus den folgenden Optionen ändern

Option Beschreibung
Start Startet die der Instanz zugewiesene Zeitleiste (wenn die Zeitleiste angehalten wurde, wird die Startposition wieder auf den Zeitpunkt 0 gesetzt).
Lebenslauf Fortsetzen einer angehaltenen Timeline (wenn die Timeline angehalten und nicht pausiert wurde, bewirkt dies nichts).
Pausiert Halten Sie die der Instanz zugewiesene Zeitleiste an (im angehaltenen Zustand bewegen sich die Momente in einer Zeitleiste nicht mit den Spielframes weiter).
Stopp Stoppt die der Instanz zugewiesene Zeitleiste.

Syntax:

Argumente:

Argument Beschreibung
State Der Zustand, in den die Zeitleiste gesetzt werden soll (siehe Tabelle oben)

Der obige Aktionsblock code prüft den Wert in einer globalen Variablen und Wenn er true ist, wird die Zeitleiste angehalten, und wenn er false ist, wird die Zeitleiste fortgesetzt.
