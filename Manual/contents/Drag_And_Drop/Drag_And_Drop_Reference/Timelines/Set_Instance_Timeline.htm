Zeitleiste der Instanz festlegen

Mit dieser Aktion wird der Instanz eine Zeitleiste aus dem Asset-Browser zugewiesen. Sie geben die zuzuweisende Zeitleiste an und können dann wählen, ob die Zeitleiste loop beim Erreichen des letzten Moments (das nächste Bild nach dem letzten Moment wird loop zurück zum wieder zum ersten Moment der Zeitleiste zurück) und ob die Zeitleiste sofort gestartet werden soll oder nicht. Wenn Sie die Zeitleiste nicht sofort starten, können Sie die Aktion Zeitleistenstatus setzen verwenden, um sie später in einem anderen Ereignis oder script zu starten.

Aktion Syntax:

Argumente:

Argument Beschreibung
Timeline Die Ressource der Zeitleiste, die einer Instanz zugewiesen werden soll
Loop Markierung, dass die Zeitleiste loop nach dem letzten Moment angezeigt werden soll (Standard ist markiert)
Start Timeline Markierung, ob die Zeitleiste sofort beginnen soll oder nicht (Standard ist markiert)
