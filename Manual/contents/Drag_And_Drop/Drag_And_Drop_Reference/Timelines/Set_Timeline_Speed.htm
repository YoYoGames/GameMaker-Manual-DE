Geschwindigkeit der Zeitleiste einstellen

Mit dieser Aktion können Sie die Geschwindigkeit festlegen, mit der sich eine Zeitleiste durch die darin enthaltenen Momente bewegt. Normalerweise soll die Position in der Zeitleiste bei jedem Spielframe um 1 erhöht werden, aber Sie können diesen Wert ändern, indem Sie diese Aktion auf einen anderen Wert setzen. Der von Ihnen gewählte Wert wird wie ein Multiplikator behandelt, d. h. bei einem Wert von 2 läuft die Zeitleiste beispielsweise mit doppelter Geschwindigkeit, während sie bei einem Wert von 0,5 nur halb so schnell läuft. Für die Momente bedeutet das: Wenn der Wert größer als eins ist, können mehrere Momente innerhalb desselben Spielframes auftreten (es werden nie Momente übersprungen), und wenn die Geschwindigkeit kleiner als eins ist, gibt es nach dem Ablauf eines Moments Frames Wenn Sie die Geschwindigkeit der Zeitachse beispielsweise auf 0,1 einstellen, gibt es einen Moment in einem Spiel-Frame, dann 9 Spiel-Frames ohne Momente, und im zehnten Spiel-Frame wird der nächste Moment ausgeführt.

Wenn Sie eine Zeitleiste mit den Aktionen Instanzzeitleiste setzen oder Zeitleistenstatus setzen starten, aber keine die Geschwindigkeit nicht festlegen, dann wird die Zeitleiste standardmäßig auf 1 Moment pro Spielframe eingestellt.

Argumente:

Argument Beschreibung
Speed Die Geschwindigkeit - als Multiplikator - mit der die Zeitleiste laufen soll (Standard ist 1)
