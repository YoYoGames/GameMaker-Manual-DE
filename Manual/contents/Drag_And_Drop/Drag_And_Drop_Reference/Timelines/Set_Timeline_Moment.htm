Zeitleiste einstellen Moment

Mit dieser Aktion kann eine Zeitleiste vom aktuellen Moment zu einem neuen Moment übersprungen werden. Zeitleisten bewegen sich durch verschiedene "Momente" vorwärts, und jeder Moment nimmt in der Regel einen Spiel-Frame ein (wie das Step-Event eines Objekts). Sobald eine Zeitleiste begonnen hat beginnt, durchläuft sie die Momente, bis sie das Ende erreicht. An diesem Punkt kehrt sie entweder zum ersten Moment in der Zeitleiste zurück ( loop ) oder bleibt ganz stehen. Dieser lineare Ansatz für Zeitleisten kann jedoch mit dieser Aktion geändert werden, um jederzeit einen anderen Zeitpunkt zu setzen. einen anderen Zeitpunkt zu setzen, so dass Sie Ihre eigene Zeitleiste loops erstellen oder Abschnitte einer Zeitleiste unter bestimmten Umständen überspringen können usw.. Mit dieser Aktion legen Sie einfach den Zeitpunkt fest, zu dem innerhalb der Zeitleiste gesprungen werden soll, und das nächste Spielbild wird (Beachten Sie, dass Sie der Instanz mit der Aktion Instanz-Zeitleiste setzen eine Zeitleiste zugewiesen haben müssen).
