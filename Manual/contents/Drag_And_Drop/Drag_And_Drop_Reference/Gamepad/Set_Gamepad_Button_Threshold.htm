Schwellenwert für Gamepad-Tasten festlegen

Mit dieser Aktion können Sie den "Schwellenwert" für die analogen Tasten des Gamepads festlegen. Du gibst den Gamepad-Index an, der eingestellt werden soll, und stellst dann einen Wert zwischen 0 und 1 ein, und wenn der Eingabebetrag der analogen Taste niedriger als der angegebene Wert ist, wird die Gamepad-Taste als 0 betrachtet. Beachten Sie, dass es sich hierbei um eine globale Einstellung handelt, die sich auf alle analogen Tasten des Gamepads auswirkt. Dieser Wert wird bei allen Prüfungen auf "gedrückt", " gedrückt" und " losgelassen" für das jeweilige Gamepad verwendet, wird aber von der Aktion " Get Gamepad Trigger" ignoriert.

Aktion Syntax:

Argumente:

Argument Beschreibung
Gamepad Der Gamepad-Index.
Deadzone Der Schwellenwert (0 - 1)
