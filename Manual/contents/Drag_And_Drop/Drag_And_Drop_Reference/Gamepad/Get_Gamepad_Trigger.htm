Gamepad-Trigger holen

Mit dieser Aktion können Sie den Wert der verschiedenen Auslöser eines bestimmten Gamepads ermitteln. Sie geben den Gamepad-Index an (das ist die Nummer des "Slots", den ein Gamepad belegt) und wählen den zu prüfenden Auslöser (links oder rechts). Der zurückgegebene Wert liegt zwischen 0 und 1, wobei 0 für keinen Druck auf den Auslöser und 1 für vollen Druck steht. Der zurückgegebene Wert wird in der Zielvariablen gespeichert, die Sie zur späteren Verwendung angeben. Beachten Sie, dass diese Funktion die Schwellenwerteinstellung nicht berücksichtigt.

Aktion Syntax:

Argumente:

Argument Beschreibung
Gamepad Der Gamepad-Index.
Trigger Der zu prüfende Auslöser (links/rechts)
Target Die Zielvariable, in der der zurückgegebene Wert gespeichert wird.

Beispiel:

Der obige Aktionsblock code erstellt eine lokale (temporäre) Variable und verwendet sie dann zum Speichern den aktuellen Wert für den linken Trigger. Wenn der Triggerwert über 0,5 liegt, wird eine Instanz an der Position der aufrufenden Instanz erstellt.
