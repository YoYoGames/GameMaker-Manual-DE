Wenn die Gamepad-Taste gedrückt wird

Mit dieser Aktion können Sie das Gamepad abfragen, um zu sehen, ob gerade eine Taste gedrückt wird. Die Aktion gibt bei jedem Schritt, bei dem die Taste des Gamepads gedrückt (und gehalten) wird, true zurück, andernfalls false. Wenn du allerdings auf den not Modifikator klicken, können Sie prüfen, ob die Taste nicht gedrückt wird, d. h. die Aktion gibt true zurück, wenn keine Taste gedrückt wird, und false, wenn eine Taste gedrückt wird. Wenn Sie nur prüfen wollen, ob eine einzelne Taste gedrückt wird, dann verwenden Sie die Aktion if gamepad button down.

Wenn es sich bei der zu prüfenden Taste um eine analoge Taste handelt, wird die Betätigung erst dann erkannt, wenn der Wert der Taste über dem Schwellenwert liegt, der mit der Aktion Schwellenwert für Gamepad-Tasten festlegen festgelegt wurde.

Aktion Syntax:

Argumente:

Argument Beschreibung
Gamepad Der Gamepad-Index.
Button Der Name der zu prüfenden Schaltfläche.
Not Negieren der Prüfung (aus wahr wird falsch und umgekehrt)

Der obige Aktionsblock code fragt den Tastenzustand des Gamepads 0 bei jedem Schritt ab und wenn eine Taste gedrückt gehalten wird, prüft er, ob die Taste zum ersten Mal gedrückt wird. Wenn die Prüfung true auf den ersten Tastendruck erfolgt, wird ein Ton abgespielt und die Instanzgeschwindigkeit auf 10 gesetzt, während die Taste gedrückt wird. Wenn die Taste nicht gedrückt wird wird beim Loslassen der Taste eine Prüfung durchgeführt, um die Geschwindigkeit wieder auf 0 zurückzusetzen.
