Wenn Gamepad-Taste losgelassen

Diese Aktion prüft, wann eine Gamepad-Taste für den angegebenen Gamepad-Index losgelassen wurde. Sie gibt true bei dem einen Spieltick zurück, bei dem die Taste losgelassen wird, und false zu allen anderen Zeiten. Wenn du den Modifikator not aktivierst, aktivieren, wird diese Aktion umgekehrt und überprüft, ob die Taste nicht losgelassen wurde, und gibt false bei dem Spieltick zurück, bei dem sie losgelassen wurde, und true zu allen anderen Zeiten.

Handelt es sich bei der zu prüfenden Taste um eine analoge Taste, wird das Loslassen erst dann erkannt, wenn der Wert der Taste unter den Schwellenwert fällt, der mit der Aktion Schwellenwert für Gamepad-Tasten festlegen festgelegt wurde.

Beachten Sie, dass Aktionen, die Sie in den "if"-Block einfügen möchten, an der Seite der Aktion abgelegt werden müssen, wie in der folgenden Abbildung gezeigt:

Diese Aktionen werden nun ausgeführt, wenn der "if"-Block den Wert true ergibt, während alle an anderer Stelle ausgelassenen Aktionen nach dem "if"-Block ausgeführt werden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Gamepad Der Gamepad-Index.
Button Der Name der zu prüfenden Schaltfläche.
Not Negieren der Prüfung (aus wahr wird falsch und umgekehrt)

Der obige Aktionsblock code fragt den Tastenzustand des Gamepads 0 bei jedem Schritt ab und wenn eine Taste gedrückt gehalten wird, prüft er, ob die Taste zum ersten Mal gedrückt wird. Wenn die Prüfung true auf den ersten Tastendruck erfolgt, wird ein Ton abgespielt und die Instanzgeschwindigkeit auf 10 gesetzt, während die Taste gedrückt wird. Wenn die Taste nicht gedrückt wird wird beim Loslassen der Taste eine Prüfung durchgeführt, um die Geschwindigkeit wieder auf 0 zurückzusetzen.
