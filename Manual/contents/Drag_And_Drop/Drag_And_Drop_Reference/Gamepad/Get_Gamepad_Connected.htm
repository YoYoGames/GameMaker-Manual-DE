Gamepad-Verbindung herstellen

Mit dieser Aktion können Sie überprüfen, ob an einem bestimmten Gamepad-"Steckplatz" ein Gamepad angeschlossen ist. Sie geben den Gamepad-Index an (dies ist die Nummer des "Steckplatzes", den ein Gamepad belegt) und der zurückgegebene Wert wird in der Zielvariablen gespeichert, die Sie für die künftige Verwendung angeben (diese kann als temporäre lokale Variable gekennzeichnet werden, die Sie für dieses Ereignis erstellen oder script).

Aktion Syntax:

Argumente:

Argument Beschreibung
Gamepad Der zu prüfende Gamepad-Index.
Target Die Zielvariable, in der der zurückgegebene Wert gespeichert wird.

Beispiel:

Der obige Aktionsblock code führt eine for loop aus, um zu prüfen, ob ein Gamepad an einen der 12 verfügbaren "Steckplätze" angeschlossen ist. Wenn ein Gamepad gefunden wird, wird eine globale Variable auf die aktuelle Steckplatznummer gesetzt und die loop wird unterbrochen.
