Gamepad-Anzahl erhalten

Mit dieser Aktion können Sie die Anzahl der Gamepads ermitteln, die an das Gerät angeschlossen sind, auf dem Ihr Spiel läuft. Die zurückgegebene Zahl liegt zwischen 0 und 12.

Aktion Syntax:

Argumente:

Argument Beschreibung
Target Die Zielvariable, in der der zurückgegebene Wert gespeichert wird.

Beispiel:

Der obige Aktionsblock code ermittelt die Anzahl der angeschlossenen Gamepads und ordnet den Wert einer eine temporäre lokale Variable zu. Anschließend wird geprüft, ob der Wert größer als 0 ist, und eine globale Variable entsprechend gesetzt.
