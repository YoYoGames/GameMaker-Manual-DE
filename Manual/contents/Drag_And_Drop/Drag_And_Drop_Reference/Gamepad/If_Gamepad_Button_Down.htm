Wenn Gamepad-Taste unten

Mit dieser Aktion können Sie ein Gamepad abfragen, um zu sehen, ob eine bestimmte Taste gerade gedrückt wird. Die Aktion gibt bei jedem Schritt, bei dem die Taste gedrückt gehalten wird, true zurück, andernfalls false. Wenn Sie jedoch auf den Modifikator not klicken, können Sie prüfen, ob die Taste nicht gedrückt wird, d. h. die Aktion gibt true zurück, wenn keine Taste gedrückt ist, und false, wenn eine Taste gedrückt ist. Wenn Sie nur auf einen einzelnen Tastendruck prüfen wollen, verwenden Sie die Aktion If Gamepad Button Pressed.

Handelt es sich bei der zu prüfenden Taste um eine analoge Taste, wird sie erst dann als gedrückt betrachtet, wenn der Tastenwert den mit der Aktion Gamepad-Tastenschwelle festlegen festgelegten Schwellenwert überschritten hat.

Diese Aktionen werden nun ausgeführt, wenn das "if" den Wert true ergibt, während alle Aktionen, die an anderer Stelle ausgelassen werden, nach dem "if"-Block ausgeführt werden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Gamepad Der Gamepad-Index.
Button Der Name der zu prüfenden Schaltfläche.
Not Negieren der Prüfung (aus wahr wird falsch und umgekehrt)

Der obige Aktionsblock code fragt den Tastenzustand des Gamepads 0 bei jedem Schritt ab und prüft, ob eine Taste gedrückt gehalten wird, prüft er, ob die Taste zum ersten Mal gedrückt wird. Wenn die Prüfung true auf den ersten Tastendruck erfolgt, wird ein Ton abgespielt und die Instanzgeschwindigkeit auf 10 gesetzt, während die Taste gedrückt gehalten wird. Wenn die Taste nicht gedrückt gehalten wird wird beim Loslassen der Taste eine Prüfung durchgeführt, um die Geschwindigkeit wieder auf 0 zurückzusetzen.
