Fall

Mit dieser Aktion wird ein neuer Fall innerhalb eines Schalters erstellt. Sie geben den Wert für den Fall an und dieser wird mit dem Eingabewert für Switch verglichen. Wenn sie übereinstimmen, werden die unter dem Fall hinzugefügten Aktionen ausgeführt. Aktionen, die zu einem Case hinzugefügt werden und nur in diesem Case ausgeführt werden sollen, sollten an die Seite der Aktion gestellt werden, so wie Sie eine Aktion zu einem "if" hinzufügen würden:

Beachten Sie, dass ein Case-Wert eine Konstante sein muss, die einen beliebigen Datentyp haben kann (real, string, etc...), solange er beim Kompilieren zu einem einzigen Wert ausgewertet wird - Sie können also keine Ausdrücke verwenden, die von Variablen abhängen. Beachten Sie auch, dass Case nicht außerhalb von Switch verwendet werden kann und der Versuch, dies zu tun, zu einem Fehler führt.

Aktion Syntax:

Argumente:

Argument Beschreibung
Constant Der konstante Wert, der mit dem Eingangswert Switch verglichen wird.

Beispiel:

Der obige Aktionsblock code ruft den in einer Variablen gespeicherten Wert ab und vergleicht ihn dann mit den verschiedenen möglichen Fällen. Wenn einer der Fälle mit dem Wert übereinstimmt, wird die Aktion unter diesem Fall ausgeführt, andernfalls wird der nächste Fall ausgewertet. Nachdem switch ausgeführt wurde, wird ein Alarm auf 30 gesetzt.
