Aktionsbibliothek wechseln

Die Aktionsbibliothek Switches enthält die Aktionen, die erforderlich sind, um Erstellung von switch Anweisungen in Ihrer Spielaktion code erforderlich sind. In einer Reihe von Situationen möchten Sie Ihre Instanzen eine Aktion in Abhängigkeit von einem bestimmten Wert ausführen lassen, und obwohl Sie dies mit einer Reihe von aufeinanderfolgenden "if"-Anweisungen tun können, kann dies kann unübersichtlich werden, wenn die Anzahl der möglichen Optionen zwei oder drei übersteigt. In diesen Fällen ist es besser, die "switch"-Aktion zu verwenden. Eine Aktion switch funktioniert wie folgt:

  • Sie liefern den Wert a (in der Regel aus einem Ausdruck oder einer Variablen)
  • Der Wert wird mit dem Wert verglichen, der den verschiedenen "case"-Anweisungen zugeordnet ist
  • Wenn der Wert mit dem Fallwert übereinstimmt, wird der Inhalt des Falles ausgeführt, andernfalls wird der nächste Fall ausgewertet, usw..
  • Sobald ein Fall durchgeführt wurde oder alle Fälle ausgewertet wurden und fehlgeschlagen sind, wird die code vom Ende des Schalters an fortgesetzt.
  • Wenn alle Fälle fehlgeschlagen sind und Sie einen Standardfall haben, wird dieser ausgeführt, bevor die code vom Ende des Schalters an weiterläuft.
