Schalter

Mit dieser Aktion wird eine neue Anweisung Switch erstellt, zu der Sie dann weitere Fälle zur Auswertung hinzufügen können. Sie geben den Wert an, der für den Vergleich mit den Fällen in Switch verwendet werden soll, und fügen dann die verschiedenen Case-Aktionen hinzu, ähnlich Sie fügen die verschiedenen Case-Aktionen hinzu, ähnlich wie Sie eine Aktion zu einem "if" hinzufügen würden, d. h. Sie fügen sie neben der Aktion ein und nicht unter ihr:

Beachten Sie, dass Sie keine andere Art von Aktion in eine switch einfügen können, ohne vorher mindestens einen Fall hinzugefügt haben, der sie enthält.

Aktion Syntax:

Argumente:

Argument Beschreibung
Value Der Wert, der von switch für die Bewertung verwendet wird

Beispiel:

Der obige Aktionsblock code ruft den in einer Variablen gespeicherten Wert ab und vergleicht ihn dann mit den verschiedenen möglichen Fällen. Wenn einer der Fälle mit dem Wert übereinstimmt, wird die Aktion unter diesem Fall ausgeführt, andernfalls wird der nächste Fall ausgewertet. Nachdem switch ausgeführt wurde, wird ein Alarm auf 30 gesetzt.
