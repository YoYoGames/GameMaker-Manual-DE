Partikel-Alpha einstellen

Diese Aktion legt den Alphawert für die Erstellung des Partikels sowie den Alphawert während seiner Lebensdauer fest. Zuerst geben Sie den ID-Wert des Partikeltyps an (der mit dem Wert übereinstimmen sollte, der bei der Erstellung des Partikels mit der Aktion Partikeltyp erstellen zurückgegeben wurde) und dann können Sie die zu verwendenden Alphawerte festlegen.

Sie können drei Alphawerte auswählen, zwischen denen der Partikel während seiner Lebensdauer überblendet. Wenn zum Beispiel ein Partikel die Alphawerte 0,5, 1 und 0 hat und eine Lebensdauer von 60 Spielframes, dann wird er bei seiner Erstellung (Frame 0) einen Alpha-Wert von 0,5, dann einen Alpha-Wert von 1 (Bild 30) und schließlich einen Alpha-Wert von 0 (Bild 60), um unsichtbar zu werden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Type Der eindeutige ID-Wert für den Partikeltyp, für den die Farbe festgelegt werden soll
Start Der zu verwendende Anfangsalpha
Middle Der Zwischenalpha zur Verwendung
End Das endgültige Alpha, das zu verwenden ist

Beispiel:

Der obige Aktionsblock code erstellt einen neuen Partikeltyp und weist seinen eindeutigen ID-Wert einer globalen Variablen zu. Anschließend werden alle Eigenschaften des Partikeltyps festgelegt.
