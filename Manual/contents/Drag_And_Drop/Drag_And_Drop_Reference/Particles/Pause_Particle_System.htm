Pause Partikelsystem

Mit dieser Aktion wird das angegebene Partikelsystem angehalten. Du gibst den eindeutigen ID-Wert für das zu pausierende System an, der zurückgegeben wurde, als du das System mit der Aktion Partikelsystem erstellen erstellt hast. Ein pausiertes Partikelsystem zeichnet immer noch die Partikel, die gespawnt wurden, aber sie aktualisieren nicht mehr ihre Farbe, Position oder Alpha, noch werden sie ihre Lebensdauer beenden und verschwinden, bis Sie das System wieder aufnehmen. Sie können wählen, ob Sie das System anhalten oder fortsetzen möchten, indem Sie die Option Option "Zustand".

Beachten Sie, dass dadurch weder Emitter angehalten werden, die Partikel streamen, noch wird ein Emitter gestoppt, der Partikel ausstößt - Emitter erzeugen weiterhin Partikel, wenn sie aufgerufen werden, aber wenn das System angehalten wird, tun die Partikel nichts und werden genau dort gezeichnet gezeichnet, bis du den pausierten Zustand zurücksetzt oder das Partikelsystem aktualisierst.

Aktion Syntax:

Argumente:

Argument Beschreibung
System Der eindeutige ID-Wert für das System, das angehalten werden soll
State ob das Partikelsystem angehalten oder fortgesetzt werden soll
