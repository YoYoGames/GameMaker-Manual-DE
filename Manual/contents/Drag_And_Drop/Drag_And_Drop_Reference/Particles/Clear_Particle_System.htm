Klares Partikelsystem

Mit dieser Aktion können Sie das gegebene Partikelsystem in seinen Standardzustand zurückversetzen, indem Sie alle Emitter entfernen (es besteht auch keine Notwendigkeit, die Aktion Partikelemitter zerstören aufzurufen, da dies automatisch erledigt wird) und alle Partikel vom Bildschirm entfernen. Sie geben den eindeutigen ID-Wert für das zu löschende System an, der bei der Erstellung des Systems mit der Aktion Partikelsystem erstellen zurückgegeben wurde. Vorsicht ist geboten bei der Verwendung Diese Aktion zerstört alle Emitter, die mit dem Partikelsystem verbunden sind - das bedeutet, dass jede Instanz im Raum, die einen Emitter verwendet, eine Fehlermeldung ausgeben kann, da dieser Emitter nicht mehr existiert, also stelle sicher, dass die Aktion nur aufgerufen wird, wenn du keine neuen Partikel von diesem Emitter erstellen wirst (für weitere Informationen über Partikelemitter siehe Partikelemitter erstellen).

HINWEIS: Diese Aktion löscht nur die sichtbaren Partikel im Raum, aber sie löscht weder die Partikeleigenschaften noch entfernt sie diese aus dem Speicher. Hierfür sollten Sie die Aktion Partikeltyp zerstören verwenden.

Aktion Syntax:

Argumente:

Argument Beschreibung
System Der eindeutige ID-Wert für das zu vernichtende System
