Partikel emittieren

Diese Aktion wird verwendet, um Partikel aus einem Emitter zu erstellen. Du gibst die ID des Partikelsystems an, zu dem der Emitter gehört (wie von der Aktion Partikelsystem erstellen zurückgegeben), dann die eigentliche Emitter-ID (wie von der Aktion der Aktion Partikelemitter erstellen) und schließlich den zu erstellenden Partikeltyp (der von der Aktion Partikeltyp erstellen zurückgegeben wird). Sie können dann wählen zwischen "Burst" oder "Streamen" der Partikel aus dem Emitter wählen und die Anzahl der zu erstellenden Partikel angeben.

Wenn Sie sich für das Streamen von Partikeln entscheiden, müssen Sie diese Aktion nicht in ein Spielframe-Ereignis wie das Step-Ereignis oder das Draw-Ereignis einfügen, sondern rufen sie einfach einmal im Create-Ereignis auf (z. B.), woraufhin der Emitter in jedem Spielframe Partikel streamt. Frame, bis du diese Aktion erneut aufrufst, um sie zu stoppen (indem du den Zählwert auf 0 setzt). Wenn Sie die Partikel platzen lassen möchten, können Sie diese Aktion in jedem beliebigen Ereignis aufrufen, und die Partikel werden einmal erstellt.

Aktion Syntax:

Argumente:

Argument Beschreibung
System Die ID des Systems, in dem sich der Emitter befindet
Emitter Die ID des zu verwendenden Emitters
Type Die ID des zu erstellenden Partikeltyps
Event Der Ereignistyp für die Partikel, entweder "Burst" oder "Stream".
Count Die Anzahl der Partikel zum Bersten oder Strömen

Beispiel:

Der obige Aktionsblock code erstellt einen Partikelemitter, setzt seine Region, und setzt ihn dann so, dass er einen Strom von Partikeln aussendet.
