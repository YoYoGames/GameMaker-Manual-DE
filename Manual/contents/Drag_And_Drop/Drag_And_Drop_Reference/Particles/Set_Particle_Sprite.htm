Partikel-Sprite einstellen

Diese Aktion kann verwendet werden, um die sprite zu setzen, die ein Partikel beim Zeichnen verwenden kann. Du gibst den eindeutigen ID-Wert für den Typ an, dessen sprite gesetzt werden soll (wie er zurückgegeben wurde, als du den Typ mit der Aktion Partikeltyp erstellen erstellt hast) und die zu verwendende Ressource sprite. Wenn die sprite Unterbilder hat, dann wird der Partikel durch die Bilder mit der Geschwindigkeit animiert, die im Bildeditor für die sprite eingestellt ist. Beachte auch, dass wenn du einen Farbwert für den Partikel einstellst (mit der Aktion Partikelfarbe einstellen ), dann wird diese Farbe mit der verwendeten sprite gemischt, ähnlich wie das Einstellen der Bildüberblendung auf einer Instanz eine Farbe mit der Instanz sprite mischt (verwende Weiß für keine Überblendung).

HINWEIS: Wenn Sie diese Aktion verwenden, sollten Sie nicht die Aktion Partikelform festlegen verwenden, da Sie nur eine sprite oder eine Form haben können, aber nicht beides.

Aktion Syntax:

Argumente:

Argument Beschreibung
Type Der ID-Wert für den Typ, dessen sprite eingestellt werden soll.
Sprite Die sprite, die dem Partikeltyp zugewiesen wird
