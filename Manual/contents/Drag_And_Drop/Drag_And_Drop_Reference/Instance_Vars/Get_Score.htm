Punkte sammeln

Mit dieser Aktion können Sie den Wert der Instanzvariablen abrufen, die für "score" erstellt wurde. Sie geben die Zielvariable an, um den zurückgegebenen "Score"-Wert zu speichern (die als temporäre lokale Variable gekennzeichnet werden kann), aber beachten Sie, dass wenn die Variable nicht vorher gesetzt wurde (mit Set Score), dann wird die Variable für Sie erstellt und die Aktion gibt 0 zurück (im Gegensatz zu regulären Instanzvariablen, die einen Fehler anzeigen, wenn Sie versuchen, auf sie zuzugreifen ohne sie vorher gesetzt zu haben).

Aktion Syntax:

Argumente:

Argument Beschreibung
Target Die Zielvariable zur Speicherung des zurückgegebenen "Score"-Werts (kann als temporäre lokale Variable gekennzeichnet werden)

Beispiel:

Der obige Aktionsblock code ruft den aktuellen Punktwert ab und speichert ihn in einer temporären lokalen Variable. Diese Variable wird dann in eine Zeichenkette umgewandelt und in einer anderen temporären lokalen Variable gespeichert, bevor sie auf dem Bildschirm angezeigt wird.
