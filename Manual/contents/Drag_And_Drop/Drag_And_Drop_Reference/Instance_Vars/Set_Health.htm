Gesundheit einstellen

Mit dieser Aktion können Sie eine Instanzvariable für Gesundheit erstellen oder setzen. Sie geben den Wert an, auf den die Gesundheitsvariable gesetzt oder zu der Gesundheitsvariable hinzugefügt werden soll (wenn Sie das Kontrollkästchen " relativ" ankreuzen, wird der Wert zu dem aktuellen Wert "health" hinzugefügt). Beachten Sie, dass dieser Wert zwischen 0 und 100 liegen sollte, wenn Sie die Aktion " Instanz-Gesundheitsleiste zeichnen " verwenden möchten, andernfalls können Sie einen beliebigen Wert angeben. Beachte auch, dass im Gegensatz zu regulären Instanzvariablen der Variablenname für "health" intern gespeichert wird. Um den Wert abzurufen, musst du also die Aktion Get Health verwenden.
