Spielstand festlegen

Mit dieser Aktion können Sie eine Instanzvariable für "score" erstellen oder setzen. Sie geben den Wert an, auf den die "score"-Variable gesetzt oder zu der "score"-Variable hinzugefügt werden soll (wenn Sie das Kontrollkästchen " relativ" ankreuzen, wird der Wert zu dem aktuellen "score"-Wert Wert hinzugefügt). Beachten Sie, dass im Gegensatz zu regulären Instanzvariablen der Variablenname für "score" intern gespeichert wird. Um den Score-Wert abzurufen, müssen Sie also die Aktion Get Score verwenden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Score Der Betrag, auf den die Punktzahl gesetzt wird (oder der zur Punktzahl hinzuaddiert wird, wenn " relativ" markiert ist)
