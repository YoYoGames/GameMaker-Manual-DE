Gesundheit erhalten

Mit dieser Aktion können Sie den Wert der Instanzvariablen abrufen, die für den "Gesundheitszustand" des Objekts erstellt wurde. Sie geben die Zielvariable an, in der der zurückgegebene "Health"-Wert gespeichert werden soll (die als temporäre lokale Variable gekennzeichnet werden kann), Beachten Sie jedoch, dass die Variable für Sie erstellt wird, wenn sie noch nicht gesetzt wurde (mit Set Health), und die Aktion 0 zurückgibt (im Gegensatz zu normalen Instanzvariablen, die einen Fehler anzeigen, wenn Sie versuchen auf sie zuzugreifen, ohne sie vorher gesetzt zu haben).

Aktion Syntax:

Argumente:

Argument Beschreibung
Target Die Zielvariable zur Speicherung des zurückgegebenen "Health"-Werts (kann als temporäre lokale Variable gekennzeichnet werden)

Beispiel:

Der obige Aktionsblock code ruft den Wert für "health" ab und speichert ihn in einer temporären lokalen Variable. Diese Variable wird dann verwendet, um eine Gesundheitsleiste für die Instanz zu zeichnen.
