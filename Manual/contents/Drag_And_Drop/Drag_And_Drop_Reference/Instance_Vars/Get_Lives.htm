Leben erhalten

Mit dieser Aktion können Sie den Wert der Instanzvariable abrufen, die für die "Leben" erstellt wurde. Sie geben die Zielvariable an, in der der zurückgegebene "Leben"-Wert gespeichert werden soll (die als temporäre lokale Variable gekennzeichnet werden kann), aber beachten Sie, dass wenn die Variable nicht vorher gesetzt wurde (mit Set Lives), dann wird die Variable für Sie erstellt und die Aktion gibt 0 zurück (im Gegensatz zu normalen Instanzvariablen, die einen Fehler anzeigen, wenn Sie versuchen, auf sie zuzugreifen ohne sie vorher gesetzt zu haben).

Argumente:

Argument Beschreibung
Target Die Zielvariable, in der der zurückgegebene "Leben"-Wert gespeichert wird (kann als temporäre lokale Variable gekennzeichnet werden)

Der obige Aktionsblock code ruft den Wert für "lives" ab und speichert ihn in einer temporären lokalen Variable. Diese Variable wird dann verwendet, um zu entscheiden, wie viele gestapelte sprites gezeichnet werden sollen.
