Leben einstellen

Mit dieser Aktion können Sie eine Instanzvariable für Leben erstellen oder setzen. Sie geben den Wert an, auf den die Variable "lives" gesetzt werden soll oder der zur Variablen "lives" hinzugefügt werden soll (wenn Sie das Kontrollkästchen " relativ" aktivieren, wird der Wert zum aktuellen Wert "lives" hinzugefügt Wert hinzugefügt). Beachten Sie, dass im Gegensatz zu regulären Instanzvariablen der Variablenname für "Leben" intern gespeichert wird, so dass Sie zum Abrufen des Wertes die Aktion " Leben abrufen" verwenden müssen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Lives Der Betrag, auf den die Leben gesetzt werden sollen (oder der zu den Leben hinzuaddiert wird, wenn " relativ" markiert ist)

Der obige Aktionsblock code prüft, ob der Raum der erste im Spiel ist, und wenn das der Fall ist, dann werden die Variablen "score", "lives" und "health" für die Instanz eingerichtet.
