Funktion Aufruf

Mit dieser Aktion können Sie eine beliebige GML-Funktion oder eine benutzerdefinierte Funktion aufrufen. Sie geben die aufzurufende Funktion an und liefern dann alle zusätzlichen Argumente, die sie benötigt, d. h. alle Werte (Strings, Ganzzahlen, Ressourcenindizes usw.), die Sie in die Funktion eingeben möchten, damit sie sie verwenden kann. Sie können weitere argument Eingaben hinzufügen, indem Sie auf das Plus-Symbol neben klicken, und Sie sollten sicherstellen, dass die Anzahl der argument mit der der verwendeten Funktion übereinstimmt (d. h., wenn die Funktion 3 arguments benötigt, sollten Sie 3 arguments eingeben). Einige Funktionen können auch einen Wert zur späteren Verwendung in der Aktionsblockliste zurückgeben. Daher können Sie auch die Zielvariable angeben, die diesen Rückgabewert speichern soll. Diese kann zuvor erstellt worden sein oder eine neue temporäre lokale Variable sein (wenn Sie das Kontrollkästchen "Temp" aktivieren). Sowohl das Ziel und das Feld argument können leer gelassen werden, wenn sie nicht benötigt werden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Function Die aufzurufende Funktion GML.
Argument0 ... ArgumentX Die verschiedenen arguments (Werte), die an die Funktion übergeben werden sollen.
Target Der Name der Variablen, auf die die zurückgegebenen Werte abzielen sollen (kann leer bleiben).
