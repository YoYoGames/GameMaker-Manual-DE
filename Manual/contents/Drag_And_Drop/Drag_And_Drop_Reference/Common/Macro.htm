Makro

Diese Aktion ermöglicht es Ihnen, ein oder mehrere Makros zu definieren. Makros sind einfach eindeutige konstante Werte - oder Ausdrücke, die zu einem eindeutigen konstanten Wert ausgewertet werden -, die dann in ähnlicher Weise wie eine reguläre Konstante verwendet werden können. Sie können auch einen Funktionsaufruf in dieser Aktion verwenden, um einen einzigen Rückgabewert zu erzeugen. In diesem Fall wertet das Makro die Funktion bei jedem Aufruf aus. Beachten Sie, dass Makros global im Geltungsbereich sind und einmal deklariert, können sie von jeder Instanz, jedem Skript oder Aktionsblock in Ihrem Spiel verwendet werden, aber sie können nach ihrer Definition nicht mehr geändert werden. Beachten Sie, dass Sie zusätzliche Makros in derselben Aktion definieren können, indem Sie auf das Plus-Symbol neben der Aktion klicken. Für weitere Informationen über Makros siehe hier.
