Globale Variable setzen

Diese Aktion wird verwendet, um entweder einer globalen Variable einen Wert zuzuweisen (und sie dabei zu erstellen) oder um den Wert einer bestehenden globalen Variable zu ändern. Globale Variablen sind Variablen, die zu keiner bestimmten Instanz "gehören", sondern zum gesamten Projekt "gehören" und als solche von jeder Instanz, script oder code, aufgerufen und geändert werden können. Weitergehende Informationen zu globalen Variablen finden Sie hier.

Sie geben den Namen der globalen Variablen an und dann den Wert, der bei Bedarf relativ zum ursprünglichen Wert hinzugefügt werden kann. Wenn Sie einen neuen Variablennamen angeben (d. h. eine Variable, die im gesamten Spiel noch nie vergeben wurde), wird die neue globale Variable und weist ihr den von dir angegebenen Wert zu. Gibst du hingegen den Namen einer bestehenden globalen Variable an, wird der Wert einfach in den neuen Wert geändert. Beachten Sie, dass bei der Auswahl von Relativ der Wert zur bestehenden globalen Variablenwert addiert wird (verwenden Sie eine negative Zahl, um zu subtrahieren) und Sie können auch Ausdrückeund Variablen, die zuvor erstellt wurden, als Wert verwenden.

Es ist erwähnenswert, dass wir mit "Werten" nicht nur numerische Werte meinen, da eine Variable eine Zeichenkette, ein Zeiger, eine Ressourcen-ID oder etwas anderes sein kann, das eine Funktion zurückgeben oder verwenden kann. Wenn es sich um eine Zeichenkette handelt, wird der Wert bei der Relativprüfung mit dem Wert verknüpft, den die Variable zuvor hatte. Wenn Ihre Variable also "Hello" war und Sie ihr mit der Relativprüfung "World" zuweisen, lautet der endgültige Variablenwert "Hello World". Sie müssen Es muss jedoch darauf geachtet werden, dass die Variablenwerte beide vom gleichen Datentyp sind, da der Versuch, z. B. eine Ganzzahl und eine Zeichenkette zu addieren, zu einem compiler Fehler führt.

Sie können auch weitere Variablen in derselben Aktion hinzufügen, indem Sie auf das Plus-Symbol neben der Aktion klicken und eine weitere globale Variable Namen und Wert geben. Weitere Informationen zu Variablen finden Sie im Abschnitt über Variablen und Variablenumfang.

Aktion Syntax:

Argumente:

Argument Beschreibung
Name Der Name der neuen globalen Variable, die erstellt werden soll, oder der Name einer vorhandenen globalen Variable, die geändert werden soll.
Value Der Wert, den die globale Variable enthalten soll.
Relative Aktivieren Sie dieses Kontrollkästchen, um den Wert relativ zum vorherigen Wert zu setzen.
