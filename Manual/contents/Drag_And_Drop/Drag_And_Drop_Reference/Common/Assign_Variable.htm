Variable zuweisen

Diese Aktion wird verwendet, um entweder einer Variablen einen Wert zuzuweisen (und sie dabei zu erstellen) oder um den Wert einer bestehenden Variablen zu ändern. Sie geben den Namen der Variablen und dann den Wert an, der bei Bedarf relativ zum ursprünglichen Wert hinzugefügt werden kann. erforderlich. Wenn Sie einen neuen Variablennamen angeben (d. h. eine Variable, die noch nie zuvor in der Instanz zugewiesen wurde), wird die neue Variable erstellt und ihr der von Ihnen angegebene Wert zugewiesen, während, wenn Sie den Namen einer bestehenden Variable angeben, diese einfach den Wert in den neuen Wert ändern. Beachten Sie, dass das Ankreuzen von Is Relative den Wert zu dem bestehenden Variablenwert addiert (verwenden Sie eine negative Zahl, um zu subtrahieren) und Sie können auch Ausdrücke und Variablen, die zuvor erstellt wurden, als Wert verwenden.

Wenn wir von "Werten" sprechen, meinen wir nicht nur numerische Werte, denn eine Variable kann eine Zeichenkette, ein Zeiger, eine Ressourcen-ID oder etwas anderes sein, das eine Funktion zurückgeben oder verwenden kann. Im Falle einer Zeichenkette wird die Überprüfung von Relative Wenn Ihre Variable also "Hello" war und Sie ihr mit der Relativprüfung "World" zuweisen, lautet der endgültige Variablenwert "Hello World". Es muss darauf geachtet werden, Es muss jedoch darauf geachtet werden, dass die Variablenwerte beide vom gleichen Datentyp sind, da der Versuch, z. B. eine Ganzzahl und eine Zeichenkette zu addieren, zu einem compiler Fehler führt.

Beachten Sie, dass Sie weitere Variablen in dieselbe Aktion einfügen können, indem Sie auf das Plus-Symbol neben der Aktion klicken und einen weiteren Variablennamen und Wert angeben. Weitere Informationen über Variablen finden Sie im Abschnitt über Variablen und Variablenumfang.

Aktion Syntax:

Argumente:

Argument Beschreibung
Name Der Name der neuen Variable, die zugewiesen werden soll, oder der Name einer vorhandenen Variable, die geändert werden soll.
Value Der Wert, den die Variable enthalten soll.
Relative Prüfen Sie, ob der Wert relativ zum vorherigen Wert gesetzt werden soll.
