Wenn Variable

Diese Aktion wird verwendet, um den Zustand einer bestimmten Variablen anhand eines bestimmten Ausdrucks mit einem bestimmten Wert zu vergleichen. Eine "if"-Anweisung gibt immer entweder true oder false zurück, und die zu prüfenden Ausdrücke können eine der folgenden sein folgenden sein:

  • Gleich - Die Variable und der Wert sind beide genau gleich
  • Kleiner als - Die Variable ist kleiner als der Wert
  • Größer als - Die Variable ist größer als der Wert
  • Kleiner als oder gleich - Die Variable ist kleiner als oder gleich dem Wert
  • Größer als oder gleich - Die Variable ist größer als oder gleich dem Wert

Wenn Sie das Feld "Not" argument markieren, werden die obigen Ausdrücke negiert, z. B. wird aus "gleich" "nicht gleich". Beachten Sie auch, dass weder die Variable noch der Wert arguments auf numerische Werte beschränkt sind sondern auch eine Zeichenkette, ein Zeiger, eine Ressource-ID oder etwas anderes sein können, das eine Funktion zurückgeben oder verwenden kann (weitere Informationen finden Sie im Abschnitt über Datentypen ), aber Sie können nicht zwei Sie können jedoch nicht zwei verschiedene Datentypen miteinander vergleichen, ohne einen Fehler zu erhalten - wenn die Variable beispielsweise eine Zeichenkette enthält und Sie versuchen zu prüfen, ob sie gleich einer Zahl ist, erhalten Sie einen syntax Fehler.

WICHTIGER HINWEIS: Aufgrund von Problemen mit der Fließkommagenauigkeit kann die Prüfung, ob zwei Werte genau gleich sind, false zurückgeben, da einer genau 1 sein kann, während der andere 1,00000000000001 sein kann. Dies kann vermieden werden, indem man vor der Prüfung die Aktion Dezimal nach Ganzzahl verwendet oder die Ausdrücke "größer als" oder "kleiner als" benutzt.

Aktion Syntax:

Argumente:

Argument Beschreibung
Variable Der Name der zu prüfenden Variablen.
Not Wird gesetzt, um zu prüfen, ob der Ausdruck nicht zu true ausgewertet werden kann.
Expression Die Art des Ausdrucks, der für die Prüfung verwendet werden soll.
Value Der Wert, gegen den die Variable geprüft werden soll.
