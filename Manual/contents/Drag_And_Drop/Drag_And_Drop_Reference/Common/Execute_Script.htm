Skript ausführen

Diese Aktion kann zum Aufrufen einer script Funktion oder einer benutzerdefinierten Funktion verwendet werden. Im Allgemeinen wird diese Aktion nur für Legacy-Support verwendet und sollte nicht wirklich in aktuellen oder neuen Projekten eingesetzt werden. Zur Verwendung der Aktion zu verwenden, geben Sie das aufzurufende Asset script oder den Variablennamen der benutzerdefinierten Funktion an und liefern dann alle erforderlichen Argumente, d. h. alle Werte (Strings, Ganzzahlen, Ressourcenindizes usw.), die Sie in das Skript oder Funktion eingeben möchten, damit es sie verwenden kann. Sie können bei Bedarf weitere arguments hinzufügen, indem Sie auf das Plus-Symbol neben der Aktion klicken.

Einige scripts und benutzerdefinierte Funktionen können auch einen Wert für die spätere Verwendung in der Aktionsblockliste zurückgeben, und so können Sie die Zielvariable angeben, die diesen Rückgabewert speichern soll. Diese kann zuvor erstellt worden sein oder eine neue temporäre lokale Variable sein (wenn Sie das Kontrollkästchen "Temp" aktivieren). Sowohl das Ziel- als auch das argument -Feld können leer bleiben, wenn sie nicht benötigt werden.

Beachten Sie, dass es im Allgemeinen besser ist, die Aktion Funktionsaufruf zu verwenden, um benutzerdefinierte Funktionen aufzurufen, und dass Sie diese Aktion im Allgemeinen nicht verwenden oder ein Asset von script direkt in code aufrufen müssen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Script Der Name der aufzurufenden script oder benutzerdefinierten Funktion.
Argument0 ... Argument3 Die verschiedenen arguments (Werte), die an die script oder die Funktion übergeben werden sollen (unbenutzte arguments können leer gelassen werden)
Target Der Name der Variablen, auf die die zurückgegebenen Werte abzielen sollen (kann leer bleiben)
