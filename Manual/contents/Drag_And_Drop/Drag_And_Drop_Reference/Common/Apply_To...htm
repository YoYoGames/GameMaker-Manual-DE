Bewerben bei...

Diese Aktion kann verwendet werden, um den Aktionsumfang für alle weiteren Aktionen zu ändern, die mit dieser Aktion verkettet sind. Wie bei einer if Aktion, wenn Sie weitere Aktionen rechts neben der Aktion Anwenden auf... ablegen, werden sie von dem Objekt oder der Instanz mit dem Geltungsbereich ausgeführt. Der Geltungsbereich wird aus dem Popup-Menü ausgewählt, das angezeigt wird, wenn Sie auf den kleinen Pfeil oben rechts neben der Aktion klicken, und auf diese Weise können Sie Auf diese Weise können Sie einem anderen Satz von Objekten - oder sogar einer einzelnen Instanz, wenn Sie die Instanz-ID haben - mitteilen, dass mehrere Aktionen ausgeführt werden sollen, bevor Sie mit der code des aktuellen Objekts fortfahren.

Beachten Sie, dass Aktionen, die Sie in den Block "Gilt für" einfügen möchten, an der Seite der Aktion abgelegt werden müssen, wie in der folgenden Abbildung gezeigt:

Diese Aktionen werden nun von dem Objekt oder der Instanz ausgeführt, auf die der "Gilt für", während alle Aktionen, die an anderer Stelle abgelegt werden, nach dem "Gilt für" Block ausgeführt werden.
