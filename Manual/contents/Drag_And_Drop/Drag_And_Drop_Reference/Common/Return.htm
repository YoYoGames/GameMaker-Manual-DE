Rückkehr

Diese Aktion wird bei der Erstellung von Aktionsblockfunktionen verwendet und dient der Rückgabe eines Wertes an die Instanz, die die script aufruft. Der Rückgabewert kann alles sein, was eine Variable sein - wie eine Zeichenkette, ein numerischer Wert, ein Zeiger oder eine Ressourcen-ID - oder er kann selbst eine Variable sein. Beachten Sie, dass beim Aufruf der Rückgabe-Aktion in einer Funktion die Funktion beendet wird - und den angegebenen Wert zurückgibt - und keine weiteren Aktionen nach der Rückgabe aufgerufen werden, Wenn Sie jedoch nach dem Funktionsaufruf Aktionen im Ereignis haben, werden diese weiterhin ausgeführt.

Aktion Syntax:

Argumente:

Argument Beschreibung
Return Der Rückgabewert.

Beispiel:

Die obige Aktionsblockfunktion setzt eine temporäre Variable auf das Schlüsselwort noone, und prüft dann eine Eingangsvariable, um festzustellen, welchen Wert sie derzeit hat. Wenn die Variable größer als 100 ist, wird eine Instanz erstellt und ihre eindeutige Instanz-ID in der temporären Variable gespeichert, die dann von der Funktion zurückgegeben wird.
