Ausfahrt

Je nach dem Kontext, in dem sie aufgerufen wird, führt diese Aktion eine von zwei Aktionen aus. Wenn sie innerhalb eines Ereignisses aufgerufen wird, verlässt sie das Ereignis und geht zum nächsten über, wobei sie alle nachfolgenden Aktionsblöcke code überspringt. Jedoch Wenn sie jedoch innerhalb einer Aktionsblockfunktion verwendet wird, wird sie nur die Funktion verlassen und zu dem Ereignis zurückkehren, das die Funktion aufgerufen hat, wobei alle nachfolgenden Aktionen ausgeführt werden. Beachten Sie, dass der Aufruf in der Mitte einer Funktion keinen Wert zurückgibt (verwenden Sie dafür die return Aktion).
