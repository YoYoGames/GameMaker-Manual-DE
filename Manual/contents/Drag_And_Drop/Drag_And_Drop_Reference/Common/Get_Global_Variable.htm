Globale Variable holen

Diese Aktion wird verwendet, um den Wert einer globalen Variablen abzurufen und ihn einer anderen Instanz oder einer temporären (lokalen) Variablen zuzuweisen. Die Variable, der Sie den globalen Wert zuweisen, kann eine zuvor erstellte sein, oder Sie können das Kontrollkästchen "Temp" ankreuzen und einen neuen Variablennamen angeben, damit die Aktion eine neue temporäre (lokale) Variable speziell für den zurückgegebenen Wert anlegt.

Beachten Sie, dass Sie in derselben Aktion weitere Variablen hinzufügen können, indem Sie auf das Plus-Symbol neben der Aktion klicken und eine weitere globale Variablennamen und einen temporären lokalen Variablennamen angeben. Weitere Informationen über Variablen finden Sie im Abschnitt über Variablen und Variablenumfang.

Aktion Syntax:

Argumente:

Argument Beschreibung
Name Der Name der globalen Variable, deren Wert ermittelt werden soll.
Target Die Zielvariable, die den zurückgegebenen Wert enthält - sie sollte zuvor erstellt worden sein, wenn "Temp" nicht angekreuzt ist (siehe unten).
Temp Erstellen Sie eine temporäre (lokale) Variable, die den zurückgegebenen Wert enthält (standardmäßig ausgeschaltet).
