Die "else"-Aktion wird immer nur in Verbindung mit der "if"-Variable (oder einer anderen "if"-Aktion) verwendet und ist eine Möglichkeit, eine Reihe von Aktionen zu definieren, die ausgeführt werden sollen, wenn die "if"-Auswertung false ergibt. Du könntest zum Beispiel eine "if"-Aktion haben, die prüft, ob die Gesundheitsvariable des Spielers kleiner oder gleich 0 ist, und wenn das Ergebnis "true" ist, zerstörst du die Instanz. Wenn du jedoch eine andere Aktion durchführen möchtest code Aktionen durchführen willst, wenn die Gesundheit des Spielers nicht kleiner als 0 ist, kannst du ein "else" hinzufügen und dann die Aktionen angeben, die durchgeführt werden sollen, während das "if" false auswertet.
