Eine neue Funktion deklarieren

Diese Aktion kann verwendet werden, um eine neue benutzerdefinierte Funktion zu erstellen, die später jederzeit in Ihrem code aufgerufen werden kann. Eine benutzerdefinierte Funktion ist eine Funktion, die Sie für eine Aufgabe definieren, die nicht von den eingebauten GML Visual-Aktionen oder GML runner Funktionen abgedeckt wird, und die global, instanziell oder lokal sein kann (weitere Informationen über den Anwendungsbereich finden Sie hier ); sie kann auch als Konstruktor zur Erstellung von Strukturen verwendet werden. Um mehr über die Verwendung dieser Aktion zur Erstellung globaler Funktionen zu erfahren, lesen Sie bitte den Abschnitt über die Erstellung von Aktionsfunktionen.

Um diese Aktion zu verwenden, können Sie sie zu einem Ereignis oder einem Skript hinzufügen und der neuen Funktion einen Namen geben. Anschließend geben Sie die erforderlichen Argumente/Parameter an, die die Funktion übernehmen soll, sowie die Standardwerte für alle optionalen arguments.

Wenn die Funktion als Konstruktor verwendet werden soll (der zur Erstellung einer Struktur verwendet wird), müssen Sie die Option Konstruktor markieren und später die Aktion Neu verwenden, um eine Struktur aus der Funktion zu erstellen. Sie können die Funktion auch als statisch kennzeichnen, was bedeutet, dass sie nur einmal definiert wird, egal wie oft diese Aktion ausgeführt wird. Wenn die Funktion nur lokal in demselben Ereignis und nirgendwo anders verwendet werden soll, können Sie die Option Temp aktivieren.

Sobald Sie den Namen und die Parameter definiert haben, können Sie die Aktionen, die beim Aufruf der Funktion in Ihrem code ausgeführt werden sollen, anhängen, indem Sie sie an der Seite der Aktion ablegen, wie in der folgenden Abbildung gezeigt:

Beispiele für die Verwendung dieser Aktion sowohl als reguläre Funktion als auch als Konstruktorfunktion finden Sie im Abschnitt "Erweiterte Beispiele" weiter unten.

Aktion Syntax:

Argumente:

Argument Beschreibung
Name Der Name der neuen Funktion.
Argument [+] Die verschiedenen arguments, die an die Funktion übergeben werden können, und ihre Standardwerte, wenn sie optional sind.
Constructor Ob diese Funktion ein Konstruktor ist.
Static Ob diese Funktion statisch ist.
Temp Ob die Funktion temporär ist (d.h. nur innerhalb des aktuellen Ereignisses verwendet wird).
