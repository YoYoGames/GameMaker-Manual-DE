Neue Struktur

Diese Aktion wird nur verwendet, wenn Strukturen mit einer Funktion erstellt werden, die als Konstruktorfunktion gekennzeichnet wurde (siehe Deklarieren einer neuen Funktion für weitere Informationen). Diese Aktion übernimmt die Funktionsmethode für eine zuvor definierte Funktion sowie eine Reihe von arguments, die Sie bei Bedarf mit dem Symbol auf der linken Seite erweitern können, um weitere hinzuzufügen. Diese arguments sollten den Eingaben entsprechen, die von der Funktion benötigt werden, und werden verwendet, um die zu erstellende Struktur zu füllen. Die Struktur wird an die Zielvariable zurückgegeben, die als temporäre lokale Variable gekennzeichnet sein kann oder nicht.

Aktion Syntax:

Argumente:

Argument Beschreibung
Function Der Name der aufzurufenden script oder benutzerdefinierten Funktion.
Argument0 ... Argument3 Die verschiedenen arguments (Werte), die an script oder die Funktion übergeben werden sollen (nicht verwendete arguments können leer gelassen werden)
Target Der Name der Variablen, auf die die zurückgegebenen Werte abzielen sollen (kann leer bleiben)
