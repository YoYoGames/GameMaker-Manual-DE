Wenn ein Objekt am Ort

Diese Aktion wird verwendet, um zu prüfen, ob sich eine Instanz eines beliebigen Objekts an einer Position befindet, die auf der Kollisionsmaske der aufrufenden Instanz basiert. Sie geben die Position an, die eine absolute Position oder eine Position relativ zur Instanz sein kann, und die Aktion gibt true zurück, wenn es eine Kollision gibt, oder false, wenn es keine gibt. Du kannst auch das "Not"-Flag ankreuzen, um zu prüfen, ob es an der angegebenen Position keine Kollision gibt, und die Aktion gibt dann true zurück, wenn keine Kollisionen gefunden wurden, und andernfalls false. Kollisionen werden auf der Grundlage der Kollisionsmaske der aufrufenden Instanz berechnet, die sich mit der Kollisionsmaske einer beliebigen Instanz an der Position überschneidet (als ob sie an der Position "platziert" würde).

WICHTIG! Kollisionen werden nur für die Instanzen registriert, die eine gültige Kollisionsmaske haben, d.h. sie haben eine sprite, die der sprite_index zugeordnet ist, oder eine sprite, die der mask_index zugeordnet ist. Wenn eine der Instanzen in der Kollision keine Kollisionsmaske hat, wird die Kollision nicht erkannt, unabhängig davon, was die Instanz zu diesem Zeitpunkt zeichnet.
