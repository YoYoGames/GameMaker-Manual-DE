Wenn Buffer vorhanden ist

Diese Aktion kann verwendet werden, um zu prüfen, ob eine gegebene buffer ID gültig ist und die buffer tatsächlich im Speicher existiert. Wenn die buffer ID gültig ist und existiert, gibt die Aktion true zurück, andernfalls false. Sie können "not" markieren angeben, damit die Aktion prüft, ob eine buffer nicht existiert.
