Buffer Größe abrufen

Diese Aktion kann verwendet werden, um die Größe eines buffer in Bytes abzurufen. Sie geben den eindeutigen buffer ID-Wert an, der von der Aktion Puffer erstellen zurückgegeben wird, und geben dann eine Zielvariable an, an die der Größenwert zurückgegeben werden soll. Die Zielvariable kann als temporäre lokale Variable gekennzeichnet werden. In diesem Fall wird sie für Sie erstellt und am Ende der aktuellen Aktion script oder des Ereignisses wieder aus dem Speicher entfernt.

Aktion Syntax:

Argumente:

Argument Beschreibung
Buffer Der Index buffer (gespeichert in einer Variablen)
Target Die Zielvariable, die den zurückgegebenen Wert enthält
