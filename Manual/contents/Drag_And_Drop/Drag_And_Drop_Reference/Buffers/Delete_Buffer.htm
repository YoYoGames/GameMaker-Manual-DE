Puffer löschen

Mit dieser Aktion kann ein buffer aus dem Speicher gelöscht werden, wobei die zu seiner Erstellung verwendeten Ressourcen freigegeben werden und alle Daten, die er derzeit enthält, entfernt werden. Sie geben den eindeutigen buffer ID-Wert an, der von der Aktion Puffer erstellen zurückgegeben wird. Beachten Sie, dass Sie mehrere buffers zum Löschen auswählen können, indem Sie auf das Plus-Symbol neben der Aktion klicken, und eine weitere eine weitere buffer ID hinzufügen.

HINWEIS: Es ist wichtig, dass Sie dynamisch erstellte Ressourcen immer aus dem Speicher entfernen, wenn Sie sie nicht mehr benötigen, um Speicherlecks zu vermeiden.
