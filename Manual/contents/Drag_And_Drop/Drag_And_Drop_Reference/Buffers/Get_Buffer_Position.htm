Buffer Position abrufen

Diese Aktion kann verwendet werden, um die aktuelle Position für das Lesen/Schreiben auf buffer abzurufen. Wenn Sie eine buffer erstellen, wird die Lese-/Schreibposition auf 0 Bytes gesetzt, und wenn Sie Daten zu buffer hinzufügen, bewegt sich diese Position sequentiell weiter, was bedeutet, dass Sie können diese Aktion verwenden, um die Anzahl der Bytes in buffer abzurufen, auf die sie zu jeder Zeit eingestellt ist. Sie geben den eindeutigen buffer ID-Wert an, der von der Aktion Puffer erstellen zurückgegeben wird, und geben dann eine Zielvariable an, an die den Positionswert zurückzugeben. Die Zielvariable kann als temporäre lokale Variable gekennzeichnet werden. In diesem Fall wird sie für Sie erstellt und am Ende der aktuellen Aktion script oder des Ereignisses wieder aus dem Speicher entfernt.

Aktion Syntax:

Argumente:

Argument Beschreibung
Buffer Der Index buffer (gespeichert in einer Variablen)
Position Die Lese-/Schreibposition (in Bytes) innerhalb des Puffers
