Schreibpuffer

Diese Aktion kann verwendet werden, um Daten in einen buffer zu schreiben. Sie geben die eindeutige buffer ID an, in die geschrieben werden soll (wie von der Aktion Puffer erstellen zurückgegeben), und dann den zu schreibenden Wert, der entweder ein realer Wert oder ein String sein kann, aber beachten Sie, dass Beachten Sie jedoch, dass alle Werte als Strings gespeichert werden. Wenn Sie also aus buffer lesen (mit Read Buffer), müssen Sie möglicherweise eine der Datentyp-Aktionen verwenden, um den Wert von einem String zurück in den ursprünglichen Wert zu konvertieren. Denken Sie auch daran, dass buffers sequentiell gelesen und beschrieben wird, so dass das Schreiben auf buffer die "Suchposition" um eine Position verschiebt. so oft wie nötig aufrufen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Buffer Der Index buffer (gespeichert in einer Variablen)
Value Der Wert, der dem Puffer hinzugefügt werden soll
