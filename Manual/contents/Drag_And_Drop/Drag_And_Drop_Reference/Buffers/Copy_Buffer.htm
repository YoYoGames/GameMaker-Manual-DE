Puffer kopieren

Mit dieser Aktion können alle Daten, die in einem buffer gespeichert sind, in einen anderen kopiert werden. Wenn Sie zwei buffers verwenden und von einem in den anderen kopieren, müssen beide buffers zuvor mit der Aktion Puffer erstellen erstellt worden sein, und Sie geben die eindeutige buffer ID an, die als von/bis arguments zurückgegeben wird.

HINWEIS: Sie können nicht auf dieselbe buffer kopieren.

Aktion Syntax:

Argumente:

Argument Beschreibung
Buffer Der buffer Index (gespeichert in einer Variablen), der von
Copy To Der Index buffer (in einer Variablen gespeichert), der nach

Beispiel:

Der obige Aktionsblock code erstellt eine neue buffer und speichert deren ID in einer temporären Variablen. Anschließend wird geprüft, ob die in der angegebenen globalen Variablen indizierte buffer existiert. Ist dies der Fall, wird sie in die temporäre buffer kopiert und die Lese-/Schreibposition auf den Anfang zurückgesetzt; ist sie nicht vorhanden, wird sie erstellt.
