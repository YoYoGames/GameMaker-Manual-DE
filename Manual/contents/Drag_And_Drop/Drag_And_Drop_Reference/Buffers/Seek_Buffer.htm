Suchpuffer

Diese Aktion kann verwendet werden, um eine Position innerhalb von buffer anzusteuern (zu suchen). Sie geben den eindeutigen buffer ID-Wert an, der von der Aktion " Puffer erstellen" zurückgegeben wird, und dann den Versatzwert in Bytes, der angesteuert werden soll. Der Versatzwert wird Der Offset-Wert wird vom Anfang des buffer aus angewendet, so dass der Aufruf dieser Funktion die Lese-/Schreibposition an den Anfang des buffer verschiebt, dann die angegebene Anzahl von Bytes weiterrückt und die neue Lese-/Schreibposition dort positioniert. Wenn Sie jedoch als relativ kennzeichnen, wird die Lese-/Schreibposition von der aktuellen Position aus nach vorne (oder nach hinten, wenn der Offset ein negativer Wert ist) verschoben.
