Puffer erstellen

Diese Aktion kann verwendet werden, um einen Teil des Speichers als buffer in Ihrem Spiel zuzuweisen, wobei die Funktion den eindeutigen Puffer-ID-Wert zurückgibt, der in einer Variablen gespeichert und für alle weiteren Funktionsaufrufe an den buffer verwendet werden sollte. Sie können eine globale Variable oder eine Instanzvariable angeben oder die Aktion eine temporäre lokale Variable zum Speichern der buffer ID erstellen lassen. Beachten Sie, dass Sie weitere buffers erstellen können, indem Sie auf das Plus-Symbol neben der Aktion klicken und eine weitere Variable für die buffer ID auswählen.

HINWEIS: Es ist wichtig, dass Sie alle dynamisch erstellten Ressourcen wie diese aus dem Speicher entfernen, wenn Sie sie nicht mehr benötigen, um Speicherlecks zu vermeiden. Wenn Sie also mit der von Ihnen erstellten buffer fertig sind, sollten Sie sie wieder freigeben mit der Aktion Puffer löschen (z. B. im Ereignis Sonstige - Auf räumen). Dies ist besonders wichtig, wenn Sie eine temporäre lokale Variable, denn wenn Sie die buffer nicht vor dem Ende des Ereignisses oder der Aktion script wieder entfernen, können Sie nicht mehr auf sie zugreifen, was später zu Problemen führen kann.

Aktion Syntax:

Argumente:

Argument Beschreibung
Buffer Der Index buffer (gespeichert in einer Variablen)
Value Der Wert, der in die Liste aufgenommen werden soll
