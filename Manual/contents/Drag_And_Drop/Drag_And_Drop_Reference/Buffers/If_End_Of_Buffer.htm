Wenn Ende des Puffers

Mit dieser Aktion kann geprüft werden, ob die Lese-/Schreibposition das Ende des buffer erreicht hat. Sie geben den eindeutigen buffer ID-Wert an, der von der Aktion Puffer erstellen zurückgegeben wird, und die Aktion gibt true zurück, wenn die Lese-/Schreibposition das Ende aller verfügbaren Daten im buffer erreicht hat, oder false, wenn dies nicht der Fall ist. Sie können "not" markieren, damit die Aktion prüft, ob die Lese-/Schreibposition nicht am Ende des buffer liegt.
