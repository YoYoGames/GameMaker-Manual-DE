Kachelindex bei Pixel abrufen

Diese Aktion wird verwendet, um einen Kachelindex auf einer bestimmten Ebene zu erhalten. Die Ebene wird anhand des Ebenennamens angegeben (eine Zeichenkette, wie sie für die Benennung der Ebene im Raumeditor verwendet wird), und dann geben Sie eine x- und y-Position an, um die Kachel zu finden, deren Index ermittelt werden soll. Die Kachel ist diejenige, auf die die angegebene Position im Raum fällt, und der Index wird an die Zielvariable zurückgegeben, wobei 0 für keine Kachel steht und eine positive ganze Zahl den Index der Kachel angibt. Die x und y Position kann als relativ zur Position der aufrufenden Instanz gekennzeichnet werden der Position der aufrufenden Instanz gekennzeichnet werden, und die Zielvariable kann als temporäre (lokale) Variable gekennzeichnet werden, die bis zum Ende des Ereignisses verwendet werden kann.

Aktion Syntax:

Argumente:

Argument Beschreibung
Layer Die Ebene, auf die die Kachel gesetzt werden soll
x Die zu prüfende x-Position
y Die zu prüfende y-Position
Target Die Zielvariable zum Speichern des Kachelindexes

Beispiel:

Der obige Aktionsblock code prüft, ob die linke Maustaste und ermittelt in diesem Fall den Index der Kachelzelle, in die die Mausposition fällt. Dieser Kachelindex wird dann überprüft, und wenn er nicht 0 ist, wird er auf 0 gesetzt.
