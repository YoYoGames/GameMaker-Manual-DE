Satz Kachelsatz

Mit dieser Aktion können Sie eine Ebene so einstellen, dass sie andere Kachelsätze verwendet als die, die im Asset-Browser erstellt wurden. Sie geben den Namen der Ebene an (eine Zeichenkette, wie im Raum-Editor definiert), und dann Alle Kacheln im Raum auf dieser Ebene werden dann mit dem neuen Kachelsatz gezeichnet.

Aktion Syntax:

Argumente:

Argument Beschreibung
Layer Die Ebene, auf die das neue Kachelset gesetzt werden soll.
Tileset Der Kachelsatz, der für die Ebene verwendet werden soll.

Beispiel:

Der obige Aktionsblock code prüft, ob die Fliesen auf der Ebene "Floor_Tiles" das das Kachelset "tl_PalaceRuins" verwenden, und wenn nicht, dann werden sie so eingestellt, dass sie es verwenden.
