Kachelindex in Zelle setzen

Mit dieser Aktion können Sie eine Kachel auf einer bestimmten Ebene auf einen neuen Index setzen. Die Ebene wird anhand des Ebenennamens angegeben (eine Zeichenkette, wie sie zur Benennung der Ebene im Raumeditor verwendet wird), und dann geben Sie die Zeile und Spalte der Zelle innerhalb der Ebene an, um die die Kachel zu setzen. Schließlich geben Sie den Kachelindex an, auf den die Kachel gesetzt werden soll (die Kacheln werden von links nach rechts von oben nach unten indiziert, mit 0 oben links). Beachten Sie, dass Sie den Kachelindex jederzeit auf 0 setzen können, um die Kachel zu löschen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Layer Die Ebene, auf die die Kachel gesetzt werden soll
Column Die Spalte (von links nach rechts) der Zelle
Row Die Zeile (von oben nach unten) der Zelle
Tile Der Kachelindex, auf den die Zelle gesetzt werden soll
