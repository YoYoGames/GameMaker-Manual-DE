Kacheldaten in Zelle abrufen

Mit dieser Aktion können Sie die Kacheldaten für eine Kachelzelle auf einer Kachelkartenebene abrufen. Die Kacheldaten sind einfach ein Wert, der den Index der Kachel zusammen mit den Werten für Spiegeln, Drehen und Spiegeln wiedergibt, sowie alle benutzerdefinierten Kachelmasken, die Sie verwendet haben. Mit dieser Aktion geben Sie zunächst die Ebene an, die Sie anvisieren möchten, indem Sie den Namen der Ebene (eine Zeichenkette) wie im Raumeditor definiert angeben, und dann die Spalte und Zeile der Kachelkartenebene, für die Sie die Kacheldaten zu erhalten. Die "Zelle" ist der Bereich auf dem Kachelkartenraster, der die Kachel enthält, von der Sie die Daten erhalten möchten. Wenn Ihre Kacheln beispielsweise 16x16 groß sind und der Raum 1024x768 groß ist, hat die Kachelkarte 64 Spalten und 48 Zeilen.

HINWEIS: Die ausgewählte Ebene muss im Raumeditor als Kachelkartenebene definiert worden sein, andernfalls können Fehler auftreten.

Der zurückgegebene Wert wird dann in der Zielvariablen gespeichert, die zuvor erstellt worden sein kann oder eine neue temporäre Variable sein kann (wenn Sie das Kontrollkästchen "Temp" aktivieren). Weitere Informationen zu Kacheldaten finden Sie unter GML im Abschnitt über Kachelkartenfunktionen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Layer Die Ebene, aus der die Kachel stammen soll
Column Die Spalte (von links nach rechts) der Zelle

Der obige Aktionsblock code loops durchläuft jede Kachelzelle im Raum, ruft die die Kacheldaten für die Zelle, spiegelt sie und setzt die Zelle dann wieder.
