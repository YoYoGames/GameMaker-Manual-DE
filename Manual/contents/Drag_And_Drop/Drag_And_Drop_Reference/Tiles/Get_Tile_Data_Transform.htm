Kacheldaten abrufen Transformieren

Mit dieser Aktion können Sie herausfinden, ob die Kacheldaten für eine Kachel auf eine der drei folgenden Arten transformiert wurden: gespiegelt, gedreht oder gedreht. Die Kacheldaten sind einfach ein Wert, der den Index der Kachel zusammen mit den angewandten Transformationen widerspiegelt, und Sie werden Normalerweise sollten Sie die Kacheldaten zuerst mit Get Tile Data In Cell oder Get Tile Data At Pixel abrufen, bevor Sie diese Aktion verwenden. Die Aktion gibt true zurück, wenn Die Aktion gibt false zurück, wenn die gewählte Transformation auf die Kacheldaten angewendet wurde, oder andernfalls . Der zurückgegebene Wert wird dann in der Zielvariablen gespeichert, die zuvor erstellt worden sein kann oder eine neue temporäre Variable sein kann (wenn Sie das Kontrollkästchen "Temp" Kontrollkästchen). Weitere Informationen über Kacheldaten finden Sie unter GML im Abschnitt über Kachelkartenfunktionen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Tile Die Kacheldaten für die Kachel, von der die Transformation abgerufen werden soll
Transform Die zu prüfende Transformation
