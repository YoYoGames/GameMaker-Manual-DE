Kachelindex bei Pixel einstellen

Mit dieser Aktion können Sie eine Kachel auf einer bestimmten Ebene auf einen neuen Index setzen. Die Ebene wird anhand des Ebenennamens angegeben (eine Zeichenkette, wie sie für die Benennung der Ebene im Raumeditor verwendet wird), und dann geben Sie eine x- und y-Position an, um die zu setzende Kachel zu erhalten. Die x- und y-Position können als relativ zur Position der aufrufenden Instanz gekennzeichnet werden, und die zu setzende Kachel wird diejenige sein, in die die angegebene Position im Raum fällt, und der verwendete Kachelsatz wird derjenige sein, der der Ebene im Raum Editor. Schließlich gibst du den Kachelindex an, auf den die Kachel gesetzt werden soll (Kacheln werden von links nach rechts von oben nach unten indiziert, mit 0 oben links). Beachten Sie, dass Sie den Kachelindex jederzeit auf 0 setzen können, um die Kachel zu löschen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Layer Die Ebene, auf die die Kachel gesetzt werden soll
x Die zu prüfende x-Position
y Die zu prüfende y-Position
Tile Der neue Index der zu verwendenden Kachel

Der obige Aktionsblock code prüft, ob die linke Maustaste gedrückt wurde und wird der Index der Kachelzelle ermittelt, in die die Mausposition fällt. Dieser Kachelindex wird dann überprüft, und wenn er nicht 0 ist, wird er auf 0 gesetzt.
