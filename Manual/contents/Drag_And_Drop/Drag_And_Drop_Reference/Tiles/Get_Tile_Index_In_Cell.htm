Kachelindex in Zelle abrufen

Mit dieser Aktion können Sie einen Kachelindex auf einer bestimmten Ebene abrufen. Die Ebene wird anhand des Ebenennamens angegeben (eine Zeichenkette, wie sie für die Benennung der Ebene im Raumeditor verwendet wird), und dann geben Sie die Zeile und Spalte der Zelle innerhalb der Ebene an, aus der Sie den Kachel Index zu erhalten. Schließlich geben Sie die Zielvariable an, in der der zurückgegebene Kachelindex gespeichert werden soll, wobei 0 für keine Kachel und eine positive Ganzzahl für den Index der Kachel steht. Beachten Sie, dass die Zielvariable als temporäre (lokale) Variable gekennzeichnet werden kann, die bis zum Ende des Ereignisses verwendet werden kann.

Aktion Syntax:

Argumente:

Argument Beschreibung
Layer Die Ebene, aus der die Kachel stammen soll
Column Die Spalte (von links nach rechts) der Zelle
Row Die Zeile (von oben nach unten) der Zelle
Target Die Zielvariable für den Rückgabewert
