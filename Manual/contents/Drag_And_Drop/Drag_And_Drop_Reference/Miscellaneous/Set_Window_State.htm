Fensterstatus festlegen

Mit dieser Aktion können Sie den Status des Spielfensters entweder auf Vollbild oder Fenster einstellen. Hier gibt es drei Optionen:

  • Umschalten - Wenn diese Option ausgewählt ist, wird das Fenster bei jedem Aufruf der Aktion zwischen dem Vollbild- und dem Fensterzustand umgeschaltet
  • Fullscreen - Damit wird das Fenster vom Fensterzustand in den Vollbildmodus versetzt
  • Windowed - Das Fenster kehrt in den Fenstermodus zurück, wenn es sich im Vollbildmodus befindet.

Beachten Sie, dass das Aussehen des Spiels im Vollbildmodus von den Spieloptionen - Grafikeinstellungen Ihres Ziels platforms abhängt, denn wenn das Spiel keinen Mechanismus zur Anpassung an unterschiedliche Wenn das Spiel keinen Mechanismus zur Anpassung an unterschiedliche Seitenverhältnisse und Auflösungen hat, kann es zu einer Streckung kommen (wenn die Spieloption auf Vollbild eingestellt ist), oder es können schwarze Balken hinzugefügt werden (wenn die Spieloption auf Seitenverhältnis beibehalten eingestellt ist). Wenn Sie keine Vorkehrungen für unterschiedliche Auflösungen getroffen haben mit unterschiedlichen Auflösungen umgehen können, sollten Sie für das Ziel platforms, für das Sie kompilieren möchten, die Option Seitenverhältnis be ibehalten verwenden.
