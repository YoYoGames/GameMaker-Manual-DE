Debug-Meldung anzeigen

Mit dieser Aktion können Sie eine Debug-Meldung an die Compiler-Ausgabe senden. Diese Meldung kann entweder eine Zeichenkette oder eine reelle Zahl sein, aber wenn Sie beides verwenden möchten, muss die Zahl in eine Zeichenkette umgewandelt werden (siehe die Aktion Zahl in Zeichenkette). Beachten Sie, dass Sie mit dieser Aktion mehrere Meldungen ausgeben können, indem Sie auf das Symbol klicken, um die Aktion mit weiteren Ausgabefeldern zu erweitern.

Aktion Syntax:

Argumente:

Argument Beschreibung
Message Die Meldung, die im Ausgabefenster angezeigt werden soll
