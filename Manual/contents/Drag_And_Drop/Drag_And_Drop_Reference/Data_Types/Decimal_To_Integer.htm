Dezimal in Integer

Mit dieser Aktion wird eine Dezimalzahl in eine ganze Zahl umgewandelt. Sie geben den zu konvertierenden Wert an und wählen dann die Art der Rundung aus, die vorgenommen werden soll, um ihn in einen ganzzahligen Wert umzuwandeln. Folgende Rundungsarten sind möglich:

  • Runden: Nimmt eine Dezimalzahl und rundet sie auf die nächste Ganzzahl auf oder ab. In dem besonderen Fall, dass die angegebene Zahl genau eine halbe ganze Zahl ist (z. B. 1,5, 17,5, -2,5 usw.), wird die Zahl auf geraden Wert gerundet, z. B. würde 2,5 auf 2 gerundet, während 3,5 auf 4 gerundet würde. Diese Art der Rundung wird Banker-Rundung genannt und ergibt bei einer großen Anzahl von Iterationen oder bei Verwendung von Fließkomma-Mathematik eine statistisch eine statistisch bessere Rundung als der traditionellere Ansatz "aufrunden, wenn über 0,5, sonst abrunden".
  • Ceil: Nimmt einen Dezimalwert und rundet ihn auf die nächste Ganzzahl auf. Im Gegensatz zu round (oben) wird hier der Dezimalteil des Wertes nicht berücksichtigt, so dass eine Zahl wie 5.0000001 immer noch auf 6 aufgerundet wird. Es ist erwähnenswert Wenn Sie die Zufallsaktion " Zufallszahl ermitteln " verwenden, um einen zufälligen Dezimalwert zu erzeugen, und dann diese Aktion aufrufen, um ihn aufzurunden, können Sie unerwartete Ergebnisse erhalten. Die Aktion Random Aktion könnte möglicherweise 0 zurückgeben, und die Verwendung von "Ceil" auf diese Aktion ergibt nicht 1, sondern 0. Wenn Sie also eine zufällige Dezimalzahl von 0 bis 9 erhalten und diese Funktion verwenden, erwarten Sie möglicherweise, dass Sie ganze Zahlen von 1 bis 10 erhalten, aber in Wirklichkeit könnten Sie ganze Zahlen von 0 bis 10 erhalten. Dies ist eine unwahrscheinliche Möglichkeit, sollte aber bei der Verwendung dieser Aktion in Betracht gezogen werden.
  • Stockwerk: Mit dieser Funktion wird ein Dezimalwert auf den nächsten ganzzahligen Wert abgerundet. Im Gegensatz zum Runden (oben) wird der Dezimalteil des Wertes nicht berücksichtigt, so dass eine Zahl wie 5,9999999 trotzdem auf 5.

Nachdem Sie den Rundungstyp ausgewählt haben, müssen Sie die zu rundende Zahl angeben (dies kann eine Variable sein) und dann eine Zielvariable, an die der neue Ganzzahlwert zurückgegeben werden soll (die als temporäre lokale Variable gekennzeichnet werden kann).

Aktion Syntax:

Argumente:

Argument Beschreibung
Rounding Die Art der Rundung, die verwendet wird, um die ganze Zahl zu erhalten (siehe Beschreibung oben)
Number Die Dezimalzahl, die auf eine ganze Zahl gerundet werden soll
Target Die Zielvariable zur Speicherung der zurückgegebenen Ganzzahl
