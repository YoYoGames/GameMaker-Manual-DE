Nummer in Zeichenkette

Diese Aktion wandelt eine reelle Zahl in eine Zeichenkette um. Wenn die Zahl keine ganze Zahl ist und Dezimalstellen hat, werden diese auf die nächsten zwei Stellen gerundet, so dass zum Beispiel die Zahl 123,45678 die Zeichenkette "123,46" ergeben würde.

Aktion Syntax:

Argumente:

Argument Beschreibung
Value Die zu konvertierende reelle Zahl
Target Die Zielvariable, die den zurückgegebenen Wert enthält

Beispiel:

Der obige Aktionsblock code nimmt einen Dezimalwert und wandelt ihn in eine ganze Zahl um, und speichert den zurückgegebenen Wert in einer temporären lokalen Variablen. Dieser Wert wird dann in eine Zeichenkette umgewandelt und auf dem Bildschirm dargestellt.
