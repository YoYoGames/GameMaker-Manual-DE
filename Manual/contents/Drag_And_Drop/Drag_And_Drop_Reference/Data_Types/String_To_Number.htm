String in Zahl

Diese Aktion kann eine Zeichenkette in eine Zahl umwandeln. Sie geben die zu ändernde Zeichenkette und dann eine Zielvariable an, an die die reale Zahl zurückgegeben werden soll (diese Variable kann als temporäre lokale Variable gekennzeichnet werden). Wenn die Zeichenkette Zeichen enthält Wenn die Zeichenkette jedoch mit einer Zahl beginnt, wird diese umgewandelt und der Rest der Zeichenkette ausgeschlossen. Es ist wichtig zu beachten, dass bei der Übergabe von Dezimalzahlen an diese Aktion Dezimalzahlen übergeben werden, wird die zurückgegebene Zeichenkette nur auf zwei Dezimalstellen gerundet. Nachstehend finden Sie eine Tabelle mit den Eingabewerten und den zu erwartenden Rückgaben

Zeichenfolge Zurückgegebener Wert
"123" 123
"123 Hello World" 123
"Hello World" 0 (standardmäßig wird 0 zurückgegeben, wenn der String nicht konvertiert werden kann)
"Hello World 123" 0
"123.45678" 123.46

Aktion Syntax:

Argumente:

Argument Beschreibung
Value Die zu konvertierende Zeichenkette
Target Die Zielvariable, die den zurückgegebenen Wert enthält

Beispiel:

Der obige Aktionsblock code initialisiert eine globale Variable und lädt dann eine Datei mit mit einer darin gespeicherten buffer. Zwei Zeichenketten werden aus der buffer gelesen und in einer temporären lokalen Variablen gespeichert, bevor eine von ihnen in eine reelle Zahl umgewandelt und in einer anderen temporären lokalen Variablen gespeichert wird. Diese werden dann globalen Variablen zugewiesen, und die buffer wird gelöscht.
