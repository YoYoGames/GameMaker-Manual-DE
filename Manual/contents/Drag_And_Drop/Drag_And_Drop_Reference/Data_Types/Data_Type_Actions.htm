Datentypen Aktionsbibliothek

GameMaker hat eine Reihe verschiedener Datentypen, wie z.B. Integer, Strings, Pointer, etc... und einige von ihnen können in andere umgewandelt werden. Dieser Abschnitt des Handbuchs zeigt die verschiedenen Aktionen, die für den Umgang mit verschiedenen Datentypen und deren Konvertierung zur Verfügung stehen.

HINWEIS: Weitere Informationen zu den verschiedenen Datentypen finden Sie hier.

Die Bibliothek " Typen " verfügt über die folgenden Aktionen:

Dezimal in Integer
String in Zahl
Nummer in Zeichenkette
