Draw Alpha erhalten

Mit dieser Aktion können Sie den allgemeinen Alphawert abrufen, der beim Zeichnen verwendet wird. Die Aktion gibt den aktuellen Alphawert an die von Ihnen angegebene Zielvariable zurück, die zuvor erstellt worden sein kann oder eine neue temporäre Variable sein kann (wenn Sie das Kontrollkästchen "Temp" Kontrollkästchen).
