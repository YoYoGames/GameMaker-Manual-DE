Zeichnen Sprite Transformiert

Diese Funktion zeichnet das angegebene sprite ähnlich wie die Aktion Sprite zeichnen, aber mit zusätzlichen Optionen, um die Skalierung, das Blending, die Rotation und den Rahmen des sprite zu ändern. Das Ändern dieser Werte verändert nicht Die Ressource wird in keiner Weise verändert (nur die Art und Weise, wie sie gezeichnet wird), und Sie können alle verfügbaren Sprite-Eigenschaftsvariablen anstelle von direkten Werten für alle arguments in der Funktion verwenden. Das folgende Bild veranschaulicht wie sich verschiedene Werte auf das Zeichnen des Sprites auswirken:

HINWEIS Die Farbüberblendung wird für das HTML5-Ziel nur empfohlen, wenn WebGL aktiviert ist, obwohl Sie die Überblendungsfarbe auch dann einstellen können, wenn sie nicht aktiviert ist, und die sprite wird normal überblendet. Allerdings wird beim Überblenden auf diese Weise ein Duplikat sprite das dann in cache gespeichert und bei Bedarf verwendet wird. Dies ist bei weitem nicht optimal, und wenn Sie mehrere Farbwechsel verwenden, wird die Leistung Ihres Spiels verlangsamt, es sei denn, Sie aktivieren WebGL. Wenn Sie WebGL nicht verwenden möchten, können Sie die die Größe der Schrift cache einstellen, um dies zu begrenzen, wenn es notwendig ist. sprite_set_cache_size().

HINWEIS: Diese Aktion ist nur für die verschiedenen Zieh-Ereignisse gedacht und wird nicht gezogen, wenn sie anderweitig verwendet wird.

Aktion Syntax:

Argumente:

Argument Beschreibung
Sprite Die sprite zu zeichnen
Frame Der sprite Frame, der gezeichnet werden soll (verwenden Sie die eingebaute Variable image_index für den aktuellen Frame)
X Die X-Position, an der innerhalb des Raums gezeichnet werden soll
Y Die y-Position, an der innerhalb des Raums gezeichnet werden soll
Scale X Der Skalierungsfaktor für die X-Achse (verwenden Sie die eingebaute Variable image_xscale für die aktuelle X-Skala)
Scale Y Der Skalierungsfaktor für die Y-Achse (verwenden Sie die eingebaute Variable image_yscale für die aktuelle Y-Skala)
Rotation Der Ziehwinkel für die sprite (von 0 bis 360, gegen den Uhrzeigersinn drehend, wobei 0 nach rechts ist - verwenden Sie die eingebaute Variable image_angle für die aktuelle Drehung)
Colour Die Farbe, die mit sprite gemischt werden soll (Standard ist weiß, und Sie können die eingebaute Variable image_blend für die aktuelle Farbe verwenden)

Der obige Aktionsblock code prüft eine Variable, und wenn sie auf true gesetzt ist, wird sprite mit zufälliger Skalierung entlang der x- und y-Achse gezeichnet und mit der Farbe Rot überblendet, andernfalls wird sprite mit den Standardtransformationen gezeichnet.
