Schriftart einstellen

Mit dieser Aktion können Sie die Schriftart festlegen, die für alle nachfolgenden Aktionen zum Zeichnen von Text verwendet werden soll. Wenn Sie auf den Dialog klicken, öffnet sich das folgende Fenster, in dem Sie die gewünschte Schriftart auswählen können:

Beachten Sie, dass damit die Schriftart für alle nachfolgenden Aufrufe zum Zeichnen von Text festgelegt wird und kann also einmal zu Beginn des Spiels aufgerufen werden - in jedem Fall, nicht nur beim Draw Event - und alle Draw Text Aktionen werden die eingestellte Schriftart verwenden. Wenn Sie jedoch mehrere Schriftarten in Ihrem Spiel benötigen, müssen Sie diese Aktion aufrufen, bevor Sie Text mit der gewünschten Schriftart zu zeichnen.
