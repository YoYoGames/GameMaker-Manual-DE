Sprite zeichnen

Mit dieser Aktion wird ein sprite irgendwo im Raum gezeichnet. Du gibst die sprite, die Position (entweder eine absolute Position innerhalb des Raumes oder eine relative Position zu der Instanz, die die Zeichnung ausführt) und den Animationsrahmen der sprite an, wenn sie mehrere Frames hat (beachte, dass Bildframes von 0 aufwärts nummeriert sind). Wenn du eine animierte sprite hast und du willst, dass sie die Frames so zeichnet, als ob sie dem Objekt zugewiesen wäre, dann verwende -1 für den Frame-Index oder die eingebaute Instanzvariable image_index, Aber beachte, dass wenn der Instanz keine sprite zugewiesen ist, sie nicht animiert wird, und wenn die zugewiesene sprite eine andere Anzahl von Frames hat als die, die gezeichnet wird, dann wird die sprite mit zusätzlichen Frames oder fehlenden Frames gezeichnet.

WICHTIG: Diese Aktion zeichnet das sprite nicht transformiert, auch wenn Sie die Bildüberblendung oder den Maßstab geändert haben. Siehe dazu die Aktion Sprite transformiert zeichnen.

HINWEIS: Diese Aktion ist nur für die verschiedenen Zieh-Ereignisse gedacht und wird nicht gezogen, wenn sie anderweitig verwendet wird.

Aktion Syntax:

Argumente:

Argument Beschreibung
Sprite Die sprite zu zeichnen
x Die x-Position, an der gezeichnet werden soll
y Die y-Position, an der gezeichnet werden soll
Frame Index Der zu zeichnende Rahmenindex
