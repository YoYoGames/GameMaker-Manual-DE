Selbst zeichnen

Wenn Sie dem Draw Event eines Objekts eine beliebige Aktion oder code hinzufügen, teilen Sie GameMaker mit, dass Sie die Kontrolle übernehmen und ihm sagen, was er von nun an zeichnen soll (standardmäßig zeichnet eine Instanz die sprite, die ihr im Objekteditor zugewiesen wurde im Objekteditor, wenn es keine Aktionen im Hauptzeichenereignis gibt). Das bedeutet, dass jede zugewiesene sprite nicht mehr gezeichnet wird, es sei denn, du sagst es explizit, dann würdest du diese Aktion verwenden. Dies zeichnet einfach die zugewiesene sprite für das Objekt mit allen Transformationen, die in früheren Ereignissen festgelegt wurden (wie Skalierung, Überblendung oder Alpha-Änderungen) in das aktuelle Teilbild.

HINWEIS: Diese Aktion ist nur für die verschiedenen Zieh-Ereignisse gedacht und wird nicht gezogen, wenn sie anderweitig verwendet wird.
