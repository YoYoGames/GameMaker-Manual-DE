Linie zeichnen

Mit dieser Aktion wird eine Linie an einer bestimmten Position im Raum gezeichnet. Sie geben die anfängliche x/y-Position und dann die endgültige x/y-Position an und die Linie wird zwischen diesen beiden Positionen mit der aktuellen Zeichenfarbe gezeichnet. Die Positionen können entweder eine absolute Position innerhalb des Raumes sein oder eine Position relativ zu der Instanz, die die Aktion aufruft.

HINWEIS: Diese Aktion ist nur für die verschiedenen Draw-Events gedacht und wird bei einer anderen Verwendung nichts zeichnen.

Aktion Syntax:

Argumente:

Argument Beschreibung
x Die x-Startposition
y Die y-Startposition
x2 Die endgültige X-Position
y2 Die letzte y-Position

Beispiel:

Der obige Aktionsblock code zeichnet eine rosafarbene Linie über den oberen Teil des Raums.
