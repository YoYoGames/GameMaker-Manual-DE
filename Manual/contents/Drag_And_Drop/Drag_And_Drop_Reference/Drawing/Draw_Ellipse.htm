Ellipse zeichnen

Mit dieser Aktion wird eine Ellipse an einer bestimmten Position im Raum gezeichnet. Sie geben die obere linke Position und die untere rechte Position des Bereichs an, in den die Ellipse "passen" soll, und die Ellipse wird zwischen diesen beiden Positionen gezeichnet. Die Positionen können entweder eine absolute Position innerhalb des Raums oder eine Position relativ zu der Instanz sein, die die Aktion aufruft, und Sie können auch festlegen, ob die Ellipse gefüllt oder umrandet werden kann, indem Sie das Füllkästchen am unteren Rand aktivieren.

HINWEIS: Diese Aktion ist nur für die verschiedenen Zieh-Ereignisse gedacht und wird nicht gezogen, wenn sie anderweitig verwendet wird.

Aktion Syntax:

Argumente:

Argument Beschreibung
Left Die linke Position, von der aus gezeichnet werden soll
Top Die oberste Position, von der aus gezeichnet wird
Right Die richtige Position, um die Zeichnung zu beenden
Bottom Die unterste Position zum Beenden der Zeichnung von
