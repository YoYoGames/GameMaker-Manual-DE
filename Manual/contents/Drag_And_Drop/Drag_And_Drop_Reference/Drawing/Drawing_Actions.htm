Bibliothek der Zeichnungsaktionen

In der Bibliothek der Zeichnungsaktionen finden Sie die Aktionen, die zum Zeichnen von sprites, Text oder Formen sowie zum Setzen bestimmter Zeicheneigenschaften. Die meisten dieser Aktionen sind nur für die Verwendung in den verschiedenen Zeichnungsereignissen eines Objekts vorgesehen und funktionieren möglicherweise nicht, wenn sie außerhalb des des Draw Events verwendet werden. Eine Ausnahme bilden die Set-Aktionen, die zu jedem Ereignis hinzugefügt werden können und sich anschließend auf das Zeichnen aller Instanzen auswirken.

Es ist wichtig zu beachten, dass, wenn Sie irgendwelche Aktionen in das Hauptzeichen-Ereignis eines Objekts einfügen, es nicht die sprite zeichnen wird, die der Instanz zugewiesen wurde, es sei denn, Sie sagen GameMaker explizit, dass er sie zeichnen soll, indem Sie eine Aktion wie Draw Self. Grundsätzlich zeichnet GameMaker standardmäßig alle sprite, die einer Instanz zugewiesen sind, nur dann, wenn es nichts anderes im Draw Event gibt.
