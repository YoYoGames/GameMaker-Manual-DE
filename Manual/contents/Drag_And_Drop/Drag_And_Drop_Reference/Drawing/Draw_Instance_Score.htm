Draw Instance Score

Mit dieser Aktion kannst du den Punktestand der Instanz zeichnen. Wenn du eine Punkteaktion in GameMaker verwendest, wird der Instanz, die sie aufruft, eine neue Instanzbereichsvariable namens score hinzugefügt. Diese Variable kann dann geprüft werden und gesetzt werden, wie jede andere Variable auch. Im Fall dieser Aktion wird die Variable an der angegebenen Position auf den Bildschirm gezeichnet, zusammen mit einem beliebigen Beschriftungstext, den du angeben möchtest. Der Beschriftungstext wird vor dem Punktwert gezeichnet, und wenn Wenn es sich um Text handelt, sollte er mit "" umschlossen werden, und beide Texte werden mit der aktuellen Schriftart, Farbe und Ausrichtung formatiert.

HINWEIS: Diese Aktion ist nur für die verschiedenen Zieh-Ereignisse gedacht und wird nicht gezogen, wenn sie anderweitig verwendet wird.

Aktion Syntax:

Argumente:

Argument Beschreibung
Caption Die Zeichenkette, die zusammen mit dem Punktwert gezeichnet werden soll (vorangestellt und kann leer gelassen werden)
x Die x-Position, an der die Partitur gezeichnet werden soll
y Die y-Position, an der die Partitur gezeichnet werden soll
