Transformierten Wert zeichnen

Mit dieser Aktion können Sie eine Zeichenkettenbeschriftung und/oder eine Variable oder einen Wert auf den Bildschirm zeichnen. Bei der Beschriftung argument handelt es sich um eine Zeichenfolge, die Sie dem Wert voranstellen möchten, und die bei Bedarf leer gelassen werden kann. eine andere Zeichenkette sein - in diesem Fall sollte sie in Anführungszeichen gesetzt werden und kann bei Bedarf auch leer gelassen werden. Dann geben Sie die Position an, an der die Zeichenkette gezeichnet werden soll, entweder als absolute Position oder als Position relativ zu der Instanz, die den Text zeichnet. Zusätzlich können Sie die Skalierung des Textes festlegen, indem Sie ihn entweder entlang der horizontalen oder der vertikalen Achse strecken, sowie die Drehung (die Winkel für die Drehung des Textes sind gegen den Uhrzeigersinn, so dass die Einstellung 90° ihn nach links und -90° nach rechts dreht nach rechts). Die Skalierung erfolgt mit dem angegebenen Faktor, d. h. eine Skalierung von 0,5 halbiert die Breite des Textes, eine Skalierung von 2 verdoppelt die Breite, wobei 1 der Standardwert für keine Skalierung ist. Beachten Sie, dass das Aussehen des gezeichneten Textes von folgenden Faktoren abhängt der verwendeten Schriftart, der Farbe und der Ausrichtung ab. All diese Dinge können vor dem Zeichnen geändert werden, wie im folgenden Beispiel gezeigt.

HINWEIS: Diese Aktion ist nur für die verschiedenen Zieh-Ereignisse gedacht und wird nicht gezogen, wenn sie anderweitig verwendet wird.

Aktion Syntax:

Argumente:

Argument Beschreibung
Caption Die zu zeichnende Beschriftung (eine Zeichenkette), kann leer gelassen werden
Value Der zu zeichnende Wert (Real, Variable oder String)
x Die x-Position, an der gezeichnet werden soll
y Die y-Position, an der gezeichnet werden soll
xscale Der x (horizontale) Skalierungsfaktor, Standard ist 1
yscale Der y (vertikale) Skalierungsfaktor, Standard ist 1
Rotation Der Drehwinkel für das Zeichnen des Textes
