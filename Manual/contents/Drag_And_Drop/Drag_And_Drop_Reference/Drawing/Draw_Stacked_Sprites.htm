Gestapelte Sprites zeichnen

Diese Aktion zeichnet eine sprite eine bestimmte Anzahl von sprites nacheinander an einer bestimmten Position im Raum. Sie geben die zu zeichnenden sprite und die Stapelreihenfolge an, die entweder Zeile (horizontal, von links nach rechts) oder Spalte (vertikal, von oben nach unten), sowie die Anzahl der zu zeichnenden sprites und die Position. Die Position kann eine absolute Position innerhalb des Raumes sein, oder eine relative zur Position der Instanz, die die Zeichnung ausführt, und der Abstand zwischen den Bildern basiert auf Die Abstände zwischen den Bildern basieren auf der Breite oder Höhe von sprite. Beachte, dass dies einfach ein statisches Bild zeichnet - das anfängliche Einzelbild (Frame 0) des gegebenen sprite - und alle weiteren Frames werden ignoriert, ebenso wie alle Transformationen, die durch das Ändern der Instanzvariablen hinzugefügt wurden (wie image_xscale oder image_blend).

HINWEIS: Diese Aktion ist nur für die verschiedenen Zieh-Ereignisse gedacht und wird nicht gezogen, wenn sie anderweitig verwendet wird.

Aktion Syntax:

Argumente:

Argument Beschreibung
Sprite Die sprite zu zeichnen
Stack Order Die Reihenfolge, in der gezeichnet werden soll (entweder Zeile oder Spalte)
Number Die Anzahl der zu zeichnenden sprites
x Die x-Position, an der gezeichnet werden soll
y Die y-Position, an der gezeichnet werden soll

Beispiel:

Der obige Aktionsblock code ermittelt die Anzahl der Instanzen des Objekts "obj_Player" und verwendet diese dann, um eine Anzahl von Markern sprites auf den Bildschirm zu zeichnen.
