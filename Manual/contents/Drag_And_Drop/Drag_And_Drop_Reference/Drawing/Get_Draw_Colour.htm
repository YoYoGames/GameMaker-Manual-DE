Get Draw Farbe

Mit dieser Aktion können Sie die Farbe ermitteln, die für beliebige Formen (Rechteck/Ellipse/Linie) oder für gezeichneten Text verwendet wird. Die Aktion gibt die Farbe an die von Ihnen angegebene Zielvariable zurück, die zuvor erstellt worden sein kann oder eine neue temporäre Variable sein (wenn Sie das Kontrollkästchen "Temp" aktivieren).
