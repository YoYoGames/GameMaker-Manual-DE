Zeichnen Alpha einstellen

Mit dieser Aktion können Sie den Alphawert für alle Formen (Rechteck/Ellipse/Linie) oder für jeden Text, der später gezeichnet wird, festlegen. Beachten Sie, dass diese Aktion nicht vor jeder Zeichnungsaktion aufgerufen werden muss, sondern nur einmal zu Beginn des Spiels gesetzt werden kann. Alpha ist die Einstellung, die steuert, wie "transparent" Dinge gezeichnet werden und basiert auf einer Skala von 0 bis 1, wobei 0 völlig transparent und 1 völlig undurchsichtig ist.

Aktion Syntax:

Argumente:

Argument Beschreibung
Alpha Der zu verwendende Alphawert (von 0 bis 1, Standard ist 1)
