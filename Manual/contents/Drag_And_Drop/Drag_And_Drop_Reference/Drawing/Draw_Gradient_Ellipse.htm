Ellipse mit Farbverlauf zeichnen

Mit dieser Aktion wird eine Ellipse an einer bestimmten Position im Raum gezeichnet, wobei eine Reihe von Mischfarben verwendet wird, um einen Farbverlauf zu erstellen. Sie geben die obere linke Position und die untere rechte Position des Bereichs an, in den die Ellipse "passen" soll, und die Ellipse wird zwischen diesen beiden Positionen gezeichnet. Die Position kann entweder eine absolute Position innerhalb des Raums sein oder eine Position relativ zu der Instanz, die die Aktion aufruft, und Sie können die Farben so einstellen, dass sie vom Rand der Ellipse und von ihrer Mitte aus übergehen. Die Ellipse kann gefüllt oder umrandet gezeichnet werden, indem man das Füllkästchen am unteren Rand aktiviert.

HINWEIS: Diese Aktion ist nur für die verschiedenen Zieh-Ereignisse gedacht und wird nicht gezogen, wenn sie anderweitig verwendet wird.

Aktion Syntax:

Argumente:

Argument Beschreibung
Left Die linke Position, von der aus gezeichnet werden soll
Top Die oberste Position, von der aus gezeichnet wird
Right Die richtige Position, um die Zeichnung zu beenden
Bottom Die unterste Position zum Beenden der Zeichnung von
Center Die Farbe zum Überblenden von der Mitte aus
Edge Die Farbe zum Überblenden vom Rand

Beispiel:

Der obige Aktionsblock code zeichnet eine Ellipse mit Farbverlauf an der Position der Instanz von "obj_Player".
