Textausrichtung abrufen

Diese Aktion ermittelt die Schriftausrichtung für alle aktuellen Aktionen zum Zeichnen von Text. Man kann wählen, ob die horizontale oder vertikale Ausrichtung geprüft werden soll, und die Aktion wird einen der folgenden konstanten Werte zurückgeben, die eine unterschiedliche Typ der horizontalen oder vertikalen Ausrichtung repräsentieren:

Der Text ist horizontal nach links ausgerichtet.
Text wird horizontal in der Mitte ausgerichtet
Der Text ist horizontal nach rechts ausgerichtet.
Text ist vertikal nach oben ausgerichtet
Text ist vertikal zur Mitte ausgerichtet
Text ist vertikal nach unten ausgerichtet

Die Rückgabekonstante wird in der von Ihnen angegebenen Zielvariablen gespeichert, die zuvor erstellt worden sein kann oder eine neue temporäre Variable sein kann (wenn Sie das Kontrollkästchen "Temp" aktivieren).
