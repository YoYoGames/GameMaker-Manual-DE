Textausrichtung festlegen

Mit dieser Aktion wird die Schriftausrichtung für alle nachfolgenden Aktionen zum Zeichnen von Text festgelegt. Sie können die horizontale Ausrichtung auf links, mittig oder rechts und die vertikale Ausrichtung auf oben, mittig oder unten einstellen, und der Text wird relativ zur x/y Position der Zeichenaktion ausgerichtet, wie in den folgenden Abbildungen gezeigt:

Der Text ist horizontal nach links ausgerichtet.
Text wird horizontal in der Mitte ausgerichtet
Der Text ist horizontal nach rechts ausgerichtet.
Text ist vertikal nach oben ausgerichtet
Text ist vertikal zur Mitte ausgerichtet
Text ist vertikal nach unten ausgerichtet

Beachten Sie, dass damit die Ausrichtung für alle nachfolgenden Zeichentextaufrufe festgelegt wird. Sie können diese Aktion also zum Beispiel einmal zu Beginn des Spiels aufrufen - in jedem Fall, nicht nur beim Ereignis "Zeichnen" - und alle Zeichentextaktionen werden die festgelegte Ausrichtung verwenden. Wenn Sie jedoch mehrere Zeichentext-Aktionen auf unterschiedliche Weise ausrichten möchten, müssen Sie diese Aktion vor jeder Zeichentext-Aktion aufrufen, die Sie anders ausrichten möchten.

Aktion Syntax:

Argumente:

Argument Beschreibung
H.Align Der Wert für die horizontale Ausrichtung (Links, Mitte, Rechts)
V.Align Der Wert für die vertikale Ausrichtung (Oben, Mitte, Unten)

Beispiel:

Der obige Aktionsblock code legt die Schriftart, die Zeichenfarbe und die Ausrichtung für einen Text der gezeichnet wird.
