Zeichnen Wert

Mit dieser Aktion können Sie eine Zeichenkettenbeschriftung und/oder eine Variable oder einen Wert auf den Bildschirm zeichnen. Die Beschriftung argument ist eine Zeichenkette, die dem Wert vorangestellt werden soll, und kann bei Bedarf leer gelassen werden. Der Wert kann eine Variable, ein fester Wert oder eine andere Zeichenkette sein - in diesem Fall sollte er ebenfalls in Anführungszeichen gesetzt werden -, darf aber nicht leer bleiben. Dann gibst du die Position an, an der die Zeichenkette gezeichnet werden soll, entweder als absolute Position oder als eine Position relativ zu der Instanz, die den Text zeichnet. Wie der Text aussieht, wenn er gezeichnet wird, hängt von der verwendeten Schriftart, der verwendeten Farbe und der Ausrichtung ab. All diese Dinge können vor dem Zeichnen geändert werden, wie im folgenden Beispiel gezeigt.

HINWEIS: Diese Aktion ist nur für die verschiedenen Draw-Events gedacht und wird bei einer anderen Verwendung nichts zeichnen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Caption Die zu zeichnende Beschriftung (eine Zeichenkette), kann leer gelassen werden
Value Der zu zeichnende Wert (Real, Variable oder String)
x Die x-Position, an der gezeichnet werden soll
y Die y-Position, an der gezeichnet werden soll

Beispiel:

Der obige Aktionsblock code legt eine Schriftart und die Schriftfarbe fest und richtet den Text so aus, dass er von der Mitte aus gezeichnet wird. Zentrum zu zeichnen, bevor der Wert "level_num" zusammen mit der Beschriftung "Level:" gezeichnet wird.
