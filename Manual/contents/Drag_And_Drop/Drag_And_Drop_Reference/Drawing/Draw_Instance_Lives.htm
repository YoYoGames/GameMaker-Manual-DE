Zeichnen Instanz lebt

Mit dieser Aktion wird ein sprite gezeichnet, um jedes der Leben darzustellen, die die Instanz gerade hat. Die zu zeichnende Zahl wird aus dem aktuellen Wert der lokalen Instanzvariable lives entnommen, die eine spezielle Variable ist, die automatisch innerhalb der Instanz erstellt wird der Instanz erstellt wird, wenn du eine der Aktionen verwendest, die sie benötigen. Sie geben die sprite für die Leben und die Stapelreihenfolge an, die entweder Zeile (horizontal, von links nach rechts) oder Spalte (vertikal, von oben nach unten) sein kann, sowie die Position. Die Position kann eine absolute Position innerhalb des Raumes sein, oder eine relative zur Position der Instanz die die Zeichnung macht, und der Abstand zwischen den Bildern basiert auf der Breite oder Höhe der sprite. Beachte das dies einfach ein statisches Bild zeichnet - das anfängliche Einzelbild (Frame 0) der angegebenen sprite - und alle weiteren Frames werden ignoriert, ebenso wie alle Transformationen, die der Instanz durch Ändern der Instanzvariablen (wie image_xscale oder image_blend) hinzugefügt wurden. Der Beschriftungstext wird vor dem Wert von lives gezeichnet, und wenn es sich um Text handelt, sollte er mit "" umschlossen werden, und beides wird mit der aktuellen Schriftart, Farbe und Ausrichtung formatiert.

HINWEIS: Diese Aktion ist nur für die verschiedenen Zieh-Ereignisse gedacht und wird nicht gezogen, wenn sie anderweitig verwendet wird.

Aktion Syntax:

Argumente:

Argument Beschreibung
Caption Die Zeichenkette, die zusammen mit dem Punktwert gezeichnet werden soll (vorangestellt und kann leer gelassen werden)
Stack Order Die Reihenfolge, in der gezeichnet werden soll (entweder Zeile oder Spalte)
x Die x-Position, an der das Leben gezeichnet werden soll
y Die y-Position, an der das Leben gezeichnet werden soll
