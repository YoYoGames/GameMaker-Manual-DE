Umgekehrt

Diese Aktion wird verwendet, um bestimmte Werte in Bezug auf Richtung und Bewegung umzukehren. Die verfügbaren Optionen sind:

Eine Richtungsumkehrung addiert einfach 180° zur aktuellen Richtung (ggf. in einer Schleife um 360°). Umkehrung der horizontalen oder vertikalen Geschwindigkeit multipliziert die angegebene Geschwindigkeit vector einfach mit -1, und die Wahl der Schwerkraft bewirkt dasselbe wie die Wahl der Richtung, nur für die die Richtung der Schwerkraft.

Aktion Syntax:

Argumente:

Argument Beschreibung
direction Die Richtung umkehren

Beispiel:

Der obige Aktionsblock code prüft auf eine Kollision der Instanz mit "obj_Wall" relativ zur aktuellen Position nach links und nach rechts. Wenn eine gefunden wird, wird die horizontale Geschwindigkeit umgekehrt. Das Gleiche gilt für den Bereich oberhalb und unterhalb der Instanz, wobei die vertikale Geschwindigkeit umgekehrt wird, wenn eine Kollision festgestellt wird.
