Richtung zufällig festlegen

Diese Aktion dient dazu, die Bewegungsrichtung der Instanz auf eine oder mehrere zufällige Richtungen aus den 8 Himmelsrichtungen zu setzen. Du kannst einen oder mehrere der 8 Richtungspfeile innerhalb der Aktion anklicken und die Aktion wählt eine davon zufällig als Richtung für die Instanz zu verwenden. Wenn du keinen der Pfeile auswählst, hat die Aktion keinen Einfluss auf die aktuelle Richtung der Instanz.

Beispiel:

Der obige Aktionsblock code wählt entweder true oder false und speichert das Ergebnis in einer temporären (lokalen) Variablen. Diese Variable wird dann überprüft, und wenn sie true lautet, wird ein Satz zufälliger Richtungen gewählt, wenn sie false lautet, wird ein anderer Satz gewählt.
