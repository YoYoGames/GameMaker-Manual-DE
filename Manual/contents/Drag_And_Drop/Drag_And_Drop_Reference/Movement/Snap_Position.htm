Fangposition

Diese Aktion wird verwendet, um eine Instanz an einer bestimmten Gitterkoordinatenposition "einzufangen". Sie geben die Größe der "Zelle" des Gitters an und die Instanz wird an diesem Gitter "gefangen". Wenn Sie beispielsweise die Werte für den Rasterfang horizontal und vertikal auf 8 setzen und die Instanz sich auf (4, 6) befindet, wird sie an der Position (0, 8) gefangen. vertikal auf 8 gesetzt hast, und die Instanz sich bei (4, 6) befindet, dann wird sie an der Position (0, 8) gefangen - im Wesentlichen wird die aktuelle Position auf den nächsten Wert gerundet, der durch die horizontalen und vertikalen Fangeinstellungen teilbar ist, wobei die Gitterzellen an der oberen linken Ecke des Raums (0, 0) beginnen. Die Standardwerte sind auf einen Fang von (32, 32) eingestellt, aber Sie können sie auf jeden beliebigen positiven ganzzahligen Wert ändern.

Aktion Syntax:

Argumente:

Argument Beschreibung
horizontal Der Wert der horizontalen Zelle, an dem gefangen werden soll
vertical Der Wert der vertikalen Zelle, an dem gefangen werden soll
