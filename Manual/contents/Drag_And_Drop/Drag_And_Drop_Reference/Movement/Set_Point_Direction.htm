Sollwert Richtung

Diese Aktion wird verwendet, um eine Richtung von einer vector zu bestimmen, die durch die aktuelle Position der Instanz und die angegebene Position gebildet wird. Du gibst die x- und y-Position an, auf die "gezeigt" werden soll - diese können als relativ zur Position der Instanz gekennzeichnet werden - und die Richtung von der Instanz zu diesem Punkt wird für die Instanz verwendet.

Aktion Syntax:

Argumente:

Argument Beschreibung
x Die x-Komponente der Richtung vector.
y Die y-Komponente der Richtung vector.

Beispiel:

Der obige Aktionsblock code prüft, ob eine Instanz des Objekts "obj_Player" im Raum existiert, und wenn dies der Fall ist, wird die Richtung der Instanz so eingestellt, dass sie auf die x/y-Position von "obj_Player" zeigt.
