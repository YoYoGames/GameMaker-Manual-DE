Reibung einstellen

Diese Aktion wird verwendet, um die "Reibung" für die Instanz einzustellen. Reibung bedeutet einfach, dass bei jedem Spielframe ein bestimmter Betrag von der Geschwindigkeit vector abgezogen wird, wobei du den abzuziehenden Betrag angibst. Normalerweise funktionieren kleinere Werte am besten, wie 0.2, aber der tatsächliche Betrag hängt von den Einstellungen deines Projekts ab. Wenn Sie das relative Flag setzen, wird der angegebene Betrag zum aktuellen Reibungswert addiert, anstatt ihn zu setzen.

Aktion Syntax:

Argumente:

Argument Beschreibung
friction Die anzuwendende Reibung (nur positive Werte)

Der obige Aktionsblock code legt verschiedene Instanzeigenschaften für die Bewegung fest: Die Richtung der Schwerkraft wird auf abwärts (270°) gesetzt, die Schwerkraft auf 0,05 und die Reibung auf 0,2.
