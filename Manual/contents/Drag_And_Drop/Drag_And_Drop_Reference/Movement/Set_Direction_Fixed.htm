Richtung festlegen Fest

Diese Aktion dient dazu, eine feste Bewegungsrichtung festzulegen. Die verfügbaren Richtungen werden durch die verschiedenen Schaltflächen im Aktionsblock angezeigt, und ein Klick auf eine von ihnen legt diese Richtung fest. Beachten Sie, dass Richtungen in GameMaker in Grad gemessen werden, wobei rechts 0° ist, oben 90°, links 180° und unten 270°.
