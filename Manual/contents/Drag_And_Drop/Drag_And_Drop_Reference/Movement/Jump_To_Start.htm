Sprung zum Start

Diese Aktion wird verwendet, um eine Instanz von ihrer aktuellen Position an die Position zu verschieben, an der sie ursprünglich erstellt wurde. Die ursprüngliche Position, an der eine Instanz erstellt wurde, wird in zwei Instanzvariablen namens xstart und ystart gespeichert. Diese Variablen können in jeder anderen Aktion verwendet werden und können auch gesetzt werden, um die aufgezeichnete Startposition zu ändern.
