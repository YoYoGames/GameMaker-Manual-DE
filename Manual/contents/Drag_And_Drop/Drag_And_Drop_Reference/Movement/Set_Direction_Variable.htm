Richtungsvariable einstellen

Diese Aktion wird verwendet, um eine Richtung für die Instanz festzulegen, die nicht auf 45°-Schritte festgelegt ist. Sie geben den Richtungswert von 0° bis 360° an - wobei rechts 0°, oben 90°, links 180° und unten 270° ist - und Sie können auch das relative Flag setzen, um den von Ihnen angegebenen Wert zur aktuellen Richtung zu addieren/subtrahieren.

Aktion Syntax:

Argumente:

Argument Beschreibung
Direction Richtung von 0° bis 360° (Werte außerhalb des Bereichs werden umbrochen).

Beispiel:

Der obige Aktionsblock code prüft, ob die linke Maustaste gedrückt ist. gedrückt wird. Ist dies der Fall, wird die Richtung auf einen Zufallswert von 0 bis 360 und die Geschwindigkeit auf 5 gesetzt.
