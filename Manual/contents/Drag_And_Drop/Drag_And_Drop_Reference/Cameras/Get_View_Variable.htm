Ansichtsvariable holen

Diese Aktion ermöglicht es Ihnen, den Wert bestimmter eingebauter Variablen in Bezug auf einen bestimmten Viewport abzurufen. Sie wählen die abzurufende Viewport-Variable aus, geben dann den Viewport als Ziel an (von 0 bis 7) und schließlich die Zielvariable, die den zurückgegebenen Wert enthalten soll (diese kann als temporäre lokale Variable gekennzeichnet werden). Der Ansichtsanschluss ist der Bereich des Bildschirms, der zum Zeichnen einer bestimmten Kameraansicht verwendet wird (wie im Raumeditor eingestellt), und die verschiedenen Variablen, die Sie auf diese Weise erhalten können sind in der folgenden Tabelle aufgeführt:

Variabel Beschreibung
Camera Der zurückgegebene Wert ist der eindeutige ID-Wert der Kamera, die dem Anschluss zugewiesen werden soll (weitere Informationen finden Sie hier ).
Visibility Dies gibt einen Wert von true oder false zurück, wobei true bedeutet, dass der View Port sichtbar ist (d.h. er wird auf den Bildschirm gezeichnet), und false bedeutet, dass nichts gezeichnet wird.
View Port X Coordinate Dies ist die X-Position des View Ports innerhalb des Spielfensters.
View Port Y Coordinate Dies ist die Y-Position des View Ports innerhalb des Spielfensters.
View Port Width Dies ist die Breite (in Pixel) des Ansichtsfensters.
View Port Height Dies ist die Höhe (in Pixeln) des View Ports.
View Port Surface ID Dies gibt den eindeutigen ID-Wert für die dem View Port zugewiesene Oberfläche zurück oder -1, wenn keine Oberfläche zugewiesen wurde.

Aktion Syntax:

Argumente:

Argument Beschreibung
Variable Die eingebaute Ansichtsvariable zum Abrufen
View Der anzuvisierende Ansichtsanschluss (von 0 bis 7)
Target Die Zielvariable, die den zurückgegebenen Wert enthält (kann als temporäre lokale Variable gekennzeichnet werden)

Beispiel:

Der obige Aktionsblock code ermittelt die aktuelle Kamera-ID, die dem View port [0] zugewiesen ist, und prüft dann, ob sie mit der in einer globalen Variablen gespeicherten identisch ist. Ist dies nicht der Fall, wird die Kamera auf die neue eingestellt.
