Variable Ansicht einstellen

Mit dieser Aktion können Sie bestimmte eingebaute Variablen einstellen, die sich darauf beziehen, wie das Ansichtsfenster die Kameraansichten des Raums anzeigt. Sie wählen die zu bearbeitende Ansichtsanschlussvariable, geben dann den Zielwert für den Ansichtsanschluss (von 0 bis 7) und schließlich den neuen Wert für die Variable. Der Ansichtsanschluss ist der Bereich des Bildschirms, der zum Zeichnen einer bestimmten Kameraansicht verwendet wird (wie im Raum-Editor eingestellt), und die verschiedenen Variablen, die Sie auf diese Weise einstellen können, sind in der folgenden Tabelle aufgeführt:

Variabel Beschreibung
Camera Hier können Sie das Ansichtsfenster so einstellen, dass eine andere Kameraansicht angezeigt wird. Sie müssen den eindeutigen ID-Wert der Kamera angeben, zu der Sie wechseln möchten (siehe hier für mehr Informationen).
Visibility Dies ist der Sichtbarkeitsschalter für den View Port und erwartet einen Wert von true oder false, wobei ein Wert von true den View Port sichtbar macht (d.h. er wird auf den Bildschirm gezeichnet) und ein Wert von false nichts auf den Bildschirm zeichnet. nichts auf den Bildschirm.
View Port X Coordinate Dies ist die X-Position des View Ports innerhalb des Spielfensters. Beachten Sie jedoch, dass dieser Wert nur benötigt wird, wenn mehr als ein Viewport aktiv ist - zum Beispiel, um einen Split-Screen-Effekt zu erzeugen - und wenn nur ein Port aktiv ist, erhalten Sie unerwartete Ergebnisse.
View Port Y Coordinate Dies ist die Y-Position des View Ports innerhalb des Spielfensters. Beachten Sie jedoch, dass dieser Wert nur benötigt wird, wenn mehr als ein Viewport aktiv ist - zum Beispiel, um einen Split-Screen-Effekt zu erzeugen - und wenn nur ein Port aktiv ist, erhalten Sie unerwartete Ergebnisse.
View Port Width Dies ist die Breite (in Pixeln) des Ansichtsfensters. Die Breite des View Ports (oder kombinierter View Ports, wenn mehr als einer aktiv ist) definiert die Breite des Spielfensters oder der Hintergrundleinwand zu Beginn des Spiels, so dass eine Änderung dieses Wertes nach dem Das Ändern dieses Wertes nach dem Spielstart hat keinen sichtbaren Effekt auf die Größe des Spielfensters, es sei denn, es wird zusammen mit der Funktion GML window_set_size(). Beachten Sie auch, dass die Kameraansicht verkleinert - oder vergrößert - wird, wenn Sie eine größere oder kleinere Portgröße als die der Kamera zugewiesene haben.
View Port Height Dies ist die Höhe (in Pixeln) des Ansichtsfensters. Die Höhe des View Ports (oder kombinierter View Ports, wenn mehr als einer aktiv ist) definiert die Höhe des Spielfensters oder der Hintergrundleinwand zu Beginn des Spiels, so dass eine Änderung dieses Wertes nach Das Ändern dieses Wertes nach dem Start des Spiels hat keinen sichtbaren Effekt auf die Größe des Spielfensters, es sei denn, es wird zusammen mit der Funktion GML window_set_size(). Beachten Sie auch, dass die Kameraansicht verkleinert oder vergrößert wird, wenn Sie eine größere oder kleinere Portgröße als die der Kameraansicht zugewiesene haben.
View Port Surface ID Mit dieser Variable können Sie den Inhalt eines gegebenen View Ports auf eine Oberfläche zeichnen, wobei der von Ihnen angegebene Wert derjenige der Oberfläche ist, auf die der View Port gezeichnet werden soll. Für weitere Informationen lesen Sie bitte hier.
