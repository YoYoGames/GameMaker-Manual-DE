Das Git Plugin einrichten

Bevor wir das Plugin Git verwenden können, müssen wir die ausführbare Datei Git einrichten, indem wir den Abschnitt Plugins - Versorgungskontrolle (Git) in den Voreinstellungen aufrufen:

Der Git Ausführungspfad sollte automatisch ausgefüllt werden, wenn er von GameMaker gefunden wird. Wenn das Feld jedoch leer ist, können Sie auf die Schaltfläche klicken und manuell die Datei git.exe auswählen, die sich im Ordner cmd Ihrer Git -Installation befindet, der in der Regel innerhalb von C:\Program Files\Git\ liegt.

Die übrigen Optionen von Git finden Sie auf der Seite Plugin-Einstellungen.
