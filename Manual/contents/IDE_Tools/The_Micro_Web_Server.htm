Der Micro Web Server

Die Hauptfunktion dieses kleinen Tools ist es, eine Verbindung zu einigen der verfügbaren Ziele von platforms für GameMaker (wie das HTML5-Ziel) herzustellen und Dateien bereitzustellen. Es stellt auch Dateien für das GameMaker-Handbuch bereit, wenn es offline verwendet wird.

Wenn Sie ein Spiel auf einem Ziel ausführen, für das Micro Web Server erforderlich ist, wird ein neues Fenster geöffnet, während das Spiel auf dem ausgewählten Gerät angezeigt wird:

HINWEIS Wenn Sie zum ersten Mal ein Spiel auf einem Ziel ausführen, für das das Micro Web Server erforderlich ist, erhalten Sie möglicherweise eine Firewall-Warnung. In diesem Fall sollten Sie dem Web server erlauben, durch die Firewall zu kommunizieren.

Wenn das Micro Web Server läuft, können Sie von jedem lokalen Gerät aus auf Ihr Spiel zugreifen (und sogar aus der Ferne, wenn die Portweiterleitung richtig eingerichtet ist). Greifen Sie einfach über den in diesem Fenster angezeigten Port auf die lokale IP-Adresse Ihres Computers zu, und schon können Sie Ihr Spiel von einem anderen Gerät, z. B. einem Mobiltelefon oder einem anderen Computer, spielen.

Nehmen wir an, die lokale IP-Adresse Ihres Computers lautet 192.168.0.110(lesen Sie hier, wie Sie sie herausfinden können) und der im Fenster von Micro Web Server angezeigte Port ist 51264. Nachdem Sie das Spiel auf Ihrem Computer gestartet haben, würden Sie sich über einen Browser auf Ihrem Handy mit 192.168.0.110:51264 verbinden und Ihr Spiel spielen.

Das Web Server zeigt eine bestimmte Anzahl von Debug-Informationen zu den Dateien an, die ausgeliefert wurden, und verfügt über eine Schaltfläche zum Löschen dieser Informationen am unteren Rand. Neben der Schaltfläche "Löschen" befindet sich der Pfad zur Stamm-JS-Datei für das Projekt, und darunter befindet sich die Option " Netzwerkgeschwindigkeit ". Diese ist standardmäßig auf "No Limit" eingestellt, aber wenn Sie darauf klicken, öffnet sich eine Liste mit verschiedenen Netzwerkoptionen:

Wenn Sie diese Einstellung ändern, ändert sich auch die Option Paketgröße auf der rechten Seite. Sie können diese Einstellung auch manuell ändern. In diesem Fall wird die Netzwerkgeschwindigkeit auf "Benutzerdefiniert" gesetzt. Wenn Sie hier einen anderen Wert als den Maximalwert einstellen, wird die Netzwerkübertragungsgeschwindigkeit effektiv gedrosselt und Sie können Spiele in einer Umgebung mit geringer Bandbreite testen.

Sie können auch die Liste der zulässigen IPs zusammen mit der verwendeten Portnummer sehen, aber diese können nicht im Micro Web Server bearbeitet werden. Sie sollten stattdessen in den Einstellungen des Gerätemanagers für das jeweilige Zielgerät oder in den HTML5-Voreinstellungen bei Verwendung des "Standard"-Geräts festgelegt werden.
