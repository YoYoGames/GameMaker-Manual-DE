Projekt Image Generator

Der Projektbild-Generator wird über das Menü "Werkzeuge" am am oberen Rand des GameMaker IDE. Mit diesem Tool können Sie schnell alle verschiedenen Splash-Screens, Promo-Bilder und Icons, die für die verschiedenen Zielsysteme benötigt werden, aus einem einzigen Satz von Bildern erstellen platforms. Sie geben das Basisbild und das Basissymbol ein und klicken dann auf die Schaltfläche " Generieren ". Die Spieloptionen für jedes platform werden dann mit den neuen Bildern aktualisiert.

Damit dies korrekt funktioniert, müssen Sie einen hochauflösenden Splash-Screen und ein hochauflösendes Symbolbild bereitstellen. Die empfohlenen Größen sind:

  • Splash Screen: mindestens 1920x1080px, oder 1080x1920px für Hochformat, und erstellt als 24bit *.png ohne Transparenzen.
  • Icons: mindestens 1024x1024px (das ist die Größe des größten Icons, das benötigt wird), und als 24bit *.png erstellt, das Transparenzen haben kann

Sobald Sie diese Bilder zur Verfügung gestellt haben, werden sie im Vorschaufenster angezeigt, und Sie können dann die Art der "Anpassung" einstellen, die bei der Anwendung auf die verschiedenen platforms verwendet werden soll. Sie können aus den folgenden Möglichkeiten wählen:

  • Seitenverhältnis beibehalten - das Bild wird so skaliert, dass es passt, wobei das Seitenverhältnis beibehalten wird und eine Hintergrundfarbe hinzugefügt wird, um die Seiten auszufüllen, wenn das Bild nicht 1:1 passt.
  • Auf Größe strecken - damit werden die Bilder auf die volle Größe des Bildes/Icons gestreckt.
  • Zuschneiden - das Basisbild wird entweder entlang der horizontalen oder der vertikalen Achse zugeschnitten und der Rest wird zugeschnitten. Wenn Sie zum Beispiel ein Bild von 640x480 haben, das als Splash-Screen für 480x800 verwendet werden soll, dann wird das Bild so skaliert, dass es auf die vertikale Achse passt (480 -> 800), und der Rest des Bildes wird weggeschnitten. Wenn Sie ein Bild mit einer Größe von 1080x1920 haben, das auf 800x800 passen muss, wird das Bild verkleinert, damit es auf die horizontale Achse angepasst und der Rest weggeschnitten.

Sie können auch die Hintergrundfarbe sowohl für Symbole als auch für Bilder festlegen. Diese Farbe wird verwendet, um das Bild bei der Skalierung mit dem Seitenverhältnis "aufzufüllen". Das folgende Bild zeigt, wie dies aussehen wird, wenn alles eingerichtet ist:

HINWEIS: Eine Ausnahme von der obigen Aussage bilden die adaptiven Android-Symbole. Diese werden aufgrund ihrer Struktur nicht mit diesem Tool erstellt.

WICHTIG! Auch wenn dieses Tool alle von Ihnen benötigten Bilder generiert, bedeutet das nicht, dass es perfekt ist! Es wird sein Bestes tun, um die Bilder an die verschiedenen Größen und Auflösungen der einzelnen platform anzupassen, aber manchmal kann es sein, dass sie nicht aber manchmal sieht es vielleicht nicht so aus, wie Sie es sich wünschen, denn es gibt keine "Einheitsgröße" für alle. Daher empfehlen wir Ihnen, nach dem Ausführen dieses Tools alle Bilder in den platforms Spieloptionen zu überprüfen und diejenigen, die nicht genau so aussehen, wie Sie es wünschen, manuell zu ändern. wünschen.
