Der Inspektor

Das Inspektorfenster zeigt alle Eigenschaften, die mit den ausgewählten Elementen in IDE verbunden sind.

Sie können in jedem Editor ein oder mehrere Elemente auswählen und ihre verschiedenen Eigenschaften anzeigen oder ändern:

Der Inspektor kann über das Menü Windows am oberen Rand des Fensters GameMaker geöffnet werden:

Wählen Sie diese Option, um ein neues Inspektor-Fenster zu öffnen - es können auch mehrere gleichzeitig geöffnet sein.

Nachdem Sie ein Inspektorfenster geöffnet haben, können Sie es an einer beliebigen Stelle auf IDE andocken oder es als schwebendes Fenster verwenden:

Verwendung des Inspektors

Mit dem Inspektor können Sie Assets anzeigen und bearbeiten, ohne sich ausschließlich auf die Arbeitsbereiche verlassen zu müssen.

So bearbeiten Sie z. B. Ihr Ereignis code in der Regel im Vollbildmodus und navigieren häufig zurück zu Workspace, um die Eigenschaften dieses Objekts zu bearbeiten.

Mit dem Inspektor können Sie die Details eines Objekts anzeigen, während Sie in der Vollbildansicht bleiben:

Dadurch ist es möglich, die Eigenschaften eines Objekts zu modifizieren und Ereignisse zu löschen/zu ändern, während die code im Vollbildmodus bearbeitet wird, und auch Elternschaft, Variablendefinitionen und mehr einzurichten.

In diesem Beispiel geht es zwar nur um Objekte, aber der Inspektor kann noch viel mehr und ermöglicht Ihnen die Bearbeitung vieler verschiedener Arten von Assets (z. B. Sprites, Schriftarten, Animationskurven usw.).

Editor-Elemente

Zusätzlich zu den Assets können Sie auch Elemente in einem Editor untersuchen, z. B. Ebenen in einem Raum, Spuren in einem Sequence usw.

Mit dem Inspektor können Sie einen Filter oder Effekt auf eine Raumebene anwenden, was im Raumeditor selbst nicht möglich ist:

HINWEIS Wenn Sie einen Filter/Effekt mit dem Inspektor auf eine Nicht-FX-Ebene anwenden, wird er immer als "Ein-Ebenen-Effekt" angewendet, d. h. seine Wirkung wird nur auf diese spezielle Ebene und nicht auf andere darunter liegende Ebenen angewendet.

Die Schaltfläche "Editor öffnen

Wenn Sie ein Asset ausgewählt haben, können Sie im Inspektor auf "Editor öffnen" klicken, um dessen Asset-Editor zu öffnen.

Wenn Sie dies zum Beispiel bei einem Objekt tun, wird der Objekteditor geöffnet, in dem Sie die Eigenschaften des Objekts ändern, Ereignisse hinzufügen usw. können.

Wenn Sie jedoch etwas innerhalb eines Raums bearbeiten, z. B. eine Instanz oder eine Ebene, konzentriert sich der Inspektor nur auf die Editoren innerhalb des Raumeditors.

Wenn du eine Instanz in einem Raum auswählst und im Inspektor auf "Editor öffnen" klickst, wird der Instanz-Editor geöffnet, wie oben gezeigt.

Wenn Sie eine Pfadebene auswählen und auf "Editor öffnen" klicken, wird der Fokus auf das Bedienfeld "Ebenen" verschoben, wo Sie die Eigenschaften der Ebene bearbeiten können, anstatt den Asset-Editor "Pfad" zu öffnen.

Mehrfachauswahl

Mit dem Inspektor können Sie die Eigenschaften mehrerer Assets und Elemente (z. B. Raumebenen und sequence Spuren) gleichzeitig anzeigen und bearbeiten.

Wählen Sie die gewünschten Assets oder Elemente aus, indem Sie STRG / CMD gedrückt halten und mit der linken Maustaste klicken. Die ausgewählten Elemente werden im Inspektor in einer "Mehrfachauswahl"-Liste angezeigt:

Drücken Sie auf das X neben einem Element, um es aus Ihrer Auswahl zu entfernen.

Werte editieren

Wenn Ihre ausgewählten Artikel denselben Wert für eine Eigenschaft haben, wird dieser gemeinsame Wert in diesem Feld angezeigt.

Wenn die Elemente jedoch unterschiedliche Werte für eine Eigenschaft haben, zeigt der Editor einfach - anstelle dieses Wertes an.

Sie können jedes dieser Felder ändern. Wenn Sie einen Wert ändern, wird er auf alle von Ihnen ausgewählten Elemente angewendet.

Auswahl innerhalb der Liste

Sie können ein beliebiges Element in Ihrer Mehrfachauswahlliste auswählen, um dessen Eigenschaften anzuzeigen und zu bearbeiten, ohne die Auswahl zurücksetzen zu müssen.

Klicken Sie auf den leeren Bereich in der Liste, um zur Mehrfachauswahl-Ansicht zurückzukehren.

Beachten Sie, dass Sie in dieser Liste nicht mehrere Einträge auswählen können.

Auswahl Highlight

Wenn Sie ein Element in der Mehrfachauswahlliste ausgewählt haben, wird es im Editor mit einer roten Umrandung angezeigt:

Sie können die Farbe dieses Umrisses in den Raum- und Sequenzvorgaben ändern.

Mehrere Inspektoren

Sie können jederzeit mehrere Inspektor-Fenster öffnen, und jedes kann mit einem bestimmten Element in IDE verknüpft werden.

Um sicherzustellen, dass sich nicht jedes Mal, wenn Sie auf ein Element klicken, alle geöffneten Inspektoren ändern, um die neue Auswahl anzuzeigen, müssen Sie zunächst jedes Inspektorenfenster für sich sperren .

Dazu öffnen Sie einen Inspektor, wählen ein zu inspizierendes IDE Element und klicken dann auf die Schaltfläche , um den Inspektor für das Element zu sperren (siehe Abbildung unten). Sobald Sie einen Inspektor gesperrt haben, können Sie einen anderen öffnen und den Vorgang wiederholen.

Beachten Sie, dass ein gesperrtes Inspektorfenster nicht die Bearbeitung von Werten für das zu inspizierende Element sperrt. Es sperrt lediglich den Inspektor für das ausgewählte Element und Sie können es jederzeit weiter bearbeiten, auch wenn das ursprüngliche Element nicht der Hauptfokus von IDE ist.
