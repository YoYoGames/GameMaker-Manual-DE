YoYo Compiler

Der YoYo Compiler (YYC) ist ein spezieller compiler für die Erstellung von ausführbaren Paketen, die Maschinencode anstelle von interpretiertem code und einem runner verwenden (was die GameMaker VM-Kompilierung verwendet). Die Kompilierung von code ist schneller, benötigt aber mehr Zeit für die Kompilierung der ausführbaren Datei.

Lizenz

Ihre GameMaker Lizenz kann die YoYo Compiler enthalten, muss es aber nicht. Weitere Informationen finden Sie auf der Seite Welches Abonnement ist das richtige für mich?

Das SDK einrichten

Damit der YoYo Compiler für das Ziel platform kompilieren kann, müssen Sie die richtigen SDKsinstalliert haben.

Eine allgemeine Übersicht hierzu finden Sie auf der folgenden Help Center Seite: GameMaker Erforderliche SDKs.

Informationen zu den erforderlichen SDKs für platform finden Sie ebenfalls im Help Center:

  • Oper GX
  • Windows
  • macOS
  • Ubuntu
  • Android
  • iOS (einschließlich iPadOS)
  • tvOS
