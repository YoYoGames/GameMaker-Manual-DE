Vorlage Info

In diesem Bereich können Sie Informationen für Ihr Projekt festlegen, die beim Export als Vorlage verwendet werden.

Hier können Sie dem Vorlagenprojekt ein Bild, ein Symbol und eine Beschreibung zuweisen, die jedoch nur verwendet werden, wenn es tatsächlich als Vorlage exportiert wird. Öffnen Sie dazu das Menü "Datei", klicken Sie auf "Projekt exportieren" und wählen Sie "Als Vorlage".
