Opera GX Spieloptionen

Die Opera GX-Spieloptionen ermöglichen es Ihnen, die Eigenschaften Ihres Spiels zu ändern. Die folgenden Abschnitte sind verfügbar:

Allgemein

Dieses Fenster zeigt den Status Ihrer Opera GX-Anmeldung an. Sie können auf das Symbol Aktualisieren klicken, um sich mit Ihrem Opera-Konto anzumelden.

Sie haben dann die folgenden Möglichkeiten:

  • Spiel auswählen: In dieser Liste können Sie entweder ein vorhandenes Spiel zur Aktualisierung auswählen oder ein neues Spiel hochladen.
  • Spiel-Name: Dies ist der Name (Titel) Ihres Spiels auf GX.games.

    WICHTIG Damit der Upload auf GX.games erfolgreich ist, ist es wichtig, dass Sie nur erlaubte Zeichen im Namen des Spiels verwenden (der standardmäßig auf den Namen Ihres GameMaker Projekts eingestellt ist, stellen Sie also sicher, dass Sie entweder einen gültigen Projektnamen wählen oder den Namen des Spiels anschließend manuell ändern).

    Wenn Sie ein Spiel mit einem Titel hochladen, der verbotene Zeichen enthält, erhalten Sie diese Fehlermeldung: Falsche Daten. Prüfen Sie die Ausgabe

    Die erlaubten Zeichen sind:
    • a-z, A-Z (mit Unterstützung von lokalen Schemata wie å)
    • 0-9
    • Whitespace
    • : - ! ? . , ' & ( )
  • Version: Dies ist die Version des Spiels, die zuletzt hochgeladen wurde. Sie können dieses Feld nur ändern, bevor Ihr Spiel auf GX.games hochgeladen wurde.
  • Nächste Version: Dies ist die Versionsnummer, die für Ihren nächsten GX.games-Upload verwendet wird. Sie können diese Versionsnummer erst ändern, nachdem Ihr Spiel auf GX.games hochgeladen wurde. Sie müssen dies nicht bei jedem Upload manuell ändern, da GameMaker die letzte Ziffer der Versionsnummer automatisch für Sie erhöht.
  • Auf GX.games registrieren: Klicken Sie auf diese Schaltfläche, um einen neuen Eintrag bei GX.games zu registrieren. Dies ist nur möglich, wenn "Spiel auswählen" auf "Neues Spiel" eingestellt ist.
  • Gruppe auswählen: Bevor Sie ein neues Spiel hochladen, können Sie das Studio auswählen, zu dem das Spiel gehören soll.
  • Spiel auf Opera bearbeiten: Öffnet GX.games DevCloud und führt Sie zur Seite "Spieldetails" Ihres Spiels. Diese ist nur anklickbar, wenn Sie eingeloggt sind und Ihr Spiel hochgeladen wurde.
  • Interne Freigabe-URL: Öffnet die private Version deines Spiels, nur wenn sie in der GX.games DevCloud aktiviert wurde.
  • Öffentliche Freigabe-URL: Öffnet die öffentliche Version deines Spiels, wenn sie in der GX.games DevCloud aktiviert wurde.

    Beide URLs können mit anderen geteilt werden, damit sie Ihr Spiel spielen können.

Grafiken

Hier können Sie die folgenden Details bezüglich der Anzeige Ihres Spiels ändern:

  • Mauszeiger anzeigen: Legt fest, ob der Mauszeiger beim Start des Spiels sichtbar ist. Nützlich für Spiele, die keine Mauseingabe erfordern oder ein eigenes Maus-Sprite verwenden.
  • Interpoliert die Farben zwischen den Pixeln: Schaltet die lineare Interpolation ein, die im Grunde die Pixel "glättet". Für scharfe Pixelgrafiken sollte sie ausgeschaltet sein, aber wenn Sie schöne Alphamischungen und geglättete Kanten haben, ist es besser, sie eingeschaltet zu lassen. Diese Option ist standardmäßig aktiviert.
  • Skalierung: Ihr Spiel kann so konfiguriert werden, dass die Zeichenfläche automatisch skaliert wird, um das Seitenverhältnis im Browser beizubehalten , oder Sie können wählen, dass es in voller Größe ausgeführt wird. Bei der Option "Voller Maßstab" wird das Spiel im Browser nicht im Vollbildmodus angezeigt, sondern das Gezeichnete wird so gestreckt, dass es in die Leinwandgröße passt, die durch den ersten Raum des Spiels definiert ist. Das Seitenverhältnis wird standardmäßig beibehalten.

Schließlich gibt es noch die Möglichkeit, die Größe der Texturseite festzulegen. Die Standardgröße (und die kompatibelste) ist 2048x2048, aber Sie können eine Größe zwischen 256x256 und 8192x8192 wählen. Es gibt auch eine Schaltfläche mit der Bezeichnung Vorschau die die Texturseiten für diese platform generiert und dann ein Fenster öffnet, in dem Sie sehen können, wie sie aussehen. Dies kann sehr nützlich sein, wenn Sie sehen möchten, wie die Texturseiten aufgebaut sind, und um zu verhindern, dass die Texturseiten größer (oder kleiner) als nötig sind. Weitere Informationen über Texturseiten finden Sie hier.
