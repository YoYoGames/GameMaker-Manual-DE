Der Raummanager

Wenn Sie den Asset-Browser verwenden, können Sie Ihre Räume erstellen und sie zwischen den verschiedenen Ordnergruppen verschieben, um das Projekt so zu organisieren, wie es Ihnen am besten passt. Spiele in GameMaker haben jedoch in der Regel Räume, die einer bestimmten Reihenfolge folgen, und müssen zumindest einen bestimmten Startraum haben (nachdem Nachdem das Spiel in diesem Raum begonnen hat, können Sie mit code oder Aktionen in jeden beliebigen Raum des Spiels wechseln, wenn Sie möchten.

Sie können den Raummanager öffnen, indem Sie oben im Asset-Browser auf die Schaltfläche "Extras" klicken und ihn aus dem sich öffnenden Menü auswählen, oder Sie können Sie können auch einfach auf klicken, das Symbol links neben einem Raum im Asset-Browser:

Sobald der Raummanager geöffnet ist, sehen Sie zwei verschiedene Registerkarten, eine für die Raumreihenfolge und eine für die Raumvererbung. Die Raumreihenfolge legt die Reihenfolge fest, in der auf die Räume zugegriffen wird, wenn Sie das Spiel buchstäblich durchqueren, indem Sie die Funktion room_goto_next() auf Der oberste Raum in der Liste ist immer mit dem "Home"-Symbol gekennzeichnet, das anzeigt, dass dieser Raum der Startraum des Spiels ist. Startraum für das Spiel ist. Dieses Symbol wird auch im Asset-Browser angezeigt, so dass Sie eindeutig sehen können, um welchen Raum es sich handelt, unabhängig davon, wie Sie Ihre Assets gefiltert oder eingerichtet haben.

Hinweis: Die IDs Ihrer Raum-Assets werden nicht in der gleichen Reihenfolge sein, wie sie in dieser Liste definiert sind, und daher sollten Sie nicht direkt einen Zahlenwert angeben, um den Raum unter runtime zu ändern. Verwenden Sie immer die Raumkonstante für das Asset, auf das Sie verweisen wollen (das ist einfach sein Name), oder rufen Sie es über eine Funktion ab.

Sie können die Reihenfolge der Räume ändern, indem Sie einfach auf klicken und den gewünschten Raum in der Liste nach oben oder unten ziehen. Sie können auch und/oder / verwenden, um mehrere Räume auszuwählen und sie dann durch Klicken und Ziehen insgesamt zu verschieben:

Die Registerkarte Raumvererbung dient zum Einrichten von "untergeordneten" Räumen. Ein Kindraum ist ein Raum, der von einem "Elternraum" kopiert wurde einem "Eltern"-Raum kopiert und so eingestellt wurde, dass er alle seine Eigenschaften erbt. Wenn Sie an diesem Raum nichts ändern, ist dieser Raum tatsächlich eine Kopie des Originalraums, aber Sie können dann in den Raumeditor gehen und die Vererbung für bestimmte Dinge wie Ebenen oder Instanzen ein- oder ausschalten und dem Raum neue Ebenen, Instanzen und Eigenschaften hinzufügen.

Wie bei der Registerkarte "Raumreihenfolge" können Sie auf klicken und ziehen, um die Räume neu anzuordnen. Beachten Sie jedoch, dass durch die Auswahl eines untergeordneten Raums und das Verschieben zu einem anderen übergeordneten Raum die Raumvererbung mit dem ursprünglichen übergeordneten Raum aufgehoben wird und er stattdessen die Eigenschaften des neuen Raums erbt. Beachten Sie auch, dass Sie einen untergeordneten Raum in einen "vollständigen" Raum ohne Vererbung umwandeln können, indem Sie ihn nach links aus der Liste ziehen (der Listenrand wird hervorgehoben), wodurch er zu einem eigenständigen Raum wird.

Schließlich verfügt der Raummanager über einige zusätzliche Werkzeuge, die Ihnen die Navigation und Organisation Ihrer Räume erleichtern. Oben finden Sie eine Suchleiste, mit der Sie einen Suchbegriff eingeben und dann schnell zu den Räumen gehen können, die den Suchbegriff im Namen tragen Namen enthalten. Räume, in denen der Suchbegriff vorkommt, werden hervorgehoben, und Sie können mit den seitlichen Auf-/Ab-Tasten sofort zu ihnen navigieren:

Es gibt auch einige hilfreiche Schaltflächen am unteren Rand des Raummanagers, mit denen Sie Räume in der Raumordnungsliste nach oben oder unten verschieben können:

Neben diesen Schaltflächen gibt es auch eine Schaltfläche Gruppe erstellen, mit der Sie einen Gruppenordner erstellen können. Diese Ordner können verwendet werden, um die Räume in Ihrem Spiel besser zu organisieren (die Reihenfolge der Räume bleibt erhalten), und Sie können Räume anklicken und in den Ordner ziehen, um sie hinzuzufügen. Sie können auch mehrere Räume auswählen, indem Sie bei gedrückter Maustaste auf und/oder / klicken. Klicken Sie dann auf die Schaltfläche Gruppe erstellen, und alle ausgewählten Räume werden der Gruppe hinzugefügt. Beachten Sie, dass sich dadurch die Reihenfolge der Räume in Ihrem Spiel ändert, und beachten Sie auch, dass, wenn Sie den Gruppenordner später löschen, alle Räume innerhalb des Ordners ebenfalls gelöscht werden (standardmäßig wird GameMaker warnt Sie vor diesem Vorgang).

Beachten Sie auch, dass Sie mit der rechten Maustaste auf einen beliebigen Raum im Raum-Manager klicken können, um das folgende Menü aufzurufen:

Dieses Menü bietet Ihnen die folgenden Optionen:

  • Untergeordneten Raum erstellen: Damit wird ein neuer Raum als Kind des angeklickten Raums erstellt, wobei der neue Raum die Eigenschaften des übergeordneten Raums erbt.
  • Bearbeiten: Dadurch wird der Raumeditor für den Raum geöffnet.
  • Tags bearbeiten: Dadurch wird der Tag-Editor geöffnet, damit Sie dem Raum Tags hinzufügen können.
  • Umbenennen: Hiermit können Sie die Raumanlage umbenennen (Sie können auch einen langsamen Doppelklick auf den Raum machen, um ihn umzubenennen).
  • Duplizieren: Hiermit wird der ausgewählte Raum dupliziert und der neue Raum unter dem ausgewählten Raum platziert.
  • Bevorzugt: Hiermit wird der ausgewählte Raum zum Abschnitt Favoriten des Asset-Browsers hinzugefügt.
  • Löschen: Durch Klicken auf diese Schaltfläche wird der ausgewählte Raum (oder die ausgewählten Räume) gelöscht. Bei Verwendung für einen Gruppenordner werden alle Räume innerhalb des Ordners zusammen mit dem Ordner gelöscht. Bei Verwendung für einen übergeordneten Raum wird nur der übergeordnete Raum gelöscht, und alle untergeordneten Räume bleiben erhalten, auch wenn sie jetzt unabhängig sind und nicht mehr zu den untergeordneten Räumen gehören.
  • Kinder aufklappen: Wenn ein Gruppenordner ausgewählt ist, wird mit dieser Option der Inhalt des Ordners erweitert.
  • Alle erweitern: Mit dieser Option werden alle Ordnergruppen erweitert.
  • Alle schließen: Mit dieser Option werden alle Ordnergruppen geschlossen.
  • Anordnen: Mit dem Untermenü Anordnen können Sie Räume im Raummanager nach oben oder unten verschieben, ähnlich wie mit den Schaltflächen am unteren Rand des Managerfensters.
  • Im Explorer öffnen: Damit wird ein OS-Dateiexplorer für das Asset im Spielprojekt geöffnet.
  • Asset-Farbe: Hier können Sie eine Farbe für den Raum (oder die Räume) auswählen, um sie im Raum-Manager zu identifizieren. Beachten Sie, dass diese Farbe auch im Asset-Browser verwendet wird und alle zuvor eingestellten Farben überschreibt.
