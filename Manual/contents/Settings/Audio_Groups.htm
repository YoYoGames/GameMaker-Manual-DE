Audio-Gruppen

Der Audiogruppen-Manager ist über das Menü "Extras" auf IDE verfügbar. Hier können Sie Audiogruppen hinzufügen, löschen und umbenennen sowie ihre platform Exportoptionen einstellen. Der GameMaker erlaubt es Ihnen, die einzelnen Audio-Ressourcen (Soundeffekte und Musik) verschiedenen Audiogruppen zuzuordnen zuzuordnen, um die Anzahl der gleichzeitig abgespielten Sounds zu optimieren und um zu bestimmen, in welches platforms sie exportiert werden.

Um eine Audiogruppe zu definieren, müssen Sie auf auf die Schaltfläche Neu hinzufügen klicken, wodurch eine neue Gruppe erstellt wird, die Sie dann benennen können. Um eine Gruppe zu ändern, klicken Sie auf auf das Dropdown-Menü und wählen Sie die Gruppe aus, die Sie bearbeiten möchten.

HINWEIS: Wenn Sie im Sound-Editor Streaming-Audio für einen Sound auswählen, können Sie den Sound nicht einer Audiogruppe zuweisen, da Streaming-Sounds nicht auf die gleiche Weise wie andere Sounds verpackt sind. Sie haben die volle Kontrolle darüber, wann sie mit code geladen, abgespielt und entladen werden.

Um einen Sound zu einer Audiogruppe hinzuzufügen, müssen Sie den Sound-Editor für diesen Sound öffnen und dann die Gruppe aus der dortigen Option auswählen, oder Sie können alternativ Assets zu einer Audiogruppe hinzufügen über das Menü der rechten Maustaste im Asset-Browser hinzufügen. Sie können ein oder mehrere Sound-Assets (oder einen ganzen Asset-Gruppen-Ordner) auswählen und dann das RMB-Menü verwenden, um die Sounds zu einer beliebigen Gruppe hinzuzufügen. Sie können auch auf die Schaltfläche " Ressource hinzufügen " am unteren Rand des Audiogruppen-Editors klicken, um auf diese Weise Assets hinzuzufügen.

Sobald Sie einen Sound zu einer benutzerdefinierten Audiogruppe hinzugefügt haben, können Sie in diesem Fenster auswählen, in welche platform Sie diesen Sound exportieren möchten. Es kann sein, dass Sie standardmäßig alle Ihre Sounds in der höchsten Qualität haben, aber für HTML5 (zum Beispiel) (z. B. für HTML5) eine Sounddatei mit geringerer Qualität verwenden möchten, um weniger Speicherplatz zu benötigen. In diesem Fall fügen Sie die Dateien mit geringerer Qualität zu GameMaker hinzu und weisen sie einer neuen Audiogruppe zu. Dann wählst du diese Audiogruppe aus dem Dropdown-Menü aus und stellst nur in HTML5 exportieren und den HTML5-Export aus den Exportoptionen für Audiogruppen höherer Qualität entfernen. Es ist wichtig zu beachten, dass Sie die Exportoptionen für die "Standard"-Audiogruppe nicht ändern können und dass sie immer immer in alle verfügbaren platforms exportiert wird, wenn Sie ein endgültiges Spielpaket erstellen.

Sobald Sie Audiogruppen definiert und ihnen Sounds zugewiesen haben, können Sie diese in der Liste auf der linken Seite sehen, wenn Sie die Gruppe auswählen. Es wird immer eine "Standard"-Audiogruppe zur Verfügung stehen, und alle Klänge, die sich in dieser Gruppe befinden, werden immer sind immer im Spielpaket für alle platforms enthalten und werden beim Start in den Speicher geladen (es sei denn, sie sind in den Eigenschaften des Sound-Editors als "gestreamt" gekennzeichnet), aber wenn Sie eine benutzerdefinierte Audiogruppe erstellen, werden die Dateien, die dieser Gruppe nicht in den Speicher geladen, bis Sie die Funktion audio_group_load(). Auf diese Weise können Sie kontrollieren welche Audiodateien jederzeit im Speicher vorhanden sind. Beachten Sie auch, dass Sie für alle Audiogruppenfunktionen den Wert der Audiogruppen-ID angeben müssen. Dies ist einfach der Name, den Sie der Audiogruppe gegeben haben.

Audiogruppen sind auch mit den Einstellungen der Konfigurationen verknüpft. Wenn Sie eine Konfiguration auswählen, können Sie das Fenster Audiogruppen öffnen und auf der rechten Seite die Exportziele für diese Konfiguration auswählen. Wenn Sie dann die Konfiguration ändern, werden diese Ausgabeziele geändert. Beachten Sie, dass Sie die Tonressourcen nicht pro Konfiguration auf verschiedene Gruppen festlegen können, sondern nur das Exportziel für die jeweilige Gruppe.
