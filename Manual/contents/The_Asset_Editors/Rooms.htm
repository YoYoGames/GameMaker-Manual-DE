Der Raum-Editor

Mit dem Raumeditor erstellen Sie Ihre Spielräume. Jedes Spiel benötigt mindestens einen Raum, in dem Sie Instanzen, sprites, Kacheln, Pfade und Hintergründe platzieren können. Jedes dieser verschiedenen Elemente kann auf einer eigenen Ebene platziert werden, die dann im Ebenen-Editor nach Belieben angeordnet werden kann. Aufgrund der Komplexität des Raum-Editors geben wir Ihnen zunächst einen kurzen Überblick über die wichtigsten Funktionen, und dann können Sie unter den unten aufgeführten Links zu den einzelnen Abschnitten weitere Einzelheiten finden.

Wenn Sie eine Raumressource erstellen, können Sie im Asset-Browser mit der rechten Maustaste auf klicken, um das Kontextmenü zu öffnen, über das Sie untergeordnete Räume erstellen können (weitere Informationen finden Sie auf der Seite über Vererbung ), den Raum zur Bearbeitung öffnen, eine neue Ressourcengruppe hinzufügen, um die Räume besser zu organisieren, den Raum umbenennen oder ihn löschen können. Beachten Sie, dass Sie zum Ändern der Raumreihenfolge und/oder der Vererbung den Raummanager verwenden müssen, den Sie über das Menü oben rechts im Asset-Browser öffnen können.

Der Raum-Editor ist selbst ein Arbeitsbereich, und als solcher können Sie auf klicken und die Registerkarte aus dem Hauptfenster in ein neues, eigenes Fenster ziehen - zum Beispiel in eine andere Anzeige. Sie können ihn auch wieder in das Hauptfenster einfügen, indem Sie die Registerkarte an den oberen Rand von IDE ziehen und die Maustaste loslassen.

Die Benutzeroberfläche des Raumeditors ist einfach zu navigieren und in verschiedene diskrete Bereiche aufgeteilt. Diejenigen Teile des Editors, die angedockt sind - der Raumeigenschaften- und der Ebenen-Editor sowie die verschiedenen Ebenen-Eigenschaftsbereiche - können auch aus dem Dock entfernt werden, indem sie einfach in das workspace gezogen werden, und sie können wieder in die Docks eingefügt werden, indem sie an die Seiten oder an den Boden des workspace gezogen werden.

Im Folgenden finden Sie einen kurzen Überblick über die einzelnen Bereiche des Raumeditors:

Lineale/Führer

Die Lineale am Rand der Leinwand zeigen Ihnen die Position der darin platzierten Objekte an. Sie sind von (0, 0) aus markiert, dem Mittelpunkt der Leinwand und dem Ursprung des Raums. Sie können auf die Lineale klicken und ziehen, um eine horizontale oder vertikale Hilfslinie in den Raum zu ziehen. Diese Hilfslinie kann dann verwendet werden, um die verschiedenen verwendeten Elemente genau zu positionieren, da das Bewegen eines Elements in die Nähe der Hilfslinie dieses "einrasten" lässt. Während der Positionierung von Assets im Raum werden vorübergehend auch intelligente Hilfslinien angezeigt, die den Abstand zwischen den Assets sowie den Abstand von der Raumgrenze oder dem Mittelpunkt angeben.

Der Abstand, in dem Assets an Hilfslinien "einrasten", sowie andere Eigenschaften können in den Raumeditor-Voreinstellungen festgelegt werden.

Ebenen-Editor

Der Raumeditor platziert Dinge auf Ebenen innerhalb des Raums. Jede Ebene hat eine bestimmte "Tiefe", d. h. die Ebenen, die am unteren Ende der Liste im Ebenenfenster erscheinen, werden unter den Ebenen gezeichnet, die am oberen Ende erscheinen.

WICHTIG! Es gibt eine minimale und maximale Ebenentiefe von -16000 bis 16000. Alles, was auf einer Ebene außerhalb dieses Bereichs platziert wird, wird nicht gezeichnet, obwohl alle Ereignisse weiterhin wie gewohnt ablaufen.

Ebenen werden erstellt, indem Sie auf die entsprechende Schaltfläche für den Ebenentyp klicken, den Sie erstellen möchten:

Instanzen Auf dieser Ebene werden die Instanzen platziert, die in dem Raum verwendet werden sollen.
Kachel-Karten Diese Ebene dient zum Platzieren von Kacheln aus Kachelsätzen alsKachelkarten.
Pfade Diese Ebene dient zum Anzeigen und Hinzufügen von Pfaden im Raum.
Vermögenswerte Diese Ebene dient dazu, dem Raum visuelle Elemente wie Sprites und Sequenzen hinzuzufügen.
Filter/Effekt Dieser Ebenentyp definiert einen Filter/Effekt, der auf alle darunter liegenden Ebenen angewendet wird.
Hintergrund Dieser Ebenentyp definiert einen Hintergrund, bei dem es sich im Wesentlichen um ein einfarbiges oder sprite Bild handelt, das über den gesamten Raum gekachelt werden kann.
UI This creates a global UI layer that is shared among rooms.

Sie können auch mit der Schaltfläche Ordner einen Ebenenordner erstellen, in dem Sie ausgewählte Ebenen zusammenfassen können, und mit der Schaltfläche Löschen die ausgewählten Ebenen löschen. Jeder dieser Ebenentypen wird hier ausführlicher behandelt.

Beachten Sie, dass Sie die Vererbung für den Ebenen-Editor umschalten können, was sich auf die Ebenenreihenfolge und die Sichtbarkeit auswirkt. Die Sichtbarkeit selbst kann durch Anklicken des Augensymbols neben den einzelnen Ebenen eingestellt werden, oder Sie können auf das Symbol ganz oben klicken, um alle Ebenen auf einmal zu aktivieren/deaktivieren. Sie haben auch die Möglichkeit, eine Ebene (oder alle) zu "sperren", so dass sie nicht geändert werden können, indem Sie auf das Vorhängeschloss-Symbol klicken. Beachten Sie, dass Ebenen, die als unsichtbar markiert sind, bei der Ausführung des Spiels nicht angezeigt werden.

Layer-Eigenschaften

Das Fenster mit den Eigenschaften der Raumebene ändert sich je nach der aktuell im Ebenen-Editor ausgewählten Ebene.

Jedes Fenster enthält die verschiedenen Eigenschaften und Listen, die mit dem aktuell ausgewählten Ebenentyp verbunden sind, und Sie können grundlegende Details bearbeiten, wie die Ebene angezeigt wird und was sich tatsächlich auf der Ebene befindet. Für weitere Details siehe hier.

Raum-Eigenschaften

In diesem Abschnitt des Raum-Editors können Sie alle grundlegenden Eigenschaften für den Raum festlegen, wie z. B. seine Größe, ob er Physik verwendet oder welche Kameras und Ansichtsfenster er benötigt. Dieser Abschnitt wird hier ausführlicher behandelt.

Layer-Toolbox

Bestimmte Ebenentypen verfügen über zusätzliche Werkzeuge, die oben auf IDE in der Ebenen-Werkzeugpalette angezeigt werden (z. B. Kachelebenen oder Pfadebenen). Die genauen Werkzeuge ändern sich je nach Ebenentyp, der gerade bearbeitet wird, und werden daher auf der Seite erläutert, auf der die einzelnen Ebenen erklärt werden, hier.

Zimmer-Toolbox

Oben im Hauptfenster des Raum-Editors gibt es einige Steuerelemente, mit denen Sie die Darstellung der Dinge beeinflussen können. Sie sind:

  • Canvas-Raster einschalten: Hiermit schalten Sie das Canvas-Raster des Raumeditors ein/aus. Dies ist ein Raster, das GameMaker über die Hauptleinwand zeichnet, um sie in Abschnitte zu unterteilen, und das standardmäßig auf eine Größe von 32x32px eingestellt ist. Wenn Sie jedoch auf das Rastermenüsymbol klicken, öffnen Sie die Rasteroptionen:Mit diesen Optionen können Sie die Rasterfarbe und das Alpha sowie die Zellenwerte für das Raster entlang der X- und Y-Achse einstellen. Sie haben hier auch die Möglichkeit, das Einrasten des Rasters zu aktivieren oder zu deaktivieren (standardmäßig aktiviert). Mit den Tastenkombinationen "G" und + "G" können Sie die Sichtbarkeit des Rasters bzw. das Einrasten am Raster umschalten.
  • Leinwand-Zoom-Steuerungen: Diese Schaltflächen steuern die aktuelle Zoomstufe der Leinwand. Sie können ein- oder auszoomen, und wenn Sie auf die Schaltfläche klicken, wird die Leinwand im Verhältnis 1:1 zum bearbeiteten Raum zurückgesetzt. Sie können auch auf die Schaltfläche Window Fit (Fensteranpassung) klicken, um die gesamte Raum-Leinwand in den aktuellen Editor workspace einzupassen (dabei wird die Leinwand entsprechend vergrößert/verkleinert, damit sie passt). Beachten Sie, dass Sie auch mit / und dem Mausrad zoomen können. Durch Drücken von / + wird die Raumleinwand auf 1:1 mit den Raumeinstellungen eingestellt.
  • Ansichten anzeigen: Durch Anklicken dieser Option wird das Ansichtsrechteck aktiviert bzw. deaktiviert. Wenn es aktiviert ist, sehen Sie einen hervorgehobenen Bereich, der die verschiedenen Ansichten, die für den Raum aktiviert sind, kennzeichnet.
  • Animationen abspielen: Wenn Sie sprites, sequences oder animierte Kacheln hinzufügen, möchten Sie sich vielleicht ein Bild davon machen, wie diese im Raum selbst aussehen werden, ohne das gesamte Projekt kompilieren zu müssen, und deshalb können Sie auf diese Schaltfläche klicken, um alle verschiedenen Animationen abzuspielen. Wenn Sie erneut darauf klicken, wird die Animation gestoppt.
  • Canvas Von jeder Ebene auswählen: Wenn Sie auf ein Element im Raum-Canvas klicken, können Sie standardmäßig nur die Assets auswählen, die sich auf der gerade bearbeiteten Ebene befinden. Wenn Sie diese Option jedoch aktivieren, können Sie auf ein beliebiges Element klicken und es wird ausgewählt, wobei die Zielebene in die Ebene geändert wird, auf der sich das Element befindet. Sie können die Tastenkombination " P" verwenden, um diese Option vorübergehend zu aktivieren (halten Sie "P" gedrückt, um sie zu aktivieren und lassen Sie sie los, um sie zu deaktivieren).

Zimmer-Leinwand

Die Mitte des Raumeditor-Fensters wird von dem Bereich eingenommen, in dem die eigentliche Bearbeitung stattfindet. Hier platzieren Sie Ihre Instanzen und Assets, zeichnen Ihre Kacheln oder positionieren Ihre Pfade. Sie können mit dem Mausrad oder den Raumsteuerungen oben zoomen, und Sie können den Raum verschieben, indem Sie die mittlere Maustaste gedrückt halten und die Maus bewegen oder + verwenden, während Sie sich bewegen.

Bei der Erstellung von Asset- oder Instanzebenen können Sie das Asset oder die Instanz platzieren, indem Sie es einfach aus dem Asset-Browser ziehen und dann an der gewünschten Stelle ablegen. Alternativ können Sie ein Asset oder eine Instanz aus dem Asset-Browser auswählen und dann drücken und gedrückt halten, um eine Vorschau der Ressource "an Ort und Stelle" zu erhalten. Wenn Sie zusätzlich die linke Maustaste drücken, können Sie die Instanz hinzufügen und sogar in die Raumebene "malen", indem Sie die Taste gedrückt halten und die Maus ziehen. Beachten Sie, dass Instanzen und sprites, die einem Raum auf diese Weise hinzugefügt werden, standardmäßig an den definierten Raumrastereinstellungen ausgerichtet werden, aber Sie können / gedrückt halten, damit die Instanz der Maus genau folgt und nicht einrastet (wenn Sie / loslassen, wird das Einrasten wieder möglich).

Für Pfade können Sie einen neuen Pfad-Layer und eine Pfad-Ressource direkt im Raum-Editor erstellen - oder Sie erstellen eine neue Pfad-Ressource und ziehen sie dann in das Haupt-Editor-Fenster, wie Sie es für eine Instanz tun würden - und bearbeiten dann den Pfad und seine Verbindungen auch im Editor-Fenster. Für Kachelsätze können Sie sie aus dem Kachelsatz-Editor "einzeichnen".

Beachten Sie, dass Sie mehrere Assets auf derselben Ebene auswählen und verschieben oder löschen können, indem Sie / + auf den gewünschten Assets gedrückt halten. Die meisten Ressourcen im Raumeditor können Sie drehen und skalieren (einzeln oder als Gruppe), indem Sie entlang der Außenkanten oder Ecken anklicken und die Maus ziehen. Sie können auch einen Doppelklick auf ein einzelnes Element (Kacheln, Instanzen, sprites) ausführen, um den Editor darauf schwenken zu lassen und das Eigenschaftenfenster für das ausgewählte Element zu öffnen.

Statusleiste

Die Statusleiste dient dazu, kontextspezifische Informationen anzuzeigen. Die Statusleiste zeigt immer an, wo im Raum sich der Mauszeiger befindet, aber sie zeigt auch zusätzliche Informationen auf der Grundlage der bearbeiteten Ebene, des verwendeten Werkzeugs und des Status dieses Werkzeugs.

Zimmer-Menü

Abgesehen von den Werkzeugen im Editor wurde ein neues Dropdown-Menü am oberen Rand von IDE hinzugefügt.

Diese Menüs werden im Folgenden erläutert:

  • Ebenen-Ansicht: Damit wird das Ebenen-Editor-Fenster wieder geöffnet, falls Sie es zu einem beliebigen Zeitpunkt geschlossen haben.
  • Raumeigenschaften: Damit wird das Fenster Raumeigenschaften wieder geöffnet, falls Sie es zu irgendeinem Zeitpunkt geschlossen haben.
  • Ebeneneigenschaften: Damit wird das Fenster Ebeneneigenschaften wieder geöffnet, falls Sie es zu einem beliebigen Zeitpunkt geschlossen haben.
  • Instanz-Erstellungsreihenfolge: Dadurch wird das Fenster Instance Creation Order geöffnet (das auch über die Schaltfläche im Fenster Room Properties geöffnet werden kann):
    In diesem Fenster werden alle Instanzen des Raums in der Reihenfolge aufgelistet, in der sie erstellt werden (von oben nach unten). Wenn Sie möchten, dass eine bestimmte Instanz vor einer anderen erstellt wird, können Sie einfach auf klicken und sie an die gewünschte Position ziehen.
  • Windows auf dem aktuellen Desktop zurücksetzen: Setzt das Layout des Raumeditor-Fensters auf seine Standardwerte für den aktuell fokussierten Desktop workspace zurück.

  • Kachelbearbeitung: Wenn Sie mit einer Kachelebene arbeiten, wird diese Option im Dropdown-Menü hervorgehoben. Sie hat das folgende Untermenü:
    Bei der Bearbeitung der Kachelebene können Sie mehrere Kacheln auswählen, indem Sie die linke Maustaste anklicken, gedrückt halten und ziehen , oder Sie können einzelne Kacheln auswählen und gruppieren mit / + , oder einzelne Kacheln aus der Auswahl entfernen mit + . In diesem Menü können Sie dann die Auswahl aufheben oder umkehren sowie die Auswahl ausschneiden, kopieren oder auf einer anderen Ebene (oder sogar auf derselben Ebene, aber an einer anderen Position) einfügen. Mit der letzten Option können Sie das angedockte Fenster des Kachelsatz-Editors wieder öffnen, wenn Sie es geschlossen haben.
  • Bild in Kachelplan konvertieren: Die Option Bild in Kachelplan konvertieren ist ein leistungsfähiges Werkzeug, mit dem ein einzelnes Bild importiert und dann die verwendeten Kacheln aus dem Bild extrahiert und als Kachelplan-Ebene im Raumeditor neu erstellt werden können, wobei das erforderliche Kachelset und sprite als Teil des Prozesses erstellt werden. Wenn Sie dieses Werkzeug auswählen, werden Sie aufgefordert, eine unkomprimierte Bilddatei (PNG, GIF oder BMP) anzugeben, die dann geladen wird. Nach dem Laden wird das Bildimport-Tool angezeigt:
    Hier können Sie festlegen, wie die Kachel bei der Erstellung des Kachelsatzes aufgeteilt werden soll, und Sie können die Zellenbreite und -höhe sowie die an den Rändern erforderlichen Versätze einstellen. Sie können auch die ungefähre Breite (in Pixeln) für das endgültige sprite festlegen. Wenn der Kachelsatz beispielsweise aus 64x64 Kachelzellen besteht und Sie hier eine Breite von 200 einstellen, ist die endgültige sprite, die für die Verwendung im Kachelsatz erstellt wird, 192 Pixel breit (d. h.: 3 Kachelzellen). Wenn Sie den Standardwert von 0 beibehalten, versucht GameMaker, sprite so "quadratisch" wie möglich zu gestalten, mit ungefähr der gleichen Anzahl an horizontalen und vertikalen Kachelzellen.

    Nachdem Sie festgelegt haben, wie das Bild aufgeteilt werden soll, klicken Sie auf das Feld neben der Option " Farbe entfernen ", um den Farbwähler zu öffnen und eine Farbe auszuwählen, die aus dem endgültigen sprite entfernt werden soll. In der Regel handelt es sich dabei um eine Hintergrundfarbe, die Sie entfernen möchten. Wenn Sie dieses Feld auf etwas anderes als 100 % transparent (Alpha 0) einstellen, wird die ausgewählte Farbe beim Import entfernt.

    Schließlich haben Sie die Möglichkeit, die sprite, den Kachelsatz und die Kachelkartenebene zu benennen, die mit diesem Werkzeug für Sie erstellt werden. Wenn Sie den Import abschließen, wird ein sprite mit allen in einem Raster angeordneten Bildern sowie ein Kachelsatz aus diesem sprite erstellt. Im Raum wird eine neue Kachelkartenebene erstellt und das Bild mit dem erstellten Kachelsatz reproduziert. Beachten Sie, dass das Werkzeug keine Kacheln dupliziert und stattdessen erkennt, wenn eine Zelle Bilddaten hat, die mit einer anderen Zelle übereinstimmen (dies schließt gedrehte Kacheln ein).
