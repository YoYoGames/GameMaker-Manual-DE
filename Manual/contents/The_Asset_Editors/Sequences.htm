Der Sequence Editor

Eine Sequenz ist im Wesentlichen eine Sammlung von Elementen, die im Laufe der Zeit eine dynamische Animation ausführen. Sie können sprites, Instanzen, Sounds und andere Dinge enthalten, und jedes dieser Elemente kann so eingestellt werden, dass es sich bewegt, die Farbe ändert oder die Animation im Laufe der Zeit auf sequence selbst startet/stoppt. Die von Ihnen hinzugefügten Elemente werden bestimmten Spuren innerhalb von sequence zugewiesen, und diesen Spuren können dann verschiedene Attribute zugewiesen werden - sogenannte Keyframes -, die statisch sein oder sich im Laufe der Zeit ändern können. Die gesamte Bearbeitung einer sequence erfolgt im Sequenzeditor, der im Wesentlichen aus drei Bereichen besteht:

  • Leinwand - hier platzieren Sie die Elemente, aus denen die Sequenz besteht
  • Track Panel - hier können Sie Asset-Tracks und Parameter-Tracks zu Ihrer Sequenz hinzufügen/entfernen
  • Dope Sheet - hier können Sie die Keyframes der Spuren hinzufügen/entfernen/bearbeiten und steuern, wie sie sich im Laufe der Zeit verhalten.

Beachten Sie auch, dass Sie sequences innerhalb von sequences haben können, um komplexe Animationen und Effekte zu erstellen, und sequences kann auch über code aufgerufen und geändert werden, um Ihnen so viel Kontrolle wie möglich darüber zu geben, wie sie in Ihrem Spiel angezeigt und verwendet werden. Weitere Informationen über den kodierten Aspekt von sequences finden Sie im Abschnitt über die Sequenzfunktionen.

Bevor wir die Details des Editors erläutern, nehmen wir uns einen Moment Zeit, um zu erklären, wie ein sequence erstellt wird und welche Schlüsselwörter in den Beschreibungen verwendet werden, damit Sie ein besseres Verständnis für die Funktionsweise des Sequences Editors bekommen. Der allgemeine Arbeitsablauf für die Erstellung eines sequence ist wie folgt:

  • Zuerst erstellen Sie Ihre sequence im Asset-Browser, der den Sequences Editor öffnet.
  • Als Nächstes ziehen Sie ein Element (sprite, ein Objekt, einen Sound oder sequence) in die Arbeitsfläche des Editors.
  • Dadurch wird eine Asset-Spur im Spur-Panel erstellt und ein Asset-Schlüssel zum Dope Sheet hinzugefügt. Die Asset-Spur ist einfach der Name der Spur, die das Asset enthält, während der Asset-Schlüssel ein Balken ist, der gezeichnet wird, um die Anzahl der Frames zu symbolisieren, über die das Asset in sequence animiert wird.
  • Zu diesem Zeitpunkt sollten Sie das Asset im Canvas an seiner Anfangsposition für den Start von sequence positionieren und alle anfänglichen Transformationen wie Skalierung oder Drehung hinzufügen.
  • Wenn Sie mit den anfänglichen Einstellungen zufrieden sind, sollten Sie die Parameterschlüssel (oder Keyframes) aufzeichnen - das ist einfach die Bezeichnung für die Werte für Skalierung, Drehung und Position, die ein bestimmter Asset-Schlüssel an einem beliebigen Punkt in der Zeitleiste von dope sheet hat.
  • Als Nächstes ändern Sie die Position des Abspielkopfes (die aktuelle Abspielposition in der Zeitleiste dope sheet ) und verschieben oder ändern dann die Asset-Transformationen und nehmen einen weiteren Parameterschlüssel auf.
  • Alle Parameterschlüssel werden in zusätzlichen Parameterspuren gespeichert, die eine Untergruppe der Spuren unter der Haupt-Asset-Spur im Spurbereich sind, und Sie können die Parameterschlüsseldaten für jeden Parameter im Track Editor bearbeiten.
  • Nun wiederholen Sie den obigen Vorgang für die gesamte Länge von sequence, verschieben, drehen und skalieren das Element nach Bedarf und fügen weitere Elemente hinzu, wenn Sie sie benötigen.

Lineale/Führer

Die Lineale am Rand der Leinwand zeigen Ihnen die Position der Dinge, die darin platziert sind, und sind von (0, 0) aus markiert, dem Zentrum der Leinwand und dem Standardursprung für sequence. Sie können auf die Lineale klicken und ziehen, um eine horizontale oder vertikale Hilfslinie in die zu ziehen. Sie können auf die Lineale klicken und ziehen, um eine horizontale oder vertikale Hilfslinie in die sequence zu ziehen. Diese Hilfslinie kann dann verwendet werden, um die verschiedenen verwendeten Elemente genau zu positionieren, da das Bewegen eines Elements in die Nähe der Hilfslinie dieses "einrasten" lässt. Bei der Positionierung Assets innerhalb von sequence werden vorübergehend auch intelligente Hilfslinien eingeblendet, die den Abstand zwischen den Assets sowie den Abstand zur Grenze oder zum Mittelpunkt von sequence angeben.

Beachten Sie, dass die Begrenzung von sequence und die Mittelpunktsmarkierungslinien für sequence ebenfalls als als Hilfslinien gelten und sich genauso verhalten wie Hilfslinien, die über die Lineale hinzugefügt werden, mit der Ausnahme, dass sie nicht bearbeitet werden können.

Der Abstand, in dem Assets an Hilfslinien "einrasten", sowie andere Eigenschaften können in den Voreinstellungen für Sequenzen festgelegt werden.

Leinwand-Ansicht

In der Canvas-Ansicht sehen Sie eine Vorschau des sequence Canvas und können die verschiedenen Assets, aus denen Ihr sequence besteht, bearbeiten und positionieren. Sie können dem sequence ein Asset hinzufügen, indem Sie es einfach aus dem Asset-Browser auf die Leinwand ziehen. Dadurch wird eine neue Spur im Spur-Bedienfeld erstellt und ein Asset-Schlüssel für das Asset an der aktuellen Abspielposition in der Zeitleiste hinzugefügt. Die folgenden Assets können auf diese Weise auf die Leinwand gezogen werden:

  • Sprites
  • Objekte
  • Klingt
  • Sequenzen

Sie können auch das Werkzeug Text in der Toolbox verwenden, um eine Textspur zu erstellen. Lesen Sie Text in Sequenzen für weitere Informationen.

Jedes zum Canvas hinzugefügte Element kann auf verschiedene Weise manipuliert werden - gedreht, transformiert oder übersetzt - je nachdem, wie Sie mit ihm interagieren, da jedes Element über einen Begrenzungsrahmen und Transformationsgimmicks sowie Begrenzungsrahmen-Steuerungen verfügt. All dies wird auf der Seite über den Sequenzeditor Canvas näher erläutert.

HINWEIS Sound-Assets, die zu einem sequence hinzugefügt werden, sind im Wesentlichen Sound-Emitter , die nur das jeweilige Sound-Asset abspielen.

Für die Navigation in der Leinwandansicht können Sie dieselben grundlegenden Steuerelemente wie für die allgemeine Workspaces verwenden, d. h.: Verwenden Sie die mittlere Maustaste und ziehen Sie, um die Leinwand zu verschieben (oder verwenden Sie alternativ die Kombination + ), und Sie können mit dem Mausrad horizontal scrollen oder mit der Taste / und dem Mausrad vergrößern und verkleinern. Sie können auch Assets ausschneiden, kopieren und einfügen, indem Sie die Tastenkombinationen / + "X", / + "C" und / + "V" verwenden.

Spur als Parameterspuren hinzugefügt, und die Parametertasten werden als Punkte in der dope sheet Timeline an der Abspielkopfposition hinzugefügt:

Reihenfolge der Ereignisse

Ähnlich wie bei regulären Objekten können sequences Ereignisse zugewiesen werden, die eine code ausführen können. Die code wird in Form einer geskripteten Funktion zugewiesen, die keine arguments und wird aufgerufen, wenn das Ereignis ausgelöst wird.

Ereignisse werden hinzugefügt, indem Sie auf die Schaltfläche Ereignis hinzufügen klicken, wodurch sich das folgende Fenster öffnet:

Die hier aufgeführten Veranstaltungen sind wie folgt:

  • Erstellen - Dieses Ereignis tritt ein, wenn eine Instanz von sequence zum ersten Mal erstellt wird, und ist das allererste, was in einem sequence passiert, das über den Raumeditor in den Raum gestellt wird, wenn ein Raum betreten wird.
  • Destroy - Dieses Ereignis wird ausgeführt, wenn eine Instanz von sequence zerstört wird, und wird vor dem Ereignis Clean Up (siehe unten) ausgeführt.
  • Aufräumen - Dieses Ereignis wird nach jedem Ereignis aufgerufen, das eine Instanz von sequence aus dem Raum entfernt. Es wird also ausgelöst, wenn sequence zerstört wird, wenn der Raum endet, oder wenn das Spiel endet, und ist dafür gedacht jede Aufgabe auszuführen, die einmal ausgeführt werden muss, wenn die Instanz sequence aus irgendeinem Grund aus dem Spiel entfernt wird. Wenn du die sequence Instanz z.B. mit layer_sequence_destroy() zerstört hast, dann wird dieses Ereignis nach dem Destroy Ereignis (siehe oben).
  • Begin Step / Step / End Step - Das Step-Ereignis ist ein Ereignis, das bei jedem einzelnen Schritt (Frame) des Spiels überprüft wird, solange die Instanz sequence existiert, und ist in drei Teile unterteilt: begin, step und Ende. Für die meisten Dinge ist das Standard-Schritt-Ereignis gut geeignet, aber manchmal möchte man ein bisschen mehr Kontrolle darüber haben, was code zu welchem Zeitpunkt ausführt, und dafür gibt es die Schritt-Ereignisse "Beginn" und "Ende". Diese Ereignisse werden bei jedem Schritt (Frame) des Spiels immer in der gleichen Reihenfolge ausgelöst. Wichtig! Wenn sequence pausiert, werden diese Ereignisse nicht ausgelöst, und wenn das Spiel fortgesetzt wird, werden sie im nächsten Frame ausgelöst , nachdem sequence wieder zu spielen beginnt. Beachten Sie auch, dass die Reihenfolge der Ereignisse nicht von der Richtung des Abspielkopfes beeinflusst wird. Selbst wenn sequence rückwärts abgespielt wird, werden die Ereignisse immer noch als Beginn, Schritt und Ende ausgeführt.
  • Asynchrones Ereignis: Dies ist das Äquivalent zum Objekt Asynchronous System Event. Wichtig! Wenn die sequence pausiert, wird dieses Ereignis nicht ausgelöst.

Jedem Ereignis kann eine einzelne Funktion zugewiesen werden, die aufgerufen wird, wenn das Ereignis ausgelöst wird. Sie können die Funktion über das Eingabefeld für das Ereignis zuweisen, und ein Klick auf die Pfeilschaltfläche öffnet den script Editor für die zu bearbeitende Funktion. Sie können auch auf die Schaltfläche Neu erstellen am unteren Rand klicken, um eine neue script Ressource mit bereits definierten und ausfüllbaren "Boilerplate"-Funktionen zu erstellen. Beachten Sie, dass Sie die Funktionsnamen nach Belieben ändern können und nicht Sie müssen nicht die vordefinierten Namen verwenden, und Sie können auch alle nicht benötigten Funktionsdefinitionen entfernen. Es ist wichtig zu beachten, dass Funktionen, die für sequence Ereignisse verwendet werden, keine arguments annehmen können.

Sequenzereignisse können auch mit code hinzugefügt und bearbeitet werden. Für weitere Informationen siehe hier.

Gleis-Panel

In der Spurtafel wird jedes Asset in Ihrem sequence als Spur aufgelistet, wobei jede Spur Unterspuren für alle Parameter hat, die für das Asset geändert werden. Sie können auf klicken und auf eine beliebige Asset-Spur ziehen, um ihre Position über oder unter den anderen Spuren zu ändern (alle zugewiesenen Parameterspuren werden mitbewegt), und Sie können / + verwenden, um mehrere Spuren auszuwählen.

Sie können eine Spur erstellen, indem Sie ein Asset aus dem Asset-Browser (entweder ein sprite, ein Objekt, einen Sound oder ein sequence) in den sequence Canvas ziehen, oder Sie klicken auf das Symbol am unteren Rand, um eine neue Spur hinzuzufügen, und wählen das zu verwendende Asset aus dem sich öffnenden Asset-Explorer-Fenster. Die neue Spur wird erstellt und das Asset an der aktuellen Abspielposition in der Zeitleiste dope sheet hinzugefügt:

Canvas-Werkzeugkasten

Die Sequence Editor-Toolbox enthält die verschiedenen Visualisierungs- und Einrichtungsoptionen für die zu bearbeitende sequence.

  • Textspur hinzufügen : Hiermit können Sie eine Textspur in das Sequence Canvas einfügen. Nachdem Sie diese Option ausgewählt haben, können Sie auf eine beliebige Stelle im Canvas klicken (und optional ziehen), um eine neue Textspur zu erstellen.

    Ausführliche Informationen zu Textspuren finden Sie unter: Text in Sequenzen.
  • Canvas-Raster umschalten : Hiermit schalten Sie das Canvas-Raster des Sequence Editors ein/aus. Dies ist ein Raster, das GameMaker über die Vorschau-Leinwand zieht, um sie in Abschnitte zu unterteilen, und ist standardmäßig auf eine Größe von 32x32px eingestellt. Wenn Sie auf das Symbol für das Rastermenü klicken, werden die Rasteroptionen angezeigt:
    Mit diesen Optionen können Sie die Farbe und das Alpha des Rasters sowie die Zellenwerte für das Raster entlang der X- und Y-Achse festlegen. Sie haben hier auch die Möglichkeit, das Einrasten des Rasters zu aktivieren oder zu deaktivieren (standardmäßig deaktiviert). Mit den Tastenkombinationen "G" und + "G" können Sie die Sichtbarkeit des Rasters bzw. das Einrasten am Raster umschalten.
  • Canvas-Zoom-Steuerungen : Diese Schaltflächen steuern die aktuelle Zoomstufe der Leinwand. Sie können ein- oder auszoomen, und wenn Sie auf die Schaltfläche klicken, wird die Leinwand 1:1 mit der zu bearbeitenden sequence zurückgesetzt. Sie können auch auf die Schaltfläche Window Fit (Fensteranpassung) klicken, um die gesamte sequence Leinwand in den aktuellen Editor workspace einzupassen (dadurch wird sie entsprechend vergrößert/verkleinert). Beachten Sie, dass Sie auch mit / und dem Mausrad zoomen können. Durch Drücken von / + wird die Leinwand im Verhältnis 1:1 mit dem Bildschirm ausgerichtet.
  • Gizmos umschalten : Durch Anklicken dieser Option werden die verschiedenen im Canvas sichtbaren Gizmos ein- oder ausgeschaltet. Wenn diese Funktion aktiviert ist, sehen Sie alle Gizmos, die mit den verschiedenen Assets und dem Canvas selbst verbunden sind. Wenn sie deaktiviert ist, werden sie nicht angezeigt und Sie erhalten einen besseren Überblick über das Aussehen von sequence. Beachten Sie, dass diese Funktion standardmäßig aktiviert ist. Über die Optionsschaltfläche steht Ihnen ein Menü zur Verfügung, mit dem Sie filtern können, was sichtbar ist, ohne alle verschiedenen Widgets zusammen deaktivieren zu müssen. Eine vollständige Liste dieser Filteroptionen und eine Erklärung ihrer Funktionsweise finden Sie hier.
  • Umschalten desTransform-Gizmo-Modus : Jedes Asset, das zu sequence hinzugefügt wird, hat ein Transformations-Gizmo an seinem Ursprung:
    Dieses Gizmo kann steuern, wie das Asset skaliert oder gedreht wird, sowie seine Position und die Position des Asset-Ursprungs steuern, es zeigt jedoch immer nur eine dieser Optionen an. Wenn Sie auf diese Schaltfläche switch klicken, können Sie zwischen den verschiedenen Widget-Typen wechseln. Sie können auch auf das Optionsmenü klicken, um direkt die Art der Steuerungsaktion auszuwählen, die das Transformations-Gizmo durchführen soll.
  • Canvas-Rahmen umschalten : Mit dieser Schaltfläche können Sie den Canvas-Rahmen ein- oder ausschalten (er ist standardmäßig aktiviert). Der Canvas-Rahmen ist lediglich eine Orientierungshilfe, die dazu dient, zu beurteilen, wo Elemente im sequence Canvas platziert werden. Diese Schaltfläche hat auch das folgende Optionsmenü :
    Hier können Sie den Canvas-Rahmen umschalten und die Größe des Canvas einstellen. Sie haben auch die Möglichkeit, ein Referenzbild festzulegen, d. h. ein Bild, das Sie auf Ihrem Computer auswählen (es wird nicht in das Projekt importiert) und das als Hintergrundreferenz für das sequence verwendet wird. Dieses Bild ist nur im Sequence Editor selbst sichtbar und wird nicht gerendert, wenn das sequence in einem Projekt verwendet wird. Wenn Sie ein Referenzbild hinzufügen, können Sie auch die Deckkraft festlegen, mit der es gezeichnet wird, und seine Position innerhalb des Leinwandrahmens verschieben.

Sequenz-Tools

Diese Leiste enthält die verschiedenen Wiedergabesteuerungen von sequence:

  • Abspielgeschwindigkeit: Dies ist die Abspielgeschwindigkeit der sequence in Bildern (Schritten) pro Sekunde.
  • Abspielkopfposition: Dies ist die aktuelle Abspielkopfposition, die in Zeit (h:m:s) oder in Bildern (f) angezeigt werden kann.
  • Zeitanzeigeoptionen: Hier können Sie einstellen, ob die Zeiten auf sequence in Stunden, Minuten und Sekunden (h:m:s) oder in Frames (f) angezeigt werden sollen.
  • Nachricht senden: Wenn Sie auf diese Schaltfläche klicken, können Sie an der aktuellen Abspielposition eine Nachricht einfügen (dies wird hier näher erläutert).
  • Moment hinzufügen: Wenn Sie auf diese Schaltfläche klicken, können Sie an der aktuellen Abspielkopfposition einen sequence Moment hinzufügen (dies wird weiter unten näher erläutert).
  • Kurveneditor öffnen: Mit dieser Schaltfläche schalten Sie für die aktuell ausgewählte Spur zwischen der Kurvenansicht und der Keyframe-Ansicht um. Diese Option ist nur auf Parameterspuren anwendbar und wenn sie aktiviert ist stehen Ihnen weitere Optionen zur Verfügung, je nachdem, ob die Parameterspur Keyframes oder eine Animationskurve verwendet. Weitere Informationen finden Sie im Abschnitt über die Verwendung von Animationskurven.
  • Wiedergabesteuerung: Mit diesen Schaltflächen können Sie die Vorschau von sequence starten und stoppen sowie den Abspielkopf an den Anfang oder das Ende bewegen. Es gibt auch eine Schaltfläche, mit der Sie einstellen können, ob die sequence loop oder nicht, und ob die Sequenz in einer Schleife "ping-pong" oder wiederholen soll. Beachten Sie, dass sich diese Einstellung darauf auswirkt , wie die sequence abgespielt wird, wenn das Projekt kompiliert und ausgeführt wird.
  • Sequenzlänge: Hier wird die mögliche Gesamtlänge von sequence in Bildern (f) oder in Zeit (m:h:s) angezeigt. Beachten Sie, dass sequence möglicherweise nicht bis zu dieser Länge läuft, wenn Sie den Clipbereich kleiner eingestellt haben.

Wie bereits erwähnt, können Sie einem sequenceMomente hinzufügen. Ein sequence Moment ist eine Position auf der Zeitachse, an der sequence eine Funktion aufrufen kann. Dies ist eine geskriptete Funktion, die Es handelt sich um eine skriptgesteuerte Funktion, die keine Parameter haben muss und die aufgerufen wird, wenn der Zeitpunkt (Frame) auf der Zeitachse erreicht ist. Um einen Zeitpunkt festzulegen, bewegen Sie einfach den Abspielkopf an die gewünschte Position und klicken dann auf die Schaltfläche Zeitpunkt hinzufügen. Dialogfeld geben Sie den Namen der aufzurufenden Funktion an.

Wenn Sie eine Momentfunktion hinzufügen, können Sie auf die Schaltfläche klicken, um zu script mit der angegebenen Funktion zu wechseln, oder Sie können auf die Schaltfläche klicken, um eine neue Ressource script mit einer leeren Funktion zu erstellen, die Sie bearbeiten können. Sie können auch den Moment und den darin enthaltenen Funktionsaufruf entfernen, indem Sie auf die Schaltfläche klicken.

Sequenzmomente können auch mit code hinzugefügt und bearbeitet werden. Für weitere Informationen siehe hier.

Dopesheet

Auf dem Dope Sheet werden alle Asset- undParameter-Tasten für jeden Track angezeigt werden. Sie können auf klicken und die Schlüssel nach links oder rechts ziehen, um sie in der dope sheet Zeitleiste zu positionieren, und Sie können auch auf auf den rechten Rand eines Asset-Schlüssels klicken, um seine Länge zu ändern. Beachten Sie, dass bei sprites und Objekten, wenn Sie die Länge länger als die Animation einstellen, die Animation loop über die zusätzlichen Frames. dope sheet verfügt auch über Steuerelemente zum Festlegen des Beginns und des Endes des Clip-Bereichs für sequence, und Sie können auf klicken, um den Abspielkopf an diese Position zu setzen. Orientierung ändern

Mit der Schaltfläche Ausrichtung ändern können Sie die Art und Weise ändern, in der der Sequence Editor die Anzeige der verschiedenen Abschnitte ändern. Standardmäßig befindet sich die Cnavas-Ansicht oben und der Track-Editor und Dopesheet unten. Dies kann jedoch geändert werden, indem Sie auf die Schaltfläche klicken, um die verschiedenen Layouts zu durchlaufen, oder indem Sie auf die Schaltfläche Schaltfläche Optionen klicken, um ein Menü zu öffnen und das gewünschte Layout auszuwählen. gewünschten Layout. Die folgenden Layouts sind verfügbar:

Geteiltes Fenster

Mit dieser Option wird der Sequence Editor in zwei separate Fenster aufgeteilt, wobei das Hauptfenster IDE die Canvas-Ansicht auf dem gesamten Desktop zeigt und das neue Fenster den Dope Sheet und den Track Editor. Ein erneuter Klick auf die Schaltfläche wird das Layout wieder geändert und das zusätzliche Fenster entfernt.
Schnappverschluss

Mit dieser Option werden der Track-Editor und Dopesheet an den oberen Rand der IDE angeheftet.
Rechts einrasten

Mit dieser Option werden der Track-Editor und Dopesheet rechts neben IDE eingeblendet.
Schnapp-Boden

Mit dieser Option werden der Track-Editor und Dopesheet am unteren Rand von IDE fixiert (dies ist die Standardansicht).
Links einrasten

Mit dieser Option werden der Track-Editor und Dopesheet auf der linken Seite des IDE eingerastet.

Sie können das Layout des Sequence Editors auch jederzeit zurücksetzen, indem Sie auf die Option Zurücksetzen klicken. Dadurch wird der Editor auf seinen Standardzustand zurückgesetzt und alle Abschnitte, die geschlossen waren, werden wieder geöffnet.
