Animierte Kacheln

Kacheln werden im Allgemeinen als statische Zellen im Spielraum betrachtet, aber mit GameMaker ist es möglich, sie zu animieren, genau wie ein sprite. Warum nicht einfach sprites verwenden, fragen Sie sich? Nun, sprites hat einen gewissen Verarbeitungs-Overhead, da da sie mit einem Begrenzungsrahmen und der Art und Weise, wie sie gerendert werden, verbunden sind, aber Kacheln haben einen viel geringeren Verarbeitungsaufwand und sind daher schneller zu rendern. Dies hat jedoch seinen Preis, denn mit Kachelsatz-Animationen sind Sie beschränkt auf Animationen zu erstellen, deren Länge eine Potenz von zwei ist - d. h. 2, 4, 8, 16 usw. Frames -, und sie müssen natürlich innerhalb des Kachelrasters erstellt werden, und sie müssen alle mit der gleichen Geschwindigkeit animiert werden (auf einer Kachel-Set-Basis). Doch selbst mit diesen Einschränkungen sind animierte Kacheln eine leistungsstarke Funktion, die verwendet werden kann, um ansonsten statischen Umgebungen und Hintergründen Leben und Farbe zu verleihen.

Um eine Kachelanimation zu erstellen, müssen Sie zunächst ein Kachelset erstellen, das die benötigten Kacheln alle in einem sprite Bild enthält (aber nicht als sprite Frames, also ein einziges großes sprite Frame mit allen Animationsbildern darin), und dann im Kachelset-Editor auf die Schaltfläche Kachelanimation klicken, um den Animationseditor aufzurufen:

Das obige Beispielbild ist ein einzelnes Bild mit allen darin enthaltenen Animationsbildern, aber es ist zu beachten, dass das Bild auch andere auch andere Kacheln enthalten kann, die keine Animationen enthalten, da Sie ein einziges großes Kachelset verwenden und dann die Teile davon "herauspicken" können, aus denen Sie die Animation erstellen möchten. Zum Beispiel könnte man ein großes Landschaftskachelset mit Bäumen, Erde und Wasser haben, Erde und Wasser, und innerhalb des Kachelsatzes mehrere Kacheln haben, die animiert werden sollen, um das Wasser plätschern zu lassen.

Um eine Kachelanimation zu erstellen, müssen Sie zunächst eine Animation zur Animationsbibliothek hinzufügen, indem Sie auf die Schaltfläche Animation hinzufügen klicken. einen leeren Animationsstreifen zur Bibliothek hinzufügt und Sie auffordert, die Anzahl der zu verwendenden Bilder auszuwählen:

Sie erhalten dann eine Ansicht jedes leeren Animationsrahmens, wobei der Anfangsrahmen für Sie hervorgehoben ist:

Wenn Sie dann zum Kachelsatzbild auf der linken Seite des Editors zurückkehren, können Sie auf auf eine beliebige Kachel klicken, um sie der Animation hinzuzufügen, und das Bild wird zum nächsten übergehen. Wenn Sie auf auf aufeinanderfolgende Kacheln klicken, füllt sich die Sie können die Animation dann in der Vorschau anzeigen, indem Sie auf die Schaltfläche "Abspielen" neben den Bildern klicken:

Sobald Sie dies getan haben, wird Ihre Animation Teil der Animationsbibliothek und wenn Sie den Raumeditor aufrufen, können Sie sie auswählen und im Raum platzieren. Beachten Sie, dass, wenn Sie eine der Kacheln aus einer Animation auswählen, um sie in einem Raum zu platzieren, diese Kachel ab der ausgewählten Kachel animiert wird, so dass Sie eine Animation "desynchronisieren" können, indem Sie separate Rahmen innerhalb des Raums platzieren und sie werden alle animiert. Der einzige Nachteil dabei ist, dass die Animationsgeschwindigkeit immer die gleiche ist, da sie für den gesamten Kachelsatz und nicht für einzelne Animationen innerhalb der Bibliothek definiert ist.

Ein paar Dinge sind bei Kacheln, die in Animationen verwendet werden, zu beachten:

  • Wenn für die Kachel eine eigene Animation festgelegt wurde (d. h. es gibt eine Animation, in der sie als erste Kachel festgelegt ist), wird diese Animation verwendet.
  • Wenn die Kachel keine eigene Animation hat, sondern nur in einer anderen Animation verwendet wird, dann wird diese Animation ab der ersten Position verwendet, an der die Kachel in der Animation vorkommt.
  • Wenn die Kachel keine eigene Animation hat und in mehreren anderen Animationen verwendet wird, dann wird sie überhaupt nicht animiert. Sie müssen dann explizit eine Animation angeben, in der sie die erste Kachel ist.
