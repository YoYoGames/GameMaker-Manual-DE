Der Fliesensatz-Pinsel-Builder

Wenn Sie im Raumeditor Kacheln auf eine Kachelkartenebene "malen", wählen Sie standardmäßig eine einzelne Kachel aus und malen mit dieser. Kachelsätze sind jedoch fast immer so konzipiert, dass sie Abschnitte enthalten die auf unterschiedliche Weise zusammenpassen, um ganze Bereiche oder Gegenstände zu bilden. Ein RPG-Kachelsatz kann zum Beispiel Landschaftskacheln enthalten, die je nach Anzahl der verwendeten Kacheln zu größeren oder kleineren Objekten wie Bäumen und Gebäuden verbunden werden können. verwendet werden. Würden Sie nun mehrere solcher Features auf einer Raumebene platzieren, müssten Sie viele Male hin- und hergehen, um die Kacheln zu wechseln, was für Ihren Arbeitsablauf ungünstig ist. Um dieses Problem zu lösen, ermöglicht GameMaker das Erstellen von Kachelpinseln im dem Kachelsatz-Editor zu erstellen, der verfügbar ist, wenn Sie auf auf die Schaltfläche " Brush Builder " klicken:

Im Brush Builder sehen Sie auf der linken Seite das ursprüngliche Kachelset und auf der rechten Seite eine leere "Leinwand". Sie können nun Sie können nun eine beliebige Kachel auf der linken Seite auswählen und sie auf der rechten Seite malen, um benutzerdefinierte "Pinsel" zu erstellen, die Sie dann im Raumeditor verwenden können. Beachten Sie, dass Sie mit der linken Maustaste auf klicken und diese gedrückt halten und dann auf dem Kachelset ziehen können, um mehrere Kacheln auszuwählen, die Sie auf die Pinsel-Leinwand malen möchten. Sie können auch / + gedrückt halten, um bestimmte einzelne Kacheln hinzuzufügen oder + gedrückt halten, um bestimmte Kacheln zu entfernen.

Rechts sehen Sie drei von uns erstellte Funktionen (im Bild mit einem orangefarbenen Kästchen hervorgehoben im Bild). Beachten Sie, dass wir eine Lücke von einer Kachel zwischen den einzelnen Merkmalen gelassen haben - das liegt daran, dass jede sich berührende Gruppe von Kacheln im Raumeditor als ein einzelner Pinsel behandelt wird. Pinsel ist, den wir erstellen wollen. Während Sie Ihre Pinsel erstellen, können Sie mit der linken Maustaste malen und mit der rechten Maustaste löschen. Sie können auch das Kachelblatt oder die Pinselfläche zoomen, indem Sie / und das Mausrad verwenden, wenn sich die Maus wenn sich die Maus über einem der Fenster befindet, oder Sie können mit + oder die mittlere Maustaste und Ziehen. Beachten Sie, dass Sie auch die Zoom-Werkzeuge in der Toolbox innerhalb der Editorfenster verwenden können, um die Sie können auch die Zoomstufen der beiden Fenster ändern und die Sichtbarkeit und Farbe des Rasters umschalten.

Oben rechts sehen Sie das aktuell ausgewählte Werkzeug, und Sie können auch die Größe des Pinsels einstellen, mit dem Sie malen möchten. Die Standardgröße ist 1, was einer einzelnen Kachel entspricht, aber wenn Sie einen höheren Wert einstellen, können Sie mit einem größeren Pinsel malen (und löschen), der aus der größeren Pinsel malen (und löschen), der aus der Wiederholung der ausgewählten Kachel besteht, wie in der folgenden Abbildung gezeigt:

In der Toolbox können Sie das Werkzeug auswählen, das Sie für viele verschiedene Aufgaben im Kachelsatz-Editor verwenden möchten, von denen einige davon abhängen, ob Sie etwas in Ihrer automatischen Kachelbibliothek definiert haben. Nachfolgend finden Sie eine kurze Beschreibung der einzelnen Werkzeuge (beachten Sie, dass diese Toolbox oben im Raum angezeigt wird, wenn Sie eine Fliesenschicht im Raumeditor ausgewählt haben workspace zusammen mit weiteren Zeichenwerkzeugen angezeigt):

Bleistift Dies ist das Bleistift-Werkzeug. Es verwendet die ausgewählte Kachel, um im Pinsel-Fenster mit der linken Maustaste zu malen und Sie können mit der rechten Maustaste löschen .
Radiergummi Mit dem Radiergummi-Werkzeug können Sie die linke Maustaste verwenden, um eine bestimmte Kachel im Pinsel-Fenster zu löschen. Im Wesentlichen wird dadurch nur der Kachelindex auf 0 gesetzt, was die reservierte "leere" Kachel.
Auswahl Dies ist das Auswahlwerkzeug, mit dem Sie einen Bereich des Pinselfensters für die Bearbeitung festlegen können. Sie können mit der linken Maustaste klicken und dann die Maus ziehen Sie können auch / + drücken, um mehrere Auswahlen hinzuzufügen, und + , um einen Teil der Auswahl zu löschen. Um die gesamte Auswahl zu löschen, können Sie drücken. Wenn Sie einen Bereich des Pinselfensters ausgewählt haben, funktionieren die übrigen Werkzeuge (Stifte, Spiegeln, Drehen usw.) nur innerhalb des ausgewählten Bereichs. Beachten Sie, dass Sie auch Kacheln im Fenster auswählen und dann kopieren ( / + "C"), ausschneiden ( / + "X") und einfügen ( / + " V"), wodurch das Werkzeug switch zum Stift wird und Sie damit "malen" können.
AutoTile Wenn Sie auf dieses Werkzeug klicken, wird der Malstil "Automatisch kacheln " aktiviert. Wenn dies aktiviert ist, können Sie eine beliebige Kachel aus der Registerkarte "Automatische Kacheln" auswählen und sie dann in das Pinsel-Fenster malen, und GameMaker passt sie automatisch an die umliegenden Kacheln an. und GameMaker passt sie automatisch an die umliegenden Kacheln an, sofern du die Registerkarte"Automatische Kacheln" richtig eingestellt hast. Beachten Sie, dass die Auswahl einer Kachel aus dem Kachelsatz, die nicht Teil der automatischen Kachelbibliotheken ist, das Zeichenwerkzeug auf das Standard Bleistift-Werkzeug.
Flip Wenn Sie mit der linken Maustaste auf das Werkzeug "Spiegeln" klicken (oder die Tastenkombination "X" verwenden), wird die Kachel bzw. werden die Kacheln Kacheln, die zum Zeichnen ausgewählt sind, entlang der horizontalen Achse, ohne das Zeichenwerkzeug zu wechseln (wenn Sie einen benutzerdefinierten Pinsel zum Zeichnen ausgewählt haben, wird der gesamte Pinsel gespiegelt). Wenn Sie keine Kachel zum Zeichnen ausgewählt haben, aber eine Gruppe von Kacheln im Pinselfenster ausgewählt haben im Pinselfenster ausgewählt haben, werden die ausgewählten Kacheln mit dem Flip-Werkzeug gespiegelt.
Spiegel Wenn Sie mit der linken Maustaste auf das Werkzeug "Spiegeln" klicken (oder das Tastaturkürzel " Y" verwenden), werden die aktuell zum Zeichnen ausgewählten Kacheln entlang der vertikalen Achse gespiegelt, ohne das Zeichenwerkzeug zu wechseln (wenn Sie einen benutzerdefinierten Pinsel zum Zeichnen ausgewählt haben, wird der gesamte Pinsel gespiegelt). Wenn Sie keine Kachel zum Zeichnen ausgewählt haben, aber eine Gruppe von Kacheln im Pinselfenster ausgewählt haben im Pinselfenster ausgewählt haben, dann spiegelt das Werkzeug Spiegeln die ausgewählten Fliesen.

Wenn Sie keine Kachel zum Zeichnen ausgewählt haben, aber eine Gruppe von Kacheln im Pinselfenster eine Gruppe von Kacheln ausgewählt haben, dann werden die ausgewählten Kacheln mit dem Werkzeug Drehen gedreht.

Unterhalb der Werkzeuge finden Sie zwei verschiedene Abschnitte zur Auswahl beliebiger Auto-Kachel oder animierten Kacheln, die unter Verwendung des aktuellen Kachelsatzbildes erstellt wurden. Ein einzelnes sprite, das für ein Kachelset verwendet wird, kann viele, viele Einzelzellenbilder enthalten, und diese können im Animations- oder Autotile-Editor kombiniert werden, um benutzerdefinierte Pinsel zu erstellen, die in diesen Abschnitten angezeigt werden und in Verbindung mit folgenden Elementen verwendet werden können regulären statischen Kacheln verwendet werden (beachten Sie, dass eine animierte Kachel unabhängig davon animiert wird, ob Sie sie aus der Bibliothek oder aus dem Basis-Kachelsatz ausgewählt haben).

Sobald Sie alle benötigten Pinsel eingerichtet haben, können Sie diese verwenden, um Kacheln auf eine beliebige Kachelkartenebene im Raumeditor zu malen.
