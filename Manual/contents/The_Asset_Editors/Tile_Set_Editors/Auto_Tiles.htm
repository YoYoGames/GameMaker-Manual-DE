Autofliesen

Die Funktion Auto-Kacheln ist ein unglaublich leistungsfähiges Werkzeug, das die Erstellung von Ebenen erheblich vereinfacht, egal ob es sich um ein Projekt in der Draufsicht oder in der Seitenansicht handelt. Im Wesentlichen erstellen Sie eine automatische Kachelbibliothek, und wenn Sie dann Wenn Sie dann eine Kachel aus dieser Bibliothek verlegen, "verbindet" sie sich mit den umliegenden Kacheln und bildet eine nahtlose Wand oder platform.

Bevor Sie die automatische Kachelfunktion verwenden, ist es jedoch sehr wichtig, dass Sie den Kachelsatz sprite richtig eingerichtet haben und dass Sie einen geeigneten Typ von automatischen Kacheln wählen. Sie können zwischen folgenden Typen wählen: 47-Kachel-Autokacheln oder 16-Kachel-Autokacheln , wobei (im Allgemeinen) 16-Kacheln für Top-Down oder Landschaften verwendet werden (da sie schönere Übergänge bieten) und 47-Kacheln für Plattformer/Seitenansichten oder Top-Down, wo Wände mehrere Verbindungen erfordern, aber das aber das ist keineswegs eine feste Regel und hängt davon ab, wie das endgültige Projekt aussehen soll.

Beachten Sie, dass das verwendete Kachelset sprite ALLE benötigten Kacheln enthält und auch andere Kacheln enthalten kann, da Sie auswählen können welche Kacheln in der automatischen Kachel enthalten sein sollen. Sie können nun auf auf die Schaltfläche "47 Kacheln" im Bereich "Vorlagen" klicken, um einen neuen Bibliothekseintrag mit 47 Kacheln zu erstellen. die Vorlage für die Auswahl der benötigten Kacheln aufrufen:

In der Vorlage stellt der hellgraue Bereich die Außenkante der verwendeten Fliese dar, während das dunklere Grau der "Füllbereich" ist. Um eine Kachel in die Vorlage einzufügen, klicken Sie einfach auf die erste verfügbare Zelle der Vorlage und wählen Sie dann die hinzuzufügende Kachel von links aus:

Wenn Sie einen Fehler machen, wählen Sie einfach die zu entfernende Autokachel aus und wählen Sie die "leere" Kachel aus aus dem Kachelsatz. Nachdem Sie alle Kacheln hinzugefügt haben, können Sie die Kacheln mit der Vorlage vergleichen, indem Sie auf die Schaltfläche "Vorlage anzeigen" oben auf der Seite Dadurch wird die Vorlage auf der endgültigen Autokachel eingeblendet, wie in der Abbildung unten gezeigt:

Wenn Sie ein Autotile mit 16 Kacheln herstellen möchten, ist die Vorgehensweise genau dieselbe wie oben beschrieben, nur dass Sie statt der der 47-Kachel-Vorlage verwenden Sie die 16-Kachel-Vorlage:

Wir haben jetzt einige automatische Kacheln in unserer automatischen Kachelbibliothek, die aus einem Kachelsatz stammen. Diese können im Raumeditor verwendet werden, indem Sie einfach eine Kachel Map-Ebene erstellen und dann eine der Auto-Kacheln aus der Bibliothek auswählen (die automatisch den Auto-Kachel-Pinsel oben im Raum-Editor auswählt) und in Ihrem Raum malen. Die Kacheln verbinden sich nun "auto-magisch" und erstellen Kachelkarten in der richtigen Reihenfolge zu erstellen.

Bitte beachten Sie, dass das Verhalten einer Autokachel an den Raumkanten von der Schaltfläche Offene oder geschlossene Kanten abhängt. Wenn eine automatische Fliese entlang der Kante eines Raumes platziert wird, wählt sie standardmäßig eine Fliese, als ob die Außenseite des Raumes leer wäre (d.h.: Sie gibt dem Raum eine "Kante"). Wenn Sie jedoch auf diese Schaltfläche klicken, wird die Autokachel den Raum so kacheln, als ob der Bereich außerhalb des Raumes mit Kacheln gefüllt wäre, und somit Kacheln auswählen, die sich vermischen und keinen "Rand" haben.
