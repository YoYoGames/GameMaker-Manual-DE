Der Notiz-Editor

Der Notiz-Editor öffnet ein Code-/Skript-Fenster, in dem Sie Notizen und code Schnipsel oder andere Informationen speichern können, die Sie für das Projekt als relevant erachten. für das Projekt. Sie können mehrere Notizen erstellen und sie über das Menü der rechten Maustaste im Asset-Browser benennen. Notizen können auch gruppiert und neu geordnet werden wie jedes andere Asset.

Wenn Sie ein Projekt importiert haben, das mit einer früheren Version von GameMaker erstellt wurde, finden Sie in diesem Abschnitt auch einen Kompatibilitätsbericht als Hinweis.
