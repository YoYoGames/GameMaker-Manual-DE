Der Schriftarten-Editor

Wenn du Text in deinem Spiel zeichnen möchtest, wird dieser Text in der Standardschriftart Arial 12 Punkt gezeichnet, aber um interessantere oder einzigartig aussehende Texte zu erstellen, wirst du wahrscheinlich andere Schriftarten verwenden wollen. Um also verschiedene Schriftarten zu verwenden, die du auf deinem Computer hast, musst du in GameMaker ein Schriftarten-Asset erstellen. Dazu klickst du mit der rechten Maustaste auf im Asset-Browser und wählst "Erstellen" -> "Schriftart". Für jede Schriftart, die du erstellst, gibst du eine bestimmte Schriftart von deinem Computer an, die dann in deinem Spiel verwendet werden kann, indem du die Aktion oder code benutzt, um eine Schriftart zum Zeichnen festzulegen.

Wenn Sie zum ersten Mal eine Schriftressource erstellen, öffnet sich das Fenster des Schrifteditors mit den folgenden Optionen:

Einzelheiten

Hier können Sie Ihrer Schriftart einen eindeutigen Namen geben (nur alphanumerische Zeichen und das "_"-Zeichen unter dem Balken) und die zu verwendende Schriftart auswählen. Die Schriftart muss auf Ihrem Rechner installiert sein, und aus lizenzrechtlichen Gründen speichert GameMaker nicht die eigentlichen Schriftdateien standardmäßig nicht mit der Projektdatei, sondern die Schrift wird als eine Reihe von Bitmap-Glyphen auf einer Texturseite gerendert. von Bitmap-Glyphen gerendert. Die Texturseite wird einer Texturgruppe zugewiesen, die Sie auch hier über die Option Gruppe auswählen können.

Wenn Sie die eigentliche Schriftdatei (nur *.ttf oder *.otf ) zusammen mit dem Projekt verteilen möchten, wenn Sie ein Marketplace Asset erstellen oder aus einem anderen Grund, können Sie die Option In Projekt kopieren aktivieren. Diese öffnet den Datei-Explorer, in dem Sie die Schriftdatei auswählen können, die dann zu den mit dem Projekt gebündelten Dateien hinzugefügt wird.

WICHTIG! Sie sollten nur Schriftarten verwenden, deren Weitergabe oder Verwendung in Ihrem Projekt rechtlich zulässig und lizenziert ist.

Voreinstellungen

In diesem Bereich können Sie die Größe (in Punkt - Standardwert ist 12pt) einstellen und festlegen, ob die Schrift fett oder kursiv gezeichnet oder mit Anti-Aliasing versehen werden soll. Das Vorschaufenster auf der rechten Seite zeigt Ihnen den Text ungefähr so an, wie er in Ihrem Spiel mit der von Ihnen angegebenen Größe und den Transformationen aussehen wird.


Bereichskontrollen

Bei der Erstellung einer Schriftart-Ressource ist es selten, dass man jede einzelne Glyphe der Schriftart einbeziehen möchte. Aus diesem Grund verwendet GameMaker standardmäßig nur die Zeichen von 32 bis 127 der gewählten Schriftart, da dies die Zahlen und die Klein- und Großbuchstaben sowie bestimmte Satzzeichen. Sie können jedoch den verwendeten Zeichenbereich ändern, um Ihre Spiele zu optimieren: Wenn Sie nur die Zahlen einer bestimmten Schriftart benötigen, wählen Sie nur die Zahlen aus. Um dies klicken Sie zunächst auf die Schaltfläche Löschen , um den aktuellen Bereich zu löschen (möglicherweise müssen Sie den Bereich markieren, damit dies funktioniert, indem Sie im Vorschaufenster auf Vorschaufenster) und dann auf die Schaltfläche Hinzufügen klicken, um einen neuen Bereich hinzuzufügen. Daraufhin wird das folgende verkettete Fenster geöffnet:

Dieses Fenster enthält einige Schaltflächen, mit denen Sie einen Standardbereich für Ihre Schrift festlegen können:

- Der normale Bereich von 32 bis 127

- Der ASCII-Bereich von 0 bis 255

- Der Ziffernbereich , der nur die 10 Ziffern enthält

- Der Buchstabenbereich , der nur die Groß- und Kleinbuchstaben enthält

Andere Bereiche können verwendet werden, indem Sie den ersten und letzten Zeichenindex des gewünschten Bereichs in die Eingabefelder für den Zeichenbereich eingeben (wenn ein Zeichen nicht im Bereich liegt, wird es durch ein Leerzeichen ersetzt). Abgesehen von diesen Schaltflächen und Eingabefeldern, haben Sie noch zwei weitere Optionen, die besonders nützlich sind:

  • Aus Code: Wenn Sie auf diese Schaltfläche und dann auf OK klicken, analysiert GameMaker Ihr Spiel code automatisch nach Zeichenketten und erstellt dann verschiedene Zeichenbereiche, um den gesamten Text in Ihrem Spiel abzudecken. Beachten Sie, dass er nach alle Zeichenketten (entweder innerhalb von "" oder '') und kann daher auch Dateinamen in die Zeichenbereiche aufnehmen.
  • Aus Datei: Diese Option funktioniert ähnlich wie die oben beschriebene Option "Aus Code". Wenn Sie darauf klicken, werden Sie aufgefordert, eine Datei anzugeben, und wenn das geschehen ist, sollten Sie auf OK klicken. GameMaker parst dann die Datei und erstellt Zeichenbereiche für den darin enthaltenen Text.

Beachten Sie, dass Sie den Textbereich manuell bearbeiten können, indem Sie direkt im Vorschaufenster Textglyphen hinzufügen oder löschen.

Textur-Steuerelemente

Wie oben (im Abschnitt über Schriftartdetails) erwähnt, werden standardmäßig alle Schriftartglyphen auf eine Texturseite als Teil einer Texturgruppe gerendert, um sie in grafische Ressourcen umzuwandeln. Wenn das Spiel kompiliert wird, wird die Schriftart zu einer temporären Textur hinzugefügt, um ein einzelnes Bild mit allen erforderlichen Schriftartglyphen zu erstellen, das dann der Haupttexturgruppe zum Exportieren hinzugefügt wird. Mit den Schaltflächen hier können Sie einstellen, wie die Schrifttextur generiert wird, oder eine neue Schrifttextur generieren.

Über die Schaltfläche Optionen können Sie die erweiterten Optionen für die Schrifttextur aufrufen:

Diese Optionen können die Darstellung einer Schriftart im Spiel manchmal radikal verändern, und nicht alle Schriftarten erfordern dieselben Einstellungen. Die folgenden Optionen sollen Ihnen die größtmögliche Anpassung Ihrer Schriftart ermöglichen:

  • Hinting: Unter Hinting (auch Instruktion genannt) versteht man die Verwendung von mathematischen Anweisungen, um die Anzeige einer Umrissschrift so anzupassen, dass sie mit einem gerasterten Raster übereinstimmt. Bei niedrigen Bildschirmauflösungen ist das Hinting entscheidend für die Erzeugung von klarem, lesbarem Text, und hier können Sie zwischen verschiedenen Hinting-Algorithmen wählen, um das bestmögliche Ergebnis zu erzielen. Die Voreinstellung ist "normal".
  • Glyphen-Optionen: Mit diesen Optionen können Sie die Art und Weise, wie die Glyphen geladen werden, ändern, indem Sie verschiedene Operationen aktivieren/deaktivieren. Diese sind:
    1. Hinting deaktivieren: Damit werden die schriftsprachlichen Hintings und der Auto-Hintergrund deaktiviert.
    2. Auto-Hinterlegung deaktivieren: Dies deaktiviert den Auto-Hintergrund und verwendet stattdessen den nativen Hintergrund der Schriftart (falls vorhanden). Beachten Sie, dass dies die Option "Auto-Hinterlegung bevorzugen" außer Kraft setzt.
    3. Auto-Hintergrund bevorzugen: Diese Option weist GameMaker an, den Auto-Hintergrund zu bevorzugen und nicht den nativen Hintergrund der Schriftart. Beachten Sie, dass dies durch die Optionen "Auto-Hintergrund deaktivieren" und "Hinting deaktivieren" überschrieben wird.
    4. Verzerren: Wenn Sie diese Option aktivieren, werden Glyphen leicht verzerrt, um die Rasterausrichtung zu verbessern. Beachten Sie, dass hierfür der Auto-Hintergrund und der Hinting-Modus "Normal" erforderlich sind.
    5. Keine Skalierung: Wenn diese Option aktiviert ist, bleiben die Glyphen in Schrifteinheiten und ignorieren die angeforderte Größe. Dies bedeutet, dass das Hinting deaktiviert ist.
    Die Standardeinstellung für alle oben genannten Optionen ist deaktiviert.
  • Interpreter: Hier können Sie die TrueType-Version auswählen, die GameMaker zum Rendern von Schriften verwenden soll. Standard ist TrueType v40.
  • Rundung: Mit dieser Option wird die Rundungsmethode bei der Konvertierung von einer Festkomma-Dezimalgröße in eine Ganzzahl festgelegt. Standard ist "floor".
  • Kerning anwenden: Hier wird festgelegt, ob die Kerning-Daten aus der Schriftartdatei für das Rendern im Spiel verwendet werden sollen. Dies sollte deaktiviert werden, wenn die Kerning-Daten für die Quellschriftart falsch sind und dazu führen, dass die Schriftart nicht korrekt angezeigt wird.

Mit der Schaltfläche Neu generieren können Sie die Schrifttextur neu generieren. Dies ist erforderlich, wenn Sie Änderungen an einer bestehenden Schrift aktualisieren möchten oder wenn Sie eine neue Schrift erstellt haben und sehen möchten, wie sie im Vorschaufenster aussieht. Beachten Sie, dass Sie Schriften auch über das Menü der rechten Taste für die Schrift(en) im Asset-Browser neu generieren können.

Wenn Sie eine neue Schriftart zum ersten Mal erstellen und die Textur nicht im Editor neu generieren, wird die Textur beim ersten Ausführen oder Testen des Spiels erstellt, aber das ist das einzige Mal, dass sie automatisch generiert wird. Das bedeutet, dass, sobald eine Schriftart eine generierte Textur hat, alle Änderungen, die an der Schriftart-Ressource vorgenommen werden, erfordern, dass die Schriftart neu generiert wird, damit die Änderungen im Spiel sichtbar werden, andernfalls wird die "alte" Schriftart-Textur weiterhin verwendet.

GameMaker rendert die generierte Schrifttextur als PNG Datei zusammen mit den übrigen Projektdateien. Wenn Sie das Projekt laden und die Basisschriftart nicht gefunden wird, zeigt das Spiel den Text trotzdem korrekt mit der gespeicherten Textur an. Eine fehlende Schriftart bedeutet jedoch, dass Sie außer dem Namen, der Schriftfamilie und der Texturgruppe (über den Texturgruppeneditor) keine weiteren Eigenschaften mehr ändern können.

Vorschau-Modus

In diesem Abschnitt können Sie wählen, ob im Vorschaufenster ein Beispieltext der ausgewählten Schriftart oder das gerenderte Texturbild, das für die Schriftart erstellt wurde, angezeigt werden soll.

Vorschau

Das Vorschaufenster können Sie eine Vorschau der Schriftart sehen, entweder als Textmuster oder als Textur. Wenn der Vorschaumodus eingestellt ist auf Muster eingestellt ist, können Sie jeden Text, den Sie testen möchten, direkt in das Vorschaufenster eingeben, um zu sehen, wie er aussieht, während die Auswahl von Spielvorschau zeigt das gerenderte Texturbild an, das für die Schriftart erstellt wurde. Wenn kein Bild vorhanden ist, sollten Sie auf die Schaltfläche Neu generieren klicken, um die Textur zu erzeugen.

Beachten Sie, dass das Texturbild vom Muster abweichen kann, da die Textur nicht neu generiert wird, es sei denn, dies wird ausdrücklich gewünscht. Das bedeutet, dass nach der anfänglichen Erstellung einer Schrifttextur Änderungen an der Schriftressource nicht in das Spiel übernommen werden, es sei denn, die Textur wird neu erstellt.

Bereich Vorschau

Hier werden die verschiedenen Zeichenbereiche, die Sie für die Schriftart hinzugefügt haben, aufgelistet. Die Werte der Bereiche werden auf der linken Seite angezeigt und die Glyphen, die diesen Werten entsprechen, werden auf der rechten Seite angezeigt.
