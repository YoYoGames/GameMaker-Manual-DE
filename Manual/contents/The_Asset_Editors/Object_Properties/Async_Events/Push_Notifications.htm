Push-Benachrichtigungen

Das Push-Benachrichtigungsereignis wird ausgelöst durch den Rückruf von Push-Benachrichtigungen auf dem Betriebssystem des Geräts ausgelöst, entweder von einer lokalen Quelle oder einer entfernten Quelle. Es erzeugt eine DS-Map, die exklusiv für dieses Ereignis ist und in der speziellen Variable async_load.

Dieses Ereignis wird ausschließlich von Erweiterungen (z. B. der Firebase Cloud Messaging-Erweiterung ) verwendet. Anweisungen zur Verwendung dieses Ereignisses finden Sie daher in der Dokumentation Ihrer Erweiterung.
