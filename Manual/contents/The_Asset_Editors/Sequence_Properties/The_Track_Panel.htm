Das Gleis-Panel

Was Ihre sequences tun, hängt von den Assets und Parametern ab, die dem Track Panel hinzugefügt wurden. Sie können eine Spur zum Editor hinzufügen, indem Sie entweder:

  • Ziehen eines Assets in den sequence Canvas
  • Ziehen eines Assets in den Spurbereich selbst
  • indem Sie auf die Schaltfläche "Asset hinzufügen" am unteren Rand des Spurfensters klicken.

Diese letzte Methode öffnet den Asset Explorer, in dem Sie das hinzuzufügende Asset auswählen können.

Unabhängig von der gewählten Methode wird eine neue Asset-Spur für das Asset erstellt. Diese Spur ist die Hauptspur für das Asset auf sequence, und es wird eine entsprechende Asset-Key-Spur für das Asset auf Dope Sheet an der Abspielposition erstellt:

Beachten Sie, dass die Farbe des Asset-Schlüssels auf Dope Sheet über das Spurbedienfeld eingestellt werden kann, indem Sie auf den farbigen Balken am Ende klicken. Dadurch wird der Farbwähler geöffnet, in dem Sie die Farbe für die Schlüsselspuren auf Dope Sheet auswählen können.

Jedem Asset im Track-Editor kann auch eine Reihe von Parameterspuren hinzugefügt werden. Diese Spuren steuern die verschiedenen Eigenschaften (Parameter), die im Laufe der Zeit für die Spur geändert werden können. Die verschiedenen verfügbaren Parameter werden im Abschnitt "Hinzufügen von Parameterspuren" weiter unten auf dieser Seite ausführlich behandelt:

  • Alle Assets: Position und Rotation.
  • Töne: Tonhöhe und Lautstärke.
  • Sequenzen: Ursprung und Umfang.
  • Sprites und Objekte: Ursprung, Maßstab, Bildindex, Bildgeschwindigkeit und Farbmultiplikation.
  • Text: Ursprung, Maßstab, Farbmultiplikation, Zeichenabstand, Rahmengröße, Zeilenabstand und Absatzabstand.

Um eine Parameterspur im Editor hinzuzufügen, können Sie die Option "Änderungen automatisch aufzeichnen" in der Spurtafel aktivieren, die dann Parameterschlüssel für alle Änderungen, die Sie an einem Asset im Canvas an der aktuellen Abspielkopfposition vornehmen, wie hier gezeigt, hinzufügt:

Alternativ können Sie auch auf die Schaltfläche "Parameterspur hinzufügen" klicken, woraufhin sich ein Menü mit den verschiedenen verfügbaren Optionen öffnet:

Wenn Sie eine Option aus diesem Menü auswählen oder direkt auf IDE aufnehmen, werden eine oder mehrere Parameterspuren unterhalb der Haupt-Asset-Spur erstellt und außerdem eine Parametertaste auf Dope Sheet hinzugefügt, die die gleiche Farbe wie die Haupt-Asset-Spur hat:

Wenn Sie eine Parameterspur zu Ihrer Haupt-Assetspur hinzufügen, wird auf dope sheet ein farbiger Balken angezeigt, der die Länge der Asset-Spur in Frames angibt, und es werden Parameterschlüssel (Keyframes) an den Stellen hinzugefügt, an denen eine Änderung des Parameters erfolgen soll. Diese Schlüssel sind in der obigen Abbildung als einfache "Punkte" auf einem Frame zu sehen. Wenn eine Spur über solche Parameterschlüssel verfügt, wird der Parameter, auf den sie sich beziehen, standardmäßig zwischen jedem der angezeigten Punkte interpoliert (mit linearer Interpolation). Wenn Sie z. B. einen Rotationsparameter haben und dieser bei Frame 0 auf 90° eingestellt ist, dann ist er bei Frame 30 auf 180° eingestellt, und bei Frame 15 auf 135°. Sie können die Veränderung der Werte im Laufe der Zeit sehen, indem Sie die Parameterspur auswählen und dann mit dem Abspielkopf im Dope Sheet "scrubben":

Beachten Sie, dass Sie für alle Parameterspuren die standardmäßige lineare Interpolation zwischen verschiedenen Tasten über das Menü der rechten Maustaste auf der Parameterspur deaktivieren können, was bedeutet, dass die Übergänge zwischen den Parameterwerten sofort auf dem gegebenen Frame und nicht schrittweise über die Zeit erfolgen:

Neben der Auswahl des Übergangstyps für Parameterspuren bietet Ihnen das RMB-Menü auch folgende Optionen:

  • Ausschneiden, Kopieren und Einfügen: Mit diesen Optionen können Sie Asset- oder Parameterspuren ausschneiden/kopieren/einfügen.
  • Spur(en) duplizieren: Hiermit wird ein Duplikat der aktuell ausgewählten Spur(en) erstellt. Wenn es sich um eine Asset-Spur handelt, werden auch die damit verbundenen Parameterspuren dupliziert.
  • Spur(en) löschen: Dies löscht die ausgewählte(n) Spur(en). Wenn Sie eine Kühlstellenspur löschen, werden auch die mit ihr verbundenen Parameterspuren entfernt.
  • Gruppieren: Wenn Sie diese Option auf einem leeren Platz im Spurenkontrollzentrum verwenden, wird ein Gruppenordner erstellt, den Sie dann verwenden können, um Ihre Assets in einem geordneten Format zu halten. Sie können ein Asset zu einer Gruppe hinzufügen, indem Sie es einfach anklicken und in den Ordner ziehen. Wenn Sie bereits eine oder mehrere Spuren ausgewählt haben, wird mit dieser Option nicht nur ein Gruppenordner erstellt, sondern auch die Spur(en) hinzugefügt.
  • Umbenennen: Mit dieser Option können Sie jeden Asset-Track oder Gruppenordner umbenennen.

Eine wichtige Funktion von Asset- und Parameterspuren ist die Möglichkeit, Spuren und Gruppen zu überordnen . Wenn Sie eine Asset-Spur auf eine andere im Track-Panel ziehen, wird die erste Asset-Spur zu einem Kind der Asset-Spur, auf der sie abgelegt wurde, die wiederum zur übergeordneten Asset-Spur wird. Alle Parameterspuren, die der übergeordneten Spur zugewiesen sind, werden dann automatisch auf die untergeordneten Spuren übertragen, so dass Sie z. B. mehrere sprites haben können, die alle eine einzige Positionsspur zum Verschieben verwenden:. Das Gleiche gilt für Gruppen. Sie können einen Gruppenordner erstellen, indem Sie auf die Schaltfläche am unteren Rand des Spurfensters klicken und dann GML die Asset-Spuren, die Sie in dem Gruppenordner zusammenfassen möchten, visuell darstellen. Diese Kühlstellen behalten ihre eigenen Parameterspuren, aber Sie können der Gruppe zusätzliche Parameterspuren zuweisen und alle Kühlstellen in der Gruppe werden von diesen Parameterspuren sowie von ihren eigenen individuellen Parameterspuren beeinflusst.

Bei der Arbeit mit dem Track-Panel können auch die üblichen Tastenkombinationen verwendet werden, d.h.: / und die linke Maustaste , um einen oder mehrere Tracks in beliebiger Reihenfolge auszuwählen, und die linke Maustaste , um aufeinanderfolgende Tracks auszuwählen, und Sie können sie auch ausschneiden, kopieren und einfügen, indem Sie die Tastenkombinationen / +"X", / + "C" und / + "V" verwenden.

Das sind die Grundlagen für die Verwendung des Track-Editors. Im Folgenden gehen wir etwas ausführlicher darauf ein, wofür die einzelnen Abschnitte da sind und wie man sie verwendet:

Orientierung ändern

Mit der Schaltfläche "Ausrichtung ändern" können Sie die Art und Weise ändern, wie der Sequence Editor die verschiedenen Abschnitte anzeigt. Standardmäßig befindet sich die Cnavas-Ansicht oben und der Track-Editor und Dopesheet unten. Sie können dies jedoch ändern, indem Sie auf die Schaltfläche klicken, um die verschiedenen Layouts zu durchlaufen, oder indem Sie auf die Schaltfläche Optionen klicken, um ein Menü zu öffnen und das gewünschte Layout auszuwählen. Die folgenden Layouts sind verfügbar:

Geteiltes Fenster

Mit dieser Option wird der Sequence Editor in zwei getrennte Fenster aufgeteilt, wobei das Hauptfenster IDE die Canvas-Ansicht auf dem gesamten Desktop und das neue Fenster den Dope Sheet und den Track Editor anzeigt. Wenn Sie erneut auf die Schaltfläche klicken, wird das Layout wieder geändert und das zusätzliche Fenster entfernt.
Schnappverschluss

Mit dieser Option werden der Track-Editor und Dopesheet an den Anfang der IDE gesetzt.
Rechts einrasten

Mit dieser Option werden der Track-Editor und Dopesheet rechts neben IDE eingeblendet.
Schnapp-Boden

Mit dieser Option werden der Track-Editor und Dopesheet am unteren Rand von IDE fixiert (dies ist die Standardansicht).
Links einrasten

Mit dieser Option werden der Track-Editor und Dopesheet auf der linken Seite des IDE eingerastet.

Sie können das Layout des Sequence Editors auch jederzeit zurücksetzen, indem Sie auf die Option Zurücksetzen klicken. Dadurch wird der Editor auf seinen Standardzustand zurückgesetzt und alle Abschnitte, die geschlossen waren, werden wieder geöffnet.

Spurensuche

Mit diesem Suchfeld können Sie schnell die verschiedenen Spuren auf sequence durchsuchen, um die Spur zu finden, die Sie bearbeiten möchten. Wenn Sie hier einen beliebigen Text eingeben, wird angezeigt, wie viele Asset-Spuren mit dieser Zeichenfolge gefunden wurden, und die Pfeile neben dem Suchfeld werden hervorgehoben. Wenn Sie auf diese Pfeile klicken, springen Sie zu den ersten und nachfolgenden Kühlstellen, die die Suchkriterien erfüllen.

Asset-Tracks

Jedes Asset, das zu sequence hinzugefügt wird, erhält eine eigene Spur, die hier aufgelistet wird. Wenn für die Asset-Spur Parameter festgelegt wurden, werden diese ebenfalls als Parameterspuren darunter angezeigt. Sie können die Parameterspuren erweitern oder verkleinern, indem Sie auf auf das Pfeilsymbol links neben dem Spurnamen klicken, und Sie können den Spurnamen bearbeiten, indem Sie einen langsamen Doppelklick auf die Spur ausführen und ihr dann einen Namen Ihrer Wahl geben. Beachten Sie, dass es keine Einschränkungen für den Namen gibt, den Sie vergeben, und Sie können Leerzeichen, nicht-alphanumerische Zeichen verwenden und mit einer Zahl beginnen, wenn Sie möchten.

Die verschiedenen Arten von Parameterspuren werden im Abschnitt "Hinzufügen von Parameterspuren" weiter unten auf dieser Seite näher erläutert.

Schlüsselkontrollen

Sobald Sie eine Parameterspur hinzugefügt haben, können Sie Parameterschlüssel zu bestimmten Frames hinzufügen und entfernen, um den Wert für diesen Parameter festzulegen. Dies kann über die Tasten "Record Key" oder "Delete key" Taste (weitere Informationen finden Sie im Abschnitt "Track Key Controls"), oder Sie können die Key Controls am Parameter selbst verwenden. Um diese Steuerelemente zu verwenden, müssen Sie den Abspielkopf auf Dope Sheet auf den Frame positionieren, den Sie bearbeiten möchten, und dann auf das kreisförmige "Key"-Symbol auf der linken Seite der Parameterspur klicken. Wenn dem Frame keine Taste zugewiesen ist, ist der Kreis "leer". Wenn Sie darauf klicken, wird eine neue Taste hinzugefügt und Sie können den Parameterwert bearbeiten. Wenn dem Rahmen bereits eine Taste zugewiesen ist, ist der Kreis "gefüllt", und wenn Sie darauf klicken, wird die Taste aus dem Rahmen entfernt:

Sie werden auch feststellen, dass Sie im obigen Bild die Pfeiltasten auf beiden Seiten der Parametertastenmarkierung verwenden können, um schnell zwischen den Schlüsselbildern auf dope sheet zu springen, ohne den Abspielkopf manuell positionieren zu müssen.

Parameter Werte

Wenn eine Parameterspur hinzugefügt wurde, können Sie die Werte für einige von ihnen bearbeiten, indem Sie einfach den Abspielkopf in Dope Sheet auf den Frame mit dem zu ändernden Key setzen und dann die verschiedenen Canvas-Widgets verwenden, um die Position, Drehung, Skalierung oder den Ursprung des Assets zu bearbeiten. Diese Änderungen werden in den Schlüsselwert für dieses Bild übernommen. In diesen Fällen können Sie den Parameterwert direkt bearbeiten, indem Sie ihn in der Spur anklicken und auf den gewünschten Wert für den Parametertyp setzen.

Beachten Sie, dass einige Parameterspuren ein "Link"-Symbol neben ihren Werten haben können, was bedeutet, dass sie miteinander verknüpft werden können. Klicken Sie dazu auf das Symbol, um die Verknüpfung zu aktivieren/deaktivieren. Wenn es aktiviert ist, wird beim Bearbeiten eines Wertes auch der andere eingestellt.

Veranstaltungen

Ähnlich wie bei regulären Objekten können sequences Ereignisse zugewiesen werden, die einige code ausführen können. Die code wird in Form einer geskripteten Funktion zugewiesen, die keine arguments annehmen kann und aufgerufen wird, wenn das Ereignis ausgelöst wird.

Ereignisse werden hinzugefügt, indem Sie auf die Schaltfläche Ereignis hinzufügen klicken, wodurch sich das folgende Fenster öffnet:

Die hier aufgeführten Veranstaltungen sind wie folgt:

  • Erstellen - Dieses Ereignis tritt ein, wenn eine Instanz von sequence zum ersten Mal erstellt wird, und ist das allererste, was in einem sequence passiert, das über den Raumeditor in den Raum gestellt wird, wenn ein Raum betreten wird.
  • Destroy - Dieses Ereignis wird ausgeführt, wenn eine Instanz von sequence zerstört wird, und wird vor dem Ereignis Clean Up (siehe unten) ausgeführt.
  • Aufräumen - Dieses Ereignis wird nach jedem Ereignis aufgerufen, das eine Instanz von sequence aus dem Raum entfernt. Es wird also ausgelöst, wenn sequence zerstört wird, wenn der Raum endet, oder wenn das Spiel endet, und ist dafür gedacht, jede Aufgabe auszuführen, die einmal ausgeführt werden muss, wenn die sequence Instanz aus irgendeinem Grund aus dem Spiel entfernt wird. Wenn du die sequence Instanz z.B. mit layer_sequence_destroy() zerstört hast, dann wird dieses Ereignis nach dem Destroy Event (siehe oben) aufgerufen.
  • Begin Step / Step / End Step - Das Step-Ereignis ist ein Ereignis, das bei jedem einzelnen Schritt (Frame) des Spiels überprüft wird, solange die Instanz sequence existiert, und ist in drei Teile unterteilt: begin, step und end. Für die meisten Dinge ist das Standard-Schritt-Ereignis gut geeignet, aber manchmal möchte man etwas mehr Kontrolle darüber haben, was code zu welchem Zeitpunkt ausführt. Dafür gibt es die Schritt-Ereignisse Begin und End, die immer in der gleichen Reihenfolge bei jedem Schritt (Frame) des Spiels ausgelöst werden. Wichtig! Wenn sequence pausiert, werden diese Ereignisse nicht ausgelöst, und wenn das Spiel fortgesetzt wird, werden sie im nächsten Frame ausgelöst , nachdem sequence wieder zu spielen beginnt. Beachten Sie auch, dass die Reihenfolge der Ereignisse nicht von der Richtung des Abspielkopfes beeinflusst wird. Selbst wenn sequence rückwärts abgespielt wird, werden die Ereignisse immer noch als Beginn, Schritt und Ende ausgeführt.
  • Asynchrones Ereignis: Dies ist das Äquivalent zum Objekt Asynchronous System Event. Wichtig! Wenn die sequence pausiert, wird dieses Ereignis nicht ausgelöst.

Jedem Ereignis kann eine einzelne Funktion zugewiesen werden, die aufgerufen wird, wenn das Ereignis ausgelöst wird. Sie können die Funktion mit Hilfe des Eingabefeldes für die Entlüftung zuweisen, und ein Klick auf die Pfeilschaltfläche öffnet den script Editor für die zu bearbeitende Funktion. Sie können auch auf die Schaltfläche Neu erstellen am unteren Rand klicken, um eine neue script Ressource mit bereits definierten und ausfüllbaren "Boilerplate"-Funktionen zu erstellen. Beachten Sie, dass Sie die Funktionsnamen nach Belieben ändern können und nicht die vordefinierten Namen verwenden müssen, und Sie können auch alle nicht benötigten Funktionsdefinitionen entfernen. Es ist wichtig zu beachten, dass Funktionen, die für sequence Ereignisse verwendet werden, keine arguments annehmen können.

Sequenzereignisse können auch mit code hinzugefügt und bearbeitet werden. Für weitere Informationen siehe hier.

Sichtbar / Sperr-Schalter

Sowohl Asset-Tracks als auch Parameter-Tracks verfügen über eine Schaltfläche zum Umschalten der Sichtbarkeit von Tracks und zum Sperren des Tracks . Wenn Sie die Sichtbarkeit der Spur deaktivieren, wird sie nicht angezeigt, wenn sequence in einem Raum ausgeführt wird, und sie wird auch nicht im sequence Canvas während des Testens angezeigt, und das Umschalten der Sichtbarkeit einer Asset-Spur schaltet auch die Sichtbarkeit der zugehörigen Parameter-Spuren um. Das Sperren einer Spur bedeutet einfach, dass diese Spur nicht mehr bearbeitet werden kann. Beachten Sie, dass unsichtbare Spuren auf dope sheet verblassen, während gesperrte Spuren mit einem Overlay gezeichnet werden, so dass Sie sofort sehen können, in welchem Zustand sich jede Spur befindet:

Spur Farbe

Jeder von Ihnen erstellten Kühlstellenspur wird von IDE eine zufällige Farbe zugewiesen, damit sie auf Dope Sheet leichter zu erkennen ist. Diese Farbe kann jederzeit geändert werden, indem Sie einfach auf den Farbbalken für die Spur klicken und dann die gewünschte Farbe aus dem sich öffnenden Farbauswahlfenster auswählen. Jede für eine Kühlstellenspur gewählte Farbe wird automatisch auf alle zugehörigen Parameterspuren angewendet. Sie können jedoch die Farbe jeder Parameterspur einzeln bearbeiten, um sie bei Bedarf zu ändern. Beachten Sie, dass eine erneute Änderung der Farbe der Kühlstellenspur alle Änderungen an den Parameterspuren außer Kraft setzt.

Hinzufügen von Parameterspuren

Die Schaltfläche Parameter hinzufügen kann verwendet werden, um eine einzelne Parameterspur zur ausgewählten Kühlstellenspur hinzuzufügen. Wie bereits auf dieser Seite erwähnt, gibt es verschiedene Arten von Parameterspuren, je nach Asset, auf das sie sich beziehen. Nachfolgend finden Sie eine Liste der verschiedenen verfügbaren Parameterspuren und ihrer Bezeichnungen sowie der Kühlstellen, auf die sie angewendet werden können:

  • Position: Diese Parameterspur kann zu jedem der vier sequence Asset-Typen(Sprites, Objekte, Sounds oder Sequenzen) hinzugefügt werden und wird verwendet, um die räumliche Position innerhalb von sequence für das Asset festzulegen. Er nimmt einen X- und einen Y-Wert an.
  • Drehung: Wie der Parameter "Position" ist auch der Parameter "Drehung" für alle Sequenz-Assets verfügbar und kann verwendet werden, um die Drehung in Grad für das Asset festzulegen. Beachten Sie, dass GameMaker die Drehung gegen den Uhrzeigersinn berechnet, beginnend mit 0° rechts, also 90° ist oben, 180° ist links und 270° ist unten.
  • Ursprung: Der Ursprungsparameter ist für Sprite-, Objekt- und Sequenz-Assets verfügbar und kann verwendet werden, um die X- und Y-Position des Ursprungs des Assets festzulegen. Eine Änderung des Ursprungs wirkt sich darauf aus, wie das Asset positioniert, gedreht und skaliert wird.
  • Maßstab: Der Parameter scale ist für Sprite-, Objekt- und Sequenz-Assets verfügbar und wird verwendet, um den horizontalen und vertikalen Maßstab des Assets festzulegen. Dies wirkt sich darauf aus, wie das Asset auf IDE und runtime gerendert wird, und wird als skalarer Wert berechnet, wobei 100 % die Standardgröße des Assets ist, 50 % wäre die halbe Größe und 200 % wäre die doppelte Größe. Beachten Sie, dass Sie die Möglichkeit haben, die horizontalen und vertikalen Skalenwerte zu verknüpfen, indem Sie auf die Schaltfläche "verknüpfen" klicken. Das bedeutet, dass die Änderung eines Wertes automatisch auch die Änderung des anderen Wertes zur Folge hat.
  • Bildindex und Bildgeschwindigkeit: Diese Parameter sind nur für Sprite- und Objekt-Assets verfügbar und können verwendet werden, um den aktuellen Animations-Frame zu ändern (siehe die Seite über die image_index für weitere Informationen) sowie die aktuelle Animationsgeschwindigkeit, die mit FPS eingestellt wird und genauso funktioniert wie der Wert, den Sie für das sprite im sprite Editor einstellen.
  • Farbe multiplizieren: Dieser Parameter ist nur für Sprite- und Objekt-Assets verfügbar und kann verwendet werden, um die aktuelle Mischfarbe für das gezeichnete sprite zu ändern. Dies ist das Äquivalent zur Einstellung des image_blend Wertes und alle hier eingestellten Farben werden mit der Farbe von sprite"multipliziert" (d.h. überblendet). Ein Wert von Weiß (#ffffffff) bedeutet, dass keine Überblendung stattfinden soll. Beachten Sie, dass Sie bei dieser Art von Spur die Farbe durch manuelle Eingabe des Hexadezimalwerts festlegen können, oder Sie können auf das Farbfeld klicken, um den GameMaker-Farbwähler zu öffnen. Beachten Sie, dass Farben im ARGB-Format angegeben werden, so dass Sie mit dieser Art von Parameterspur einen Wert für Alpha einstellen können (der Farbwähler erlaubt Ihnen auch, den Alpha-Wert einzustellen). Dies wäre das Äquivalent zum Setzen des image_alpha für das Sprite.
  • Lautstärke: Der Lautstärke-Parameter gilt nur für Sound-Asset-Spuren und kann verwendet werden, um die Lautstärke des Sounds einzustellen (die tatsächliche Lautstärke des Sound-Assets wird nicht verändert, da es sich im Wesentlichen um einen Emitter handelt, der für die Asset-Spur in sequences verwendet wird). Der Lautstärkewert wird als Prozentsatz berechnet, wobei 0% keine Lautstärke und 100% volle Lautstärke bedeutet.
  • Tonhöhe: Der Pitch-Parameter gilt nur für Sound-Asset-Spuren und kann verwendet werden, um die Tonhöhe des Klangs einzustellen (wie bei der Lautstärke wird die tatsächliche Verstärkung des Sound-Assets nicht verändert, da es sich im Wesentlichen um einen Emitter handelt, der für die Asset-Spur in sequences verwendet wird). Der Tonhöhenwert wird als Prozentsatz berechnet, 100 % ist die "natürliche" Tonhöhe des Klangs, und ein Wert unter 100 % senkt sie ab, während ein Wert über 100 % sie anhebt.
  • Falloff: Der Falloff-Parameter gilt nur für Sound-Asset-Spuren und ist derzeit nicht implementiert.
  • Es gibt einige zusätzliche Parameter, die nur für Textspuren verfügbar sind. Bitte lesen Sie Text in Sequenzen, um Informationen dazu zu erhalten.

Aktionen verfolgen

Die Schaltflächen am unteren Rand des Track-Editors sind die verschiedenen Track-Aktionen und können für die folgenden Zwecke verwendet werden:

  • - Diese Schaltfläche öffnet den Asset-Explorer, in dem Sie ein Asset auswählen können, das Sie als neue Spur verwenden möchten. Nach der Auswahl wird die Asset-Spur erstellt und die Frames werden an der aktuellen Abspielkopfposition in Dope Sheet eingefügt.
  • - Wenn Sie eine oder mehrere Asset-Spuren im Track-Editor ausgewählt haben, können Sie sie durch Klicken auf diese Schaltfläche duplizieren (zusammen mit den darin enthaltenen Parameterspuren).
  • - Wenn Sie darauf klicken, wird eine spezielle Schnittmaskengruppe erstellt, in der Sie sprites als Maske verwenden und sprites maskieren können. Siehe hier für weitere Informationen.
  • - Wenn Sie darauf klicken, wird dem Track-Editor ein neuer Gruppenordner hinzugefügt, der dann zur besseren Organisation der Tracks verwendet werden kann. Sie können eine Spur zu einer Gruppe hinzufügen, indem Sie die Spur in den Gruppenordner ziehen, und Sie können sie wieder entfernen, indem Sie sie aus dem Ordner ziehen. Beachten Sie, dass jedes Asset, das einem Gruppenordner hinzugefügt wird, automatisch die Farbe des Ordners erhält, obwohl Sie die Farbe der einzelnen Spuren bei Bedarf ändern können.
  • - Wie zu erwarten, wird durch Anklicken dieser Schaltfläche die ausgewählte(n) Spur(en) gelöscht. Wenn es sich um eine Asset-Spur handelt, werden auch alle darin enthaltenen Parameterspuren gelöscht. Sie können die Funktion jedoch auch auf ausgewählte Parameterspuren anwenden, um sie allein zu löschen, ohne die Asset-Spuren zu entfernen, mit denen sie verknüpft sind. Wenn Sie diese Funktion auf einen leeren Gruppenordner anwenden, wird der Ordner gelöscht. Wenn sich jedoch Spuren in diesem Ordner befinden, wird eine Warnung angezeigt, da beim Löschen eines Ordners auch alle darin enthaltenen Spuren gelöscht werden.
