Die Sequence Leinwand

Bei der Erstellung von sequences für Ihre Projekte wird ein großer Teil der Arbeit im Canvas des Sequenzeditors erledigt. Die Leinwand ist einfach der Bereich, der für die sequence Elemente verwendet wird, und sie definiert auch, was angezeigt wird, wenn die sequence zu einem Raum hinzugefügt wird, der angezeigt werden soll. Wenn Sie zum ersten Mal eine sequence erstellen, wird die Leinwand auf die Standardgröße von 1366px x 768px eingestellt, aber dies kann von der Toolbox aus geändert werden, indem Sie auf neben der Schaltfläche Canvas Frame Toggle klicken:

Die Größe der Leinwand definiert einen visuellen Bereich, in dem die Assets platziert werden, aus denen sequence besteht. Der Rahmen ist jedoch nur ein Richtwert für die Größe und hat keinen Einfluss darauf, wie die Dinge gerendert werden, d. h. wenn Sie ein Asset sprite außerhalb des Rahmens platzieren und sequence zu einem Raum hinzufügen, wird das Asset trotzdem gerendert.

Der Rahmen wird auch zur Aufnahme von Referenzbildern verwendet, die Sie für die Positionierung von Assets innerhalb von sequence verwenden möchten. Ein Referenzbild kann über dasselbe Menü " Leinwandoptionen " hinzugefügt werden, in dem Sie auch die Größe des Leinwandrahmens festlegen, und wird als Hintergrundbild für die sequence Leinwand verwendet. Auf diese Weise können Sie die Elemente z. B. entsprechend einem UI-Mock-up eines Projektkünstlers positionieren:

Beachten Sie, dass Referenzbilder nicht mit dem Projekt gespeichert werden und nicht gerendert werden, wenn sequence zu einem Raum hinzugefügt wird.

Sobald Sie die Leinwand nach Ihren Wünschen eingerichtet haben, können Sie damit beginnen, ihr Assets hinzuzufügen. Dazu ziehen Sie sie einfach aus dem Asset-Browser in die Arbeitsfläche, wodurch eine Asset-Spur im Spurbedienfeld erstellt wird:

Folgende Vermögenswerte können zu einer sequence hinzugefügt werden:

  • Sprites
  • Objekte
  • Klingt
  • Sequenzen

Sie können auch das Werkzeug Text in der Toolbox verwenden, um eine Textspur zu erstellen. Lesen Sie Text in Sequenzen für weitere Informationen.

Gizmos

Sobald Sie die benötigten Elemente hinzugefügt haben, können Sie diese an ihren Anfangspositionen für die Animation platzieren und mit Hilfe der verschiedenen Transformationssteuerungs-Gizmos transformieren. Auf diese kann auf zwei verschiedene Arten zugegriffen werden:

  • Bewegen Sie die Maus zu den Rändern des Assets im Canvas, klicken Sie auf und ziehen Sie (beachten Sie, dass sich der Cursor ändert, um die verschiedenen Transformationen anzuzeigen, die geändert werden):
  • Indem Sie das Transformationsgizmo am Ursprung des Assets verwenden, das Sie in der Canvas Toolbox ändern können:

Parameter-Tasten

Wenn Sie die anfänglichen Transformationen für das Asset festgelegt haben (falls erforderlich), können Sie das Canvas verwenden, um der Asset-Spur im SpurbedienfeldParameterschlüssel hinzuzufügen. Ein einfaches Beispiel dafür wäre eine einfache Positionstransformation, bei der Sie das Asset an seiner Startposition platzieren, dann den Abspielkopf im Dope Sheet zu dem Frame bewegen, an dem die Bewegung enden soll, und dann das Asset erneut bewegen:

Wie Sie in der obigen Abbildung sehen können, werden die neuen Positionen automatisch auf der Spur aufgezeichnet, da wir die Option "Änderungen automatisch aufzeichnen" im Spurbereich aktiviert haben. Sie können auch sehen, dass der Leinwandansicht eine gepunktete Linie hinzugefügt wurde, um die Bewegung des Assets durch die Zeit anzuzeigen. Wenn Sie die automatische Aufzeichnung nicht aktiviert haben, können Sie immer noch Punkte hinzufügen, indem Sie das Asset verschieben, dann den Abspielkopf setzen und schließlich auf die Schaltfläche "Einen neuen Key aufnehmen" klicken.

Intelligente Leitfäden

Ein letzter erwähnenswerter Punkt ist, dass der Canvas die Verwendung und das Hinzufügen von intelligenten Hilfslinien erlaubt, d. h. Hilfslinien, die dem Canvas hinzugefügt werden können, damit die Assets an ihnen ausgerichtet werden können. Standardmäßig zeigt der Canvas horizontale und vertikale Hilfslinien um den Mittelpunkt sowie eine rechteckige Hilfslinie um den "Rand" des sequence Canvas an, je nach Größe des Canvas. Diese Hilfslinien werden in der endgültigen Version von sequence nicht gerendert, wenn sie zu einem Raum hinzugefügt werden, und dienen nur dazu, Ihnen eine Vorstellung vom Raum zu vermitteln und die Positionierung von Dingen zu erleichtern. Sie können auch Ihre eigenen Hilfslinien hinzufügen, indem Sie auf auf ein Lineal klicken und es dann auf die Leinwand ziehen:

Um eine Hilfslinie zu entfernen, klicken Sie einfach darauf und ziehen Sie sie dann wieder in Richtung der Lineale zurück. Wenn Sie einem sequence Hilfslinien hinzugefügt haben, rastet das Verschieben eines Assets an der Hilfslinie ein, wenn der Begrenzungsrahmen oder der Ursprung nahe genug ist, und es werden zusätzliche intelligente Hilfslinien angezeigt, um die Abstände zwischen den Assets zu markieren sowie um Assets zu markieren, die sich alle auf derselben Hilfslinie oder an derselben Position befinden (diese sind unterschiedlich gefärbt):

RMB-Menü und Navigation

Um in der Leinwandansicht zu navigieren, können Sie dieselben grundlegenden Steuerelemente wie für eine reguläre Workspace verwenden, d.h.: Verwenden Sie die mittlere Maustaste und ziehen Sie, um die Leinwand zu verschieben (oder verwenden Sie alternativ die Tasten + ), und Sie können mit dem Mausrad horizontal scrollen oder mit der Taste / und dem Mausrad zoomen und verkleinern.

Um eine Kühlstelle auszuwählen, klicken Sie einfach mit der linken Maustaste darauf. Sie können auch mehrere Kühlstellen auswählen, indem Sie / und verwenden, um auf jede Kühlstelle zu klicken. Sie können auch mehrere Kühlstellen auswählen, indem Sie verwenden und dann auf klicken und ziehen. Dadurch wird ein "Kasten" in der Arbeitsfläche erstellt, und jedes Asset, das den Kastenbereich berührt, wird ausgewählt.

Wenn Sie ein oder mehrere Assets ausgewählt haben, können Sie diese mit den Tastenkombinationen / +"X", / + "C" und / + "V" ausschneiden, kopieren und einfügen.

Wenn mehrere Assets im Canvas ausgewählt sind, wird für jedes einzelne Asset ein Begrenzungsrahmen und für alle zusammen ein größerer Begrenzungsrahmen angezeigt. Sie können das Verhalten dieses Begrenzungsrahmens mit den Symbolen in der oberen linken Ecke ändern:

Wenn Sie auf das Kreissymbol klicken, wird das Transformationssteuerungs-Widget auf den Ursprung des gesamten Begrenzungsrahmens für alle ausgewählten Kühlstellen gesetzt, während ein Klick auf das Kastensymbol ein individuelles Steuerungs-Widget für jede Kühlstelle in der Auswahl anzeigt. Sie können gruppierte Kühlstellen verschieben und sie kopieren/einfügen usw., wie Sie es bei einer einzelnen Kühlstelle tun würden.

Wenn Sie mehrere Assets haben, die sich überschneiden, und ein bestimmtes auswählen möchten, ohne die anderen zu stören, können Sie mit der Maus darüber fahren, um die Objekt-Toolbox aufzurufen, die eine Liste von Symbolen für jedes Objekt unter der Maus anzeigt. Sie können dann auf eines dieser Elemente klicken, um es auszuwählen und zu bearbeiten:

Wenn Sie mit der rechten Maustaste auf eine beliebige Kühlstelle in der Sequenz klicken, stehen Ihnen außerdem eine Reihe zusätzlicher Optionen zur Verfügung:

Die hier verfügbaren Optionen sind:

  • Aufzeichnen: Mit dieser Option können Sie Parameterschlüssel für das/die Asset(s) aufzeichnen. Wenn Sie den Mauszeiger darüber bewegen, werden Optionen zum Aufzeichnen der Skalierung, Drehung oder Position angezeigt.
  • Gruppe: Wenn Sie diese Option wählen, wird ein neuer Gruppenordner im Spurenbedienfeld erstellt, in den das/die ausgewählte(n) Asset(s) verschoben werden.
  • Umbenennen: Wenn Sie auf diese Schaltfläche klicken, wird die Spur in der Spurtafel hervorgehoben und der Name kann geändert werden.
  • Im Blatt suchen: Wenn Sie auf diese Schaltfläche klicken, wird der Schlüssel des ausgewählten Assets in Dope Sheet hervorgehoben.
  • Andere suchen: Wenn Sie auf diese Schaltfläche klicken, wird eine Liste mit allen anderen Kühlstellen desselben Typs angezeigt, aus der Sie eine auswählen können.
  • Auswählen: Wenn Sie darauf klicken, wird eine Liste aller Assets angezeigt, die sich an der angeklickten Position überschneiden, und Sie können jedes davon zur Bearbeitung auswählen.
  • Schlüssel entfernen: Wenn Sie diese Option wählen, wird der Asset-Track-Schlüssel an der aktuellen Abspielkopfposition von dope sheet entfernt. Die Spur selbst bleibt bestehen, ebenso wie alle Parameterspuren, die sie enthält, und Sie können der Spur ein anderes Asset wieder hinzufügen, indem Sie es aus dem Asset-Browser in die dope sheet ziehen.
  • Spur entfernen: Dies entfernt die Asset-Spur (und eventuell zugewiesene Parameter-Spuren) aus dem Spur-Panel, sowie alle zugehörigen Tasten im Dope Sheet.
  • Editor öffnen: Damit wird das Asset zur Bearbeitung im Standard-Editor für den jeweiligen Asset-Typ geöffnet.
