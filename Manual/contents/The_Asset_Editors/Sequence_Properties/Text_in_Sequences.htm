Text in Sequenzen

Textspuren können über den Editor zu einem Sequence hinzugefügt werden. Dazu wählen Sie in der Canvas-Toolbox die Schaltfläche Textspur hinzufügen und klicken (und ziehen) dann in die Leinwand, um eine Textspur zu erstellen.

Diese Seite behandelt die folgenden Themen:

  1. Text erstellen
  2. Ändern von Text
  3. Schriftart Assets
  4. Parameter
  5. Inspektor

Text erstellen

Es gibt zwei Möglichkeiten, eine Textspur zu erstellen. Nachdem Sie das Werkzeug Text aus dem Werkzeugkasten ausgewählt haben, können Sie:

  • Klicken Sie einmal in die Leinwand: Dadurch wird eine "nicht umrahmte" Textspur erstellt. Das heißt, der Text kann so lang sein, wie Sie wollen, und er bleibt in einer Zeile. Der Umbruch kann später leicht über den Inspektor aktiviert werden.
  • Klicken und ziehen Sie im Canvas: Dadurch wird eine "umrahmte" Textspur erstellt, d. h. der Text wird nur innerhalb eines Rahmens angezeigt. Die Größe dieses Rahmens wird durch Ziehen bei der Erstellung der Textspur festgelegt. Die Größe kann auch später über Parameterspuren oder den Inspektor geändert werden. Es wird empfohlen, umrahmte Textspuren zu verwenden, da es dadurch viel einfacher ist, den Text auszurichten und konsistente Ergebnisse bei der Animation zu erzielen.

Ändern von Text

Wenn Sie eine neue Textspur erstellen, öffnet GameMaker den Inspektor und wählt automatisch das Textfeld aus, damit Sie mit dem Schreiben Ihres Textes beginnen können.

Um den Text für eine Spur jederzeit zu bearbeiten, wählen Sie einfach diese Spur aus, gehen in den Inspektor und ändern das Textfeld erneut.

Verhalten bei Größenänderung

Die Größenänderung Ihrer Textspur im Canvas funktioniert unterschiedlich, je nachdem, ob Ihre Textspur umbrochen oder nicht umbrochen ist.

Wenn sie nicht umrahmt ist, wird Ihre Textspur skaliert, ähnlich wie Sie sprites und Objekte skalieren. Wenn Sie jedoch die Größe einer umrahmten Textspur ändern, wird die Rahmengröße geändert, nicht die Skalierungswerte.
Um eine umrahmte Textspur zu skalieren, verwenden Sie stattdessen Gizmos, oder bearbeiten Sie die Werte in der Parameterspur Skalierung manuell.

Schriftart Assets

Textspuren sind an Font-Assets gebunden. Wenn Sie eine neue Textspur erstellen, wird in Ihrem Projekt nach einem Schrift-Asset gesucht und das zuletzt hinzugefügte verwendet. Wenn Ihr Projekt keine Schriftarten enthält, wird automatisch ein neues Schriftarten-Asset erstellt.

Wenn Sie eine Schriftart löschen, werden alle Textspuren, die diese Schriftart verwenden, ebenfalls gelöscht. Stellen Sie also sicher, dass Sie nicht versehentlich Schriftarten löschen, die in Sequences verwendet werden.

Parameter

Sie können alle üblichen Parameter, die für eine sprite Spur zur Verfügung stehen, wie Position, Drehung, Skalierung, Ursprung und Farbmultiplikation ändern und animieren. Mit diesen Parametern können Sie Ihren Text verschieben und drehen, ihn vergrößern oder verkleinern und seine Farbe und sein Alpha ändern.

Es gibt auch einige textspezifische Parameter, die ebenfalls animiert werden können:

  • Zeichenabstand: Dies ist der Abstand zwischen den einzelnen Zeichen (in Pixeln).
  • Zeilenabstand: Dies ist der Abstand zwischen den einzelnen Textzeilen (in Pixeln).
  • Absatzskalierung: Dies ist der Abstand zwischen den einzelnen Textabschnitten (in Pixeln). Ein neuer Absatz beginnt, wenn Sie manuell einen Zeilenumbruch hinzufügen, indem Sie drücken.
  • Rahmengröße: Dies ist die Größe des Textrahmens und wird nur verwendet, wenn der Umbruch im Inspektor aktiviert ist (weitere Informationen dazu finden Sie im Abschnitt unten).

Inspektor

Es gibt einige Eigenschaften, die nur über den Inspektor geändert werden können. Solche Eigenschaften können nicht animiert werden, da sie keine Parameterspuren sind, sondern einfache Optionen, die für die gesamte Spur gelten.

Der Inspektor zeigt Ihnen die folgenden Eigenschaften an, wenn eine Textspur ausgewählt ist:

HINWEIS Der Inspektor sollte sich automatisch öffnen, wenn Sie eine Textspur erstellen oder auswählen. Wenn der Inspektor jedoch geschlossen ist, können Sie ihn manuell über das Windows-Menü oben öffnen.

Einige der Parameterspuren (z. B. Farbe, Zeichenabstand, Zeilenabstand usw.) sind hier enthalten, und ihre Änderung erzeugt einen Keyframe am Abspielkopf, wenn Änderungen automatisch aufzeichnen aktiviert ist, oder ändert den ausgewählten Keyframe. Die übrigen Optionen sind "statisch" (da sie nicht animiert werden können) und lauten wie folgt:

  • Im oberen Bereich wird das ausgewählte Schrift-Asset angezeigt. Mit dieser Option können Sie ein vorhandenes Schrift-Asset auswählen oder ein neues erstellen und es der ausgewählten Textspur zuweisen.
  • Text: Dies ist der Text, der auf Ihrer Textspur angezeigt wird. Sie können hier mehrere Zeilen eingeben, leere Zeilen werden jedoch nicht angezeigt.
  • Ausrichten: Hier können Sie festlegen, wie Ihr Text innerhalb des Rahmens horizontal (links, mittig, rechts, ausgerichtet) und vertikal (unten, oben, Mitte) ausgerichtet wird. Dies gilt nur, wenn die Option Umbruch aktiviert ist.
  • Umbruch: Hier wird festgelegt, ob der Text innerhalb des festgelegten Rahmens automatisch umbrochen werden soll. Wenn dies aktiviert ist, springt Ihr Text automatisch in die nächste Zeile, wenn er das Ende des Rahmens erreicht, und wird immer abgeschnitten, wenn der Text länger als die Höhe des Rahmens ist. Wenn diese Funktion deaktiviert ist, wird Ihr gesamter Text in einer einzigen Zeile angezeigt, und der Rahmen wird nicht mehr im Canvas angezeigt.
