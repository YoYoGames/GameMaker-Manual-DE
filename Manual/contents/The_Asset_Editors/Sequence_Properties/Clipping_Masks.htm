Freistellungsmasken

Wie bereits auf der Seite des Spurpanels erwähnt, gibt es eine Aktion, mit der Sie im Spureditor eine so genannte Schnittmaskengruppe erstellen können. Dazu klicken Sie auf die Schaltfläche am unteren Rand des Spurpanels:

Wenn Sie auf diese Schaltfläche klicken, wird eine Schnittmaskengruppe erstellt, die aus zwei Untergruppen von Assets besteht, die entweder als das Motiv oder die Maske der Gruppe. Im Grunde ist das Motiv das Asset, das maskiert werden soll, und die Maske ist das Asset, das die Maskierung vornimmt:

Die Idee hinter der Verwendung einer Schnittmaske ist, dass Sie ein Motiv sprite und dann eine Maske sprite hinzufügen, wobei die Maske sprite das Motiv "ausblendet". (Maske) das Motiv. Sie können dann die Animation der Maske sprite verwenden oder die Maske sprite verschieben oder skalieren, um die Sichtbarkeit des Motivs (oder von Teilen des Motivs) zu ändern. Jede Gruppe (Maske und Motiv) kann mehrere Elemente enthalten, und jedes Element kann mehrere Parameterspuren haben.

Um ein Motiv und eine Maske hinzuzufügen, ziehen Sie einfach das sprite, Objekt oder sequence aus dem Asset-Browser auf den Ordner, dem Sie es hinzufügen möchten:

Die Maske sprite alpha wird verwendet, um das darunter liegende Objekt zu "verstecken", wobei ein Wert von 100 % alpha (undurchsichtig) es vollständig maskiert und ein Wert von 0 % alpha (transparent) es sichtbar macht. Zum Beispiel haben wir diese sprite für die Maskierung:

Wenn die Kombination und die sequence abgespielt werden, enthüllt die Schnittmaskengruppe das darunter liegende Basis-Sprite darunter und erzeugt so unseren Abkling-Effekt.

Beachten Sie, dass Schnittmaskengruppen in der Spurliste und auf dope sheet als ein einzelnes Element behandelt werden können, und Sie können der Gruppe als Ganzes Dreh- und Positionsparameterspuren hinzufügen. Sie können Schnittmaskengruppen auch auf Gruppenordner von Assets anwenden oder sogar auf andere Schnittmaskengruppen anwenden, was sie zu einem sehr leistungsfähigen Werkzeug macht.
