Der Pfad-Editor

Bei der Erstellung deiner Spiele kann es vorkommen, dass eine deiner Instanzen einem Pfad durch ein Level folgen muss. Nun, das kann in code gemacht werden, indem man einfach ein Array von Positionen im Raum erstellt und dann eine Instanz sich zwischen ihnen bewegen lässt, aber das kann eine lästige Aufgabe sein und ist schwierig zu ändern, zu testen und für verschiedene Instanzen anzupassen. Aus diesem Grund hat GameMaker Pfad-Ressourcen und einen eigenen Pfad-Editor.

Die Grundidee ist recht einfach: Sie definieren einen Pfad, indem Sie ihn im Pfad-Editor zeichnen. Dann können Sie eine Aktion (oder code) in das Erstellungsereignis eines Objekts einfügen, um die Instanz anzuweisen, diesem bestimmten Pfad zu folgen. Sie können auch die Geschwindigkeit festlegen, mit der dem Pfad gefolgt wird, sowie eine Reihe anderer Aktionen, die sich auf die Position und Ausrichtung des Pfades im Raum beziehen. Pfad-Eigenschaften

Wie bei allen Ressourcen in GameMaker sollten Sie Ihrem neuen Pfad einen eindeutigen Namen geben, da dieser zur Identifizierung im gesamten Spiel verwendet werden wird. voranzustellen (oder zu ergänzen), um die Wiedererkennung in code oder Aktionen zu erleichtern (z. B.: path_Enemy_1 oder PowerUp_path).

Sobald Sie den Pfad benannt haben, können Sie ihn definieren, indem Sie Punkte im Haupteditor auf der rechten Seite platzieren, wobei jeder Punkt mit seiner Position und Geschwindigkeit in die Liste aufgenommen wird. Der Geschwindigkeitswert für Pfadpunkte wird durch einen Prozentsatz definiert der Geschwindigkeit, mit der sich die Instanz entlang des Pfades bewegt (Standard ist 100%). Wenn du also eine Instanz mit einer Geschwindigkeit von 2 (2 Pixel pro Schritt) entlang eines Pfades startest und dann einer der Pfadpunkte einen Geschwindigkeitswert von 50% hat, wird die Instanz die Geschwindigkeit bei der Annäherung an diesen Punkt auf eine Geschwindigkeit von 1 (50% der Pfadgeschwindigkeit). Genauso kannst du die Geschwindigkeit auf 150% setzen und die Instanz wird beschleunigen, wenn sie den Punkt erreicht. Auf diese Weise kannst du eine schöne, dynamisch aussehende Bewegung ohne jeglichen Code wie auch immer.

Es ist erwähnenswert, dass Sie einen schnellen Doppelklick auf einen der definierten Punkte in der Liste machen können, um den Editor auf oder durch einen langsamen Doppelklick auf auf einen der Werte, um diesen manuell zu bearbeiten (x, y oder Geschwindigkeit).

Sie können auch die Art der Verbindung für den gesamten Pfad angeben und ob der Pfad geschlossen sein soll oder nicht (ein geschlossener Pfad verbindet den letzten Punkt des Pfades mit dem ersten). Die Art der Verbindung kann entweder geradlinig oder glatt sein, wobei eine gerade Verbindung einfach die Punkte eines Pfades auf linearer Basis verbindet, mit Winkeländerungen an jedem Punkt, während eine glatte Verbindung einen viel weicheren Pfad erzeugt, der möglicherweise nicht direkt durch einen der Punkte verläuft, sondern eher eine eine ungefähre Linie, die durch weiche Interpolation schöne Kurven erzeugt. Sie können auch die Genauigkeit für die Kurven angeben (Standardwert ist 4), die zwischen 1 und 8 liegen kann, wobei 8 die am glattesten ist. Pfad-Bearbeitung

Sie können dem Pfad Punkte hinzufügen, indem Sie auf an einer beliebigen Stelle im Bearbeitungsfenster klicken, und alle nachfolgenden Klicks fügen weitere Punkte hinzu. Sobald ein Punkt im Editor platziert ist, kann er verschoben werden, indem Sie Klicken Sie auf und ziehen Sie ihn, oder ändern Sie die Werte, die Ihnen links für die X- und Y-Position und die Geschwindigkeit angezeigt werden (siehe Pfadeigenschaften, oben). Sie Sie können / verwenden und mit der linken Maustaste auf mehrere Punkte klicken, um sie gemeinsam auszuwählen, und diese Gruppe von Punkten kann dann beliebig verschoben und platziert werden. Sie können auch drücken, um den ausgewählten Punkt oder die ausgewählten Punkte zu löschen.

Im Editor-Fenster steht Ihnen ein zusätzliches Menü zur Verfügung, wenn Sie mit der rechten Maustaste auf klicken (siehe unten):

Die einzelnen Punkte in diesem Menü funktionieren wie folgt:

  • Punkte löschen - Löscht den oder die ausgewählten Punkte.
  • Auf Pfad schwenken - Schwenkt die Ansicht im Editor auf die Mitte des Pfades.
  • Umkehren - Kehrt die Reihenfolge der Pfadpunkte um. Dadurch werden sie nicht verschoben, sondern die Reihenfolge, in der sie verarbeitet werden, wird geändert, so dass der Anfang zum Ende wird und umgekehrt.
  • Spiegeln - Spiegelt den Pfad entlang der horizontalen Achse.
  • Spiegeln - Spiegelt den Pfad entlang der vertikalen Achse.

Der Werkzeugkasten

Mit den Schaltflächen des Werkzeugkastens am oberen Rand können Sie steuern, wie die Dinge im Editorfenster gezeichnet werden. Sie können das Hintergrundraster ein- oder ausschalten und festlegen, ob die zusätzlichen Pfadpunkte an diesem Raster ausgerichtet werden sollen oder nicht:

Die Fangwerte geben die Größe der Gitterzellen an und kann auf jeden beliebigen Wert geändert werden. Die übrigen Werkzeuge dienen zum Vergrößern und Verkleinern sowie zum Schwenken des Editorbereichs. Sie können auf die Schaltfläche klicken, um die Anzeige auf 1:1 zurückzusetzen. Wenn Sie sich zu weit von der Stelle entfernt haben, an der Sie den Pfad zeichnen möchten, können Sie das Menü der rechten Maustaste verwenden und dann auf Auf Pfad schwenken klicken, um zu der Stelle zurückzuscrollen, an der Sie gearbeitet haben.


Wegpunkte

Pfade sind einfach eine Sammlung von Punkten, die durch eine Linie miteinander verbunden sind. Im Editor platzieren Sie diese Punkte mit der linken Maustaste . Sie können Punkte an beliebiger Stelle im Raum-Editor hinzufügen und sie werden nacheinander in die Punkteliste eingefügt, um einen Pfad zu bilden. Wenn Sie einen Punkt in der Nähe einer Verbindungslinie oder eines anderen Knotens platzieren, wird der neue Punkt in den Pfad eingefügt und nicht zu ihm hinzugefügt. (Sie können den Erkennungsabstand dafür in den Voreinstellungen festlegen). Sie können / drücken und auf auf bereits platzierte Punkte klicken, um sie auszuwählen, und dann werden alle Operationen, die an den Pfadpunkten vorgenommen werden, wie z. B. das Verschieben, auf alle die ausgewählten Punkte als Gruppe angewendet. Um einen Punkt oder eine Auswahl von Punkten zu entfernen, können Sie mit der rechten Maustaste auf klicken und "Punkt löschen" wählen. Punkt" wählen oder die Taste drücken.

Beachten Sie, dass Sie auch die folgenden Tastenkombinationen verwenden können, um einige zusätzliche Operationen durchzuführen, die nicht über das Menü der rechten Maustaste auf dem/den ausgewählten Pfadpunkt(en) verfügbar sind:

  • T + : Verschieben der Pfadpunkte
  • R + : Drehen der Pfadpunkte
  • S + : Skalieren der Pfadpunkte
