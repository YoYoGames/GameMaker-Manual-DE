Die Animationskurven-Bibliothek

Im Fenster Animationskurvenbibliothek können Sie den Kurventyp für die Animationskurve ändern und eine Voreinstellung auf den ausgewählten Kanal anwenden. eine Voreinstellung auf den ausgewählten Kanal anwenden. Es kann geöffnet werden, indem Sie mit der linken Maustaste auf die Schaltfläche im Animationskurven-Editor oder im Sequenzdiagramm klicken oder indem Sie mit der rechten Maustaste irgendwo im Kanal-Editor klicken.

Kurve Typ

Der Kurventyp steuert, wie ein Kanal von Punkt zu Punkt fließt. Er kann auf eine der folgenden Optionen eingestellt werden:

  • Linear: Damit wird ein linearer Verlauf zwischen den Punkten des Kanals erzeugt.
  • Glatt: Diese Option verwendet die Katmull-Rom-Interpolation, um einen glatten Verlauf zwischen den Punkten des Kanals zu erzeugen.
  • Bezier: Hier wird die Mathematik der Bezier-Kurve verwendet, um einen sanften Verlauf zwischen den Punkten des Kanals zu erstellen, wobei Sie das Aussehen der Kurve durch die Bearbeitung von "Griffen" an den Punkten steuern können.

Beachten Sie, dass durch das Ändern des Kurventyps dieser Typ auf alle Kanäle in der Animationskurve angewendet wird.

Kurven-Voreinstellungen

EineKurvenvorgabe ändert die Form Ihrer Kurve. Wenn Sie mit der linken Maustaste auf eine Vorgabe im Menü Vorgaben klicken, wird diese Vorgabe auf den ausgewählten Kanal angewendet. Wenn Sie mit der linken Maustaste auf eine Voreinstellung im Menü Voreinstellungen klicken, wird diese Voreinstellung auf den ausgewählten Kanal angewendet.

HINWEIS: Voreinstellungen können nur angewendet werden, wenn der Kurventyp auf Bezier eingestellt ist.

Standardmäßig wird eine Voreinstellung auf den gesamten Kanal angewendet. Sie können einen Bereich Ihres Kanals auswählen, indem Sie mit der linken Maustaste im Kanal-Editor ziehen und dann das Fenster Animationskurven-Bibliothek über die Schaltfläche oder durch einen Rechtsklick auf öffnen, wodurch alle Voreinstellungen nur auf den ausgewählten Bereich angewendet werden.

Beachten Sie, dass Ihr Kurvenkanal (oder eine Auswahl innerhalb dieses Kanals) eine Steigung haben muss, damit eine Voreinstellung eine Wirkung hat, da jede angewandte Interpolation auf der vertikalen Differenz zwischen dem Anfangs- und Endpunkt basiert. Bei einer flachen Kurve gibt es keinen solchen Unterschied zwischen den beiden Punkten, so dass Sie nach der Anwendung einer Voreinstellung auf eine solche Kurve keine Ergebnisse sehen werden.

Sie können zwischen zwei Anwendungsmodi wählen, die sich darauf auswirken, wie eine Voreinstellung auf Ihren Kanal (oder auf Ihre Auswahl innerhalb dieses Kanals) angewendet wird:


  • Überschreiben: Dies ist die Standardeinstellung, wenn das Fenster über die Schaltfläche im Animationskurven-Editor oder über Sequence Dope Sheet geöffnet wird. Es überschreibt alle Punkte zwischen dem Anfangs- und Endpunkt Ihres Kanals oder Ihrer Auswahl.
  • Zwischen: Dies ist die Standardeinstellung, wenn das Fenster durch einen Rechtsklick auf eine beliebige Stelle im Kanal-Editor ( ) geöffnet wird. Es wendet die Voreinstellung zwischen jedem Paar ausgewählter Punkte an, und fügt alle neuen Punkte hinzu, die benötigt werden, ohne vorhandene Punkte zu entfernen.

Außerdem können Sie zwischen drei Entspannungsmodi wählen, die steuern, wie eine Kurve beschleunigt oder verlangsamt wird. Der Standardmodus ist " Abschwächen".

  • Leichtes Anfahren: Die Kurve beginnt mit einer schnelleren Beschleunigung und verlangsamt sich dann
  • Abschwächen: Die Kurve beginnt mit einer langsameren Beschleunigung und beschleunigt sich dann mit der Zeit
  • Langsam ein- und aussteigen: Die Kurve beginnt mit langsamer Beschleunigung, beschleunigt mit der Zeit und verlangsamt sich dann
