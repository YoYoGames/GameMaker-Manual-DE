Der Zeitleisten-Editor

In den meisten Spielen möchte man, dass bestimmte Dinge zu bestimmten Zeitpunkten passieren. Man kann versuchen, dies mit Hilfe von Alarmereignissen zu erreichen, aber wenn die Dinge zu kompliziert werden, funktioniert das nicht mehr, zumal man auf nur zwölf Alarme beschränkt ist. Aus diesem Grund gibt es die Zeitleiste . In einer Zeitleiste legen Sie fest, welche Aktionen zu bestimmten Zeitpunkten im Spiel stattfinden müssen, und Sie können alle Aktionen verwenden, die auch für ein Objekt in seinen verschiedenen Ereignissen oder code verfügbar sind, um dies zu definieren. Sobald du eine Zeitleiste erstellt hast, kannst du sie einem Objekt zuweisen, und die Instanz dieses Objekts wird dann die Aktionen zu den angegebenen Zeitpunkten ausführen, wenn sie in einem Raum platziert oder erstellt wird.

Nehmen Sie zum Beispiel an, dass Sie einen Wächter erstellen möchten und dass sich dieser Wächter 20 Zeitschritte nach links, dann 10 nach oben, dann 20 nach rechts, dann 10 nach unten und schließlich anhalten soll. Um dies zu erreichen, können Sie eine Zeitleiste erstellen, in der Sie zunächst eine Bewegung nach links festlegen. Dann erstellen Sie den Zeitpunkt 20 (ein "Zeitpunkt" ist nur eine andere Bezeichnung für "Spielschritt" oder "Spielframe") und setzen eine Bewegung nach oben, dann zum Zeitpunkt 30 eine Bewegung nach rechts, zum Zeitpunkt 50 eine Bewegung nach unten und schließlich zum Zeitpunkt 60 halten Sie die Bewegung an. Danach können Sie diese Zeitleiste dem Wächterobjekt zuweisen, und der Wächter wird genau das tun, was Sie geplant haben, wenn er sich in einem Raum befindet. Sie können in der Zeitleiste sogar einen Alarm einstellen, so dass sie nach einer bestimmten Zeit wiederholt wird, und auf diese Weise kann Ihr Wächter einen Bereich patrouillieren. Du kannst eine Zeitleiste auch verwenden, um dein Spiel globaler zu steuern, indem du zum Beispiel ein unsichtbares Controller-Objekt erstellst, dann eine Zeitleiste, die zu bestimmten Zeitpunkten Feinde erzeugt, und sie dann dem Controller-Objekt zuweist und eine Instanz in den Raum setzt. Dies ist ein einfacher Weg, um zeitgesteuertes Spawnen von Feinden in einem Weltraum-Shooter-Spiel oder ähnlichem zu erreichen.

Wenn Sie einen neuen Moment hinzufügen, werden Sie möglicherweise aufgefordert, zwischen GML Visual und GML Code zu wählen. Siehe GameMaker-Sprache für weitere Informationen.

Um eine Zeitleiste zu erstellen, klicken Sie mit der rechten Maustaste auf im Asset-Browser und wählen Sie "Zeitleiste erstellen". Daraufhin wird das oben auf dieser Seite abgebildete Fenster mit den folgenden Abschnitten geöffnet:

Name

Hier geben Sie Ihrer Zeitleistenressource einen Namen. Dieser darf nur alphanumerische Zeichen oder den Unterstrich "_" enthalten und darf nicht mit einer Zahl beginnen, da er zur Referenzierung Ihrer Zeitleiste über code verwendet wird. Dieser Name wird auch die eindeutige Zeitleisten-ID sein, die in GML Visuelle Aktionen oder code Funktionen zur Identifizierung der Zeitleiste verwendet werden kann.


Moment

Hier können Sie den Momentwert einstellen. Der Momentwert wird als die Anzahl der Spielschritte nach dem Start der Zeitleiste berechnet. Wenn Sie also eine Zeitleiste mit Zeitpunkten an den Positionen 0 und 60 haben, wird der Zeitpunkt 0 ausgelöst, wenn die Zeitleiste gestartet wird, und der Zeitpunkt 60 wird 60 Schritte später eintreten. Wenn die Raumgeschwindigkeit ebenfalls auf 60 eingestellt ist, wird der Zeitpunkt 60 eine Sekunde nach dem Start der Zeitleiste ausgelöst. Sie können einen Doppelklick auf den Zeitwert machen oder Sie können das Menü der rechten Taste verwenden und " Ändern" wählen (siehe den Abschnitt über das Tastenmenü unten), um ihn zu ändern. Um ein neues Moment hinzuzufügen, klicken Sie einfach auf die Schaltfläche Hinzufügen am unteren Rand des Editors. Beim Hinzufügen neuer Momente wird der Code-Editor (oder Aktions-Editor) geöffnet, in dem Sie die Funktionen oder GML Visual hinzufügen können, die Sie in diesem Moment aufrufen möchten.


Moment Name

Damit Sie besser verstehen, was Ihre Zeitleiste tut, ist es erlaubt, Momenten einen Namen zu geben, um sie zu identifizieren. Dabei handelt es sich lediglich um eine Zeichenkette, die weder im Spiel code noch an anderer Stelle verwendet wird, da sie lediglich der Organisation und dem Verständnis der Vorgänge in der Zeitleiste dient. Um Ihrem Moment einen Namen/eine Beschreibung zu geben, müssen Sie das Format /// @description in der ersten Zeile des Blocks code verwenden, zum Beispiel:

/// @description Hello World

Der Zeitleisten-Editor ermöglicht es Ihnen, mit + alle Momente zwischen dem ersten Klick und dem zweiten Klick auszuwählen, oder Sie können mit / + auf einzelne Momente klicken, um sie unabhängig voneinander auszuwählen. Die Momente der Zeitleiste können über das Menü der rechten Maustaste auf einem der ausgewählten Elemente bearbeitet werden:

  • Hinzufügen: Mit dieser Option können Sie einen neuen Moment zur Zeitleiste hinzufügen. Er wird direkt nach dem letzten Moment hinzugefügt. Wenn also der letzte Moment in Ihrer Zeitleiste 120 ist, wird der nächste Moment, den Sie hinzufügen, 121 sein. Sie können die Position des Zeitpunkts ändern, indem Sie auf auf den Wert des Zeitpunkts im Editor doppelklicken oder indem Sie mit der rechten Maustaste auf klicken und "Ändern" wählen.
  • Bearbeiten: Öffnet den aktuellen Moment der Zeitleiste zur Bearbeitung.
  • Zeitpunkt ändern: Ändern Sie den Wert für den ausgewählten Timeline-Moment. Dies ist dasselbe wie ein langsamer Doppelklick auf den Timeline-Wert im Editorfenster .
  • Beschreibung ändern: Ändern Sie den beschreibenden Namen für den ausgewählten Timeline-Moment. Dies ist dasselbe wie ein langsamer Doppelklick auf auf die Beschreibung der Zeitleiste im Editorfenster.
  • Duplizieren: Hiermit werden alle ausgewählten Momente der Zeitleiste dupliziert und nacheinander an das Ende der aktuellen Zeitleiste angehängt.
  • Zusammenführen: Wenn Sie zwei oder mehr Momente in einer Zeitleiste auswählen, können Sie diese zusammenführen. Dadurch wird ein einziger Timeline-Moment an der Position des ersten ausgewählten Moments erstellt, in dem alle code und Aktionen gleichzeitig enthalten sind.
  • Entfernen: Damit wird der ausgewählte Moment (oder die ausgewählten Momente) aus der Zeitleiste entfernt.
  • Ausschneiden / Kopieren / Einfügen: Mit diesen Optionen können Sie einen oder mehrere Momente der Zeitleiste ausschneiden, kopieren oder einfügen. Sie können auch hier die Standard-Tastaturkürzel verwenden: / + "X", / + "C", und / + "V".
  • Alles auswählen: Mit dieser Option können Sie alle Momente in einer Zeitleiste auswählen. Die ausgewählten Momente können dann entfernt, zusammengeführt, dupliziert, ausgeschnitten oder kopiert werden.
  • Konvertieren Sie in GML Visual / Code: Timeline-Momente werden mit der Methode erstellt, die Sie bei der Erstellung des Projekts gewählt haben, entweder mit code oder mit GML Visual-Aktionen. Mit dieser Option können Sie unabhängig vom Projekttyp von der einen zur anderen Methode wechseln.
