Der Bild-Editor

ImBild-Editor können Sie Ihre eigenen Sprites erstellen oder solche bearbeiten, die Sie zuvor importiert oder erstellt haben. Wenn Sie bereits andere Bildbearbeitungsprogramme verwendet haben, sollte Ihnen dies vertraut sein, aber es gibt ein paar Dinge zu beachten, wenn Sie den GameMaker-Bildeditor verwenden, die vielleicht nicht sofort ersichtlich sind:

  • Wenn Sie mehrere Rahmen erstellen, können Sie während der Animation auf diese Rahmen zeichnen! Wenn Sie also in der Rahmensteuerungsleiste auf die Wiedergabetaste drücken, dann einen Pinsel auswählen und im Editorfenster zu zeichnen beginnen, werden Sie sehen, dass die Rahmen im Editor animiert werden und dass das, was Sie zeichnen, in diesem Moment dem entsprechenden Rahmen hinzugefügt wird. Sie können mit verschiedenen Bildfrequenzen experimentieren, bis Sie etwas finden, das Ihnen gefällt, aber wenn Sie einmal den Dreh raus haben, wie das funktioniert, werden Sie im Handumdrehen dynamische sprites und Effekte erstellen können.
  • Sie können die Registerkarte Bildeditor aus dem Hauptfenster workspace in ein eigenes Fenster ziehen. Dies kann z. B. bei der Verwendung mehrerer Monitore praktisch sein.
  • Wenn Sie die Kopierfunktionen verwenden, um z. B. eine Auswahl eines Bildes zu kopieren, wird die kopierte Auswahl zu Ihrer Pinselliste hinzugefügt. Das bedeutet, dass Sie einen Teil eines Bildes auswählen und ihn dann in den übrigen Malwerkzeugen zum Zeichnen von Linien, Polygonen oder einfach zum freien Malen verwenden können. Der Pinsel kann auch skaliert und mit einer Farbe überblendet werden. Beachten Sie, dass die Funktion " Einfügen" nicht für die interne Verwendung im Editor gedacht ist, da alles, was Sie kopieren, der Pinselliste hinzugefügt wird und auf diese Weise zum "Einfügen" ausgewählt werden kann. Stattdessen können Sie mit der Funktion "Einfügen" einfach etwas aus einem externen Programm erfassen und in den Bildeditor einfügen.
  • Dank des praktischen Ebenen-Editors können Sie mehrere Ebenen für jede sprite haben. Diese Ebenen können geordnet und sogar in Ordnern abgelegt werden, um die Erstellung komplexerer Bilder zu erleichtern.
  • Sie können im Editor mit den Tasten "1" und "2" schnell zwischen Teilbildern (Frames) wechseln.

Dies sind nur einige der erweiterten Funktionen, die der Bildeditor bietet. Um den vollen Funktionsumfang kennenzulernen, lesen Sie bitte die folgenden Artikel, in denen die Funktionen der einzelnen Teile des Editors beschrieben werden:

Frame-Ansicht

In diesem Teil des Fensters werden die Bilder in der Reihenfolge angezeigt, in der sie animiert werden sollen. Sie können auf die Schaltfläche auf der linken Seite klicken, um weitere Bilder hinzuzufügen - oder das Bildmenü verwenden - und Sie können auch auf eines der Bilder klicken und es ziehen, um seine Position auf der Zeitachse zu ändern. Oben links in der Frame-Ansicht können Sie switch auf Onion Skinning klicken. Das bedeutet, dass Sie in einem Einzelbild einer beliebigen sprite Animation jedes der vorhergehenden oder nachfolgenden Bilder als halbtransparente Ebene unter dem Bildframe sehen können, das Sie gerade bearbeiten. Das Tolle daran ist, dass jedes Bild etwas heller gezeichnet wird als das letzte, so dass Sie sehen können, wie sich die Animation von Bild zu Bild entwickelt hat, während Sie das nächste Bild zeichnen.

Wenn Sie dies aktivieren, werden einige zusätzliche Steuerelemente über den Bildern angezeigt, die Sie anklicken und ziehen können, um die Anzahl der anzuzeigenden vorangehenden Bilder sowie die Anzahl der folgenden Bilder festzulegen.

Sie können bestimmte Eigenschaften für das Zwiebelschälen über das Kontextmenü Ansicht einstellen, das weiter unten auf dieser Seite erläutert wird.

Unter der Schaltfläche "Onion Skin" befindet sich die Schaltfläche " Loop Control" (Schleifensteuerung ) , mit der Sie einstellen können, ob sprite loops (bei Erreichen des letzten Bildes zurück zu Bild 1) oder "Ping-Pong" (bei Erreichen des letzten Bildes rückwärts durch die Bilder gehen). Die Schaltfläche Play am unteren Rand dient zum Starten/Stoppen der Vorschau der Animation.

Sie können auf einen beliebigen Rahmen klicken, um ihn für die Bearbeitung auszuwählen, und ein ausgewählter Rahmen hat in der oberen Ecke eine Schaltfläche Löschen , auf die Sie klicken können, um den Rahmen aus sprite zu entfernen. Sie können auch auf / + auf mehrere Rahmen klicken, um sie gemeinsam auszuwählen; in diesem Fall werden durch Klicken auf die Schaltfläche Löschen alle ausgewählten Rahmen gelöscht. Schließlich können Sie die Standardtastaturkürzel für Ausschneiden, Kopieren und Einfügen verwenden, um ausgewählte Rahmen zu bearbeiten: / + "X", / + "C", und / + "V".

Rahmen-Informationen

Diese Leiste enthält die Bildinformationen. Hier können Sie die Geschwindigkeit einstellen, mit der die Animationsbilder wiedergegeben werden sollen, indem Sie das Eingabefeld auf der linken Seite verwenden. Die Bildgeschwindigkeit kann entweder auf "Bilder pro Sekunde" oder auf "Bilder pro Spielbild" basieren. Die Gesamtzahl der verfügbaren Frames wird rechts zusammen mit dem aktuell ausgewählten Frame angezeigt. Sie können diese Leiste einklappen (ausblenden), indem Sie auf das Symbol auf der linken Seite klicken. In den Voreinstellungen des Bildeditors können Sie einstellen, ob die Leiste immer offen oder immer geschlossen sein soll.

Beachte, dass der Wert, der hier für die Animationsgeschwindigkeit verwendet wird, auch die Art und Weise festlegt, wie sprite animiert wird, wenn es in einem Spiel verwendet wird, entweder als sprite Asset auf einer Ebene oder als sprite, das einer Instanz zugewiesen ist, obwohl du dies unter runtime mit den entsprechenden Funktionen und Variablen ändern kannst.

Leinwand-Steuerelemente

Die Canvas-Steuerelemente regeln verschiedene Aspekte der Darstellung der Hauptzeichenfläche und ihres Inhalts. Die Optionen sind:

  • Canvas-Raster umschalten : Hiermit wird das Canvas-Raster ein- und ausgeschaltet. Dies ist ein Raster, das GameMaker über die Hauptleinwand zieht, um sie in Abschnitte zu unterteilen, und das standardmäßig auf eine Größe von 1 Pixel eingestellt ist. Wenn Sie jedoch auf das Symbol für das Rastermenü klicken, werden die Rasteroptionen geöffnet:
    Mit diesen Optionen können Sie die Farbe und das Alpha des Rasters sowie die Zellenwerte für das Raster entlang der X- und Y-Achse einstellen. Sie haben hier auch die Möglichkeit, das Einrasten des Rasters zu aktivieren oder zu deaktivieren (standardmäßig deaktiviert).
  • Canvas-Zoom-Steuerungen : Diese Schaltflächen steuern die aktuelle Zoomstufe der Leinwand. Sie können ein- oder auszoomen, und wenn Sie auf die Schaltfläche klicken, wird die Leinwand auf 1:1 mit dem bearbeiteten Bild zurückgesetzt. Sie können auch auf die Schaltfläche Window Fit (Fensteranpassung) klicken, um die gesamte Bildleinwand in den aktuellen Editor workspace einzupassen (dabei wird das Bild entsprechend vergrößert/verkleinert, damit es passt). Beachten Sie, dass Sie auch mit / und dem Mausrad zoomen können. Durch Drücken von / + wird die Raumleinwand auf 1:1 mit dem Bild eingestellt.
  • Leinwand teilen : Wenn Sie auf diese Schaltfläche klicken, wird die Leinwand abwechselnd horizontal und vertikal geteilt, so dass sie eine einzige Leinwand bleibt. In der Ansicht "Geteilte Leinwand" hat jede Leinwand ihre eigenen Steuerelemente, so dass Sie z. B. eine Leinwand vergrößern und die andere verkleinern können. Um die Ansicht zu wechseln, klicken Sie auf die Schaltfläche Split Options und wählen Sie die gewünschte Ansicht aus oder klicken Sie einfach auf die Schaltfläche Split Canvas, bis die gewünschte Ansicht angezeigt wird.

Leinwand zeichnen

Dies ist das Hauptbearbeitungsfenster für Ihre Bilder. Hier können Sie jedes der Werkzeuge auf der rechten Seite anwenden, um auf der aktuell ausgewählten Ebene zu malen. Sie können entweder mit der linken oder der rechten Maustaste malen (und jeder Taste eine Farbe aus dem Farbwähler zuweisen) und einen der oben rechts im Editor definierten Pinsel verwenden. Sie können auch / und das Mausrad zum Vergrößern oder Verkleinern verwenden, und durch Drücken von / + wird die Leinwand 1:1 mit dem Bild eingestellt (oder Sie können die Leinwandsteuerungen wie oben beschrieben verwenden). Beachten Sie auch, dass Sie mit der mittleren Maustaste oder + auf der Leinwand herumschwenken können.

Leinwand-Werte

Dieser kleine Balken am unteren Rand zeigt die verschiedenen Werte für die Mausposition und die Breite und Höhe der Auswahlwerkzeuge im Editor an.

Bürsten

Hier können Sie den Pinsel auswählen, mit dem Sie malen möchten. Sie können aber auch ganz einfach einen eigenen Pinsel erstellen, indem Sie auf die Bildebene malen und dann den Teil auswählen, den Sie für den neuen Pinsel verwenden möchten, und ihn mit / + "C" kopieren. Der kopierte Bereich wird dann zu Ihren Pinseln hinzugefügt, und Sie können damit wie mit jedem anderen Pinsel malen, auch mit den verschiedenen verfügbaren Malwerkzeugen, wie z. B. dem Zeichnen einer Linie.

Beachten Sie, dass die Standardmischfarbe für den neuen Pinsel weiß ist, aber wenn Sie die Farbe mit dem Farbwähler ändern und mit dem Pinsel malen, wird er mit dieser neuen Farbe gemischt . Wenn Sie einen Pinsel aus der Liste entfernen möchten, klicken Sie ihn einfach an und drücken Sie die Taste , um ihn zu entfernen.

HINWEIS: Pinsel werden zwischen den Sitzungen nicht beibehalten. Wenn Sie Ihr Projekt schließen, werden alle Pinsel, die Sie erstellt haben, entfernt.

Farbwähler

Oben in diesem Bereich finden Sie eine Reihe von vordefinierten Farbfeldern, aus denen Sie wählen können. Sie können sie der linken und der rechten Maustaste zuweisen, indem Sie mit einer der beiden Tasten darauf klicken (Sie können die Farben auch zwischen den einzelnen Tasten austauschen, indem Sie auf das Pfeilsymbol neben den Feldern klicken). Die ausgewählten Farben werden unten als Farbfelder neben dem Bild der Maus angezeigt, und Sie können darauf klicken, um das Fenster Farbauswahl zu öffnen:

In diesem Fenster können Sie die Farbe mit einem der verfügbaren Schieberegler und Gadgets definieren oder direkt Werte für die ARGB- oder HSV-Werte eingeben.

Wenn Sie bei gedrückter STRG-/CMD-Taste auf auf eines der Standard-Farbfelder klicken, wird das Farbauswahlwerkzeug angezeigt, mit dem Sie die Standardpalette für sprite bearbeiten können. Wenn Sie in diesem Fall in der Farbauswahl auf "Okay" klicken, wird die Palette auf die neue Farbe eingestellt und zusammen mit dem bearbeiteten Asset sprite gespeichert.

Diese Farbpaletten können dann über das Menü Farbpalettenoptionen ausgetauscht und auf verschiedene sprites angewendet werden:

Die einzelne Farbzeile unterhalb der Palette zeigt Ihren Farbverlauf, wobei die zuletzt ausgewählte Farbe ganz links steht. Dieser Abschnitt wird nicht kopiert, wenn Sie die Option "Farben importieren" oder "Farben kopieren" verwenden, da dies nur für die Hauptpalette oben gilt.

Sie können also eine benutzerdefinierte Farbpalette in einem sprite Bild erstellen und diese dann verwenden, um die Palette für jedes andere sprite Bild im Asset-Browser festzulegen, oder Sie können ein sprite Bild öffnen und die Farbpalette aus einem anderen sprite Bild im Asset-Browser importieren. Werkzeuge

Nachfolgend finden Sie eine Liste der Werkzeuge, die standardmäßig in GameMaker verfügbar sind - beachten Sie, dass alle Werkzeuge mit der linken und der rechten Maustaste verwendet werden können, und die meisten von ihnen haben Tastenkombinationen, die mit ihnen verbunden sind (die wichtigsten sind hier zusammen mit dem Werkzeug aufgeführt, aber für eine vollständige Liste sehen Sie bitte die Seite über Tastenkombinationen). Die meisten Werkzeuge haben auch zusätzliche Optionen, die am oberen Rand des Bildbearbeitungsprogramms sichtbar sind, wie unten gezeigt:

Bleistift Dies ist das Bleistift-Werkzeug. Es verwendet den ausgewählten Pinsel, um in der ausgewählten Farbe für die Maustaste zu malen, und Sie können die Größe in der oberen Leiste des Bildeditors einstellen sowie die Glättung (Interpolation) aktivieren/deaktivieren. Beachten Sie, dass Sie auch die Größe und Skalierung jedes Pinsels ändern können, indem Sie + "Z" gedrückt halten, dann auf klicken und die Maus ziehen, um die Proportionen des Pinsels zu ändern, oder alternativ einen der vordefinierten Pinsel oben rechts auswählen.
Radiergummi Dies ist das Radiergummi-Werkzeug. Es verwendet den ausgewählten Pinsel, um einen Bereich zu löschen, wenn Sie mit einer der beiden Maustasten klicken. Wenn Sie dieses Werkzeug auswählen, werden in der oberen Leiste des Editors zusätzliche Werkzeuge angezeigt, mit denen Sie den Pinsel skalieren oder die Glättung(lineare Interpolation) aktivieren/deaktivieren können.
Farbe/Füllung Dies ist das Werkzeug Farbeimer (oder Füllung). Damit wird ein Bereich des Bildes auf der Grundlage der gewählten Alphatoleranz-Einstellung gefüllt, die Sie mit dem Schieberegler in der oberen Leiste des Editors ändern können.
Farbe entfernen Dies ist das Werkzeug zum Entfernen von Farben. Wenn Sie es auswählen, können Sie auf oder auf ein beliebiges Pixel im Bild klicken und alle Pixel mit derselben Farbe werden entfernt (sie werden technisch gesehen durch die Farbe Schwarz mit einem Alpha-Wert von 0 ersetzt). In der oberen Leiste befindet sich ein Schieberegler, mit dem Sie die Farbtoleranz für die Pixel einstellen können. Je niedriger dieser Wert ist, desto genauer muss eine Farbe mit der angeklickten Farbe übereinstimmen, während ein höherer Wert alle Farben einschließt, die der angeklickten Farbe ähnlich sind. Wenn Sie zuvor mit einem der Auswahlwerkzeuge eine Auswahl getroffen haben, wirkt sich nur das Klicken auf Pixel innerhalb der Auswahl aus, und nur die Pixel innerhalb der Auswahl werden entfernt.
Farbe Ersetzen Dies ist das Werkzeug zum Ersetzen von Farben. Wenn Sie es auswählen, können Sie auf oder auf ein beliebiges Pixel im Bild klicken und alle Pixel, die dieselbe Farbe haben, werden durch die Farbe ersetzt, die der verwendeten Maustaste zugeordnet ist. In der oberen Leiste befindet sich ein Schieberegler, mit dem Sie die Toleranz für die Pixelfarbe einstellen können - je niedriger dieser Wert ist, desto genauer muss eine Farbe mit der angeklickten Farbe übereinstimmen, während ein höherer Wert die Farben einschließt, die der angeklickten Farbe ähnlich sind. Wenn Sie zuvor mit einem der Auswahlwerkzeuge eine Auswahl getroffen haben, wirkt sich nur das Klicken auf Pixel innerhalb der Auswahl aus, und nur die Pixel innerhalb der Auswahl sind von der Farbänderung betroffen.
Leitung Mit diesem Werkzeug können Sie eine Linie zwischen zwei Punkten zeichnen. Klicken Sie einmal mit einer der beiden Schaltflächen, um den ersten Punkt zu erstellen, und klicken Sie dann an einer anderen Stelle erneut, um einen weiteren Punkt zu erstellen und die beiden Punkte zu verbinden. Die Linie wird mit dem ausgewählten Pinsel gezeichnet, um den Umriss zu erstellen, und Sie können einige der Pinsel-Eigenschaften in der oberen Leiste des Editors bearbeiten (Größe, lineare Interpolation usw.).
Rechteck Mit diesem Werkzeug können Sie entweder ein ausgefülltes oder ein umrandetes Rechteck zeichnen. Wenn Sie in die obere linke Ecke des Symbols klicken, wird es umrandet, wenn Sie in die untere rechte Ecke klicken, wird es ausgefüllt. Wenn Sie beides wünschen, können Sie diese Option in der oberen Leiste des Bildbearbeitungsprogramms auswählen (oder + verwenden, um erst das eine und dann das andere auf dem Werkzeugkasten-Symbol auszuwählen) und die Größe und Glättung des verwendeten Pinsels einstellen. Das Rechteck wird dann mit dem ausgewählten Pinsel gezeichnet. Beachten Sie, dass Sie während des Zeichnens gedrückt halten und klicken können, um von der Mitte der Mausposition aus zu zeichnen.
Ellipse Mit diesem Werkzeug können Sie entweder eine gefüllte oder eine umrissene Ellipse zeichnen. Wenn Sie in die obere linke Ecke des Symbols klicken, wird die Ellipse umrandet, wenn Sie in die untere rechte Ecke klicken, wird sie ausgefüllt. Wenn Sie beides möchten, können Sie diese Option in der oberen Leiste des Bildbearbeitungsprogramms auswählen (oder + verwenden, um erst die eine und dann die andere Option auf dem Werkzeugkasten-Symbol auszuwählen) und die Größe und Glättung des verwendeten Pinsels einstellen. Die Ellipse wird mit dem ausgewählten Pinsel gezeichnet, um die Kontur zu erstellen. Beachten Sie, dass Sie während des Zeichnens gedrückt halten und klicken können, um von der Mitte der Mausposition aus zu zeichnen.
Polygon Mit diesem Werkzeug können Sie entweder ein ausgefülltes oder ein umrandetes Polygon zeichnen. Wenn Sie oben links auf das Symbol klicken, wird es umrandet, wenn Sie unten rechts klicken, wird es ausgefüllt. Wenn Sie beides möchten, können Sie diese Option in der oberen Leiste des Bildeditors auswählen (oder + verwenden, um das eine und dann das andere auf dem Werkzeugkasten-Symbol auszuwählen) und die Größe und Glättung des verwendeten Pinsels einstellen. Klicken Sie auf oder , um mit dem ausgewählten Pinsel Punkte zum Polygon hinzuzufügen. Wenn Sie mit dem Ergebnis zufrieden sind, wählen Sie ein anderes Werkzeug, um die Änderungen zu übernehmen und das Bild zu erstellen. Das Polygon wird mit dem ausgewählten Pinsel gezeichnet. Sie können jederzeit drücken, um die Definition des Polygons abzubrechen.
Spline Dies ist das Spline-Werkzeug. Es verwendet den ausgewählten Pinsel, um einen Spline zu erstellen. Sie klicken und halten eine der beiden Maustasten gedrückt, um einen Punkt zu setzen, und ziehen dann die Maus, um den Spline zu erzeugen; lassen Sie die Maus los und wiederholen Sie den Vorgang, um weitere Punkte und Kurven hinzuzufügen. Der Spline wird mit dem ausgewählten Pinsel gezeichnet, der in der oberen Leiste des Editors bearbeitet werden kann, um Größe und Glättung einzustellen. In der oberen Leiste gibt es auch eine Einstellung für Iterationen , die festlegt, wie glatt der resultierende Spline ist (niedrigere Werte sind gröber, höhere glatter, und der Standardwert ist 10).
Text
Dies ist das Texteingabewerkzeug. Es erstellt eine neue Textebene, in die Sie alles eingeben können, was Sie brauchen. Sie können die zu verwendende Schriftart und die Formatierung in der oberen Leiste des Bildeditors festlegen.
Augentropfer Mit der Pipette können Sie eine Farbe aus dem Bild, das Sie gerade bearbeiten, auswählen.
Rechteckige Auswahl Mit diesem Werkzeug können Sie eine rechteckige Auswahl erstellen. Sie können eine Auswahl über das Menü Bild oder durch Drücken von aufheben.
Auswahl der Farbe Mit diesem Werkzeug können Sie einen ausgewählten Bereich mit einem beliebigen Pinsel, einschließlich benutzerdefinierter Pinsel, "malen". Es gelten die gleichen Tastenkombinationen wie für das Werkzeug "Stift". Sie können eine Auswahl über das Menü Bild oder durch Drücken von
Zauberstab Das Auswahlwerkzeug des Zauberstabs wählt Bereiche gleicher Farbe auf der Grundlage des in der oberen Leiste des Editors eingestellten Toleranzwerts aus. Wenn Sie "Nur Farbe" ankreuzen, wird der Alphawert nicht in Betracht gezogen. Sie können eine Auswahl über das Menü "Bild" oder durch Drücken von aufheben.
Drehen Sie Dies ist das Werkzeug zum Drehen. Wenn Sie einen Bereich mit einem der Auswahlwerkzeuge ausgewählt haben, können Sie den ausgewählten Bereich um einen beliebigen Wert zwischen 0° und 360° drehen. Wenn Sie keinen Bereich des Bildes ausgewählt haben, können Sie damit den aktuellen Pinsel drehen. Zum Drehen halten Sie einfach die linke Maustaste gedrückt und ziehen nach links oder rechts oder stellen den Wert direkt in der oberen Leiste ein. Um die Änderungen zu übernehmen, können Sie entweder drücken oder ein anderes Werkzeug auswählen. Beachten Sie, dass durch die Annahme der Änderung das Ausgangsmaterial nicht verändert wird, sondern eine neue Pinselressource erstellt wird, die Sie mit dem ausgewählten Bild oder einem um den von Ihnen festgelegten Wert gedrehten Pinsel verwenden können. Beachten Sie auch, dass Sie mit dem Tastaturkürzel "Z" eine beliebige Auswahl oder einen Pinsel um 90° drehen können und dass diese Option keinen neuen Pinsel erstellt, sondern nur die Art und Weise, wie der aktuelle Pinsel gezeichnet wird, beeinflusst.
Spiegel Dies ist das Spiegelungswerkzeug. Wenn Sie diese Option auswählen, wird der aktuelle Pinsel entlang der vertikalen Achse gespiegelt. Mit dieser Option wird kein neuer Pinsel erstellt, sondern nur die Art und Weise, wie der aktuelle Pinsel gezeichnet wird, beeinflusst.
Flip Dies ist das Spiegelungswerkzeug. Wenn Sie es auswählen, wird der aktuelle Pinsel entlang der horizontalen Achse gespiegelt. Mit dieser Option wird kein neuer Pinsel erstellt, sie wirkt sich nur darauf aus, wie der aktuelle Pinsel gezeichnet wird.
Pan Dies ist das Schwenkwerkzeug. Wenn Sie es auswählen, können Sie auf klicken und die Maus ziehen, um den Ebeneninhalt innerhalb der Ebene zu verschieben. Beachten Sie, dass alle Pixel außerhalb der Zeichenfläche nicht sichtbar sind und das Bild auf die Größe der Zeichenfläche beschnitten wird, wenn Sie das Werkzeug erneut wechseln. Beachten Sie auch, dass, wenn Sie eine Auswahl getroffen haben, die ausgewählten Pixel und nicht der Inhalt der Ebene verschoben werden.

Ebenen-Editor

Sie können im Bildeditor Ebenen für verschiedene Teile Ihres Bildes erstellen, und diese Ebenen können mit den verschiedenen Pinseln und Werkzeugen bearbeitet und gezeichnet werden. Um eine neue Ebene zu erstellen, klicken Sie einfach auf das Symbol , und um sie wieder zu entfernen, auf das Symbol . Ebenen können auch geordnet werden, indem Sie auf auf eine Ebene klicken und dann die Ebene nach oben oder unten an die gewünschte Position ziehen. Sie können die Ebenen auch weiter organisieren, indem Sie mit der Schaltfläche Ordner erstellen und dann Ebenen in diese Ordner ziehen. Wenn Sie auf auf eine Ebene doppelklicken (oder verwenden und Ebeneneigenschaften bearbeiten wählen), erhalten Sie Zugriff auf das Fenster Ebeneneigenschaften:

Hier können Sie der Ebene einen Namen geben und den Mischmodus festlegen (die Art und Weise, wie sie mit den darunter liegenden Ebenen "gemischt" werden soll), wobei Sie zwischen normalen, additiven, subtraktiven oder multiplen Mischungen wählen können. Schließlich können Sie die Deckkraft (Transparenz) der Ebene mit einem Schieberegler einstellen. Beachten Sie jedoch, dass sich dies mit dem tatsächlichen Alphawert der einzelnen Pixel kumuliert - wenn Sie beispielsweise Pixel mit einer Deckkraft von 50 % haben und die Deckkraft der Ebene auf 50 % einstellen, werden die endgültigen Pixel mit einer Deckkraft von 25 % gezeichnet.

Neben den Werkzeugen im Bildbearbeitungsprogramm selbst stehen Ihnen über die Menüleiste am oberen Rand der IDE eine Reihe von zusätzlichen Werkzeugen zur Verfügung:

Diese Menüs werden in den folgenden Abschnitten erläutert:

Bild

Das Menü "Bild" enthält die folgenden Optionen (von denen einige je nach aktuellem Fokus des Editors verfügbar sein können oder nicht):

  • Rahmen / Auswahl ausschneiden - Schneidet den Rahmen oder die Auswahl aus dem Bild aus, je nach Fokus des Editors. Wenn Sie eine Auswahl ausschneiden, wird sie zu den Pinseln hinzugefügt.
  • Rahmen/Auswahl kopieren - Kopieren Sie den Rahmen oder die Auswahl aus dem Bild, je nachdem, ob der Editor im Fokus ist. Wenn Sie eine Auswahl kopieren, wird diese zu den Pinseln hinzugefügt.
  • Rahmen / Auswahl einfügen - Fügt den Rahmen aus der Zwischenablage ein oder fügt die Auswahl aus der Zwischenablage in das Bild ein, je nach aktuellem Fokus des IDE. Beim Einfügen in ein Bild sollten Sie beachten, dass diese Option nur für die Verwendung mit Daten aus der Zwischenablage von externen Programmen vorgesehen ist. Wenn Sie im Bildeditor Daten aus einem Bild ausschneiden oder in die Zwischenablage kopieren, werden diese zu den Pinseln oben rechts im Editor hinzugefügt, und Sie sollten einen Pinsel auswählen, um mit der ausgeschnittenen/kopierten Auswahl zu zeichnen, anstatt zu versuchen, sie in das Bild einzufügen. Beachten Sie auch, dass einige Programme Probleme beim Einfügen von Transparenzen aus der Zwischenablage in den GameMaker-Bildeditor haben, da die Zwischenablage ein anderes Format verwendet.
  • Alles auswählen - Das gesamte Bild auswählen.
  • Auswahl aufheben - Hebt die Auswahl des Bildes auf.
  • Auswahl umkehren - Kehrt die aktuelle Auswahl um.
  • Rahmen hinzufügen - Fügt einen neuen Rahmen zu sprite hinzu.
  • Rahmen einfügen - Fügt einen neuen Rahmen nach dem aktuell angezeigten ein.
  • Ausgewählte Bilder löschen - Entfernen Sie das ausgewählte Bild aus sprite (Sie können / + und/oder + verwenden, um mehrere Bilder auszuwählen).
  • Bild(er) importieren - Laden Sie ein Bild, das als Rahmen verwendet werden soll, von der Festplatte. Beachten Sie, dass Sie mehrere Bilder auf einmal laden können, indem Sie im Explorer-Fenster mehrere einzelne Dateien zum Importieren auswählen. Nachdem Sie Ihre Datei(en) ausgewählt haben, wird das folgende Importfenster angezeigt:
    Hier können Sie auswählen, wie das importierte Bild zur aktuellen sprite hinzugefügt werden soll. Sie können die Größe der gesamten Zeichenfläche ändern, sie zuschneiden oder auffüllen und auch die "Ankerposition" für die Platzierung des neuen Bildes/der neuen Bilder festlegen.
  • In Frames konvertieren - Mit dieser Option wird ein einzelnes Bild in eine Reihe von Einzelbildern umgewandelt. Sie müssen zunächst ein Einzelbild mit allen Teilen der Animation erstellen (oder ein einzelnes Streifenbild importieren), wie das unten gezeigte:
    Da es sich um ein Einzelbild handelt und wir es in Einzelbilder umwandeln wollen, können wir diese Option verwenden, die dann das folgende Fenster öffnet:
    Hier können Sie einstellen, wie das Einzelbild aufgeteilt werden soll, um die Einzelbilder zu erstellen, indem Sie die Anzahl der Einzelbilder, die Breite und Höhe der Einzelbilder und eventuelle Versatzwerte festlegen.
  • Bildstreifen importieren - Diese Option öffnet einen Datei-Explorer und fordert Sie auf, nach einer einzelnen Bildstreifendatei zu suchen, aus der Sie Rahmen erstellen möchten. Sobald Sie ein Bild ausgewählt haben, öffnet sich das Fenster "Streifen importieren", genau wie bei der oben gezeigten Option " In Rahmen konvertieren ".
  • In PNG exportieren - Wenn Sie diese Option wählen, wird sprite als ein einziges PNG Bild exportiert. Wenn sprite mehr als ein Bild enthält, wird ein "Streifen" sprite mit allen Bildern nacheinander erstellt, und dem Dateinamen wird automatisch der Zusatz "_stripX" hinzugefügt, wobei "X" die Anzahl der exportierten Bilder angibt. Zum Beispiel wird ein Bild mit fünf Bildern sprite als "sprite_name_strip5.png" exportiert.
  • Resize All Frames - Wenn Sie diese Option wählen, öffnet sich das folgende Fenster, in dem Sie festlegen können, wie Sie die Größe aller Animationsframes, aus denen das Sprite besteht, ändern möchten:
    Sie können wählen, ob Sie das Bild skalieren oder die Größe der Leinwand ändern möchten. Wenn Sie sich für die Skalierung des Bildes entscheiden, wählen Sie einfach die neue Breite und/oder Höhe für jedes Einzelbild aus und legen fest, ob das Seitenverhältnis beibehalten werden soll und ob die Skalierung in Pixel oder Prozent erfolgen soll. Beachten Sie, dass Sie bei der Skalierung die lineare Interpolation ein- oder ausschalten können. Wenn Sie die lineare Interpolation einschalten, verschwimmen die Pixel bei der Skalierung des Bildes (bei hochauflösender Kunst sollten Sie diese Option einschalten, bei Pixelkunst sollten Sie sie ausschalten).

    Die Option zur Größenänderung der Leinwand verändert nicht die Pixel im Bild, sondern vergrößert oder verkleinert die Leinwand, auf der die Bilder gezeichnet werden. Mit den Pfeilsymbolen können Sie festlegen, wo der "Anker" für die Größenänderung liegen soll, und dann die neue Größe (in Pixeln oder als Prozentsatz) sowie die Beibehaltung des Seitenverhältnisses der ursprünglichen Rahmen festlegen. Beachten Sie, dass die Pixel beschnitten werden, wenn Sie den Rahmen kleiner als die ursprünglichen sprite Bilder machen.
  • Alle Frames auf Auswahl beschneiden - Diese Option ist nur verfügbar, wenn Sie einen Bereich des Bildes in der Hauptzeichenfläche ausgewählt haben. Wenn Sie diese Option wählen, wird das zu bearbeitende Bild (und alle anderen Bilder in der Animation) auf die Grenzen der Auswahl beschnitten.
  • Alle Rahmen automatisch zuschneiden - Mit dieser Option können Sie alle Null-Alpha-Pixel an den Rändern von sprite zuschneiden. Dabei werden alle Bildrahmen innerhalb von sprite berücksichtigt, so dass jede Seite auf den Rand des Rahmens mit dem am weitesten entfernten Nicht-Null-Alpha-Pixel zugeschnitten wird.
  • Frames umkehren - Wenn Sie diese Option auswählen, wird die Reihenfolge der Frames auf sprite einfach umgekehrt.
  • Spiegeln - Mit dieser Option können Sie entweder das aktuelle Bild oder alle Bilder, aus denen sprite besteht, an der vertikalen Achse spiegeln.
  • Spiegeln - Mit dieser Option können Sie entweder das aktuelle Bild oder alle Bilder, aus denen die sprite besteht, entlang der horizontalen Achse spiegeln.
  • Alle Frames drehen (im Uhrzeigersinn 90°) - Mit dieser Option werden alle Frames der Animation um 90° im Uhrzeigersinn gedreht.
  • Alle Bilder drehen (gegen den Uhrzeigersinn 90°) - Mit dieser Option werden alle Bilder der Animation um 90° gegen den Uhrzeigersinn gedreht.

Siehe

Das Menü Ansicht bietet die folgenden Optionen:

  • 1:1 - Skalieren Sie das Bild auf 1:1 mit der Auflösung (d.h. die tatsächliche Größe) und zentrieren Sie es im Bildbearbeitungsfenster.
  • Bildschirm anpassen - Skaliert das Bild so, dass es das Bildbearbeitungsfenster ausfüllt.
  • Raster anzeigen - Schaltet das Pixelraster ein und aus. Dies ist dasselbe wie die Schaltfläche "Raster" in den Canvas-Steuerungen (weiter oben auf dieser Seite erklärt).
  • Zwiebelhaut-Einstellungen - Ändern Sie die Einstellungen der Zwiebelhaut. Dies öffnet das folgende Fenster, in dem Sie die von der Schaltfläche "Zwiebelhaut" angezeigten Rahmen sowie die Deckkraft und Farbe der umhüllten Rahmen einstellen können.

Auswirkungen

Im Menü "Effekte" können Sie Ihre eigenen benutzerdefinierten Effekt-Plug-ins hinzufügen, die standardmäßig die folgenden Effekte enthalten:

  • Weichzeichnen - Wendet einen Weichzeichnungseffekt auf das Bild an. Sie können die Intensität und die Art des Weichzeichners (Rechteck oder Gauß) festlegen und bestimmen, ob er nur entlang der horizontalen oder vertikalen Achse oder entlang beider Achsen weichgezeichnet werden soll. Sie können dann wählen, ob der Effekt nur auf die aktuelle Ebene, die sichtbaren Ebenen oder auf alle Ebenen angewendet werden soll.
  • Graustufen - Entsättigt das Bild um einen bestimmten Wert. Sie können wählen, ob dies nur auf die aktuelle Ebene, die sichtbaren Ebenen oder alle Ebenen angewendet werden soll.
