Der Animationskurven-Editor

Eine Animationskurve ist ein Asset, das eine oder mehrere Kurven enthält, die darstellen, wie sich ein Wert im Laufe der Zeit verändert, und kann lineare Interpolation, glatte Interpolation oder Bezier-Interpolation verwenden, um zwischen verschiedenen Punkten in der Kurve zu wechseln. Die Werte, die Sie auf der vertikalen Achse einstellen, können zwischen zwei beliebigen Werten liegen (standardmäßig zwischen -1 und 1), und die Dauer entlang der horizontalen Achse ist immer von 0 bis 1 normalisiert, was es einfach macht verschiedene Zeitbereiche mit Hilfe von Multiplikatoren in Ihrem Spiel code anzusteuern oder sie zu strecken, damit sie in einen Track im Sequenzeditor passen.

Animationskurven bestehen aus "Kanälen", und jeder Kanal kann seine eigenen Kurveneinstellungen haben, so dass Sie z. B. eine räumliche Position mit zwei Kanälen für die x/y-Position oder einen Farbverlauf mit 4 Kanälen für ein Farbformat beschreiben können. ein Farbformat darzustellen. Animationskurven-Assets können bei der Arbeit mit Sequences verwendet werden und sind auch über code zugänglich, was sie zu einem leistungsstarken Werkzeug bei der Erstellung Ihrer Spiele macht.

Wenn Sie zum ersten Mal ein Animationskurven-Asset erstellen, öffnet sich das Editorfenster und zeigt Ihnen die folgenden Abschnitte:

Name der Kurve

Hier können Sie Ihrer Animationskurve einen eindeutigen Namen geben(nur alphanumerische Zeichen und das Unterstrichsymbol "_"). Dieser Name wird verwendet, um auf die Kurve und alle ihre Kanäle zu verweisen wenn sie über code oder in sequence aufgerufen werden.

Kurvenbibliothek

Die Schaltfläche öffnet das Animationskurven-Bibliothek Fenster, in dem Sie den Kurventyp der Animationskurve ändern und eine Voreinstellung auf den ausgewählten Kanal anwenden können.

Informationen zu Kurventypen und Voreinstellungen finden Sie im Abschnitt über die über Kurventypen und Voreinstellungen. Beachten Sie, dass beim Öffnen des Fensters über die Schaltfläche standardmäßig der Anwendungsmodus "Überschreiben " ausgewählt ist.

Kanal-Liste

In der Kanalliste können Sie die verschiedenen Kanäle hinzufügen und benennen, aus denen Ihr Animationskurven-Asset bestehen wird. Wenn Sie zum ersten Mal eine Animationskurve erstellen, wird standardmäßig ein Kanal für Sie hinzugefügt, und Sie können weitere Kanäle hinzufügen, indem Sie auf die Schaltfläche links neben dem Kurvennamen klicken. Jeder Kanal muss einen eindeutigen Namen haben ( nur alphanumerische Zeichen und das Unterstrichsymbol "_" verwenden), Sie können auch die Farbe für den Kanal festlegen, indem Sie auf auf das Farbfeld auf der linken Seite doppelklicken, wodurch sich der Farbwähler öffnet und Sie eine neue Farbe auswählen können. Um den Kanal umzubenennen, können Sie einen langsamen Doppelklick auf machen und umbenennen, und Sie können auch die rechte Maustaste benutzen, um ein Menü zu öffnen, in dem Sie wählen können, ob Sie den Kanal umbenennen, die Farbe ändern oder löschen wollen.

Kanäle können durch Klicken auf den Pfeil neben dem Namen erweitert werden, um die einzelnen Punkte auf der Kanalkurve anzuzeigen, und diese Punkte können manuell bearbeitet werden, indem man auf den aktuellen Wert klickt und den neuen Wert eintippt.

Sie können die Sichtbarkeit des Kanals durch einen Klick auf das "Auge"-Symbol umschalten oder den Kanal durch einen Klick auf das "Mülleimer"-Symbol löschen.

Kanal Bereich

Mit der Schaltfläche Kanalbereich können Sie den visuellen vertikalen Bereich für den Kanal-Editor einstellen. Standardmäßig ist dieser Bereich von -1 bis 1 eingestellt, aber ein Klick auf diese Schaltfläche öffnet das folgende Fenster:

Wenn Sie diese Werte ändern, ändert sich die Art und Weise, wie die Daten im Bereich des Kanal-Editors dargestellt werden. höhere/niedrigere Werte entlang der Kanalkurve als nur -1 bis 1 zu haben. Beachten Sie, dass es sich hierbei um eine rein visuelle Änderung handelt und die Werte für den Kanal nicht geklemmt werden.

Kanal-Editor

Der Kanal-Editor zeigt eine visuelle Darstellung der verschiedenen Kanäle, die in der Animationskurve verwendet werden. Standardmäßig werden hier alle Kanäle angezeigt, aber Sie können nur den aktuell in der Kanalliste ausgewählten Kanal bearbeiten, der im Editor hervorgehoben wird. im Editor hervorgehoben wird. Sie können die Position eines jeden Punktes entlang der Kanalkurve ändern, indem Sie auf klicken und sie im Editor verschieben. Beachten Sie jedoch, dass der erste und der letzte Punkt nur entlang der vertikalen Achse geändert werden können und ihr horizontaler Wert immer 0 bzw. 1 ist.

Sie können Punkte direkt zur Kanalkurve hinzufügen, indem Sie die Maus in die Nähe der Hauptkanallinie bewegen und dann auf klicken, wenn sich der Cursor in das Werkzeug "Punkt hinzufügen" Sie können diese Punkte dann bearbeiten, indem Sie auf klicken und ziehen oder indem Sie ihre Werte in den erweiterten Kanaloptionen in der Kanalliste ändern:

Sie können mehrere Punkte auf einmal bearbeiten, indem Sie auf klicken und innerhalb des Editors ziehen, um alle zu bearbeitenden Punkte auszuwählen, oder indem Sie / + verwenden, um Punkte einzeln zur Auswahl hinzuzufügen. Wenn Sie die Maustaste loslassen und dann erneut auf einen der Punkte in der Auswahl klicken und ziehen, werden sie alle zusammen verschoben:

Sie können mit der rechten Maustaste auf eine beliebige Stelle im Kanal-Editor klicken, um das Fenster Animationskurven-Bibliothek zu öffnen (nur, wenn der ausgewählte Kurventyp Bezier ist). Das Anwenden einer Voreinstellung auf diese Weise wirkt sich auf den gesamten Kanal aus. Um nur einen Teil des Kanals zu ändern, können Sie mit der rechten Maustaste auf eine einzelne Linie klicken, um eine Vorgabe auf sie anzuwenden, oder mehrere Linien auswählen, indem Sie auf klicken und ziehen und dann mit der rechten Maustaste auf die Auswahl klicken, um eine Vorgabe auf die ausgewählten Linien anzuwenden.

Beachten Sie, dass beim Öffnen des Fensters "Animationskurvenbibliothek" über den Kanal-Editor standardmäßig der Anwendungsmodus "Zwischen " ausgewählt ist.
