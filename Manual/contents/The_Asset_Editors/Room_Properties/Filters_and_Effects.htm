Filter und Effekte

Filter/Effekt-Ebenen

Im Raum-Editor können Sie eine neue Filter-/Effekt-Ebene hinzufügen, indem Sie auf die Schaltfläche klicken, die eine neue Ebene hinzufügt und deren Eigenschaften unten öffnet:

Eine Filter/Effekt-Ebene (oder FX-Ebene) wird verwendet, um einen visuellen Filter oder Effekt auf einige Ebenen anzuwenden. Sie können einen Effekt aus der Dropdown-Liste "Effekttyp" auswählen, dessen Parameter im Fenster "Ebeneneigenschaften" angezeigt werden, so dass Sie sein Aussehen und Verhalten ändern können.

Wenn Sie z. B. den Effekt "Bildschirmverwacklung" auswählen, können Sie die Stärke und die Geschwindigkeit der Verwacklung ändern und eine sprite als Rauschtextur anwenden:

Wenn Sie eine Effekt-Ebene hinzufügen, wird der ausgewählte Effekt auf alle anderen Ebenen unterhalb der Effekt-Ebene angewendet, wie in dieser Abbildung gezeigt:

Das bedeutet, dass auf alle "betroffenen" Ebenen der ausgewählte Filter (hier "Entsättigen") angewendet wird, da sie sich unterhalb einer Effekt-Ebene befinden. Alle Ebenen oberhalb der Effekt-Ebene sind nicht betroffen. Eine Ebene kann von mehreren Effektebenen betroffen sein, die sich in der Ebenenreihenfolge über ihr befinden.

Zur Erstellung von Filtern/Effekten, die nur auf eine Ebene angewendet werden, siehe: Ein-Ebenen-FX.

HINWEIS Wenn Sie die Tiefe einer Ebene oder einer Instanz auf runtime ändern, können Sie die darauf angewendeten Effektebenen ändern, indem Sie entweder die Tiefe der Ebene höher als die Tiefe einer Effektebene einstellen (d. h. sie liegt unter der Effektebene und wird beeinflusst) oder die Tiefe der Ebene niedriger als die Tiefe einer Effektebene einstellen (d. h. sie liegt über der Effektebene und wird nicht beeinflusst).

Filter vs. Effekt

Ein Filter verwendet nur Shader und kann im Raumeditor in der Vorschau angezeigt werden.

Ein Effekt hat zusätzlich zu den Shadern auch interne code für die Verwaltung seiner selbst. Aus diesem Grund können sie im Raumeditor nicht in der Vorschau angezeigt werden und sind nur unter runtime (im Spiel) funktionsfähig.

Ein Beispiel: "Einfärben" ist ein Filter, "Windverwehte Partikel" ist ein Effekt.

Effekt-Typen

Alle Filter und Effekte in der Dropdown-Liste "Effekttyp" sind auf dieser Seite dokumentiert: FX-Typen & Parameter.

Einschichtige FX

Sie können zwar eine FX-Ebene erstellen, um einen Filter auf mehrere Ebenen anzuwenden, Sie können aber auch einen Filter nur einer bestimmten Ebene zuweisen, indem Sie den Inspektor verwenden. Wenn Sie einen Inspektor öffnen und eine Raumebene auswählen, können Sie einen Filter/Effekt auf diese Ebene anwenden. Wenn es sich bei dieser Ebene um eine Nicht-FX-Ebene handelt (z. B. Instanzebene, Asset-Ebene usw.), wird der Effekt nur auf diese Ebene und nicht auf andere Ebenen darunter angewendet.

Sie können auch auf das Symbol klicken, um die Effekte für die ausgewählte Ebene zu aktivieren/deaktivieren.

Laufzeit-Nutzung

Mit den Filter/Effekt-Funktionen runtime können Sie Effekte auf GML hinzufügen und steuern.

Lesen Sie Filter- und Effektfunktionen für weitere Informationen.
