FX-Typen und Parameter

Auf dieser Seite sind alle Filter-/Effekttypen aufgelistet, die in GameMaker zur Verfügung stehen, zusammen mit den folgenden Informationen zu jedem Effekt:

  • Interner Name (z.B. "_filter_tintfilter"), der an fx_create() übergeben werden kann, um einen FX zur Laufzeit zu erstellen
  • Beschreibung
  • Beispielbild
  • IDE-Parameter, die über die Ebeneneigenschaften bearbeitet werden können
  • Laufzeit-Parameter, die zum Ändern von Effekten auf runtime verwendet werden können, unter Verwendung der Filter- und Effektfunktionen
    • Die Runtime Parameter geben auch ihre Typen an, so dass Sie wissen, welche Art von Daten Sie auf welchen Parameter anwenden müssen.
