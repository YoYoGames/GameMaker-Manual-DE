Code-Schnipsel

Ein sehr praktisches Werkzeug, das Ihnen bei der Bearbeitung Ihrer code zur Verfügung steht, ist die Verwendung von Code-Snippets. Wenn Sie den Editor script verwenden, können Sie F4 drücken, um das Pop-up-Fenster code zu öffnen, in dem Sie eine der häufig verwendeten Methoden code aus einer Liste auswählen können:

In diesem Fenster können Sie das zu verwendende Snippet mit der Maus auswählen oder den zugehörigen Hotkey auf der rechten Seite drücken. Dadurch wird das Snippet zur Bearbeitung zu Ihrem code hinzugefügt.

Sie können auch Ihre eigenen code Schnipsel definieren, wenn Sie das möchten. Dazu müssen Sie eine Datei mit dem Namen "snippets.txt" in einem der folgenden Verzeichnisse erstellen:

  • Fenster: %ProgramData%\GameMaker\User\
  • macOS: /Users/Shared/GameMaker/User/

Dieser Ordner wird bei einem Update von GameMaker nicht verändert, so dass Ihre bearbeitete Datei erhalten bleibt (aber alle Änderungen an der Snippets-Basisdatei im Installationsverzeichnis werden rückgängig gemacht).

Sobald die Datei im Benutzerverzeichnis erstellt ist, können Sie sie mit einem beliebigen Texteditor bearbeiten, indem Sie diese Regeln befolgen:

  • Jedes Snippet muss in einer eigenen Zeile stehen (es sollten keine Leerzeilen vorhanden sein)
  • Jedes Snippet muss mit dem zu verwendenden Hotkey beginnen, gefolgt von "-" und dann dem Namen des Snippets (der im Menü angezeigt wird), gefolgt von einem Doppelpunkt ":", z. B:

I - Instanz erstellen:

Nach dem Doppelpunkt fügen Sie den folgenden Codeschnipsel ein

I - Instanz erstellen:instance_create_layer(x, y, |layer|, object);

Die code, die Sie hinzufügen, müssen auch ein bestimmtes Format haben, bei dem:

  • Alle code befinden sich in einer einzigen Zeile.
  • Sie verwenden "$" für einen Zeilenumbruch
    • HINWEIS Wenn Sie mehrere Zeilenumbrüche erstellen, muss zwischen jedem "$" Zeichen ein Leerzeichen stehen. Sie würden zum Beispiel $ $ $ schreiben, um drei Zeilenumbrüche einzufügen.
  • Sie setzen den Text, der hervorgehoben werden soll, zwischen die Symbole "|". Der Code Editor wählt diesen Text automatisch zur Bearbeitung aus, wenn das Snippet hinzugefügt wird.

Sehen Sie sich die Beispiele an, die bereits in der Basisdatei der Snippets enthalten sind, um zu sehen, wie sie nach den oben genannten Regeln aufgebaut sind. Sie finden die Basisdatei im GameMaker-Installationsverzeichnis:

  • Windows: GameMaker\TextEditor\snippets.txt
  • macOS: /Applications/GameMaker.app/Contents/MonoBundle/TextEditor/snippets.txt
