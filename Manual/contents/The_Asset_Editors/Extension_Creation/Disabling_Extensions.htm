Wie deaktiviere ich Erweiterungen?

GameMaker hat Erweiterungen für einige Ziele platforms, z.B. Steamworks und GDK.

Wenn Sie beide Erweiterungen verwenden, sollten Sie die GDK-Erweiterung deaktivieren, wenn Sie für Steam bauen, und andersherum.

Dieses Problem ist nicht auf diese beiden Erweiterungen beschränkt, sondern gilt für jede Erweiterung, die für ein bestimmtes Ziel gedacht ist.

Selektives Deaktivieren von Erweiterungen

Richten Sie zunächst für jedes Ihrer Ziele eine Konfiguration ein platforms, falls Sie dies nicht bereits getan haben.

Dieses Beispiel verwendet nur Steamworks und GDK.

Wählen Sie die Konfiguration für ein Ziel platform. Nehmen wir an, Sie haben die GDK-Konfiguration ausgewählt.

Öffnen Sie eine Erweiterung, die nicht für GDK verwendet werden soll. In diesem Beispiel öffnen Sie die Steamworks-Erweiterung.

Deaktivieren Sie alle Ziele unter Kopieren nach. Dadurch wird sichergestellt, dass die Steamworks-Erweiterung bei Verwendung der GDK-Konfiguration nicht ausgeführt wird.

Wechseln Sie nun unter switch zu Ihrer Steamworks-Konfiguration. Öffnen Sie die GDK-Erweiterung, und deaktivieren Sie alle ihre Ziele.

Erledigt!

Wenn Sie nun für GDK bauen, verwenden Sie die GDK-Konfiguration; wenn Sie für Steam bauen, verwenden Sie die Steamworks-Konfiguration usw.

Wenn Sie Konfigurationen auf diese Weise verwenden, können Sie steuern, welche Erweiterungen für die Erstellung welcher platforms verwendet werden.

Wenn Sie weitere Erweiterungen haben, behandeln Sie sie genauso: Deaktivieren Sie ihre Exporte in Konfigurationen, die sie nicht benötigen.

Wenn Sie mehrere Ziele haben platforms, fügen Sie Konfigurationen für diese hinzu und ändern Sie, welche Erweiterungen für sie verwendet werden.
