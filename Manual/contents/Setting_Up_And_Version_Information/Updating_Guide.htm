Leitfaden zur Aktualisierung

Der GameMaker IDE sucht regelmäßig nach Updates und lädt diese automatisch herunter. Einige dieser Aktualisierungen sind optional und Sie können sie nicht installieren, wenn Sie dies nicht möchten oder glauben, dass einige der Änderungen ein Projekt beschädigen könnten, das Sie gerade veröffentlichen wollen. Es gibt jedoch einige Updates, die wichtige Korrekturen oder Verbesserungen enthalten, die nicht übersprungen werden können, und GameMaker erzwingt in diesen Fällen ein Update.

GameMaker ist mehr als nur ein IDE und verfügt über eine Reihe verschiedener runtime Tools, die hinter den Kulissen verwendet werden, um Spiele für die verschiedenen verfügbaren Ziele platforms zu erstellen. Diese Werkzeuge sind hauptsächlich die verschiedenen runners und compilers, die jeder platform benötigt, und Sie können sogar mehrere davon gleichzeitig haben. Das bedeutet, dass Sie sich aussuchen können, welche compiler Version und welche runner Version Sie verwenden möchten, wenn Sie Probleme mit der aktuellen Version haben oder auf eine ganz bestimmte Zielversion oder platform abzielen. Die Tools aktualisieren sich automatisch, wenn eine neue Version verfügbar ist, aber Sie können dann die Runtime Feeds Preferences verwenden, um auszuwählen, welche Sie tatsächlich verwenden möchten. Weitere Informationen zu runtime Aktualisierungen finden Sie unter Voreinstellungen - Runtime Feeds.
