Installationsanleitung

GameMaker ist für die Verwendung auf macOS- und Windows-Desktop-Computern mit den hier genannten Mindestanforderungen konzipiert.

Sie können den GameMaker-Installer von Ihrer YoYo Konto-Seite auf der YoYo Games Website herunterladen. Nach der Installation werden Sie beim ersten Start von GameMaker möglicherweise aufgefordert, ein Update durchzuführen, bevor Sie das Produkt verwenden können. Nachdem alle Updates installiert wurden, müssen Sie die Lizenzdaten für Ihr YoYo Games Konto eingeben (weitere Informationen finden Sie hier ), bevor Sie das Produkt verwenden können, was bedeutet, dass Sie bei der ersten Verwendung des GameMaker eine Internetverbindung haben müssen. Nach der ersten Anmeldung sollten Sie in der Lage sein, das Produkt offline zu nutzen, es wird jedoch in regelmäßigen Abständen verlangen, dass Sie eine Verbindung herstellen, um das Anmeldekonto zu verifizieren.

Bitte beachten Sie, dass Sie - je nach Lizenztyp - möglicherweise zusätzliche Komponenten installieren oder zusätzliche Schritte ausführen müssen, um GameMaker für bestimmte platforms einzurichten und zu verwenden. Ausführliche Informationen darüber, was die einzelnen platform benötigen, finden Sie in diesem Artikel auf YoYo Games Helpdesk: GameMaker Erforderliche SDKs.

Stabile und Beta-Versionen

Die Version, die Sie von der Webseite YoYo Games herunterladen, sollte die aktuelle Stable-Version von GameMaker sein, aber es kann auch eine andere " Beta"-Version verfügbar sein, die dazu dient, neue Funktionen vorzustellen oder zu testen, bevor sie in eine endgültige Stable-Version übernommen wird. Die vollständigen Versionsangaben lauten wie folgt:

  • Stabil: Diese Version ist diejenige, die Sie für die allgemeine Entwicklung verwenden würden und wird für die Verwendung mit jedem kommerziellen Produkt empfohlen. Bestimmte Funktionen, die in den Beta-Versionen verfügbar sind, können fehlen, aber das liegt daran, dass sie noch nicht vollständig getestet wurden oder zusätzliche Arbeit erfordern.
  • Beta: Die Beta-Version von GameMaker ist für diejenigen gedacht, die neue Funktionen ausprobieren möchten, bevor sie in den Stable-Build aufgenommen werden, oder für diejenigen, die eine baldige Lösung für ein Problem benötigen. Es wird nicht empfohlen, den Beta-Build für die endgültige Spielproduktion zu verwenden, es sei denn, er enthält eine wichtige Korrektur für einen Fehler, der dein Projekt aufhält. Du kannst den Beta-Build von GameMaker von der "Downloads"-Seite deines YoYo -Kontos herunterladen und ihn parallel zum Stable-Build installieren und ausführen. Bitte beachten Sie, dass Sie, wenn Sie GameMaker über Steam betreiben, keinen Zugriff auf den Beta-Build des Produkts über Steam selbst haben, obwohl Sie die Desktop-Version weiterhin herunterladen und verwenden können.
