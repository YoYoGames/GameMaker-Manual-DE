Informationen zu Lizenzierung und Version

Es gibt verschiedene Versionen der Lizenzen, die Sie für GameMaker erwerben können. Die Wahl der Lizenzen hängt weitgehend von der Zielgruppe ab, die Sie ansprechen wollen, und davon, ob Sie nur Spaß an der Entwicklung von Spielen haben oder sie auch verkaufen und vertreiben wollen.

Aktuelle Informationen zu den Lizenzen von GameMaker finden Sie auf der folgenden Seite:

  • Welcher Führerschein ist der richtige für mich?

Unabhängig von der Version von GameMaker, die Sie verwenden, benötigen sie alle die gleichen grundlegenden Einstellungen vor der Verwendung. Die folgenden Abschnitte befassen sich damit und enthalten Informationen über die Aktualisierung von IDE und Runtimes, die erforderlichen SDKs für bestimmte Zielsysteme modules und die IDE Voreinstellungen, mit denen Sie die Funktionsweise von GameMaker an Ihre eigene Arbeitsweise anpassen können.
