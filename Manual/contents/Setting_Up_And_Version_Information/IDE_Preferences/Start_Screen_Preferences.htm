Startbildschirm-Einstellungen

Mit den Einstellungen für den Startbildschirm können Sie festlegen, wie sich die Startseite verhält. Die wichtigsten Optionen sind:

  • Tour anzeigen: Hier können Sie auswählen, ob die Einführungstour beim Start von IDE angezeigt werden soll. Dies wird automatisch deaktiviert, wenn Sie die Tour zum ersten Mal überspringen.
  • Einrichtungsassistenten anzeigen: Hier können Sie auswählen, ob der Einrichtungsassistent beim Start von IDE angezeigt werden soll. Der Einrichtungsassistent führt Sie im Wesentlichen durch die Schritte, die zum Erstellen eines neuen Projekts erforderlich sind.
  • Stil der Liste der letzten Projekte: Hier können Sie die Anzeige der Liste der letzten Projekte auf der Startseite zwischen "Liste" und "Raster" ändern.
