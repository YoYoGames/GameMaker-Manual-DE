Sprite-Editor-Einstellungen

In den Sprite-Editor-Voreinstellungen wird festgelegt, wie der Sprite-Editor aussieht und funktioniert. Es gibt die folgenden Optionen:

  • Standardursprung sprite: Mit dieser Option können Sie die Standard-Ursprungsposition für alle neuen sprite Ressourcen festlegen, die Sie erstellen. Der Standardwert ist "Oben links".
  • Standard sprite Art der Kollisionsmaske: Mit dieser Option können Sie die Art der Kollisionsmaske festlegen, die standardmäßig für neue sprites verwendet werden soll. Zur Auswahl stehen Rechteck, gedrehtes Rechteck, Ellipse, Raute oder präzise / präzise pro Frame. Die Standardeinstellung ist Rechteck.
  • Standard-Animationsmodus sprite: Mit dieser Option können Sie den Standard-Animationsmodus für sprites im sprite -Editor festlegen. Sie können entweder " Frames pro Sekunde" oder " Frames pro Spielframe" wählen, wobei die Standardeinstellung " Bilder pro Sekunde" ist.
  • Standard-Animationsgeschwindigkeit sprite : Mit dieser Option können Sie den Standardwert für die Animationsgeschwindigkeit beim Erstellen eines neuen Assets sprite festlegen. Beachten Sie, dass dieser Wert mit dem Animationsmodus verknüpft ist und - zum Beispiel - 3 Bilder pro Sekunde ist nicht dasselbe wie 30 Bilder pro Spielframe. Stellen Sie also sicher, dass Sie diesen Wert in Bezug auf den bevorzugten Modus richtig einstellen. Der Standardwert ist 30.
  • Standardbreite sprite : Mit dieser Option können Sie die Standardbreite für alle neu erstellten sprites festlegen. Der Standardwert ist 64.
  • Standardhöhe sprite: Mit dieser Option können Sie die Standardhöhe für alle neuen sprites festlegen. Der Standardwert ist 64.
  • Größenänderung und Skalierung auf sprite Ursprung anwenden: Wenn diese Option aktiviert ist, wird der Ursprung von sprite geändert, wenn sprite in der Größe verändert oder skaliert wird, wobei der neue Ursprung so verschoben/skaliert wird, dass er mit den neuen Werten für sprite übereinstimmt. Beachten Sie, dass dies nur für benutzerdefinierte sprite Ursprünge gilt, die im sprite Editor nicht gesperrt sind. Wenn der Ursprung gesperrt ist oder sprite eine der Standardpositionen verwendet - z. B. "Mitte" oder "Oben links" usw. - dann wird diese Option ausgegraut und kann nicht geändert werden. Standardmäßig ist sie aktiviert.

Die folgende Unterkategorie existiert auch für den Sprite Editor:

Dialoge bestätigen

Die hier verfügbaren Optionen steuern die Meldungen, die angezeigt werden, wenn Sie eine Aktion bestätigen müssen. Sie können diese Einstellungen so ändern, dass die Meldung immer angezeigt wird oder dass sie bei einer bestimmten Aktion (entweder Ja oder Nein) ausgeblendet wird:

  • Automatische Reaktion auf den Datei-Import: Wenn Sie ein neues Bild importieren und bereits eines vorhanden ist, werden Sie standardmäßig aufgefordert, mit dem Import fortzufahren und das vorhandene Bild zu überschreiben oder den Vorgang abzubrechen. Mit dieser Option können Sie diese Meldung unterdrücken und Sie können entweder den Import abbrechen, indem Sie "Nein" wählen, oder das aktuelle sprite überschreiben, indem Sie "Ja" wählen. Der Standardwert ist "Meldung anzeigen".
  • Automatische Reaktion auf Fehler beim Laden von Bildern: Wenn Sie ein neues Bild importieren und der Import fehlschlägt, wird standardmäßig eine Warnmeldung angezeigt, die Sie darüber informiert, dass ein Fehler aufgetreten ist. Mit dieser Option können Sie diese Meldung unterdrücken, indem Sie "Okay" wählen. Der Standardwert ist "Meldung anzeigen".
