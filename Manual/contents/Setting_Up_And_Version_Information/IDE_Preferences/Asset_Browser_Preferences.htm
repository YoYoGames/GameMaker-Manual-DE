Asset-Browser-Einstellungen

Die Asset-Browser-Voreinstellungen dienen dazu, bestimmte Eigenschaften für die Funktionsweise des Asset-Browsers und seiner Elemente festzulegen. Die wichtigsten Optionen sind:

  • Schnellzugriff anzeigen: Wenn diese Option aktiviert ist, wird der Abschnitt "Schnellzugriff" am oberen Rand des Asset-Browsers angezeigt. Diese Option ist standardmäßig aktiviert.
  • Standardordner im neuen Projekt erstellen: Wenn diese Option aktiviert ist, werden alle neuen Projekte mit den bereits erstellten Standard-Asset-Ordnern gestartet; wenn sie deaktiviert ist, wird nur ein Raumordner mit dem für alle Projekte erforderlichen Standardraum erstellt. Diese Option ist standardmäßig aktiviert.
  • Maximale Anzahl von Assets, die gleichzeitig ohne Warnung geöffnet werden können: Sie können mehrere Assets zum gleichzeitigen Öffnen auswählen, indem Sie / + verwenden, um sie auszuwählen, und dann mit der rechten Maustaste auf klicken und Alle öffnen wählen. Wenn Sie mehr als die in dieser Option angegebene Anzahl ausgewählt haben, wird eine Warnung angezeigt, da das gleichzeitige Öffnen von zu vielen Assets zu Leistungsproblemen auf leistungsschwächeren Rechnern führen kann. Der Standardwert ist 10.
  • Maximal kürzlich bearbeitete Elemente: Diese Einstellung regelt die Anzahl der Assets, die im Abschnitt "Zuletzt bearbeitet" des Asset-Browsers aufgeführt werden. Der Standardwert ist 10.
  • Größe der Farbpalette: Hier können Sie die Anzahl der Farben festlegen, die Ihnen in der Asset-Browser-Palette für Asset-Gruppen zur Verfügung stehen. Der Standardwert ist 16, mit einem Mindestwert von 16 und einem Höchstwert von 100.
  • Asset-Farben im Schnellzugriff anwenden: Wenn diese Option aktiviert ist, werden die für einen bestimmten Asset-Typ eingestellten Farben verwendet, um das Asset im Schnellzugriffsbereich des Browsers einzufärben. Wenn Sie diese Option deaktivieren, werden die Asset-Farben nicht verwendet. Diese Option ist standardmäßig aktiviert.
  • Gruppenfarben haben Vorrang vor den Farben der untergeordneten Assets: Wenn dieses Kontrollkästchen aktiviert ist, wird die Farbe der obersten Gruppenebene für die Einfärbung aller Assets innerhalb der Gruppe verwendet, ohne dass eine für das einzelne Asset eingestellte Farbe berücksichtigt wird. Diese Option ist standardmäßig deaktiviert.
  • Asset-Dialogstil erstellen: Wenn Sie im Asset-Browser auf die Schaltfläche "Asset hinzufügen" klicken, wird ein Fenster mit den verschiedenen Assets angezeigt, die zu Ihrem Projekt hinzugefügt werden können. Dieses Fenster kann in einer Listen- oder einer Rasteransicht dargestellt werden, und diese Einstellung bestimmt, was standardmäßig angezeigt wird. Der Standardwert ist Raster.
  • Standard-Sortierreihenfolge: Hier kann die Standard-Sortierreihenfolge für Assets im Browser festgelegt werden, entweder alphabetisch oder von A bis Z oder von Z bis A. Standard ist von A bis Z.
  • Standardmäßig werden Ordner zuerst sortiert: Wenn diese Option aktiviert ist, sortiert der Asset-Browser zuerst die Ordner und dann die "losen" Assets, wenn sie deaktiviert ist, werden alle Assets und Ordner zusammen sortiert. Diese Option ist standardmäßig aktiviert.
  • Sprite-Doppelklick öffnet Bild-Editor: Standardmäßig öffnet ein Doppelklick auf auf ein beliebiges Asset dieses Asset zur Bearbeitung, aber oft öffnen Sie nach der anfänglichen Einrichtung, wenn Sie das neue Asset erstellen, sprites, um das Bild und nicht die sprite Eigenschaften zu bearbeiten. Wenn Sie diese Option wählen, wird der Bildeditor für das Asset sprite geöffnet und nicht das Asset sprite selbst. Sie können den sprite Editor immer noch öffnen, indem Sie mit der rechten Maustaste auf klicken und Bearbeiten wählen. Diese Option ist standardmäßig deaktiviert.
  • EinDoppelklick auf ein Objekt öffnet alle Ereignisse des Objekts: Standardmäßig öffnet ein Doppelklick auf auf ein beliebiges Asset dieses Asset zur Bearbeitung, aber oft öffnen Sie bei der Arbeit mit Objekten nach der anfänglichen Einrichtung, wenn Sie ein neues Asset erstellen, dieses nur, um die Ereignisse und nicht die Objekteigenschaften zu bearbeiten. Wenn Sie diese Option wählen, werden die verschiedenen Objektereignisse in einem neuen workspace geöffnet, wobei jedes Ereignis eine eigene Registerkarte innerhalb des neuen workspace erhält. Sie können den Objekteditor immer noch öffnen, indem Sie mit der rechten Maustaste auf klicken und Bearbeiten wählen. Diese Option ist standardmäßig deaktiviert.
  • Raumreihenfolge aufheben entfernt die leere Gruppe: Wenn aktiviert, wird beim Aufheben der Gruppierung aller Räume, die zu einem Gruppenordner gehören, auch der Gruppenordner entfernt. Wenn diese Funktion deaktiviert ist, wird die Gruppierung der Räume aufgehoben, der leere Gruppenordner bleibt jedoch im Asset-Browser erhalten. Dies ist standardmäßig aktiviert.
  • Standard-Zoom (%): Hier wird der Standard-Zoom-Prozentsatz für den Asset-Browser festgelegt, wenn Sie ein neues Projekt starten. Der Standardwert ist 100 %.
  • Minimaler Zoom (%): Hier wird der kleinste zulässige Zoomwert für den Asset-Browser festgelegt. Der Standardwert ist 50 %.
  • Maximaler Zoom (%): Hier wird der maximal zulässige Zoomwert für den Asset-Browser festgelegt. Der Standardwert ist 250 %.
  • Baumlinien anzeigen: Hier können Sie die Baumlinien, die neben den Assets im Asset-Browser angezeigt werden, aktivieren/deaktivieren. Diese Option ist standardmäßig aktiviert, um die Baumlinien anzuzeigen.
  • Baum bei Filterwechsel aufklappen: Mit dieser Option wird der Asset-Browser gezwungen, alle kollabierten Baumknoten zu erweitern, wenn Sie die Filteroptionen ändern. Wenn Sie die Option deaktivieren, wird dies verhindert. Standardmäßig ist sie aktiviert.

Dialoge bestätigen

Hier können Sie das Verhalten der Bestätigungsdialoge bei der Arbeit mit einem Asset auswählen aus dem Asset-Browser. Die verfügbaren Optionen sind:

  • Automatische Reaktion auf das Löschen von Assets: Wenn Sie ein Asset aus dem Asset-Browser löschen, erhalten Sie normalerweise eine Warnung, in der Sie gefragt werden, ob Sie sicher sind, dass Sie fortfahren möchten. Wenn Sie diese Option auf "OK" setzen, wird die Meldung unterdrückt als ob Sie auf die Schaltfläche "OK" geklickt hätten, um fortzufahren. Der Standardwert ist "Meldung anzeigen".
  • Automatische Reaktion auf das Löschen von Raumordnungsgruppen: Wenn Sie einen Gruppenordner, der Räume enthält, aus dem Asset-Browser löschen, erhalten Sie normalerweise eine Warnung, in der Sie gefragt werden, ob Sie sicher sind und den Vorgang fortsetzen wollen (in der alle Räume der Gruppe aufgelistet sind die gelöscht werden sollen). Wenn Sie diese Option auf "OK" setzen, wird die Meldung unterdrückt, als ob Sie auf die Schaltfläche OK geklickt hätten, um fortzufahren und alle Räume in der Gruppe sowie den Gruppenordner selbst zu löschen. Der Standardwert ist "Show Meldung".
  • Automatische Reaktion auf das Öffnen mehrerer Assets: Wie bereits in den allgemeinen Einstellungen des Asset-Browsers erwähnt, können Sie beim gleichzeitigen Öffnen mehrerer Assets im Asset-Browser eine Warnung erhalten, wenn die Anzahl der Assets die angegebene Anzahl überschreitet. Wenn Sie diese Option auf "OK" setzen, wird die Warnung unterdrückt, als ob Sie auf die Schaltfläche "OK" geklickt hätten, um fortzufahren. Der Standardwert ist "Meldung anzeigen".
  • Automatische Reaktion auf die Konvertierung von GML Visual in GameMaker Language: Wenn Sie die Option wählen, einen GML Visual-Knoten in GML zu konvertieren, werden Sie aufgefordert, die Änderung zu bestätigen, da diese nicht rückgängig gemacht werden kann. Wenn Sie diese Option auf "OK" setzen wird die Warnung unterdrückt, als ob Sie auf die Schaltfläche "OK" geklickt hätten, um fortzufahren. Der Standardwert ist "Meldung anzeigen".
  • Automatische Reaktion auf das Setzen eines übergeordneten Raums durch Ziehen eines Knotens: Im Raum-Manager-Fenster, Abschnitt "Vererbung", können Sie einen Raum auf einen anderen Raum ziehen, um ihn als Kind des Raums zu setzen, auf den er gezogen wird. Wenn Sie dies erhalten Sie eine Warnung, in der Sie gefragt werden, ob Sie sicher sind und den Vorgang fortsetzen möchten. Wenn Sie diese Option auf "OK" setzen, wird die Meldung unterdrückt, als ob Sie zum Fortfahren auf die Schaltfläche "OK" geklickt hätten. Der Standardwert ist "Meldung anzeigen".
  • Automatische Reaktion auf Fehler bei doppeltem Gruppennamen: Wenn Sie einen Gruppenordner für die Organisation Ihrer Assets erstellen und diesen gleich benennen wie einen anderen Ordner im Browser, wird eine Meldung angezeigt, in der Sie gefragt werden, ob Sie fortfahren möchten und zwei Ordner mit demselben Namen haben möchten. Wenn Sie diese Option auf "OK" setzen, wird die Warnung unterdrückt, als ob Sie auf die Schaltfläche "OK" geklickt hätten, um fortzufahren. Der Standardwert ist "Meldung anzeigen".
  • Automatische Reaktion auf das Festlegen eines übergeordneten Raums durch Ziehen eines Knotens: Wenn Sie im Raum-Manager einen Raum über einen anderen Raum ziehen und ihn dann loslassen, wird eine Meldung angezeigt, in der Sie gefragt werden, ob Sie den Raum zu einem "Kind" des Raums machen möchten, auf den Raumes machen wollen. Wenn Sie diese Option auf "OK" setzen, wird die Warnung unterdrückt, als ob Sie auf die Schaltfläche "OK" geklickt hätten, um fortzufahren. Der Standardwert ist "Nachricht anzeigen".
