Schriftarten-Editor-Einstellungen

In den Einstellungen für den Schriftarten-Editor können Sie festlegen, wie sich der Schriftarten-Editor verhält. Die verschiedenen verfügbaren Optionen sind unten aufgeführt, zusammen mit einer Beschreibung der jeweiligen Funktion.

  • Aus GameMaker 1-Projekten importierte Schriftarten aktualisieren: Wenn diese Option aktiviert ist, versucht GameMaker, die Schriftarten zu aktualisieren und ihre Textur neu zu generieren. Dies geschieht nur, wenn:
    • Diese Voreinstellung ist aktiviert
    • Die Schriftart ist auf dem Rechner verfügbar
    • GameMaker kann eine passende Schrift in der Schriftfamilie finden (dies ist die wahrscheinlichste Ursache für Inkonsistenzen beim Import)
    Beachten Sie, dass dies auch geschieht, wenn die GameMaker 1-Schriftart keine zwischengespeicherte Schrifttextur hat. Wenn eine Legacy-Schrift nicht aktualisiert werden kann, verwendet GameMaker standardmäßig die zwischengespeicherte Schrifttextur zu verwenden, und der Schrift-Editor erlaubt Ihnen nicht mehr, die Schrift zu bearbeiten, sondern zeigt eine Option zum Sichern und Aktualisieren an:

    Wenn Sie auf diese Option klicken, können Sie die Schriftart bearbeiten und entweder die fehlende Schriftart hinzufügen oder eine andere einstellen. Die gesicherte Schriftart wird im Font-Asset-Ordner gespeichert, als <font name>.yy.gms1 und <font name>.png.gms1. Um diese Dateien wiederherzustellen, löschen Sie manuell die Dateien <font name>.yy und ersetzen Sie sie durch diese, indem Sie die Erweiterung .gms1 aus den Dateinamen entfernen. Wichtig! Die Namen aller Assets werden in der Datei .yy gespeichert, daher müssen Sie diese bearbeiten und den richtigen Namen der Ressource auch in das Feld <name> eintragen (falls abweichend).
  • Automatische Auswahl des Dialogs für die fehlgeschlagene Regeneration von Inhalten: Wenn Sie eine Font-Asset-Textur neu generieren wollen, wird eine Warnung angezeigt. Wenn Sie diese Option auf "OK" setzen, wird die Schrift neu generiert, ohne dass die Warnung angezeigt wird. angezeigt.
  • Automatische Auswahl des Dialogs für fehlgeschlagene Schriftenregenerierung: Wenn die Regeneration einer Font-Asset-Textur fehlschlägt, wird eine Warnung angezeigt. Wenn Sie diese Option auf "OK" setzen, wird die Meldung nicht mehr angezeigt.
  • Automatische Auswahl für das Upgrade von importierten Legacy-Schriften: Wenn Sie ein Legacy-Projekt importieren, erhalten Sie möglicherweise eine Warnung über Legacy-Schriften und das Angebot, diese zu aktualisieren. Wenn Sie diese Option auf "OK" setzen, wird die Aktualisierung automatisch genehmigt, ohne diese Meldung angezeigt wird.
