Bildeditor-Einstellungen

Mit den Voreinstellungen für den Bildeditor können Sie festlegen, wie das Bildeditor-Fenster aussehen und funktionieren soll. Im Folgenden sind die allgemeinen Optionen aufgeführt:

  • Erweitern Sie sprite Steuerelemente standardmäßig überlagern: Wenn diese Option aktiviert ist, werden die sprite Steuerelemente und Rahmeninformationen jedes Mal im Bildeditor angezeigt, wenn Sie ein sprite zur Bearbeitung öffnen (dieses kann mit der Schaltfläche wieder geschlossen werden, wird aber später beim Bearbeiten des sprite wieder geöffnet). Wenn diese Option nicht aktiviert ist, werden die Steuerelemente und Rahmeninformationen von sprite beim Öffnen von einer sprite ausgeblendet. Standardmäßig ist diese Option nicht aktiviert.
  • Standardmodus Loop: Die Schaltfläche loop im Bildeditor kann so eingestellt werden, dass sie in zwei verschiedenen Modi funktioniert: entweder einmalige Wiedergabe, bei der die Animation beim letzten Bild anhält, loop von Anfang an, wobei die Animation nach Beendigung zum ersten Bild zurückspringt, oder Ping-Pong, wobei die Animation bis zum Ende und dann wieder zurück zum Anfang läuft. von Anfang an, wobei die Animation zum ersten Bild zurückkehrt, wenn sie beendet ist, oder Ping Pong, wobei die Animation durch die Bilder bis zum Ende und dann wieder zurück zum Anfang läuft. Der Standardwert ist hier "Schleife".
  • Dünne Auswahlkontur zeichnen: Wenn diese Option aktiviert ist, hat der Umriss der Auswahl eine absolute Breite von 1px, unabhängig von der Skalierung des Bildes; wenn sie nicht aktiviert ist, hat der Umriss der Auswahl eine relative Breite von 1px und skaliert mit dem Bild. Standardmäßig ist diese Option aktiviert.
  • Grundfarbe der Auswahlkontur: Die Auswahlkontur ist eine gestrichelte Linie, die zwischen zwei Farben wechselt. Mit dieser Option können Sie die erste dieser Farben (einschließlich Alpha) festlegen. Der Standardwert ist ARGB #FFFFFD0.
  • Sekundärfarbe der Auswahlkontur: Die Kontur des Abschnitts ist eine gestrichelte Linie, die zwischen zwei Farben wechselt. Mit dieser Option können Sie die zweite dieser Farben festlegen (einschließlich Alpha). Der Standardwert ist ARGB #0000D0.
  • Hintergrundfarbe des Editors: Während der Bearbeitung eines sprite Rahmens kann das Bild vergrößert oder verkleinert werden oder nicht in das gesamte Zeichenfenster passen. In diesem Fall wird eine Hintergrundfarbe angezeigt, die Sie hier einstellen können. Der Standardwert ist ARGB #282828FF.

Die Bildeditor-Voreinstellungen haben auch die folgenden Unterkategorien:

Hintergrund

Hier können Sie die Optionen für den Hintergrund festlegen, vor dem Sie Ihre sprite Bilder bearbeiten. Standardmäßig ist dies ein Gitter aus grauen Quadraten, aber Sie können die folgenden Einstellungen verwenden, um es zu ändern:

  • Stil der Leinwand: Hier stellen Sie den "Stil" für den Hintergrund ein, der entweder " Block-Muster" oder " Einfarbig". Bei "Einfarbig" wird der Hintergrund einfach auf die von Ihnen gewählte Farbe eingestellt (mit der Einstellung "Farbe 1", wie unten erläutert), während "Blockmuster" die Standardeinstellung ist (die auch mit den übrigen Optionen des Dialogs Optionen bearbeitet werden kann). Die Voreinstellung ist " Blockmuster".
  • Größe des Leinwandblocks: Wenn Sie für den Hintergrund ein Blockmuster für den Hintergrund ausgewählt haben, können Sie hier die Größe des einzelnen Blocks festlegen. Der Standardwert ist 16px, wobei ein Mindestwert von 2px und ein Höchstwert von 256px zulässig sind.
  • Grundfarbe der Leinwand: Wenn Sie Single Colour für den Hintergrund gewählt haben, wird diese Farbe verwendet. Sie können auf das Farbfeld klicken, um den Farbeditor zu öffnen und die Farbe zu ändern. Wenn Sie die Option Blockmuster wird hier die erste Farbe für abwechselnde Blöcke festgelegt. Die Standardfarbe ist ( ARGB) #FFC3C3C3.
  • Sekundärfarbe der Leinwand (Blockmuster): Wenn Sie Blockmuster für den Hintergrund gewählt haben, wird diese Farbe im Wechsel mit Farbe 1 (oben) verwendet. Sie können auf das Farbfeld klicken, um den Farbeditor zu öffnen und die Farbe zu ändern. Die Standardfarbe ist (ARGB) #FF808080.

Dialoge bestätigen

Hier können Sie das Verhalten der Bestätigungsdialoge auswählen, wenn Sie verschiedene Aktionen auf der aktuellen sprite durchführen. Diese sind:

  • Automatische Reaktion auf das Löschen von Rahmen: Wenn Sie einen Rahmen löschen, wird standardmäßig eine Meldung angezeigt, in der Sie gefragt werden, ob Sie sicher sind, dass Sie dies tun wollen. Wenn Sie diese Option auf "OK" setzen, können Sie diese Meldung entfernen und das Löschen von Rahmen ohne dass weitere Meldungen angezeigt werden. Die Voreinstellung ist "Meldung anzeigen".
  • Automatische Reaktion auf das Ersetzen von Bildern während des Streifenimports: Wenn Sie im Dropdown-Menü "Bild" die Option "Streifen importieren" wählen, wird eine Meldung angezeigt, die Sie darauf hinweist, dass alle Bilder, die sich bereits in der Ressource sprite befinden, überschrieben werden, mit der Option, fortzufahren oder nicht. Sie können hier festlegen, ob diese Meldung angezeigt werden soll, oder ob IDE einfach die Option "Ja" oder "Nein" wählen soll. Der Standardwert ist "Nachricht anzeigen".
  • Automatische Reaktion auf das Festschreiben von Änderungen beim Werkzeugwechsel: Wenn Sie nach der Bearbeitung eines Rahmens das Zeichenwerkzeug wechseln, wird standardmäßig eine Meldung angezeigt, in der Sie gefragt werden, ob Sie die vorgenommenen Änderungen übernehmen möchten. Wenn Sie diese Option auf "OK" setzen, können Sie können Sie diese Meldung entfernen, so dass die Rahmenbearbeitung ohne weitere Meldungen beim Werkzeugwechsel übernommen wird. Wenn Sie "Nein" wählen, wird die Rahmenbearbeitung bei einem Werkzeugwechsel stillschweigend abgebrochen. Die Voreinstellung ist "Anzeigen Meldung".
  • Automatische Reaktion auf Warnung vor gesperrten Ebenen: Wenn Sie versuchen, eine gesperrte Ebene zu bearbeiten, erhalten Sie standardmäßig eine Meldung, die Sie darüber informiert, dass Sie dies nicht tun können. Wenn Sie diese Option auf "OK" setzen, wird keine Meldung angezeigt und die Bearbeitung wird stillschweigend abgebrochen.
  • Automatische Reaktion auf die Warnung bei Gruppenauswahl: Wenn Sie im Ebenen-Editor mehrere Ebenen auswählen und dann versuchen, zu zeichnen oder zu bearbeiten, wird standardmäßig eine Meldung angezeigt, die Sie darüber informiert, dass dies nicht möglich ist. Sie können "OK" wählen wählen, um diese Meldung zu entfernen und die Bilder erst dann gemeinsam zu bearbeiten, wenn nur eine einzige Ebene ausgewählt ist. Die Voreinstellung ist "Meldung anzeigen".



Raster-Einstellungen

Raster Mit den Voreinstellungen wird festgelegt, wie das Bildraster dargestellt wird, wobei folgende Optionen zur Verfügung stehen:

  • Horizontaler Abstand des Gitters (px): Dieser Wert legt den Abstand für die horizontalen Linien des Bildrasters fest. Der Standardwert ist 1px.
  • Vertikaler Abstand des Gitters (px): Dieser Wert legt den Abstand für die vertikalen Linien des Bildrasters fest. Der Standardwert ist 1px.
  • Raster anzeigen: Wenn diese Option aktiviert ist, wird der Bildeditor mit eingeschaltetem Bildraster geöffnet. Die Voreinstellung ist aus.
  • Am Raster ausrichten: Wenn diese Option aktiviert ist, rastet der Bildeditor bestimmte Werkzeuge automatisch am Bildraster ein. Die Voreinstellung ist aus.
  • Rasterlinienfarbe: Hier können Sie die Farbe (einschließlich Alphawert) für das Bildraster festlegen. Standard ist ( ARGB) #FF000000.

Werkzeuge

Hier können Sie die verschiedenen Optionen einstellen einstellen, wie einige der Zeichenwerkzeuge funktionieren und angezeigt werden.

Wenn Sie das Bogen-Werkzeug verwenden, können Sie hier können Sie die verschiedenen Farben einstellen, die zum Zeichnen der Knoteneigenschaften verwendet werden:

  • Ankerpunktfarbe: Legen Sie die Farbe für die verschiedenen Ankerpunkte auf dem Spline/Bogen fest. Standard ist ( ARGB) #FFFF0000.
  • Farbe des Kontrollpunkts: Legen Sie die Farbe für jeden der Kontrollpunkt-"Griffe" an einem Ankerpunkt fest. Standard ist (ARGB) #FFFF0000.
  • Farbe der Tangentenlinie: Legen Sie die Farbe für die Linien fest, die die Kontrollpunkte mit den Ankern verbinden. Standard ist (ARGB) #FF00FFFF.
  • Farbe des ausgewählten Punktes: Legen Sie die Farbe fest, die für alle ausgewählten Ankerpunkte auf dem Spline/Bogen verwendet werden soll. Standard ist (ARGB) #FF00FF00.
  • Punktgröße: Legen Sie die Größe der verschiedenen Anker- und Kontrollpunkte fest, die verwendet werden sollen. Standard ist 10px.
