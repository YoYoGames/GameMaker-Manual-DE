Zeitleisten-Einstellungen

In den Zeitleisten-Voreinstellungen können Sie bestimmte Eigenschaften für die Arbeit mit dem Zeitleisten-Editor festlegen. Die verfügbaren Optionen sind:

  • Automatische Reaktion auf die Konvertierung von GML Visual in GameMaker Language: Wenn Sie die Menüoption der rechten Maustaste in einem Moment der Zeitleiste verwenden, um GML Visuelle Aktionen in script (code) zu konvertieren, werden Sie gefragt, ob Sie fortfahren möchten, da es sich um eine einseitige Konvertierung handelt. Wenn Sie diese Option in "Okay" ändern, wird die Meldung unterdrückt und die Konvertierung wird fortgesetzt, als ob Sie im Dialogfeld auf "Okay" geklickt hätten. Die Standardeinstellung ist "Meldung anzeigen".
  • Eine automatische Reaktion auf eine Warnung bei einer permanenten Aktion: Bestimmte Aktionen innerhalb einer Zeitleiste können nicht rückgängig gemacht werden, und wenn dies geschieht, erhalten Sie eine Warnmeldung, in der Sie gefragt werden, ob Sie fortfahren möchten. Wenn Sie diese Option auf "Okay" ändern, wird die Meldung unterdrückt und die Aktion wird fortgesetzt, als ob Sie im Dialog auf "Okay" geklickt hätten. Die Standardeinstellung ist "Meldung anzeigen".
  • Neue Momente bei der Erstellung zur Bearbeitung öffnen: Wenn Sie einen neuen Moment zu einer Zeitleiste hinzufügen, öffnet GameMaker normalerweise automatisch den Editor, damit Sie Ihr code oder GML Visual hinzufügen können. Wenn Sie diese Option jedoch deaktivieren, wird der Moment hinzugefügt, ohne den Editor zu öffnen (Sie können ihn öffnen, indem Sie auf auf den Moment doppelklicken). Standardmäßig ist diese Option aktiviert.
