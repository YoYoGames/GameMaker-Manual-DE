Marktplatz-Präferenzen

Die Marktplatz-Einstellungen werden verwendet, um das Marktplatz-Zertifikat für das Hochladen oder Kaufen von Assets einzurichten. Das Zertifikatsformular besteht aus den folgenden Abschnitten, wenn Sie bereits ein Zertifikat erstellt haben (siehe weiter unten auf dieser Seite für Details zur Erstellung eines Zertifikats):

  • Zertifikatsdatei: In diesem Abschnitt geben Sie die Datei *.pem an, die Marketplace für die Zertifizierung von Assets verwendet. Wenn Sie ein bereits erstelltes Zertifikat haben, können Sie hier auf die Schaltfläche Dateibrowser klicken und es dem Forum hinzufügen. Wenn Sie jedoch kein Zertifikat haben und eines erstellen möchten, sollten Sie hier keine Angaben machen und stattdessen auf die Schaltfläche Neues Zertifikat erstellen klicken (siehe unten).
  • Zertifikatspasswort: Hier müssen Sie das Zertifikatspasswort für die oben ausgewählte Datei eingeben. Das Kennwort wird beim erstmaligen Erstellen der Datei *.pem erstellt (siehe Neues Zertifikat erstellen, unten) und wird unwiderruflich mit der erstellten Zertifikatsdatei verknüpft.
  • Prüfen: Nachdem Sie die obigen Felder ausgefüllt haben, sollten Sie auf diese Schaltfläche klicken, um sicherzustellen, dass das Passwort und die Dateiauswahl korrekt sind und zusammen funktionieren.

Dialoge bestätigen

Hier können Sie das Verhalten der Bestätigungsdialoge bei der Arbeit mit einer Ressource aus dem Asset-Browser auswählen. Die verfügbaren Optionen sind:

  • Automatische Reaktion auf das Löschen von Offline-Inhalten: Wenn Sie ein Marktplatz-Asset ohne Webverbindung aus Ihrer Bibliothek löschen, erhalten Sie normalerweise eine Warnung, in der Sie gefragt werden, ob Sie sicher sind, dass Sie fortfahren möchten. Wenn Sie diese Option auf "OK" setzen wird die Meldung unterdrückt, als ob Sie auf die Schaltfläche "OK" geklickt hätten, um fortzufahren. Der Standardwert ist "Meldung anzeigen".

Ein neues Zertifikat erstellen

Wenn Sie noch keine Zertifikatsdatei für die Verwendung von Marketplace erstellt haben, sollten Sie auf die Schaltfläche Neues Zertifikat erstellen am unteren Rand dieses Formulars klicken. Daraufhin wird ein neues Fenster geöffnet, in dem Sie aufgefordert werden die folgenden Informationen einzugeben:

  • Name des Herausgebers: Dies ist der Name, der als Herausgebername für das Asset verwendet werden soll. Im Marketplace Dashboard können Sie bei Bedarf verschiedene Verleger einrichten, und jeder sollte eine eigene *.pem Zertifikatsdatei haben.
  • Zertifikatspasswort: Hier müssen Sie ein eindeutiges Passwort für die zu erstellende Zertifikatsdatei angeben. Achten Sie darauf, was Sie wählen, da Sie es später nicht mehr ändern können, und wenn Sie nicht das richtige Passwort für die Zertifikatsdatei in Wenn Sie in den obigen Abschnitten nicht das richtige Passwort für die Zertifikatsdatei angeben, können Sie keine bestehenden Marketplace Assets aktualisieren (und keine neuen hochladen).

Mit diesen Angaben können Sie dann auf die Schaltfläche Zertifikat erstellen klicken, um um die Datei *.pem zu erstellen. Sie werden aufgefordert, einen Speicherort und einen Namen für die Datei anzugeben, den Sie sich für die Zukunft notieren sollten. Sobald Sie dies getan haben, werden die neue Datei und das Passwort automatisch Sie können dann auf die Schaltfläche " Prüfen " klicken, um zu testen, ob das Zertifikat korrekt generiert wurde.

HINWEIS: Wir empfehlen Ihnen, einen Screenshot der aktuellen Einstellungen zu machen und eine Kopie der Datei *.pem zu erstellen. Beide sollten zusammen an einem sicheren Ort gespeichert werden, damit Sie später darauf zurückgreifen können, oder für den Fall, dass Sie GameMaker zu einem späteren Zeitpunkt neu installieren müssen. Sie können keine bereits hochgeladenen Dateien aktualisieren, es sei denn, das aktualisierte Paket ist mit demselben Zertifikat signiert, mit dem das Original erstellt wurde.
