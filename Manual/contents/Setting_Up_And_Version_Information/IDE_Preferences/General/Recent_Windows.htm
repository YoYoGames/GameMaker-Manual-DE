Neueste Windows Präferenzen

Die Einstellungen hier enthalten Optionen dafür, wie die Liste der zuletzt geöffneten Fenster Elemente anzeigt:

  • Fokussierte Fenster an den Anfang verschieben: Wenn diese Option aktiviert ist und Sie auf ein beliebiges Element in der Liste der letzten Fenster klicken, wird das ausgewählte Element an den Anfang der Liste verschoben. Standardmäßig ist diese Funktion deaktiviert.
  • Nur aktuell geöffnete Fenster anzeigen: Wenn Sie diese Option aktivieren, werden in der Liste "Zuletzt geöffnete Fenster" von Windows alle Elemente entfernt, die nicht mehr in workspace geöffnet sind (d. h., wenn Sie ein Fenster schließen, wird es ebenfalls aus der Liste entfernt). In der Standardeinstellung ist dies deaktiviert.
  • Maximal anzuzeigende Fenster: Hier wird die maximale Anzahl der zuvor geöffneten Fenster festgelegt, die in der Liste "Zuletzt geöffnet" angezeigt werden Windows. Der Standardwert ist 10.
