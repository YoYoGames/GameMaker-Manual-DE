Hintergrund-Präferenzen

Die Hintergrund-Einstellungen betreffen den Hintergrund von Workspace und was und wie er angezeigt wird. Die Optionen sind:

  • Hintergrundbild: Dies ist die Bilddatei, die als Hintergrund für die verschiedenen Workspaces verwendet wird. Es wird ein Standardbild mitgeliefert, das standardmäßig aktiviert ist, aber Sie können hier Ihr eigenes Bild auswählen. Die Arten von Bildern die Sie auswählen können, sind *.png, *.gif, *.jpg, *.jpeg, *.tif, *.tiff, *.ico, *.bmp, *.2bp oder *.mdi, aber beachten Sie, dass unabhängig vom ursprünglichen Bildformat alle Dateien zur Verwendung als *.png gespeichert werden, So werden beispielsweise animierte *.gif Dateien nur als ein einziges Anfangsbild gerendert.
  • Bild skalieren: Hier wird festgelegt, wie das Hintergrundbild in den verschiedenen workspaces skaliert werden soll. Die Auswahlmöglichkeiten sind:
    • Dehnen - Das Bild wird gedehnt, um es an die workspace anzupassen, wobei die Skalierung in Abhängigkeit von den sichtbaren Docks geändert wird.
    • Kacheln - Das Bild wird so gekachelt, dass es den Arbeitsbereich abdeckt.
    • Best Fit - Das Bild wird so skaliert, dass es proportional auf die workspace passt, wobei um den Rand herum Balken hinzugefügt werden, wenn es nicht 1:1 passt.
    • Ausdehnung des Desktops - Der Hintergrund deckt die gesamte workspace ab und wird beim Öffnen/Schließen von Docks eher verschoben als skaliert.
    • Bildschirmausdehnung - Der Hintergrund deckt einen Bereich in der Größe des Fensters ab und wird beschnitten/erweitert, um mehr oder weniger des Bildes zu zeigen, wenn die Docks geöffnet/geschlossen werden.
  • Hintergrundbilder auf Arbeitsflächen anzeigen: Diese Option steuert, ob das ausgewählte Hintergrundbild angezeigt werden soll oder nicht. Standardmäßig ist diese Option aktiviert.
