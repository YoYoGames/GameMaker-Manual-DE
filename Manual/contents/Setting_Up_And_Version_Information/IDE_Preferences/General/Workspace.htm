Arbeitsbereich-Einstellungen

Die Einstellungen hier wirken sich darauf aus, wie Sie sich in den verschiedenen GameMaker Arbeitsbereiche. Die Optionen sind:

  • Tastaturnavigationsbereich im Arbeitsbereich: Innerhalb eines workspace können Sie mehrere Fenster für verschiedene Objekte und Ressourcen geöffnet haben, und um sich schnell zwischen ihnen zu bewegen, können Sie die Tastenkombination / + + <Arrow Keys> verwenden. Da die Fenster innerhalb eines workspace nicht am Raster ausgerichtet sind, muss IDE entscheiden, welches Fenster Sie meinen, wenn Sie eine der 8 möglichen Richtungen drücken. Mit dieser Option können Sie den "Blickwinkel" einstellen, der verwendet wird. Der Standardwert ist 40°.
  • Workspace chain column padding (px): Workspace Elemente sind oft miteinander "verkettet" (z.B. ist der Code Editor mit dem Event-Editor mit dem Objekt-Editor verkettet), und mit dieser Option können Sie den den horizontalen Abstand zwischen zwei verketteten Fenstern festzulegen. Der Standardwert ist 30px.
  • Arbeitsbereichsketten-Polsterung (px): Wie bereits erwähnt, werden die Elemente von workspace oft miteinander "verkettet". Mit dieser Option können Sie den vertikalen Abstand zwischen zwei verketteten Fenstern festlegen. Der Standardwert ist 20px.
  • Anzahl der Rendersegmente für Arbeitsbereichsketten: Hier können Sie die maximale Anzahl von Segmenten für Rendering-Ketten festlegen. Ein niedrigerer Wert kann die Leistung auf leistungsschwächeren Rechnern verbessern. Der Standardwert ist 20.
  • Farbe der Arbeitsbereich-Kettenglieder: Dies ist die Farbe, die für die Anzeige von Ketten zwischen workspace Elementen verwendet wird. Die Standardfarbe ist ARGB $99DD00.
  • DieTastaturnavigation im Arbeitsbereich setzt den Zoom zurück: Wie bereits erwähnt, können Sie innerhalb des workspace mit Hilfe von Tastaturkürzeln von Fenster zu Fenster springen. Sie können die workspace auch mit / + dem Mausrad vergrößern und verkleinern. Wenn Sie diese Option aktivieren, bedeutet dies, dass, wenn Sie die workspace gezoomt haben und die Tastenkombinationen verwenden, um zu einem anderen Fenster zu wechseln, die workspace in eine 1:1-Ansicht zurückgesetzt und auf das ausgewählte Fenster zentriert wird. Diese Option ist standardmäßig deaktiviert.
  • Arbeitsbereichsketten können sich überschneiden: Wenn diese Option aktiviert ist, können sich die Ketten von workspace kreuzen und überlappen. Diese Option ist standardmäßig deaktiviert.
