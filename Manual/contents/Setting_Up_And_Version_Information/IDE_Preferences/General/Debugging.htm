Debugging-Einstellungen

Die Einstellungen hier beziehen sich auf den GameMaker-Debugger und Sie haben die folgenden Optionen:

  • Automatische Reaktion auf das Schließen des Debuggers: Mit dieser Option können Sie die automatische Reaktion beim Schließen des Debuggers einstellen. Wenn Sie die Option "Meldung anzeigen" wählen, wird der Dialog angezeigt, wenn Sie den Debugger schließen möchten, während Wenn Sie diese Option auf "Ja" oder "Nein" setzen, wird dieser Dialog nicht angezeigt und die Aufgabe wird wie angegeben ausgeführt.
  • Echtzeit-Debugging aktivieren: Wenn Sie diese Option aktivieren, können Sie Ihre Spielprojekte in Echtzeit debuggen, während Sie sie auf IDE ausführen. Diese Option ist standardmäßig aktiviert.
  • Debugger-Anschluss: Hier wird die Anschlussnummer für den Debugger Module festgelegt. Der Standardwert ist 6509.
  • Ereignissymbole in der Debugger-Ressourcenansicht anzeigen: Legen Sie fest, ob die Symbole in der Debugger-Ressourcenansicht angezeigt werden sollen. Dies ist standardmäßig aktiviert.
