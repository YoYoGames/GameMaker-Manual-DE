Ausgabefenster-Einstellungen

Die Ausgabe-Fenster Voreinstellungen steuern die verschiedenen Farb- und Formatierungsoptionen für die verschiedenen Ausgabefenster in IDE. Jedes der verschiedenen Fenster hat die gleichen Optionen, die im Folgenden aufgelistet sind:

  • Farbe: Die Standardfarbe für den Text, der in das Fenster geschrieben wird.
  • Name der Schriftart: Der Name der zu verwendenden Schriftart (die verfügbaren Schriftarten hängen von den auf Ihrem System installierten ab).
  • Stil: Der zu verwendende Schriftstil, z. B. fett oder kursiv (beachten Sie, dass nicht alle Fenster die gleichen Stile zulassen und dass dies auch von der verwendeten Schriftart abhängt).
  • Größe: Die Größe, auf die die Schrift für das angegebene Ausgabefenster eingestellt werden soll.
  • Anti-Aliasing: Hier können Sie den Anti-Aliasing-Modus für die Darstellung der Schrift im Ausgabefenster festlegen. Sie können IDE die Entscheidung überlassen, indem Sie die Option Standard verwenden wählen, oder Sie können die Option Ein oder Aus erzwingen, indem Sie eine der beiden Optionen wählen.
