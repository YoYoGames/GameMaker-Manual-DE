Hilfe-Einstellungen

Auch wenn die Dateien lokal gespeichert werden (je nach Ihren Einstellungen), ist ein externer Browser erforderlich, um sie anzuzeigen. Die folgenden Einstellungen, die sich auf das Hilfe-Handbuch beziehen, können hier geändert werden:

  • Sprache des Handbuchs: Hier wird die Sprache des Handbuchs eingestellt. Die Standardeinstellung ist "Englisch".
  • Online-Handbuch verwenden: Aktivieren Sie diese Option, um die Online-Handbuch-Website zu verwenden, wenn das Handbuch geöffnet wird, oder deaktivieren Sie sie, um das Handbuch lokal auszuführen. Diese Option ist standardmäßig deaktiviert.
  • Hilfe Handbuch Port: Hier wird die Portnummer für das Micro-Web-Handbuch server festgelegt, wenn "Online-Handbuch verwenden" deaktiviert ist. Der Standardwert ist 51290.
