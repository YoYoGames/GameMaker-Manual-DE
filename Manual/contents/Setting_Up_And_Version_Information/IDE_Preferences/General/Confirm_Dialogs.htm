Dialogvorgaben bestätigen

Die Einstellungen für den Bestätigungsdialog befassen sich mit einigen der allgemeinen Bestätigungs-Popups, die beim Ausführen bestimmter Aktionen in GameMaker IDE erscheinen. Die Optionen sind:

  • Automatische Reaktion auf das Schließen eines workspace über eine Tastenkombination: Die oben erwähnte Tastenkombination kann zum Schließen jeder geöffneten Registerkarte verwendet werden, aber manchmal kann es passieren, dass Sie versehentlich eine leere Registerkarte schließen, die Sie benötigen. Um dies zu verhindern, können Sie Um dies zu verhindern, können Sie diese Option auf "Meldung anzeigen" (die Standardoption) setzen, um zu fragen, ob Sie die leere Registerkarte wirklich schließen möchten. Die anderen Einstellungen sind "Nein", um einfach nichts zu tun, wenn die Verknüpfung verwendet wird, oder "Okay", um die Registerkarte ohne Meldung zu schließen. Meldung zu schließen.
  • Automatische Reaktion auf den Neustart eines laufenden Builds: Wenn Sie einen Build neu starten und ein anderer, früherer Build noch läuft, werden Sie gefragt, ob Sie den aktuellen Build zuerst stoppen wollen. Wenn Sie diese Meldung nicht ankreuzen, wird diese Meldung unterdrückt Meldung unterdrückt und zwingt GameMaker, davon auszugehen, dass Sie als Antwort auf "Okay" geklickt haben. Diese Option ist standardmäßig auf "Meldung anzeigen" eingestellt.
  • Automatische Reaktion auf Warnung beim Versuch, YYC zu debuggen: Das YYC-Ziel wird nach code kompiliert und kann daher nicht mit dem eingebauten Debugger debuggt werden. Wenn Sie versuchen, den Debugger für das YYC-Ziel zu verwenden, erhalten Sie eine Warnung die Sie darüber informiert, dass das Projekt stattdessen mit der VM kompiliert wird. Diese Option ist standardmäßig auf "Meldung anzeigen" eingestellt, aber wenn Sie sie auf "OK" setzen, wird diese Warnung unterdrückt.
  • Automatische Reaktion auf das Zurücksetzen des Projektlayouts: Wenn Sie das Projektlayout zurücksetzen (aus dem Menü Layouts), wird standardmäßig eine Warnmeldung angezeigt darüber angezeigt, dass dadurch alle geöffneten Ressourcenfenster geschlossen werden. Sie können diese Meldung jedoch unterdrücken und das Layout automatisch zurücksetzen, indem Sie "Okay" wählen, oder nichts tun, indem Sie "Nein" wählen. Der Standardwert ist "Show Meldung".
