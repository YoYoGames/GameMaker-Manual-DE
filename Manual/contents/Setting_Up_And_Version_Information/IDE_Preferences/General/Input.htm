Eingabe-Einstellungen

Die Eingabeeinstellungen steuern bestimmte Aspekte der Maus- und Tastatureingabe innerhalb von IDE. Die verfügbaren Optionen sind:

  • Langsame Doppelklickzeit (ms): Einige Bereiche von IDE benötigen einen langsameren Doppelklick, um korrekt zu funktionieren, zum Beispiel beim Umbenennen einer Ressource (ein schneller Doppelklick öffnet den Ressourceneditor, während ein langsamer Doppelklick das Umbenennen ermöglicht). Mit dieser Option können Sie die Erkennungsgeschwindigkeit einstellen, mit einem Standardwert von 500 ms.
  • Scroll-Lock-Zeit (ms): Diese Option betrifft die Zeit, die GameMaker zwischen sich überlappenden Scrollfenstern warten soll. Wenn Sie in Fenster "A" scrollen, kann es sein, dass der Cursor über einem anderen scrollbaren Bereich in Fenster "B" schwebt. Diese Einstellung legt also fest, wie lange Sie warten müssen, nachdem Sie mit dem Scrollen in "A" aufgehört haben, bevor Sie in "B" scrollen können, damit Sie nicht versehentlich mit dem Scrollen in verschiedenen Bereichen beginnen. Der Standardwert ist 500ms.
  • Cursor beim Schwenken um den Bildschirm herumführen: Wenn diese Option aktiviert ist, wird der Cursor beim Schwenken des Bildschirms um die Bildschirmgrenzen herumgeführt workspaces. Standardmäßig ist diese Option aktiviert.
  • Bildlaufgeschwindigkeit: Steuert die Bildlaufgeschwindigkeit innerhalb der verschiedenen workspaces. Der Standardwert ist 1.
  • Entfernung der Registerkarte zum Aufheben der Maximierung des Desktops (px): Wenn Sie eine Registerkarte workspace von ihrer Standardposition innerhalb von IDE wegziehen, wird das GameMaker-Fenster, wenn nur eine workspace aktiv ist, entmaximiert und mit der Maus verschoben (wenn Sie mehr als eine workspace aktiv haben, wird eine neue Instanz von IDE erstellt, um die workspace aufzunehmen). Diese Einstellung steuert, wie weit Sie die Registerkarte ziehen müssen, bevor die workspace sie erkennt, und der Standardwert ist 50px.
  • Minimale Zoomstufe für die Interaktion mit Fensterinhalten (%): Diese Option steuert die Zoomgrenze, ab der die Maus nicht mehr in der Lage ist, mit den Fenstern von workspace zu interagieren (d. h.: Knoten hinzufügen, Fenster ziehen usw.). Der Standardwert ist 40.
  • Mausrad-Zoom erfordert Strg-Taste: Wenn diese Option aktiviert ist, können alle Editoren (und das Hauptfenster workspace) mit / und dem Mausrad gezoomt werden. Wenn die Option nicht aktiviert ist, kann nur mit dem Mausrad gezoomt werden MMB Icon.
  • Strg + MMB zum Öffnen des vererbten Ereignisses anstelle des Handbuchs: Wenn diese Option aktiviert ist, wird durch Gedrückthalten von / + der mittleren Maustaste in einem Objekt das geerbte Ereignis des Objekts (sofern es eines hat) anstelle des Handbuchs geöffnet.
  • Vorspannung zur Textauswahl hinzufügen, um Zeilensprünge zu verhindern: Wenn diese Option aktiviert ist, wird die gedachte Zeilenhöhe bei der Auswahl einer einzelnen Zeile in einem beliebigen Textfenster aufgefüllt, so dass sie weniger anfällig für vertikale Ungenauigkeiten ist. Diese Option ist standardmäßig aktiviert.
  • Aktivieren Sie den Laptop-Modus auf allen Geräten: Wenn Sie dieses Kontrollkästchen aktivieren, wird die Schaltfläche " Laptop-Modus" auf IDE angezeigt, so dass Sie sie unter switch je nach Bedarf ein- oder ausschalten können, unabhängig davon, ob das Gerät, auf dem GameMaker läuft, ein Laptop ist oder nicht. Diese Option ist standardmäßig nicht aktiviert.
  • Laptop-Modus erzwingen: Diese Einstellung legt fest, ob der Laptop-Modus verwendet werden kann oder nicht, und ist standardmäßig aktiviert. Sie können diese Einstellung deaktivieren, um sicherzustellen, dass der Laptop-Modus nicht automatisch aktiviert wird, wenn Sie IDE auf einem Laptop starten.
