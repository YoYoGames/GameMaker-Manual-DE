Sequenzen Präferenzen

Die Voreinstellungen für Sequenzen werden verwendet, um bestimmte Eigenschaften für die Funktionsweise des Sequenz-Editors und seiner Elemente zu definieren. Die wichtigsten Optionen sind:

  • Canvas-Rahmen bei neu erstellten Sequenzen anzeigen: Wenn diese Option aktiviert ist, wird die Smart-Guide-Leinwand im Canvas des Sequence Editors für alle neuen sequences angezeigt. Diese Option kann auch über die Canvas-Eigenschaften im Editor selbst aktiviert/deaktiviert werden. Der Standardwert ist aktiviert.
  • Bildschirmbreite bei neu erstellten Sequenzen: Hier können Sie die Standardrahmenbreite für alle neuen sequences festlegen. Der Standardwert ist 1366px.
  • Rahmenhöhe derLeinwand bei neu erstellten Sequenzen: Hier können Sie die Standard-Rahmenhöhe für alle neuen sequences festlegen. Der Standardwert ist 768px.
  • Pause beim Klicken auf die Leinwand: Wenn diese Option aktiviert ist, wird die Wiedergabe von Sequence angehalten, wenn Sie auf eine beliebige Stelle im sequence Editor-Canvas klicken. Diese Option ist standardmäßig aktiviert.
  • Verzögerung der Überlappungsauswahl in Millisekunden: Hier können Sie festlegen, wie lange Sie warten müssen, bis die Item Toolbox bei sich überlappenden Elementen auf der Arbeitsfläche des Sequence Editors angezeigt wird. Der Standardwert ist 1000 ms.
  • Einfügemarke für den Translationspfadpunkt: Wenn Sie im Canvas des Sequence Editors eine Kühlstelle hinzugefügt und dann eine Positionsparameterspur erstellt haben, werden gepunktete Linien gezeichnet, um den Pfad zu zeigen, den die Kühlstelle bei ihrer Bewegung nehmen wird. Wenn Sie Ihre Maus nahe an dieser Linie platzieren, können Sie einen neuen Parameter-Track-Punkt hinzufügen, und diese Option legt den Abstand von der Translationslinie fest, den die Maus haben muss, bevor der Punktmarker angezeigt wird. Der Standardwert ist 6px.
  • Standard-Parameterspuren automatisch erstellen: Wenn diese Option aktiviert ist, werden allen neuen Tracks, die im Track Panel von sequence erstellt werden, automatisch Parametertracks für Position, Skalierung, Drehung und Ursprung hinzugefügt.
  • Automatisch neu erstellte Assets strecken: Mit dieser Option werden alle neu erstellten Asset-Schlüssel gestreckt, so dass sie der Länge von sequence entsprechen. Die Standardeinstellung ist deaktiviert.
  • Höhe der Anim-Kurven-Spur im Dopesheet: Mit dieser Option können Sie festlegen, wie viele Spurzeilen Parameter-Spuren, die Kurven verwenden, in der Keyframe-Ansicht im Dopw-Sheet einnehmen sollen. Die Voreinstellung ist 2, aber Sie können auch 1 oder 3 wählen.
  • Standardtext für eine neue Textspur: Die Textzeichenfolge, die verwendet wird, wenn Sie eine neue Textspur in einer Sequence erstellen.
  • Inspektor-Auswahlfarbe: Die Farbe, mit der die in der Inspektor-Multi-Select-Liste ausgewählte Spur hervorgehoben wird.

Dialoge bestätigen

Mit den Einstellungen für die Bestätigungsdialoge können Sie die Meldungen ändern, die im Editor sequence angezeigt werden, wenn Sie bestimmte Aktionen durchführen. Die Meldungen, die Sie ändern können, sind:

  • Automatische Auswahl zum Einbetten einer bearbeiteten Kurve: Wenn Sie versuchen, eine von Ihnen bearbeitete Animationskurve einzubetten, wird Ihnen standardmäßig eine Meldung angezeigt, um die Aktion zu bestätigen. Wenn Sie diese Option jedoch auf "Ja" setzen, wird die Meldung nicht angezeigt und die Kurve wird jedes Mal eingebettet. Wenn Sie die Option auf "Nein" setzen, wird die Einbettung jedes Mal verhindert.
  • Automatische Auswahl zur Umwandlung von Kurven in Keyframes: Wenn Sie versuchen, eine Animationskurve in Keyframes zu konvertieren, wird standardmäßig eine Meldung angezeigt, in der Sie gefragt werden, ob Sie fortfahren möchten. Wenn Sie diese Option jedoch auf "Ok" setzen, wird die Meldung nicht angezeigt und Sie können die Kurve einfach nicht konvertieren.

Intelligente Leitfäden

Hier können Sie das Verhalten der intelligenten Hilfslinien bei der Arbeit im Sequences Editor festlegen. Die verfügbaren Optionen sind:

  • Intelligente Hilfslinien aktivieren: Hier können Sie die intelligenten Hilfslinien im Sequences Editor Canvas aktivieren oder deaktivieren. Diese Funktion ist standardmäßig aktiviert.
  • Lineale aktivieren: Hier können Sie die intelligenten Hilfslinien im Sequences Editor Canvas aktivieren oder deaktivieren. Diese Option ist standardmäßig aktiviert.
  • Lineale an Rändern einrasten: Mit dieser Option können Lineale an den Kanten von Assets im sequences Canvas "einrasten" (und umgekehrt). Standardmäßig ist sie aktiviert.
  • Fangabstand: Dies ist der Abstand (in Pixel), den ein Element zu einem Lineal (oder ein Lineal zu einem Element) haben muss, damit es an der Position "einrastet". Der Standardwert ist 6px.
