Einstellungen des Erweiterungseditors

Die Einstellungen des Erweiterungseditors umfassen einige Optionen, die sich auf die Meldungen beziehen, die Ihnen bei der angezeigt werden, wobei die genauen Details weiter unten beschrieben werden:

  • Automatische Reaktion auf das Löschen von Dateien: Standardmäßig wird hier eine Meldung angezeigt, in der Sie gefragt werden, ob Sie sicher sind, dass Sie die Datei löschen möchten, aber Sie können diese Einstellung auf "OK" setzen, so dass alle Löschvorgänge ohne diese Meldung abgeschlossen werden, als ob Sie bereits bereits auf "OK" geklickt hätten.
  • Automatische Reaktion auf das Löschen von Proxydateien: Wie oben, steuert dies die Meldung, die Ihnen beim Löschen von Proxydateien angezeigt wird.
