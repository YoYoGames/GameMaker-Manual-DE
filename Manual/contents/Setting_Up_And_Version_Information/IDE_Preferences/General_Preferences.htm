Allgemeine Präferenzen

Die allgemeinen Einstellungen werden verwendet, um das allgemeine Verhalten der GameMaker-IDE zu definieren. Die allgemeinsten sind auf der Hauptseite aufgeführt, während viele andere in mehrere Unterkategorien aufgeteilt sind, die weiter unten auf dieser Seite erläutert werden.

  • IDE-Sprache: Mit dieser Option können Sie die Sprache für IDE aus der Liste der mitgelieferten Optionen auswählen. Die Voreinstellung ist Englisch.
  • IDE-Schriftart: Mit dieser Option können Sie die in der IDE verwendete Schriftart ändern. Standardmäßig ist Open Sans eingestellt.
  • Maximale Framerate des GameMaker-Editors: Mit dieser Option können Sie die maximale Framerate einstellen, mit der der GameMaker IDE aktualisiert wird, so dass Sie die Framerate an die Aktualisierungsgeschwindigkeit Ihres Monitors anpassen können. Der Standardwert ist 60.
  • IDE-Oberfläche: Hier können Sie zwischen den verschiedenen Skins wählen, die IDE verwenden kann. Standardmäßig wird GameMaker mit zwei Skins geliefert, einem hellen und einem dunklen, und bei der Ersteinrichtung wird der dunkle Skin verwendet.
  • Standard-Audiogerät: Hier können Sie das Standard-Audiogerät auswählen, das GameMaker IDE verwenden soll. Sie können die Liste aktualisieren, indem Sie auf die Schaltfläche Aktualisieren klicken.
  • Automatische Antwort auf Beenden-Bestätigung: Mit dieser Option können Sie einstellen, ob eine Bestätigungsmeldung angezeigt werden soll, wenn Sie GameMaker verlassen. Standardmäßig ist diese Option auf "Meldung anzeigen" eingestellt, kann aber auch so eingestellt werden, dass die Meldung übersprungen wird, als ob Sie auf OK geklickt hätten.
  • Automatische Reaktion auf den Dialog "Systemschriftart geändert": Mit dieser Option können Sie festlegen, ob eine Bestätigungsmeldung angezeigt werden soll, wenn eine Systemschriftart hinzugefügt, entfernt oder bearbeitet wurde. Standardmäßig ist diese Option auf "Meldung anzeigen" eingestellt, kann aber auch so eingestellt werden, dass die Meldung übersprungen wird, als ob Sie auf OK geklickt hätten.
  • Benachrichtigungen aktivieren: Wenn Sie diese Option aktivieren, werden auf IDE Benachrichtigungen über neue runtimes usw. angezeigt. Diese Option ist standardmäßig aktiviert.
  • Zeitüberschreitung für Benachrichtigungen (ms): Legt die Zeitspanne (in Millisekunden) fest, für die eine Benachrichtigung angezeigt wird. Der Standardwert ist 6000.
  • Beim Beenden abmelden: Wenn diese Option aktiviert ist, meldet IDE die Benutzersitzung automatisch ab, wenn GameMaker geschlossen wird. Dies ist in erster Linie für Benutzer aus dem Bildungsbereich gedacht, da - wenn diese Option aktiviert ist - der Benutzer jedes Mal, wenn das Programm beendet wird, abgemeldet wird und somit der Sitzungsplatz für andere Benutzer frei wird. Diese Option ist standardmäßig ausgeschaltet.
  • Willkommensbildschirm beim Starten anzeigen: Diese Option ist standardmäßig aktiviert und weist GameMaker lediglich an, beim Programmstart einen kurzen Begrüßungsbildschirm anzuzeigen. Wenn Sie diese Option deaktivieren, wird der Begrüßungsbildschirm nicht angezeigt.
  • Schließen von leeren Tabs per Tastenkombination erlauben: Sie können alle offenen Tabs mit der Tastenkombination / + W schließen, aber wenn Sie diese Option deaktivieren, wird diese Tastenkombination nicht mehr funktionieren. Diese Option ist standardmäßig aktiviert.
  • Öffnen Sie neue Ressourcen zur Bearbeitung, indem Sie Dateien in die IDE ziehen: Bestimmte Ressourcen (z. B. sprites oder Sounds) können auf das Hauptfenster Workspace im GameMaker IDE gezogen werden und werden als neue Ressource im Asset-Browser erkannt und hinzugefügt. Standardmäßig wird in diesem Fall auch der entsprechende Ressourcen-Editor zur Bearbeitung geöffnet, aber wenn Sie diese Option deaktivieren, werden die Editoren nicht geöffnet und Sie müssen sie manuell öffnen.
  • Neue Ressourcen bei der Erstellung zur Bearbeitung öffnen: Wenn diese Option aktiviert ist, wird jedes Mal, wenn Sie eine neue Ressource im Asset-Browser erstellen, automatisch der entsprechende Ressourceneditor geöffnet. Diese Option ist standardmäßig aktiviert. Wenn Sie sie deaktivieren, müssen Sie alle neu erstellten Ressourcen manuell öffnen.
  • Geänderte Dateien automatisch neu laden: Wenn diese Option aktiviert ist, überprüft IDE automatisch, ob Dateien außerhalb von GameMaker geändert wurden, und lädt sie gegebenenfalls neu.
  • Deaktivieren Sie IDE Übergangsanimationen: Der GameMaker IDE verfügt über eine Reihe von animierten Übergängen für Dinge wie das Öffnen von Fenstern usw., die jedoch nicht jedermanns Geschmack sind und auf älteren oder leistungsschwachen Rechnern zu einem Leistungsschub führen können, wenn sie deaktiviert sind. Sie können sie hier deaktivieren, und sie sind standardmäßig aktiviert.
  • Opt-in für Feedback-Umfragen: Wenn diese Option aktiviert ist, teilen Sie GameMaker mit, dass Sie an Feedback-Umfragen teilnehmen möchten. Das bedeutet, dass gelegentlich während der Arbeit eine Nachricht zu dem Bereich angezeigt wird, an dem Sie gerade arbeiten, und um eine Bewertung und ein Feedback gebeten wird. Diese Option ist standardmäßig aktiviert.
  • DPI-Überschreibung aktivieren: Mit dieser Option können Sie die Standard-DPI-Einstellung mit Ihrer eigenen überschreiben. Sie müssen zuerst die Option aktivieren und dann entweder den Wert " Raw DPi Override " (der Standardwert ist 96) oder den Prozentsatz des nativen DPi (Standardwert 100 %) einstellen. Nach der Änderung werden Sie aufgefordert, IDE neu zu starten, was Sie dazu veranlassen kann, Ihre Arbeit an dem derzeit geöffneten Projekt zu speichern. Beachten Sie, dass der DPi einen Mindestprozentsatz von 50% und einen Höchstprozentsatz von 500% hat. Wenn Sie mit der Maus über das Informationssymbol fahren, sehen Sie die empfohlene Maximaleinstellung für die verwendete Anzeige:

    Wenn Sie die empfohlene DPi-Einstellung für den Monitor überschreiten, wird eine Meldung angezeigt, mit der Sie den DPi zurücksetzen können.
  • Use US Date Format: This setting changes the format used to display dates in the IDE. When enabled, dates are displayed as e.g. "Thursday, April 27, 2023", when disabled, dates are displayed as e.g. "27 April 2023" instead. By default this preference is unchecked.

Die oben genannten Optionen sind die Hauptoptionen, die in den allgemeinen Einstellungen zur Verfügung stehen, aber es gibt eine Reihe von zusätzlichen Optionen, die in den unten aufgeführten Unterkategorien zur Verfügung stehen:

  • Hintergrund
  • Kompilieren
  • Dialoge bestätigen
  • Fehlersuche
  • Dateiüberwacher
  • GoTo
  • Hilfe
  • Eingabe
  • Ausgabefenster
  • Pfade
  • Energieeinsparung
  • Neue Fenster
  • Ressourcen
  • Arbeitsbereiche
