Laufzeit-Feed-Einstellungen

In den Runtime-Voreinstellungen können Sie die aktuell installierte Runtime sowie die aktuell verwendete runtime anzeigen und ändern. GameMaker kann mehrere verschiedene runtime Tools gleichzeitig installiert haben, und Sie können in dieser Voreinstellung festlegen, welches davon verwendet werden soll. Idealerweise möchten Sie das aktuellste runtime verwenden, aber manchmal kann es Probleme geben oder Sie müssen aus Gründen der Vergleichbarkeit ein älteres Programm verwenden usw... Daher können Sie hier die runtime Tools ändern, die GameMaker zum Erstellen Ihres Spiels verwendet.

Der erste Abschnitt dieser Einstellung bezieht sich darauf, woher Sie die Tools von runtime beziehen, und erfordert die Eingabe einer URL für einen RSS-Feed. Dieser Feed wird dann nach Updates für die runtime Tools durchsucht, die dann heruntergeladen und installiert werden können. In diesem Abschnitt wird standardmäßig die URL des Master-Feeds runtime angezeigt, aber Sie können auf die Schaltfläche Hinzufügen klicken, um einen neuen Runtime Feed zu erstellen, in dem Sie den Namen und die URL für den RSS-Feed angeben, und dann auf Übernehmen klicken, um GameMaker den Zugriff darauf zu ermöglichen. Sie müssen das Fenster "Einstellungen" schließen und erneut öffnen, damit der neue Feed angezeigt wird.

Sobald ein Runtime Feed erstellt wurde, wird er in diesem Abschnitt als Unterkategorie aufgeführt, und Sie sollten immer mindestens einen Feed aufgeführt haben (standardmäßig wäre dies der Master-Feed ). Die Runtime Feed-Einstellungen dienen zur Auswahl, Installation und Deinstallation der verschiedenen runtime Tools. Oben sehen Sie die aktuell verwendeten runtime Tools, das Erstellungsdatum und den Speicherort auf der Festplatte. Unterhalb dieser Angaben finden Sie eine Liste aller verfügbaren runtimes, aus der Sie ein beliebiges auswählen können, um es als "laufendes" runtime für die Kompilierung Ihrer Projekte zu verwenden.

Um ein anderes runtime zu verwenden, suchen Sie es einfach in der Liste und doppelklicken Sie auf . Wenn runtime noch nicht heruntergeladen und installiert wurde, werden Sie aufgefordert, dies zu tun. Nach der Installation werden Sie aufgefordert, IDE zu schließen und neu zu starten, damit das neue runtime initialisiert werden kann.

Diese Liste enthält alle verfügbaren runtimes, ihre Versionsnummer, das Veröffentlichungsdatum und - wenn sie installiert sind - den Pfad, in dem sie installiert wurden. Auf der rechten Seite werden auch die verschiedenen Statussymbole angezeigt, die wie folgt lauten:

Dies markiert die aktuelle runtime, die für die Verwendung bei der Erstellung Ihrer Projekte ausgewählt wurde.
Wenn Sie darauf klicken, wird der Download und die Installation der ausgewählten runtime gestartet (eine Statusleiste wird oben rechts in der IDE angezeigt).
Wenn Sie auf diese Schaltfläche klicken, wird die ausgewählte runtime von Ihrer Festplatte entfernt.

Sie können die Liste runtime jederzeit über den RSS-Feed aktualisieren, indem Sie auf die Schaltfläche "Aktualisieren" klicken, und Sie können Ihre Festplatte von alten runtimes bereinigen, indem Sie auf die Schaltfläche "Jetzt bereinigen Runtimes " klicken.
