Tastenpräferenzen neu definieren

Im Abschnitt " Tasten neu definieren " der Voreinstellungen können Sie die Zuordnungen für die verschiedenen Tastenkombinationen in der GameMaker-IDE ändern.

Suche

Oben befindet sich die Suchleiste, mit der Sie nach bestimmten Befehlen suchen können, indem Sie einen Namen (z. B. "Kopieren"), eine Tastenkombination (z. B. "STRG+C") oder einen Ort (z. B. "Texteditor") eingeben.

Wenn Sie die Schaltfläche neben dem Suchfeld aktivieren, können Sie die Suche durch die Eingabe einer Tastenkombination durchführen. Alle Tasten, die Sie drücken, während diese Funktion aktiviert ist, werden als Tastenkürzel anstelle von normalem Text eingegeben.

Neben der Suchleiste befindet sich das Menü Filter, das in einem eigenen Abschnitt weiter unten auf dieser Seite beschrieben wird.

Befehlsliste

Die "Befehlsliste" zeigt alle verfügbaren Tastenkombinationen an, die neu zugeordnet werden können, und ist in verschiedene Kategorien unterteilt, die auf den Positionen dieser Tastenkombinationen basieren. Wenn Sie auf eine beliebige Stelle in dieser Liste klicken, werden die Befehle angezeigt, die an dieser Stelle verwendet werden können:

Die Informationen für jeden Befehl sind in zwei Spalten unterteilt:

  • Befehl: Dies ist der Name des Befehls/Kurzbefehls und beschreibt die Aktion, die er ausführt.
  • Tasten: Hier wird die aktuelle Tastaturbelegung für den Befehl angezeigt. Sie können auf ein beliebiges Element in dieser Spalte doppelklicken, um den Befehl neu zuzuweisen, indem Sie einfach die entsprechende Tastenkombination auf der Tastatur drücken (sie müssen gleichzeitig gehalten werden, ähnlich wie bei der tatsächlichen Verwendung des Tastaturkürzels). Sobald Sie Ihre neue Tastenkombination loslassen, wird sie registriert und auf den Befehl angewendet. Wenn Sie während der Bearbeitung eines Befehls drücken, verlassen Sie den Bearbeitungsmodus und ändern den ausgewählten Befehl nicht.

Sie können die Sortierung der Befehlsliste ändern, indem Sie auf auf eine Spaltenüberschrift (entweder "Befehl" oder "Schlüssel") klicken, um die Liste nach den Werten dieser Spalte zu sortieren, wobei bei jedem Klick zwischen auf- und absteigender Sortierung gewechselt wird.

Clash-Fenster

Wenn Sie versuchen, eine Tastenkombination festzulegen, die bereits für andere Befehle verwendet wird, wird ein spezielles "Clash"-Fenster angezeigt:
In diesem Fenster werden die Position und der Name des Befehls angezeigt, der bereits die von Ihnen gewünschte Tastenkombination verwendet. Falls die Tastenkombination von mehreren Befehlen verwendet wird, werden alle Befehle in einer Liste angezeigt.

Hier haben Sie zwei Möglichkeiten: "Abbrechen" und "Überschreiben (ALLE) und neu zuweisen":

  • Durch Drücken von "Abbrechen" wird das Clash-Fenster geschlossen und es werden keine Änderungen an den Befehlen vorgenommen.
  • Wenn Sie auf "Überschreiben (ALLE) und neu zuweisen" drücken, wird die neue Tastenkombination dem ausgewählten Befehl zugewiesen und die Bindungen aller anderen Befehle, die zuvor dieselbe Kombination verwendet haben, gelöscht .

Nach der Verwendung von "Überschreiben und neu zuweisen" markiert GameMaker den überschriebenen Befehl in der Befehlsliste, so dass Sie ihm eine neue Tastenkombination zuweisen können. Falls mehrere Befehle überschrieben wurden, wird der Filter so geändert, dass nur diese Befehle angezeigt werden, damit Sie ihnen neue Tastenkombinationen zuweisen können. Dieser Filter kann durch Klicken auf "Löschen" wieder aufgehoben werden.

Wenn Sie eine andere Bindung für den ausgewählten Befehl verwenden möchten, können Sie den Abschnitt "Vorgeschlagene freie Tasten" erweitern, um eine Liste von Tastenkombinationen zu sehen, die nirgendwo auf IDE verwendet werden. Wenn Sie auf "Verwenden" neben einer dieser Kombinationen klicken, wird diese dem ausgewählten Befehl zugewiesen und der Konflikt gelöst.

Menü "Filter

Im Menü Filter können Sie festlegen, welche Art von Befehlen in der Befehlsliste erscheinen soll. Oben haben Sie zwei Optionen:

  • "Ungebundene Befehle anzeigen": Wenn Sie diese Option aktivieren, werden alle Befehle angezeigt, die ungebunden sind, d.h. deren Bindung aufgehoben wurde. Ungebundene Befehle können nicht verwendet werden, da ihnen keine Tastenkombination zugewiesen ist.
  • "Nur neu definierte Befehle anzeigen": Wenn Sie diese Option aktivieren, werden nur die Befehle angezeigt, die manuell geändert wurden, d.h. deren Bindungen sich von den Standardbindungen unterscheiden.

Darunter gibt es zahlreiche Optionen, mit denen Sie die in der Befehlsliste angezeigten Orte festlegen können. Standardmäßig ist "Alle Orte" markiert, d. h., Sie sehen Befehle von allen Orten auf IDE. Wenn Sie die Markierung von "Alle Orte" aufheben, werden alle anderen Einträge darunter deaktiviert, so dass Sie die Befehle, die Sie sehen möchten, manuell auswählen können; wenn Sie z. B. die Markierung von "Alle Orte" aufheben und dann "Bildeditor" markieren, werden Ihnen nur Befehle angezeigt, die im Bildeditor verwendet werden können.

Löschen von Befehlen

Mit einem Doppelklick auf eine bestehende Tastenkombination können Sie diese ändern. Während Sie dies tun, können Sie auf die Schaltfläche "Löschen" unterhalb der Befehlsliste klicken, um die Bindung für diesen Befehl zu löschen.

Wenn Sie versehentlich einige Tastenkombinationen überschrieben haben und diese auf die Standardeinstellungen zurücksetzen möchten, können Sie auf die Schaltfläche "Standardwerte wiederherstellen" oben im Fenster "Einstellungen" klicken, um alle Befehle auf die Standardbelegung zurückzusetzen, oder die Seite " Tastenkombinationen" aufrufen, um die Standardbelegung zu überprüfen und Ihre Befehle manuell zu ändern.
