Windows-Einstellungen

Die Einstellungen von Windows bieten die folgenden Optionen:

  • Visual Studio-Speicherort: Diese Option verweist GameMaker auf die Microsoft-Installation Visual Studio auf dem Erstellungs-PC.
  • Standard-Paketierungsart: Wenn Sie ein endgültiges Windows ausführbares Paket erstellen, können Sie wählen, ob Sie ein Installationsprogramm (mit NSIS) oder eine komprimierte Datei ZIP die das Spiel und alle zusätzlichen Dateien enthält. Standardmäßig wird Ihnen beim Kompilieren ein Dialog angezeigt, in dem Sie zwischen diesen beiden Optionen wählen können, aber Sie können diese Einstellung so wählen, dass immer die eine oder die andere Option verwendet wird. Beachten Sie, dass wenn diese Option auf Dialog anzeigen" eingestellt ist, im eigentlichen Dialogfenster, das beim Kompilieren erscheint, die Einstellung "Verpackungsoption merken" auf die von Ihnen gewählte Option gesetzt wird.
