HTML5-Einstellungen

Die HTML5-Einstellungen bieten die folgenden Optionen:

  • Standard-Paketierungsart: Bei der Erstellung eines endgültigen ausführbaren HTML5-Pakets können Sie zwischen "Paket als lose Dateien " - bei dem einfach alle erforderlichen Dateien in einem Ordner erstellt werden - und " Paket als Zip " - bei dem eine komprimierte Datei ZIP erstellt wird, die das Spiel und alle zusätzlichen Dateien enthält - wählen. Standardmäßig können Sie wählen, welche dieser beiden Optionen Sie beim Kompilieren wählen möchten, aber Sie können diese Einstellung so wählen, dass immer die eine oder die andere verwendet wird. Beachten Sie, dass, wenn diese Option auf Dialog anzeigen eingestellt ist, im eigentlichen Dialogfenster, das beim Kompilieren erscheint, die Auswahl von "Verpackungsoption merken" diese Voreinstellung auf die von Ihnen gewählte Option setzt.
  • Standard-Web Server Port: Hier können Sie den Standard-Port für das HTML5-Web server ändern. Der Standardwert ist 51264.
  • Standard Web Server Adresse: Hier können Sie die Standard-Webadresse server für das HTML5-Web server ändern. Der Standardwert ist 127.0.0.1.
  • Standard Web Server Erlaubte IPs: Hier können Sie alle zulässigen IPs für das Web server eingeben, die dann als Standard für alle über den Target Manager hinzugefügten Geräte festgelegt werden.

WICHTIG Wenn Sie ein bestimmtes Zielgerät (z. B. einen Browser) verwenden, werden die für dieses Gerät festgelegten IP- und Port-Werte anstelle dieser Einstellungen verwendet. Überprüfen Sie den Gerätemanager, um eine Liste Ihrer Geräte mit deren Details zu sehen.
