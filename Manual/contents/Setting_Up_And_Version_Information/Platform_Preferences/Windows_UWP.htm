Windows UWP-Einstellungen

Die Einstellungen von Windows UWP bieten die folgenden Optionen:

  • Visual Studio-Speicherort: Diese Option verweist GameMaker auf die Microsoft-Installation Visual Studio auf dem Erstellungs-PC.
  • Win UWP Tools-Speicherort: Mit dieser Option verweist GameMaker auf den Installationspfad der UWP-Tools auf dem Build-PC.
  • Standard-Paketarchitektur: Standardmäßig wird beim Kompilieren einer ausführbaren UWP-Datei ein Dialogfenster angezeigt, in dem Sie zwischen den Architekturen x86, x64 und ARM wählen können. Sie können diese Option auf eine dieser drei Architekturen setzen, die dann standardmäßig ausgewählt wird, wenn Sie eine ausführbare Datei kompilieren.
  • Automatische Reaktion auf die Erstellung eines Speicherpakets: Im Dialogfenster, das beim Kompilieren angezeigt wird, können Sie wählen, ob ein Speicherpaket erstellt werden soll. Sie können diese Option auf "Ja" oder "Nein" setzen, die dann standardmäßig verwendet wird wenn Sie eine ausführbare Datei kompilieren wollen.
