tvOS-Einstellungen

Die Einstellungen von tvOS bieten die folgenden Optionen:

  • Automatische Reaktion auf tvOS persistent storage limitation warning: tvOS Spiele haben einen begrenzten lokalen Speicherplatz für Dateien, und wenn Sie für dieses Ziel kompilieren, wird standardmäßig eine Warnung angezeigt, um Sie an diese Tatsache zu erinnern. Sie können die Sie können die Einstellung hier ändern, um diese Warnung zu unterdrücken, indem Sie "OK" wählen. Die Standardeinstellung ist "Meldung anzeigen".
  • Bauen und Ausführen unterdrücken: Wenn Sie ein tvOS Projekt erstellen und dann ein endgültiges ausführbares Paket erstellen, wird GameMaker standardmäßig das Paket erstellen und dann versuchen, es auf allen angeschlossenen Geräten auszuführen. Durch Aktivieren dieser Option wird unterdrückt das Erstellen und Ausführen der Anwendung auf dem Gerät und erstellt lediglich ein XCode-Projekt auf dem Host-Mac.
  • macOS-Installationspfad: Der Installationspfad für die Projektdateien auf dem erstellten Mac.
  • Standard-Team-Kennung: Hier können Sie Ihre Standard-Team-Kennung hinzufügen, die Ihnen von Apple zugewiesen wurde. Diese Team-ID wird verwendet, wenn Ihre Spieldateien an Xcode gesendet werden, um die App zu erstellen, und ermöglicht es Xcode, die erforderlichen Signierzertifikate zu generieren. Beachten Sie, dass diese Einstellung standardmäßig auf alle Spiele angewendet wird, die für tvOS erstellt werden, aber sie kann projektspezifisch in den Allgemeinen tvOS Spiel-Optionen.
