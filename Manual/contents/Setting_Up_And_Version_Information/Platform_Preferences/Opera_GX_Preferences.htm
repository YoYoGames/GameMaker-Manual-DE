Opera GX-Einstellungen

Die Voreinstellungen von Opera GX bieten die folgenden Optionen:

  • Emscripten SDK Speicherort: Dies ist der Speicherort des Emscripten SDK, das für YYC-Builds für Opera GX benötigt wird. Lesen Sie Einrichten für Opera GX für Informationen zum Einrichten des SDKs.
  • Standard-Web Server Port: Hier können Sie den Standard-Port für das Opera GX-Web server ändern. Der Standardwert ist 51264.
  • Standard Web Server Adresse: Hier können Sie die Standard-Webadresse server für das Opera GX-Web server ändern. Der Standardwert ist 127.0.0.1.
  • Standard Web Server Erlaubte IPs: Hier können Sie alle zulässigen IPs für das Web server eingeben, die dann als Standard für alle über den Target Manager hinzugefügten Geräte festgelegt werden.
