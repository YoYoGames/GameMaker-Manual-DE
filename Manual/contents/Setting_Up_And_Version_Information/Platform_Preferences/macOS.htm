macOS-Einstellungen

Die Einstellungen von macOS bieten die folgenden Optionen:

  • Standard-Team-Kennung: Hier können Sie Ihre Standard-Teamkennung hinzufügen, die Ihnen von Apple zugewiesen wurde. Diese Team-Kennung wird verwendet, wenn Ihre Spieldateien an Xcode gesendet werden, um die App zu erstellen, und ermöglicht Xcode, die erforderlichen Signierzertifikate zu generieren. Beachten Sie, dass diese Einstellung standardmäßig auf alle Spiele angewendet wird, die für macOS erstellt werden. Sie kann jedoch projektspezifisch in den allgemeinen macOS Spieloptionen überschrieben werden.
  • Automatische Paketierungswahl: Hier können Sie eine Standardeinstellung für die Erstellung einer ausführbaren Datei für das Ziel macOS festlegen. Die Option "Dialog anzeigen" zeigt vor der Erstellung einer ausführbaren Datei immer ein Dialogfeld an, in dem Sie zwischen DMG und Zip wählen können, aber Sie können diese Option in den Voreinstellungen auf eine der beiden Optionen setzen, um dieses Dialogfeld zu überspringen. Beachten Sie, dass durch die Aktivierung der Option "Paketoption merken" im Dialogfeld auch diese Option auf den ausgewählten Pakettyp gesetzt wird.
