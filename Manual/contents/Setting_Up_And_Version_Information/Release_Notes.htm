Anmerkungen zur Veröffentlichung

Zu jeder Version von GameMaker gibt es einen eigenen Satz von Versionshinweisen. Diese enthalten detaillierte Informationen über die vorgenommenen Änderungen sowie eine vollständige Liste aller behobenen Fehler. Dies ist ein unschätzbares Hilfsmittel, um zu entscheiden ob Sie Ihre aktuelle Version aktualisieren sollten, oder wenn Sie nach einer neuen Version einen Fehler finden, der ein Fehler sein könnte.

Die Versionshinweise sind in zwei verschiedene Sätze aufgeteilt, da sowohl die IDE als auch die Runtimes unabhängig voneinander aktualisiert werden können. Die aktuellen Versionshinweise für IDE finden Sie unter dem nachstehenden Link:

  • IDE-Versionshinweise

Die aktuellen Release Notes für die Runtimes finden Sie unter dem unten stehenden Link:

  • Runtime Release Notes
