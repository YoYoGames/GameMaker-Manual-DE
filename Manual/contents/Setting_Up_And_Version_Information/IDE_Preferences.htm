IDE-Einstellungen

Das Fenster "Einstellungen" ist ein wesentlicher Teil von GameMaker, da Sie hier Ihre Erfahrungen mit IDE einrichten und anpassen können. Außerdem können Sie hier alle spezifischen platform SDKs usw. einrichten. In diesem Abschnitt werden die IDE -spezifischen Bereiche des Einstellungsfensters behandelt - für die einzelnen Ziele platforms lesen Sie bitte den Abschnitt über die Zieleinstellungen Platform . Beachten Sie, dass Sie nach dem Ändern von Einstellungen in diesem Fenster auf die Schaltfläche Apply (Übernehmen) klicken müssen, damit sie wirksam werden.

Wenn Sie hier die Optionen ändern, um den GameMaker an Ihren Geschmack anzupassen, und die Einstellungen sichern möchten, müssen Sie diesen Ordner unter Windows aufrufen:

%AppData%/GameMaker/<your_account_name>/

Oder diesen Ordner auf dem Mac:

~/.config/GameMaker

Kopieren Sie dann die beiden folgenden Dateien an einen sicheren Ort:

  • local_settings.json
  • cloud_settings.json

Wenn Sie jedoch die von Ihnen geänderten Einstellungen zurücksetzen möchten, müssen Sie oben im Fenster auf die Schaltfläche Standardwerte wiederherstellen klicken. Beachten Sie, dass dadurch nur die Standardeinstellungen für den aktuell bearbeiteten Bereich wiederhergestellt werden.
