netzwerk_verbinden_async

Mit dieser Funktion können Sie eine Verbindungsanfrage an server senden. Die Funktion benötigt die Socket-ID, über die die Verbindung hergestellt werden soll (siehe network_create_socket()) und erfordert die Angabe der IP-Adresse, zu der eine Verbindung aufgebaut werden soll (eine Zeichenkette) sowie den Port, über den die Verbindung hergestellt werden soll. Wenn die Verbindung fehlschlägt, wird ein Wert kleiner als 0 zurückgegeben. Die Verbindung verwendet ein spezielles Protokoll, das sicherstellt, dass sich nur GameMaker-Spiele miteinander verbinden, aber wenn Sie eine Verbindung zu einem server herstellen, das kein GameMaker-Spiel ist, können Sie network_connect_async_raw(). Beachten Sie, dass diese Funktion asynchron ist und ein Asynchronous Networking-Ereignis vom Typ network_type_non_blocking_connect erzeugt.

Syntax:

network_connect_async(socket, url, port);

ArgumentTyp Beschreibung
socket The id of the socket to use.
url The URL or IP to connect to (a string).
port The port to connect to.

Rückgabe:

Beispiel:

client = network_create_socket(network_socket_tcp);
network_connect_async(client, "192.134.0.1", 6510);

Die obige code erstellt einen neuen TCP-Socket und versucht dann, über diesen eine asynchrone Verbindung zu der angegebenen IP-Adresse an Port 6510 herzustellen.
