netzwerk_erzeugen_socket_ext

Diese Funktion wird verwendet, um einen neuen Client-Socket für Ihr Spiel zur Kommunikation über das Netzwerk zu erstellen. Sie müssen den Socket-Typ definieren (siehe die Liste der Konstanten unten) und einen zu verwendenden Port angeben. Die Funktion gibt eine eindeutige ID zurück, die in allen weiteren Funktionsaufrufen für diesen Socket verwendet werden sollte, oder einen Wert kleiner als 0, wenn die Verbindung fehlschlägt.

Syntax:

network_create_socket_ext(protocol, port);

Argument Typ Beschreibung
protocol Steckdosentyp Konstant The network protocol to use
port Real The port to use

Rückgabe:

Network Socket ID

Beispiel:

client = network_create_socket_ext(network_socket_udp, 6510);

Die obige code erstellt einen neuen UDP-Socket an Port 6510.
