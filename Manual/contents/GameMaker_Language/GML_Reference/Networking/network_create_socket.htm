network_create_socket

Diese Funktion wird verwendet, um einen neuen Client-Socket für Ihr Spiel zur Kommunikation über das Netzwerk zu erstellen. Sie müssen den Socket-Typ definieren (siehe die Liste der Konstanten unten) und die Funktion gibt eine eindeutige ID für diesen Socket zurück, die in allen weiteren Funktionsaufrufen für diesen Socket verwendet werden sollte, oder einen Wert kleiner als 0, wenn die Verbindung fehlschlägt.

TIPP Sie können die IP "127.0.0.1" verwenden, um sich wieder mit demselben Gerät zu verbinden, auf dem das Spiel läuft.

Syntax:

network_create_socket(type);

Argument Typ Beschreibung
type Steckdosentyp Konstant The type of socket connection to create (see the constants listed above).

Rückgabe:

Network Socket ID

Beispiel:

if os_browser == browser_not_a_browser
{
    client = network_create_socket(network_socket_tcp);
    network_connect( client, "192.134.0.1", 6510 );
}
else
{
    client = network_create_socket(network_socket_ws);
    network_connect_raw_async( client, "192.134.0.1", 6520 );
}

Das obige code prüft, ob das Spiel in einem Browser läuft oder nicht, und erstellt einen neuen TCP- oder Web-Socket, bevor es versucht, über diesen eine Verbindung mit der angegebenen IP-Adresse am angegebenen Port herzustellen.
