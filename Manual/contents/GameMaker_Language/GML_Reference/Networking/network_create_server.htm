network_create_server

Diese Funktion wird verwendet, um ein neues Netzwerk server für Ihr Spiel zu erstellen, das eines der zulässigen Verbindungsprotokolle verwendet (siehe die unten aufgeführten Konstanten). Sie geben den Typ server an, dann einen Port, der verwendet werden soll, und schließlich die Anzahl der maximalen Verbindungen, die zu einem beliebigen Zeitpunkt zum server zugelassen werden sollen (beachten Sie, dass diese Zahl Ihnen überlassen ist, aber zu viele verbundene Clients werden das Netzwerk sättigen oder die CPU des Geräts wird nicht in der Lage sein, die Verarbeitung dieser Anzahl von Spielern zu bewältigen, also verwenden Sie sie mit Vorsicht). Die Funktion gibt eine eindeutige ID zurück, die in einer Variablen gespeichert und zur Identifizierung von server in allen weiteren Netzwerkfunktionen verwendet werden sollte, oder einen Wert von weniger als 0, wenn die Verbindung fehlschlägt.

Syntax:

network_create_server(type, port, max_client);

Argument Typ Beschreibung
type Steckdosentyp Konstant The type of server to create (see the constants listed above).
port Real The port that the server will use.
max_client Real The maximum number of clients that can connect at once.

Rückgabe:

Network Socket ID

Beispiel:

var port = 6510;
server = network_create_server(network_socket_tcp, port, 32);
while (server < 0 && port < 65535)
{
    port++
    server = network_create_server(network_socket_tcp, port, 32);
}

Die obige code versucht, eine server mit TCP über Port 6510 zu erstellen. Wenn dieser Port nicht verfügbar ist, wird loop durch die Ports durchsucht, um einen zu finden, der verfügbar ist.
