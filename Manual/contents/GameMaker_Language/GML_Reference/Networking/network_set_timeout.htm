netzwerk_zeitüberschreitung_setzen

Mit dieser Funktion können Sie den Timeout für das Lesen und Schreiben von Daten auf/von server über den angegebenen Socket festlegen. Beachten Sie, dass die Zeitüberschreitung keine Art von Ereignis erzeugt, so dass Sie sich selbst um Zeitüberschreitungen kümmern müssen, indem Sie z. B. Alarme verwenden. Beachten Sie, dass sich dieser Wert nur auf das Senden und Empfangen von Daten auswirkt. Wenn Sie den Timeout-Wert für die Verbindung ändern möchten, sollten Sie die Funktion network_set_config().

Syntax:

network_set_timeout(socket, read_timeout, write_timeout);

ArgumentTyp Beschreibung
socket The id of the socket to use.
read_timeout The milliseconds in which a transfer from a server will timeout.
write_timeout The milliseconds in which a transfer to a server will timeout.

Rückgabe:

Beispiel:

network_set_timeout(sock, 3000, 3000);

Die obige code setzt den Timeout für das Lesen und Schreiben von Daten über den in der Variable "sock" indizierten Socket auf 3 Sekunden.
