network_connect_raw_async

Mit dieser Funktion können Sie eine Verbindungsanfrage an server senden. Die Funktion nimmt die Socket-ID an, über die die Verbindung hergestellt werden soll (siehe network_create_socket()) und erfordert die Angabe der IP-Adresse, zu der eine Verbindung aufgebaut werden soll (eine Zeichenkette) sowie den Port, über den die Verbindung hergestellt werden soll, anzugeben. Schlägt die Verbindung fehl, wird ein Wert kleiner 0 zurückgegeben. Der Unterschied zwischen dieser Funktion und network_connect_async()ist, dass diese Funktion eine Verbindung zu jeder beliebigen server herstellen kann und nichts mit den Rohdaten macht, was bedeutet, dass Sie die Protokolle auf der Seite server selbst implementieren müssen. Beachten Sie, dass diese Funktion asynchron ist und ein Asynchronous Networking-Ereignis vom Typ network_type_non_blocking_connect erzeugt.

Syntax:

network_connect_raw_async(socket, url, port);

ArgumentTyp Beschreibung
socket The id of the socket to use.
url The URL or IP to connect to (a string).
port The port to connect to.

Rückgabe:

Beispiel:

client = network_create_socket(network_socket_tcp);
network_connect_raw_async(client, "www.macsweeneygames.com", 6510);

Die obige code erstellt einen neuen TCP-Socket und versucht dann, über diesen eine "rohe" asynchrone Verbindung zu der angegebenen URL an Port 6510 herzustellen.
