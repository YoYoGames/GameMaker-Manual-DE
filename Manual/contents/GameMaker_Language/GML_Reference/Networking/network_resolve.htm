Diese Funktion gibt die IP-Adresse der angegebenen URL zurück.

Syntax:

network_resolve(url);

ArgumentTyp Beschreibung
url The URL to get the IP of (a string).

Rückgabe:

Beispiel:

game_ip = network_resolve("www.macsweeneygames.com");

Die obige code gibt die IP-Adresse der angegebenen URL zurück und speichert sie in der Variablen "game_ip".
