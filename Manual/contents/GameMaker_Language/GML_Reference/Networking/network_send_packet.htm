netzwerk_senden_pakete

Mit dieser Funktion können Sie ein Daten "paket" über das Netzwerk senden. Die Funktion benötigt die Socket-ID, über die eine Verbindung hergestellt werden soll, und dann die Puffer-ID, die die zu sendenden Daten enthält (weitere Informationen zu buffers finden Sie unter Referenz - Puffer), und schließlich die Größe (in Bytes) des Datenpakets. Die mit dieser Funktion gesendeten Pakete werden so formatiert, dass das GameMaker-Spiel, das die Daten empfängt, die Pakete korrekt "aufteilen" kann, und die Funktion gibt die Anzahl der gesendeten Datenbytes zurück oder eine Zahl kleiner als 0, wenn das Senden fehlgeschlagen ist. Es ist erwähnenswert, dass die endgültige Größe der gesendeten Daten, die von dieser Funktion zurückgegeben wird, auch die GameMaker Header-Informationen enthält, die zusätzlich 12 Byte betragen.

Syntax:

network_send_packet(socket, bufferid, size);

ArgumentTyp Beschreibung
socket The id of the socket to use.
bufferid The id of the buffer to get the data from.
size The size (in bytes) of the data.

Rückgabe:

Real

Beispiel:

buff = buffer_load("player_save.dat");
network_send_packet(sock, buff, buffer_get_size(buff));

Die obigen Informationen laden eine zuvor gespeicherte buffer in den Speicher und geben die buffer id zurück, die in der Variablen "buff" gespeichert wird. Diese vollständige buffer wird dann als Paket über das Netzwerk unter Verwendung des Sockets, der durch die Variable "sock" identifiziert wird, gesendet.
