Binäre Dateien

In seltenen Fällen kann es erforderlich sein, Daten aus Binärdateien zu lesen. Diese Funktionen können für diesen Zweck verwendet werden, obwohl wir im Allgemeinen empfehlen, dass Sie die speziellen Pufferfunktionen zum Erstellen, Speichern und Laden von Binärdaten verwenden, anstatt diese Funktionen.

HINWEIS: Diese Funktionen funktionieren nicht, wenn das Ziel module HTML5 ist.

Hierfür gibt es die folgenden Low-Level-Routinen:

  • file_bin_open
  • file_bin_rewrite
  • file_bin_close
  • file_bin_size
  • file_bin_position
  • file_bin_seek
  • file_bin_write_byte
  • file_bin_read_byte
