file_bin_close

Schließt eine zuvor geöffnete Binärdatei. Sie geben den Wert der Datei-ID an, wie er von der Funktion file_bin_open().

HINWEIS: Diese Funktionen funktionieren nicht, wenn das Ziel module HTML5 ist.

Syntax:

file_bin_close(binfile);

ArgumentTyp Beschreibung
binfile The ID of the file to close.

Rückgabe:

Beispiel:

file = file_bin_open("myfile.bin", 2);
file_bin_rewrite(file);
file_bin_close(file);

Dies würde eine Datei aus dem gleichen Verzeichnis wie das Spiel öffnen und ihren Index der Variablen "file" zuweisen. Dann wird die Datei neu geschrieben (die Daten werden gelöscht) und wieder geschlossen.
