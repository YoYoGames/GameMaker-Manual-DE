file_bin_seek

Diese Funktion verschiebt die aktuelle Leseposition innerhalb der Datei an die angegebene Position. Sie geben den Wert der Datei-ID an, der von der Funktion file_bin_open()zurückgegeben wird, und um eine Datei anzuhängen, verschieben Sie die Position um die Größe der Datei vor dem Schreiben.

HINWEIS: Diese Funktionen funktionieren nicht, wenn das Ziel module HTML5 ist.

Syntax:

file_bin_seek(binfile, pos);

ArgumentTyp Beschreibung
binfile The ID of the file to read from.
pos The position in the file to move to.

Rückgabe:

Beispiel:

file = file_bin_open("myfile.bin", 2);
size = file_bin_size(file);
file_bin_seek(file, size);

Dies würde eine Datei aus dem gleichen Verzeichnis wie das Spiel öffnen und ihren Index der Variablen "file" zuweisen, dann die Größe der Datei ermitteln und die nächste Schreibposition auf diese Größe setzen.
