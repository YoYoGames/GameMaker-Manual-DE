file_bin_read_byte

Diese Funktion gibt ein Datenbyte von der aktuellen Position innerhalb der Datei mit der angegebenen Datei-ID zurück. Sie geben den Wert der Datei-ID an, wie er von der Funktion file_bin_open().

HINWEIS: Diese Funktionen funktionieren nicht, wenn das Ziel module HTML5 ist.

Syntax:

file_bin_read_byte(binfile);

ArgumentTyp Beschreibung
binfile The ID of the file to get the data from.

Rückgabe:

Beispiel:

file = file_bin_open("myfile.bin", 2);
data = file_bin_read_byte(file);
file_bin_close(file);

Dies würde eine Datei aus dem gleichen Verzeichnis wie das Spiel öffnen und ein Byte an Daten abrufen, bevor es wieder geschlossen wird.
