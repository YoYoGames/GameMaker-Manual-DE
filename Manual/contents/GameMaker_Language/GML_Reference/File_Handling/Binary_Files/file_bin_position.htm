file_bin_position

Diese Funktion gibt die aktuelle Position der Datei mit der angegebenen Datei-ID in Bytes zurück, wobei 0 die erste Position ist. Sie geben den Wert der Datei-ID an, wie er von der Funktion file_bin_open().

HINWEIS: Diese Funktionen funktionieren nicht, wenn das Ziel module HTML5 ist.

Syntax:

file_bin_position(binfile);

ArgumentTyp Beschreibung
binfile The ID of the file to get the position in.

Rückgabe:

Beispiel:

pos = file_bin_position(file);

Dadurch wird die aktuelle Position in der Variablen "pos" gespeichert.
