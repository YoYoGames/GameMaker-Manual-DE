file_bin_rewrite

Diese Funktion nimmt den Dateinamen-Handle, wie er von der Funktion file_bin_open() zurückgegeben wurde, und schreibt dann die Datei neu, wobei sie alle vorherigen Daten löscht, um mit dem Schreiben am Anfang der Datei zu beginnen.

HINWEIS: Diese Funktionen funktionieren nicht, wenn das Ziel module HTML5 ist.

Syntax:

file_bin_rewrite(binfile);

ArgumentTyp Beschreibung
binfile The ID of the file to rewrite.

Rückgabe:

Beispiel:

file = file_bin_open("myfile.bin", 2);
file_bin_rewrite(file);

Dies würde eine Datei aus dem gleichen Verzeichnis wie das Spiel öffnen und ihren Index der Variablen "file" zuweisen. Dann wird die Datei neu geschrieben (die Daten werden gelöscht).
