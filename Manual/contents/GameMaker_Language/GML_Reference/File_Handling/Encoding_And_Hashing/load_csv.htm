load_csv

Diese Funktion lädt eine Datei im CSV-Format, konvertiert sie in ein DS-Gitter und gibt den eindeutigen ID-Wert für das erstellte Gitter zurück.

Ihre CSV-Datei sollte der CSV-Spezifikation entsprechen. Wenn Ihre CSV syntax nicht korrekt ist, werden Sie beim Laden der Datei unerwartete Ergebnisse erhalten.

Syntax:

load_csv(filename)

Argument Typ Beschreibung
filename Zeichenfolge The name of the file to open (as a string)

Rückgabe:

DS Grid ID

Beispiel:

file_grid = load_csv("spreadsheet.csv");
var ww = ds_grid_width(file_grid);
var hh = ds_grid_height(file_grid);
var xx = 32;
var yy = 32;
for (var i = 0; i < ww; i++)
{
    for (var j = 0; j < hh; j++)
    {
        draw_text(xx, yy, file_grid[# i, j]);
        yy += 32;
    }
    yy = 32;
    xx += 32;
}

Die obige code öffnet die angegebene CSV-Datei und speichert das zurückgegebene DS-Gitter in der Variablen "file_grid". Dieses Raster wird dann in ein paar for lops geparst und der Inhalt auf den Bildschirm gezeichnet.
