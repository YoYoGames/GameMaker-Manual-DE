sha1_datei

In der Kryptografie ist SHA-1 eine kryptografische Hash-Funktion, die von der Nationalen Sicherheitsbehörde der Vereinigten Staaten entwickelt wurde und in mehreren weit verbreiteten Anwendungen zum Einsatz kommt, z. B. in Git, wo sie zur Überprüfung von Dateiänderungen verwendet wird, sowie in den Protokollen TLS und SSL, PGP, SSH, S/MIME und IPsec. Diese Funktion nimmt eine Eingabedatei und gibt einen 160-Bit-Message-Digest im ASCII-Format zurück, der nur für diese Datei gilt und zu einem späteren Zeitpunkt zur Integritätsprüfung verwendet werden kann.

HINWEIS: SHA-1 ist nicht völlig sicher und kann geknackt werden. Siehe diese Seite für weitere Informationen.

Syntax:

sha1_file(filename)

Argument Typ Beschreibung
filename Zeichenfolge The file to generate the sha1 hash for.

Rückgabe:

String

Beispiel:

hash = sha1_file(working_directory + "game_data.ini")

Die obige code erzeugt einen sha1-Hash für die angegebene Datei und speichert den zurückgegebenen Wert in der Variablen "hash".
