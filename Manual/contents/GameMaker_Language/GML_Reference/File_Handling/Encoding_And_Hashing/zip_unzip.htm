zip_unzip

Diese Funktion öffnet eine gespeicherte ZIP-Datei und entpackt ihren Inhalt in das angegebene Verzeichnis. Beachten Sie, dass, wenn Sie keinen vollständigen Pfad zum ZIP-Verzeichnis angeben, das aktuelle Stammverzeichnis des Laufwerks verwendet wird, und wenn Sie es in einem relativen Pfad zum Arbeitsverzeichnis des Spiele-Bundles platzieren möchten, sollten Sie die Variable working_directory als Teil des Pfades verwenden (relative Pfade mit "." oder ".." funktionieren nicht wie erwartet und sollten daher vermieden werden). Beachten Sie auch, dass die Zip-Datei entweder Teil des Spielepakets sein muss (d.h. eine eingeschlossene Datei) oder mit http_get_file() in den Speicherbereich heruntergeladen wurde.

Die Funktion gibt einen Wert zurück, der die Anzahl der extrahierten Dateien angibt, oder sie gibt 0 oder weniger zurück, wenn die Extraktion fehlgeschlagen ist.

Syntax:

zip_unzip(zip_file, target_directory)

ArgumentTyp Beschreibung
zip_file The zip file to open
target_directory The target directory to extract the files to

Rückgabe:

Beispiel:

var num = zip_unzip("/downloads/level_data.zip", working_directory + "extracted/");
if num <= 0
{
    show_debug_message("Extraction Failed!");
}

Die obige code öffnet die im Verzeichnis "downloads" gespeicherte Zip-Datei und extrahiert ihren Inhalt in das Verzeichnis "extracted" (wobei dieses Verzeichnis erstellt wird, falls es noch nicht existiert) und prüft dann, ob die Extraktion korrekt durchgeführt wurde.
