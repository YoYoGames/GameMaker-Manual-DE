md5_datei

In der Kryptografie ist MD5 (Message-Digest-Algorithmus 5) eine weit verbreitete kryptografische Hash-Funktion mit einem 128-Bit-Hash-Wert, die in einer Vielzahl von Sicherheitsanwendungen eingesetzt wird. Sie wird auch häufig verwendet, um die Integrität von Dateien und Zeichenketten zu überprüfen. Mit dieser Funktion wird aus einer gegebenen Datei ein eindeutiger MD5-Wert für diese Datei generiert, der dann zur späteren Verwendung gespeichert werden kann.

HINWEIS: MD5 ist nicht völlig sicher und kann geknackt werden. Siehe diese Seite für weitere Informationen.

Syntax:

md5_file(filename)

ArgumentTyp Beschreibung
filename The file to generate the MD5 hash for.

Rückgabe:

Beispiel:

hash = md5_file(working_directory + "game_data.ini")

Die obige code erzeugt einen MD5-Hash für die angegebene Datei und speichert den zurückgegebenen Wert in der Variablen "hash".
