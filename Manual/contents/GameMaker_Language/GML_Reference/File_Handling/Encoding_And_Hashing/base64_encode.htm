base64_encode

Diese Funktion wandelt eine Zeichenkette in eine im base64-Format kodierte Zeichenkette um. Dies ist ein häufig verwendetes Kodierungsschema, das für alle Medien verwendet wird, die als Text gespeichert oder über das Internet übertragen werden müssen, und das die Ausgabe für das menschliche Auge unlesbar macht. das menschliche Auge unlesbar.

Syntax:

base64_encode(string)

ArgumentTyp Beschreibung
string The string to encode.

Rückgabe:

Beispiel:

var str, file;
str = base64_encode(game_data);
file = file_text_open_write("save.txt");
file_text_write_string(file, str);
file_text_close(file);

Die obige code wandelt die in "game_data" gespeicherte Zeichenfolge in eine base64-kodierte Zeichenfolge um, die dann in einer externen Textdatei gespeichert wird.
