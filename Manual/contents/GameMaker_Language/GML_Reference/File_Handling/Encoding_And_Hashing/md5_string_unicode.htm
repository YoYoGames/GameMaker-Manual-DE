md5_string_unicode

In der Kryptografie ist MD5 (Message-Digest-Algorithmus 5) eine weit verbreitete kryptografische Hash-Funktion mit einem 128-Bit-Hash-Wert, die in einer Vielzahl von Sicherheitsanwendungen eingesetzt wird. Sie wird auch häufig verwendet, um die Integrität von Dateien und Zeichenketten zu überprüfen. Diese Funktion nimmt eine Unicode-Zeichenkette (mit 16 Bits pro Zeichen) als Eingabe und gibt den 32-stelligen hexadezimalen MD5-Hashwert zurück, der für diese Zeichenkette eindeutig ist. Auf diese Weise können Sie einen sicheren Schlüssel erzeugen, der gespeichert und verwendet werden kann, um die Integrität der an eine externe server gesendeten (oder von ihr empfangenen) Informationen zu überprüfen.

HINWEIS Es gibt zwei Formate für die MD5-Kodierung, UTF-8 und Unicode. Beide Formate sind vorgesehen, um die Kommunikation mit verschiedenen server zu erleichtern, aber am häufigsten wird Unicode verwendet.

HINWEIS: MD5 ist nicht völlig sicher und kann geknackt werden. Siehe diese Seite für weitere Informationen.

Syntax:

md5_string_unicode(string)

ArgumentTyp Beschreibung
string The string to hash.

Rückgabe:

Beispiel:

var hash, str;
str = base64_encode(game_data);
hash = md5_string_unicode(str);
http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?hash=" + hash); http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?data=" + str);

Die obige code kodiert eine Zeichenkette mit base64 und erzeugt dann einen MD5-Hash. Schließlich werden sowohl der Hash als auch die kodierte Zeichenkette an eine server gesendet.
