Datei-Verzeichnisse

Wenn Sie mit Dateien arbeiten, werden Sie manchmal auf das Verzeichnis verweisen müssen, in dem die Dateien gespeichert sind, und GameMaker verfügt über einige spezielle Funktionen und Variablen, die sich auf die verschiedenen möglichen Verzeichnisse beziehen, die Sie benötigen könnten:

  • directory_exists
  • directory_create
  • directory_destroy
  • temp_directory
  • working_directory
  • program_directory
  • game_save_id (aus Allgemeine Spielsteuerung)

HINWEIS Sie können auf HTML5 keine Verzeichnisse erstellen oder löschen.
