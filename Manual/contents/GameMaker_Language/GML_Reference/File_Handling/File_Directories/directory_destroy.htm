verzeichnis_zerstören

Mit dieser Funktion wird ein Verzeichnis mit dem angegebenen Namen aus dem Speicherbereich entfernt.

HINWEIS: Diese Funktion funktioniert nicht mit HTML5, da Sie im lokalen Speicher des Browsers keine Verzeichnisse erstellen oder entfernen können.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht so, wie Sie es erwarten, da GameMaker in einer Sandbox läuft! Bitte lesen Sie den Abschnitt über das Dateisystem für weitere Informationen.

Syntax:

directory_destroy(dname)

ArgumentTyp Beschreibung
dname The name of the directory to remove.

Rückgabe:

Beispiel:

if directory_exists("DLC")
{
    directory_destroy("DLC");
}

Damit wird geprüft, ob das angegebene Verzeichnis im lokalen Datenordner existiert, und wenn ja, wird es entfernt.
