Programm_Verzeichnis

Damit wird das Verzeichnis zurückgegeben, in dem die ausführbare Datei des Spiels gespeichert ist. Dies ist jedoch nicht immer nützlich, insbesondere da einige Geräte die Exe-Datei aus einer *.zip-Datei ausführen, so dass dies das gleiche zurückgeben würde, unabhängig davon, wo das Spiel tatsächlich ausgeführt wird.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht so, wie Sie es erwarten, da GameMaker in einer Sandbox läuft! Bitte lesen Sie den Abschnitt über das Dateisystem für weitere Informationen.

HINWEIS: Wenn Sie sandbox deaktivieren, können Sie trotzdem nicht in dieses Verzeichnis schreiben, um mögliche Probleme mit dem Löschen benötigter Spieldateien zu vermeiden.

Syntax:

program_directory

Rückgabe:

Beispiel:

dir = program_directory;

Dadurch wird das Verzeichnis, in dem die ausführbare Datei gespeichert ist, in einer Variablen gespeichert.
