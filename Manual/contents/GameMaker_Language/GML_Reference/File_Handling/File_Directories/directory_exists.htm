directory_exists

Diese Funktion gibt true zurück, wenn das angegebene Verzeichnis existiert, oder false, wenn es nicht existiert. Der angegebene Name muss den vollständigen Pfad enthalten, nicht einen relativen Pfad, und standardmäßig können Sie nicht auf Verzeichnisse außerhalb des Spielepakets zugreifen, da alle Spiele in einer Sandbox untergebracht sind (weitere Informationen finden Sie im Abschnitt über das Dateisystem ).

Syntax:

directory_exists(dname)

ArgumentTyp Beschreibung
dname The name of the directory to look for.

Rückgabe:

Beispiel:

if directory_exists(working_directory + "Saves/")
{
    file = file_find_first(working_directory + "Saves/*.doc", fa_readonly);
}

Diese Funktion prüft, ob das angegebene Verzeichnis existiert, und wenn ja, geht sie dorthin und gibt die erste gefundene "schreibgeschützte" Datei doc zurück.
