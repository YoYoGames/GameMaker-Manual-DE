temp_Verzeichnis

Damit kann das temporäre Verzeichnis zurückgegeben werden, das für Ihr Spiel bei jedem Start erstellt wird (das Stammverzeichnis enthält nicht das endgültige "\"). Dieses Verzeichnis enthält Dateien und kann aufgerufen werden, während das Spiel läuft, aber es wird (zusammen mit allen darin enthaltenen Dateien) mit allen darin enthaltenen Dateien) entfernt, wenn das Spiel geschlossen wird.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht so, wie Sie es erwarten, da GameMaker in einer Sandbox läuft! Bitte lesen Sie den Abschnitt über das Dateisystem für weitere Informationen.

Syntax:

temp_directory

Rückgabe:

Beispiel:

ini_open(temp_directory + "\temp_ini.ini");

Dies öffnet eine ini-Datei im temporären Verzeichnis des Spiels (und erstellt sie, falls sie noch nicht vorhanden ist).
