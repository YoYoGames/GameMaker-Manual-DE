Dateiname_pfad

Diese Funktion gibt den Pfadteil des angegebenen Dateipfads zurück, einschließlich des abschließenden Backslashs.

Syntax:

filename_path(fname);

ArgumentTyp Beschreibung
fname The file name and path to use.

Rückgabe:

Beispiel:

path = filename_path(working_directory + "Test.ini");

Die obige code ruft den Pfad (als String) der angegebenen Datei ab.
