Dateiname_name

Diese Funktion gibt den Namensteil der angegebenen Datei zurück, mit der Erweiterung, aber ohne den Pfad.

Syntax:

filename_name(fname);

ArgumentTyp Beschreibung
fname The file to use.

Rückgabe:

String

Beispiel:

name = filename_name(file_find_first("C:/Games/*.doc", 0));

Die obige code ruft den Namen (als Zeichenkette) der ersten im angegebenen Verzeichnis gefundenen Datei vom Typ "doc" ab.
