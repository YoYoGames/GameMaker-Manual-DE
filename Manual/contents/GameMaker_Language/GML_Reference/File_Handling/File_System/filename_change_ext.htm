Dateiname_ändern_ext

Diese Funktion gibt den angegebenen Dateinamen zurück, wobei die Erweiterung (einschließlich des Punktes) in die neue Erweiterung geändert wird. Wenn Sie eine leere Zeichenkette als neue Erweiterung verwenden, können Sie den Erweiterungsteil ganz entfernen.

Syntax:

filename_change_ext(fname, newext);

ArgumentTyp Beschreibung
fname The file to use.
newext The new extension to use.

Rückgabe:

Beispiel:

ext = filename_change_ext(file_find_first(working_directory + "*.*", 0), "");

Die obige code ruft den Namen der Datei (als Zeichenkette) ab, wobei der Erweiterungsteil entfernt wird.
