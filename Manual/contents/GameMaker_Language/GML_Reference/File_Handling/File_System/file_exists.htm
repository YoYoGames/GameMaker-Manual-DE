file_exists

Diese Funktion gibt true zurück, wenn die angegebene Datei existiert, und false, wenn sie nicht existiert. Beachten Sie, dass die Funktion nur zum Überprüfen lokaler Dateien verwendet werden kann, nicht aber von Dateien, die auf einem entfernten server gespeichert sind.

Syntax:

file_exists(fname);

ArgumentTyp Beschreibung
fname The name of the file to check for.

Rückgabe:

Beispiel:

if file_exists("level.txt")
{
    file = file_text_open_read("level.txt");
}

Damit wird geprüft, ob eine Datei vorhanden ist, und wenn ja, wird sie zum Lesen geöffnet.
