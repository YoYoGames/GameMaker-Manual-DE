dateiname_ext_speichern

Diese Funktion öffnet einen Dialog und fragt den Spieler nach einem Dateinamen, der mit dem angegebenen Filter gespeichert werden soll. Der Filter hat die Form "Name1|Maske1|Name2|Maske2|...", wobei die Maske die verschiedenen Optionen mit einem Semikolon dazwischen enthalten kann und ein "*" für eine beliebige Zeichenfolge stehen kann. Zum Beispiel: "bitmaps|*.bmp;*.wmf", würde den Benutzer auffordern, den Namen einer Bitmap-Datei aus dem System auszuwählen, aber nur solche, die die beiden angegebenen Erweiterungen haben (bmp und png). Diese Funktion speichert die Datei selbst nicht! Sie gibt nur eine Zeichenkette mit dem vollständigen Namen und Pfad der Datei zurück. Wenn der Benutzer auf "Abbrechen" drückt, wird eine leere Zeichenkette "" zurückgegeben. Sie können auch ein Verzeichnis angeben, in dem die Suche beginnen soll, und eine Überschrift, die am oberen Rand des Fensters anstelle der Standardüberschrift angezeigt werden soll.

Wichtig bei der Verwendung dieser Funktion ist, dass sie Ihnen für die Dauer Ihres Spiels bestimmte Rechte für diese Datei gewährt. Sobald Sie also den Dateipfad vom Benutzer erhalten haben, können Sie immer wieder darauf zugreifen, ohne fragen zu müssen, und die Berechtigung gilt für das Lesen und Schreiben gleichermaßen.

HINWEIS: Diese Funktion ist nur für die Ziele Windows und macOS gültig.

Syntax:

get_save_filename_ext(filter, fname, directory, caption);

ArgumentTyp Beschreibung
filter The file type filter and/or name.
fname The suggested filename to use.
directory The directory to start the search from.
caption The caption for the window.

Rückgabe:

Beispiel:

var file;
file = get_save_filename_ext("screenshot|*.png", "", working_directory, "Save a level here");
if file != ""
{
    screen_save(file);
}

Die obige code fordert den Benutzer auf, eine Bilddatei auszuwählen, die im Arbeitsverzeichnis gespeichert werden soll, und prüft dann den zurückgegebenen Dateinamen und den Pfad. Wenn es sich nicht um eine leere Zeichenfolge handelt, wird ein Bildschirmfoto erstellt und unter dem angegebenen Namen gespeichert.
