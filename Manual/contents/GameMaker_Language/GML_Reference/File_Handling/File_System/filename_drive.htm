Dateiname_Laufwerk

Diese Funktion gibt die Laufwerksinformationen des Dateinamens zurück.

Syntax:

filename_drive(fname);

ArgumentTyp Beschreibung
fname The file to use.

Rückgabe:

Beispiel:

drive = filename_drive(file_find_first(working_directory + "*.doc", 0));

Die obige code ruft die Laufwerksinformationen (als String) der angegebenen Datei ab.
