Dateisystem

In GameMaker gibt es eine Reihe von Funktionen, mit denen Sie Dateien aller Art auf unterschiedliche Weise manipulieren können:

  • file_exists
  • file_delete
  • file_rename
  • file_copy
  • file_find_first
  • file_find_next
  • file_find_close
  • file_attributes

Mit den folgenden Funktionen können Sie Dateinamen ändern:

HINWEIS Diese Funktionen arbeiten nicht mit den eigentlichen Dateien! Sie bearbeiten nur die Zeichenketten, die sich auf die verschiedenen Pfadelemente beziehen, und sollten in Verbindung mit den anderen Dateifunktionen verwendet werden.

  • filename_name
  • filename_path
  • filename_dir
  • filename_drive
  • filename_ext
  • filename_change_ext

Schließlich gibt es noch einige spezielle Funktionen zur Ermittlung von Dateinamen, die zum Laden und Speichern von Dateien mit Dateityp-Filtern verwendet werden können. Diese Funktionen öffnen ein Explorer-Fenster, was dazu führt, dass das Spiel den Fokus verliert. Das bedeutet, dass alle Eingaben, von denen das Spiel abhängt, möglicherweise nicht korrekt registriert werden. Daher empfehlen wir, dass Sie (wenn Sie diese Funktionen in einem Tastaturereignis verwenden) das Ereignis "Taste hoch" und nicht das Ereignis "Taste gedrückt" oder "Taste runter" verwenden.

HINWEIS Diese Funktionen gelten nur für Windows und macOS.

WARNUNG Diese Funktionen funktionieren möglicherweise nicht so, wie Sie es erwarten, da GameMaker in einer Sandbox arbeitet! Bitte lesen Sie den Abschnitt über das Dateisystem für weitere Informationen.

  • get_open_filename
  • get_open_filename_ext
  • get_save_filename
  • get_save_filename_ext
