datei_attribute

Mit dieser Funktion können Sie die Attribute einer bestimmten Datei überprüfen. Sie können die folgenden Konstanten addieren, um den gewünschten Dateityp zu ermitteln:

Konstante Beschreibung
fa_readonly Schreibgeschützte Dateien
fa_hidden Versteckte Dateien
fa_sysfile System-Dateien
fa_volumeid Volume-id-Dateien
fa_directory Verzeichnisse
fa_archive Archivierte Dateien

HINWEIS: Dies ist eine Funktion, die nur auf Windows verfügbar ist.

Syntax:

file_attributes(fname, attr);

Argument Typ Beschreibung
fname Zeichenfolge The name of the file to check.
attr Datei-Attribut Konstante The attributes to check for.

Rückgabe:

Boolean

Beispiel:

if !file_attributes(file, fa_hidden)
{
    file_delete(file);
}

Damit wird geprüft, ob eine Datei versteckt ist oder nicht, und wenn nicht, wird sie gelöscht.
