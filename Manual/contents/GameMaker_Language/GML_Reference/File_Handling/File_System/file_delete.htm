datei_löschen

Diese Funktion löscht die angegebene Datei aus dem System. Es ist zu beachten, dass diese Funktion nur die Dateien löscht, die GameMaker erstellen und parsen kann: ini-Dateien, Textdateien und Binärdateien oder Dateien, die zum Speichern von im Spiel erstellten Ressourcen wie sprites oder Oberflächen erstellt wurden. Es werden jedoch keine anderen Dateien gelöscht. Die Funktion gibt außerdem true zurück, wenn die Datei erfolgreich gelöscht wurde, oder false, wenn dies nicht der Fall ist.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht so, wie Sie es erwarten, da GameMaker in einer Sandbox arbeitet! Weitere Informationen finden Sie im Abschnitt über Dateisystembeschränkungen.

Syntax:

file_delete(fname);

ArgumentTyp Beschreibung
fname The name of the file to delete.

Rückgabe:

Beispiel:

if file_exists("level.txt")
{
    file_delete("level.txt");
}

Dabei wird nach einer Datei gesucht, und wenn sie existiert, wird sie gelöscht.
