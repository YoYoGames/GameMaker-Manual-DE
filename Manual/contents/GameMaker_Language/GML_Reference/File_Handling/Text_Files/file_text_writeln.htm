file_text_writeln

Mit dieser Funktion können Sie eine neue Zeile in eine geöffnete Textdatei schreiben. Auf diese Weise können Sie Zeilen überspringen oder Informationen zeilenweise schreiben (siehe Beispiel code unten).

Syntax:

file_text_writeln(fileid);

ArgumentTyp Beschreibung
fileid The id of the file to edit.

Rückgabe:

Beispiel:

var file = file_text_open_write(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
{
    file_text_write_real(file, scr[i]);
    file_text_writeln(file);
    file_text_write_string(file, scr_name[i]);
    file_text_writeln(file);
}
file_text_close(file);

Die obige code öffnet eine Datei zum Schreiben und loops durchläuft dann zwei Arrays, wobei jeder Array-Wert in eine neue Zeile der Datei geschrieben wird. Die Datei wird dann geschlossen, wenn die loop beendet ist.
