file_text_readln

Mit dieser Funktion können Sie den Rest der aktuellen Zeile aus einer bestimmten geöffneten Textdatei überspringen und zum Anfang der nächsten Zeile springen. Die Funktion gibt auch die gesamte Zeile als Zeichenkette zurück, so dass es eine einfache Möglichkeit ist, komplette Datenpakete zu lesen und später zu parsen.

Syntax:

file_text_readln(fileid);

ArgumentTyp Beschreibung
fileid The id of the file to read from.

Rückgabe:

Beispiel:

var file = file_text_open_read(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
{
    scr[i] = file_text_read_real(file);
    file_text_readln(file);
    scr_name[i] = file_text_read_string(file);
    file_text_readln(file);
}
file_text_close(file);

Die obige code öffnet eine Datei zum Lesen und liest dann loops durch die Zeilen der Datei, indem sie abwechselnd einen reellen Zahlenwert und eine Zeichenkette in zwei verschiedene Arrays zur späteren Verwendung einliest. Die Datei wird dann geschlossen, wenn die loop beendet ist.
