file_text_write_string

Mit dieser Funktion können Sie eine Zeichenkette in eine zuvor geöffnete Textdatei schreiben. Wenn die Datei bereits Informationen enthält, werden diese Informationen gelöscht und die Zeichenkette wird an den Anfang der Datei geschrieben, es sei denn, Sie haben die Datei mit der Funktion file_text_open_append(). Sie können dies auch vermeiden, indem Sie die Funktion file_text_readln() Funktion zusammen mit der file_text_eof() die Funktion loop verwenden, um den Inhalt der Datei bis zum Ende zu durchsuchen und dann mit dem Schreiben zu beginnen.

Syntax:

file_text_write_string(fileid, str);

ArgumentTyp Beschreibung
fileid The id of the file to edit.
str The string to write to the file.

Rückgabe:

Beispiel:

var file = file_text_open_write(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i)
{
    file_text_write_real(file, scr[i]);
    file_text_writeln(file);
    file_text_write_string(file, scr_name[i]);
    file_text_writeln(file);
}
file_text_close(file);

Die obige code öffnet eine Datei zum Schreiben und loops durchläuft dann zwei Arrays, wobei jeder Array-Wert in eine neue Zeile der Datei geschrieben wird. Die Datei wird dann geschlossen, wenn die loop beendet ist.
