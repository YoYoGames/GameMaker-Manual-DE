file_text_open_write

Diese Funktion öffnet die Textdatei mit dem angegebenen Dateinamen nur zum Schreiben (wenn die Datei nicht existiert, wird sie erstellt) und gibt die eindeutige ID der Datei zurück, die in einer Variablen gespeichert werden sollte, da sie für alle weiteren Aktionen im Zusammenhang mit dieser Datei verwendet wird.

Beachten Sie, dass die Funktion -1 zurückgibt, wenn die Datei nicht erstellt werden kann (z. B. wegen eines unzulässigen Dateinamens).

Wenn die Datei bereits vorhanden ist, wird sie beim Speichern überschrieben, wenn Sie diese Funktion verwenden. Wenn Sie Daten zu einer Datei hinzufügen möchten, während der vorhandene Inhalt intakt bleibt, verwenden Sie file_text_open_append().

HINWEIS: Es können immer nur maximal 32 Dateien gleichzeitig geöffnet sein. Außerdem sollten Sie Dateien immer schließen, wenn Sie fertig sind, da dadurch die Informationen geschrieben werden und der mit der Datei verbundene Speicher freigegeben wird.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht so, wie Sie es erwarten, da GameMaker in einer Sandbox läuft! Bitte lesen Sie den Abschnitt über das Dateisystem für weitere Informationen.

Syntax:

file_text_open_write(fname);

Argument Typ Beschreibung
fname Zeichenfolge The name of the file to write to.

Rückgabe:

Text File ID or -1

Beispiel:

var file;
file = file_text_open_write(working_directory + "level.txt");
file_text_write_string(file, level_data);
file_text_close(file);

Die obige code öffnet die Datei "level.txt" zum Schreiben und schreibt dann die in der Variablen "level_data" gespeicherte Zeichenfolge, bevor sie die Datei wieder schließt.
