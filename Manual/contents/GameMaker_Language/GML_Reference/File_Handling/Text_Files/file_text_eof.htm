file_text_eof

Diese Funktion gibt true zurück, wenn das Ende einer bestimmten geöffneten Textdatei erreicht wurde, oder false, wenn nicht.

Syntax:

file_text_eof(fileid);

ArgumentTyp Beschreibung
fileid The id of the file to check.

Rückgabe:

Beispiel:

var num = 0;
var file = file_text_open_read(working_directory + "Game_Data.txt");
while (!file_text_eof(file))
{
    str[num++] = file_text_readln(file);
}
file_text_close(file);

Die obige code öffnet eine Datei zum Schreiben und geht dann loops durch die bereits in die Datei geschriebenen Textzeilen, bis sie das Ende erreicht hat, wobei jede Zeile im Array "str" gespeichert wird.
