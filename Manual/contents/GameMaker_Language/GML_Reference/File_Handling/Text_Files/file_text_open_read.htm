file_text_open_read

Diese Funktion öffnet die Textdatei mit dem angegebenen Dateinamen nur zum Lesen und gibt die eindeutige ID der Datei zurück, die in einer Variablen gespeichert werden sollte, da sie für alle weiteren Aktionen im Zusammenhang mit dieser Datei verwendet wird. Wenn die Datei nicht existiert, gibt die Funktion den Wert -1 zurück.

HINWEIS: Es können immer nur maximal 32 Dateien gleichzeitig geöffnet sein. Außerdem sollten Sie Dateien immer schließen, wenn Sie fertig sind, da dadurch die Informationen geschrieben werden und der mit der Datei verbundene Speicher freigegeben wird.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht so, wie Sie es erwarten, da GameMaker in einer Sandbox läuft! Bitte lesen Sie den Abschnitt über das Dateisystem für weitere Informationen.

Syntax:

file_text_open_read(fname);

ArgumentTyp Beschreibung
fname The name of the file to read from.

Rückgabe:

Text File ID or -1

Beispiel:

file = file_text_open_read(working_directory + "level.txt");

Dies würde die Datei "level.txt" aus demselben Verzeichnis wie das Spiel öffnen und die Datei-ID in der Variablen "file" speichern.
