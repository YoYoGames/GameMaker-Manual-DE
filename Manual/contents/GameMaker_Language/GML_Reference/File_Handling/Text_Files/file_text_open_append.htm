file_text_open_append

Diese Funktion öffnet die Textdatei mit dem angegebenen Dateinamen zum Schreiben (wenn die Datei nicht existiert, wird sie erstellt) und gibt die eindeutige ID der Datei zurück, die in einer Variablen gespeichert werden sollte, da sie für alle weiteren Aktionen im Zusammenhang mit dieser Datei verwendet wird. Die Position innerhalb der Datei, in die geschrieben werden soll, wird auf die letzte in der Datei enthaltene Textzeile gesetzt, so dass vorhandene Daten nicht überschrieben werden.

Beachten Sie, dass die Funktion -1 zurückgibt, wenn die Datei nicht erstellt werden kann (z. B. wegen eines unzulässigen Dateinamens).

HINWEIS: Es können immer nur maximal 32 Dateien gleichzeitig geöffnet sein. Außerdem sollten Sie Dateien immer schließen, wenn Sie fertig sind, da dadurch die Informationen geschrieben werden und der mit der Datei verbundene Speicher freigegeben wird.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht so, wie Sie es erwarten, da GameMaker in einer Sandbox läuft! Bitte lesen Sie den Abschnitt über das Dateisystem für weitere Informationen.

Syntax:

file_text_open_append(fname);

Argument Typ Beschreibung
fname Zeichenfolge The name of the file to append to.

Rückgabe:

Text File ID or -1

Beispiel:

file = file_text_open_append(working_directory + "save.txt");

Dies würde die Datei "save.txt" aus demselben Verzeichnis wie das Spiel öffnen und die Datei-ID in der Variablen "file" speichern.
