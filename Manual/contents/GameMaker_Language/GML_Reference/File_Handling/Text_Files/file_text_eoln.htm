file_text_eoln

Mit dieser Funktion können Sie GameMaker dazu veranlassen, die aktuell geöffnete Datei daraufhin zu überprüfen, ob die zu lesende Zeile zu Ende ist. Die Funktion gibt true zurück, wenn das Ende der Zeile erreicht wurde, und andernfalls false.

Syntax:

file_text_eoln(fileid);

ArgumentTyp Beschreibung
fileid The id of the file to check.

Rückgabe:

Beispiel:

var file = file_text_open_read(working_directory + "Game_Data.txt");
var num = 0; while (!file_text_eoln(file))
{
    score_array[num] = file_text_read_real(file);
    num++;
}
file_text_close(file);

Die obige code öffnet eine Datei zum Lesen und liest dann die Werte aus einer einzigen Zeile in ein Array.
