file_text_open_from_string

Diese Funktion erstellt eine Textdatei aus einer Zeichenkette und öffnet sie zum Lesen. Sie gibt das "Handle" der Datei zurück, das bei allen weiteren Aufrufen von Dateifunktionen zum Lesen aus dieser Datei verwendet werden sollte. Beachten Sie, dass diese Datei temporär und nur lesbar ist und daher aus dem Speicher entfernt wird, sobald sie geschlossen wird.

HINWEIS: Es können immer nur maximal 32 Dateien gleichzeitig geöffnet sein. Außerdem sollten Sie Dateien immer schließen, wenn Sie fertig sind, da dadurch die Informationen geschrieben werden und der mit der Datei verbundene Speicher freigegeben wird.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht so, wie Sie es erwarten, da GameMaker in einer Sandbox läuft! Bitte lesen Sie den Abschnitt über das Dateisystem für weitere Informationen.

Syntax:

file_text_open_from_string(string);

ArgumentTyp Beschreibung
string The string to create the file from.

Rückgabe:

File ID or -1

Beispiel:

file = file_text_open_from_string(reset_str);

Die obige code nimmt die in der Variablen "reset_str" gespeicherte Zeichenkette und verwendet sie, um eine schreibgeschützte Textdatei zu erstellen. Das "Handle" für diese Datei wird dann in der Variablen "file" gespeichert, damit alle weiteren Dateifunktionen darauf zugreifen können.
