ini_key_delete

Mit dieser Funktion können Sie den ausgewählten Schlüssel (und den dazugehörigen Wert) aus einer Ini-Datei entfernen.

Syntax:

ini_key_delete(section, key);

ArgumentTyp Beschreibung
section The section to delete a key from.
key The key to delete.

Rückgabe:

Beispiel:

ini_open("savedata.ini");
ini_write_real("save1","Score",score);
ini_key_delete("save1","Score");
ini_close();

In diesem Beispiel wird "savedata.ini" geöffnet und ein Wert in "save1" > "Score" geschrieben. Anschließend wird der Schlüssel "Score" gelöscht und die .ini-Datei geschlossen.
