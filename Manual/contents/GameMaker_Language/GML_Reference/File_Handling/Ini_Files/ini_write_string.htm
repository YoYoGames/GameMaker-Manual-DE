ini_schreib_zeichenkette

Mit dieser Funktion können Sie einen String (Text) in eine Ini-Datei schreiben. Ini-Dateien sind in Abschnitte unterteilt und jeder Abschnitt wird anschließend in Schlüssel-Wert-Paare aufgeteilt. Eine typische Ini-Datei würde also etwa so aussehen:

Es ist erwähnenswert, dass die Verwendung von Anführungszeichen und Escape-Zeichen beim Schreiben von Ini-Strings zu Problemen führen kann, wenn die Daten später zurückgelesen werden, da sie möglicherweise vorzeitig abgeschnitten werden. Dies ist zum Beispiel beim Schreiben ein Problem:

ini_write_string("test2", "section", "hello \"Fritz\"");

Syntax:

ini_write_string(section, key, value);

Argument Typ Beschreibung
section Zeichenfolge The section of the .ini to write to.
key Zeichenfolge The key within the relevant section of the .ini to write to.
value Zeichenfolge The string to write to the relevant destination.

Rückgabe:

N/A

Beispiel:

ini_open("savedata.ini");
ini_write_string("Save", "Player", global.Name);
ini_close();

Die obige code öffnet eine Ini-Datei zum Lesen und Schreiben, schreibt dann die in der globalen Variable "Name" gespeicherte Zeichenkette in den Abschnitt "Save" mit dem Schlüssel "Player" und schließt die Datei wieder.
