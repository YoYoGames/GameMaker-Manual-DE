ini_write_real

Mit dieser Funktion können Sie einen Wert (numerisch) in eine Ini-Datei schreiben. Ini-Dateien sind in Abschnitte unterteilt und jeder Abschnitt wird anschließend in Schlüssel-Wert-Paare aufgeteilt. Eine typische Ini-Datei würde also etwa so aussehen:

Syntax:

ini_write_real(section, key, value);

ArgumentTyp Beschreibung
section The section of the .ini to write to.
key The key within the relevant section of the .ini to write to.
value The real value to write to the relevant destination.

Rückgabe:

Beispiel:

score = 1000;
ini_open("savedata.ini");
ini_write_real("save1", "Score", score );
score2 = ini_read_real("save1", "Score", 0 );
ini_close();

Dadurch wird score auf 1000 gesetzt, dann wird "savedata.ini" geöffnet und dieser Wert in "save1" > "Score" geschrieben. Dann wird score2 auf den tatsächlichen Wert unter "save1" > "score" gesetzt, 1000, und die .ini wieder. Hätte es diesen Wert nicht gefunden, wäre score2 am Ende auf 0 gesetzt worden.
