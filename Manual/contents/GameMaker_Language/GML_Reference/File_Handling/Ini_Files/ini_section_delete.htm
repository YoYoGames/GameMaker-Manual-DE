ini_section_delete

Mit dieser Funktion können Sie einen ganzen Abschnitt einer ini-Datei löschen, wodurch auch alle damit verbundenen Schlüssel-Wert-Paare entfernt werden.

Syntax:

ini_section_delete(section);

ArgumentTyp Beschreibung
section The section to delete.

Rückgabe:

Beispiel:

ini_open("savedata.ini");
ini_write_real("save1", "Score", score );
ini_section_delete("save1");
ini_close();

In diesem Beispiel wird "savedata.ini" geöffnet und ein Wert in "save1" > "Score" geschrieben. Anschließend wird der Abschnitt "save1" gelöscht und die .ini-Datei geschlossen.
