ini_öffnen_aus_Zeichenfolge

Diese Funktion erstellt eine temporäre Ini-Datei aus einer Zeichenkette und öffnet sie zum Lesen/Schreiben. Die Zeichenkette sollte korrekt als Ini-Datei formatiert sein (d.h. mit Abschnitten, Zeilenumbrüchen, Schlüsseln und Werten), ansonsten wird die Ini-Datei nicht korrekt erstellt. Beachten Sie, dass diese Ini-Datei temporär ist und aus dem Speicher entfernt wird, sobald sie geschlossen wird, wodurch alle darin gespeicherten Informationen verloren gehen. ini_close() Funktion gibt jedoch eine Zeichenkette der vollständigen Ini-Datei zurück, die dann auf server oder auf der Festplatte gespeichert werden kann.

Syntax:

ini_open_from_string(string);

ArgumentTyp Beschreibung
string The string containing all the ini information.

Rückgabe:

N/A

Beispiel:

ini_open_from_string(str)
global.sound = ini_read_string("Options", "Sound", true);
ini_close();

Die obige code legt eine temporäre Ini an, um die Informationen aus der Zeichenkette "str" zu speichern, liest dann aus der Ini und schließt sie wieder.
