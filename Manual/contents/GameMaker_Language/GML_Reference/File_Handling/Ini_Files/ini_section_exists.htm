ini_section_exists

Diese Funktion prüft, ob ein Abschnitt in der aktuell geöffneten ini-Datei vorhanden ist und gibt in diesem Fall true zurück, andernfalls false. Diese Prüfung ist nicht notwendig, um Fehler zu vermeiden, denn wenn ein Abschnitt nicht existiert, wird beim Lesen aus einem nicht existierenden Abschnitt einfach ein Standardwert zurückgegeben. Es kann jedoch nützlich sein, um festzustellen, ob eine Ini-Datei bestimmte Daten gespeichert hat.

Syntax:

ini_section_exists(section);

ArgumentTyp Beschreibung
section The section in the open .ini file to check for.

Rückgabe:

Beispiel:

ini_open("savedata.ini");
if (!ini_section_exists("save1"))
{
    global.savegame = "1";
}
ini_close();

Dadurch wird die Variable global.savegame auf "1" gesetzt, wenn "save1" NICHT gefunden wird. Mit einem for loop könnte dies als grundlegende Methode verwendet werden, um herauszufinden, wie viele Savegames es für ein Spiel gibt (d.h.: ein System für mehrere gleichzeitige Savegames für ein Spiel erstellen).
