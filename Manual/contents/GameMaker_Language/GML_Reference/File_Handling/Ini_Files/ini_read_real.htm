ini_lesen_real

Mit dieser Funktion können Sie eine Zahl aus einer Ini-Datei lesen. Ini-Dateien sind in Abschnitte unterteilt und jeder Abschnitt wird anschließend in Schlüssel-Wert-Paare aufgeteilt. Eine typische Ini-Datei würde also etwa so aussehen:

Syntax:

ini_read_real(section, key, default);

ArgumentTyp Beschreibung
section The section of the .ini to read from.
key The key within the relevant section of the .ini to read from.
default The value to return if a value is not found in the defined place (or the .ini file does not exist). Must be a real number.

Rückgabe:

Beispiel:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0 );
ini_close();

Dadurch wird die Datei "savedata.ini" geöffnet und der Wert "score" auf den Wert unter "save1" > "score" gesetzt und die Datei wieder geschlossen. Sollte unter "save1" > "score" kein Wert stehen oder keine "savedata.ini" Datei vorhanden sein, wird score auf 0 gesetzt.
