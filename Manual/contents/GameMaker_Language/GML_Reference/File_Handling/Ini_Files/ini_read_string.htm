ini_read_string

Mit dieser Funktion können Sie einen String (Text) aus einer Ini-Datei lesen. Ini-Dateien sind in Abschnitte unterteilt und jeder Abschnitt wird anschließend in Schlüssel-Wert-Paare aufgeteilt. Eine typische Ini-Datei würde also etwa so aussehen:

Syntax:

ini_read_string(section, key, default);

Argument Typ Beschreibung
section Zeichenfolge The section of the .ini to read from.
key Zeichenfolge The key within the relevant section of the .ini to read from.
default Zeichenfolge The string to return if a string is not found in the defined place (or the .ini file does not exist). Must be a string.

Rückgabe:

String

Beispiel:

ini_open("savedata.ini");
global.name = ini_read_string("player", "name', "Player1");
ini_close();

Dadurch wird die Datei "savedata.ini" geöffnet und global.name auf die Zeichenkette unter "save1" > "Name" gesetzt, dann wird die Datei wieder geschlossen. Sollte keine Zeichenkette unter "save1" > "Name" vorhanden sein, oder keine "savedata.ini"-Datei vorhanden sein, wird global.name auf "Player1" gesetzt.
