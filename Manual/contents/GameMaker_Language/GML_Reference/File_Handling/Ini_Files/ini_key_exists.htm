ini_key_exists

Diese Funktion prüft, ob ein Schlüssel in der aktuell geöffneten ini-Datei vorhanden ist und gibt in diesem Fall true zurück, andernfalls false. Diese Prüfung ist nicht notwendig, um Fehler zu vermeiden, denn wenn ein Schlüssel nicht existiert, wird beim Lesen aus einem nicht existierenden Schlüssel einfach ein Standardwert zurückgegeben. Es kann jedoch nützlich sein, um zu sehen, ob eine Ini-Datei bestimmte Daten und einige andere Dinge gespeichert hat.

Syntax:

ini_key_exists(section, key);

ArgumentTyp Beschreibung
section The section in the open .ini file to check a key in.
key The key to check for.

Rückgabe:

Beispiel:

ini_open("savedata.ini");
if !ini_key_exists("save1", "name")
{
    global.name = "Player1";
}
ini_close();

Dadurch wird die Variable global.name auf "Player1" gesetzt, wenn kein Schlüssel mit dem Namen "name" existiert.
