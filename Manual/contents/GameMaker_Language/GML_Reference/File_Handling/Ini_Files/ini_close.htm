ini_schließen

Diese Funktion sollte aufgerufen werden, sobald Sie eine geöffnete Ini-Datei gelesen oder geschrieben haben. Wenn Sie die Funktion nicht verwenden, nachdem Sie eine der Ini-Schreibfunktionen verwendet haben, wird nichts auf die Festplatte geschrieben, da die Dateiinformationen im Speicher gehalten werden, bis diese Funktion aufgerufen wird, die das Schreiben erzwingt. Wenn Sie versuchen, eine ini-Datei zu öffnen, ohne vorher eine andere (oder dieselbe) geschlossen zu haben, erhalten Sie ebenfalls einen Fehler.

Die Funktion gibt auch eine Zeichenkette zurück, in der die Ini-Datei kodiert ist. Diese Zeichenkette kann dann auf server gespeichert und/oder zusammen mit der Funktion ini_open_from_string() verwendet werden, um die ini-Datei neu zu erstellen.

Syntax:

ini_close();

Rückgabe:

String

Beispiel:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

Dadurch wird die Datei "savedata.ini" geöffnet und ein Wert aus dem Abschnitt "save1" mit dem Schlüssel "score" ausgelesen, dann wird die Ini-Datei geschlossen. Wenn es unter "save1" keinen Wert namens "score" gibt oder keine Datei "savedata.ini" vorhanden ist, wird 0 zurückgegeben und eine neue Ini-Datei erstellt.
