ds_grid_shuffle

Diese Funktion kann verwendet werden, um die Positionen aller Werte in allen Zellen innerhalb eines Gitters zu randomisieren.

HINWEIS Diese Funktion mischt die Gitterzellen bei jedem neuen Durchlauf des Spiels an die gleichen Positionen, da GameMaker jedes Mal den gleichen anfänglichen Zufallswert erzeugt, um debugging code zu vereinfachen. Um dies zu vermeiden dieses Verhalten zu vermeiden, verwenden Sie randomise() beim Start des Spiels. Dies gilt nur für das Testen und debugging des Spiels, da das endgültige ausführbare Paket wird dieses Verhalten nicht zeigen und wird bei jedem Spiel zufällig sein.

Syntax:

ds_grid_shuffle(index);

ArgumentTyp Beschreibung
index The index of the grid to shuffle.

Rückgabe:

Beispiel:

ds_grid_shuffle(grid)

Dies würde alle Werte im DS-Raster, die in der Variablen "grid" indiziert sind, nehmen und sie umherschieben, um eine andere Reihenfolge zu erhalten.
