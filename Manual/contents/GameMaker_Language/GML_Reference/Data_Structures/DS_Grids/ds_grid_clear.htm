ds_grid_clear

Diese Funktion kann verwendet werden, um ein bestimmtes DS-Gitter auf einen bestimmten Wert zu setzen. Alle Zellen innerhalb des Rasters enthalten dann diesen Wert, der eine reelle Zahl oder eine Zeichenkette sein kann. Die folgende Abbildung veranschaulicht, wie dies funktioniert:

Syntax:

ds_grid_clear(index, val);

ArgumentTyp Beschreibung
index This index of the grid to clear.
val The new value for all grid cells.

Rückgabe:

Beispiel:

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1)

Die obige code ändert die Größe des DS-Gitters, das in der globalen Variable "Grid" indiziert ist, und löscht es dann, so dass jede Zelle den Wert -1 hat.
