ds_grid_value_disk_exists

Mit dieser Funktion können Sie prüfen, ob ein bestimmter Wert (real oder string) innerhalb eines kreisförmigen Bereichs eines bestimmten DS-Gitters vorhanden ist. Wenn dies der Fall ist, gibt die Funktion true zurück, andernfalls wird false ausgegeben.

Syntax:

ds_grid_value_disk_exists(index, xm, ym, r, val);

ArgumentTyp Beschreibung
index The index of the grid.
xm The x position of the disk on the grid.
ym The y position of the disk on the grid.
r The radius of the disk on the grid.
val The value to find.

Rückgabe:

Beispiel:

if ds_grid_value_disk_exists(grid, 5, 5, 5, val)
{
    xpos = ds_grid_value_disk_x(grid, 5, 5, 5, val);
    ypos = ds_grid_value_disk_y(grid, 5, 5, 5, val);
}

Die obige code prüft ein DS-Gitter auf einen bestimmten Wert innerhalb einer Plattenregion. Wird er gefunden, speichert sie die x- und y-Position des Wertes in zwei Variablen zur späteren Verwendung.
