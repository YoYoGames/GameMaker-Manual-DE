ds_grid_value_y

Mit dieser Funktion können Sie die y-Koordinate (innerhalb des angegebenen rechteckigen Gitterbereichs) des gesuchten Wertes ermitteln. Sie geben den DS-Gitterindex (wie er von ds_grid_create()zurückgegeben wird) zusammen mit den x/y-Positionen für die obere linke und untere rechte Ecke des zu durchsuchenden Gitterbereichs an, bevor Sie den zu suchenden Wert angeben. Wenn der gesuchte Wert nicht existiert, gibt die Funktion -1 zurück.

Syntax:

ds_grid_value_y(index, x1, y1, x2, y2, val);

ArgumentTyp Beschreibung
index The index of the grid.
x1 The x position of the left of the region in the grid, from 0 to (grid width - 1).
y1 The y position of the top of the region in the grid, from 0 to (grid height - 1).
x2 The x position of the right of the region in the grid, from 0 to (grid width - 1).
y2 The y position of the bottom of the region in the grid, from 0 to (grid height - 1)
val The value to find.

Rückgabe:

Beispiel:

if ds_grid_value_exists(grid, 0, 1, 5, 6, val)
{
    xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val);
    ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val);
}

Die obige code prüft ein ds_grid auf einen bestimmten Wert innerhalb eines rechteckigen Bereichs. Wird er gefunden, speichert sie die x- und y-Position des Werts in zwei Variablen zur späteren Verwendung.
