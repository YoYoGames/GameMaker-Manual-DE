ds_grid_value_exists

Mit dieser Funktion können Sie prüfen, ob ein bestimmter Wert (Real oder String) in einem rechteckigen Bereich eines bestimmten DS-Gitters vorhanden ist. Ist er vorhanden, gibt die Funktion true zurück, andernfalls false.

Syntax:

ds_grid_value_exists(index, x1, y1, x2, y2, val);

ArgumentTyp Beschreibung
index The index of the grid.
x1 The x position of the left of the region in the grid.
y1 The y position of the top of the region in the grid.
x2 The x position of the right of the region in the grid.
y2 The y position of the bottom of the region in the grid.
val The value to find.

Rückgabe:

Beispiel:

if ds_grid_value_exists(grid, 0, 1, 5, 6, val)
{
    xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val);
    ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val);
}

Die obige code prüft ein DS-Gitter auf einen bestimmten Wert innerhalb eines rechteckigen Bereichs. Wenn er gefunden wird, speichert sie die x- und y-Position des Werts in zwei Variablen zur späteren Verwendung.
