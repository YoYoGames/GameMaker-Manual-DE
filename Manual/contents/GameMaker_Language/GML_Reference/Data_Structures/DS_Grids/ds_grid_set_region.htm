ds_grid_set_region

Diese Funktion kann verwendet werden, um einen rechteckigen Bereich eines gegebenen Gitters auf einen bestimmten Wert zu setzen (der entweder ein realer Wert oder eine Zeichenkette sein kann), wie in der unten stehenden Abbildung dargestellt:

Syntax:

ds_grid_set_region(index, x1, y1, x2, y2, val);

ArgumentTyp Beschreibung
index The index of the grid.
x1 The x position of the left of the region in the grid.
y1 The y position of the top of the region in the grid.
x2 The x position of the right of the region in the grid.
y2 The y position of the bottom of the region in the grid.
val The value to set the region cells to.

Rückgabe:

Beispiel:

ds_grid_set_region(grid, 5, 5, 10, 10, 99)

Dies würde alle Zellen innerhalb des in der Variablen "grid" indizierten Bereichs von (5,5) bis (10,10) auf 99 setzen.
