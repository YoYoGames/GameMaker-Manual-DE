ds_grid_set

Diese Funktion kann verwendet werden, um eine bestimmte Zelle innerhalb des gegebenen DS-Gitters auf einen beliebigen Wert zu setzen, der eine reelle Zahl oder eine Zeichenkette sein kann. Das folgende Bild veranschaulicht dies:

Syntax:

ds_grid_set(index, x, y, value);

ArgumentTyp Beschreibung
index This index of the grid.
x The x position of the cell to set.
y The y position of the cell to set.
value The value with which to set the cell.

Rückgabe:

Beispiel:

grid = ds_grid_create(5, 5);
var i = 0;
var j = 0;

repeat (ds_grid_width(grid))
{
    repeat (ds_grid_height(grid))
    {
        ds_grid_set(grid, i, j, irandom(9));
        j += 1;
    }

    j = 0;
    i += 1;
}

Die obige code erstellt ein Raster und speichert dessen Index in der Variablen "grid". Anschließend wird dieses Gitter mit zufälligen ganzen Zahlen von 0 bis 9 gefüllt.
