ds_grid_create

Mit dieser Funktion können Sie eine neue DS-Grid-Datenstruktur mit der angegebenen Zellenbreite und -höhe erzeugen. Diese Funktion gibt eine ID zurück, die in allen weiteren Funktionen, die mit diesem DS-Grid arbeiten, verwendet werden muss.

WICHTIG! Wenn Sie eine Datenstruktur anlegen, ist der Indexwert zu ihrer Identifizierung ein ganzzahliger Wert, der bei 0 beginnt. Das bedeutet, dass Datenstrukturen unterschiedlichen Typs denselben Indexwert haben können, so dass Sie im Zweifelsfall die ds_exists() Funktion verwenden, bevor Sie auf sie zugreifen. Beachten Sie auch, dass Indizes wiederverwendet werden, so dass ein Indexwert einer zerstörten Datenstruktur anschließend von einer neu erstellten verwendet werden kann.

Syntax:

ds_grid_create(w, h);

ArgumentTyp Beschreibung
w The width of the grid to be created.
h The height of the grid to be created.

Rückgabe:

DS Grid ID

Beispiel:

mygrid = ds_grid_create(10, 10)

Dadurch entsteht ein 10 Zellen hohes und 10 Zellen breites Gitter.
