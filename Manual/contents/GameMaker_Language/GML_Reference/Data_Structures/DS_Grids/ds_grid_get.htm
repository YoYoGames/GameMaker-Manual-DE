ds_grid_get

Diese Funktion kann verwendet werden, um den Wert einer beliebigen Zelle innerhalb des angegebenen DS-Gitters zu ermitteln. Wenn Sie der Funktion ungültige Gitterkoordinaten übergeben, lautet der zurückgegebene Wert undefined und es wird eine Fehlermeldung im Ausgabefenster angezeigt.

Syntax:

ds_grid_get(index, x, y);

ArgumentTyp Beschreibung
index The index of the grid.
x The x position of the cell you want to find the value of.
y The y position of the cell you want to find the value of.

Rückgabe:

Variable

Beispiel:

var xx = irandom(ds_grid_width(grid) - 1);
var yy = irandom(ds_grid_height(grid) - 1);
val = ds_grid_get(grid, xx, yy)

Die obige code wählt eine zufällige Zelle aus dem DS-Gitter aus, die in der Variablen "grid" indiziert ist, und speichert ihren Wert in der Variablen "val".
