ds_grid_write

Mit dieser Funktion kann das angegebene ds_grid in einen String umgewandelt werden, der dann z.B. in einer externen Datei gespeichert werden kann. Sie können die von dieser Funktion zurückgegebene Zeichenkette wieder in ein ds_grid einlesen, indem Sie die Funktion ds_grid_read().

HINWEIS Die zurückgegebene Zeichenkette ist keine vom Menschen lesbare Zeichenkette, sondern ein Dump des Inhalts der Datenstruktur.

Syntax:

ds_grid_write(index);

ArgumentTyp Beschreibung
index The index of the grid to write.

Rückgabe:

Beispiel:

ini_open("Save.ini");
ini_write_string("Save", "0", ds_grid_write(mygrid));
ini_close()

Die obige code öffnet eine ini-Datei (und erstellt sie, falls sie noch nicht existiert) und schreibt dann das angegebene ds_grid als String in diese Datei.
