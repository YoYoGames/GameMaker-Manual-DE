ds_grid_sort

Diese Funktion kann verwendet werden, um ein DS-Gitter auf der Grundlage der Werte einer bestimmten Spalte zu sortieren (ähnlich wie Sie Dateien im OS-Dateiexplorer nach Datum, Größe usw. sortieren würden). Das folgende Bild zeigt ein Beispiel:

Syntax:

ds_grid_sort(index, column, ascending);

ArgumentTyp Beschreibung
index The index of the grid to sort.
column The column to use for sorting the rows
ascending Whether to sort lowest to highest (true), oder von oben nach unten (false).

Rückgabe:

Beispiel:

ds_grid_sort(grid, 3, false)

Dies würde alle Werte im DS-Gitter nehmen, die in der Variablen "grid" indiziert sind, und sie nach den Werten in der dritten Spalte des Gitters sortieren (wie im obigen Bild gezeigt).
