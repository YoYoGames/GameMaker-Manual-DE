ds_grid_read

Diese Funktion kann verwendet werden, um einen String, der zuvor mit der Funktion ds_grid_write() zurück in ein DS-Gitter. Das DS-Gitter muss zuvor erstellt worden sein (siehe das Beispiel unten).

Beachten Sie, dass Sie, wenn die angegebene DS-Zeichenkette von GameMaker geschrieben wurde: Studio 1.2.x runtime (oder älter) geschrieben wurde, sollten Sie das optionale argument "legacy" angeben und es auf true setzen, da sich das String-Format danach geändert hat.

Syntax:

ds_grid_read(index, string [, legacy]);

Argument Typ Beschreibung
index DS-Raster-ID The index of the grid to read.
string Zeichenfolge The string to read into the DS grid.
legacy Boolesche OPTIONAL Kann entweder true oder false sein oder ganz weggelassen werden.

Rückgabe:

N/A

Beispiel:

grid = ds_grid_create(room_width div 32, room_height div 32);
ini_open("Save.ini");
ds_grid_read(grid, ini_read_string("Save", "0", ""));
ini_close();

Die obige code erstellt ein DS-Gitter basierend auf der Größe des Raumes (jedes 32x32 Pixelquadrat stellt eine Gitterzelle dar) und liest dann einen zuvor gespeicherten Satz von Gitterdaten aus einer ini-Datei in das neue DS-Gitter ein.
