ds_grid_value_disk_x

Mit dieser Funktion können Sie die x-Koordinate (innerhalb des angegebenen scheibenförmigen Rasters) des gesuchten Wertes ermitteln. Sie geben den DS-Gitterindex (wie er von ds_grid_create()zurückgegeben wird) zusammen mit den x/y-Positionen für die zentrale Zelle der Scheibe. Dann geben Sie den Radius (als ganzzahligen Wert) um die zu suchende Mittelzelle an, bevor Sie den zu suchenden Wert angeben. Wenn der gesuchte Wert nicht vorhanden ist, gibt die Funktion -1 zurück.

Syntax:

ds_grid_value_disk_x(index, xm, ym, r, val);

ArgumentTyp Beschreibung
index The index of the grid.
xm The x position of the disk on the grid.
ym The y position of the disk on the grid.
r The radius of the disk on the grid.
val The value to find.

Rückgabe:

Beispiel:

if ds_grid_value_disk_exists(grid, 5, 5, 5, val)
{
    xpos = ds_grid_value_disk_x(grid, 5, 5, 5, val);
    ypos = ds_grid_value_disk_y(grid, 5, 5, 5, val);
}

Die obige code prüft ein ds_grid auf einen bestimmten Wert innerhalb einer Plattenregion. Wenn er gefunden wird, speichert sie die x- und y-Position des Wertes in zwei Variablen zur späteren Verwendung.
