ds_grid_copy

Mit dieser Funktion können Sie den Inhalt eines Gitters in ein anderes kopieren. Beide Grids müssen zuvor mit der Funktion ds_grid_create() Funktion erstellt worden sein.

Syntax:

ds_grid_copy(destination, source);

Argument Typ Beschreibung
destination DS-Raster-ID This index of the grid to copy to.
source DS-Raster-ID This index of the grid to copy from.

Rückgabe:

N/A

Beispiel:

n_grid = ds_grid_create(ds_grid_width(a_grid), ds_grid_height(a_grid));
ds_grid_copy(n_grid, a_grid);
ds_grid_clear(a_grid, -1)

Die obige code erstellt ein neues DS-Gitter, basierend auf der Breite und Höhe eines zuvor erstellten Gitters, und kopiert dann die Informationen aus dem vorherigen Gitter in das neue. Schließlich wird das alte Raster gelöscht, so dass alle Zellen den Wert -1 haben.
