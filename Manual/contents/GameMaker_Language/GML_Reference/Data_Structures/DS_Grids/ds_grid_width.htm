ds_grid_width

Diese Funktion gibt die Breite des angegebenen Gitters zurück. Dieser Wert ist die Anzahl der Zellen, die das Gitter entlang der x-Achse hat, und ist immer eine ganze Zahl, wie in der Abbildung unten gezeigt:

Syntax:

ds_grid_width(index);

ArgumentTyp Beschreibung
index This index of the grid to find the width of.

Rückgabe:

Beispiel:

for (var i = 0; i < ds_grid_width(grid); ++i)
{
    for (var j = 0; j < ds_grid_height(grid); ++j)
    {
        if (ds_grid_get(grid, i, j) == 1)
        {
            instance_create_layer(i * 32, j * 32, "Walls", obj_Wall);
        }
    }
}

code Die obige loop wird durch das DS-Gitter gehen, das in der Variablen "grid" indiziert ist, und wenn der Wert in einer bestimmten Zelle gleich 1 ist, wird eine Instanz von "obj_Wall" an der entsprechenden Position im Raum erstellt.
