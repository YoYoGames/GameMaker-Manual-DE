ds_grid_get_disk_sum

Diese Funktion kann verwendet werden, um alle Werte aller Zellen innerhalb des definierten Bereichs eines Gitters zu addieren, wie in der folgenden Abbildung gezeigt:

Syntax:

ds_grid_get_disk_sum(index, xm, ym, r);

ArgumentTyp Beschreibung
index The index of the grid.
xm The x position of the disk on the grid.
ym The y position of the disk on the grid.
r The radius of the disk on the grid.

Rückgabe:

Real or String

Beispiel:

val = ds_grid_get_disk_sum(grid, 5, 5, 2)

Die obige code setzt die Variable "val" auf die Summe aller Werte, die in der angegebenen Scheibe des DS-Gitters enthalten sind, das in der Variablen "grid" indiziert ist.
