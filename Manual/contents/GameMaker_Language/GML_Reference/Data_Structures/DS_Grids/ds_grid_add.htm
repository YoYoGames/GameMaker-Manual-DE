ds_grid_add

Mit dieser Funktion kann ein bestimmter Wert (Real oder String) zum Wert der angegebenen Zelle im Raster addiert werden. Der hinzuzufügende Wert muss vom gleichen Typ sein wie der in der Gitterzelle enthaltene Wert, d. h. Sie können keine Zeichenkette zu einer reellen Zahl oder umgekehrt hinzufügen. Strings entspricht dies einer Verkettung.

Syntax:

ds_grid_add(index, x, y, val);

ArgumentTyp Beschreibung
index The index of the grid.
x The x position of the cell in the grid.
y The y position of the cell in the grid.
val The value to add to the cell.

Rückgabe:

Beispiel:

ds_grid_add(grid, 5, 5, 6)

Dies würde der angegebenen Zelle innerhalb des DS-Gitters, auf das die Variable "grid" verweist, 6 hinzufügen.
