ds_priority_delete_max

Diese Funktion gibt den Wert mit der höchsten Priorität in der Warteschlange zurück und entfernt den Wert (und die Priorität) aus der Datenstruktur. Wenn mehr als ein Wert die gleiche Priorität hat, kann jeder von ihnen in beliebiger Reihenfolge zurückgegeben werden, aber alle anderen Werte mit der gleichen Priorität bleiben in der Warteschlange.

Syntax:

ds_priority_delete_max(id);

ArgumentTyp Beschreibung
id The id of the priority queue to use.

Rückgabe:

(Data type stored in the priority)

Beispiel:

if ai_move
{
    script_execute(ds_priority_delete_max(ai_priority));
}

Die obige code überprüft eine Instanzvariable, und wenn sie true zurückgibt, führt sie ein script aus, das in der Prioritätswarteschlange mit dem höchsten Prioritätswert indiziert ist, und entfernt dann dieses script aus der Warteschlange.
