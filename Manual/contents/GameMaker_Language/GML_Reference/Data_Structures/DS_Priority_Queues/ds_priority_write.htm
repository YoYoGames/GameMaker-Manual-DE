ds_prioritaet_schreiben

Diese Funktion gibt eine Zeichenkette zurück, die dann gespeichert oder mit der Funktion ds_priority_read() Funktion.

HINWEIS: Die zurückgegebene Zeichenkette ist keine von Menschen lesbare Zeichenkette, sondern ein Dump des Inhalts der Datenstruktur

Syntax:

ds_priority_write(id);

ArgumentTyp Beschreibung
id The id of the data structure to check.

Rückgabe:

Beispiel:

var str;
ini_open("save.ini");
str = ds_priority_write(p_queue);
ini_write_string("P_Queues", "0", str);
ds_priority_clear(p_queue);
ini_close();

Die obige code öffnet eine Ini-Datei und schreibt dann eine Zeichenkette mit den Informationen, die in der DS-Prioritätswarteschlange gespeichert sind, die in der Variablen "p_queue" indiziert ist. Anschließend wird die Prioritätswarteschlange geleert und die Ini-Datei geschlossen.
