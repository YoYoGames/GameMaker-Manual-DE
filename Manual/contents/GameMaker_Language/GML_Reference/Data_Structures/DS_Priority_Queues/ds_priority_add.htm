ds_prioritaet_hinzufuegen

Mit dieser Funktion können Sie einen Wert (entweder eine reelle Zahl oder eine Zeichenkette) zu einer Prioritätswarteschlange hinzufügen und ihr gleichzeitig einen Prioritätswert zuweisen.

Syntax:

ds_priority_add(id, val, priority);

Argument Typ Beschreibung
id DS-Prioritäts-ID The id of the priority queue to add to.
val Variabel The value to add to the priority queue.
priority Real The priority of the value to add.

Rückgabe:

N/A

Beispiel:

ds_priority_add(ai_priority, AI_Search, 5);

Die obige code fügt der in der Variablen "ai_priority" indizierten Prioritätswarteschlange eine Funktion script hinzu und weist ihr eine Priorität von 5 zu.
