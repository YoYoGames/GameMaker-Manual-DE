DS-Warteschlangen mit Vorrang

Eine DS-Prioritätswarteschlange ist eine spezielle Datenstruktur ähnlich einer DS queuenur dass hier die Warteschlange der Werte nach den Prioritäten (immer reelle Zahlen), die den Werten zugewiesen wurden, geordnet ist. Diese Struktur kann sehr nützlich sein, um Ranglisten oder Informationslisten zu erstellen, in denen die Wichtigkeit der einzelnen Einträge gewichtet werden kann.

HINWEIS: Wie alle dynamischen Ressourcen belegen auch Datenstrukturen Speicherplatz und sollten daher immer zerstört werden, wenn sie nicht mehr benötigt werden, um Speicherlecks zu vermeiden, die das Spiel verlangsamen und schließlich zum Absturz bringen.

HINWEIS: Wenn Sie prüfen müssen, ob eine Datenstruktur existiert, können Sie die ds_exists() Funktion verwenden.

Es gibt die folgenden Funktionen, die sich mit DS-Prioritätswarteschlangen befassen:

  • ds_priority_create
  • ds_priority_destroy
  • ds_priority_clear
  • ds_priority_empty
  • ds_priority_size
  • ds_priority_add
  • ds_priority_change_priority
  • ds_priority_delete_max
  • ds_priority_delete_min
  • ds_priority_delete_value
  • ds_priority_find_max
  • ds_priority_find_min
  • ds_priority_find_priority
  • ds_priority_copy
  • ds_priority_read
  • ds_priority_write
