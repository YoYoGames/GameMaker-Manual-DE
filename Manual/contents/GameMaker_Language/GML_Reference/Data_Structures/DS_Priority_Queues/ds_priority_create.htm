ds_priority_create

Mit dieser Funktion wird eine neue Datenstruktur für die Prioritätswarteschlange erstellt und der Indexwert zurückgegeben. Dieser Wert sollte in einer Variablen gespeichert und bei allen weiteren Funktionsaufrufen im Zusammenhang mit der Prioritätswarteschlange verwendet werden.

WICHTIG! Wenn Sie eine Datenstruktur anlegen, ist der Indexwert zu ihrer Identifizierung ein ganzzahliger Wert, der bei 0 beginnt. Das bedeutet, dass Datenstrukturen unterschiedlichen Typs denselben Indexwert haben können, daher sollten Sie im Zweifelsfall die ds_exists() Funktion verwenden, bevor Sie auf sie zugreifen. Beachten Sie auch, dass Indizes wiederverwendet werden, so dass ein Indexwert einer zerstörten Datenstruktur anschließend von einer neu erstellten verwendet werden kann.

Syntax:

ds_priority_create();

Rückgabe:

DS Priority ID

Beispiel:

p_queue = ds_priority_create();

Dadurch wird eine neue Prioritätswarteschlange erstellt und ihre Index-ID der Instanzvariablen "p_queue" zugewiesen.
