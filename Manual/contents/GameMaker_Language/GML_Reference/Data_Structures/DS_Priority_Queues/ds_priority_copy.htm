ds_priorität_kopie

Mit dieser Funktion können Sie den Inhalt einer Prioritätswarteschlange in eine andere kopieren. Beachten Sie, dass der Inhalt der ursprünglichen Warteschlange NICHT entfernt wird und die ursprüngliche Warteschlange auch nicht zerstört wird. Wenn Sie diese Funktion verwenden, muss die Warteschlange, in die kopiert werden soll, zuvor erstellt worden sein, und wenn sie vor dem Kopieren Elemente enthielt, werden diese zuerst gelöscht (d. h. diese Informationen gehen verloren).

Syntax:

ds_priority_copy(destination, source);

ArgumentTyp Beschreibung
id The id of the priority queue to copy to.
source The id of the priority queue to copy from.

Rückgabe:

Beispiel:

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
p_queue = ds_priority_create();
ds_priority_copy(p_queue, other.p_queue);
}

Die obige Funktion erstellt eine neue Instanz und erstellt dann in dieser Instanz eine neue DS-Prioritätswarteschlange und kopiert den Inhalt der Prioritätswarteschlange in der Instanz, auf der der code -Block läuft, in die neu erstellte Instanz-Prioritätswarteschlange.
