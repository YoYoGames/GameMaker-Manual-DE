ds_prioritaet_loeschen_wert

Diese Funktion löscht einfach den angegebenen Wert zusammen mit seiner Priorität aus der indizierten Prioritätswarteschlange.

Syntax:

ds_priority_delete_value(id,val);

ArgumentTyp Beschreibung
id The id of the priority queue to use.
val The value to delete from the priority queue.

Rückgabe:

Beispiel:

if (ai_move == false)
{
    ds_priority_delete_value(ai_priority, AI_Move);
}

Die obige code prüft eine Instanzvariable, und wenn sie false zurückgibt, wird die indizierte Funktion script aus der Prioritätswarteschlange entfernt.
