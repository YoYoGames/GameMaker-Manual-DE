ds_prioritaet_aendern_prioritaet

Diese Funktion nimmt einen gegebenen Wert und ändert dessen Priorität innerhalb der referenzierten Prioritätswarteschlange. Der angegebene Wert sollte bereits in der Prioritäts-Warteschlange vorhanden sein.

Syntax:

ds_priority_change_priority(id, val, priority);

Argument Typ Beschreibung
id DS-Prioritäts-ID The id of the priority queue to change.
val Variabel The value to change the priority of.
priority Real The new priority of the value.

Rückgabe:

N/A

Beispiel:

if (global.Game_Time < 1000)
{
    ds_priority_change(ai_priority, AI_Search, 1);
}

Die obige code prüft eine globale Variable, und wenn sie unter einem bestimmten Wert liegt, ändert sie die Priorität des script Funktionsindexes in der Prioritätswarteschlange.
