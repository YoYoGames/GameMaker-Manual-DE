ds_prioritaet_loeschen_min

Diese Funktion gibt den Wert mit der niedrigsten Priorität in der Warteschlange zurück und entfernt den Wert (und die Priorität) aus der Datenstruktur. Wenn mehr als ein Wert die gleiche Priorität hat, kann jeder von ihnen in beliebiger Reihenfolge zurückgegeben werden, aber alle anderen Werte mit der gleichen Priorität bleiben in der Warteschlange.

Syntax:

ds_priority_delete_min(id);

ArgumentTyp Beschreibung
id The id of the priority queue to use.

Rückgabe:

(Data type stored in the priority)

Beispiel:

if ai_move
{
    script_execute(ds_priority_delete_min(ai_priority));
}

Die obige code prüft eine Instanzvariable, und wenn sie true zurückgibt, führt sie eine Funktion script aus, die in der Prioritätswarteschlange mit dem niedrigsten Prioritätswert indiziert ist, und entfernt dann diese script aus der Warteschlange.
