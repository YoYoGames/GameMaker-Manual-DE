ds_priority_clear

Mit dieser Funktion können Sie alle Daten aus der angegebenen Datenstruktur der Prioritätswarteschlange löschen. Dabei wird die Datenstruktur NICHT zerstört (dafür sollten Sie die Funktion ds_priority_destroy()), sondern löscht nur alle Daten aus der Datenstruktur und gibt eine leere Warteschlange zurück.

Syntax:

ds_priority_clear(id);

ArgumentTyp Beschreibung
id The id of the data structure to clear.

Rückgabe:

Beispiel:

if (count = 15) && (!ds_priority_empty(command_queue))
{
    ds_priority_clear(command_queue);
    alarm[0] = room_speed;
    ai_count = 0;
}

Die obige code prüft eine Variable, um zu sehen, ob sie einen bestimmten Wert erreicht hat, und wenn dies der Fall ist, löscht sie die DS-Prioritätswarteschlange, die in der Variablen "command_queue" indiziert ist, setzt einen Alarm und setzt die Variable auf 0 zurück.
