ds_prioritaet_find_prioritaet

Mit dieser Funktion können Sie die Priorität eines beliebigen Wertes abrufen. Wenn der Wert nicht in der Prioritätswarteschlange vorhanden ist, wird undefined zurückgegeben.

Syntax:

ds_priority_find_priority(id, val);

ArgumentTyp Beschreibung
id The id of the priority queue to use.
val The value to find the priority of.

Rückgabe:

Real or undefined

Beispiel:

p = ds_priority_find_priority(ai_priority, "intelligence");

Die obige code speichert die zurückgegebene Priorität für den angegebenen Wert in der Instanzvariablen "p".
