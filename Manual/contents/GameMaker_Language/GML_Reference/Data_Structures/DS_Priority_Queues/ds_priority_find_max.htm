ds_prioritaet_find_max

Mit dieser Funktion können Sie den in der Prioritätswarteschlange gespeicherten Wert mit der höchsten Priorität finden, und wenn mehr als ein Wert die gleiche Priorität hat, kann jeder von ihnen in beliebiger Reihenfolge zurückgegeben werden. Im Gegensatz zu ds_priority_delete_max()entfernt diese Funktion jedoch nicht den Wert aus der Warteschlange.

Syntax:

ds_priority_find_max(id);

ArgumentTyp Beschreibung
id The id of the priority queue to use.

Rückgabe:

Real or String

Beispiel:

if ai_move
{
    script_execute(ds_priority_find_max(ai_priority));
}

Die obige code prüft eine Instanzvariable, und wenn sie true zurückgibt, wird eine Funktion script ausgeführt, die in der Prioritätswarteschlange mit dem höchsten Prioritätswert indiziert ist.
