ds_stack_pop

Diese Funktion holt den obersten Wert vom DS-Stapel, entfernt ihn vom Stapel und gibt den Wert zurück, der in einer Variablen gespeichert werden soll. Wenn der Stapel leer ist, gibt die Funktion die Konstante undefined zurück, andernfalls den im Stapel enthaltenen reellen Wert oder String.

Syntax:

ds_stack_pop(id);

ArgumentTyp Beschreibung
id The id of the data structure to pop from.

Rückgabe:

Variable (Data type value that is stored in the stack) or undefined

Beispiel:

if !ds_stack_empty(move_stack)
{
    var xx = ds_stack_pop(move_stack);
    var yy = ds_stack_pop(move_stack);
    move_towards_point(xx, yy, 4);
}

Die obige code prüft den DS-Stapel, der in der Variablen "move_stack" indiziert ist, um zu sehen, ob er leer ist. Ist dies nicht der Fall, werden die beiden obersten Werte aus dem Stapel gepoppt und verwendet, um eine Richtung für die Bewegung festzulegen.
