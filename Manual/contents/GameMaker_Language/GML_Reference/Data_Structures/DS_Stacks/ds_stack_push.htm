ds_stack_push

Mit dieser Funktion wird ein Wert, der einen beliebigen Datentyp haben kann, oben auf den Stapel geschoben (addiert). Die Funktion kann weitere 14 optionale arguments (also insgesamt 15 mögliche Hinzufügungen) annehmen, so dass Sie mehrere Werte nacheinander mit einem einzigen Aufruf auf den Stapel legen.

Syntax:

ds_stack_push(id, val [, val2, ... val15]);

ArgumentTyp Beschreibung
id The id of the data structure to push onto.
val The value to push onto the stack.
[val2, ... val13] Optional values to be added to the stack.

Rückgabe:

Beispiel:

move_stack = ds_stack_create();
ds_stack_push(move_stack, x, y, x, y + 200, x + 200, y + 200, x +200, y);

Die obige code erstellt einen neuen DS-Stapel und speichert seinen Index in der Variablen "move_stack". Anschließend wird eine Reihe von Werten zur späteren Verwendung auf den Stapel geschoben.
