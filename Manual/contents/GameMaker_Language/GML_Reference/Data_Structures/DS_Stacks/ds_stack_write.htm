ds_stack_write

Diese Funktion gibt eine Zeichenkette zurück, die dann gespeichert oder mit der Funktion ds_stack_read() Funktion.

HINWEIS: Die zurückgegebene Zeichenkette ist keine von Menschen lesbare Zeichenkette, sondern ein Dump des Inhalts der Datenstruktur

Syntax:

ds_stack_write(id);

ArgumentTyp Beschreibung
id The id of the data structure to write.

Rückgabe:

Beispiel:

ini_open("save.ini");
var str = ds_stack_write(stack);
ini_write_string("Stacks", "0", str);
ds_stack_clear(stack);
ini_close();

Die obige code öffnet eine Ini-Datei und schreibt dann eine Zeichenkette mit den Informationen, die im DS-Stack gespeichert sind, der in der Variablen "stack" indiziert ist. Anschließend wird der Stack geleert und die Ini-Datei geschlossen.
