ds_stack_top

Diese Funktion liest nur den ersten Wert des Stapels (den, der "oben" liegt). Der Wert wird nicht gepoppt, d.h. er kann auch in Zukunft noch von dieser Funktion oder der Funktion ds_stack_pop(). Wenn der Stack leer ist, gibt die Funktion die Konstante undefined zurück, andernfalls den im Stack enthaltenen Real- oder String-Wert.

Syntax:

ds_stack_top(id);

ArgumentTyp Beschreibung
id The id of the data structure to read from.

Rückgabe:

Variable (Data type value that is stored in the stack) or undefined

Beispiel:

num = ds_stack_top(control_stack);

Die obige code liest den Wert aus dem Stapel, der in der Variablen "control_stack" indiziert ist, und speichert den Rückgabewert in der Variablen "num".
