ds_stack_copy

Diese Funktion kann verwendet werden, um den Inhalt eines Stapels in einen anderen zu kopieren. Beachten Sie, dass der Inhalt des ursprünglichen Stapels dadurch NICHT entfernt wird und der ursprüngliche Stapel auch nicht zerstört wird. Wenn Sie diese Funktion verwenden, muss der Stapel, in den kopiert werden soll, zuvor erstellt worden sein, und wenn er vor dem Kopieren Elemente enthielt, werden diese zuerst gelöscht (d. h. diese Informationen gehen verloren).

Syntax:

ds_stack_copy(id, source);

Argument Typ Beschreibung
id DS-Stapel-ID The id of the new stack.
source DS-Stapel-ID The original stack to copy from.

Rückgabe:

N/A

Beispiel:

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
    stack = ds_stack_create();
    ds_stack_copy(stack, other.stack);
}

Die obige Funktion erstellt eine neue Instanz und erstellt dann in dieser Instanz einen neuen DS-Stapel und kopiert den Inhalt des Stapels in der Instanz, in der der code -Block läuft, in den neu erstellten Instanz-Stapel.
