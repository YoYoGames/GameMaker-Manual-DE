ds_stack_destroy

Diese Funktion entfernt die angegebene Stack-Datenstruktur aus dem Speicher, gibt die von ihr verwendeten Ressourcen frei und entfernt alle darin enthaltenen Werte. Diese Funktion sollte immer verwendet werden, wenn Sie den DS-Stack nicht mehr verwenden, um Speicherlecks zu vermeiden, die Ihr Spiel verlangsamen und zum Absturz bringen können.

WICHTIG! Wenn Sie eine Datenstruktur anlegen, ist der Indexwert zu ihrer Identifizierung ein ganzzahliger Wert, der bei 0 beginnt. Das bedeutet, dass Datenstrukturen unterschiedlichen Typs denselben Indexwert haben können, daher sollten Sie im Zweifelsfall die ds_exists() Funktion verwenden, bevor Sie auf sie zugreifen. Beachten Sie auch, dass Indizes wiederverwendet werden, so dass ein zerstörter Indexwert einer Datenstruktur anschließend von einer neu erstellten verwendet werden kann. Wir empfehlen daher, die Variable, die den DS-Index enthielt, nach der Zerstörung immer auf -1 zu setzen.

Syntax:

ds_stack_destroy(id);

ArgumentTyp Beschreibung
id The id of the data structure to remove.

Rückgabe:

Beispiel:

if (lives == 0)
{
    ds_stack_destroy(AI_stack);
    AI_stack = -1;
    room_goto(rm_Menu);
}

Die obige code prüft den Wert der eingebauten globalen Variable "lives", und wenn er 0 ist, wird der DS-Stapel, der in der Variable "AI_stack" indiziert ist, zerstört und dann der Raum gewechselt.
