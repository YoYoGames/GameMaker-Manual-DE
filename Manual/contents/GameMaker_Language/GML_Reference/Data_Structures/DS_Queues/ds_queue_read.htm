ds_queue_read

Mit dieser Funktion können Sie eine gespeicherte DS-Warteschlange wiederherstellen (eine, die zuvor als Zeichenkette mit ds_queue_write()). Sie müssen zunächst eine neue DS-Warteschlange erstellen, in die Sie die Zeichenkette einlesen können. Wenn die DS-Warteschlange bereits existiert und Informationen darin gespeichert sind, werden diese vor dem Einlesen gelöscht. Diese Funktion ist von entscheidender Bedeutung bei der Erstellung von Speicher-/Lademechanismen für Ihr Spiel.

Beachten Sie, dass Sie, wenn die angegebene DS-Zeichenkette von GameMaker geschrieben wurde: Studio 1.2.x runtime (oder älter) geschrieben wurde, sollten Sie das optionale argument "legacy" angeben und es auf true setzen, da sich das String-Format danach geändert hat.

Syntax:

ds_queue_read(id, str [, legacy]);

Argument Typ Beschreibung
id DS-Warteschlangen-ID The id of the data structure to read into.
str Zeichenfolge The string to read from.
legacy Boolesche OPTIONAL Kann entweder true oder false sein oder ganz weggelassen werden.

Rückgabe:

N/A

Beispiel:

queue = ds_queue_create();
ini_open("save.ini");
var str = ini_read_string("Queues", "0", "");
if (str != "")
{
    ds_queue_read(queue, str);
}
ini_close();

Die obige code erstellt eine Warteschlange und speichert den Index in der Variablen "queue". Dann wird eine Datei ini geöffnet und eine Zeichenkette daraus gelesen, wobei zunächst geprüft wird, ob die Zeichenkette nicht als leer zurückgegeben wird. Diese Zeichenkette wird dann in die neu erstellte DS-Warteschlange eingelesen.
