ds_queue_copy

Diese Funktion kann verwendet werden, um den Inhalt einer Warteschlange in eine andere zu kopieren. Beachten Sie, dass dadurch weder der Inhalt der ursprünglichen Warteschlange entfernt noch die ursprüngliche Warteschlange zerstört wird. Wenn Sie diese Funktion verwenden, muss die Warteschlange, in die kopiert werden soll, zuvor erstellt worden sein, und wenn sie vor dem Kopieren Elemente enthielt, werden diese zuerst gelöscht (d. h. diese Informationen gehen verloren).

Syntax:

ds_queue_copy(destination, source);

Argument Typ Beschreibung
id DS-Warteschlangen-ID The id of the new queue.
source DS-Warteschlangen-ID The original queue to copy from.

Rückgabe:

N/A

Beispiel:

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
    queue = ds_queue_create();
    ds_queue_copy(queue, other.queue);
}

Die obige Funktion erstellt eine neue Instanz und erstellt dann in dieser Instanz eine neue DS-Warteschlange und kopiert den Inhalt der Warteschlange in der Instanz, in der der code -Block läuft, in die neu erstellte Instanz-Warteschlange.
