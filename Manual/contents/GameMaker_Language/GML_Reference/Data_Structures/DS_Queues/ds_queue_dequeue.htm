ds_queue_dequeue

Mit dieser Funktion wird der Kopfwert aus der DS-Warteschlange entfernt und der Wert zurückgegeben, der in einer Variablen gespeichert werden soll. Wenn die Warteschlange leer ist, gibt die Funktion die Konstante undefined zurück, andernfalls den in der Warteschlange enthaltenen Wert.

Syntax:

ds_queue_dequeue(id);

ArgumentTyp Beschreibung
id The id of the queue to dequeue from.

Rückgabe:

Variable (Data type value stored in the queue) or undefined

Beispiel:

if !ds_queue_empty(move_queue)
{
    var xx = ds_queue_dequeue(move_queue);
    var yy = ds_queue_dequeue(move_queue);
    move_towards_point(xx, yy, 4);
}

Die obige code prüft die DS-Warteschlange, die in der Variablen "move_queue" indiziert ist, um festzustellen, ob sie leer ist. Ist dies nicht der Fall, werden die beiden Werte aus dem Kopf der Warteschlange entnommen und verwendet, um eine Richtung für die Bewegung festzulegen.
