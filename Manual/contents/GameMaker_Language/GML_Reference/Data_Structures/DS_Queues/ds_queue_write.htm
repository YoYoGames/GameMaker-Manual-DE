ds_queue_write

Diese Funktion gibt eine Zeichenkette zurück, die dann gespeichert oder mit der Funktion ds_queue_read() Funktion.

HINWEIS: Die zurückgegebene Zeichenkette ist keine von Menschen lesbare Zeichenkette, sondern ein Dump des Inhalts der Datenstruktur

Syntax:

ds_queue_write(id);

ArgumentTyp Beschreibung
id The id of the data structure to write.

Rückgabe:

Beispiel:

ini_open("save.ini");
var str =ds_queue_write(queue);
ini_write_string("Queues", "0", str);
ds_queue_clear(queue);
ini_close();

Die obige code öffnet eine Ini-Datei und schreibt dann eine Zeichenkette mit den Informationen, die in der DS-Warteschlange gespeichert sind, die in der Variablen "queue" indiziert ist. Anschließend wird die Warteschlange geleert und die ini-Datei geschlossen.
