ds_queue_clear

Mit dieser Funktion können Sie alle Daten aus der angegebenen Warteschlangen-Datenstruktur löschen. Dabei wird die Datenstruktur NICHT zerstört (dafür sollten Sie die Funktion ds_queue_destroy()), sondern löscht nur alle Daten aus der Datenstruktur und gibt eine leere Warteschlange zurück.

Syntax:

ds_queue_clear(id);

ArgumentTyp Beschreibung
id The id of the data structure to clear.

Rückgabe:

Beispiel:

if (count = 15) && (!ds_queue_empty(command_queue))
{
    ds_queue_clear(command_queue);
    alarm[0] = room_speed;
    ai_count = 0;
}

Die obige code prüft eine Variable, um zu sehen, ob sie einen bestimmten Wert erreicht hat. Wenn dies der Fall ist, wird die DS-Warteschlange, die in der Variablen "command_queue" indiziert ist, gelöscht, ein Alarm ausgelöst und die Variable auf 0 zurückgesetzt.
