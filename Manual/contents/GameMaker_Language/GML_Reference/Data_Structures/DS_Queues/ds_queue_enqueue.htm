ds_queue_enqueue

Diese Funktion fügt einen Wert (real oder string) an das Ende der DS-Warteschlange an. Die Funktion kann weitere 14 optionale arguments (also insgesamt 15 mögliche Hinzufügungen) annehmen, so dass Sie mehrere Werte in einem einzigen Aufruf mehrere Werte nacheinander an das Ende der Warteschlange anhängen.

Syntax:

ds_queue_enqueue(id, val [, val2, ... val15]);

ArgumentTyp Beschreibung
id The id of the queue to add to.
val The value to add to the queue.
[val2, ... val15] Optional values to be added to the queue.

Rückgabe:

Beispiel:

move_queue = ds_queue_create();
ds_queue_enqueue(move_queue, x + 200);
ds_queue_enqueue(move_queue, y);
ds_queue_enqueue(move_queue, x + 200);
ds_queue_enqueue(move_queue, y + 200);
ds_queue_enqueue(move_queue, x);
ds_queue_enqueue(move_queue, y + 200);
ds_queue_enqueue(move_queue, x);
ds_queue_enqueue(move_queue, y);

Die obige code erstellt eine neue DS-Warteschlange und speichert ihren Index in der Variablen "move_queue". Anschließend wird eine Reihe von Werten zur künftigen Verwendung in die Warteschlange verschoben.
