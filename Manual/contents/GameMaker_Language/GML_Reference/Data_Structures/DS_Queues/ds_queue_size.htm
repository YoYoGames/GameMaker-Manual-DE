ds_queue_size

Diese Funktion gibt die "Größe" der Warteschlange zurück, d. h. die Anzahl der Elemente, die in die Warteschlange aufgenommen wurden.

Syntax:

ds_queue_size(id);

ArgumentTyp Beschreibung
id The id of the data structure to check.

Rückgabe:

Beispiel:

if !ds_queue_empty(control_queue)
{
    num = ds_queue_size(control_queue);
}

Die obige code prüft eine DS-Warteschlange, um festzustellen, ob sie leer ist oder nicht. Ist dies nicht der Fall, wird die Anzahl der darin enthaltenen Elemente ermittelt und der Wert in einer Variablen gespeichert.
