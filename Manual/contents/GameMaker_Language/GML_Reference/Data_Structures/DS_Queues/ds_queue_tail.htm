ds_queue_tail

Diese Funktion liest nur den letzten Wert der Warteschlange (den, der "am Ende" steht). Der Wert wird nicht aus der Warteschlange entfernt, d.h. er kann auch in Zukunft noch von dieser Funktion oder der Funktion ds_queue_dequeue().

Syntax:

ds_queue_tail(id);

ArgumentTyp Beschreibung
id The id of the data structure to read from.

Rückgabe:

(Data type value stored in the queue)

Beispiel:

num = ds_queue_tail(control_queue);

Die obige code liest den Wert aus der in der Variablen "control_queue" indizierten Warteschlange und speichert den Rückgabewert in der Variablen "num".
