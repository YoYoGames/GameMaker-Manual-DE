ds_exists

Mit dieser Funktion können Sie prüfen, ob eine Datenstruktur des angegebenen Typs existiert. Sie geben den "Index"-Wert (wie er in einer Variablen gespeichert ist) und den DS-"Typ" an, der eine der unten aufgeführten Konstanten sein kann, und die Funktion gibt true zurück, wenn die Datenstruktur existiert, und andernfalls false.

DS Typ Konstante
Konstante Beschreibung
ds_type_map Eine Kartendatenstruktur
ds_type_list Eine Listendatenstruktur
ds_type_stack Eine Stack-Datenstruktur
ds_type_grid Eine Gitterdatenstruktur
ds_type_queue Eine Warteschlangen-Datenstruktur
ds_type_priority Eine vorrangige Datenstruktur

HINWEIS Sie können diese Funktion nicht zur Überprüfung des Typs einer Datenstruktur verwenden, da dieselbe Indexnummer von mehreren Datenstrukturen unterschiedlichen Typs verwendet werden kann.

Syntax:

ds_exists(ind, type)

Argument Typ Beschreibung
ind Jede DS-ID The variable index to check for the data structure
type DS Typ Konstante The type of data structure to check for (see the list of constants above)

Rückgabe:

Boolean

Beispiel:

if !ds_exists(ai_grid, ds_type_grid)
{
    ai_grid = ds_grid_create(room_width / 32, room_height / 32);
}

Die obige code prüft die (zuvor initialisierte) Variable "ai_grid", um festzustellen, ob sie eine Datenstruktur vom Typ DS-Grid indiziert, und wenn nicht, wird eine solche erstellt und ihr Index in der Variablen gespeichert.
