ds_list_is_list

Mit dieser Funktion können Sie prüfen, ob an der angegebenen Position innerhalb einer DS-Liste eine andere DS-Liste gespeichert ist. Wenn die angegebene Position eine DS-Listen-ID enthält, dann gibt die Funktion true zurück, andernfalls false.

Beachten Sie, dass dadurch nur Listen erkannt werden, die mit der Funktion ds_list_mark_as_list() manuell markiert wurden.

Syntax:

ds_list_is_list(id, pos);

ArgumentTyp Beschreibung
id The id of the list to check.
pos The position within the list to check.

Rückgabe:

Beispiel:

var size = ds_list_size(ships);
for (var i = 0; i < size; i++)
{
    if (ds_list_is_list(ships, i))
    {
        ds_list_destroy(ships[| i]);
    }
}
ds_list_destroy(ships);

Die obige code loops durchläuft eine DS-Liste und prüft, ob einer der Einträge andere Listen-IDs enthält. Wenn dies der Fall ist, werden diese Listen vernichtet, und die Hauptliste wird vernichtet, nachdem die loop beendet ist.
