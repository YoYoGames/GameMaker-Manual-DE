ds_list_size

Diese Funktion gibt die "Größe" der Liste zurück, d. h. die Anzahl der hinzugefügten Elemente.

Syntax:

ds_list_size(id);

ArgumentTyp Beschreibung
id The id of the data structure to check.

Rückgabe:

Beispiel:

if !ds_list_empty(control_list)
{
    num = ds_list_size(control_list);
}

Die obige code prüft eine DS-Liste, um zu sehen, ob sie leer ist oder nicht, und wenn nicht, erhält sie die Anzahl der Elemente, die sie enthält, und speichert den Wert in einer Variablen.
