ds_liste_lesen

Mit dieser Funktion können Sie eine gespeicherte DS-Liste wiederherstellen (eine Liste, die zuvor als String mit ds_list_write()). Sie müssen zuerst eine neue DS-Liste erstellen, um die Zeichenkette einzulesen. Wenn die DS-Liste bereits existiert und Informationen darin gespeichert sind, werden diese vor dem Einlesen gelöscht. Diese Funktion ist von entscheidender Bedeutung bei der Erstellung von Speicher-/Lademechanismen für Ihr Spiel.

Beachten Sie, dass Sie, wenn die angegebene DS-Zeichenkette von GameMaker geschrieben wurde: Studio 1.2.x runtime (oder älter) geschrieben wurde, sollten Sie das optionale argument "legacy" angeben und es auf true setzen, da sich das String-Format danach geändert hat.

Syntax:

ds_list_read(id, str [, legacy]);

Argument Typ Beschreibung
id DS-Liste ID The id of the data structure to read into.
str Zeichenfolge The string to read from.
legacy Boolesche OPTIONAL Kann entweder true oder false sein oder ganz weggelassen werden.

Rückgabe:

N/A

Beispiel:

list = ds_list_create();
ini_open("save.ini");
var str = ini_read_string("Lists", "0", "");
if (str != "")
{
    ds_list_read(list, str);
}
ini_close();

Die obige code erstellt eine Liste und speichert den Index in der Variablen "list". Dann wird eine Datei ini geöffnet und eine Zeichenkette daraus gelesen, wobei zunächst geprüft wird, ob die Zeichenkette nicht als leer zurückgegeben wird. Diese Zeichenkette wird dann in die neu erstellte ds_list eingelesen.
