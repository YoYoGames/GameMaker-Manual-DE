ds_liste_erstellen

Diese Funktion erstellt eine neue Listendatenstruktur und gibt den Indexwert zurück. Dieser Wert sollte in einer Variablen gespeichert und in allen weiteren Funktionsaufrufen in Bezug auf die Liste verwendet werden. Beachten Sie, dass der Rückgabewert 0 sein kann, wenn Sie versuchen, auf einen Wert in der Liste zuzugreifen , bevor diese gefüllt wurde - d.h. die Liste ist "leer" -, da intern die ersten paar Einträge in der Liste bei der Erstellung gesetzt werden, um Leistungsprobleme beim anfänglichen Hinzufügen von Elementen zur Liste zu minimieren (obwohl die ds_list_size() Funktion bei einer neu erstellten Liste immer noch 0 zurückgeben wird). Wenn Sie sicherstellen wollen, dass die Liste bei der Erstellung "wirklich" leer ist, sollten Sie die Funktion ds_list_clear() nach der Erstellung der Liste aufrufen, was dann bedeutet, dass alle Werte, die für nicht ausgefüllte Listenplätze zurückgegeben werden, undefined sind.

WICHTIG! Wenn Sie eine Datenstruktur anlegen, ist der Indexwert zu ihrer Identifizierung ein ganzzahliger Wert, der bei 0 beginnt. Das bedeutet, dass Datenstrukturen unterschiedlichen Typs denselben Indexwert haben können, daher sollten Sie im Zweifelsfall die ds_exists() Funktion verwenden, bevor Sie auf sie zugreifen. Beachten Sie auch, dass Indizes wiederverwendet werden, so dass ein Indexwert einer zerstörten Datenstruktur anschließend von einer neu erstellten verwendet werden kann.

Syntax:

ds_list_create();

Rückgabe:

DS List ID

Beispiel:

list = ds_list_create();

Dadurch wird eine neue Liste erstellt und ihre Index-ID der Instanzvariablen "list" zugewiesen.
