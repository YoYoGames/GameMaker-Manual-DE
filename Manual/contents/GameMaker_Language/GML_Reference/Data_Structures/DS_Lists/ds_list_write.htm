ds_list_write

Diese Funktion gibt eine Zeichenkette zurück, die dann gespeichert oder mit der Funktion ds_list_read() Funktion.

HINWEIS: Die zurückgegebene Zeichenkette ist keine vom Menschen lesbare Zeichenkette, sondern ein Dump des Inhalts der Datenstruktur.

Syntax:

ds_list_write(id);

ArgumentTyp Beschreibung
id The id of the data structure to write.

Rückgabe:

Beispiel:

ini_open("save.ini");
var str = ds_list_write(list);
ini_write_string("Lists", "0", str);
ds_list_clear(list);
ini_close();

Die obige code öffnet eine Ini-Datei und schreibt dann eine Zeichenkette mit den Informationen, die in der DS-Liste gespeichert sind, die in der Variablen "list" indiziert ist. Anschließend wird die Liste geleert und die Ini-Datei geschlossen.
