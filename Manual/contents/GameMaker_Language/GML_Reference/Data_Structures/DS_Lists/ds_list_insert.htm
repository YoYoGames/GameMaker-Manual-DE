ds_list_insert

Diese Funktion fügt den angegebenen Wert an der angegebenen Position in die Liste ein. Wenn die Liste nach der angegebenen Position weitere Werte enthält, wird deren Position um eins nach oben verschoben, um Platz zu schaffen, wodurch die Liste um eins größer wird.

Syntax:

ds_list_insert(id, pos, val);

Argument Typ Beschreibung
id DS-Liste ID The id of the list to add to.
pos Real The position to add the value, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.
val Variabel The value to add to the list.

Rückgabe:

N/A

Beispiel:

ds_list_insert(list, 9, score);

Die obige code fügt den in der Variablen "score" gespeicherten Wert an die 10. Stelle der durch die Variable "list" indizierten Liste hinzu (Listen werden von 0 an gezählt, eine Liste mit zehn Werten ist also von 0 bis 9 nummeriert).
