ds_list_add

Mit dieser Funktion kann ein neuer Wert (real oder string) zur Liste hinzugefügt werden, der am Ende angefügt wird. Die Funktion kann weitere optionale arguments annehmen (so viele wie Sie benötigen), so dass Sie mit einem einzigen Aufruf mehrere Werte nacheinander zur Liste hinzufügen können.

Syntax:

ds_list_add(id, val1 [, val2, ... max_val]);

Argument Typ Beschreibung
id DS-Liste ID The id of the list to add to.
val Variabel The value to add to the list.
[val2, ... max_val] Variabel Optional values to be added to the list.

Rückgabe:

N/A

Beispiel:

ds_list_add(sc_list, score);

Die obige code fügt den in der Variablen "score" gespeicherten Wert in die in der Variablen "sc_list" indizierte Liste ein.
