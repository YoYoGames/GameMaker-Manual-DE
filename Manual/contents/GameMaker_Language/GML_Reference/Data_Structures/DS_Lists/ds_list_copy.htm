ds_list_copy

Mit dieser Funktion können Sie den Inhalt einer Liste in eine andere kopieren. Beide Listen müssen zuvor erstellt worden sein, und wenn die Liste, in die kopiert werden soll, bereits Informationen enthält, wird diese Liste zuerst gelöscht. Das Endergebnis sind zwei unabhängige Listen, die die gleichen Informationen enthalten.

Syntax:

ds_list_copy(destination, source);

ArgumentTyp Beschreibung
id The id of the list being copied to.
source The id of the list to be copied from.

Rückgabe:

Beispiel:

if !ds_list_empty(main_list)
{
old_list = ds_list_create();
ds_list_copy(old_list, main_list);
ds_list_clear(main_list);
}

Die obige code prüft eine DS-Liste, um festzustellen, ob sie leer ist. Wenn sie nicht leer ist, wird sie in eine andere DS-Liste kopiert (die zuvor erstellt wurde) und dann wird die ursprüngliche Liste geleert.
