ds_liste_markieren_als_liste

Diese Funktion "markiert" (oder "kennzeichnet") eine bestimmte Position innerhalb einer zuvor erstellten DS-Liste als eine andere DS-Liste enthaltend. Diese Funktionalität ist erforderlich, wenn JSON-Strings kodiert werden (die Sie mit json_encode()), kann aber auch bei der Verschachtelung von Datenstrukturen nützlich sein, da auf diese Weise markierte Elemente automatisch entsorgt (zerstört) werden, wenn die übergeordnete DS-Liste zerstört wird. Dies bedeutet, dass Sie nicht manuell durch den Listeninhalt gehen und die markierten Datenstrukturen einzeln zerstören müssen, bevor Sie die "übergeordnete" Liste zerstören. Wenn Sie jedoch die Listenposition einzeln löschen, wird die darin enthaltene DS-Liste nicht gelöscht, und Sie sollten die entsprechende Funktion zum Zerstören der DS-Liste aufrufen, bevor Sie das Element der übergeordneten Listenposition löschen. Beachten Sie auch, dass bei einem Aufruf der Funktion ds_list_clear() für eine Liste aufrufen, werden alle Elemente, die als Listen gekennzeichnet sind, ebenfalls zerstört, wenn die Liste geleert wird.

Wenn sich die markierte Position innerhalb der Liste ändert, verschiebt sich auch die "Markierung". Wenn Sie also z. B. die Position 3 in der Liste markiert haben und dann zwei weitere Elemente darunter einfügen, so dass die Liste auf Position 5 rückt, wird sie weiterhin als Liste markiert.

Syntax:

ds_list_mark_as_list(id, pos);

ArgumentTyp Beschreibung
id The id of the list to mark.
pos The position within the list to mark.

Rückgabe:

DS List ID or -1 (if it fails)

Beispiel:

var j_list = ds_list_create();
var sub_list = ds_list_create();
ds_list_add(sub_list, health);
ds_list_add(sub_list, lives);
ds_list_add(sub_list, score);
ds_list_add(j_list, sub_list);
ds_list_mark_as_list(j_list, 0);

Die obige code erstellt zwei DS-Listen und füllt dann eine mit verschiedenen Werten aus globalen Variablen. Diese Liste wird dann in die zweite Liste eingefügt und die Position als solche "markiert", damit sie später korrekt kodiert werden kann.
