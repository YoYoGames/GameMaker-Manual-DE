ds_list_find_index

Mit dieser Funktion können Sie die angegebene Liste auf einen Wert prüfen, und die Position dieses Wertes innerhalb der Liste wird zurückgegeben. Beachten Sie, dass bei mehreren Einträgen in der Liste mit demselben Wert die Position eines beliebigen Eintrags zurückgegeben werden kann, und dass, wenn der Wert nicht existiert, -1 zurückgegeben wird.

Syntax:

ds_list_find_index(id, val);

Argument Typ Beschreibung
id DS-Liste ID The id of the list to use.
val Variabel The value to find.

Rückgabe:

Real

Beispiel:

pos = ds_list_find_index(list, "Player1");

Die obige code überprüft die in der Variablen "list" indizierte Liste auf den Wert "Player1" und speichert die zurückgegebene Position in der Variablen "pos".
