ds_list_shuffle

Mit dieser Funktion können Sie eine Liste mischen, wodurch alle Komponentenwerte in zufällige Positionen umgeordnet werden, die von denen abweichen, an denen sie ursprünglich der Liste hinzugefügt wurden.

HINWEIS: Diese Funktion mischt die Listenelemente bei jedem neuen Durchlauf des Spiels an denselben Positionen, da GameMaker jedes Mal denselben anfänglichen Zufallswert erzeugt, um debugging code zu vereinfachen. Um dieses Verhalten zu vermeiden, verwenden Sie randomise() beim Start des Spiels. Dies gilt nur für das Testen und debugging des Spiels, da das endgültige ausführbare Paket dieses Verhalten nicht zeigt und bei jedem Spiel zufällig ist.

Syntax:

ds_list_shuffle(id);

ArgumentTyp Beschreibung
id The id of the list to shuffle.

Rückgabe:

Beispiel:

if restart
{
    ds_list_shuffle(card_list);
}

Die obige code wird die in der Variablen "card_list" indizierte Liste mischen, wenn die Variable "restart" als true gekennzeichnet ist.
