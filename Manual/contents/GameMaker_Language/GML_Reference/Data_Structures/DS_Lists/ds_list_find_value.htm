ds_list_find_value

Mit dieser Funktion können Sie die angegebene Listenposition überprüfen, und der in der Liste enthaltene Wert für diese Position wird zurückgegeben. Beachten Sie, dass die Funktion undefined oder 0 zurückgeben kann, wenn Sie eine Position angeben, die außerhalb der angegebenen Listengröße liegt (z.B.: Position 11 in einer Liste mit 10 Werten). Das liegt daran, dass beim Erstellen der Liste intern die ersten paar Einträge in der Liste auf 0 gesetzt werden, um Leistungsprobleme beim anfänglichen Hinzufügen von Elementen zur Liste zu minimieren (obwohl die ds_list_size() Funktion auch bei einer neu erstellten Liste 0 zurückgibt). Wenn Sie sicherstellen wollen, dass die Liste beim Erstellen "wirklich" leer ist, sollten Sie die Funktion ds_list_clear() aufrufen, nachdem Sie die Liste erstellt haben. Das bedeutet, dass alle Werte, die für nicht ausgefüllte Listenplätze zurückgegeben werden, undefined sind, was Sie dann mit der is_undefined() Funktion.

Syntax:

ds_list_find_value(id, pos);

Argument Typ Beschreibung
id DS-Liste ID The id of the list to use.
pos Real The position to look at, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.

Rückgabe:

Variable or undefined

Beispiel:

val = ds_list_find_value(list, ds_list_size(list) - 1);

Die obige code prüft die in der Variablen "list" indizierte Liste an der letzten Position in der Liste und speichert den zurückgegebenen Wert in der Variablen "val".
