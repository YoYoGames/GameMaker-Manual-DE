ds_list_sort

Mit dieser Funktion können Sie alle Werte in einer Liste entweder in aufsteigender oder absteigender Reihenfolge sortieren. Wenn die Liste Zeichenketten enthält, werden diese alphabetisch sortiert, basierend auf dem englischen Alphabet mit 26 Buchstaben.

Syntax:

ds_list_sort(id, ascend);

ArgumentTyp Beschreibung
id The id of the list to sort.
ascend Whether the values should be ascending (true) oder in absteigender (false) Reihenfolge.

Rückgabe:

Beispiel:

if newgame
{
    ds_list_sort(name_list, true);
}

Die obige code sortiert die in der Variablen "name_list" indizierte Liste, wenn die Variable "newgame" als true gekennzeichnet ist.
