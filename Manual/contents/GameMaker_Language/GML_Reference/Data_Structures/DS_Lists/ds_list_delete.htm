ds_list_delete

Mit dieser Funktion können Sie den Wert entfernen, der an einer bestimmten Position in der Liste gespeichert ist.

Syntax:

ds_list_delete(id, pos);

ArgumentTyp Beschreibung
id The id of the list to change.
pos Where in the list to delete the value.

Rückgabe:

Beispiel:

if (ds_list_size(sc_list) > 10)
{
    while (ds_list_size(sc_list) > 10)
    {
        ds_list_delete(sc_list, 0);
    }
}

Die obige code prüft die Größe einer DS-Liste, und wenn sie größer als zehn ist, wird loops durch die Liste geführt, wobei der oberste Wert (Position 0) entfernt wird, bis die Liste nur noch 10 Einträge hat.
