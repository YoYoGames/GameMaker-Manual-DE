ds_list_set

Diese Funktion kann verwendet werden, um einen zuvor hinzugefügten Listeneintrag zu setzen. Sie geben die Listen-ID (wie sie bei der Erstellung der Liste zurückgegeben wurde) und die Position innerhalb der Liste sowie den Wert an, auf den er gesetzt werden soll. Liegt der zu setzende Eintrag außerhalb der Liste (d.h. Sie setzen den Listeneintrag 20, die aktuelle Liste enthält aber nur 10 Einträge), wird die Liste bis zur angegebenen Position gefüllt und jeder Eintrag auf 0 gesetzt. Diese Funktion entspricht der Verwendung des DS-Listen-Accessors.

Syntax:

ds_list_set(id, pos, val);

Argument Typ Beschreibung
id DS-Liste ID The id of the list to add to.
pos Variabel The position within the list to set.
[val2, ... max_val] Variabel The value to set in the list.

Rückgabe:

N/A

Beispiel:

for (var i = 0; i < ds_list_size(list); i++;)
{
    ds_list_set(list, i, -1);
}

Die obige code fügt den in der Variablen "score" gespeicherten Wert in die in der Variablen "sc_list" indizierte Liste ein.
