ds_list_mark_as_map

Diese Funktion "markiert" (oder "kennzeichnet") eine bestimmte Position innerhalb einer zuvor erstellten DS-Liste als eine DS-Map enthaltend. Diese Funktionalität ist erforderlich, wenn JSON-Strings kodiert werden (die Sie mit json_encode()), kann aber auch bei der Verschachtelung von Datenstrukturen nützlich sein, da auf diese Weise markierte Elemente automatisch entsorgt (zerstört) werden, wenn die übergeordnete DS-Liste zerstört wird. Dies bedeutet, dass Sie nicht manuell durch den Listeninhalt gehen und die markierten Datenstrukturen einzeln zerstören müssen, bevor Sie die "übergeordnete" Liste zerstören. Wenn Sie jedoch die Listenposition einzeln löschen, wird die darin enthaltene Datenstruktur nicht gelöscht, und Sie sollten die entsprechende DS-Map-Zerstörungsfunktion aufrufen, bevor Sie die übergeordnete Listenposition löschen. Beachten Sie auch, dass beim Aufruf der Funktion ds_list_clear() für eine Liste aufrufen, werden alle als Maps gekennzeichneten Elemente ebenfalls zerstört, wenn die Liste geleert wird.

Wenn sich die markierte Position innerhalb der Liste ändert, verschiebt sich auch die "Markierung". Wenn Sie also z. B. die Position 3 in der Liste markiert haben und dann zwei weitere Elemente darunter einfügen, so dass die Position 5 erreicht wird, ist sie immer noch als Karte markiert.

Syntax:

ds_list_mark_as_map(id, pos);

ArgumentTyp Beschreibung
id The id of the list to mark.
pos The position within the list to mark.

Rückgabe:

DS Map ID or -1 (if it fails)

Beispiel:

var sub_map = ds_map_create();
ds_map_add(sub_map, "player", player_array);
ds_map_add(sub_map, "enemy", enemy_array);
ds_list_add(j_list, sub_map);
ds_list_mark_as_map(j_list, 0);

Die obige code erstellt eine DS-Map und füllt sie dann mit zwei Schlüsseln, die jeweils ein Array von Werten enthalten. Diese Karte wird dann in die angegebene DS-Liste eingefügt und die Position als solche "markiert", damit sie später korrekt kodiert werden kann.
