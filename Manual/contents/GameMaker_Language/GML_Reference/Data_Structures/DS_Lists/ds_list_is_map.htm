ds_list_is_map

Mit dieser Funktion können Sie prüfen, ob eine DS-Map an der angegebenen Position in einer DS-Liste gespeichert ist. Wenn die angegebene Position eine DS-Map-ID enthält, dann gibt die Funktion true zurück, andernfalls false.

Beachten Sie, dass dadurch nur Maps erkannt werden, die manuell über die Funktion ds_list_mark_as_map() markiert wurden.

Syntax:

ds_list_is_map(id, pos);

ArgumentTyp Beschreibung
id The id of the list to check.
pos The position within the list to check.

Rückgabe:

Beispiel:

var size = ds_list_size(ships);
for (var i = 0; i < size; i++)
{
    if (ds_list_is_map(ships, i))
    {
        ds_map_destroy(ships[| i]);
    }
}
ds_list_destroy(ships);

Die obige code loops durchläuft eine DS-Liste und prüft, ob einer der Einträge Karten-IDs enthält. Wenn dies der Fall ist, werden diese Karten vernichtet, und die Hauptliste wird vernichtet, nachdem loop beendet ist.
