ds_map_replace_list

Mit dieser Funktion können Sie eine DS-Liste, die im angegebenen "Schlüssel" gespeichert wurde, durch eine andere, zuvor erstellte Liste ersetzen. Diese Funktion ist für die Erstellung von JSON-kompatiblen Maps gedacht, die Sie dann mit json_encode() kodieren und sollte nur in Verbindung mit dieser Funktion verwendet werden.

Syntax:

ds_map_replace_list(id, key, value)

Argument Typ Beschreibung
id DS-Karten-ID The id of the map to use.
key Zeichenfolge The key to replace.
value DS-Liste ID The id of the ds_list to use to replace the one previously stored in the given key.

Rückgabe:

N/A

Beispiel:

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
ds_map_replace_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_list_destroy(j_list);

Die obige code erstellt eine DS List und füllt sie mit den Werten verschiedener globaler Variablen, bevor sie eine zuvor gespeicherte Liste in der DS Map "j_map" ersetzt.
