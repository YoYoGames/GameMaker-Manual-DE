ds_map_keys_to_array

Mit dieser Funktion können Sie alle Schlüssel abrufen, die eine DS-Map enthält. Sie geben die ID der DS-Map an, von der Sie die Schlüssel abrufen wollen (wie sie von ds_map_create()) und die Funktion gibt ein Array zurück, bei dem jeder Eintrag im Array ein Schlüssel aus der DS-Map ist. Die Funktion hat eine optionale zweite argument, in der Sie ein von Ihnen erstelltes Array angeben können. In diesem Fall werden die Schlüsseldaten der DS-Map an alle vorhandenen Daten im Array angehängt. Beachten Sie, dass die Funktion das übergebene Array direkt ändert, aber auch einen Verweis darauf zurückgibt (oder einen Verweis auf ein neues Array, wenn keines übergeben wird).

Syntax:

ds_map_keys_to_array(id, [array])

Argument Typ Beschreibung
id DS-Karten-ID The id of the map to use.
[array] Array OPTIONAL Das Array, an das die DS-Map-Schlüssel angehängt werden.

Rückgabe:

Array

Beispiel:

map_keys = ds_map_keys_to_array(inventory);

Die obige code ruft die Schlüssel für eine DS-Map ab und speichert sie dann als Array zur späteren Verwendung.
