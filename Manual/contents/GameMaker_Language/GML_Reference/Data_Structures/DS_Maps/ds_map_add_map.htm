ds_map_add_map

Mit dieser Funktion können Sie eine (zuvor erstellte) DS-Map einem Schlüssel innerhalb der angegebenen DS-Map zuordnen. Diese Funktion ist für die Erstellung von JSON-kompatiblen Maps gedacht, die Sie dann mit json_encode() kodieren, und sollte nur in Verbindung mit dieser Funktionalität verwendet werden. Wenn einer DS-Map eine andere Map auf diese Weise hinzugefügt wird, dann wird das Zerstören der übergeordneten Map auch die enthaltenen Maps zerstören und deren Speicher freigeben, und der Aufruf von ds_map_clear() auf der übergeordneten Map werden auch alle markierten Maps zerstört und bereinigt.

Syntax:

ds_map_add_map(id, key, value)

Argument Typ Beschreibung
id DS-Karten-ID The id of the map to use.
key Zeichenfolge The key for the added map.
value DS-Karten-ID The id of the map to add.

Rückgabe:

N/A

Beispiel:

var j_map = ds_map_create();
var j_list = ds_list_create();
var sub_map = ds_map_create();
ds_map_add_list(sub_map, "list", j_list);
ds_map_add(sub_map, "array", j_array);
ds_map_add_map(j_map, "map", sub_map);
var j = json_encode(j_map);
ds_map_destroy(j_map);

Die obige code erstellt zwei DS-Maps und füllt eine davon mit einer Liste und einem Array, bevor sie der zweiten hinzugefügt wird, die dann in einen JSON-String kodiert wird. Die Map wird dann zerstört, um sie und alle anderen Maps oder Listen, die sie enthält, aus dem Speicher zu entfernen.
