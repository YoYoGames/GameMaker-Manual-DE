ds_map_add

Diese Funktion sollte verwendet werden, um Sätze von Schlüssel/Wertpaaren in die angegebene DS-Map einzufügen. Sie können diese Funktion überprüfen, um zu sehen, ob sie erfolgreich war oder nicht (bei Erfolg gibt sie true zurück, ansonsten false ), da sie fehlschlagen kann, wenn derselbe Schlüssel bereits in der DS-Map existiert oder Sie eine nicht existierende DS-Map als ID der Map angeben, zu der hinzugefügt werden soll. Die Schlüssel und Werte, die Sie angeben, können aus einer beliebigen Kombination von Datentypen bestehen, so dass alle der folgenden - und mehr - sind akzeptabel (obwohl Sie in der Praxis am häufigsten eine Zeichenkette für den Schlüssel verwenden):

ds_map_add(map, 5, undefined);
ds_map_add(map, "level", 1);
ds_map_add(map, 89.6, "hello world");
ds_map_add(map, 5, buffer_get_address(buff));

Sie können eine Karte auch mit dem Accessor "?" ergänzen, wie unten gezeigt:

map[? 5] = undefined;
map[? "level"] = 1;
map[? 89.6] = "hello world";
map[? 5] = buffer_get_address(buff);

HINWEIS: Im Gegensatz zu anderen Datenstrukturen in GameMaker wird dieser Schlüssel nicht an den Anfang (und auch nicht an das Ende) der DS-Map gesetzt, sondern er wird einfach irgendwo in die DS-Map gesetzt.

Syntax:

ds_map_add(id, key, val);

ArgumentTyp Beschreibung
id The id of the map to add to.
key The key of the value to add.
val The value to add to the map.

Rückgabe:

Beispiel:

inventory = ds_map_create();
ds_map_add(inventory, "hp potion", 1);
ds_map_add(inventory, "gold", 100);

Dadurch wird eine neue Karte erstellt, deren ID der Variablen "inventory" zugewiesen wird. Dann werden der Karte zwei neue Schlüssel hinzugefügt, "hp potion" und "gold", und ihre Anfangswerte auf 1 und 100 gesetzt.
