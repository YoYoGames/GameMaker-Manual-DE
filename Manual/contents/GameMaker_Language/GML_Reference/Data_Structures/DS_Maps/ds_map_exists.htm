ds_map_exists

Diese Funktion gibt true zurück, wenn der angegebene Schlüssel in der (zuvor erstellten) DS-Map existiert, und false, wenn dies nicht der Fall ist.

Syntax:

ds_map_exists(id, key);

Argument Typ Beschreibung
id DS-Karten-ID The id of the data structure to check
key Zeichenfolge The key to check for

Rückgabe:

Boolean

Beispiel:

if !ds_map_exists(inventory, "potions")
{
    ds_map_add(inventory, "potions", 1);
}

Die obige code prüft die DS-Karte, die in der Variable "inventory" indiziert ist, auf den Schlüssel "trions" und fügt ihn, falls er nicht existiert, der Karte hinzu.
