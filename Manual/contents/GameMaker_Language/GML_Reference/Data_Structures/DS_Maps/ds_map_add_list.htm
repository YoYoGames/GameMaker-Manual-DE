ds_map_add_list

Mit dieser Funktion können Sie eine (zuvor erstellte) DS-Liste einem Schlüssel innerhalb der angegebenen DS-Map zuordnen. Diese Funktion ist für die Erstellung von JSON-kompatiblen Maps gedacht, die Sie dann mit json_encode() kodieren, und sollte nur in Verbindung mit dieser Funktionalität verwendet werden. Wenn einer DS-Map auf diese Weise eine Liste hinzugefügt wurde, werden beim Zerstören der übergeordneten Map auch die enthaltenen Listen zerstört und ihr Speicher freigegeben, und der Aufruf von ds_map_clear() auf der übergeordneten Map wird auch alle markierten Listen zerstören und bereinigen.

Syntax:

ds_map_add_list(id, key, value)

Argument Typ Beschreibung
id DS-Karten-ID The id of the map to use.
key Zeichenfolge The key for the added list.
value DS-Liste ID The id of the list to add.

Rückgabe:

N/A

Beispiel:

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
var j_map = ds_map_create();
ds_map_add_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_map_destroy(j_map);

Die obige code erstellt eine Liste und füllt sie mit den verschiedenen Werten globaler Variablen. Diese Liste wird dann in einer DS-Map "verschachtelt", und die Map wird dann in eine JSON-Zeichenfolge kodiert, bevor die Map zerstört wird, wobei sie und alle darin enthaltenen Listen aus dem Speicher entfernt werden.
