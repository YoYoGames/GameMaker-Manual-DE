ds_map_clear

Diese Funktion löscht die (zuvor erstellte) DS-Map von allen Schlüssel/Wert-Paaren, aber sie löscht nicht die DS-Map selbst aus dem Speicher. Hierfür sollten Sie die Funktion ds_map_destroy(). Beachten Sie, dass das Löschen einer Map alle in der Map gespeicherten Datenstrukturen de-referenziert, was zu einem Speicherleck führt. Dies ist nur dann nicht erforderlich, wenn Sie Elemente in der Map als DS-Liste oder als andere DS-Map gekennzeichnet haben. In diesem Fall werden diese Elemente zerstört (nicht gelöscht!) und ihr Speicher automatisch bereinigt, wenn die übergeordnete Map gelöscht wird.

Syntax:

ds_map_clear(id);

ArgumentTyp Beschreibung
id The id of the data structure to clear.

Rückgabe:

Beispiel:

if global.new_game
{
    ds_map_clear(inventory);
}

Die obige Funktion prüft eine globale Variable, um zu sehen, ob sie wahr ist, und wenn dies der Fall ist, wird die DS-Map, die in der Instanzvariablen "inventory" indiziert ist, von allen Schlüssel/Wertpaaren befreit (aber nicht zerstört).
