ds_map_copy

Mit dieser Funktion können Sie den Inhalt einer Karte in eine andere Karte kopieren, die Sie zuvor mit ds_map_create() erstellt haben. Wenn die DS-Karte, in die kopiert wird, nicht leer ist, dann wird diese Funktion sie vor dem Kopieren löschen. Die ursprüngliche DS-Karte bleibt bei diesem Vorgang unverändert.

Syntax:

ds_map_copy(id, source);

ArgumentTyp Beschreibung
id The id of the map you are copying to
source The id of the map you are copying from

Rückgabe:

Beispiel:

inventory_2 = ds_map_create();
ds_map_copy(inventory_2, inventory_1);

Die obige code erstellt eine neue Karte und weist sie der Variablen "inventory_2" zu. Anschließend wird der Inhalt der in der Variablen "inventory_1" indizierten DS-Karte in diese neue Karte kopiert.
