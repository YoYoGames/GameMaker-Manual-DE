ds_map_ist_map

Mit dieser Funktion können Sie prüfen, ob eine DS-Map im angegebenen Map-Schlüssel gespeichert ist. Wenn der angegebene Schlüssel eine DS-Map-ID enthält, dann gibt die Funktion true zurück, andernfalls false.

Beachten Sie, dass dies nur Maps erkennt, die mit der Funktion ds_map_add_map() hinzugefügt wurden.

Syntax:

ds_map_is_map(id, key)

ArgumentTyp Beschreibung
id The id of the map to use.
key The key to replace.

Rückgabe:

Beispiel:

var size = ds_map_size(inventory);
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++)
{
    if (ds_map_is_map(inventory, key))
    {
        ds_map_destroy(inventory[? key]);
    }
    key = ds_map_find_next(inventory);
}
ds_map_destroy(inventory);

Die obige code loops durchläuft eine DS-Map und prüft, ob einer der Schlüssel darin für andere DS-Maps bestimmt ist. Ist dies der Fall, wird die gespeicherte DS-Map zerstört, und am Ende von loop wird auch die Haupt-DS-Map zerstört.
