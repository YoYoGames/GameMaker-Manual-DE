ds_map_create

Diese Funktion wird verwendet, um eine neue, leere DS-Map zu erstellen, und gibt ihre ID zurück, die dann für den Zugriff auf die Datenstruktur in allen anderen DS-Map-Funktionen verwendet wird.

WICHTIG! Wenn Sie eine Datenstruktur anlegen, ist der Indexwert zu ihrer Identifizierung ein ganzzahliger Wert, der bei 0 beginnt. Das bedeutet, dass Datenstrukturen unterschiedlichen Typs denselben Indexwert haben können, so dass Sie im Zweifelsfall die ds_exists() Funktion verwenden, bevor Sie auf sie zugreifen. Beachten Sie auch, dass Indizes wiederverwendet werden, so dass ein Indexwert einer zerstörten Datenstruktur anschließend von einer neu erstellten verwendet werden kann.

Syntax:

ds_map_create();

Rückgabe:

DS Map ID

Beispiel:

inventory = ds_map_create();

Die obige code erstellt eine neue, leere DS-Map und speichert ihren ID-Index in der Variablen "inventory".
