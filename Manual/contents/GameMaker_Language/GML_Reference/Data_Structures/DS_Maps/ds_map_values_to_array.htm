ds_map_values_to_array

Mit dieser Funktion können Sie alle Werte abrufen, die eine DS-Map für jeden Schlüssel in der Map enthält. Sie geben die ID der DS-Map an, von der Sie die Schlüssel abrufen möchten (wie sie von ds_map_create()) und die Funktion gibt ein Array zurück, in dem jeder Eintrag ein Wert aus der DS-Map ist (wenn die Map beispielsweise 10 Schlüssel hat, wird ein Array mit 10 Einträgen zurückgegeben, wobei jeder Eintrag einem Wert für einen Schlüssel entspricht). Die Funktion verfügt über eine optionale zweite argument, in der Sie ein von Ihnen erstelltes Array angeben können. In diesem Fall werden die Schlüsselwertdaten der Map an alle vorhandenen Daten im Array angehängt. Beachten Sie, dass die Funktion das übergebene Array direkt ändert, aber auch einen Verweis darauf zurückgibt (oder einen Verweis auf ein neues Array, wenn keines übergeben wird).

Syntax:

ds_map_values_to_array(id, [array])

Argument Typ Beschreibung
id DS-Karten-ID The id of the map to use.
[array] Array OPTIONAL Das Array, an das die DS-Map-Werte angehängt werden.

Rückgabe:

Array

Beispiel:

var _values = ds_map_values_to_array(lvl_score);
var _total = 0;
var _length = array_length(_values);

for (var i = 0; i < _length; i ++)
{
    _total += _values[i];
}

draw_text(32, 32, "Total score for all levels: " + string(_total));

Die obige Funktion ruft alle Werte aus einer DS-Map für jeden Schlüssel in der Map ab und speichert sie in einer lokalen Variablen als Array. Anschließend durchläuft loops das Array und fügt jeden der Werte zu einer lokalen Variablen hinzu, die dann verwendet wird, um die Summe aller Werte anzuzeigen.
