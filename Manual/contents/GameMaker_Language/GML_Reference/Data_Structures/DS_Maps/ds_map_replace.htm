ds_map_replace

Mit dieser Funktion können Sie den Wert für den angegebenen Schlüssel innerhalb einer DS-Map ändern. Sie geben den Index der Map an (wie er von der Funktion ds_map_create()) und dann den zu ersetzenden Schlüssel - entweder eine Zeichenkette oder eine ganze Zahl - und den Wert, durch den der Schlüsselwert ersetzt werden soll. Wenn der angegebene Schlüssel nicht existiert, wird er für Sie erstellt, und wenn er existiert, wird der aktuelle Wert durch den neuen Wert ersetzt. Wert ersetzt. Die Funktion gibt true zurück, wenn der Schlüssel existiert und der Wert ersetzt wurde, und false, wenn der Schlüssel nicht existiert und ein neuer Schlüssel mit dem Wert erstellt wurde.

Syntax:

ds_map_replace( id, key, val );

ArgumentTyp Beschreibung
id The id of the map to change.
key The key with the value that should be replaced by the new one
val The new value to replace the given value with

Rückgabe:

Beispiel:

ds_map_replace(inventory, "torso", 55);

Die obige code sucht in der DS-Map nach dem Schlüssel "torso", und wenn sie ihn findet (oder wenn er nicht existiert, wird er erstellt), wird der aktuelle Wert durch den angegebenen ersetzt.
