ds_map_empty

Diese Funktion gibt einfach false zurück, wenn die angegebene (zuvor erstellte) DS-Map irgendwelche Schlüssel/Wert-Paare enthält, oder true, wenn sie keine enthält.

Syntax:

ds_map_empty(id);

ArgumentTyp Beschreibung
id The id of the data structure to check.

Rückgabe:

Beispiel:

if ds_map_empty(inventory)
{
    weight = 0;
}

Die obige code prüft, ob die in der Variable "inventory" indizierte DS-Map Schlüssel/Wert-Paare enthält, und wenn nicht, setzt sie die Variable "weight" auf 0.
