ds_map_secure_load

Mit dieser Funktion wird eine zuvor gespeicherte DS-Karte aus der angegebenen Datei geladen. Diese Datei muss zuvor mit der Funktion ds_map_secure_save() Funktion erstellt worden sein, und nach dem Laden gibt die Funktion den Index der DS-Map zurück, die aus den geladenen Daten erstellt wurde. Dieser DS-Map-Index sollte in einer Variablen gespeichert werden und für alle weiteren Funktionsaufrufe zu dieser Map verwendet werden. Wurde die zu ladende DS-Map mit einem Array als einem der Schlüsselwerte gespeichert, wird dieses Array beim Laden in eine DS-Liste umgewandelt.

WICHTIG! Eine der Eigenschaften einer sicher gespeicherten Datei ist, dass sie an das Gerät gebunden ist, auf dem sie erstellt wurde. Sie können also eine auf einem Gerät gespeicherte Datei nicht in ein Projekt laden, das auf einem anderen Gerät läuft.

Syntax:

ds_map_secure_load(filename);

ArgumentTyp Beschreibung
filename The name of the file to load the map data from.

Rückgabe:

DS Map ID

Beispiel:

p_map = ds_map_secure_load("p_data.dat");

Die obige code lädt eine sicher gespeicherte DS-Map und speichert ihren Indexwert in einer Variablen zur späteren Verwendung.
