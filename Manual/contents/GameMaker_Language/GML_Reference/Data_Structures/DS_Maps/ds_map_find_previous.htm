ds_map_find_previous

Diese Funktion gibt den vorherigen Schlüssel zurück, der in der DS-Map vor dem in der Funktion angegebenen Schlüssel gespeichert ist. Dies kann nützlich sein, wenn Sie die DS-Map auf der Suche nach etwas durchlaufen müssen, sollte aber nach Möglichkeit vermieden werden, da es langsam sein kann. Wenn kein solcher Schlüssel existiert, gibt die Funktion undefined zurück. Sie sollten dies immer mit der is_undefined() Funktion.

Syntax:

ds_map_find_previous(id, key)

ArgumentTyp Beschreibung
id The id of the map to use.
key The key to find the previous one to.

Rückgabe:

Variable or undefined

Beispiel:

var size = ds_map_size(inventory) - 1;
var key = ds_map_find_last(inventory);
for (var i = size; i > 0; i--;)
{
    if (key != "gold")
    {
        _key = ds_map_find_previous(inventory, _key);
    }
    else break;
}

Die obige code erstellt einige temporäre Variablen, ruft dann die Größe der ds_map ab und findet den letzten Schlüssel, den der Computer in der map gespeichert hat. Dann wird ein for loop verwendet, um die ds_map zu durchlaufen und nach dem Schlüsselwert "gold" zu suchen. Wenn er ihn findet, gibt er die loop aus.
