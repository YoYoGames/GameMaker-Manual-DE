ds_map_find_value

Mit dieser Funktion können Sie den Wert eines bestimmten Schlüssels ermitteln. Die Eingabewerte der Funktion sind die (zuvor erstellte) DS-Map, die verwendet werden soll, und der Schlüssel, auf den geprüft werden soll.

HINWEIS: Wenn kein solcher Schlüssel existiert, gibt die Funktion undefined zurück. Sie sollten dies immer mit der is_undefined() Funktion

Syntax:

ds_map_find_value(id, key)

ArgumentTyp Beschreibung
id The id of the map to use.
key The key to find.

Rückgabe:

Variable or undefined

Beispiel:

amount = ds_map_find_value(inventory, "food");

Oder Sie verwenden den Map-Accessor "?":

amount = inventory[? "food"];

Die obige code ermittelt den Wert des Schlüssels "food" und speichert ihn in der Variablen "amount".
