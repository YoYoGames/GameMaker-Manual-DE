ds_map_set

Mit dieser Funktion können Sie den Wert eines Schlüssels innerhalb einer bestimmten DS-Map setzen. Sie geben den ID-Wert der DS-Map an (wie er von der Funktion ds_map_create()zurückgegeben wird), dann geben Sie den Schlüssel an, den Sie setzen wollen, und den Wert, auf den er gesetzt werden soll. Schlüssel können Ganzzahlen oder Strings sein, und wenn der angegebene Schlüssel nicht existiert, wird er für Sie erstellt und auf den Wert gesetzt. Diese Funktion entspricht der Verwendung des DS-Map-Zugriffs zum Setzen/Erstellen eines Schlüssel/Wert-Paares für eine Map. Die Funktion gibt nichts zurück. Wenn Sie also prüfen müssen, ob der Schlüsselwert ersetzt oder ein neuer Schlüssel erstellt wurde, sollten Sie die Funktion ds_map_replace() verwenden.

Syntax:

ds_map_set(id, key, value)

ArgumentTyp Beschreibung
id The id of the map to use.
key The key to set.
value The value to set the key to.

Rückgabe:

Beispiel:

if is_undefined(ds_map_find_value(map, "score"))
{
    ds_map_set(map, "score", 0);
}

Die obige code prüft, ob der angegebene Schlüssel existiert, und wenn nicht, wird er erstellt und gesetzt.
