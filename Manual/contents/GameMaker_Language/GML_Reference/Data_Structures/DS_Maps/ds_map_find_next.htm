ds_map_find_next

Diese Funktion gibt den nächsten in der DS-Map gespeicherten Schlüssel nach dem in der Funktion angegebenen Schlüssel zurück. Dies kann nützlich sein, wenn Sie die DS-Map auf der Suche nach etwas durchlaufen müssen, sollte aber nach Möglichkeit vermieden werden, da es langsam sein kann. Wenn kein solcher Schlüssel existiert, gibt die Funktion undefined zurück. Sie sollten dies immer mit der is_undefined() Funktion.

Syntax:

ds_map_find_next(id, key);

ArgumentTyp Beschreibung
id The id of the map to use.
key The key to find the next one to.

Rückgabe:

Variable or undefined

Beispiel:

var size = ds_map_size(inventory);
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
{
    if (key != "gold")
    {
        key = ds_map_find_next(inventory, key);
    }
    else break;
}

Die obige code erstellt einige temporäre Variablen, ruft dann die Größe der DS-Map ab und findet den ersten Schlüssel, den der Computer in der Map gespeichert hat. Anschließend wird mit for loop die DS-Map auf der Suche nach dem Schlüsselwert "gold" durchlaufen. Wenn er ihn findet, wird die loop geöffnet.
