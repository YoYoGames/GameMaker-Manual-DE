ds_map_read

Diese Funktion nimmt eine Zeichenkette, die zuvor von der Funktion ds_map_write() erstellt wurde, und liest sie dann in eine zuvor erstellte DS-Map ein. Wenn die Karte, in die die Zeichenkette eingelesen wird, Schlüssel/Wert-Paare enthält, werden diese zuerst gelöscht, bevor die gespeicherte Karte neu aufgebaut wird.

Beachten Sie, dass Sie, wenn die angegebene DS-Zeichenkette von GameMaker geschrieben wurde: Studio 1.2.x runtime (oder älter) geschrieben wurde, sollten Sie das optionale argument "legacy" angeben und es auf true setzen, da sich das String-Format danach geändert hat.

Syntax:

ds_map_read(id, str, [legacy]);

Argument Typ Beschreibung
id DS-Karten-ID The id of the data structure to read the string into
str Zeichenfolge The string to read
legacy Boolesche OPTIONAL Kann entweder true oder false sein oder ganz weggelassen werden.

Rückgabe:

N/A

Beispiel:

inventory = ds_map_create();
ini_open("map.ini");
var t_string = ini_read_string("Saved", "0", "");
if (t_string != "")
{
    ds_map_read(inventory, t_string);
}
ini_close();

Die obige code erstellt eine neue DS-Map und speichert ihren ID-Index in der Variablen "inventory". Dann wird eine Ini-Datei geöffnet und ein String aus dieser Datei in die temporäre Variable "t_string" eingelesen. Abschließend wird überprüft, ob die Zeichenkette gültig ist (nicht der Standard-Ini-Wert ""). Ist dies der Fall, wird die Zeichenkette in die neu erstellte DS-Map eingelesen, bevor die Ini-Datei wieder geschlossen wird.
