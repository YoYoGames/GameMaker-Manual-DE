ds_map_is_list

Mit dieser Funktion können Sie prüfen, ob eine DS-Liste im angegebenen Map-Schlüssel gespeichert ist. Wenn der angegebene Schlüssel eine DS-Listen-ID enthält, dann gibt die Funktion true zurück, andernfalls false.

Beachten Sie, dass dies nur Listen erkennt, die mit der Funktion ds_map_add_list() hinzugefügt wurden.

Syntax:

ds_map_is_list(id, key)

ArgumentTyp Beschreibung
id The id of the map to use.
key The key to check.

Rückgabe:

Beispiel:

var size = ds_map_size(inventory);
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++)
{
    if (ds_map_is_list(inventory, key))
    {
        ds_list_destroy(inventory[? key]);
    }
    key = ds_map_find_next(inventory);
}
ds_map_destroy(inventory);

Die obige code loops durchläuft eine DS-Map und prüft, ob einer der darin enthaltenen Schlüssel für eine DS-Liste bestimmt ist. Wenn ja, wird die DS-Liste zerstört, und am Ende von loop wird auch die DS-Map zerstört.
