ds_map_find_last

Diese Funktion gibt den letzten in der DS-Map gespeicherten Schlüssel zurück. Dies ist nicht der letzte Schlüssel in der Reihenfolge, in der Sie ihn hinzugefügt haben! DS-Maps werden nicht in einer linearen Form gespeichert, sondern in DS-Listen, so dass diese Funktion nur den letzten vom Computer gespeicherten Schlüssel findet. Dies kann nützlich sein, wenn Sie die DS-Map nach etwas durchsuchen müssen, sollte aber nach Möglichkeit vermieden werden, da es langsam sein kann.

Beachten Sie, dass diese Funktion undefined zurückgibt, wenn die angegebene DS-Map leer ist.

Syntax:

ds_map_find_last(id);

ArgumentTyp Beschreibung
id The id of the map to use.

Rückgabe:

Variable or undefined

Beispiel:

var size = ds_map_size(inventory);
var key = ds_map_find_last(inventory);
for (var i = size; i > 0; i--;)
{
    if (key != "gold")
    {
        key = ds_map_find_previous(inventory, key);
    }
    else break;
}

Die obige code erstellt einige temporäre Variablen, ruft dann die Größe der DS-Map ab und findet den letzten Schlüssel, den der Computer in der Map gespeichert hat. Anschließend wird mit for loop die DS-Map auf der Suche nach dem Schlüsselwert "gold" durchlaufen. Wenn er ihn findet, gibt er die loop aus.
