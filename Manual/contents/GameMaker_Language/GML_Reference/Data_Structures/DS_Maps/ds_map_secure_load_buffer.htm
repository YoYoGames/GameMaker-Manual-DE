ds_map_secure_load_buffer

Diese Funktion lädt eine sicher gespeicherte DS-Map von einer buffer. Sie müssen zuvor die buffer in den Speicher geladen haben (mit buffer_load()), und die Übergabe an diese Funktion gibt eine DS-Map zurück, die mit dem Inhalt von buffer gefüllt ist. Beachten Sie, dass buffer mit der Funktion ds_map_secure_save_buffer() erstellt worden sein muss, damit dies korrekt funktioniert, und beachten Sie auch, dass, wenn die zu ladende DS-Map ein Array enthielt, dieses beim Laden stattdessen in eine DS-Liste umgewandelt wird.

WICHTIG! Eine der Eigenschaften einer sicher gespeicherten Datei ist, dass sie an das Gerät gebunden ist, auf dem sie erstellt wurde. Sie können also eine auf einem Gerät gespeicherte Datei nicht in ein Projekt laden, das auf einem anderen Gerät läuft.

HINWEIS: Diese Funktion wird von HTML5 und UWP nicht unterstützt.

Syntax:

ds_map_secure_load_buffer(buffer);

Argument Typ Beschreibung
buffer Puffer-ID The buffer ID of the buffer to load the map data from.

Rückgabe:

DS Map ID

Beispiel:

var buff = buffer_load("save.dat");
map = ds_map_secure_load_buffer(buff);
buffer_delete(buff);

Die obige code lädt eine sicher gespeicherte DS-Map von einer buffer und speichert ihren Indexwert in einer Variablen zur späteren Verwendung.
