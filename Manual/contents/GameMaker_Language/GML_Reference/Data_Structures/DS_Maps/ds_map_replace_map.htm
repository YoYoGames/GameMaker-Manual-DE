ds_map_replace_map

Mit dieser Funktion können Sie eine DS Map, die im angegebenen "Schlüssel" gespeichert wurde, durch eine andere, zuvor erstellte Karte ersetzen. Diese Funktion ist für die Erstellung von JSON-kompatiblen Maps gedacht, die Sie dann mit json_encode() kodieren, und sollte nur in Verbindung mit dieser Funktion verwendet werden.

Syntax:

ds_map_replace_map(id, key, value)

Argument Typ Beschreibung
id DS-Karten-ID The id of the ds_map to use.
key Zeichenfolge The key to replace.
value DS-Karten-ID The id of the ds_map to use to replace the one previously stored in the given key.

Rückgabe:

N/A

Beispiel:

var temp_map = ds_map_create();
ds_map_add_list(temp_map, "list", j_list);
ds_map_add(temp_map, "array", j_array);
ds_map_replace_map(j_map, "maps", temp_map);
var j = json_encode(j_map);
ds_map_destroy(temp_map);

Die obige code erstellt eine DS Map und füllt sie mit einem Array und einer DS List, bevor sie eine zuvor gespeicherte Karte in der DS Map "j_map" ersetzt.
