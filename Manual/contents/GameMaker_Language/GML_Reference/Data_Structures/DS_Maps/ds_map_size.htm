ds_map_size

Mit dieser Funktion können Sie herausfinden, wie viele Schlüssel/Wertpaare die (zuvor erstellte) DS-Map enthält.

Syntax:

ds_map_size(id);

ArgumentTyp Beschreibung
id The id of the data structure to check.

Rückgabe:

Beispiel:

if (ds_map_size(inventory) > 49)
{
    full = true;
}

Die obige code prüft die Größe der DS-Map (d.h. die Anzahl der Schlüssel/Wertpaare) und setzt, wenn sie größer als 49 ist, die Variable "full" auf true.
