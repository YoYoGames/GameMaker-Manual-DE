ds_map_write

Diese Funktion wandelt die DS-Map-Daten der angegebenen Map in ein String-Format um, das dann zur einfachen Speicherung in eine *.ini- oder *.txt-Datei geschrieben werden kann. Diese Zeichenkette kann dann später in eine neue ds_map zurückgelesen werden mit ds_map_read().

HINWEIS: Die zurückgegebene Zeichenkette ist keine vom Menschen lesbare Zeichenkette, sondern ein Dump des Inhalts der Datenstruktur.

Syntax:

ds_map_write(id);

ArgumentTyp Beschreibung
id The id of the data structure to use

Rückgabe:

Beispiel:

ini_open("map.ini");
var t_string;
t_string = ds_map_write(inventory);
ini_write_string("Saved", "0", t_string);
ini_close();

Die obige code öffnet eine Ini-Datei, in die geschrieben werden kann. Dann wird mit ds_map_write() eine Zeichenkette erzeugt, die in der temporären Variablen "t_string" gespeichert wird. Schließlich wird diese Zeichenfolge in die Ini-Datei geschrieben, bevor sie geschlossen wird.
