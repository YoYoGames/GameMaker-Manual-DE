string_char_at

Mit dieser Funktion können Sie ein bestimmtes Zeichen an einer bestimmten Position innerhalb einer Zeichenkette zurückgeben, wobei der Index bei 1 für das erste Zeichen beginnt. Wird kein Zeichen gefunden oder ist die Zeichenkette kürzer als der angegebene Indexwert, wird eine leere Zeichenkette "" zurückgegeben; ist der angegebene Index jedoch gleich oder kleiner als 0, wird das erste Zeichen der Zeichenkette zurückgegeben.

Syntax:

string_char_at(str, index);

ArgumentTyp Beschreibung
str The string to check.
index The position to get the character from.

Rückgabe:

Beispiel:

str1 = "Hello World";
str2 = string_char_at(str1, 7);

Damit wird str2 auf das siebte Zeichen ("H" zählt als 1. Zeichen) in der Zeichenfolge str1 gesetzt, in diesem Fall "W".
