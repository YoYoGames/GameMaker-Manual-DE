zwischenablage_set_text

Mit dieser Funktion wird die Zwischenablage so eingestellt, dass sie die angegebene Zeichenfolge enthält. Sie können sie auf eine leere Zeichenkette "" setzen, um die Zwischenablage effektiv von Text zu befreien.

Syntax:

clipboard_set_text(string);

Argument Typ Beschreibung
string Zeichenfolge The text (as a string) to set the clipboard to hold.

Rückgabe:

N/A

Beispiel:

if clipboard_has_text()
{
    str = clipboard_get_text();
    clipboard_set_text("");
}

Die obige code prüft die Zwischenablage auf Text, und wenn sie Text enthält, wird dieser als String in die Variable "str" eingelesen. Schließlich wird die Zwischenablage gelöscht, indem sie auf einen leeren String gesetzt wird.
