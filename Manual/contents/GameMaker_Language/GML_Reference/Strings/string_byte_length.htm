string_byte_length

Diese Funktion gibt die Anzahl der Bytes in einer Zeichenkette zurück. Da sie in UTF8 gehalten werden, entspricht dies nicht der Anzahl der Zeichen in der Zeichenkette.

Syntax:

string_byte_length(string);

Argument Typ Beschreibung
string Zeichenfolge The string to measure the number of bytes of.

Rückgabe:

Real

Beispiel:

bytesize = string_byte_length("Hello World");

Dies würde bytesize auf die Anzahl der Bytes in "Hello World" setzen.
