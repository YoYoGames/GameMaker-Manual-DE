string_niedrig

Mit dieser Funktion können Sie erzwingen, dass eine Zeichenkette nur Kleinbuchstaben enthält.

HINWEIS Diese Funktion erkennt nur das englische Alphabet mit 26 Buchstaben von A - Z.

Syntax:

string_lower(string);

ArgumentTyp Beschreibung
string The string to convert to lowercase.

Rückgabe:

Beispiel:

str1 = "Hello World";
str2 = string_lower(str1);

Die obige code setzt str2 auf "hello world".
