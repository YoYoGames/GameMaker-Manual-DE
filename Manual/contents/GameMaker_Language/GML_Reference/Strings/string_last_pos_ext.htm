string_last_pos_ext

Diese Funktion gibt die Zeichenposition einer Instanz einer Teilzeichenkette innerhalb einer Zeichenkette zurück, indem sie die Zeichenkette von der als Startposition angegebenen Position aus rückwärts durchsucht. Die Funktion gibt 0 zurück, wenn die Suchzeichenfolge nicht gefunden wird, oder die Position des ersten Zeichens der Suchzeichenfolge, wenn sie gefunden wird. Beachten Sie, dass bei der Legacy-Unterstützung Zeichenketten von 1 an indiziert werden, so dass 1 die erste Position in der Zeichenkette ist, nicht 0, wie Sie vielleicht erwarten.

Syntax:

string_last_pos_ext(substr, str, start_pos);

ArgumentTyp Beschreibung
substr The substring to look for in the string.
str The string.
start_pos The starting position to search from.

Rückgabe:

Beispiel:

if (string_last_pos_ext(",", text, 20) != 0)
{
    string_insert(name, text, string_last_pos_ext(",", text, 20));
}

Die obige code sucht in der Zeichenkette, die in der Variablen "text" gespeichert ist, nach einem Komma vor dem 20. Zeichen und fügt an dieser Stelle die Teilzeichenkette "name" ein.
