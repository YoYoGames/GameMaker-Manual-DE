string_byte_at

Gibt den Byte-Rohwert als Realwert an einer bestimmten Position in der angegebenen Zeichenkette zurück.

Syntax:

string_byte_at(str, index);

ArgumentTyp Beschreibung
str The string to check.
index The position to get the byte from.

Rückgabe:

Beispiel:

newbyte = string_byte_at("Hello World", 5);

Damit wird newbyte auf den Rohbyte-Wert des sechsten Buchstabens von "Hello World" gesetzt.
