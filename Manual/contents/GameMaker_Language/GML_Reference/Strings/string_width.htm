string_width

Diese Funktion gibt die Breite (in Pixeln) der eingegebenen Zeichenkette zurück, wobei eventuelle Zeilenumbrüche im Text berücksichtigt werden. Sie ist sehr praktisch für die Berechnung von Abständen zwischen Textelementen auf der Grundlage der Gesamtbreite der Buchstaben, aus denen die Zeichenkette besteht wie sie mit draw_text() unter Verwendung der aktuell definierten Schriftart gezeichnet würde.

Syntax:

string_width(string);

ArgumentTyp Beschreibung
string The string to measure the width of.

Rückgabe:

Beispiel:

var ww = string_width(str_Name + " ");
draw_text(32, 32, str_Name));
draw_text(32 + ww, 32, "has won the game!");

Die obige code ermittelt die Breite der angegebenen Zeichenkette und zeichnet dann zwei Textzeilen, wobei die zurückgegebene Zeichenkettenbreite als Trennzeichen verwendet wird.
