ord

Diese Funktion nimmt eine einzelne Zeicheneingabezeichenfolge entgegen und gibt den Unicode (UTF8)-Wert für dieses Zeichen zurück. Beachten Sie, dass bei der Verwendung mit den keyboard_check* Funktionen verwendet wird, die Eingabezeichenfolge nur ein Zeichen lang sein kann und nur eine Zahl von 0 bis 9 oder ein großgeschriebenes römisches Zeichen von A bis Z sein kann.

Syntax:

ord(string)

Argument Typ Beschreibung
string Zeichenfolge The string with which to find the Unicode value.

Rückgabe:

Real

Beispiel:

if keyboard_check(ord("W"))
{
    y -= 4;
}

Dadurch wird die aufrufende Instanz um vier Pixel nach oben verschoben, wenn die Taste W gedrückt gehalten wird.
