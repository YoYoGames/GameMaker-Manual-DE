string_copy

Mit dieser Funktion können Sie ganz einfach eine Anzahl von Zeichen aus einer Zeichenkette auswählen, die in eine andere kopiert werden sollen. Das erste Zeichen in einer Zeichenkette wird immer als 1 indiziert und nicht als 0, wie Sie vielleicht erwarten. Um also (zum Beispiel) die ersten fünf Zeichen einer Zeichenkette zu kopieren, müssten Sie string_copy(str, 1, 5) verwenden. Ein weiteres Beispiel finden Sie weiter unten.

Syntax:

string_copy(str, index, count);

ArgumentTyp Beschreibung
str The string to copy from.
index The position of the first character in the string to copy from (numbered from 1)
count The number of characters, starting from the position of the first, to copy

Rückgabe:

Beispiel:

name = keyboard_string;
if (string_length(name) > 15)
{
    keyboard_string = string_copy(name, 1, 15);
}

Die obige code ermöglicht es dem Spieler, eine Zeichenkette über die Tastatur einzugeben, die dann in der Variablen name gespeichert wird. Diese Variable wird dann daraufhin überprüft, ob sie die maximale Länge von 15 Zeichen überschreitet, und wenn dies der Fall ist, wird die Tastaturzeichenkette durch eine 15 Zeichen lange Kopie der Zeichenkette name ersetzt. Dadurch wird die Eingabe des Spielers auf 15 Zeichen begrenzt.
