string_delete

Diese Funktion wird verwendet, um einen Teil der angegebenen Zeichenkette zu entfernen. Sie geben die Eingabezeichenkette, die Startposition innerhalb der Zeichenkette, von der Zeichen entfernt werden sollen (der Index beginnt bei 1), und die Anzahl der zu entfernenden Zeichen an. Die Funktion gibt eine neue Zeichenkette ohne diesen Teil zurück.

Syntax:

string_delete(str, index, count);

ArgumentTyp Beschreibung
str The string to copy and delete from.
index The position of the first character to remove (from 1).
count The number of characters to remove.

Rückgabe:

String

Beispiel:

str1 = "Helloo World";
str2 = string_delete(str1, 5, 1);

Dadurch wird str2 auf "Hello World" gesetzt, da das zusätzliche "o" von "Hello" (das an Position 5 steht) entfernt wird.
