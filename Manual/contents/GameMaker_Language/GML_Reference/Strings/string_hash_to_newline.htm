string_hash_to_newline

Diese Funktion gibt eine Zeichenkette zurück, bei der das Symbol "#" in eine neue Zeile umgewandelt wurde. Also eine Zeichenkette formatiert als:

"Hello#World"

Wird auf dem Bildschirm ausgedruckt als:

Hello
World

WICHTIG! Diese Funktion dient der Importkompatibilität zwischen GameMaker und früheren Versionen der Software und sollte daher nicht verwendet werden, es sei denn, es ist notwendig, veraltete Funktionen zu replizieren. Stattdessen sollten Sie die Escape-Zeichen verwenden, um eine neue Zeile zu markieren. Weitere Informationen zur Importkompatibilität finden Sie hier.

Syntax:

string_hash_to_newline(string);

ArgumentTyp Beschreibung
string The string to convert over multiple lines

Rückgabe:

Beispiel:

var str = string_hash_to_newline("Hello#World");
draw_text(32, 32, str);

Die obige code wandelt die Zeichenkette mit dem Raute-Symbol in eine auf zwei Zeilen aufgeteilte Zeichenkette um und gibt sie auf dem Bildschirm aus.
