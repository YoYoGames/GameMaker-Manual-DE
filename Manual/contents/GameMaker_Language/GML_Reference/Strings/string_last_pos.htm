string_last_pos

Diese Funktion gibt die Zeichenposition einer Instanz einer Teilzeichenkette innerhalb einer Zeichenkette zurück, wobei vom Ende der Zeichenkette zum Anfang gesucht wird (also die Umkehrung von string_pos(). Die Funktion gibt 0 zurück, wenn die Suchzeichenkette nicht gefunden wird, oder die Position des ersten Zeichens der Suchzeichenkette, wenn sie gefunden wird. Beachten Sie, dass für die Legacy-Unterstützung Zeichenketten von 1 an indiziert werden, so dass 1 die erste Position in der Zeichenkette ist, nicht 0, wie Sie vielleicht erwarten.

Syntax:

string_last_pos(substr, str);

ArgumentTyp Beschreibung
substr The substring to look for in the string.
str The string.

Rückgabe:

Beispiel:

if (string_last_pos(",", text) != 0)
{
    string_insert(name, text, string_last_pos(",", text));
}

Die obige code sucht in der in der Variablen "text" gespeicherten Zeichenfolge nach einem Komma und beginnt die Suche am Ende der Zeichenfolge. Wird eines gefunden, wird die Teilzeichenkette "name" an dieser Stelle eingefügt.
