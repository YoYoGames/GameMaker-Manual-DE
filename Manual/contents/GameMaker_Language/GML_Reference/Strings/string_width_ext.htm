string_width_ext

Diese Funktion gibt die maximale Breite (in Pixeln) der eingegebenen Zeichenkette zurück, wobei der Zeilenabstand und die Breite des Zeilenumbruchs (definiert als die Anzahl der Pixel, die die Zeichenkette einnehmen kann, bevor ein Zeilenumbruch eingefügt wird) berücksichtigt werden. Es ist sehr für die Berechnung von Abständen zwischen Textelementen auf der Grundlage der maximalen Breite einer Zeichenkette, die über mehrere Zeilen aufgeteilt ist, wie sie mit draw_text_ext() unter Verwendung der aktuell definierten Schriftart. Trennung und Breite können auf -1 gesetzt werden, um die Standardabstände zu erhalten.

Syntax:

string_width_ext(string, sep, w);

ArgumentTyp Beschreibung
string The string to measure the width of.
sep The distance in pixels between lines of text as if the string was being drawn.
w The maximum width (in pixels) of the string before a line break as if the string was bring drawn.

Rückgabe:

Beispiel:

var ww = string_width_ext(str_Story_Text[1], -1, 100);
draw_text_ext(32, 32, str_Story_Text[1], -1, 100);
draw_text_ext(32 + ww, 32, str_Story_Text[2], -1, 100);

Die obige code ermittelt die Breite der angegebenen Zeichenkette unter Berücksichtigung der Zeilentrennung und der Breite des Zeilenumbruchs und zeichnet dann zwei Textzeilen, wobei die zurückgegebene Gesamtbreite der Zeichenkette als Trennzeichen verwendet wird.
