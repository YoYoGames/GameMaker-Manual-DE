String

Mit dieser Funktion können Sie jede reelle Zahl in eine Zeichenkette umwandeln. Wenn die reelle Zahl eine ganze Zahl ist, wird sie ohne Nachkommastellen gespeichert, andernfalls wird sie mit zwei Nachkommastellen gespeichert. Wenn Sie mehr Dezimalstellen benötigen, verwenden Sie die Funktion string_format(). Beachten Sie auch, dass die Verwendung dieser Funktion auf eine Variable, die ein Array, eine Datenstruktur oder eine Struktur speichert, den Inhalt dieser Variablen in eine Zeichenkette umwandelt, die dann auf der Konsole ausgegeben oder in einer Datei für debugging gespeichert werden kann.

Syntax:

string(val);

Argument Typ Beschreibung
val Real The real number to be turned into a string.

Rückgabe:

String

Beispiel:

draw_text(100, 100, "Score: " + string(score) + " / Health: " + string(health));

Die obige code verwendet die Funktion string, um sowohl reelle Zahlen als auch Zeichenketten zusammen zu zeichnen (draw akzeptiert nur entweder eine Zeichenkette oder eine reelle Zahl, aber nicht beides).
