Zeichenfolge_Buchstaben

Mit dieser Funktion können Sie alle Zeichen entfernen, die nicht als Buchstaben klassifiziert sind. Dies ist praktisch, um zu verhindern, dass jemand unerwünschte Zeichen in einen Texteintrag eingibt (z. B. die Eingabe von "#", um eine neue Zeile zu erzwingen).

HINWEIS Diese Funktion erkennt und gibt nur das englische Alphabet mit 26 Buchstaben von A - Z (Groß- oder Kleinschreibung) wieder.

Syntax:

string_letters(string);

ArgumentTyp Beschreibung
string The string to convert to letters.

Rückgabe:

Beispiel:

username = string_letters(username);

Die obige code setzt die Variable "username" so, dass sie nur die Buchstaben der ursprünglichen Zeichenkette enthält (wenn die ursprüngliche Zeichenkette "MacSw33ny" war, wäre die zurückgegebene Zeichenkette "MacSwny").
