string_digits

Mit dieser Funktion können Sie eine gegebene Zeichenkette analysieren und beliebige Zahlen daraus ermitteln. Ein Beispiel: Sie haben diesen Text - "Ich bin 81 Jahre alt". Mit dieser Funktion würden Sie die Zeichenkette "81" zurückerhalten.

Syntax:

string_digits(string);

ArgumentTyp Beschreibung
string The string to get the digits from.

Rückgabe:

Beispiel:

var t_str = string_digits(input_str);
age = real(t_str);

Die obige code nimmt die Eingabezeichenkette, entfernt alle Zeichen, die keine Zahlen sind, und setzt dann die Variable age so, dass sie den realen Zahlenwert der Rückgabezeichenkette enthält (wenn die Eingabezeichenkette also zum Beispiel "Ich bin 18" wäre, würde die Funktion "18" zurückgeben).
