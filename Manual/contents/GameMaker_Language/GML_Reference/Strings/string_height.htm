string_height

Diese Funktion gibt die Höhe (in Pixeln) der eingegebenen Zeichenkette zurück und berücksichtigt dabei den Zeilenabstand und eventuelle Zeilenumbrüche im Text. Sie ist sehr praktisch für die Berechnung von Abständen zwischen Textelementen auf der Grundlage des höchsten Buchstabens die die Zeichenkette bilden, wie sie mit draw_text() unter Verwendung der aktuell definierten Schriftart gezeichnet würde.

Syntax:

string_height(string);

ArgumentTyp Beschreibung
string The string to measure the height of.

Rückgabe:

Beispiel:

var hh = string_height("ABCDEFGHIJKLMNOPQRSTUVWXYZ");
draw_text(32, 32, string(score));
draw_text(32, 32 + hh, string(lives);

Die obige code ermittelt die Höhe der angegebenen Zeichenkette und zeichnet dann zwei Textzeilen, wobei die zurückgegebene Höhe der Zeichenkette als Trennzeichen verwendet wird.
