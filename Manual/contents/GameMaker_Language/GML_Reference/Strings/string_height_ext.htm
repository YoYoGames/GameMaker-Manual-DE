string_height_ext

Diese Funktion gibt die Höhe (in Pixeln) der eingegebenen Zeichenkette zurück, wobei der Zeilenabstand und die Breite des Zeilenumbruchs (definiert als die Anzahl der Pixel, die die Zeichenkette einnehmen kann, bevor ein Zeilenumbruch eingefügt wird) berücksichtigt werden. Es ist sehr praktisch für die Berechnung von Abständen zwischen Textelementen auf der Grundlage des höchsten Buchstabens, aus dem die Zeichenkette besteht, sowie für die Berechnung der tatsächlichen Länge der Zeichenkette selbst, wenn sie sich über mehrere Zeilen erstreckt, wie sie mit draw_text_ext() unter Verwendung der aktuell definierten Schriftart gezeichnet würde. Abstand und Breite können auf -1 gesetzt werden, um die Standardabstände zu erhalten.

Syntax:

string_height_ext(string, sep, w);

ArgumentTyp Beschreibung
string The string to measure the height of.
sep The distance in pixels between lines of text as if the string was being drawn.
w The maximum width (in pixels) of the string before a line break as if the string was bring drawn.

Rückgabe:

Beispiel:

var hh = string_height_ext(str_Story_Text[1], -1, 100);
draw_text_ext(32, 32, str_Story_Text[1], -1, 100);
draw_text_ext(32, 32 + hh, str_Story_Text[2], -1, 100);

Die obige code ermittelt die Höhe der angegebenen Zeichenkette unter Berücksichtigung des Zeilenabstands und der Breite des Zeilenumbruchs und zeichnet dann zwei Textzeilen, wobei die zurückgegebene Gesamthöhe der Zeichenkette als Trennzeichen verwendet wird.
