zwischenablage_holen_text

Diese Funktion gibt eine Zeichenkette des in der Zwischenablage enthaltenen Textes zurück. Wenn kein Text gespeichert ist, wird eine leere Zeichenkette "" zurückgegeben.

Syntax:

clipboard_get_text();

Rückgabe:

String

Beispiel:

if clipboard_has_text()
{
    str = clipboard_get_text();
    clipboard_set_text("");
}

Die obige code prüft die Zwischenablage auf Text, und wenn sie Text enthält, wird dieser als String in die Variable "str" eingelesen. Schließlich wird die Zwischenablage gelöscht, indem sie auf einen leeren String gesetzt wird.
