string_insert

Mit dieser Funktion können Sie eine neue Zeichenfolge erstellen, die aus zwei Zeichenfolgen besteht, von denen eine an einer bestimmten Stelle in die andere eingefügt wurde. Das kann zum Beispiel nützlich sein, um einen Benutzernamen in einen vordefinierten Text einzufügen und so den Spieler Ihres Spiels mehr in das Spielgeschehen einzubeziehen. Beachten Sie bei der Berechnung der Position, an der eingefügt werden soll, dass Strings von 1 an indiziert werden, so dass das erste Zeichen in einem String 1 ist und nicht 0, wie Sie vielleicht erwarten.

Syntax:

string_insert(substr, str, index);

ArgumentTyp Beschreibung
substr The substring to be inserted.
str The string to be copied.
index The position in characters of the string to insert the substring.

Rückgabe:

Beispiel:

str2 = string_insert(username, "Hello, , how are you?", 8);

Dies fügt die Zeichenkette in der Variable "username" in die angegebene Phrase ein, wobei die resultierende Zeichenkette wie folgt aussieht: "Hallo, NAME, wie geht es dir?".
