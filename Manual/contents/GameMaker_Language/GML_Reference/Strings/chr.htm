chr

Diese Funktion gibt eine Zeichenkette zurück, die das Zeichen enthält, das sich auf die Unicode-Eingabe code bezieht, die angezeigt werden soll. Dieses Zeichen hängt von der aktuellen Zeichensatzseite code ab, und wenn keine Schriftart eingestellt ist, wird die Standardseite code für die Maschine.

Syntax:

chr(val);

ArgumentTyp Beschreibung
val The Unicode code value to get the string from

Rückgabe:

Beispiel:

mystring = chr(53) + chr(48);

Dies würde mystring auf "50" (eine Zeichenkette, keine ganze Zahl) setzen, da chr(53) "5" und chr(48) "0" ist.
