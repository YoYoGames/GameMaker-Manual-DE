string_lettersdigits

Diese Funktion gibt eine Kopie einer gegebenen Zeichenkette zurück, bei der alles außer den Buchstaben und Ziffern entfernt wurde, d.h. sie kann verwendet werden, um unerwünschte Zeichen (wie "#" oder "?") zu entfernen, z.B. aus einem Anmeldenamen oder einem Passwort.

HINWEIS Diese Funktion gibt nur die 26 Buchstaben des englischen Alphabets von A - Z (Groß- oder Kleinschreibung) und die Zahlen 0 - 9 wieder.

Syntax:

string_lettersdigits(string);

ArgumentTyp Beschreibung
string The string to convert to letters and digits.

Rückgabe:

Beispiel:

if (string_length(password) > string_length(string_lettersdigits(password)))
{
    draw_text(32,32, "Invalid Password! Only Letters and numbers please!");
}

Die obige code prüft die Länge einer Zeichenkette mit der Länge der gleichen Zeichenkette, bei der jedoch alle Symbole entfernt wurden. Wenn sie nicht übereinstimmen, wird eine Meldung auf dem Bildschirm angezeigt.
