string_set_byte_at

Diese Funktion setzt ein Byte direkt in einen String (basierend auf dem UTF8-Format) und gibt eine Kopie des Strings mit den Änderungen zurück.

HINWEIS Diese Funktion ist unglaublich langsam. Überlegen Sie daher sorgfältig, ob sie notwendig ist und wo Sie sie verwenden.

Syntax:

string_set_byte_at(str, pos, byte);

ArgumentTyp Beschreibung
str The string to change the byte of.
pos The position within the string (starting at 1) to change the byte of.
byte The new byte value.

Rückgabe:

Beispiel:

str = string_set_byte_at("hello", 2, 97);

Die obige code würde den Byte-Wert des zweiten Buchstabens in der Zeichenkette ändern und so die Variable str auf den Wert "hallo" setzen.
