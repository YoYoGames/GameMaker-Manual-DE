ansi_char

Diese Funktion gibt eine Zeichenkette zurück, die das Zeichen mit dem gesetzten BYTE-Rohwert enthält. Diese Zeichenkette wird nicht angezeigt und sollte auch nicht angezeigt werden, aber sie wird für die Verwendung bei der Kodierung korrekt auf der Festplatte gespeichert.

Syntax:

ansi_char(val);

ArgumentTyp Beschreibung
val The raw byte value.

Rückgabe:

(Single character)

Beispiel:

var str1 = ansi_char($EF);
var str2 = ansi_char($BB);
var str3 = ansi_char($BF);
file_text_write_string(global.saveFile, str1 + str2 + str3);

Die obige code erstellt eine Zeichenkette aus Byte-Rohdaten und schreibt sie in eine (zuvor geöffnete) Datei.
