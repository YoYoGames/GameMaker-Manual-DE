string_pos

Diese Funktion gibt die Zeichenposition einer Instanz einer Teilzeichenkette innerhalb einer Zeichenkette zurück, wobei vom Anfang der Zeichenkette aus gesucht wird (um vom Ende aus zu suchen, verwenden Sie die Funktion string_last_pos()). Die Funktion gibt 0 zurück, wenn sie nicht gefunden wird, oder die Position des ersten Zeichens der Teilzeichenkette, wenn sie gefunden wird. Denken Sie daran, dass die Indizierung von Zeichenketten bei 1 beginnt, so dass 1 die erste Position in der Zeichenkette ist, nicht 0, wie Sie vielleicht erwarten. Eine Verwendung hierfür ist das Filtern von Wörtern, die als beleidigend angesehen werden könnten, oder das Finden der richtigen Stelle zum Einfügen eines Textes in eine andere Zeichenkette.

Syntax:

string_pos(substr, str);

ArgumentTyp Beschreibung
substr The substring to look for in the string.
str The string.

Rückgabe:

Beispiel:

if (string_pos(",", text) != 0)
{
    string_insert(name, text, string_pos(",", text));
}

Die obige code sucht in der in der Variablen "text" gespeicherten Zeichenkette nach einem Komma, und wenn sie eines findet, fügt sie die Teilzeichenkette "name" an dieser Stelle ein.
