string_replace_all

Mit dieser Funktion können Sie eine Zeichenkette nach bestimmten Teilen durchsuchen, die dann an allen Stellen, an denen sie vorkommen, durch die von Ihnen angegebene neue Zeichenkette ersetzt werden können.

Syntax:

string_replace_all(str, substr, newstr);

ArgumentTyp Beschreibung
str The string to be copied.
substr The substring within the string to be replaced in all instances.
newstr The new substring to replace the previous one(s).

Rückgabe:

Beispiel:

str1 = "Hexxo Worxd";
str2 = string_replace_all(str1, "x", "l");

Dadurch wird str2 auf str1 gesetzt, wobei jedoch die Instanzen von "x" durch "l" ersetzt werden, so dass str2 zu "Hello World" wird.
