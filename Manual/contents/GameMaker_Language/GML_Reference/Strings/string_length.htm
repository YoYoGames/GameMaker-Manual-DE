string_length

Diese Funktion gibt die Anzahl der Zeichen zurück, aus denen eine gegebene Zeichenkette besteht. Sie kann z.B. nützlich sein, um herauszufinden, wann die Zeichenlänge eines benutzerdefinierten Texteintrags begrenzt werden soll (z.B.: Begrenzung des Spielernamens auf 10 Zeichen). Beachten Sie, dass dies ein Unterschied zu string_width() insofern unterscheidet, als es die Anzahl der Zeichen in der Zeichenkette misst und nicht deren Breite, wie sie auf dem Bildschirm in Pixeln dargestellt wird.

Syntax:

string_length(string);

Argument Typ Beschreibung
string Zeichenfolge The string to measure the number of characters of.

Rückgabe:

Real

Beispiel:

if (string_length(name) > 10)
{
    name = string_copy(name, 1, 10);
}

Dabei wird geprüft, ob die Zeichenkette des Namens mehr als zehn Zeichen umfasst. Ist dies der Fall, werden nur die ersten zehn Zeichen kopiert und verwendet.
