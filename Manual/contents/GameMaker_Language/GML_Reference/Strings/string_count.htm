string_count

Diese Funktion gibt zurück, wie oft die angegebene Teilzeichenkette in einer bestimmten Zeichenkette vorkommt. Auf diese Weise können Sie überprüfen, wie oft ein einzelner Buchstabe oder eine Phrase in einem Abschnitt des gespeicherten Textes wiederholt wird.

Syntax:

string_count(substr, str);

ArgumentTyp Beschreibung
substr The substring to check the string for.
str The string to check.

Rückgabe:

Beispiel:

str1 = "Hello World";
ocount = string_count( "o", str1 );

Damit wird die Variable ocount auf die Anzahl der "o "s in str1 gesetzt, in diesem Fall 2.
