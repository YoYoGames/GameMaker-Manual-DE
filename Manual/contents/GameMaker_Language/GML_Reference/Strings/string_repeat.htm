string_repeat

Diese Funktion gibt einfach die gleiche Zeichenkette zurück, die eine bestimmte Anzahl von Malen über sich selbst wiederholt wird.

Syntax:

string_repeat(str, count);

ArgumentTyp Beschreibung
str The string to repeat.
count The number of times to repeat the string.

Rückgabe:

Beispiel:

str1 = "Hello World";
str2 = string_repeat(str1, 3);

Die obige code wird str2 auf "Hello WorldHello WorldHello World" setzen.
