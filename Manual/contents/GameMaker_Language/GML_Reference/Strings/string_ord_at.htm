string_ord_at

Sie können diese Funktion verwenden, um ein bestimmtes Zeichen code an einer bestimmten Position innerhalb einer Zeichenkette zurückzugeben, wobei der Index bei 1 für das erste Zeichen beginnt. Wenn kein Zeichen gefunden wird oder die Zeichenkette kürzer ist als der für index angegebene Wert, wird -1 zurückgegeben.

Syntax:

string_ord_at(str, index);

ArgumentTyp Beschreibung
str The string to check.
index The position to get the character code from.

Rückgabe:

Beispiel:

str = "Hello World";
char_code = string_ord_at(str, 7);

Damit wird das Zeichen code für das siebte Zeichen (wobei "H" als erstes Zeichen zählt) in der Zeichenkette "str" ermittelt und in der Variablen "char_code" gespeichert.
