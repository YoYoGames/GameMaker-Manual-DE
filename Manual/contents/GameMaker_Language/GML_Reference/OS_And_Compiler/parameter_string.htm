Parameter_Zeichenfolge

Befehlszeilenparameter sind die zusätzlichen Befehle, die Sie einer Exe hinzufügen können, um die Ausführung des Programms zu ändern. Mit dieser Funktion können Sie den gewählten Befehlszeilenparameter als String erhalten. Sie können die Anzahl der Parameter für das aktuelle Spiel mit der Funktion parameter_count()herausfinden, wobei der erste Parameter den Index 1 und der letzte den von der Funktion zurückgegebenen Index hat (ein Wert von 0 ist insofern etwas Besonderes, als es sich um den Dateinamen der ausführbaren Datei des Spiels handelt, einschließlich des Pfads). Es sollte beachtet werden, dass diese Funktion für den HTML5 platform funktioniert und die Url-Parameter abruft.

Syntax:

parameter_string(n);

Rückgabe:

Beispiel:

var p_num;
p_num = parameter_count();
if p_num > 0
{
    var i;
    for (i = 0; i < p_num; i += 1)
    {
        p_string[i] = parameter_string(i + 1);
    }
}

Das obige code ermittelt die Anzahl der Befehlszeilenparameter, und wenn es 1 oder mehr sind, wird es loop durch sie hindurchgehen und sie als Zeichenketten in einem Array speichern.
