scheduler_resolution_set

Diese Funktion wird verwendet, um die Auflösung des Windows Thread Schedulers zu ändern. Der Auflösungswert, der in argument angegeben wird, muss in Millisekunden angegeben werden.

Standardmäßig, GameMaker eine Auflösung von 1 Millisekunde, was die kleinste erlaubte Auflösung ist und die beste Leistung ergibt. Das Ändern der Auflösung auf einen höheren Wert kann sich negativ auf die Leistung auswirken, aber es kann auch den Stromverbrauch reduzieren.

Sie können -1 als argument angeben, um die Auflösung auf den Standardwert zurückzusetzen (wie von Windows festgelegt). Beachten Sie, dass sich eine Änderung der Auflösung des Schedulers auf alle Prozesse auswirkt, die zur gleichen Zeit wie Ihr Spiel laufen.

HINWEIS: Diese Funktion ist nur für Windows verfügbar.

Syntax:

scheduler_resolution_set(milliseconds);

ArgumentTyp Beschreibung
milliseconds The new resolution value (in milliseconds) or -1 for default

Rückgabe:

Beispiel:

scheduler_resolution_set(2);

In diesem Beispiel wird die Auflösung des Thread-Schedulers Windows auf 2 Millisekunden eingestellt.
