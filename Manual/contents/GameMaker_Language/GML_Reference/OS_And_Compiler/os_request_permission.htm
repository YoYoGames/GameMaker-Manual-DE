os_check_permission

Mit dieser Funktion können Sie eine bestimmte Berechtigung anfordern. Sie geben die anzufordernde Berechtigung als Zeichenkette im Format "android.permission.<permission>" an. Um zum Beispiel die Berechtigung RECORD_AUDIO anzufordern, rufen Sie also

os_request_permission("android.permission.RECORD_AUDIO");

Die Funktion löst ein asynchrones Systemereignis aus, wobei die eingebaute async_load DS-Map die folgenden Schlüssel/Wert-Paare enthält:

  • "Typ": Dies ist der Ereignistyp, der ausgelöst wurde, und der Wert ist die Zeichenfolge "permission_request_result".
  • "<Erlaubnis>": Dies ist die von der Funktion angeforderte Erlaubnis in Form eines Strings, der eine der folgenden Konstanten enthält:
Konstante Beschreibung
os_permission_granted Dies bedeutet, dass die Genehmigung erteilt wurde.
os_permission_denied Dies bedeutet, dass die Genehmigung nicht erteilt wurde.

Es ist erwähnenswert, dass die folgenden Berechtigungen von GameMaker nativ unterstützt werden, aber von Google als "gefährlich" eingestuft werden und als solche explizit angefordert werden müssen (beachten Sie auch, dass einige Berechtigungen über die Android-Spieloptionen angefordert werden können, ohne dass diese Funktion benötigt wird):

  • android.permission.WRITE_EXTERNAL_STORAGE
  • android.permission.READ_PHONE_STATE
  • android.permission.RECORD_AUDIO

Weitere Informationen zu App-Berechtigungen finden Sie in der Android-Dokumentation.

WICHTIG! Diese Funktion ist nur für das Android-Ziel.

Syntax:

os_request_permission(permission)

Argument Typ Beschreibung
permission Zeichenfolge The permission to request (a string)

Rückgabe:

N/A

Beispiel:

if (os_type == os_android)
{
    if (os_check_permission("android.permission.INTERNET") == os_permission_denied)
    {
        os_request_permission("android.permission.INTERNET");
    }
}

Das obige code prüft den Betriebssystemtyp, und wenn er Android ist, führt es eine Überprüfung der Berechtigungen durch, und wenn die Berechtigung "INTERNET" noch nicht erteilt wurde, fordert es sie an.
