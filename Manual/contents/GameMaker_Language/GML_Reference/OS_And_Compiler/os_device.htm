os_device

Diese schreibgeschützte Variable enthält einen der unten aufgeführten konstanten Werte, um Ihnen mitzuteilen, auf welchem Gerät Sie das Spiel gerade spielen. Beachten Sie, dass diese Variable zu Gunsten der Funktion os_get_info() die genauere Informationen über das Gerät liefert, auf dem das Spiel läuft.

Syntax:

os_device

Rückgabe:

Device Type Constant

Gerätetyp Konstante
Konstante Beschreibung
device_ios_ipad iPad
device_ios_ipad_retina Neueres iPad mit Retina-Display in der Größe von 2048 x 1536
device_ios_iphone6 iPhone6 mit Bildschirmgröße 1334 x 750
device_ios_iphone6plus Größeres iPhone 6 mit Display 1920 x 1080
device_ios_iphone5 iPhone5 mit Displaygröße 640 x 1136)
device_ios_iphone Älteres iPhone/iPod Touch (480 x 320 Bildschirm) oder Android Telefon
device_ios_iphone_retina Neueres iPhone/iPod Touch mit Retina-Display von 960 x 640
device_emulator Das Gerät ist eigentlich ein Emulator (Windows Phone oder Android)
device_tablet Android-Tablet
device_ios_unknown Unbekannt oder nicht iOS

Beispiel:

if (os_browser = browser_not_a_browser)
{
    switch (os_device)
    {
        case device_ios_ipad: global.Config = 2; break;
        case device_ios_iphone: global.Config = 3; break;
        case device_ios_iphone_retina: global.Config = 4; break;
        case device_ios_unknown: global.Config = 5; break;
    }
}
else
{
    global.Config = 1;
}

Die obige code prüft, ob das Spiel in einem Browser läuft oder nicht und setzt eine globale Variable auf einen Wert, der vom Ergebnis der Prüfung abhängt.
