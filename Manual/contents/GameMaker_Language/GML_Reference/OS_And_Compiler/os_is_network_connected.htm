os_is_network_connected

Mit dieser Funktion können Sie überprüfen, ob Ihr Gerät derzeit eine Internetverbindung hat. Wenn ja, wird true zurückgegeben, wenn nicht, false, und je nach Betriebssystem wird möglicherweise versucht, eine Verbindung herzustellen, bevor ein Wert zurückgegeben wird.

Die Funktion verfügt über eine optionale argument, mit der der Versuch, eine Verbindung herzustellen, aktiviert/deaktiviert werden kann. Diese Option ist derzeit nur für das Ziel Nintendo Switch verfügbar. Wenn sie auf true gesetzt ist, versucht die Funktion, eine Verbindung herzustellen, und wenn sie auf false gesetzt ist, beschränkt sie sich darauf, die Verbindung zu überprüfen. Beachten Sie, dass der Versuch, eine Verbindung herzustellen, dazu führen kann, dass das Betriebssystem Switch für einige Sekunden hängen bleibt.

HINWEIS Diese Funktion prüft die interne Geräte-API, die Verbindungen steuert, und kann daher true zurückgeben, wenn eine Bluetooth-Verbindung, eine Wi-Fi-Verbindung oder auch nur eine normale Netzwerkverbindung besteht, die den Internetzugang ermöglicht.

Syntax:

os_is_network_connected([attempt_connection]);

Argument Typ Beschreibung
attempt_connection Boolesche OPTIONAL Nur für das Nintendo Switch-Ziel. Setzen Sie diese Option auf true, um beim Aufruf eine Verbindung auf Betriebssystemebene zu versuchen.

Rückgabe:

Boolean

Beispiel:

if os_is_network_connected()
{
    global.connected = true;
}

Die obige code prüft, ob das Gerät eine Verbindung zum Internet hat, und wenn ja, setzt sie eine globale Variable.
