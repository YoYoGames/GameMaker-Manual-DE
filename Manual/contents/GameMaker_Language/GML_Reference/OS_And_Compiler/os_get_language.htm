os_get_language

Diese Funktion gibt eine Zeichenkette mit den zwei Buchstaben der Sprache Code für das Betriebssystem zurück, auf dem das Spiel läuft, wie es der ISO639-Standard vorschreibt. Wenn die Informationen nicht verfügbar sind, wird einfach eine leere Zeichenkette "" oder "en" für die Sprache "Englisch" zurückgegeben. Beachten Sie, dass einige Sprachen auch eine entsprechende Region Code haben. Um also zwischen verschiedenen Regionen desselben Landes zu unterscheiden, verwenden Sie die Funktion os_get_region().

Die folgende Tabelle enthält Beispiele für einige der wichtigsten Sprachen mit zwei Buchstaben codes, wie sie in ISO 639 definiert sind:

Sprache Code
Arabisch ar
Chinesisch zh
Dänisch da
Englisch en
Französisch fr
Deutsch de
Griechisch el
Italienisch es
Japanisch ja
Norwegisch keine
Polnisch pl
Portugiesisch pt
Russisch ru
Spanisch es
Schwedisch sv

HINWEIS: Es handelt sich nicht um das Land code, das zurückgegeben wird, sondern um die Sprache code des Betriebssystems.

Syntax:

os_get_language()

Rückgabe:

Beispiel:

switch (os_get_language())
{
    case "es": ini_open("spanish.ini"); break;
    case "fr": ini_open("french.ini"); break;
    case "it": ini_open("italian.ini"); break;
    default: ini_open("english.ini"); break;
}

Die obige code prüft die Betriebssystemsprache und öffnet je nach dem zurückgegebenen Wert eine andere *.ini Datei.
