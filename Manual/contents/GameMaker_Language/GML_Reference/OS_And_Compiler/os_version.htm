os_version

Diese Variable gibt Ihnen die Versionsnummer des Betriebssystems an, auf dem Ihr Spiel läuft. Wenn Sie es zum Beispiel auf Windows 10 ausführen, ist os_version gleich 655360.

HINWEIS: Sollten Sie weitere Informationen über das Betriebssystem Windows benötigen, können Sie die environment_get_variable().

Syntax:

os_version

Rückgabe:

Beispiel:

if (os_type == os_android) && (os_version > 10)
{
    global.GFX = 1;
}

Die obige code prüft den Betriebssystemtyp und die Versionsnummer und wenn beide korrekt sind, wird die globale Variable auf 1 gesetzt.
