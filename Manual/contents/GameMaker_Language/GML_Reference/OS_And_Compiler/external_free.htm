extern_frei

Diese Funktion gibt den Speicher frei, der mit der Datei dll oder dylib mit dem angegebenen Namen verbunden ist. Dies sollte immer dann geschehen, wenn die betreffende Datei im Spiel nicht mehr benötigt wird, normalerweise (z.B.) bei einem Game End Event.

Syntax:

external_free(id);

Argument Typ Beschreibung
id Externe Funktion The name of the dll oder dylib, die Sie befreien möchten

Rückgabe:

N/A

Beispiel:

external_free("MyDLL.dll");

Das obige Beispiel code gibt den mit der angegebenen DLL verbundenen Speicher frei.
