externer_Aufruf

Wenn Sie einen externen Funktionsaufruf zu einer dll oder dylib mit external_define()erstellt haben, können Sie diese Funktion verwenden, um sie aufzurufen. Sie geben den Namen der zuvor definierten Funktion sowie jede der arguments an, die sie benötigt (jede argument muss vom richtigen Typ sein, entweder real oder string), und die Funktion gibt das Ergebnis des externen Aufrufs zurück.

Syntax:

external_call(id, args[0...15]);

Argument Typ Beschreibung
id Externe Funktion The name of the function that you want to call
args[0...10] Real oder String The different arguments that you want to pass through to the external function

Rückgabe:

Variable (the type of value returned will depend on the defined function)

Beispiel:

my_function = external_define("MyDLL.dll", "MyMin", dll_cdecl, ty_real, 2, ty_real, ty_real);
var _a = external_call(my_function, x, y);

Das obige Beispiel code ruft eine zuvor definierte externe Funktion auf und speichert den zurückgegebenen Wert in einer lokalen Variablen.
