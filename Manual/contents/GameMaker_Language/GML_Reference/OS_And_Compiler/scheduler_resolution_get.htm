scheduler_resolution_get

Diese Funktion wird verwendet, um die Auflösung des Windows Thread Schedulers in Millisekunden abzurufen. Wenn die Auflösung des Schedulers auf den Standardwert gesetzt ist (wie von Windows festgelegt), gibt die Funktion -1 zurück.

Informationen über den Thread-Scheduler finden Sie auf der Seite scheduler_resolution_set().

HINWEIS: Diese Funktion ist nur für Windows verfügbar.

Syntax:

scheduler_resolution_get();

Rückgabe:

(or -1 for default)

Beispiel:

scheduler_resolution = scheduler_resolution_get();

Dieses Beispiel ruft die Auflösung des Thread-Schedulers Windows ab und speichert sie in der Variablen "scheduler_resolution".
