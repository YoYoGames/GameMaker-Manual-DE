GM_Laufzeit_version

Diese Konstante enthält die runtime Versionsnummer, wie sie in den Voreinstellungen als runtime definiert ist, mit der das Projekt erstellt wird. Der Wert wird als String gespeichert.

Syntax:

GM_runtime_version;

Rückgabe:

String

Beispiel:

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);
draw_text(32, 96, "Runtime " + GM_runtime_version);

Die obige code nimmt die Konstanten GM-Datum, Version und runtime und zeichnet sie auf den Bildschirm.
