os_get_region

Diese Funktion gibt eine Zeichenkette mit dem zwei- oder dreibuchstabigen Regionalcode Code für das Betriebssystem zurück, auf dem das Spiel läuft, wie in der Norm ISO3166-1 festgelegt. Wenn die Informationen nicht verfügbar sind, wird einfach eine leere Zeichenkette "" zurückgegeben.

HINWEIS: Es handelt sich nicht um die regionale code, die zurückgegeben wird, sondern um die regionale Sprache code des Betriebssystems.

Syntax:

os_get_region()

Rückgabe:

Beispiel:

switch (os_get_language())
{
    case "zh":
        var region = os_get_region();
        if (region == "HK" || region == "MO" || region == "TW")
        {
            ini_open("chinese_traditional.ini");
        }
        else
        {
            ini_open("chinese_simplified.ini");
        }
    break;

    case "fr":
        ini_open("french.ini");
    break;

    case "it":
        ini_open("italian.ini");
    break;

    default:
        ini_open("english.ini");
    break;
}

Die obige code prüft die Sprache des Betriebssystems, und wenn sie Chinesisch ist, prüft sie die Betriebssystemregion und öffnet je nach den zurückgegebenen Werten eine andere *.ini Datei.
