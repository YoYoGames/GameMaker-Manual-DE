external_define

Diese Funktion kann verwendet werden, um einen externen Funktionsaufruf zu einer bestimmten dll (für Windows) oder dylib (für Mac) zu definieren. Diese Datei kann entweder eine eingeschlossene Datei oder Teil einer Erweiterung sein.

Sie geben den Namen (und den Pfad) der Datei an, dann den Namen der Funktion, die Sie definieren möchten. Als nächstes müssen Sie die zu verwendende Aufrufkonvention (siehe Konstantenliste unten) sowie die Art des zu erwartenden Ergebnisses (ebenfalls eine Konstante, wie unten aufgeführt) festlegen.

Schließlich müssen Sie die Anzahl der arguments angeben, die die Funktion annehmen kann (von 0 bis 15) und für jede der arguments müssen Sie auch ihren Typ angeben. Bitte beachten Sie, dass bei Funktionen mit 4 oder mehr arguments alle vom Typ ty_real sein müssen.

HINWEIS: Dies gilt nur für dll oder dylib, die als "Included Files" zum GameMaker IDE hinzugefügt wurden. Es funktioniert nicht mit Dateien, die als Erweiterungen hinzugefügt wurden, da diese erfordern, dass Sie die Funktionen im Erweiterungspaket selbst definieren.

Externe Aufruftypkonstanten
Konstante Beschreibung
dll_cdecl Dies ist der Standardaufruf in C, C++
dll_stdcall Dies ist der Standard-WinAPI-Aufruf (nurWindows dll )
Externe Datentypkonstanten
Konstante Beschreibung
ty_real Ein reelles Zahlenargument
ty_string Ein null-terminiertes String-Argument

Syntax:

external_define(dll, name, calltype, restype, argnumb, argtype[0], argtype[1], ...argtype[10]);

Argument Typ Beschreibung
dll Zeichenfolge The name of the DLL file (string)
name Zeichenfolge The name of the function (string)
calltype Externe Aufrufart Konstante (dll_*) The calling convention used
restype Externe Datentypkonstante (ty_*) The type of the result to expect
argnumb Real The number of arguments (0 - 10)
argtype[0 ... 10] Externe Datentypkonstante (ty_*) The types of the arguments being used

Rückgabe:

External Function

Beispiel:

my_funcion = external_define("MyDLL.dll", "MyMin", dll_cdecl, ty_real, 2, ty_real, ty_real);

Im obigen Beispiel code wird eine externe Funktion namens "MyMin" mit zwei arguments definiert.
