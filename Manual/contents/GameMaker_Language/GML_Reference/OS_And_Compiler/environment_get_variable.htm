environment_get_variable

Diese Funktion gibt den Wert (eine Zeichenkette) der Umgebungsvariablen mit dem angegebenen Namen (ebenfalls eine Zeichenkette) zurück. Sie können die verfügbaren Umgebungsvariablen auf macOS und Ubuntu (Linux) abrufen, indem Sie "env" in die Terminal-App eingeben, und für Informationen über die Umgebungsvariablen von Windows erhalten Sie, wenn Sie die Eingabeaufforderung verwenden, indem Sie "echo %PATH%" eingeben, und in der PowerShell ist es "ls env". Beachten Sie, dass sowohl auf macOS als auch auf Ubuntu (Linux) die Umgebungsvariable "HOME" Umgebungsvariable den Pfad "~/" zurückgibt, der auf "/Users/<username>" auf macOS und "/home/<username>" auf Ubuntu (Linux) abgebildet wird.

HINWEIS: Diese Funktion ist nur für Windows, macOS und Ubuntu (Linux) verfügbar.

Syntax:

environment_get_variable(name);

ArgumentTyp Beschreibung
name The name (a string) of the environment variable to check.

Rückgabe:

Beispiel:

e_str = environment_get_variable("APPDATA");

Die obige code gibt den vollständigen Pfad für die Umgebungsvariable "%appdata%" zurück, der normalerweise "C:\Users\{username}\AppData\Roaming" lautet.
