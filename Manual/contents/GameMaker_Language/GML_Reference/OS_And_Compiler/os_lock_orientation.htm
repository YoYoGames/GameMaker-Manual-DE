os_lock_orientation

Mit dieser Funktion können Sie Ihr Gerät in der aktuellen Ausrichtung "sperren", bis Sie es "freigeben", um alle in den Spieloptionen aktivierten Ausrichtungen für dieses Ziel wieder zuzulassen platform. Beachten Sie, dass Sie sich vor dem Sperren wahrscheinlich vergewissern sollten, dass die Ausrichtung wie gewünscht ist.

Syntax:

os_lock_orientation(flag)

ArgumentTyp Beschreibung
flag Set to true oder false, um die Ausrichtungssperre zu aktivieren oder zu deaktivieren.

Rückgabe:

Beispiel:

if (os_type == os_android) || (os_type == os_ios)
{
    os_lock_orientation(true);
}

Die obige code prüft den OS-Typ und wenn er entweder Android oder iOS ist, wird die Orientierungssperre als true gekennzeichnet.
