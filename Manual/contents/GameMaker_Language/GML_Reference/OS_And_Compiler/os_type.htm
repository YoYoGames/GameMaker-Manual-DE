os_type

Diese schreibgeschützte Variable enthält eine der verschiedenen Konstanten, mit denen GameMaker Ihnen mitteilt, für welches Betriebssystem das Spiel erstellt wurde. Beachten Sie, dass dies nicht notwendigerweise mit dem Betriebssystem des Geräts übereinstimmt, auf dem es läuft, da - zum Beispiel - Ihr Spiel auf einem Amazon Fire Betriebssystem laufen könnte, aber für das Android platform erstellt wurde (in diesem Fall wird os_type zu os_android).

Die folgenden Konstanten können zurückgegeben werden:

OS-Typ Konstante
Konstante Beschreibung
os_windows Windows-Betriebssystem
os_uwp Windows 10 Universal Windows Plattform
os_operagx Oper GX
os_linux Linux
os_macosx macOS X
os_ios iOS (iPhone, iPad, iPod Touch)
os_tvos Apple tvOS
os_android Android
os_ps4 Sony PlayStation 4
os_ps5 Sony PlayStation 5
os_xboxone Microsoft Xbox One
os_xboxseriesxs Microsoft Xbox Serie X/S
os_switch Nintendo Switch
os_unknown Unbekanntes OS

Syntax:

os_type

Rückgabe:

OS Type Constant

Beispiel:

switch (os_type)
{
    case os_windows: global.Config = 0; break;
    case os_android: global.Config = 1; break;
    case os_linux: global.Config = 2; break;
    case os_macosx: global.Config = 3; break;
    case os_ios: global.Config = 4; break;
    case os_winphone: global.Config = 5; break;
}

Die obige code überprüft das Betriebssystem, auf dem das Spiel läuft, und setzt eine globale Variable entsprechend.
