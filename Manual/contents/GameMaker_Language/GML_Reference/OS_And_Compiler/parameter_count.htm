Parameter_Zahl

Befehlszeilenparameter sind die zusätzlichen Befehle, die Sie einer Exe hinzufügen können, um die Ausführung des Programms zu ändern. Sie können die Anzahl der Parameter für das aktuelle Spiel mit dieser Funktion ermitteln, wobei der erste Parameter den Index 1 und der letzte den Index der von der Funktion zurückgegebene Index (ein Wert von 0 ist insofern besonders, als es sich um den Dateinamen der ausführbaren Datei des Spiels handelt, einschließlich des Pfads). Es sollte beachtet werden, dass diese Funktion für HTML5 platform funktioniert und die url-Parameter abruft.

Syntax:

parameter_count();

Rückgabe:

Beispiel:

p_num = parameter_count();

Die obige code speichert die Anzahl der Befehlszeilenparameter, die für das Spiel verwendet wurden, in der Variablen "p_num".
