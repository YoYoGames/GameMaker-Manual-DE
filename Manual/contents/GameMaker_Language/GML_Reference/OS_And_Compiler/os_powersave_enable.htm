os_powersave_enable

Mit dieser Funktion können Sie die Energiesparfunktionen des Geräts ein- oder ausschalten. Dies ist wichtig, da bestimmte Spiele (z. B. solche, die die Neigungsfunktionen verwenden) möglicherweise keine Ereignisse erzeugen, die das Betriebssystem als Benutzereingabe interpretieren kann, und so den Bildschirm abschalten oder das Spiel beenden. Wenn Sie diese Funktion auf false setzen, können Sie die Energiesparfunktionen deaktivieren und sicherstellen, dass der Bildschirm (und das Spiel) immer funktioniert.

HINWEIS: Dies gilt nur für die Zielscheiben iOS und Android.

Syntax:

os_powersave_enable(flag)

ArgumentTyp Beschreibung
flag Set to true oder false können Sie die Energiesparfunktionen aktivieren oder deaktivieren.

Rückgabe:

Beispiel:

if (os_type == os_android) || (os_type == os_ios)
{
    os_powersave_enable(false);
}

Die obige code prüft den Betriebssystemtyp und wenn es entweder Android oder iOS ist, werden die Energiesparfunktionen deaktiviert.
