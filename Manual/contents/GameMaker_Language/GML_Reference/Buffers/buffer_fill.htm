Puffer_füllen

Diese Funktion kann verwendet werden, um ein zuvor erstelltes buffer mit einem bestimmten Datentyp und Wert zu füllen. Die Daten, mit denen Sie buffer füllen, müssen mit dem "Typ" argument dieser Funktion übereinstimmen, was bedeutet, dass Sie nicht versuchen können, mit einer String zu füllen und z.B. den vorzeichenlosen 16-Bit-Integer-Typ zu verwenden. Die Typkonstanten sind die gleichen, die von den Funktionen buffer_read() und buffer_write() verwendeten Funktionen. Die "Größe" ist die Größe der buffer (in Bytes), die Sie füllen möchten, während "offset" der Offset-Wert (ebenfalls in Bytes) vom Anfang der buffer ist, ab dem die Füllung beginnen soll.

Syntax:

buffer_fill(buffer, offset, type, value, size);

ArgumentTyp Beschreibung
buffer The index of the buffer to fill.
offset The data offset value (in bytes).
type The type of data that is to be written to the buffer (see the list of constants here).
value The data to write.
size The size of the buffer (in bytes) that you wish to fill.

Rückgabe:

Beispiel:

map_buffer = buffer_create(16384, buffer_fixed, 0);
buffer_fill(map_buffer, 0, buffer_u16, 0, 16384);

Die obige code findet den Anfang der buffer mit der in der Variablen "buff" gespeicherten ID und schreibt eine Reihe von vorzeichenbehafteten 16-Bit-Ganzzahlwerten hinein.
