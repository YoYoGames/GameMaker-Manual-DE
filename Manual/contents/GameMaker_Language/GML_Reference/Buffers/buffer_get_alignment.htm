Buffer_get_alignment

Mit dieser Funktion können Sie die Byte-Ausrichtung für die angegebene buffer ID ermitteln.

Syntax:

buffer_get_alignment(buffer);

ArgumentTyp Beschreibung
buffer The index of the buffer to check.

Rückgabe:

Beispiel:

alignment = buffer_get_alignment(buff);

Die obige code ermittelt die Ausrichtung der buffer aus dem in der Variablen "buff" indizierten Wert und speichert sie in einer Variablen.
