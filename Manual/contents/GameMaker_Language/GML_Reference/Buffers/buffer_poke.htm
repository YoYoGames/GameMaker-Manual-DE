buffer_poke

Mit der Funktion buffer_write() können Sie Daten in die angegebene buffer an der aktuellen "Suchposition" schreiben, wobei jedes Datenstück diese Position um die geschriebenen oder gelesenen Bytes vorverlegt. Es kann jedoch notwendig sein, ein bestimmtes Datenstück zu ändern, ohne die aktuelle Suchposition zu ändern, und in diesem Fall würden Sie diese Funktion verwenden. Sie geben der Funktion einfach einen buffer -Index und die Offset-Position vom buffer -Start (in Bytes) innerhalb dieses buffer an, in das geschrieben werden soll, sowie den Datentyp und den zu schreibenden Wert.

Syntax:

buffer_poke(buffer, offset, type, value);

Argument Typ Beschreibung
buffer Puffer-ID The index of the buffer to use.
offset Real The offset position (in bytes) within the buffer to write the given data to.
type Puffer Datentyp Konstante The type of data that is to be written to the buffer (see the list of constants here).
value Variabel The data to add to the buffer, in accordance with the type specified.

Rückgabe:

N/A

Beispiel:

buffer_poke(buff, 3, buffer_u8, colour_get_blue(image_blend));

Die obige code fügt den Wert der Blaukomponente der für die Bildüberblendung verwendeten Farbe in die buffer ein, die in der Variablen "buff" an der dritten Position in der buffer indiziert ist und einen vorzeichenlosen 8-Bit-Wert darstellt.
