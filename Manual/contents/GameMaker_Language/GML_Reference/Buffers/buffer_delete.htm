Puffer_Löschen

Mit dieser Funktion können Sie eine buffer, die Sie zuvor mit der Funktion buffer_create() erstellt wurde, aus dem Speicher löschen. Dabei werden die zu seiner Erstellung verwendeten Ressourcen freigegeben und alle Daten, die er möglicherweise noch enthält, entfernt.

HINWEIS: Es ist wichtig, dass Sie dynamisch erstellte Ressourcen immer aus dem Speicher entfernen, wenn Sie sie nicht mehr benötigen, um Speicherlecks zu vermeiden.

WICHTIG! Wenn Sie ein buffer erstellen, ist der Indexwert, der zur Identifizierung verwendet wird, ein ganzzahliger Wert, der bei 0 beginnt. Diese Indizes werden von GameMaker wiederverwendet, so dass ein zerstörter buffer Indexwert anschließend von einem neu erstellten verwendet werden kann, Wir empfehlen, dass Sie jede Variable, die einen Pufferindex enthält, auf -1 setzen, nachdem Sie den buffer zerstört haben.

Syntax:

buffer_delete(buffer)

ArgumentTyp Beschreibung
buffer The index of the buffer to delete.

Rückgabe:

Beispiel:

buffer_delete(player_buffer);
player_buffer = -1;

Die obige code löscht die zuvor erstellte buffer mit der in der Variablen "player_buffer" gespeicherten ID und setzt die Variable dann auf -1.
