buffer_tell

Wenn Sie Daten auf einer buffer lesen oder schreiben, indem Sie die buffer_read() oder buffer_write() wird die aktuelle "seek"-Position um die geschriebenen oder gelesenen Bytes vorverlegt, und mit können Sie die aktuelle "seek"-Position zur Verwendung in anderen buffer Funktionen ermitteln. Wenn beispielsweise die Ausrichtung von buffer auf 4 Bytes eingestellt ist und Sie ein einzelnes Datenstück mit einer Größe von 1 Byte schreiben, führen Sie eine buffer_tell() aus, ausführen, erhalten Sie den Rückgabewert 1. Wenn Sie jedoch ein weiteres Datenstück schreiben, das ebenfalls 1 Byte groß ist, und dann buffer_tell() ausführen, erhalten Sie den Rückgabewert 5, da die Ausrichtung die Daten auf diese Position "aufgefüllt" hat.

Syntax:

buffer_tell(buffer);

ArgumentTyp Beschreibung
buffer The index of the buffer to use.

Rückgabe:

Beispiel:

var pos = buffer_tell(buff); buffer_seek(buff, buffer_seek_start, 0);
val[0] = buffer_read(buff, buffer_S16);
val[1] = buffer_read(buff, buffer_S16);
val[2] = buffer_read(buff, buffer_S16);
buffer_seek(buff, buffer_seek_start, pos);

Die obige code speichert die aktuelle Suchposition innerhalb der buffer, die in der Variablen "buff" indiziert ist, in der lokalen Variablen "pos". Die buffer Suchposition wird dann auf den Anfang der buffer gesetzt, und drei Daten werden in ein Array gelesen, bevor die Suchposition von buffer wieder auf die vorherige Position gesetzt wird.
