puffer_laden_teilweise

Diese Funktion lädt einen Teil der Daten von buffer, die zuvor mit den buffer_save() Funktionen gespeichert wurden, in eine bestehende buffer. Sie geben die ID der zuvor erstellten buffer an, in die geladen werden soll, dann die gespeicherte buffer Datei, die geladen werden soll, und dann den Offset vom Startpunkt der buffer (in Bytes), von dem aus Sie die Daten laden möchten. Die folgenden arguments dienen zum Einstellen der Länge der buffer Daten (in Bytes) ab dem anfänglichen Offset-Punkt, den Sie laden möchten, und des Offset-Punktes, bis zu dem die Daten in buffer geladen werden sollen (ebenfalls in Bytes).

Bitte lesen Sie die Seite buffer_load() für plattformspezifische Hinweise.

Syntax:

buffer_load_partial(buffer, filename, offset, src_len, dest_offset);

ArgumentTyp Beschreibung
buffer The index of the buffer to load into (destination).
filename The name of the file to load from (source).
offset The offset within the destination buffer to load to (in bytes).
src_len The length of the part of the source buffer to load (in bytes).
dest_offset The offset where to start putting the partial data in the destination buffer (in bytes).

Rückgabe:

N/A

Beispiel:

buff = buffer_create(256, buffer_grow, 1);
var _file = "save.dat";
var _so = 6;
var _sl = 5;
var _do= 0;
buffer_load_partial(buff, _file, _so, _sl, _do);

Die obige code erstellt eine neue "grow" buffer und lädt dann einen Teil der in der Datei "save.dat" gespeicherten Daten in diese Datei.
