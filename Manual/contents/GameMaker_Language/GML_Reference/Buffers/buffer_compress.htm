puffer_komprimieren

Mit dieser Funktion können Sie einen Teil (oder die gesamte Datei) von buffer mit zlib-Kompression komprimieren. Sie geben die ID der zu komprimierenden buffer an (wie sie von buffer_create()), den zu verwendenden Offset innerhalb von buffer in Bytes und die Größe der zu komprimierenden buffer Daten (ebenfalls in Bytes). Die Funktion gibt einen neuen buffer ID-Wert für die komprimierte buffer zurück, oder einen Wert kleiner als 0, wenn sie aus irgendeinem Grund fehlgeschlagen ist. Diese Funktion verändert die ursprüngliche buffer nicht.

Syntax:

buffer_compress(buffer, offset, size);

ArgumentTyp Beschreibung
buffer The index of the buffer to compress.
offset The offset within the buffer to compress (in bytes).
size The size of the buffer area to compress (in bytes).

Rückgabe:

Buffer ID

Beispiel:

var srcBuff = buffer_create(1024, buffer_grow, 1);
buffer_write(srcBuff, global.DataString);
var cmpBuff = buffer_compress(srcBuff, 0, buffer_tell(srcBuff));
buffer_save(cmpBuff, "Player_Save.sav");
buffer_delete(srcBuff);
buffer_delete(cmpBuff);

Die obige code erstellt eine buffer und füllt sie mit den Daten einer Zeichenkette. Diese buffer wird dann komprimiert und gespeichert, und sowohl die Quelle als auch die komprimierte buffers werden gelöscht.
