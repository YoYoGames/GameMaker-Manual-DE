buffer_base64_encode

Diese Funktion konvertiert die Daten aus der angegebenen buffer in eine Zeichenkette im base64-Format. Dies ist ein häufig verwendetes Kodierungsschema, das für alle Medien verwendet wird, die als Text gespeichert oder über das Internet übertragen werden müssen, und macht die Ausgabe für das menschliche Auge unlesbar. Dazu müssen Sie ein bereits erstelltes buffer, den Offset-Wert (den Punkt innerhalb von buffer, an dem Sie mit der Kodierung beginnen möchten) sowie die Größe des zu kodierenden buffer Speichers in Bytes angeben.

Syntax:

buffer_base64_encode(buffer, offset, size);

ArgumentTyp Beschreibung
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

Rückgabe:

Beispiel:

var b_str = buffer_base64_encode(buff, 0, buffer_get_size(buff))

Die obige code kodiert die vollständigen Daten, die in der buffer gespeichert sind und durch die Variable "buff" indiziert werden, und speichert die zurückgegebene Zeichenkette in der lokalen Variable "b_str".
