Puffer_crc32

Diese Funktion nimmt Eingabedaten von buffer entgegen und gibt einen crc32-Prüfsummen-Hash zurück. Sie geben die buffer ID der zu verwendenden buffer an, dann einen Offset-Wert (in Bytes) für den Anfang und dann eine Größe (wieder in Bytes) für die zu hashende Region, und die Funktion gibt einen 32-Bit-Ganzzahlwert für diese Region zurück.

Syntax:

buffer_crc32(buffer, offset, size);

ArgumentTyp Beschreibung
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

Rückgabe:

Beispiel:

check_val = buffer_crc32(buff, 0, buffer_get_size(buff));

Die obige code erstellt einen crc32-Prüfsummen-Hashwert für die vollständigen Daten, die in der durch die Variable "buff" indizierten buffer gespeichert sind, und speichert den zurückgegebenen Integer-Hashwert in der Variablen "check_val".
