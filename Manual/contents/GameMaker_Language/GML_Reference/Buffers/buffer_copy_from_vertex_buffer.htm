puffer_kopieren_vom_vertex_puffer

Mit dieser Funktion können einige (oder alle) Scheitelpunktdaten, die in einem Scheitelpunkt buffer gespeichert sind, in ein zuvor erstelltes reguläres buffer kopiert werden. Wenn Sie mit dieser Funktion von einem Scheitelpunkt buffer in ein reguläres buffer kopieren, müssen beide buffers zuvor erstellt worden sein (mit den Befehlen vertex_create_buffer() und buffer_create() Funktionen). Sie können den Bereich der Scheitelpunktdaten angeben, den Sie in die buffer kopieren möchten, wobei der Startscheitelpunkt irgendwo zwischen 0 und der Anzahl der Scheitelpunkte -1 liegen kann, und Sie können die Anzahl der Scheitelpunkte angeben, die von diesem Punkt an kopiert werden sollen. Sie können die Funktion vertex_get_number() für den Scheitelpunkt buffer verwenden, um die Gesamtzahl der gespeicherten Scheitelpunkte zu ermitteln. Schließlich geben Sie den Index buffer an, in den die Scheitelpunktdaten kopiert werden sollen, sowie einen Datenoffset, um die Position zu definieren, an die die Scheitelpunktdaten im Ziel buffer kopiert werden sollen.

Syntax:

buffer_copy_from_vertex_buffer(vertex_buffer, start_vertex, num_vertices, dest_buffer, dest_offset);

Argument Typ Beschreibung
vertex_buffer Vertex Buffer ID The index of the vertex buffer to copy from.
start_vertex Real The starting vertex.
num_vertices Real The total number of vertices to use.
dest_buffer Puffer-ID The index of the buffer to copy to.
dest_offset Real The offset position to copy the data to (in bytes).

Rückgabe:

N/A

Beispiel:

var v_num = vertex_get_number(model_buff); buffer_copy_from_vertex_buffer(model_buffer, 0, v_num - 1, player_buffer, 0);

Die obige code kopiert die Scheitelpunktdaten, die im Scheitelpunkt buffer gespeichert sind, der in der Variable "model_buffer" indiziert ist, und fügt sie dann in den buffer ein, der in der Variable "player_buffer" indiziert ist.
