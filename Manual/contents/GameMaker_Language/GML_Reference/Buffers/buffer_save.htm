puffer_sichern

Mit dieser Funktion können Sie den Inhalt von buffer in einer Datei speichern, die Sie dann mit der Funktion buffer_load() Funktion.

HINWEIS: Bei HTML5 wird der Inhalt von buffer als base64-kodierte Zeichenkette gespeichert, wenn diese Funktion verwendet wird.

Syntax:

buffer_save(buffer, filename);

ArgumentTyp Beschreibung
buffer The index of the buffer to save.
filename The name of the file to save as.

Rückgabe:

Beispiel:

buffer_save(buff, "Player_Save.sav");

Speichert den aktuellen Inhalt von buffer mit der in der Variablen "buff" gespeicherten ID in eine Datei.
