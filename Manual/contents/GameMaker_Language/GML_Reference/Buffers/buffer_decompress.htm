puffer_dekomprimieren

Mit dieser Funktion können Sie eine zuvor komprimierte buffer mit zlib-Kompression dekomprimieren. Sie geben die buffer ID an, die dekomprimiert werden soll, und die Funktion gibt eine neue buffer ID zurück, die die unkomprimierten Daten enthält. Wenn die Dekomprimierung fehlgeschlagen ist (z. B. wenn Sie eine buffer übergeben, die nicht komprimiert wurde), gibt die Funktion stattdessen einen Wert kleiner als 0 zurück.

Syntax:

buffer_decompress(buffer);

ArgumentTyp Beschreibung
buffer The index of the buffer to decompress.

Rückgabe:

Buffer ID

Beispiel:

var cmpBuff = buffer_load("Player_Save.sav");
var srcBuff = buffer_decompress(cmpBuff);
global.DataString = buffer_read(srcBuff, buffer_string);

Die obige code lädt zuerst eine gespeicherte buffer, dekomprimiert sie dann und liest schließlich die String-Daten aus der dekomprimierten buffer in eine globale Variable.
