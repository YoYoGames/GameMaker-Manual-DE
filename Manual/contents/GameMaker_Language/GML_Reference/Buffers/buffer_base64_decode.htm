buffer_base64_decode

Mit dieser Funktion können Sie eine base64-kodierte Zeichenkette (erstellt mit der Funktion buffer_base64_encode() Funktion) in einen buffer dekodieren. Diese Funktion erstellt den buffer (als 1 Byte ausgerichteten "grow"-Puffer) und gibt den eindeutigen Index für den buffer zurück, der in allen weiteren Funktionsaufrufen verwendet werden sollte.

HINWEIS: Es ist wichtig, dass Sie alle dynamisch erstellten Ressourcen wie diese aus dem Speicher entfernen, wenn Sie sie nicht mehr benötigen, um Speicherlecks zu vermeiden. Wenn Sie also mit der von Ihnen erstellten buffer fertig sind, sollten Sie sie wieder freigeben mit buffer_delete().

Syntax:

buffer_base64_decode(string);

ArgumentTyp Beschreibung
string The base64 encoded string to decode

Rückgabe:

Buffer ID

Beispiel:

ini_open("Save.ini");
buff = buffer_base64_decode(ini_read_string("Save", "Slot1", ""));
ini_close();

Die obige code öffnet eine ini-Datei und liest dann eine Zeichenkette daraus in die Decode-Funktion. Diese Funktion gibt einen Index buffer zurück, der in der Variablen "buff" gespeichert wird und die zuvor kodierten und gespeicherten Daten enthält. Die Ini-Datei wird dann geschlossen.
