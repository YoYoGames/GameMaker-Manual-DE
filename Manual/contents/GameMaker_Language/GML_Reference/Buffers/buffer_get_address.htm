buffer_get_address

Mit dieser Funktion können Sie einen Zeiger auf die rohe, ausgerichtete Adresse buffer erhalten. Dies ist in erster Linie für die Verwendung mit Erweiterungen gedacht, da Sie diesen Wert an sie weitergeben können, damit sie auf die Daten von buffer zugreifen können. Beachten Sie, dass es ein paar Dinge gibt die Sie mit buffer nicht tun können, wenn Sie von einer Erweiterung aus darauf zugreifen:

  • Sie können den Speicher nicht freigeben.
  • Sie können die Größe des Speichers nicht ändern.
  • Sie können nicht über die Adresse buffer plus die Größe buffer hinaus schreiben (Sie können die Funktion buffer_get_size() Funktion verwenden), sonst erhalten Sie einen Out-of-Bounds-Fehler.

Syntax:

buffer_get_address(buffer);

ArgumentTyp Beschreibung
buffer The index of the buffer to use.

Rückgabe:

Beispiel:

var b_address = buffer_get_address(buff_model);
var b_end_address = b_address + buffer_get_size(buff_model);

Die obige code ermittelt die Speicheradresse von buffer, die in der Variablen "buff_model" indiziert ist, und ermittelt dann die Speicheradresse für das Ende von buffer mit Hilfe der Funktion buffer_get_size(), wobei beide Werte in lokalen Variablen für weitere Verwendung.
