Puffer_Kopie

Mit dieser Funktion können Sie ein Segment (oder alle) der in einem buffer gespeicherten Daten in ein anderes kopieren. Wenn Sie zwei buffers verwenden und von einem zum anderen kopieren, müssen beide buffers zuvor mit der Funktion buffer_create() Funktion erstellt worden sein (zum Beispiel), und Sie können einen Datenoffset (in Bytes) für den Startpunkt der zu kopierenden Daten von der Quelle buffer relativ zum Anfang der buffer angeben, sowie einen weiteren Datenoffset, um die Position zu definieren, an die die Daten kopiert werden sollen im Ziel buffer zu kopieren.

HINWEIS: Sie können nicht auf dieselbe buffer kopieren.

Syntax:

buffer_copy(src_buffer, src_offset, size, dest_buffer, dest_offset);

ArgumentTyp Beschreibung
src_buffer The index of the buffer to copy from.
src_offset The data offset to start copying from (in bytes).
size The size of the data to copy (in bytes).
dest_buffer The index of the buffer to copy to.
dest_offset The offset position to copy the data to (in bytes).

Rückgabe:

Beispiel:

buffer_copy(buff1, 0, 2048, buff2, 2048);

Die obige code kopiert die in buffer gespeicherten Daten, die in der Variablen "buff1" indiziert sind, und fügt sie dann in buffer ein, das in der Variablen "buff2" indiziert ist, jedoch um 2048 Bytes versetzt.
