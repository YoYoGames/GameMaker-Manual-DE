buffer_set_surface

Mit dieser Funktion können Sie Informationen von einer buffer auf eine bestimmte Oberfläche schreiben. Sowohl die buffer, von der gelesen wird, als auch die Oberfläche, auf die geschrieben wird, müssen zuvor erstellt worden sein, und Sie können einen Offset-Wert in der buffer angeben, von dem aus gelesen werden soll. Beachten Sie, dass das Lesen immer am Anfang von buffer plus dem Offset-Wert beginnt und nicht an der aktuellen Suchposition plus dem Offset-Wert.

HINWEIS: Diese Funktion hat sich ab dem GameMaker-Update 2.3.1 geändert. Frühere Versionen verwendeten die Konstante buffer_surface_copy, die nun veraltet ist.

Syntax:

buffer_set_surface(buffer, surface, offset);

ArgumentTyp Beschreibung
buffer The index of the buffer to use.
surface The index of the surface to use.
offset The data offset value.

Rückgabe:

Beispiel:

buffer_set_surface(buff, application_surface, 0);

Die obige code kopiert alle in buffer gespeicherten Daten, die in der Variablen "buff" indiziert sind, ohne Versatz auf die Anwendungsoberfläche.
