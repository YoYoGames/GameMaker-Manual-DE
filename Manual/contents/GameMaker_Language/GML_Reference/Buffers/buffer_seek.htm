Puffersuchlauf

Diese Funktion kann verwendet werden, um sich durch eine buffer zu bewegen und den Anfang, das Ende oder eine Position relativ zu derjenigen zu finden, die zuletzt beim Lesen oder Schreiben von Daten verwendet wurde. Der "Offset"-Wert ist der Offset (in Bytes), der zur gegebenen Suchposition hinzuzufügen ist. Wenn beispielsweise die Basis relativ ist und der Offset 4 beträgt, dann wird die Position buffer um 4 Bytes von der aktuellen Position aus verschoben. Bitte beachten Sie das Folgende:

  • Sie können sowohl negative Werte für den Offset verwenden, um durch die buffer zurückzusuchen, als auch positive Werte.
  • Wenn die buffer vom Typ "Wrap" ist und Sie über das Ende der buffer hinausgehen, wird die Suchposition ebenfalls umbrochen.
  • Ist buffer nicht vom Typ "Wrap", wird der Suchlauf am Anfang oder Ende von buffer festgehalten, auch wenn der Offset den Suchlauf außerhalb der Grenzen von buffer führen würde.

Die folgenden Konstanten werden als "Basis" argument für die Suche nach akzeptiert:

Puffer-Suchkonstante
Konstante Beschreibung
buffer_seek_start Der Anfang des Puffers
buffer_seek_relative Eine Position relativ zur aktuellen Lese-/Schreibposition
buffer_seek_end Das Ende des Puffers

Syntax:

buffer_seek(buffer, base, offset);

Argument Typ Beschreibung
buffer Puffer-ID The index of the buffer to use.
base Real The base position to seek.
offset Real The data offset value.

Rückgabe:

N/A

Beispiel:

buffer_seek(buff, buffer_seek_start, 0);
buffer_write(buff, buffer_s16, 0);
buffer_write(buff, buffer_s16, x);
buffer_write(buff, buffer_s16, y);

Die obige code findet den Anfang der buffer mit der in der Variablen "buff" gespeicherten ID und schreibt eine Reihe von vorzeichenbehafteten 16-Bit-Ganzzahlwerten hinein.
