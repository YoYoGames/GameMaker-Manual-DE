puffer_async_gruppe_option

Mit dieser Funktion können Sie einige platform spezifische Optionen für die zu speichernde/ladende buffer Gruppe einstellen. Die verfügbaren Optionen sind wie folgt:

  • "subtitle" oder "slottitle" - Der Wert für diese Option ist eine Zeichenkette, die dem Benutzer angezeigt wird, wenn er seine Speicherdaten im Betriebssystem verwaltet. Dies ist nur beim Speichern von Daten wichtig, nicht beim erneuten Laden.
  • "showdialog" - Der Wert für diese Option ist ein boolescher Wert true oder false. Wenn er auf true gesetzt ist, zeigt GameMaker beim Speichern oder Laden die System-Benutzeroberfläche an, andernfalls wird ein QuickSave/Quickload durchgeführt, ohne dass die Benutzeroberfläche angezeigt wird. Normalerweise brauchen Sie diese Option nur, wenn Sie mehrere Speicherslots unterstützen, damit der Benutzer einen Slot auswählen kann, aber wenn Sie nur einen Slot pro Benutzer unterstützen, setzen Sie diese Option auf false.
  • "savepadindex" - Bei dieser Option geben Sie den Pad-Index des Spielers an, der gerade speichert oder lädt, und das System schreibt Daten in den Speicherordner dieses Spielers und liest Daten daraus.
  • "saveslotsize" - Diese Option erfordert, dass Sie die tatsächliche Größe in Bytes angeben, die Sie speichern möchten (Sie können also eine buffer_seek() und buffer_tell() um diese Größe zu erhalten). Beachten Sie, dass die Angabe dieses Wertes nicht zwingend erforderlich ist, da allen Speicherungen ein Mindestplatz zugewiesen wird, der in der Regel mit dem Ziel platform variiert.
  • "ps_create_backup" - (nur PlayStation) - Wenn Sie dies auf true setzen, wird sichergestellt, dass die Speicherdaten gesichert werden, wenn das Speicherverzeichnis ausgehängt wird (d.h. wenn die Funktion buffer_async_group_end() aufgerufen wird). Weitere Informationen finden Sie in der Konsolendokumentation.

WICHTIG! Diese Funktion ist derzeit nur für die Ziele PSVita, PS4 und XBox One gültig modules. Bei allen anderen Zielen wird sie nichts bewirken.

Syntax:

buffer_async_group_option(option, value);

Argument Typ Beschreibung
option Zeichenfolge The option to set.
value Real oder Zeichenfolge The value to set (can be a real or string, depending on the option).

Rückgabe:

N/A

Beispiel:

buffer_async_group_begin("save_folder_name");
buffer_async_group_option("showdialog", false);
buffer_async_group_option("slottitle", "Catch The Haggis Save");
buffer_async_group_option("subtitle", "All your haggis are saved here!");
save = buffer_save_async(buff, "Player_Save.sav", 0, 16384);
buffer_async_group_end();

Die obige code startet eine buffer Gruppe und legt dann die Gruppenoptionen fest, bevor sie 4 Dateien für die asynchrone Speicherung festlegt. Die Gruppendefinition wird dann beendet (und das Speichern beginnt).
