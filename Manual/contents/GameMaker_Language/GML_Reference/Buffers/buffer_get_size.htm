puffer_holen_grösse

Mit dieser Funktion können Sie die Größe der angegebenen buffer in Bytes ermitteln.

Syntax:

buffer_get_size(buffer);

Argument Typ Beschreibung
buffer Puffer-ID The index of the buffer to get the size of.

Rückgabe:

Real

Beispiel:

var _size = buffer_get_size(player_data);
var _temp = buffer_create(_size, buffer_fixed, 0);

Die obige code erstellt eine neue buffer und speichert ihren Index in der lokalen Variablen "_temp", wobei die Größe dieser neuen buffer derjenigen der buffer entspricht, die in der Variablen "player_data" indiziert ist.
