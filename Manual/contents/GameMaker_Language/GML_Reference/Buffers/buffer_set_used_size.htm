buffer_set_used_size

Diese Funktion ist in erster Linie für die Verwendung innerhalb von Erweiterungen gedacht und ermöglicht es Ihnen, die "benutzte" Größe der angegebenen buffer einzustellen, d.h. die Anzahl der Bytes, die in sie geschrieben wurden.

Wenn Sie Daten von einer Erweiterung in ein buffer schreiben, weiß GameMaker nicht, wie viel von buffer von der Erweiterung code gefüllt wurde und kann diese Daten nicht lesen. Diese Funktion kann von der Erweiterung aufgerufen werden, um der Engine mitzuteilen, wie viele Bytes an Daten in die buffer geschrieben wurden, so dass die Maschine diese Daten lesen kann.

Syntax:

buffer_set_used_size(buffer, size);

ArgumentTyp Beschreibung
buffer The index of the buffer to use.
size The number of bytes to set as the "used" size.

Rückgabe:

Beispiel:

buffer_write(_bufferAddress, buffer_u8, 1);
buffer_write(_bufferAddress, buffer_u8, 2);
buffer_write(_bufferAddress, buffer_u16, 400);

buffer_set_used_size(_bufferAddress, 4);

Die obigen Funktionen würden in einer Erweiterung aufgerufen, um Daten in eine GameMaker buffer zu schreiben (über ihre Speicheradresse) und dann die Anzahl der geschriebenen Bytes zu setzen, damit die Engine diese Daten lesen kann.
