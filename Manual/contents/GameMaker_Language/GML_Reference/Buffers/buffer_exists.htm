Puffer_vorhanden

Mit dieser Funktion kann eine Variable daraufhin überprüft werden, ob sie einen gültigen buffer ID-Wert enthält oder nicht. Ist dies der Fall, gibt die Funktion true zurück, andernfalls wird false ausgegeben.

Syntax:

buffer_exists(buffer)

ArgumentTyp Beschreibung
buffer The index of the buffer to check.

Rückgabe:

Beispiel:

if buffer_exists(buff)
{
    buffer_delete(buff);
}

Die obige code prüft, ob die Variable "buff" eine buffer ID enthält, und wenn dies der Fall ist, wird die buffer gelöscht.
