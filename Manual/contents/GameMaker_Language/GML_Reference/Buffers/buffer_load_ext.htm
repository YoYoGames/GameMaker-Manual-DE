buffer_load_ext

Diese Funktion lädt die Daten von buffer, die zuvor mit den buffer_save() Funktionen gespeichert wurden, in eine bestehende buffer. Sie geben die ID der zuvor erstellten buffer an, in die geladen werden soll, dann die zu ladende gespeicherte Datei buffer und schließlich den Offset vom Anfang der buffer (in Bytes), in die Sie die Daten laden möchten.

Bitte lesen Sie die Seite buffer_load() für plattformspezifische Hinweise.

Syntax:

buffer_load_ext(buffer, filename, offset);

ArgumentTyp Beschreibung
buffer The index of the buffer to load into.
filename The name of the file to load from.
offset The offset within the buffer to load to (in bytes).

Rückgabe:

Beispiel:

var pos = buffer_seek(player_buffer, buffer_seek_end, 0);
buffer_load(player_buffer, "Data_Save.sav", pos);

Das obige code ermittelt zunächst die Position des Endes des buffer, das in der Variablen player_buffer" indiziert ist, und lädt dann die Daten aus dem angegebenen in diese Position (beachten Sie, dass dieses Beispiel nur mit den Typen grow" oder wrap" buffer funktioniert).
