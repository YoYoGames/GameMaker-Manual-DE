puffer_lesen

Mit dieser Funktion können Sie Daten aus einer zuvor erstellten buffer lesen. Der Rückgabewert hängt von der Art der gelesenen Daten ab, die wiederum durch die folgenden Konstanten definiert wird:

Konstante Beschreibung
buffer_u8 Eine 8-Bit-Ganzzahl ohne Vorzeichen. Dies ist ein positiver Wert von 0 bis 255.
buffer_s8 Eine 8-Bit-Ganzzahl mit Vorzeichen. Dies kann ein positiver oder negativer Wert von -128 bis 127 sein (0 wird als positiv eingestuft).
buffer_u16 Eine 16-Bit-Ganzzahl ohne Vorzeichen. Dies ist ein positiver Wert von 0 - 65.535.
buffer_s16 Eine ganze Zahl mit Vorzeichen und 16 Bit. Dies kann ein positiver oder negativer Wert zwischen -32.768 und 32.767 sein (0 wird als positiv eingestuft).
buffer_u32 Eine 32-Bit-Ganzzahl ohne Vorzeichen. Dies ist ein positiver Wert von 0 bis 4.294.967.295.
buffer_s32 Eine ganze Zahl mit Vorzeichen und 32 Bit. Dies kann ein positiver oder negativer Wert zwischen -2.147.483.648 und 2.147.483.647 sein (0 wird als positiv eingestuft).
buffer_u64 Eine 64-Bit-Ganzzahl ohne Vorzeichen.
buffer_f16 Ein 16-Bit-Float. Dies kann ein positiver oder negativer Wert innerhalb des Bereichs von +/- 65504 sein. (Wird derzeit nicht unterstützt!)
buffer_f32 Eine 32-Bit-Fließkommazahl. Dies kann ein positiver oder negativer Wert innerhalb des Bereichs von +/-16777216 sein.
buffer_f64 Ein 64-Bit-Float.
buffer_bool Ein boolescher Wert. Kann nur entweder 1 oder 0 sein (true oder false)
buffer_string Eine Zeichenkette von beliebiger Größe.
buffer_text Eine Zeichenkette beliebiger Größe, ohne das abschließende Null-Zeichen.

Wenn die Funktion erfolgreich ist, gibt sie einen Wert des angegebenen Typs zurück, wenn sie jedoch fehlschlägt, wird ein runner Fehler verursacht.

HINWEIS Die Verwendung des falschen Datentyps für die zu lesenden Daten führt dazu, dass fehlerhafte Werte zurückgegeben werden.

Syntax:

buffer_read(buffer, type)

Argument Typ Beschreibung
buffer Puffer-ID The index of the buffer to read from.
type Puffer Datentyp Konstante The type of data that is to be read from the buffer (see the list of constants above).

Rückgabe:

Real, Boolean or String

Beispiel:

var cmd = buffer_read(buff, buffer_s16);

Die obige code liest von der buffer mit der in der Variable "buff" gespeicherten ID einen vorzeichenbehafteten 16-Bit-Wert in die lokale Variable "cmd".
