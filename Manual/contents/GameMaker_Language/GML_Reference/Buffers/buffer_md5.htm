Puffer_md5

In der Kryptografie ist MD5 (Message-Digest-Algorithmus 5) eine weit verbreitete kryptografische Hash-Funktion mit einem 128-Bit-Hash-Wert, die in einer Vielzahl von Sicherheitsanwendungen eingesetzt wird. Sie wird auch häufig zur Überprüfung der Integrität von Daten verwendet. Diese Funktion nimmt die Eingabedaten von einer bestimmten buffer (oder einem Teil der buffer) und gibt den 32-stelligen hexadezimalen MD5-Hashwert zurück, der für diese Daten eindeutig ist. Auf diese Weise können Sie einen einen sicheren Schlüssel, der gespeichert und verwendet werden kann, um die Integrität der Informationen zu prüfen, die z. B. an eine externe server gesendet (oder von dieser empfangen) werden. Wenn Sie diese Funktion auf buffers anwenden, müssen Sie die buffer id des zu verwendenden buffer angeben. verwenden, dann einen Offset-Wert (in Bytes) für den Anfang und dann eine Größe (wiederum in Bytes) für den zu hashenden Bereich.

HINWEIS: MD5 ist nicht völlig sicher und kann geknackt werden. Siehe diese Seite für weitere Informationen.

Syntax:

buffer_md5(buffer, offset, size);

ArgumentTyp Beschreibung
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

Rückgabe:

Beispiel:

check_string = buffer_md5(buff, 0, buffer_get_size(buff));

Die obige code erstellt einen md5-Hash für die vollständigen Daten, die in der durch die Variable "buff" indizierten buffer gespeichert sind, und speichert den zurückgegebenen Hash in der Variablen "check_string".
