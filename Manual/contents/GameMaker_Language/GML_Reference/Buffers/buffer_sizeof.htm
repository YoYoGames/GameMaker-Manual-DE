puffer_größe

Diese Funktion gibt die Größe (in Bytes) der angegebenen Puffer-Datentyp-Konstante ( hier aufgelistet) zurück.

Syntax:

buffer_sizeof(type);

Argument Typ Beschreibung
type Puffer Datentyp Konstante The type of data that is to be checked (see the list of constants here).

Rückgabe:

Real

Beispiel:

var b = 12 * buffer_sizeof(buffer_u8);
buff = buffer_create(b, buffer_fixed, 1);

Die obige code berechnet zunächst die Größe des zu erstellenden buffer, indem sie den vorzeichenlosen 8-Bit-Datentyp mit 12 multipliziert (da wir buffer verwenden werden, um 12 Daten zu speichern), und verwendet dann diesen Wert, um ein festes buffer auf die richtige Größe zu setzen.
