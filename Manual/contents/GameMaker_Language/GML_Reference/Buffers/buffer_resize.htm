buffer_resize

Mit dieser Funktion können Sie die Größe einer gegebenen buffer auf die von Ihnen angegebene Größe (in Bytes) ändern.

Syntax:

buffer_resize(buffer, newsize);

ArgumentTyp Beschreibung
buffer The index of the buffer to change the size of.
newsize The new size of the buffer (in bytes).

Rückgabe:

Beispiel:

if (buffer_get_size(buff) < 16384)
{
    buffer_resize(buff, 16384);
}

Die obige code prüft die Größe der buffer, die in der Variablen "buff" indiziert ist, und wenn sie kleiner ist als der angegebene Wert, wird die buffer in der Größe angepasst.
