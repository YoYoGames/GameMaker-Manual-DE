puffer_base64_decode_ext

Mit dieser Funktion können Sie eine base64-kodierte Zeichenkette (erstellt mit der Funktion buffer_base64_encode() Funktion) in eine buffer dekodieren. Im Gegensatz zur Funktion buffer_base64_decode(), wird hier keine buffer für Sie erstellt, sondern Sie sollten bereits die buffer erstellt haben (siehe buffer_create()), dessen ID Sie dann mit dieser Funktion verwenden würden. Der "Offset" ist die Position innerhalb der buffer zur Dekodierung der angegebenen Zeichenkette (in Bytes).

Syntax:

buffer_base64_decode_ext(buffer, string, offset);

ArgumentTyp Beschreibung
buffer The index of the buffer to decode the string into.
string The base64 encoded string to decode.
offset The data offset value.

Rückgabe:

Beispiel:

buff = buffer_create(16384, buffer_grow, 2);
ini_open("Save.ini");
var str = ini_read_string("Save", "Slot1", "");
buffer_base64_decode_ext(buff, str, 0);
ini_close();

Die obige code erstellt eine buffer und speichert die eindeutige Kennung in der Variablen "buff", öffnet dann eine Ini-Datei und liest eine Zeichenkette daraus in die lokale Variable "str". Diese Zeichenkette wird dann in die neu erstellte buffer dekodiert, bevor bevor die ini-Datei wieder geschlossen wird.
