Pufferspeichersuche

Mit der Funktion buffer_read() können Sie Daten von der angegebenen Adresse buffer an der aktuellen "Suchposition" lesen, wobei jedes zu lesende Datenstück diese Position um die gelesenen oder geschriebenen Bytes vorverlegt. Es kann jedoch notwendig sein, ein bestimmtes Datenstück zu lesen, ohne die aktuelle Suchposition zu ändern, und in diesem Fall würden Sie diese Funktion verwenden. Sie geben der Funktion einfach eine buffer id und dann eine Offset-Position (vom Start von buffer aus) innerhalb von buffer an, von der aus gelesen werden soll, sowie den Datentyp, den Sie lesen möchten.

HINWEIS: Die Verwendung des falschen Datentyps für die zu lesenden Daten führt zu fehlerhaften Werten!

Wenn die Funktion erfolgreich ist, wird ein Wert (Real, String, Boolean, etc...) je nach Datentyp zurückgegeben, während bei einem Fehler einfach 0 zurückgegeben wird.

Syntax:

buffer_peek(buffer, offset, type);

Argument Typ Beschreibung
buffer Puffer-ID The index of the buffer to use.
offset Real The offset position (in bytes) within the buffer to read the given data from.
type Puffer Datentyp Konstante The type of data that is to be read from the buffer (see the list of constants here).

Rückgabe:

Real, Boolean, String or 0 (if it fails)

Beispiel:

var red = buffer_peek(buff, 1, buffer_u8);
var green = buffer_peek(buff, 2, buffer_u8);
var blue = buffer_peek(buff, 3, buffer_u8);
image_blend = make_colour_rgb(red, green, blue);

Die obige code holt sich drei Werte von drei verschiedenen Positionen innerhalb der buffer, die in der Variablen "buff" indiziert sind, und verwendet diese Werte dann, um die Bildüberblendung der Instanz einzustellen.
