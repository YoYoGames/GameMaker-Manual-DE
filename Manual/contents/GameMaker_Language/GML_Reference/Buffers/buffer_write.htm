buffer_write

Diese Funktion kann verwendet werden, um Daten in eine zuvor erstellte buffer zu schreiben. Die Daten, die Sie schreiben, müssen mit dem "Typ" argument dieser Funktion übereinstimmen, d.h. Sie können z.B. nicht versuchen, eine Zeichenkette als vorzeichenlose 16-Bit-Ganzzahl zu schreiben. Die folgenden Konstanten können verwendet werden, um den Datentyp zu definieren:

Puffer Datentyp Konstante
Konstante Beschreibung
buffer_u8 Eine 8-Bit-Ganzzahl ohne Vorzeichen. Dies ist ein positiver Wert von 0 bis 255.
buffer_s8 Eine 8-Bit-Ganzzahl mit Vorzeichen. Dies kann ein positiver oder negativer Wert von -128 bis 127 sein (0 wird als positiv eingestuft).
buffer_u16 Eine 16-Bit-Ganzzahl ohne Vorzeichen. Dies ist ein positiver Wert von 0 - 65.535.
buffer_s16 Eine ganze Zahl mit Vorzeichen und 16 Bit. Dies kann ein positiver oder negativer Wert zwischen -32.768 und 32.767 sein (0 wird als positiv eingestuft).
buffer_u32 Eine 32-Bit-Ganzzahl ohne Vorzeichen. Dies ist ein positiver Wert von 0 bis 4.294.967.295.
buffer_s32 Eine ganze Zahl mit Vorzeichen und 32 Bit. Dies kann ein positiver oder negativer Wert zwischen -2.147.483.648 und 2.147.483.647 sein (0 wird als positiv eingestuft).
buffer_u64 Eine 64-Bit-Ganzzahl ohne Vorzeichen.
buffer_f16 Ein 16-Bit-Float. Dies kann ein positiver oder negativer Wert innerhalb des gleichen Bereichs wie eine 16-Bit-Ganzzahl mit Vorzeichen sein.
buffer_f32 Eine 32-Bit-Fließkommazahl. Dies kann ein positiver oder negativer Wert innerhalb des gleichen Bereichs wie eine 32-Bit-Ganzzahl mit Vorzeichen sein.
buffer_f64 Ein 64-Bit-Float.
buffer_bool Ein boolescher Wert. Kann nur entweder 1 oder 0 sein (true oder false)
buffer_string Eine Zeichenkette beliebiger Größe, die mit einem Nullzeichen abgeschlossen wird.
buffer_text Eine Zeichenkette beliebiger Größe, ohne das abschließende Null-Zeichen.

Die Funktion gibt 0 zurück, wenn sie erfolgreich war, oder -1, wenn sie fehlgeschlagen ist.

Syntax:

buffer_write(buffer, type, value)

Argument Typ Beschreibung
buffer Puffer-ID The index of the buffer to write to.
type Real The type of data that is to be written to the buffer (see the list of constants above).
value Real The data to write.

Rückgabe:

Real (0 if success, or -1 if it fails)

Beispiel:

buffer_seek(buff, buffer_seek_start, 0);
buffer_write(buff, buffer_s16, 0);
buffer_write(buff, buffer_s16, x);
buffer_write(buff, buffer_s16, y);

Die obige code findet den Anfang der buffer mit der in der Variablen "buff" gespeicherten ID und schreibt eine Reihe von vorzeichenbehafteten 16-Bit-Ganzzahlwerten hinein.
