puffer_speichern_ext

Mit dieser Funktion können Sie einen Teil des Inhalts von buffer in einer Datei speichern, die Sie dann mit der Funktion buffer_load() Funktion gelesen werden kann. Der "Offset" definiert die Startposition innerhalb des Puffers für die Speicherung (in Bytes), und die "Größe" ist die Größe des zu speichernden Bereichs von buffer ab diesem Offset (ebenfalls in Bytes).

Syntax:

buffer_save_ext(buffer, filename, offset, size);

ArgumentTyp Beschreibung
buffer The index of the buffer to save.
filename The name of the file to save as.
offset The offset within the buffer to save from (in bytes).
size The size of the buffer area to save (in bytes).

Rückgabe:

Beispiel:

buffer_save_ext(buff, "Player_Save.sav", 0, 16384);

Speichert einen Teil des aktuellen Inhalts von buffer mit der in der Variablen "buff" gespeicherten ID in eine Datei.
