dampf_datei_schreibe_datei

Mit dieser Funktion können Sie den Inhalt einer lokal gespeicherten Datei in eine Datei kopieren, die mit der Steam Cloud synchronisiert ist. Die lokale Datei muss vorhanden sein, bevor diese Funktion verwendet werden kann, und sie gibt einen Wert von 0 zurück, wenn sie aus irgendeinem Grund fehlschlägt, und einen Wert größer als 0, wenn sie erfolgreich ist.

Syntax:

steam_file_write_file(steam_filename, local_filename);

Argument Beschreibung
steam_filename Die zu kopierende Steam Cloud-Datei.
local_filename Die lokale Datei, aus der kopiert werden soll.

Rückgabe:

real

Beispiel:

steam_file_write_file("rm_koala.png", "Koala2.png");

Die obige code wird den Inhalt der Datei "Koala2.png" in die Steam Cloud-Datei "rm_koala.png" kopieren.
