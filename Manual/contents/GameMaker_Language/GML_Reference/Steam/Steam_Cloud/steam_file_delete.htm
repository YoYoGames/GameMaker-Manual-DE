dampf_datei_loeschen

Diese Funktion löscht die angegebene Datei aus der Steam Cloud. Die Funktion gibt den Wert 0 zurück, wenn sie aus irgendeinem Grund fehlschlägt, und einen Wert größer als 0, wenn sie erfolgreich ist.

Syntax:

steam_file_delete(filename);

Argument Beschreibung
filename Der Name der zu löschenden Datei.

Rückgabe:

Real

Beispiel:

if steam_file_exists("Save.txt")
{
    steam_file_delete("Save.txt");
}

Die obige code prüft, ob eine Datei existiert, und wenn ja, löscht sie die Datei aus der Steam Cloud.
