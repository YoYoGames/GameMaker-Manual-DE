steam_is_cloud_enabled_for_app

Mit dieser Funktion können Sie überprüfen, ob der Steam Cloud-Dienst für Ihr Spiel aktiviert ist. Wenn dies der Fall ist, wird true zurückgegeben, andernfalls false. Beachten Sie, dass dies nicht automatisch bedeutet, dass Sie die Cloud-Funktionen nutzen können, da der Benutzer die Cloud-Synchronisierung auf switch in seinem Steam Client deaktivieren kann. Sie können dies mit der Funktion steam_is_cloud_enabled_for_account()Aber auch wenn sie für den Benutzer deaktiviert (und für das Spiel aktiviert) ist, funktionieren die Funktionen, um Daten aus einer lokalen Kopie aller Dateien zu speichern und abzurufen, sie werden nur nicht am Ende des Spiels in die Cloud hochgeladen und beim Spielstart nicht synchronisiert.

Syntax:

steam_is_cloud_enabled_for_app();

Rückgabe:

Boolean

Beispiel:

if steam_is_cloud_enabled_for_app()
{
    quota = steam_get_quota_total();
}

Die obige code prüft, ob die steam Cloud für das Spiel aktiviert ist, und wenn ja, ermittelt sie die Größe des Speicherkontingents und speichert sie in einer Variablen.
