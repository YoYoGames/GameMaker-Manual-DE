steam_get_quota_free

Mit dieser Funktion können Sie herausfinden, wie viel freier Speicherplatz für den Benutzer des Steam Cloud-Kontingents übrig ist. Der zurückgegebene Wert ist in Bytes.

Syntax:

steam_get_quota_free();

Rückgabe:

Real

Beispiel:

if steam_is_cloud_enabled_for_app()
{
    quota = steam_get_quota_free();
}

Die obige code prüft, ob die steam Cloud für das Spiel aktiviert ist und wenn ja, ermittelt sie die Größe des freien Speicherplatzes und speichert sie in einer Variablen.
