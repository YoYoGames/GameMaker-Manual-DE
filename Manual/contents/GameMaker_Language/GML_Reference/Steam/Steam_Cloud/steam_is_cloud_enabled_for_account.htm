steam_is_cloud_enabled_for_account

Mit dieser Funktion können Sie überprüfen, ob der Steam Cloud-Dienst vom Benutzer in seinen Steam Client-Einstellungen aktiviert wurde. Sie gibt true zurück, wenn dies der Fall ist, andernfalls false. Beachten Sie, dass dies nicht automatisch bedeutet, dass Sie Daten in der Cloud speichern können, da diese auch für Ihr Spiel aktiviert sein muss (Sie können dies mit der Funktion steam_is_cloud_enabled_for_app()). Wenn die Steam Cloud für Ihr Spiel aktiviert ist, der Benutzer sie aber lokal ausgeschaltet hat, können Sie die Cloud-Funktionen trotzdem verwenden, um Daten in einer lokalen Kopie aller Dateien zu speichern und abzurufen, sie werden dann nur nicht bei Spielende in die Cloud hochgeladen und beim Spielstart nicht synchronisiert.

Syntax:

steam_is_cloud_enabled_for_account();

Rückgabe:

Boolean

Beispiel:

if steam_is_cloud_enabled_for_account
{
    steam_file_share("Save.txt");
}

Die obige code prüft, ob der Benutzer die Steam Cloud aktiviert hat, und wenn sie true zurückgibt, wird die angegebene Datei synchronisiert.
