Dampfwolke

Die Steam Cloud bietet ein einfaches und transparentes Remote-Dateispeichersystem für Ihr Spiel. Alle Dateien, die mit den GameMaker Studio 2 Cloud-Funktionen (siehe unten) auf die Festplatte geschrieben werden, werden nach Beendigung des Spiels auf Steam servers repliziert. Wenn der Benutzer dann den Computer wechselt, werden die Dateien auf den neuen Computer heruntergeladen, bevor das Spiel gestartet wird. Das bedeutet, dass das Spiel dann auf die Dateien zugreifen kann, indem es sie mit den entsprechenden Steam Funktionen liest. Der Steam Client sorgt dafür, dass die Dateien auf allen Computern, auf die der Benutzer zugreift, synchronisiert bleiben.

HINWEIS Standardmäßig ist die Cloud für ein Spiel auf Steamworks nicht aktiviert. Sie muss zuvor auf der Registerkarte "Cloud" in der Steamworks-Spielverwaltung aktiviert werden, wo Sie die Byte- und Dateiquote festlegen sollten. Wenn du das nächste Mal die Steamworks-Konfiguration deines Spiels veröffentlichst, ist der Cloud-Speicher einsatzbereit.

Die folgenden Funktionen können verwendet werden, um von GameMaker Studio 2 aus auf die Steam Cloud zuzugreifen

  • steam_is_cloud_enabled_for_app
  • steam_is_cloud_enabled_for_account
  • steam_get_quota_total
  • steam_get_quota_free
  • steam_file_exists
  • steam_file_size
  • steam_file_persisted
  • steam_file_write
  • steam_file_write_file
  • steam_file_read
  • steam_file_share
  • steam_file_delete
