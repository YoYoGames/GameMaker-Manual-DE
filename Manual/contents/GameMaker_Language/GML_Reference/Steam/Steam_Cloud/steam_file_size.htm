dampf_datei_grösse

Mit dieser Funktion können Sie die Größe einer in der Steam Cloud gespeicherten Datei überprüfen. Die zurückgegebene reelle Zahl ist die Größe der Datei in Bytes.

Syntax:

steam_file_size(filename);

Argument Beschreibung
filename Der Name der Datei, deren Größe geprüft werden soll.

Rückgabe:

Real

Beispiel:

file_bytes = steam_file_size("Save.txt");

Die obige code speichert die Größe einer Datei aus der Steam Cloud in der Variablen "file_bytes".
