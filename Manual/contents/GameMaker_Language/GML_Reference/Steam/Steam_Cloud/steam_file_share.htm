steam_file_share

Mit dieser Funktion können Sie Ihr Spiel zwingen, die angegebene Datei mit der Steam Cloud zu synchronisieren. Dies ist normalerweise nicht notwendig, da das Spiel am Ende der Spielersitzung automatisch synchronisiert wird, und wird auch nicht von Steam empfohlen, kann aber nützlich sein, um sicherzustellen, dass sensible Informationen sofort synchronisiert werden. Die Funktion gibt einen Wert von 0 zurück, wenn sie aus irgendeinem Grund fehlschlägt, und einen Wert größer als 0, wenn sie erfolgreich ist.

Syntax:

steam_file_share(filename);

Argument Beschreibung
filename Der Name der Datei, die synchronisiert werden soll.

Rückgabe:

Real

Beispiel:

if !steam_file_persisted("Save.txt")
{
    steam_file_share("Save.txt");
}

Die obige code prüft, ob eine Datei in der Steam Cloud gespeichert wurde, und wenn nicht, wird sie synchronisiert.
