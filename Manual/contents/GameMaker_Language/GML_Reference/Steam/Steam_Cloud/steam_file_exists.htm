steam_file_exists

Mit dieser Funktion können Sie prüfen, ob eine Datei aus der Steam Cloud existiert oder nicht, mit einem Rückgabewert von true, wenn sie existiert, oder false, wenn sie nicht existiert.

Syntax:

steam_file_exists(filename);

Argument Beschreibung
filename Der Name der Datei, nach der gesucht werden soll.

Rückgabe:

Boolean

Beispiel:

if steam_file_exists("Save.txt")
{
    save_str = steam_file_read("Save.txt");
}

Die obige code prüft, ob eine Datei auf der Steam Cloud existiert, und wenn ja, öffnet sie diese und liest ihren Inhalt in die Variable "save_str".
