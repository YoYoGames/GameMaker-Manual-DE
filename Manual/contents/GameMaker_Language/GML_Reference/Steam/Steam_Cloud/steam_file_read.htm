dampf_datei_lesen

Diese Funktion liest den Inhalt der angegebenen Datei in eine Zeichenkette ein, die später in Ihrem Spiel analysiert werden kann.

Syntax:

steam_file_read(filename);

Argument Beschreibung
filename Der Name der Datei, aus der gelesen werden soll.

Rückgabe:

String

Beispiel:

if steam_file_exists("Save.txt")
{
    save_str = steam_file_read("Save.txt");
}

Die obige code prüft, ob eine Datei auf der Steam Cloud existiert, und wenn ja, öffnet sie diese und liest ihren Inhalt in die Variable "save_str".
