steam_file_write

Sie können diese Funktion verwenden, um Daten in eine Datei zu schreiben, die dann mit der Steam Cloud synchronisiert wird, wenn der Benutzer das Spiel beendet. Daten, die bereits in der Datei gespeichert sind, mit dem neuen Datenstring. Die Funktion gibt einen Wert von 0 zurück, wenn sie aus irgendeinem Grund fehlschlägt, und einen Wert größer als 0, wenn sie erfolgreich ist.

Syntax:

steam_file_write(filename, data, size);

Argument Beschreibung
filename Der Name der Datei, in die geschrieben werden soll.
data Die zu schreibenden Daten (eine Zeichenkette).
size die Größe der zu schreibenden Daten.

Rückgabe:

Real

Beispiel:

var fname="SaveData.txt";
var data = string(global.Level) + "|" + string(global.Points) + "|" + string(global.HP);
var len = string_length(data);
steam_file_write_file(fname, data, len);

Die obige code bereitet eine Reihe lokaler Variablen vor und verwendet sie dann, um in eine Datei zu schreiben (oder zu erstellen), die dann mit der Steam Cloud synchronisiert wird.
