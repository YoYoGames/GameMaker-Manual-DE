steam_file_persisted

Mit dieser Funktion können Sie prüfen, ob die angegebene Datei mit der Steam Cloud synchronisiert wurde. Ein Rückgabewert von true bedeutet, dass sie synchronisiert ist, während false bedeutet, dass sie nicht synchronisiert ist.

Syntax:

steam_file_persisted(filename);

Argument Beschreibung
filename Der Name der zu prüfenden Datei.

Rückgabe:

Boolean

Beispiel:

if !steam_file_persisted("Save.txt")
{
    steam_file_share("Save.txt");
}

Die obige code prüft, ob eine Datei in der Steam Cloud gespeichert wurde, und wenn nicht, wird sie synchronisiert.
