leistung_dampf_setzen

Mit dieser Funktion können Sie der Steam API mitteilen, dass ein Erfolg für den Spieler vergeben ("gesetzt") werden soll. Diese Errungenschaften sollten auf der Kontenseite des Steamworks-Kontrollpanels für Ihr Spiel definiert worden sein, und die Zeichenkette, die an die Funktion übergeben wird, sollte mit der übereinstimmen, die als API-Name im Kontrollpanel verwendet wird. Das Steam Game Overlay zeigt dem Benutzer ein Benachrichtigungsfeld an, das ihn über die erhaltene Errungenschaft informiert, es sei denn, die Errungenschaft wurde bereits verliehen; in diesem Fall geschieht nichts.

Syntax:

steam_set_achievement(ach_name);

Argument Beschreibung
ach_name Der Name der zu setzenden Leistung (String).

Rückgabe:

N/A

Beispiel:

if hp <= 0
{
    global.Deaths += 1;
    if global.Deaths == 10
    {
        if !steam_get_achievement("ach_Player_Dies_Ten_Times") steam_set_achievement("ach_Player_Dies_Ten_Times");
    }
}

Die obige code belohnt den Spieler mit einer Errungenschaft, wenn die globale Variable "Tode" gleich 10 ist und wenn die Errungenschaft nicht bereits vergeben wurde.
