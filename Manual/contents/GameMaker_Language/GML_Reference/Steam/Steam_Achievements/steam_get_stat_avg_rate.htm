steam_get_stat_avg_rate

Mit dieser Funktion können Sie den Wert einer bestimmten Durchschnittsstatistik abrufen. Die Statistik sollte zuvor auf der Kontenseite des Steamworks-Kontrollpanels für dein Spiel definiert worden sein, und die Zeichenkette, die an die Funktion übergeben wird, sollte mit der als API-Name im Kontrollpanel verwendet wird.

Syntax:

steam_get_stat_avg_rate(stat_name);

Argument Beschreibung
stat_name Der Name der abzurufenden Statistik (eine Zeichenkette).

Rückgabe:

Real

Beispiel:

var avg = steam_get_stat_avg_rate("PointsPerHour");
draw_text(8, 8, "PPH = " + string(avg);

Die obige code ermittelt den aktuellen Wert für die Durchschnittsstatistik "PointsPerHour" und zeichnet ihn auf den Bildschirm.
