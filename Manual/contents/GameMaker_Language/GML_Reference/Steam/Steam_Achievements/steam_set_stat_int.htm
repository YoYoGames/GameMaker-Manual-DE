steam_set_stat_int

Mit dieser Funktion können Sie eine bestimmte Statistik auf einen neuen, vorzeichenbehafteten Ganzzahlwert setzen. Die Statistik sollte zuvor auf der Kontenseite des Steamworks-Kontrollpanels für Ihr Spiel definiert worden sein, und die Zeichenkette, die an die Funktion übergeben wird, sollte mit der übereinstimmen, die als API-Name im Kontrollpanel verwendet wird. Beispiele für die Verwendung dieser Funktion sind die Nachverfolgung, wie oft der Spieler stirbt, oder die Nachverfolgung des Fortschritts bei der Erreichung eines Ziels.

Syntax:

steam_set_stat_int(stat_name, value);

Argument Beschreibung
stat_name Der Name der zu setzenden Statistik (eine Zeichenkette).
value Der Wert (eine 32-Bit-Ganzzahl), auf den die Statistik gesetzt werden soll.

Rückgabe:

N/A

Beispiel:

xp += 100;
steam_set_stat_int("Total_XP", steam_get_stat_int("Total_XP") + 100);
if steam_get_stat_int("Total_XP") > 1000
{
    if !steam_get_achievement("Ach_1000XP") steam_set_achievement("Ach_1000XP");
}

Die obige code setzt eine Statistik und prüft dann den Endwert, um zu entscheiden, ob eine Leistung vergeben wird oder nicht.
