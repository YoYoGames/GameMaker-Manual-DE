steam_get_achievement

Mit dieser Funktion können Sie die Steam API überprüfen, um zu sehen, ob eine bestimmte Leistung vergeben wurde. Die Errungenschaft sollte zuvor auf der Kontenseite des Steamworks-Kontrollpanels für Ihr Spiel definiert worden sein, und die Zeichenkette, die an die Funktion übergeben wird, sollte mit der übereinstimmen, die als API-Name im Kontrollpanel verwendet wird.

Syntax:

steam_get_achievement(ach_name);

Argument Beschreibung
ach_name Der Name der zu erhaltenden Leistung (String).

Rückgabe:

Boolean

Beispiel:

if hp <= 0
{
    global.Deaths += 1;
    if global.Deaths == 10
    {
        if !steam_get_achievement("ach_Player_Dies_Ten_Times") steam_set_achievement("ach_Player_Dies_Ten_Times");
    }
}

Die obige code belohnt den Spieler mit einer Errungenschaft, wenn die globale Variable "Tode" gleich 10 ist und wenn die Errungenschaft noch nicht vergeben wurde.
