Errungenschaften und Statistiken

Die Steam Stats and Achievements API bietet eine einfache Möglichkeit für Ihr Spiel, eine dauerhafte, roamingfähige Leistungs- und Statistikverfolgung für Ihre Benutzer bereitzustellen. Die Daten des Nutzers werden mit seinem Steam Konto verknüpft, und die Leistungen und Statistiken jedes Nutzers können formatiert und in seinem Steam Community-Profil angezeigt werden.

Achievements sind nicht nur wertvolle Belohnungen für die Spieler Ihrer Spiele, sondern auch nützlich, um Teamwork und Spielerinteraktion zu fördern und zu belohnen, den Spielzielen eine zusätzliche Dimension zu verleihen und die Benutzer dafür zu belohnen, dass sie mehr Zeit im Spiel verbringen, und daher wird empfohlen, dass Ihr Spiel einige davon enthält. Sie lassen sich leicht über das Steam Dashboard einrichten, erfordern aber, dass Sie spezielle Icons für sie erstellen.

Statistiken erfassen feinkörnige Informationen wie Spielzeit, Anzahl der verwendeten Power-Ups usw. Sie können sie auch einfach zur Verfolgung interner Spieldaten verwenden - so können Sie zum Beispiel eine Leistung auf der Grundlage von Spielstatistiken, die vom Benutzer über mehrere Computer hinweg gesammelt wurden, vergeben. Oder Sie können interessante Spieldaten für die Anzeige auf der Community-Seite des Benutzers Steam aufzeichnen, wo die Benutzer ihre eigenen Statistiken mit denen ihrer Freunde vergleichen können.

Es gibt drei Arten von Statistiken - Ganzzahlen, Fließkommazahlen und Durchschnittswerte - und GameMaker Studio 2 ermöglicht es dir, jede dieser Statistiken von deinem Spiel aus zu ändern, aber sie sollten zuvor über das Steamworks-Kontrollfeld für dein Spiel initialisiert worden sein.

HINWEIS Sie müssen warten, bis steam_stats_ready() true zurückgegeben hat, bevor Sie versuchen, Statistiken und Erfolge zu lesen oder zu schreiben.

Weitere Einzelheiten und Anleitungen zur Verwendung von Statistiken und Leistungen finden Sie auf den folgenden Seiten, auf denen die Funktionen beschrieben werden:

  • steam_set_achievement
  • steam_get_achievement
  • steam_clear_achievement
  • steam_set_stat_int
  • steam_set_stat_float
  • steam_set_stat_avg_rate
  • steam_get_stat_int
  • steam_get_stat_float
  • steam_get_stat_avg_rate
  • steam_reset_all_stats
  • steam_reset_all_stats_achievements

Wenn der Benutzer im Offline-Modus ist, speichert Steam eine lokale cache der Statistiken und Leistungsdaten, so dass die APIs wie gewohnt verwendet werden können. Alle Statistiken, die nicht übertragen werden können, werden für das nächste Mal, wenn der Benutzer online ist, gespeichert. Für den Fall, dass Änderungen auf mehreren Rechnern vorgenommen wurden, fasst Steam automatisch die Leistungen zusammen und wählt den Satz von Statistiken aus, der die meisten Fortschritte gemacht hat. Da Steam eine lokale cache der Statistikdaten speichert, ist es nicht notwendig, dass das Spiel auch eine lokale cache der Daten auf der Festplatte speichert, zumal solche caches oft in Konflikt geraten und es dann für den Benutzer so aussieht, als ob sein Fortschritt rückgängig gemacht worden wäre, was eine frustrierende Erfahrung ist.
