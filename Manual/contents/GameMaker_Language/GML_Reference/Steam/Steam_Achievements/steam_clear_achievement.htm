steam_clear_achievement

Mit dieser Funktion können Sie die Steam API anweisen, eine bestimmte Leistung zu löschen (zurückzusetzen). Die Errungenschaft sollte zuvor auf der Kontenseite des Steamworks-Kontrollpanels für Ihr Spiel definiert worden sein, und die Zeichenkette, die an die Funktion übergeben wird, sollte mit der übereinstimmen, die als API-Name im Kontrollpanel verwendet wird.

Syntax:

steam_clear_achievement(ach_name);

Argument Beschreibung
ach_name Der Name der zu löschenden Leistung (String).

Rückgabe:

N/A

Beispiel:

if mouse_check_button_pressed(mb_left)
{
    steam_clear_achievement("Ach_Game_Win");
    steam_clear_achievement("Ach_Died_10_Times");
    steam_clear_achievement("Ach_Killed_100_Enemies");
    steam_clear_achievement("Ach_Beat_Boss_Level_1");
}

Die obige code wird die Erfolge des Spiels zurücksetzen, wenn der Benutzer die linke Maustaste drückt.
