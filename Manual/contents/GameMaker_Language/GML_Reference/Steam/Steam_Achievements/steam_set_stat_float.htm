steam_set_stat_float

Mit dieser Funktion können Sie eine bestimmte Statistik auf einen neuen Fließkommawert setzen. Die Statistik sollte zuvor auf der Kontenseite des Steamworks-Kontrollpanels für dein Spiel definiert worden sein und die Zeichenkette, die an die Funktion übergeben wird, sollte mit dem API-Namen im Kontrollpanel übereinstimmen. Sie können diese Funktion beispielsweise verwenden, um zu verfolgen, wie weit Ihr Spieler gereist ist oder wie viel Prozent des Spiels abgeschlossen sind.

Syntax:

steam_set_stat_float(stat_name, value);

Argument Beschreibung
stat_name Der Name der zu setzenden Statistik (eine Zeichenkette).
value Der Wert (eine 32-Bit-Fließkommazahl), auf den der Wert gesetzt werden soll.

Rückgabe:

N/A

Beispiel:

var dist_pc = (dist / dist_max) * 100;
steam_set_stat_float("Travelled", dist_pc);

Die obige code berechnet einen Prozentsatz auf der Grundlage der Variable "dist" für die zurückgelegte Entfernung und der maximalen Entfernung "dist_max", die Sie zurücklegen können, und setzt dann die Statistik "Zurückgelegt" auf den neuen Wert.
