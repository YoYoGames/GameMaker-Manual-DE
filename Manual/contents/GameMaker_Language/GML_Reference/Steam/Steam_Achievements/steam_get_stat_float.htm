steam_get_stat_float

Mit dieser Funktion können Sie den Wert einer bestimmten Fließkommastatistik abrufen. Die Statistik sollte zuvor auf der Kontenseite des Steamworks-Kontrollpanels für Ihr Spiel definiert worden sein und die Zeichenkette, die an die Funktion übergeben wird, sollte mit dem API-Namen im Kontrollpanel übereinstimmen.

Syntax:

steam_get_stat_float(stat_name);

Argument Beschreibung
stat_name Der Name der abzurufenden Statistik (eine Zeichenkette).

Rückgabe:

Real

Beispiel:

var dist_pc = (dist / dist_max) * 100;
if steam_get_stat_float("Travelled") < dist_pc
{
    steam_set_stat_int("Travelled", dist_pc);
}

Die obige code berechnet einen Prozentsatz auf der Grundlage der Variablen "dist" für die zurückgelegte Entfernung und der maximalen Entfernung, die Sie zurücklegen können, "dist_max". Anschließend wird der aktuelle Wert für die Statistik "Zurückgelegte Strecke" abgefragt, und wenn er kleiner ist als der berechnete Wert, wird die Statistik erneut gesetzt.
