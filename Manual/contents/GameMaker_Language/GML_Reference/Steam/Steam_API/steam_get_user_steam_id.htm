steam_get_user_steam_id

Sie können diese Funktion verwenden, um die eindeutige Steam Benutzerkennung des Benutzers zurückzugeben, der gerade im Steam Client angemeldet ist. Dies ist ein eindeutiger 64-Bit-Wert, der verwendet werden kann, um einzelne Benutzer zu identifizieren und nicht ihren Benutzernamen auf dem Bildschirm (dieser kann mit der Funktion steam_get_persona_name()). Da es sich hierbei um einen int64-Wert handelt, müssen Sie ihn möglicherweise als solchen umwandeln, wenn Sie ihn in nachfolgenden Funktionen mit der int64() Funktion.

Syntax:

steam_get_user_steam_id();

Rückgabe:

Integer (int64)

Beispiel:

if steam_initialised()
{
    global.u_id = steam_get_user_steam_id();
}

Die obige code setzt eine globale Variable auf die eindeutige ID des aktuellen Benutzers Steam, wenn die Steam Client-API korrekt initialisiert ist.
