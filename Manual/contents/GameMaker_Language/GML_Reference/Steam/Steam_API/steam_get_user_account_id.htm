steam_get_user_account_id

Diese Funktion dient dazu, die eindeutige Benutzer-ID abzurufen, die Steam jedem Benutzer zuweist und die für die Nutzung einiger der Funktionen für benutzergenerierte Inhalte erforderlich ist.

Syntax:

steam_get_user_account_id();

Rückgabe:

Real

Beispiel:

global.user_id = steam_get_user_account_id();

Die obige code ruft die eindeutige Benutzer-ID für die Person ab, die das Spiel besitzt, und speichert sie in einer globalen Variablen.
