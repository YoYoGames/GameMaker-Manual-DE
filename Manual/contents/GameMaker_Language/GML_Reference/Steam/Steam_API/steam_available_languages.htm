steam_available_languages

Diese Funktion kann verwendet werden, um eine Liste aller verfügbaren Sprachen für Steam abzurufen. Der zurückgegebene Wert ist einfach eine durch Kommata getrennte Liste von Sprachen.

Syntax:

steam_available_languages();

Rückgabe:

String

Beispiel:

language = steam_available_languages();

Der obige Befehl ruft die verfügbaren Sprachen für Steam als Zeichenkette ab und speichert sie in einer Variablen.
