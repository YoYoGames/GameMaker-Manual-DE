steam_current_game_language

Diese Funktion wird verwendet, um die aktuelle Sprache, die Steam verwendet, abzurufen (als String), z. B. "english".

Syntax:

steam_current_game_language();

Rückgabe:

String

Beispiel:

language = steam_current_game_language();

Die obige code liefert die für das aktuelle Spiel verwendete Sprache.
