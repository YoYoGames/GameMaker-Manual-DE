Allgemein Steam API

Die folgenden Funktionen dienen alle dazu, die Verfügbarkeit bestimmter Aspekte des Steam -Clients oder der server -API zu überprüfen und geben entweder true oder false zurück, je nachdem, ob der server -Client beschäftigt ist oder der Benutzer eingeloggt ist usw. Das bedeutet, dass diese Funktionen vor allen anderen Steam API-Funktionsaufrufen verwendet werden sollten, um sicherzustellen, dass die Client/Server-Einrichtung korrekt ist und mit Ihrem Spiel kommuniziert:

  • steam_initialised
  • steam_stats_ready
  • steam_get_app_id
  • steam_get_user_account_id
  • steam_get_user_steam_id
  • steam_get_persona_name
  • steam_get_user_persona_name
  • steam_is_user_logged_on
  • steam_current_game_language
  • steam_available_languages
