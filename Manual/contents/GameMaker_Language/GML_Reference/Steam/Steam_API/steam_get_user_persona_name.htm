steam_get_user_persona_name

Diese Funktion kann verwendet werden, um den Benutzernamen (Bildschirmname) für eine bestimmte Benutzer-ID abzurufen. Sie geben die Benutzer-ID an, und die Funktion gibt einen eindeutigen Async-ID-Wert zurück, der verwendet werden kann, um die Details im Async-Ereignis von Steam korrekt zu überprüfen. Dieses Ereignis enthält die async_load DS-Map, die mit den folgenden Schlüssel/Wert-Paaren befüllt wird:

  • "id" - Die asynchrone ID, die von der Funktion zurückgegeben wird, die das Ereignis ausgelöst hat
  • "event_type" - Die Zeichenfolge "user_persona_name"
  • "steamid"" - Der Steam ID-Wert, der verwendet wird, um den Benutzernamen zu erhalten.
  • "persona_name" - Der Benutzername für die angegebene Steam ID

Syntax:

steam_get_user_persona_name(steamID);

Argument Beschreibung
steamID Die eindeutige Steam ID für einen Benutzer.

Rückgabe:

Real

Beispiel:

request = steam_get_user_persona_name(global.UGC_UserID);

Die obige code fragt den Benutzernamen der in der globalen Variablen "UGC_UserID" gespeicherten Benutzer-ID ab und speichert den zurückgegebenen Wert in einer Variablen, die im Async-Ereignis geparst wird.
