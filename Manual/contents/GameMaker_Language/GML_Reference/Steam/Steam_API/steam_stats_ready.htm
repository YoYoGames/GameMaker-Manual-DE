steam_stats_ready

Bei Verwendung der Steam API kann diese Funktion aufgerufen werden, um zu überprüfen, ob die Steam Client-API die Statistiken für Ihr Spiel korrekt initialisiert hat.

Syntax:

steam_stats_ready();

Rückgabe:

Boolean

Beispiel:

global.steam_api = false;
if steam_initialised()
{
    if steam_stats_ready() && steam_is_overlay_enabled()
    {
        global.steam_api = true;
    }
}

Die obige code setzt eine globale Variable auf true, wenn die Steam Client-API zusammen mit der Steam Statistik- und Overlay-Funktionalität korrekt initialisiert wurde, andernfalls setzt sie die Variable auf false.
