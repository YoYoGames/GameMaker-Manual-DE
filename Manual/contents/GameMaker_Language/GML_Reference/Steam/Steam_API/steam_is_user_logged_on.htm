steam_is_user_logged_on

Diese Funktion gibt true zurück, wenn der Steam -Client derzeit eine aktive Verbindung zum Steam servers hat. Wenn sie false zurückgibt, bedeutet dies, dass keine aktive Verbindung besteht, weil entweder ein Netzwerkproblem auf dem lokalen Rechner vorliegt oder die Steam server nicht erreichbar oder besetzt ist.

Syntax:

steam_is_user_logged_on();

Rückgabe:

String

Beispiel:

if steam_is_user_logged_on()
{
    global.user_id = steam_get_user_account_id();
}

Die obige code prüft, ob der Benutzer auf Steam server angemeldet ist, und speichert die Benutzer-ID in einer globalen Variablen.
