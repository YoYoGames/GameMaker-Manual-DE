steam_activate_overlay

Das Steam Overlay ist ein Teil der Steam Benutzeroberfläche, die über fast jedes Spiel, das über Steam gestartet wird, aktiviert werden kann. Es ermöglicht dem Benutzer den Zugriff auf seine Freundesliste, den Webbrowser, den Chat und den Kauf von DLC im Spiel. Die Standardtaste für den Zugriff auf das Overlay während eines Spiels ist SHIFT+TAB, aber Sie können mit dieser Funktion auch eine beliebige Seite des Overlays aufrufen. Sie nimmt eine der sechs unten aufgeführten Konstanten an:

Konstante Beschreibung
ov_friends Die Freundeseite für den aktuellen Benutzer
ov_community Die Gemeinschaftsseite für Ihr Spiel
ov_players Die Seite mit den anderen Spielern, die dasselbe Spiel spielen oder mit denen Sie kürzlich gespielt haben
ov_settings Die Steam Client-Overlay-Einstellungen
ov_gamegroup Öffnet den Steam Community-Webbrowser zur offiziellen Spielgruppe für dieses Spiel
ov_achievements Die Seite mit den Erfolgen für Ihr Spiel

Syntax:

steam_activate_overlay(overlayindex);

Argument Beschreibung
overlayindex Der Seitenindex der Steam API UI, der angezeigt werden soll (siehe unten aufgeführte Konstanten).

Rückgabe:

N/A

Beispiel:

var key = keyboard_lastkey;
switch (key)
{
    case vk_f1: steam_activate_overlay(ov_friends); break;
    case vk_f2: steam_activate_overlay(ov_community); break;
    case vk_f3: steam_activate_overlay(ov_players); break;
    case vk_f4: steam_activate_overlay(ov_settings); break;
    case vk_f5: steam_activate_overlay(ov_gamegroup); break;
    case vk_f6: steam_activate_overlay(ov_achievements); break;
}

Das obige code fragt die zuletzt gedrückte Taste ab, und wenn es eine der Funktionstasten von 1 bis 6 ist, wird die entsprechende Seite des Overlays Steam geöffnet.
