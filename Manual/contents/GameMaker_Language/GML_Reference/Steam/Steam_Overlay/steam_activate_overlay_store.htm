steam_activate_overlay_store

Mit dieser Funktion können Sie das Overlay Steam auf der Shop-Seite für ein Spiel öffnen, damit die Nutzer z. B. DLC kaufen oder herunterladen können. Sie müssen die eindeutige App-ID für das Spiel oder den DLC angeben, die Sie bei der Einrichtung über das Steam Dashboard erhalten.

Syntax:

steam_activate_overlay_store(app_id);

Argument Beschreibung
app_id Die eindeutige App-ID für Ihr Spiel.

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(vk_f1)
{
    steam_activate_overlay_store(global.DLC_id);
}

Die obige code fragt die Tastatur nach der F1-Taste ab, und wenn dies der Fall ist, wird das Overlay Steam auf der Seite für den Spielinhalt geöffnet, wobei die in der globalen Variablen gespeicherte App-ID verwendet wird.
