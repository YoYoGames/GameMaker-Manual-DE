steam_activate_overlay_user

Mit dieser Funktion wird das Overlay von Steam zu einem der ausgewählten Dialoge geöffnet, die sich auf die angegebene Benutzer-ID beziehen. Die möglichen Dialogstrings sind:

  • "steamid" - öffnet den Webbrowser der Steam Community auf der Seite des Benutzers oder der Gruppe
  • "Chat" - öffnet ein Chat-Fenster für den angegebenen Benutzer

Beachten Sie, dass Steam IDs große Zahlen sein können und Sie daher Ihren ID-Wert als int64() umwandeln, bevor Sie ihn an die Funktion übergeben.

Syntax:

steam_activate_overlay_user(dialog_name, steamid);

Argument Beschreibung
dialog_name Der Dialog, in dem das Overlay geöffnet werden soll (siehe unten).
steamid Die zu verwendende Steam Benutzer-ID oder Gruppen-ID.

Rückgabe:

N/A

Beispiel:

var key = keyboard_lastkey;
switch (key)
{
    case vk_f1: steam_activate_overlay_user("steamid", global.GameGroupID); break;
    case vk_f2: steam_activate_overlay_user("chat", global.FriendID); break;
}

Das obige code fragt die zuletzt gedrückte Taste ab, und wenn es die Funktionstaste 1 oder die Funktionstaste 2 ist, öffnet es das Overlay Steam, um entweder die in der globalen Variable "GamegroupID" gespeicherte Gruppe Steam zu sehen, oder es öffnet das Chat-Fenster, um mit dem in der globalen Variable "FriendID" gespeicherten Benutzer zu chatten.
