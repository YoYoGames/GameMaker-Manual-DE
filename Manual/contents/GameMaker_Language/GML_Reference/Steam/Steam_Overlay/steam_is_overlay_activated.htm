steam_is_overlay_activated

Diese Funktion kann verwendet werden, um herauszufinden, ob der Benutzer das Steam Overlay aktiv hat oder nicht. Wenn das Overlay aktiv und für den Benutzer sichtbar ist, gibt die Funktion true zurück, und wenn nicht, dann false. Ein Beispiel für die Verwendung dieser Funktion wäre die Abfrage der Steam API für das Overlay, so dass Sie Ihr Spiel pausieren können, während das Overlay angezeigt wird.

Syntax:

steam_is_overlay_activated();

Rückgabe:

Boolean

Beispiel:

if steam_is_overlay_activated()
{
    global.Pause = true;
}

Die obige code prüft, ob das Overlay Steam aktiv ist, und wenn ja, setzt sie die globale Variable "Pause" auf true.
