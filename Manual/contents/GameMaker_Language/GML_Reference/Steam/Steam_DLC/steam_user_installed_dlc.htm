steam_user_installed_dlc

Wenn für Ihr Spiel DLC erstellt wurde, können Sie diese Funktion verwenden, um zu prüfen, ob der Benutzer den DLC installiert hat, bevor Sie auf die dazugehörigen Dateien zugreifen. Die Funktion gibt true zurück, wenn der Spieler den Inhalt installiert hat, und false, wenn der Benutzer ihn nicht installiert hat, aber beachten Sie, dass der Benutzer auch den DLC besitzen muss. steam_user_owns_dlc() verwenden, um zu prüfen, ob er auch im Besitz des DLCs ist, bevor Sie ihn verwenden. Beachten Sie, dass Steam IDs große Zahlen sein können und Sie daher Ihren ID-Wert als int64() umwandeln, bevor Sie ihn an die Funktion übergeben.

Syntax:

steam_user_installed_dlc(dlc_id);

Argument Beschreibung
dlc_id Der eindeutige Bezeichner für den zu prüfenden DLC (eine ganze Zahl).

Rückgabe:

Boolean

Beispiel:

global.Level_Max = 100;
if steam_user_owns_dlc(10354)
{
    if steam_user_installed_dlc(10354)
    {
        global.Level_max = 200;
    }
}

Die obige code prüft, ob der Benutzer den DLC mit der ID 10354 gekauft und installiert hat, und setzt in diesem Fall eine globale Variable auf einen anderen Wert.
