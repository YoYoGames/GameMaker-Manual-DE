Die Steam API unterstützt dauerhafte Ranglisten mit automatisch geordneten Einträgen. Diese Ranglisten können verwendet werden, um globale und Freundes-Ranglisten in Ihrem Spiel und auf der Community-Webseite für Ihr Spiel anzuzeigen. Jedes Spiel kann bis zu 10.000 Bestenlisten haben, und jede Bestenliste kann sofort abgerufen werden, nachdem der Punktestand eines Spielers in sie eingefügt wurde, aber beachten Sie, dass ein Spieler für jede Bestenliste nur einen Eintrag haben kann, obwohl es keine Begrenzung für die Anzahl der Spieler pro Bestenliste gibt.

Jeder Leaderboard-Eintrag enthält einen Namen, eine Punktzahl und einen Rang für das Leaderboard. Diese Daten werden ersetzt, wenn ein neuer Leaderboard-Eintrag für den Benutzer erstellt wird, und die folgenden Funktionen können zum Hinzufügen und Abrufen dieser Daten aus den Leaderboards für Ihr Spiel verwendet werden:

  • steam_create_leaderboard
  • steam_upload_score
  • steam_upload_score_buffer
  • steam_download_scores
  • steam_download_scores_around_user
  • steam_download_friends_scores
