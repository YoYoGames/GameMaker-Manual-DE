steam_download_friends_scores

Mit dieser Funktion können Sie nur die Spielstände auf der Bestenliste abrufen, die zu den Personen gehören, die im Steam Client als "Freunde" markiert sind. Wenn Ihre Bestenliste also 200 Einträge hat und 50 davon Ihre Freunde sind, wird diese Funktion nur diese 50 Ergebnisse abrufen. Der Name der Bestenliste ist eine Zeichenkette, die Sie beim Erstellen der Bestenliste mit der Funktion steam_create_leaderboard()definiert haben, und die Funktion gibt einen Wert zurück, der dann zur Identifizierung des Rückrufs im Steam-Async-Ereignis verwendet werden kann, oder sie gibt -1 zurück, wenn sie fehlgeschlagen ist.

Syntax:

steam_download_friends_scores(lb_name);

Argument Beschreibung
lb_name Der Name der Rangliste, von der Sie die Ergebnisse herunterladen (eine Zeichenkette).

Rückgabe:

Real

Beispiel:

score_get = steam_download_friends_scores("Game Scores");

Dies sendet eine Anfrage an Steam Server für die Spielstände der Freunde des Benutzers aus dem angegebenen Leaderboard und speichert die Async-ID der Anfrage in der Variablen "score_get". Dies wird dann in dem Steam Async Event behandelt, wie in dem erweiterten Beispiel für steam_download_scores().
