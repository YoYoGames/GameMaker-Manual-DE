steam_create_leaderboard

Mit dieser Funktion können Sie ein neues Leaderboard für Ihr Spiel erstellen. Die erste argument ist eine Zeichenkette, die den Namen Ihres Leaderboards definiert. Dieser Name sollte in allen weiteren Funktionsaufrufen verwendet werden, die sich auf das zu erstellende Leaderboard beziehen. Sie können können Sie dann die Sortiermethode (siehe die unten aufgeführten Konstanten) sowie die Art und Weise, wie die Informationen angezeigt werden, festlegen (wiederum eine der unten aufgeführten Konstanten). Die Funktion gibt den Wert -1 zurück, wenn sie die Rangliste nicht erstellen kann, oder einen Wert größer als -1 zurück, wenn sie erfolgreich war.

HINWEIS: Wenn Sie bereits eine Rangliste mit demselben Namen erstellt haben (entweder über code oder über Ihre Steam Seite für das Spiel), wird diese Funktion ignoriert!

Konstante sortieren Beschreibung
lb_sort_none Keine Sortierung. Die Informationen werden "wie sie sind" angezeigt.
lb_sort_ascending Sortieren Sie die Rangliste in aufsteigender Reihenfolge.
lb_sort_descending Sortieren Sie die Rangliste in absteigender Reihenfolge.

Konstante anzeigen Beschreibung
lb_disp_none Zeigen Sie die Rangliste "wie sie ist".
lb_disp_numerisch Zeigt die Bestenliste als numerische Anzeige an.
lb_disp_zeit_sec Zeigt die Werte der Bestenliste als Zeiten an, wobei der Basiswert Sekunden sind.
lb_disp_zeit_ms Zeigt die Werte der Bestenliste als Zeiten an, wobei der Basiswert Millisekunden sind.

Syntax:

steam_create_leaderboard(lb_name, sort_method, display_type);

Argument Beschreibung
lb_name Der Name der Rangliste, die Sie erstellen (eine Zeichenkette).
sort_method Die Methode zur Sortierung der Einträge in der Rangliste (siehe unten aufgeführte Konstanten).
display_type Die Art und Weise, wie die Bestenliste dem Benutzer angezeigt wird (siehe unten aufgeführte Konstanten).

Rückgabe:

Real

Beispiel:

steam_create_leaderboard("Game Times", lb_sort_ascending, lb_disp_time_sec);

Die obige code erstellt eine Rangliste mit dem Namen "Spielzeiten" und zeigt die Ergebnisse in aufsteigender Reihenfolge und mit einer Anzeige in Sekunden an.
