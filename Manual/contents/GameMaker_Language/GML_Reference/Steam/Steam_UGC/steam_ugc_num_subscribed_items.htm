steam_ugc_num_subscribed_items

Diese Funktion kann verwendet werden, um die Anzahl der Artikel zu ermitteln, die der aktuelle Benutzer abonniert hat.

Syntax:

steam_ugc_num_subscribed_items();

Rückgabe:

Real

Beispiel:

numSub = steam_ugc_num_subscribed_items();

In der obigen code wird die Anzahl der abonnierten Artikel in einer Variablen gespeichert.
