steam_ugc_query_set_return_total_only

Diese Funktion kann verwendet werden, um eine beliebige UGC-Abfrage weiter zu filtern, insbesondere um nur die Anzahl der Ergebnisse ohne weitere Informationen abzufragen (was bedeutet, dass die von der Sendefunktion erzeugte DS-Map den Schlüssel "num_results" ohne ohne weitere Map-Daten). Das Abfragehandle ist der Wert, der bei der Erstellung der Abfrage zurückgegeben wurde (z.B. mit, steam_ugc_create_query_user()) und die zweite argument ist entweder true oder falsch. Die Funktion gibt true zurück, wenn der Abfragefilter korrekt gesetzt wurde, andernfalls false.

Syntax:

steam_ugc_query_set_return_total_only(ugc_query_handle , total_only);

Argument Beschreibung
ugc_query_handle Das zu verwendende Abfragehandle.
total_only Ob die Abfrage nur die Gesamtzahl der Treffer zurückgeben soll oder nicht.

Rückgabe:

Boolean

Beispiel:

var query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1);
steam_ugc_query_set_return_total_only(query_handle, true);
steam_ugc_query_set_allow_cached_response(query_handle, true);
query_ID = steam_ugc_send_query(query_handle);

Die obige code erstellt eine Abfrage und speichert deren Handle in einer lokalen Variable zur späteren Verwendung in den übrigen Funktionen, die die Abfrage vor dem Senden der Abfrage weiter definieren.
