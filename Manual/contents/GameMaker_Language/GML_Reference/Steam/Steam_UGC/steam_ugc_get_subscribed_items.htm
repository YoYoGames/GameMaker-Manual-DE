steam_ugc_get_subscribed_items

Diese Funktion füllt eine DS-Liste mit allen veröffentlichten Datei-IDs für die Artikel, die der Benutzer derzeit abonniert hat. Sie müssen zunächst die Liste erstellen und den Index in einer Variablen speichern und diese dann an die Funktion übergeben. Die Funktion gibt true zurück, wenn alles korrekt ist und die Steam API initialisiert wurde, oder false, wenn ein Fehler aufgetreten ist.

Syntax:

steam_ugc_get_subscribed_items(item_list);

Argument Beschreibung
item_list Ein (zuvor erstellter) DS-Listenindex.

Rückgabe:

Boolean

Beispiel:

steam_list = ds_list_create();
steam_ugc_get_subscribed_items(steam_list);

Die obige code erstellt eine leere DS-Liste und füllt sie dann mit den Datei-IDs für alle abonnierten Elemente für den Benutzer.
