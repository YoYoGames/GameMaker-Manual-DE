steam_ugc_query_set_return_long_description

Diese Funktion kann verwendet werden, um eine beliebige UGC-Abfrage weiter zu filtern, insbesondere um den langen Beschreibungstext im Rückrufereignis abzurufen, das beim Senden der Abfrage ausgelöst wurde. Das Abfragehandle ist der Wert, der zurückgegeben wurde, als Sie die Abfrage erstellt haben (z. B. mit Beispiel, steam_ugc_create_query_user()) und das zweite argument ist entweder true oder false. Die Funktion gibt true zurück, wenn der Abfragefilter korrekt gesetzt wurde, andernfalls false.

Syntax:

steam_ugc_query_set_return_long_description(ugc_query_handle , long_description);

Argument Beschreibung
ugc_query_handle Das zu verwendende Abfragehandle.
long_description Ob die Abfrage den langen Beschreibungstext zurückgeben soll.

Rückgabe:

Boolean

Beispiel:

var query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1);
steam_ugc_query_set_return_long_description(query_handle, true);
steam_ugc_query_set_allow_cached_response(query_handle, true);
query_ID = steam_ugc_send_query(query_handle);

Die obige code erstellt eine Abfrage und speichert deren Handle in einer lokalen Variable zur späteren Verwendung in den übrigen Funktionen, die die Abfrage vor dem Senden der Abfrage weiter definieren.
