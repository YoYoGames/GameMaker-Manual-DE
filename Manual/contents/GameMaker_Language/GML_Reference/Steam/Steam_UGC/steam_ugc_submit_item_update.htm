steam_ugc_submit_item_update

Diese Funktion überträgt das UGC-Element, das mit dem angegebenen Handle angegeben wurde, an den Steam Workshop servers und fügt die Änderungsnotizen hinzu, die für das angegebene Element verwendet werden sollen. Die Änderungsnotizen sollten eine Zeichenkette sein und das Update-Handle ist der Wert, der beim Aufruf der Funktion Funktion steam_ugc_start_item_update(). Bei Verwendung dieser Funktion wird ein asynchroner ID-Wert zurückgegeben, der geparst werden kann, wenn das asynchrone Steam-Ereignis ausgelöst wird, um die Erstellung des Elements zu melden. Das Ereignis enthält die folgenden Schlüssel/Map-Werte in der async_load DS-Map:

  • "id" - Die von der aufrufenden Funktion zurückgegebene asynchrone ID
  • "result" - Das Ergebnis der Operation (ein reeller Wert). Dies ist entweder die Konstante GML ugc_result_success oder eine andere reelle Zahl. Sie sollten also auf diese Konstante prüfen, um sicherzustellen, dass der Aufruf erfolgreich war, und wenn andernfalls irgendetwas nicht korrekt funktioniert hat. Die übrigen möglichen Werte, die zurückgegeben werden, werden als Ergebnis des Steam "EResult"-Wertes angezeigt, und Sie sollten in der Datei steamclientpublic.h in den SDK-Headern alle 89 möglichen Werte finden.
  • "event_type" - Dieser Schlüssel enthält den Wert"ugc_update_item".
  • "legal_agreement_required" - Wird true oder false sein (siehe die Steam-Dokumente für weitere Details)

Syntax:

steam_ugc_submit_item_update(ugc_update_handle, change_note);

Argument Beschreibung
ugc_update_handle Der eindeutige Handle für die zu aktualisierende UGC.
change_note Die Änderungsnotizen (eine Zeichenkette), die für den Artikel verwendet werden sollen.

Rückgabe:

Async ID

Beispiel:

var app_id = steam_get_app_id();
var updateHandle = steam_ugc_start_item_update(app_id, global.Publish_ID);
steam_ugc_set_item_title(updateHandle, "My workshop item(3)!");
steam_ugc_set_item_description( updateHandle, "testing workshop...");
steam_ugc_set_item_visibility(updateHandle, ugc_visibility_public);
var tagArray;
tagArray[0] = "Test";
tagArray[1] = "New";
steam_ugc_set_item_tags(updateHandle, tagArray);
steam_ugc_set_item_preview(updateHandle, "promo.jpg");
steam_ugc_set_item_content(updateHandle, "WorkshopContent1");
requestId = steam_ugc_submit_item_update(updateHandle, "Version 1.2");

Die obige code ruft die Spiel-ID ab und verwendet diese zusammen mit einer zuvor gespeicherten veröffentlichten Datei-ID, um ein Aktualisierungs-Handle für das Objekt zu erzeugen. Dieses Handle wird dann verwendet, um verschiedene Informationen zu aktualisieren, bevor die Aktualisierung an den Workshop übermittelt wird servers .
