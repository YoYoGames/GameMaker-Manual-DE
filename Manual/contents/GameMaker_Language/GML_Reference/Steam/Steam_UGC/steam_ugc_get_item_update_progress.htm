steam_ugc_get_item_update_progress

Diese Funktion kann verwendet werden, um den Aktualisierungsstatus für einen Artikel zu verfolgen. Sie geben das Item-Handle (wie von der Funktion steam_ugc_start_item_update()) und eine leere DS-Map, die dann mit den folgenden Schlüssel/Wertpaaren gefüllt wird:

  • "status_code" - Der Statuscode Steam
  • "status_string" - Eine Zeichenkette für den aktuellen Status
  • "bytes_processed" - Die bisher verarbeiteten Bytes
  • "bytes_total" - Die Gesamtzahl der Bytes in der Aktualisierung

Wenn ein Fehler auftritt, gibt die Funktion false zurück und die Karte ist leer, andernfalls gibt die Funktion true zurück.

Syntax:

steam_ugc_get_item_update_progress(ugc_update_handle, info_map);

Argument Beschreibung
ugc_update_handle Der eindeutige Handle für die zu aktualisierende UGC.
info_map Ein (zuvor erstellter) DS-Map-Index.

Rückgabe:

Boolean

Beispiel:

var uploadMap = ds_map_create();
steam_ugc_get_item_update_progress(global.itemHandle, uploadMap);
var statusCode = uploadMap[? "status_code"];
var status = uploadMap[? "status_string"];
var processed = uploadMap[? "bytes_processed"];
var total = uploadMap[? "bytes_total"];
draw_text(32, 0, "Upload info for item:" + string(global.itemHandle));
draw_text(32, 15, "status code:" + string(statusCode));
draw_text(32, 30, "status:" + string(status));
draw_text(32, 45, "bytes processed:" +string(processed));
draw_text(32, 60, "bytes total:" + string( total));
ds_map_destroy(uploadMap);

Die obige code fragt den Upload-Status des in der globalen Variable "itemHandle" indizierten Elements ab, wobei eine DS-Map zum Speichern der Informationen verwendet wird. Diese wird dann geparst und die resultierenden Werte werden auf dem Bildschirm angezeigt.
