steam_ugc_create_query_user

Diese Funktion kann zur Abfrage der UGC-Datenbank verwendet werden. Die Funktion verwendet automatisch die Standard-ID für die App und den Benutzer und geht davon aus, dass die Abfrage vom Verbraucher (und nicht vom Ersteller) durchgeführt wird. Die Funktion erfordert die Verwendung der folgenden Konstanten für die Art der abzufragenden Daten, die Art des abzufragenden Elements und die Reihenfolge, in der die zurückgegebenen Elemente sortiert werden sollen, sowie eine Seitenzahl - beachten Sie, dass eine Abfrage maximal 50 Elemente zurückgibt.

Die Funktion gibt einen eindeutigen Abfrage-Handle-Wert zurück, der in einer Variablen zur Verwendung in den anderen Abfragefunktionen gespeichert werden sollte. Beachten Sie, dass diese Funktion die Abfrage nur vorbereitet, aber nicht tatsächlich sendet - dafür müssen Sie die Funktion steam_ugc_send_query() - aufrufen, und Sie können weitere steam_ugc_query_*() -Funktionen verwenden, um die Suchanfrage zu verfeinern, bevor sie tatsächlich gesendet wird.

Syntax:

steam_ugc_create_query_user(list_type, match_type, sort_order, page);

Argument Beschreibung
list_type Der Typ der zu erstellenden Datenliste (siehe unten aufgeführte Konstanten).
match_type Der Typ der abzufragenden UGC-Elemente (siehe unten aufgeführte Konstanten).
sort_order Die Art und Weise, wie die Daten geordnet werden sollen (siehe unten aufgeführte Konstanten).
page Die abzufragende Seitenzahl.

Rückgabe:

Query Handle

Beispiel:

query_handle = steam_ugc_create_query_user(ugc_list_Published, ugc_match_Items, ugc_sortorder_TitleAsc, 1);

Die obige code erstellt eine Abfrageanfrage und speichert deren Handle in einer Variablen zur späteren Verwendung.
