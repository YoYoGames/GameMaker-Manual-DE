steam_ugc_query_add_required_tag

Diese Funktion kann verwendet werden, um eine gegebene UGC-Abfrage weiter zu filtern, insbesondere um nur UGC-Elemente mit dem angegebenen Tag zu suchen. Das Abfrage-Handle ist der Wert, der zurückgegeben wird, wenn Sie die Abfrage erstellen (z. B. mit, steam_ugc_create_query_user()) und die zweite argument ist eine Zeichenkette, die Sie als einzuschließenden Tag verwenden möchten. Die Funktion gibt true zurück, wenn der Abfragefilter korrekt gesetzt wurde, andernfalls false.

Syntax:

steam_ugc_query_add_required_tag(ugc_query_handle , tag_name);

Argument Beschreibung
ugc_query_handle Das zu verwendende Abfragehandle.
tag_name Der einzuschließende Tag-Name (eine Zeichenkette).

Rückgabe:

Boolean

Beispiel:

var query_handle = steam_ugc_create_query_all(ugc_query_RankedByTrend, ugc_match_Items, 1);
steam_ugc_query_add_required_tag(query_handle, "RPG");
steam_ugc_query_set_return_long_description(query_handle, true);
steam_ugc_query_set_allow_cached_response(query_handle, true);
query_ID = steam_ugc_send_query(query_handle);

Die obige code erstellt eine Abfrage und speichert deren Handle in einer lokalen Variable zur späteren Verwendung in den übrigen Funktionen, die die Abfrage vor dem Senden der Abfrage weiter definieren.
