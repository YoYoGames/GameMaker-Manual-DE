steam_ugc_get_item_install_info

Diese Funktion kann verwendet werden, um Informationen über ein bestimmtes veröffentlichtes Dateielement abzurufen, das abonniert und in den lokalen Speicherbereich Steam für Ihr Spiel heruntergeladen wurde. Sie geben die Element-ID an und liefern den Index einer leeren DS-Map, die dann mit den folgenden Schlüssel/Wert-Paaren gefüllt wird, wenn das Element existiert:

  • "size_on_disk" - Die Dateigröße auf der Festplatte (in Bytes)
  • "legacy_item" - Wird true oder false sein, je nachdem, ob es sich um eine Legacy-Datei handelt oder nicht.
  • "folder" - Dies ist der vollständige Pfad zum installierten Inhalt (siehe "Item Installation" in Steam SDK docs, da "legacy" Artikel, die mit der alten Methode hochgeladen wurden, anders behandelt werden).

Wenn das Element existiert, gibt die Funktion true zurück und füllt die Karte auf, andernfalls gibt sie false zurück und die Karte bleibt leer.

Syntax:

steam_ugc_get_item_install_info(published_file_id, info_map);

Argument Beschreibung
published_file_id Der eindeutige Handle für die zu aktualisierende UGC.
info_map Ein (zuvor erstellter) DS-Map-Index.

Rückgabe:

Boolean

Beispiel:

var item_map = ds_map_create();
steam_ugc_get_item_install_info(global.fileID, item_map);

Die obige code fragt den Installationsstatus des Elements ab, das in der globalen Variable "fileID" indiziert ist, und verwendet eine DS Map, um die Informationen zu speichern.
