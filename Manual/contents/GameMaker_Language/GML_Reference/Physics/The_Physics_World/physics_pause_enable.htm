physik_pause_aktivieren

Damit eine Physiksimulation funktioniert, muss sie normalerweise kontinuierlich ablaufen und darf nicht angehalten und wieder gestartet werden, und die Instanzen dürfen nicht plötzlich von einem Ort zum anderen im Raum verschoben werden. Es gibt jedoch Momente, in denen die Simulation angehalten werden muss, weil z. B. das Betriebssystem des Geräts pausiert hat, und dann wird diese Funktion verwendet. Sie unterbricht die Simulation, wenn das Flag auf true gesetzt ist, und es werden keine weiteren Physikberechnungen durchgeführt, bis das Flag wieder auf false gesetzt wird.

HINWEIS: Dies ist besonders nützlich, wenn Sie alle Instanzen in einem Raum deaktivieren möchten, da ein physischer Körper auch nach der Deaktivierung weiterhin in der Physikwelt berechnet und simuliert wird.

Syntax:

physics_pause_enable(flag)

ArgumentTyp Beschreibung
flag This can be set to true um die Simulation anzuhalten, oder false, um sie erneut zu starten.

Rückgabe:

Beispiel:

if os_is_paused()
{
    physics_pause_enable(true);
    instance_deactivate_all(true);
    instance_create_layer(x, y, "Controllers", obj_PauseMenu);
}

Das obige code prüft, ob das Betriebssystem angehalten wurde, und wenn dies der Fall ist, hält es die Physikwelt an, deaktiviert alles außer sich selbst und erstellt dann eine Instanz des Pausenmenüs.
