physics_draw_debug

Beim Erstellen eines Physiksystems in GameMaker ist es oft wichtig, eine visuelle Darstellung dessen zu haben, was mit einer Instanz tatsächlich passiert. Dies kann durch die Verwendung von physics_draw_debug() erreicht werden, das einen schematischen Umriss der physikalischen Eigenschaften zeichnet, die mit der Instanz verbunden sind, auf der die code läuft. Hier ist ein Bild davon, wie es aussieht:

Es sollte beachtet werden, dass dies nur dann funktioniert , wenn es im Draw-Ereignis der Instanz sein muss, und es wird mit der aktuell definierten Farbe gezeichnet.

Syntax:

physics_draw_debug()

Rückgabe:

Beispiel:

draw_set_colour(c_red);
physics_draw_debug();

Die obige code zeichnet eine Darstellung der physikalischen Eigenschaften, die mit dieser Instanz verbunden sind, unter Verwendung der Farbe Rot.
