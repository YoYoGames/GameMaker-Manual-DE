Die Welt der Physik

Bevor physikalische Reaktionen in GameMaker auftreten können, müssen Sie zunächst die physikalische Welt definieren, in der sie ablaufen sollen. Wenn Sie dies nicht tun, verwendet GameMaker standardmäßig die grundlegenden Kollisionsfunktionen, und alle anderen physikbezogenen code oder Einstellungen werden ignoriert. Sie können die Physikwelt im Raum-Editor einrichten, aber Sie können die gleichen Eigenschaften auch über code mit den folgenden Funktionen einstellen:

  • physics_world_create
  • physics_world_gravity
  • physics_world_update_iterations
  • physics_world_update_speed
  • physics_world_draw_debug
  • physics_draw_debug
  • physics_pause_enable
