phy_bullet

Diese Variable legt fest, ob sich die Instanz extrem schnell bewegt (z. B. ein Geschoss) oder nicht. Der Standardwert ist false, aber wenn sie auf true gesetzt wird, teilt dies GameMaker mit, dass sich die Instanz mit so hoher Geschwindigkeit bewegt, dass sie eine aufwendigere Kollisionserkennung benötigt, um sicherzustellen, dass sie nicht unerkannt durch andere Instanzen hindurchgeht

Syntax:

phy_bullet;

Rückgabe:

(or undefined if the instance is not physics enabled)

Beispiel:

with (instance_create_layer(x, y, "Bullets", obj_Shoot))
{
    phy_bullet = true;
    physics_apply_local_impulse(0, 10, 0, 200);
}

Die obige code erstellt eine neue Instanz und definiert sie dann als "Geschoss" in der Physikwelt, bevor sie ihr einen massiven Impuls entlang der y-Achse gibt.
