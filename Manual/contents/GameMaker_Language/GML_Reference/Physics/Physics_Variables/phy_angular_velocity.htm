phy_angular_velocity

Diese Variable kann verwendet werden, um die Winkelgeschwindigkeit der Instanz festzulegen, oder sie kann verwendet werden, um die aktuelle Winkelgeschwindigkeit in Grad pro Sekunde zu erhalten, und der verwendete Wert kann entweder positiv (für eine Drehung im Uhrzeigersinn) oder negativ (für eine Drehung gegen den Uhrzeigersinn) sein. Wenn du dies auf eine Instanz setzt, die vorher statisch war (d.h. sie hat eine Dichte von 0), wird sie zu einem kinematischen Objekt und beginnt sich zu drehen

Syntax:

phy_angular_velocity;

Rückgabe:

(single precision floating point value, or undefined if the instance is not physics enabled)

Beispiel:

if abs(phy_angular_velocity) > 0
{
    phy_angular_velocity -= sign(phy_angular_velocity) * 0.01;
}
else
{
    phy_angular_velocity = 0;
}

Die obige code prüft die Winkelgeschwindigkeit der Instanz und wenn sie nicht 0 ist, wird bei jedem Schritt ein kleiner Betrag addiert (oder subtrahiert), bis der Wert 0 ist.
