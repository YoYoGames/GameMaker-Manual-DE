phy_col_normal_y

Diese schreibgeschützte Variable gibt die y-Komponente der Kollisionsnormale zurück, die dem phy_collision_y Array-Wert entspricht. Für jeden Kontaktpunkt gibt es eine zugehörige Kontaktnormale (die in der Regel die gleiche Normale für alle Kontaktpunkte in der Kollision ist). Diese Kontaktnormale ist eine Einheit vector, die von einer Instanz in der Kollision zu einer anderen zeigt, und kann zum Beispiel verwendet werden, um die korrekte "Push"-Richtung zur Auflösung von Kollisionen zu berechnen

HINWEIS: Diese Variable ist nur im Kollisionsfall einer physikalisch aktivierten Instanz verfügbar.

Syntax:

phy_col_normal_y;

Rückgabe:

(single precision floating point value, or undefined if the instance is not physics enabled)

Beispiel:

repeat(5 + irandom(5))
{
    with (instance_create_layer(x, y, "Effects", obj_Debris)
    {
        physics_apply_local_impulse(0, 0, other.phy_col_normal_x[0], other.phy_col_normal_y[0]);
    }
}

Die obige code verwendet die Kontaktnormale, um die Bewegungsrichtung für eine Instanz festzulegen, die im Kollisionsereignis zwischen zwei physikfähigen Instanzen erstellt wurde.
