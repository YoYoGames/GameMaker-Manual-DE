phy_kinematisch

Diese schreibgeschützte Variable gibt true zurück, wenn die Instanz als kinematisches Objekt klassifiziert ist, oder false, wenn dies nicht der Fall ist. Eine kinematische Instanz ist eine Instanz, die eine unendliche Masse (eine Dichte von 0) hat, sich aber bewegen kann. Um eine Instanz kinematisch zu machen, erstellen Sie also zunächst eine statische Instanz und setzen dann eine oder mehrere Instanzvariablen, die sich auf die Bewegung beziehen (z. B: phy_speed_x, phy_speed_y, oder phy_angular_velocity)

Syntax:

phy_kinematic;

Rückgabe:

(or undefined if the instance is not physics enabled)

Beispiel:

if !phy_kinematic
{
    phy_speed_x = 5;
}

Die obige code prüft, ob die Instanz kinematisch ist, und wenn nicht, setzt sie die horizontale Geschwindigkeit auf 5.
