phy_position_x

Diese Variable kann verwendet werden, um die x-Position der Instanz innerhalb der Spielraum-Physikwelt zu erhalten (oder zu setzen). Bitte beachten Sie, dass die Physikwelt Fehler anzeigen kann, wenn Instanzen durch direktes Setzen dieser Variable bewegt werden, da dies die kontinuierliche Simulation unterbricht. Diese Variable ist das physikalische Äquivalent der Instanzvariable x.

Syntax:

phy_position_x;

Rückgabe:

(single precision floating point value, or undefined if the instance is not physics enabled)

Beispiel:

xx = phy_position_x;
yy = phy_position_y;

Die obige code speichert die x- und y-Position der Instanz in zwei Variablen.
