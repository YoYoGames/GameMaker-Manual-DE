phy_collision_y

Dieses schreibgeschützte Array gibt die y-Position aller Punkte zurück, die bei einer Kollision zwischen zwei physikfähigen Instanzen entdeckt wurden.

HINWEIS: Diese Variable ist nur im Kollisionsfall einer physikalisch aktivierten Instanz verfügbar.

Syntax:

phy_collision_y;

Rückgabe:

(single precision floating point value, or undefined if the instance is not physics enabled)

Beispiel:

for(var i = 0; i < phy_collision_points; i += 1;)
{
    part_particles_create(global.Sname, phy_collision_x[i], phy_collision_y[i], global.Spark, 5);
}

Die obige code erzeugt Partikel an allen definierten Punkten einer Kollision zwischen zwei physikfähigen Instanzen.
