phy_fixed_rotation

Mit dieser Variablen kann eingestellt werden, ob die Instanz von Rotationskräften beeinflusst werden kann oder nicht (Standard ist false). Wenn diese Variable auf true gesetzt ist, wird keine externe Kraft (weder von kodierten Impulsen oder Kräften, noch von Kollisionen) den Rotationswert der Instanz beeinflussen. den Rotationswert der Instanz und dieser müsste manuell mit der phy_rotation Variable

Syntax:

phy_fixed_rotation

Rückgabe:

(or undefined if the instance is not physics enabled)

Beispiel:

phy_rotation = 0;
phy_fixed_rotation = true;

Die obige code wird switch die Instanz eine feste Drehung haben, dann setzen Sie den Drehwinkel.
