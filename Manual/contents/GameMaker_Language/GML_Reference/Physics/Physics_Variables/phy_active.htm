phy_aktiv

Diese Variable steuert, ob die Instanz gerade "aktiv" ist oder nicht. Wenn sie auf false gesetzt wird, nimmt die Instanz nicht mehr an der Physikwelt teil, und wenn sie auf true gesetzt wird, nimmt sie wieder teil. Bitte beachte, dass dies nicht dasselbe ist wie das Deaktivieren der Instanz, da die Instanz immer noch auf dem Bildschirm sichtbar ist und immer noch über code verändert werden kann, sondern diese Funktion verhindert nur, dass sie an der Physiksimulation teilnimmt.

Syntax:

phy_active;

Rückgabe:

(or undefined if the instance is not physics enabled)

Beispiel:

if keyboard_check_pressed(ord"P")
{
    global.Pause = !global.Pause;
    with (obj_Parent)
    {
        phy_active = !global.Pause;
    }
}

Die obige code erkennt einen Tastendruck des Buchstabens "P" und schaltet dann die globale Variable "Pause" von true auf false und wieder zurück. Diese Variable wird dann verwendet, um festzulegen, ob die Physik in den Kindinstanzen des in der Variablen "obj_Parent" indizierten Objekts aktiv ist oder nicht.
