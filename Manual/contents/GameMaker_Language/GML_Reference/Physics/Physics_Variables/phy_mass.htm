phy_mass

Diese schreibgeschützte Variable gibt die Masse der Instanz in Kilogramm zurück. Dieser Wert wird automatisch auf der Grundlage der Oberfläche der zugewiesenen Einrichtungsgegenstände und ihrer Dichtewerte berechnet, kann aber mit der Funktion physics_mass_properties().

Syntax:

phy_mass;

Rückgabe:

(single precision floating point value, or undefined if the instance is not physics enabled)

Beispiel:

if (phy_mass < other.phy_mass)
{
    instance_destroy();
}

Die obige code ist von dem Kollisionsereignis der Instanz mit einer anderen und es vergleicht die Masse jeder Instanz und zerstört diejenige, die weniger Masse hat.
