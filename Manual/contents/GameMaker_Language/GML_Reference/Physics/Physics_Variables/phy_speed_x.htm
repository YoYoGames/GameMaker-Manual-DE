phy_speed_x

Diese Variable kann verwendet werden, um die x-Komponente der linearen Geschwindigkeit der Instanz zu erhalten oder zu ändern vector und ist in Pixeln pro Schritt definiert (für Pixel pro Sekunde siehe phy_linear_velocity_x). Wenn du diese Variable für eine statische Instanz änderst (d.h. eine Instanz mit 0 Dichte), wird sie zu einer kinematischen Instanz.

Syntax:

phy_speed_x;

Rückgabe:

(single precision floating point value, or undefined if the instance is not physics enabled)

Beispiel:

if phy_speed_x != 0
{
    phy_speed_x = 0;
}

Die obige code prüft die x-Komponente der linearen Geschwindigkeit vector und setzt sie auf 0, wenn sie einen anderen Wert als 0 hat.
