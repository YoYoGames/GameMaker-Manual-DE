phy_collision_points

Diese schreibgeschützte Variable gibt die Anzahl der erkannten Kollisionspunkte zwischen den beiden Objekten in der Kollision zurück

HINWEIS: Diese Variable ist nur im Kollisionsfall einer physikalisch aktivierten Instanz verfügbar.

Syntax:

phy_collision_points;

Rückgabe:

(integer value, or undefined if the instance is not physics enabled)

Beispiel:

for(var i = 0; i < phy_collision_points; i += 1;)
{
    part_particles_create(global.Sname, phy_collision_x[i], phy_collision_x[1], global.Spark, 5);
}

Die obige code erzeugt Partikel an allen definierten Punkten einer Kollision zwischen zwei physikfähigen Instanzen.
