phy_speed

Diese schreibgeschützte Variable gibt die aktuelle Geschwindigkeit der physikalisch aktivierten Instanz zurück, definiert in Pixel pro Schritt. Sollten Sie diesen Wert ändern wollen, müssen Sie dies tun, indem Sie die x- und y-Werte vectors mit den Variablen phy_speed_x und phy_speed_y.

Syntax:

phy_speed;

Rückgabe:

(single precision floating point value, or undefined if the instance is not physics enabled)

Beispiel:

if phy_speed > 10
{
    phy_linear_damping += 0.01;
}
else
{
    phy_linear_damping = 2;
}

Die obige code prüft die Geschwindigkeit der physikalisch aktivierten Instanz und ändert dann die lineare Dämpfung basierend auf dem zurückgegebenen Wert.
