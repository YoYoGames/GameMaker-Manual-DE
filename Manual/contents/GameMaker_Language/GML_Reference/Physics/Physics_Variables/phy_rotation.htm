phy_rotation

Diese Variable kann verwendet werden, um den Winkel der Instanzenbefestigung in Grad zu erhalten (oder zu setzen), ähnlich wie das Setzen oder Erhalten der image_angle. Beachten Sie jedoch, dass Rotationen in der Physikwelt anders als in der normalen GameMaker-Spielwelt berechnet werden, was bedeutet, dass vector Funktionen wie point_direction()ihre Rückgabewerte ändern müssen (einfach positive Werte in negative umwandeln sollte dies beheben).

Syntax:

phy_rotation;

Rückgabe:

(single precision floating point value, or undefined if the instance is not physics enabled)

Beispiel:

if (phy_speed_x > 0) || (phy_speed_y > 0)
{
    phy_rotation += sqrt(sqr(phy_speed_x) + sqr(phy_speed_y)) /10;
}

Die obige code prüft die lineare Geschwindigkeit, und wenn eine der beiden vector nicht 0 ist, berechnet sie die tatsächliche Geschwindigkeit und verwendet diese, um die Drehung einzustellen.
