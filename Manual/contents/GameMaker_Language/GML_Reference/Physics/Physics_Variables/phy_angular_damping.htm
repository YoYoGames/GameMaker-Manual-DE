phy_angular_damping

Diese Variable kann verwendet werden, um die Winkeldämpfung der Instanz einzustellen, oder sie kann verwendet werden, um die aktuelle Winkeldämpfung zu erhalten. Die Dämpfung ist der "Widerstand" der physikalisch aktivierten Instanz gegen die Winkelrotation, wobei ein niedriger Wert es der Instanz erlaubt, sich nach einer Kollision (zum Beispiel) schneller zu drehen, und ein höherer Wert einen stärkeren Stoß erfordert und sie sich langsamer dreht

Syntax:

phy_angular_damping;

Rückgabe:

(single precision floating point value, or undefined if the instance is not physics enabled)

Beispiel:

if place_meeting(phy_position_x, phy_position_y, obj_Water)
{
    phy_angular_damping = 10;
}
else
{
    phy_angular_damping = 3;
}

Die obige code prüft auf eine Kollision zwischen der aufrufenden Instanz und Instanzen von "obj_Water" und ändert die Winkeldämpfung entsprechend.
