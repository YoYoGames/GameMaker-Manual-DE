phy_sleeping

Diese schreibgeschützte Variable gibt zurück, ob die Instanz gerade "schläft" (true) oder nicht (false). Eine "schlafende" Instanz ist eine, die nicht aktiv an einer physikalischen Simulation beteiligt ist. GameMaker versetzt Objekte in den Ruhezustand, um Simulationszyklen zu sparen, wenn eine Instanz in Ruhe ist und nicht mit einer anderen Instanz kollidiert

Syntax:

phy_sleeping

Rückgabe:

(or undefined if the instance is not physics enabled)

Beispiel:

if phy_sleeping
{
    instance_destroy();
}

Die obige code überprüft, ob das Objekt aktiv simuliert wird oder nicht, und wenn nicht, wird es zerstört.
