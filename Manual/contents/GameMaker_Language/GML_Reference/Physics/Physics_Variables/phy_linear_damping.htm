phy_linear_damping

Diese Variable kann verwendet werden, um die lineare Dämpfung der Instanz einzustellen, oder sie kann verwendet werden, um die aktuelle lineare Dämpfung zu erhalten. Die Dämpfung ist der "Widerstand", den die physikalisch aktivierte Instanz der Vorwärtsbewegung entgegensetzt, wobei ein niedriger Wert der Instanz erlaubt, sich schneller zu bewegen und zu beschleunigen, und ein höherer Wert einen kräftigeren Stoß erfordert

Syntax:

phy_linear_damping

Rückgabe:

(single precision floating point value, or undefined if the instance is not physics enabled)

Beispiel:

if place_meeting(phy_position_x, phy_position_y, obj_Water)
{
    phy_linear_damping = 10;
}
else
{
    phy_linear_damping = 3;
}

Die obige code prüft auf eine Kollision zwischen der aufrufenden Instanz und Instanzen von "obj_Water" und ändert die lineare Dämpfung entsprechend.
