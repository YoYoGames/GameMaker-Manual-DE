physik_impuls_anwenden

Mit der GameMaker-Physik können Sie nicht nur Kraft und Schwerkraft auf ein Objekt anwenden, sondern auch einen Impuls. Dieser unterscheidet sich insofern von einer Kraft, als dass er sich sofort auf die Geschwindigkeit und möglicherweise auf das Drehmoment (oder den "Spin") des Objekts auswirkt, insbesondere wenn der Punkt, auf den der Impuls angewendet wird, nicht mit dem Massenschwerpunkt übereinstimmt (Hinweis: Der Massenschwerpunkt ist nicht unbedingt mit dem Ursprung identisch!). Hier ist eine Illustration:

Wie Sie sehen können, hat der Spieler auf den Ball geklickt und ihm dadurch einen Impuls gegeben, sich nach rechts oben zu bewegen und sich dabei zu drehen. Der genaue Impuls wird durch die vector definiert, die wir aus den Komponenten ximpulse/yimpulse im Verhältnis zu den xpos/ypos-Koordinaten erhalten - was einfach bedeutet, dass der Impuls als die Entfernung von xpos/ypos zu ximpulse/yimpulse in Newton berechnet wird, und die Richtung ist der Winkel, den wir von xpos/ypos zu ximpulse/yimpulse erhalten.

HINWEIS: Die Physikwelt verwendet Meter für alle Berechnungen, und das Verhältnis von Pixeln pro Meter wird im Raumeditor oder mit physics_world_create().

Syntax:

physics_apply_impulse(xpos, ypos, ximpulse, yimpulse)

ArgumentTyp Beschreibung
xpos The x coordinate in the room where the impulse will be applied
ypos The y coordinate in the room where the impulse will be applied
ximpulse the x component of the impulse vector
yimpulse the y component of the impulse vector

Rückgabe:

Beispiel:

if mouse_check_button_pressed(mb_left)
{
    with (place_meeting(mouse_x, mouse_y, all))
    {
        physics_apply_impulse(mouse_x, mouse_y, -10 + irandom(20), -10 + irandom(20));
    }
}

Die obige code wendet einen Impuls mit einem zufälligen vector auf eine Instanz an, die sich an der Mausposition befindet, wenn die linke Taste gedrückt wird.
