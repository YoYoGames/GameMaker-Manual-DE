physics_apply_torque

Wenn wir von Drehmoment sprechen, meinen wir in der Regel die Tendenz einer Kraft oder eines Impulses, eine Instanz um ihren Massenschwerpunkt zu drehen, aber in GameMaker können Sie auch unabhängig davon ein Drehmoment auf eine Instanz anwenden, um sie in eine bestimmte Richtung drehen zu lassen. Wie viel Drehmoment du aufbringst, um die gewünschte Drehung zu erreichen, hängt von der Größe und der Dichte der Halterung ab, die du deiner Instanz zuweist, da diese beiden Eigenschaften ihre Masse definieren, die wiederum ihre Trägheit definiert, die das Drehmoment beeinflusst! Als allgemeine Faustregel gilt: Wenn eine Instanz eine große Trägheit hat, brauchst du ein großes Drehmoment, um sie in Drehung zu versetzen. Es sollte auch beachtet werden, dass andere Faktoren wie Schwerkraft und Kollisionen das Drehmoment einer Instanz ebenfalls beeinflussen können. Hier ist ein Bild zur Erklärung:

Wie Sie sehen können, hängt das Drehmoment, das Sie benötigen, um das Objekt zu drehen, von der zugrunde liegenden Masse und Trägheit ab. Dies alles wird von GameMaker auf der Grundlage der Größe und Dichte des Fixtures berechnet, das der Instanz beim Erstellen zugewiesen wird. Beachte auch, dass für eine Drehung gegen den Uhrzeigersinn der Wert negativ sein sollte.

Syntax:

physics_apply_torque(torque)

ArgumentTyp Beschreibung
torque The torque (in Newton metres) to be applied

Rückgabe:

Beispiel:

if keyboard_check(vk_left)
{
    physics_apply_torque(-100);
}

Die obige code wird die Instanz gegen den Uhrzeigersinn (nach links) mit einem Drehmoment von 100 Nm drehen, wenn die linke Pfeiltaste gedrückt wird.
