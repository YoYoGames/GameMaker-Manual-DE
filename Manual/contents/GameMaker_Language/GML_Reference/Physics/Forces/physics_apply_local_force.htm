physics_apply_local_force

Eine andere Möglichkeit, Kraft in der Physikwelt zu verwenden, ist, sie lokal auf eine Instanz anzuwenden. Das bedeutet, dass die Stärke und Richtung der Kraft auf der Grundlage des Ursprungs (oder der Position, wenn sie keine sprite hat) der Instanz berechnet wird, ohne die Richtung oder Drehung zu berücksichtigen, die sie im Spielraum oder in der Physikwelt haben kann. Es sollte beachtet werden, dass mit dieser Funktion die Kräfte nicht auf den Massenschwerpunkt des Objekts angewendet werden, sondern auf einen Punkt relativ zur Instanz, und sie werden nicht sofort angewendet, da sie von anderen Kräften abhängig sind, die auf das Objekt wirken (wie die Schwerkraft). Diese Illustration zeigt, wie eine lokale Kraft funktioniert:

Wie Sie sehen können, werden die xlocal- und ylocal-Koordinaten auf der Grundlage der Achsenposition berechnet, die immer als Ursprung der Instanz (0,0) betrachtet wird, d.h. egal, wo im Raum sich das Objekt befindet, xlocal und ylocal werden immer die gleichen sein. Du musst dann entscheiden, wo die vector Komponenten der Kraft (xforce und yforce) relativ zu der xlocal/ylocal Position positioniert werden sollen. Die genaue Kraft wird durch die vector definiert, die wir aus den Komponenten xforce/yforce im Verhältnis zu diesen Koordinaten erhalten - was einfach bedeutet, dass die Kraft als der Abstand von xlocal/ylocal zu xforce/yforce in Newton berechnet wird, und die Richtung ist der Winkel, den wir von xlocal/ylocal zu xforce/yforce erhalten. Das mag etwas verwirrend erscheinen, aber Sie müssen sich nur die Abbildung ansehen, um zu sehen, wie das funktioniert, und es erspart Ihnen einige komplizierte Rechenoperationen!

HINWEIS: Die Physikwelt verwendet Meter für alle Berechnungen, und das Verhältnis von Pixeln pro Meter wird im Raumeditor oder mit physics_world_create().

Syntax:

physics_apply_local_force(xlocal, ylocal, xforce, yforce)

ArgumentTyp Beschreibung
xlocal The x coordinate relative to the origin where the force will be applied
ylocal The y coordinate relative to the origin where the force will be applied
xforce the x component of the force vector
yforce the y component of the force vector

Rückgabe:

Beispiel:

if keyboard_check(vk_right)
{
    physics_apply_local_force(-25, 0, -50, 0);
}

Die obige code übt eine Kraft von 50 Newton von links nach rechts relativ zum Ursprung auf die Instanz aus, während die rechte Pfeiltaste gedrückt wird.
