physics_apply_angular_impulse

Diese Funktion gibt einen Winkelimpuls an eine physikalisch aktivierte Instanz. Dieser Impuls setzt die Winkeldrehung um den angegebenen Betrag, wobei das aktuelle Drehmoment ignoriert wird, wodurch im Wesentlichen der "Spin" einer Vorrichtung festgelegt wird. Wenn du eine Winkelkraft auf eine Instanz mit Drehmoment anwenden willst, solltest du die Funktion physics_apply_torque().

Syntax:

physics_apply_angular_impulse(impulse)

ArgumentTyp Beschreibung
impulse The impulse (in Newton metres) to be applied

Rückgabe:

Beispiel:

if keyboard_check(vk_up)
{
    physics_apply_angular_impulse(10);
}

Die obige code gibt der physikalisch aktivierten Vorrichtung einen Winkelimpuls, wenn eine Taste gedrückt wird.
