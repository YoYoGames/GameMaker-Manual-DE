physics_particle_get_data_particle

Diese Funktion gibt verschiedene Informationen über ein einzelnes Teilchen in der Physiksimulation zurück, indem sie die angegebene flags überprüft. Der Partikelindex (seine ID) ist derjenige, der von der Funktion physics_particle_create()zurückgegeben wurde, und das verwendete buffer muss zuvor mit der Funktion buffer_create(). Es sollte vom Typ "grow" sein, mit einer Größe, die ungefähr der der erwarteten Rückgabedaten entspricht. Die flags selbst werden mit den unten angegebenen Konstanten gesetzt, und Sie würden die bitweise oder "|" verwenden, um einen einzelnen Flag-Wert zu erstellen, um die gewünschten Informationen zu erhalten.

Konstante Beschreibung Datenart
phy_particle_data_flag_typeflags Der flags Wert für das Partikel. buffer_u32
phy_particle_data_flag_position Die x- und y-Position des Partikels. 2 x buffer_f32
phy_particle_data_flag_velocity Die horizontale und vertikale Geschwindigkeit. 2 x buffer_f32
phy_particle_data_flag_colour Die Farbe und der Alphawert (hexadezimal). buffer_f32
phy_particle_data_flag_category Die Partikelkategorie (wie bei der Erstellung des Partikels definiert). buffer_u32

Syntax:

physics_particle_get_data_particle(ind, buffer, flags)

Argument Typ Beschreibung
ind Physik Teilchen-ID The index (ID) of the particle to get the data from.
buffer Puffer-ID The (previously created) buffer to use to store the data.
flags Physik Partikeldaten Flagge(n)Konstante(n) The flags to use to extract data about specific particle types.

Rückgabe:

N/A

Beispiel:

var part = physics_particle_create(flags, x, y, x_vel, y_vel, c_white, 1, 1)
var flags = phy_particle_data_flag_position | phy_particle_data_flag_velocity;
var buffer = buffer_create(16, buffer_grow, 4);
physics_particle_get_data_particle(part, buffer, flags);
px = buffer_read(buffer, buffer_f32);
py = buffer_read(buffer, buffer_f32);
pvelx = buffer_read(buffer, buffer_f32);
pvely = buffer_read(buffer, buffer_f32);
buffer_delete(buffer);

Die obige code ermittelt die Position und Geschwindigkeit des durch die Variable "part" indizierten Partikels und speichert die Daten in einer Reihe von Variablen.
