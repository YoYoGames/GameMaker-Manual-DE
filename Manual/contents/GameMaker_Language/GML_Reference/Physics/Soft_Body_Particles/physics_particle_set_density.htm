physics_particle_set_density

Mit dieser Funktion können Sie die Dichte der Partikel in einer Physiksimulation einstellen. Die Einstellung der Dichte eines Partikels wirkt sich direkt darauf aus, wie viel Trägheit es hat und wie es auf Kollisionen reagiert. Wenn Sie also ein kleines Teilchen mit einer hohen Dichte erstellen, wird es eine sehr große Masse haben, aber wenn Sie ein großes Teilchen mit einer niedrigen Dichte definieren, wird es eine viel kleinere Masse haben. Diese Funktion ist global, d.h. sie ändert die Dichte nicht nur für neue Partikel, die nach der der Änderung erzeugt werden, sondern auch für die bereits in der Simulation vorhandenen.

Syntax:

physics_particle_set_density(density)

ArgumentTyp Beschreibung
density The density of the particle fixture.

Rückgabe:

Beispiel:

physics_particle_set_radius(15);
physics_particle_set_density(0.5);
physics_particle_set_damping(1);
physics_particle_set_gravity_scale(1);

Mit der obigen code werden die Basiseigenschaften für alle Partikel in der Simulation festgelegt.
