physics_particle_group_get_inertia

Mit dieser Funktion können Sie die Trägheit einer ganzen Gruppe von Partikeln abrufen. Der Wert group ist derjenige, der zurückgegeben wurde, als Sie die Gruppe von Partikeln mit der Funktion physics_particle_group_end()erstellt haben, und die Funktion gibt einen Wert zurück, der dem kombinierten Wert der aktuell eingestellten flags entspricht.

Syntax:

physics_particle_group_get_inertia(group)

Argument Typ Beschreibung
group Physik Teilchengruppen-ID The particle group to get.

Rückgabe:

Real

Beispiel:

i = physics_particle_group_get_inertia(group1);

Die obige code ermittelt die Trägheit der in der Variablen "group1" indizierten Partikelgruppe und speichert sie in einer Variablen.
