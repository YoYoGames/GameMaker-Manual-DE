physics_particle_set_category_flags

Mit dieser Funktion können Sie die Partikel flags für eine Kategorie von Partikeln ändern. Der Wert category ist derjenige, den Sie in der Funktion physics_particle_create() oder (wenn Sie eine Gruppe erstellt haben) physics_particle_group_begin(), während die flags der Rückgabewert einer Kombination der folgenden Konstanten ist:

Konstante Beschreibung
phy_particle_flag_water Die Standardeigenschaften für einen Weichkörperpartikel.
phy_particle_flag_zombie Ein Zombiepartikel ist ein Partikel, der in einem einzigen Schritt zerstört wird, während alle anderen auf diese Weise gekennzeichnet sind.
phy_particle_flag_wall Damit wird das Partikel als statisch definiert, was es zu einem unbeweglichen Objekt in der Physiksimulation macht, da es in einer festen Position bleibt, egal was mit ihm kollidiert. Sie sollten dieses Flag verwenden, anstatt die Dichte auf 0 zu setzen.
phy_particle_flag_spring Federpartikel erzeugen den Effekt, dass sie wie durch eine Feder miteinander verbunden sind. Teilchen, die mit dieser Flagge erzeugt werden, sind paarweise "verbunden", wobei jedes Teilchen mit demjenigen verbunden ist, das ihm zum Zeitpunkt der Erzeugung am nächsten war. Einmal gepaart, wechseln die Teilchen ihre "Partner" nicht mehr, und je weiter eine äußere Kraft sie voneinander wegzieht, desto größer ist die Kraft, mit der sie kollidieren, wenn diese äußere Kraft wegfällt. Beachten Sie, dass die Verbindung zwischen gepaarten Teilchen nicht abreißt, egal wie weit sie sich voneinander entfernen.
phy_particle_flag_elastic Elastische Partikel verformen sich und können auch abprallen, wenn sie mit anderen starren Körpern in der Physiksimulation zusammenstoßen.
phy_particle_flag_viscous Ein zähflüssiges Teilchen ist ein Teilchen, das "klebrig" ist, wie Öl. Viskose Partikel verklumpen und kleben stärker zusammen.
phy_particle_flag_powder Puderpartikel erzeugen einen Streueffekt, wie man ihn bei Sand oder Staub beobachten kann.
phy_particle_flag_tensile Zugpartikel werden verwendet, um den Effekt der Oberflächenspannung oder die gespannte Krümmung der Oberfläche einer Flüssigkeit zu erzeugen. Sie können zum Beispiel verwendet werden, um die Oberflächenspannung eines Wassertropfens zu erzeugen. Sobald die Spannung gebrochen ist, prallen die Teilchen ab, als ob sie elastisch wären, ziehen sich aber auch weiterhin gegenseitig an. Infolgedessen neigen die Teilchen dazu, beim Aufprall Cluster zu bilden.
phy_particle_flag_colourmixing Farbmischende Partikel nehmen einen Teil der Farbe anderer Partikel an, mit denen sie kollidieren. Wenn nur eines der beiden kollidierenden Teilchen ein farbmischendes Teilchen ist, behält das andere Teilchen seine Farbe vor der Kollision.

Diese flags verwenden Bit-Maskierung, um einen endgültigen Ausgabewert zu erzeugen, der dann geprüft wird, um die verschiedenen grundlegenden Eigenschaften der Partikelkategorie einzustellen (wobei die Basiseigenschaft immer die von phy_particle_flag_water ist). Wenn Sie z. B. eine viskose Flüssigkeit mit Oberflächenspannung simulieren wollen, maskieren Sie die entsprechenden Bits bitweise oder mit "|", wie im folgenden Beispiel gezeigt. Auf diese Weise können Sie für jede erstellte Partikelkategorie unterschiedliche Eigenschaften (außer den globalen Eigenschaften) festlegen.

Syntax:

physics_particle_set_category_flags(category, flags)

Argument Typ Beschreibung
category Real The particle category to set.
flags Physik Teilchen Flagge Konstante(n) The flags to set on the particle category.

Rückgabe:

N/A

Beispiel:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_set_category_flags(1, flags);

Die obige code erstellt eine Variable, um den Wert flags zu speichern, und verwendet sie dann, um den Wert flags aller Partikel zu setzen, die mit dem Kategoriewert 1 erstellt wurden.
