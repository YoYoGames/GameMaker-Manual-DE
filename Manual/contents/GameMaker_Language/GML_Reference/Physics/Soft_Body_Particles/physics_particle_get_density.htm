physics_particle_get_density

Mit dieser Funktion können Sie herausfinden, wie hoch die Stromdichte der Teilchen in der Physiksimulation ist (Sie können diesen Wert mit physics_particle_set_density()).

Syntax:

physics_particle_get_damping()

Rückgabe:

Beispiel:

if (physics_particle_get_density() < 1)
{
    physics_particle_set_density(physics_particle_get_density() + 0.01);
}

Die obige code prüft den Wert der Stromdichte für alle Partikel im System, und wenn er kleiner als 1 ist, wird er um 0,01 erhöht.
