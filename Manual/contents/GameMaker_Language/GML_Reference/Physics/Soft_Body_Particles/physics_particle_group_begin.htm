physics_particle_group_begin

Mit dieser Funktion können Sie eine Gruppe von Partikeln in einem Raum erstellen. Eine Partikelgruppe ist eine Gruppe, in der alle Partikel, die zur Erzeugung einer bestimmten Form notwendig sind, auf einmal erzeugt werden. Dies ermöglicht die Erzeugung von simulierten Weichkörpern mit verschiedenen Eigenschaften, die durch die verwendete flags festgelegt werden. Die "Flags" sind der Rückgabewert einer Kombination aus den Partikelkonstanten (siehe die Funktion physics_particle_create() ), während die "groupFlags" eine Kombination aus den folgenden Gruppenkonstanten sind (oder 0 für keine flags):

Physik Teilchengruppe Flagge Konstante
Konstante Beschreibung
phy_particle_group_flag_solid Eine feste Partikelgruppe verhindert, dass sich andere Einrichtungsgegenstände in ihr festsetzen. Sollte etwas in sie eindringen, so drückt die Feststoffgruppe das anstößige Teil zurück an ihre Oberfläche, wodurch eine Feststoffgruppe eine besonders starke Abstoßungskraft besitzt.
phy_particle_group_flag_rigid Starre Partikelgruppen sind solche, deren Form sich nicht verändert, selbst wenn sie mit anderen Einrichtungsgegenständen zusammenstoßen.

Diese Gruppe flags verwendet Bit-Maskierung, um einen endgültigen Ausgabewert zu erzeugen, der dann überprüft wird, um die verschiedenen grundlegenden Eigenschaften der Gruppe einzustellen, abgesehen von denen, die durch die Partikel flags eingestellt werden. Sie können keine, eine oder beide dieser Konstanten verwenden, indem Sie die entsprechenden Bits bitweise oder mit "|" maskieren (wie im folgenden Beispiel gezeigt).

Neben flags können Sie auch die Position im Raum festlegen, an der die Partikelgruppe erstellt werden soll (die angegebene Position ist der Massenschwerpunkt der fertigen Gruppe), die anfängliche horizontale und vertikale Geschwindigkeit sowie die Winkelgeschwindigkeit der Gruppe, so dass Sie bewegliche, rotierende Gruppen erstellen können.

Sie können auch die anfängliche Farbe und den Alphawert für alle Partikel in der Gruppe festlegen, wobei die Werte beim Zeichnen der Partikel mit der Funktion physics_particle_draw(). Danach legen Sie die Kohäsionsstärke der Partikel in der Gruppe fest, was eine andere Art ist zu sagen, wie stark sie aneinander "kleben". Die Stärke kann ein Wert zwischen 0 und 1 sein, wobei 0 für keinen Zusammenhalt und 1 für maximalen Zusammenhalt steht.

Schließlich musst du die Kategorie für deine Partikelgruppe angeben. Dieser Wert ist ein willkürlicher ganzzahliger Wert, den du jeder Partikelgruppe gibst, die ähnliche Eigenschaften haben soll, und er kann verwendet werden, um später flags für diese spezifische Kategorie sowie eine Reihe anderer Dinge einzustellen.

HINWEIS: Sie können jeden ganzzahligen Wert für die Kategorie verwenden , mit Ausnahme von 0, der von der Simulation reserviert ist und für die Auswahl aller Kategorien in anderen Funktionen (z. B. für das Zeichnen) verwendet wird.

Es ist wichtig zu beachten, dass der Aufruf dieser Funktion die Gruppe im Physikraum nicht erstellt. Diese Funktion startet lediglich den Prozess und legt die Basis-Partikel-Eigenschaften für die Gruppe fest. Um die Partikelgruppe tatsächlich zu erstellen, müssen Sie ihr eine Form geben (siehe das Beispiel unten) und dann die Funktion physics_particle_group_end()aufrufen, woraufhin die Gruppe erstellt wird.

HINWEIS: Der von Ihnen eingestellte Partikeltyp flags beeinflusst auch den Zusammenhalt der Partikelgruppe. Wenn Sie die Partikel z. B. als Wassertyp kennzeichnen, bricht die Gruppe auseinander, wenn sie mit einem anderen Gerät kollidiert, während sie ihren Zusammenhalt bei einer Kollision beibehält, wenn Sie die Partikel als Federtyp kennzeichnen.

Syntax:

physics_particle_group_begin(flags, groupflags, x, y, ang, xv, yv, ang_velocity, col, alpha, strength, category)

Argument Typ Beschreibung
flags Physik Teilchen Flagge Konstante(s) The flags to set on the particle.
groupflags Physik Teilchengruppe Flagge Konstante(s) The group flags to set on the particle group (see table at the top).
x Real The x position to create the particle group at.
y Real The y position to create the particle group at.
ang Real The angle (in degrees) to rotate the group to on create.
xv Real The initial horizontal velocity.
yv Real The initial vertical velocity.
ang_velocity Real The initial angular velocity.
col Farbe The base colour to use for the particle group.
alpha Real The base alpha to use for the particle group.
strength Real Set the cohesion strength between particles in the group.
category Real The user defined category to which the group belongs.

Rückgabe:

N/A

Beispiel:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

Die obige code speichert die flags für den Partikeltyp und die Partikelgruppeneigenschaften in Variablen und verwendet diese, um eine kreisförmige Partikelgruppe mit einem Radius von 100px an der Mausposition zu erstellen.
