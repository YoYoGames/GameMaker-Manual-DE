physics_particle_group_join

Mit dieser Funktion können Sie zwei Partikelgruppen miteinander verbinden. Die verbundenen Gruppen verhalten sich dann so, als ob sie beide Teil einer einzigen Einheit wären. Die Gruppen sollten mit überlappenden Kanten erstellt worden sein, denn wenn sie sich nicht bereits berühren, werden sie nicht verbunden. Die Funktion nimmt die eindeutigen Gruppen-IDs der zu verbindenden Gruppen, wie sie von der Funktion physics_particle_group_end()zurückgegeben werden, und Sie können die Funktion beliebig oft für eine einzelne Gruppe verwenden, um verschiedene Weichkörper miteinander zu verbinden.

Syntax:

physics_particle_group_join(to, from)

Argument Typ Beschreibung
to Physik Teilchengruppen-ID The first particle group to join.
from Physik Teilchengruppen-ID The second particle group to join.

Rückgabe:

N/A

Beispiel:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x- 45, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(50);
var g1 = physics_particle_group_end();
physics_particle_group_begin(flags, groupflags, mouse_x + 45, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(50);
var g2 = physics_particle_group_end();
physics_particle_group_join(g1, g2);

Die obige code erstellt zwei kreisförmige Partikelgruppen und fügt sie zusammen.
