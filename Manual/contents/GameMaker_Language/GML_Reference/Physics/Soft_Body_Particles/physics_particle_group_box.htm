physik_teilchen_gruppen_box

Diese Funktion legt die Form der Partikelgruppe fest, die gerade erstellt wird. Sie müssen zuerst die Gruppendefinition mit der Funktion physics_particle_group_begin()beginnen und dann mit dieser Funktion die ungefähre halbe Breite und halbe Höhe der Gruppe in Pixeln festlegen - ungefähr, weil die genaue Breite und Höhe von der Größe der Basispartikel abhängt, wie sie mit der Funktion physics_particle_set_radius() Funktion, da die Physiksimulation versucht, so viele Partikel wie möglich in die definierte Form "einzupassen". Schließlich müssen Sie die Funktion physics_particle_group_end() aufrufen, um die Gruppe von Partikeln im Raum zu erstellen.

Syntax:

physics_particle_group_box(halfWidth, halfHeight)

ArgumentTyp Beschreibung
halfWidth The half width of the box.
halfHeight The half height of the box.

Rückgabe:

N/A

Beispiel:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_box(100, 100);
mLastGroup = physics_particle_group_end();

Die obige code speichert die flags für den Partikeltyp und die Partikelgruppeneigenschaften in Variablen und verwendet diese dann, um eine rechteckige Partikelgruppe mit Seiten von 200px an der Mausposition zu erstellen.
