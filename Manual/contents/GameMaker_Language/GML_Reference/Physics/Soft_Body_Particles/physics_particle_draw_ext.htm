physik_teilchen_zeichnen_ext

Diese Funktion zeichnet eine sprite an der Position aller Partikel, die denselben benutzerdefinierten Kategoriewert haben und deren flags mit den für die Maske festgelegten Werten übereinstimmt. Der Maskenwert wird auf die gleiche Weise definiert, wie Sie flags bei der Erstellung der Partikel definieren würden (siehe die Funktionen physics_particle_create() für ein Beispiel), d. h. mit Hilfe der bitweisen oder "|"-Methode werden die entsprechenden Bits für jedes Flag maskiert.

HINWEIS: Diese Funktion zeichnet alle Partikel in der Physiksimulation mit flags, die mit denen der Maske übereinstimmen, unabhängig davon, ob es sich um einzelne Partikel oder Gruppen handelt, wenn die Benutzerkategorie dieselbe ist.

Der gezeichnete Partikel ignoriert die Farb- und Alphawerte, die bei seiner Erstellung festgelegt wurden, und wird stattdessen mit den in dieser Funktion angegebenen Parametern Skala, Winkel, Farbe und Alpha gezeichnet. Beachte, dass dies nur zum Zeichnen einer sprite an der Partikelposition dient und das Verhalten des Partikels in keiner Weise beeinflusst.

Syntax:

physics_particle_draw_ext(typemask, category, sprite, subimg, xscale, yscale, ang, col, alpha)

Argument Typ Beschreibung
typemask Physik Teilchen Flagge Konstante(n) The mask to compare the particles flags with.
category Real The user defined category to draw (use 0 to draw all categories).
sprite Sprite-Asset The sprite to use for each particle.
subimg Real The image_index des zu zeichnenden Sprites.
xscale Real The scale factor to apply to the sprite along the x axis.
yscale Real The scale factor to apply to the sprite along the y axis.
ang Real The angle (in degrees) to draw the sprite at.
col Farbe The colour to blend with the sprite (default is c_white).
alpha Real The alpha value to use when drawing (0 - 1).

Rückgabe:

N/A

Beispiel:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_draw_ext(flags, 2, sprBlob, 0, 2, 2, 0, c_white, 0.5);

Die obige code wird eine sprite für alle Partikel in der benutzerdefinierten Kategorie "2" zeichnen, die die gleiche flags wie die angegebene Maske haben, skaliert um 2 entlang beider Achsen und mit einem Alpha von 0,5.
