physics_particle_group_polygon

Diese Funktion legt die Form der Partikelgruppe fest, die gerade erstellt wird. Sie müssen zuerst die Gruppendefinition mit der Funktion physics_particle_group_begin()begonnen haben, und dann würden Sie diese Funktion verwenden, um GameMaker mitzuteilen, dass Sie die ungefähren Punkte hinzufügen werden, die zur Definition der Polygonform verwendet werden - ungefähr, weil die genaue Form von der Größe der Basispartikel abhängt, die durch der Funktion physics_particle_set_radius() Funktion definiert ist, da die Physiksimulation versucht, so viele Partikel wie möglich in die definierte Form "einzupassen".

Sie würden dann die Punkte des Polygons definieren, indem Sie die Funktion physics_particle_group_add_point()aufrufen und für jeden Punkt die x/y-Position im Raum festlegen (mindestens 3, höchstens 8, und sie können als konkav definiert werden, aber die Punkte innerhalb der Form werden ignoriert). Schließlich müssen Sie physics_particle_group_end() aufrufen, um die Gruppe von Partikeln im Raum zu erstellen.

Syntax:

physics_particle_group_polygon()

Rückgabe:

Beispiel:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_polygon();
physics_particle_group_add_point(200, 200);
physics_particle_group_add_point(300, 300);
physics_particle_group_add_point(100, 300);
mLastGroup = physics_particle_group_end();

Die obige code speichert die flags für den Partikeltyp und die Partikelgruppeneigenschaften in Variablen und verwendet diese, um eine Polygonpartikelgruppe mit drei Seiten an der Mausposition zu erstellen.
