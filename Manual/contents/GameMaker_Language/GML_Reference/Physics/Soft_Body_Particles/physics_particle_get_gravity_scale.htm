physics_particle_get_gravity_scale

Mit dieser Funktion können Sie herausfinden, wie hoch der aktuelle Schwerkraft-Skalierungsfaktor für Partikel in der Physiksimulation ist (Sie können diesen Wert mit physics_particle_set_gravity_scale()).

Syntax:

physics_particle_get_gravity_scale()

Rückgabe:

Beispiel:

physics_particle_set_gravity_scale(physics_particle_get_gravity_scale() + 0.1);

Unter code wird die Schwerkraft-Skala eingestellt.
