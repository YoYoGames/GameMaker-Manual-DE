physics_particle_group_get_mass

Mit dieser Funktion können Sie die Masse einer ganzen Gruppe von Partikeln abrufen. Der Wert group ist derjenige, der zurückgegeben wurde, als Sie die Gruppe von Partikeln mit der Funktion physics_particle_group_end().

Syntax:

physics_particle_group_get_mass(group)

Argument Typ Beschreibung
group Physik Teilchengruppen-ID The particle group to get.

Rückgabe:

Real

Beispiel:

var _mass = physics_particle_group_get_mass(group1);

Die obige code ermittelt die Masse der in der Variablen "group1" indizierten Partikelgruppe und speichert sie in einer Variablen.
