physik_teilchen_zeichnen

Diese Funktion zeichnet ein sprite an der Position aller Partikel, die denselben benutzerdefinierten Kategoriewert haben und deren flags mit den für die Maske festgelegten Werten übereinstimmt. Der Maskenwert wird auf die gleiche Weise definiert, wie Sie flags beim Erstellen des Partikels definieren würden (siehe die Funktion physics_particle_create() für ein Beispiel), d.h. mit Hilfe der bitweisen oder "|"-Methode werden die entsprechenden Bits für jedes Flag maskiert.

HINWEIS: Diese Funktion zeichnet alle Partikel in der Physiksimulation mit flags, die mit denen der Maske übereinstimmen, unabhängig davon, ob es sich um einzelne Partikel oder Gruppen handelt, wenn die Benutzerkategorie dieselbe ist.

Das gezeichnete Partikel verwendet die Farb- und Alphawerte, die Sie bei der Erstellung festgelegt haben, und dreht sich mit dem Drehimpuls des Partikels. Wenn Sie jedoch das sprite skalieren oder andere Eigenschaften des gezeichneten Partikels ändern möchten, sollten Sie die Funktion physics_particle_draw_ext().

Syntax:

physics_particle_draw(typemask, category, sprite, subimg)

Argument Typ Beschreibung
typemask Physik Teilchen Flagge Konstante(n) The mask to compare the particles flags with.
category Real The user defined category to draw (use 0 to draw all categories).
sprite Sprite-Asset The sprite to use for each particle.
subimg Real The image_index des zu zeichnenden Sprites.

Rückgabe:

N/A

Beispiel:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_draw(flags, 2, sprBlob, 0);

Die obige code wird eine sprite für alle Partikel in der benutzerdefinierten Kategorie "2" zeichnen, die die gleiche flags wie die angegebene Maske haben.
