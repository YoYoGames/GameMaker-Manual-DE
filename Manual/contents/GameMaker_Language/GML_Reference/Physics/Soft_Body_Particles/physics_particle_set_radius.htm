physik_teilchen_setzen_radius

Mit dieser Funktion können Sie den Radius (in Pixeln) für die Partikel in einer Physiksimulation festlegen. Diese Funktion ist global, d.h. sie ändert den Radius nicht nur für neue Partikel, die nach der Änderung erstellt werden, sondern auch für die bereits in der Simulation vorhandenen die bereits in der Simulation vorhanden sind.

Syntax:

physics_particle_set_radius(radius)

ArgumentTyp Beschreibung
radius The radius (in pixels) of the particle fixture.

Rückgabe:

Beispiel:

physics_particle_set_radius(15);
physics_particle_set_density(0.5);
physics_particle_set_damping(1);
physics_particle_set_gravity_scale(1);

Mit der obigen code werden die Basiseigenschaften für alle Partikel in der Simulation festgelegt.
