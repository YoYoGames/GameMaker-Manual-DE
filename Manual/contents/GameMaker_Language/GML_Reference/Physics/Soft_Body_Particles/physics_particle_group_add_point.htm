physics_particle_group_add_point

Diese Funktion setzt einen Punkt im Raum, um die Form eines Polygons zu definieren, das zur Erzeugung einer Gruppe von Weichkörperpartikeln verwendet wird. Sie müssen dem GameMaker zuvor signalisiert haben, dass Sie eine Polygonform mit der Funktion physics_particle_group_polygon() eine Polygonform definieren und dann mit dieser Funktion die einzelnen Punkte des Polygons festlegen. Sie müssen bei der Definition der Polygonform mindestens drei Punkte angeben, können aber auch bis zu acht Punkte angeben, und die Funktion erlaubt die Definition von konkaven Polygonen. Wenn Sie jedoch ein Polygon mit Hohlräumen erzeugen, werden die Punkte darin ignoriert und eine konvexe Form für die Partikelgruppe erstellt.

Syntax:

physics_particle_group_add_point(x, y)

ArgumentTyp Beschreibung
x The x position in the room for the point.
y The y position in the room for the point.

Rückgabe:

Beispiel:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_polygon();
physics_particle_group_add_point(200, 200);
physics_particle_group_add_point(300, 300);
physics_particle_group_add_point(100, 300);
mLastGroup = physics_particle_group_end();

Die obige code speichert die flags für den Partikeltyp und die Partikelgruppeneigenschaften in Variablen und verwendet diese, um eine Polygonpartikelgruppe mit drei Seiten an der Mausposition zu erstellen.
