physics_particle_delete_region_poly

Mit dieser Funktion können Sie alle Partikel, die innerhalb der Grenzen des definierten Polygonbereichs liegen, aus der Physiksimulation im aktuellen Raum löschen (entfernen). Die Funktion nimmt eine (zuvor erstellte) ds_list() die die x/y-Position jedes Punktes des Polygons enthält, wobei die geradzahligen Positionen in der Liste die x-Koordinaten und die ungeradzahligen Positionen die y-Koordinaten sind, d.h. für ein Dreieck würde Ihre Liste sechs Einträge haben, wobei die Einträge 0, 2 und 4 die x-Koordinaten und 1, 3 und 5 die y-Koordinaten sind. Das Polygon wird dann verwendet, um alle Partikel zu löschen, die in den definierten Bereich fallen.

Beachten Sie, dass das definierte Polygon mindestens drei und höchstens 8 Punkte haben muss.

Syntax:

physics_particle_delete_region_poly(pointList)

ArgumentTyp Beschreibung
pointList A ds_list of points to use to create the polygon.

Rückgabe:

Beispiel:

var list = ds_list_create(p_list);
for (var i = 0; i < 5; i ++;)
{
    ds_list_add(p_list, mx[i]);
    ds_list_add(p_list, my[i]);
}
physics_particle_delete_region_poly(p_list);
ds_list_destroy(p_list);

Die obige code löscht alle Partikel, die sich in dem polygonalen Bereich befinden, der durch die zur DS List hinzugefügten Punkte definiert ist.
