physics_particle_get_group_flags

Mit dieser Funktion können Sie die Gruppe flags für eine Gruppe von Partikeln abrufen. Der Wert group ist derjenige, der zurückgegeben wurde, als Sie die Gruppe von Partikeln mit der Funktion physics_particle_group_end()erstellt haben, und die Funktion gibt einen Wert zurück, der dem kombinierten Wert der aktuell eingestellten flags entspricht.

Syntax:

physics_particle_get_group_flags(group)

Argument Typ Beschreibung
group Physik Teilchengruppen-ID The particle group to get.

Rückgabe:

Real

Beispiel:

var flags = physics_particle_get_group_flags(group, flags);
if flags != (phy_particle_group_flag_solid | phy_particle_group_flag_rigid)
{
    flags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
    physics_particle_set_group_flags(group, flags);
}

Die obige code erstellt eine Variable, in der der Wert flags gespeichert wird, und verwendet diese dann, um die flags der in der Variablen "group" indizierten Gruppe zu überprüfen. Wenn sie nicht übereinstimmen, wird die Gruppe mit diesen flags gesetzt.
