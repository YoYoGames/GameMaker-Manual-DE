physics_particle_set_max_count

Mit dieser Funktion wird die zulässige Gesamtzahl der Partikel in einer Physiksimulation festgelegt. Wenn Sie diesen Wert auf 0 setzen, dann gibt es keine Obergrenze für die erzeugten Partikel, und wenn Sie ihn auf einen anderen Wert setzen, dann werden keine weiteren Partikel mehr erzeugt, wenn die Gesamtanzahl der Partikel dem Wert entspricht. Beachten Sie, dass die Mindestanzahl von Partikeln, die Sie in einer Simulation erstellen können, 128 beträgt, so dass ein niedrigerer Wert keine Auswirkungen hat (es sei denn, er wird auf 0 gesetzt).

Syntax:

physics_particle_set_max_count(count)

ArgumentTyp Beschreibung
count The maximum number of particles to permit.

Rückgabe:

Beispiel:

physics_particle_set_max_count(500);

Mit der obigen code wird die Partikelobergrenze der Physiksimulation auf 500 gesetzt.
