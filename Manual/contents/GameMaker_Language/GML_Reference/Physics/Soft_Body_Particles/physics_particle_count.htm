physik_teilchen_zahl

Diese Funktion gibt die Anzahl der Partikel zurück, die in einem physikfähigen Raum aktiv sind.

Syntax:

physics_particle_count()

Rückgabe:

Beispiel:

if (physics_particle_count() < physics_particle_get_max_count())
{
    physics_particle_create(0, x, y, 0, 0, c_white, 1, 1)
}

Das obige code prüft, ob weniger als die maximal zulässige Anzahl von Partikeln im Raum vorhanden ist, und erstellt gegebenenfalls ein weiteres.
