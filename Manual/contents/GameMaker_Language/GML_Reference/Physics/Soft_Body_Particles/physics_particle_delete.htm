physik_teilchen_loeschen

Mit dieser Funktion können Sie ein Partikel aus der Physiksimulation im aktuellen Raum löschen (entfernen). Die Funktion nimmt die eindeutige ID des zu löschenden Partikels, wie sie von der Funktion physics_particle_create().

Syntax:

physics_particle_group_delete(ind)

Argument Typ Beschreibung
ind Physik Teilchen-ID The index (ID) of the particle to delete.

Rückgabe:

N/A

Beispiel:

physics_particle_delete(part);

Die obige code löscht das Partikel mit der in der Variablen "part" gespeicherten ID aus der Simulation.
