physik_teilchen_gruppe_ende

Diese Funktion wird verwendet, um die Definition einer Partikelgruppenform zu beenden. Der Aufruf dieser Funktion erzeugt die Partikel innerhalb der gegebenen Formparameter und gibt außerdem einen ID-Wert zurück, der gespeichert und in weiteren Funktionen für Interaktionen mit der Partikelgruppe verwendet werden kann.

Syntax:

physics_particle_group_end()

Rückgabe:

Physics Particle Group ID

Beispiel:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

Die obige code speichert die flags für den Partikeltyp und die Partikelgruppeneigenschaften in Variablen und verwendet diese, um eine kreisförmige Partikelgruppe mit einem Radius von 100px an der Mausposition zu erstellen. Die ID für die erstellte Gruppe wird in der Variablen "mLastGroup" gespeichert.
