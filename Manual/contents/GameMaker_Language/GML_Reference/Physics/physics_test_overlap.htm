physik_prüfung_überlappung

Diese Funktion kann verwendet werden, um zu prüfen, ob sich ein physischer Körper (d.h. die Halterung einer Instanz) überschneidet oder überschneiden wird, wenn er gedreht und an einer bestimmten Position im Raum platziert wird. Der "angle" argument ist der Drehwinkel, den die aufrufende Instanz an der zu prüfenden Position hat (oder haben wird), und der "obj" argument kann entweder eine einzelne Instanz-ID und ein Objektindex sein oder die Schlüsselwörter alle oder andere.

Syntax:

physics_test_overlap(xpos, ypos, angle, obj);

Argument Typ Beschreibung
xpos Real The x position in the room to check
ypos Real The y position in the room to check
angle Real The angle to check (of the calling instance)
obj Objekt Asset- oder Instanz-ID The object to check for

Rückgabe:

Boolean

Beispiel:

if physics_test_overlap(x+20, y-35, 0, obj_Bomb)
{
    alarm[0] = room_speed;
    ignited = true;
}

Die obige code prüft eine Position auf eine Überschneidung mit einer physikalischen Vorrichtung, und wenn es eine gibt, setzt sie eine Variable und einen Alarm.
