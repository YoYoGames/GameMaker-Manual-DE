physik_gelenke_aktivieren_motor

Wenn Sie ein Gelenk mit einem Motor(prismatisch oder drehbar) haben, möchten Sie vielleicht switch den Motor abhängig von Variablen und Bedingungen im Spiel ein- oder ausschalten können. Dazu müssen Sie den Index des Gelenks zuvor in einer Variablen gespeichert haben und dann können Sie switch den Motor ein- oder ausschalten, indem Sie diese Funktion verwenden und den "Motor" argument auf true oder false setzen.

Syntax:

physics_joint_enable_motor(joint, motor)

Argument Typ Beschreibung
joint Physik Gemeinsame ID The joint that you wish to enable or disable the motor on
motor Boolesche Whether you wish to turn the motor on (true) or off (false)

Rückgabe:

N/A

Beispiel:

var t_fix = physics_fixture_create();
physics_fixture_set_circle_shape(t_fix, sprite_get_width(sprite_index) / 2);
var o_id=instance_create_layer(x+300, y, "Instances", obj_Door);
physics_fixture_bind(t_fix, id);
physics_fixture_bind(t_fix, o_id);
perma_joint = physics_joint_revolute_create(id, o_id, x+25, y, -90, 90, 1, 10, 2, 0, 0);
physics_joint_enable(perma_joint, 1);
physics_fixture_delete(t_fix);

Die obige code erstellt und definiert ein neues Fixture und erstellt dann eine Instanz von "obj_Door", wobei das erstellte Fixture mit den beiden Instanzen verbunden wird. Die beiden Instanzen werden dann durch ein Drehgelenk ohne Motor verbunden, wobei die Winkel auf +/- 90 Grad begrenzt sind, und der Gelenkindex wird in der Variablen "perma_joint" gespeichert. Anschließend wird der Motor über die Variable switch eingeschaltet, bevor die Halterung aus dem Speicher gelöscht wird, da sie nicht mehr benötigt wird.

Rücken: Gelenke
Nächste: physics_joint_get_value
