physics_joint_delete

Sobald zwei Instanzen mit physikalischen Darstellungen durch eine Verbindung gebunden sind, kann diese jederzeit wieder gelöscht werden. Normalerweise geschieht dies automatisch, wenn eine der beiden Instanzen zerstört wird, oder wenn der Raum endet, aber es gibt Zeiten, in denen du dies manuell tun möchtest. In solchen Fällen können Sie diese Funktion verwenden.

Es sollte auch beachtet werden, dass immer dann, wenn eine Instanz, die Teil eines Zahnradgelenks ist, zerstört wird, das Zahnradgelenk mit dieser Funktion vor allen anderen Instanzen, die an der Bildung des Zahnrads beteiligt sind, gelöscht werden sollte (die verbleibenden Gelenke werden jedoch automatisch gelöscht), zum Beispiel im Zerstörungsereignis der Instanz.

Syntax:

physics_joint_delete(joint)

Argument Typ Beschreibung
joint Physik Gemeinsame ID The index of the joint that you wish to delete

Rückgabe:

N/A

Beispiel:

var reactionForceX, reactionForceY, reactionForce;
if (shipJoint)
{
    _reaction_force_x = physics_joint_get_value(ship_joint, phy_joint_reaction_force_x);
    _reaction_force_y = physics_joint_get_value(ship_joint, phy_joint_reaction_force_y);
    _reaction_force = sqrt((_reaction_force_x + _reaction_force_x) + (_reaction_force_y + _reaction_force_y));
    if (_reaction_force > 2)
    {
        physics_joint_delete(ship_joint);
        ship_joint = -1;
    }
}

Die obige code prüft, ob die Variable "shipJoint" einen Gelenkindex enthält, und wenn ja, berechnet sie die Kraft, die auf dieses Gelenk wirkt, unter Verwendung der beiden Konstanten. Wenn die Gesamtkraft größer als 2 ist, wird das Gelenk schließlich gelöscht.

Rücken: Gelenke
