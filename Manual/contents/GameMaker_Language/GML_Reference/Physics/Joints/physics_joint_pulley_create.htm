physik_gelenk_pulley_erstellen

Das Flaschenzuggelenk wird verwendet, um zwei Instanzen innerhalb der physikalischen Welt so zu verbinden, dass die Bewegung der einen Instanz die Bewegung der anderen direkt beeinflusst. Diese Gelenke werden zunächst an zwei Punkten im Weltraum verankert, von denen jeder mit einer Instanz in deren Massenschwerpunkt verbunden ist. Dies kann geändert werden, indem die Koordinaten des lokalen Ankers relativ zum Ursprung der Instanz gesetzt werden, was bedeutet, dass die tatsächliche Verbindung von der Instanz zum Ankerpunkt der Physikwelt an einen anderen Ort als den Ursprung verlegt werden kann. Die folgende Abbildung veranschaulicht dies:

Wenn du dir inst2 ansiehst, kannst du sehen, dass sein lokaler Ankerpunkt um 20 Pixel nach "links" von seinem Ursprung verschoben wurde (Zur Erinnerung! Die weltweiten Ankerpunkte werden anhand der Raumkoordinaten definiert, während die lokalen Ankerpunkte relativ zum Ursprung der Instanz definiert werden). Du kannst auch ein Verhältnis für das Flaschenzug-Gelenk angeben, das der einen oder anderen Seite sagt, dass sie sich schneller bewegen soll, wobei ein Wert unter 1 die erste Instanz schneller bewegen lässt (z.B. 0,5 lässt sie sich doppelt so schnell bewegen) und ein Wert über 1 die zweite Instanz schneller bewegen lässt (z.B. 2 lässt sie sich doppelt so schnell bewegen).

Wie bei allen Gelenken, wenn Sie die "col"-Wert auf true dann die beiden Instanzen können interagieren und miteinander kollidieren, aber nur, wenn sie Kollisions-Ereignisse haben, aber wenn es auf false eingestellt ist, werden sie nicht kollidieren, egal was.

Syntax:

physics_joint_pulley_create(inst1, inst2, w_anchor1_x, w_anchor1_y, w_anchor2_x, w_anchor2_y, l_anchor1_x, l_anchor1_y, l_anchor2_x, l_anchor2_y, ratio, col)

Argument Typ Beschreibung
inst1 Instanz-ID The first instance to connect with the joint
inst2 Instanz-ID The second instance to connect with the joint
w_anchor1_x Real The x coordinate where the first part of joint is anchored, within the game world
w_anchor1_y Real The y coordinate where the first part of the joint is anchored, within the game world
w_anchor2_x Real The x coordinate where the second part of the joint is anchored, within the game world
w_anchor2_y Real The y coordinate where the second part of the joint is anchored, within the game world
l_anchor1_x Real The local x coordinate where joint 1 is anchored to the first instance
l_anchor1_y Real The local y coordinate where joint 1 is anchored to the first instance
l_anchor2_x Real The local x coordinate where joint 2 is anchored to the second instance
l_anchor2_y Real The local y coordinate where joint 2 is anchored to the second instance
ratio Real Set the velocity ratio between two instances
col Boolesche Whether the two instances can collide (true) or not (false)

Rückgabe:

Physics Joint ID

Beispiel:

var t_fix, inst1, inst2;
t_fix = physics_fixture_create();
physics_fixture_set_circle_shape(t_fix, 20);
inst1 = instance_create_layer(150, room_height - 90, "Instances", obj_Block);
inst2 = instance_create_layer(300, room_height - 90, "Instances", obj_block);
physics_fixture_bind(t_fix, inst1);
physics_fixture_bind(t_fix, inst2);
physics_joint_pulley_create(inst1, inst2, 150, room_height - 140, 300, room_height - 140, 0, 0, 0, 0, 2, 0);
physics_fixture_delete(t_fix);

Die obige code erstellt und definiert ein neues Fixture, erstellt zwei Instanzen von "obj_Block" und bindet dann dieses Fixture an diese beiden. Dann wird eine Seilzugverbindung zwischen diesen Instanzen definiert, ohne Versatz für die Verbindungen, mit einem Verhältnis von 2:1 (was bedeutet, dass inst2 sich schneller bewegt). Es kommt zu keinen Kollisionen zwischen den beiden Instanzen des Flaschenzugs.
