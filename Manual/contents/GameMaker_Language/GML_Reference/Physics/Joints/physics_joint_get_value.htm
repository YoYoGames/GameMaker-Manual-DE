physics_joint_get_value

Durch die Verwendung einer Reihe vordefinierter Konstanten können Sie GameMaker bitten, Ihnen eine Reihe von Informationen über den Zustand eines bestimmten Gelenks zu geben. Dies ist sehr nützlich, da es Ihnen die Möglichkeit gibt, Gelenke zu löschen oder das Verhalten einer Instanz zu ändern, je nachdem, was Sie gerade brauchen. Es gibt eine Reihe von Konstanten, die in dieser Funktion verwendet werden können und die Sie hier finden können: Physics Joint Constants, aber seien Sie sich bewusst, dass komplexe Berechnungen durchgeführt werden, wenn Sie diese aufrufen, daher sollten sie mit Vorsicht und nur bei Bedarf verwendet werden.

Wenn die Eigenschaft nicht vorhanden ist (wenn Sie z. B. ein Riemenscheibengelenk auf prismatische Verschiebung prüfen), wird der Rückgabewert 0 sein.

Syntax:

physics_joint_get_value(joint, value)

Argument Typ Beschreibung
joint Physik Gemeinsame ID The index of the joint that you wish to test
value Physik Gemeinsame Konstante The constant for the joint property that you wish to test

Rückgabe:

Real

Beispiel:

if shipJoint
{
    var reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
    var reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_y);
    var reactionForce = point_distance(0, 0, reactionForceX, reactionForceY);
    if (reactionForce > 2)
    {
        physics_joint_delete(shipJoint);
        shipJoint = -1;
    }
}

Die obige code prüft, ob die Variable "shipJoint" einen Gelenkindex enthält, und wenn ja, berechnet sie die Kraft, die auf dieses Gelenk wirkt, unter Verwendung der beiden Konstanten. Wenn die Gesamtkraft größer als 2 ist, wird das Gelenk schließlich gelöscht.

Rücken: Gelenke
Nächste: physics_joint_set_value
