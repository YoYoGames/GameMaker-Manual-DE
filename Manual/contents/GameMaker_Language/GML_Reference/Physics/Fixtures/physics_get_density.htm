physics_get_density

Wenn Sie ein Fixture mit einer Instanz binden, indem Sie physics_fixture_bind() wird eine "id" für das gebundene Gerät zurückgegeben. Mit dieser ID kann man jederzeit den Dichtewert des gebundenen Geräts(nicht des "Basis"-Einrichtung) jederzeit mit dieser Funktion abrufen.

Syntax:

physics_get_density(fixture)

ArgumentTyp Beschreibung
fixture the id of the bound fixture

Rückgabe:

Beispiel:

var density = physics_get_density(fix_id);
physics_set_density(fix_id, density - 0.1);

Die obige code ruft den Wert der Stromdichte für die gebundenen physikalischen Eigenschaften der Instanz ab und setzt sie dann auf einen anderen Wert.
