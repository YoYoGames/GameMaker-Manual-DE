physics_fixture_delete

Da ein Fixture einmal an eine Instanz gebunden wurde und diese Instanz die Eigenschaften des Fixtures beibehält, können wir das Fixture mit dieser Funktion aus dem Speicher entfernen, insbesondere wenn wir nicht mehr vorhaben, es wieder zu verwenden. Bitte beachten Sie, dass das Nichtentfernen von Fixtures nachdem sie nicht mehr benötigt werden, zu einem Speicherleck führen kann, das das Spiel verlangsamt und schließlich zum Absturz bringt.

Syntax:

physics_fixture_delete(fixture)

ArgumentTyp Beschreibung
fixture the fixture that is to be deleted from memory

Rückgabe:

Beispiel:

var fix, inst;
fix = physics_fixture_create();
physics_fixture_set_circle_shape(fix, 16);
physics_fixture_set_density(fix, 1.0);
inst = instance_create_layer(x, y, "Instances", genericBodyObject);
physics_fixture_bind(fix, inst);
physics_fixture_delete(fix);

Die obige code erstellt eine Halterung und weist ihren Index der Variablen "fix" zu. Anschließend werden die Form und die Dichte der Vorrichtung definiert, bevor sie an die Instanz gebunden wird, die mit dem in der Variablen "inst" gespeicherten Index erstellt wurde. Schließlich wird die Vorrichtung gelöscht, um Speicherlecks zu vermeiden, da sie nicht mehr benötigt wird.
