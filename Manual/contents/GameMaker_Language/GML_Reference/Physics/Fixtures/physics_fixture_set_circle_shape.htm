physics_fixture_set_circle_shape

Diese Funktion definiert eine Kreisform für Ihr Gerät mit einem Radius, der durch die argument "rad" definiert wird.

Syntax:

physics_fixture_set_circle_shape(fixture, rad)

ArgumentTyp Beschreibung
fixture the index of the fixture
rad radius of the circle

Rückgabe:

Beispiel:

physics_fixture_set_circle_shape(fix_Ball, sprite_get_width(spr_Ball) / 2);

Die obige code wendet eine Kreisform auf die in der Variable "fix_Ball" indizierte Vorrichtung an, deren Radius der Breite der sprite "spr_Ball" geteilt durch 2 entspricht.
