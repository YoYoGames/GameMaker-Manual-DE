physics_fixture_set_polygon_shape

Diese Funktion legt eine Polygonform für das Gerät fest, aber Sie müssen mit physics_fixture_add_point() die Form des Polygons in Bezug auf den Ursprung der Halterung definieren. Das Polygon wird geschlossen, wenn das Fixture an eine Instanz gebunden wird. Beachten Sie auch, dass diese Funktion aufgerufen werden muss, bevor Sie Punkte definieren, und dass Sie mindestens drei Punkte für Ihr Polygon definiert haben müssen, bevor Sie es an eine Instanz binden, sonst erhalten Sie eine Fehlermeldung.

Syntax:

physics_fixture_set_polygon_shape(fixture)

ArgumentTyp Beschreibung
fixture the index of the fixture

Rückgabe:

Beispiel:

physics_fixture_set_polygon_shape(fix_Ship);
physics_fixture_add_point(fix_Ship, 0,0);
physics_fixture_add_point(fix_Ship, -40, 100);
physics_fixture_add_point(fix_Ship, 40, 100);

Die obige code wendet eine Polygonform auf die in der Variable "fix_Ship" indizierte Vorrichtung an und definiert dann drei Punkte, um eine dreieckige Form zu erzeugen.
