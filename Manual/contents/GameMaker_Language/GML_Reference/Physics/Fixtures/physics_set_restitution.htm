physik_set_restitution

Wenn Sie ein Fixture mit einer Instanz binden, indem Sie physics_fixture_bind() wird eine "id" für das gebundene Fixture zurückgegeben. Sie können diese ID verwenden, um den Restitutionswert des gebundenen Fixtures, nicht des "Basis"-Fixtures, jederzeit mit dieser Funktion zu setzen. Die Restitution wird normalerweise als Wert zwischen 0 und 1 gesetzt, aber Sie können auch höhere Werte verwenden, obwohl die Ergebnisse unvorhersehbar sein können.

Syntax:

physics_set_restitution(fixture, restitution)

ArgumentTyp Beschreibung
fixture the id of the bound fixture
restitution the new restitution value to apply

Rückgabe:

Beispiel:

var rest = physics_get_restitution(fix_id);
physics_set_restitution(fix_id, rest * 2);

Die obige code ruft den aktuellen Rückgabewert für die gebundenen physikalischen Eigenschaften der Instanz ab und setzt sie dann auf einen anderen Wert.
