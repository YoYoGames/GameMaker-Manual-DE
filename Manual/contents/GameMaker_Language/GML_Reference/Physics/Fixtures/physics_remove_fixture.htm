physics_remove_fixture

Diese Funktion entfernt (oder "entbindet") ein Fixture von einer Instanz oder Instanzen. Sie benötigt die eindeutige "id" des gebundenen Fixtures (wie von der Funktion physics_fixture_bind() zurückgegeben wird) und entfernt alle aktuell definierten physikalischen Eigenschaften für die Instanz, so dass Sie ein neues Fixture definieren und dieses an die Instanz binden können. Auf diese Weise können Sie die physikalischen Eigenschaften der Instanz ändern, ohne sie zerstören und neu erstellen zu müssen.

Syntax:

physics_remove_fixture(id, fixture)

Argument Typ Beschreibung
id Instanz-ID The ID of the instance to remove the fixture from
fixture Physik Fixture ID The ID of the fixture that is to be removed from the instance

Rückgabe:

N/A

Beispiel:

physics_remove_fixture(id, my_fix);

Die obige code entfernt die Vorrichtung mit der "id", die in der Variablen "my_fix" gespeichert ist, aus der Instanz.
