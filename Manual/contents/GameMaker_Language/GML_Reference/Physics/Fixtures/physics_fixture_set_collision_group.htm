physics_fixture_set_collision_group

Sobald Sie Ihr Fixture erstellt haben, können Sie es als Teil einer Kollisionsgruppe definieren. Das bedeutet, dass du Instanzen mit diesem Fixture anweisen kannst, immer oder nie mit anderen Instanzen zu kollidieren, die Fixtures mit der gleichen Kollisionsgruppe an sich gebunden haben. Die Werte, die zugewiesen werden können, basieren auf positiven oder negativen Zahlen, wobei 0 der Standardwert ist (was bedeutet, dass die Instanz mit dieser Fixture-Einstellung ignoriert wird, es sei denn, es wird ein Kollisionsereignis für sie erstellt). Hier ist ein Bild zur Erläuterung:

Wie Sie sehen, kann das Festlegen von Kollisionsgruppen sehr nützlich sein, um komplexe physische Körper zu erstellen, die mit der physischen Welt um sie herum interagieren, aber nicht miteinander. Es sollte jedoch beachtet werden, dass es aufgrund der Komplexität der Berechnung von Kollisionen auf der Grundlage solcher Gruppen empfehlenswert ist, die Anzahl der in einem Raum verwendeten Gruppen auf ein absolutes Minimum zu beschränken.

Die Werte für die Kollisionsgruppe können zwischen -32.768 und 32.767 (jeweils einschließlich) liegen.

Syntax:

physics_fixture_set_collision_group(fixture, group);

Argument Typ Beschreibung
fixture Physik Fixture ID The fixture that is to be bound
group Real The group that this fixture is assigned to (either a positive value, 0, or a negative value)

Rückgabe:

N/A

Beispiel:

physics_fixture_set_collision_group(fix_Wheel, -1);

Die obige code setzt das in der Variable "fix_Wheel" indizierte Gerät in die Gruppe "-1".
