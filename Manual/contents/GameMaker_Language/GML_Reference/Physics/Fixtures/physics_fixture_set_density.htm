physics_fixture_set_density

Die Dichte einer Sache ist definiert als ihre Masse pro Volumeneinheit, was im Grunde bedeutet, wie viel Masse in dem Raum, den sie in der Welt einnimmt, eingepfercht ist. Ein Luftballon hätte also eine geringe Dichte, da er wenig Masse hat, die einen großen Raum einnimmt, während ein Bleistab eine hohe Dichte hat, da er eine große Masse hat, aber wenig Raum einnimmt.

In der 2D-Physikwelt von GameMaker wird die Masse automatisch berechnet aus den Werten, die Sie für die Dichte eingeben, und der Oberfläche der Form, die Sie für die Vorrichtung definieren. Dies hat einen direkten Einfluss auf die Trägheit des Spielgeräts und darauf, wie es auf Kollisionen und Kräfte reagiert. Wenn Sie eine kleine Form mit einer hohen Dichte erstellen, hat sie eine sehr große Masse (wie ein Stab aus Blei), aber wenn Sie eine große Form mit einer niedrigen Dichte definieren, hat sie eine viel geringere Masse (wie ein Luftballon).

HINWEIS: Wenn du möchtest, dass die Instanz, die die Halterung erhält, statisch ist (d.h. unbeweglich in der Spielwelt), dann sollte die Dichte auf 0 gesetzt werden, was ihr eine unendliche Dichte verleiht.

HINWEIS Eine statische Vorrichtung kann kinematisch gemacht werden (d.h. sie kann sich bewegen, aber ohne die Auswirkungen von Kraft oder Schwerkraft), indem sie eine Dichte von 0 erhält und dann die phy_speed_x , phy_speed_y , oder phy_angular_velocity Variablen.

Syntax:

physics_fixture_set_density(fixture, density)

ArgumentTyp Beschreibung
fixture the index of the fixture
density the density of the fixture calculated as kg/m²

Rückgabe:

Beispiel:

physics_fixture_set_density(fix_Balloon, 0.1);

Die obige code setzt die Dichte der in "fix_Ballon" indizierten Halterung auf 0,1.
