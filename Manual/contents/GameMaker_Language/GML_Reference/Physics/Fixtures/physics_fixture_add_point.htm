physics_fixture_add_point

Diese Funktion definiert die Punkte eines Polygons, das zuvor entweder mit physics_fixture_set_polygon_shape() oder sie kann verwendet werden, um Punkte zu einer Kette von Kanten hinzuzufügen Einbauten mit der Funktion physics_fixture_set_chain_shape(). Bei einer Polygonform sind die Punkte relativ zum Ursprung des Fixtures, wobei die Position 0,0 der x/y-Achse das Zentrum ist, und die nachfolgenden Koordinaten werden wie in GameMaker üblich berechnet, wobei unten/rechts +x,+y und oben/links -x,-y ist. Zur Veranschaulichung dieses Vorgangs finden Sie hier eine Illustration:

Wie Sie sehen können, kann jedes einfache konvexe Polygon auf diese Weise modelliert werden, aber konkave Polygone werden derzeit noch nicht unterstützt. Um das Polygon zu schließen, müssen Sie das Fixture an eine Instanz binden. Sie sollten auch beachten, dass Sie mindestens drei Punkte für Ihr Polygon definiert haben müssen, bevor Sie es an eine Instanz binden Sie erhalten sonst eine Fehlermeldung, und das Hinzufügen eines Punktes zu einem Fixture, das noch keine Polygonform hat, führt ebenfalls zu einer Fehlermeldung. Beachten Sie auch, dass die maximal zulässige Anzahl von Punkten für ein Fixture acht ist.

Bei Kettenformen (die einfach eine Reihe verbundener Kantenformen sind) setzt diese Funktion einen Punkt als "Verbindung", an dem die Kanten miteinander verkettet werden. Daher müssen Sie mindestens zwei Punkte definiert haben, bevor Sie die Halterung an die Instanz sonst erhalten Sie einen Fehler. Anders als bei Polygonen sind Sie jedoch nicht auf maximal acht Punkte beschränkt, sondern können so viele Punkte hinzufügen, wie Sie für die Kette benötigen, und die Punkte werden in tatsächlichen Raumkoordinaten und nicht relativ zur Instanz.

Syntax:

physics_fixture_add_point(fixture, xpos, ypos)

ArgumentTyp Beschreibung
fixture the index of the fixture
xpos the x position of the point (relative to the fixture origin for polygon shapes)
ypos the y position of the point (relative to the fixture origin for polygon shapes)

Rückgabe:

Beispiel:

physics_fixture_set_polygon_shape(fix_Ship);
physics_fixture_add_point(fix_Ship, 0,0);
physics_fixture_add_point(fix_Ship, -40, 100);
physics_fixture_add_point(fix_Ship, 40, 100);

Die obige code wendet eine Polygonform auf die in der Variable "fix_Ship" indizierte Vorrichtung an und definiert dann drei Punkte, um eine dreieckige Form zu erzeugen.
