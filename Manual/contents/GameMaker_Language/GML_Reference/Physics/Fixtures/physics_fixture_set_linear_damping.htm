physics_fixture_set_linear_damping

Dämpfung wird verwendet, um die Geschwindigkeit von Instanzen in der physikalischen Welt zu reduzieren, und unterscheidet sich von Reibung dadurch, dass Reibung nur auftritt, wenn zwei Instanzen mit Einbauten in Kontakt sind. Dämpfung ist auch viel billiger zu simulieren als Reibung, aber bitte beachten Sie, dass Dämpfung kein Ersatz für Reibung ist; die beiden Effekte können und sollten zusammen verwendet werden. Hier ist ein Bild zur Veranschaulichung der Dämpfung:

Die Dämpfungsparameter sollten zwischen 0 und unendlich liegen, wobei 0 keine Dämpfung bedeutet, und unendlich bedeutet volle Dämpfung. Normalerweise werden Sie einen Dämpfungswert zwischen 0 und 1 verwenden, aber Sie können bei Bedarf auch einen beliebigen nicht-negativen Wert verwenden.

Syntax:

physics_fixture_set_linear_damping(fixture, damping)

ArgumentTyp Beschreibung
fixture the index of the fixture
damping the damping of the fixture, usually between 0 and 1

Rückgabe:

Beispiel:

physics_fixture_set_linear_damping(fix_Ball, 0.1);

Die obige code setzt die lineare Dämpfung der in "fix_ball" indizierten Vorrichtung auf 0,1.
