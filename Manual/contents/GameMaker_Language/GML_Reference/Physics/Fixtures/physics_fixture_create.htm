physics_fixture_create

Der erste Schritt bei der Einrichtung einer Vorrichtung ist ihre Erstellung mit dieser Funktion. Der zurückgegebene Index sollte in einer Variablen gespeichert werden, die in allen weiteren Funktionen verwendet wird, die zur Definition und Verwendung dieser Vorrichtung dienen.

Syntax:

physics_fixture_create()

Rückgabe:

Physics Fixture ID

Beispiel:

fix_Ball = physics_fixture_create();

Die obige code erstellt eine Vorrichtung und speichert ihren Index in der Variablen "fix_Ball".
