physics_get_restitution

Wenn Sie ein Fixture mit einer Instanz binden, indem Sie physics_fixture_bind() bindet, wird eine "id" für das gebundene Fixture zurückgegeben. Sie können diese id verwenden, um die Restitution (die "bounciness" Eigenschaft) Wert des gebundenen Fixtures, nicht des "Basis"-Fixtures, jederzeit mit dieser Funktion abrufen.

Syntax:

physics_get_restitution(fixture)

ArgumentTyp Beschreibung
fixture the id of the bound fixture

Rückgabe:

Beispiel:

var rest = physics_get_restitution(fix_id);
physics_set_restitution(fix_id, rest * 2);

Die obige code ruft den aktuellen Rückgabewert für die gebundenen physikalischen Eigenschaften der Instanz ab und setzt sie dann auf einen anderen Wert.
