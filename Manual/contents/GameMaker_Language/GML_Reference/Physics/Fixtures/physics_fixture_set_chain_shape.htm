physics_fixture_set_chain_shape

Diese Funktion definiert eine "Ketten"-Halterungsform. Eine Kettenbefestigung besteht aus einer Reihe von Punkten, die mit Hilfe von Kantenformen miteinander verbunden werden, um sie zu verbinden. Die Funktion selbst nimmt den Index (ID) des Fixtures und kann das Fixture auf loop setzen oder nicht. Bei der Einstellung true wird der letzte Punkt mit dem ersten Punkt durch eine Kante verbunden, bei der Einstellung false hingegen nicht. Beachten Sie, dass diese Funktion das Gerät auf die Aufnahme der für die Kette erforderlichen Punkte vorbereitet. physics_fixture_add_point()aufgerufen wird, ähnlich wie bei der Erstellung einer Polygonbefestigung.

Wenn Sie eine Kettenbefestigung erstellen, müssen Sie ihr mindestens zwei Punkte geben, aber Sie sind in der Anzahl der nachfolgenden Punkte, die Sie hinzufügen können, um die fertige Befestigung zu bilden, nicht beschränkt.

Syntax:

physics_fixture_set_chain_shape(fixture, loop)

ArgumentTyp Beschreibung
fixture The index of the fixture
loop Whether to close the chain (true) oder nicht (false).

Rückgabe:

Beispiel:

var xx = -100;
var yy = room_height / 2;
var fix = physics_fixture_create(); physics_fixture_set_chain_shape(fix, false);
for (var i = 0; i < 10; i++;)
{
    physics_fixture_add_point(_fix, _xx, _yy);
    _xx += 50 + random(150);
    _yy = (room_height / 2) - 250 + irandom(500);
}
physics_fixture_bind(fix, id);

Mit der obigen code wird eine Reihe von "Rand"-Leuchten mit unterschiedlichen Höhen über die gesamte Länge des Raumes geschaffen.
