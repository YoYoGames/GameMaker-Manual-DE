physics_fixture_set_edge_shape

Diese Funktion definiert eine "Rand"-Fixture-Form. Eine Kantenform ist einfach eine Linie, die eine Kollision erzeugt, wenn andere Fixtures sie überlappen, und kann sehr nützlich sein, um (zum Beispiel) Gelände zu erzeugen oder um Grenzen um einen Raum herum zu schaffen. Die Position der Kante wird über den lokalen Raum definiert, d.h. die x/y-Position der Instanz wird als (0,0) betrachtet, also sollte dies bei der Erstellung berücksichtigt werden (im code Beispiel unten wurde die Instanz bei (0,0) im Raum platziert, um Komplikationen zu vermeiden).

Syntax:

physics_fixture_set_edge_shape(fixture, local_x1, local_y1, local_x2, local_y2)

ArgumentTyp Beschreibung
fixture the index of the fixture
local_x1 start x position for the edge
local_y1 start y position for the edge
local_x2 end x position for the edge
local_y2 end y position for the edge

Rückgabe:

Beispiel:

var xx = 0;
var y1 = room_height - 100;
var y2 = room_height - 50 - irandom(100);
for (var i = 0; i < 10; i++;)
{
    var _fix = physics_fixture_create();
    physics_fixture_set_edge_shape(_fix, _xx, _y1, _xx + 50, _y2);
    physics_fixture_bind(_fix, id);
    physics_fixture_delete(_fix);
    _xx += 50;
    _y1 = y2;
    _y2 = room_height - 50 - irandom(100);
}

Mit der obigen code wird eine Reihe von "Rand"-Leuchten mit unterschiedlichen Höhen über die gesamte Länge des Raumes geschaffen.
