physics_get_friction

Wenn Sie ein Fixture mit einer Instanz binden, indem Sie physics_fixture_bind() wird eine "id" für die gebundene Vorrichtung zurückgegeben. Sie können diese ID verwenden, um den Reibungswert des gebundenen Geräts(nicht des "Basisgeräts") jederzeit mit dieser Funktion zu ermitteln.

Syntax:

physics_get_friction(fixture)

Argument Typ Beschreibung
fixture Physik Fixture ID the id of the bound fixture

Rückgabe:

Real

Beispiel:

var fric = physics_get_friction(fix_id);
physics_set_friction(fix_id, fric + 0.1);

Die obige code ruft den aktuellen Reibungswert für die gebundenen Physikeigenschaften der Instanz ab und setzt sie dann auf einen anderen Wert.
