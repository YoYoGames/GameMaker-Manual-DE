physics_fixture_set_box_shape

Diese Funktion definiert eine Kastenform für Ihre Vorrichtung. Sie nimmt die halbe Breite und Höhe, da die Physik diese Werte viel mehr als ganze Breiten-/Höhenwerte verwendet, um Dinge wie Kollisionen zu bestimmen.

Syntax:

physics_fixture_set_box_shape(fixture, half_width, half_height)

ArgumentTyp Beschreibung
fixture the index of the fixture
halfWidth the half width of the box
halfHeight the half height of the box

Rückgabe:

Beispiel:

physics_fixture_set_box_shape(fix_Border, room_width/2, 10);

Die obige code wendet eine Kastenform auf das in der Variable "fix_Border" indizierte Gerät mit einer Breite des Raums und einer Höhe von 20 Pixeln an.
