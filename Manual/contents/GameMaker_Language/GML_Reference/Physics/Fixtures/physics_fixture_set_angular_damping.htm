physics_fixture_set_angular_damping

Jedes rotierende Objekt in der "realen Welt", sofern es nicht über einen Motor verfügt oder sich im Weltraum befindet, verlangsamt sich mit der Zeit durch den Einfluss äußerer Kräfte (wie Reibung mit der umgebenden Luft). Wir können die Funktion physics_fixture_set_angular_damping() im GameMaker verwenden, um diesen Effekt zu simulieren und die Rotationsgeschwindigkeit von Instanzen in der physikalischen Welt zu reduzieren, da sich sonst jede rotierende Instanz unendlich weiterdrehen würde. Die Dämpfungsparameter sollten zwischen 0 und unendlich liegen, wobei 0 für keine Dämpfung und unendlich für volle Dämpfung steht. Normalerweise werden Sie einen Dämpfungswert zwischen 0 und 1 verwenden, aber Sie können bei Bedarf einen beliebigen nicht-negativen Wert verwenden.

Syntax:

physics_fixture_set_angular_damping(fixture, damping)

ArgumentTyp Beschreibung
fixture the index of the fixture
damping the angular damping of the fixture, usually between 0 and 1

Rückgabe:

Beispiel:

physics_fixture_set_angular_damping(fix_Ball, 0.1);

Mit der obigen code wird die Winkeldämpfung der in "fix_ball" indizierten Vorrichtung auf 0,1 gesetzt.
