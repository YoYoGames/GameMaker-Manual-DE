draw_getpixel_ext

Mit dieser Funktion können Sie den vollständigen AGBR-32-Bit-Wert jedes Pixels abrufen, das auf das aktuelle Rendering-Ziel gezeichnet wird. Das bedeutet, dass die Ergebnisse von dem Ereignis abhängen, bei dem die Funktion aufgerufen wird, und auch von der verwendeten Zieloberfläche.

WICHTIG Diese Funktion verursacht einen enormen Leistungsabfall und sollte daher nur verwendet werden, wenn es unbedingt notwendig ist.

Syntax:

draw_getpixel_ext(x, y);

ArgumentTyp Beschreibung
x The x coordinate of the pixel to check
y The y coordinate of the pixel to check

Rückgabe:

Real

Beispiel:

col = draw_getpixel_ext(mouse_x, mouse_y);
alpha = (col >> 24) & 255;
blue = (col >> 16) & 255;
green = (col >> 8) & 255;
red = col & 255;

Die obige code holt sich den 32-Bit-Farbwert an der Position der Maus und teilt ihn dann in seine Komponentenwerte auf und speichert sie in Variablen.
