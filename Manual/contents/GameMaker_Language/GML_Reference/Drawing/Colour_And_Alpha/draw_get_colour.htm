draw_get_colour

Diese Funktion gibt die aktuelle Zeichenfarbe zurück, die zum Zeichnen von Formen, Text, Primitiven und nicht texturierten 3D-Modellen verwendet wird. Sie kann mit der Funktion draw_set_colour() Funktion eingestellt werden.

Syntax:

draw_get_colour()

Rückgabe:

Colour Constant

Beispiel:

var _cur_color = draw_get_color();
draw_set_color(text_color);
draw_text(x, y, text);
draw_set_color(_cur_color);

Die obige code speichert die aktuelle Zeichenfarbe in einer lokalen Variable und ändert die Zeichenfarbe auf der Grundlage einer Instanzvariablen. Nach dem Zeichnen von Text wird die Farbe wieder auf den in der lokalen Variable gespeicherten Wert zurückgesetzt.
