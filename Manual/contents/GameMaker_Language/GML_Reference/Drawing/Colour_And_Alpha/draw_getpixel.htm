draw_getpixel

Mit dieser Funktion können Sie den Farbwert jedes Pixels ermitteln, das auf das aktuelle Rendering-Ziel gezeichnet wird. Das bedeutet, dass die Ergebnisse vom Ereignis abhängen, in dem die Funktion aufgerufen wird, und auch von der verwendeten Zieloberfläche. Beachten Sie, dass diese Funktion keine Alphawerte zurückgibt, dafür sollten Sie draw_getpixel_ext().

WICHTIG Diese Funktion verursacht einen enormen Leistungsabfall und sollte daher nur verwendet werden, wenn es unbedingt notwendig ist.

Syntax:

draw_getpixel(x, y);

ArgumentTyp Beschreibung
x The x coordinate of the pixel to check
y The y coordinate of the pixel to check

Rückgabe:

Colour Constant

Beispiel:

draw_set_colour(draw_getpixel(500,200));

Die obige code setzt die Zeichenfarbe für alle Texte, Formen und Primitive auf die Farbe des Pixels bei (500,200) auf dem Spielbildschirm.
