colour_get_red

Diese Funktion gibt die Menge an Rot zurück, die für die angegebene Farbe verwendet wurde, wobei der Wert zwischen 0 und 255 liegt, wobei 0 kein Rot und 255 alles Rot bedeutet. Das folgende Bild veranschaulicht dies:

Syntax:

colour_get_red(col);

ArgumentTyp Beschreibung
col The colour to check

Rückgabe:

Beispiel:

r_comp = colour_get_red(c_teal);

Die obige code ermittelt den Rotanteil der Farbkonstante "c_teal" und speichert den Wert in der Variablen "r_comp".
