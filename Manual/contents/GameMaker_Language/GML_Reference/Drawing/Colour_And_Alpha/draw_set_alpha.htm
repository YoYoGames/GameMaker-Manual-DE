draw_set_alpha

Mit dieser Funktion können Sie den Basis-Alpha-Wert für das Spiel festlegen. Dieser Wert kann von 0 bis 1 eingestellt werden, wobei 0 völlig transparent und 1 völlig undurchsichtig ist (der Standardwert), und wirkt sich auf alle weiteren Zeichnungen aus, einschließlich Hintergründe, sprites, Schriftarten, Primitive und 3D.

Beachten Sie, dass, wenn Sie eines der Assets zeichnen und einen Alphawert argument als Teil der Funktion angeben (z. B. bei Verwendung von draw_sprite_ext()), dann wird der hier eingestellte Alphawert überschrieben, so dass beim Zeichnen eines sprite mit draw_sprite_ext() und einem Alpha-Wert argument von 1 das sprite vollständig undurchsichtig gezeichnet wird, unabhängig von dem mit draw_set_alpha() eingestellten Alpha-Wert (dies funktioniert genauso, wenn Sie draw_self() und Setzen des image_alpha der Instanz). Beachte auch, dass wenn eine Instanz standardmäßig gezeichnet wird (d.h.: sie hat ein sprite aber kein Draw Event), dann wird die image_alpha Variable auf das mit draw_set_alpha() eingestellte Alpha geklemmt wird, so dass ein image_alpha von 1 und ein Draw Alpha von 0.5 das sprite bei Alpha 0.5 zeichnen wird, während ein image_alpha von 0.25 und ein Draw Alpha von 0,5 die sprite mit 0,25 Alpha zeichnet.

Syntax:

draw_set_alpha(alpha);

ArgumentTyp Beschreibung
alpha The alpha to set (between 0 and 1)

Rückgabe:

Beispiel:

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

Die obige code wird einen Text an der angegebenen Position mit einem Schatteneffekt zeichnen, der durch eine Änderung von Alpha und Farbe erzeugt wird.
