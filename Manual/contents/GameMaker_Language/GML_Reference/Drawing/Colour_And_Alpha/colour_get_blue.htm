farbe_holen_blau

Diese Funktion gibt die Menge an Blau zurück, die für die gegebene Farbe verwendet wurde, wobei der Wert zwischen 0 und 255 liegt, wobei 0 kein Blau und 255 alles Blau bedeutet. Das folgende Bild veranschaulicht dies:

Syntax:

colour_get_blue(col);

ArgumentTyp Beschreibung
col The colour to check

Rückgabe:

Beispiel:

b_comp = colour_get_blue(c_teal);

Die obige code ermittelt den Blauanteil der Farbkonstante "c_teal" und speichert den Wert in der Variablen "b_comp".
