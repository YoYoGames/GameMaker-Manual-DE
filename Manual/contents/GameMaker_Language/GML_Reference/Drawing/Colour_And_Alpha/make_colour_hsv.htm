make_colour_hsv

GameMaker bietet diese Funktion (und andere), um dem Benutzer die Möglichkeit zu geben, seine eigenen Farben zu erstellen. Diese Funktion setzt sich aus drei Komponenten zusammen, dem Farbton, der Sättigung und dem Wert (auch als "Leuchtkraft" bezeichnet), um die gewünschte Farbe zu erzeugen. Diese Werte werden zwischen 0 und 255 angenommen, so dass Sie damit 16.777.216 (256*256*256) Farben erzeugen können! Unten sehen Sie ein Bild, das zeigt, wie diese Komponenten aussehen, wenn sie getrennt werden:

Die linke Abbildung zeigt eine Aufschlüsselung der einzelnen Komponenten der Funktion, während die rechte Abbildung veranschaulicht, wie sich die Änderung dieser Komponenten auf die Endfarbe auswirkt. Wie Sie sehen, können wir die Farbe auch ohne Änderung des Farbtons (er bleibt bei 0, rot) mit den beiden anderen Einstellungen drastisch verändern, wobei die Sättigung den Farbton von grau (0) zu einem reinen Ton (255) und der Wert von dunkler (0) zu heller (255) werden lässt.

Syntax:

make_colour_hsv(hue, sat, val)

ArgumentTyp Beschreibung
hue The hue of the colour
sat How saturated the colour is
val How dark the colour is

Rückgabe:

Colour Constant

Beispiel:

col = make_colour_hsv(100, 145, 255);

Die obige code verwendet die Funktion, um eine Farbe zu erstellen und ihren Wert zur späteren Verwendung in der Variablen "col" zu speichern.
