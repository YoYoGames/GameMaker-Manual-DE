draw_get_alpha

Diese Funktion gibt den aktuellen Wert von Draw Alpha zurück, der zwischen 0 (vollständig transparent) und 1 (vollständig undurchsichtig) liegt. Das Zeichen-Alpha beeinflusst die Transparenz aller Zeichenfunktionen und kann mit der Funktion draw_set_alpha() eingestellt werden.

Syntax:

draw_get_alpha()

Rückgabe:

Beispiel:

var _cur_alpha = draw_get_alpha();
draw_set_alpha(text_alpha);
draw_text(x, y, text);
draw_set_alpha(_cur_alpha);

Die obige code speichert den aktuellen Alphawert in einer lokalen Variable und ändert den Alphawert in Abhängigkeit von einer Instanzvariablen. Nach dem Zeichnen von Text wird der Alphawert wieder auf den in der lokalen Variable gespeicherten Wert zurückgesetzt.
