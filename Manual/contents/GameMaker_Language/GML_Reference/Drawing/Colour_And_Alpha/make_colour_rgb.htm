make_colour_rgb

GameMaker stellt diese Funktion (und andere) zur Verfügung, damit der Benutzer seine eigenen Farben erstellen kann. Diese Funktion besteht aus drei Komponenten, der roten, der grünen und der blauen Komponente der Farbe, die Sie erstellen möchten. Diese Werte werden zwischen 0 und 255 angenommen, so dass Sie damit 16.777.216 (256*256*256) Farben erzeugen können! Unten sehen Sie ein Bild, das zeigt, wie diese Komponenten aussehen, wenn sie getrennt werden:

Die linke Abbildung zeigt eine Aufschlüsselung der einzelnen Komponenten der Funktion, während die rechte Abbildung veranschaulicht, wie sich eine Änderung dieser Komponenten auf die Endfarbe auswirkt.

Syntax:

make_colour_rgb(red, green, blue)

ArgumentTyp Beschreibung
red The red component of the colour
green The green component of the colour
blue The blue component of the colour

Rückgabe:

Colour Constant

Beispiel:

col = make_colour_rgb(100, 145, 255);

Die obige code verwendet die Funktion, um eine Farbe zu erstellen und ihren Wert zur späteren Verwendung in der Variablen "col" zu speichern.
