draw_clear_alpha

Mit dieser Funktion kann der gesamte Bildschirm mit einer bestimmten Farbe geleert werden, und die Alphakomponente des Ziels wird auf den von Ihnen eingestellten Wert gesetzt - diese Funktion führt keine Überblendung durch, aber alle nachfolgenden Überblendungsoperationen können so eingestellt werden, dass sie die von Ihnen eingestellte Alphakomponente verwenden. können aber so eingestellt werden, dass sie die von Ihnen eingestellte Alphakomponente des Ziels verwenden. Dies ist nur für die Verwendung im Draw-Ereignis einer Instanz (es wird nicht angezeigt, wenn es in einem anderen Ereignis verwendet wird), und es kann auch sehr nützlich für das Löschen von Oberflächen sein, wenn sie neu erstellt werden.

Es ist erwähnenswert, dass Sie bei der Verwendung von HTML5 module diese Funktion verwenden können, um den Canvas-Hintergrund transparent zu machen, so dass alle Bilder, die im Host-HTML verwendet werden, sichtbar sind, wenn das Spiel über ihnen läuft. Um diesen Effekt zu erzielen müssen Sie zuerst in den Raum-Editor gehen und alle Hintergrundebenen aus dem Raum-Editor entfernen und dann die Option Clear View Background in den View Properties deaktivieren. Schließlich müssen Sie ein neues Objekt erstellen, es im ersten Raum des Spiels platzieren und ihm eine sehr hohe Tiefe zuweisen (z. B. 10000). Verwenden Sie dann im Pre Draw-Ereignis diese Funktion, um die Farbe und Transparenz der Leinwand auf die gewünschte Farbe und Transparenz zu setzen. die Sie haben möchten.

Syntax:

draw_clear_alpha(col, alpha);

ArgumentTyp Beschreibung
col The colour with which the screen will be cleared
alpha The transparency of the colour with which the screen will be cleared

Rückgabe:

Beispiel:

draw_clear_alpha(c_red, 0.5);

Dadurch wird der Bildschirm mit 50 % (0,5) Transparenz in der Farbe Rot dargestellt.
