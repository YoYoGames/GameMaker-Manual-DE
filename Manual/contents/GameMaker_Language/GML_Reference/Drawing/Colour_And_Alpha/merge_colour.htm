merge_colour

Mit dieser Funktion können Sie zwei Farben nehmen und sie dann zu einer neuen Farbe zusammenfügen. Die Menge jeder der Komponentenfarben kann durch Ändern der "Menge" argument festgelegt werden, wobei ein Wert von 0 die erste Farbe (col1), ein Wert von 1 die zweite Farbe (col2) und ein Wert dazwischen die entsprechende Mischung ergibt. Ein Wert von 0,5 zum Beispiel mischt die beiden Farben gleichmäßig. Das folgende Bild veranschaulicht die Funktionsweise, indem es die Farben Rot und Blau miteinander mischt:

Syntax:

merge_colour(col1, col2, amount);

ArgumentTyp Beschreibung
col1 The first colour to merge
col2 The second colour to merge
amount How much of each colour should be merged. For example, 0 will return col1, 1 will return col2, and 0.5 would return a merge of both colours equally

Rückgabe:

Colour Constant

Beispiel:

col = merge_colour(c_lime, c_orange, 0.5);

Die obige code verwendet die Funktion, um eine Farbe zu erstellen, indem sie Limette und Orange 50/50 mischt, und speichert dann ihren Wert in der Variablen "col" zur späteren Verwendung.
