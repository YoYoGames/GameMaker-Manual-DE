draw_set_swf_aa_level

Mit dieser Funktion kann der Anti-Aliasing-Level (AA) für das SWF-Format vector sprites eingestellt werden. Dies kann ein realer Wert zwischen 0 und 1 sein und wird die Ränder dieser sprites"glätten". Um diesen Effekt zu sehen, müssen Sie zuerst AA mit der Funktion draw_enable_swf_aa().

Syntax:

draw_set_swf_aa_level(AA);

ArgumentTyp Beschreibung
AA The anti-aliasing value to use from 0 to 1.

Rückgabe:

Beispiel:

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

Die obige code überprüft den AA-Wert für das SWF-Format sprites, und wenn er 0 ist, wird AA aktiviert und der Wert auf 0,5 gesetzt.
