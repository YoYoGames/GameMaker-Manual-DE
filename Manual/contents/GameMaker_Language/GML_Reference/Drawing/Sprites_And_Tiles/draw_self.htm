zeichnen_selbst

Diese Funktion zeichnet die sprite, die der Instanz zugewiesen ist, genau so, wie sie gezeichnet werden würde, wenn das Draw-Ereignis keine code oder Aktionen enthielte, und spiegelt alle Änderungen wider, die an den Sprite-Variablen in anderen Ereignissen vorgenommen wurden.

HINWEIS: Wenn der Instanz keine sprite zugewiesen ist, wird diese Funktion einen Fehler auslösen!

Syntax:

draw_self();

Rückgabe:

Beispiel:

draw_self();

Dadurch wird die Instanz selbst mit den Eigenschaften gezeichnet, die durch die eingebauten sprite Variablen definiert sind.
