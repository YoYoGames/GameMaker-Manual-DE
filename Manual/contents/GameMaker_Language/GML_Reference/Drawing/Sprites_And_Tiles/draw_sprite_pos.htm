sprung_position_ziehen

Mit dieser Funktion können Sie ein sprite über den durch die vier Eckkoordinaten definierten Bereich verzerrt zeichnen. Die ersten beiden arguments sind das zu zeichnende sprite und das Teilbild des sprite (das gleiche wie in der Funktion draw_sprite()) aber die nächsten sind diejenigen, die die Position jeder der vier Ecken des Begrenzungsrahmens des angegebenen sprite definieren. Diese sollten im Uhrzeigersinn angegeben werden, so dass die erste Koordinate die obere linke, dann die obere rechte, dann die untere rechts und schließlich die untere linke. Sie können auch einen Wert für den Alphawert der sprite festlegen, um sie mit Transparenz zu zeichnen. Das folgende Bild zeigt, wie diese Funktion funktioniert:

WARNUNG! Das obige Bild dient nur zur Veranschaulichung. Wenn Sie diese Funktion auf einer sprite verwenden, erhalten Sie andere Ergebnisse und es kann zu einer "Scherung" der Textur kommen, da sprite aus einem Quad von Primitiven aufgebaut ist.

HINWEIS: Diese Funktion ist nur für Bitmaps nützlich sprites und funktioniert nicht mit SWF oder JSON (Spine) sprites.

HINWEIS: Wenn Sie diese Funktion verwenden, sollte die Einstellung Automatisch beschneiden für Texturseiten deaktiviert sein.

Syntax:

draw_sprite_pos(sprite, subimg, x1, y1, x2, y2, x3, y3, x4, y4, alpha);

ArgumentTyp Beschreibung
sprite The index of the sprite to draw.
subimg The subimg (frame) of the sprite to draw (image_index or -1 correlate to the current frame of animation in the object).
x1 The first x coordinate.
y1 The first y coordinate.
x2 The second x coordinate.
y2 The second y coordinate.
x3 The third x coordinate.
y3 The third y coordinate.
x4 The fourth x coordinate.
y4 The fourth y coordinate.
alpha The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

Rückgabe:

Beispiel:

draw_sprite_pos(sprite_index, image_index, x - 100, y - 50, x - 50, y +150, x + 100, y + 200, x + 100, y, 1);

Das obige code zeichnet die sprite, die mit der Instanz verbunden ist, auf der code läuft, verzerrt um die x/y-Position der Instanz und mit einem vollständig undurchsichtigen Alpha.
