draw_sprite_part

Mit dieser Funktion können Sie einen Teil eines beliebigen sprite an einer bestimmten Stelle im Raum zeichnen. Wie bei draw_sprite() können Sie ein sprite und ein Teilbild zum Zeichnen angeben, dann müssen Sie die relativen Koordinaten innerhalb des sprite für den Bereich angeben, der zum Zeichnen ausgewählt werden soll. Das bedeutet, dass eine linke Position von 0 und eine obere Position von 0 die linke obere Ecke des sprite wäre und alle weiteren Koordinaten von dieser Position aus genommen werden sollten. Das folgende Bild zeigt ein Beispiel wie dies funktioniert:

Beachten Sie, dass diese Funktion möglicherweise nicht wie erwartet funktioniert, wenn die Texturseite eine automatische Beschneidung zulässt, da der zusätzliche "leere" Raum um die sprite für die Erstellung der Texturseite entfernt wurde. Um dieses Problem zu beheben, müssen Sie in den Einstellungen der Texturseite (im Texturgruppen-Editor) die Option Automatisches Zuschneiden deaktivieren.

HINWEIS: Beim Zeichnen mit dieser Funktion werden der sprite x-Offset und y-Offset ignoriert und der sprite Teil wird mit der linken oberen Ecke an der angegebenen x/y-Position im Raum gezeichnet.

HINWEIS: Diese Funktion ist nur für Bitmaps nützlich sprites und funktioniert nicht mit SWF oder JSON (Spine) sprites.

Syntax:

draw_sprite_part(sprite, subimg, left, top, width, height, x, y);

ArgumentTyp Beschreibung
sprite The index of the sprite to draw.
subimg The subimg (frame) of the sprite to draw (image_index or -1 correlate to the current frame of animation in the object).
left The x position on the sprite of the top left corner of the area to draw.
top The y position on the sprite of the top left corner of the area to draw.
width The width of the area to draw.
height The height of the area to draw.
x The x coordinate of where to draw the sprite.
y The y coordinate of where to draw the sprite.

Rückgabe:

Beispiel:

draw_sprite_part(sprite_index, image_index, 4, 0, sprite_width-16, sprite_height-16, x, y );

Dies wird die Instanzen zugewiesen sprite (sprite_index) und seine aktuelle Frame der Animation (image_index), aber es wird ein 4px Rand aus der Breite auf beiden Seiten, und ein 8 Pixel Rand aus der Höhe von der Unterseite des ursprünglichen 24x24 rasieren Pixel sprite.
