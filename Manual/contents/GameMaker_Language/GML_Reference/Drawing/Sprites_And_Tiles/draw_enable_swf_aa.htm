draw_enable_swf_aa

Mit dieser Funktion können Sie Anti-Aliasing (AA) für das SWF-Format vector sprites aktivieren oder deaktivieren. AA glättet einfach die Kanten von vector Bildern, um ihnen ein schöneres Aussehen zu verleihen. Die Stärke von AA hängt von dem Wert ab, den Sie mit der Funktion draw_set_swf_aa_level(). Standardmäßig ist diese Funktion deaktiviert.

Syntax:

draw_enable_swf_aa(enable);

ArgumentTyp Beschreibung
enable Enable (true) oder deaktivieren (false) Sie AA für alle SWF sprites.

Rückgabe:

Beispiel:

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

Die obige code überprüft den AA-Wert für das SWF-Format sprites, und wenn er 0 ist, wird AA aktiviert und der Wert auf 0,5 gesetzt.
