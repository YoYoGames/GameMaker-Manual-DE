draw_tilemap

Diese Funktion kann verwendet werden, um eine bestimmte Tilemap irgendwo im Raum zu zeichnen. Sie geben die ID des Tilemap-Elements an (die Sie erhalten, wenn Sie ein Tilemap-Element mit layer_tilemap_create() oder bei Verwendung der Funktion layer_tilemap_get_id()), dann gibst du die x/y-Position zum Zeichnen an (im Raum). Beachte, dass dies einfach die Tilemap an dem angegebenen Punkt zeichnet, unter Verwendung der Ebenentiefe der Instanz, die die Funktion aufruft. Es bewegt die Tilemap nicht - noch verändert es sie in irgendeiner Weise - und es spielt keine Rolle ob die Tilemap als sichtbar markiert ist oder nicht.

Syntax:

draw_tilemap(tilemap_element_id, x, y);

Argument Typ Beschreibung
tilemap_element_id Kachelkarte Element-ID The unique ID value of the tilemap element to draw
x Real The x position within the room to draw at
y Real The y position within the room to draw at

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Tiles_Effects");
var map_id = layer_tilemap_get_id(lay_id);
draw_tilemap(map_id, mouse_x, mouse_y);

Die obige code ruft die Layer-ID ab und verwendet diese dann, um eine bestimmte Tilemap-ID zu erhalten, die wiederum verwendet wird, um die Tilemap an der Position der Maus zu zeichnen.
