Sprites und Kacheln

Auch wenn Sie im Raumeditor Hintergründe einrichten und Instanzen sprites zuweisen können, ist dies oft nicht flexibel genug für das Spiel, das Sie erstellen. Wenn du zum Beispiel eine Instanz hast, die ein Menü zeichnen muss, möchtest du vielleicht verschiedene Schaltflächen sprites gleichzeitig anzeigen, oder wenn du ein Spiel mit scrollenden Parallaxen-Hintergründen hast, musst du in der Lage sein, sie über code in einer Controller-Instanz irgendeiner Art einzustellen. In diesen Situationen sollten Sie die folgenden Funktionen verwenden, mit denen Sie sprite und Hintergrund-Assets an beliebiger Stelle im Raum zeichnen können, wobei einige der Funktionen auch die Möglichkeit bieten, Form, Größe, Farbe und Alpha der sprite zu manipulieren, um spezielle Effekte zu erzielen.

Du kannst einer Instanz eine sprite zuweisen und standardmäßig wird GameMaker sie mit den Eigenschaften zeichnen, die durch die verschiedenen Sprite-Variablen gegeben sind. Wenn du jedoch etwas zum Draw-Ereignis hinzufügst, "übernimmst" du die Kontrolle und musst GameMaker genau sagen, was er zeichnen soll. Das bedeutet, dass du die folgenden Funktionen verwenden musst, um genau zu definieren, was die Instanz zeichnen soll und wo im Raum sie es zeichnen soll:

  • draw_self
  • draw_sprite
  • draw_sprite_ext
  • draw_sprite_general
  • draw_sprite_part
  • draw_sprite_part_ext
  • draw_sprite_stretched
  • draw_sprite_stretched_ext
  • draw_sprite_pos
  • draw_sprite_tiled
  • draw_sprite_tiled_ext

Wenn Sie das SWF-Format sprites verwenden, gibt es auch zusätzliche Optionen zur Steuerung des beim Zeichnen verwendeten Anti-Aliasings (siehe Importieren von Nicht-Bitmap-Sprites für weitere Informationen über diese Arten von sprites). Beachten Sie, dass diese Funktionen auf globaler Ebene arbeiten und das AA für alle vector SWF sprites gezeichneten Bilder einstellen, nachdem sie aufgerufen wurden.

  • draw_enable_swf_aa
  • draw_set_swf_aa_level
  • draw_get_swf_aa_level

Neben den oben genannten Funktionen gibt es auch zwei spezielle Funktionen, die für die Verwendung mit Skelett-Animationen entwickelt wurden (siehe Importieren von Nicht-Bitmap-Sprites für weitere Informationen über diese Arten von sprites und den Abschnitt Skelett-Animationen für spezifische Funktionen im Zusammenhang mit diesen Arten von sprites):

  • draw_skeleton
  • draw_skeleton_instance
  • draw_skeleton_collision
  • draw_skeleton_time

Kacheln werden automatisch gezeichnet, basierend auf den Daten, die sie enthalten, dem verwendeten Kachelsatz und den Eigenschaften der Kachelkarte. Sie können jedoch einzelne Kacheln oder Kachelkarten mit den folgenden Funktionen zeichnen:

  • draw_tile
  • draw_tilemap
