draw_get_swf_aa_level

Diese Funktion kann verwendet werden, um den Anti-Aliasing-Level (AA) für das SWF-Format vector sprites zu ermitteln. Der Rückgabewert liegt zwischen 0 und 1 und zeigt an, wie "glatt" die Kanten dieser sprites gezeichnet werden. Sie können den AA-Level mit der Funktion draw_set_swf_aa_level().

Syntax:

draw_get_swf_aa_level();

Rückgabe:

Beispiel:

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

Die obige code überprüft den AA-Wert für das SWF-Format sprites, und wenn er 0 ist, wird AA aktiviert und der Wert auf 0,5 gesetzt.
