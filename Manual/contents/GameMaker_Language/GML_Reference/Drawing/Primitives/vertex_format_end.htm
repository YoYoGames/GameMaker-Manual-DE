vertex_format_end

Diese Funktion muss nach der Definition eines neuen Scheitelpunktformats aufgerufen werden. Sie gibt den "Handle" (Index) des neuen Formats zurück, der in allen weiteren Scheitelpunktfunktionen verwendet werden muss, die sich auf dieses neue Format beziehen.

Syntax:

vertex_format_end();

Rückgabe:

Vertex Format ID

Beispiel:

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
my_format = vertex_format_end();

Die obige code erstellt ein neues Scheitelpunktformat nur mit Farb- und Positionswerten und speichert dann die Format-ID in der Variablen "my_format".
