draw_primitive_begin_texture

Diese Funktion muss aufgerufen werden, bevor Sie die Scheitelpunkte eines texturierten Primitivs definieren. Sie müssen die Art des zu verwendenden Primitivs angeben (siehe draw_primitive_begin() für weitere Informationen) und die ID einer zu verwendenden Textur, die ein sprite oder ein Hintergrundbild-Asset sein kann. Diese Asset-ID kann mit den Funktionen abgefragt werden sprite_get_texture()abgerufen werden (verwenden Sie -1 für keine Textur).

HINWEIS Damit eine Textur wiederholt werden kann, muss sie eine Zweierpotenz der Größe sein, d.h. 32x32, 128x128, usw...

Syntax:

draw_primitive_begin_texture(kind, tex)

Argument Typ Beschreibung
kind Primitiver Typ Konstante The kind of primitive you are going to draw.
tex Textur-Zeiger The texture to use with the primitive.

Rückgabe:

N/A

Beispiel:

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

Die obige code zeichnet einen Dreiecksstreifen mit 4 Scheitelpunkten (der ein Rechteck ergibt), der mit der in der Variable "tex" gespeicherten Textur texturiert ist, und die gesamte Textur wird verwendet, um das fertige Primitiv abzudecken.
