vertex_end

Mit dieser Funktion beenden Sie die Erstellung des benutzerdefinierten Primitivs. Sobald Sie diesen Befehl aufrufen, kann das Primitiv in der vertex_submit() Funktion für die Verwendung in einem Shader verwendet werden oder Sie können die buffer einfrieren (wodurch der Vertex buffer schreibgeschützt und viel schneller verwendet wird), indem Sie vertex_freeze().

Syntax:

vertex_end(buffer);

Argument Typ Beschreibung
buffer Vertex Buffer ID The buffer to end writing to.

Rückgabe:

N/A

Beispiel:

vertex_end(buff);

Mit der obigen code wird die Definition eines benutzerdefinierten Primitivs beendet.
