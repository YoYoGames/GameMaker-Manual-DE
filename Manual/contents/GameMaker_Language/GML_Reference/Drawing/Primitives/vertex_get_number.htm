vertex_get_number

Mit dieser Funktion können Sie die Anzahl der einzelnen Scheitelpunkte ermitteln, die in einem beliebigen Scheitelpunkt buffer definiert sind.

Syntax:

vertex_get_number(buffer);

Argument Typ Beschreibung
buffer Vertex Buffer ID The vertex buffer to check.

Rückgabe:

Real

Beispiel:

v_num = vertex_get_number(v_buffer);

Die obige code speichert die Anzahl der Scheitelpunkte, die in dem angegebenen Scheitelpunkt buffer gespeichert sind, in einer Variablen.
