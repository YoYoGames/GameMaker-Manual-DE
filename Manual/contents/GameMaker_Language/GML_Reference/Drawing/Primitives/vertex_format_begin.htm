vertex_format_begin

Bevor Sie Ihr neues Scheitelpunktformat definieren können, müssen Sie GameMaker mitteilen, dass Sie dies mit dieser Funktion tun. Sie müssen diese Funktion zuerst aufrufen, dann die Formatwerte mit den entsprechenden Funktionen definieren und schließlich die Funktion vertex_format_end() auf, um die Definition abzuschließen und das neue Format "handle" zurückzugeben.

Syntax:

vertex_format_begin();

Rückgabe:

Beispiel:

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
my_format = vertex_format_end();

Die obige code erstellt ein neues Scheitelpunktformat nur mit Farb- und Positionswerten und speichert dann die Format-ID in der Variablen "my_format".
