vertex_argb

Diese Funktion legt die ARGB-Werte für den Scheitelpunkt fest, der gerade für das benutzerdefinierte Primitiv definiert wird. Sie geben die buffer an, in die die Daten geschrieben werden sollen, sowie die Rot-, Grün-, Blau- und Alpha-Werte, die Sie als einzelne 32-Bit-Ganzzahl ohne Vorzeichen verwenden möchten - Alpha-Sample in den höchsten 8 Bits, gefolgt vom Rot-Sample, Grün-Sample und schließlich dem Blau-Sample in den niedrigsten 8 Bits. Sie können die Hex-Notation ($AARRGGBB) oder eine reelle Zahl verwenden oder eine der make_colour_*() Funktionen zur Definition des Farbwerts nutzen.

Syntax:

vertex_argb(buffer, argb);

Argument Typ Beschreibung
buffer Vertex Buffer ID The buffer to write the information to.
argb Farbe The ARGB value to set.

Rückgabe:

N/A

Beispiel:

vertex_argb(buff, $FFFFFFFF);

Die obige code setzt die ARGB-Werte des zu definierenden Scheitelpunktes auf Weiß.
