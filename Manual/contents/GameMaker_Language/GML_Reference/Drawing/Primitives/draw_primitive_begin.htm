draw_primitive_begin

Diese Funktion muss aufgerufen werden, bevor Sie irgendwelche Primitive definieren können. Es gibt 6 Arten von Primitiven, die Sie als die folgenden Konstanten definieren können:

Primitiver Typ Konstante
Konstante Beschreibung
pr_pointlist Eine Punkteliste
pr_linelist Eine Zeilenliste
pr_linestrip Ein Leitungsstreifen
pr_trianglelist Eine Dreiecksliste
pr_trianglestrip Ein dreieckiger Streifen
pr_trianglefan Ein dreieckiger Ventilator

Das folgende Bild veranschaulicht, wie diese im Wesentlichen aussehen sollten und in welcher Reihenfolge Sie die Scheitelpunkte definieren sollten:

Bitte beachten Sie, dass auf einigen platforms (Windows, UWP, XBox) der Typ pr_trianglefan nicht nativ unterstützt wird und GameMaker daher eine Konvertierung vornimmt, wenn das Spiel kompiliert wird, damit sie funktionieren. Das bedeutet, dass auf diesen platforms der Typ pr_trianglefan viel langsamer ist als die anderen. Beachten Sie auch, dass Sie WebGL in den Spieloptionen aktivieren müssen, um diese Funktion auf HTML5 zu verwenden.

Syntax:

draw_primitive_begin(kind);

Argument Typ Beschreibung
kind Primitiver Typ Konstante The kind of primitive you are going to draw.

Rückgabe:

N/A

Beispiel:

var _steps = 20;
var _xx = 50;
var _yy = 50;
var _radius = 30;
draw_primitive_begin(pr_trianglefan);
draw_vertex(_xx, _yy);
for(var i = 0; i <= _steps; ++i;)
{
    draw_vertex(_xx + lengthdir_x(_radius, 270 * i / _steps), _yy + lengthdir_y(_radius, 270 * i / _steps));
}
draw_primitive_end();

Die obige code zeichnet drei Viertel eines Kreises, der aus Primitiven besteht.
