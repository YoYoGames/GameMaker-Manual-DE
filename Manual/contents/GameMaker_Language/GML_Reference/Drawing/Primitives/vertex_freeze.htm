vertex_freeze

Diese Funktion kann verwendet werden, um einen Scheitelpunkt buffer"einzufrieren". Diese buffer wird schreibgeschützt, was bedeutet, dass Sie, wenn Sie sie ändern möchten, die gesamte buffer löschen und neu erstellen müssen. Ein eingefrorenes buffer kann dem Shader schneller übergeben werden als ein normales, dynamisches buffer und wird für jene Effekte empfohlen, die ein unveränderliches benutzerdefiniertes Primitiv für die Dauer eines Levels oder des Spiels benötigen.

Die Funktion gibt 0 zurück, wenn das Einfrieren des angegebenen Scheitelpunkts buffer erfolgreich war. Wenn es jedoch aus irgendeinem Grund fehlgeschlagen ist, gibt sie -1 zurück.

Syntax:

vertex_freeze(buffer);

Argument Typ Beschreibung
buffer Vertex Buffer ID The vertex buffer to freeze.

Rückgabe:

Real (0 for success, -1 for failure)

Beispiel:

// Create event
vertex_freeze(vbuff);

Die obige code friert den angegebenen Vertex buffer im Create-Ereignis ein, so dass er als "statisches" buffer in anderen Ereignissen für schnelleren Zugriff verwendet werden kann.
