Primitive und Vertex-Formate

Ein Primitiv ist eine Art von Zeichnung, bei der Punkte verwendet werden, um zu definieren, was auf dem Bildschirm angezeigt wird. Sie können aus einzelnen Punkten, Linien oder Dreiecken bestehen und können eingefärbt, mit Alpha-Blending versehen und sogar mit einer Textur versehen werden, was bedeutet, dass sie verwendet werden können, um einige sehr aufregende Effekte in einem 2D-Spiel zu erzeugen! Sie können zum Beispiel ein Rechteck aus mehreren Dreiecks-Primitiven erstellen, es dann mit einer Flagge sprite texturieren und im Draw-Ereignis einer Instanz die Flagge im Wind "wehen" lassen und die von Ihnen verwendete sprite verzerren, indem Sie einfach die Punkte ändern, die das Primitiv bilden.

WARNUNG! Diese Funktionen funktionieren nicht mit dem HTML5 module, es sei denn, Sie haben OpenGL in den Spieloptionen aktiviert.

WICHTIG! Die Primitive, die Sie erstellen, sollten dem Format folgen, das Sie mit den Funktionen für das Scheitelpunktformat festgelegt haben. Wenn Sie also ein Scheitelpunktformat nur mit Positionsdaten definiert haben, hat es keinen Sinn, Ihr Primitiv mit Farbdaten zu erstellen. Sie sollten beachten, dass die Reihenfolge, in der Sie dem Primitiv, das Sie erstellen, Eigenschaften hinzufügen, durch die Reihenfolge definiert wird, in der Sie diese Eigenschaften beim Erstellen des Scheitelpunktformats hinzugefügt haben. Wenn Sie also das Format vector mit der Reihenfolge Position, Farbe und Texturkoordinate definiert haben, müssen Sie diese Eigenschaften dem Primitiv, das Sie erstellen, in derselben Reihenfolge hinzufügen, da Sie sonst Fehler erhalten.
