vertex_erstellen_puffer_aus_puffer_ext

Wie bei der Funktion vertex_create_buffer()erstellt diese Funktion einen neuen Scheitelpunkt buffer, nur dass jetzt die Scheitelpunktdaten, die sie speichert, von der regulären buffer kopiert werden, die als Quelle angegeben ist. Die buffer ist mit dem Scheitelpunktformat für die Erstellung von Primitiven für die Verwendung mit (zum Beispiel) Shadern vorformatiert, und Sie können auch einen Offset innerhalb der Quelle buffer angeben, von dem kopiert werden soll, und die Anzahl der Scheitelpunkte, die die endgültige buffer haben soll. Beachten Sie, dass, wenn die Anzahl der Scheitelpunkte nicht mit denen übereinstimmt, die kopiert werden, Sie möglicherweise beschädigte Scheiteldaten erhalten.

Syntax:

vertex_create_buffer_from_buffer_ext(buffer, format, src_offset, vert_num);

Argument Typ Beschreibung
buffer Puffer-ID The buffer to create the vertex buffer from.
format Primitiver Typ Konstante The primitive vertex format to use.
src_offset Real The offset within the the source buffer to copy from.
vert_num Real The number of vertices the buffer should have.

Rückgabe:

Vertex Buffer ID

Beispiel:

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat, 0, 512);

Die obige Funktion code erstellt ein neues Scheitelpunktformat und erzeugt dann einen neuen Scheitelpunkt buffer aus einem zuvor erstellten regulären buffer, wobei das benutzerdefinierte Scheitelpunktformat mit einem Offset von 0 darauf angewendet wird. Die Funktion teilt dem neuen Scheitelpunkt buffer mit, dass er 512 Scheitelpunkte erwarten soll.
