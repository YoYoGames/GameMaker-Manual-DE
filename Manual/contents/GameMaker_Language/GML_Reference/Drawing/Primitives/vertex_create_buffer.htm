vertex_create_buffer

Mit dieser Funktion können Sie einen neuen Vertex buffer erstellen. Dabei handelt es sich um ein spezielles , von GameMaker erstelltes buffer, das für die Verwendung bei der Erstellung von Primitiven (z. B. für die Verwendung mit Shadern) vorformatiert ist. Die Funktion gibt einen Index für buffer zurück, der bei allen weiteren Aufrufen der Funktion verwendet werden sollte.

Wenn Sie einen mit dieser Funktion erstellten Vertex buffer verwenden, rufen Sie einfach vertex_begin() auf, um ihm Scheitelpunktdaten zuzuweisen und damit zu beginnen, Ihr benutzerdefiniertes Primitiv zu definieren, das dann in buffer für die Übergabe an den Shader bereitgehalten wird. Die buffer kann bei Bedarf wiederverwendet werden (es sei denn, Sie haben die vertex_freeze() Funktion), wobei jeder Aufruf von vertex_begin() die vorherigen Daten von buffer löscht, um die neuen Daten aufzunehmen.

Syntax:

vertex_create_buffer();

Rückgabe:

Vertex Buffer ID

Beispiel:

v_buff = vertex_create_buffer();

Die obige code erstellt einen neuen Scheitelpunkt buffer und speichert dessen Handle in der Variablen "v_buff".
