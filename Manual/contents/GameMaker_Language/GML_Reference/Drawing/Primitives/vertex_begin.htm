vertex_begin

Mit dieser Funktion beginnen Sie die Definition eines benutzerdefinierten Primitivs. Sie weisen eine buffer zu, in die das Primitiv geschrieben wird, und das zu verwendende Scheitelpunktformat (das zuvor mit den Scheitelpunktformatfunktionen definiert wurde). Anschließend definieren Sie die erforderlichen Punkte für jeden Scheitelpunkt des Primitivs, bevor Sie vertex_end() aufrufen, um die Erstellung des Primitivs abzuschließen.

Syntax:

vertex_begin(buffer, format);

Argument Typ Beschreibung
buffer Vertex Buffer ID The buffer to be written to.
format Scheitelpunkt-Format-ID The vertex format to use.

Rückgabe:

N/A

Beispiel:

vertex_format_begin();
vertex_format_add_position();
vertex_format_add_colour();
vertex_format_add_textcoord();
v_format = vertex_format_end();
v_buff = vertex_create_buffer();
vertex_begin(v_buff, v_format);

Die obige code definiert ein neues Scheitelpunktformat, erstellt ein neues buffer und startet den Definitionsprozess eines neuen Primitivs.
