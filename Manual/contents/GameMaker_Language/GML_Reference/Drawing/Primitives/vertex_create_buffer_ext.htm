vertex_erstellen_puffer_ext

Wie bei der Funktion vertex_create_buffer()erzeugt diese Funktion einen neuen Vertex buffer. Dabei handelt es sich um ein spezielles , von GameMaker erstelltes buffer, das für die Verwendung bei der Erstellung von Primitiven zur Verwendung mit Shadern vorformatiert ist. Sie können eine anfängliche Startgröße für buffer (in Bytes) angeben, und es wird ein Index für buffer zurückgegeben, der dann in allen weiteren Aufrufen von buffer verwendet werden sollte.

Syntax:

vertex_create_buffer_ext(size);

ArgumentTyp Beschreibung
size The initial size of the buffer (in bytes).

Rückgabe:

Vertex Buffer ID

Beispiel:

v_buff = vertex_create_buffer_ext(1024 * 1024);

Die obige code erstellt einen neuen Vertex buffer mit einer Größe von zunächst 1MB und speichert dessen Handle in der Variablen "v_buff".
