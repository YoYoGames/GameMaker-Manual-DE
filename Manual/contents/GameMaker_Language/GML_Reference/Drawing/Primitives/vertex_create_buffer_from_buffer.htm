vertex_erstellen_puffer_aus_puffer

Wie bei der Funktion vertex_create_buffer()erstellt diese Funktion einen neuen Scheitelpunkt buffer, nur dass jetzt die Scheitelpunktdaten, die sie speichert, von der regulären buffer kopiert werden, die als Quelle angegeben ist. Die erstellte buffer ist eine spezielle wachsende buffer, die mit dem Scheitelpunktformat vorformatiert ist, um Primitive für die Verwendung mit (zum Beispiel) Shadern zu erstellen. Diese Funktion erfordert, dass Sie den Zeiger auf eine zuvor erstellte reguläre buffer und ein Scheitelpunktformat, das auf die kopierten Daten angewendet werden soll, angeben.

Syntax:

vertex_create_buffer_from_buffer(buffer, format);

Argument Typ Beschreibung
buffer Puffer-ID The buffer to create the vertex buffer from.
format Primitiver Typ Konstante The primitive vertex format to use.

Rückgabe:

Vertex Buffer ID

Beispiel:

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat);

Die obige code wird ein neues Scheitelpunktformat erstellen und dann einen neuen Scheitelpunkt buffer aus einem zuvor erstellten regulären buffer erstellen und das benutzerdefinierte Scheitelpunktformat darauf anwenden.
