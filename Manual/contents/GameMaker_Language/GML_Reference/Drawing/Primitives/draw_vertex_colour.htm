draw_vertex_colour

Diese Funktion definiert die Position eines Scheitelpunkts für ein Primitiv, mit seiner eigenen Farbe und Alpha-Einstellung. Das endgültige Aussehen des Primitivs hängt von dem zum Zeichnen gewählten Primitivtyp und der Reihenfolge ab, in der Sie die Scheitelpunkte dazu hinzufügen (siehe draw_primitive_begin() für weitere Informationen). Die Scheitelpunkte mit unterschiedlichen Farben und Alphas gehen fließend ineinander über. Um das Primitiv zu beenden und zu zeichnen, müssen Sie den Befehl draw_primitive_end().

Syntax:

draw_vertex_colour(x, y, col, alpha)

ArgumentTyp Beschreibung
x The x coordinate of the vertex.
y The y coordinate of the vertex.
col The colour to draw this vertex with.
alpha The alpha to draw this vertex with (0-1).

Rückgabe:

Beispiel:

draw_primitive_begin(pr_trianglelist);
draw_vertex_colour(100, 100, c_blue, 0.1);
draw_vertex_colour(100, 200, c_red, 0.1);
draw_vertex_colour(150, 150, c_green, 1);
draw_primitive_end();

Die obige code zeichnet ein halbtransparentes Dreieck, bei dem jeder Scheitelpunkt eine andere Farbe hat.
