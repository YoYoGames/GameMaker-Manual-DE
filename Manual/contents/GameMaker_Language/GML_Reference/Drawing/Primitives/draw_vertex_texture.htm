draw_vertex_texture

Diese Funktion definiert die Position eines texturierten Scheitelpunktes für ein Primitiv. Das endgültige Aussehen des Primitivs hängt von dem zum Zeichnen gewählten Primitivtyp ab (siehe draw_primitive_begin() für weitere Informationen), von der Reihenfolge, in der Sie die Scheitelpunkte hinzufügen, und von der Position des Startpunkts, den Sie für die Textur festlegen. Um das Primitiv zu beenden und zu zeichnen, müssen Sie die Funktion draw_primitive_end().

Sie können sich auch dafür entscheiden, nur einen Teil der Textur über die Oberfläche des Primitivs zu zeichnen, indem Sie die entsprechende xtex/ytex-Position für die Abtastung festlegen (diese Koordinaten werden auch als UV-Punkte bezeichnet). Dieser Punkt wird durch einen Wert zwischen 0 und 1 definiert, wobei die obere linke Ecke des zu verwendenden Bildes der (0,0)-Punkt und die untere rechte Ecke der (1,1)-Punkt ist. Auf diese Weise können Sie ein Primitiv mit nur einem Teil des Quellbild-Assets texturieren.

Beim Zeichnen eines texturierten Primitivs wird die Textur mit der durch draw_set_colour eingestellten Farbe überblendet . Das bedeutet, dass zum Zeichnen der Textur ohne Überblendung diese auf c_white gesetzt werden sollte, wie im folgenden Beispiel gezeigt.

HINWEIS Damit eine Textur wiederholt werden kann, muss sie eine Zweierpotenz der Größe sein, d.h. 32x32, 128x128, usw...

Syntax:

draw_vertex_texture(x, y, xtex, ytex)

Argument Typ Beschreibung
x Real The x coordinate of the vertex.
y Real The y coordinate of the vertex.
xtex Real The width of the texture sample to use.
ytex Real The height of the texture sample to use.

Rückgabe:

N/A

Beispiel:

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

Die obige code zeichnet einen Dreiecksstreifen mit 4 Scheitelpunkten (der ein Rechteck ergibt), der mit der in der Variable "tex" gespeicherten Textur texturiert ist, und die gesamte Textur wird verwendet, um das fertige Primitiv abzudecken.
