vertex_format_delete

Diese Funktion muss immer dann aufgerufen werden, wenn Sie die Verwendung der erstellten vector Formate beendet haben. Sie geben den Wert der Format-ID an (wie er von der Funktion vector_format_end()zurückgegeben wird), und diese Funktion gibt den damit verbundenen Speicher frei. Wenn Sie nach dem Aufruf dieser Funktion versuchen, das Format erneut zu verwenden, erhalten Sie eine Fehlermeldung.

Syntax:

vertex_format_delete(formatID);

Argument Typ Beschreibung
formatID Scheitelpunkt-Format-ID The vertex format to delete

Rückgabe:

N/A

Beispiel:

vertex_format_delete(myFormat);

Die obige code wird das in der Variablen "myFormat" erstellte Scheitelpunktformat aus dem Speicher entfernen.
