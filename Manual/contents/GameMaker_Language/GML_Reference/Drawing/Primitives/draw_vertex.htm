draw_vertex

Diese Funktion definiert die Position eines Scheitelpunkts für ein Primitiv. Das endgültige Aussehen des Primitivs hängt von dem zum Zeichnen gewählten Primitivtyp und der Reihenfolge ab, in der Sie die Scheitelpunkte dazu hinzufügen. Siehe draw_primitive_begin() für weitere Informationen. Um das Primitiv zu beenden und zu zeichnen, müssen Sie den Befehl draw_primitive_end().

Syntax:

draw_vertex(x, y)

ArgumentTyp Beschreibung
x The x coordinate of the vertex.
y The y coordinate of the vertex.

Rückgabe:

Beispiel:

draw_primitive_begin(pr_trianglelist);
draw_vertex(100, 100);
draw_vertex(100, 200);
draw_vertex(150, 150);
draw_primitive_end();

Die obige code zeichnet ein einfaches primitives Dreieck.
