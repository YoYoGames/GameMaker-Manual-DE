vertex_get_buffer_size

Mit dieser Funktion können Sie die Größe des angegebenen Vertex buffer in Bytes ermitteln.

Syntax:

vertex_get_buffer_size(buffer);

Argument Typ Beschreibung
buffer Vertex Buffer ID The buffer to get the size of.

Rückgabe:

Real

Beispiel:

bufferBytes = vertex_get_buffer_size(buff);

Die obige code ermittelt die Anzahl der Bytes, die von dem angegebenen Vertex buffer verwendet werden und speichert den Wert in der Variablen "bufferBytes".
