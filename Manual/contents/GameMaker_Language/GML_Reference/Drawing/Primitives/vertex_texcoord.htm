vertex_texcoord

Diese Funktion legt die Texturkoordinaten fest, die für den Scheitelpunkt verwendet werden sollen, der derzeit für das benutzerdefinierte Primitiv definiert ist. Sie geben die buffer an, in die die Daten geschrieben werden sollen, sowie die zu verwendende UV-Position innerhalb der Textur.

Syntax:

vertex_texcoord(buffer, u, v);

Argument Typ Beschreibung
buffer Vertex Buffer ID The buffer to write the information to.
u Real The first texture coordinate to use (0 - 1).
v Real The second texture coordinate to use (0 - 1).

Rückgabe:

N/A

Beispiel:

vertex_texcoord(buff, 0, 0);

Die obige code setzt die UV-Werte des zu definierenden Scheitelpunktes.
