scheitelpunkt_position

Diese Funktion fügt 2D-Positionsdaten zu dem Scheitelpunkt hinzu, der gerade für das benutzerdefinierte Primitiv definiert wird. Sie geben die buffer an, in die die Daten geschrieben werden sollen, sowie die x- und y-Position zum Zeichnen.

Syntax:

vertex_position(buffer, x, y);

Argument Typ Beschreibung
buffer Vertex Buffer ID The buffer to write the information to.
x Real The x position.
y Real The y position.

Rückgabe:

N/A

Beispiel:

vertex_position(buff, x - 100, y - 125);

Die obige code legt die Position des zu definierenden Scheitelpunkts fest.
