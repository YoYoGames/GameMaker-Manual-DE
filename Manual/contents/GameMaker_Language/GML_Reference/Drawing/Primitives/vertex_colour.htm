vertex_colour

Wenn Ihr definiertes Scheitelpunktformat einen Farbwert enthält, können Sie diese Funktion verwenden, um diese Daten dem Scheitelpunkt hinzuzufügen, der für das aktuelle Primitiv definiert ist. Die Funktion benötigt eine buffer, um die Daten zu speichern, und nimmt entweder eine Farbkonstante oder einen Hex-Wert (unter Verwendung des GameMaker-Standardformats BGR, z. B. $FF0000 für Blau) sowie einen Alphawert von 0 (transparent) bis 1 (völlig undurchsichtig).

Syntax:

vertex_colour(buffer, colour, alpha);

Argument Typ Beschreibung
buffer Vertex Buffer ID The buffer to write the information to.
colour Farbe The colour for this vertex (can be a constant or a hex value).
alpha Real The alpha value for the vertex (from 0 to 1).

Rückgabe:

N/A

Beispiel:

vertex_colour(b, c_white, 1);

Die obige code setzt den zu definierenden Scheitelpunkt auf Weiß mit einem Alphawert von 1.
