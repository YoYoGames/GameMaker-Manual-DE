vertex_delete_buffer

Diese Funktion kann verwendet werden, um einen zuvor erstellten Vertex buffer (siehe vertex_create_buffer()) aus dem Systemspeicher zu entfernen. Dabei werden auch alle Scheitelpunktdaten für alle benutzerdefinierten Primitive entfernt, die sie enthielten.

Syntax:

vertex_delete_buffer(buffer);

Argument Typ Beschreibung
buffer Vertex Buffer ID The vertex buffer to be removed.

Rückgabe:

N/A

Beispiel:

vertex_delete_buffer(v_buff);

Mit der obigen code wird die in der Variablen "v_buff" indizierte buffer aus dem Speicher gelöscht.
