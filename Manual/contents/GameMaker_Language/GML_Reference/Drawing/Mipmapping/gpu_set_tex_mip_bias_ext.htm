gpu_set_tex_mip_bias_ext

Mit dieser Funktion können Sie den Mipmap-Bias-Wert für einen bestimmten Shader-Sampler einstellen. Sie geben den Indexwert für den Shader-Sampler an (wie er von der Funktion shader_get_sampler_index()zurückgegeben wird, und dann den Bias-Wert, wobei 0 für keinen Bias steht, 1 für die erste Mipmap, 2 für die zweite Mipmap usw... Dies steuert die Geschwindigkeit, mit der die Mipmap ausgetauscht wird, und macht die Shader-Texturen im Allgemeinen umso unschärfer, je höher der Wert und je größer die "Entfernung" ist, die betrachtet wird. Beachten Sie, dass diese Funktion auch negative Werte annehmen kann. In diesem Fall werden die Shader-Texturen über eine größere Entfernung schärfer, je niedriger der Wert ist.

Syntax:

gpu_set_tex_mip_bias_ext(sampler_index, bias);

ArgumentTyp Beschreibung
sampler_index The index of the shader sampler to get
bias The mipmap bias value to use (default: 0)

Rückgabe:

Beispiel:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if (gpu_get_tex_mip_bias_ext(_sampleIndex) != 0)
{
    gpu_set_tex_mip_bias_ext(_sampleIndex, 0);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

Die obige code setzt den Mip-Filter-Bias für den angegebenen Shader-Textur-Sampler auf 0, wenn er nicht bereits auf 0 gesetzt wurde.
