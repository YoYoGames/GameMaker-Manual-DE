gpu_set_tex_mip_bias

Mit dieser Funktion können Sie den Mipmap-Bias-Wert einstellen, wobei ein Wert von 0 keiner Verzerrung entspricht, 1 der ersten Mipmap, 2 der zweiten Mipmap usw.. Dies steuert die Rate, mit der die Mipmap ausgetauscht wird, und macht die gerenderten Texturen im Allgemeinen umso unschärfer, je höher der Wert und je größer die "Entfernung" ist, die betrachtet wird. Beachten Sie, dass diese Funktion auch negative Werte annehmen kann. In diesem Fall werden die gerenderten Texturen über eine größere Entfernung schärfer, je niedriger der Wert ist.

Syntax:

gpu_set_tex_mip_bias(bias);

ArgumentTyp Beschreibung
bias The mipmap bias value to use (default: 0)

Rückgabe:

Beispiel:

if gpu_get_tex_mip_bias() != 0
{
    gpu_set_tex_mip_bias(0);
}

Die obige code prüft den aktuellen Mipmap-Bias, und wenn er nicht 0 ist, wird er auf 0 gesetzt.
