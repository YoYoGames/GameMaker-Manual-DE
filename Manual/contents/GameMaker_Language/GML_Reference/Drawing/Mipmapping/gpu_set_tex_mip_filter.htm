gpu_set_tex_mip_filter

Mit dieser Funktion können Sie den aktuellen mip-Filtermodus auf einen der drei unterstützten Typen einstellen. Sie geben die Konstante an, die sich auf den gewünschten mip-Filtermodus bezieht, wobei Sie zwischen Punktfilterung (Standardeinstellung, d. h. keine Filterung), linearer Filterung (auch als trilineare Filterung bezeichnet) oder aniostroper Filterung wählen können (siehe die Konstantentabelle unten).

Konstante Beschreibung
tf_point Dadurch wird die Filterung zwischen den Mipmap-Ebenen deaktiviert, was zu sichtbaren Texturübergängen führen kann, aber die beste Leistung bietet.
tf_linear Dies ermöglicht eine lineare Filterung zwischen den Mipmap-Ebenen (auch bekannt als trilineare Filterung), wodurch die Texturübergänge geglättet werden, was jedoch zu einer geringfügigen Beeinträchtigung der Leistung führt.
tf_anisotropic Dadurch wird die anisotrope Filterung zwischen den Mipmap-Ebenen aktiviert, was die Qualität der Texturübergänge erheblich verbessert und die bei anderen Filterungsmodi sichtbare Unschärfe reduzieren kann, allerdings hat dies den größten Einfluss auf die Leistung.

Syntax:

gpu_set_tex_mip_filter(filter);

Argument Typ Beschreibung
filter Mipmapping Filter Konstante The mip filter mode to use (a constant, default: tf_point).

Rückgabe:

N/A

Beispiel:

if keyboard_check(vk_enter)
{
    switch(gpu_get_tex_mip_filter())
    {
        case tf_point: gpu_set_tex_mip_filter(tf_linear); break;
        case tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); break;
        case tf_anisotropic: gpu_set_tex_mip_filter(tf_point); break;
    }
}

Das obige code prüft die Tastatur und wenn die angegebene Taste gedrückt wird, wird der aktuelle mip-Filter abgerufen und der Wert auf den nächsten umgeschaltet, wobei die verschiedenen Modi durchlaufen werden.
