gpu_set_tex_min_mip

Mit dieser Funktion können Sie die minimale Mipmap-Stufe einstellen, die derzeit verwendet wird, wobei ein Wert von 0 die höchste Auflösung ist, 1 bedeutet, dass die erste Mipmap verwendet wird, 2 ist die zweite usw...

Syntax:

gpu_set_tex_min_mip(minmip);

ArgumentTyp Beschreibung
minmip The minimum mipmap level to use

Rückgabe:

Beispiel:

if gpu_get_tex_min_mip() != 0
{
    gpu_set_tex_min_mip(0);
}

Die obige code prüft den aktuellen minimalen Mipmap-Level und wenn dieser nicht 0 ist, wird er auf 0 gesetzt.
