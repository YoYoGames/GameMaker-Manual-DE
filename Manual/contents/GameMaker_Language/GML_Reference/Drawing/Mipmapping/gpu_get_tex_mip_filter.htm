gpu_get_tex_mip_filter

Mit dieser Funktion können Sie den aktuellen mip-Filtermodus abfragen. Der von der Funktion zurückgegebene Moduswert ist eine der unten aufgeführten Konstanten.

Syntax:

gpu_get_tex_mip_filter();

Rückgabe:

Mipmapping Filter Constant (listed below):

Mipmapping Filter Konstante
Konstante Beschreibung
tf_point Das bedeutet, dass das Überblenden zwischen den Mipmap-Ebenen deaktiviert ist, was zu sichtbaren Texturübergängen führen kann, aber die beste Leistung bringt.
tf_linear Das bedeutet, dass das Überblenden zwischen den Mipmap-Ebenen aktiviert ist (dies ist auch als trilineare Filterung bekannt), wodurch die Texturübergänge geglättet werden, was jedoch zu einer geringfügigen Beeinträchtigung der Leistung führt.
tf_anisotropic Dies bedeutet, dass die anisotrope Filterung aktiviert ist, was die Qualität der Texturübergänge erheblich verbessert und die bei anderen Filterungsmodi sichtbare Unschärfe reduzieren kann, aber die Leistung am stärksten beeinträchtigt.

Beispiel:

if keyboard_check(vk_enter)
{
    switch(gpu_get_tex_mip_filter())
    {
        case tf_point: gpu_set_tex_mip_filter(tf_linear); break;
        case tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); break;
        case tf_anisotropic: gpu_set_tex_mip_filter(tf_point); break;
    }
}

Das obige code prüft die Tastatur und wenn die angegebene Taste gedrückt wird, wird der aktuelle mip-Filter abgerufen und der Wert auf den nächsten umgeschaltet, wobei die verschiedenen Modi durchlaufen werden.
