gpu_set_tex_mip_enable_ext

Mit dieser Funktion können Sie einstellen, ob Mipmapping ausgeschaltet, für alles eingeschaltet oder nur für die im Texturgruppenmanager ausgewählten Texturgruppen eines Shader-Samplers eingeschaltet werden soll. Sie geben den Indexwert für den Shader-Sampler an (wie er von der Funktion shader_get_sampler_index()zurückgegeben wird, und dann eine der unten aufgeführten Konstanten.

Konstante Beschreibung
mip_off Mipmapping ist deaktiviert.
mip_on Mipmapping für alle Texturen ist aktiviert.
mip_markedonly Mipmapping ist für Texturen aktiviert, die es in den Texturgruppenoptionen aktiviert haben (Standard).

Syntax:

gpu_set_tex_mip_enable_ext(sampler_index, setting);

Argument Typ Beschreibung
sampler_index Shader Sampler Handle The index of the shader sampler to get
setting Mipmapping-Konstante The mipmap setting (a constant, default: mip_markedonly)

Rückgabe:

N/A

Beispiel:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if (gpu_get_tex_mip_enable_ext(_sampleIndex) != mip_on)
{
    gpu_set_tex_mip_enable_ext(_sampleIndex, mip_on);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

Die obige code aktiviert Mipmapping für den angegebenen Shader-Textur-Sampler, wenn es nicht bereits aktiviert ist.
