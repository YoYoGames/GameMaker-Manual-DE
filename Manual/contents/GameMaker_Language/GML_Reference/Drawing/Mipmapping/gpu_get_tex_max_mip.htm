gpu_get_tex_max_mip

Mit dieser Funktion können Sie die aktuell eingestellte maximale Mipmap-Stufe ermitteln, die verwendet werden soll, wobei ein Wert von 0 die höchste Auflösung ist, 1 bedeutet, dass die erste Mipmap verwendet wird, 2 ist die zweite usw.. Beachten Sie, dass dies sehr nützlich sein kann, um Bleeding-Artefakte beim Rendern von Texturen zu vermeiden. Wenn Sie z.B. den Seitenrand der Textur auf 8px einstellen und dann die maximale Mipmap-Stufe auf 3 setzen, wird sichergestellt, dass Sie bei größeren Renderabständen keine Bleeding-Probleme bekommen.

Syntax:

gpu_get_tex_max_mip();

Rückgabe:

(default: 16)

Beispiel:

if gpu_get_tex_max_mip() != 4
{
    gpu_set_tex_max_mip(4);
}

Die obige code prüft den aktuellen maximalen Mipmap-Level und setzt ihn, falls er nicht 4 ist, auf 4.
