gpu_get_tex_max_aniso

Mit dieser Funktion können Sie den aktuellen Höchstwert der Anisotropie ermitteln, wenn Sie den Filtermodus tf_anisotropic verwenden (siehe gpu_get_tex_mip_filter() für weitere Informationen). Der zurückgegebene Wert liegt zwischen 1 und 16.

Syntax:

gpu_get_tex_max_aniso();

Rückgabe:

(default: 16)

Beispiel:

if gpu_get_tex_max_aniso() != 8
{
    gpu_set_tex_max_aniso(8);
}

Die obige code prüft die aktuelle maximale anisotrope Filterstufe, und wenn sie nicht 8 ist, wird sie auf 8 gesetzt.
