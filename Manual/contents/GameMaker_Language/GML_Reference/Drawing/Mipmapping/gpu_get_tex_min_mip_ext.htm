gpu_get_tex_min_mip_ext

Mit dieser Funktion können Sie die minimale Mipmap-Ebene ermitteln, die derzeit für einen bestimmten Shader-Sampler verwendet wird. Sie geben den Indexwert für den Shader-Sampler an (wie er von der Funktion shader_get_sampler_index()zurückgegeben wird, und die Funktion gibt einen Wert von 0 für volle Auflösung, 1 für die erste Mipmap, 2 für die zweite usw. zurück.

Syntax:

gpu_get_tex_min_mip_ext(sampler_index);

ArgumentTyp Beschreibung
sampler_index The index of the shader sampler to get

Rückgabe:

(default: 0)

Beispiel:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if (gpu_get_tex_min_mip_ext(_sampleIndex) != 0)
{
    gpu_set_tex_min_mip_ext(_sampleIndex, 0);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

Die obige code setzt den minimalen Mipmap-Level für den angegebenen Shader-Textursampler auf 0, falls er nicht bereits auf 0 gesetzt wurde.
