gpu_set_tex_mip_enable

Mit dieser Funktion können Sie ändern, ob Mipmapping ausgeschaltet, für alles eingeschaltet oder nur für die im Texturgruppen-Manager ausgewählten Texturgruppen eingeschaltet ist. Die Funktion erfordert eine der unten aufgeführten Konstanten.

Konstante Beschreibung
mip_off Mipmapping ist deaktiviert.
mip_on Mipmapping für alle Texturen ist aktiviert.
mip_markedonly Mipmapping ist für Texturen aktiviert, die es in den Texturgruppenoptionen aktiviert haben (Standard).

Syntax:

gpu_set_tex_mip_enable(setting);

Argument Typ Beschreibung
setting Mipmapping-Konstante The mipmap setting (a constant, default: mip_markedonly)

Rückgabe:

N/A

Beispiel:

if gpu_get_tex_mip_enable != mip_on
{
    gpu_set_tex_mip_enable(mip_on);
}

Die obige code prüft, ob mipmapping aktiviert ist, und wenn nicht, wird es aktiviert.
