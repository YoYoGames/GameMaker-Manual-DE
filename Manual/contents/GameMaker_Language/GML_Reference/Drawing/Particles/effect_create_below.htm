effect_create_below

Mit dieser Funktion können Sie einen einfachen Effekt unter allen Instanzen Ihres Raums erstellen (er wird tatsächlich in einer Tiefe von 100000 erstellt). Wenn es sich bei dem Effekt um etwas anderes als ef_rain oder ef_snow handelt, können Sie eine x/y-Position definieren, um den Effekt zu erzeugen, und die Größe kann ein Wert von 0, 1 oder 2 sein, wobei 0 für klein, 1 für mittel und 2 für groß steht.

Es ist erwähnenswert, dass das Zeichnen dieser Effekte ein- und ausgeschaltet werden kann, sowie das Zeichnen angehalten werden kann, indem man die Funktionen part_system_automatic_draw() und part_system_automatic_update() mit dem entsprechenden Wert für den Partikelsystemindex (wobei 0 für Effekte unterhalb und 1 für Effekte oberhalb steht).

Die verfügbaren Konstanten für die verschiedenen Partikelarten sind:

Konstante Beispiel Beschreibung
ef_cloud An effect that creates random cloud particles of varying sizes
ef_ellipse An effect that creates expanding ellipses
ef_explosion An effect that creates expanding fading explosions
ef_firework An effect that creates multiple small particles to generate a firework explosion
ef_flare An effect that generates a brilliant point that flares up and fades out
ef_rain An effect that generates rain particles coming down from the top of the screen
ef_ring An effect that generates expanding and fading circles
ef_smoke An effect that generates little puffs of smoke
ef_smokeup An effect that creates a smoke plume that rises up the screen
ef_snow An effect that generates multiple snow particles falling down the screen
ef_spark An effect that generates a small spark
ef_star An effect that generates star particles

Syntax:

effect_create_below(kind, x, y, size, colour);

Argument Typ Beschreibung
kind Effekttyp Konstante The kind of effect (use one of the constants listed above).
x Real The x positioning of the effect if relevant.
y Real The y positioning of the effect if relevant.
size Real The size of the effect.
colour Farbe The colour of the effect.

Rückgabe:

N/A

Beispiel:

if speed > 0
{
    effect_create_below(ef_smoke, x, y, 0, c_gray);
}

Die obige code wird bei jedem Schritt, bei dem die Geschwindigkeit der Instanz an den x,y-Koordinaten der Instanz größer als 0 ist, eine kleine graue Rauchwolke erzeugen.
