part_type_direction

Diese Funktion wird verwendet, um die Richtung des Partikels zu bestimmen, wenn er erstellt wird, und kann auch verwendet werden, um die Richtung des Partikels während seiner Lebensdauer zu erhöhen oder zu verringern. Die Werte für die minimale und maximale Richtung sind standardmäßig auf 0 eingestellt, was bedeutet, dass sich das Partikel ohne Veränderung nach rechts bewegt. Diese Werte können jedoch geändert werden, um die Bewegungsrichtung entsprechend der Standard-GameMaker-Richtungen 0 nach rechts, 90 nach oben, 180 nach links und 270 nach unten zu begrenzen. Wenn du sie auf denselben Wert setzt, bewegen sich die Partikel nur in eine Richtung.

Sie können auch einen Inkrementwert festlegen, der während der Lebensdauer der Richtung einen bestimmten Gradbetrag addiert (bei einer positiven Zahl) oder subtrahiert (bei einer negativen Zahl). Dieser Wert kann ein Minimum von (+/-) 0,01 sein.

Schließlich können Sie den "Wiggle"-Faktor einstellen. Dies ist ein Wert, der bei jedem Schritt der Partikel-Lebensdauer zufällig zur Richtung addiert oder subtrahiert wird. Größere Werte sind natürlich stärker ausgeprägt als kleinere, und dieser Wert kann sogar negativ sein, wobei der maximale Bereich zwischen -20 und 20 liegt.

Syntax:

part_type_direction(ind, dir_min, dir_max, dir_incr, dir_wiggle);

Argument Typ Beschreibung
ind Partikel Typ ID The index of the particle type to change.
dir_min Real The minimum direction the particle can start at.
dir_max Real The maximum direction the particle can start at.
dir_incr Real How much the particle direction should increase or decrease per step.
dir_wiggle Real How much should randomly be added or subtracted from the particle's direction per step.

Rückgabe:

N/A

Beispiel:

part_type_shape(particle1, pt_shape_pixel);
part_type_size(particle1, 0.19, 0.79, 0.01, 0);
part_type_scale(particle1, 5, 5);
part_type_colour3(particle1, 32768, 8454016, 8454016);
part_type_alpha3(particle1, 0, 0.80, 0);
part_type_speed(particle1, 8, 10, -0.50, 1);
part_type_direction(particle1, 0, 359, 1, 20);
part_type_gravity(particle1, 0, 270);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 10, 50);

Die obige code stellt verschiedene Partikelwerte ein, einschließlich der Richtung, die ein Zufallswert zwischen 0 und 359 Grad sein wird. Es wird auch um 1 Grad jeden Schritt gehen und hat eine zufällige Menge hinzugefügt, um es von irgendwo zwischen 0 und 20 jeden Schritt zu.
