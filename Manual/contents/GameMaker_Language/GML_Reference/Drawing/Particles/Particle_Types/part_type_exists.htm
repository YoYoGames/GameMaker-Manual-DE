teil_art_vorhanden

Mit dieser Funktion kannst du überprüfen, ob der angegebene Partikeltyp im Spiel existiert oder nicht. Beachte, dass diese Funktion einen Fehler auslöst, wenn die zu prüfende Variable eine nicht initialisierte Variable ist (der sonst ein Partikeltyp seinen Index zugewiesen hätte).

Syntax:

part_type_exists(ind);

Argument Typ Beschreibung
ind Partikel Typ ID The index of the particle type to check for.

Rückgabe:

Boolean

Beispiel:

if !part_type_exists(global.p1)
{
    global.p1 = part_type_create();
}

Die obige code prüft, ob die globale Variable "p1" den Index eines Partikeltyps speichert, und wenn nicht, erstellt sie einen.
