part_type_step

Diese Funktion kann verwendet werden, um einen anderen Partikeltyp in jedem Schritt bis zum Ende seiner Lebensdauer auszustoßen. Du kannst einen negativen Wert für die Schrittnummer angeben und ein Partikel wird in einem zufälligen Intervall basierend auf diesem Wert emittiert, so dass zum Beispiel ein Wert von -3 eine 1:3 Chance hat, ein Partikel pro Schritt zu emittieren.

HINWEIS: Verwenden Sie NIEMALS denselben Partikeltyp, da dies zu einer unendlichen loop führt und das Spiel innerhalb von Sekunden zum Absturz bringen kann! Seien Sie außerdem sehr vorsichtig, wenn Sie diese Funktion verwenden, da sie die Anzahl der Partikel auf dem Bildschirm stark erhöhen kann und Ihr Spiel dadurch merklich verlangsamt wird.

Syntax:

part_type_step(ind, step_number, step_type);

Argument Typ Beschreibung
ind Partikel Typ ID The index of the particle type to change.
step_number Real The quantity of the new particle to create per step.
step_type Partikel Typ ID The index of the particle type to create per step.

Rückgabe:

N/A

Beispiel:

particle2 = part_type_create();
part_type_shape(particle2, pt_shape_smoke);
part_type_size(particle2, 0.01, 0.01, 0.01, 0);
part_type_scale(particle2, 1, 1);
part_type_colour3(particle2, 4227327, 8454143, 4227327);
part_type_alpha3(particle2,1, 0.50, 0.80);
part_type_speed(particle2, 0, 0, 0, 0);
part_type_direction(particle2, 0, 359, 0, 0);
part_type_gravity(particle2, 0, 270);
part_type_orientation(particle2, 0, 0, -2, 0, 1);
part_type_blend(particle2, 1);
part_type_life(particle2, 160, 160);
particle3 = part_type_create();
part_type_shape(particle3, pt_shape_flare);
part_type_size(particle3, 0.20, 0.20, 0.02, 0);
part_type_scale(particle3, 1, 1);
part_type_colour3(particle3, 16777215, 8454143, 16777215);
part_type_alpha3(particle3, 0, 0.50, 0);
part_type_speed(particle3, 3, 3, -0.03, 0);
part_type_direction(particle3, 0, 359, 0, 0);
part_type_gravity(particle3, 0, 270);
part_type_orientation(particle3, 0, 0, 0, 0, 1);
part_type_blend(particle3, 1);
part_type_life(particle3, 40, 80);
part_type_step(particle2, 2, particle3);

Die obige code erstellt und definiert zwei Partikel und setzt den Partikel, der in der Variable "particle2" indiziert ist, um 2 Partikel von "particle3" in jedem Schritt seiner Lebensdauer zu erstellen.
