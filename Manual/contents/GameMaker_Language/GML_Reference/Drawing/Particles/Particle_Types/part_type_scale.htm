teil_art_skala

Mit dieser Funktion wird die horizontale und vertikale Skalierung des Partikels festgelegt, bevor andere Größenänderungen vorgenommen werden. Diese Funktion arbeitet mit dem Verhältnis der sprite Größe, so dass ein Wert von 1 (der Standardwert) ein 1:1 Verhältnis darstellt, was bedeutet, dass 0,5 die Hälfte und 2 das Doppelte wäre.

Syntax:

part_type_scale(ind, xscale, yscale);

Argument Typ Beschreibung
ind Partikel Typ ID The index of the particle type to change.
xscale Real The horizontal scaling of the base particle.
yscale Real The vertical scaling of the base particle.

Rückgabe:

N/A

Beispiel:

part_type_shape(particle1, pt_shape_flare);
part_type_size(particle1, 2, 2, 0.10, 1);
part_type_scale(particle1, 8, 0.50);
part_type_colour3(particle1, 8388608, 4227327, 8454143);
part_type_alpha3(particle1, 0.10, 0.10, 0);
part_type_speed(particle1, 0, 1, 0.18, 20);
part_type_direction(particle1, 0, 359, 0, 0);
part_type_orientation(particle1, 0, 0, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 10, 40);

Die obige code stellt verschiedene Partikelwerte ein, einschließlich der Skala, die die x-Skala auf das Achtfache und die y-Skala auf die Hälfte der Größe einstellt.
