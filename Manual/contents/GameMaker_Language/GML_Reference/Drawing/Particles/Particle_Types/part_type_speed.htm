teil_art_geschwindigkeit

Diese Funktion wird verwendet, um die Bewegungsgeschwindigkeit des Partikels bei seiner Erstellung festzulegen, und kann auch verwendet werden, um die Geschwindigkeit des Partikels während seiner Lebensdauer zu erhöhen oder zu verringern. Die Werte für die minimale und maximale Geschwindigkeit sind standardmäßig auf 1 eingestellt, was bedeutet, dass sich das Partikel bei jedem Schritt um 1 Pixel bewegt. Wenn sie auf denselben Wert eingestellt sind, ist die Anfangsgeschwindigkeit für alle Partikel gleich, aber wenn sie auf unterschiedliche Werte eingestellt sind, startet jedes Partikel mit einer zufälligen Geschwindigkeit innerhalb des angegebenen Bereichs. Wenn du zum Beispiel ein Minimum von 3 und ein Maximum von 6 angibst, werden alle Partikel mit einer Geschwindigkeit zwischen drei und sechs Pixeln pro Schritt erstellt.

Sie können auch einen Inkrementwert festlegen, der das Partikel während seiner Lebensdauer schneller (bei einer positiven Zahl) oder langsamer (bei einer negativen Zahl) werden lässt, wobei der angegebene Betrag bei jedem Schritt zur Partikelgeschwindigkeit addiert wird. Dieser Wert kann ein Minimum von (+/-)0,01 sein.

Schließlich können Sie den "Wiggle"-Faktor einstellen. Dies ist ein Wert, der bei jedem Schritt der Partikel-Lebensdauer zufällig zur Geschwindigkeit addiert oder subtrahiert wird. Größere Werte sind natürlich stärker ausgeprägt als kleinere, und dieser Wert kann sogar negativ sein, wobei der maximale Bereich zwischen -20 und 20 liegt.

Syntax:

part_type_speed(ind, speed_min, speed_max, speed_incr, speed_wiggle);

Argument Typ Beschreibung
ind Partikel Typ ID The index of the particle type to change.
speed_min Real The minimum speed the particle can start at.
speed_max Real The maximum speed the particle can start at.
speed_incr Real How much the particle speed should increase or decrease per step.
speed_wiggle Real How much should randomly be added or subtracted from the particle's speed per step.

Rückgabe:

N/A

Beispiel:

part_type_shape(particle2, pt_shape_flare);
part_type_size(particle2, 0.01, 0.50, 0.01, 20);
part_type_scale(particle2, 0.10, 0.10);
part_type_colour3(particle2, 4210816, 4227327, 8240895);
part_type_alpha3(particle2, 0.20, 0.80, 0);
part_type_speed(particle2, 5, 10, -0.20, 20);
part_type_direction(particle2, 0, 359, -1, 20);
part_type_gravity(particle2, 0, 270);
part_type_orientation(particle2, 0, 359, 0, 20, 1);
part_type_blend(particle2, 1);
part_type_life(particle2, 1, 20);

Die obige code stellt verschiedene Partikelwerte ein, einschließlich der Geschwindigkeit, die eine minimale Geschwindigkeit von fünf und eine maximale Geschwindigkeit von zehn festlegt, wobei sich die Geschwindigkeit bei jedem Schritt um einen Wert von -0,2 verlangsamt. Die Geschwindigkeit wackelt außerdem bei jedem Schritt um einen Zufallswert zwischen 0 und 20.
