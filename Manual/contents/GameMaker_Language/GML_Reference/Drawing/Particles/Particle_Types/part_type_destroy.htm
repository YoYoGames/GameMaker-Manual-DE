teil_art_zerstören

Mit dieser Funktion kannst du den angegebenen Partikeltyp aus dem Spiel entfernen. Wenn du diese Funktion verwendest, verschwinden alle Partikel des angegebenen Typs aus dem Raum und der Partikel selbst wird aus dem Speicher entfernt, daher sollte diese Funktion nur verwendet werden, wenn du den Partikel nicht mehr benötigst.

Syntax:

part_type_destroy(ind);

Argument Typ Beschreibung
ind Partikel Typ ID The index of the particle type to destroy.

Rückgabe:

N/A

Beispiel:

if part_particles_count(smoke_sys) = 0
{
    part_type_destroy(smoke_part);
    part_system_destroy(smoke_sys);
    instance_destroy();
}
else alarm[0] = 1;

Die obige code prüft ein Partikelsystem, um zu sehen, ob derzeit Partikel im Raum sichtbar sind, und wenn nicht, werden die Partikel, das System und die Instanz zerstört.
