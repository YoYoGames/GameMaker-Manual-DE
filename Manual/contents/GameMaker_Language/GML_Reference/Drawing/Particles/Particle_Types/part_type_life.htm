part_type_life

Dies ist die Funktion, die bestimmt, wie lange jedes einzelne Partikel des angegebenen Typs auf dem Bildschirm bleibt. Wie bei anderen Partikelfunktionen geben Sie einen Minimal- und einen Maximalwert (in Schritten) an, und jede Partikel-Lebensdauer wird eine zufällige Anzahl von Schritten innerhalb des angegebenen Bereichs sein. Um alle Partikel mit der gleichen Lebensdauer zu haben, setzen Sie die beiden Werte gleich.

Syntax:

part_type_life(ind, life_min, life_max);

Argument Typ Beschreibung
ind Partikel Typ ID The index of the particle type to change.
life_min Real The minimum lifespan of the particles.
life_max Real The maximum lifespan of the particles.

Rückgabe:

N/A

Beispiel:

part_type_life(part_Flare, room_speed, room_speed * 2);

Die obige code setzt alle Partikel, die mit dem in der Variable "part_Flare" indizierten Partikeltyp erstellt werden, auf eine Lebensdauer zwischen 1 und 2 Sekunden, basierend auf der aktuellen Raumgeschwindigkeit.
