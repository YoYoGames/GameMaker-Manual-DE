part_type_clear

Mit dieser Funktion kannst du ein Partikel "zurücksetzen", indem du alle Werte für jede der Funktionen, die sich auf das Partikel beziehen (Leben, Farbe, Alpha, Ausrichtung etc...) auf ihre Standardwerte zurücksetzt. Beachte, dass diese Funktion keine Partikel, die gerade im Raum sichtbar sind, vom Bildschirm entfernt, dafür solltest du part_particles_clear().

Syntax:

part_type_clear(ind);

Argument Typ Beschreibung
ind Partikel Typ ID The index of the particle type to clear.

Rückgabe:

N/A

Beispiel:

part_type_clear(global.explode_part);

Die obige code setzt den in der globalen Variable "explode_part" indizierten Partikeltyp auf seine Standardwerte zurück.
