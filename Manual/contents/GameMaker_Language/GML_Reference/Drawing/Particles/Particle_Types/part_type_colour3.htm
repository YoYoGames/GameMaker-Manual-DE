teil_art_farbe3

Mit dieser Funktion kann ein dreifacher Farbverlauf für jedes erzeugte Partikel des angegebenen Typs festgelegt werden. Die erste Farbe ist diejenige, mit der alle Partikel beginnen, die zweite Farbe ist diejenige, die nach der Hälfte der Lebensdauer des Partikels überblendet wird, und die dritte Farbe ist diejenige, mit der der Partikel endet. Im Laufe der Lebensdauer des Partikels wird ein sanfter Farbübergang von einer Farbe zur anderen stattfinden.

Syntax:

part_type_colour3( ind, colour1, colour2, colour3 );

Argument Typ Beschreibung
ind Partikel Typ ID The index of the particle type to change.
colour1 Farbe The colour the particle will start at.
colour2 Farbe The colour the particle will be halfway through its lifespan.
colour3 Farbe The colour the particle will end at.

Rückgabe:

N/A

Beispiel:

part_type_colour3( part_Health, c_red, c_white, c_maroon);

Die obige code sorgt dafür, dass alle Partikel, die mit dem in der Variablen "part_Health" indizierten Typ erstellt werden, während ihrer Lebensdauer eine Farbmischung von rot über weiß bis kastanienbraun aufweisen.
