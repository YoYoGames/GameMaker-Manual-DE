teil_art_alpha1

Mit dieser Funktion kann ein Partikeltyp so eingestellt werden, dass er für die gesamte Lebensdauer jedes einzelnen Partikels einen einzigen Alphawert (Transparenz) hat, der zwischen 0 (transparent) und 1 (undurchsichtig) liegen kann.

Syntax:

part_type_alpha1(ind, alpha1);

Argument Typ Beschreibung
ind Partikel Typ ID The index of the particle type to change.
alpha1 Real The alpha of the particle.

Rückgabe:

N/A

Beispiel:

part_type_alpha1(global.Snow_Part, 0.5);

Die obige code setzt alle Partikel, die mit dem in der globalen Variable "Snow_Part" indizierten Partikeltyp erstellt werden, auf einen Alpha-Wert von 0,5 (halbtransparent).
