part_type_colour_hsv

Mit dieser Funktion können Sie einen Farbton, eine Sättigung und einen Wertebereich für alle Partikel des angegebenen Typs festlegen. Sie geben einen Minimal- und einen Maximalwert für jede der drei Komponenten an und die erzeugten Partikel haben eine zufällige Farbe, die auf dem angegebenen Parameterbereich basiert. Auf diese Weise können Sie Partikel mit gleichem Farbton, aber unterschiedlicher Sättigung oder mit unterschiedlichem Farbton, aber gleichem Wert (Luminanz) usw. erstellen. Alle Werte müssen zwischen 0 und 255 liegen.

Syntax:

part_type_colour_hsv(ind, hmin, hmax, smin, smax, vmin, vmax);

Argument Typ Beschreibung
ind Partikel Typ ID The index of the particle type to change.
hmin Real The minimum the final colour's hue component can be.
hmax Real The maximum the final colour's hue component can be.
smin Real The minimum the final colour's saturation component can be.
smax Real The maximum the final colour's saturation component can be.
vmin Real The minimum the final colour's value component can be.
vmax Real The maximum the final colour's value component can be.

Rückgabe:

N/A

Beispiel:

part_type_hsv(global.Stars, 0, 255, 0, 255, 255, 255 );

Die obige code legt fest, dass jedes emittierte Partikel des Partikeltyps, der in der globalen Variable "Stars" indiziert ist, unterschiedliche Farben und Sättigungen, aber denselben Wert (Leuchtkraft) hat.
