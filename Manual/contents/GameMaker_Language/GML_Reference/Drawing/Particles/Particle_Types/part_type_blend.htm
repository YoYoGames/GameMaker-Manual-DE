part_type_blend

Mit dieser Funktion können Sie festlegen, ob Ihre Partikel mit einem additiven Überblendungsmodus (true) gezeichnet werden oder nicht (false). Bei der additiven Überblendung handelt es sich um einen speziellen Überblendungsmodus, bei dem die Helligkeitswerte der einzelnen Partikel addiert werden, wenn sie sich überlappen, so dass helle Farben mit zunehmender Überlappung immer heller werden (bis sie weiß erscheinen) und dunkle Farben immer transparenter werden, bis Schwarz fast unsichtbar wird. Diese Funktion kann schöne Partikeleffekte erzeugen, ist aber nicht immer notwendig. Bei einer Rauchfahne zum Beispiel würde man die additive Überblendung ausschalten, um den Effekt grau zu halten, aber bei einem Flammeneffekt würde man sie wahrscheinlich einschalten, um die Partikel durchsichtiger und heller zu machen.

Syntax:

part_type_blend(ind, additive);

Argument Typ Beschreibung
ind Partikel Typ ID The index of the particle type to change.
additive Boolesche Whether the particles should be blended additively (true) or normally (false).

Rückgabe:

N/A

Beispiel:

part_type_blend(part_Fire, true);

Die obige code setzt alle Partikel, die mit dem in der Variable "part_Fire" indizierten Partikeltyp erstellt werden, auf einen additiven Mischmodus.
