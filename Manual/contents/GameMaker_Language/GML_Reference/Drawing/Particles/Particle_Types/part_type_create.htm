part_type_create

Mit dieser Funktion können Sie einen neuen Partikeltyp erstellen. Der Rückgabewert sollte in einer Variablen gespeichert werden, die in allen nachfolgenden Partikelfunktionen verwendet werden kann.

Syntax:

part_type_create();

Rückgabe:

Particle Type ID

Beispiel:

mypart = part_type_create();

Dadurch wird ein neuer Partikeltyp erstellt und sein Index in der Variablen "mypart" gespeichert.
