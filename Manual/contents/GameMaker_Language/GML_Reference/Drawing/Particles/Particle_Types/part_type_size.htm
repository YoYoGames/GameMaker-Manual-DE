teil_art_grösse

Diese Funktion wird verwendet, um die Größe des Partikels zu bestimmen, wenn er erstellt wird, und kann auch verwendet werden, um den Partikel im Laufe seiner Lebensdauer zu vergrößern oder zu verkleinern. Die Werte für die minimale und maximale Größe sind standardmäßig auf 1 gesetzt (was bedeutet, dass die Größe das gleiche Verhältnis wie das Basispartikel ist, nachdem die part_type_scale() Wenn sie auf denselben Wert eingestellt sind, ist die Anfangsgröße für alle Partikel gleich, aber wenn sie auf unterschiedliche Werte eingestellt sind, beginnt jedes Partikel mit einer zufälligen Größe innerhalb des Bereichs, z. B. wenn du ein Minimum von 0,5 und ein Maximum von 2 angibst, werden alle Partikel mit einer Größe zwischen der Hälfte und dem Doppelten der Basisskala erstellt.

Sie können auch einen Inkrementwert festlegen, der das Partikel sprite während der Lebensdauer des Partikels wachsen (wenn es sich um eine positive Zahl handelt) oder schrumpfen lässt (wenn es sich um eine negative Zahl handelt), wobei der angegebene Betrag bei jedem Schritt zur Partikelgröße hinzugefügt wird. Dieser Wert kann ein Minimum von (+/-)0,01 sein.

Schließlich können Sie den "Wackelfaktor" festlegen. Dies ist ein Wert, der bei jedem Schritt der Partikellebensdauer zufällig zur Größe addiert oder subtrahiert wird. Größere Werte sind natürlich stärker ausgeprägt als kleinere, und dieser Wert kann sogar negativ sein, wobei der maximale Bereich zwischen -20 und 20 liegt.

Syntax:

part_type_size(ind, size_min, size_max, size_incr, size_wiggle);

Argument Typ Beschreibung
ind Partikel Typ ID The index of the particle type to change.
size_min Real The minimum size the particle can start at.
size_max Real The maximum size the particle can start at.
size_incr Real How much the particle should increase or decrease per step.
size_wiggle Real How much should randomly be added or subtracted from the particles size per step.

Rückgabe:

N/A

Beispiel:

part_type_shape(particle1, pt_shape_explosion);
part_type_size(particle1, 2, 2, 0.10, 0.01);
part_type_scale(particle1, 1, 1);
part_type_colour2(particle1, 16777215, 12632256);
part_type_alpha3(particle1, 0, 0.20, 0);
part_type_speed(particle1, 0.20, 0.20, 0, 0);
part_type_direction(particle1, 0, 359, 0, 20);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 0);
part_type_life(particle1, 40, 60);

Die obige code stellt verschiedene Partikelwerte ein, einschließlich der Größe, die bei der Erstellung jedes Partikels auf die doppelte Größe eingestellt wird, und erhöht dann die Größe bei jedem Schritt um 0,01 mit einem Wackelschritt von ebenfalls 0,1.
