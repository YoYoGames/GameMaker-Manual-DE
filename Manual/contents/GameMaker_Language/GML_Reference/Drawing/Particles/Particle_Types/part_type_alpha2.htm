teil_art_alpha2

Mit dieser Funktion kann ein Alphawert-Gradient (Transparenz) für jedes erzeugte Partikel des angegebenen Typs festgelegt werden. Der erste Alphawert ist derjenige, mit dem alle Partikel beginnen, und der zweite Alphawert ist derjenige, mit dem die Partikel enden, und es wird ein sanfter Farbverlauf für den Alphawert während der Lebensdauer der Partikel von einem zum anderen auftreten. Dieser kann zwischen 0 (transparent) und 1 (undurchsichtig) liegen.

Syntax:

part_type_alpha2(ind, alpha1, alpha2);

Argument Typ Beschreibung
ind Partikel Typ ID The index of the particle type to change.
alpha1 Real The start alpha of the particle.
alpha2 Real The end alpha of the particle.

Rückgabe:

N/A

Beispiel:

part_type_alpha2(global.Snow_Part, 0, 1);

Die obige code setzt alle Partikel, die mit dem in der globalen Variable "Snow_Part" indizierten Partikeltyp erstellt werden, auf einen Alphawert von 0 (transparent) und blendet dann am Ende ihrer Lebensdauer auf einen Alphawert von 1 (undurchsichtig) ein.
