part_type_gravity

Diese Funktion legt die Schwerkraft fest, die auf jedes erzeugte Partikel des angegebenen Typs wirken soll. Der Wert für die Stärke der Schwerkraft wird bei jedem Schritt zur Partikelgeschwindigkeit addiert und ist normalerweise ein kleiner Wert wie 0,5, während die Richtung die Richtung der Schwerkraft ist und den Standard-GameMaker-Richtungen folgt: 0° ist rechts, 90° ist oben, 180° ist links und 270° ist unten.

Syntax:

part_type_gravity(ind, grav_amount, grav_direction);

Argument Typ Beschreibung
ind Partikel Typ ID The index of the particle type to change.
grav_amount Real Strength of the gravity.
grav_direction Real The direction of the gravity.

Rückgabe:

N/A

Beispiel:

part_type_shape(global.p1, pt_shape_pixel);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

Die obige code stellt verschiedene Partikelwerte ein, einschließlich der Schwerkraft, die bei jedem Schritt 0,2 zur Geschwindigkeit hinzufügt und eine Richtung von 270º hat, so dass die Partikel nach unten zum unteren Rand des Bildschirms gezogen werden.
