part_type_sprite

Mit dieser Funktion kann ein Partikeltyp so eingestellt werden, dass er eine eigene sprite aus den Spiel-Assets verwendet. Du kannst sprite auswählen und dann GameMaker sagen, dass er sprite verwenden und es animieren soll oder nicht. Wenn du animiert wählst, kannst du auch festlegen, dass die Unterbilder in einem Intervall geändert werden, so dass die Animation am Anfang und am Ende der Lebenszeit jedes erstellten Partikels beginnt und endet (wenn du also eine Lebenszeit mit einem Minimum von 15 Schritten und einem Maximum von 30 Schritten einstellst, wird die Animation, egal welchen Wert die Lebenszeit hat, angepasst, wobei sie für die kürzeren Lebenszeiten schneller und für die längeren langsamer läuft). Sie können auch ein zufälliges Teilbild auswählen, so dass, wenn sprite nicht animiert ist, ein anderes Bild zufällig ausgewählt wird, um die sprite zu erstellen, oder wenn das Teilchen animiert ist, wird die Animation an einem zufälligen Punkt beginnen.

Syntax:

part_type_sprite(ind, sprite, animate, stretch, random);

Argument Typ Beschreibung
ind Partikel Typ ID The index of the particle type to change.
sprite Sprite-Asset The index of the sprite to set the particle type to.
animate Boolesche Whether to follow the sprite's animation normally (true) or not (false).
stretch Boolesche Whether to stretch the sprite's animation to match the particle lifespan (true) or not (false).
random Boolesche Whether to choose a random sub-image (true) or not (false).

Rückgabe:

N/A

Beispiel:

global.p1 = part_type_create();
part_type_sprite(global.p1 , spr_Coins, 1, 0, 0);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

Die obige code setzt den Partikeltyp, der in der globalen Variable "p1" indiziert ist, auf die Form der sprite, die in "spr_Coins" indiziert ist, und setzt dann verschiedene andere Partikeleigenschaften.
