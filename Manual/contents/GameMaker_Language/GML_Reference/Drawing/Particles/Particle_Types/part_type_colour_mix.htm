teil_art_farbe_mix

Mit dieser Funktion können Sie den angegebenen Partikeltyp auf eine zufällige Mischung aus zwei Farben einstellen.

Syntax:

part_type_colour_mix(ind, colour1, colour2);

Argument Typ Beschreibung
ind Partikel Typ ID The index of the particle type to change.
colour1 Farbe The first colour the blend will take from.
colour2 Farbe The second colour the blend will take from.

Rückgabe:

N/A

Beispiel:

part_type_colour_mix(global.P_Damage, c_red, c_yellow);

Die obige code setzt die Farbe für jedes emittierte Partikel des in der globalen Variable "P_Damage" indizierten Partikeltyps auf eine zufällige Mischung aus den Farben Rot und Gelb.
