teil_art_alpha3

Diese Funktion kann verwendet werden, um einen Gradienten mit drei Alpha-Werten (Transparenz) für jedes Partikel des angegebenen Typs festzulegen. Der erste Alphawert ist derjenige, mit dem alle Partikel beginnen, der zweite Alphawert ist derjenige, der nach der Hälfte der Lebensdauer des Partikels erreicht wird, und der dritte Alphawert ist derjenige, mit dem der Partikel endet. Während der Lebensdauer des Partikels wird ein sanfter Übergang von einem Alpha zum anderen stattfinden.

Syntax:

part_type_alpha3(ind, alpha1, alpha2, alpha3);

Argument Typ Beschreibung
ind Partikel Typ ID The index of the particle type to change.
alpha1 Real The starting alpha of the particle.
alpha2 Real The halfway point alpha of the particle.
alpha3 Real The ending alpha of the particle.

Rückgabe:

N/A

Beispiel:

part_type_alpha3( part_Health, 0.5, 1, 0);

Die obige code sorgt dafür, dass alle Partikel, die von dem in der Variablen "part_Health" indizierten Typ erstellt werden, während ihrer Lebensdauer eine Alpha-Mischung von 0,5 bis 1 bis 0 aufweisen.
