teil_art_farbe_rgb

Mit dieser Funktion können Sie die Mischung der Farben Rot, Grün und Blau für alle erzeugten Partikel des angegebenen Typs festlegen. Sie geben einen Minimal- und einen Maximalwert für jede der drei Komponenten an, und die erzeugten Partikel erhalten eine zufällige Farbe, die auf dem angegebenen Parameterbereich basiert. Alle Werte müssen zwischen 0 und 255 liegen.

Syntax:

part_type_colour_rgb(ind, rmin, rmax, gmin, gmax, bmin, bmax p>

Argument Typ Beschreibung
ind Partikel Typ ID The index of the particle type to change.
rmin Real The minimum the final colour's red component can be.
rmax Real The maximum the final colour's red component can be.
gmin Real The minimum the final colour's green component can be.
gmax Real The maximum the final colour's green component can be.
bmin Real The minimum the final colour's blue component can be.
bmax Real The maximum the final colour's blue component can be.

Rückgabe:

N/A

Beispiel:

part_type_rgb(global.Blood_Part, 0, 255, 0, 0, 0, 0);

Die obige code setzt jedes emittierte Partikel des Partikeltyps, der in der globalen Variable "Blood_Part" indiziert ist, nur in verschiedenen Rottönen.
