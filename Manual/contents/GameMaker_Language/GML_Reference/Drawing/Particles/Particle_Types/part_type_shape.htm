teil_art_form

Mit dieser Funktion können Sie die für den Partikeltyp zu verwendende Form sprite einstellen, wobei die folgenden Konstanten zulässig sind:

Partikelform Konstante
Konstante Beschreibung
pt_shape_pixel Ein 1x1-Pixel. (Dies ist die Standardeinstellung.)
pt_shape_disk Ein gefüllter Kreis.
pt_shape_square Ein gefülltes Quadrat.
pt_shape_line Eine 8px breite horizontale Linie.
pt_shape_star Ein fünfzackiger gefüllter Stern.
pt_shape_circle Ein 3px umrandeter Kreis.
pt_shape_ring Ein Kreis, der nach innen leuchtet (sieht aus wie eine Blase).
pt_shape_sphere Ein Kreis, der nach außen leuchtet ' in der Mitte fest, nach außen leuchtend.
pt_shape_flare Ein grell leuchtender Punkt (sieht aus wie ein echter Stern in der Nacht).
pt_shape_spark Ein Funken-Effekt ' wie ein Stern mit mehreren verblassenden Punkten.
pt_shape_explosion Eine viereckige Rauchwolke ' erfordert mehrere Farben, um einer Explosion zu ähneln.
pt_shape_cloud Eine dünne Wolke erfordert eine Hochskalierung und mehrere Partikel, um einer Wolke zu ähneln.
pt_shape_smoke Eine sanfte Version des Explosionseffekts. Verwenden Sie mehrere, um eine Rauchwolke zu erzeugen.
pt_shape_snow Eine generische Schneeflockenform.

Syntax:

part_type_shape(ind, shape);

Argument Typ Beschreibung
ind Partikel Typ ID The index of the particle type to set.
shape Partikelform Konstante The shape to choose.

Rückgabe:

N/A

Beispiel:

global.p1 = part_type_create();
part_type_shape(global.p1, pt_shape_square);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

Die obige code setzt den Partikeltyp, der in der globalen Variable "p1" indiziert ist, auf die Form eines Quadrats und setzt dann verschiedene andere Partikeleigenschaften.
