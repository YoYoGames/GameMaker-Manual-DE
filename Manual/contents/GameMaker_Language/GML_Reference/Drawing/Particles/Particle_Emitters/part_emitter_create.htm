part_emitter_create

Diese Funktion wird verwendet, um einen neuen Partikelemitter im angegebenen Partikelsystem zu erstellen. Die Funktion gibt den Indexwert für den neuen Emitter zurück, der in einer Variablen gespeichert und in allen weiteren Funktionen, die den Emitter referenzieren, verwendet werden muss. Der Emitter selbst muss zerstört werden, wenn er nicht mehr verwendet wird, um Speicherlecks zu vermeiden (dies kann mit den spezifischen Emitter-Destroy-Funktionen oder durch Zerstörung des gesamten Partikelsystems, zu dem der Emitter gehört, erreicht werden).

HINWEIS Die von dieser Funktion zurückgegebenen Emitter-IDs werden wiederverwendet, d.h. wenn Sie zunächst drei Partikelemitter erstellen, werden diesen die IDs 0, 1 und 2 zugewiesen. Wenn du dann den zweiten Emitter (der eine ID von 1 hat) mit part_emitter_destroy zerstören und dann einen weiteren Emitter erstellen, wird dieser Emitter die ID 1 wiederverwenden. Da die Emitter-ID 2 in Gebrauch ist, wird der nächste Emitter die ID 3 erhalten usw.

Syntax:

part_emitter_create(ps);

Argument Typ Beschreibung
ps Partikel System ID The particle system to create the emitter in.

Rückgabe:

Particle Emitter ID

Beispiel:

p_emit1 = part_emitter_create(Sname);

Dadurch wird ein neuer Partikelemitter erstellt und sein Index in der Variablen "p_emit" gespeichert.
