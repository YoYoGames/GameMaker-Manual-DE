part_emitter_clear

Mit dieser Funktion kannst du den angegebenen Emitter aus dem angegebenen Partikelsystem in seinen Standardzustand zurückversetzen. Dies wird auch alle Partikel stoppen, die zu diesem Zeitpunkt vom Emitter gestreamt werden, und wenn du den Emitter wieder verwenden möchtest, musst du die Position der Region und den Partikeltyp mit der part_emitter_region() Funktion.

Syntax:

part_emitter_clear(ps, ind);

Argument Typ Beschreibung
ps Partikel System ID The particle system that the emitter is in.
ind Partikel-Emitter-ID The index of the emitter to clear.

Rückgabe:

N/A

Beispiel:

part_emitter_clear(Sname, p_emit1 );

Die obige code löscht den Partikelemitter, der in der Variablen "p_emit1" indiziert ist.
