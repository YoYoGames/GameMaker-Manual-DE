part_emitter_destroy

Diese Funktion entfernt den angegebenen Emitter aus dem angegebenen System und löscht ihn aus dem Speicher (dadurch werden auch keine Partikel mehr von dem angegebenen Emitter produziert, aber sie werden NICHT aus dem Raum entfernt). Diese Funktion sollte immer aufgerufen werden, wenn der angegebene Emitter nicht mehr für das System benötigt wird, um Speicherlecks und Fehler zu vermeiden.

Syntax:

part_emitter_destroy( ps, ind );

Argument Typ Beschreibung
ps Partikel System ID The particle system to destroy the emitter from.
ind Partikel-Emitter-ID The index of the emitter to destroy.

Rückgabe:

N/A

Beispiel:

if part_emitter_exists(global.Sname, p_emit)
{
    part_emitter_destroy(global.Sname, p_emit);
}

Die obige code prüft, ob der Partikelemitter, der in der Variable "p_emit" indiziert ist, im gegebenen Partikelsystem existiert und wenn ja, wird er zerstört.
