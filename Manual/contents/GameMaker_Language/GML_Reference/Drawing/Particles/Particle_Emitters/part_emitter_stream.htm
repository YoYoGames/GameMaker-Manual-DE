part_emitter_stream

Mit dieser Funktion können Sie einen Emitter so einstellen, dass er einen bestimmten Partikeltyp ausstrahlt, und sie wird in der Regel für das Ereignis "Erstellen" verwendet, da es sich um ein einmaliges Ereignis handelt ( code ), das den Emitter so einstellt, dass er die angegebene Anzahl von Partikeln bei jedem nachfolgenden Spielschritt erzeugt. Die Partikel werden entsprechend der Verteilung, Form und Position emittiert, die durch die Funktion part_emitter_region(). Du kannst die Anzahl der Partikel, die gestreamt werden, auf Null setzen und diese Funktion erneut aufrufen, um das Partikel-Streaming "auszuschalten". Die Funktion akzeptiert auch negative Zahlen für die Menge, in diesem Fall produziert der Emitter Partikel auf Basis des Zufalls. Wenn du zum Beispiel die Partikelanzahl auf -5 gesetzt hast, besteht eine 1:5 Chance, dass bei jedem Schritt ein Partikel produziert wird. Wenn Sie möchten, dass die Partikel auf einmal und nicht bei jedem Schritt erscheinen, sollten Sie die Funktion part_emitter_burst().

Syntax:

part_emitter_stream(ps, ind, parttype, number);

Argument Typ Beschreibung
ps Partikel System ID The particle system that the emitter is in.
ind Partikel-Emitter-ID The index of the emitter to burst from.
parttype Partikel Typ ID The index (type) of the particles to be created.
number Real The number of particles to create per step.

Rückgabe:

N/A

Beispiel:

part_emitter_stream(global. Sname, p_emit1, p1, 1);

Die obige code wird bei jedem Spielschritt 1 Partikel streamen, bis der Emitter zerstört wird oder der Stream auf 0 gesetzt wird.
