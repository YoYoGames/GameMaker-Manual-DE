part_emitter_burst

Diese Funktion ermöglicht es Ihnen, einen Emitter so einzustellen, dass er einen bestimmten Partikeltyp ausstößt. Sie wird typischerweise in Alarmen und Zerstörungsereignissen verwendet, da es sich um eine einmalige code handelt, die die angegebene Anzahl von Partikeln auf einmal erzeugt und dabei die von der Funktion festgelegte Verteilung, Form und Position einhält part_emitter_region(). Sollten Sie die Partikel bei jedem Schritt benötigen, sollten Sie die Funktion part_emitter_stream() verwenden, anstatt diese Funktion bei jedem Schritt aufzurufen.

Syntax:

part_emitter_burst(ps, ind, parttype, number);

Argument Typ Beschreibung
ps Partikel System ID The particle system that the emitter is in.
ind Partikel-Emitter-ID The index of the emitter to burst from.
parttype Partikel Typ ID The index (type) of the particles to be created.
number Real The number of particles to create.

Rückgabe:

N/A

Beispiel:

part_emitter_burst(global. Sname, p_emit1, p1, 30 + irandom(30));

Die oben genannte code wird eine zufällige Anzahl von Partikeln zwischen 30 und 60 aus dem gewählten Emitter ausstoßen.
