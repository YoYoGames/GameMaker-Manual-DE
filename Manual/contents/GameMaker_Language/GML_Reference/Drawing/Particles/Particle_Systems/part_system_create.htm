teil_system_erstellen

Diese Funktion wird verwendet, um ein neues Partikelsystem zu erstellen und gibt eine eindeutige Indexnummer zurück, die gespeichert und in allen weiteren Funktionen, die sich auf dieses System beziehen, verwendet werden sollte. Dem System wird eine verwaltete Ebene zugewiesen und es wird auf eine Tiefe von 0 gesetzt. Verwaltete Ebenen sind für den Benutzer nicht zugänglich und werden nur für die interne Verwaltung verwendet, wenn Tiefe anstelle von Ebenen verwendet wird. Normalerweise würden Sie die Funktion part_system_create_layer() anstelle dieser Funktion verwenden.

Syntax:

part_system_create();

Rückgabe:

Particle System ID

Beispiel:

global.Sname = part_system_create();

Dadurch wird ein neues Partikelsystem erstellt und der Index in der globalen Variable "Sname" gespeichert.
