teil_system_position

Mit dieser Funktion können Sie die Basisposition für das Partikelsystem relativ zur (0,0)-Position des Raums festlegen, was bedeutet, dass alle weiteren Partikelfunktionen, die sich auf dieses System beziehen, nun relativ zu der neuen Position gezeichnet werden. Standardmäßig ist diese Position immer (0,0), aber unter sehr speziellen Umständen kann es sein, dass Sie dies auf etwas anderes ändern möchten.

HINWEIS: Diese Funktion ändert alles innerhalb des Partikelsystems. Wenn du also einen Emitter an der Position (100,100) hast und dann die Position des Partikelsystems auf (0,100) setzt, wird der Emitter jetzt an (100,200) gezeichnet. Dasselbe gilt, wenn du das System verschiebst und dann den Emitter erstellst, denn obwohl du ihn an (100,100) erstellst, wird er an (100,200) gezeichnet.

Syntax:

part_system_position(ind, x, y);

Argument Typ Beschreibung
ind Partikel System ID The index of the particle system to change.
x Real The new x coordinate of the particle system.
y Real The new y coordinate of the particle system.

Rückgabe:

N/A

Beispiel:

if mouse_check_button_pressed(mb_left)
{
    part_system_position(global.Sname, mouse_x, mouse_y);
}

Die obige code prüft, ob die Maustaste gedrückt wurde, und wenn dies der Fall ist, wird das in der globalen Variable "Sname" indizierte Partikelsystem an die x/y-Position der Maus verschoben
