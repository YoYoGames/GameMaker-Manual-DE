teil_system_aktualisierung

Mit dieser Funktion wird das angegebene Partikelsystem um einen Schritt in der Spielzeit weitergeschaltet. Sie kann verwendet werden, um das Verhalten des Partikelsystems zu simulieren, wenn die automatische Aktualisierung ausgeschaltet ist, oder sie kann verwendet werden, um ein Partikelsystem in einem Schritt zu einem bestimmten Punkt weiterzuschalten. Wenn du z. B. ein Partikelsystem für Schnee hast und möchtest, dass es so aussieht, als würde der Schnee gleich in dem Moment fallen, in dem der Spieler den Raum betritt, dann möchtest du vielleicht deine Schneepartikel im Erstellungsereignis einer Instanz um einige Sekunden vorverlegen, so dass beim ersten Mal, wenn sie gezeichnet werden, überall auf dem Bildschirm Partikel zu sehen sind (das code Beispiel unten zeigt dies). Sie können diese Funktion auch verwenden, wenn die automatische Aktualisierung des Systems mit der Funktion part_system_automatic_update() um das System voranzutreiben.

Syntax:

part_system_update(ind);

Argument Typ Beschreibung
ind Partikel System ID The index of the particle system to update.

Rückgabe:

N/A

Beispiel:

repeat (room_speed * 3)
{
    part_system_update(global.RainSys);
}

Die obige code wird das Partikelsystem, das in der globalen Variable "RainSys" indiziert ist, um 3 Sekunden vorstellen.
