teil_partikel_zahl

Mit dieser Funktion kannst du überprüfen, ob ein Partikelsystem gerade Partikel im Raum erzeugt hat, und sie gibt auch die Anzahl der Partikel zurück.

Syntax:

part_particles_count(ind);

Argument Typ Beschreibung
ind Partikel System ID The index of the particle system.

Rückgabe:

Real

Beispiel:

if (part_particles_count(Sname) == 0)
{
    part_system_destroy(Sname);
    instance_destroy();
}

Die obige code prüft die Anzahl der Partikel in dem lokalen Partikelsystem, das in der Variable "Sname" indiziert ist, und wenn es keine gibt, wird das System und dann die Instanz zerstört.
