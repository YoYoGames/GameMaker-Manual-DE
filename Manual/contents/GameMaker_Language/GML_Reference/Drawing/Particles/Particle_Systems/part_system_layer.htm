part_system_layer

Diese Funktion kann verwendet werden, um switch ein Partikelsystem von seiner aktuellen Ebene auf eine neue Ebene zu übertragen. Sie geben den Wert der Partikelsystem-ID (wie von der Funktion part_system_create_layer()) und dann die eindeutige Layer-ID (wie von der Funktion layer_create() zurückgegeben wird, oder den Namen der Ebene, der als String verwendet werden soll - zum Beispiel "instance_layer" - wie im Raumeditor definiert), und das System wird in die neue Ebene verschoben.

Syntax:

part_system_layer(ps, layer);

Argument Typ Beschreibung
ps Partikel System ID The particle system ID
layer String oder Ebenen-ID The new layer ID

Rückgabe:

N/A

Beispiel:

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
    part_system_layer(global.p_sys, "effects_layer";
}

Die obige code überprüft ein Partikelsystem, um zu sehen, auf welcher Ebene es sich befindet, und wenn es nicht auf der angegebenen Ebene ist, wird es geändert.
