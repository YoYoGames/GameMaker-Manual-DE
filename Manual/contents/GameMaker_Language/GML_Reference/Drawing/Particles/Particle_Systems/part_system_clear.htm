teil_system_klar

Mit dieser Funktion können Sie das indizierte System auf den Standardzustand zurücksetzen, indem Sie alle Emitter entfernen und die Tiefe und Position zurücksetzen (falls sie geändert wurden). Seien Sie vorsichtig bei der Verwendung dieser Funktion, da Sie einen Fehler erhalten, wenn Sie eine Instanz setzen oder einen Emitter verwenden, der mit diesem System verbunden ist, es sei denn, Sie verwenden die part_emitter_exists() Funktion zu überprüfen. Es besteht auch keine Notwendigkeit, die Funktion part_emitter_destroy() aufzurufen, da dies ebenfalls automatisch erledigt wird.

HINWEIS: Diese Funktion löscht die sichtbaren Partikel im Raum, aber sie löscht nicht die Partikeleigenschaften und entfernt sie auch nicht aus dem Speicher. Hierfür solltest du die Funktionen part_type_clear() und verwenden. part_type_destroy().

Syntax:

part_system_clear(ind);

Argument Typ Beschreibung
ind Partikel System ID The index of the particle system to clear.

Rückgabe:

N/A

Beispiel:

part_system_clear(global.Sname);

Mit der obigen code wird das in der globalen Variable "Sname" indizierte Partikelsystem auf seinen Standardzustand zurückgesetzt.
