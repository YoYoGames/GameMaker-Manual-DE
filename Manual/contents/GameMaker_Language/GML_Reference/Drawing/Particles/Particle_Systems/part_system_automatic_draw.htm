teil_system_automatisch_ziehen

Diese Funktion kann verwendet werden, um switch das Zeichnen eines Partikelsystems abzuschalten, so dass alle Aktualisierungen, die am System vorgenommen werden (automatisch oder anderweitig), nicht gesehen werden. Dies ist eine rein visuelle Option, und wenn sie auf false gesetzt ist, kannst du die Partikel nicht sehen, da sie nicht gezeichnet werden, aber sie existieren immer noch und ändern ihre Position, Farbe etc... wie sie es normalerweise tun würden. Wenn das automatische Zeichnen ausgeschaltet ist, kannst du GameMaker explizit anweisen, den aktuellen Zustand des Partikelsystems zu zeichnen, indem du die Funktion part_system_drawit()den aktuellen Zustand des Partikelsystems zu zeichnen, und wenn du diese Funktion wieder auf true setzt, kannst du switch das automatische Zeichnen wieder einschalten.

Zu beachten ist, dass Sie bei der Verwendung der einfachen Effekte, die durch die Funktionen effect_create_above() oder effect_create_below() erstellten einfachen Effekte verwenden, können Sie die Werte 0 (für die unteren Effekte) oder 1 (für die oberen Effekte) als Partikelsystemindex verwenden und so die automatische Zeichnung auch für diese umschalten (dies funktioniert auch für die GML Visual-Partikel-Effekte).

Syntax:

part_system_automatic_draw(ind, automatic);

Argument Typ Beschreibung
ind Partikel System ID The index of the particle system to change.
automatic Boolesche Whether automatic drawing is on (true) or not (false).

Rückgabe:

N/A

Beispiel:

part_system_automatic_draw(global.Sname, false);

Die obige code schaltet das automatische Zeichnen für das in der globalen Variable "Sname" indizierte Partikelsystem aus.
