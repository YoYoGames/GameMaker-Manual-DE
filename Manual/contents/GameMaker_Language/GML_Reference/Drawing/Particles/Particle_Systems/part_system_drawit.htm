teil_system_aktualisierung

Mit dieser Funktion kannst du ein Partikelsystem dazu zwingen, alle aktuellen Partikel auf den Bildschirm zu zeichnen. Wenn part_system_automatic_draw() ausgeschaltet ist, zeigt diese Funktion die Partikel an, wenn sie im Draw-Ereignis einer Instanz verwendet wird, oder sie kann verwendet werden, wenn das Zeichenziel auf eine Oberfläche gesetzt ist (mit surface_set_target()), um die Partikel im System auf diese Oberfläche zu zeichnen.

Syntax:

part_system_drawit(ind);

Argument Typ Beschreibung
ind Partikel System ID The index of the particle system to draw the particles of.

Rückgabe:

N/A

Beispiel:

if surface_exists(surf)
{
    surface_set_target(surf);
    part_system_drawit(global.psys);
    surface_reset_target();
}

Die obige code prüft, ob die in der Variablen "surf" indizierte Oberfläche existiert, und wenn dies der Fall ist, setzt sie das Zeichenziel auf die Oberfläche, zeichnet das Partikelsystem mit der in der globalen Variablen gespeicherten ID und setzt dann das Zeichenziel zurück, so dass alle normalen Zeichnungen wieder auf dem Bildschirm erscheinen.
