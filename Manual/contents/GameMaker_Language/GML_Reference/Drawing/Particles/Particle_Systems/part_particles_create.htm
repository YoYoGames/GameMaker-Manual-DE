teil_partikel_erzeugen

Diese Funktion ist ideal für Effekte, die keine der von Partikelemittern angebotenen Funktionen benötigen (z. B. um Rauch aus einer Rakete oder einen einfachen Explosionseffekt zu erzeugen), da sie es Ihnen ermöglicht, schnell und einfach Partikel an jeder beliebigen Position im Spielraum zu erzeugen. Beachten Sie, dass Sie das Partikelsystem und den Partikeltyp zuvor erstellt haben müssen, um diese Funktion nutzen zu können.

Syntax:

part_particles_create(ind, x, y, parttype, number);

Argument Typ Beschreibung
ind Partikel System ID The index of the particle system.
x Real The x coordinate of where to create the particles.
y Real The y coordinate of where to create the particles.
parttype Partikel Typ ID The index (type) of the particles to be created.
number Real The number of particles to create.

Rückgabe:

N/A

Beispiel:

if mouse_check_button(mb_left)
{
    part_particles_create(sname, mouse_x, mouse_y, p_CursorEffect, 5);
}

Die obige code prüft, ob die Maustaste gedrückt wird, und wenn sie true zurückgibt, werden 5 Partikel an der Mausposition erzeugt.
