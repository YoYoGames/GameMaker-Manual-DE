teil_system_tiefe

Mit dieser Funktion können Sie die Zeichnungstiefe für das Partikelsystem einstellen, ähnlich wie Sie die Rendertiefe der verschiedenen Ebenen im Raum einstellen können, wobei eine niedrige Zeichnungstiefe bedeutet, dass es über allen Dingen erscheint, die mit einer höheren Tiefe gezeichnet wurden, und eine hohe Zeichnungstiefe, die es unter allem mit einer niedrigeren Zeichnungstiefe platziert.

Syntax:

part_system_depth(ind, depth);

Argument Typ Beschreibung
ind Partikel System ID The index of the particle system to change.
depth Real The depth at which to set the particle system.

Rückgabe:

N/A

Beispiel:

global.Sname = part_system_create();
part_system_depth(global.Sname, -1000 );

Die obige code erstellt ein Partikelsystem und speichert dessen Index in der globalen Variable "Sname". Diesem System wird dann eine niedrige Tiefe von -1000 zugewiesen, was bedeutet, dass es über allem erscheint, was eine höhere Zeichentiefe hat.
