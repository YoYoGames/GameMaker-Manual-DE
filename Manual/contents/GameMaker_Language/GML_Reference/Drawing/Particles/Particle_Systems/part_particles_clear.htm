part_particles_clear

Mit dieser Funktion können Sie alle Partikel, die derzeit vom System erzeugt werden, aus dem Raum entfernen. Es werden nicht die Partikeltypen selbst zurückgesetzt oder entfernt, sondern nur ihre visuelle Darstellung. Wenn Sie ein Objekt haben, das Partikel von einem Emitter ausstrahlt, verschwinden diese Partikel, werden aber im nächsten Schritt nach dem Aufruf dieser code wieder erscheinen.

Syntax:

part_particles_clear(ind);

Argument Typ Beschreibung
ind Partikel System ID The index of the particle system.

Rückgabe:

N/A

Beispiel:

if (lives <= 0)
{
    part_particles_clear(global.Sname);
    room_goto(rm_intro);
}

Die obige code prüft den Wert der Variable "lives" und wenn dieser gleich 0 ist, werden alle Partikel aus dem System gelöscht und der Raum gewechselt.
