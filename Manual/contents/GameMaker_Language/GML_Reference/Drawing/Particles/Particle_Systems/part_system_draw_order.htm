part_system_draw_order

Mit dieser Funktion können Sie die Art und Weise einstellen, in der Partikel bei der Erstellung auf dem Bildschirm gezeichnet werden. Das Standardsystem verwendet ein altes > neues Aussehen (die Funktion ist auf true eingestellt), wobei alte Partikel in einer höheren Tiefe als neuere gezeichnet werden und somit "unter" den neuen Partikeln erscheinen. Wenn Sie diese Funktion auf false einstellen, können Sie diese Reihenfolge umkehren und die neuen Partikel höher zeichnen lassen, so dass sie "unter" den älteren Partikeln erscheinen. Die folgenden Bilder veranschaulichen dies, wobei das linke Bild den Standardwert true und das rechte Bild den Wert false hat:

HINWEIS: Wenn die Partikel mit einem additiven Überblendungsmodus gezeichnet werden, ist die Wirkung dieser Funktion möglicherweise nicht immer offensichtlich.

Syntax:

part_system_draw_order(ind, oldtonew);

Argument Typ Beschreibung
ind Partikel System ID The index of the particle system to change.
oldtonew Boolesche Whether old particles should be drawn behind newer ones (true) or vice versa (false).

Rückgabe:

N/A

Beispiel:

mysystem = part_system_create();
part_system_draw_order(mysystem, true);

Dadurch wird ein neues Partikelsystem mit dem Index "mysystem" erstellt, und dann werden die Partikel so eingestellt, dass neuere Partikel über ältere gezeichnet werden.
