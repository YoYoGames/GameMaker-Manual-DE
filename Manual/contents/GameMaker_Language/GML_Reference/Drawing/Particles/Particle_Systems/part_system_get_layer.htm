part_system_get_layer

Diese Funktion kann verwendet werden, um den eindeutigen Layer-ID-Wert für das angegebene Partikelsystem abzurufen. Sie geben den ID-Wert des Partikelsystems an (wie er von der Funktion part_system_create_layer()) und die Funktion gibt den ID-Wert der Ebene zurück.

Syntax:

part_system_get_layer(ind);

Argument Typ Beschreibung
ind Partikel System ID The particle system ID value to get the layer ID of

Rückgabe:

Layer ID or 0 (if the layer is an internally managed one)

Beispiel:

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
    part_system_layer(global.p_sys, "effects_layer";
}

Die obige code überprüft ein Partikelsystem, um zu sehen, auf welcher Ebene es sich befindet, und wenn es nicht auf der angegebenen Ebene ist, wird es geändert.
