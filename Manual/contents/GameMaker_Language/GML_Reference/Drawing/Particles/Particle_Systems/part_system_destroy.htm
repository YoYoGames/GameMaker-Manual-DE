part_system_destroy

Mit dieser Funktion kannst du ein bestimmtes Partikelsystem zerstören und aus dem Speicher entfernen. Diese Funktion sollte immer dann aufgerufen werden, wenn das System nicht mehr benötigt wird, z.B. am Ende eines Raumes oder im Zerstörungsereignis einer Instanz. Andernfalls kann es passieren, dass Partikel in späteren Räumen auftauchen und nicht mehr entfernt werden können, was zu einem Speicherleck führt und das Spiel zum Absturz bringt.

HINWEIS: Diese Funktion zerstört und entfernt auch alle Emitter, die möglicherweise erstellt wurden und mit dem zu zerstörenden System verbunden sind.

Syntax:

part_system_destroy(ind);

Argument Typ Beschreibung
ind Partikel System ID The index of the particle system to destroy.

Rückgabe:

N/A

Beispiel:

if (global.player_hp <= 0)
{
    part_system_destroy(p_sys);
    room_goto_next();
}

Die obige code prüft, ob der Wert einer globalen Variable kleiner oder gleich Null ist, und wenn dies der Fall ist, wird das Partikelsystem, auf das in der gegebenen Variable verwiesen wird, zerstört und dann zum nächsten Raum übergegangen.
