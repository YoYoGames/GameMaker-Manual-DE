zeichnen_spülen

Mit dieser Funktion können Sie die gesamte Zeichenpipeline leeren. Dies ist eine reine Debug-Funktion und sollte im Allgemeinen nicht verwendet werden, es sei denn, sie wird von einem Mitglied des YoYo Games Supports verwendet werden, da eine wahllose Verwendung zu ernsthaften Leistungsproblemen in Ihrem Spiel führen kann.

Syntax:

draw_flush();

Rückgabe:

Beispiel:

draw_flush();

Die obige code spült die Draw-Pipeline.
