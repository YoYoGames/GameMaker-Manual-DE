oberfläche_erstellen_ext

Mit dieser Funktion können Sie eine Fläche an ein Canvas-Element anhängen, das bereits in Ihrer Webseite vorhanden ist. Das bedeutet, dass Sie Teile Ihres Spiels effektiv aufteilen können, um sie an verschiedenen Stellen innerhalb der Seite zu zeichnen. Zu diesem Zweck müssen Sie das Canvas-Element auf der Seite *.html Ihres Spiels korrekt definieren, indem Sie die richtigen Größen und Namen verwenden, die den Flächen entsprechen, die Sie erstellen möchten. Sie haben also Ihre "Haupt"-Leinwand und dann Ihre sekundären Oberflächen-Leinwandelemente, die mit dieser Funktion den richtigen Oberflächen zugewiesen werden. Das folgende Bild ist ein Beispiel dafür, wie eine Seite mit drei Canvas-Elementen aufgebaut sein könnte:

Das Seitenlayout kann knifflig sein, vor allem, wenn alle Elemente korrekt ausgerichtet sein sollen. Sobald die schwierige Aufgabe der Layouterstellung abgeschlossen ist, können Sie diese html Datei als Standardseitendatei für das Spiel hinzufügen, indem Sie die Registerkarte HTML5 der Spieloptionen verwenden. Als Nächstes solltest du deinen Raum und deine Ansichten einrichten, da jede Oberfläche mit einer bestimmten Ansicht verknüpft werden muss, um die Bilder des Spiels zu "erfassen" (siehe die Ansichtsvariable view_surface_id[0...7]). Die folgende Abbildung zeigt, wie der Spielraum für das obige Canvas-Beispiel eingerichtet werden würde:

Schließlich würden Sie diese Funktion verwenden, um die Flächen zu erstellen, wobei der Name derselbe ist wie bei den Canvas-Elementen und die Größe der Größe des Canvas entspricht. Die Funktion gibt den Index der Fläche zurück, der in einer Variablen für zukünftige Funktionsaufrufe gespeichert werden sollte. Wenn die Oberfläche zum ersten Mal erstellt wird, kann sie "Rauschen" enthalten, da es sich im Grunde nur um einen Speicherbereich handelt, der für diesen Zweck beiseite gelegt wird (und dieser Speicher kann immer noch Informationen enthalten), daher sollten Sie die Oberfläche vor der Verwendung mit einer Funktion wie draw_clear_alpha().

HINWEIS: Diese Funktion ist nur für die Verwendung mit dem HTML5 module verfügbar.

Syntax:

surface_create_ext(name, w, h);

ArgumentTyp Beschreibung
name The name of the canvas element to link the surface to.
w The width of the surface to be created.
h The height of the surface to be created.

Rückgabe:

Surface ID

Beispiel:

s1 = surface_create_ext("surface1", 192, 550);
s2 = surface_create_ext("surface2", 608, 186);
view_surface_id[1] = s1;
view_surface_id[2] = s2;

Die obige code erstellt zwei unterschiedlich große Flächen, die jeweils einem anderen Canvas-Element zugewiesen werden, und diese Flächen werden dann zwei Ansichten zugewiesen, damit der richtige Teil des Raums erfasst wird.
