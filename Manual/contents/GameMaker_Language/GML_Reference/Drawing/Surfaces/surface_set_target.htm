surface_set_target

Mit dieser Funktion legen Sie alle weiteren Zeichnungen auf die Zielfläche und nicht auf den Bildschirm fest und können so GameMaker anweisen, nur bestimmte Dinge auf die angegebene Fläche zu zeichnen. Bitte beachten Sie, dass, wenn Sie nicht surface_reset_target() aufrufen, wird nichts auf dem Bildschirm gezeichnet, da alle weiteren Zeichnungen (auch in anderen Instanzen) auf der Oberfläche erfolgen. Du solltest dir auch darüber im Klaren sein, dass nichts zu sehen sein wird, wenn die Oberfläche selbst nicht im draw event einer Instanz auf den Bildschirm gezeichnet wird. Sie können den Rückgabewert dieser Funktion auch als Debug-Tool verwenden, um zu prüfen, ob das Oberflächenziel gesetzt wurde oder nicht, wobei ein Rückgabewert von 0 bedeutet, dass das Ziel nicht gesetzt werden konnte, und jeder andere positive Wert ein Erfolg ist.

Dabei ist zu beachten, dass die Flächen gestapelt sind, so dass man nicht von Ziel zu Ziel springen und am Ende wieder zum normalen Ziehziel zurückkehren kann, sondern dass man Rendering-Ziele öffnen und schließen muss. Dies wird zum Beispiel nicht korrekt funktionieren:

surface_set_target(surf1);
draw_text(32, 32, "surface1");
surface_set_target(surf2);
draw_text(32, 64, "surface2");
surface_reset_target();

Stattdessen müssen Sie das Ziel für jede der von Ihnen festgelegten Flächen zurücksetzen, ähnlich wie Sie öffnende und schließende Klammern {} für code Blöcke verwenden müssen. Der obige Text sollte also entweder so oder so geschrieben werden:

surface_set_target(surf1);
draw_text(32, 32, "surface1");
surface_reset_target();
surface_set_target(surf2);
draw_text(32, 64, "surface2");
surface_reset_target();

oder als dies:

surface_set_target(surf1);
draw_text(32, 32, "surface1");
surface_set_target(surf2);
draw_text(32, 64, "surface2");
surface_reset_target();
surface_reset_target();

HINWEIS: Bei der Arbeit mit Flächen besteht die Möglichkeit, dass sie jederzeit nicht mehr existieren, da sie im Texturspeicher abgelegt sind. Sie sollten IMMER überprüfen, ob eine Fläche existiert, indem Sie surface_exists() überprüfen, bevor Sie sie direkt referenzieren.

Syntax:

surface_set_target(surface_id);

ArgumentTyp Beschreibung
surface_id The ID of the surface to set as the drawing target.

Rückgabe:

Beispiel:

if view_current = 0
{
    surface_set_target(surf);
    with (obj_Effect)
    {
        draw_self();
    }
    surface_reset_target();
}
else
{
    draw_surface(surf, 0, 0);
}

Die obige code prüft, welche Ansicht gerade gezeichnet wird, und wenn es view[0] ist, setzt sie das Zeichenziel auf eine Oberfläche und zeichnet alle Instanzen des Objekts "obj_Effect", bevor sie das Zeichenziel wieder zurücksetzt. Wenn die Ansicht nicht view[0] ist, wird die Oberfläche auf den Bildschirm gezeichnet.
