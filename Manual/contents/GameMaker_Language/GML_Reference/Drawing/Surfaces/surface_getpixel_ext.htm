oberfläche_getpixel_ext

Mit dieser Funktion können Sie den vollen abgr 32bit Wert eines beliebigen Pixels einer (zuvor erstellten) Oberfläche erhalten.

HINWEIS: Diese Funktion hat einen enormen Leistungsverlust zur Folge und sollte daher nur verwendet werden, wenn dies unbedingt erforderlich ist.

Syntax:

surface_getpixel_ext(surface_id, x, y);

ArgumentTyp Beschreibung
surface_id The ID of the surface to use.
x The x coordinate of the pixel to check
y The y coordinate of the pixel to check

Rückgabe:

Beispiel:

col = surface_getpixel_ext(mouse_x, mouse_y);
alpha = (col >> 24) & 255;
blue = (col >> 16) & 255;
green = (col >> 8) & 255;
red = col & 255;

Die obige code holt sich den 32-Bit-Farbwert an der Position der Maus und teilt ihn dann in seine Komponentenwerte auf und speichert sie in Variablen.
