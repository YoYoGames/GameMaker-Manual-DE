draw_surface_ext

Diese Funktion zeichnet die angegebene Oberfläche wie in der Funktion draw_surface() gezeichnet, jedoch mit zusätzlichen Optionen zum Ändern von Skalierung, Blending, Rotation und Alpha der gezeichneten Fläche. Das Ändern dieser Werte verändert die Ressource in keiner Weise (nur wie sie gezeichnet wird).

Syntax:

draw_surface_ext(id, x, y, xscale, yscale, rot, col, alpha);

Argument Typ Beschreibung
id Oberfläche ID The unique ID value of the surface to draw.
x Real The x position of where to draw the surface.
y Real The y position of where to draw the surface.
xscale Real The horizontal scale.
yscale Real The vertical scale.
rot Real The rotation or angle to draw the surface.
col Farbe The colour with which to blend the surface.
alpha Real The alpha transparency for drawing the surface.

Rückgabe:

N/A

Beispiel:

draw_surface_ext(surf, 0, 0, 2, 2, 0, c_red, 0.5);

Die obige code zeichnet eine Oberfläche, die in der Variablen "surf" an der Position (0,0) im Raum indiziert ist, mit dem doppelten Originalmaßstab, rot überblendet und halbtransparent.
