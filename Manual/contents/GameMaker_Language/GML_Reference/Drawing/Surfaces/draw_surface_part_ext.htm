oberfläche_teil_ext_zeichnen

Diese Funktion zeichnet einen Teil der gewählten Fläche an der angegebenen Position nach den gleichen Regeln wie bei draw_surface_part()nur dass Sie jetzt den Teil skalieren, mit einer Farbe überblenden oder sein Alpha verändern können, wenn Sie ihn auf den Bildschirm zeichnen (genau wie beim Zeichnen einer Fläche mit draw_surface_ext()).

Syntax:

draw_surface_part_ext(surface, left, top, w, h, x, y, xscale, yscale, colour, alpha);

Argument Typ Beschreibung
id Oberfläche ID The unique ID value of the surface to draw.
left Real The left position in the surface of the part to be drawn.
top Real The top position in the surface of the part to be drawn.
w Real The width of the part to be draw, from left.
h Real The height of the part to be drawn, from top.
x Real The x position of where to draw the surface.
y Real The y position of where to draw the surface.
xscale Real The horizontal scaling the part should be drawn with.
yscale Real The vertical scaling the part should be drawn with.
colour Farbe The colour blending the part should be drawn with.
alpha Real The alpha transparency the part should be drawn with.

Rückgabe:

N/A

Beispiel:

draw_surface_part_ext(surf, 8, 8, 32, 32, x, y, 2, 0.5, c_black, 1);

Damit wird ein 32x32 Pixel großer Bereich von 8x8 Pixeln in die in der Variablen "surf" indizierte Fläche gezeichnet. Er wird auf die doppelte Breite, aber die halbe Höhe gestreckt. Er ist undurchsichtig und wird mit Schwarz überblendet (was ihn zu einer Silhouette macht).
