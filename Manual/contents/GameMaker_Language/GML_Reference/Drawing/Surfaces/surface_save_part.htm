oberfläche_speichern_teil

Diese Funktion speichert einen Teil einer Oberfläche unter dem angegebenen Dateinamen auf der Festplatte. Die Oberfläche muss als Datei im Format *.png gespeichert werden, und die (x,y)-Position muss als lokale Koordinaten der Oberfläche angegeben werden, wobei zu beachten ist, dass die obere linke Ecke der Oberfläche immer (0,0) ist.

Syntax:

surface_save_part(surface_id, fname, x, y, width, height);

ArgumentTyp Beschreibung
surface_id The ID of the surface to set as the drawing target.
fname The name of the saved image file.
x The starting x position within the surface.
y The starting y position within the surface.
width The width of the part to save.
height The height of the part to save.

Rückgabe:

Beispiel:

if keyboard_check_pressed(ord("S")
{
    surface_save_part(surf, "test.png", 0, 0, 100, 100);
}

Die obige code prüft, ob der Benutzer die Taste "S" auf der Tastatur drückt, und wenn dies der Fall ist, wird ein Teil der in der Variablen "surf" indizierten Oberfläche auf der Festplatte gespeichert.
