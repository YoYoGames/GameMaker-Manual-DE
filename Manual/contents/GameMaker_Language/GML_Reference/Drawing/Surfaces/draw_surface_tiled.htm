zeichne_Fläche_gekippt

Diese Funktion nimmt eine Fläche und kachelt sie dann wiederholt über den gesamten Raum, ausgehend von den Koordinaten, die Sie in der Funktion angeben.

Syntax:

draw_surface_tiled(id, x, y);

Argument Typ Beschreibung
id Oberfläche ID The unique ID value of the surface to draw.
x Real The x position of where to draw the surface.
y Real The y position of where to draw the surface.

Rückgabe:

N/A

Beispiel:

draw_surface_tiled(surf, x, y);

Dadurch wird die in "surf" indizierte Oberfläche an der eigenen x- und y-Position der Instanz gezeichnet und in jede Richtung des Raumes gekachelt.
