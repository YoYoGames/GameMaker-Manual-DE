surface_copy_part

Diese Funktion übernimmt einfach das Bild von einer Fläche und kopiert es auf eine andere an der angegebenen lokalen Position innerhalb dieser Fläche (wobei die Position (0,0) die obere linke Ecke der Zielfläche ist). Sie können eine lokale x- und y-Position angeben, von der aus kopiert werden soll, sowie die Breite und Höhe des Ausschnitts. Bitte beachten Sie, dass es sich hierbei um Koordinaten handelt, die auf der Größe der Fläche basieren und nicht auf der Position, an der die Fläche im Raum gezeichnet wird. Wenn die Zielfläche bereits Informationen enthält, werden diese durch die Kopie überschrieben, und die Funktion verändert die Quellfläche in keiner Weise.

HINWEIS: Bei der Arbeit mit Flächen besteht die Möglichkeit, dass sie jederzeit nicht mehr existieren, da sie im Texturspeicher abgelegt sind. Sie sollten IMMER überprüfen, ob eine Fläche existiert, indem Sie surface_exists() überprüfen, bevor Sie sie direkt referenzieren.

Syntax:

surface_copy_part(destination, x, y, source, xs, ys, ws, hs);

Argument Typ Beschreibung
destination Oberfläche ID The ID of the surface to copy the other surface to.
x Real The x position to copy to.
y Real The y position to copy to.
source Oberfläche ID The ID of the surface to be copied.
xs Real The x position in the source surface to copy from.
ys Real The y position in the source surface to copy from.
ws Real The width of the area in the source surface to copy from.
hs Real The height of the area in the source surface to copy from.

Rückgabe:

N/A

Beispiel:

if view_current == 0
{
    var _cam1_x = camera_get_view_x(view_camera[1]);
    var _cam1_y = camera_get_view_y(view_camera[1]);

    surface_copy_part(surf, 0, 0, temp_surf, 0, 0, _cam1_x - mouse_x, _cam1_y - mouse_y);
}
else
{
    draw_surface(surf, 0, 0);
}

Die obige code prüft die aktuelle Ansicht, die gezeichnet wird, und wenn es view[0] ist, kopiert sie die in der Variablen "temp_surf" indizierte Fläche auf die in der Variablen "surf" indizierte Fläche. Der kopierte Bereich entspricht einem Rechteck, das durch die relative Position der Maus innerhalb der Fläche gebildet wird, wie sie in view[1] gezeichnet würde. Wenn die aktuelle Ansicht eine andere als view[0] ist, wird die Oberfläche "surf" auf den Bildschirm gezeichnet.
