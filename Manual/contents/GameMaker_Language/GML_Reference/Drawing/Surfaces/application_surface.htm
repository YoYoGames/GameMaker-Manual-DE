anwendung_oberflaeche

Diese globale, eingebaute Variable kann verwendet werden, um auf die Anwendungsoberfläche zuzugreifen, um sie in einer der Oberflächenfunktionen zu verwenden. Diese Oberfläche ist ständig verfügbar und wird von GameMaker für das Spiel verwendet.

Die regulären Draw-Ereignisse(Draw Begin, Draw und Draw End) zeichnen standardmäßig auf diese Oberfläche. Andere Draw-Ereignisse (wie Pre/Post Draw und Draw GUI) zeichnen nicht auf die Anwendungsoberfläche. Weitere Informationen über die Funktionsweise dieser Ereignisse finden Sie unter Draw Events.

Syntax:

application_surface;

Rückgabe:

Surface ID

Beispiel:

surface_resize(application_surface, display_get_gui_width(), display_get_gui_height())

Die obige code ändert die Größe der Anwendungsoberfläche so, dass sie ein Verhältnis von 1:1 mit der GUI-Ebene hat.
