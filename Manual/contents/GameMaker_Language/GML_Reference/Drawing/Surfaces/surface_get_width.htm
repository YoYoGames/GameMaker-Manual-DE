oberfläche_get_width

Diese Funktion gibt einfach die Breite der indizierten Fläche in Pixeln zurück. Es ist zu beachten, dass, wenn Sie diese Funktion aufrufen, um die application_surface zu überprüfen, unmittelbar nachdem Sie ihre Größe mit surface_resize() geändert haben, wird sie nicht den neuen Wert zurückgeben, da die Änderung ein oder zwei Schritte benötigt, um vollständig verarbeitet zu werden. Nach einer Wartezeit von einem Schritt sollte die neue Größe korrekt zurückgegeben werden.

HINWEIS: Bei der Arbeit mit Flächen besteht die Möglichkeit, dass sie jederzeit nicht mehr existieren, da sie im Texturspeicher abgelegt sind. Sie sollten IMMER überprüfen, ob eine Fläche existiert, indem Sie surface_exists() überprüfen, bevor Sie sie direkt referenzieren.

Syntax:

surface_get_width(surface_id);

ArgumentTyp Beschreibung
surface_id The ID of the surface to get the width of.

Rückgabe:

Beispiel:

sw = surface_get_width(surf);

Die obige code speichert die Breite der in der Variablen "surf" indizierten Fläche in der Variablen "sw".
