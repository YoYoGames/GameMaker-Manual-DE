oberfläche_zurueck_ziel

Mit dieser Funktion setzen Sie alle weiteren Zeichnungen von der Zielfläche zurück auf den Bildschirm. Bitte beachten Sie, dass Sie, um mit dem Zeichnen auf eine Oberfläche zu beginnen, zuerst die Funktion surface_set_target() und danach diese Funktion für jede Oberfläche, die Sie als Ziel festgelegt haben, aufgerufen haben müssen. Andernfalls wird nichts auf dem Bildschirm gezeichnet, da alle weiteren Zeichnungen (auch in anderen Instanzen) auf der Oberfläche ausgeführt werden. Sie sollten sich auch darüber im Klaren sein, dass nichts zu sehen sein wird, wenn die Oberfläche selbst nicht im Draw-Ereignis einer Instanz auf den Bildschirm gezeichnet wird.

HINWEIS: Wenn Sie nicht zuvor ein Rendering-Ziel mit der Funktion surface_set_target()kein Rendering-Ziel festgelegt haben, beendet die Verwendung dieser Funktion stillschweigend (d.h. ohne Fehlermeldung) jede weitere Ausführung von code für das Ereignis.

Syntax:

surface_reset_target();

Rückgabe:

Beispiel:

if view_current = 0
{
    surface_set_target(surf);
    with (obj_Effect)
    {
        draw_self();
    }
    surface_reset_target();
}
else
{
    draw_surface(surf, 0, 0);
}

Die obige code prüft, welche Ansicht gerade gezeichnet wird, und wenn es view[0] ist, setzt sie das Zeichenziel auf eine Oberfläche und zeichnet alle Instanzen des Objekts "obj_Effect", bevor sie das Zeichenziel wieder zurücksetzt. Wenn die Ansicht nicht view[0] ist, wird die Oberfläche auf den Bildschirm gezeichnet.
