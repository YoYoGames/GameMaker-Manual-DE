application_surface_draw_enable

Mit dieser Funktion können Sie das automatische Zeichnen der Anwendungsoberfläche aktivieren oder deaktivieren. Standardmäßig ist diese Funktion aktiviert, aber in vielen Fällen werden Sie selbst bestimmen wollen, wann und wie die Oberfläche gezeichnet wird (z. B. bei der Verwendung von Shadern); in diesem Fall In diesem Fall können Sie diese Funktion auf false setzen, so dass Sie die Oberfläche selbst zeichnen können, wann und wie Sie es wünschen. Es ist wichtig zu beachten, dass, wenn Sie switch das automatische Zeichnen abschalten und die Anwendungsoberfläche selbst zeichnen, Sie bestimmte Probleme sehen können mit der Alpha-Komponente von sprites und der Oberfläche selbst auftreten können. Das liegt daran, dass GameMaker die Programmoberfläche ohne Alpha-Blending zeichnet, wenn das automatische Zeichnen eingeschaltet ist. Wenn du switch das automatische Zeichnen ausschaltest, musst du müssen Sie dies selbst tun, zum Beispiel mit der folgenden code:

gpu_set_blendenable(false);
draw_surface_ext(application_surface, 0, 0, 1, 1, 0, c_white, 1);
gpu_set_blendenable(true);

Syntax:

application_surface_draw_enable(flag);

ArgumentTyp Beschreibung
flag The flag will be either true (aktiviert, der Standardwert) oder false (deaktiviert).

Rückgabe:

Beispiel:

application_surface_draw_enable(false);

code Die obige Seite switch schaltet das automatische Zeichnen der Anwendungsfläche aus.

Rückseite: Oberflächen
Nächste: surface_exists
