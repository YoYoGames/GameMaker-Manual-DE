zeichne_Fläche

Mit dieser Funktion können Sie eine Fläche an einer bestimmten Position im Raum zeichnen, wobei die obere linke Ecke der Fläche an der angegebenen x/y-Position gezeichnet wird.

Syntax:

draw_surface(id, x, y);

Argument Typ Beschreibung
id Oberfläche ID The unique ID value of the surface to draw.
x Real The x position of where to draw the surface.
y Real The y position of where to draw the surface.

Rückgabe:

N/A

Beispiel:

var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
draw_surface(surf, _vx, _vy);

Die obige code zeichnet die in "surf" indizierte Oberfläche an derselben Position wie die Kameraansicht[0].
