zeichne_Fläche_gestreckt

Diese Funktion nimmt einfach eine Fläche und dehnt sie über die angegebene Breite und Höhe aus, so dass sie den Bereich einnimmt. Wie bei draw_surface() können Sie eine Fläche und dann die x/y-Position im Raum angeben, an der die Fläche gezeichnet werden soll, und schließlich eine Breite und eine Höhe (die Pixelwerte sein müssen).

Syntax:

draw_surface_stretched(id, x, y, w, h);

Argument Typ Beschreibung
id Oberfläche ID The unique ID value of the surface to draw.
x Real The x position of where to draw the surface.
y Real The y position of where to draw the surface.
w Real The width at which to draw the surface.
h Real The height at which to draw the surface.

Rückgabe:

N/A

Beispiel:

draw_surface_stretched(surf, 10, 10, 100, 100);

Dadurch wird die in der Variablen "surf" indizierte Fläche mit ihrer linken Ecke bei (10,10) gezeichnet. Die Breite und die Höhe werden beide auf 100 gesetzt, was bedeutet, wie viel Platz sie unabhängig von der tatsächlichen Breite und Höhe der Fläche einnehmen wird.
