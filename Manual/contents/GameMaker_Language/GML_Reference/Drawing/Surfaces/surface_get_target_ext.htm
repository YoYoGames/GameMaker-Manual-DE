oberfläche_ziel_ext_holen

Diese Funktion ruft die Oberflächen-ID ab, die einem der 4 für Oberflächen verfügbaren Rendering-Ziele zugewiesen ist. Sie geben den Index des zu prüfenden Rendering-Ziels an, und die Funktion gibt -1 zurück, wenn keine Oberfläche zugeordnet ist, oder einen ganzzahligen Wert >= 0, der die ID der zugeordneten Oberfläche darstellt (wie von der Funktion surface_create()).

Syntax:

surface_get_target_ext(index);

ArgumentTyp Beschreibung
index The render target index to check (from 0 to 3).

Rückgabe:

Surface ID or -1 (if no target surface is set)

Beispiel:

if surface_get_target_ext(0) == -1
{
    surface_set_target_ext(0, global.Surf);
}

Die obige code prüft zunächst, ob das Shader-Rendering-Ziel 0 auf eine Oberfläche gesetzt wurde, und wenn nicht, wird eine zugewiesen.
