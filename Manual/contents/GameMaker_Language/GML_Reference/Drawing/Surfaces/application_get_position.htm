anwendung_get_position

Wenn Sie in den Spieloptionen für ein Ziel platform das Häkchen bei "Seitenverhältnis beibehalten" gesetzt haben, stellt GameMaker die Zeichenposition für die Programmoberfläche automatisch so ein, dass sie auf dem Bildschirm korrekt zentriert und skaliert dargestellt wird. dem gegebenen Bildschirm angezeigt wird. Wenn Sie diese Oberfläche jedoch selbst zeichnen möchten, können Sie mit dieser Funktion auf einfache Weise herausfinden, wo genau in der Anzeige oder im Fenster die Oberfläche gezeichnet wurde, so dass Sie sie dann selbst zeichnen oder GUI-Bilder oder Post-Draw-Bilder an ihr ausrichten können.

Die Funktion gibt ein Array mit vier Schlüsseln zurück, wobei die Schlüssel 0 und 1 die x- und y-Position der oberen linken Ecke der Oberfläche und die Schlüssel 2 und 3 die x- und y-Position der unteren rechten Ecke der Oberfläche sind. der Oberfläche sind, alle relativ zur Größe der Anzeige oder des Fensters.

Syntax:

application_get_position();

Rückgabe:

Beispiel:

var a = application_get_position();
xx = a[0];
yy = a[1];
ww = a[2] - a[0];
hh = a[3] - a[1];

Die obige code ermittelt die Position der Anwendungsoberfläche sowie die absolute Breite und Höhe in Bezug auf das Anzeigefenster und speichert sie in vier Variablen zur späteren Verwendung.

Rückseite: Oberflächen
Nächste: application_surface_draw_enable
