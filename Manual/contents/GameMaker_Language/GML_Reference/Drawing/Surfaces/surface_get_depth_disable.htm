surface_get_depth_disable

Diese Funktion prüft, ob die automatische Tiefenerzeugung buffer für Oberflächen aktiviert ist. Normalerweise haben alle Oberflächen die Tiefe buffers, wenn Sie also 3D-Objekte darauf zeichnen, werden diese korrekt nach Tiefe sortiert. Allerdings verdoppelt die Zuweisung von Tiefe buffers im Wesentlichen die Größe der Oberflächen, was ein übermäßiger und unnötiger Overhead sein kann, besonders wenn Ihr Spiel sehr speicherintensiv oder überwiegend 2D ist. In diesen Fällen können Sie dies mit dieser Funktion überprüfen und die Tiefe buffer für Oberflächen deaktivieren, falls erforderlich mit der Funktion surface_depth_disable().

Syntax:

surface_get_depth_disable();

Rückgabe:

Beispiel:

if (!surface_exists(surf))
{
    if (surface_get_depth_disable() == false)
    {
        surface_depth_disable(true);
    }
    surf = surface_create(room_width, room_height);
}

Die obige code prüft, ob die angegebene Oberfläche existiert, und wenn nicht, dann prüft sie den aktuellen Status der Oberflächentiefe buffer und wenn sie aktiviert ist, wird sie stattdessen deaktiviert, bevor sie die Oberfläche schließlich erstellt.
