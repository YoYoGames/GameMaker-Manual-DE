oberfläche_get_target

Mit dieser Funktion können Sie die ID der aktuellen Oberfläche ermitteln, auf die gezeichnet werden soll. Wenn Sie diese Fläche mit surface_create()erstellt haben, ist dies die positive Ganzzahl, die die eindeutige ID für diese Fläche darstellt

Wenn keine benutzerdefinierte Oberfläche eingestellt ist, aber die standardmäßige application_surface noch aktiviert ist, gibt die Funktion normalerweise 0 zurück. Wenn Sie jedoch zuvor die application_surface manipuliert haben, sollten Sie überprüfen, ob die erhaltene ID nicht mit dem aktuellen Wert der Variable application_surface übereinstimmt, falls dieser nicht mehr 0 ist.

Wenn keine benutzerdefinierte Oberfläche eingestellt ist und die Standard-Anwendungsoberfläche ebenfalls deaktiviert wurde, gibt die Funktion -1 zurück.

Syntax:

surface_get_target();

Rückgabe:

Surface ID or -1 (if no target surface is set)

Beispiel:

if surface_get_target() != -1
{
    surface_reset_target();
}
draw_surface(surf, 0, 0);

Die obige code prüft, ob das aktuelle Rendering-Ziel eine Fläche ist oder nicht, und wenn ja, setzt sie das Ziel zurück und zeichnet dann eine Fläche.
