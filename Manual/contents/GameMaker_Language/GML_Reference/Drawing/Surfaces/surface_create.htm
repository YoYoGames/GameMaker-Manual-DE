Oberfläche_erstellen

Diese Funktion wird verwendet, um eine Oberfläche zu erstellen und gibt den Index der Oberfläche zurück, der in einer Variablen für zukünftige Funktionsaufrufe gespeichert werden sollte. Wenn die Oberfläche zum ersten Mal erstellt wird, kann sie "Rauschen" enthalten, da es sich im Grunde nur um einen Bereich des Speichers handelt, der für diesen Zweck beiseite gelegt wird (und dieser Speicher kann immer noch Informationen enthalten), daher sollten Sie die Oberfläche vor der Verwendung mit einer Funktion wie draw_clear_alpha().

Es wird dringend empfohlen, dass alle Oberflächen mit einer Größe erstellt werden, die eine Potenz von 2 ist, d.h. 16, 128, 512 oder 1024 Pixel groß, zum Beispiel. Dies ist bei bestimmten platforms (wie Windows und MacOS) nicht unbedingt erforderlich, erhöht aber sicherlich die Kompatibilität auf diesen Zielsystemen, während es für HTML5 und Geräte unerlässlich ist, und es ist sehr wichtig, dass Sie daran denken, sonst könnten Sie später Probleme bekommen.

Syntax:

surface_create(w, h);

ArgumentTyp Beschreibung
w The width of the surface to be created.
h The height of the surface to be created.

Rückgabe:

Surface ID

Beispiel:

if !surface_exists(surf)
{
    surf = surface_create(1024, 1024);
    surface_set_target(surf);
    draw_clear_alpha(c_black, 0);
    surface_reset_target();
    view_surface_id[0] = surf;
}

Die obige code prüft, ob eine Oberfläche existiert, und wenn nicht, wird eine Oberfläche mit einer Breite von 1024 Pixeln und einer Höhe von 1024 Pixeln erstellt, wobei der Index der Variablen "surf" zugewiesen wird. Das Zeichenziel wird dann auf die neue Fläche gesetzt, die gelöscht und transparent gemacht wird, bevor das Zeichenziel auf die Anzeige zurückgesetzt wird. Schließlich wird die Fläche einer Ansicht zugewiesen.
