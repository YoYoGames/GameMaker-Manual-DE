anwendung_oberflaeche_ist_aktiviert

Diese Funktion gibt true zurück, wenn die Anwendungsoberfläche zum Zeichnen verwendet wird, oder false, wenn der Bildschirm buffer verwendet wird.

Syntax:

application_surface_is_enabled();

Rückgabe:

Beispiel:

if keyboard_check_pressed(vk_space)
{
    if application_surface_is_enabled()
    {
        application_surface_enable(false);
    }
    else
    {
        application_surface_enable(true);
    }
}

Die oben genannten code prüfen, ob eine Taste gedrückt wurde, und schalten die Anwendungsoberfläche je nach ihrem Zustand ein oder aus (wie in einem Optionsmenü).
