oberfläche_getpixel

Diese Funktion kann verwendet werden, um die Farbe eines bestimmten Pixels von einer Oberfläche zu erhalten, indem die lokalen Koordinaten der Oberfläche verwendet werden, wobei (0,0) die obere linke Ecke ist. Diese Funktion sollte nicht sehr oft verwendet werden, da sie extrem langsam ist und eine Pause im Spiel verursachen kann.

HINWEIS: Bei der Arbeit mit Flächen besteht die Möglichkeit, dass sie jederzeit nicht mehr existieren, da sie im Texturspeicher abgelegt sind. Sie sollten IMMER überprüfen, ob eine Fläche existiert, indem Sie surface_exists() überprüfen, bevor Sie sie direkt referenzieren.

Syntax:

surface_getpixel(surface_id, x, y);

ArgumentTyp Beschreibung
surface_id The ID of the surface.
x The x position on the surface from which to get the pixel.
y The y position on the surface from which to get the pixel.

Rückgabe:

Colour Constant

Beispiel:

col = surface_getpixel(surf, 56, 78 );

Dies gibt die Farbe des Pixels an den Koordinaten (56,78) der in der Variablen "surf" indizierten Oberfläche zurück.
