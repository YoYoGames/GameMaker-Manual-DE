surface_get_texture

Diese Funktion gibt einen speziellen Zeiger für die Textur der Oberfläche zurück. Dieser Wert kann dann in anderen Zeichenfunktionen verwendet werden, insbesondere in allgemeinen 3D- und einigen der 2D-Primitivfunktionen.

HINWEIS Bei der Arbeit mit Flächen besteht die Möglichkeit, dass sie jederzeit nicht mehr existieren, da sie im Texturspeicher abgelegt sind. Sie sollten IMMER überprüfen, ob eine Fläche existiert, indem Sie surface_exists() überprüfen, bevor Sie sie direkt referenzieren.

Syntax:

surface_get_texture(surface_id);

Argument Typ Beschreibung
surface_id Oberfläche ID The ID of the surface to get the width of.

Rückgabe:

Texture

Beispiel:

var tex;
tex = surface_get_texture(surf);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 480, 0, 0);
draw_vertex_texture(640, 480, 1, 0);
draw_vertex_texture(640, 480, 1, 1);
draw_vertex_texture(0, 480, 0, 1);
draw_primitive_end();

Die obige code zeichnet einen dreieckigen Streifen mit 4 Eckpunkten, der mit der Textur in der Variable "tex" texturiert ist, die wiederum von einer zuvor erstellten Oberfläche stammt.
