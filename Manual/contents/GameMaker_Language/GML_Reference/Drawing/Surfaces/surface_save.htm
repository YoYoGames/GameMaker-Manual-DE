Oberfläche_sichern

Mit dieser Funktion wird eine Oberfläche unter dem angegebenen Dateinamen auf der Festplatte gespeichert. Die Oberfläche muss als Datei im Format *.png gespeichert werden.

Syntax:

surface_save(surface_id, fname);

ArgumentTyp Beschreibung
surface_id The ID of the surface to set as the drawing target.
fname The name of the saved image file.

Rückgabe:

Beispiel:

if keyboard_check_pressed(ord("S")
{
    surface_save(surf, "test.png");
}

Die obige code prüft, ob der Benutzer die Taste "S" auf der Tastatur drückt, und wenn dies der Fall ist, wird die in der Variablen "surf" indizierte Oberfläche auf der Festplatte gespeichert.
