anwendung_oberflaeche_aktivieren

Mit dieser Funktion können Sie die Anwendungsoberfläche aktivieren oder deaktivieren. Standardmäßig ist die Anwendungsoberfläche aktiviert und alle Zeichnungen für die Draw-Ereignisse werden auf dieser Oberfläche ausgeführt und die Oberfläche wird auf den Bildschirm gezeichnet. Auf einigen älteren Geräten und bestimmten Chipsätzen kann dies jedoch zu einer schlechteren Leistung führen. In diesen Fällen können Sie unter switch das Zeichnen auf der Anwendungsoberfläche deaktivieren. Alle Zeichenereignisse werden dann direkt auf den Bildschirm übertragen buffer.

Sie können diese Funktion verwenden, um die Anwendungsoberfläche jederzeit in Ihrem Spiel zu aktivieren oder zu deaktivieren, aber Sie sollten beachten, dass es mindestens einen Frame (Schritt) braucht, bis die Änderungen wirksam werden. Ausschalten der Funktion switch schaltet alle Seitenverhältniskorrekturen aus.Die Anwendungsoberfläche muss in den Spieloptionen für das Ziel platform eingestellt werden. Das bedeutet, dass Sie über eine Methode verfügen sollten, die das Spiel auf die erforderlichen Geräte-, Fenster- oder Bildschirmabmessungen skaliert. Wenn Sie switch wieder einschalten, werden die Spieloptionen im nächsten Schritt wieder wirksam.

Es wird empfohlen, diese Funktion mit Bedacht zu verwenden, da sie sich darauf auswirkt, wie alles auf dem Bildschirm dargestellt wird. Im Allgemeinen sollten Sie diese Funktion als Option in einem Menü Ihres Spiels einrichten oder sie nur einmal zu Beginn des Spiels aufrufen und unter code angeben, ob Sie sie aktiviert oder deaktiviert haben.

Syntax:

application_surface_enable(enable);

ArgumentTyp Beschreibung
enable Set to enable (true) oder deaktivieren (false) Sie die Anwendungsoberfläche.

Rückgabe:

Beispiel:

if keyboard_check_pressed(vk_space)
{
    if application_surface_is_enabled()
    {
        application_surface_enable(false);
    }
    else
    {
        application_surface_enable(true);
    }
}

Die oben genannten code prüfen, ob eine Taste gedrückt wurde, und schalten die Anwendungsoberfläche je nach ihrem Zustand ein oder aus (wie in einem Optionsmenü).
