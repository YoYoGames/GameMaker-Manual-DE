texturegroup_get_sprites

Mit dieser Funktion können Sie den sprite Index jeder der sprites zugewiesenen Texturseiten innerhalb der angegebenen Texturgruppe abrufen. Sie geben den Texturgruppen-ID-String an (wie im Texturgruppen-Editor definiert) und die Funktion gibt ein 1D-Array zurück, in dem jeder Eintrag den sprite Index für eine sprite Ressource enthält. Wenn die Funktion fehlschlägt - d.h. wenn eine ungültige Gruppe angegeben wird oder der Gruppe keine Textur zugewiesen ist - dann ist das Array leer (0 Länge).

Syntax:

texturegroup_get_sprites(tex_id)

Argument Typ Beschreibung
tex Zeichenfolge The name of the texture group to check (a string)

Rückgabe:

Array

Beispiel:

var _tex_array = texturegroup_get_sprites( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i)
{
    show_debug_message("Sprite " + string(i) + " Index:" + string(tex_array[i]));
}

Die obige code ruft die sprite Indizes für die Texturgruppe "MainMenu" ab und zeigt diese IDs dann im Konsolenausgabefenster an.
