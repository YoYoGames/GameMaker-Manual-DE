textur_bündig

Diese Funktion kann verwendet werden, um eine Texturseite oder eine Gruppe von Texturseiten zu "flushen", d.h. sie aus dem VRAM zu entfernen, wenn sie nicht mehr benötigt werden. Sie geben die eindeutige Texturseiten-ID an (wie von den texturegroup_* Funktionen zurückgegeben), um eine einzelne Seite zu flushen, oder Sie können einen Texturgruppennamen angeben (wie im Texturgruppeneditor definiert), um alle Texturseiten in der Gruppe zu flushen.

Syntax:

texture_flush(tex_id);

Argument Typ Beschreibung
tex_id Textur Zeiger oder String The texture page pointer or a texture group name (a string)

Rückgabe:

N/A

Beispiel:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i)
{
    texture_flush(_tex_array[i]);
}

Die obige code wird alle Texturseiten unter der Texturgruppe "MainMenu" spülen.
