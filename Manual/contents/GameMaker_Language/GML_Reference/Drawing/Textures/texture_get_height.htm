texture_get_height

Gibt die Höhe der Textur mit der angegebenen ID zurück, die immer ein Wert im Bereich von 0 - 1 ist und beim Mappen von Texturen auf Modelle oder Primitive verwendet werden kann.

Syntax:

texture_get_height(tex);

Argument Typ Beschreibung
tex Textur-Zeiger The texture page asset pointer to use

Rückgabe:

Real

Beispiel:

tex_h = texture_get_height(surface_get_texture(global.Surf));

Mit der obigen code wird die Höhe der Textur von einer zuvor erstellten Oberfläche übernommen.
