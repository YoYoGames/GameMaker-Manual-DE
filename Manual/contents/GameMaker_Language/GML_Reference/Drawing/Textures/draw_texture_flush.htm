zeichnen_textur_bündig

Mit dieser Funktion können Sie alle Texturen aus dem Videospeicher entfernen, und sie werden dann bei der ersten Verwendung neu geladen. Dies ist der einzige effektive Weg, um den Videospeicher zu verwalten, wenn Sie mehrere Texturseiten für ein Spiel haben, und Sie sollten den Texturspeicher Sie sollten den Texturspeicher zwischen den Levels Ihres Spiels leeren und die Grafiken mit Hilfe der Texturgruppenfunktion organisieren, um sicherzustellen, dass so wenig Texturen wie möglich verwendet werden.

Es ist erwähnenswert, dass standardmäßig auf allen Targets Textur-Seiten nur geladen werden, wenn sie benötigt werden, und Sie können die verschiedenen Prefetch- und Flush-Funktionen für sprites ( hier zu finden) verwenden verwenden, um die Dinge ebenso zu verwalten wie diese Funktion.

Syntax:

draw_texture_flush();

Rückgabe:

Beispiel:

draw_texture_flush();

Das obige code löscht den Videospeicher von Texturdaten und würde wahrscheinlich im Erstellungsereignis der ersten Instanz eines im Raum platzierten Objekts platziert werden.
