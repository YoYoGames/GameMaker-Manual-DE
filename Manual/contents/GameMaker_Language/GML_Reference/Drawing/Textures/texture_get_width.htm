textur_get_width

Gibt die Breite der Textur mit der angegebenen ID zurück, die immer ein Wert im Bereich von 0 - 1 ist und beim Zuordnen von Texturen zu Modellen oder Primitiven verwendet werden kann.

Syntax:

texture_get_width(tex);

Argument Typ Beschreibung
tex Textur-Zeiger The texture page asset pointer to use

Rückgabe:

Real

Beispiel:

tex_w = texture_get_width(sprite_get_texture(spr_Model_tex, 0));

Die obige code erhält die Breite der Textur aus einem sprite Asset.
