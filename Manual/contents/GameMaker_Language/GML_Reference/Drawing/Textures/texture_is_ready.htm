textur_ist_bereit

Diese Funktion kann verwendet werden, um zu prüfen, ob eine bestimmte Texturseite entpackt wurde und zur Verwendung bereit ist oder ob eine Gruppe von Texturseiten entpackt wurde und zur Verwendung bereit ist. Sie geben die eindeutige Texturseiten-ID (wie von den Funktionen texturegroup_* zurückgegeben) oder die Texturgruppen-ID-Zeichenkette (wie im Texturgruppen-Editor definiert) an, und die Funktion gibt true zurück, wenn sie ausgepackt wurden, oder andernfalls false.

Syntax:

texture_is_ready(tex_id);

Argument Typ Beschreibung
tex_id Textur Zeiger oder String The texture page pointer or a texture group name (a string)

Rückgabe:

Boolean

Beispiel:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i)
{
    if !texture_is_ready(_tex_array[i])
    {
        texture_prefetch(_tex_array[i]);
    }
}

Die obige code ruft die Texturseiten-IDs für die Texturgruppe "MainMenu" ab, prüft dann, ob sie entpackt sind, und wenn nicht, werden sie in den VRAM vorgeladen.
