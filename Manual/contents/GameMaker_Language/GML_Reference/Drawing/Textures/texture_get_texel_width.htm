textur_get_texel_width

Ein Texel oder Texturelement ist die grundlegende Einheit des in der Computergrafik verwendeten Texturraums. Texturen werden durch Arrays von Texeln dargestellt, so wie Bilder durch Arrays von Pixeln dargestellt werden. Diese Funktion gibt die Breite eines einzelnen Texels aus der Texturseite des verwendeten Bildes zurück.

Syntax:

texture_get_texel_width(tex);

Argument Typ Beschreibung
tex Textur-Zeiger The texture page asset pointer to use

Rückgabe:

Real

Beispiel:

var tex = sprite_get_texture(sprite_index, 0);
tex_w = texture_get_texel_width(tex);
tex_h = texture_get_texel_height(tex);

Die obige code erhält die Texel-Breite und -Höhe der Textur aus einem sprite Asset.
