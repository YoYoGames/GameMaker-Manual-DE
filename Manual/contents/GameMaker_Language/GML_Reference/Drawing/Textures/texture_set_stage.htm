textur_set_stage

Diese Funktion legt für den angegebenen "Slot" eine Textur fest, die verwendet werden soll. Die Anzahl der verfügbaren Stage-"Slots" hängt von der platform ab, für die Sie kompilieren, wobei maximal 8 für Windows, Mac und Linux zur Verfügung stehen, aber auf weniger leistungsfähigen Android -Geräten (z. B.) kann diese Anzahl nur 2 betragen. Es ist auch erwähnenswert, dass der erste Stage-Slot (1) immer automatisch von GameMaker verwendet wird.

HINWEIS: Diese Funktion bewirkt nichts außerhalb des Kontextes eines laufenden Shaders! Siehe Shader für weitere Informationen.

Syntax:

texture_set_stage(stage, tex);

Argument Typ Beschreibung
stage Real The texture "slot" to use.
tex Textur-Zeiger The texture to use.

Rückgabe:

N/A

Beispiel:

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set(GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

Die obige code holt sich das Handle für den Sampler innerhalb des Shaders, der als "GlassShader" indiziert ist, und setzt dann diese Shader-Konstante auf die angegebene sprite Textur.
