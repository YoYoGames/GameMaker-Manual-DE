texture_prefetch

Diese Funktion kann verwendet werden, um eine Texturseite oder eine Gruppe von Texturseiten "vorzuholen", d.h. sie bei Bedarf in den VRAM zu laden. Sie geben die eindeutige ID der Texturseite an (wie sie von den texturegroup_* Funktionen zurückgegeben wird), um eine einzelne Seite vorzuholen, oder Sie können einen Texturgruppennamen angeben (wie im Texturgruppen-Editor definiert), um alle Texturseiten der Gruppe vorzuholen.

Syntax:

texture_prefetch(tex_id);

Argument Typ Beschreibung
tex_id Textur Zeiger oder String The texture page pointer or a texture group name (a string)

Rückgabe:

N/A

Beispiel:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    texture_prefetch(_tex_array[i]);
}

Die obige code wird alle Texturseiten unter der Texturgruppe "MainMenu" vorwegnehmen.
