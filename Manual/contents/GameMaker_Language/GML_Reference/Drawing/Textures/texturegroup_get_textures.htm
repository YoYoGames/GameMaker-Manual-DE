texturegroup_get_textures

Diese Funktion kann verwendet werden, um die Texturseiten-IDs der einzelnen Seiten einer Texturgruppe abzurufen. Sie geben den Texturgruppen-ID-String an (wie im Texturgruppen-Editor definiert), und die Funktion gibt ein 1D-Array zurück, wobei jeder Eintrag im Array eine einzelne Texturseiten-ID ist. Wenn die Funktion fehlschlägt - d.h. wenn eine ungültige Gruppe angegeben wird oder der Gruppe keine Textur zugewiesen ist - dann ist das Array leer (0 Länge).

Syntax:

texturegroup_get_textures(tex_id)

Argument Typ Beschreibung
tex_id Zeichenfolge The name of the texture group to check (a string)

Rückgabe:

Array of Texture Pointers

Beispiel:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i)
{
    if texture_is_ready(_tex_array[i])
    {
        texture_prefetch(_tex_array[i]);
    }
}

Die obige code ruft die Texturseiten-IDs für die Texturgruppe "MainMenu" ab, prüft dann, ob sie ausgepackt sind, und wenn ja, werden sie im VRAM abgelegt.
