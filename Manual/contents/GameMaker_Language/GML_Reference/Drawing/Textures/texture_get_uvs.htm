textur_get_uvs

Diese Funktion gibt ein 1D-Array mit 4 Elementen zurück, die die UV-Koordinaten für den angegebenen Texturzeiger repräsentieren, und füllt das Array mit den folgenden Werten auf:

  • [0] = links
  • [1] = oben
  • [2] = rechts
  • [3] = unten
  • [4] = Anzahl der Pixel, die das Asset compiler von der linken Seite sprites abgeschnitten hat (nursprite Assets)
  • [5] = Anzahl der Pixel, die das Asset compiler von der oberen Seite sprites abgeschnitten hat (nursprite Assets)
  • [6] = normalisierter Prozentsatz der Pixeldaten der ursprünglichen sprites Breite, die auf der Texturseite gespeichert wurden (nursprite Assets)
  • [7] = normalisierter Prozentsatz der Pixeldaten von der ursprünglichen Höhe sprites, die auf der Texturseite gespeichert wurden (nursprite assets)

Dieser Wert kann dann in anderen Zeichenfunktionen verwendet werden, insbesondere in allgemeinen 3D- und einigen der 2D-Primitivfunktionen sowie den Shader-Funktionen. Wenn Sie die UVS für eine sprite benötigen, können Sie die sprite_get_uvs() und für eine Schriftart verwenden, font_get_uvs().

Syntax:

texture_get_uvs(texid)

Argument Typ Beschreibung
texid Textur The texture pointer to get the UVS for

Rückgabe:

Array (4 - 8 elements)

Beispiel:

var _tex = surface_get_texture(surf_back);
var _uvs = texture_get_uvs(tex);
var _uvs_left = _uvs[0];
var _uvs_top = _uvs[1];
var _uvs_right = _uvs[2];
var _uvs_bottom = _uvs[3];

Die obige code ruft zunächst die Textur für die in surf_back gespeicherte Oberfläche ab und ermittelt dann die UV-Koordinaten für diese Textur. Anschließend werden die linken, oberen, rechten und unteren UV-Koordinaten aus dem zurückgegebenen Array abgerufen und in lokalen Variablen gespeichert.
