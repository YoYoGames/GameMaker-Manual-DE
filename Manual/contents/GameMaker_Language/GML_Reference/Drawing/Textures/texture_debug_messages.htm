texture_debug_messages

Diese Funktion kann verwendet werden, um Textur-Debug-Meldungen zu aktivieren oder zu deaktivieren. Wenn sie aktiviert ist (auf true gesetzt), werden zusätzliche Informationen über die Verwendung von Texturseiten an das Ausgabefenster gesendet. Setzen Sie sie auf false, um diese Ausgabe wieder zu deaktivieren.

Syntax:

texture_debug_messages(enable);

ArgumentTyp Beschreibung
enable Enable or disable the texture debug messages

Rückgabe:

Beispiel:

if global.debug
{
    texture_debug_messages(true);
}

Die obige code prüft den Wert einer globalen Variablen, und wenn sie auf true gesetzt ist, werden Texturdebugmeldungen aktiviert.
