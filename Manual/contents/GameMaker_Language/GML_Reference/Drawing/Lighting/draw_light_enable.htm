draw_light_enable

Diese Funktion wird verwendet, um ein definiertes Licht zu aktivieren. Wenn Sie ein Positions- oder Richtungslicht definieren, müssen Sie ihm eine Indexnummer zuweisen, die dann von dieser Funktion switch verwendet wird, um das Licht ein- oder auszuschalten. Die Voreinstellung ist deaktiviert (false).

HINWEIS: Es stehen nur 8 Hardware-Lichter zur Verfügung, so dass nur 8 definierte Lichter gleichzeitig aktiviert werden können (obwohl mehr definiert werden können).

Syntax:

draw_light_enable(ind, enable);

Argument Typ Beschreibung
ind Real The index number of the light (from 0 to 7)
enable Boolesche Enable or disable all lighting (true oder false)

Rückgabe:

N/A

Beispiel:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

Die obige code wird die Beleuchtung für die gesamte Szene aktivieren, dann ein weißes gerichtetes Licht im Raum definieren und schließlich dieses Licht einschalten.
