draw_light_define_direction

Mit dieser Funktion kann ein gerichtetes Licht definiert werden, wobei die Richtung und die Intensität durch die Werte für die x-, y- und z-Einheiten vector festgelegt werden (d. h. sie müssen zwischen -1 und 1 liegen, wobei 1 oder -1 die volle Intensität und 0 keine Intensität bedeutet). Die Lichtern kann auch eine Farbe zugewiesen werden, was sich ebenfalls auf die wahrgenommene Intensität des Lichts auswirkt, da bestimmte Farben "dunkler" erscheinen als andere. Sie müssen dem Licht auch eine Indexnummer geben, die in anderen Funktionen verwendet wird, um es zu referenzieren. referenzieren.

HINWEIS: Es stehen nur 8 Hardware-Lichter zur Verfügung, so dass nur 8 definierte Lichter gleichzeitig aktiviert werden können (obwohl mehr definiert werden können).

Syntax:

draw_light_define_direction(ind, x, y, z, col)

ArgumentTyp Beschreibung
ind The index number of the light (arbitrary)
x The x component of the light vector
y The y component of the light vector
z The z component of the light vector
col The colour to use for the light (either a constant, a real or a hex value).

Rückgabe:

Beispiel:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

Die obige code wird die Beleuchtung für die gesamte Szene aktivieren, dann ein weißes gerichtetes Licht im Raum definieren und schließlich dieses Licht einschalten.
