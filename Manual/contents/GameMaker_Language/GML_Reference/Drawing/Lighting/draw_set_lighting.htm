beleuchtung_zeichnen_setzen

Mit dieser Funktion können Sie alle Lichteffekte aktivieren. Die Voreinstellung ist deaktiviert (false).

Syntax:

draw_set_lighting(enable);

Argument Typ Beschreibung
enable Boolesche Enable or disable all lighting (true oder falsch)

Rückgabe:

N/A

Beispiel:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

Die obige code wird die Beleuchtung für die gesamte Szene aktivieren, dann ein weißes gerichtetes Licht im Raum definieren und schließlich dieses Licht einschalten.
