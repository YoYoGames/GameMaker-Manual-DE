draw_light_define_ambient

Diese Funktion dient zur Steuerung des Umgebungslichts einer Szene, d. h. des Lichts, das in einer Szene vorhanden ist, auch wenn keine punktuellen oder gerichteten Lichtquellen definiert wurden. Es handelt sich dabei um die Gesamtfarbe und -helligkeit (oder Dunkelheit) einer Szene. Die Standardfarbe ist c_black.

Syntax:

draw_light_define_ambient(col);

ArgumentTyp Beschreibung
col The colour to use (either a constant, a real or a hex value).

Rückgabe:

Beispiel:

draw_light_define_ambient(c_white);

Die obige code definiert die Umgebungsbeleuchtung als weiß.
