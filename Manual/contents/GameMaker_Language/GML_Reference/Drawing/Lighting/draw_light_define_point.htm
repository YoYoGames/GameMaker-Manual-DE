draw_light_define_point

Mit dieser Funktion können Sie ein Positionslicht definieren, wobei Sie die x-, y- und z-Position des Lichts, den Lichtbereich und seine Farbe festlegen können (was sich auch auf die wahrgenommene Intensität des Lichts auswirkt, da bestimmte Farben "dunkler" erscheinen als andere). andere). Sie müssen dem Licht auch eine Indexnummer geben, die in anderen Funktionen verwendet wird, um es zu referenzieren.

HINWEIS: Es stehen nur 8 Hardware-Lichter zur Verfügung, so dass nur 8 definierte Lichter gleichzeitig aktiviert werden können (obwohl mehr definiert werden können).

Syntax:

draw_light_define_point(ind, x, y, z, range, col);

ArgumentTyp Beschreibung
ind The index number of the light (arbitrary)
x The x position of the light
y The y position of the light
z The z position of the light
range The light range (in pixels)
col The colour to use for the light (either a constant, a real or a hex value).

Rückgabe:

Beispiel:

draw_set_lighting(true);
draw_light_define_point(1, 200, 123, 50, 2000, c_white);
draw_light_enable(1, true);

Die obige code wird die Beleuchtung für die gesamte Szene aktivieren, dann ein weißes Licht an einem bestimmten Punkt im Raum definieren und schließlich dieses Licht einschalten.
