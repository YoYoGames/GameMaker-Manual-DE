zeichnen_licht_holen

Diese Funktion liefert die angegebenen Lichtparameter als Array mit den folgenden 6 Elementen -

  • [0] = aktiviert / deaktiviert (true / false)
  • [1] = Lichttyp Konstante (siehe Tabelle unten)
  • [2] = Position x
  • [3] = y-Position
  • [4] = z-Position
  • [5] = Lichtradius (nur für Punktlichter, bei gerichteten Lichtern ist er 0)
  • [6] = helle Farbe (eine echte)

Der Lichttyp (Element [1]) kann eine der beiden folgenden Konstanten sein:

Lichttyp Konstant
Konstante Beschreibung
lighttype_dir Das Licht ist ein gerichtetes Licht
lighttype_point Das Licht ist ein Punktlicht

Syntax:

draw_light_get(ind);

Argument Typ Beschreibung
ind Real The index number of the light (from 0 to 7)

Rückgabe:

Array

Beispiel:

light_a = draw_light_get(1);
if light_a[5] < 200
{
    light_a[5] += 5;
    draw_light_define_point(1, 200, 123, 50, light_a[5], c_white);
}

Die obige code erhält die Werte, die zur Definition des Lichts verwendet werden, indiziert als "1", prüft dann den Radius und wenn dieser kleiner als 200 ist, wird er erhöht und der Lichtradius auf den neuen Wert gesetzt.
