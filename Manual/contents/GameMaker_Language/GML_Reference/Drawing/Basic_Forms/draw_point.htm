Punkt_zeichnen

Mit dieser Funktion können Sie ein einzelnes Pixel an einer beliebigen Stelle des Bildschirms zeichnen, wobei die aktuell eingestellte Zeichenfarbe und Alpha verwendet werden.

Syntax:

draw_point(x, y);

ArgumentTyp Beschreibung
x The x coordinate of the point to be drawn.
y The y coordinate of the point to be drawn.

Rückgabe:

Beispiel:

draw_set_colour(c_yellow);
draw_point(100,100);

Dadurch wird ein gelbes Pixel an der Position (100,100) gezeichnet.
