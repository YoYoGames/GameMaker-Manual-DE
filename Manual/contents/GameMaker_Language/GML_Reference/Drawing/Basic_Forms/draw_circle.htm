zeichnen_kreis

Mit dieser Funktion können Sie entweder einen Umriss eines Kreises oder einen gefüllten Kreis zeichnen. Wie genau die Zeichnung ist, können Sie mit der Funktion draw_set_circle_precision().

HINWEIS: Wenn Sie eine Form mit einem Shader zeichnen wollen, sollten Sie wissen, dass die meisten Shader die folgenden Eingaben erwarten: Scheitelpunkt, Textur, Farbe. Wenn Sie diese Funktion verwenden, werden jedoch nur Scheitelpunkt- und Farbdaten übergeben, so dass der Shader möglicherweise nichts zeichnet (oder etwas zeichnet, aber nicht korrekt). Wenn Sie Formen auf diese Weise zeichnen müssen, sollte der Shader entsprechend angepasst werden.

Syntax:

draw_circle(x, y, r, outline);

Argument Typ Beschreibung
x Real The x coordinate of the center of the circle.
y Real The y coordinate of the center of the circle.
r Real The circle's radius (length from its center to its edge)
outline Boolesche Whether the circle is drawn filled (false) oder als ein Pixel breiter Umriss (true).

Rückgabe:

N/A

Beispiel:

draw_set_colour(c_white);
draw_circle(100, 100, 50, true);

Damit wird ein ein Pixel breiter weißer Kreis mit einem Radius von 50 Pixeln gezeichnet.
