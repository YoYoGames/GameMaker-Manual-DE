draw_set_circle_precision

Um die Leistung beim Zeichnen von Kreisen zu optimieren, zeichnet GameMaker grundsätzlich eine Polygonform mit genügend Seiten, um sie kreisförmig erscheinen zu lassen. Je nachdem, wie groß oder klein Ihre Kreise sein sollen, können Sie jedoch feststellen, dass eine Änderung dieses Werts diesen Wert ändern, um die Leistung Ihres Spiels zu verbessern oder die Kreise besser aussehen zu lassen. Der Präzisionswert, den Sie in diese Funktion eingeben können, ist die Anzahl der Seiten, die das Kreispolygon hat, und diese Zahl kann zwischen 4 und 64 liegen, muss aber durch 4 teilbar sein, der Standardwert ist 24.

HINWEIS: Wenn Sie eine Form mit einem Shader zeichnen wollen, sollten Sie wissen, dass die meisten Shader folgende Eingaben erwarten: Scheitelpunkt, Textur, Farbe. Wenn Sie diese Funktion verwenden, werden jedoch nur Vertex- und Farbdaten übergeben, so dass der Shader möglicherweise nichts zeichnet (oder etwas zeichnet, aber nicht richtig). Wenn Sie Formen auf diese Weise zeichnen müssen, sollte der Shader entsprechend angepasst werden.

Syntax:

draw_set_circle_precision(precision);

ArgumentTyp Beschreibung
precision The precision of all circles drawn after this function is called.

Rückgabe:

Beispiel:

draw_set_circle_precision(12);

Die obige code zeichnet alle Kreise unter Verwendung eines 12-seitigen Polygons.
