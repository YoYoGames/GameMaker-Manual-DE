draw_arrow

Diese Funktion zeichnet einen Pfeil vom Punkt (x1,y1) zum Punkt (x2,y2). Der Schaft des Pfeils wird entlang dieser Punkte gezeichnet, wobei die eigentliche Pfeilspitze am Ende gezeichnet wird, wobei die Größe der Pfeilspitze durch die argument "size" definiert ist und als Teil des Stiels berechnet wird, so dass der Endpunkt immer an der durch x2,y2 definierten Position ausgerichtet ist. Die Breite der Pfeilspitze wird automatisch im Verhältnis zur Länge berechnet.

HINWEIS: Wenn Sie eine Form mit einem Shader zeichnen wollen, sollten Sie wissen, dass die meisten Shader folgende Eingaben erwarten: Scheitelpunkt, Textur, Farbe. Wenn Sie diese Funktion verwenden, werden jedoch nur Vertex- und Farbdaten übergeben, so dass der Shader möglicherweise nichts zeichnet (oder etwas zeichnet, aber nicht richtig). Wenn Sie Formen auf diese Weise zeichnen müssen, sollte der Shader entsprechend angepasst werden.

Syntax:

draw_arrow(x1, y1, x2, y2, size);

ArgumentTyp Beschreibung
x1 The x coordinate of the start of the line.
y1 The y coordinate of the start of the line.
x2 The x coordinate of the end of the line (where the arrowhead ends).
y2 The y coordinate of the end of the line (where the arrowhead ends).
size The length of the arrow in pixels.

Rückgabe:

Beispiel:

draw_arrow(x, y, mouse_x, mouse_y, 10);

Die obige code wird einen Pfeil von der Position der Instanz zur Position der Maus zeichnen, wobei die Pfeilspitze 10 Pixel lang ist.
