draw_point_colour

Mit dieser Funktion können Sie ein einzelnes Pixel an einer beliebigen Stelle des Bildschirms mit einer von Ihnen definierten Farbe zeichnen. Die Farbeinstellungen überschreiben die Grundfarbe, die mit der Funktion draw_set_colour().

HINWEIS: Wenn Sie eine Form mit einem Shader zeichnen wollen, sollten Sie wissen, dass die meisten Shader folgende Eingaben erwarten: Scheitelpunkt, Textur, Farbe. Wenn Sie diese Funktion verwenden, werden jedoch nur Vertex- und Farbdaten übergeben, so dass der Shader möglicherweise nichts zeichnet (oder etwas zeichnet, aber nicht richtig). Wenn Sie Formen auf diese Weise zeichnen müssen, sollte der Shader entsprechend angepasst werden.

Syntax:

draw_point_colour(x, y, col1);

ArgumentTyp Beschreibung
x The x coordinate of the point.
y The y coordinate of the point.
col1 The colour of the point.

Rückgabe:

Beispiel:

draw_point_colour(50, 50, c_red);

Dies würde ein rotes Pixel bei (50,50) zeichnen.
