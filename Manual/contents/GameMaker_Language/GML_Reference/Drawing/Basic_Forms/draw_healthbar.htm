draw_healthbar

Mit dieser Funktion können Sie einen farbigen Balken zeichnen, um einen konstanten Wert anzuzeigen. Obwohl die Funktion das Wort "Gesundheitsbalken" verwendet, können Sie damit alles anzeigen, was Sie möchten, solange der anzuzeigende Wert ein Prozentwert zwischen 0 und 100 ist (ein Prozentsatz eines beliebigen Wertes kann mit der Formel (AKTUELLER_WERT / MAXIMALER_WERT) * 100 berechnet werden), so dass Sie damit zum Beispiel Macht, Gesundheit, Mana, Zeit oder alles andere anzeigen können, was von einer Balkenanzeige profitieren würde.

Sie können verschiedene Einstellungen vornehmen, die den visuellen Aspekt des Balkens verändern. Diese sind in der folgenden Abbildung dargestellt (beachten Sie, dass die hier verwendeten Farben c_green für den Höchstwert und c_red für den Mindestwert sind, aber Sie können die von Ihnen gewünschten Farben verwenden):

HINWEIS: Wenn Sie eine Form mit einem Shader zeichnen wollen, sollten Sie wissen, dass die meisten Shader die folgenden Eingaben erwarten: Scheitelpunkt, Textur, Farbe. Wenn Sie diese Funktion verwenden, werden jedoch nur Scheitelpunkt- und Farbdaten übergeben, so dass der Shader möglicherweise nichts zeichnet (oder etwas zeichnet, aber nicht korrekt). Wenn Sie Formen auf diese Weise zeichnen müssen, sollte der Shader entsprechend angepasst werden.

Syntax:

draw_healthbar(x1, y1, x2, y2, amount, backcol, mincol, maxcol, direction, showback, showborder);

Argument Typ Beschreibung
x1 Real The x coordinate of the left of the healthbar
y1 Real The y coordinate of the top of the healthbar
x2 Real The x coordinate of the right of the healthbar
y2 Real The y coordinate of the bottom of the healthbar
amount Real The variable which defines total health (between 0 and 100)
backcol Farbe The colour of the background for the bar
mincol Farbe The colour of the bar when at no health (0)
maxcol Farbe The colour of the bar when at full health (100)
direction Real Where the bar is "anchored" (0 = left, 1 = right, 2 = top, 3 = bottom)
showback Boolesche Whether the bar's background is drawn (true) oder nicht (false). Bei "false" wird die Backcol argument nicht beachtet.
showborder Boolesche Whether the bar's elements have a 1px wide black border (true) oder nicht (false).

Rückgabe:

N/A

Beispiel:

var pc;
pc = (hp / max_hp) * 100;
draw_healthbar(100, 100, 500, 200, pc, c_black, c_red, c_lime, 0, true, true)

Die obige code verwendet den prozentualen Wert, den ihr aus den Variablen "hp" und "hp_max" erhaltet, um eine standardmäßige rot/grüne Gesundheitsleiste zu zeichnen.
