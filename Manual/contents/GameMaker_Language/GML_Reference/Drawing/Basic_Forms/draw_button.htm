draw_button

Diese Funktion zeichnet einen sehr einfachen, rechteckigen "Knopf" unter Verwendung der aktuell gewählten Zeichenfarbe und Alpha, wobei die obere argument definiert, wie der abgeschrägte Randeffekt aussieht, wie im Bild unten gezeigt:

HINWEIS: Wenn Sie eine Form mit einem Shader zeichnen wollen, sollten Sie wissen, dass die meisten Shader die folgenden Eingaben erwarten: Scheitelpunkt, Textur, Farbe. Wenn Sie diese Funktion verwenden, werden jedoch nur Scheitelpunkt- und Farbdaten übergeben, so dass der Shader möglicherweise nichts zeichnet (oder etwas zeichnet, aber nicht korrekt). Wenn Sie Formen auf diese Weise zeichnen müssen, sollte der Shader entsprechend angepasst werden.

Syntax:

draw_button(x1, y1, x2, y2, up);

Argument Typ Beschreibung
x1 Real The x coordinate of the left of the button
y1 Real The y coordinate of the top of the button
x2 Real The x coordinate of the right of the button
y2 Real The y coordinate of the bottom of the button
up Boolesche Whether the button is up (true) oder nach unten (false)

Rückgabe:

N/A

Beispiel:

draw_button(100, 100, 200, 150, !mouse_check_button(mb_left));

Damit wird eine Schaltfläche gezeichnet, die gedrückt erscheint, wenn die linke Maustaste gedrückt gehalten wird.
