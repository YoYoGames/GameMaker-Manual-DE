draw_ellipse_colour

Mit dieser Funktion können Sie entweder einen Umriss einer Ellipse oder eine gefüllte Ellipse zeichnen, indem Sie einen rechteckigen Bereich definieren, in den die Ellipse dann eingepasst wird. Bei einer gefüllten Ellipse können Sie die inneren und äußeren Füllfarben festlegen. Wenn diese Farben nicht identisch sind, erhalten Sie einen Farbverlauf (die Farbeinstellungen überschreiben die mit der Funktion draw_set_colour()). Wie genau die Zeichnung ist, können Sie mit der Funktion draw_set_circle_precision().

HINWEIS: Wenn Sie eine Form mit einem Shader zeichnen wollen, sollten Sie wissen, dass die meisten Shader die folgenden Eingaben erwarten: Scheitelpunkt, Textur, Farbe. Wenn Sie diese Funktion verwenden, werden jedoch nur Scheitelpunkt- und Farbdaten übergeben, so dass der Shader möglicherweise nichts zeichnet (oder etwas zeichnet, aber nicht korrekt). Wenn Sie Formen auf diese Weise zeichnen müssen, sollte der Shader entsprechend angepasst werden.

Syntax:

draw_ellipse_colour(x1, y1, x2, y2, col1, col2, outline);

Argument Typ Beschreibung
x1 Real The x coordinate of the left of the ellipse.
y1 Real The y coordinate of the top of the ellipse.
x2 Real The x coordinate of the right of the ellipse.
y2 Real The y coordinate of the bottom of the ellipse.
col1 Farbe The colour of the center.
col2 Farbe The colour of the edge.
outline Boolesche Whether the ellipse is an outline (true) or filled in (false).

Rückgabe:

N/A

Beispiel:

draw_ellipse_colour(50, 50, 200, 100, c_black, c_white, false);

Dies würde eine ausgefüllte, horizontal gestreckte Ellipse mit schwarzer Mitte und weißem Rand zeichnen.
