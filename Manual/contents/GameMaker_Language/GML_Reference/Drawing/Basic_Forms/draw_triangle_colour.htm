draw_triangle_colour

Mit dieser Funktion können Sie entweder einen Umriss eines Dreiecks oder ein ausgefülltes Dreieck zeichnen. Bei einem gefüllten Dreieck können Sie die einzelnen Farben für jeden Eckpunkt festlegen, und wenn diese Farben nicht gleich sind, erhalten Sie einen Verlaufseffekt von einem zum anderen (die Farbeinstellungen überschreiben die mit der Funktion draw_set_colour()).

HINWEIS: Wenn Sie eine Form mit einem Shader zeichnen wollen, sollten Sie wissen, dass die meisten Shader die folgenden Eingaben erwarten: Scheitelpunkt, Textur, Farbe. Wenn Sie diese Funktion verwenden, werden jedoch nur Scheitelpunkt- und Farbdaten übergeben, so dass der Shader möglicherweise nichts zeichnet (oder etwas zeichnet, aber nicht korrekt). Wenn Sie Formen auf diese Weise zeichnen müssen, sollte der Shader entsprechend angepasst werden.

Syntax:

draw_triangle_colour(x1, y1, x2, y2, x3, y3, col1, col2, col3, outline);

Argument Typ Beschreibung
x1 Real The x coordinate of the triangle's first corner.
y1 Real The y coordinate of the triangle's first corner.
x2 Real The x coordinate of the triangle's second corner.
y2 Real The y coordinate of the triangle's second corner.
x3 Real The x coordinate of the triangle's third corner.
y3 Real The y coordinate of the triangle's third corner.
col1 Farbe The colour of the first corner.
col2 Farbe The colour of the second corner.
col3 Farbe The colour of the third corner.
outline Boolesche Whether the triangle is an outline (true) or filled in (false).

Rückgabe:

N/A

Beispiel:

draw_triangle_colour(200, 200, 300, 200, 200, 300, c_red, c_blue, c_blue, false);

Damit würde ein ausgefülltes gleichschenkliges rechtwinkliges Dreieck gezeichnet, mit Rot am rechten Winkel und Blau an den beiden anderen Ecken.
