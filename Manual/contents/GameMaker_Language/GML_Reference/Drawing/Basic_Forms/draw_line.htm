zeichnen_linie

Mit dieser Funktion können Sie eine 1 Pixel breite Linie zwischen zwei beliebigen Punkten im Spielraum zeichnen. Bitte beachten Sie, dass die zu zeichnende Linie möglicherweise unterschiedliche Werte (+/-1 auf x, y) benötigt, um mit den gewünschten Abmessungen gezeichnet zu werden, da die verschiedenen unterstützten platforms.

HINWEIS: Wenn Sie eine Form mit einem Shader zeichnen wollen, sollten Sie wissen, dass die meisten Shader folgende Eingaben erwarten: Scheitelpunkt, Textur, Farbe. Wenn Sie diese Funktion verwenden, werden jedoch nur Vertex- und Farbdaten übergeben, so dass der Shader möglicherweise nichts zeichnet (oder etwas zeichnet, aber nicht richtig). Wenn Sie Formen auf diese Weise zeichnen müssen, sollte der Shader entsprechend angepasst werden.

Syntax:

draw_line(x1, y1, x2, y2);

ArgumentTyp Beschreibung
x1 The x coordinate of the start of the line.
y1 The y coordinate of the start of the line.
x2 The x coordinate of the end of the line.
y2 The y coordinate of the end of the line.

Rückgabe:

Beispiel:

draw_set_colour(c_lime);
draw_line(50,50,150,50);

Dadurch wird eine hellgrüne horizontale Linie von Punkt (50,50) zu Punkt (150,50) gezeichnet.
