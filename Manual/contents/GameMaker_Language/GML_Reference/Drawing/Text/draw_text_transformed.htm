draw_text_transformed

Diese Funktion zeichnet Text auf ähnliche Weise wie draw_text() nur dass Sie jetzt wählen können, ob der Text entlang der horizontalen oder vertikalen Achse skaliert werden soll (also effektiv gestreckt oder geschrumpft werden soll) und ob GameMaker ihn in einem Winkel zeichnen (wobei 0 normal ist und jedes Grad über 0 den Text gegen den Uhrzeigersinn dreht).

Syntax:

draw_text_transformed(x, y, string, xscale, yscale, angle);

ArgumentTyp Beschreibung
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
xscale The horizontal scale (default 1).
yscale The vertical scale(default 1).
angle The angle of the text.

Rückgabe:

Beispiel:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_transformed(room_width / 2, room_height / 2, "GAME OVER!", 2, 2, image_angle);

Das obige code zeichnet den angegebenen Text in der Mitte des Raumes, wobei er sich dreht und auf das Doppelte seiner ursprünglichen Größe skaliert wird.
