zeichnen_text

Mit dieser Funktion können Sie eine beliebige Zeichenkette an einer beliebigen Stelle im Raum zeichnen (um reelle Zahlen zu zeichnen, sollten Sie die Funktion string() Funktion verwenden, um sie in Text umzuwandeln). Um Zeichenketten zu kombinieren, können Sie + verwenden (siehe Beispiel unten), und Sie können auch \n innerhalb einer Zeichenkette verwenden, um einen Zeilenumbruch hinzuzufügen, so dass sie über mehrere Zeilen gezeichnet wird (Informationen über die richtige Formatierung einer Zeichenkette und welche Escape-Zeichen Sie verwenden können, finden Sie hier). Die Farbe des Textes und der Alphawert richten sich nach den aktuellen Basis-Alpha- und Farbwerten, die mit draw_set_alpha() und draw_set_colour().

HINWEIS: Die tatsächliche Position des Textes wird von den Ausrichtungswerten beeinflusst, die mit draw_set_halign() und draw_set_valign().

Syntax:

draw_text(x, y, string);

ArgumentTyp Beschreibung
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.

Rückgabe:

Beispiel:

draw_text(x, y, "Hello, " + global.Name + "!\nI hope you are well!");

Die obige code zeichnet eine Zeichenkette an der x/y-Position der Instanz, die die in der globalen Variable "Name" gespeicherte Zeichenkette verwendet und sie auf zwei Zeilen aufteilt.
