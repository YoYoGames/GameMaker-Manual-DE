zeichne_get_schrift

Diese Funktion ermittelt die aktuell zugewiesene Schriftart zum Zeichnen von Text. Die Funktion gibt -1 zurück, wenn keine Schriftart eingestellt ist, oder den ID-Wert (eine positive ganze Zahl) für die zugewiesene Schriftart.

Syntax:

draw_get_font();

Rückgabe:

Font Asset or -1

Beispiel:

var _cur_font = draw_get_font();
var _y_offset = 0;

switch (_cur_font)
{
    case ft_small:
        _y_offset = 10;
    break;

    case ft_medium:
        _y_offset = 22;
    break;

    case ft_big:
        _y_offset = 34;
    break;

    default:
        _y_offset = 8;
}

draw_text(room_width / 2, 200 + _y_offset, "MENU");

Die obige code ruft die aktuell verwendete Schriftart ab und führt eine switch -Anweisung darauf aus, wobei je nach Schriftart ein anderer Y-Versatzwert angewendet wird. Dieser Offset-Wert wird dann beim Zeichnen von Text verwendet.
