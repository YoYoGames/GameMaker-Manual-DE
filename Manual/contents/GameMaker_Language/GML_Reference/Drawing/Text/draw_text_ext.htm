zeichne_text_ext

Diese Funktion zeichnet Text auf ähnliche Weise wie draw_text() nur können Sie jetzt den Abstand zwischen den einzelnen Textzeilen festlegen - falls der Text mehr als eine Zeile einnimmt - und die Breite (in Pixeln) der Zeichenkette pro Zeile begrenzen, so dass der GameMaker den Text automatisch in die nächste Zeile an der nächsten freien Stelle aufteilt, wenn eine Zeile diesen Wert überschreitet (wenn der Text keine freien Stellen hat, überschreitet er diesen maximalen Breite überschritten). Beachten Sie, dass alle Leerzeichen am Anfang der Zeichenkette entfernt werden, bevor sie zum Zeichnen geparst werden. Beachten Sie auch, dass ein Wert von -1 für die Zeilentrennung argument standardmäßig auf eine Trennung basierend auf der Höhe des Zeichens "M" in der gewählten Schriftart basiert.

Syntax:

draw_text_ext(x, y, string, sep, w);

ArgumentTyp Beschreibung
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
sep The distance in pixels between lines of text.
w The maximum width in pixels of the string before a line break.

Rückgabe:

Beispiel:

draw_text_ext(100, 50, keyboard_string, 3, 300);

Die obige code zeichnet den Text, den der Benutzer über die Tastatur eingibt, und teilt ihn jedes Mal auf neue Zeilen auf, wenn die Länge der Zeichenkette für diese Zeile 300 Pixel überschreitet. code hält auch einen Abstand von 3 Pixeln zwischen den Zeilen ein, wenn dies der Fall ist.
