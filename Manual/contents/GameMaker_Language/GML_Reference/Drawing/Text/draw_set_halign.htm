draw_set_halign

Diese Funktion wird verwendet, um Text entlang der horizontalen Achse auszurichten. Eine Änderung der horizontalen Ausrichtung ändert die Position und die Richtung, in der alle weiteren Texte gezeichnet werden, wobei der Standardwert fa_left ist. Die folgenden Konstanten werden akzeptiert:

Konstante Ausrichtung
fa_left
fa_center
fa_right

Syntax:

draw_set_halign(halign);

Argument Typ Beschreibung
halign Horizontale Ausrichtung Konstante Horizontal alignment constant (from the table above).

Rückgabe:

N/A

Beispiel:

draw_set_halign(fa_left);
draw_text(100, 32, "Score: " + string(score));
draw_set_halign(fa_right);
draw_text(room_width - 100, 32, "Health: " + string(health));

Die obige code zeichnet zwei Strings auf der gleichen Linie, wobei die Partitur links und die Gesundheit rechts ausgerichtet ist.
