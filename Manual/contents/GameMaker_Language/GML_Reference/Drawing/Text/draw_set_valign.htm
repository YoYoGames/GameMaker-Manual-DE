draw_set_valign

Diese Funktion wird verwendet, um Text entlang der vertikalen Achse auszurichten. Eine Änderung der vertikalen Ausrichtung ändert die Position und die Richtung, in der alle weiteren Texte gezeichnet werden, wobei der Standardwert fa_top ist. Die folgenden Konstanten werden akzeptiert:

Konstante Ausrichtung
fa_top
fa_middle
fa_bottom

Syntax:

draw_set_valign(valign);

Argument Typ Beschreibung
valign Vertikale Ausrichtung Konstante Vertical alignment constant (from the table above).

Rückgabe:

N/A

Beispiel:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text(100, 32, "Score: " + string(score));

Mit der obigen code wird die Partitur genau in der Mitte des Textes gezeichnet.
