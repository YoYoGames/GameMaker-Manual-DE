draw_text_ext_transformed_colour

Diese Funktion ist eine Kombination aus der Basisfunktion draw_text() Funktion mit der draw_text_ext(), draw_text_transformed() und draw_text_colour() Funktionen. Als solche ist sie die allgemeinste aller Funktionen zum Zeichnen von Text und bietet ein hohes Maß an Flexibilität beim Zeichnen. Beachten Sie, dass die "Breite" argument auf einem Maßstab von 1 basiert. Wenn der Maßstab also anders ist, sollte dieser Wert proportional geändert werden. Wenn zum Beispiel die Basisbreite für einen Zeilenumbruch 300 beträgt und Sie den Maßstab auf 2 setzen, wird der Text falsch dargestellt, da er die angegebene Breite. Stattdessen sollten Sie die Breite auf 150 einstellen, um die Skalierung zu kompensieren.

HINWEIS: Das Überblenden von Farbverläufen ist für das HTML5-Ziel nicht verfügbar, es sei denn, WebGL ist aktiviert, obwohl Sie die Überblendungsfarben immer noch einstellen können und die Schrift mit der ersten angegebenen Farbe überblendet wird. Jede Überblendung auf diese Weise erzeugt jedoch ein Duplikat Schriftart, die dann in cache gespeichert und bei Bedarf verwendet wird. Das ist alles andere als optimal und verlangsamt die Leistung des Spiels, wenn Sie mehrere Farbwechsel verwenden. Sie können die Größe der Schrift cache einstellen, um dies einzuschränken, falls es notwendig sein sollte indem Sie die Funktion font_set_cache_size().

Syntax:

draw_text_ext_transformed_colour(x, y, string, sep, w, xscale, yscale, angle, c1, c2, c3, c4, alpha);

ArgumentTyp Beschreibung
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
sep The distance in pixels between lines of text.
w The maximum width in pixels of the string before a line break.
xscale The horizontal scale.
yscale The vertical scale.
angle The angle of the text.
c1 The colour for the top left of the drawn text.
c2 The colour for the top right of the drawn text.
c3 The colour for the bottom right of the drawn text.
c4 The colour for the bottom left of the drawn text.
alpha The alpha for the text.

Rückgabe:

Beispiel:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text_ext_transformed_colour(200, 200, "My name is" + global.name +".", 20, 200, 0.5, 0.5, 180, c_blue, c_blue, c_navy, c_navy, 1);

Damit wird der angegebene Text mit einem blauen bis dunkelblauen Farbverlauf nach unten gezeichnet, an der Position (200,200) im Raum, auf den Kopf gedreht und mit der Hälfte der normalen Punktgröße der Schrift. Der Abstand zwischen den einzelnen Zeilen beträgt 20 Pixel, falls mehr als eine Zeile eine Linie gezeichnet werden und jede Linie hat eine maximale Breite von 200 Pixeln.
