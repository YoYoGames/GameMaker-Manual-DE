draw_text_colour

Diese Funktion zeichnet Text auf ähnliche Weise wie draw_text() nur dass Sie jetzt die Farben, die Sie zum Einfärben des Textes verwenden wollen, sowie den Alphawert auswählen können, und diese neuen Werte werden anstelle von anstelle der Grundfarbe und des Alphawertes verwendet.

HINWEIS: Das Überblenden von Farbverläufen ist für das HTML5-Ziel nicht verfügbar, es sei denn, WebGL ist aktiviert, obwohl Sie die Überblendungsfarben immer noch einstellen können und die Schrift mit der ersten angegebenen Farbe überblendet wird. Jede Überblendung auf diese Weise erzeugt jedoch ein Duplikat Schriftart, die dann in cache gespeichert und bei Bedarf verwendet wird. Das ist alles andere als optimal und verlangsamt die Leistung des Spiels, wenn Sie mehrere Farbwechsel verwenden. Sie können die Größe der Schrift cache einstellen, um dies einzuschränken, falls es notwendig sein sollte indem Sie die Funktion font_set_cache_size().

Syntax:

draw_text_colour(x, y, string, c1, c2, c3, c4, alpha);

ArgumentTyp Beschreibung
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
c1 The colour for the top left of the drawn text.
c2 The colour for the top right of the drawn text.
c3 The colour for the bottom right of the drawn text.
c4 The colour for the bottom left of the drawn text.
alpha The alpha for the text.

Rückgabe:

Beispiel:

draw_set_colour(c_white);
draw_text(100, 100, "Health");
draw_text_colour(100, 200, string(health), c_lime, c_lime, c_green, c_green, 1);

Die obige code zeichnet zwei Textabschnitte auf derselben Linie, wobei der erste Text weiß gezeichnet wird (da dies die Grundfarbe ist) und der zweite Text mit einem lindgrünen bis normalgrünen Farbverlauf gezeichnet wird.
