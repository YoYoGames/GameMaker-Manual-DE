zeichen_setzen_schrift

Diese Funktion legt die Schriftart fest, die für alle weiteren Textzeichnungen verwendet werden soll. Diese Schriftart muss zu den Schriftart-Assets des Spiels hinzugefügt worden sein oder entweder mit der font_add() oder font_add_sprite().

Syntax:

draw_set_font(font);

ArgumentTyp Beschreibung
font The name of the font to use.

Rückgabe:

Beispiel:

draw_set_colour(c_blue);
draw_set_font(fnt_Game);
draw_text(200, 200, "Hello World");

Die obige code zeichnet den angegebenen Text mit der in der Variablen "fnt_Game" indizierten Schriftart in blauer Farbe.
