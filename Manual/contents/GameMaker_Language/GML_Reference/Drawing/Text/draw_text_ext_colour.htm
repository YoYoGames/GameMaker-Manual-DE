draw_text_ext_colour

Diese Funktion ist eine Kombination aus der Basisfunktion draw_text() Funktion mit den Funktionen draw_text_ext() und draw_text_colour() und ermöglicht es Ihnen, Farbverläufe für Text sowie den Zeilenabstand und die maximale Breite pro Zeile zu definieren.

HINWEIS Das Überblenden von Farbverläufen ist für das HTML5-Ziel nicht verfügbar, es sei denn, WebGL ist aktiviert, obwohl Sie immer noch die Überblendfarben einstellen können und die Schrift mit der ersten angegebenen Farbe überblendet wird. Allerdings wird beim Überblenden auf diese Weise eine doppelte Schriftart erstellt, die dann auf cache gespeichert und bei Bedarf verwendet wird, was alles andere als optimal ist. Sie können die Größe der Schrift cache einstellen, um dies einzuschränken, falls es notwendig sein sollte, indem Sie die Funktion font_set_cache_size().

Syntax:

draw_text_ext_colour(x, y, string, sep, w, c1, c2, c3, c4, alpha);

Argument Typ Beschreibung
x Real The x coordinate of the drawn string.
y Real The y coordinate of the drawn string.
string Zeichenfolge The string to draw.
sep Real The distance in pixels between lines of text.
w Real The maximum width in pixels of the string before a line break.
c1 Farbe The colour for the top left of the drawn text.
c2 Farbe The colour for the top right of the drawn text.
c3 Farbe The colour for the bottom right of the drawn text.
c4 Farbe The colour for the bottom left of the drawn text.
alpha Real The alpha for the text.

Rückgabe:

N/A

Beispiel:

draw_text_ext_colour(200, 200, keyboard_string, 5, 300, c_blue, c_blue, c_navy, c_navy, 1);

Das obige code zeichnet den Text, den der Benutzer über die Tastatur eingibt, und teilt ihn jedes Mal auf neue Zeilen auf, wenn die Länge der Zeichenkette für diese Zeile 300 Pixel überschreitet. code hält auch einen Abstand von 5 Pixeln zwischen den Zeilen ein, wenn dies der Fall ist. Jede Zeile des Textes wird mit einem blauen Farbverlauf eingefärbt, wobei der obere Teil hellblau und der untere Teil dunkelblau ist.
