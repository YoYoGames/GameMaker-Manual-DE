draw_text_transformed_colour

Diese Funktion ist eine Kombination aus der Basisfunktion draw_text() Funktion mit den Funktionen draw_text_transformed() und draw_text_colour() und ermöglicht es Ihnen, Text zu skalieren und zu drehen sowie ihn mit einem Farbverlauf zu füllen und seinen Alphawert zu ändern, wobei die Basis-Alpha- und Farbeinstellungen für das Zeichnen ignoriert werden.

HINWEIS Das Überblenden von Farbverläufen ist für das HTML5-Ziel nicht verfügbar, es sei denn, WebGL ist aktiviert, obwohl Sie die Überblendungsfarben immer noch einstellen können und die Schrift mit der ersten angegebenen Farbe überblendet wird. Alle Überblendungen auf diese Weise erzeugen jedoch ein Duplikat Schriftart, die dann in cache gespeichert und bei Bedarf verwendet wird. Das ist alles andere als optimal und verlangsamt die Leistung des Spiels, wenn Sie mehrere Farbwechsel verwenden. Sie können die Größe der Schrift cache einstellen, um dies einzuschränken, falls es notwendig sein sollte indem Sie die Funktion font_set_cache_size().

Syntax:

draw_text_transformed_colour(x, y, string, xscale, yscale, angle, c1, c2, c3, c4, alpha);

ArgumentTyp Beschreibung
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
xscale The horizontal scale.
yscale The vertical scale.
angle The angle of the text.
c1 The colour for the top left of the drawn text.
c2 The colour for the top right of the drawn text.
c3 The colour for the bottom right of the drawn text.
c4 The colour for the bottom left of the drawn text.
alpha The alpha for the text.

Rückgabe:

Beispiel:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_transformed_colour(room_width / 2, room_height / 2, keyboard_string, 2, 2, image_angle, c_red, c_red, c_yellow, c_yellow, 0.5);

Die obige code zeichnet den angegebenen Text in der Mitte des Raumes, kreisend und auf die doppelte Größe skaliert, mit einem Farbverlauf von Gelb nach Rot sowie einem Alpha-Wert von 0,5.
