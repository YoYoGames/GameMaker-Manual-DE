video_get_position

Diese Funktion gibt die aktuelle Position des geladenen Videos in Millisekunden zurück. Dies zeigt an, wie viel von der Gesamtdauer des Videos bereits abgespielt wurde.

Syntax:

video_get_position();

Rückgabe:

Real

Beispiel:

if (keyboard_check_pressed(vk_right))
{
    var _video_position = video_get_position();
    _video_position += 2000;

    if (_video_position < video_get_duration()) 
    {
        video_seek_to(_video_position);
    }
    else
    {
        video_close();
    }
}

Die obige code verschiebt das abspielende Video um 2 Sekunden nach vorne, wenn die rechte Pfeiltaste gedrückt wird und die neue Position kleiner ist als die Dauer des Videos. Wenn sie jedoch nicht innerhalb der Dauer liegt, wird das Video geschlossen.

In diesem Beispiel wird video_get_position() verwendet, um die Position des abgespielten Videos zu ermitteln.
