video_pause

Mit dieser Funktion wird die aktuell geladene Videodatei angehalten. Sie können sie jederzeit nach dieser Funktion durch den Aufruf von video_resume() fortsetzen.

Syntax:

video_pause();

Rückgabe:

N/A

Beispiel:

var _status = video_get_status();

if (keyboard_check_pressed(vk_space))
{
    if (_status == video_status_playing)
    {
        video_pause();
    }
    else if (status == video_status_paused)
    {
        video_resume();
    }
}

Die obige code ruft den Status des Videos ab und prüft dann, ob der Spieler die Leertaste gedrückt hat. In diesem Fall wird das Video angehalten, wenn es abgespielt wird, und fortgesetzt, wenn es angehalten wurde.
