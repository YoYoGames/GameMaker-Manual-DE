video_enable_loop

Diese Funktion aktiviert oder deaktiviert die Schleifenbildung für das aktuell geladene Video. Setzen Sie den Parameter enable argument auf true, um die Schleife zu aktivieren, und false, um sie zu deaktivieren.

Standardmäßig ist die Schleifenfunktion deaktiviert.

Syntax:

video_enable_loop(enable);

Argument Typ Beschreibung
enable Boolesche Whether to enable (true) oder deaktivieren(false) Schleifenbildung

Rückgabe:

N/A

Beispiel:

video_open("Loading.mp4");
video_enable_loop(true);

Die obige code lädt ein Video und aktiviert die Schleifenfunktion.
