video_get_status

Diese Funktion gibt den Status des aktuell geladenen Videos zurück. Der Status kann eine der folgenden Konstanten sein:

Videostatus Konstant
Konstante Beschreibung
video_status_closed Es ist kein Video geladen, oder das Video wurde geschlossen mit video_close()
video_status_preparing Das Video wird gerade vorbereitet und ist noch nicht abgespielt worden.
video_status_playing Das Video wird gerade abgespielt
video_status_paused Das Video wird pausiert (siehe video_pause())

Syntax:

video_get_status();

Rückgabe:

Video Status Constant

Beispiel:

var _status = video_get_status();

if (keyboard_check_pressed(vk_space))
{
    if (_status == video_status_playing)
    {
        video_pause();
    }
    else if (status == video_status_paused)
    {
        video_resume();
    }
}

Die obige code ruft den Status des Videos ab und prüft dann, ob der Spieler die Leertaste gedrückt hat. In diesem Fall wird das Video angehalten, wenn es abgespielt wird, und fortgesetzt, wenn es angehalten wurde.
