video_is_looping

Diese Funktion gibt zurück, ob das aktuell geladene Video auf loop eingestellt ist(true) oder nicht(false). Sie können ein Video auf loop setzen, indem Sie video_enable_loop() aufrufen.

Syntax:

video_is_looping();

Rückgabe:

Boolean

Beispiel:

if (video_is_looping()) {
    obj_player.control_enabled = false;
}

Die obige code prüft, ob das aktuell geladene Video in einer Schleife läuft, und deaktiviert in diesem Fall die Steuerelemente des Players, indem eine benutzerdefinierte Instanzvariable auf false gesetzt wird.
