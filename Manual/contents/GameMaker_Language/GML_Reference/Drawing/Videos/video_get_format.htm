video_get_format

Diese Funktion gibt das Farbformat des aktuell geladenen Videos zurück. Dies kann eine der folgenden Konstanten sein:

Videoformat Konstant
Konstante Beschreibung
video_format_rgba Die Videooberfläche verwendet das RGBA-Farbmodell
video_format_yuv Die Videooberfläche verwendet das YUV-Farbmodell

Windows, macOS, Opera GX, Android, iOS, und HTML5 verwenden das RGBA-Modell für Videos. Alle anderen platforms verwenden das YUV-Modell.

Da die Zeichenmethoden für RGBA- und YUV-Videos unterschiedlich sind, kann der Rückgabewert dieser Funktion verwendet werden, um je nach Format des abgespielten Videos unterschiedliche code auszuführen. Siehe Draw Video für ein Beispiel.

Syntax:

video_get_format();

Rückgabe:

Video Format Constant
