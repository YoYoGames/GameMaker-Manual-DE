video_öffnen

Mit dieser Funktion wird eine Videodatei unter dem angegebenen Pfad geöffnet. Der Pfad kann sich auf eine Datei in den enthaltenen Dateien Ihres Projekts, eine lokal auf dem Gerät des Benutzers vorhandene Datei oder eine URL zum Laden einer Datei aus dem Internet beziehen (was möglicherweise nicht auf allen platforms funktioniert).

Da immer nur ein Video gleichzeitig geladen werden kann, lädt diese Funktion das angegebene Video in den einzigen verfügbaren "Video-Slot", der dann mit den anderen Video-Funktionen bearbeitet wird.

Es gibt asynchrone Callbacks, die mit der Videowiedergabe verbunden sind. Weitere Informationen finden Sie unter Async-Aufrufe.

Syntax:

video_open(path);

Argument Typ Beschreibung
path Zeichenfolge Path to the video file

Rückgabe:

N/A

Beispiel:

my_video = video_open("splash.mp4");

Die obige code lädt splash.mp4 aus den mitgelieferten Dateien des Spiels.
