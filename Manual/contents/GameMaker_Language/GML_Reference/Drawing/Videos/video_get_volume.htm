video_get_volume

Diese Funktion gibt die Lautstärke des aktuell geladenen Videos zurück, also einen Wert zwischen 0 und 1. Sie können die Lautstärke des geladenen Videos mit video_set_volume() ändern.

Syntax:

video_get_volume();

Rückgabe:

Real

Beispiel:

/// Step event
var _video_volume = video_get_volume();

if (_video_volume < 1) {
    _video_volume += 0.02;
    video_set_volume(_video_volume);
}

Die obige code würde in einem Step-Ereignis ausgeführt. Es ermittelt die Lautstärke für das aktuell abgespielte Video, und wenn sie niedriger als 1 ist, wird sie um 0,02 erhöht, wobei die Lautstärke schrittweise bis zum Maximalwert erhöht wird.
