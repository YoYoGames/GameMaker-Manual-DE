video_schließen

Diese Funktion schließt die aktuell geladene Videodatei. Stellen Sie sicher, dass diese Funktion nur nach einem Aufruf von video_open() aufgerufen wird, da sie sonst keine Wirkung zeigt.

Syntax:

video_close();

Rückgabe:

N/A

Beispiel:

if (keyboard_check_pressed(vk_escape))
{
    video_close();
}

Die obige code schließt das Video, wenn die Escape-Taste gedrückt wird.
