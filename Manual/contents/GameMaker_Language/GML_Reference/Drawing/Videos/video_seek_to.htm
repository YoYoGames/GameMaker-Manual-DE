video_seek_to

Mit dieser Funktion können Sie die angegebene Position im aktuell geladenen Video ansteuern. Sie geben den Zeitwert für die Suche an, der in Millisekunden angegeben wird.

Diese Funktion funktioniert am besten, wenn das Video angehalten wird und der Abstand zwischen der aktuellen Position und der neuen Suchposition nicht zu klein ist.

Syntax:

video_seek_to(time);

Argument Typ Beschreibung
time Real The time to seek to (in milliseconds).

Rückgabe:

N/A

Beispiel:

if (keyboard_check_pressed(vk_right))
{
    var _video_position = video_get_position();
    _video_position += 2000;

    if (_video_position < video_get_duration()) 
    {
        video_seek_to(_video_position);
    }
    else
    {
        video_close();
    }
}

Die obige code verschiebt das abspielende Video um 2 Sekunden nach vorne, wenn die rechte Pfeiltaste gedrückt wird und die neue Position kleiner ist als die Dauer des Videos. Wenn sie jedoch nicht innerhalb der Dauer liegt, wird das Video geschlossen.

In diesem Beispiel wird video_seek_to() verwendet, um das Video an der neuen Position zu suchen.
