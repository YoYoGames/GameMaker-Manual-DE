video_set_volume

Diese Funktion ändert die Lautstärke des aktuell geladenen Videos. Sie nimmt einen Wert zwischen 0 und 1 an, wobei 0 stumm ist und 1 die maximale Lautstärke darstellt.

Stellen Sie sicher, dass diese Funktion nur nach einem Aufruf von video_open(), aber vor einem Aufruf von video_close() aufgerufen wird, da sie sonst nichts bewirkt (da kein Video geladen wird).

Auf einigen platforms funktioniert diese Funktion erst, nachdem das Video vollständig geladen wurde. Verwenden Sie daher Async-Callbacks, um herauszufinden, wann das Video fertig ist, und ändern Sie dann die Lautstärke.

Syntax:

video_set_volume(value);

Argument Typ Beschreibung
value Real Volume value between 0 and 1

Rückgabe:

N/A

Beispiel:

my_video = video_open("splash.mp4");
video_set_volume(0.5);

Die obige code lädt splash.mp4 aus den mitgelieferten Dateien des Spiels und ändert die Lautstärke auf 0,5 (50% der maximalen Lautstärke).
