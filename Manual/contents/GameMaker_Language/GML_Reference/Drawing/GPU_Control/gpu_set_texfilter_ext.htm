gpu_set_texfilter_ext

Diese Funktion kann verwendet werden, um die lineare Interpolation für einen einzelnen Sampler-"Slot" bei der Verwendung von Shadern in GameMaker einzustellen. Wenn diese Funktion aktiviert ist (true), wird die Sampler-Textur geglättet, und wenn sie deaktiviert ist (false), werden die Bilder auf der Grundlage des nächstgelegenen Pixels gezeichnet. Der Standardwert ist derjenige, der in den globalen Spieloptionen für Ihr Spiel eingestellt ist, oder derjenige, der mit der Funktion gpu_set_texfilter().

HINWEIS: Diese Einstellung wird durch den Wert überschrieben, der beim Aufruf der Funktion gpu_set_texfilter().

HINWEIS: Auf dem HTML5-Ziel funktioniert diese Funktion nur, wenn WebGL aktiviert ist.

Syntax:

gpu_set_texfilter_ext(sampler_id, enable);

ArgumentTyp Beschreibung
sampler_id The sampler id from the shader.
enable Enable or disable texture filtering (true / false)

Rückgabe:

Beispiel:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if gpu_get_texfilter_ext(s_tex)
{
    gpu_set_texfilter_ext(s_tex, false);
}
else
{
    gpu_set_texfilter_ext(s_tex, true);
}

Die obige code prüft, ob die Texturfilterung für eine bestimmte Sampler-ID (die in einer lokalen Variable gespeichert ist) ein- oder ausgeschaltet ist und schaltet sie entsprechend um.
