gpu_get_state

Diese Funktion ermittelt den aktuellen Zustand der GPU und gibt ihn als DS-Map zurück. Diese kann dann manipuliert oder sogar gespeichert werden, und Sie können diese Map mit der Funktion "DS Map" an die GPU zurückgeben. gpu_set_state().

Syntax:

gpu_get_state();

Rückgabe:

DS Map ID

Beispiel:

gpu_map = gpu_get_state();

Die obige code speichert den aktuellen Zustand der GPU in einer Variablen.
