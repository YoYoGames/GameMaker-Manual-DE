gpu_get_fog

Diese Funktion kann verwendet werden, um die Nebeleinstellungen abzurufen. Die Funktion gibt ein 4-Element-1D-Array mit den folgenden Informationen zurück:

  • [0] = aktivierter Toggle (ein Boolescher Wert, entweder true oder false), Standard false
  • [1] = Farbkonstante, Standard c_black
  • [2] = Startabstand(Real), Standardwert 0
  • [3] = Endabstand(Real), Voreinstellung 1

Beachten Sie, dass Sie diese Werte ändern können und das gesamte Array an die gpu_set_fog() Funktion als ein einziges argument übergeben.

Syntax:

gpu_get_fog();

Rückgabe:

Array (4 elements only; see above for details)

Beispiel:

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

Die obige code ruft die aktuellen Nebeleinstellungen ab und setzt dann das Farbelement des Arrays auf c_red, bevor der Nebel mit dem geänderten Array erneut eingestellt wird.
