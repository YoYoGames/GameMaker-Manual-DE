gpu_set_alphatestenable

Mit dieser Funktion können Sie den Alphatest für Ihr Spiel aktivieren oder deaktivieren (standardmäßig ist er deaktiviert). Wenn Sie den Alphatest einschalten, können Sie mit der Begleitfunktion gpu_set_alphatestref() den "Cut-Off"-Wert einstellen, bei dem alle Alphawerte auf 0 gesetzt werden. Das Bild unten zeigt den Unterschied, der zu sehen ist, wenn der Alphatest ein- oder ausgeschaltet ist:

HINWEIS: Diese Funktion kann die Leistung auf iOS und Android-Geräten beeinträchtigen.

Syntax:

gpu_set_alphatestenable(enable);

ArgumentTyp Beschreibung
enable Enable or disable alpha testing (true / false)

Rückgabe:

Beispiel:

if !gpu_get_alphatestenable()
{
    gpu_set_alphatestenable(true);
    gpu_set_alphatestref(128);
}

Die obige code prüft, ob der Alphatest aktiviert ist, und wenn nicht, wird switch auf Alphatest und den Testschwellenwert auf 128 gesetzt (nur Pixel mit einem Alphawert über 0,5 werden gezeichnet).
