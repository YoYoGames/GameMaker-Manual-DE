gpu_get_alphatestenable

Mit dieser Funktion können Sie prüfen, ob der Alphatest aktiviert ist (gibt true zurück) oder nicht (gibt false zurück). Weitere Informationen über Alpha-Tests finden Sie in der Funktion gpu_set_alphatestref().

Syntax:

gpu_get_alphatestenable();

Rückgabe:

Beispiel:

if !gpu_get_alphatestenable()
{
    gpu_set_alphatestenable(true);
    gpu_set_alphatestref(128);
}

Die obige code prüft, ob der Alphatest aktiviert ist, und wenn nicht, wird switch auf Alphatest und den Testschwellenwert auf 128 gesetzt (nur Pixel mit einem Alphawert über 0,5 werden gezeichnet).
