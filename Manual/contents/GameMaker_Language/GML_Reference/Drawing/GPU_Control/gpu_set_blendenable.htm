gpu_set_blendenable

Mit dieser Funktion können Sie das Alpha-Blending ein- und ausschalten. Wenn Sie diese Funktion auf false einstellen, werden alle Bilder, die gezeichnet werden, zu 100 % undurchsichtig gezeichnet, was bedeutet, dass alle transparenten oder halbtransparenten Bereiche eines sprite oder Hintergrunds als Volltonfarbe sichtbar sind. Es ist eine gute Idee, Alpha Blending auszuschalten, wann immer dies möglich ist (insbesondere bei der Entwicklung für mobile Geräte), da dies die Zeichengeschwindigkeit erheblich erhöht.

Syntax:

gpu_set_blendenable(enable);

ArgumentTyp Beschreibung
enable Enable or disable alpha blending value (true oder false).

Rückgabe:

Beispiel:

gpu_set_blendenable(false);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_blendenable(true);

Die obige code schaltet das Alpha-Blending aus, um einen Hintergrund sprite zu zeichnen, und schaltet es dann wieder ein, um weiter zu zeichnen.
