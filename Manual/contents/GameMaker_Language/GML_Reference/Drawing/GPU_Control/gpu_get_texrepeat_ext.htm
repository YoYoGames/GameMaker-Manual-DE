gpu_get_texrepeat_ext

Mit dieser Funktion können Sie prüfen, ob die Texturwiederholung für eine bestimmte Shader-Sampler-Textur aktiviert ist (gibt true zurück) oder nicht (gibt false zurück).

Syntax:

gpu_get_texrepeat_ext(sampler_id);

ArgumentTyp Beschreibung
sampler_id The sampler id from the shader.

Rückgabe:

Beispiel:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if (!gpu_get_texrepeat_ext(s_tex))
{
    gpu_set_texrepeat_ext(s_tex, true);
}

Die obige code prüft, ob die Texturfilterung für eine bestimmte Sampler-ID (die in einer lokalen Variable gespeichert ist) ausgeschaltet ist und schaltet sie ein, wenn dies nicht der Fall ist.
