gpu_get_blendmode_ext

Diese Funktion kann verwendet werden, um den aktuellen erweiterten Überblendungsmodus abzurufen, der zum Zeichnen verwendet wird. Die Funktion gibt ein 1D-Array mit 2 Elementen zurück, das die folgenden Elemente enthält:

  • [0] = Quelle Blend Mode Factor Constant (Standard ist bm_src_alpha)
  • [1] = Ziel Überblendungsmodus-Faktor Konstante (Standard ist bm_inv_src_alpha)

Die Werte des Arrays sind eine der folgenden Konstanten ("s" steht für einen Wert aus der Quelle, "d" für einen Wert aus dem Ziel):

Blend Mode Faktor Konstante
Konstante Überblendfaktor (Rot, Grün, Blau, Alpha)
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-As)
bm_src_alpha (As, As, As, As)
bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As)
bm_dest_alpha (Anzeige, Anzeige, Anzeige, Anzeige)
bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) mit f = min(As, 1-Ad)

Beachten Sie, dass Sie diese Werte ändern können und das gesamte Array an die gpu_set_blendmode_ext() Funktion als ein einziges argument übergeben.

Syntax:

gpu_get_blendmode_ext();

Rückgabe:

Array (2 elements only; see above for constants)

Beispiel:

var bm = gpu_get_blendmode_ext();
bm[0] = bm_src_alpha;
gpu_set_blendmode_ext(bm);

Die obige code ruft den aktuellen erweiterten Überblendungsmodus ab, ändert die Quelle und setzt dann den erweiterten Überblendungsmodus erneut.
