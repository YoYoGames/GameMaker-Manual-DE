gpu_set_texrepeat

Mit dieser Funktion kann man GameMaker mitteilen, ob sich Texturen bei Modellen und Primitiven wiederholen sollen (true) oder nicht (false). Normalerweise liegen Texturbreiten und -höhen im Bereich 0 - 1, aber wenn Sie einen Wert größer als 1 angeben, wird der Rest standardmäßig nicht gezeichnet (die gesamte Textur wird mit einem "leeren" Abschnitt gezeichnet, der dem verwendeten Prozentsatz über 1 entspricht. So würde ein Wert von 2 für eine Texturbreite oder -höhe die Textur in der Hälfte des Raums zeichnen und die andere Hälfte leer lassen). Wenn Sie jedoch diese Funktion verwenden, um repeat auf true zu setzen, wird die Textur automatisch über alle Modelle und Primitive hinweg wiederholt. Der wahrscheinliche Anwendungsfall für diese Funktionen ist die Wiederholung einer Textur in 3D, aber damit es funktioniert und keine Bilder vom Rest der Texturseite abzieht, muss die verwendete sprite im Sprite Editor als "Separate Texturseite" markiert werden.

HINWEIS: Sprites und Hintergründe werden immer ohne Wiederholung gezeichnet. Sobald Sie also einen sprite oder einen Hintergrund zeichnen, wird dieser Wert auf false zurückgesetzt. Dieses Standardverhalten hat keine Auswirkungen auf die Einstellungen der Texturstufe (d.h. der Aufruf der Funktion hat Auswirkungen, aber die interne Einstellung auf false nicht).

HINWEIS: Diese Einstellung hat Vorrang vor allen Texturstufenwiederholungen, die für Shader festgelegt wurden, wenn Sie die Funktion gpu_set_texrepeat_ext().

Syntax:

gpu_set_texrepeat(enable);

ArgumentTyp Beschreibung
enable Enable or disable texture filtering (true / false)

Rückgabe:

Beispiel:

if gpu_get_texrepeat()
{
    gpu_set_texrepeat(false);
}
else
{
    gpu_set_texrepeat(true);
}

Die obige code prüft, ob die Texturwiederholung ein- oder ausgeschaltet ist und schaltet sie entsprechend.
