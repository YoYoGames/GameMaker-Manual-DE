gpu_get_texfilter_ext

Mit dieser Funktion können Sie prüfen, ob die Texturinterpolation (lineare Interpolation) für eine bestimmte Shader-Sampler-Textur aktiviert ist (gibt true zurück) oder nicht (gibt false zurück).

HINWEIS: Auf dem HTML5-Ziel funktioniert diese Funktion nur, wenn WebGL aktiviert ist.

Syntax:

gpu_get_texfilter_ext(sampler_id);

Argument Typ Beschreibung
sampler_id Shader Sampler Handle The sampler id from the shader.

Rückgabe:

Boolean

Beispiel:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if (gpu_get_texfilter_ext(s_tex))
{
    gpu_set_texfilter_ext(s_tex, false);
}
else
{
    gpu_set_texfilter_ext(s_tex, true);
}

Die obige code prüft, ob die Texturfilterung für eine bestimmte Sampler-ID (die in einer lokalen Variable gespeichert ist) ein- oder ausgeschaltet ist und schaltet sie entsprechend um.
