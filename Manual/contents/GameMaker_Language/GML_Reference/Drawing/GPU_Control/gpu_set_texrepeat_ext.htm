gpu_set_texrepeat_ext

Diese Funktion kann verwendet werden, um festzulegen, ob ein einzelner Sampler-"Slot" die angegebene Textur wiederholt, wenn Shader in GameMaker verwendet werden. Die Einstellung true wiederholt die Textur, wenn die UV-Koordinaten außerhalb des Bereichs 0-1 liegen, während die Einstellung false keine Wiederholung bedeutet. Der wahrscheinliche Anwendungsfall für diese Funktionen ist die Wiederholung einer Textur in 3D, aber damit es funktioniert und nicht Bilder vom Rest der Texturseite zieht, muss die verwendete sprite im Sprite Editor als "Separate Texturseite" markiert werden.

HINWEIS Diese Einstellung wird durch den Wert überschrieben, der beim Aufruf der Funktion gpu_set_texrepeat().

Syntax:

gpu_set_texrepeat_ext(sampler_id, enable);

ArgumentTyp Beschreibung
sampler_id The sampler id from the shader.
enable Enable or disable texture filtering (true / false)

Rückgabe:

Beispiel:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if (gpu_get_texrepeat_ext(s_tex))
{
    gpu_set_texrepeat_ext(s_tex, false);
}
else
{
    gpu_set_texrepeat_ext(s_tex, true);
}

Die obige code prüft, ob die Texturfilterung für eine bestimmte Sampler-ID (die in einer lokalen Variable gespeichert ist) ein- oder ausgeschaltet ist und schaltet sie entsprechend um.
