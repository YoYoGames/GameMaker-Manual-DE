gpu_set_cullmode

Diese Funktion kann verwendet werden, um den Modus für die Rückseitenauslese einzustellen. Ein Polygon (Primitiv) hat eine Vorder- und eine Rückseite. Die Vorderseite ist die Seite, auf der die Scheitelpunkte gegen den Uhrzeigersinn definiert sind. Normalerweise werden beide Seiten gezeichnet, aber wenn Sie eine geschlossene Form (wie eine Pyramide) erstellen, ist dies eine Verschwendung von Rechenleistung, da die Rückseite des Dreiecks nie zu sehen ist. In diesen Fällen können Sie switch auf Backface Culling zurückgreifen, da dies etwa die Hälfte der Zeichnungszeit einspart, obwohl es bedeutet, dass Sie Ihre Polygone auf die richtige Weise definieren müssen, um Probleme zu vermeiden.

Der verwendete Cull-Modus-Wert muss eine der folgenden Konstanten sein (der Standardwert ist cull_noculling):

Konstante Beschreibung
cull_noculling Es wird keine Keulung vorgenommen
cull_clockwise Alle Dreiecke im Uhrzeigersinn werden herausgefiltert
cull_counterclockwise Alle Dreiecke, die gegen den Uhrzeigersinn liegen, werden aussortiert.

Syntax:

gpu_set_cullmode(cullmode);

Argument Typ Beschreibung
cullmode Tötungsmodus Konstant The cull mode to use (see list above)

Rückgabe:

N/A

Beispiel:

if gpu_get_cullmode() != cull_clockwise
{
    gpu_set_cullmode(cull_clockwise);
}

Die obige code ruft den aktuellen Cull-Modus ab, und wenn er nicht cull_clockwise ist, wird er auf diese Konstante gesetzt.
