gpu_get_blendmode

Diese Funktion kann verwendet werden, um den aktuellen Überblendungsmodus abzurufen, der zum Zeichnen verwendet wird. Der zurückgegebene Wert ist eine der folgenden Konstanten (der Standardwert ist bm_normal):

Blend Mode Constant
Konstante Beschreibung Erweiterter Blend-Modus
bm_normal Normales Überblenden (der Standardüberblendungsmodus). (bm_src_alpha, bm_inv_src_alpha)
bm_add Additives Überblenden. Die Helligkeitswerte der hellen Bereiche werden addiert. (bm_src_alpha, bm_one)
bm_subtract Subtraktives Überblenden. Die Helligkeitswerte der hellen Bereiche werden subtrahiert. (bm_zero, bm_inv_src_colour)
bm_max Max Blending. Ähnlich wie beim additiven Blending. (bm_src_alpha, bm_inv_src_colour)

Syntax:

gpu_get_blendmode();

Rückgabe:

Blend Mode Constant (see above for constants)

Beispiel:

if gpu_get_blendmode() != bm_normal
{
    gpu_set_blendmode(bm_normal);
}

Die obige code ruft den aktuellen Mischmodus ab, und wenn dieser nicht bm_normal ist, wird er auf diese Konstante gesetzt.
