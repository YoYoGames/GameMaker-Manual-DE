gpu_get_blendenable

Diese Funktion kann verwendet werden, um den Alpha-Blending-Status abzufragen. Wenn sie true zurückgibt, ist Alpha-Blending aktiviert, und wenn sie false zurückgibt, ist es deaktiviert. Standardmäßig ist dies eingeschaltet und die Funktion gibt daher zurück true

Syntax:

gpu_get_blendenable();

Rückgabe:

Beispiel:

if gpu_get_blendenable() == false
{
    gpu_set_blendenable(true);
}

Die obige code prüft den Status der Alpha-Überblendung und wenn sie deaktiviert ist, wird sie wieder aktiviert.
