gpu_get_blendmode_destalpha

Diese Funktion kann verwendet werden, um den aktuellen Alphafaktor des erweiterten Überblendungsmodus des Ziels abzurufen. Der zurückgegebene Wert ist eine der folgenden Konstanten ("s" steht für einen Wert aus der Quelle, "d" für einen Wert aus dem Ziel), wobei beim Zeichnen nur die Komponente "A" verwendet wird:

Konstante Überblendfaktor (Rot, Grün, Blau, Alpha)
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-As)
bm_src_alpha (As, As, As, As)
bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As)
bm_dest_alpha (Anzeige, Anzeige, Anzeige, Anzeige)
bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) mit f = min(As, 1-Ad)

Syntax:

gpu_get_blendmode_destalpha();

Rückgabe:

Blend Mode Factor Constant (see above table)

Beispiel:

var bm;
bm[0] = gpu_get_blendmode_srcalpha();
bm[1] = gpu_get_blendmode_destalpha();
gpu_set_blendmode_ext_sepalpha(bm_inv_src_alpha, bm_inv_dest_colour, bm[0], bm[1]);

Die obige code erstellt ein lokales Array und ruft die aktuellen Quell- und Zielüberblendfaktoren für die Alphakomponente ab. Dieses Array wird dann verwendet, um die RGB-Komponente der Überblendfaktoren zu bearbeiten.
