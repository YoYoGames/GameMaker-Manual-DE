gpu_set_state

Diese Funktion setzt den aktuellen GPU-Status mit Hilfe der übergebenen DS Map. Die übergebene Map kann mit der Funktion gpu_get_state().

Syntax:

gpu_set_state(ds_map);

ArgumentTyp Beschreibung
ds_map The GPU state to set as a DS Map.

Rückgabe:

Beispiel:

gpu_set_state(gpu_map);

Die obige code setzt den GPU-Status unter Verwendung der in der Variablen "gpu_map" angegebenen Karte.
