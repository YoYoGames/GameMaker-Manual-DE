gpu_set_blendmode

Wenn GameMaker ein Pixel zeichnet, gibt es eine Quellfarbe (die Farbe des Pixels, das wir zeichnen wollen) und eine Zielfarbe (die Farbe, die bereits in dem Pixel enthalten ist, zu dem wir zeichnen), so dass bei der Bestimmung der neuen Farbe des Pixels die Quell- und Zielfarben entsprechend dem gewählten Mischmodus berechnet werden. Jede Komponente der Farben wird als Fließkommawert zwischen 0 und 1 gespeichert, und die neue Farbe wird berechnet, indem jede Komponente der Ausgangsfarbe mit einem Faktor und jede Komponente der Zielfarbe mit einem anderen Faktor multipliziert wird und die Ergebnisse dann Komponente für Komponente addiert werden.

Mit dieser Funktion können Sie den Überblendungsmodus auf vier grundlegende Arten einstellen, indem Sie die folgenden Konstanten verwenden:

Konstante Beschreibung Erweiterter Blend-Modus
bm_normal Normales Überblenden (der Standardüberblendungsmodus). (bm_src_alpha, bm_inv_src_alpha)
bm_add Additives Überblenden. Die Helligkeitswerte der hellen Bereiche werden addiert. (bm_src_alpha, bm_one)
bm_subtract Subtraktives Überblenden. Die Helligkeitswerte der hellen Bereiche werden subtrahiert. (bm_zero, bm_inv_src_colour)
bm_max Max Blending. Ähnlich wie beim additiven Blending. (bm_src_alpha, bm_inv_src_colour)

Wie Sie aus der obigen Tabelle ersehen können, handelt es sich bei diesen vier Mischmodi in Wirklichkeit um Zusammensetzungen erweiterter Mischmodi, die Sie auf der Seite mit der Beschreibung gpu_set_blendmode_ext().

Damit Sie die Mischmodi optimal nutzen können und um zu verstehen, wie sie funktionieren und wie sie sich auf das endgültige Bild auf dem Bildschirm auswirken, empfehlen wir Ihnen, den folgenden Leitfaden zu lesen:

  • Anleitung zur Verwendung von Blendmodes

Syntax:

gpu_set_blendmode(mode);

Argument Typ Beschreibung
mode Blend Mode Constant The blend mode to use (see the table above)

Rückgabe:

N/A

Beispiel:

gpu_set_blendmode(bm_add);
draw_circle_colour(100, 100, 50, c_white, c_black, 0);
gpu_set_blendmode(bm_normal);

Dadurch wird das Schwarz transparent und es entsteht ein "Glüheffekt", da das Weiß außen stark ist und von der Kreismitte aus allmählich schwächer wird. Der Mischmodus wird nach dem Zeichnen des Kreises zurückgesetzt, um sicherzustellen, dass die additive Überblendung nicht auch auf alles andere nach dem Kreis angewendet wird.
