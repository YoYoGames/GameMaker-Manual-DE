gpu_get_zfunc

Mit dieser Funktion können Sie den z-Vergleichsmodus abrufen. Der zurückgegebene Wert ist eine der folgenden Konstanten (der Standardwert ist cmpfunc_lessequal):

Vergleich Funktion Konstante
Konstante Beschreibung
cmpfunc_never Niemals
cmpfunc_less Weniger
cmpfunc_equal Gleichberechtigt
cmpfunc_lessequal Weniger oder gleich
cmpfunc_greater Größere
cmpfunc_notequal Nicht gleichwertig
cmpfunc_greaterequal Größer oder gleich
cmpfunc_always Immer

Syntax:

gpu_get_zfunc();

Rückgabe:

Comparison Function Constant (see table above)

Beispiel:

if gpu_get_zfunc() != cmpfunc_greater
{
    gpu_set_zfunc(cmpfunc_greater);
}

Die obige code prüft, ob die z-Testmethode auf cmpfunc_greater eingestellt ist, und wenn nicht, wird sie auf diese Konstante gesetzt.
