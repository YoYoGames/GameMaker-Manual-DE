gpu_set_texfilter

Diese Funktion kann verwendet werden, um die Texturfilterung (lineare Interpolation) aller auf dem Spielbildschirm gezeichneten Bilder einzustellen. Wenn diese Funktion aktiviert ist (true), werden alle Texturen beim Zeichnen geglättet (dies schließt sprites ein, da auch sie als Texturen betrachtet werden), was bedeutet, dass beim Skalieren oder Verschieben, wenn das Pixelverhältnis nicht 1:1 ist, ein "Verschmieren" über verschiedene Pixel erfolgt, was die Bilder je nach verwendetem Kunststil verschwommen erscheinen lässt. Wenn diese Option deaktiviert ist (false), werden die Bilder beim Skalieren oder Verschieben auf der Grundlage des nächstgelegenen Pixels gezeichnet, was zu "blockigen" Bildern führen kann. Der Standardwert ist false, und dies kann auch in den Globalen Spieloptionen für einzelne Ziele geändert werden platforms.

HINWEIS:Diese Einstellung überschreibt jede Texturstufen-Interpolation, die für Shader mit der Funktion gpu_set_texfilter_ext().

HINWEIS: Auf dem HTML5-Ziel funktioniert diese Funktion nur, wenn WebGL aktiviert ist.

Syntax:

gpu_set_texfilter(enable);

ArgumentTyp Beschreibung
enable Enable or disable texture filtering (true / false)

Rückgabe:

Beispiel:

if gpu_get_texfilter()
{
    gpu_set_texfilter(false);
}
else
{
    gpu_set_texfilter(true);
}

Die obige code prüft, ob die Texturfilterung ein- oder ausgeschaltet ist und schaltet sie entsprechend um.
