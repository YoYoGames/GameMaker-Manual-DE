gpu_get_colourwriteenable

Mit dieser Funktion können die aktuellen Werte für die Schreibfreigabe von Farben abgerufen werden. Die Funktion gibt ein 4-elementiges 1D-Array mit den folgenden Elementen zurück, die entweder true (aktiviert) oder false (deaktiviert) sein werden. Standardmäßig ist das Schreiben aller Farben auf true eingestellt:

  • [0] = Roter Kanal aktiviert/deaktiviert
  • [1] = Grüner Kanal aktiviert/deaktiviert
  • [2] = Blauer Kanal aktiviert/deaktiviert
  • [3] = Alphakanal aktiviert/deaktiviert

Syntax:

gpu_get_colourwriteenable();

Rückgabe:

(4 elements only; see above for details)

Beispiel:

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

Die obige code holt sich die aktuellen Farbschreibwerte und setzt dann die Alpha-Komponente auf false.
