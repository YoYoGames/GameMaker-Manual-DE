gpu_get_texrepeat

Mit dieser Funktion können Sie prüfen, ob die Texturwiederholung aktiviert ist (gibt true zurück) oder nicht (gibt false zurück). Weitere Informationen zur Texturwiederholung finden Sie in der Funktion gpu_set_texrepeat().

Syntax:

gpu_get_texrepeat();

Rückgabe:

Beispiel:

if gpu_get_texrepeat()
{
    gpu_set_texrepeat(false);
}
else
{
    gpu_set_texrepeat(true);
}

Die obige code prüft, ob die Texturwiederholung ein- oder ausgeschaltet ist und schaltet sie entsprechend.
