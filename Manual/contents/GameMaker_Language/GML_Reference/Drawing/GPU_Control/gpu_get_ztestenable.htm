gpu_get_ztestenable

Mit dieser Funktion kann abgefragt werden, ob die z-Prüfung aktiviert ist (die Funktion gibt true zurück) oder nicht (die Funktion gibt false zurück). Der Standardwert ist, dass die z-Prüfung deaktiviert ist, daher gibt die Funktion false zurück.

Syntax:

gpu_get_ztestenable();

Rückgabe:

Beispiel:

if gpu_get_ztestenable() == false
{
    gpu_set_ztestenable(true);
}

Die obige code prüft, ob der z-Test aktiviert ist oder nicht, und wenn er deaktiviert ist, wird er wieder aktiviert.
