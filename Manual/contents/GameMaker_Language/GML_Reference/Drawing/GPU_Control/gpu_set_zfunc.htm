gpu_set_zfunc

Mit dieser Funktion kann der Vergleichsmodus für den Z-Puffer-Test eingestellt werden (siehe gpu_set_ztestenable() für weitere Informationen). Als Werte stehen die folgenden Konstanten zur Verfügung (der Standardwert ist cmpfunc_lessequal):

Konstante Beschreibung
cmpfunc_never Niemals
cmpfunc_less Weniger
cmpfunc_equal Gleichberechtigt
cmpfunc_lessequal Weniger oder gleich
cmpfunc_greater Größere
cmpfunc_notequal Nicht gleichwertig
cmpfunc_greaterequal Größer oder gleich
cmpfunc_always Immer

Syntax:

gpu_set_zfunc(cmp_func);

Argument Typ Beschreibung
cmp_func Vergleich Funktion Konstante The comparison mode to use (see list above)

Rückgabe:

N/A

Beispiel:

gpu_set_ztestenable(true);
gpu_set_zfunc(cmpfunc_always);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

Die obige code schaltet den Z-Puffer-Test ein und setzt seinen Vergleichsmodus, bevor ein Hintergrund gezeichnet wird sprite und schaltet ihn dann wieder aus, um mit dem Zeichnen fortzufahren.
