gpu_get_blendmode_ext_sepalpha

Diese Funktion kann verwendet werden, um den aktuellen erweiterten Überblendungsmodus abzurufen, der zum Zeichnen verwendet wird, wobei die Überblendungsmodi in zwei Teile aufgeteilt werden - die RGB-Komponente und die Alpha-Komponente. Die Funktion gibt ein 1D-Array mit 4 Elementen zurück, das die folgenden Elemente enthält:

  • [0] = Quell-Mischmodus abzüglich der Alphakomponente (Standard ist bm_src_alpha (As, As, As))
  • [1] = Ziel-Mischmodus abzüglich der Alphakomponente (Standard ist bm_inv_src_alpha (1-As, 1-As, 1-As))
  • [2] = Quell-Alpha-Komponente (Standard ist bm_src_alpha (As))
  • [3] = Ziel-Alpha-Komponente (Standard ist bm_inv_src_alpha (1-As))

Die Werte des Arrays sind eine der folgenden Konstanten ("s" steht für einen Wert aus der Quelle, "d" für einen Wert aus dem Ziel):

Konstante Überblendfaktor (Rot, Grün, Blau, Alpha)
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-As)
bm_src_alpha (As, As, As, As)
bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As)
bm_dest_alpha (Anzeige, Anzeige, Anzeige, Anzeige)
bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) mit f = min(As, 1-Ad)

Wenn Sie die erweiterten Mischmodi auf diese Weise erstellen, werden die RGB-Komponenten von einem Mischmodus und die Alphakomponente von einem anderen übernommen. Wenn Sie also beispielsweise bm_zero (0, 0, 0, 0) als RGB-Mischmodus und bm_one als Alphamodus verwenden, erhalten Sie eine zusammengesetzte Mischung von (0,0,0,1).

Beachten Sie, dass Sie diese Werte direkt innerhalb des Arrays ändern und das gesamte Array als einzelnes argument an die Funktion übergeben können (wie im folgenden Beispiel gezeigt).

Syntax:

gpu_get_blendmode_ext_sepalpha();

Rückgabe:

(4 elements only; see above for constants)

Beispiel:

var bm = gpu_get_blendmode_ext_sepalpha();
bm[0] = bm_src_alpha;
bm[1] = bm_inv_src_alpha;
gpu_set_blendmode_ext_sepalpha(bm);

Die obige code erstellt ein lokales Array und ruft den aktuellen erweiterten Mischmodus mit separaten Alphawerten ab. Dieses Array wird dann bearbeitet und zurückgegeben, um den Mischmodus festzulegen.
