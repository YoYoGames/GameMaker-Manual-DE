gpu_set_ztestenable

Diese Funktion kann verwendet werden, um den Z-Puffer-Test ein- oder auszuschalten, was sich auf die Art und Weise auswirkt, wie Dinge gezeichnet werden (im Allgemeinen nur von Nutzen, wenn man mit 3D-Projekten arbeitet). Wenn die Z-Prüfung standardmäßig ausgeschaltet ist und zwei Objekte in denselben Raum gezeichnet werden, werden beide Objekte gerendert, werden beide Objekte gerendert, unabhängig davon, ob das eine das andere überzeichnet, was zu unnötigen Zeichenaufrufen führt. Wenn Sie switch einschalten, wird der Z-Puffer getestet, um festzustellen, ob ein Objekt "sichtbar" ist und nicht gezeichnet wird, wenn dies nicht der Fall ist. nicht. Beachten Sie, dass dies das Standardverhalten ist, aber Sie können dies ändern, indem Sie die Art des Vergleichs, der für den Z-Buffer-Test verwendet wird, ändern (siehe die Funktion gpu_set_zfunc().

Standardmäßig ist der z-Puffer-Test ausgeschaltet (false).

Syntax:

gpu_set_ztestenable(enable);

ArgumentTyp Beschreibung
enable Enable or disable z-buffer testing (true oder false).

Rückgabe:

Beispiel:

gpu_set_ztestenable(true);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

Die obige code schaltet den Z-Buffer-Test ein, um einen Hintergrund sprite zu zeichnen, und schaltet ihn dann wieder aus, um weiter zu zeichnen.
