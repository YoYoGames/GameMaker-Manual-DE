gpu_get_alphatestref

Sie können diese Funktion verwenden, um den aktuellen Wert für die Alphatestreferenz zu ermitteln (Standardwert ist 0, aber Sie können mit gpu_set_alphatestref() diesen Wert auf einen anderen Wert setzen).

Syntax:

gpu_get_alphatestref();

Rückgabe:

(from 0 - 255)

Beispiel:

if gpu_get_alphatestenable()
{
    if gpu_get_alphatestref() < 254
    {
        gpu_set_alphatestref(254);
    }
}

Die obige code prüft, ob der Alphatest aktiviert ist, und wenn ja, dann prüft sie den aktuellen Alphatest-Referenzwert und setzt ihn, wenn er kleiner als 254 ist.
