gpu_set_fog

Mit dieser Funktion können Sie das Zeichnen von Nebel aktivieren oder deaktivieren. Nebel kann in 3D-Spielen verwendet werden, um Instanzen in der Ferne verblasst oder sogar unsichtbar aussehen zu lassen, was sowohl bei der Schaffung von Atmosphäre hilft als auch die Tatsache verdeckt, dass Sie Instanzen, die weit entfernt sind, nicht zeichnen.

Sie legen fest, ob die Funktion aktiviert (true) oder deaktiviert (false) ist, welche Farbe der Nebel für die Überblendung verwenden soll, sowie die Start- und Endentfernung. Die Entfernungswerte sind im Wesentlichen Tiefenwerte in Pixeln und beziehen sich auf die Position der Kamera.

Eine Position von 0 ist zum Beispiel am nächsten zur Kamera, und mit jeder Erhöhung wird sie weiter entfernt. Standardmäßig befindet sich die Kamera einer Ansicht in einer Tiefe von -16000. Wenn Sie also möchten, dass der Nebel in Tiefe 0 beginnt und in Tiefe 1000 endet, müssen Sie den Startabstand auf 16000 und den Endabstand auf 17000 einstellen.

Die Funktion kann vier einzelne arguments (siehe oben) oder ein einzelnes Array mit der folgenden Struktur annehmen (das folgende Beispiel code zeigt diese Methode):

  • [0] = aktivierter Toggle (ein boolescher Wert, entweder true oder false), Standard false
  • [1] = Farbe (real), Standard c_black
  • [2] = Startabstand (real), Standardwert 0
  • [3] = Endabstand (real), Standardwert 1

Syntax:

gpu_set_fog(enable, colour, start, end);

Argument Typ Beschreibung
enable Boolesche Enable or disable fog
colour Farbe The fog colour
start Real The distance to start applying fog from (relative to the camera)
end Real The distance to end the fog (relative to the camera)

Rückgabe:

N/A

Beispiel:

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

Die obige code ruft die aktuellen Nebeleinstellungen ab und setzt dann das Farbelement des Arrays auf c_red, bevor der Nebel mit dem geänderten Array erneut eingestellt wird.
