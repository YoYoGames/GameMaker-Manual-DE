gpu_get_cullmode

Mit dieser Funktion kann der Modus für das Backface-Culling abgefragt werden. Der zurückgegebene Wert ist eine der folgenden Konstanten (der Standardwert ist cull_noculling):

Tötungsmodus Konstant
Konstante Beschreibung
cull_noculling Es wird keine Keulung vorgenommen
cull_clockwise Alle Dreiecke im Uhrzeigersinn werden herausgefiltert
cull_counterclockwise Alle Dreiecke, die gegen den Uhrzeigersinn liegen, werden aussortiert.

Syntax:

gpu_get_cullmode();

Rückgabe:

Culling Mode Constant (see above for constants)

Beispiel:

if gpu_get_cullmode() != cull_clockwise
{
    gpu_set_cullmode(cull_clockwise);
}

Die obige code ruft den aktuellen Cull-Modus ab, und wenn er nicht cull_clockwise ist, wird er auf diese Konstante gesetzt.
