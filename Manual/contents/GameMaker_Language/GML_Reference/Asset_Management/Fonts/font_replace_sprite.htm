schrift_ersetzen_sprite

Mit dieser Funktion können Sie einen "Sprite Strip" verwenden, um ein zuvor erstelltes sprite Font-Asset zu ersetzen, wobei jedes Teilbild ein einzelnes Symbol oder einen Buchstaben darstellt. Der Index ist die UTF8-Zuordnungsnummer für das erste zu verwendende Zeichen, und die zu verwendenden Bereiche sind die gleichen wie bei einer normalen Schriftart, beginnend mit "!", aber denken Sie daran, dass Sie nur das einschließen sollten, was Sie brauchen, und dass Sie die ord() Funktion verwenden, um den richtigen UTF8-Wert für den Anfangsbuchstaben zu erhalten.

Sie können auch festlegen, ob die Schrift proportional (true) oder nicht proportional (false) sein soll. Bei einer proportionalen Schrift richtet sich der Abstand nach der tatsächlichen Breite der Buchstaben (der Buchstabe "i" nimmt also weniger Platz ein als der Buchstabe "w"), während bei einer nicht proportionalen Schrift der Abstand auf der Breite des Teilbildes basiert, wie bei einer einzeiligen Schrift. Schließlich können Sie den Abstand zwischen den einzelnen Buchstaben beim Schreiben festlegen. Dieser kann ein beliebiger ganzzahliger Wert sein, wobei 0 für keinen Abstand steht (die Buchstaben "berühren" sich, wenn sie proportional sind).

Wenn die Funktion fehlschlägt, wird sie stillschweigend abgebrochen, ohne die angegebene Schriftart sprite zu ersetzen.

Wenn Sie also z. B. '.' und '0', aber nicht '/' verwenden, müssen Sie entweder trotzdem '/' einfügen oder ein leeres Teilbild zwischen den beiden verwendeten Zeichen lassen. Die folgende Abbildung zeigt ein typisches Layout der Schrift sprite:

Syntax:

font_replace_sprite(ind, spr, first, prop, sep);

Argument Typ Beschreibung
ind Schriftart Asset The index of the font that shall be replaced (ie all references to this resource past this function will refer to the new font, not the original).
spr Sprite-Asset The sprite to add a font based on.
first Real The index of the first available character.
prop Boolesche Whether white space around each letter should be removed (if false, the created font will be monospaced, otherwise it'll be like most regular fonts).
sep Real The space between each letter, 0 meaning the right of one letter just touches the left of the next one.

Rückgabe:

N/A

Beispiel:

font_replace_sprite(global.Font, working_directory + "Sprites\spr_Font.png", ord("!"), true, 2);

Die obige code wird die in der globalen Variable "Font" indizierte Schriftart durch eine neue Schriftart aus einer externen Bilddatei ersetzen.
