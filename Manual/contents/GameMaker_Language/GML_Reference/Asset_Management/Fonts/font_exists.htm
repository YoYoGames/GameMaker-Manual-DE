font_exists

Diese Funktion gibt zurück, ob eine Schrift mit dem angegebenen Index existiert oder nicht. Sie können die im Asset-Browser definierten Schriftindizes prüfen oder Schriften, die mit Funktionen wie font_add() hinzugefügt wurden.

Syntax:

font_exists(ind);

ArgumentTyp Beschreibung
ind Index of the font to check.

Rückgabe:

Beispiel:

if font_exists(fnt_Main)
{
    draw_set_font(fnt_Main);
}

Dadurch wird die aktive Zeichenschriftart auf fnt_Main gesetzt, falls sie existiert.
