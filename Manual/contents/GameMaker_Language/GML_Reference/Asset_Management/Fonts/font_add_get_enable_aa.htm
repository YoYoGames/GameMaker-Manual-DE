font_add_get_enable_aa

Diese Funktion kann verwendet werden, um zu überprüfen, ob Anti-Aliasing (AA) für Schriften aktiviert ist, die mit font_add(). Die Funktion gibt true zurück, wenn AA aktiviert ist, und false, wenn es nicht aktiviert ist. Beachten Sie, dass AA standardmäßig aktiviert ist, aber Sie können den AA-Status für hinzugefügte Schriftarten mit der Funktion font_add_enable_aa()ändern, sofern sie vor dem Hinzufügen der Schriftart aufgerufen wird.

Syntax:

font_add_get_enable_aa();

Rückgabe:

Beispiel:

if !font_add_get_enable_aa()
{
    font_add_enable_aa(true);
}

Die obige code prüft den Status des Anti-Aliasing für hinzugefügte Schriften und wenn es nicht aktiviert ist, dann aktivieren wir es.
