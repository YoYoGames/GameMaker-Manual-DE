schrift_get_italic

Mit dieser Funktion können Sie für jede beliebige Schriftart prüfen, ob sie das Kursiv-Flag hat oder nicht. Ist dies der Fall, gibt die Funktion true zurück, andernfalls false.

Syntax:

font_get_italic(ind);

ArgumentTyp Beschreibung
ind Index of the font to check.

Rückgabe:

Beispiel:

if font_get_italic(fnt_Main)
{
    draw_set_font(fnt_Main);
}

Dadurch wird die aktive Zeichenschrift auf fnt_Main gesetzt, wenn sie in ihren Schrifteigenschaften als kursiv eingestellt ist.
