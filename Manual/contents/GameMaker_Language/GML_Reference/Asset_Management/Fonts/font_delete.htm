Schriftart_löschen

Mit dieser Funktion können Sie ein Schrift-Asset aus dem Spiel löschen. Dies ist eine dauerhafte Entfernung, und ein Raumwechsel oder ein Neustart des Spiels bringt die entfernte Schriftart nicht zurück. Dazu müsste der Spieler das Spiel verlassen und das Spiel neu starten, also sei vorsichtig, wenn du diese Funktion benutzt. Im Allgemeinen wird diese Funktion nur benötigt, um Speicher freizugeben, der von einer Schriftart belegt wurde, die dem Spiel durch die Funktionen font_add() oder font_add_sprite().

Syntax:

font_delete(ind);

ArgumentTyp Beschreibung
ind The index of the font to delete.

Rückgabe:

Beispiel:

font_delete(global.Font);

Die obige code löscht die in der globalen Variable "Font" indizierte Schriftart.
