Schriftart_holen_Info

Diese Funktion wird verwendet, um Informationen über die angegebene Schriftart abzurufen. Sie geben einen Schriftindex an (der ein über den Asset-Browser hinzugefügtes Asset oder eine unter runtimehinzugefügte Schrift sein kann) und die Funktion gibt eine Struktur mit den folgenden Variablen zurück:

Schriftart Info Struktur
Variable Name Datenart Beschreibung
ascenderOffset Real Der maximale Abstand von der Grundlinie zum oberen Rand der Schrift (in Pixel)
size Real Die ungefähre Größe der Schrift (in Pixel)
spriteIndex Sprite-Asset Der Index sprite für die Schriftart, wenn sie aus einem Sprite erstellt wurde, ansonsten -1
texture Zeiger -1, wenn die Schriftart aus einer sprite erstellt wurde, ansonsten die Textur-ID der Schriftart
name Zeichenfolge Der Name der Schriftart
bold Boolesche true wenn die Schrift fett ist, sonst false (siehe HINWEIS unten)
italic Boolesche true wenn die Schrift kursiv ist, sonst false (siehe HINWEIS unten)
glyphs Schriftart Glyphe Struct Eine Struktur, die Informationen für jede Glyphe in der Schriftart enthält (weitere Informationen finden Sie unten)
effectsEnabled Boolean Whether effects are enabled for this font
effectParams Struct The effects struct for this font, which can be changed with font_enable_effects

HINWEIS Die Variablen bold und italic spiegeln nur die Einstellungen des Benutzers für die Schriftart wider, so dass sie für Schriften, die standardmäßig fett oder kursiv erscheinen, möglicherweise nicht korrekt sind.

Wenn die angegebene Schriftart nicht existiert, gibt die Funktion undefined zurück.

Beachten Sie auch, dass das Ändern der Werte einer dieser Variablen oder einer in der Struktur glyphs enthaltenen Variablen die Darstellung der Schriftart nicht verändert, so dass diese Informationen als schreibgeschützt betrachtet werden sollten.

Glyphen Struktur

Die Variable glyphs in der zurückgegebenen Struktur ist eine eigenständige Struktur, die Informationen zu jeder in der Schrift enthaltenen Glyphe enthält. Jede Variable in dieser Struktur ist der Zeichenname einer Glyphe, der eine Struktur mit Informationen zu dieser Glyphe enthält. Sie können sich diese verschachtelten Structs in folgendem Format vorstellen (ausgehend von der Haupt-Struct):

{
    glyphs:
    {
        A: {},
        B: {},
        C: {},
        // ...other glyphs
    },
    // ...other font info
}

Sie können den Zeichennamen einer Glyphe verwenden, um ihre Informationen aus der Struktur abzurufen, wie z. B. im folgenden Code:

var _font_info = font_get_info(Font1);

var _info_A = _font_info.glyphs[$ "A"]; // $ is a struct accessor

show_debug_message(_info_A);

Die obige code ruft die Glyphenstruktur für das Zeichen "A" ab und gibt sie in das Ausgabeprotokoll ein. Sie können die Zeichenkette ("A") auch durch eine Variable ersetzen, die eine Zeichenkette enthält, wodurch Sie Informationen für jede Glyphe über diese Variable unter runtime abrufen können.

Die Struktur, die für eine Glyphe zurückgegeben wird, enthält die folgenden Variablen:

HINWEIS Bei HTML5 ist die Glyphenstruktur für alle geladenen Dateischriften (wie *.ttf oder *.otf) leer, da runtime keine Informationen über diese Glyphen enthält.

Schriftart Glyphe Struct
Variable Name Datenart Beschreibung
char real Wenn die Schrift aus einer sprite erstellt wurde, ist dies der Bildindex der Glyphe aus dieser sprite, andernfalls ist es ihre Unicode-Zeichennummer

Hinweis: Alle Variablen darunter sind in der Struktur nicht vorhanden, wenn der Font aus einem Sprite erstellt wurde.
x real Die X-Position der Glyphe auf der Texturseite (in Texel)
y real Die Y-Position der Glyphe auf der Texturseite (in Texel)
w real Die Breite der Glyphe auf der Texturseite (in Texel)
h real Die Höhe der Glyphe auf der Texturseite (in Texel)
shift real Die Anzahl der Pixel, die beim Vorrücken zum nächsten Zeichen nach rechts verschoben werden (kann negativ sein, wenn nach links verschoben wird)
offset real Die Anzahl der Pixel, um die die Darstellung dieser Glyphe horizontal versetzt wird (kann positiv oder negativ sein)
kerning Array Ein Array von Ganzzahlen, das Kerning-Informationen in Paaren (oder 2er-Gruppen) enthält. Die erste ganze Zahl in einem Paar ist der Unicode-Wert für ein Zeichen, und die zweite ganze Zahl ist der Betrag, der zum Verschiebungswert dieses Zeichens hinzuzufügen ist (kann positiv oder negativ sein), wenn ihm das Zeichen dieser Glyphe vorausgeht.

HINWEIS Für Dateischriften, die unter runtime geladen werden, können die Werte x und y für eine Glyphe -1 sein, wenn sie noch nicht gerendert wurde, da sie keine Position auf der für die Zwischenspeicherung verwendeten Texturseite hat.

Syntax:

font_get_info(font);

Argument Typ Beschreibung
font Schriftart Asset The font for which information should be retrieved.

Rückgabe:

Font Info Struct (or undefined)

Beispiel:

var _info = font_get_info(my_font);

if (_info != undefined && _info.bold)
{
    image_xscale *= 2;
    image_yscale *= 2;
}

Diese Funktion code ruft Informationen über die in der Variablen my_font gespeicherte Schriftart ab. Wenn der zurückgegebene Wert nicht undefined ist und die angegebene Schriftart fett ist, wird der Maßstab der Instanz verdoppelt.
