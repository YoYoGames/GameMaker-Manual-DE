schrift_ersetzen_schrift_ext

Diese Funktion ersetzt eine Schriftart sprite, die Sie zuvor mit der Funktion font_add_sprite_ext(). Sie verwendet einen "Sprite Strip", bei dem jedes Teilbild ein einzelnes Symbol oder ein Buchstabe ist, und die Bilder werden anhand der argument "string_map" der Funktion auf Buchstaben abgebildet. Diese argument ist eine Zeichenkette, mit der du GameMaker mitteilen kannst, in welcher Reihenfolge die Teilbilder der sprite Schriftart angeordnet sind, und die beim Schreiben von Text automatisch zugeordnet wird. Wenn du also zum Beispiel eine String-Map von "AaBbCcDdEeFfGgHh" hast, muss dein sprite Font die Unterbilder in dieser Reihenfolge haben.

Sie können auch festlegen, ob die Schrift proportional (true) oder nicht proportional (false) sein soll. Bei einer proportionalen Schrift richtet sich der Abstand nach der tatsächlichen Breite der Buchstaben (der Buchstabe "i" nimmt also weniger Platz ein als der Buchstabe "w"), während bei einer nicht proportionalen Schrift der Abstand auf der Breite des Teilbildes basiert, wie bei einer einzeiligen Schrift. Schließlich können Sie den Abstand zwischen den einzelnen Buchstaben beim Schreiben festlegen. Dieser kann ein beliebiger ganzzahliger Wert sein, wobei 0 für keinen Abstand steht (die Buchstaben "berühren" sich, wenn sie proportional sind). Die Funktion gibt einen Indexwert zurück, der in einer Variablen gespeichert werden sollte, da dieser in allen weiteren codes benötigt wird, die sich auf diese Schriftart beziehen.

Wenn die Funktion fehlschlägt, wird sie stillschweigend abgebrochen, ohne die angegebene Schriftart sprite zu ersetzen.

HINWEIS Diese Funktion setzt voraus, dass Sie zuvor eine Schriftart sprite hinzugefügt haben, indem Sie font_add_sprite()hinzugefügt haben, da sie diese ersetzt und nicht für Schriftarten verwendet werden kann, die in den Spielressourcen enthalten sind.

Syntax:

font_replace_sprite_ext(font, spr, string_map, prop, sep);

Argument Typ Beschreibung
font Schriftart Asset The font to replace.
spr Sprite-Asset The sprite to add a font based on.
string_map Zeichenfolge String from which sprite sub-image order is taken.
prop Boolesche Set as proportional font or not.
sep Real The space to leave between each letter.

Rückgabe:

N/A

Beispiel:

font_replace_sprite_ext(global.Font, spr_CalcFont, "0123456789+-*/=", true, 2);

Die obige code ersetzt ein sprite Schriftarten-Asset, das in der Variablen "global.Font" indiziert ist, durch sprite Bilder, die in der Variablen "spr_CalcFont" indiziert sind.
