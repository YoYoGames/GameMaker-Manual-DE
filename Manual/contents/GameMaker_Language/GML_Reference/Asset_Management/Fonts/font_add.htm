Schriftart_hinzufügen

Mit dieser Funktion können Sie Ihrem Spiel eine Schriftart aus einer Schriftartdatei in den Included Files hinzufügen. Sie können die Größe der Schriftart (in Punkten) festlegen, sowie ob die Schriftart fett oder kursiv sein soll, und Sie können auch den Bereich der einzuschließenden Zeichen definieren. Die Funktion gibt einen Indexwert zurück, der in einer Variablen gespeichert werden sollte, da dieser in allen weiteren Codes benötigt wird, die sich auf diese Schriftart beziehen, oder sie gibt -1 zurück, wenn die Funktion aus irgendeinem Grund fehlschlägt.

Beachten Sie, dass es sich bei der Schriftartdatei entweder um eine *.ttf - oder eine *.otf -Datei handeln muss, was auch für das Hinzufügen von Nicht-Standard-Schriftarten wie asiatischen oder arabischen Schriftarten nützlich sein kann.

WARNUNG! Wenn Sie eine solche Schriftart in die enthaltenen Dateien aufnehmen, muss sie für den Vertrieb mit dem Spiel lizenziert werden.

Für das HTML5-Ziel module kann diese Funktion nur verwendet werden, um Web Fonts hinzuzufügen, und sie funktioniert nur, wenn WebGL nicht aktiviert ist - wenn WebGL aktiviert ist, müssen Sie eine allgemeine Font-Ressource verwenden (d. h.: mit draw_set_font() zusammen mit einer Schriftart aus dem Asset-Browser) oder Bitmap-Schriften (siehe font_add_sprite()).

WICHTIG! Bei der Ausrichtung auf HTML5 wird kursiver und fetter Text nur unterstützt, wenn die Schriftart oder die Webschrift selbst diese Zeichen enthält. Sie werden nicht aus der Basisschriftart synthetisiert.

Beim Laden einer Schriftart aus einer Datei wird die Größe der Schriftart in Pixeln angegeben und der erste und der letzte Wert werden ignoriert, d.h. alle Glyphen der Schriftart werden hinzugefügt. Schriften, die auf diese Weise hinzugefügt werden, werden einer eigenen Texturseite zugewiesen. Daher ist bei der Verwendung dieser Funktion Vorsicht geboten, da sie die Anzahl der Texturwechsel beim Zeichnen erhöht. Es ist auch zu beachten, dass Schriften beim Zeichnen etwas größer erscheinen können, da die Glyphen Teile haben können, die außerhalb des für die Schrift definierten Begrenzungsrahmens gezeichnet werden. Sie sollten sich auch darüber im Klaren sein, dass es eine Begrenzung von etwa 200 Glyphen gibt, die in einem einzigen Frame für eine bestimmte Schriftart gerendert werden können.

Standardmäßig ist bei hinzugefügten Schriftarten die Kantenglättung aktiviert. Sie können den Status von AA jedoch jederzeit mit der Funktion font_add_get_enable_aa() überprüfen, und Sie können AA mit der Funktion font_add_enable_aa().

HINWEIS Wenn Sie eine Schriftart in GameMaker laden, müssen Sie daran denken, sie wieder zu entfernen (mit font_delete()) zu entfernen, wenn Sie sie nicht mehr benötigen, sonst besteht die Gefahr eines Speicherlecks, das Ihr Spiel verlangsamt und schließlich zum Absturz bringt.

Syntax:

font_add(name, size, bold, italic, first, last);

Argument Typ Beschreibung
name Zeichenfolge The path of the font file in the Included Files (*.ttf oder *.otf).
size Real The size of the font - points for Web Fonts, pixels for file fonts.
bold Boolesche Whether the font is bold (true) oder nicht (false).
italic Boolesche Whether the font is italic (true) oder nicht (false).
first Real The first character to include (if you're unsure, go for 32).
last Real The last character to include (if you're unsure, go for 128).

Rückgabe:

Font Asset

Beispiel:

new_font = font_add("gamefont.ttf", 24, true, true, 32, 128);

Dadurch wird die Datei "gamefont.ttf" , die sich in den mitgelieferten Dateien des Spiels befindet, mit einer Schriftgröße von 24pt geladen. Die Schriftart ist fett und kursiv, und der Schriftbereich umfasst Groß- und Kleinbuchstaben, Zahlen und alle gängigen Satzzeichen. Nachdem die Schriftart geladen ist, wird ihre ID in der Variablen new_font gespeichert.
