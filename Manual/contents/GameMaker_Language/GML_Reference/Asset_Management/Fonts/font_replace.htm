schrift_ersetzen

Mit dieser Funktion können Sie eine Schriftart in Ihrem Spiel durch eine der Schriftarten ersetzen, die auf dem System installiert sind, auf dem es läuft. Beachten Sie, dass die zu ersetzende Schriftart keine Schriftart aus dem Asset-Browser sein kann, sondern eine Schriftart sein muss, die zuvor mit der Funktion font_add().

Wenn Sie eine Schriftart auf diese Weise ersetzen, können Sie die Größe der Schrift (in Punkt) festlegen und bestimmen, ob die Schrift fett oder kursiv sein soll, und Sie können auch den Bereich der einzuschließenden Zeichen festlegen. Wenn die Funktion fehlschlägt, wird sie stillschweigend abgebrochen und die angegebene Schriftart wird nicht ersetzt.

Für das HTML5-Ziel modulekann diese Funktion nur verwendet werden, um Web Fonts zu ersetzen, und sie funktioniert nur, wenn WebGL nicht aktiviert ist - wenn WebGL aktiviert ist, müssen Sie eine allgemeine Font-Ressource verwenden (d.h. mit draw_set_font() zusammen mit einer Schriftart aus dem Asset-Browser) oder Bitmap-Schriften (siehe font_add_sprite()).

WICHTIG Bei der Ausrichtung auf HTML5 wird kursiver und fetter Text nur unterstützt, wenn die Schriftart oder die Webschrift selbst diese Zeichen enthält. Sie werden nicht aus der Basisschriftart synthetisiert.

In der folgenden Tabelle sind die Schriftarten aufgeführt, die in allen Browsern standardmäßig vorhanden sind und problemlos verwendet werden können sollten. Jede andere Schriftart kann auf dem Computer oder Gerät, auf dem Ihr Spiel läuft, vorhanden sein oder auch nicht, daher sollte die Verwendung dieser Funktion auf diese Standardschriften beschränkt sein:

Auf allen anderen (nicht Web-)Zielen können Sie diese Funktion verwenden, um eine Schriftart aus einer Datei zu ersetzen. Die Datei muss mit der Funktion " Enthaltene Dateien" von GameMaker in das Spielpaket aufgenommen werden und muss entweder eine Schriftartdatei im Format *.ttf oder *.otf sein, was nützlich ist, um nicht standardmäßige Schriftarten wie asiatische oder arabische hinzuzufügen.

TIPP Wenn Sie eine solche Schriftartdatei in ein Spiel einbinden, vergewissern Sie sich, dass sie ordnungsgemäß für den Vertrieb mit dem Spiel lizenziert ist.

Wenn eine Schriftart auf diese Weise aus einer Datei geladen wird, ist die Größe der Schriftart in Pixeln angegeben und der erste und letzte Wert werden ignoriert, d.h. alle Glyphen der Schriftart werden hinzugefügt. Auf diese Weise hinzugefügte Schriftarten werden einer eigenen Texturseite zugewiesen, daher ist bei der Verwendung dieser Funktion Vorsicht geboten, da sie die Anzahl der Texturwechsel beim Zeichnen erhöht. Es ist auch zu beachten, dass Schriften beim Zeichnen etwas größer erscheinen können, da Glyphen Teile haben können, die außerhalb des für die Schrift definierten Begrenzungsrahmens gezeichnet werden. Sie sollten sich auch darüber im Klaren sein, dass es eine Begrenzung von etwa 200 Glyphen gibt, die in einem einzigen Frame für eine bestimmte Schriftart gerendert werden können.

HINWEIS Wenn Sie eine Schriftart in GameMaker laden, müssen Sie daran denken, sie wieder zu entfernen (mit font_delete()) zu entfernen, wenn Sie sie nicht mehr benötigen, sonst besteht die Gefahr eines Speicherlecks, das Ihr Spiel verlangsamt und schließlich zum Absturz bringt.

Syntax:

font_replace(ind, name, size, bold, italic, first, last);

Argument Typ Beschreibung
ind Schriftart Asset The index of the (previously added) font to be replaced.
name Zeichenfolge The name of the font to be used (eg 'Arial'), or the file path if the font is an included *.ttf oder *.otf Datei.
size Real The size of the font - points for Web Fonts, pixels for file fonts.
bold Boolesche Whether the font is bold (true) or not (false).
italics Boolesche Whether the font is italic (true) or not (false).
first Real The first character to include (if you're unsure, go for 32).
last Real The last character to include (if you're unsure, go for 128).

Rückgabe:

N/A

Beispiel:

font_replace(myfont, "Arial", 24, true, true, 32, 128);

Dadurch wird die in der Variablen "myfont" indizierte Schriftart durch eine neue Schriftart in der Größe 24pt ersetzt, die "Arial" verwendet und fett und kursiv ist. Die Schriftart umfasst Groß- und Kleinbuchstaben, Zahlen und alle gängigen Interpunktionszeichen.
