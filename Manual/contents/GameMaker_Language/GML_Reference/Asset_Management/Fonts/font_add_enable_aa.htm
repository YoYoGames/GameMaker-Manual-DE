font_add_enable_aa

Diese Funktion kann verwendet werden, um Anti-Aliasing (AA) für Schriftarten zu aktivieren oder zu deaktivieren, die mit font_add(). Diese Funktion muss aufgerufen werden , bevor Sie Schriftarten hinzufügen. Wenn Sie sie auf true setzen, wird AA aktiviert, und wenn Sie sie auf false setzen, wird es deaktiviert. Standardmäßig ist AA aktiviert. Beachten Sie, dass dies keine Auswirkungen auf Schriftarten hat, die vor dem Aufruf der Funktion hinzugefügt wurden, und dass die Funktion nur einmal aufgerufen werden muss, wenn die Schriftart hinzugefügt wird, und nicht bei jedem Zeichnen/Schritt, bei dem die Schriftart verwendet wird.

Syntax:

font_add_enable_aa(enable);

ArgumentTyp Beschreibung
enable Whether to enable (true) oder deaktivieren (false) Sie AA für hinzugefügte Schriftarten.

Rückgabe:

Beispiel:

if !font_add_get_enable_aa()
{
    font_add_enable_aa(true);
}

Die obige code prüft den Status des Anti-Aliasing für hinzugefügte Schriften und wenn es nicht aktiviert ist, dann aktivieren wir es.
