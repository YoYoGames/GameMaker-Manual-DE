schrift_get_last

Wenn Sie in GameMaker eine Schriftart definieren, können Sie eine Reihe von Zeichen festlegen, die enthalten sein sollen. Das liegt daran, dass die Schriftart selbst nicht in Ihr Spiel aufgenommen wird (aus rechtlichen Gründen), sondern ein Bild der der Schriftart auf einer Texturseite enthalten ist, und das ist es, was Ihr Spiel verwenden wird (genau wie jedes andere Grafikobjekt). Das bedeutet, dass du die Anzahl der Zeichen, die du verwendest, auf ein Minimum beschränken und nur den Bereich der Zeichen angeben solltest, den die Ihr Spiel benötigt, um den Texturspeicher so optimal wie möglich zu nutzen. Diese Funktion kann verwendet werden, um das letzte Zeichen (als UTF8-Wert) zu finden, das verwendet wurde, als die Schriftart dem Spiel hinzugefügt wurde.

Syntax:

font_get_last(ind);

ArgumentTyp Beschreibung
ind Index of the font to check.

Rückgabe:

Beispiel:

num = font_get_last(fnt_Main);

Die obige code speichert den ASCII-Wert des letzten Buchstabens des Schriftbereichs für die in "fnt_Main" indizierte Schrift.
