schrift_get_textur

Diese Funktion gibt einen speziellen Zeiger für die Schrifttextur-Seite zurück. Dieser Wert kann dann in anderen Zeichenfunktionen verwendet werden, insbesondere beim allgemeinen Zeichnen unter Verwendung von Primitiven sowie den Shader-Funktionen. Sie können mehr Informationen über die zurückgegebene Texturseite erhalten, indem Sie die verschiedenen texture_ Funktionen verwenden, die Sie hier finden.

Syntax:

font_get_texture(font);

Argument Typ Beschreibung
font Schriftart Asset The index of the font to use.

Rückgabe:

Texture

Beispiel:

tex = font_get_texture(fnt_Main);

Die obige code erhält den Texturzeiger für die als "fnt_Main" indizierte Schriftart.
