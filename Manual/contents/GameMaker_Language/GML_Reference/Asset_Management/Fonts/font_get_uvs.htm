font_get_uvs

Diese Funktion gibt ein Array mit den UV-Koordinaten für die Schrifttextur auf der Texturseite zurück und füllt das Array mit den folgenden Werten auf:

  • [0] = links
  • [1] = oben
  • [2] = rechts
  • [3] = unten

Dieser Wert kann dann in anderen Zeichenfunktionen verwendet werden, insbesondere beim allgemeinen Zeichnen unter Verwendung von Primitiven sowie bei den Shader-Funktionen.

Syntax:

font_get_uvs(font);

ArgumentTyp Beschreibung
font The index of the font to use.

Rückgabe:

Beispiel:

var tex = font_get_uvs(fnt_Main);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_left = tex[3];

Die obige code speichert die UV-Koordinaten für den gegebenen Hintergrund in einem lokalen Array und weist die Werte dann den Instanzvariablen zu.
