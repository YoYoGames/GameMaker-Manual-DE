Schriftart_get_fontname

Mit dieser Funktion können Sie den aktuellen Systemnamen der angegebenen Schriftart ermitteln. Diese Funktion gibt einen String und keinen Index zurück, und der zurückgegebene Name hängt von der verwendeten Schriftart ab. Sie können zum Beispiel eine Schriftart mit dem Namen "fnt_Main" im Asset-Browser haben, und die Schriftart selbst kann die Windows Systemschriftart Arial sein. In diesem Fall wird die Funktion "Arial" zurückgeben, da dies der Systemname der Schrift ist. Wenn Sie den Namen benötigen, der im Asset-Browser erscheint, verwenden Sie font_get_name().

Das oben beschriebene Verhalten gilt nur für Schriftarten, die über IDE hinzugefügt wurden. Wenn die Funktion auf eine Schriftart angewendet wird, die über runtime (mit font_add()) geladen wurde, gibt sie den vollständigen Pfad zu dieser Schriftartdatei anstelle des Schriftartnamens zurück.

Syntax:

font_get_fontname(ind);

ArgumentTyp Beschreibung
ind Index of the font to check.

Rückgabe:

String

Beispiel:

fnt_Name = font_get_fontname(font0);

Die obige code ermittelt den Systemnamen einer Font-Ressource und speichert ihn als String in der Variablen "fnt_Name".
