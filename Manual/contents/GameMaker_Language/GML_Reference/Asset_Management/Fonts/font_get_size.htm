font_get_size

Mit dieser Funktion können Sie die Größe einer beliebigen Schriftressource ermitteln, d.h. den Punktwert, der im Schriftressourcen-Dialog angezeigt wird.

Syntax:

font_get_size(ind);

ArgumentTyp Beschreibung
ind Index of the font to get the size of.

Rückgabe:

Beispiel:

sz = font_get_size(font0);

Damit wird die Größe der durch die Variable "font0" indizierten Schriftart ermittelt und in der Variablen "sz" gespeichert.
