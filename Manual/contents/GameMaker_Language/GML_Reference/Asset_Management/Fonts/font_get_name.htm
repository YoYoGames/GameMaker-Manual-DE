font_get_name

Diese Funktion kann verwendet werden, um den Namen (als String) zu erhalten, der der Schriftart gegeben wurde, als sie als Asset zum GameMaker Asset Browser hinzugefügt wurde. Bitte beachten Sie, dass es sich nur um eine String ist und nicht verwendet werden kann, um die Schrift direkt zu referenzieren - dazu bräuchte man den Schriftindex. Sie können diese Zeichenkette jedoch verwenden, um den Schriftindex zu erhalten, indem Sie die zurückgegebene Zeichenkette zusammen mit der Funktion asset_get_index().

Syntax:

font_get_name(ind);

ArgumentTyp Beschreibung
ind Index of the font to check.

Rückgabe:

Beispiel:

fnt_Name = font_get_name(font0);

Die obige code ermittelt den Namen einer Schriftart-Ressource, wie er im Asset-Browser erscheint, und speichert ihn als String in der Variablen "fnt_Name".
