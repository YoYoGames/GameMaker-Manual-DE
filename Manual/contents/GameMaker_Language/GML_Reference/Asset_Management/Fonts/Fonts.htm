Schriftarten

GameMaker verfügt über eine Reihe von Funktionen, die verwendet werden können, um Informationen über die enthaltenen Schriftarten zu erhalten, sowie über eine Reihe weiterer Funktionen, die verwendet werden können, um Schriftarten zur Laufzeit zu ändern, hinzuzufügen oder aus dem Spiel zu entfernen. Insbesondere die Funktion font_add() kann verwendet werden, um Nicht-Standard-Schriften, wie z. B. asiatische Glyph-Schriften, aus einer *.ttf oder *.otf Datei hinzuzufügen.

Wenn Sie Ihrem Spiel eine Schriftart hinzufügen, die nicht über die Glyphen verfügt, die für den zu schreibenden Text erforderlich sind, wird das Unicode-Zeichen 9647 (▯) verwendet, um die fehlenden Glyphen zu ersetzen, wenn der Text im Draw-Event gerendert wird. Wenn Ihre Schriftart also zum Beispiel das º-Symbol nicht enthält, wird beim Schreiben von 90º tatsächlich 90▯ erzeugt.

Die folgenden Funktionen können verwendet werden, um Informationen über Schriftressourcen zu erhalten:

  • font_get_name
  • font_get_fontname
  • font_get_first
  • font_get_last
  • font_get_italic
  • font_get_bold
  • font_get_size
  • font_get_texture
  • font_get_uvs
  • Schriftart_holen_Info

Mit den folgenden Funktionen können Sie die Schriftarten während einer Partie ändern und manipulieren:

  • font_exists
  • font_set_cache_size
  • font_add_enable_aa
  • font_add_get_enable_aa
  • font_add
  • font_add_sprite
  • font_add_sprite_ext
  • font_replace
  • font_replace_sprite
  • font_replace_sprite_ext
  • font_delete
  • font_texture_page_size
