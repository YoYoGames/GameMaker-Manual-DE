erweiterung_get_options

Diese Funktion gibt eine Struktur zurück, die alle Optionen und ihre Werte für die Erweiterung mit dem angegebenen Namen enthält.

Die Funktion gibt undefiniert zurück, wenn der angegebene Erweiterungsname ungültig ist.

Syntax:

extension_get_options(extension_name);

Argument Typ Beschreibung
extension_name Zeichenfolge The name of your extension asset as a string

Rückgabe:

Struct

Beispiel:

var _options = extension_get_options("MyExtension");
var _enabled = _options.enabled;

Diese code ruft die Optionsstruktur für eine Erweiterung ab und liest einen Wert daraus.
