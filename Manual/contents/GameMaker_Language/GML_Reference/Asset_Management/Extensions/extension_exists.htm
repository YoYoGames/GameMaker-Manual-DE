erweiterung_existiert

Diese Funktion gibt an, ob eine Erweiterung mit dem angegebenen Namen existiert(true) oder nicht(false).

Syntax:

extension_exists(extension_name);

Argument Typ Beschreibung
extension_name Zeichenfolge The name of your extension asset as a string

Rückgabe:

Boolean

Beispiel:

if (extension_exists("MyExtension"))
{
    var _options = extension_get_options("MyExtension");
    var _enabled = _options.enabled;
}

Diese code prüft, ob die MyExtension-Erweiterung existiert, und wenn ja, erhält sie ihre Optionen als Struktur und liest einen Wert daraus.
