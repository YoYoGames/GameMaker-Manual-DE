schicht_sprite_get_speed

Diese Funktion kann verwendet werden, um den aktuellen Geschwindigkeitsmultiplikatorwert des Elements sprite zu ermitteln. Sie geben die sprite Element-ID an (die Sie erhalten, wenn Sie ein sprite Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()), und die Funktion gibt einen realen Wert zurück, der den Geschwindigkeitsmultiplikator darstellt, der für die Animation von sprite verwendet wird. Der Standardwert ist 1.

Syntax:

layer_sprite_get_speed(sprite_element_id);

ArgumentTyp Beschreibung
sprite_element_id The unique ID value of the sprite element to get the information from

Rückgabe:

Beispiel:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_speed(spr_id) > 0
{
    layer_sprite_speed(spr_id, 0);
}

Die obige code ermittelt die Ebenen-ID für die Ebene "sprite_sky" und verwendet diese dann, um die ID des Elements sprite auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um die Animationsgeschwindigkeit für das Element zu überprüfen, und wenn sie größer als 0 ist, wird sie auf 0 gesetzt.
