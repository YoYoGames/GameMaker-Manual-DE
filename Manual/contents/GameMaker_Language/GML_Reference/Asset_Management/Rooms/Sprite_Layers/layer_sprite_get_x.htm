schicht_sprite_get_x

Diese Funktion kann verwendet werden, um die x-Position des Elements sprite im Raum zu ermitteln. Sie geben die sprite Element-ID an (die Sie erhalten, wenn Sie ein sprite Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()), und die Funktion gibt den Wert der X-Position zurück.

Syntax:

layer_sprite_get_x(sprite_element_id);

ArgumentTyp Beschreibung
sprite_element_id The unique ID value of the sprite element to get the information from

Rückgabe:

Beispiel:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_x(spr_id) < 0)
{
    layer_sprite_x(spr_id, room_width);
}

Die obige code ermittelt die Ebenen-ID für die Ebene "sprite_sky" und verwendet diese dann, um die ID des Elements sprite auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um die x-Position des Elements zu überprüfen, und wenn sie kleiner als 0 ist, wird das Ebenenelement an eine andere x-Position verschoben.
