layer_sprite_create

Mit dieser Funktion können Sie einem Layer, der in Ihrem Projekt verwendet werden soll, eine sprite Ressource zuweisen. Sie geben die ID der Ebene an (die Sie erhalten, wenn Sie die Ebene mit layer_create() oder wenn Sie den Ebenennamen zusammen mit layer_get_id()), eine Position innerhalb des Raums und einen sprite -Index (das ist der Name der sprite, wie er im Asset-Browser angezeigt wird), und die Ressource wird der Ebene hinzugefügt. Die Funktion gibt den eindeutigen ID-Wert für das Element zurück, der dann in weiteren Ebenenfunktionen für sprites verwendet werden kann.

Syntax:

layer_sprite_create(layer_id, x, y, sprite)

Argument Typ Beschreibung
layer_id String oder Ebenen-ID The unique ID value of the layer to target
x Real The x position to use
y Real The y position to use
sprite Sprite-Asset The sprite index to be used

Rückgabe:

Sprite Element ID

Beispiel:

global.asset_layer = layer_create(10000);
for (var i = 0; i< 10; i++;)
{
    var _x = random(room_width);
    var _y = room_height - 100;
    global.asset_spr_trees[i] = layer_sprite_create(global.asset_layer, _x, _y, spr_Trees);
}

Die obige code erstellt eine neue Ebene und fügt ihr dann 10 neue sprite Elemente hinzu, wobei die ID jedes Elements in einem Array gespeichert wird.
