layer_sprite_get_id

Diese Funktion kann verwendet werden, um den eindeutigen ID-Wert eines sprite Elements auf einer Ebene abzurufen. Sie geben die ID der Ebene an (die Sie erhalten, wenn Sie die Ebene mit layer_create() oder wenn Sie den Ebenennamen zusammen mit layer_get_id()) und den Namen des Elements sprite, wie er im Raumeditor definiert ist. Die Funktion gibt den ID-Wert zurück, der mit dem Element sprite auf der Ebene verbunden ist.

Beachten Sie, dass diese Funktion speziell für die Verwendung mit sprites konzipiert ist, die in IDE auf einer Asset-Ebene hinzugefügt wurden. Wenn Sie einer Ebene ein sprite über code mit der Funktion layer_sprite_create()hinzugefügt haben, sollte die von dieser Funktion zurückgegebene ID für alle zukünftigen Referenzen verwendet werden (da das Element sprite keinen Namen hat, der an diese Funktion übergeben werden kann).

Wenn die angegebene Ebene nicht existiert oder das angegebene Element sprite nicht gefunden werden kann, gibt die Funktion -1 zurück.

Syntax:

layer_sprite_get_id(layer_id, sprite_element_name)

Argument Typ Beschreibung
layer_id String oder Ebenen-ID The unique ID value of the layer to target
sprite_element_name Zeichenfolge The unique name of the sprite element on the layer as defined in the Room Editor

Rückgabe:

Sprite Element ID

Beispiel:

var lay_id = layer_get_id("Assets_trees");
var back_id = layer_sprite_get_id(lay_id, "graphic_254367CB");
layer_sprite_change(back_id, spr_Trees_Winter);

Die obige code ruft die Ebenen-ID für die Ebene "Assets_trees" ab und verwendet diese dann, um die ID des sprite Elements "graphic_254367CB" auf dieser Ebene abzurufen. Die abgerufene sprite Element-ID wird dann verwendet, um die sprite des Elements zu ändern.
