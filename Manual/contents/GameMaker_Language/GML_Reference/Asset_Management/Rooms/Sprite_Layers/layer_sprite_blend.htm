layer_sprite_blend

Diese Funktion steuert die Überblendung (oder "Tönung") von sprite sprite. Der Standardwert ist -1 (was der Konstante c_white entspricht, die ebenfalls verwendet werden kann). Jeder andere Wert (einschließlich interner Farbkonstanten wie c_red, oder c_aqua) mischt die angegebene Farbe mit der ursprünglichen sprite. Sie geben die sprite Element-ID an (die Sie erhalten, wenn Sie ein sprite Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()), und legen dann die zu verwendende Mischfarbe fest. Unten sehen Sie ein Beispiel für ein sprite, das mit verschiedenen Farben gemischt wurde:

Bitte beachten Sie, dass Sie versuchen sollten, die Überblendung auf der HTML5-Seite platform zu begrenzen (es sei denn, Sie verwenden WebGL), da jedes überblendete sprite separat zwischengespeichert werden muss und daher viele überblendete sprites die Leistung beeinträchtigen können (Sie können die Größe von cache auch mit der Funktion sprite_set_cache_size()).

Syntax:

layer_sprite_blend(sprite_element_id, blend);

ArgumentTyp Beschreibung
sprite_element_id The unique ID value of the sprite element to change
blend The colour to blend with the sprite sprite (default is c_white)

Rückgabe:

Beispiel:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_blend(spr_id, c_gray);

Die obige code ruft den ID-Wert der sprite namens "Clouds" ab, die der Ebene "Asset_sky" zugewiesen ist, und färbt sie dann in eine Farbe ein.
