layer_sprite_get_yscale

Diese Funktion kann verwendet werden, um den aktuellen Skalenmultiplikator entlang der y-Achse für das Element sprite zu ermitteln. Sie geben die sprite Element-ID an (die Sie erhalten, wenn Sie ein sprite Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()), und die Funktion gibt einen realen Wert für die y-Skala zurück, wobei 1 keine Skalierung bedeutet.

Syntax:

layer_sprite_get_yscale(sprite_element_id);

ArgumentTyp Beschreibung
sprite_element_id The unique ID value of the sprite element to get the information from

Rückgabe:

Beispiel:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_yscale(spr_id) != 1)
{
    layer_sprite_yscale(spr_id, 1);
}

Die obige code ermittelt die Ebenen-ID für die Ebene "sprite_sky" und verwendet diese dann, um die ID des Elements sprite auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um den y-Skalenwert zu überprüfen, und wenn er einen anderen Wert als 1 hat, wird er auf 1 gesetzt.
