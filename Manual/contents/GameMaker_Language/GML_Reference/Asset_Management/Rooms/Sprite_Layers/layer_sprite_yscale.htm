layer_sprite_yscale

Mit dieser Funktion können Sie festlegen, ob das angegebene sprite Element auf einer Ebene entlang der y-Achse skaliert werden soll oder nicht. Sie geben die sprite Element-ID an (die Sie erhalten, wenn Sie ein sprite Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()), und legen dann den Skalierungswert fest. Ein Skalierungswert von 1 bedeutet keine Skalierung (1:1), kleinere Werte skalieren nach unten (0,5 zum Beispiel halbiert die Höhe des verwendeten sprite ), größere Werte skalieren nach oben, und negative Werte spiegeln das sprite und skalieren es, es sei denn, der verwendete Wert ist genau -1 (in diesem Fall wird das verwendete sprite einfach von oben nach unten um seine (0, 0) Position gespiegelt, ohne Skalierung).

Syntax:

layer_sprite_yscale(sprite_element_id, yscale);

ArgumentTyp Beschreibung
sprite_element_id The unique ID value of the sprite element to change
yscale The yscale value (default is 1)

Rückgabe:

Beispiel:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1
{
    layer_sprite_xscale(asset_sprite, 1);
    layer_sprite_yscale(asset_sprite, 1);
}

Das obige code prüft das Element sprite, das der Ebene zugewiesen ist, auf der die Instanz code läuft, und wenn es in eine der beiden Richtungen skaliert ist, werden sowohl die Skalierung der x-Achse als auch die der y-Achse auf 1 gesetzt.
