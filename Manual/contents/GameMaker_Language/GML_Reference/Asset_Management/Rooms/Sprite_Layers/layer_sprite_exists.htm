layer_sprite_exists

Mit dieser Funktion können Sie prüfen, ob ein Element sprite auf einer bestimmten Ebene existiert. Sie geben die Layer-ID (die Sie beim Erstellen des Layers mit layer_create()) oder den Ebenennamen (als String - dies wirkt sich auf die Leistung aus) und die sprite Element-ID (die Sie erhalten, wenn Sie ein sprite Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()) und die Funktion gibt entweder true zurück, wenn das Element existiert, oder false, wenn es nicht existiert.

HINWEIS: Diese Funktion arbeitet innerhalb des Bereichs des aktuellen Zielraums - standardmäßig der Raum, in dem die Funktion aufgerufen wird -, der mit der Funktion layer_set_target_room().

Syntax:

layer_sprite_exists(layer_id, sprite_element_id)

Argument Typ Beschreibung
layer_id String oder Ebenen-ID The unique ID value of the layer to target (or the layer name as a string)
sprite_element_id Sprite-Element-ID The unique ID value of the sprite element to check

Rückgabe:

Boolean

Beispiel:

var lay_id = layer_get_id("Asset_Trees");
if (layer_sprite_exists(lay_id, global.TreesSprites))
{
    layer_sprite_destroy(lay_id, global.TreesSprites);
}

Die obige code prüft die Ebene "Asset_trees", um zu sehen, ob das angegebene Element sprite existiert, und wenn ja, wird es zerstört (aber nicht die Ebene).
