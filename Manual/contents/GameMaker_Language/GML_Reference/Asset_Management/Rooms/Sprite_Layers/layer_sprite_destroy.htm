layer_sprite_destroy

Diese Funktion zerstört das angegebene sprite Element. Sie geben die sprite ID an (die Sie erhalten, wenn Sie das sprite mit layer_sprite_create() oder wenn Sie die Ebenen-ID zusammen mit layer_get_sprite_id()) und diese Funktion entfernt sie. Beachte, dass dies nicht die Ebene entfernt, sondern nur die sprite, und wenn die sprite eine ist, die im Raum-Editor hinzugefügt wurde, dann wird das nächste Mal, wenn du den Raum verlässt und dann zurückkehrst, die sprite wieder neu erstellt. Wenn der Raum jedoch persistent ist, wird die sprite entfernt, es sei denn, die Raumpersistenz wird wieder ausgeschaltet.

Syntax:

layer_sprite_destroy(sprite_element_id)

ArgumentTyp Beschreibung
sprite_element_id The unique ID value of the sprite to be destroyed

Rückgabe:

Beispiel:

var lay_id = layer_get_id("Asset_Trees");
var spr_id = layer_sprite_get_id(lay_id, "graphic_254367CB");

if (layer_sprite_exists(lay_id, spr_id))
{
    layer_sprite_destroy(lay_id);
}

Die obige code prüft die Ebene "Asset_Trees", um zu sehen, ob das angegebene Element sprite existiert, und wenn ja, wird es zerstört (aber nicht die Ebene).
