Sprite-Ebenen

Mit dem GameMaker-Raumeditor können Sie sprites mithilfe von Ebenen in jeden beliebigen Raum einfügen. Wie der Name schon sagt, ist eine Ebene einfach eine Sammlung von Elementen, die sich alle in der gleichen Tiefe innerhalb des Raums befinden. Sie können auch bestimmte Aspekte der Ebenen von code aus steuern, Dinge hinzufügen oder entfernen oder bestimmte Eigenschaften der Ebene oder ihrer Inhalte zur Laufzeit ändern. Nachfolgend finden Sie eine Liste aller Funktionen, die zum Bearbeiten von Sprites verwendet werden können, die zu einer Ebene Asset-Ebene:
