layer_sprite_get_alpha

Diese Funktion kann verwendet werden, um den Alphawert des Elements sprite zu ermitteln. Sie geben die sprite Element-ID an (die Sie erhalten, wenn Sie ein sprite Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()), und die Funktion gibt einen Wert zwischen 0 (völlig transparent) und 1 (völlig undurchsichtig) zurück.

Syntax:

layer_sprite_get_alpha(sprite_element_id);

ArgumentTyp Beschreibung
sprite_element_id The unique ID value of the sprite element to get the information from

Rückgabe:

(from 0 to 1)

Beispiel:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_alpha(spr_id) < 0.1)
{
    layer_sprite_destroy(spr_id);
}

Die obige code ermittelt die Ebenen-ID für die Ebene "sprite_sky" und verwendet diese dann, um die ID des Elements sprite auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um den Alphawert des Elements zu prüfen, und wenn er kleiner als 0,1 ist, wird das Ebenenelement zerstört.
