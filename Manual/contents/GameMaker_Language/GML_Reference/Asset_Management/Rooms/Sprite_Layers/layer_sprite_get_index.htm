layer_sprite_get_index

Diese Funktion kann verwendet werden, um den aktuellen Bildindexwert des Elements sprite zu ermitteln. Sie geben die sprite Element-ID an (die Sie erhalten, wenn Sie ein sprite Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()), und die Funktion gibt einen realen Wert zurück, der den Bildindex darstellt, der für das sprite-Element angezeigt wird. Die Funktion gibt -1 zurück, wenn entweder das sprite -Element nicht existiert oder dem Element kein gültiges sprite zugewiesen ist.

Syntax:

layer_sprite_get_index(sprite_element_id);

ArgumentTyp Beschreibung
sprite_element_id The unique ID value of the sprite element to get the information from

Rückgabe:

Real or -1

Beispiel:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_index(spr_id) < 4)
{
    layer_sprite_index(spr_id, 4);
}

Die obige code ermittelt die Ebenen-ID für die Ebene "sprite_sky" und verwendet diese dann, um die ID des Elements sprite auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um zu prüfen, ob der Bildindex für das Element kleiner als 4 ist, und wenn ja, wird er auf 4 gesetzt.
