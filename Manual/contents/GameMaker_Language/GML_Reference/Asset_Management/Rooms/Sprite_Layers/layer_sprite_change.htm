layer_sprite_change

Mit dieser Funktion können Sie die sprite Ressource ändern, die einem bestimmten sprite Element auf einer Ebene zugeordnet ist. Sie geben die sprite Element-ID an (die Sie erhalten, wenn Sie ein sprite Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()), und geben dann ein sprite Indexelement an, das geändert werden soll. Beachten Sie, dass, wenn sprite Unterbilder hat, auch diese animiert werden (dies kann mit den Optionen layer_sprite_index() und layer_sprite_speed() Funktionen gesteuert werden). Beachten Sie, dass Sie einen Wert von -1 als neuen sprite -Index zuweisen können und kein sprite angezeigt wird, obwohl das Element weiterhin existiert und später erneut geändert werden kann.

Syntax:

layer_sprite_change(sprite_element_id, sprite_index)

Argument Typ Beschreibung
sprite_element_id Sprite-Element-ID The unique ID value of the sprite element to change
sprite_index Sprite-Asset The new sprite index to use

Rückgabe:

N/A

Beispiel:

var s = layer_sprite_get_id(layer, global.Asset_sprite);
if (layer_sprite_get_sprite(s) != spr_Nighttime)
{
    layer_sprite_change(s, spr_nighttime);
}

Die obige code prüft den Index sprite des Elements mit der in der globalen Variablen global.Asset_sprite gespeicherten ID, und wenn diese nicht "spr_Nighttime" ist, wird sprite dem Element zugewiesen.
