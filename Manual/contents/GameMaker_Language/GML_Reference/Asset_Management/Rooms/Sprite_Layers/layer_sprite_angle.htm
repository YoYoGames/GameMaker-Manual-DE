layer_sprite_angle

Mit dieser Funktion können Sie den Winkel für das angegebene sprite Element auf einer Ebene ändern. Sie geben die sprite Element-ID an (die Sie erhalten, wenn Sie ein sprite Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()), und stellen Sie dann den Winkelwert ein, von 0 bis 359 gegen den Uhrzeigersinn, wobei 0 nach rechts, 90 nach oben, 180 nach links und 270 nach unten zeigt. Wenn Sie einen Wert größer als 360 einstellen, wird dieser in einer Schleife in den Bereich von 0 bis 359 gebracht.

Syntax:

layer_sprite_angle(sprite_element_id, angle);

Argument Typ Beschreibung
sprite_element_id Sprite-Element-ID The unique ID value of the sprite element to change
angle Real The angle of the sprite (default is 0)

Rückgabe:

N/A

Beispiel:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if (layer_sprite_get_angle(asset_sprite) != 0)
{
    layer_sprite_angle(asset_sprite, 0);
}

Das obige code prüft das Element sprite, das der Ebene zugewiesen ist, auf der die Instanz code läuft, und wenn sein Winkel nicht 0 ist, wird er auf 0 gesetzt.
