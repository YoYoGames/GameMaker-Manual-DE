schicht_sprite_geschwindigkeit

Mit dieser Funktion kann die Geschwindigkeit eines animierten Bildes eingestellt werden, das einem sprite Element zugewiesen wurde. Sie geben die sprite Element-ID an (die Sie erhalten, wenn Sie ein sprite Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()), und legen dann die Animationsgeschwindigkeit fest. Wenn IDE die Animationsgeschwindigkeit für sprite als Frames Per Game Frame (Frames pro Spielframe) festgelegt hat, dann ist der angegebene Geschwindigkeitswert ein Multiplikator, wobei 1 der Standardwert ist und eine Einstellung von 0,5 die Animationsgeschwindigkeit halbiert, während eine Einstellung von 1,5 die Geschwindigkeit verdoppelt. Wenn jedoch IDE die Animationsgeschwindigkeit für sprite so eingestellt hat, dass sie als Frames pro Sekunde berechnet wird, dann ist der angegebene Geschwindigkeitswert ein buchstäblicher Wert für Frames pro Sekunde. Wenn Sie diesen Wert also zum Beispiel auf 5 setzen, wird die Animation auf loop in jeder Sekunde, in der das Spiel läuft, 5 Frames anzeigen. Wenn das verwendete sprite keine Unterbilder hat, hat diese Funktion keine Wirkung.

Syntax:

layer_sprite_speed(sprite_element_id, speed);

Argument Typ Beschreibung
sprite_element_id Sprite-Asset The unique ID value of the sprite element to change
speed Real The speed multiplier/value to use for the element

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Assets_water");
var spr_id = layer_sprite_get_id(lay_id, "gfc_water");
layer_sprite_speed(spr_id, 0.5);

Die obige code ruft die Ebenen-ID für die Ebene "Assets_water" ab und verwendet diese dann, um die ID des angegebenen sprite Elements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um die Bildgeschwindigkeit des Elements zu ändern.
