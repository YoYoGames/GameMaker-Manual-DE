schicht_sprite_x

Diese Funktion steuert die Position entlang der X-Achse des Raums des Elements sprite auf der Ebene. Sie geben die sprite Element-ID an (die Sie erhalten, wenn Sie ein sprite Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()), und legen dann den zu verwendenden x-Wert fest (basierend auf den Raumkoordinaten).

Syntax:

layer_sprite_x(sprite_element_id, x);

ArgumentTyp Beschreibung
sprite_element_id The unique ID value of the sprite element to change
x The x position for the sprite

Rückgabe:

Beispiel:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_x(spr_id, irandom(room_width));

Die obige code ruft den ID-Wert des sprite -Assets "Clouds" ab, das der Ebene "Asset_sky" zugewiesen ist, und setzt dann seine x-Position auf einen Zufallswert zwischen 0 und der Breite des Raums.
