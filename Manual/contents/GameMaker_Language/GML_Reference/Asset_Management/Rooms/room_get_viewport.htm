room_get_viewport

Mit dieser Funktion können Sie die Details eines View Ports in einem anderen Raum als dem aktuellen abrufen. Sie geben die Raum-ID und den Index des abzurufenden View Ports an (von 0 bis 7) und die Funktion gibt ein Array mit 5 Indizes zurück, wobei:

  • [0] = sichtbar (Boolescher Wert: true / false)
  • [1] = x-Position (real)
  • [2] = y-Position (real)
  • [3] = Breite (real)
  • [4] = Höhe (real)

Syntax:

room_get_viewport(rm, vind);

Argument Typ Beschreibung
rm Zimmer Asset The index of the room to get viewport data from
vind Real The index of the view port to get

Rückgabe:

Array (5 elements: visible, x, y, width, height)

Beispiel:

v_vals = room_get_viewport(rm_Game, 0);

if v_vals[0] == false
{
    room_set_view(rm_Game, true, v_vals[1], v_vals[2], v_vals[3], v_vals[4]);
}

Die obige code ruft die Daten der Sichtfenster für den angegebenen Raum ab und prüft dann, ob das Fenster als sichtbar gekennzeichnet ist. Ist dies nicht der Fall, werden die Daten des Anschlusses so eingestellt, dass er sichtbar wird.
