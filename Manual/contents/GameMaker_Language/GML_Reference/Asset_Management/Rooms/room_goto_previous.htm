raum_zur_vorherigen

Mit dieser Funktion können Sie Ihr Spiel zum vorherigen Raum springen lassen, der zum Zeitpunkt der Kompilierung des Spiels im Raum-Manager aufgeführt war. Wenn dieser Raum nicht existiert, wird ein Fehler ausgelöst und das Spiel wird gezwungen, sich zu beenden. Beachte, dass der Raum erst am Ende des Ereignisses, in dem die Funktion aufgerufen wurde, geändert wird, so dass code nach dem Aufruf dieser Funktion immer noch ausgeführt wird, wenn es sich um dasselbe Ereignis handelt. Diese Funktion löst auch das Ereignis Raumende aus.

Syntax:

room_goto_previous();

Rückgabe:

Beispiel:

if room_exists(room_previous(room))
{
    room_goto_previous();
}

Das obige code prüft, ob es einen anderen Raum vor dem aktuellen gibt, und wenn ja, geht es zu diesem Raum.
