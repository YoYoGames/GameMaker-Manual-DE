layer_get_fx

Diese Funktion wird verwendet, um die FX-Struktur für eine Ebene abzurufen. Sie geben entweder die ID oder den Namen der Ebene an, auf die Sie abzielen wollen, und die Funktion gibt eine Struktur mit Informationen über den angewandten Effekt zurück. Diese Struktur ähnelt der Struktur, die Sie von fx_create() erhalten, und den Funktionen fx_get_parameter/s und fx_set_parameter/s können zum Lesen und Ändern der Parameter verwendet werden.

Wenn auf die angegebene Ebene keine Filter/Effekte angewendet wurden, gibt die Funktion -1 zurück.

Syntax:

layer_get_fx(layer_name_or_id);

Argument Typ Beschreibung
layer_name_or_id String oder Ebenen-ID The name or ID of the layer to read

Rückgabe:

FX Struct (or -1 if not found)

Beispiel:

var layers = layer_get_all();
for(var i = 0; i < array_length(layers); i ++)
{    
    var layer_fx = layer_get_fx(layers[i]);
    
    if (layer_fx != -1)
    {
        if (fx_get_name(layer_fx) == "_filter_tintfilter")
        {            
            fx_set_parameter(_fx_tint, "g_TintCol", [0, 0, 1, 1]);
        }
    }
}

Die obige code führt eine for Schleife durch alle im Raum vorhandenen Ebenen und überprüft jede Ebene auf eine FX-Struktur. Wenn eine Ebene eine FX-Struktur hat, prüft sie den Namen des Filters/Effekts, indem sie fx_get_name() aufruft; wenn er gleich "_filter_tintfilter" ist, was bedeutet, dass es sich um einen "Farbton"-Filter handelt, ändert er seine Farbtonfarbe in Blau.
