layer_set_fx

Diese Funktion wird verwendet, um eine FX-Struktur einem Raum-Layer zuzuweisen. Sie geben entweder die ID oder den Namen des Layers an, den Sie ändern möchten, und dann eine FX-Struktur (wie von fx_create() oder layer_get_fx() zurückgegeben), die auf den angegebenen Layer angewendet wird und den Effekt im Raum sichtbar macht.

Wenn Sie eine Ebene angeben, die keine Filter-/Effekt-Ebene ist, sondern von einem anderen Typ (z. B. Instanz-Ebene, Asset-Ebene, Kachelebene usw.), wird der Effekt auf den Inhalt dieser Ebene sowie auf alle darunter liegenden Ebenen angewendet.

Wenn der angegebenen Ebene bereits eine FX-Struktur zugewiesen ist, wird sie durch die neue ersetzt.

HINWEIS Der Aufruf dieser Funktion auf der Stammebene eines script bewirkt nichts, da scripts ausgeführt wird, bevor der erste Raum geladen wird. Sie können zuerst einen Zielraum festlegen und dann diese Funktion aufrufen.

Syntax:

layer_set_fx(layer_name_or_id, filter_or_effect);

Argument Typ Beschreibung
layer_name_or_id Zeichenfolge oder Ebene ID The name or ID of the layer to modify
filter_or_effect FX Struktur The FX struct to apply to the layer

Rückgabe:

N/A

Beispiel:

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

Die obige code erstellt eine neue FX-Struktur aus dem Effekt "_filter_tintfilter", dem "Colour Tint"-Filter aus dem Raumeditor. Er weist seinem Parameter "g_TintCol" einen Wert zu, der die Farbe des Farbtons darstellt, und da es sich intern um vec4 handelt, nimmt er ein Array mit vier Werten (entsprechend den Rot-, Grün-, Blau- und Alpha-Werten). Die FX-Struktur für den Farbton wird dann auf eine FX-Ebene angewendet.
