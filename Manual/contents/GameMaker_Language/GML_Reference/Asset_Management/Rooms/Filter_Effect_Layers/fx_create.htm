fx_create

Mit dieser Funktion können Sie eine neue FX-Struktur für einen bestimmten Effekttyp erstellen, die dann auf eine Ebene angewendet werden kann.

Die Namen der Filter/Effekte, die Sie an diese Funktion übergeben können, sind auf dieser Seite aufgeführt: Alle Filter-/Effekttypen.

Verwendung

Sie können die Parameter für den erzeugten Effekt lesen, indem Sie fx_get_parameter/s für die zurückgegebene Struktur lesen und ihnen neue Werte zuweisen, indem Sie fx_set_parameter/s aufrufen.

Beachten Sie, dass die von dieser Funktion zurückgegebene Struktur keine sichtbaren Mitglieder enthält und nur durch eine der Filter-/Effektfunktionen geändert werden kann.

Funktionsausfälle

Die Funktion gibt -1 zurück, wenn die Erstellung der Struktur aus irgendeinem Grund fehlgeschlagen ist; dies kann passieren, wenn der angegebene FX-Name falsch war oder der FX nicht in das Spielpaket geladen wurde.

Um sicherzustellen, dass diese Funktion richtig funktioniert, stellen Sie sicher, dass der benötigte Effekt in Ihr Spielpaket geladen ist, wie auf der Seite Filter- und Effektfunktionen beschrieben:

Wenn diese Bedingungen nicht erfüllt sind, d. h. wenn Sie kein Stringliteral übergeben und der Filter/Effekt in keinem Raum vorhanden ist, kann die Funktion diesen Filter/Effekt nicht laden und gibt -1 zurück.

Syntax:

fx_create(filter_or_effect_name);

Argument Typ Beschreibung
filter_or_effect_name Zeichenfolge The name of the filter/effect to create (get them from All Filter/Effect Types).

Rückgabe:

FX Struct or -1 (on failure)

Beispiel:

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

Die obige code erstellt eine neue FX-Struktur aus dem Effekt "_filter_tintfilter", dem "Colour Tint"-Filter aus dem Raumeditor. Er weist seinem Parameter "g_TintCol" einen Wert zu, der die Farbe des Farbtons darstellt, und da es sich intern um vec4 handelt, nimmt er ein Array mit vier Werten (entsprechend den Rot-, Grün-, Blau- und Alpha-Werten). Die FX-Struktur für den Farbton wird dann auf eine FX-Ebene angewendet.
