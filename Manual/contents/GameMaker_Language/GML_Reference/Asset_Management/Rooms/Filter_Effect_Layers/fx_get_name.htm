fx_get_name

Diese Funktion wird verwendet, um den Namen eines Filters/Effekts aus seiner Struktur abzurufen. Sie geben die zu lesende FX-Struktur an (wie von fx_create() oder layer_get_fx() zurückgegeben) und die Funktion gibt den Namen als String zurück. Dieser Name kann dann an fx_create() übergeben werden, um eine neue FX-Struktur mit demselben Filter-/Effekttyp zu erstellen.

Syntax:

fx_get_name(filter_or_effect);

ArgumentTyp Beschreibung
filter_or_effect The FX struct to get the name of

Rückgabe:

Beispiel:

var layers = layer_get_all();
for(var i = 0; i < array_length(layers); i ++)
{    
    var layer_fx = layer_get_fx(layers[i]);
    
    if (layer_fx != -1)
    {
        if (fx_get_name(layer_fx) == "_filter_tintfilter")
        {            
            fx_set_parameter(_fx_tint, "g_TintCol", [0, 0, 1, 1]);
        }
    }
}

Die obige code führt eine for Schleife durch alle im Raum vorhandenen Ebenen und prüft jede Ebene auf eine FX-Struktur. Wenn eine Ebene eine FX-Struktur hat, prüft sie den Namen des Filters/Effekts, indem sie fx_get_name() aufruft; wenn er gleich "_filter_tintfilter" ist, was bedeutet, dass es sich um einen "Colour Tint"-Filter handelt, ändert sie die Farbtonfarbe in Blau.
