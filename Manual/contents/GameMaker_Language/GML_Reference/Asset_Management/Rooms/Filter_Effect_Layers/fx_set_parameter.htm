fx_set_parameter

Diese Funktion wird verwendet, um den Wert eines Parameters in der angegebenen FX-Struktur zu ändern. Sie geben eine FX-Struktur (wie von fx_create() oder layer_get_fx() zurückgegeben), einen Parameternamen als String und den neuen Wert an. Dadurch werden Ihre Änderungen in der Ebene sichtbar, der die FX-Struktur zugewiesen ist.

Sie können mehrere arguments für den Parameter übergeben, falls es sich um eine vector handelt, Sie können aber auch einfach ein Array als einziges argument angeben, das Ihre mehreren Werte enthalten würde.

Auf dieser Seite finden Sie die Parameternamen für einen Filter/Effekt: FX-Typen & Parameter.

Syntax:

fx_set_parameter(filter_or_effect, parameter_name, val, ...);

Argument Typ Beschreibung
filter_or_effect FX Struktur The FX struct to modify
parameter_name Zeichenfolge The name of the parameter to change the value of (as a string)
val... Echt oder Array The new value(s) for that parameter as a real or array

Rückgabe:

N/A

Beispiel:

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

Die obige code erstellt eine neue FX-Struktur aus dem Effekt "_filter_tintfilter", dem "Colour Tint"-Filter aus dem Raumeditor. Er weist seinem Parameter "g_TintCol" einen Wert zu, der die Farbe des Farbtons darstellt, und da es sich intern um vec4 handelt, nimmt er ein Array mit vier Werten (entsprechend den Rot-, Grün-, Blau- und Alpha-Werten). Die FX-Struktur für den Farbton wird dann auf eine FX-Ebene angewendet.

Beachten Sie, dass Sie auch in der Lage sind, Werte an vector als separate arguments anstelle eines Arrays zu übergeben:

fx_set_parameter(_fx_tint, "g_TintCol", 1, 0, 0, 1);
