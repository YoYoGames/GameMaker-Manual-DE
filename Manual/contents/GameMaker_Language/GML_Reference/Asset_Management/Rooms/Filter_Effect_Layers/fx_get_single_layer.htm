fx_get_single_layer

Diese Funktion gibt zurück, ob der gelieferte Filter/Effekt den Single-Layer-Modus (true) verwendet oder nicht (false). Siehe fx_set_single_layer() für Informationen über den Ein-Ebenen-Modus.

Sie geben eine FX-Struktur an (wie von fx_create() oder layer_get_fx() zurückgegeben) und die Funktion gibt einen booleschen Wert zurück, der angibt, ob der Single-Layer-Modus für den FX aktiviert ist oder nicht.

Syntax:

fx_get_single_layer(filter_or_effect);

ArgumentTyp Beschreibung
filter_or_effect The FX struct to read

Rückgabe:

Boolean

Beispiel:

var _fx = layer_get_fx("Distort");

if (!fx_get_single_layer(_fx))
{
    fx_set_single_layer(_fx, true);
}

Die obige code holt sich die FX-Struktur von einer Ebene namens "Verzerren" und prüft, ob der Ein-Ebenen-Modus für diese Ebene aktiviert ist. Ist dies nicht der Fall, wird der Ein-Ebenen-Modus für diesen Effekt aktiviert.
