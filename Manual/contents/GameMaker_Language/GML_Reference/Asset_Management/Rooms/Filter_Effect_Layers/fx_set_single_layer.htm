fx_set_single_layer

Diese Funktion wird verwendet, um den Modus "Einzelne Ebene" für einen Filter/Effekt zu aktivieren oder zu deaktivieren. Standardmäßig wird ein Filter/Effekt auf die Ebene, der er zugewiesen ist, und auf alle Ebenen darunter angewendet. Sie können jedoch den Modus "Einzelne Ebene" für eine Effektstruktur aktivieren, um sicherzustellen, dass er nur auf die Ebene angewendet wird, der er zugewiesen ist.

Sie geben eine FX-Struktur (wie von fx_create() oder layer_get_fx() zurückgegeben) und einen booleschen Wert an, um den Single-Layer-Modus zu aktivieren (true) oder zu deaktivieren (false).

Die folgende Abbildung zeigt einen Filter, der mit deaktiviertem Einzelebenenmodus verwendet wird (dies ist das Standardverhalten für alle FX-Ebenen), und denselben Filter mit aktiviertem Einzelebenenmodus:

HINWEIS Wenn Sie den Modus "Einzelne Ebene" für einen Effekt aktivieren, stellen Sie sicher, dass Sie ihn auf eine Ebene anwenden, die keine Effekt-Ebene ist, da eine Effekt-Ebene selbst nichts enthält und daher kein Filter/Effekt sichtbar ist.

Syntax:

fx_set_single_layer(filter_or_effect, enable);

ArgumentTyp Beschreibung
filter_or_effect The FX struct to modify
enable A boolean value to enable (true) oder deaktivieren (false) Einschichtmodus

Rückgabe:

Beispiel:

shake_fx = fx_create("_filter_screenshake");
fx_set_single_layer(shake_fx, true);
layer_set_fx("ShakeyThings", shake_fx);

Die obige code erstellt einen neuen Screenshake-FX, aktiviert den Ein-Ebenen-Modus und wendet ihn dann auf eine Raumebene an. Das bedeutet, dass der Screenshake-Filter nur auf die Ebene "ShakeyThings" angewendet wird.
