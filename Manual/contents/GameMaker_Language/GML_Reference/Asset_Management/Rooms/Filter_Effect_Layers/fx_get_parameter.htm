fx_get_parameter

Diese Funktion wird verwendet, um den Wert eines Parameters aus einer FX-Struktur abzurufen. Sie geben die zu lesende FX-Struktur (wie von fx_create() oder layer_get_fx() zurückgegeben) und den Namen des Parameters als String an, und die Funktion gibt den aktuellen Wert zurück. Dabei kann es sich um einen realen Wert handeln oder um ein Array, wenn der Parameter ein vector ist (eine Gruppe von Werten, wie z.B. vec4, die eine RGBA-Farbe speichert).

Auf dieser Seite finden Sie die Parameternamen für einen Filter/Effekt: FX-Typen & Parameter.

Syntax:

fx_get_parameter(filter_or_effect, parameter_name);

Argument Typ Beschreibung
filter_or_effect FX Struktur The FX struct to get the parameter from
parameter_name Zeichenfolge The name of the parameter to get the value of (as a string)

Rückgabe:

Real or Array

Beispiel:

var _fx_struct = layer_get_fx("TintEffect");
var _tint_colour = fx_get_parameter(_fx_struct, "g_TintCol");
show_debug_message("The currently active tint colour is: " + string(_tint_colour));

Die obige code ruft die FX-Struktur von einer Ebene ab, auf die der Filter "Farbton" angewendet wurde, und ruft den Wert des Parameters "g_TintCol" ab, der ein Array mit den vier Werten für die Farbtonfarbe (Rot, Grün, Blau und Alpha) zurückgibt. Anschließend wird dieser Wert in das Ausgabeprotokoll gedruckt.
