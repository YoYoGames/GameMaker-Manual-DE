layer_clear_fx

Diese Funktion wird verwendet, um die FX-Struktur zu löschen, die auf die angegebene Ebene angewendet wird. Sie geben entweder die ID oder den Namen der Ebene an, die Sie ändern möchten, und diese Funktion entfernt die FX-Struktur, die auf diese Ebene angewendet wird, und schaltet ihre Effekte vollständig aus.

Wenn Sie diese Funktion verwenden, um einen Effekt vorübergehend zu deaktivieren und ihn auf der Ebene wiederherzustellen, sollten Sie zunächst die FX-Struktur für die Ebene durch Aufruf von layer_get_fx() speichern und dann diese Funktion aufrufen, um den Effekt auf der Ebene zu löschen. Später können Sie den Effekt wieder auf die Ebene anwenden, indem Sie layer_set_fx() aufrufen, wodurch der Effekt wieder aktiviert wird.

Syntax:

layer_clear_fx(layer_name_or_id);

Argument Typ Beschreibung
layer_name_or_id Zeichenfolge oder Ebene ID The name or ID of the layer to clear the FX for

Rückgabe:

N/A

Beispiel:

// Disable effect
if (keyboard_check_pressed(ord("E")))
{
    if (layer_get_fx("EffectLayer") != -1)
    {
        temp_fx = layer_get_fx("EffectLayer");
        layer_clear_fx("EffectLayer");
    }
}
// Re-enable effect
else if (keyboard_check_pressed(ord("F")))
{
    if (temp_fx != -1)
    {
        layer_set_fx("EffectLayer", temp_fx);
        temp_fx = -1;
    }
}

Die obige code prüft, ob die Taste "E" gedrückt wird, und prüft dann, ob auf die Ebene "EffectLayer" ein Filter/Effekt angewendet wurde; in diesem Fall speichert sie ihre FX-Struktur in einer temporären Instanzvariablen und löscht die Ebene von allen Filtern/Effekten. Wenn die Taste "F" gedrückt wird, wird geprüft, ob die temporäre Variable einen anderen Wert als -1 enthält, und in diesem Fall wird dieser Effekt wieder auf die Ebene angewendet und die temporäre Variable auf -1 zurückgesetzt. Auf diese Weise können Sie einen Effekt bei Bedarf aktivieren und deaktivieren, ohne dass seine Parameter verloren gehen.
