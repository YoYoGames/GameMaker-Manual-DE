layer_enable_fx

Diese Funktion aktiviert/deaktiviert den Filter/Effekt, der einer Raumebene zugewiesen ist. Sie geben entweder die ID oder den Namen der Ebene an, die Sie ändern möchten, und dann einen booleschen Wert, der angibt, ob der Effekt aktiviert(true) oder deaktiviert(false) werden soll.

Wenn Sie false angeben, wird der Effekt nicht aus der Ebene entfernt, sondern nur unsichtbar gemacht. Verwenden Sie layer_clear_fx(), um einen Effekt aus einer Ebene zu entfernen.

Ebenso bewirkt die Übergabe von true nichts, wenn kein Effekt auf die Ebene angewendet wird. Verwenden Sie layer_set_fx(), um einen Effekt auf eine Ebene anzuwenden.

Syntax:

layer_enable_fx(layer_name_or_id, enable);

Argument Typ Beschreibung
layer_name_or_id Zeichenfolge oder Ebene ID The name or ID of the layer to modify
enable Boolesche Whether to enable or disable the FX

Rückgabe:

N/A

Beispiel:

if (hp <= 3)
{
    layer_enable_fx("DesaturateLayer", true);
}
else
{
    layer_enable_fx("DesaturateLayer", false);
}

Die obige code aktiviert eine Desaturate FX-Ebene, wenn der HP-Wert der Instanz kleiner oder gleich 3 ist, andernfalls wird sie deaktiviert. Dies zeigt dem Spieler an, dass seine HP niedrig sind, indem der Bildschirm entsättigt wird.
