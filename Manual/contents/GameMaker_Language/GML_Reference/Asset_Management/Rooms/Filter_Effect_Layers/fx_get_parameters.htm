fx_get_parameters

Diese Funktion wird verwendet, um eine Struktur abzurufen, die alle Parameter für die angegebene FX-Struktur enthält. Sie geben eine FX-Struktur an (wie von fx_create() oder layer_get_fx() zurückgegeben) und diese Funktion gibt eine Struktur zurück, die Variablen für alle Parameter enthält, die für diesen Filter/Effekt gelten. Parameter, die vectors (d.h. Gruppen von Werten) verwenden, werden als Arrays dargestellt.

Sie können die Namen aller Parameter in einem separaten Array erhalten, indem Sie fx_get_parameter_names() aufrufen, oder sie auf dieser Seite finden: FX-Typen und Parameter.

Wenn Sie Werte in dieser Struktur ändern, werden diese Änderungen nicht auf den Filter/Effekt angewendet; Sie müssen fx_set_parameters() aufrufen, um die geänderte Struktur wieder auf den Filter/Effekt anzuwenden, damit die Änderungen wirksam werden.

Syntax:

fx_get_parameters(filter_or_effect);

Argument Typ Beschreibung
filter_or_effect FX Struktur The FX struct for which the parameters will be returned

Rückgabe:

Struct (or -1 on failure)

Beispiel:

var _fx_struct = layer_get_fx("TintEffect");

if (_fx_struct != -1)
{
    var _params = fx_get_parameters(_fx_struct);
    var _osc = sin(current_time / 1000);
    _params.g_TintCol = [_osc, 0.3 + _osc, 0.6 + _osc, 1];

    fx_set_parameters(_fx_struct, _params);
}

Die obige code ruft die FX-Struktur für eine Ebene ab, auf die der Filter "Farbton" angewendet werden soll, und ruft deren Parameterstruktur durch Aufruf von fx_get_parameters() ab. Danach wird mit Hilfe von sin() und current_time ein oszillierender Wert erzeugt, der dann für die RGBA-Werte des Tönungseffekts verwendet wird, so dass dieser ständig animiert wird. Das RGBA-Array wird der Variable g_TintCol in der Parameter-Struktur zugewiesen, und die Struktur wird dann durch den Aufruf von fx_set_parameters().
