Filter- und Effektfunktionen

Übersicht

Es gibt verschiedene GML Funktionen, die zum Erstellen, Ändern und Entfernen von Filtern/Effekten auf Ebenen innerhalb eines Raums verwendet werden können. So können Sie Effekte einfach in Echtzeit verwalten und realistische und dynamische Filter/Effekte erstellen.

Die letztgenannte Methode (die Angabe des Filters/Effekts in Ihrem code) funktioniert nur mit String-Literalen, die direkt in der Funktion argument angegeben werden. Das bedeutet, wenn Sie eine Variable oder eine andere Logik verwenden, um den Namen des Filters/Effekts zu ermitteln, wird das Asset compiler dies nicht erkennen und der Filter/Effekt wird nicht geladen.

Betrachten Sie die folgenden Beispiele:

// This will work on its own
var _fxshake = fx_create("_filter_screenshake");

// This will NOT work on its own
var _myfilters = { screenshake: "_screenshake" }
var _filter_to_use = "_filter" + _myfilters.screenshake;
var _fxshake = fx_create(_filter_to_use);

Um sicherzustellen, dass die letztere Methode funktioniert, können Sie den Filter einfach zu mindestens einem Raum in Ihrem Projekt hinzufügen oder sicherstellen, dass fx_create() irgendwo in Ihrem Projekt mit dem Filternamen als String-Konstante (und nicht als Variable) aufgerufen wird.

Funktionsliste

Die folgenden Funktionen dienen der Erstellung und Verwaltung von "FX Structs", die Effektdaten enthalten:

  • fx_create
  • fx_get_parameter
  • fx_get_parameters
  • fx_get_name
  • fx_get_parameter_names
  • fx_get_single_layer
  • fx_set_parameter
  • fx_set_parameters
  • fx_set_single_layer

Die folgenden Funktionen dienen der Modifikation von Layern, die Filter/Effekte enthalten können, unter Verwendung von FX-Strukturen:

  • layer_set_fx
  • layer_get_fx
  • layer_clear_fx
  • layer_enable_fx
  • layer_fx_is_enabled

FX zur Laufzeit modifizieren

Sie können Filter/Effekte unter runtime wie folgt ändern:

  • Rufen Sie die FX-Struktur von der Ebene ab, die Sie ändern möchten, indem Sie layer_get_fx()
    • Oder Sie erstellen eine neue FX-Struktur durch Aufruf von fx_create() und wenden sie auf eine Ebene an, indem Sie layer_set_fx()
  • Rufen Sie die Parameterstruktur ab, indem Sie fx_get_parameters()
  • Ändern Sie die Parameter nach Bedarf, indem Sie den struct-Variablen Werte zuweisen
    • Die Parameternamen finden Sie hier: FX-Typen und Parameter
  • Wenden Sie die geänderte Struktur wieder auf die FX-Struktur an, indem Sie fx_set_parameters()
    • Sie müssen hier nicht die Funktion layer_set_fx() aufzurufen, da sich die Änderung der FX-Struktur direkt auf die Ebene auswirkt, der sie bereits zugewiesen ist.

Hier ist ein Beispiel code für den oben erwähnten Arbeitsablauf:

Ereignis erstellen

// Store the FX struct, and its parameters struct, in variables
pixelate_fx = layer_get_fx("Effect_1");
pixelate_fx_params = fx_get_parameters(pixelate_fx);

Schritt Ereignis

// Change param as variable
pixelate_fx_params.g_CellSize = round((mouse_x / room_width) * 64);

// Or, change param as string
pixelate_fx_params[$ "g_CellSize"] = round((mouse_x / room_width) * 64);

// Apply updated parameters struct to the FX struct
fx_set_parameters(pixelate_fx, pixelate_fx_params);

FX Runtime Parameter

Auf der Seite FX Types & Parameters sind alle Filter/Effekte mit ihren Laufzeitparametern aufgelistet.

Sie können die Runtime Parameternamen auf die folgenden drei Arten verwenden (am Beispiel des Parameters "g_CellSize" ):

  • Ändern Sie einen Parameter in einer FX-Struktur, indem Sie fx_set_parameter() aufrufen: fx_set_parameter(fx_struct, "g_CellSize", 8);
  • Ändern Sie eine Variable in einer Parameterstruktur: params_struct.g_CellSize= 8;
    • HINWEIS Vergewissern Sie sich, dass Sie zuerst die Parameterstruktur durch Aufruf von fx_get_parameters() erhalten.
  • Ändern Sie eine Variable in einer Parameterstruktur mit dem $-Accessor und einer Zeichenkette: params_struct[$ "g_CellSize"] = 8;

Einschichtiger Modus

Standardmäßig wird ein Filter/Effekt auf die Ebene, der er zugewiesen ist, und auf alle darunter liegenden Ebenen angewendet, Sie können jedoch mit fx_set_single_layer() können Sie jedoch den Modus Einzelne Ebene für einen Filter/Effekt aktivieren, um sicherzustellen, dass er nur auf die Ebene angewendet wird, der er zugewiesen ist.
