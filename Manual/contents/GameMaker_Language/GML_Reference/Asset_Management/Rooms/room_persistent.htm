raum_persistent

Diese Variable kann verwendet werden, um das Persistent-Flag für den aktuellen Raum zu erhalten und zu setzen. Wenn sie auf true gesetzt ist, wird der Raum als persistent angesehen. In diesem Fall wird jedes Mal, wenn du den Raum verlässt und wieder zurückkommst, der Zustand der Instanzen innerhalb des Raums beibehalten. Wenn er jedoch als false gekennzeichnet ist, wird er jedes Mal, wenn du in den Raum zurückkehrst, auf seinen Ausgangszustand zurückgesetzt. Du solltest beachten, dass ein persistenter Raum wesentlich mehr Speicherplatz benötigt als ein normaler Raum und es ist nicht empfehlenswert, zu viele von ihnen in deinem Spiel zu haben.

Syntax:

room_persistent;

Rückgabe:

Beispiel:

if lives < 1
{
    room_persistent = false;
    room_goto(rm_start);
}

Die obige code prüft die Variable "lives", und wenn sie kleiner als 1 ist, setzt sie die Raumpersistenz auf false und wechselt dann die Räume.
