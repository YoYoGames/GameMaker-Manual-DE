Raum_vorhanden

Mit dieser Funktion können Sie prüfen, ob der von Ihnen angegebene Raum existiert oder nicht. Diese Funktion nimmt den Raumindex (eine reelle Zahl) und nicht den Raumnamen (eine Zeichenkette). Diese Funktion ist besonders nützlich, wenn Sie Räume dynamisch mit der Funktion room_add()verwenden, aber Sie können auch die schreibgeschützten Variablen room_first und room_last oder die Funktionen room_next() und room_previous() verwenden, um einen bestimmten Raumindex zu erhalten, oder eine Variable bereitstellen, die den Index eines beliebigen anderen Raums in Ihrem Spiel gespeichert hat.

Syntax:

room_exists(index);

ArgumentTyp Beschreibung
index The index of the room to check.

Rückgabe:

Beispiel:

if room_exists(global.rm[0])
{
    room_goto(global.rm[0]);
}

Die obige code prüft, ob der Raum, der im Array "global.rm[]" indiziert ist, existiert, und wenn dies der Fall ist, geht sie zu diesem Raum.
