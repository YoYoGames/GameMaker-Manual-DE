raum_holen_name

Diese Funktion kann verwendet werden, um den Namen des angegebenen Raums als String zurückzugeben. Bitte beachten Sie, dass es sich hierbei nur um eine Zeichenkette handelt, die nicht verwendet werden kann, um den Raum direkt zu referenzieren - dafür benötigen Sie den Raumindex. Sie können diese Zeichenkette jedoch verwenden, um den Raumindex zu ermitteln, indem Sie die zurückgegebene Zeichenkette zusammen mit der Funktion asset_get_index().

Syntax:

room_get_name(index);

ArgumentTyp Beschreibung
index The index of the room to check the name of.

Rückgabe:

Beispiel:

var roomname = room_get_name(room);

draw_text(32, 32, _roomname);

Die obige code ermittelt den Namen des aktuellen Raums und zeichnet ihn auf den Bildschirm.
