Zimmer_hinzufügen

Diese Funktion erstellt einen neuen, leeren Raum und fügt ihn zu Ihrem Spiel hinzu. Sie gibt seinen Index zurück, der in einer Variablen für alle weiteren codes gespeichert wird, die mit diesem Raum zu tun haben. Beachten Sie, dass jeder Raum dauerhaft zum Spiel hinzugefügt wird, bis die ausführbare Datei geschlossen wird, d.h.: Räume, die durch code hinzugefügt werden, können nicht wieder gelöscht werden. Dies hat erhebliche Auswirkungen auf die Speichernutzung, weshalb Sie diese Funktion mit Bedacht verwenden sollten.

HINWEIS: Neue Räume sind nicht Teil der üblichen Raumordnung, d.h. sie haben keinen "vorherigen" oder "nächsten" Raum (was bedeutet, dass die Funktionen room_next() und room_previous() nicht funktionieren). Um von einem hinzugefügten Raum zu einem anderen zu springen, müssen Sie den Index des Raums selbst verwenden.

Syntax:

room_add();

Rückgabe:

Room Asset

Beispiel:

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_persistent(global.myroom, false);

Dadurch wird ein neuer Raum erstellt und sein Index in der Variablen "global.myroom" gespeichert. Anschließend wird seine Breite auf 640 Pixel, seine Höhe auf 480 Pixel und seine Dauerhaftigkeit auf false eingestellt.
