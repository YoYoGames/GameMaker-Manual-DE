zimmer_erste

Diese schreibgeschützte Variable gibt den Index des allerersten Raums im Spiel zurück (dieser wird durch die Reihenfolge definiert, in der die Räume im Raummanager erscheinen, und nicht durch die Reihenfolge, in der sie erstellt wurden).

Syntax:

room_first;

Rückgabe:

Room Asset

Beispiel:

if lives < 1
{
    room_goto(room_first);
}

Die obige code prüft die Variable "Leben" und geht, wenn sie kleiner als 1 ist, in den ersten Raum des Spiels.
