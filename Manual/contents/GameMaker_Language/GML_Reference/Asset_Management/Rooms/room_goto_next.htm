raum_zur_nächsten

Mit dieser Funktion können Sie Ihr Spiel zum nächsten Raum springen lassen, der zum Zeitpunkt der Kompilierung des Spiels im Raummanager aufgeführt war. Wenn dieser Raum nicht existiert, wird ein Fehler ausgelöst und das Spiel wird gezwungen, sich zu beenden. Beachte, dass sich der Raum erst am Ende des Ereignisses ändert, in dem die Funktion aufgerufen wurde, so dass code nach dem Aufruf dieser Funktion immer noch läuft, wenn es sich um dasselbe Ereignis handelt.

Syntax:

room_goto_next();

Rückgabe:

Beispiel:

if room_exists(room_next(room))
{
    room_goto_next();
}

Das obige code prüft, ob es nach dem aktuellen Raum noch einen weiteren gibt, und wenn ja, geht es zu diesem Raum.
