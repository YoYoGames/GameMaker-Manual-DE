schicht_hintergrund_erstellen

Mit dieser Funktion können Sie einer Ebene eine sprite Ressource zuweisen, die als Hintergrund in Ihrem Projekt verwendet werden soll. Sie geben die Ebenen-ID (die Sie erhalten, wenn Sie die Ebene mit layer_create()) oder den Ebenennamen (als String - dies wirkt sich auf die Leistung aus) und einen sprite -Index (das wäre der Name der sprite, wie er im Asset-Browser angezeigt wird), und schon wird sie der Ebene hinzugefügt. Die Funktion gibt den eindeutigen ID-Wert für das Element zurück, der dann in weiteren Ebenenfunktionen für Hintergründe verwendet werden kann.

Syntax:

layer_background_create(layer_id, sprite)

Argument Typ Beschreibung
layer_id String oder Ebenen-ID The unique ID value of the layer to target (or the layer name as a string)
sprite Sprite-Asset The sprite index to be used

Rückgabe:

Background Element ID

Beispiel:

global.back_layer = layer_create(10000);
global.back_trees = layer_background_create(global.back_layer, spr_Trees);

Die obige code erstellt eine neue Ebene und fügt dann ein neues Hintergrundelement hinzu, wobei sprite als Hintergrundbild verwendet wird.
