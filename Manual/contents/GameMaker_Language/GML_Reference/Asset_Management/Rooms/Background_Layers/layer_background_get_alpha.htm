layer_background_get_alpha

Diese Funktion kann verwendet werden, um den Alphawert des Hintergrundelements zu ermitteln. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie die Funktion layer_background_get_id()), und die Funktion gibt einen Wert zwischen 0 (völlig transparent) und 1 (völlig undurchsichtig) zurück.

Syntax:

layer_background_get_alpha(background_element_id);

ArgumentTyp Beschreibung
background_element_id The unique ID value of the background element to get the information from

Rückgabe:

(from 0 to 1)

Beispiel:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_alpha(back_id) < 0.1)
{
    layer_background_destroy(back_id);
}

Die obige code ermittelt die Ebenen-ID für die Ebene "Hintergrund_Himmel" und verwendet diese dann, um die ID des Hintergrundelements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um den Alphawert des Elements zu prüfen, und wenn er kleiner als 0,1 ist, wird das Ebenenelement zerstört.
