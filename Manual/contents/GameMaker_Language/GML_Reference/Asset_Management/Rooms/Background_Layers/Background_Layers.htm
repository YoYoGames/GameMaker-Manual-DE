Hintergrund-Ebenen

Mit dem GameMaker-Raumeditor können Sie mithilfe von Ebenen Hintergründe in jeden beliebigen Raum einfügen. Wie der Name schon sagt, ist eine Hintergrundebene einfach ein sprite Asset, das als Hintergrund auf einer Ebene in einer bestimmten Tiefe innerhalb des Raums verwendet wird, und durch das Stapeln von Ebenen können Sie einige Dinge über oder unter anderen zeichnen lassen. Sie können auch bestimmte Aspekte der Ebenen von code aus steuern, Dinge hinzufügen oder entfernen oder bestimmte Eigenschaften der Ebene oder ihrer Inhalte zur Laufzeit ändern. Im Folgenden finden Sie eine Liste aller Funktionen, die für die Bearbeitung verwendet werden können Hintergrund-Ebenen:

  • layer_background_get_id
  • layer_background_exists
  • layer_background_create
  • layer_background_destroy
  • layer_background_visible
  • layer_background_sprite
  • layer_background_htiled
  • layer_background_vtiled
  • layer_background_stretch
  • layer_background_blend
  • layer_background_alpha
  • layer_background_index
  • layer_background_speed
  • layer_background_xscale
  • layer_background_yscale
  • layer_background_get_visible
  • layer_background_get_sprite
  • layer_background_get_htiled
  • layer_background_get_vtiled
  • layer_background_get_stretch
  • layer_background_get_blend
  • layer_background_get_alpha
  • layer_background_get_index
  • layer_background_get_speed
  • layer_background_get_xscale
  • layer_background_get_yscale

Beachten Sie, dass Sie die folgenden allgemeinen Ebenenfunktionen verwenden müssen, wenn Sie die Hintergrundposition oder die horizontale oder vertikale Bildlaufgeschwindigkeit des Hintergrunds einstellen möchten:

  • layer_x
  • layer_y
  • layer_hspeed
  • layer_vspeed
