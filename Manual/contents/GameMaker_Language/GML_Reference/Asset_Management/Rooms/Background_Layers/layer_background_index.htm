schicht_hintergrund_index

Diese Funktion kann verwendet werden, um den Bildindex des Hintergrunds sprite zu setzen, der mehrere Unterbilder hat. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie die Funktion layer_background_get_id()), und setzen dann den zu verwendenden Bildindex. Wenn Sie einen Wert außerhalb des Bereichs der Unterbilder festlegen, wird der Bildindex loop umhergehen.

Syntax:

layer_background_index(background_element_id, image_index);

Argument Typ Beschreibung
background_element_id Hintergrund Element ID The unique ID value of the background element to change
index Real The image index to use for the background

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_index(back_id, 1);

Die obige code ermittelt die Ebenen-ID für die Ebene "Background_trees" und verwendet diese dann, um die ID des Hintergrundelements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um den Bildindex des Elements zu ändern.
