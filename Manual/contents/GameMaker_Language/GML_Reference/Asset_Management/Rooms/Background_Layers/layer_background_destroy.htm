layer_background_destroy

Diese Funktion zerstört das angegebene Hintergrundelement. Sie geben die Hintergrund-ID an (die Sie erhalten, wenn Sie den Hintergrund mit layer_background_create() oder wenn Sie die Ebenen-ID zusammen mit layer_get_background_id()) und diese Funktion entfernt ihn. Wenn es sich um einen Hintergrund handelt, der im Raumeditor hinzugefügt wurde, wird der Hintergrund beim nächsten Verlassen des Raums und bei der Rückkehr wieder neu erstellt. Wenn der Raum jedoch persistent ist, wird der Hintergrund entfernt, es sei denn, die Raumpersistenz wird wieder ausgeschaltet.

Syntax:

layer_background_destroy(background_element_id)

Argument Typ Beschreibung
background_element_id Hintergrund Element ID The unique ID value of the background element to be destroyed

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Background_trees");
var bck_id = layer_background_get_id("Forrest");
if layer_background_exists(lay_id, bck_id)
{
    layer_background_destroy(bck_id);
}

Die obige code prüft die Ebene "Background_trees", um zu sehen, ob das angegebene Hintergrundelement existiert, und wenn ja, wird es zerstört (aber nicht die Ebene).
