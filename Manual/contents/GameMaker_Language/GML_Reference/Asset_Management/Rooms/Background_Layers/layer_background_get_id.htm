layer_background_get_id

Diese Funktion kann verwendet werden, um den eindeutigen ID-Wert des Hintergrundelements einer Ebene abzurufen. Sie geben die Ebenen-ID an (die Sie erhalten, wenn Sie den Ebenennamen zusammen mit layer_get_id()) und die Funktion gibt den ID-Wert zurück, der mit dem Hintergrundelement auf der Ebene verbunden ist. Beachten Sie, dass diese Funktion speziell für die Verwendung mit Hintergründen gedacht ist, die in IDE hinzugefügt wurden, denn wenn Sie einer Ebene über code einen Hintergrund hinzufügen, indem Sie die Funktion layer_background_create()einen Hintergrund hinzufügen, wird die eindeutige ID für das hinzugefügte Hintergrundelement zurückgegeben.

Syntax:

layer_background_get_id(layer_id)

Argument Typ Beschreibung
layer_id String oder Ebenen-ID The unique ID value of the layer to target

Rückgabe:

Background Element ID

Beispiel:

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_sprite(back_id, bck_Trees_Winter);

Die obige code ermittelt die Ebenen-ID für die Ebene "Background_trees" und verwendet diese dann, um die ID des Hintergrundelements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um das Element sprite zu ändern.
