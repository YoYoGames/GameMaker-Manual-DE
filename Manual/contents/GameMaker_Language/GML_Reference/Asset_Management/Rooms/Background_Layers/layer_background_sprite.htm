layer_background_sprite

Mit dieser Funktion können Sie den sprite Index des Hintergrundelements einstellen. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie die Funktion layer_background_get_id()), und geben dann einen sprite Index an, der verwendet werden soll. Das Bild des Hintergrundelements wird durch den neuen sprite ersetzt. Wenn Sie den Wert -1 angeben, wird dem Element kein sprite zugewiesen (es existiert aber weiterhin und kann später wieder mit sprite versehen werden).

Diese Funktion hat auch den Alias layer_background_change().

Syntax:

layer_background_sprite(background_element_id, sprite_index)

Argument Typ Beschreibung
background_element_id Hintergrund Element ID The unique ID value of the background element to change
sprite_index Sprite-Asset The sprite index of the sprite to use for the background element

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_sprite(back_id) != spr_Clouds)
{
    layer_background_sprite(back_id, spr_Clouds);
}

Die obige code ermittelt die Ebenen-ID für die Ebene "Hintergrund_Himmel" und verwendet diese dann, um die ID des Hintergrundelements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um die dem Element zugewiesene sprite zu überprüfen und sie auf sprite "spr_Clouds" zu setzen, wenn sie es nicht bereits ist.
