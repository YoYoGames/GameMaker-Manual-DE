layer_background_get_visible

Diese Funktion kann verwendet werden, um den sichtbaren Zustand des Hintergrundelements zu ermitteln. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie die Funktion layer_background_get_id()), und die Funktion gibt entweder true zurück, wenn das Element derzeit sichtbar ist, oder false, wenn es nicht sichtbar ist. Beachten Sie, dass dieser Rückgabewert nicht davon beeinflusst wird, ob die Ebene, auf der sich das Element befindet, sichtbar ist oder nicht.

Syntax:

layer_background_get_visible(background_element_id);

ArgumentTyp Beschreibung
background_element_id The unique ID value of the background element to get the information from

Rückgabe:

Beispiel:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_visible(back_id))
{
    layer_background_visible(back_id, false);
}

Die obige code ruft die Ebenen-ID für die Ebene "Hintergrund_Himmel" ab und verwendet diese dann, um die ID des Hintergrundelements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um die Sichtbarkeit des Elements zu prüfen, und wenn es sichtbar ist, wird diese Eigenschaft auf false gesetzt.
