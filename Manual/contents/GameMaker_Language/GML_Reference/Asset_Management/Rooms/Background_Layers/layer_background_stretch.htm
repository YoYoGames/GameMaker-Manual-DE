layer_background_stretch

Mit dieser Funktion können Sie ein Hintergrundelement sprite so einstellen, dass es sich dem Raum anpasst oder dass es 1:1 mit der Auflösung bleibt. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie die Funktion layer_background_get_id()), und setzen Sie dann die Streckung argument auf true oder false. Bei der Einstellung true wird das Element sprite gestreckt, damit es in den Raum passt (entweder vergrößert oder verkleinert, je nach Größe von sprite und des Raums), und bei der Einstellung false wird das Element sprite in seiner ursprünglichen Auflösung angezeigt.

Syntax:

layer_background_stretch(background_element_id, stretch)

Argument Typ Beschreibung
background_element_id Hintergrund Element ID The unique ID value of the background element to change
stretch Boolesche The stretch toggle, which can be true oder false

Rückgabe:

N/A

Beispiel:

if mouse_check_button_pressed(mb_left)
{
    var back = layer_background_get_id(layer);
    if (layer_background_get_stretch(back))
    {
        layer_background_stretch(back, false);
    }
    else
    {
        layer_background_stretch(back, true);
    }
}

Das obige code prüft, ob eine Maustaste gedrückt wird, und wenn dies der Fall ist, wird die Dehnung des Hintergrundelements sprite umgeschaltet, das der Ebene zugewiesen ist, auf der die Instanz code läuft.
