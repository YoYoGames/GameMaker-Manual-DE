layer_background_xscale

Mit dieser Funktion können Sie die Skalierung entlang der x-Achse eines Hintergrundelements festlegen. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie die Funktion layer_background_get_id()), und legen dann den Skalenwert fest. Der von Ihnen angegebene Skalenwert ist ein Multiplikator, der verwendet wird, um die Darstellung des Hintergrundelements zu ändern, Ein Wert von 0,5 würde das Element im halben Maßstab anzeigen, ein Wert von 2 im doppelten Maßstab. Beachten Sie, dass auch negative Werte gültig sind und das Element um die Position (0,0) "spiegeln", d. h. eine x-Skala von -1 würde das Bild seitenverkehrt anzeigen.

Syntax:

layer_background_xscale(background_element_id, xscale);

ArgumentTyp Beschreibung
background_element_id The unique ID value of the background element to change
xscale The scale value to use (1 is no scaling)

Rückgabe:

Beispiel:

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_xscale(back_id, -1);

Die obige code ermittelt die Ebenen-ID für die Ebene "Hintergrund_Wasser" und verwendet diese dann, um die ID des Hintergrundelements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um das Element xscale zu ändern.
