layer_background_alpha

Diese Funktion steuert das Alpha (Transparenz) des Hintergrunds sprite. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie verwenden die Funktion layer_background_get_id()), und legen dann den zu verwendenden Alphawert fest. Alpha kann zwischen 0 (völlig transparent) und 1 (völlig undurchsichtig) liegen, wobei der Standard-Alpha-Wert für das Hintergrundelement Element ist 1. Beachten Sie, dass, wenn die Ebene, der das Hintergrundelement zugewiesen wurde, nicht sichtbar ist - oder das Element selbst unsichtbar gemacht wurde - Sie mit dieser Funktion keinen Unterschied sehen werden, bis die Ebene oder das Element wieder sichtbar gemacht wurde wieder sichtbar gemacht wird.

Syntax:

layer_background_alpha(background_element_id, alpha);

ArgumentTyp Beschreibung
background_element_id The unique ID value of the background element to change
alpha The alpha for background sprite, from 0 to 1 (default is 1)

Rückgabe:

Beispiel:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
layer_background_alpha(back_id, random(1));

Die obige code ruft den ID-Wert des Hintergrunds ab, der der Ebene "Background_sky" zugewiesen ist, und setzt dann dessen Alpha auf einen Zufallswert zwischen 0 und 1.
