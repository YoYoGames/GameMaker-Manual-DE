layer_background_get_sprite

Diese Funktion kann verwendet werden, um den aktuellen sprite Indexwert des Hintergrundelements zu erhalten. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie die Funktion layer_background_get_id()), und die Funktion gibt einen reellen Wert zurück, der den angezeigten sprite -Index darstellt. Wenn dem Element kein sprite zugewiesen ist, gibt die Funktion -1 zurück.

Syntax:

layer_background_get_sprite(background_element_id);

ArgumentTyp Beschreibung
background_element_id The unique ID value of the background element to get the information from

Rückgabe:

Sprite Asset or -1

Beispiel:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_sprite(back_id) != spr_Clouds)
{
    layer_background_sprite(back_id, spr_Clouds);
}

Die obige code ermittelt die Ebenen-ID für die Ebene "Hintergrund_Himmel" und verwendet diese dann, um die ID des Hintergrundelements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um die dem Element zugewiesene sprite zu überprüfen und sie auf sprite "spr_Clouds" zu setzen, wenn sie es nicht bereits ist.
