layer_background_yscale

Mit dieser Funktion können Sie die Skalierung entlang der y-Achse eines Hintergrundelements festlegen. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie die Funktion layer_background_get_id()), und legen dann den Skalenwert fest. Der von Ihnen angegebene Skalenwert ist ein Multiplikator, der verwendet wird, um die Darstellung des Hintergrundelements zu ändern, wobei ein Wert von 0,5 das Element im halben Maßstab und ein Wert von 2 im doppelten Maßstab darstellen würde. Beachten Sie, dass negative Werte gültig sind und das Element um die Position (0,0) "spiegeln", d.h. eine x-Skala von -1 würde das Bild umgekehrt darstellen.

Syntax:

layer_background_yscale(background_element_id, yscale);

Argument Typ Beschreibung
background_element_id Hintergrund Element ID The unique ID value of the background element to change
yscale Real The speed multiplier/value to use for the element

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_yscale(back_id, -1);

Die obige code ermittelt die Ebenen-ID für die Ebene "Hintergrund_Wasser" und verwendet diese dann, um die ID des Hintergrundelements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um die y-Skala des Elements zu ändern.
