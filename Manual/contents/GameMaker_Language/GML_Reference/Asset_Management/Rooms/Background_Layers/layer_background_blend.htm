layer_background_blend

Diese Funktion steuert die Überblendung (oder "Tönung") des Hintergrunds sprite und der Standardwert ist -1 (was der Konstante c_white entspricht, die ebenfalls verwendet werden kann). Jeder andere Wert (einschließlich interner Farbkonstanten wie c_red, oder c_aqua) mischt die angegebene Farbe mit dem Original sprite. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie die Funktion layer_background_get_id()), und legen dann die zu verwendende Mischfarbe fest. Unten sehen Sie ein Beispiel für eine sprite, die mit verschiedenen Farben überblendet wurde:

Bitte beachten Sie, dass Sie versuchen sollten, die Überblendung auf der HTML5-Seite platform zu begrenzen (es sei denn, Sie verwenden WebGL), da jede überblendete sprite separat zwischengespeichert werden muss und daher viele überblendete sprites die Leistung beeinträchtigen können (Sie können die Größe von cache auch mit der Funktion sprite_set_cache_size()).

Syntax:

layer_background_blend(background_element_id, blend);

ArgumentTyp Beschreibung
background_element_id The unique ID value of the background element to change
blend The colour to blend with the background sprite (default is c_white)

Rückgabe:

Beispiel:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
layer_background_blend(back_id, c_aqua);

Die obige code ruft den ID-Wert des Hintergrunds ab, der der Ebene "Hintergrund_Himmel" zugewiesen ist, und färbt ihn dann in eine Farbe ein.
