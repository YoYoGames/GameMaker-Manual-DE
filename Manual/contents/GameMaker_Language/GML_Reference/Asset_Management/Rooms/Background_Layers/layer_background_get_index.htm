layer_background_get_index

Diese Funktion kann verwendet werden, um den aktuellen Bildindexwert des Hintergrundelements zu ermitteln. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie die Funktion layer_background_get_id()), und die Funktion gibt einen realen Wert zurück, der den für sprite angezeigten Bildindex darstellt. Die Funktion gibt -1 zurück, wenn entweder das Hintergrundelement nicht existiert oder dem Element keine gültige sprite zugewiesen ist.

Syntax:

layer_background_get_index(background_element_id);

ArgumentTyp Beschreibung
background_element_id The unique ID value of the background element to get the information from

Rückgabe:

Real (the current sprite image index or -1)

Beispiel:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_index(back_id) < 4)
{
    layer_background_index(back_id, 4);
}

Die obige code ermittelt die Ebenen-ID für die Ebene "Hintergrund_Himmel" und verwendet diese dann, um die ID des Hintergrundelements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um zu prüfen, ob der Bildindex für das Element kleiner als 4 ist, und wenn ja, wird er auf 4 gesetzt.
