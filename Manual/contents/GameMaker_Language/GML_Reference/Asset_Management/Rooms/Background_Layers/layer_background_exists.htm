layer_background_exists

Mit dieser Funktion können Sie prüfen, ob ein Hintergrundelement auf einer bestimmten Ebene vorhanden ist. Sie geben die Ebenen-ID (die Sie erhalten, wenn Sie die Ebene mit layer_create()) oder den Ebenennamen (als String - dies wirkt sich auf die Leistung aus) und die ID des Hintergrundelements (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie die Funktion layer_background_get_id()) und die Funktion gibt entweder true zurück, wenn das Element existiert, oder false, wenn es nicht existiert.

HINWEIS: Diese Funktion arbeitet innerhalb des Bereichs des aktuellen Zielraums - standardmäßig der Raum, in dem die Funktion aufgerufen wird -, der mit der Funktion layer_set_target_room().

Syntax:

layer_background_exists(layer_id, background_element_id)

Argument Typ Beschreibung
layer_id String oder Ebenen-ID The unique ID value of the layer to target (or the layer name as a string)
background_element_id Hintergrund Element ID The unique ID value of the background element to check

Rückgabe:

Boolean

Beispiel:

var lay_id = layer_get_id("Background_trees");
if layer_background_exists(lay_id, global.TreesBackground)
{
    layer_background_destroy(lay_id, global.TreesBackground);
}

Die obige code prüft die Ebene "Background_trees", um zu sehen, ob das angegebene Hintergrundelement existiert, und wenn ja, wird es zerstört (aber nicht die Ebene).
