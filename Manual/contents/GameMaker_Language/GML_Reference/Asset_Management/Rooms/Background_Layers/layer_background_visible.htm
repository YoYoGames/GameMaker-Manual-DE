layer_background_visible

Mit dieser Funktion können Sie die Sichtbarkeit eines Hintergrundelements umschalten. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie die Funktion layer_background_get_id()), und setzen Sie dann die Sichtbarkeit argument auf true oder false. Bei true wird das Element angezeigt, bei false wird das Element nicht angezeigt. Beachten Sie, dass dies von der Sichtbarkeit der Ebene abhängt. Selbst wenn das Hintergrundelement als sichtbar gekennzeichnet ist, wird es nicht gezeichnet, wenn die Ebene, auf der es sich befindet, als nicht sichtbar gekennzeichnet ist.

Syntax:

layer_background_visible(background_element_id, visible)

Argument Typ Beschreibung
background_element_id Hintergrund Element ID The unique ID value of the background element to change
visible Boolesche The visibility toggle, which can be true oder false

Rückgabe:

N/A

Beispiel:

if mouse_check_button_pressed(mb_left)
{
    var back = layer_background_get_id(layer);
    if layer_background_get_visible(back)
    {
        layer_background_visible(back, false);
    }
    else
    {
        layer_background_visible(back, true);
    }
}

Das obige code prüft, ob eine Maustaste gedrückt wird, und wenn dies der Fall ist, wird die Hintergrundsichtbarkeit des Hintergrundelements, das der Ebene zugeordnet ist, auf der die Instanz code läuft, umgeschaltet.
