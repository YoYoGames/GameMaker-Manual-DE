layer_background_get_xscale

Mit dieser Funktion können Sie den aktuellen Skalenmultiplikatorwert des Hintergrundelements ermitteln. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie die Funktion layer_background_get_id()), und die Funktion gibt einen realen Wert zurück, der den Skalierungsmultiplikator darstellt, mit dem das sprite gezeichnet wird. Der Standardwert ist 1.

Syntax:

layer_background_get_xscale(background_element_id);

ArgumentTyp Beschreibung
background_element_id The unique ID value of the background element to get the information from

Rückgabe:

Beispiel:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_xscale(back_id) != -1)
{
    layer_background_xscale(back_id, -1);
}

Die obige code ermittelt die Ebenen-ID für die Ebene "Hintergrund_Himmel" und verwendet diese dann, um die ID des Hintergrundelements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um die x-Skala für das Element zu überprüfen, und wenn sie nicht -1 ist, wird sie auf -1 gesetzt.
