layer_background_vtiled

Mit dieser Funktion können Sie festlegen, ob das angegebene Hintergrundelement auf einer Ebene vertikal gekachelt werden soll oder nicht. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie die Funktion layer_background_get_id()), und setzen dann den Wert für die Kachelung. Wenn Sie den Wert true angeben, wird das Element vertikal im Raum angezeigt, wenn Sie den Wert false angeben, wird das Element nicht gekachelt. Diese Funktion ist nur für 2D-Projekte geeignet und funktioniert nicht korrekt, wenn eine 3D-Kameraprojektion verwendet wird.

Syntax:

layer_background_vtiled(background_element_id, tiled);

Argument Typ Beschreibung
background_element_id Hintergrund Element ID The unique ID value of the background element to change
tiled Boolesche The vertical tiling toggle, which can be true oder false

Rückgabe:

N/A

Beispiel:

var back = layer_background_get_id(layer);
if !layer_background_get_htiled(back) || !layer_background_get_vtiled(back)
{
    layer_background_htiled(back, true);
    layer_background_vtiled(back, true);
}

Die obige code prüft das Hintergrundelement, das der Ebene zugewiesen ist, auf der die Instanz code läuft, und wenn es nicht in eine der beiden Richtungen gekachelt ist, wird es so eingestellt, dass es vertikal und horizontal gekachelt wird.
