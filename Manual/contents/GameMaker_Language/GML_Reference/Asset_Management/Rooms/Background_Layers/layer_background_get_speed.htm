layer_background_get_speed

Diese Funktion kann verwendet werden, um den aktuellen Geschwindigkeitsmultiplikatorwert des Hintergrundelements zu ermitteln. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie die Funktion layer_background_get_id()), und die Funktion gibt einen realen Wert zurück, der den Geschwindigkeitsmultiplikator darstellt, der für die Animation von sprite verwendet wird. Der Standardwert ist 1.

Syntax:

layer_background_get_speed(background_element_id);

ArgumentTyp Beschreibung
background_element_id The unique ID value of the background element to get the information from

Rückgabe:

Beispiel:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_speed(back_id) > 0)
{
    layer_background_speed(back_id, 0);
}

Die obige code ermittelt die Ebenen-ID für die Ebene "Hintergrund_Himmel" und verwendet diese dann, um die ID des Hintergrundelements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um die Animationsgeschwindigkeit für das Element zu überprüfen, und wenn sie größer als 0 ist, wird sie auf 0 gesetzt.
