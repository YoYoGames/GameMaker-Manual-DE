room_set_camera

Mit dieser Funktion können Sie eine Kamera einem bestimmten Ansichtsfenster in einem anderen als dem aktuellen Raum zuweisen. Sie geben den Raumindex, den Ansichtsindex (von 0 bis 7) und dann den Index der zu verwendenden Kamera an.

Syntax:

room_set_camera(rm, vind, camera);

Argument Typ Beschreibung
rm Zimmer Asset The index of the room to set the view camera of
vind Real The index of the view port to assign the camera to
camera Kamera-ID The index of the camera to assign

Rückgabe:

N/A

Beispiel:

global.myroom = room_add();
room_set_camera(global.myroom, 0, global.MainCam);

Die obige code weist eine Kamera in einem neu erstellten Raum dem View Port [0] zu.
