room

Diese Variable enthält den Raumindex für den aktuellen Raum, in dem Ihr Spiel läuft. Diese Variable ist nicht schreibgeschützt. Wenn Sie diese Variable ändern, wird nicht der Indexwert für den aktuellen Raum geändert, sondern der Raum wird so geändert, dass er dem Index entspricht, auf den Sie die Variable gesetzt haben. Seien Sie vorsichtig, wenn Sie dies tun, denn wenn der Index, auf den Sie die Variable ändern, nicht gültig ist, wird das Spiel einen Fehler ausgeben und geschlossen. Im Allgemeinen ist es viel besser, die Funktion room_goto() um Räume zu wechseln.

Hinweis: Die Raum-IDs basieren nicht auf ihrer Reihenfolge im Asset-Browser oder im Raum-Manager, daher sollten Sie es vermeiden, einen Zahlenwert direkt anzugeben. Verwenden Sie stattdessen die Raumkonstante für das Asset, auf das Sie verweisen möchten (das ist einfach sein Name), oder rufen Sie ihn über eine Funktion ab.

Syntax:

room

Rückgabe:

Room Asset

Beispiel:

if (room == rm_level1)
{
    audio_play_sound(snd_level1, 1, 1);
}

Die obige code prüft, ob der aktuelle Raum die erste Ebene ist, und spielt in diesem Fall die Musik dieser Ebene.
