schicht_tiefe

Diese Funktion kann verwendet werden, um die Tiefe der gegebenen Ebene zu ändern und so die Reihenfolge zu ändern, in der ihr Inhalt auf dem Bildschirm dargestellt wird. Sie geben die Ebenen-ID (die Sie beim Erstellen der Ebene mit layer_create()) oder den Ebenennamen (als String - dies wirkt sich auf die Leistung aus) und geben dann die neue Tiefe an, auf die sie gesetzt werden soll (ein ganzzahliger Wert von -16000 bis 16000). Die Ebenentiefe ist so definiert, dass sie höher ist, wenn sie "weiter weg" von der Kamera ist, und niedriger, wenn sie "näher" an der Kamera ist. Wenn Sie also drei Ebenen mit den Tiefen -100, 0, 100 haben, werden die Ebenen in der Reihenfolge 100, 0, -100 gezeichnet, so dass die "oberste" Ebene (d. h. die Ebene, die der Kameraansicht am nächsten ist und daher über alles andere gezeichnet wird) die Ebene mit der Tiefe -100 ist.

Beachten Sie, dass Sie die Tiefe einer Ebene jederzeit mit der Funktion layer_get_depth(). Beachten Sie auch, dass die minimale und maximale Ebenentiefe zwischen -16000 und 16000 liegt und alles, was außerhalb dieser Tiefen liegt , nicht gerendert wird. Wenn Sie eine Tiefe außerhalb dieses Bereichs benötigen, müssen Sie die Funktion layer_force_draw_depth().

Beachten Sie, dass die Änderung der Tiefe einer Ebene die Anwendung von Filtern und Effekten auf diese Ebene verändern kann, da eine Änderung der Tiefe unter der Tiefe einer Effekt-Ebene dazu führt, dass deren Effekt nicht mehr auf die Ebene angewendet wird.

Syntax:

layer_depth(layer_id, depth)

Argument Typ Beschreibung
layer_id String oder Ebenen-ID The unique ID value of the layer to set the depth of (or the layer name as a string)
depth Real The new depth for the layer

Rückgabe:

N/A

Beispiel:

if layer_get_depth(layer) != -100
{
    layer_depth(layer, -100);
}

Die obige code ruft die Tiefe der Ebene ab, auf der sich die Instanz, auf der code läuft, befindet, und wenn sie nicht -100 ist, wird die Tiefe auf -100 gesetzt.
