layer_destroy_instances

Diese Funktion kann verwendet werden, um alle Instanzen zu zerstören, die der angegebenen Ebene zugeordnet sind. Sie geben die Layer-ID (die Sie erhalten, wenn Sie den Layer mit layer_create()) oder den Ebenennamen (als String - dies wirkt sich auf die Leistung aus), und dann werden alle Instanzen, die sich auf der Ebene befinden, aus dem Spiel entfernt und ihre Destroy- und Clean Up-Ereignisse ausgelöst.

Syntax:

layer_destroy_instances(layer_id)

Argument Typ Beschreibung
layer_id String oder Ebenen-ID The unique ID value of the layer (or the layer name as a string)

Rückgabe:

N/A

Beispiel:

if global.game_over
{
    layer_destroy_instances(layer);
}

Die obige code prüft eine globale Variable und wenn sie true ist, werden alle Instanzen, die sich auf der Ebene der aufrufenden Instanz befinden, zerstört (einschließlich der aufrufenden Instanz).
