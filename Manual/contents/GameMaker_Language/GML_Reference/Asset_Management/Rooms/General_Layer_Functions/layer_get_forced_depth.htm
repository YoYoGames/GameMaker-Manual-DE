layer_get_forced_depth

Diese Funktion kann verwendet werden, um die Z-Tiefe abzurufen, die für das Rendern von Ebenen innerhalb des Raums eingestellt ist. Siehe layer_force_draw_depth() für weitere Informationen.

Syntax:

layer_get_forced_depth()

Rückgabe:

Real

Beispiel:

if layer_get_forced_depth() != 0
{
    layer_force_draw_depth(true, 0);
}

Die obige code prüft, ob die Z-Tiefe der Ebene auf einen Wert von 0 erzwungen wird oder nicht, und wenn dies nicht der Fall ist, wird die Z-Tiefe der Ebene auf 0 gesetzt.
