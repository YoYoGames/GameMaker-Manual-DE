layer_get_id

Diese Funktion kann verwendet werden, um den eindeutigen ID-Wert für eine bestimmte Ebene zu erhalten. In IDE haben alle Ebenen einen Namen und einen Typ, und um sie über code bearbeiten oder ändern zu können, müssen Sie den Wert der Ebenen-ID angeben. Diese Funktion dient dazu, diese ID unter Verwendung des Namens (einer Zeichenkette) der Ebene (wie in IDE geschrieben) abzurufen. Wenn Sie eine neue Ebene über code mit der Funktion layer_create() erstellen, gibt diese Funktion stattdessen den eindeutigen ID-Wert zurück (dynamisch erstellte Ebenen erhalten keine Namen). Beachten Sie, dass Sie einen Fehler erhalten und das Projekt abstürzt, wenn Sie den Namen einer Ebene angeben, die im aktuellen Raum nicht existiert.

Syntax:

layer_get_id(layer_name)

ArgumentTyp Beschreibung
layer_name The name of the layer (a string)

Rückgabe:

Layer ID or -1 (if the layer specified doesn't exist)

Beispiel:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

Die obige code ermittelt zunächst den Index der Instanz, die der angegebenen x/y-Position am nächsten liegt, und speichert ihn in einer lokalen Variablen. Anschließend wird die eindeutige Instanz-Layer-ID für die Ebene "Instances Front" ermittelt und die gefundene Instanz auf diese Ebene verschoben.
