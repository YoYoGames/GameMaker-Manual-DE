schicht_element_verschieben

Mit dieser Funktion können Sie ein Element von einer Ebene in eine andere verschieben. Sie geben die Element-ID an, wie sie von der Funktion zum Erstellen des Elements oder des Raumeditors oder der Funktion layer_get_all_elements()zurückgegeben wird, und dann geben Sie die Ebenen-ID (die Sie erhalten, wenn Sie die Ebene mit layer_create()) oder den Ebenennamen (als String - dies hat Auswirkungen auf die Leistung).

Syntax:

layer_element_move(element_id, layer_id);

Argument Typ Beschreibung
element_id Ebene Element-ID The unique ID value of the element to move
layer_id String oder Ebenen-ID The unique ID value of the layer to move the element to (or the layer name as a string)

Rückgabe:

N/A

Beispiel:

var a = layer_get_all_elements(layer);
asset_layer = layer_create(-100);
for (var i = 0; i < array_length(a); i++)
{
    if layer_get_element_type(a[i]) == layerelementtype_sprite
    {
        layer_element_move(a[i], asset_layer)
    }
}

Die obige code ruft die Elemente auf der Ebene ab, der die Instanz, auf der code läuft, zugewiesen ist, und prüft dann, ob es sich um sprite Assets handelt. Wenn dies der Fall ist, werden sie auf die Ebene mit der in der Variablen "asset_layer" gespeicherten ID verschoben.
