layer_get_vspeed

Mit dieser Funktion können Sie die vertikale Geschwindigkeit (in Pixeln pro Spielframe) der Ebene innerhalb des aktuell ausgewählten Raums abfragen. Du gibst die Ebenen-ID (die du erhältst, wenn du die Ebene mit layer_create()) oder den Ebenennamen (als String - dies hat Auswirkungen auf die Leistung) und die Funktion gibt eine reelle Zahl für die vertikale Geschwindigkeit zurück, wobei ein positiver Wert nach unten und ein negativer Wert nach oben zeigt. Die Voreinstellung ist 0 (es sei denn, sie wurde im Raumeditor festgelegt).

Syntax:

layer_get_vspeed(layer_id)

Argument Typ Beschreibung
layer_id String oder Ebenen-ID The unique ID value of the layer to get the vertical speed from

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
    layer_hspeed(lay_id, 0);
    layer_vspeed(lay_id, 0);
}

Die obige code prüft die horizontalen und vertikalen Geschwindigkeiten der Ebene und setzt sie auf 0, wenn sie nicht beide auf 0 gesetzt sind.
