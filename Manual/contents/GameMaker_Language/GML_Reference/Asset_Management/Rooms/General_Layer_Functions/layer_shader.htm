schicht_shader

Mit dieser Funktion können Sie jeder beliebigen Ebene eine Shader-Ressource zuweisen, und die Ebene wird dann mit diesem Shader gerendert. Sie geben entweder den eindeutigen ID-Wert der Ebene oder den Namen der Ebene (als String - dies wirkt sich auf die Leistung aus) zusammen mit der ID des zu verwendenden Shaders an. Der Shader muss zuvor im Asset-Browser erstellt worden sein und der Shader-Index (der Name der Shader-Ressource) wird dann an diese Funktion übergeben. Wenn der zugewiesenen Ebene Instanzen hinzugefügt wurden, wird der Shader auf alle Zeichnungsereignisse angewendet, die die Instanz verwendet - wenn die Instanz beispielsweise ein Ereignis "Draw GUI Begin" hat, wird der Shader automatisch auf dieses Ereignis angewendet. Der Shader wirkt sich auch auf alle anderen grafischen Elemente aus, die auf dieser Ebene gezeichnet werden, z. B. sprite Assets oder Tile Maps.

Beachten Sie, dass die Funktion nicht in Zeichnungsereignissen oder Step-Ereignissen aufgerufen werden soll, sondern nur zu Beginn des Raumes im Room Creation Code oder im Create Event / Room Start Event einer Instanz aufgerufen werden muss.

Syntax:

layer_shader(layer_id, shader)

Argument Typ Beschreibung
layer_id String oder Ebenen-ID The unique ID value of the layer to target (or the layer name as a string)
shader Shader-Asset The shader index to assign to the layer

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Instances");
layer_shader(lay_id, shd_Sepia);

Die obige code wird die Shader-Ressource "shd_Sepia" dem angegebenen Layer für alle Zeichnungen zuweisen.
