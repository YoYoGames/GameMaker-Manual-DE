Ebene_existiert

Mit dieser Funktion können Sie prüfen, ob die angegebene Ebene existiert. Sie geben die Ebenen-ID (die Sie beim Erstellen der Ebene mit layer_create()) oder den Ebenennamen (als String - dies wirkt sich auf die Leistung aus) und die Funktion gibt den booleschen Wert true zurück, wenn die Ebene existiert, oder false, wenn sie nicht existiert.

HINWEIS: Diese Funktion arbeitet innerhalb des Bereichs des aktuellen Zielraums - standardmäßig der Raum, in dem die Funktion aufgerufen wird -, der mit der Funktion layer_set_target_room().

Syntax:

layer_exists(layer_name)

Argument Typ Beschreibung
layer_name String oder Ebenen-ID The name of the layer (a string or ID value)

Rückgabe:

Boolean

Beispiel:

if !layer_exists(global.tileLayer)
{
    global.tileLayer = layer_create(1000);
}

Die obige code prüft, ob die in der globalen Variable gespeicherte Ebene tatsächlich existiert, und wenn nicht, wird sie erstellt.
