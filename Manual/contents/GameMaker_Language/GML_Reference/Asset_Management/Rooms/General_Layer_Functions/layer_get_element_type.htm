Ebene_Element_Typ_holen

Sie können diese Funktion verwenden, um den Elementtyp für das angegebene Element zu ermitteln. Sie geben den eindeutigen ID-Wert des Elements an (z. B. wie er von der Funktion, die das Element erstellt hat, oder vom Raumeditor zurückgegeben wurde), und die Funktion gibt eine der folgenden Konstanten zurück (oder -1, wenn das Element nicht existiert oder der ID-Wert fehlerhaft ist):

Ebene Elementtyp Konstante
Konstante Beschreibung
layerelementtype_background Das Element ist ein Hintergrund.
layerelementtype_instance Das Element ist eine Instanz.
layerelementtype_sprite Das Element ist ein sprite asset.
layerelementtype_tilemap Das Element ist eine Kachelkarte.
layerelementtype_particlesystem Das Element ist ein Partikelsystem.
layerelementtype_tile Das Element ist eine Legacy-Hintergrundkachel (dies gilt nur für Projekte, die aus früheren Versionen von GameMaker importiert wurden).
layerelementtype_sequence Das Element ist ein sequence asset.

Beachten Sie, dass diese Funktion am nützlichsten ist, wenn Sie mehrere verschiedene Elementtypen einer Ebene zugewiesen haben, und dass Sie eine Liste aller Elemente auf einer bestimmten Ebene mit der Funktion layer_get_all_elements().

Syntax:

layer_get_element_type(element_id)

Argument Typ Beschreibung
layer Ebene Element-ID The unique ID value of the element to get the type of

Rückgabe:

Layer Element Type Constant or -1 (if element does not exist or is invalid)

Beispiel:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if layer_get_element_type(a[i]) == layerelementtype_sprite
    {
        layer_sprite_destroy(a[i]);
    }
}

Die obige code ruft die IDs für alle Instanzelemente ab, die der Ebene der Instanz zugeordnet sind, auf der code läuft. code prüft dann, ob eines der zurückgegebenen Elemente sprite Assets sind, und wenn ja, werden sie zerstört.
