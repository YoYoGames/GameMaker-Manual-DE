layer_has_instance

Diese Funktion kann verwendet werden, um zu prüfen, ob eine bestimmte Instanz derzeit dem angegebenen Layer zugeordnet ist. Sie geben die Layer-ID (die Sie erhalten, wenn Sie den Layer mit layer_create()) oder den Ebenennamen (als String - dies hat Auswirkungen auf die Leistung) und die Instanz-ID der zu prüfenden Instanz. Sie können auch object_index angeben (d. h. den Namen des Objekts im Asset-Browser), und die Funktion prüft, ob sich Instanzen dieses Objekts auf der angegebenen Ebene befinden. Die Funktion gibt true zurück, wenn sich die Instanz auf der Ebene befindet, und false, wenn dies nicht der Fall ist.

Syntax:

layer_has_instance(layer_id, instance_id)

Argument Typ Beschreibung
layer_id String oder Ebenen-ID The unique ID value of the layer to target (or the layer name as a string)
instance_id Instanz-ID The unique instance ID or the object index of the instance to check for

Rückgabe:

Boolean

Beispiel:

if !layer_has_instance(global.Bullet_Layer, obj_Bullet_Parent)
{
    layer_destroy(global.Bullet_Layer);
}

Die obige code prüft, ob die angegebene Ebene Instanzen des Objekts "obj_Bullet_Parent" enthält, und wenn nicht, wird die Ebene zerstört.
