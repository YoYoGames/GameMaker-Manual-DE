schicht_hinzufuegen_instanz

Diese Funktion kann verwendet werden, um eine bestimmte Instanz von der Ebene, auf der sie sich gerade befindet, auf eine andere Ebene zu verschieben. Sie geben die Layer-ID (die Sie erhalten, wenn Sie den Layer mit layer_create()) oder den Ebenennamen (als String - dies hat Auswirkungen auf die Leistung) und die Instanz-ID der Instanz, die zwischen den Ebenen verschoben werden soll. Ein Beispiel: Ihr Player befindet sich auf einer Ebene, die tiefer liegt als eine andere Ebene, und Sie möchten, dass er hinter den Ebenen zwischen den beiden Ebenen erscheint. Mit dieser Funktion switch können Sie den Player von der Vordergrundebene auf die Hintergrundebene verschieben und ihn hinter den anderen gezeichneten Ebenen erscheinen lassen.

HINWEIS Diese Funktion kann nicht verwendet werden, um eine neue Instanz zu einer Ebene hinzuzufügen. Sie müssen die Instanz zuvor erstellt und ihre ID in einer Variablen gespeichert haben.

HINWEIS: Diese Funktion ist nur im Bereich des aktuellen Raums gültig und kann nicht verwendet werden, wenn der Bereich mit der Funktion layer_set_target_room() geändert wurde.

Syntax:

layer_add_instance(layer_id, instance_id)

Argument Typ Beschreibung
layer_id String oder Ebenen-ID The unique ID value of the instance layer to target (or the layer name as a string)
instance_id Instanz-ID The unique instance ID value of the instance to move over to the target layer

Rückgabe:

N/A

Beispiel:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

Die obige code ermittelt zunächst den Index der Instanz, die der angegebenen x/y-Position am nächsten liegt, und speichert ihn in einer lokalen Variablen. Anschließend wird die eindeutige Instanz-Layer-ID für die Ebene "Instances Front" ermittelt und die gefundene Instanz auf diese Ebene verschoben.
