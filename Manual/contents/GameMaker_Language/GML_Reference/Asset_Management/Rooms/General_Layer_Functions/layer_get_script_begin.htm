layer_get_script_begin

Sie geben die Ebenen-ID (die Sie beim Erstellen der Ebene mit layer_create()) oder den Ebenennamen (als String - dies wirkt sich auf die Leistung aus) und diese Funktion gibt den Skriptfunktionsindex der Funktion zurück, die zu Beginn des Renderings für diese Ebene ausgeführt werden soll, oder sie gibt -1 zurück, wenn keine Funktion zugewiesen ist. Mit layer_script_begin() und layer_script_end() können Sie einer Ebene script Funktionen zuweisen.

Syntax:

layer_get_script_begin(layer_id);

Argument Typ Beschreibung
layer_id Zeichenfolge oder Ebene ID The unique ID value of the layer to target (or the layer name as a string)

Rückgabe:

Script Function or -1 (if no function is assigned)

Beispiel:

if layer_get_script_begin(layer) == -1
{
    layer_script_begin(layer, scr_SetShaderValues);
}

Die obige code prüft, ob der Ebene, auf der die Instanz code läuft, eine script Funktion zugewiesen ist, und wenn nicht, wird eine zugewiesen.
