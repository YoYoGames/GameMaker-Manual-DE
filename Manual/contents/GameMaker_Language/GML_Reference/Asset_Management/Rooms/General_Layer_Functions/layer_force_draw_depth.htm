layer_force_draw_depth

Diese Funktion erzwingt, dass alle Ebenen mit der angegebenen z-Tiefe gezeichnet werden. Dies ändert nicht die Reihenfolge, in der die Ebenen gerendert werden und sie werden immer noch in der Reihenfolge der Tiefe gezeichnet werden, es ändert einfach, welche z-Wert verwendet wird. Im Allgemeinen brauchen Sie sich darüber keine Gedanken zu machen, aber wenn Sie Ebenen haben, die eine Tiefe außerhalb des zulässigen Bereichs (-16000 bis 16000) haben, dann werden sie nicht gerendert, also können Sie die Z-Tiefe auf einen vernünftigen Wert setzen - zum Beispiel 0 - und sie werden alle gut gerendert. Beachten Sie, dass dies im Allgemeinen nur für die Verwendung mit älteren Projekten aus früheren GameMaker-Versionen gedacht ist, bei denen die Zeichentiefe höher oder niedriger als der zulässige Ebenenbereich sein kann.

Syntax:

layer_force_draw_depth(force, depth)

Argument Typ Beschreibung
force Boolesche Whether to enable (set to true oder deaktivieren (auf false setzen) Z-Tiefenzwang
depth Real The new Z depth

Rückgabe:

N/A

Beispiel:

if !layer_is_draw_depth_forced()
{
    layer_force_draw_depth(true, 0);
}

Die obige code prüft, ob die Z-Tiefe der Ebene erzwungen ist oder nicht, und wenn nicht, setzt sie die Z-Tiefe auf 0 und aktiviert sie.
