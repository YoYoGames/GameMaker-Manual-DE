layer_set_target_room

Wenn du diese Funktion aufrufst, teilst du dem GameMaker mit, dass alle weiteren Layerfunktionen auf den angegebenen Raum angewendet werden sollen. Auf diese Weise können Sie Layer und Layerinhalte in einem Raum prozedural verändern oder erzeugen, der nicht der aktuelle Raum ist. Wenn du mit dem Hinzufügen von Ebenen oder Ebenenelementen in einem Raum fertig bist, rufe die Funktion layer_reset_target_room() auf, um das Raumziel zurückzusetzen (oder rufen Sie diese Funktion erneut mit einem Raum argument von -1 auf). Beachten Sie, dass diese Funktion nur für Räume verwendet werden kann , die nicht der aktuelle Raum sind, und dass sie so konzipiert ist, dass Sie Schichten und Schichtelemente in anderen Räumen als dem aktuellen Raum hinzufügen/entfernen können.

WARNUNG! Während Sie einen anderen Raum anvisieren, können Sie alle normalen Ebenenfunktionen verwenden , außer dass Sie keine Instanzen mit instance_create_layer() oder instance_create_depth()erstellen, noch ist die Ebenenfunktion layer_add_instance() verfügbar sein.

Syntax:

layer_set_target_room(room)

ArgumentTyp Beschreibung
room The room to target for all further layer functions

Rückgabe:

Beispiel:

layer_set_target_room(rm_Game);
var l = layer_get_id("SpriteAssets");
repeat(50)
{
    var _x = irandom(1000);
    var _y = irandom(1000);
    layer_sprite_create(l, _x, _y, spr_Trees);
}
layer_reset_target_room();

Die obige code setzt den Zielraum auf den Raum "rm_Game" und erhält dann die Ebenen-ID für die Ebene "SpriteAssets" in diesem Raum. Diese Ebenen-ID wird dann verwendet, um der Ebene 50 zufällige sprite Assets hinzuzufügen, bevor das Ebenenziel auf den aktuellen Raum zurückgesetzt wird.
