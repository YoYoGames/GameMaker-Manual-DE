layer_get_depth

Mit dieser Funktion können Sie den mit einer bestimmten Ebene verbundenen Tiefenwert ermitteln. Sie geben die Ebenen-ID (die Sie beim Erstellen der Ebene mit layer_create()) oder den Ebenennamen (als String - dies wirkt sich auf die Leistung aus) und die Funktion gibt die Tiefe der Ebene als reelle Zahl zurück. Wenn Sie also drei Ebenen mit den Tiefen -100, 0, 100 haben, werden die Ebenen in der Reihenfolge 100, 0, -100 gezeichnet, so dass die "oberste" Ebene (d.h. diejenige, die der Kameraansicht am nächsten ist und daher über alles andere gezeichnet wird) die Ebene mit der Tiefe -100 sein wird.

Die folgende Abbildung zeigt schematisch, wie die Tiefe bei Ebenen funktioniert:

Beachten Sie, dass Sie eine Fehlermeldung erhalten, wenn Sie einen ungültigen Wert für die Ebenen-ID angeben.

Syntax:

layer_get_depth(layer_id)

Argument Typ Beschreibung
layer_id String oder Ebenen-ID The unique ID value of the layer to get the depth of (or the layer name as a string)

Rückgabe:

Real

Beispiel:

if layer_get_depth(global.temp_layer) != -100
{
    layer_destroy(global.temp_layer);
    global.temp_layer = layer_create(-100);
}

Die obige code prüft die Tiefe einer in einer globalen Variablen gespeicherten Ebenen-ID, und wenn sie nicht -100 ist, wird die Ebene zerstört und mit der Tiefe von -100 neu erstellt.
