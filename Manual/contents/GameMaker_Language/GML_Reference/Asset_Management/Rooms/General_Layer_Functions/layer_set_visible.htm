layer_set_visible

Mit dieser Funktion können Sie die Sichtbarkeit einer Ebene ein- und ausschalten. Sie geben die Ebenen-ID (die Sie beim Erstellen der Ebene mit layer_create()) oder den Ebenennamen (als String - dies hat Auswirkungen auf die Leistung) sowie den Umschaltwert für die Ebene, wobei sichtbar für true und unsichtbar für false steht.

Wenn eine Ebene unsichtbar ist, wird nichts, was der Ebene zugewiesen ist, gezeichnet, und wenn irgendwelche Instanzen der Ebene zugewiesen sind, werden sie nicht einmal ihr Zeichnungsereignis ausführen (ähnlich wie wenn Sie die Instanzvariable visible auf false setzen). Beachten Sie, dass alle Instanzen, die bereits als unsichtbar gekennzeichnet sind, nicht als sichtbar gekennzeichnet werden, wenn die Ebene, auf der sie sich befinden, auf sichtbar gesetzt ist, da dies zwei unabhängige Einstellungen sind, obwohl ihr Verhalten das gleiche ist.

Syntax:

layer_set_visible(layer_id, visible)

Argument Typ Beschreibung
layer_id String oder Ebenen-ID The unique ID value of the layer to target (or the layer name as a string)
visible Boolesche Whether the layer should be visible (true) oder nicht (false)

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Instances");
if (layer_get_visible(lay_id))
{
    layer_set_visible(lay_id, false);
}
else
{
    layer_set_visible(lay_id, true);
}

Die obige code ruft den ID-Wert für die Ebene mit dem Namen "Instances" im Raumeditor ab und verwendet dann die ID, um zu prüfen, ob die Ebene sichtbar ist oder nicht, wobei die Sichtbarkeit der Ebene je nach dem zurückgegebenen Wert umgeschaltet wird.
