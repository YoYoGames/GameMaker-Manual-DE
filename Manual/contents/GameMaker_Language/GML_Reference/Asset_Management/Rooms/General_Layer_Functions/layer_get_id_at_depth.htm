Ebene_get_id_at_depth

Mit dieser Funktion können Sie die IDs aller Ebenen ermitteln, die einer bestimmten Tiefe zugeordnet sind. Sie geben die zu prüfende Tiefe an und die Funktion gibt ein Array mit 1 oder mehreren Einträgen zurück, je nachdem, ob es Ebenen in der angegebenen Tiefe gibt oder nicht. Wenn es keine Ebenen in der angegebenen Tiefe gibt, enthält das Array einen einzigen Eintrag an der Position [0] mit dem Wert -1. Wenn es jedoch Ebenen in der Tiefe gibt, wird für jede gefundene Ebene ein Eintrag im Array erstellt - der Eintragswert ist der eindeutige ID-Wert für eine Ebene.

Syntax:

layer_get_id_at_depth(depth)

Argument Typ Beschreibung
depth Real The depth to check and retrieve the layer IDs from.

Rückgabe:

Array

Beispiel:

var a = layer_get_id_at_depth(0);
if (a[0] != -1)
{
    for (var i = 0; i < array_length(a); i++;)
    {
        layer_destroy(a[i]);
    }
}

Die obige code ruft Daten über die Ebenen mit einer Tiefe von 0 ab. Es wird geprüft, ob Ebenen in dieser Tiefe vorhanden sind, und wenn ja, wird das zurückgegebene Array geparst und jede der gefundenen Ebenen zerstört.
