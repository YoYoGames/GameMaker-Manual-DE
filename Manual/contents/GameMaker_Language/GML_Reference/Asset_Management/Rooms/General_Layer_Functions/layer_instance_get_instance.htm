layer_instance_get_instance

Diese Funktion kann verwendet werden, um die eindeutige Instanz-ID des angegebenen Instanzelements zu ermitteln. Sie geben die ID des Instanzelements an (siehe das code Beispiel unten), und die Funktion gibt einen reellen Wert zurück, der die eindeutige Instanz-ID für das Element darstellt. Wenn das Element keine Instanz ist, gibt die Funktion -1 zurück.

Syntax:

layer_instance_get_instance(element_id)

Argument Typ Beschreibung
element_id Instanz Element ID The unique ID value of the instance element to get the ID of

Rückgabe:

Instance ID

Beispiel:

elements = layer_get_all_elements("Instances");
for (var i = 0; i < array_length(elements); i++)
{
     if (layer_get_element_type(elements[i]) == layerelementtype_instance)
     {
         var layerelement = elements[i];
         var inst = layer_instance_get_instance(layerelement);
         inst.x = inst.x + 10;
     }
}

Die obige code wird alle Instanz-Elemente auf einer Ebene zu überprüfen, dann erhalten ihre eindeutige ID-Wert und verwenden, dass sie 10px nach rechts zu bewegen.
