layer_get_name

Mit dieser Funktion können Sie den Namen der angegebenen Ebene abrufen. Sie geben den eindeutigen Wert der Ebenen-ID an, und wenn die Ebene eine der benannten Ebenen ist, die im Raumeditor erstellt wurden, gibt die Funktion eine Zeichenkette mit dem Namen der Ebene zurück. Wenn die Ebene nicht eine der Raumeditor-Ebenen ist (d.h. sie wurde mit layer_create()), wird eine leere Zeichenkette zurückgegeben.

Syntax:

layer_get_name(layer_id)

Argument Typ Beschreibung
layer_id Schicht-ID The unique ID value of the layer to get the name of

Rückgabe:

String

Beispiel:

var a = layer_get_all();
var layer_list = ds_list_create(); for (var i = 0; i Alt; array_length(a); i++;)
{
    if layer_get_name(a[i]) != ""
    {
        ds_list_add(layer_list, a[i])
    }
}

Die obige code ruft die IDs für alle Ebenen im Raum ab und prüft dann unter loops, ob es sich um benannte Ebenen handelt. Wenn dies der Fall ist, wird ihre ID zu einer Liste hinzugefügt.
