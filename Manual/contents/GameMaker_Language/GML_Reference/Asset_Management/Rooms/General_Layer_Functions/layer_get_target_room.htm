layer_get_target_room

Diese Funktion gibt den aktuellen Raum zurück, auf den die Ebenenfunktionen abzielen. Siehe layer_set_target_room() für weitere Informationen.

Syntax:

layer_get_target_room()

Rückgabe:

Room Asset

Beispiel:

if layer_get_target_room() != room
{
    layer_reset_target_room();
}

Die obige code prüft den aktuellen Zielraum, und wenn es nicht der aktuelle Raum ist, wird der Zielraum der Ebene zurückgesetzt.
