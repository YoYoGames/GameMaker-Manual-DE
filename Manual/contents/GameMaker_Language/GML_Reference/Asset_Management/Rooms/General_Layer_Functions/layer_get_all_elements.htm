layer_get_all_elements

Mit dieser Funktion können Sie die Element-IDs der angegebenen Ebene abrufen. Sie geben die ID der Ebene an (die Sie erhalten, wenn Sie die Ebene mit layer_create()) oder den Layernamen (als String - dies hat Auswirkungen auf die Leistung) und die Funktion gibt ein Array von IDs zurück, wobei jeder Eintrag im Array eine eindeutige ID für ein Element auf diesem Layer ist. Wenn es sich bei der Ebene beispielsweise um eine Vermögensebene handelt, wird das Array mit den ID-Werten für jedes sprite Vermögen aufgefüllt, das der Ebene zugewiesen ist. Beachten Sie, dass die Verwendung von code für die Arbeit mit Ebenen bedeutet, dass Sie der gleichen Ebene verschiedene Elementtypen zuweisen können - so können Sie z. B. sprite Assets zusammen mit Instanzen haben - in diesem Fall können Sie dann die Funktion layer_get_element_type() verwenden, um den Typ des Elements zu ermitteln, auf den sich die ID bezieht.

Syntax:

layer_get_all_elements(layer_id)

Argument Typ Beschreibung
layer_id Zeichenfolge oder Ebene ID The unique ID value of the layer to get the elements from (or the layer name as a string)

Rückgabe:

Array of Layer Element IDs

Beispiel:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if (layer_get_element_type(a[i]) == layerelementtype_sprite)
    {
        layer_sprite_destroy(a[i])
    }
}

Die obige code ruft die IDs für alle Instanzelemente ab, die der Ebene der Instanz zugeordnet sind, auf der code läuft. code prüft dann, ob eines der zurückgegebenen Elemente sprite Assets sind, und wenn ja, werden sie zerstört.
