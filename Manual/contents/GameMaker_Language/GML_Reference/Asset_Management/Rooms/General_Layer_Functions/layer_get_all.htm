layer_get_all

Diese Funktion gibt ein Array zurück, das die eindeutigen ID-Werte der einzelnen Ebenen im Raum enthält.

Syntax:

layer_get_all()

Rückgabe:

Array (populated with Layer IDs)

Beispiel:

var a = layer_get_all();
for (var i = 0; i < array_length(a); i++;)
{
    layer_destroy(a[i]);
}

Die obige code ruft alle Ebenen in einem Raum ab und fügt ihre ID-Werte in ein Array ein. Dieses Array wird dann geparst, um die Raumebenen zu zerstören.
