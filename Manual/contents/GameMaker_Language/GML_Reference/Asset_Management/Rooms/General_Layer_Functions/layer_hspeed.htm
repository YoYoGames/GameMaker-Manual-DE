Schicht_Geschwindigkeit

Mit dieser Funktion können Sie die horizontale Geschwindigkeit (in Pixeln pro Spielframe) der Ebene innerhalb des aktuell ausgewählten Raums einstellen. Du gibst die Ebenen-ID (die du erhältst, wenn du die Ebene mit layer_create()) oder den Ebenennamen (als String - dies wirkt sich auf die Leistung aus) und den einzustellenden Geschwindigkeitswert, wobei ein positiver Wert nach rechts und ein negativer Wert nach links zeigt.

Syntax:

layer_hspeed(layer_id, hspd)

Argument Typ Beschreibung
layer_id String oder Ebenen-ID The unique ID value of the layer to set the horizontal speed of
hspd Real The horizontal speed (in pixels per game frame) to set

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
    layer_hspeed(lay_id, 0);
    layer_vspeed(lay_id, 0);
}

Die obige code prüft die horizontalen und vertikalen Geschwindigkeiten der Ebene und setzt sie auf 0, wenn sie nicht beide auf 0 gesetzt sind.
