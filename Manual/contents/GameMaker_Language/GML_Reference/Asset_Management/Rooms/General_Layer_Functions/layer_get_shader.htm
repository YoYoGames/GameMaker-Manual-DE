layer_get_shader

Diese Funktion kann verwendet werden, um zu prüfen, ob der angegebenen Ebene ein Shader zugewiesen ist. Sie geben die Ebenen-ID (die Sie erhalten, wenn Sie die Ebene mit layer_create()) oder den Ebenennamen (als String - dies hat Auswirkungen auf die Leistung), und die Funktion gibt entweder den Shader-Index des zugewiesenen Shaders oder -1 zurück, wenn kein Shader zugewiesen ist.

Syntax:

layer_get_shader(layer_id)

Argument Typ Beschreibung
layer_id String oder Ebenen-ID The unique ID value of the layer to target (or the layer name as a string)

Rückgabe:

Shader Asset or -1 (if no shader is assigned)

Beispiel:

if layer_get_shader(layer) == -1
{
    layer_shader(layer, shd_Sepia);
}

Die obige code prüft, ob der Ebene, auf der die Instanz code läuft, ein Shader zugewiesen ist, und wenn nicht, wird ein Shader zugewiesen.
