Ebene_erstellen

Diese Funktion wird verwendet, um eine neue Ebene im aktuellen Raum zu erstellen. Sie geben die Tiefe an, die die neue Ebene einnehmen soll, und die Funktion gibt den eindeutigen Wert der Ebenen-ID zurück, der dann für den Zugriff auf die Ebene in weiteren Funktionen verwendet werden kann. Sie können auch einen optionalen "Namen" argument angeben, der es Ihnen ermöglicht, der Ebene einen bestimmten Namen (als String) zu geben, und anstelle der Ebenen-ID für den Zugriff auf die Ebene diesen Namensstring zu verwenden, obwohl zu beachten ist, dass die Verwendung von Namensstrings größere Auswirkungen auf die Leistung hat als die Verwendung nur des ID-Werts. Wenn Sie den Namen der Ebene nicht angeben, wird der erstellten Ebene ein Name im Format "_layer_XXX" zugewiesen, wobei "XXX" ein Hexadezimalwert ist, um der Ebene einen eindeutigen Namen zu geben.

Wenn Sie einen Tiefenwert angeben, ist dieser umso höher, je weiter die Ebene von der Kamera entfernt ist, und umso niedriger, je näher die Ebene an der Kamera liegt. Wenn Sie also drei Ebenen mit den Tiefen -100, 0, 100 haben, werden die Ebenen in der Reihenfolge 100, 0, -100 gezeichnet, so dass die "oberste" Ebene (d. h. die Ebene, die der Kameraansicht am nächsten ist und daher über alles andere gezeichnet wird) die Ebene mit der Tiefe -100 ist.

Es ist zu beachten, dass Sie zwar mehr als eine Ebene einer einzigen Tiefe zuweisen können, aber nicht garantieren können, dass die Ebenen auf derselben Tiefe in einer bestimmten Reihenfolge gezeichnet werden, weshalb es am besten ist, jeder Tiefe nur eine Ebene zuzuweisen. Beachten Sie auch, dass die minimale und maximale Ebenentiefe -16000 bis 16000 beträgt und alles außerhalb dieser Tiefen nicht gerendert wird. Wenn Sie eine Tiefe außerhalb dieses Bereichs benötigen, müssen Sie die Funktion layer_force_draw_depth().

Syntax:

layer_create(depth, [name])

Argument Typ Beschreibung
depth Real The depth that the layer has to be created at.
name Zeichenfolge OPTIONAL Der Name, den die Ebene erhalten soll (eine Zeichenkette).

Rückgabe:

Layer ID

Beispiel:

global.Bullet_Layer = layer_create(-100);

Die obige code erstellt eine neue Ebene mit einem Tiefenwert von -100 und weist ihren ID-Wert einer globalen Variablen zu.
