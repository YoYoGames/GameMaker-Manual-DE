layer_get_element_layer

Mit dieser Funktion können Sie die Layer-ID ermitteln, auf der sich das angegebene Element befindet. Sie geben den eindeutigen Wert der Element-ID an (z. B. wie von der Funktion, die das Element erstellt hat, oder vom Raumeditor zurückgegeben) und die Funktion gibt die eindeutige ID der Ebene zurück, auf der sich das Element befindet. Wenn die angegebene Element-ID nicht gültig ist, gibt die Funktion -1 zurück.

Syntax:

layer_get_element_layer(element_id)

Argument Typ Beschreibung
element_id Ebene Element-ID The unique ID value of the element to get the layer of

Rückgabe:

Layer ID or -1 (if the element is invalid)

Beispiel:

element = layer_get_element_layer(asset_1);

Die obige code ruft die Ebenen-ID für das Element mit der in der Variablen "asset_1" gespeicherten ID ab und speichert sie in einer Variablen.
