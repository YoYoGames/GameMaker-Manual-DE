room_instance_add

Mit dieser Funktion können Sie eine Instanz in einen beliebigen Raum, der nicht der aktuelle ist, und an einer beliebigen Position innerhalb dieses Raums hinzufügen. Die Funktion gibt die eindeutige id der Instanz zurück, die dann in weiteren Funktionsaufrufen verwendet werden kann, um Eigenschaften etc. der Instanz zu setzen, aber erst, wenn das Spiel den angegebenen Raum betreten hat. Wenn du eine Instanz im aktuellen Raum erstellen möchtest, solltest du die Funktion instance_create_layer(). Beachte, dass der Aufruf dieser Funktion bei einem Raum-Asset, das im Asset-Browser erstellt wurde , die Instanz dauerhaft zum Raum hinzufügt, und dass selbst der Aufruf von game_restart() den Raum nicht in seinen ursprünglichen Zustand zurückversetzt (nur wenn du das Spiel beendest und es erneut öffnest, wird der Raum wieder in seinem ursprünglichen Zustand beginnen).

Syntax:

room_instance_add(index, x, y, obj);

ArgumentTyp Beschreibung
index The index of the room to add an object instance to.
x The x position of the new instance.
y The y position of the new instance.
obj The object to add an instance of.

Rückgabe:

Instance ID

Beispiel:

global.rm = room_add();
room_assign(rm_Base, global.rm);
room_instance_add(global.rm, 100, 100, obj_Player);

Die obige code fügt dem Spiel einen neuen Raum hinzu und kopiert dann den Inhalt des Raums mit dem Index "rm_Base" in diesen Raum. Anschließend wird eine Instanz des Objekts "obj_player" an der Position 100,100 dieses neuen Raums hinzugefügt.
