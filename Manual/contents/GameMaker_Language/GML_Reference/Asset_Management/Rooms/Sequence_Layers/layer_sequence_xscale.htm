schicht_sequenz_xskala

Mit dieser Funktion können Sie die X-Skala für das angegebene Element sequence einstellen. Sie geben die sequence Element-ID an, wie sie von layer_sequence_create() oder von einer der Ebenenelement-Funktionen zurückgegeben wird, zusammen mit der neuen Skalierung, die auf der X-Achse eingestellt werden soll, und die sequence wird um diesen Betrag skaliert. Ein Maßstab von 1 bedeutet keine Skalierung (1:1), kleinere Werte skalieren nach unten (0,5 zum Beispiel halbiert die Breite von sequence), größere Werte skalieren nach oben und negative Werte spiegeln das sequence um seinen Ursprung und skalieren es, es sei denn, der verwendete Wert ist genau -1 (in diesem Fall wird das sequence nur um seinen Ursprung gespiegelt und nicht skaliert).

Es ist sehr wichtig zu beachten, dass die Anwendung einer ungleichmäßigen Skalierung (z.B. Skalierung der X-Achse um 3 und der Y-Achse um 2) auf sequence Elemente, die eine Instanz enthalten, die Rotation verwendet, zu Problemen mit der Instanzzeichnung, Kollisionen, Culling und vielen anderen Dingen führen kann. Grundsätzlich gilt: Wenn Ihre sequence auf Instanz-Eigenschaften angewiesen ist, empfehlen wir Ihnen nicht, eine ungleichmäßige Skalierung mit einer Instanz-Rotation zu kombinieren.

Beachten Sie auch, dass Sie die Variablen image_xscale \ image_yscale \ image_angle \ x \ y für alle Instanzen in einer sequence, die diese Funktion verwendet, nicht ändern sollten, da sie überschrieben werden, sobald die sequence nach dem Aufruf der Funktion aktualisiert wird.

Syntax:

layer_sequence_xscale(sequence_element_id, xscale)

ArgumentTyp Beschreibung
sequence_element_id The unique ID value of the sequence element to target
xscale The new X axis scale value to apply to the sequence element

Rückgabe:

Beispiel:

if seq_scale < 2
{
    seq_scale += 0.01;
    layer_sequence_xscale(my_seq, seq_scale);
    layer_sequence_yscale(my_seq, seq_scale);
}

Die obige code prüft den Wert in der Variablen seq_scale, und wenn er kleiner als 2 ist, dann addiert sie ihn und verwendet den Wert, um die X- und Y-Skala des Elements sequence einzustellen, auf das in der Variablen my_seq verwiesen wird.
