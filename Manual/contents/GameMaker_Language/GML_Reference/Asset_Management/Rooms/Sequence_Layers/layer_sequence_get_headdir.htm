layer_sequence_get_headdir

Mit dieser Funktion geben Sie die sequence Element-ID an - wie sie von layer_sequence_create() oder von einer der Funktionen für Ebenenelemente zurückgegeben wird - und sie gibt die aktuelle Abspielrichtung zurück, die eine der unten aufgeführten Konstanten ist.

Syntax:

layer_sequence_get_headdir(sequence_element_id)

ArgumentTyp Beschreibung
sequence_element_id The unique ID value of the sequence element to target

Rückgabe:

Konstante Beschreibung Wert
seq_dir_right Die sequence spielt die Bilder in einer inkrementellen Reihenfolge von links nach rechts ab. 1
seq_dir_left Die sequence spielt die Bilder in absteigender Reihenfolge von rechts nach links ab. -1

Beispiel:

if layer_sequence_get_headdir(title_sequence) != seq_dir_left
{
    layer_sequence_headdir(title_sequence, seq_dir_left);
}

Die obige code prüft die aktuelle Abspielrichtung der sequence Element-ID, die in der Variablen "title_sequence" gespeichert ist, und wenn sie nicht auf seq_dir_left gesetzt ist, wird sie auf diesen Wert gesetzt.
