layer_sequence_is_paused

Mit dieser Funktion geben Sie die Element-ID sequence an - wie sie von layer_sequence_create() oder von einer der Ebenen-Element-Funktionen zurückgegeben wird - und sie prüft, ob das Element sequence gerade pausiert oder nicht, und gibt true zurück, wenn es pausiert, und false, wenn es nicht pausiert.

Syntax:

layer_sequence_is_paused(sequence_element_id)

ArgumentTyp Beschreibung
sequence_element_id The unique ID value of the sequence element to target

Rückgabe:

Beispiel:

if layer_sequence_is_paused(title_sequence) != 0
{
    layer_sequence_play(title_sequence);
}

Die obige code prüft, ob die in der Variablen "title_sequence" gespeicherte sequence Element-ID pausiert ist, und wenn ja, beginnt sie mit der Wiedergabe.
