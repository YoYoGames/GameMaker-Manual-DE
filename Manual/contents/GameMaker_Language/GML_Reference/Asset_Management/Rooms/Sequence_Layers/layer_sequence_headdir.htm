layer_sequence_headdir

Mit dieser Funktion können Sie die Richtung für den angegebenen sequence Abspielkopf festlegen. Sie geben die sequence Element-ID an, wie sie von layer_sequence_create() oder einer der Ebenenelementfunktionen zurückgegeben wird, und geben dann die Abspielkopfrichtung an, die eine der folgenden Konstanten sein sollte:

Konstante Beschreibung Wert
seq_dir_right Auf sequence werden die Bilder in einer inkrementellen Reihenfolge von links nach rechts abgespielt. 1
seq_dir_left Die sequence spielt die Bilder in absteigender Reihenfolge von rechts nach links ab. -1

Syntax:

layer_sequence_headdir(sequence_element_id, direction)

Argument Typ Beschreibung
sequence_element_id Sequenz Element ID The unique ID value of the sequence element to target
direction Sequenz Richtung Konstante The playhead direction, a constant, listed above

Rückgabe:

N/A

Beispiel:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_headdir(_seq, seq_dir_left);

Die obige code erstellt eine neue sequence und speichert ihre ID in einer lokalen Variablen. Diese ID wird dann verwendet, um die Richtung des Abspielkopfes so einzustellen, dass Frames dekrementiert werden (Wiedergabe von rechts nach links).
