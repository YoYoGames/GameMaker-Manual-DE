layer_sequence_headpos

Mit dieser Funktion können Sie die Abspielposition eines sequence Elements auf einen bestimmten Frame setzen. Sie geben die sequence Element-ID an, wie sie von layer_sequence_create() oder von einer der Ebenenelement-Funktionen zurückgegeben wird, zusammen mit der neu zu setzenden Position. Beachten Sie, dass die Position in Frames angegeben wird. Wenn Sie einen Wert angeben, der größer als die Gesamtzahl der Frames (oder kleiner als 0) ist, hängt die tatsächliche endgültige Abspielposition von der Art der sequence -Wiedergabe ab, die gemäß diesen Regeln ausgewählt wurde:

  • Keine Schleifenbildung: Der Abspielkopf wird am Ende von loop (oder am Anfang, wenn der angegebene Wert negativ ist) festgehalten.
  • Schleifen: Der Abspielkopf springt zurück an den Anfang von sequence plus die zusätzlichen Frames, z. B.: Wenn Sie den Abspielkopf auf einem sequence mit 30 Frames auf eine Position von Frame 45 setzen, ist die tatsächliche Position des Abspielkopfes 15. Wenn Sie einen negativen Wert angeben, wird der sequence Abspielkopf in umgekehrter Richtung umlaufen.
  • Ping-Pong-Schleife: Der Abspielkopf rückt bis zum Ende vor und geht dann um die zusätzliche Anzahl von Bildern zurück, die über die gesamte Bildlänge hinausgehen, wobei die Wiedergaberichtung umgekehrt wird, z. B.: Wenn die sequence 30 Frames hat und Sie den Abspielkopf auf 40 setzen, ist die endgültige Abspielkopfposition 20 und die Richtung wird umgekehrt (bis 0). Beachten Sie, dass Sie für die Ping-Pong-Wiedergabe von sequenceniemals einen negativen Wert angeben sollten, da dies zu undefinierten Ergebnissen führen kann.

Wenn Sie die Kopfposition auf diese Weise einstellen, wird die Wiedergabe nicht gestoppt, sondern sequence wird einfach an der neuen Abspielkopfposition fortgesetzt, es sei denn, die Wiedergabe wird angehalten.

Syntax:

layer_sequence_headpos(sequence_element_id, position)

Argument Typ Beschreibung
sequence_element_id Sequenz Element ID The unique ID value of the sequence element to target
position Real The position within the sequence (in frames) to set the playhead position to (can be a decimal value)

Rückgabe:

N/A

Beispiel:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if layer_get_element_type(a[i]) == layerelementtype_sequence
    {
        layer_sequence_headpos(a[i], 0)
    }
}

Die obige code ruft die IDs für alle Elemente ab, die der Ebene der Instanz zugewiesen sind, auf der code läuft. code prüft dann, ob eines der zurückgegebenen Elemente sequence Assets sind, und wenn ja, wird die Abspielposition auf 0 gesetzt.
