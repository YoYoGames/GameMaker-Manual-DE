layer_sequence_get_angle

Mit dieser Funktion geben Sie die Element-ID von sequence an - wie sie von layer_sequence_create() oder von einer der Funktionen für Ebenenelemente zurückgegeben wird - und sie gibt den aktuellen Winkel des Elements sequence im Spielraum zurück. Beachten Sie, dass die Winkel in Grad zurückgegeben werden, wobei 0º nach rechts, 90º nach oben, 180º nach links und 270º nach unten zeigt.

Syntax:

layer_sequence_get_angle(sequence_element_id)

ArgumentTyp Beschreibung
sequence_element_id The unique ID value of the sequence element to target

Rückgabe:

Beispiel:

var _ang = layer_sequence_get_angle(title_sequence)
if (_ang > 0)
{
    _ang -= 1;
    layer_sequence_angle(title_sequence, _ang);
}

Die obige code ruft den aktuellen Winkel des sequence -Elements mit der in der Variablen "title_sequence" gespeicherten ID ab, und wenn dieser nicht 0 ist, wird 1 vom aktuellen Winkel subtrahiert.
