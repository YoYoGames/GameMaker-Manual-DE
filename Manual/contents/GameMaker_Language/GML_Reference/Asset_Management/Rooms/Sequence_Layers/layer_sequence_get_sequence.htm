layer_sequence_get_sequence

Mit dieser Funktion geben Sie die Element-ID von sequence an - wie sie von layer_sequence_create() oder von einer der Ebenen-Elementfunktionen zurückgegeben wird - und sie gibt die sequence Objektstruktur zurück. Diese Funktion umgeht die Notwendigkeit, zuerst die sequence Instanzstruktur zu erhalten und erlaubt Ihnen, direkt auf die sequence Daten zuzugreifen. Weitere Informationen über das Format der sequence object struct finden Sie auf dieser Seite.

Syntax:

layer_sequence_get_sequence(sequence_element_id)

ArgumentTyp Beschreibung
sequence_element_id The unique ID value of the sequence element to target

Rückgabe:

Sequence Object Struct

Beispiel:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_sequence(_seq);
_struct.playbackSpeedType = spritespeed_framespersecond;
_struct.playbackSpeed = 30;

Die obige code erstellt ein neues sequence Element und ruft dann dessen sequence Datenstruktur ab. Diese wird dann verwendet, um die Abspielgeschwindigkeit und den Typ des sequence zu ändern.
