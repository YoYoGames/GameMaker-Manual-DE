layer_sequence_get_instance

Mit dieser Funktion geben Sie die Element-ID von sequence an - wie sie von layer_sequence_create() oder von einer der Ebenen-Element-Funktionen zurückgegeben wird - und sie gibt die sequence instance struct zurück. Mehr über das Format der sequence Instanzstruktur erfahren Sie auf dieser Seite.

Syntax:

layer_sequence_get_instance(sequence_element_id)

Argument Typ Beschreibung
sequence_element_id Sequenz Element ID The unique ID value of the sequence element to target

Rückgabe:

Sequence Instance Struct

Beispiel:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_instance(_seq);
_struct.speedScale = 0.5;

Die obige code erstellt ein neues sequence Element und ruft dann dessen sequence Instanz-ID ab. Diese wird dann verwendet, um die Skalierungseigenschaft der Wiedergabegeschwindigkeit der sequence Instanz zu ändern.
