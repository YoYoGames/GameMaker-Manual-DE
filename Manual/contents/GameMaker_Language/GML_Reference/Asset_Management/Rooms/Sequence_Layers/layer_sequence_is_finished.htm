layer_sequence_is_finished

Mit dieser Funktion geben Sie die Element-ID sequence an - wie sie von layer_sequence_create() oder von einer der Ebenen-Element-Funktionen zurückgegeben wird - und sie prüft, ob die Wiedergabe von sequence beendet ist oder nicht und gibt true zurück, wenn dies der Fall ist, und false, wenn dies nicht der Fall ist. Beachten Sie, dass dies nur anwendbar ist, wenn sequence nicht auf loop oder ping-pong im Wiedergabemodus eingestellt ist.

Syntax:

layer_sequence_is_finished(sequence_element_id)

ArgumentTyp Beschreibung
sequence_element_id The unique ID value of the sequence element to target

Rückgabe:

Beispiel:

if layer_sequence_is_finished(title_sequence) != 0
{
    alarm[0] = room_speed * 3;
    layer_sequence_play(title_sequence);
}

Die obige code prüft, ob die in der Variablen "title_sequence" gespeicherte sequence Element-ID beendet ist, und wenn dies der Fall ist, wird die Wiedergabe erneut gestartet und ein Alarm ausgelöst.
