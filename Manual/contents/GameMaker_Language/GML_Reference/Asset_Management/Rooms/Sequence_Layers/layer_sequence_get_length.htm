schicht_sequenz_holen_laenge

Mit dieser Funktion geben Sie die Element-ID von sequence an - wie sie von layer_sequence_create() oder von einer der Ebenen-Element-Funktionen zurückgegeben wird - und sie gibt die Länge des sequence zurück. Dies ist die Anzahl der Frames, für die das sequence läuft.

Syntax:

layer_sequence_get_length(sequence_element_id)

ArgumentTyp Beschreibung
sequence_element_id The unique ID value of the sequence element to target

Rückgabe:

Beispiel:

var _frames = layer_sequence_get_length(my_seq);
alarm[0] = frames;

Die obige code ruft die Anzahl der Frames ab, die ein sequence durchlaufen wird, und verwendet diesen Wert, um einen Alarm zu setzen.
