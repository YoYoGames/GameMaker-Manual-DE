layer_sequence_get_xscale

Mit dieser Funktion geben Sie die Element-ID von sequence an - wie sie von layer_sequence_create() oder von einer der Funktionen für Ebenenelemente zurückgegeben wird - und sie gibt den aktuellen Maßstab entlang der X-Achse des Elements sequence im Spielraum zurück.

Syntax:

layer_sequence_get_xscale(sequence_element_id)

ArgumentTyp Beschreibung
sequence_element_id The unique ID value of the sequence element to target

Rückgabe:

Beispiel:

var _xs = layer_sequence_get_xscale(title_sequence)
if _xs < 1
{
    _xs += 0.01;
    layer_sequence_xscale(title_sequence, _xs);
}

Die obige code ruft die aktuelle Skala entlang der X-Achse des sequence -Elements mit der in der Variablen "title_sequence" gespeicherten ID ab, und wenn sie kleiner als 1 ist, wird 0,01 zur aktuellen X-Skala hinzugefügt.
