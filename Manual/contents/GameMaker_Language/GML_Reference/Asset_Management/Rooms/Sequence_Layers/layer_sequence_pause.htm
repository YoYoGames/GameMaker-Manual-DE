schicht_sequenz_pause

Mit dieser Funktion können Sie die Wiedergabe der angegebenen sequence unterbrechen. Sie geben die Element-ID von sequence an, wie sie von layer_sequence_create() oder von einer der Ebenen-Element-Funktionen zurückgegeben wurde, und die Funktion hält die Wiedergabe von sequence an, bis Sie die Wiedergabe mit der Funktion layer_sequence_play().

Syntax:

layer_sequence_pause(sequence_element_id)

Argument Typ Beschreibung
sequence_element_id Sequenz Element ID The unique ID value of the sequence element to target

Rückgabe:

N/A

Beispiel:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
if (global.Pause)
{
    layer_sequence_pause(_seq);
}

Die obige code erstellt eine neue sequence und speichert ihre ID in einer lokalen Variablen. Dann wird geprüft, ob das Spiel pausiert, und wenn ja, wird die Wiedergabe von sequence angehalten.
