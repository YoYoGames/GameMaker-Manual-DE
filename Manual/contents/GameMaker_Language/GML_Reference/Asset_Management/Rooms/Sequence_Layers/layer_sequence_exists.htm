layer_sequence_exists

Mit dieser Funktion können Sie prüfen, ob ein sequence Element auf der angegebenen Ebene existiert. Sie geben die Ebenen-ID an, die eine Zeichenkette des Ebenennamens sein kann - wie im Raumeditor definiert - oder die eindeutige Ebenen-ID - wie von der Funktion layer_get_id()zurückgegeben wird, sowie die sequence Element-ID - wie sie von layer_sequence_create() oder von einer der Ebenen-Element-Funktionen zurückgegeben wird - und es wird true zurückgegeben, wenn das gegebene Element existiert oder false andernfalls.

Syntax:

layer_sequence_exists(layer_id, sequence_element_id)

Argument Typ Beschreibung
layer_ID String oder Ebenen-ID The unique ID or name of the layer to check
sequence_element_id Sequenz Element ID The unique ID value of the sequence element to target

Rückgabe:

Boolean

Beispiel:

if !layer_sequence_exists("Background", my_seq)
{
    my_seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
    layer_sequence_pause(my_seq);
}

Das obige code prüft, ob das angegebene Element sequence existiert, und wenn nicht, wird ein neues sequence auf der angegebenen Ebene erstellt und dann angehalten.
