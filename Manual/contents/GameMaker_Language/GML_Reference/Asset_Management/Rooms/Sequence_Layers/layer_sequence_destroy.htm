layer_sequence_destroy

Mit dieser Funktion können Sie ein sequence Element zerstören (aus dem Raum entfernen). Sie geben die sequence Element-ID an, wie sie von layer_sequence_create() oder von einer der Funktionen für Ebenenelemente zurückgegeben wurde, und das sequence wird zerstört.

Syntax:

layer_sequence_destroy(sequence_element_id)

ArgumentTyp Beschreibung
sequence_element_id The unique ID value of the sequence element to target

Rückgabe:

Beispiel:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if layer_get_element_type(a[i]) == layerelementtype_sequence
    {
        layer_sequence_destroy(a[i])
    }
}

Die obige code ruft die IDs für alle Elemente ab, die der Ebene der Instanz zugeordnet sind, auf der code läuft. code prüft dann, ob eines der zurückgegebenen Elemente sequence Assets sind, und wenn ja, werden sie zerstört.
