schicht_sequenz_erstellen

Mit dieser Funktion können Sie eine Instanz eines sequence Assets auf der angegebenen Ebene erstellen. Sie geben die Ebenen-ID an, die eine Zeichenkette des Ebenennamens sein kann - wie im Raumeditor definiert - oder die eindeutige Ebenen-ID - wie von der Funktion layer_get_id()zurückgegeben wird, sowie die X- und Y-Position im Raum, an der die sequence erstellt werden soll, und schließlich die ID der sequence, die erstellt werden soll. Die sequence ID ist die Namenskonstante, die Sie im Asset Browser für das sequence definiert haben. Die Funktion gibt die eindeutige ID des sequence Elements zurück, die dann in allen weiteren Layerfunktionen für sequences verwendet werden kann, oder sie kann verwendet werden, um die sequence Instanzstruktur mit der Funktion layer_sequence_get_instance().

Es ist erwähnenswert, dass, wenn sequence Objektspuren enthält, ihre Instanzen erstellt werden, sobald das sequence Element selbst erstellt wird, unabhängig davon, wo ihre Asset-Keys auf dem Dope Sheet positioniert sind. Der Sequence Controller schaltet einfach die Sichtbarkeit der Instanz um, um sie je nach Position und Dauer des Asset-Schlüssels der Spur ein- und auszublenden, und erstellt und zerstört sie nicht wiederholt. Daher werden die Instanzen ihre "Create"-Ereignisse ausführen, wenn das Element sequence erstellt wird und nicht, wenn ihre Asset-Keys beginnen.

Syntax:

layer_sequence_create(layer_id, x, y, sequence_id)

Argument Typ Beschreibung
layer_id String oder Ebenen-ID The unique ID value of the instance layer to target
x Real The x position in the room to create the sequence at
y Real The y position in the room to create the sequence at
sequence_id Sequenz Asset The sequence asset to use, as defined in the Asset Browser

Rückgabe:

Sequence Element ID

Beispiel:

var _s = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_pause(_s);

Die obige code erstellt eine neue sequence auf der Ebene "Hintergrund" und hält sie dann an.
