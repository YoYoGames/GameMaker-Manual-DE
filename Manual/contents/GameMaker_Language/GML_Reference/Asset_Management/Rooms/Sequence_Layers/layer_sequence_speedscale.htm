schicht_sequenz_geschwindigkeitsskala

Mit dieser Funktion können Sie die Abspielgeschwindigkeit der angegebenen sequence ändern. Sie geben die sequence Element-ID an, wie sie von layer_sequence_create() oder von einer der Funktionen für Ebenenelemente zurückgegeben wird, und dann einen Wert für die Geschwindigkeitsskala. Dieser Wert ist ein Multiplikator, wobei 1 die Standard-Wiedergabegeschwindigkeit ist und Werte kleiner als 1 die Wiedergabe verlangsamen und Werte größer als 1 sie beschleunigen, z.B.: ein Wert von 0,5 wäre die halbe Wiedergabegeschwindigkeit, während ein Wert von 2 die doppelte Wiedergabegeschwindigkeit wäre.

Syntax:

layer_sequence_speedscale(sequence_element_id, speedscale)

Argument Typ Beschreibung
sequence_element_id Sequenz Element ID The unique ID value of the sequence element to target
speedscale Real The speed scale to use (a multiplier), default is 1

Rückgabe:

N/A

Beispiel:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_speedscale(_seq, 0.75);

Die obige code erstellt eine neue sequence auf der Ebene der aufrufenden Instanz und setzt dann ihren Geschwindigkeitsskalierungswert auf 0,75 (drei Viertel der Standard-Wiedergabegeschwindigkeit).
