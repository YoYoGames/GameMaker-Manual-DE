Sequenz-Ebenen

In diesem Abschnitt finden Sie die Funktionen zum Hinzufügen, Entfernen und Bearbeiten des Verhaltens von Sequenzen in einem Spielraum in Bezug auf die Ebene, auf der sie sich befinden. Es sei darauf hingewiesen, dass sequences den Objekten insofern ähnelt, als dass es, wenn es auf einer Ebene in einem Raum erstellt wird, als eine Instanz des Hauptobjekts sequence aus dem Asset-Browser betrachtet wird. Wenn Sie also ein sequence zu einer Ebene in einem Raum hinzufügen, erstellen Sie eine Instanz des Basisobjekts sequence, und diese sequence Instanz wird als Element innerhalb einer Raumebene hinzugefügt. Die sequence Instanz steuert Dinge wie die Abspielgeschwindigkeit und -richtung und enthält selbst eine andere sequence Datenstruktur, die die eigentlichen sequence Trackdaten enthält. Dies ist wichtig, da die unten aufgeführten Funktionen sowohl auf die sequence Element-ID (die ID des sequence auf der Ebene) als auch auf die sequence Instanz-ID (die tatsächliche ID des sequence, auf die das Element verweist) und die sequence Datenstruktur (die alle sequence Daten enthält) verweisen.
