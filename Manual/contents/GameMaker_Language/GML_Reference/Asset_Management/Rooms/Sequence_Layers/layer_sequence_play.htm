schicht_sequenz_abspielen

Mit dieser Funktion können Sie die Wiedergabe der angegebenen sequence starten. Sie geben die Element-ID von sequence an, wie sie von layer_sequence_create() oder von einer der Ebenen-Element-Funktionen zurückgegeben wird, und die Funktion spielt die sequence ab, die Sie dann bei Bedarf mit der Funktion layer_sequence_pause().

WICHTIG! Wenn dein sequence irgendwelche Instanzen enthält, sollten diese Instanzen ihre image_xscale / image_yscale / image_angle / x / y Variablen nicht ändern, da sie überschrieben werden, wenn die sequence jeden Schritt nach dem Start des Abspielens aktualisiert.

Syntax:

layer_sequence_play(sequence_element_id)

Argument Typ Beschreibung
sequence_element_id Sequenz Element ID The unique ID value of the sequence element to target

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    var a = layer_get_all_elements(layer);
    for (var i = 0; i < array_length(a); i++;)
    {
        if layer_get_element_type(a[i]) == layerelementtype_sequence
        {
            if (global.Pause)
            {
                layer_sequence_pause(a[i]);
            }
            else
            {
                layer_sequence_play(a[i]);
            }
        }
    }
}

Die obige code prüft, ob das Spiel angehalten wurde oder nicht, wenn eine Taste gedrückt wird. Wenn das Spiel pausiert ist, durchläuft loops alle sequence Elemente auf der aktuellen Ebene (der Ebene der aufrufenden Instanz) und pausiert ihre Wiedergabe. Wenn das Spiel nicht pausiert ist, beginnt die loop ihre Wiedergabe erneut.
