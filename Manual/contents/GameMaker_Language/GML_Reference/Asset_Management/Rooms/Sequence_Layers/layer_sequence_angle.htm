layer_sequence_angle

Mit dieser Funktion können Sie den Drehwinkel für das angegebene sequence Element festlegen. Sie geben die sequence Element-ID an, wie sie von layer_sequence_create() oder von einer der Ebenenelement-Funktionen zurückgegeben wird, zusammen mit dem neu zu setzenden Winkel, und sequence wird um seinen Ursprung in die neue Position gedreht. Die Winkel in GameMaker werden mit 0º nach rechts berechnet und verlaufen gegen den Uhrzeigersinn - 90º ist also nach oben, 180º nach links und 270º nach unten - und der Standardwinkel für sequence wäre 0º.

Beachten Sie, dass Sie die Variablen image_xscale \ image_yscale \ image_angle \ x \ y für alle Instanzen in einer sequence, die diese Funktion verwendet, nicht ändern sollten, da sie überschrieben werden, sobald die sequence nach dem Aufruf der Funktion aktualisiert wird.

Syntax:

layer_sequence_angle(sequence_element_id, angle)

Argument Typ Beschreibung
sequence_element_id Sequenz Element ID The unique ID value of the sequence element to target
angle Real The new angle to rotate the sequence element to

Rückgabe:

N/A

Beispiel:

if current_angle < 90
{
    current_angle += 1;
    layer_sequence_angle(my_seq, current_angle);
}

Die obige code prüft den Wert in der Variablen current_angle, und wenn er kleiner als 90 ist, addiert sie ihn und verwendet den Wert, um den Winkel des Elements sequence einzustellen, auf das in der Variablen my_seq verwiesen wird.
