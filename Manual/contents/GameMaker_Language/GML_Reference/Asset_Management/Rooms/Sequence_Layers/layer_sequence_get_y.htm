layer_sequence_get_y

Mit dieser Funktion geben Sie die Element-ID von sequence an - wie sie von layer_sequence_create() oder von einer der Funktionen für Ebenenelemente zurückgegeben wird - und die Funktion gibt die aktuelle Y-Position im Spielraum für sequence zurück.

Syntax:

layer_sequence_get_y(sequence_element_id)

Argument Typ Beschreibung
sequence_element_id Sequenz Element ID The unique ID value of the sequence element to target

Rückgabe:

Real

Beispiel:

if layer_sequence_get_y(title_sequence) != room_height / 2
{
    layer_sequence_y(title_sequence, room_height / 2);
}

Die obige code prüft die Y-Position der sequence Element-ID, die in der Variablen "title_sequence" gespeichert ist, und wenn sie nicht auf die halbe Raumhöhe gesetzt ist, wird sie auf diesen Wert gesetzt.
