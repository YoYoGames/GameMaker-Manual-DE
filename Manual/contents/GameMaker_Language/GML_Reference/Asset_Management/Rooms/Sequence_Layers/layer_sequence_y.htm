schicht_sequenz_y

Mit dieser Funktion können Sie die Position entlang der Y-Achse (vertikal) des Raumes für das angegebene sequence Element festlegen. Sie geben die sequence Element-ID an, wie sie von layer_sequence_create() oder von einer der Layer-Element-Funktionen zurückgegebene Element-ID zusammen mit der zu setzenden Y-Position, und sequence wird an die neue Position verschoben.

Syntax:

layer_sequence_y(sequence_element_id, pos_y)

ArgumentTyp Beschreibung
sequence_element_id The unique ID value of the sequence element to target
pos_y The Y position to move the sequence element to

Rückgabe:

Beispiel:

if layer_sequence_exists(my_seq)
{
    layer_sequence_x(my_seq, x);
    layer_sequence_y(my_seq, y);
}

Die obige code prüft, ob das sequence Element, auf das in der Variable "my_seq" verwiesen wird, existiert, und wenn ja, setzt es die x/y-Position auf die der Instanz, die code ausführt.
