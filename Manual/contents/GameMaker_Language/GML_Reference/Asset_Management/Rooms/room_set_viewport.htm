room_set_viewport

Mit dieser Funktion können Sie die Eigenschaften des Sichtfensters für jeden Raum in Ihrem Spiel einstellen , außer dem aktuellen. Du gibst den Raumindex an, in dem du den Viewport setzen willst, den Viewport-Index (von 0 bis 7) und dann, ob der Viewport am Anfang des Raumes sichtbar sein soll oder nicht (setze auf true, um den Port sichtbar zu machen, und false andernfalls). Danach werden die x- und y-Koordinaten (entsprechend der oberen linken Position des Sichtfensters) und dann die Breite und Höhe in Pixeln festgelegt.

Syntax:

room_set_viewport(rm, vind, vis, xport, yport, wport, hport);

Argument Typ Beschreibung
rm Zimmer Asset The index of the room to set
vind Real The index of the view port to set
vis Boolesche The visibility of the view port (true ist sichtbar, false ist unsichtbar)
xport Real The x position for the view port in the room
yport Real The y position of the view port in the room
wport Real The width (in pixels) of the view port
hport Real The height (in pixels) of the view port

Rückgabe:

N/A

Beispiel:

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_viewport(global.myroom, 0, true, 0, 0, 640, 480);

Dadurch werden die Eigenschaften des Ansichtsports [0] in dem Raum festgelegt, der in der Variablen "global.myroom" indiziert ist.
