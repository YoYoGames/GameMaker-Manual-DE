tilemap_get_tile_height

Mit dieser Funktion können Sie die Höhe (in Pixel) jeder Kachelzelle des Kachelkartenelements abrufen. Sie geben die ID des Kachelmappenelements an (die Sie erhalten, wenn Sie ein Kachelmappenelement mit layer_tilemap_create() oder wenn Sie die Funktion layer_tilemap_get_id()), und die Funktion gibt die Höhe der Kachelzelle zurück.

Syntax:

tilemap_get_tile_height(tilemap_element_id)

ArgumentTyp Beschreibung
tilemap_element_id The unique ID value of the tile map element to get the tile cell height of

Rückgabe:

Beispiel:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
global.snap_x = tilemap_get_tile_width(map_id);
global.snap_y = tilemap_get_tile_height(map_id);

Die obige code verwendet die abgefragte Kachelkarten-ID, um die Breite und Höhe der Kachelzellen zu ermitteln und zwei globale Variablen zu setzen.
