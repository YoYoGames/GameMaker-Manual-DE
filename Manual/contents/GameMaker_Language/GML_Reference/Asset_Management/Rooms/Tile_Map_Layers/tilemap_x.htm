tilemap_x

Diese Funktion steuert die Position entlang der X-Achse des Raumes des Asset-Kachelelementes auf der Ebene. Sie geben die ID des Kachelkartenelements an (die Sie erhalten, wenn Sie ein Kachelkartenelement mit layer_tilemap_create() oder bei Verwendung der Funktion layer_tilemap_get_id()), und legen dann den zu verwendenden x-Wert fest (basierend auf den Raumkoordinaten).

Syntax:

tilemap_x(tilemap_element_id, x);

Argument Typ Beschreibung
tilemap_element_id Kachelkarte Element-ID The unique ID value of the tile map element to change
x Real The x position for the tile map

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Asset_sky");
var map_id = layer_tilemap_get_id(lay_id);
tilemap_x(map_id, irandom(room_width));

Die obige code ruft den ID-Wert des Kachelkarten-Assets ab, das der Ebene "Asset_sky" zugewiesen ist, und setzt dann seine x-Position auf einen Zufallswert zwischen 0 und der Breite des Raums.
