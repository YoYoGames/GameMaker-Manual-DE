tilemap_get_height

Mit dieser Funktion können Sie die Höhe (in Zellen) des Kachelkartenelements abrufen. Sie geben die ID des Kachelkartenelements an (die Sie erhalten, wenn Sie ein Kachelkartenelement mit layer_tilemap_create() oder wenn Sie die Funktion layer_tilemap_get_id()), und die Funktion gibt die Höhe (Anzahl der Zellen) zurück.

Syntax:

tilemap_get_height(tilemap_element_id)

ArgumentTyp Beschreibung
tilemap_element_id The unique ID value of the tile map element to get the height of

Rückgabe:

Beispiel:

var lay_id = layer_get_id("espadrilles");
var map_id = layer_tilemap_get_id(lay_id);
var _w = tilemap_get_width(map_id);
var _h = tilemap_get_height(map_id);
total_tiles = (_w * _h);

Die obige code verwendet die abgerufene Kachelkarten-ID, um die Kachelbreite und -höhe der Kachelkarte zu ermitteln, und verwendet diese Werte dann, um eine Instanzvariable zu setzen.
