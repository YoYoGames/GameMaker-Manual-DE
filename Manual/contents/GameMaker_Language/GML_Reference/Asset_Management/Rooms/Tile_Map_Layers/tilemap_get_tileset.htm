tilemap_get_tileset

Mit dieser Funktion können Sie den Indexwert des Kachelset-Assets abrufen, das einem bestimmten Kachelkartenelement auf einer Ebene zugewiesen ist. Sie geben die ID des Tile-Map-Elements an (die Sie erhalten, wenn Sie ein Tile-Map-Element mit layer_tilemap_create() oder bei Verwendung der Funktion layer_tilemap_get_id()), und die Funktion gibt den Index des Tile Set-Assets zurück.

Syntax:

tilemap_get_tileset(tilemap_element_id)

Argument Typ Beschreibung
tilemap_element_id Kachelkarte Element-ID The unique ID value of the Tile Map element to get the tile set from

Rückgabe:

Tile Set Asset

Beispiel:

var lay_id = layer_get_id("Tiles_trees");
var map_id = layer_tilemap_get_id(lay_id);
if (tilemap_get_tileset(map_id) != ts_Nighttime)
{
    tilemap_tileset(map_id, ts_Nighttime);
}

Die obige code prüft den aktuellen Kachelsatz, der der Ebene "Tiles_trees" zugewiesen ist, und wenn es nicht "ts_Nighttime" ist, wird dieser Kachelsatz der Kachelkarte zugewiesen.
