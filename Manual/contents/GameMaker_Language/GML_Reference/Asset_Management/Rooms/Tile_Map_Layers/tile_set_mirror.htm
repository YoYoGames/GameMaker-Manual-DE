tile_set_mirror

Mit dieser Funktion kann ein bestimmter Satz von Kacheldaten so eingestellt werden, dass die Kachel gespiegelt wird oder nicht. Sie geben die Kacheldaten an, die Sie mit der Funktion tilemap_get()abgerufen werden können, und setzen dann die Spiegelung argument entweder auf true, wenn die Kachel gespiegelt werden soll, oder auf false, wenn die Kachel in ihrem standardmäßigen, nicht gespiegelten Zustand bleiben soll. Die Funktion gibt einen geänderten Kacheldatensatz zurück, der dann mit der Funktion tilemap_set() Funktion angewendet werden kann.

Syntax:

tile_set_mirror(tiledata, mirror)

ArgumentTyp Beschreibung
tiledata the tile-data to set
mirror Set to true zu spiegeln und false so zu belassen, wie es ist

Rückgabe:

Tile Data (Real)

Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_mirror(data);
data = tile_set_mirror(data, bool);
tilemap_set(map_id, data, mx, my);

Die obige code holt sich die ID der Kachelkarte von der angegebenen Ebene und ermittelt dann die x- und y-Zellenposition für die Kachel unter der Maus. Diese Position wird dann verwendet, um die Kacheldaten zu erhalten, die gespiegelt werden und dann verwendet werden, um die Kachel wieder zu setzen.
