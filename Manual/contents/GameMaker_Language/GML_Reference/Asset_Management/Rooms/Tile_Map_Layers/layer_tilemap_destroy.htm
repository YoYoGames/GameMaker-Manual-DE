layer_tilemap_destroy

Diese Funktion zerstört das angegebene Kachelkartenelement. Sie geben die ID der Kachelkarte an (die Sie erhalten, wenn Sie die Kachelkarte mit layer_tilemap_create() oder wenn Sie die Layer-ID zusammen mit layer_get_tilemap_id()) und diese Funktion wird sie entfernen. Beachte, dass dies nicht die Ebene entfernt, sondern nur die Kachelkarte, und wenn die Kachelkarte eine ist, die im Raumeditor hinzugefügt wurde, dann wird die Kachelkarte beim nächsten Verlassen des Raums und der Rückkehr wieder neu erstellt. Wenn der Raum jedoch persistent ist, wird die Kachelkarte entfernt, es sei denn, die Raumpersistenz wird wieder ausgeschaltet.

Syntax:

layer_tilemap_destroy(tilemap_element_id);

Argument Typ Beschreibung
tilemap_element_id Kachelkarte Element-ID The unique ID value of the tile map to be destroyed

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if (layer_tilemap_exists(lay_id, tile_id))
{
    layer_tilemap_destroy(tile_id);
}

Die obige code prüft die Ebene "Tiles_trees", um zu sehen, ob das angegebene Kachelkartenelement existiert, und wenn ja, dann wird es zerstört (aber nicht die Ebene).
