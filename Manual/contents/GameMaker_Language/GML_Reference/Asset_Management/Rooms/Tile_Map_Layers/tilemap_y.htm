tilemap_y

Diese Funktion steuert die Position entlang der y-Achse des Raums des Asset-Kachelkartenelements auf der Ebene. Sie geben die ID des Kachelkartenelements an (die Sie erhalten, wenn Sie ein Kachelkartenelement mit layer_tilemap_create() oder bei Verwendung der Funktion layer_tilemap_get_id()) und legen dann den zu verwendenden y-Wert fest (basierend auf den Raumkoordinaten).

Syntax:

tilemap_y(tilemap_element_id, y);

Argument Typ Beschreibung
tilemap_element_id Kachelkarte Element-ID The unique ID value of the tile map element to change
y Real The y position for the tile map

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Asset_sky");
var map_id = layer_tilemap_get_id(lay_id);
tilemap_y(map_id, irandom(room_height));

Die obige code ruft den ID-Wert des Kachelkarten-Assets ab, das der Ebene "Asset_sky" zugewiesen ist, und setzt dann seine y-Position auf einen Zufallswert zwischen 0 und der Höhe des Raums.
