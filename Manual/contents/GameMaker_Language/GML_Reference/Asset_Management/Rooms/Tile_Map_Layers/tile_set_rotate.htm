tile_set_rotate

Diese Funktion kann verwendet werden, um einen gegebenen Satz von Kacheldaten so einzustellen, dass die Kachel um 90 Grad gedreht wird oder nicht. Sie geben die Kacheldaten an, die Sie mit der Funktion tilemap_get()abgerufen werden können, und setzen Sie dann die rotate argument entweder auf true, wenn die Kachel gedreht werden soll, oder auf false, wenn die Kachel in ihrem standardmäßigen, nicht gedrehten Zustand bleiben soll. Die Funktion gibt einen geänderten Kacheldatensatz zurück, der dann mit der Funktion tilemap_set() Funktion angewendet werden kann.

Syntax:

tile_set_rotate(tiledata, rotate)

Argument Typ Beschreibung
tiledata Kacheldaten The tile-data to set
rotate Boolesche Set to true zum Drehen und false zum Beibehalten

Rückgabe:

Real

Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_rotate(data);
data = tile_set_rotate(data, bool);
tilemap_set(map_id, data, mx, my);

Die obige code holt sich die ID der Kachelkarte von der angegebenen Ebene und ermittelt dann die x- und y-Zellenposition für die Kachel unter der Maus. Diese Position wird dann verwendet, um die Kacheldaten zu erhalten, die gedreht werden und dann verwendet werden, um die Kachel wieder zu setzen.
