layer_tilemap_get_id

Diese Funktion kann verwendet werden, um den eindeutigen ID-Wert des Kachelkartenelements auf einer Ebene abzurufen. Sie geben die Ebenen-ID (die Sie beim Erstellen der Ebene mit layer_create()) oder den Ebenennamen (als String - dies wirkt sich auf die Leistung aus) und die Funktion gibt den ID-Wert zurück, der mit dem Kachelkartenelement auf der Ebene verbunden ist. Beachten Sie, dass diese Funktion speziell für die Verwendung mit Kachelkarten gedacht ist, die in IDE hinzugefügt wurden. Wenn Sie nämlich eine Kachelkarte über code mit der Funktion layer_tilemap_create()hinzufügen, wird die eindeutige ID für das hinzugefügte Kachelkartenelement zurückgegeben. Wenn die angegebene Tilemap-ID falsch ist oder die Tilemap nicht existiert, gibt die Funktion -1 zurück.

Syntax:

layer_tilemap_get_id(layer_id)

Argument Typ Beschreibung
layer_id Zeichenfolge oder Ebene ID The unique ID value of the layer to target (or the layer name as a string)

Rückgabe:

Tile Map Element ID or -1

Beispiel:

var lay_id = layer_get_id("tilemap_trees");
var tile_id = layer_tilemap_get_id(lay_id);
layer_tilemap_destroy(tile_id);

Die obige code ruft die Ebenen-ID für die Ebene "tilemap_trees" ab und verwendet diese dann, um die ID des Kachelkartenelements auf dieser Ebene zu ermitteln. Diese ID wird dann zum Entfernen der Kachelkarte aus der Ebene verwendet.
