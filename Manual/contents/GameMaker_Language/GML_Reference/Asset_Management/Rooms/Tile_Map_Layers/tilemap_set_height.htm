tilemap_set_height

Diese Funktion kann verwendet werden, um die Größe eines Kachelkartenelements zu ändern. Sie geben die ID des Kachelmappenelements an (die Sie erhalten, wenn Sie ein Kachelmappenelement mit layer_tilemap_create() oder bei Verwendung der Funktion layer_tilemap_get_id()), und die neue Höhe der Kachelkarte in Kachelzellen.

Syntax:

tilemap_set_height(tilemap_element_id, height);

Argument Typ Beschreibung
tilemap_element_id Kachelkarte Element-ID The unique ID value of the tile map element to set the height of
height Real The height value (in "cells")

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
if (tilemap_get_height(map_id) != room_height div 16)
{
    tilemap_set_height(map_id, room_height div 16);
}

Die obige code prüft die Breite einer bestimmten Kachelkarte und wenn sie nicht die richtige Größe hat, wird die Breite eingestellt.
