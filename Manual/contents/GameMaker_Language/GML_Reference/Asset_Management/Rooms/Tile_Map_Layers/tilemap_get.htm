tilemap_get

Mit dieser Funktion können Sie die Kacheldaten aus einer Zelle des Kachelkartenelements abrufen. Sie geben die ID des Kachelmappenelements an (die Sie erhalten, wenn Sie ein Kachelmappenelement mit layer_tilemap_create() oder bei Verwendung der Funktion layer_tilemap_get_id()) sowie die x- und y-Position der Zelle an, und die Funktion gibt den Kacheldaten-"Blob" zurück. Bei diesen Daten handelt es sich im Wesentlichen um eine Bitmaske, die den Kachelindex, die Booleschen Werte für Spiegeln/Drehen/Spiegeln und alle angewendeten Maskendaten enthält (siehe tilemap_set_mask() für Details), und der resultierende Datenwert kann dann in den tile_* Funktionen verwendet werden, um die Eigenschaften einer Kachel zu ändern. Beachten Sie, dass wir die x/y-Zellenpositionen auf der Grundlage der Größe des Kachelkartenrasters und nicht als Position im Raum verwenden, daher ist "cell_x" ein Wert von 0 bis zur Kachelkartenbreite und "cell_y" ein Wert von 0 bis zur Kachelkartenhöhe. Wenn Sie die Daten für eine Kachel an einer bestimmten Raumposition benötigen, müssen Sie die Funktion tilemap_get_at_pixel().

WICHTIG! Wenn die Kacheln in der Kachelkarte nicht verändert wurden (d.h. sie wurden nicht gedreht oder gespiegelt etc...), dann ist der Rückgabewert der Kacheldaten genau gleich dem Index der Kachel in der Kachelkarte. So können Sie "Kollisionskarten" von Kacheln erstellen, indem Sie eine Kachel mit dem Index 1 im Kachelsatz verwenden - zum Beispiel - und dann diese Funktion verwenden, um auf 1 oder 0 (eine leere Kachel) zu prüfen, um Kollisionen zu berechnen.

Syntax:

tilemap_get(tilemap_element_id, cell_x, cell_y);

Argument Typ Beschreibung
tilemap_element_id Kachelkarte Element-ID The unique ID value of the tile map element to get the tile-data of
x_cell Real The cell along the x-axis to get the tile data from
y_cell Real The cell along the y-axis to get the tile data from

Rückgabe:

Tile Data (-1 if there is an error)

Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

Die obige code ruft die ID für die Kachelkarte auf der Ebene "Tiles_Sky" ab und verwendet diese dann, um die Daten der Kachel in der Zelle (0, 0) abzurufen. Diese Kacheldaten werden dann gespiegelt, bevor sie verwendet werden, um die Kachel auf der Kachelkarte erneut zu setzen.
