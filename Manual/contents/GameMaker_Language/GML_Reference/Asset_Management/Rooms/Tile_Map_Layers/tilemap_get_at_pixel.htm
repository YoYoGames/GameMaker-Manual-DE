tilemap_get_at_pixel

Mit dieser Funktion können Sie die Kacheldaten von einer Position (innerhalb des Raums) des Kachelkartenelements abrufen. Sie geben die ID des Tile Map Elements an (die Sie erhalten, wenn Sie ein Tile Map Element mit layer_tilemap_create() oder bei Verwendung der Funktion layer_tilemap_get_id()) sowie die x- und y-Position im Raum, von der die Kacheldaten abgerufen werden sollen, und die Funktion gibt den Kacheldaten-"Blob" zurück. Bei diesen Daten handelt es sich im Wesentlichen um eine Bitmaske, die den Kachelindex, die Booleschen Werte für "flip/rotate/mirror" und etwaige Masken enthält, die angewendet wurden (siehe tilemap_set_mask() für Details), und der resultierende Datenwert kann dann in den tile_* Funktionen verwendet werden, um die Eigenschaften einer Kachel zu ändern. Wenn Sie die Kacheldaten von einer bestimmten Kachelzelle benötigen, sollten Sie stattdessen die Funktion tilemap_get() verwenden.

WICHTIG! Wenn die Kacheln in der Kachelkarte nicht verändert wurden (d.h. sie wurden nicht gedreht oder gespiegelt etc...), dann ist der Rückgabewert der Kacheldaten genau gleich dem Index der Kachel in der Kachelkarte. So können Sie "Kollisionskarten" von Kacheln erstellen, indem Sie eine Kachel mit dem Index 1 im Kachelsatz verwenden - zum Beispiel - und dann diese Funktion verwenden, um auf 1 oder 0 (eine leere Kachel) zu prüfen, um Kollisionen zu berechnen.

Syntax:

tilemap_get_at_pixel(tilemap_element_id, x, y)

Argument Typ Beschreibung
tilemap_element_id Kachelkarte Element-ID The unique ID value of the tile map element to get the tile-data of
x Real The position along the x-axis to get the tile data from (in room pixels)
y Real The position along the y-axis to get the tile data from (in room pixels)

Rückgabe:

Tile Data (-1 if there is an error)

Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 64, 128);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 64, 128);

Die obige code ruft die ID für die Kachelkarte "Clouds" auf der Ebene "Tiles_Sky" ab und verwendet diese dann, um die Daten der Kachel an der Position (64, 128) zu erhalten. Diese Kacheldaten werden dann gespiegelt, bevor sie verwendet werden, um die Kachel auf der Kachelkarte erneut zu setzen.
