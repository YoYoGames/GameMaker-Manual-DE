tile_set_index

Diese Funktion kann verwendet werden, um den Kachelindex (die Position der Kachel innerhalb des Kachelsatzbildes) für einen Satz von Kacheldaten zu setzen. Sie geben die Kacheldaten an, die Sie mit der Funktion tilemap_get()abgerufen werden können, und dann den ganzzahligen Wert, der die Position der Kachel (ihren Index) innerhalb des Kachelsatzes darstellt. Die Funktion gibt einen geänderten Kacheldatensatz zurück, der dann mit der Funktion tilemap_set() Funktion angewendet werden kann.

Syntax:

tile_set_index(tiledata, index)

Argument Typ Beschreibung
tiledata Kacheldaten the tile-data to set
index Real The integer index value of the tile to use

Rückgabe:

Tile Data

Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var ind = tile_get_index(data);
data = tile_set_index(data, irandom(23));
tilemap_set(map_id, data, mx, my);

Die obige code holt sich die Kachel-ID von der angegebenen Ebene und verwendet diese dann, um die Kacheldaten für die Zelle unter der Mausposition zu erhalten. Diese Daten werden dann verwendet, um den Kachelindex auf eine Zufallszahl zu setzen.
