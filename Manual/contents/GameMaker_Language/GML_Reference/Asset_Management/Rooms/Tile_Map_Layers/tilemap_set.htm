tilemap_set

Diese Funktion kann verwendet werden, um eine beliebige Zelle (Rasterquadrat) innerhalb des Tile-Map-Elements auf der Ebene zu einer neuen Kachel zu machen. Sie geben die ID des Kachelkartenelements an (die Sie erhalten, wenn Sie ein Kachelkartenelement mit layer_tilemap_create() oder bei Verwendung der Funktion layer_tilemap_get_id()) und dann die Kacheldaten, die gesetzt werden sollen, sowie die Position innerhalb der Kachelkarte. Sie können Kacheldaten mit der Funktion tilemap_get() abrufen und dann die Funktionen von tile_ verwenden, um sie zu ändern, bevor Sie die Zelle mit dieser Funktion setzen. Die Funktion gibt true zurück, wenn die Kachel erfolgreich gesetzt wurde, und false, wenn es ein Problem gab und die Kachel nicht gesetzt wurde.

Syntax:

tilemap_set(tilemap_element_id, tiledata, xcell, ycell)

Argument Typ Beschreibung
tilemap_element_id Kachelkarte Element-ID The unique ID value of the tile map element to change
tiledata Kacheldaten (Real) The tile data to set
xcell Real The cell (grid) position to set along the x-axis
ycell Real The cell (grid) position to set along the y-axis

Rückgabe:

Boolean

Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

Die obige code ruft die ID für die Kachelkarte auf der Ebene "Tiles_Sky" ab und verwendet diese dann, um die Daten der Kachel in der Zelle (0, 0) abzurufen. Diese Kacheldaten werden dann gespiegelt, bevor sie verwendet werden, um die Kachel auf der Kachelkarte erneut zu setzen.
