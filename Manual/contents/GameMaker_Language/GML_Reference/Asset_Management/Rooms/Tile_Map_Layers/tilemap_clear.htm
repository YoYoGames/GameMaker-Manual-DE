tilemap_clear

Mit dieser Funktion können Sie alle Kacheln auf einer bestimmten Kachelkarte löschen/einstellen. Sie geben die ID des Tile-Map-Elements an (die Sie erhalten, wenn Sie ein Tile-Map-Element mit layer_tilemap_create() oder wenn Sie die Funktion layer_tilemap_get_id()), und geben dann die Kacheldaten an, mit denen Sie die Ebene löschen möchten. Ein Standardwert von 0 löscht alle Kacheln aus der Ebene (wodurch alle Kacheln im Wesentlichen "leer" werden), während Sie die speziellen Funktionen von tile_* verwenden können, um Ihre eigenen Kacheldaten zu erstellen, mit denen Sie die Kachelkarte löschen.

Syntax:

tilemap_clear(tilemap_element_id, tiledata);

Argument Typ Beschreibung
tilemap_element_id Kachelkarte Element-ID The unique ID value of the tile map element to change
tiledata Kacheldaten (Real) The tile data to use to clear the layer

Rückgabe:

N/A

Beispiel:

var layer_id = layer_get_id("Forest");
var tile_id = layer_tilemap_get_id(layer_id);
tilemap_clear(tile_id, 0);

Die obige code ruft den ID-Wert einer im Raumeditor erstellten Kachelkarte ab und löscht sie dann mit "leeren" Kacheln.
