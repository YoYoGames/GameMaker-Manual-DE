tilemap_get_cell_y_at_pixel

Mit dieser Funktion können Sie die y-Achsenposition einer einzelnen Kachelzelle abrufen, indem Sie die relative y-Achsenposition innerhalb des Raums angeben. Sie geben die ID des Kachelkartenelements an (die Sie erhalten, wenn Sie ein Kachelkartenelement mit layer_tilemap_create() oder bei Verwendung der Funktion layer_tilemap_get_id()) sowie die x- und y-Position innerhalb des Raums an, und die Funktion gibt die y-Position der Zelle innerhalb der Tile Map für diesen Punkt zurück. Beachten Sie, dass die Funktion -1 zurückgibt, wenn der Wert außerhalb des Tile-Map-Bereichs liegt und keine Zelle verfügbar ist.

Syntax:

tilemap_get_cell_y_at_pixel(tilemap_element_id, x, y);

ArgumentTyp Beschreibung
tilemap_element_id The unique ID value of the tile map element to get the cell y position of
x The x position within the room to use for getting the cell
y The y position within the room to use for getting the cell

Rückgabe:

(y-axis cell position or -1 if there is an error)

Beispiel:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _x = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var _y = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
tiledata = tilemap_get(map_id, _x, _y);

Die obige code verwendet die abgefragte ID der Kachelkarte, um die Position der Zelle aus einem Satz von Raumkoordinaten zu ermitteln und dann die Daten für jede dort gefundene Kachel in einer Instanzvariablen zu speichern.
