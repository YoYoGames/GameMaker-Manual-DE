tilemap_get_mask

Diese Funktion kann verwendet werden, um den Bitmaskenwert für die angegebene Tile Map abzurufen. Sie geben die ID des Tile-Map-Elements an (die Sie erhalten, wenn Sie ein Tile-Map-Element mit layer_tilemap_create() oder wenn Sie die Funktion layer_tilemap_get_id()), und die Funktion gibt den aktuellen Maskenwert zurück oder -1, wenn ein Fehler vorliegt, oder 0, wenn keine Maske angegeben ist. Weitere Informationen über Bitmasken für Kachelkarten finden Sie in der Funktion tilemap_set_mask().

Syntax:

tilemap_get_mask(tilemap_element_id)

Argument Typ Beschreibung
tilemap_element_id Kachelkarte Element-ID The unique ID value of the tile map element to get the mask of

Rückgabe:

Real (0 for no mask, -1 for an error)

Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mask = tilemap_get_mask(map_id);
var new_mask = tile_mirror | tile_flip | tile_rotate | 255;
if (mask != new_mask)
{
    tilemap_set_mask(_map_id, _new_mask);
}

Die obige code holt sich die ID der Kachelkarte von der angegebenen Ebene und überprüft dann den damit verbundenen Maskenwert. Wenn er nicht mit dem in der Variablen "new_mask" definierten Wert übereinstimmt, wird er auf diesen Wert gesetzt.
