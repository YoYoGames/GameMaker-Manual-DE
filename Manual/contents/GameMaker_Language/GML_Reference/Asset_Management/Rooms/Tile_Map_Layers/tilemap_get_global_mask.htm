tilemap_get_global_mask

Diese Funktion kann verwendet werden, um den Bitmaskenwert für alle Tile Maps abzurufen. Sie gibt den aktuellen Maskenwert oder -1 zurück, wenn ein Fehler vorliegt, oder 0, wenn keine Maske angegeben wurde. Weitere Informationen über globale Kachelkarten-Bitmasken finden Sie in der Funktion tilemap_set_global_mask().

Syntax:

tilemap_get_global_mask();

Rückgabe:

(0 for no mask, -1 for an error, mask value otherwise)

Beispiel:

var mask = tilemap_get_global_mask(map_id);
var new_mask = tile_mirror | tile_flip | tile_rotate | 255;
if (mask != new_mask)
{
    tilemap_set_global_mask(new_mask);
}

Die obige code ruft den globalen Maskenwert ab, der mit allen Kachelkarten verbunden ist. Wenn er nicht mit dem in der Variablen "new_mask" definierten Wert übereinstimmt, wird er auf diesen Wert gesetzt.
