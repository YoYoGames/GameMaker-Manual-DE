tilemap_get_frame

Da Kacheln animiert werden können, kann es manchmal nützlich sein, zu wissen, welcher Frame gerade gezeichnet wird, und entsprechend zu reagieren. Mit dieser Funktion können Sie den aktuellen Frame-Index für eine bestimmte Tile Map abrufen. Sie geben die ID des Tile-Map-Elements an (die Sie erhalten, wenn Sie ein Tile-Map-Element mit layer_tilemap_create() oder wenn Sie die Funktion layer_tilemap_get_id()), und die Funktion gibt den Frame-Index zurück.

Syntax:

tilemap_get_frame(tilemap_element_id)

Argument Typ Beschreibung
tilemap_element_id Kachelkarte Element-ID The unique ID value of the tile map element to get the frame index of

Rückgabe:

Real (between 0 (inclusive) and the maximum number of frames of animation (exclusive))

Beispiel:

var lay_id = layer_get_id("Tiles_Traps");
var map_id = layer_tilemap_get_id(lay_id);
if (tilemap_get_frame(map_id) >= 2 && tilemap_get_frame(map_id) < 4)
{
    global.activate = true;
}
else
{
    global.activate = false;
}

Die obige code prüft das aktuelle Animationsbild für die angegebene Kachelkarte auf der angegebenen Ebene und setzt eine globale Variable basierend auf dem Rückgabewert.
