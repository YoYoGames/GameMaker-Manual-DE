layer_tilemap_exists

Mit dieser Funktion können Sie prüfen, ob ein Kachelkartenelement auf einer bestimmten Ebene existiert. Sie geben die Layer-ID (die Sie beim Erstellen des Layers mit layer_create()) oder den Ebenennamen (als String - dies hat Auswirkungen auf die Leistung) und die ID des Kachelkartenelements (die Sie erhalten, wenn Sie ein Kachelkartenelement mit layer_tilemap_create() oder wenn Sie die Funktion layer_tilemap_get_id()) und die Funktion gibt entweder true zurück, wenn das Element existiert, oder false, wenn es nicht existiert.

HINWEIS: Diese Funktion arbeitet innerhalb des Bereichs des aktuellen Zielraums - standardmäßig der Raum, in dem die Funktion aufgerufen wird -, der mit der Funktion layer_set_target_room().

Syntax:

layer_tilemap_exists(layer_id, tilemap_element_id)

Argument Typ Beschreibung
layer_id String oder Ebenen-ID The unique ID value of the layer to target (or the layer name as a string)
tilemap_element_id Kachelkarte Element-ID The unique ID value of the tile map element to check

Rückgabe:

Boolean

Beispiel:

var lay_id = layer_get_id("tilemap_trees");
if (layer_tilemap_exists(lay_id, global.Treestilemap))
{
    layer_tilemap_destroy(lay_id, global.Treestilemap);
}

Die obige code prüft die Ebene "tilemap_trees", um zu sehen, ob das angegebene Kachelkartenelement existiert, und wenn ja, dann wird es zerstört (aber nicht die Ebene).
