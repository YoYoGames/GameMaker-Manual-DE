tile_set_empty

Diese Funktion kann verwendet werden, um eine gegebene Menge von Kacheldaten als leere Kachel zu setzen. Sie geben die Kacheldaten an, die Sie mit der Funktion tilemap_get()abgefragt werden können, und die Funktion gibt true zurück, wenn die Kachel leer ist, oder false, wenn es einen Kachelindex gibt. Die Funktion gibt einen geänderten Kacheldatensatz zurück, der dann mit der Funktion tilemap_set() angewendet werden kann.

Syntax:

tile_set_empty(tiledata)

Argument Typ Beschreibung
tiledata Kacheldaten the tile-data to set

Rückgabe:

Tile Data

Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
for (var i = 0; i < tilemap_get_width(map_id); i++)
{
    for (var j = 0; j < tilemap_get_height(map_id); j++)
    {
        var data = tilemap_get(map_id, i, j);
        if (!tile_get_empty(data))
        {
            data = tile_set_empty(data)
            tilemap_set(map_id, data, i, j);
        }
    }
}

Die obige code ruft die ID der Kachelkarte von der angegebenen Ebene ab und überprüft dann jede Kachelzelle auf der Karte, um festzustellen, ob sie Daten enthält oder nicht. Ist dies der Fall, wird die Kachel auf leer gesetzt.
