raum_breite

Diese Variable enthält die Breite des aktuellen Raums in Pixeln. Sie können diese Variable jederzeit ändern, um die Breite des Raums zu ändern.

Syntax:

room_width;

Rückgabe:

Beispiel:

if bbox_right > room_width
{
    x += room_width - bbox_right;
}

Die obige code prüft, ob die sprite Bounding Box der aktuellen Instanz größer ist als die Breite des Raumes, und wenn ja, wird die Instanz nach oben verschoben.
