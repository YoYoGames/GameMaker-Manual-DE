room_last

Diese schreibgeschützte Variable gibt den Index des allerletzten Raums im Spiel zurück (dieser wird durch die Reihenfolge definiert, in der die Räume im Raummanager erscheinen, und nicht durch die Reihenfolge, in der sie erstellt wurden). Beachten Sie, dass diese Variable Räume, die dynamisch über room_add() oder room_duplicate() hinzugefügt wurden, nicht erkennt oder berücksichtigt.

Syntax:

room_last

Rückgabe:

Room Asset

Beispiel:

if keyboard_check_pressed(ord("Q"))
{
    room_goto(room_last);
}

Das obige code prüft, ob eine Taste gedrückt wurde, und wenn ja, geht es zum letzten Raum im Spiel.
