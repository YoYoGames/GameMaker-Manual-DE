room_get_camera

Mit dieser Funktion können Sie die eindeutige Index-ID der Kamera ermitteln, die einer bestimmten Ansicht in einem anderen Raum als dem aktuellen zugeordnet ist. Sie geben den zu verwendenden Raum und den zu verwendenden Ansichtsanschluss (von 0 bis 7) an und die Funktion gibt einen Kameraindex zurück.

Syntax:

room_get_camera(rm, vind);

Argument Typ Beschreibung
rm Zimmer Asset The index of the room to get the view camera of
vind Real The index of the view port to get the camera of

Rückgabe:

Camera ID

Beispiel:

var cam = room_get_camera(rm_Game, 0);
if (cam != global.MainCam)
{
    room_set_camera(rm_game, 0, global.main_cam);
}

Die obige code weist eine Kamera in einem neu erstellten Raum dem View Port [0] zu.
