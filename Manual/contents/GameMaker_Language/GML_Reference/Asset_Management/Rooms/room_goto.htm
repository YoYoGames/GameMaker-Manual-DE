zimmer_suchen

Mit dieser Funktion können Sie zu einem beliebigen Raum in Ihrem Spielprojekt wechseln, unabhängig davon, ob dieser mit code oder im Asset-Browser erstellt wurde. Sie geben den Raumindex an (gespeichert in der Variablen für den Raumnamen oder als Variable, die von der Funktion room_add()). Beachten Sie, dass sich der Raum erst am Ende des Ereignisses ändert, in dem die Funktion aufgerufen wurde. Daher wird code auch nach dem Aufruf dieser Funktion noch ausgeführt, wenn es sich um dasselbe Ereignis handelt. Diese Funktion löst auch das Ereignis Raumende aus.

HINWEIS Raum-IDs basieren nicht auf ihrer Reihenfolge im Asset-Browser oder im Raum-Manager, daher sollten Sie es vermeiden, einen Zahlenwert direkt anzugeben. Verwenden Sie stattdessen die Raumkonstante für das Asset, auf das Sie verweisen möchten (das ist einfach sein Name), oder rufen Sie ihn über eine Funktion ab.

Syntax:

room_goto(index);

ArgumentTyp Beschreibung
index The index of the room to go to.

Rückgabe:

Beispiel:

switch (global.level)
{
    case 0: room_goto(rm_level1); break;
    case 1: room_goto(rm_level2); break;
    case 2: room_goto(rm_level3); break;
}

Die obige code prüft eine globale Variable und ändert den Raum auf der Grundlage des Wertes, den sie enthält.
