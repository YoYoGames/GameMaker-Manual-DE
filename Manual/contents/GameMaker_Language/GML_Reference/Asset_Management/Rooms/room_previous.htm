raum_vorher

Mit dieser Funktion können Sie den Index des Raums vor der Eingabe des Raums in die Funktion abrufen. Zum Beispiel können Sie die Variable room verwenden, um den Index des aktuellen Raums zu erhalten, und dann diese Funktion verwenden, um den Raum zu finden, der vor diesem Raum liegt, wie er im Raum-Manager aufgeführt ist. Wenn es vor dem eingegebenen Raum keinen Raum gibt, wird -1 zurückgegeben. Beachten Sie, dass diese Funktion keine Räume erkennt oder berücksichtigt, die dynamisch über room_add() oder room_duplicate() hinzugefügt wurden.

Syntax:

room_previous(numb);

ArgumentTyp Beschreibung
numb The index of the room to get the one before from.

Rückgabe:

Room Asset

Beispiel:

if room_previous(room) != -1
{
    room_goto_previous();
}

Das obige code prüft, ob der vorherige Raum existiert, und wenn ja, geht es zu ihm.
