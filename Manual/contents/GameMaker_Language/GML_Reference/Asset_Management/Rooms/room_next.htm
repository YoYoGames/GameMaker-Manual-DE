raum_nächster

Mit dieser Funktion können Sie den Index des Raums nach der Eingabe des Raums in die Funktion abrufen. Zum Beispiel können Sie die Variable room verwenden, um den Index des aktuellen Raums zu erhalten, und dann diese Funktion verwenden, um den darauf folgenden Raum zu finden, wie er im Raum-Manager aufgeführt ist. Wenn es nach dem eingegebenen Raum keinen weiteren Raum gibt, wird -1 zurückgegeben. Beachten Sie, dass diese Funktion keine Räume erkennt oder berücksichtigt, die dynamisch über room_add() oder room_duplicate() hinzugefügt wurden.

Syntax:

room_next(numb);

ArgumentTyp Beschreibung
numb The index of the room to get the next one after.

Rückgabe:

Room Asset

Beispiel:

if room_next(room) != -1
{
    room_goto_next();
}

Das obige code prüft, ob der nächste Raum existiert, und wenn ja, geht es dorthin.
