raum_neustart

Mit dieser Funktion wird der aktuelle Raum neu gestartet, als ob er gerade erst betreten worden wäre. Beachten Sie, dass der Raum erst nach dem Ende des Ereignisses, in dem die Funktion aufgerufen wurde, neu gestartet wird, so dass alle code nach dem Aufruf dieser Funktion weiterhin ausgeführt werden, wenn sie sich im selben Ereignis befinden. Diese Funktion löst auch das Ereignis Raumende aus.

Syntax:

room_restart();

Rückgabe:

Beispiel:

if lives < 1 room_restart();

Die obige code prüft, ob die Variable "lives" kleiner als 1 ist und wenn ja, wird der Raum neu gestartet.
