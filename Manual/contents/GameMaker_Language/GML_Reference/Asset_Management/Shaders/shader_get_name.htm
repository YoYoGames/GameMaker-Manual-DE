shader_get_name

Mit dieser Funktion können Sie den Namen einer Shader-Ressource abrufen. Sie geben den eindeutigen ID-Wert für den Shader an, dessen Namen Sie abrufen möchten, und die Funktion gibt den Namen der Ressource als String zurück.

Syntax:

shader_get_name(shader);

ArgumentTyp Beschreibung
shader The index of the shader to get the name of.

Rückgabe:

Beispiel:

var _shader = shader_current();
var _name = shader_get_name(_shader);
draw_text(32, 32, "Debug - Currently Rendering = " + _name);

Die obige code erhält den Namen des angegebenen Shaders und zeichnet ihn auf den Bildschirm.
