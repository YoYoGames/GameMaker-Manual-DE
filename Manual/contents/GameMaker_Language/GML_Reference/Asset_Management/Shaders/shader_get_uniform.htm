shader_get_uniform

Da Sie den Wert einer Shader-Konstante nicht im Shader selbst ändern können, müssen Sie ihn vor dem Aufruf des Shaders mit einer der verfügbaren Uniform-Set-Funktionen festlegen. Dazu müssen Sie jedoch zunächst diese Funktion aufrufen, um das "Handle" der Shader-Konstante zu erhalten, die Sie ändern möchten.

HINWEIS: Obwohl ein Shader aus zwei diskreten Programmen (Vertex und Fragment) besteht, unterscheidet diese Funktion nicht zwischen den beiden und gibt das Handle der Shader-Konstante von einem der beiden Programme zurück.

Syntax:

shader_get_uniform(shader, uniform);

Argument Typ Beschreibung
shader Shader-Asset The index of the shader to use.
uniform Zeichenfolge The shader constant to get the handle of (a string).

Rückgabe:

Shader Uniform Handle

Beispiel:

shader_params = shader_get_uniform(shd_glass, "u_vRefractColour");

Die obige code erhält das Handle der Shader-Konstante "u_vRefractColour".
