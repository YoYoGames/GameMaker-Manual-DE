shader_set

Mit dieser Funktion können Sie das Zeichnungsziel auf den angegebenen Shader setzen und alle weiteren Zeichnungen werden mit diesem durchgeführt. Sie können die Verwendung des Shaders mit der Funktion shader_reset().

Syntax:

shader_set(shader);

ArgumentTyp Beschreibung
shader The index of the shader to use.

Rückgabe:

Beispiel:

shader_set(shader_Glass);
draw_self();
shader_reset();

Die obige code legt einen Shader fest, der zum Zeichnen verwendet wird, und zeichnet dann den aktuellen sprite, der für die Instanz verwendet wird, die ihn benutzt.
