shaders_werden_unterstützt

Diese Funktion prüft, ob das gewählte Zielgerät platform Shader unterstützt und gibt true zurück, wenn dies der Fall ist, und false, wenn dies nicht der Fall ist. Es ist wichtig zu beachten, dass unter Android, wenn das Projekt keine Shader-Ressourcen definiert hat, die Funktion immer falsezurückgibt, unabhängig davon, ob das Gerät Shader unterstützt oder nicht.

Syntax:

shaders_are_supported();

Rückgabe:

Beispiel:

global.GFX = false;
if (shaders_are_supported())
{
    if (shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp))
    {
        global.GFX = true;
    }
}

Die obige code setzt eine globale Variable auf false, und dann, wenn platform Shader unterstützt und beide zu prüfenden Shader korrekt kompiliert wurden, wird sie auf true gesetzt.
