shader_current

Diese Funktion gibt den Index-ID-Wert des Shaders zurück, der gerade zum Rendern verwendet wird, oder sie gibt -1 zurück, wenn kein Shader verwendet wird.

Syntax:

shader_current();

Rückgabe:

Shader Asset or -1 (if no shader is assigned)

Beispiel:

if (shader_current() == -1)
{
    shader_set(sh_warp)
}

Die obige code prüft, was der aktuelle Shader ist, und wenn sie -1 zurückgibt (kein Shader wird verwendet), wird ein Shader gesetzt.
