shader_enable_corner_id

Mit dieser Funktion können Sie einen globalen Zustand für alle verwendeten Shader einstellen, bei dem der Shader, wenn er aktiviert ist, 2 Bits aus den Eingangsfarbwerten "stiehlt". Das erste Bit ist das untere Bit des roten Farbwerts und das zweite Bit ist das untere Bit des blauen Farbwerts. Diese Werte können dann im Shader wiederhergestellt werden, um herauszufinden, mit welchem Vertex man es zu tun hat (d.h. mit welcher Ecke).

Syntax:

shader_enable_corner_id(enable);

ArgumentTyp Beschreibung
enable Enable (true) oder deaktivieren (false) Sie diese Funktion.

Rückgabe:

Beispiel:

shader_enable_corner_id(true);

Mit der obigen code wird die Verwendung von Farbbits für die Eck-ID für alle Shader aktiviert.
