shader_reset

Diese Funktion setzt den zum Zeichnen verwendeten Shader zurück und sollte aufgerufen werden, wenn Sie den aktuellen Shader nicht mehr verwenden wollen (festgelegt mit shader_set()).

Syntax:

shader_reset();

Rückgabe:

Beispiel:

shader_set(shader_Glass);
draw_self();
shader_reset();

Die obige code setzt einen Shader, der zum Zeichnen verwendet werden soll, und zeichnet dann den aktuellen sprite, der für die Instanz verwendet wird, die ihn benutzt. Schließlich wird der Shader zurückgesetzt, um zum Standard-Shader von GameMaker zurückzukehren.
