shader_ist_kompiliert

Diese Funktion überprüft einen Shader zur Laufzeit, um sicherzustellen, dass er erfolgreich kompiliert wurde. Ist dies der Fall, gibt die Funktion true zurück, andernfalls false. Diese Funktion sollte zu Beginn des Spiels verwendet werden, um sicherzustellen, dass platform, auf dem Ihr Spiel läuft, alle verwendeten Shader erfolgreich kompiliert hat (insbesondere auf Windows, wo einige Computer möglicherweise DX9 mit Shader Level 2.0 und nicht eine spätere Version mit Shader Level 3.0 verwenden).

Wenn Ihr Shader NICHT kompiliert wurde und Sie rufen shader_set() aufruft, stürzt das Spiel ab. Es lohnt sich also, eine Art Überprüfung vorzunehmen, wenn Sie etwas anderes als einfache GLSL ES-Shader verwenden.

Syntax:

shader_is_compiled(shader);

ArgumentTyp Beschreibung
shader The shader to check.

Rückgabe:

Boolean

Beispiel:

global.GFX = false;
if (shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp))
{
    global.GFX = true;
}

Die obige code setzt eine globale Variable auf false, und wenn die beiden geprüften Shader korrekt kompiliert wurden, wird sie auf true gesetzt.
