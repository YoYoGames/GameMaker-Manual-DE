shader_set_uniform_f

Mit dieser Funktion können Sie den Wert (oder die Werte) einer Shader-Konstante setzen. Sie müssen zuvor das "Handle" der Konstante mit Hilfe der Funktion shader_get_uniform()erhalten haben, und Sie müssen wissen, um welchen Typ von Konstante es sich handelt, um die richtige Anzahl von Fließkommawerten an die Funktion zu übergeben, d.h. wenn Sie eine vec2 haben, müssen Sie zwei Werte an die Funktion übergeben.

Eine Ausnahme von der obigen Regel ist der Umgang mit 32bit-Farbwerten. Sie können dem Shader bis zu acht 32-Bit-Farbwerte übergeben, die dann automatisch in vec4-Werte von 0 - 1 umgewandelt werden. Dazu müssen Sie allerdings die Shader-Konstante in_Colour für eine einzelne Farbe verwenden, oder in_Colour0, in_Colour1, usw... bis zu in_Colour7 für mehrere Eingabewerte. Die folgenden zwei Beispiele zeigen, wie dies im Shader selbst aussehen würde:

    //Single colour
    attribute vec2 in_Position;
    attribute vec4 in_Colour;
    attribute vec2 in_TextureCoord;

    //Multiple colours
    attribute vec2 in_Position;
    attribute vec4 in_Colour0;
    attribute vec4 in_Colour1;
    attribute vec2 in_TextureCoord;

Beachten Sie auch, dass Sie, wenn Sie komprimierte Normalen usw. mit einem 4-Byte-Farbformat erstellen wollen, auch das gleiche "in_Colour"-Format wie oben gezeigt verwenden müssen.

HINWEIS: Alle Uniformen müssen nach dem Aufruf der Funktion shader_set()und vor dem Aufruf von shader_reset().

Syntax:

shader_set_uniform_f(handle, value1 [, value2, value3, value4]);

Argument Typ Beschreibung
handle Shader Uniform Handle The handle of the shader constant to set.
value1 ... value4 Real The floating point value (or values) to set the shader constant to.

Rückgabe:

N/A

Beispiel:

shader_set(shader_Glass);
shader_params = shader_get_uniform(shader_glass, "u_vParams");
shader_set_uniform_f(shader_params, 0.1, 0.8, 0.25);
draw_self();
shader_reset();

Die obige code holt sich das Handle der Shader-Konstante "u_vParams" ( vec3) und setzt diese Konstante dann auf die angegebenen Werte.
