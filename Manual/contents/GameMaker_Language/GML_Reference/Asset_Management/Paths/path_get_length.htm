path_get_length

Mit dieser Funktion können Sie die genaue Länge eines Pfades in Pixeln ermitteln. Es handelt sich dabei nicht um eine ungefähre Länge von Punkt zu Punkt, sondern um eine exakte Länge entlang der Form des Pfades, selbst wenn der Pfad glatt ist und einer hohen Kurvengenauigkeit.

Syntax:

path_get_length(index);

ArgumentTyp Beschreibung
index The index of the path to measure.

Rückgabe:

Beispiel:

path_len = path_get_length(pth_AI);

Dadurch wird "path_len" auf die Gesamtlänge des in "pth_AI" indizierten Pfades in Pixeln gesetzt.
