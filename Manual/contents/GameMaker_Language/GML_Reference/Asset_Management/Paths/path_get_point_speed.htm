path_get_point_speed

Mit dieser Funktion können Sie die Geschwindigkeit des Punktes (wie im Pfad-Editor definiert oder wenn Sie einen Pfadpunkt dynamisch hinzufügen mit path_add_point()), ausgedrückt in Prozent. Wenn Sie also einen Pfadpunkt im Pfad-Editor auf 50 gesetzt haben, gibt diese Funktion bei Verwendung 50 zurück.

Syntax:

path_get_point_speed(index, n);

Argument Typ Beschreibung
index Pfad Anlage The index of the path to check.
n Real The point number to check.

Rückgabe:

Real

Beispiel:

if (path_get_point_speed(pth_Patrol, 0) != 100)
{
    var _px = path_get_point_x(pth_Patrol, 0);
    var _py = path_get_point_x(pth_Patrol, 0);
    path_change_point(pth_Patrol, 0, _px, _py, 100);
}

Die obige code prüft den Pfadpunkt "0" des durch "pth_Patrol" indizierten Pfades daraufhin, ob die Geschwindigkeit ungleich 100 ist. Ist dies nicht der Fall, wird die Geschwindigkeit für diesen Punkt auf 100 gesetzt.
