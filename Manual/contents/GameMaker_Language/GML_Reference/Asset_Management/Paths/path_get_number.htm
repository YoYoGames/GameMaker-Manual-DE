pfad_get_number

Diese Funktion kann verwendet werden, um die Anzahl der Punkte auf einem Pfad zu ermitteln.

Syntax:

path_get_number(index);

ArgumentTyp Beschreibung
index The index of the path to check.

Rückgabe:

Beispiel:

if (path_get_number(pth_AI) > 1)
{
    path_start( pth_AI, 4, 3, 0);
}

Die obige code prüft, ob ein Pfad mehr als einen Punkt hat, und wenn ja, beginnt die Instanz, sich entlang dieses Pfades zu bewegen.
