path_speed

Du kannst diese Funktion verwenden, um die Geschwindigkeit eines Pfades zu erhalten oder zu setzen, nachdem er mit der Funktion path_start() gestartet wurde. Sie können negative Werte verwenden, um zu signalisieren, dass die Instanz dem Pfad in umgekehrter Richtung folgen soll.

Syntax:

path_speed

Rückgabe:

Beispiel:

path_speed = -1 + random(2);

Mit der obigen code wird die Instanz so eingestellt, dass sie den Pfad mit einer zufälligen Geschwindigkeit zwischen -1 und 1 durchläuft.
