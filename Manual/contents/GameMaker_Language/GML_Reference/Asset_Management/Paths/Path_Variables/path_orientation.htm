Pfad_Orientierung

Diese Variable enthält die aktuelle Ausrichtung des Pfades, der der Instanz zugewiesen wurde, als die Funktion path_start() aufgerufen wurde. Wenn ein Pfad erstellt wird, ist seine Ausrichtung der Standardwert 0 Grad, aber du kannst diesen Wert auf jeden gewünschten Wert setzen. Denke daran, dass in GameMaker (sofern du nicht mit Physik arbeitest) die Winkel gegen den Uhrzeigersinn berechnet werden, so dass die Einstellung der Pfadausrichtung auf 90° gesetzt, würde der Pfad nach links gedreht werden.

Syntax:

path_orientation;

Rückgabe:

Beispiel:

mypath = path_duplicate(choose(path_1, path_2, path_3, path_4));
path_start(path, 4, path_action_reverse, 0);
path_orientation = 90;

Die obige code dupliziert ein zufälliges, vorgefertigtes Pfad-Asset in die Variable "mypath". Dieser neue Pfad wird dann gestartet und um 90° gedreht.
