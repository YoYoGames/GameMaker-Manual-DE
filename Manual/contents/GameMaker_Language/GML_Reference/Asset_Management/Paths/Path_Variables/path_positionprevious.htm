path_positionprevious

Diese Variable kann verwendet werden, um die Position einer Instanz entlang ihres aktuellen Pfades im vorherigen Schritt zu erhalten oder zu setzen. Sie ist ein normalisierter Wert zwischen 0 und 1, d.h. 0 ist die Startposition des Pfades und 1 wäre die Endposition. Er ist ähnlich wie die xprevious und yprevious Variablen, nur ist sie spezifisch für Pfade. Sie kann z. B. nützlich sein, um einen Pfadverfolger vorübergehend anzuhalten, wenn etwas im Weg ist (siehe das Beispiel code unten).

Syntax:

path_positionprevious

Rückgabe:

Beispiel:

var _x = x + lengthdir_x(16, direction);
var _y = y + lengthdir_y(16, direction);
if (collision_circle(_x, _y, 16, obj_player, false, true))
{
    path_position = path_positionprevious;
}

Die obige code prüft einen Bereich vor der Instanz auf eine Kollision mit dem Objekt "obj_Player" und wenn es eine gibt, setzt sie die Instanz zurück auf die vorherige Position, die sie auf dem aktuellen Pfad eingenommen hat, wie von der Funktion path_start() zugewiesen.
