Pfad_Aktion

Diese Variable kann verwendet werden, um die Reaktion einer Instanz zu erhalten oder zu ändern, wenn sie das Ende des aktuellen Pfades erreicht. Normalerweise würdest du sie setzen, wenn du den Pfad beginnst, indem du path_start() setzen, aber du kannst dieses Verhalten abhängig von einer beliebigen Anzahl von Ereignissen in deinem Spiel ändern. Die verfügbaren Werte werden durch die folgenden Konstanten ausgedrückt:

Trassenende Aktion Konstante
Konstante Beschreibung
path_action_stop Den Pfad anhalten
path_action_restart Von der Startposition aus fortfahren und zum Start springen, wenn der Weg nicht geschlossen ist
path_action_continue Den Weg von der aktuellen Position aus neu beginnen
path_action_reverse Die Geschwindigkeit des Pfades umkehren (den Pfad rückwärts laufen lassen)

Syntax:

path_endaction;

Rückgabe:

Path End Action Constant

Beispiel:

if (path_endaction == path_action_stop)
{
    path_endaction = path_action_reverse;
}

Die obige code prüft die Endaktion des Pfades, und wenn sie auf "Stop" eingestellt ist, wird die Endaktion in "Reverse" geändert.
