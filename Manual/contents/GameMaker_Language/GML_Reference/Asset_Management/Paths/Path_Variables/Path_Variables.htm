Pfadvariablen

Die folgenden Variablen sind in alle Instanzen eingebaut, wenn sie erstellt werden, und können verwendet werden, um Informationen über einen Pfad zu erhalten oder um bestimmte Eigenschaften zu ändern:

  • path_index
  • path_position
  • path_positionprevious
  • path_speed
  • path_scale
  • path_orientation
  • path_endaction

Beachten Sie, dass Sie hier weitere Informationen über Instanzvariablen finden können: Instanz-Variablen.
