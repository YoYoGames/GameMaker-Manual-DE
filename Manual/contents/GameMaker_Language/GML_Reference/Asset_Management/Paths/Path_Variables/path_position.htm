path_position

Diese Funktion kann verwendet werden, um die Position einer Instanz entlang eines Pfades zu ermitteln oder festzulegen. Der Wert ist normalisiert von 0 - 1, wenn du ihn also z.B. auf 0,5 setzt, wird die Instanz genau in die Mitte des Pfades verschoben.

Syntax:

path_position

Rückgabe:

Beispiel:

path_position = random(1);

Die obige code setzt die Instanz an eine beliebige Position auf dem aktuellen Pfad.
