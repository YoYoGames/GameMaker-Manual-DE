Pfad_Skala

Dieser Wert kann verwendet werden, um den Maßstab des aktuell zugewiesenen Pfades für die Instanz zu erhalten oder einzustellen (wie von der Funktion path_start() festgelegt), mit einem Standardwert von 1. Dies ist ein skalarer Wert, also 1 ist ein Maßstab von 1:1, während ein Wert von 2 beispielsweise eine Verdoppelung des Maßstabs bedeutet und ein Wert von 0,5 eine Halbierung des Maßstabs.

Syntax:

path_scale;

Rückgabe:

Beispiel:

path_scale = 1 + random(2);

Die obige code setzt die Skalierung des aktuellen Pfades auf eine zufällige Größe zwischen 1 (seine Originalgröße) und 3 (das Dreifache seiner Originalgröße).
