Pfad_Index

Alle Ressourcen in GameMaker haben eine eindeutige Identifikationsnummer, und die path_index ist eine schreibgeschützte Variable, die den Indexwert für ein bestimmtes Pfad-Asset enthält, das einer Instanz mit der Funktion path_start() zugewiesen wurde. Wenn der Instanz kein Pfad zugewiesen ist, wird die Variable auf -1 gesetzt.

Syntax:

path_index;

Rückgabe:

Path Asset

Beispiel:

if (path_index == -1)
{
    path_start(pth_enemy3, 4, path_action_reverse, 0);
}

Die obige code prüft, ob der Instanz ein Pfad zugewiesen wurde, und wenn nicht, beginnt sie einen neuen Pfad (und weist ihn gleichzeitig der path_index zu).
