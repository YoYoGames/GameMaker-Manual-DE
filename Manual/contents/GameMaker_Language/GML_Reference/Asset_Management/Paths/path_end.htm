Pfad_Ende

Der Aufruf dieser Funktion beendet den aktuellen Pfad, dem die Instanz folgt, wie er beim Aufruf der Funktion path_start() aufgerufen wurde..

Syntax:

path_end();

Rückgabe:

Beispiel:

if place_meeting(x, y, obj_Blocker)
{
    path_end();
}

Die obige code beendet den aktuellen Pfad, wenn die Instanz eine Kollision mit einer Instanz des angegebenen Objekts feststellt.
