Pfad_get_speed

Diese Funktion gibt den Geschwindigkeitsfaktor einer bestimmten Position auf einem Pfad zurück. Die Position sollte ein Wert zwischen 0 und 1 sein (Sie können path_positionverwenden, wenn Sie z. B. die aktuelle Position der Instanz benötigen), wobei 0 die Startposition und 1 die Endposition ist. Der zurückgegebene Wert ist der Faktor, um den die aktuelle path_speed gesetzt wird. Bei einem Pfad mit nur zwei Punkten, 0 und 1, bei dem der Geschwindigkeitsfaktor für Punkt 0 100 und für Punkt 1 0 ist, gibt die Funktion path_get_speed(path, 0.5) beispielsweise 50 zurück.

Syntax:

path_get_speed(ind, pos);

Argument Typ Beschreibung
index Pfad Anlage The index of the path to check.
pos Real How far through the path to check. Between 0 (start) and 1 (end), can be decimal.

Rückgabe:

Real

Beispiel:

speed = path_get_speed(pth_Patrol, 0.5) / 20;

Dies setzt die Geschwindigkeit der aufrufenden Instanz auf den Geschwindigkeitsfaktor des Punktes, der genau in der Mitte des angegebenen Pfades liegt, geteilt durch 20. Wenn der gefundene Geschwindigkeitsfaktor also der Standardwert 100 ist, wird die Geschwindigkeit der aufrufenden Instanz auf 5 gesetzt.
