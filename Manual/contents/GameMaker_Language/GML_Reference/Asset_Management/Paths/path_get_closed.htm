path_get_closed

Mit dieser Funktion kann zurückgegeben werden, ob der Pfad als geschlossen (true) oder offen (false) gekennzeichnet ist, d.h. ob der Pfad loops oder ob er einen definitiven Anfang und ein definitives Ende hat.

Syntax:

path_get_closed(index);

Argument Typ Beschreibung
index Pfad Anlage The index of the path to check.

Rückgabe:

Boolean

Beispiel:

state = path_get_closed(pth_Patrol);

Dadurch wird "state" entweder auf true oder false gesetzt, je nach dem geschlossenen Zustand des in "pth_Patrol" indizierten Pfades.
