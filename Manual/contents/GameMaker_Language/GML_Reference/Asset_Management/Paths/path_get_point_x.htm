pfad_get_punkt_x

Diese Funktion gibt die x-Position (in Raumkoordinaten) des Punktes zurück, den Sie für den Pfad, den Sie indizieren, eingegeben haben. Wenn der Punkt außerhalb des Bereichs des Pfades liegt (z.B.: ein Pfad hat 8 Punkte und Sie fragen nach der x-Position von Punkt 10), wird der Wert 0 zurückgegeben.

Syntax:

path_get_point_x(index, n);

Argument Typ Beschreibung
index Pfad Anlage The index of the path to check.
n Real The point number to check.

Rückgabe:

Real

Beispiel:

if (path_position == 1)
{
    var _num = path_get_number(pth_Patrol);
    var _pos = floor(random(_num));
    x = path_get_point_x(pth_Patrol, _pos);
    y = path_get_point_y(pth_Patrol, _pos);
    path_position = (1 / _num) * _pos;
}

Die obige code prüft, ob sich eine Instanz am Ende eines Pfades befindet. Wenn dies der Fall ist, wird ein zufälliger Punkt auf dem Pfad ausgewählt und die Instanz dorthin verschoben.
