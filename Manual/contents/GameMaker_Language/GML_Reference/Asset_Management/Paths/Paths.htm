Pfade

GameMaker verfügt über eine umfangreiche Sammlung von Funktionen und eingebauten Variablen zur Verwendung von Pfaden in Ihrem Spiel. Dieser Abschnitt befasst sich mit den grundlegenden Funktionen, die beim Umgang mit Pfaden verwendet werden, die zuvor im Pfad-Editor erstellt wurden. Es gibt jedoch noch weitere Funktionen, die es Ihnen ermöglichen, mit code eigene Pfade zu erstellen und sie auf verschiedene Weise zu manipulieren.

Es gibt die folgenden Grundfunktionen, die sich mit Pfaden befassen:

  • path_start
  • path_end
  • path_get_closed
  • path_get_kind
  • path_get_length
  • path_get_name
  • path_get_number
  • path_get_point_speed
  • path_get_point_x
  • path_get_point_y
  • path_get_precision
  • path_get_speed
  • path_get_x
  • path_get_y

Zur Unterstützung der debugging Pfade gibt es eine Funktion, mit der sie auch auf den Bildschirm gezeichnet werden können:

  • draw_path

Jede Instanz eines Objekts verfügt auch über eine Reihe von eingebauten Variablen, die die Verwendung von Pfaden erleichtern sollen und die im folgenden Abschnitt aufgeführt sind:

  • Pfadvariablen

Schließlich ist es möglich, den Inhalt eines Pfades zu manipulieren und neue Pfade zu erstellen, während das Spiel läuft. Die Funktionen, die diese Dinge und mehr ermöglichen, sind im folgenden Abschnitt zu finden:

  • Pfadmanipulation
