Pfad_get_kind

Pfade können entweder glatt oder gerade sein (ein glatter Pfad berechnet einen gekrümmten Pfad um die definierten Punkte, während ein gerader Pfad einfach nur von einem Punkt zum anderen führt). Diese Funktion kann verwendet werden, um herauszufinden, ob der angegebene Pfad glatt ist (true) oder nicht (false).

Syntax:

path_get_kind(index);

ArgumentTyp Beschreibung
index The index of the path to check.

Rückgabe:

Beispiel:

path_kind = path_get_kind(pth_Patrol);

Dadurch wird die Variable "path_kind" auf true oder false gesetzt, je nachdem, um welche Art von Pfad es sich bei dem angegebenen Pfadindex handelt.
