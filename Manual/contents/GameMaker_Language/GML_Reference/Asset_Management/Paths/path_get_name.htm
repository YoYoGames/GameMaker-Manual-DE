path_get_name

Diese Funktion gibt den Namen des Pfades, auf den verwiesen wird, als String zurück. Der Name ist das, was auch immer verwendet wurde, um den Pfad im Editor zu definieren, oder (wenn der Pfad durch eine code Funktion erstellt wurde) gibt sie eine Zeichenkette mit dem Format " _newpathXX" zurück, wobei "XX" die Nummer des erzeugten Pfades ist, die bei 0 beginnt und jedes Mal um eins erhöht wird, wenn ein neuer Pfad erzeugt wird. Bitte beachten Sie, dass es sich nur um eine Zeichenkette ist und nicht verwendet werden kann, um direkt auf den Pfad zu verweisen - dazu bräuchten Sie die Pfadindex-ID. Sie können diese Zeichenfolge jedoch verwenden, um die Pfadindex-ID zu ermitteln, indem Sie die zurückgegebene Zeichenfolge zusammen mit mit der Funktion asset_get_index().

Syntax:

path_get_name(index);

ArgumentTyp Beschreibung
index The index of the path to check.

Rückgabe:

Beispiel:

path_name = path_get_name(path_array[0]);

Dies setzt "path_name" auf den Namen des Pfades, der im angegebenen Array an Position 0 indiziert ist.
