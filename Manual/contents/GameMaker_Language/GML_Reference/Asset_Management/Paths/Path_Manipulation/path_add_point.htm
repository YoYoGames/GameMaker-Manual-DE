pfad_hinzuf_punkt

Mit dieser Funktion können Sie einen Punkt zu dem angegebenen Pfad hinzufügen und seinen Geschwindigkeitsfaktor festlegen. Dieser Punkt wird am Ende des Pfades hinzugefügt, unabhängig von der Position des Punktes, und der Faktor ist gleich dem Prozentsatz von path_speed den die folgende Instanz tatsächlich fährt, wenn sie diesen Punkt auf dem Pfad erreicht. Wenn du einen Pfadpunkt an eine andere Position setzen willst, die nicht das Ende ist, solltest du path_insert_point().

Wenn Sie die Funktion auf einen Pfad aus dem Asset-Browser anwenden, beachten Sie, dass die Funktion das eigentliche Asset ändert und somit dauerhaft beeinflusst, wie der Pfad von allen Instanzen im Spiel ab dem Zeitpunkt der Anwendung der Funktion bis zum Ende des Spiels verwendet wird. Wenn Sie dies nicht wünschen, sollten Sie eine Funktion wie path_duplicate() verwenden, um zunächst eine Kopie des Pfades zu erstellen und dann diese Funktion für das duplizierte Asset aufzurufen (vergessen Sie nicht, path_delete() für das Asset aufzurufen, wenn es nicht mehr benötigt wird).

Syntax:

path_add_point(index, x, y, speed);

Argument Typ Beschreibung
index Pfad Anlage The index of the path to add the point to.
x Real The x coordinate of the new point.
y Real The y coordinate of the new point.
speed Real The speed factor of the point (default value is 100).

Rückgabe:

N/A

Beispiel:

my_path = path_add();
for (var i = 0; i <= 360; i += 36)
{
    path_add_point(my_path, x + lengthdir_x(50, i), y + lengthdir_y(50, i), 100);
}

Die obige code erzeugt einen kreisförmigen Pfad um die x/y-Position der Instanz, auf der die code läuft.
