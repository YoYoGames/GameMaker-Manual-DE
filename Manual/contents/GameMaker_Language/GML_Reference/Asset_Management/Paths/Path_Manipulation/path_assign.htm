Pfad_zuweisen

Mit dieser Funktion können Sie die Pfaddaten von einem Pfad auf einen anderen kopieren. Der Pfad, auf den kopiert wird, wird zuerst gelöscht (falls er noch Pfadpunkte hat) und dann komplett mit dem Pfad überschrieben, von dem kopiert wird. Keiner der beiden Pfade wird dabei gelöscht, und das Ergebnis sind zwei Pfade mit zwei verschiedenen Indizes, aber genau derselben Form und denselben Punkten. Im Allgemeinen sollten Sie diese Funktion auf einen Pfad anwenden, der mit path_add() erstellt wurde, denn wenn man sie auf ein Pfad-Asset anwendet, wirkt sie sich dauerhaft auf den Pfad für alle Instanzen im Spiel aus, von dem Moment an, in dem die Funktion verwendet wird, bis zum Ende des Spiels.

Syntax:

path_assign(index, path);

ArgumentTyp Beschreibung
index The index of the path to be overwritten.
path The index of the path that will overwrite 'index'.

Rückgabe:

Beispiel:

mypath = path_add();
path_assign(mypath, choose(path_1, path_2, path_3));

Die obige code erstellt einen neuen, leeren Pfad, der in der Variablen "mypath" indiziert ist, und kopiert dann die Pfaddaten aus einer der drei verfügbaren Pfadressourcen.
