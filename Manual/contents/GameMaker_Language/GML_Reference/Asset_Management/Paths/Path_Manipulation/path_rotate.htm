path_rotate

Mit dieser Funktion können Sie einen gegebenen Pfad um seinen Mittelpunkt drehen. Denken Sie daran, dass in GameMaker (sofern Sie nicht mit Physik arbeiten) die Winkel gegen den Uhrzeigersinn berechnet werden, so dass eine Drehung des Pfades um 90 Grad den Pfad nach links drehen würde. Diese Funktion ändert die tatsächliche Pfadressource und wirkt sich somit dauerhaft darauf aus, wie der Pfad von allen Instanzen im Spiel verwendet wird, von dem Moment an, in dem die Funktion verwendet wird, bis zum Ende des Spiels. Wenn Sie dies nicht wünschen, sollten Sie eine Funktion wie path_duplicate() verwenden, um zunächst eine Kopie des Pfades zu erstellen und dann diese Funktion für das duplizierte Asset aufrufen (vergessen Sie nicht, path_delete() für das Asset aufzurufen, wenn es nicht mehr benötigt wird).

HINWEIS: Sobald ein Pfad einer Instanz mit der Funktion path_start() Funktion zugewiesen wurde, können Sie die Variable path_orientation verwenden, um auch die Pfadrotation zu ändern.

Syntax:

path_rotate(index, angle);

Argument Typ Beschreibung
index Pfad Anlage The index of the path to flip.
angle Real The angle of rotation degrees.

Rückgabe:

N/A

Beispiel:

path_rotate(pth_Patrol, 90);

Dies würde den angegebenen Wert um 90 Grad gegen den Uhrzeigersinn drehen.
