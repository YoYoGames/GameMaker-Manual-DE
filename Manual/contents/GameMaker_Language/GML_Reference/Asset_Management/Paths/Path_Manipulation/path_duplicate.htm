Pfad_duplizieren

Diese Funktion nimmt einen Pfad und kopiert ihn in einen neuen Pfad. Der neue Pfad wird dabei erstellt, und sein Index wird zurückgegeben, um bei allen weiteren Aufrufen zur Verwendung dieses neuen Pfads verwendet zu werden.

Syntax:

path_duplicate(index);

ArgumentTyp Beschreibung
index The index of the existing path to duplicate.

Rückgabe:

Path Asset

Beispiel:

mypath = path_duplicate(choose(pth_1, pth_2, pth_3, pth_4));

Die obige code wählt eine von vier Pfadressourcen aus, dupliziert sie und speichert den Index des neuen Pfads in der Variablen "mypath".
