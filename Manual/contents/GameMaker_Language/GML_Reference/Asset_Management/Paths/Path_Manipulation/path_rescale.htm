Pfad_Umskalierung

Mit dieser Funktion kann der angegebene Pfad sowohl entlang der vertikalen als auch der horizontalen Achse neu skaliert werden, d. h. jeder der Pfadpunkte wird an eine neue Position verschoben, die dieser Skalierung um die Mitte des Pfades entspricht. Diese Funktion ändert den eigentlichen Pfadwert, und wirkt sich somit dauerhaft darauf aus, wie der Pfad von allen Instanzen im Spiel verwendet wird, von dem Moment an, in dem die Funktion verwendet wird, bis zum Ende des Spiels. Wenn Sie dies nicht wünschen, sollten Sie eine Funktion wie path_duplicate() verwenden, um zunächst eine Kopie des Pfades zu erstellen, und dann diese Funktion für das duplizierte Asset aufrufen (vergessen Sie nicht, path_delete() für das Asset aufzurufen, wenn es nicht mehr benötigt wird).

Syntax:

path_rescale(index, xscale, yscale);

ArgumentTyp Beschreibung
index The index of the path to scale.
xscale What to multiply the current horizontal scale by. Default scale is 1.
yscale What to multiply the current vertical scale by. Default scale is 1.

Rückgabe:

Beispiel:

path_rescale(pth_AI, 3, 3);

Dadurch wird der angegebene Pfad auf das Dreifache seiner ursprünglichen Größe skaliert.
