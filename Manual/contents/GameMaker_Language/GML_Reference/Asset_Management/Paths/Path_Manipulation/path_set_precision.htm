path_set_precision

Mit dieser Funktion können Sie die "Glattheit" eines Pfades ändern. Dieser Wert muss zwischen 1 und 8 liegen, wobei ein niedriger Wert geradere Kanten mit schärferen Kurven zwischen den Punkten erzeugt, während ein höherer Wert die Punkte abrundet und den Pfad viel "kurviger" macht. Beachten Sie, dass diese Funktion keinen sichtbaren Effekt hat, wenn der Pfad nicht im Pfadeditor oder mit der Funktion path_set_kind(). Diese Funktion verändert das eigentliche Pfad-Asset und wirkt sich somit dauerhaft darauf aus, wie der Pfad von allen Instanzen im Spiel verwendet wird, von dem Moment an, in dem die Funktion verwendet wird, bis zum Ende des Spiels. Wenn Sie dies nicht wünschen, sollten Sie eine Funktion wie path_duplicate() verwenden, um zunächst eine Kopie des Pfades zu erstellen und dann diese Funktion für das duplizierte Asset aufzurufen (vergessen Sie nicht, path_delete() für das Asset aufzurufen, wenn es nicht mehr benötigt wird).

Syntax:

path_set_precision(index, prec);

Argument Typ Beschreibung
index Pfad Vermögenswert The index of the path to change.
prec Real The precision of the path. Must be an integer between 1 and 8.

Rückgabe:

N/A

Beispiel:

path_set_precision(pth_Patrol, 2);

Damit wird die Genauigkeit des angegebenen Pfades auf 2 gesetzt.
