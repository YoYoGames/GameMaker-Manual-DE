Pfad_existiert

Diese Funktion gibt zurück, ob ein Pfad mit dem angegebenen Index existiert oder nicht. Beachten Sie, dass diese Funktion einen Fehler auslöst, wenn Sie die Existenz eines Pfades über eine Variable prüfen, die noch nicht deklariert wurde.

Syntax:

path_exists(index);

Argument Typ Beschreibung
index Pfad Vermögenswert The index of the path to check for.

Rückgabe:

Boolean

Beispiel:

if (path_exists(my_path))
{
    path_delete(my_path);
}

Diese code prüft, ob die angegebene Variable einen Pfadindex enthält, und wenn ja, wird der Pfad aus dem Speicher gelöscht.
