Pfad_hinzufügen

Mit dieser Funktion können Sie einen Pfad in GameMaker erstellen, ohne den Pfad-Editor zu benutzen. Diese Funktion gibt den Index des Pfades zurück, der in einer Variablen gespeichert und fortan als Referenz für diesen Pfad verwendet werden sollte. Bitte beachten Sie, dass der erstellte Pfad leer ist, d. h. er hat keine definierten Punkte, so dass Sie die anderen verfügbaren Funktionen verwenden müssen, um dem Pfad Punkte hinzuzufügen oder MP-Gitter zu verwenden, um den Pfad zu erzeugen. Wenn Sie die Verwendung des Pfades beendet haben oder einen neuen Pfad erstellen und seinen Index in derselben Variablen speichern möchten, sollten Sie zunächst den alten Pfad mit path_delete löschen, um Speicherlecks zu vermeiden, die Ihr Spiel zum Absturz bringen können.

Syntax:

path_add();

Rückgabe:

Path Asset

Beispiel:

global.newpath = path_add();

Dadurch wird ein neuer Pfad erstellt und sein Index global.newpath zugewiesen.
