pfad_set_kind

Mit dieser Funktion kann die Art des Pfades festgelegt werden, den die angegebene Ressource haben soll. Dies kann entweder ein geradliniger Pfad sein (auf 0 gesetzt) oder ein geglätteter Pfad (auf 1 gesetzt), wobei in diesem Fall auch die Pfadgenauigkeit berücksichtigt werden muss (die Genauigkeit kann ebenfalls eingestellt werden mit path_set_precision()). Diese Funktion ändert die tatsächliche Pfad-Ressource und beeinflusst somit dauerhaft, wie der Pfad von allen Instanzen im Spiel verwendet wird, von dem Moment an, in dem Wenn dies nicht das ist, was du brauchst, solltest du eine Funktion wie path_duplicate() verwenden, um zuerst eine Kopie des Pfades zu erstellen und dann diese Funktion auf dem duplizierten Asset aufrufen. für das duplizierte Asset auf (vergessen Sie nicht, path_delete() für das Asset aufzurufen, wenn es nicht mehr benötigt wird).

Syntax:

path_set_kind(index, val);

ArgumentTyp Beschreibung
index The index of the path to change.
val The kind of the path, 0 for straight or 1 for smooth.

Rückgabe:

Beispiel:

path_set_kind(pth_Patrol, true);

Damit wird der angegebene Pfad zu einem "glatten" Pfad.
