pfad_loeschen_punkt

Mit dieser Funktion können Sie einen Punkt aus dem angegebenen Pfad entfernen. Pfadpunkte werden von 0 an nummeriert, so dass der erste Punkt immer der 0. und der letzte Pfadpunkt der 1. ist. path_get_number() - Wenn Sie diese Funktion für ein Pfad-Asset aufrufen, wird der Punkt dauerhaft entfernt, und alle Instanzen, denen der Pfad in Zukunft zugewiesen wird, werden den entfernten Punkt nicht mehr haben. Wenn Sie dies nicht wünschen, sollten Sie eine Funktion wie path_duplicate() verwenden, um zunächst eine Kopie des Pfades zu erstellen und dann diese Funktion für das duplizierte Asset aufzurufen (vergessen Sie nicht, path_delete() für das Asset aufzurufen, wenn es nicht mehr benötigt wird).

Syntax:

path_delete_point(ind, n);

Argument Typ Beschreibung
index Pfad Anlage The index of the path to delete a point from.
n Real The defining point to delete.

Rückgabe:

N/A

Beispiel:

var _col = true;
while (_col)
{
    var _num = path_get_number(mypath) - 1;
    if (_num > 2)
    {
        var _x1 = path_get_point_x(my_path, _num);
        var _y1 = path_get_point_y(my_path, _num);
        var _x2 = path_get_point_x(my_path, _num -1);
        var _y2 = path_get_point_y(my_path, _num -1);
        if collision_line(_x1, _y1, _x2, _y2, obj_Wall, true, false)
        {
            path_delete_point(mypath, _num);
            _num -= 1;
        }
        else _col = false;
    }
    else _col = false;
}

Die obige code erzeugt eine loop, die die Punkte auf einer Rückwärtsstrecke vom Ende aus auf eine Kollision mit dem angegebenen Objekt überprüft. Wenn eine Kollision gefunden wird, wird ein Pfadpunkt gelöscht, bis es keine Kollision mehr gibt oder der Pfad nur noch 2 Punkte lang ist.
