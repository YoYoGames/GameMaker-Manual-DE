pfad_geschlossen

Diese Funktion kann verwendet werden, um einen gegebenen Pfad als offen (falsch) oder geschlossen (wahr) zu kennzeichnen. Bei einem geschlossenen Pfad ist der Startpunkt mit dem Endpunkt verbunden und bildet eine loop. Ein offener Pfad hat einen definitiven, unverbundenen Start- und Endpunkt. Diese Funktion ändert das eigentliche Pfad-Asset und wirkt sich somit dauerhaft darauf aus, wie der Pfad von allen Instanzen im Spiel verwendet wird, von dem Moment an, in dem die Funktion verwendet wird, bis zum Ende des Spiels. Wenn dies nicht das ist, was Sie benötigen, dann sollten Sie eine Funktion wie path_duplicate() verwenden, um zuerst eine Kopie des Pfades zu erstellen und dann diese Funktion für das duplizierte Asset aufrufen (vergessen Sie nicht, path_delete() für das Asset aufzurufen, wenn es nicht mehr benötigt wird).

Syntax:

path_set_closed(index, closed);

Argument Typ Beschreibung
index Pfad Vermögenswert The index of the path to change.
closed Boolesche Whether the path is closed (true) or not (false).

Rückgabe:

N/A

Beispiel:

path_set_closed(pth_Patrol, true);

Damit wird der angegebene Pfad zu einem geschlossenen Pfad.
