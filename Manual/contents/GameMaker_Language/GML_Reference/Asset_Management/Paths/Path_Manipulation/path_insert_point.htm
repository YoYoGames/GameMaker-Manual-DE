pfad_einfuegen_punkt

Mit dieser Funktion können Sie einen neuen Punkt in einen Pfad einfügen (der Pfad kann im Pfadeditor oder über code mit path_add()). Der Punkt wird vor dem Punkt "n", der in der Funktion angegeben ist, in den Pfad eingefügt.

Syntax:

path_insert_point(index, n, x, y, speed);

Argument Typ Beschreibung
index Pfad Anlage The index of the path to insert the point into.
n Real The defining point to insert the new point BEFORE.
x Real The x coordinate (relative to the path) of the new point.
y Real The y coordinate (relative to the path) of the new point.
speed Real The speed factor of the point.

Rückgabe:

N/A

Beispiel:

path_insert_point(mypath, 0, 50, 50, 100);

Dadurch wird ein Punkt ganz am Anfang des in der Variablen "mypath" indizierten Pfades bei (50,50) mit einem Geschwindigkeitsfaktor von 100% eingefügt.
