path_append

Mit dieser code können Sie einen Pfad an einen anderen anhängen und sie so miteinander verbinden. Damit dies funktioniert, müssen beide Pfade bereits existieren. Im Allgemeinen sollten Sie diese Funktion auf Pfade anwenden, die mit path_add() erstellt wurden, denn wenn Sie sie auf ein Pfad-Asset anwenden, wirkt sie sich dauerhaft auf den Pfad für Instanzen im Spiel aus, von dem Moment an, in dem die Funktion verwendet wird, bis zum Ende des Spiels.

HINWEIS: Dadurch wird der angehängte Pfad nicht aus dem Spiel entfernt. Er ist immer noch da, nur hat er jetzt keine Punkte mehr und wenn Sie ihn nicht mehr benutzen wollen, sollten Sie ihn mit der Funktion path_delete().

Syntax:

path_append(index, path)

ArgumentTyp Beschreibung
index The new index of the path.
path The index of the path that will be appended to 'index'.

Rückgabe:

N/A

Beispiel:

path_append(mypath, mppath);
path_delete(mppath);

Die obige code fügt die in der Variablen "mppath" indizierten Pfaddaten an den in der Variablen "mypath" indizierten Pfad an und löscht dann die Variable "mppath" aus dem Speicher.
