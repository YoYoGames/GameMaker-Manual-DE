pfad_get_präzision

Diese Funktion gibt zurück, mit welcher Genauigkeit der angegebene Pfad "geglättet" wurde, und ist ein ganzzahliger Wert von 1 bis 8. Obwohl man diesen Wert für einen geradlinigen Pfad erhalten (und einstellen) kann, hat er keinen Einfluss darauf, wie eine Instanz den Pfad verwendet, da er nur relevant ist, wenn die Pfadart auf "glatt" eingestellt ist.

Syntax:

path_get_precision(index);

ArgumentTyp Beschreibung
index The index of the path to check.

Rückgabe:

Beispiel:

if (path_get_kind(pth_Patrol))
{
    if (path_get_precision(pth_Patrol)) != 8
    {
        path_set_precision(pth_Patrol, 8);
    }
}

Die obige code prüft den in "pth_Patrol" indizierten Pfad, um festzustellen, um welche Art von Pfad es sich handelt, und wenn er glatt ist, wird geprüft, wie genau die Glättung ist. Ist er kleiner als 8, wird er auf einen Wert von 8 gesetzt.
