pfad_get_x

Mit dieser Funktion können Sie die x-Koordinate einer Position auf einem beliebigen Pfad ermitteln. 0 ist der Anfang des Pfades, 1 ist das Ende des Pfades, und alles dazwischen entspricht der entsprechenden Länge des Pfades. Dies muss kein definierter Punkt des Pfades sein, sondern kann überall auf dem Pfad liegen.

Syntax:

path_get_x(ind, pos);

Argument Typ Beschreibung
index Pfad Vermögenswert The index of the path to check.
pos Real How far through the path to check. Between 0 (start) and 1 (end).

Rückgabe:

Real

Beispiel:

x = path_get_x(pth_WallWalk, 0.5);

Dadurch wird die x-Koordinate der aufrufenden Instanz auf die x-Koordinate des Punktes gesetzt, der sich genau in der Mitte des angegebenen Pfades befindet.
