audio_sound_gain

Mit dieser Funktion können Sie einen Ton über eine bestimmte Zeitspanne ein- oder ausblenden, oder Sie können die Tonverstärkung sofort einstellen.

Die Zeit wird in Millisekunden gemessen, und die Funktion verlangt, dass Sie einen Endpegel eingeben, den der Sound am Ende dieser Zeit erreicht haben soll. Diese Verstärkung kann zwischen 0 (stumm) und einem beliebigen Wert über 0 liegen, obwohl man normalerweise die maximale Lautstärke als 1 ansieht. Alles über 1 kann verwendet werden, aber je nach verwendetem Sound und der platform, zu der kompiliert wird, kann es zu Verzerrungen oder Clipping kommen, wenn der Sound wiedergegeben wird. Beachten Sie, dass die Verstärkungsskala linear ist, und um die Verstärkung sofort zu ändern, setzen Sie einfach die Zeit argument auf 0.

Diese Funktion wirkt sich auf alle Instanzen des Klangs aus, die gerade im Raum abgespielt werden, wenn der Index eine Klangressource ist, und die Endlautstärke ist die Lautstärke, mit der alle weiteren Instanzen des Klangs abgespielt werden. Wenn du jedoch den Index, der von einer Funktion wie audio_play_sound() zurückgegeben wird, wirkt sich das nur auf diese eine Instanz des Tons aus.

Syntax:

audio_sound_gain(index, volume, time);

Argument Typ Beschreibung
index Sound Asset oder Sound Instanz ID The index of the sound to set the gain for.
volume Real Value for the music volume.
time Real The length for the change in gain in milliseconds.

Rückgabe:

N/A

Beispiel:

if val
{
    var snd = audio_play_sound(snd_fountain);
    audio_sound_gain(snd, 0, 0);
    audio_sound_gain(snd, 1, 5000);
}

Die obige code prüft eine Variable, und wenn sie true zurückgibt, wird der Index eines abzuspielenden Tons der lokalen Variable "snd" zugewiesen. Diese Variable wird dann verwendet, um die Lautstärke dieses spezifischen Tons auf 0 zu reduzieren und über 5 Sekunden auf volle Lautstärke zu erhöhen.
