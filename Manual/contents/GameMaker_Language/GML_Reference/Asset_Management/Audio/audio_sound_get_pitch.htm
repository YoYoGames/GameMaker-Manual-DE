audio_sound_get_pitch

Diese Funktion kann verwendet werden, um die Tonhöhe eines bestimmten Klangs zu ermitteln. Der Sound kann entweder aus einem Index für einen individuellen Sound referenziert werden, der in einer Variablen gespeichert wurde, wenn die Funktion audio_play_sound() oder audio_play_sound_at() Funktionen gespeichert wurde, oder ein tatsächliches Sound-Asset aus dem Asset-Browser.

Syntax:

audio_sound_get_pitch(index);

Argument Typ Beschreibung
index Sound Asset oder Sound Instanz ID The index of the sound to get the pitch of.

Rückgabe:

Real

Beispiel:

if audio_sound_get_pitch(snd_Explode) != 1
{
    audio_sound_pitch(snd_Explode, 1);
}

Die obige code ändert die Tonhöhe des Tons, der von dem als "snd_Explode" indizierten Ton abgespielt wird, wenn seine Tonhöhe ungleich 1 ist.
