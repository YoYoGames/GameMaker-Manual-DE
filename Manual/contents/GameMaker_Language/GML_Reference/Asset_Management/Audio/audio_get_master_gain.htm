audio_get_master_gain

Mit dieser Funktion können Sie den absoluten Wert für die Gesamtlautstärke aller Klänge und Musik für einen bestimmten Hörer ermitteln. Der Standard-Hörerindex ist 0, aber Sie können die Funktion audio_get_listener_info() verwenden, um die verschiedenen verfügbaren Indizes für das Ziel platform zu erhalten. Der zurückgegebene Verstärkungswert basiert auf einer linearen Skala von 0 (leise) bis 1 (volle Lautstärke). Beachten Sie, dass Sie auf einigen platforms eine Verstärkung von mehr als 1 haben können, obwohl ein Wert von 1 als "volle Lautstärke" gilt und alles, was darüber liegt, zu Audio Clipping führen kann.

Syntax:

audio_get_master_gain(listenerIndex);

Argument Typ Beschreibung
listenerIndex Real- oder Audio-Hörer-ID The index of the listener to get the gain of.

Rückgabe:

Real

Beispiel:

var num = audio_get_listener_count();
for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var ind = info[? "index"];
    if audio_get_master_gain(ind) != 1
    {
        audio_set_master_gain(info[? "index"], 1);
    }
    ds_map_destroy(info);
}

Die obige code loops durchläuft die verfügbaren Hörer und prüft, ob ihr Gain 1 ist oder nicht, und wenn nicht, setzt sie ihn für jeden von ihnen auf 1.
