audio_sound_set_listener_mask

Mit dieser Funktion können Sie die Bit-Maske für einen Sound so einstellen, dass er nur von den angegebenen Hörern abgespielt wird. Sie können eine Bit-Maske erstellen, indem Sie die Funktion audio_get_listener_info() und dann die bitweise or ("|") verwenden, um eine benutzerdefinierte Maske mit den Zuhörern zu erstellen, von denen der Sound abgespielt werden soll, und diese benutzerdefinierte Maske dann auf den Sound anwenden, nachdem Sie ihn abgespielt haben. Diese Maske überschreibt die Standardmaske oder die Maske, die Sie mit der Funktion audio_set_listener_mask().

Diese Funktion kann nur auf einen Sound angewendet werden, nachdem er zum Abspielen eingestellt wurde, und benötigt die Sound-ID, die von der Funktion zum Abspielen des Sounds zurückgegeben wird.

Syntax:

audio_sound_set_listener_mask(soundID, mask);

Argument Typ Beschreibung
soundID Sound-Instanz-ID The unique ID of the sound to set the mask of
mask Real The bitmask data to set for the sound

Rückgabe:

N/A

Beispiel:

var snd = audio_play_sound(snd_PlayerDead, 10, false);
audio_sound_set_listener_mask(snd, global.PlayerMask);

Die obige code spielt einen Ton ab und legt dann den/die Hörer fest, die mit Hilfe der in einer globalen Variablen gespeicherten Maskendaten abgespielt werden sollen.
