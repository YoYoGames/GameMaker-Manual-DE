audio_set_master_gain

Mit dieser Funktion können Sie den absoluten Wert für die Gesamtlautstärke aller Klänge und Musik für einen bestimmten Hörer einstellen. Der Standard-Hörerindex ist 0, aber Sie können die Funktion audio_get_listener_info() verwenden, um die verschiedenen Indizes für das Ziel platform zu erhalten. Der Verstärkungswert basiert auf einer linearen Skala von 0 (stumm) bis 1 (volle Lautstärke) und wirkt sich auf die relative Lautstärke aller Geräusche und Musik aus, die in Ihrem Spiel über diesen Hörer abgespielt werden.

Syntax:

audio_set_master_gain(listenerIndex, gain);

Argument Typ Beschreibung
listenerIndex Real- oder Audio-Hörer-ID The index of the listener to set the gain on.
gain Real Value for the global volume (0 to 1).

Rückgabe:

N/A

Beispiel:

var num = audio_get_listener_count();
for( var i = 0; i < num; i++;)
{
    var info = audio_get_listener_info(i);
    audio_set_master_gain(info[? "index"], 0.75);
    ds_map_destroy(info);
}

Die obige code loops durchläuft die verfügbaren Hörer und setzt dann deren Master-Gain auf 0,75.
