audio_get_recorder_count

Diese Funktion gibt die Anzahl der Audio-Aufnahmequellen (wie Mikrofone, etc...) zurück, die derzeit für Ihr Spiel verfügbar sind. Wenn der Rückgabewert z.B. vier ist, dann haben Sie Audioeingang auf den Quellen 0, 1, 2 und 3, wobei einer dieser Werte derjenige ist, mit dem Sie die Aufnahme mit der Funktion audio_start_recording(). Dieser Wert kann sich jederzeit ändern, wenn Personen Mikrofone oder andere Eingangsquellen an das Gerät anschließen oder abziehen. Beachten Sie, dass Sie mit der Funktion audio_get_recorder_info() verwenden, um Informationen über jedes angeschlossene Gerät zu erhalten.

WICHTIG Diese Funktion gibt auf dem Opera GX-Target immer 1 zurück.

Syntax:

audio_get_recorder_count();

Rückgabe:

Real

Beispiel:

if audio_get_recorder_count() > 0
{
    channel_index = audio_start_recording(0);
}

Die obige code prüft die verfügbaren Audiorecorder, und wenn es 1 oder mehrere gibt, beginnt die Aufnahme mit dem ersten indizierten (Quelle 0).
