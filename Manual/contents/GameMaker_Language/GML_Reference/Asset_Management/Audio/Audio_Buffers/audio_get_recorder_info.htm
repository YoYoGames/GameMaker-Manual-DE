audio_get_recorder_info

Diese Funktion gibt eine DS-Map mit Informationen über den angegebenen Schreiberquellenindex zurück. Sie können herausfinden, wie viele Schreiberquellen verfügbar sind, indem Sie die Funktion audio_get_recorder_count()und die zurückgegebene Karte enthält die folgenden Schlüssel/Wertpaare:

  • "Name" - ein Name zur Beschreibung des Geräts
  • "index" - der Index, der für die Aufzeichnung verwendet werden soll
  • "data_format" - das Format, in dem die Daten zurückgegeben werden (derzeit ist dies immer buffer_s16, aber andere Formate können in Zukunft unterstützt werden)
  • "sample_rate" - die Abtastrate (in Hz) der zurückgegebenen Daten (derzeit auf 16000 Hz festgelegt, dies kann sich jedoch in Zukunft ändern)
  • "Kanäle" - die Konstante audio_mono (weitere Konstanten für Stereo und 3D werden möglicherweise in Zukunft unterstützt)

Beachten Sie, dass die Funktion zwar eine DS-Map für Sie erstellt, diese aber später nicht wieder entfernt. Sie sollten die Map also selbst löschen, wenn sie nicht mehr benötigt wird, um Speicherverluste zu vermeiden.

HINWEIS Die meisten platforms unterstützen die Aufzeichnung von Audio in irgendeiner Form, aber das bedeutet nicht, dass alle Geräte dies zulassen, selbst wenn die platform dies tut, daher sollten Sie immer überprüfen, ob die audio_get_recorder_count() einen Wert größer als 0 zurückgibt, um zu prüfen, ob Aufnahmegeräte verfügbar sind, bevor Sie die übrigen Aufnahmefunktionen verwenden.

Syntax:

audio_get_recorder_info(recorder_index);

Argument Typ Beschreibung
recorder_index Real The index of the recorder source to get the information from.

Rückgabe:

DS Map ID

Beispiel:

len = async_load[? "data_len"];
audio_buff = buffer_create(len, buffer_fast, 1);
buffer_copy(async_load[? "buffer_id"], 0, len, buff, 0);
audio_queue_sound(audio_queue, audio_buff, 0, len);
audio_play_sound(audio_queue, 10, 0);

Die obige code würde im asynchronen Audio-Aufnahme-Ereignis aufgerufen werden und weist einige aufgenommene Daten einer buffer zu, die dann einer Audio-Warteschlange hinzugefügt werden. Diese wird dann abgespielt.
