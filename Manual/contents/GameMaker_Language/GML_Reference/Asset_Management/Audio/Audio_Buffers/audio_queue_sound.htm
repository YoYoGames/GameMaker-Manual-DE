audio_queue_sound

Diese Funktion fügt die Daten von einer buffer in die Audio-Warteschlange ein, die Sie zuvor mit der Funktion audio_create_play_queue(). Sie geben den Index der Warteschlange an, zu der hinzugefügt werden soll, und die buffer ID, von der die Daten stammen sollen, sowie die Position (Offset) innerhalb der buffer und die Anzahl der hinzuzufügenden Bytes. Sobald Sie Audio aus einer buffer zu einer Warteschlange hinzugefügt haben, kann die buffer erst gelöscht werden, nachdem Sie die Warteschlange mit der Funktion audio_free_play_queue()Die Eigenschaften von buffer sollten mit denen der Warteschlange übereinstimmen, zu der Sie den Ton hinzufügen.

Syntax:

audio_queue_sound(queueIndex, bufferId, bufferOffset, bufferLength);

Argument Typ Beschreibung
queueIndex Audio-Warteschlangen-ID The index of the queue to add to.
bufferId Puffer-ID The buffer to add to the queue.
bufferOffset Real The offset within the source buffer to start from.
bufferLength Real The length of the buffer (the number of the bytes in the buffer).

Rückgabe:

N/A

Beispiel:

len = async_load[? "data_len"];
audio_buff = buffer_create(len, buffer_fast, 1);
buffer_copy(async_load[? "buffer_id"], 0, len, buff, 0);
audio_queue_sound(audio_queue, audio_buff, 0, len);
audio_play_sound(audio_queue, 10, 0);

Die obige code würde im asynchronen Audio-Aufnahme-Ereignis aufgerufen werden und weist einige aufgenommene Daten einer buffer zu, die dann einer Audio-Warteschlange hinzugefügt werden. Diese wird dann abgespielt.
