audio_stop_aufzeichnung

Diese Funktion beendet die Aufzeichnung auf dem angegebenen Rekorderkanal (der Kanalindex wird beim Aufruf der Funktion audio_start_recording()). Wenn Sie die Aufzeichnung stoppen, werden keine weiteren asynchronen Audioaufzeichnungsereignisse für den angegebenen Rekorderkanal ausgelöst. Daher sollten Sie diese Funktion normalerweise im eigentlichen asynchronen Ereignis verwenden, um sicherzustellen, dass Sie alle Daten aufgezeichnet haben.

HINWEIS Die meisten platforms unterstützen die Aufzeichnung von Audio in irgendeiner Form, aber das bedeutet nicht, dass alle Geräte dies zulassen, selbst wenn die platform dies tut, daher sollten Sie immer überprüfen, ob die audio_get_recorder_count() einen Wert größer als 0 zurückgibt, um zu prüfen, ob Aufnahmegeräte verfügbar sind, bevor Sie die übrigen Aufnahmefunktionen verwenden.

Syntax:

audio_stop_recording(channel_index);

Argument Typ Beschreibung
channel_index Real The index of the recorder channel to stop.

Rückgabe:

N/A

Beispiel:

audio_stop_recording(audio_channel);

Die obige code weist GameMaker an, die Aufnahme auf dem angegebenen Audiokanalindex zu beenden.
