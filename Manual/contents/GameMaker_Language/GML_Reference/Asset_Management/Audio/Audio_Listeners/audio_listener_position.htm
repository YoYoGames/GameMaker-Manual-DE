audio_listener_position

Mit dieser Funktion können Sie die Position des Zuhörers im 3D-Audio-Raum ändern. Das Beispielbild unten zeigt die Standardposition für den Hörer im Audioraum:

HINWEISWenn Sie mehrere Hörer haben, verwenden Sie die Funktion audio_listener_set_position().

Wie du sehen kannst, ist die Standardposition (0, 0, 0), aber normalerweise würdest du diese Funktion verwenden, um den Hörer mit dem Spielerobjekt innerhalb deines Spiels zu bewegen und so die Art und Weise zu ändern, in der Audio, das von Emittern erzeugt wird, vom Spieler gehört wird, z.B. in dem Bild unten eines Top-Down-Spiels, setzt die Spielerinstanz den Hörer, was dazu führt, dass sich das Audio von den verschiedenen Emittern "ändert", wenn der Spieler sich durch das Level bewegt:

Syntax:

audio_listener_position(x, y, z);

Argument Typ Beschreibung
x Real The x position of the listener (default 0).
y Real The y position of the listener (default 0).
z Real The z position of the listener (default 0).

Rückgabe:

N/A

Beispiel:

if speed > 0
{
    audio_listener_position(x, y, 0);
}

Die obige code prüft, ob die Geschwindigkeit der Player-Instanz über 0 liegt, und wenn ja, aktualisiert sie den Audio-Hörer auf die aktuelle x/y-Position.
