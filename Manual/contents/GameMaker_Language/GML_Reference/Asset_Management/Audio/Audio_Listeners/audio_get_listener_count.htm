audio_get_listener_count

Bestimmte Ziele platforms erlauben mehr als einen Listener, daher ist es wichtig, dass Sie wissen, wie viele das Ziel hat, bevor Sie verschiedene Listener ändern oder verwenden. Diese Funktion gibt die Anzahl der verfügbaren Listener zurück.

Syntax:

audio_get_listener_count();

Rückgabe:

Beispiel:

global.listener_num = audio_get_listener_count();

Die obige code ermittelt die Anzahl der verfügbaren Zuhörer und speichert den Rückgabewert in einer globalen Variablen.
