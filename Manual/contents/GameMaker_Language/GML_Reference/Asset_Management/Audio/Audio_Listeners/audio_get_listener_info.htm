audio_get_listener_info

Diese Funktion erstellt eine DS-Map und füllt sie mit Informationen für den angegebenen Hörer.

HINWEIS Sie sind für die Vernichtung der zurückgegebenen DS Map mit der entsprechenden Funktion verantwortlich.

Die DS-Map enthält die folgenden Schlüssel:

  • "name" - Der Name des Hörers als String, wobei "default" der Standardname des Hörers auf den meisten Zielplattformen ist
  • "mask" - Die Bit-Maske für den Hörer
  • "index" - Der eindeutige Indexwert des Hörers

Der Maskenwert kann verwendet werden, um einen Sound oder Emitter so einzustellen, dass er von mehreren Hörern gleichzeitig abgespielt wird, indem man einfach die bitweise or "|" verwendet, um eine Maske für den Sound zu erzeugen (siehe das Beispiel code unten), während der Index verwendet wird, um die Eigenschaften wie Position oder Geschwindigkeit für einen bestimmten Hörer mit Funktionen wie audio_listener_set_position().

Syntax:

audio_get_listener_info(num);

Argument Typ Beschreibung
num Real- oder Audio-Hörer-ID The listener number to get the data for.

Rückgabe:

DS Map ID

Beispiel:

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i)
{
    var info = audio_get_listener_info(i);
    var m = audio_listener_get_data(info[? "mask"]);
    mask = mask | m;
    ds_map_destroy(info);
}
audio_set_listener_mask(mask);

Die obige code prüft die Anzahl der verfügbaren Listener und ermittelt dann über loops deren Maskenbits, die dann zu einer einzigen Bitmaske kombiniert werden, die auf den globalen Listener angewendet wird.
