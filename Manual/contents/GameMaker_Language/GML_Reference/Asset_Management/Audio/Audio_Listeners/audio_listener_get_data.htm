audio_listener_get_data

Diese Funktion erstellt eine DS-Map und füllt sie mit den Positions-, Geschwindigkeits- und Orientierungswerten für den angegebenen Hörer. Der Standard-Hörerindex ist 0, aber Sie können die Funktion audio_get_listener_info() verwenden, um die verschiedenen verfügbaren Indizes für das Ziel platform zu erhalten. Wenn Sie einen falschen Listener-Index angeben, gibt die Funktion -1 zurück.

HINWEIS Sie sind dafür verantwortlich, dass die zurückgegebene DS-Karte mit der entsprechenden Funktion vernichtet wird.

Die DS-Map enthält die folgenden Schlüssel:

  • "x" - Die x-Position des Hörers
  • "y" - Die y-Position des Hörers
  • "z" - Die z-Position des Hörers
  • "vx" - Die Geschwindigkeit der x-Achse des Zuhörers
  • "vy" - Die Geschwindigkeit der y-Achse des Zuhörers
  • "vz" - Die Geschwindigkeit der z-Achse des Zuhörers
  • "lookat_x" - Die x-Komponente des Look at vector des Listeners
  • "lookat_y" - Die y-Komponente des Blicks auf vector des Hörers
  • "lookat_z" - Die z-Komponente des Blicks auf vector des Hörers
  • "up_x" - Die x-Komponente von up vector des Hörers
  • "up_y" - Die y-Komponente des up vector des Hörers
  • "up_z" - Die z-Komponente des up vector des Hörers

Syntax:

audio_listener_get_data(index);

Argument Typ Beschreibung
index Real- oder Audio-Hörer-ID The listener to get the data for (default is 0).

Rückgabe:

DS Map ID

Beispiel:

var num = audio_get_listener_count();
for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var data = audio_listener_get_data(info[? "index"]);
    if data[? "x"] != 0
    {
        audio_listener_set_position(info[? "index"], 0, 0, 0);
    }
    ds_map_destroy(info);
    ds_map_destroy(data);
}

Die obige code prüft die Anzahl der verfügbaren Hörer, dann loops durch sie und wenn ihre x-Position nicht 0 ist, setzt es ihre Position auf 0, 0, 0.
