audio_set_listener_mask

Wenn Sie mehrere Hörer in einem System verwenden, können Sie die Bitmaske für einen Sound einstellen und ihn nur von dem markierten Hörer hören lassen. Sie können jedoch auch die globale Maske mit dieser Funktion einstellen und alle normal gespielten Töne werden von den durch diese Maske gekennzeichneten Hörern gehört, ohne dass Sie die Maske für jeden Ton einzeln einstellen müssen.

Syntax:

audio_set_listener_mask(mask);

Argument Typ Beschreibung
mask Real The bit-mask data to set for the listeners

Rückgabe:

N/A

Beispiel:

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var m = audio_listener_get_data(info[? "mask"]);
    mask = mask | m;
    ds_map_destroy(info);
}
audio_set_listener_mask(mask);

Die obige code prüft die Anzahl der verfügbaren Listener und ermittelt dann über loops deren Maskenbits, die dann zu einer einzigen Bitmaske kombiniert werden, die auf den globalen Listener angewendet wird.
