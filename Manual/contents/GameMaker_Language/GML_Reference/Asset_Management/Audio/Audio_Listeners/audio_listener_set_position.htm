audio_listener_set_position

Mit dieser Funktion können Sie die Position eines bestimmten Hörers innerhalb des 3D-Audio-Raums ändern. Der Standard-Hörerindex ist 0, aber Sie können die Funktion audio_get_listener_info() verwenden, um die verschiedenen verfügbaren Indizes für das Ziel platform zu erhalten. Das Beispielbild unten zeigt die Standardposition für den Hörer im Audioraum:

Wie du sehen kannst, ist die Standardposition (0, 0, 0), aber normalerweise würdest du diese Funktion verwenden, um den Hörer mit dem Spielerobjekt innerhalb deines Spiels zu bewegen und so die Art und Weise zu ändern, in der Audio, das von Emittern erzeugt wird, vom Spieler gehört wird, z.B. in dem Bild unten eines Top-Down-Spiels, setzt die Spielerinstanz den Hörer, was dazu führt, dass sich das Audio von den verschiedenen Emittern "ändert", wenn der Spieler sich im Level bewegt:

Syntax:

audio_listener_set_position(index, x, y, z);

Argument Typ Beschreibung
index Real- oder Audio-Hörer-ID The listener to set the position of (default 0).
x Real The x position of the listener (default 0).
y Real The y position of the listener (default 0).
z Real The z position of the listener (default 0).

Rückgabe:

N/A

Beispiel:

var _vmat = camera_get_view_mat(view_camera[0]);
audio_listener_set_position(global.Player_Listener, _vmat[0], _vmat[1], _vmat[2]);
audio_listener_set_orientation(info[? "index"], _vmat[3], _vmat[4], _vmat[5], _vmat[6], _vmat[7], _vmat[8]);

Die obige code ruft die Ansichtsmatrix für die Kameraansicht [0] ab und verwendet sie dann, um die Position und Ausrichtung des Audio-Hörers für den Hörer mit der in der globalen Variable "Player_Listener" gespeicherten ID festzulegen.
