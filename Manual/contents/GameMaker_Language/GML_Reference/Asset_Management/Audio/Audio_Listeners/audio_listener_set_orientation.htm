audio_listener_set_orientation

Mit dieser Funktion können Sie die Ausrichtung des angegebenen Hörers im 3D-Audio-Raum ändern. Der Standard-Hörerindex ist 0, aber Sie können die Funktion audio_get_listener_info() verwenden, um die verschiedenen verfügbaren Indizes für das Ziel platform zu erhalten.

Der Blick auf vector und auf vector basiert auf den Werten, die aus den gegebenen relativen x-, y- und z-Positionen aufgelöst werden, und ist standardmäßig (0, 0, 1) für den Blick auf vector und (0, 1, 0) für den Blick auf vector, wie in der nachstehenden Abbildung gezeigt:

Wenn Sie die Ausrichtung des Hörers mit dieser Funktion ändern, wird sich die Art und Weise ändern, wie die von den Audio-Emittern im Spielraum erzeugten Geräusche vom Spieler wahrgenommen werden. Im folgenden Beispiel würden die Geräusche, die vom Emitter erzeugt werden, wenn sich der Hörer an der Standardposition befindet, von unten und rechts vom Hörer kommen, aber mit der neuen Position und Ausrichtung des Hörers werden sie nun als von oben und rechts kommend wahrgenommen.

Syntax:

audio_listener_set_orientation(index, lookat_x, lookat_y, lookat_z, up_x, up_y, up_z);

Argument Typ Beschreibung
index Echt oder Audio-Hörer-ID The listener to set the orientation of.
lookat_x Real The x look vector (default 0).
lookat_y Real The y look vector (default 0).
lookat_z Real The z look vector (default 1).
up_x Real The x up vector (default 0).
up_y Real The y up vector (default 1).
up_z Real The z up vector (default 0).

Rückgabe:

N/A

Beispiel:

var _m = camera_get_view_mat(view_camera[0]);
audio_listener_set_position(global.Player_Listener, _m[0], _m[1], _m[2]);
audio_listener_set_orientation(info[? "index"], _m[3], _m[4], _m[5], _m[6], _m[7], _m[8]);

Die obige code ruft die Ansichtsmatrix für die Kameraansicht [0] ab und verwendet sie dann, um die Position und Ausrichtung des Audio-Hörers für den Hörer mit der in der globalen Variablen "Player_Listener" gespeicherten ID festzulegen.
