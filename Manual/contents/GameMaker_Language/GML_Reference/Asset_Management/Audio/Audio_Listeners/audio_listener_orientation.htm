audio_listener_orientation

Mit dieser Funktion können Sie die Ausrichtung des Hörers im 3D-Audio-Raum ändern. Die Blickrichtung und die Aufwärtsrichtung basieren auf den vectors, die aus den gegebenen relativen x-, y- und z-Positionen aufgelöst werden, und sind standardmäßig (0, 0, 1) für die Blickrichtung und (0, 1, 0) für die Aufwärtsrichtung, wie in der Abbildung unten gezeigt:

HINWEIS wenn Sie mehrere Hörer haben, sollten Sie die Funktion audio_listener_set_orientation().

Wenn Sie die Ausrichtung des Zuhörers mit dieser Funktion ändern, ändert sich die Art und Weise, wie die von den Audio-Emittern im Spielraum erzeugten Geräusche vom Spieler wahrgenommen werden. Im folgenden Beispiel würden die Geräusche, die vom Emitter erzeugt werden, wenn sich der Hörer an der Standardposition befindet, von unten und rechts vom Hörer kommen, aber mit der neuen Position und Ausrichtung des Hörers werden sie nun als von oben und rechts kommend wahrgenommen.

Syntax:

audio_listener_orientation(lookat_x, lookat_y, lookat_z, up_x, up_y, up_z);

Argument Typ Beschreibung
lookat_x Real The x look vector (default 0).
lookat_y Real The y look vector (default 0).
lookat_z Real The z look vector (default 1).
up_x Real The x up vector (default 0).
up_y Real The y up vector (default 1).
up_z Real The z up vector (default 0).

Rückgabe:

N/A

Beispiel:

dx = dcos(direction);
dy = - dsin(direction);
dz = - dsin(zdirection);
audio_listener_position(x, y, z);
audio_listener_orientation(dx, dy, dz, 0, 0, 1);

Die obige code setzt die Position des 3D-Audiohörers auf die x- und y-Variablen einer Instanz und eine zusätzliche z-Variable. Dann wird die Ausrichtung des Audio-Hörers auf drei Variablen gesetzt, die aus den Variablen der Instanz direction und einer zusätzlichen z-Richtungsvariablen berechnet werden.
