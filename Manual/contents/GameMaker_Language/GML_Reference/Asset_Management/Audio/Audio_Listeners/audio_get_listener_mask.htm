audio_get_listener_mask

Diese Funktion gibt die Bitmasken-Daten zurück, die die aktuelle (globale) Standardmaske für die Audio-Hörer definieren.

Syntax:

audio_get_listener_mask();

Rückgabe:

Beispiel:

var g_mask = audio_get_listener_mask();
if g_mask != global.Audio_Mask
{
    var num = audio_get_listener_count();
    global.Audio_Mask = 0;

    for(var i = 0; i < num; ++i;)
    {
        var info = audio_get_listener_info(i);
        var m = audio_listener_get_data(info[? "mask"]);
        global.Audio_Mask = global.Audio_Mask | m;
        ds_map_destroy(info);
    }
    audio_set_listener_mask(mask);
}

Die obige code ermittelt die aktuellen Hörermaskendaten und vergleicht sie mit den in einer globalen Variablen gespeicherten Daten. Wenn sie nicht übereinstimmen, prüft code die Anzahl der verfügbaren Hörer und ruft dann loops auf, um deren Maskenbits zu erhalten, die dann zu einer einzigen Maske kombiniert werden, die auf das Audiosystem angewendet wird, um die globalen Hörer zu definieren.
