audio_pause_alle

Mit dieser Funktion können Sie alle Töne, die gerade abgespielt werden, pausieren.

Syntax:

audio_pause_all();

Rückgabe:

Beispiel:

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    if (global.Pause)
    {
        audio_pause_all();
    }
    else
    {
        audio_resume_all();
    }
}

Die obige code prüft, ob die Taste "P" gedrückt wird, und wenn dies der Fall ist, setzt sie die globale Variable "Pause" auf true oder false und hält dann entweder alle Sounds an oder startet alle zuvor angehaltenen Sounds neu.
