audio_pause_ton

Mit dieser Funktion können Sie einen beliebigen Ton anhalten, der gerade abgespielt wird. Der Sound kann entweder eine einzelne Instanz eines Sounds sein (der Index für einzelne abgespielte Sounds kann in einer Variablen gespeichert werden, wenn die Funktion audio_play_sound() oder audio_play_sound_at() Funktionen gespeichert werden) oder ein Sound-Asset sein, wobei in diesem Fall alle Instanzen des gegebenen Sounds pausiert werden.

Syntax:

audio_pause_sound(index);

Argument Typ Beschreibung
index Sound Asset oder Sound Instanz ID The index of the sound to pause.

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    if global.Pause
    {
        audio_pause_sound(snd_Waterfall);
    }
    else
    {
        audio_resume_sound(snd_Waterfall);
    }
}

Die obige code prüft, ob die Taste "P" gedrückt wird, und wenn dies der Fall ist, setzt sie die globale Variable "Pause" auf true oder false und hält dann entweder den in der Variablen "snd_Waterfall" indizierten Ton an oder setzt den Ton aus dem angehaltenen Zustand fort.
