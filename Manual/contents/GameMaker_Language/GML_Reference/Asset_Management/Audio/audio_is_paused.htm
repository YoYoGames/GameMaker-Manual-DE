audio_ist_pausiert

Diese Funktion prüft, ob der angegebene Sound gerade pausiert ist. Der Sound kann entweder eine einzelne Instanz eines Sounds sein (der Index für einzelne Sounds, die abgespielt werden, kann in einer Variablen gespeichert werden, wenn die Funktion audio_play_sound() oder audio_play_sound_at() Funktionen) oder ein Sound-Asset sein. In diesem Fall werden alle Instanzen des angegebenen Sounds überprüft, und wenn eine von ihnen pausiert, gibt die Funktion true zurück, andernfalls false.

Syntax:

audio_is_paused(index);

Argument Typ Beschreibung
index Sound Asset oder Sound Instanz ID The index of the sound to check.

Rückgabe:

Boolean

Beispiel:

if audio_is_paused(snd_Waterfall)
{
    audio_resume_sound(snd_Waterfall);
}

Die obige code prüft, ob der in der Variablen "snd_Waterfall" indizierte Ton derzeit angehalten ist, und wenn sie true zurückgibt, wird die Wiedergabe des Tons fortgesetzt.
