audio_sound_pitch

Mit dieser Funktion kann die Tonhöhe eines bestimmten Klangs geändert werden. Der Ton kann entweder aus einem Index für einen einzelnen gespielten Ton referenziert werden, der in einer Variablen gespeichert wurde, wenn die Funktion audio_play_sound() oder audio_play_sound_at() Funktionen gespeichert wurde, oder ein tatsächliches Sound-Asset aus dem Asset-Browser. Wenn es sich um einen Index eines abgespielten Tons handelt, wird nur diese Instanz geändert, bei Verwendung eines Ton-Assets aus dem Asset-Browser werden jedoch alle Instanzen dieses abgespielten Ton-Assets geändert.

Die Tonhöhe argument ist ein Tonhöhenmultiplikator, d.h. der Eingabewert multipliziert die aktuelle Tonhöhe mit dem entsprechenden Wert. Der Standardwert 1 bedeutet also keine Tonhöhenänderung, während ein Wert unter 1 die Tonhöhe senkt und ein Wert über 1 die Tonhöhe erhöht. Am besten verwenden Sie für diese Funktion kleine Schrittweiten, da Werte unter 0 oder über 5 möglicherweise ohnehin nicht hörbar sind. Es ist zu beachten, dass die zulässige Gesamttonhöhenänderung auf (1/256) - 256 Oktaven begrenzt ist, so dass jeder Wert darüber oder darunter nicht registriert wird.

Der oben angegebene Wert ist der, auf den GameMaker versucht, den Bereich zu begrenzen, aber dieser Wert ist nicht für alle Ziele garantiert platforms. iOS Der Wert von (1/256) - 8 wird z.B. geklemmt, so dass Sie möglicherweise mit jedem Ziel experimentieren müssen platform und verschiedene Versionen einer Sound-Ressource haben, die jeweils vorverschiebt sind, falls Sie höhere oder niedrigere Oktavwerte benötigen.

Syntax:

audio_sound_pitch(index, pitch);

Argument Typ Beschreibung
index Sound-Asset oder Sound-Instanz-ID The index of the sound to change.
pitch Real The pitch multiplier (default 1).

Rückgabe:

N/A

Beispiel:

var s_engine = audio_play_sound(snd_CarEngine, 10, false);
switch (gear)
{
    case 1: audio_sound_pitch(s_engine, 0.8); break;
    case 2: audio_sound_pitch(s_engine, 0.9); break;
    case 3: audio_sound_pitch(s_engine, 0.95); break;
    case 4: audio_sound_pitch(s_engine, 1); break;
    case 5: audio_sound_pitch(s_engine, 1.2); break;
}

Die obige code ändert die Tonhöhe des Tons, der von dem in der Variablen "s_engine" indizierten Ton abgespielt wird, je nach dem Wert der Variablen "gear".
