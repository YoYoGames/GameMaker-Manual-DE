audio_master_gain

Mit dieser Funktion können Sie den absoluten Wert für die Gesamtlautstärke aller Sounds und Musik einstellen. Sie basiert auf einer linearen Skala von 0 (stumm) bis zu einem Wert größer als 0, obwohl man normalerweise die maximale Lautstärke als 1 ansieht. Alles über 1 kann verwendet werden, aber je nach verwendetem Sound und der platform, zu der kompiliert wird, kann es zu Verzerrungen oder Clipping kommen, wenn der Sound wiedergegeben wird. Diese Funktion beeinflusst die relative Lautstärke aller Sounds und Musik, die im Spiel abgespielt werden.

Syntax:

audio_master_gain(gain);

Argument Typ Beschreibung
gain Real Value for the global volume (0 to 1).

Rückgabe:

N/A

Beispiel:

if keyboard_check(vk_up)
{
    if vol < 1 vol += 0.05;
    audio_master_gain(vol);
}
if keyboard_check(vk_down)
{
    if vol > 0 vol -= 0.05;
    audio_master_gain(vol);
}

Das obige code prüft, ob die Pfeiltasten nach oben und unten gedrückt werden, wodurch die Variable "vol" erhöht oder verringert wird. Diese Variable wird dann verwendet, um die globale Verstärkung aller Sounds und Musik im Spiel einzustellen.
