audio_stop_sound

Diese Funktion hält den angegebenen Sound an, wenn er gerade abgespielt wird. Der Sound kann entweder eine einzelne Instanz eines Sounds sein (der Index für einzelne abgespielte Sounds kann in einer Variablen gespeichert werden, wenn die Funktion audio_play_sound() oder audio_play_sound_at() Funktionen gespeichert werden) oder ein Sound-Asset, wobei in diesem Fall alle Instanzen des angegebenen Sounds gestoppt werden.

Syntax:

audio_stop_sound(index);

Argument Typ Beschreibung
index Sound Asset oder Sound Instanz ID The index of the sound to stop.

Rückgabe:

N/A

Beispiel:

if !global.SFX
{
    audio_stop_sound(snd_Waterfall);
}
else
{
    audio_play_sound_at(snd_Waterfall, x, y, 0, 100, 300, 1, true, 1);
}

Die obige code prüft die globale Variable "SFX" und wenn sie false zurückgibt, stoppt sie den in der Variable "snd_Waterfall" indizierten Ton, der gerade abgespielt wird, und wenn sie true zurückgibt, wird der Ton loop.
