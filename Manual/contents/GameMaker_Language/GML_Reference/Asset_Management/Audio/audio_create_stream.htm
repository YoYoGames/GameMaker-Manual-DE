audio_erzeugen_strom

Mit dieser Funktion können Sie einen neuen Sound-Index erstellen, der dann in den regulären Audio-Funktionen verwendet werden kann, um Audio direkt von einer externen OGG-Dateiquelle zu streamen. Die Funktion benötigt den Dateinamen (der z.B. eine eingebundene Datei sein kann) und gibt den neuen Sound-Index zur Verwendung zurück. Beachten Sie, dass Sie, wenn Sie den Sound nicht mehr benötigen, die Funktion audio_destroy_stream() mit dem Sound-Index aufrufen, um ihn aus dem Speicher zu entfernen. Andernfalls kann es zu einem Speicherleck kommen, das Ihr Spiel verlangsamt und schließlich zum Absturz bringt.

HINWEIS: Diese Funktion ist für das HTML5-Ziel platform nicht verfügbar.

Syntax:

audio_create_stream(filename);

Argument Typ Beschreibung
filename Zeichenfolge Path to the file (OGG only) to stream the audio from.

Rückgabe:

Sound Asset

Beispiel:

snd = audio_create_stream("Music/Track1.ogg");
audio_play_sound(snd, 0, true);

Die obige code erstellt einen neuen Sound-Index in der Variablen "snd" aus der angegebenen Datei und spielt diesen Sound dann ab.
