audio_create_sync_group

Erstellt eine Sync-Gruppe und gibt einen eindeutigen ID-Wert für sie zurück, der dann in allen weiteren Audio-Funktionsaufrufen für diese Gruppe verwendet werden sollte. Wenn die Gruppe loop heißen soll, geben Sie true ein, andernfalls geben Sie false ein. Beachten Sie jedoch, dass alle später hinzugefügten Spuren die gleiche Länge haben müssen, wenn sie loop heißen sollen. Beachten Sie, dass Sie bei der Erstellung einer Sync-Gruppe den Speicher und die Sounds, die mit ihr verbunden sind, freigeben müssen, wenn sie nicht verwendet werden, indem Sie die audio_destroy_sync_group() Funktion freigeben - zum Beispiel bei den Ereignissen Raumende oder Zerstören.

HINWEIS: Diese Funktion ist für das HTML5-Ziel platform nicht verfügbar.

Syntax:

audio_create_sync_group(loop);

Argument Typ Beschreibung
loop Boolesche Whether the tracks in the group should loop (true) oder nicht (false).

Rückgabe:

Audio Sync Group ID

Beispiel:

sg = audio_create_sync_group(true);
audio_play_in_sync_group(sg, sound1);
audio_play_in_sync_group(sg, sound2);
audio_sound_gain(sound2, 0, 0);
audio_play_in_sync_group(sg, sound3);
audio_sound_gain(sound3, 0, 0);
audio_play_in_sync_group(sg, sound4);
audio_sound_gain(sound4, 0, 0);
audio_start_sync_group(sg);

Im obigen Beispiel wird eine neue Sync-Gruppe erstellt und der Index der Gruppe der Variablen "sg" zugewiesen. Dann werden der Gruppe vier Klänge hinzugefügt, wobei die Verstärkung für drei von ihnen auf 0 gesetzt wird. Schließlich wird die Sync-Gruppe abgespielt.
