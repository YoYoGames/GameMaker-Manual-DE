audio_stop_sync_group

Diese Funktion stoppt die angegebene Sync-Gruppe, wenn sie abgespielt wird, wobei der Gruppenindex der Wert ist, der bei der Erstellung der Gruppe mit der Funktion audio_create_sync_group().

HINWEIS: Diese Funktion ist für das HTML5-Ziel platform nicht verfügbar.

Syntax:

audio_stop_sync_group(group_index);

Argument Typ Beschreibung
group_index Audio-Synchronisationsgruppen-ID The group index to stop playing.

Rückgabe:

N/A

Beispiel:

if mouse_check_button_pressed(mb_left)
{
    audio_stop_sync_group(sg);
}

Die obige code prüft auf einen Mausklick, und wenn ein solcher erkannt wird, wird die in der Variablen "sg" indizierte Sync-Gruppe angehalten.
