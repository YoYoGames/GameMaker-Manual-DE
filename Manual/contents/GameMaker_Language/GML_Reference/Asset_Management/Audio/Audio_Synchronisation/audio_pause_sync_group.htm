audio_pause_sync_gruppe

Diese Funktion pausiert die angegebene Sync-Gruppe, wenn sie gerade abgespielt wird, wobei der Gruppenindex der Wert ist, der zurückgegeben wurde, als Sie die Gruppe mit der Funktion audio_create_sync_group(). Dies stoppt den Ton nicht, und der Aufruf von audio_resume_sync_group()wird die Wiedergabe an der gleichen Stelle wieder aufgenommen, an der sie unterbrochen wurde.

HINWEIS: Diese Funktion ist für das HTML5-Ziel platform nicht verfügbar.

Syntax:

audio_pause_sync_group(group_index);

Argument Typ Beschreibung
group_index Audio-Synchronisationsgruppen-ID The group index to pause.

Rückgabe:

N/A

Beispiel:

if (keyboard_check_pressed((ord)"P"))
{
    global.Pause = !global.Pause
    if (global.Pause)
    {
        audio_pause_sync_group(sg);
    }
    else
    {
        audio_resume_sync_group(sg);
    }
}

Die obige code prüft, ob die Taste "P" gedrückt wird, und wenn dies der Fall ist, wird die Variable "global.Pause" umgeschaltet und geprüft, ob die in der Variable "sg" indizierte Sync-Gruppe angehalten oder fortgesetzt werden soll.
