audio_resume_sync_group

Diese Funktion setzt die angegebene Sync-Gruppe fort, wenn sie gerade abgespielt wird und zuvor angehalten wurde (mit der Funktion audio_pause_sync_group). Der Gruppenindex ist der Wert, der bei der Erstellung der Gruppe mit der Funktion audio_create_sync_group().

HINWEIS: Diese Funktion ist für das HTML5-Ziel platform nicht verfügbar.

Syntax:

audio_resume_sync_group(group_index);

Argument Typ Beschreibung
group_index Audio-Synchronisationsgruppen-ID The group index to resume.

Rückgabe:

N/A

Beispiel:

if (keyboard_check_pressed((ord)"P"))
{
    global.Pause = !global.Pause
    if (global.Pause)
    {
        audio_pause_sync_group(sg);
    }
    else
    {
        audio_resume_sync_group(sg);
    }
}

Die obige code prüft, ob die Taste "P" gedrückt wird, und wenn dies der Fall ist, wird die Variable "global.Pause" umgeschaltet und geprüft, ob die in der Variable "sg" indizierte Sync-Gruppe angehalten oder fortgesetzt werden soll.
