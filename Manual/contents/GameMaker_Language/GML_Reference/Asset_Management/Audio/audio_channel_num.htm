audio_channel_num

Mit dieser Funktion können Sie festlegen, wie viele Audiokanäle für die Wiedergabe von Audio in GameMaker zur Verfügung stehen. Das bedeutet im Grunde, dass Sie die Anzahl der gleichzeitig abspielbaren Töne angeben. Übersteigt die Anzahl diesen Wert, werden die Töne mit einer niedrigeren Priorität angehalten, um einen Kanal für die Töne mit einer höheren Priorität freizugeben. Sie können diese Funktion nutzen, um Ihr Spiel für Geräte zu optimieren, denn je niedriger die Anzahl der Audiokanäle ist, desto besser ist die Leistung, aber bedenken Sie, dass dies auch dazu führen kann, dass bestimmte Klänge abgeschnitten werden, wenn viele gleichzeitig gespielt werden.

Syntax:

audio_channel_num(num);

ArgumentTyp Beschreibung
num Number of available audio channels (default is 128).

Rückgabe:

Beispiel:

switch (os_browser)
{
    case browser_not_a_browser:
        switch (os_type)
        {
            case os_windows:
            case os_macos:
                audio_channel_num(200);
            break;

            default:
                audio_channel_num(64);
            break;
        }
    break;

    default:
        audio_channel_num(16);
    break;
}

Das obige code überprüft die platform, auf der das Spiel läuft, und ändert die Anzahl der verfügbaren Soundkanäle, um die Leistung zu erhöhen.
