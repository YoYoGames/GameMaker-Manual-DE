audio_group_unload

Diese Funktion entlädt alle Sounds, die als zur angegebenen Audiogruppe gehörend gekennzeichnet sind, in den Speicher. Die Funktion gibt true zurück, wenn das Entladen eingeleitet wurde, und false, wenn die Gruppen-ID ungültig ist oder die Gruppe bereits zum Entladen markiert wurde. Beachten Sie, dass alle Audios, die gerade abgespielt werden, wenn diese Funktion aufgerufen wird, gestoppt werden.

Syntax:

audio_group_unload(groupID);

Argument Typ Beschreibung
groupID Audio-Gruppen-ID The index of the audio group to unload (as defined in the Audio Groups Window)

Rückgabe:

Boolean

Beispiel:

if audio_group_is_loaded(audiogroup_level1)
{
    audio_group_unload(audiogroup_level1);
}

Die obige code prüft, ob eine Audiogruppe geladen wurde und entlädt sie, wenn dies der Fall ist.
