audio_group_set_gain

Mit dieser Funktion können Sie eine Gruppe von Klängen über eine bestimmte Zeitspanne ein- oder ausblenden, oder Sie können die Gruppenverstärkung sofort einstellen.

Die Zeit wird in Millisekunden gemessen, und die Funktion erfordert, dass Sie einen Endpegel eingeben, den die Gruppe am Ende dieser Zeit erreicht haben soll. Diese Verstärkung kann zwischen 0 (leise) und 1 (volle Lautstärke) liegen, und die Skala ist linear, so dass ein Wert von 0,5 der halben Lautstärke entsprechen würde. Um die Verstärkung sofort zu ändern, setzen Sie die Zeit argument einfach auf 0. Beachten Sie, dass bei einigen platforms eine Verstärkung von mehr als 1 möglich ist, obwohl ein Wert von 1 als "volle Lautstärke" gilt und alles, was darüber hinausgeht, zu Audio-Clipping führen kann.

Syntax:

audio_group_set_gain(groupID, volume, time);

Argument Typ Beschreibung
groupID Audio-Gruppen-ID The index of the audio group to stop (as defined in the Audio Groups Window)
volume Real The final value for the group volume.
time Real The length of the change in gain in milliseconds.

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(vk_space)
{
    audio_group_set_gain(audiogroup1, 0, 5000);
}

Die obige code prüft, ob die Leertaste gedrückt wurde, und blendet dann den gesamten Ton für "audiogroup1" innerhalb von 5 Sekunden auf 0 aus.
