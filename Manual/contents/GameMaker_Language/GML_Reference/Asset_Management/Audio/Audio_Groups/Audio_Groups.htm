Audio-Gruppen

Ihre Audio-Assets können auf IDE gruppiert werden, so dass sie je nach Bedarf in den Speicher geladen und entladen werden können. Sie können diese Audiogruppen über das Fenster " Audiogruppen" erstellen, das Sie über das Menü Werkzeuge Menü oben auf IDE geöffnet werden kann - und Sie können Ihre Audio-Assets über das Fenster "Sound Properties" einer bestimmten Gruppe zuordnen.

Für die Verwendung von Audiogruppen gibt es die folgenden Funktionen:

  • audio_group_load
  • audio_group_unload
  • audio_group_is_loaded
  • audio_group_load_progress
  • audio_group_name
  • audio_group_stop_all
  • audio_group_set_gain
