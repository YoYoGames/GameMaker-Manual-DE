audio_group_stop_all

Diese Funktion stoppt alle Sounds der angegebenen Audiogruppe, die gerade abgespielt werden.

Syntax:

audio_group_stop_all(groupID);

Argument Typ Beschreibung
groupID Audio-Gruppen-ID The index of the audio group to stop (as defined in the Audio Groups Window)

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(vk_space)
{
    audio_group_stop_all(audiogroup_level1);
}

Die obige code prüft, ob die Leertaste gedrückt wurde und stoppt dann die gesamte Audiowiedergabe der angegebenen Gruppe.
