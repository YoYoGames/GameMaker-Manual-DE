audio_group_load_progress

Diese Funktion prüft den Ladefortschritt für eine Audiogruppe und gibt einen (ungefähren) Wert zwischen 0 und 100 zurück.

Syntax:

audio_group_load_progress(groupID);

Argument Typ Beschreibung
groupID Audio-Gruppen-ID The index of the audio group to check (as defined in the Audio Groups Window)

Rückgabe:

Real

Beispiel:

if load
{
    var pc = audio_group_load_progress(audiogroup_level1);
    draw_text(32, 32, "Loading: " + string(pc));
}

Die obige code prüft eine Variable, und wenn sie true zurückgibt, wird ein Text gezeichnet, der den Fortschritt des geladenen Audios anzeigt.
