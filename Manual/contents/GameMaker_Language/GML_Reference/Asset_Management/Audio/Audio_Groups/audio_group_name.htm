audio_group_name

Diese Funktion gibt eine Zeichenkette zurück, die den Namen der angegebenen Audiogruppe enthält und die angezeigt oder überprüft werden kann. Wenn Sie eine Audiogruppe in den Spieloptionen definieren, geben Sie ihr einen eindeutigen "Namen", der eigentlich eine Konstante ist, die als ID-Wertfür die Gruppe verwendet wird. Alles, was diese Funktion tut, ist, die ID zu nehmen und eine Zeichenkette des ID-Namens zurückzugeben, den du gegeben hast.

Syntax:

audio_group_name(groupID);

Argument Typ Beschreibung
groupID Audio-Gruppen-ID The index value constant of the audio group to check (as defined in the Audio Groups Window)

Rückgabe:

String

Beispiel:

var name = audio_group_name(audiogroup_level1);
draw_text(32, 32, "Now Playing Group: " + name);

Die obige code ruft den Namen der angegebenen Audiogruppenkonstante ab und zeigt ihn auf dem Bildschirm an.
