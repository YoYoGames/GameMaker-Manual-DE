audio_stop_all

Diese Funktion stoppt alle Töne, die gerade abgespielt werden.

Syntax:

audio_stop_all();

Rückgabe:

Beispiel:

if !global.SFX
{
    audio_stop_all();
}

Die obige code prüft die globale Variable "SFX" und wenn sie false zurückgibt, stoppt sie alle Sounds, die gerade abgespielt werden.
