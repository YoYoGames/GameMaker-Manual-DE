audio_klang_ist_abspielbar

Diese Funktion kann verwendet werden, um zu prüfen, ob der angegebene Tonindex derzeit abgespielt werden kann. Dies ist aufgrund der unterschiedlichen Art und Weise, wie gestreamte und nicht gestreamte Tonwiedergabe auf dem HTML5-Ziel platform gehandhabt wird, erforderlich und gibt true zurück, wenn der Ton abgespielt werden kann, und false, wenn dies nicht der Fall ist. Beachten Sie, dass die Funktion bei allen anderen platforms außer HTML5 immer true zurückgibt.

Syntax:

audio_sound_is_playable(index);

Argument Typ Beschreibung
index Sound Asset The index of the sound to check.

Rückgabe:

Boolean

Beispiel:

if audio_sound_is_playable(snd_Shot)
{
    audio_play_sound(snd_Shot, 10, false);
}

Die obige code spielt den Ton "snd_Shot" ab, wenn er abspielbar ist.
