audio_sound_get_track_position

Diese Funktion ermittelt die Position (in Sekunden) innerhalb der Sounddatei, ab der der Sound abgespielt werden soll. Der Sound kann nur eine einzelne Instanz eines Sounds sein (der Index für einzelne Sounds, die abgespielt werden, kann in einer Variablen gespeichert werden, wenn die Funktion audio_play_sound() oder audio_play_sound_at() Funktionen).

Syntax:

audio_sound_get_track_position(index);

Argument Typ Beschreibung
index Sound-Instanz-ID The index of the sound to get the play position of.

Rückgabe:

Real

Beispiel:

if audio_sound_get_track_position(global.Music) != 0
{
    audio_sound_set_track_position(global.Music, 0);
}

Die obige code überprüft die Startposition eines Tracks und setzt sie auf 0 Sekunden, wenn sie nicht 0 Sekunden beträgt.
