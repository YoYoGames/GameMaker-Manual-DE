audio_emitter_exists

Diese Funktion gibt zurück, ob ein Audioemitter existiert (true) oder nicht (false). Beachten Sie, dass diese Funktion einen Fehler verursacht, wenn der Index, nach dem Sie suchen, noch nicht initialisiert wurde, da sie nach nicht existierenden Indizes sucht.

Syntax:

audio_emitter_exists(index);

Argument Typ Beschreibung
index Audio-Sender-ID The index of the emitter to check the existence of.

Rückgabe:

Boolean

Beispiel:

if audio_emitter_exists(s_emit)
{
    audio_play_sound_on(s_emit, snd_Explode, false, 1);
}
else
{
    s_emit = audio_emitter_create();
    audio_play_sound_on(s_emit, snd_Explode, false, 1);
}

Die obige code prüft, ob ein Emitter existiert, der in der (zuvor initialisierten) Variablen "s_emit" indiziert ist. Ist dies der Fall, wird ein Ton über ihn abgespielt. Ist dies nicht der Fall, wird er erstellt und dann der Ton abgespielt.
