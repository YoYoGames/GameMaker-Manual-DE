audio_emitter_get_y

Diese Funktion gibt die aktuelle y-Position des angegebenen Audio-Emitters zurück.

Syntax:

audio_emitter_get_y(emitter);

Argument Typ Beschreibung
emitter Audio-Sender-ID The index of the emitter to use.

Rückgabe:

Real

Beispiel:

if audio_emitter_get_y(emitter_player) != x
{
    audio_emitter_position(emitter_player, x, y, 0);
}

Die obige code prüft die aktuelle y-Position eines bestimmten Emitters, und wenn sie nicht gleich der y-Position der Instanz ist, wird sie auf die Instanzposition gesetzt.
