audio_emitter_free

Mit dieser Funktion kannst du den angegebenen Emitter aus dem Speicher entfernen. Dies sollte immer dann geschehen, wenn der Emitter nicht weiter im Raum oder im Spiel verwendet werden soll, d.h. im Destroy Event der Instanz oder im Room End Event, sonst kann es zu einem Speicherleck kommen, das das Spiel verlangsamt und schließlich zum Absturz bringt.

Syntax:

audio_emitter_free(emitter);

Argument Typ Beschreibung
emitter Audio-Sender-ID The index of the emitter to free.

Rückgabe:

N/A

Beispiel:

if lives = 0
{
    audio_emitter_free(s_emit);
    room_goto(rm_Menu);
}

Die obige code prüft den Wert der globalen Variable "lives" und wenn sie 0 zurückgibt, zerstört sie den Emitter, der in der Variable "s_emit" indiziert ist, und geht dann in den Raum, der in der Variable "rm_Menu" indiziert ist.
