audio_emitter_get_listener_mask

Diese Funktion gibt die Bit-Masken-Daten zurück, die definieren, von welchen Audio-Listenern ein Emitter Töne abspielen soll. Weitere Informationen finden Sie im Abschnitt über Audio-Listener.

Syntax:

audio_emitter_get_listener_mask(emitterID);

Argument Typ Beschreibung
emitterID Audio-Sender-ID The unique ID of the emitter to get the mask of

Rückgabe:

Real

Beispiel:

s_emit = audio_emitter_create();
if (audio_emitter_get_listener_mask(s_emit) != global.PlayerMask)
{
    audio_emitter_set_listener_mask(snd, global.PlayerMask);
}

Die obige code erstellt einen Emitter und prüft dann die Masken-Daten für den Hörer. Wenn diese nicht mit denen übereinstimmen, die in einer globalen Variable gespeichert sind, werden die Hörer so eingestellt, dass sie mit den in der globalen Variable gespeicherten Masken-Daten abgespielt werden.
