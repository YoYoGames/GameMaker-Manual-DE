audio_emitter_position

Mit dieser Funktion können Sie die Position eines Audio-Emitters innerhalb des 3D-Autoraums ändern. Die Position wirkt sich unterschiedlich auf den Klang aus, je nachdem, wo sich der Zuhörer im Audio-Raum befindet (Standardposition ist (0, 0, 0)). Siehe audio_listener_position() für weitere Informationen). Wenn beispielsweise die Position des Senders auf (100, 0, 0) eingestellt ist und der aktuelle Zuhörer sich auf (200, 0, 0) befindet, erscheint der vom Sender gestreamte Ton auf der linken Seite des Audiofeldes. Die folgende Abbildung zeigt eine visuelle Darstellung der Emitter und ihrer relativen Positionen zum Hörer:

Syntax:

audio_emitter_position(emitter, x, y, z);

Argument Typ Beschreibung
emitter Audio-Sender-ID The index of the emitter to change.
x Real The x position.
y Real The y position.
z Real The z position.

Rückgabe:

N/A

Beispiel:

if speed > 0
{
    audio_emitter_position(s_emit, x, y, 0);
}

Die obige code prüft, ob die Geschwindigkeit der Instanz über 0 liegt, und wenn dies der Fall ist, wird der in der Variable "s_emit" indizierte Audioemitter auf die aktuelle x/y-Position aktualisiert.
