audio_emitter_falloff

Mit dieser Funktion können Sie die Fall-Off-Distanz für einen Emitter einstellen. Dies ist die Entfernung, die der Emitter vom Hörer entfernt sein muss, bevor der Ton auf eine Lautstärke von 0 "abfällt". Der Standardwert für diesen Wert ist 100, aber Vorsicht, wenn Sie diesen Wert auf einen niedrigeren Wert als 1 setzen, da dies jeden Ton, der durch diesen Emitter gespielt wird, für den Hörer unhörbar macht.

Wie der Klang selbst gehört wird, hängt von der Abschwächungsreferenz (das ist die Entfernung, unter der die Lautstärke der Quelle normalerweise um die Hälfte abfällt) und dem Abschwächungsfaktor (der den Klang nur über die Abschwächungsreferenzentfernung hinaus beeinflusst) ab. Der Standardfaktor ist normalerweise 1, und die Wirkung der verschiedenen Abfallwerte hängt vom gewählten Modell ab.

Eine vollständige Anleitung zu den verschiedenen Falloff-Modellen und zur Verwendung dieser Werte finden Sie in der Funktion audio_falloff_set_model().

Syntax:

audio_emitter_falloff(emitter, falloff_ref, falloff_max, falloff_factor);

Argument Typ Beschreibung
emitter Audio-Sender-ID The index of the emitter to change.
falloff_ref Real The falloff reference relative to the listener (clamp).
falloff_max Real The maximum falloff distance relative to the listener.
falloff_factor Real The falloff factor (default 1).

Rückgabe:

N/A

Beispiel:

s_emit = audio_emitter_create();
audio_emitter_position(s_emit, x, y, 0);
audio_emitter_falloff(s_emit, 100, 300, 1);

Die obige code erstellt einen Audioemitter und weist seinen Index der Variablen "s_emit" zu. Dieser Emitter wird dann an der Position der Instanz, die ihn erstellt, platziert und erhält eine Falloff-Distanz von 300, was bedeutet, dass der Emitter, wenn er weiter als 300 Pixel von der Position des Zuhörers entfernt ist, stumm wird. Sein Falloff-Faktor ist der Standardwert 1 und die Falloff-Referenz ist 100, so dass der Falloff bei 100 Pixeln vom Hörer beginnt.
