audio_emitter_gain

Mit dieser Funktion wird die maximale Verstärkung (Lautstärke) für den Ton eingestellt. Die wahrgenommene Lautstärke eines Geräuschs kann sich je nach Abfallwert und Position zum Hörer ändern, aber durch die Einstellung der Verstärkung mit dieser Funktion wird die volle Lautstärke niemals den angegebenen Verstärkungswert überschreiten. Die folgende Abbildung zeigt, wie sich die Verstärkung auf die Lautstärke des Emitters auswirkt, wenn der Abfallwert größer als 0 ist:

Diese Funktion ändert die Lautstärke des Tons, während er abgespielt wird, sowie aller nachfolgenden Töne, die über den angegebenen Emitter abgespielt werden. Beachten Sie, dass Sie auf einigen platforms eine Verstärkung von mehr als 1 haben können, obwohl ein Wert von 1 als "volle Lautstärke" angesehen wird und alles, was darüber liegt, zu Audio Clipping oder Verzerrungen führen kann.

HINWEIS Die Endlautstärke wird auch von der globalen Audioverstärkung beeinflusst, die mit der Funktion audio_master_gain().

Syntax:

audio_emitter_gain(emitter, gain);

Argument Typ Beschreibung
emitter Audio-Sender-ID The index of the emitter to change.
gain Real The maximum gain (default 1).

Rückgabe:

N/A

Beispiel:

if up
{
    gain += 0.05;
    if gain >= 1 up = false;
}
else
{
    gain += 0.05;
    if gain <= 0 up = true;
}

audio_emitter_gain(s_emit, gain);

Die obige code setzt die Variable "gain" auf verschiedene Werte und verwendet dann dieselbe Variable, um die Verstärkung des in der Variable "s_emit" indizierten Emitters einzustellen.
