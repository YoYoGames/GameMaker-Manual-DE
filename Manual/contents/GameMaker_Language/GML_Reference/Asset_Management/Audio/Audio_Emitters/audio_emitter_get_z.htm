audio_emitter_get_z

Diese Funktion gibt die aktuelle z-Position des angegebenen Audio-Emitters zurück.

Syntax:

audio_emitter_get_z(emitter);

Argument Typ Beschreibung
emitter Audio-Sender-ID The index of the emitter to use.

Rückgabe:

Real

Beispiel:

if audio_emitter_get_z(emitter_player) != 0
{
    var ex = audio_emitter_get_x(emitter_player);
    var ey = audio_emitter_get_y(emitter_player);
    audio_emitter_position(emitter_player, ex, ey, 0);
}

Die obige code prüft die aktuelle z-Position eines bestimmten Emitters und wenn sie ungleich 0 ist, wird sie auf 0 gesetzt.
