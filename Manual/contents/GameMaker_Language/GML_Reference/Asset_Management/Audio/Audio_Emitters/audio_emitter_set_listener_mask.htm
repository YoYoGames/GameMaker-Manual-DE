audio_emitter_set_listener_mask

Diese Funktion kann verwendet werden, um die Bitmaske für einen Emitter zu setzen, so dass alle Klänge, die über den Emitter abgespielt werden, nur von den angegebenen Hörern abgespielt werden. Sie können eine Bit-Maske erstellen, indem Sie die audio_get_listener_info() und dann die bitweise or ("|") verwenden, um eine benutzerdefinierte Maske mit den Hörern zu erstellen, von denen der Sound abgespielt werden soll, und diese benutzerdefinierte Maske dann auf den Emitter anwenden. Diese Maske überschreibt die Standardmaske oder die Maske, die Sie mit der Funktion audio_set_listener_mask().

Syntax:

audio_emitter_set_listener_mask(emitterID, mask);

Argument Typ Beschreibung
emitterID Audio-Sender-ID The unique ID of the emitter to set the mask of
mask Real The bitmask data to set for the sound

Rückgabe:

N/A

Beispiel:

s_emit = audio_emitter_create();
audio_emitter_set_listener_mask(snd, global.PlayerMask);

Die obige code erstellt einen Audio-Emitter und legt dann die Hörer fest, die mit Hilfe der in einer globalen Variable gespeicherten Maskendaten abgespielt werden sollen.
