audio_emitter_erstellen

Diese Funktion erstellt einen neuen Audioemitter und gibt den Index für ihn zurück. Dieser Index sollte für alle weiteren Funktionen, die sich auf diesen Emitter beziehen, in einer Variablen gespeichert werden, und wenn er nicht mehr benötigt wird, sollte er mit der Funktion audio_emitter_free() aus dem Speicher entfernt werden, um Speicherlecks zu vermeiden, die zu einem Absturz des Spiels führen können.

Der Audioemitter wird mit den folgenden Standardeinstellungen erstellt:

  • Eine Verstärkung und Tonhöhe von 1
  • Eine Listener-Maske, die nur auf den Standard-Listener (1) eingestellt ist
  • Eine Position von (0, 0, 0)
  • Eine Geschwindigkeit von 0 (vx = 0, vy = 0, vz = 0)

Syntax:

audio_emitter_create();

Rückgabe:

Audio Emitter ID

Beispiel:

s_emit = audio_emitter_create();

Die obige code erstellt einen neuen Audioemitter und weist seinen Index der Variablen "s_emit" zu.
