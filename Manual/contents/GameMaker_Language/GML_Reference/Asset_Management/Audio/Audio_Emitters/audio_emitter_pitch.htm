audio_emitter_pitch

Diese Funktion kann verwendet werden, um die Tonhöhe aller Klänge zu ändern, die von dem angegebenen Emitter abgegeben werden. Es handelt sich um einen Tonhöhenmultiplikator, d.h. der Eingabewert multipliziert die aktuelle Tonhöhe mit dem entsprechenden Wert. Der Standardwert 1 bedeutet also keine Tonhöhenänderung, während ein Wert unter 1 die Tonhöhe senkt und ein Wert über 1 die Tonhöhe erhöht. Es ist am besten, für diese Funktion kleine Schritte zu verwenden, da Werte unter 0 oder über 5 möglicherweise ohnehin nicht hörbar sind.

Töne, die bereits auf dem Audioemitter abgespielt werden, müssen nicht neu gestartet werden, damit die Änderung der Tonhöhe hörbar wird. Die Änderung der Tonhöhe wird sofort übernommen.

Syntax:

audio_emitter_pitch(emitter, pitch);

Argument Typ Beschreibung
emitter Audio-Sender-ID The index of the emitter to change.
pitch Real The pitch multiplier (default 1).

Rückgabe:

N/A

Beispiel:

switch (gear)
{
    case 1: audio_emitter_pitch(s_emit, 0.8); break;
    case 2: audio_emitter_pitch(s_emit, 0.9); break;
    case 3: audio_emitter_pitch(s_emit, 0.95); break;
    case 4: audio_emitter_pitch(s_emit, 1); break;
    case 5: audio_emitter_pitch(s_emit, 1.2); break;
}

Die obige code ändert die Tonhöhe des von dem in der Variable "s_emit" indizierten Emitter abgespielten Tons in Abhängigkeit vom Wert der Variable "gear".
