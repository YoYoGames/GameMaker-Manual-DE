audio_is_playing

Diese Funktion prüft, ob der angegebene Sound gerade abgespielt wird. Der Sound kann entweder eine einzelne Instanz eines Sounds sein (der Index für einzelne Sounds, die abgespielt werden, kann in einer Variablen gespeichert werden, wenn die Funktion audio_play_sound() oder audio_play_sound_at() Funktionen) oder ein Sound-Asset sein. In diesem Fall werden alle Instanzen des angegebenen Sounds überprüft, und wenn einer von ihnen abgespielt wird, gibt die Funktion true zurück, andernfalls false. Beachten Sie, dass diese Funktion auch dann true zurückgibt, wenn der zu prüfende Sound zuvor mit der audio_pause_sound() Funktion angehalten wurde.

Syntax:

audio_is_playing(index);

Argument Typ Beschreibung
index Sound Asset oder Sound Instanz ID The index of the sound to check.

Rückgabe:

Boolean

Beispiel:

if !audio_is_playing(snd_Waterfall)
{
    audio_play_sound_at(snd_Waterfall, x, y, 0, 300, true, 1);
}

Die obige code prüft, ob der in der Variablen "snd_Waterfall" indizierte Sound gerade abgespielt wird, und wenn sie false zurückgibt, dann wird der Sound an seiner Raumposition mit einer Fall-off-Distanz von 300 und einer niedrigen Priorität geloopt.
