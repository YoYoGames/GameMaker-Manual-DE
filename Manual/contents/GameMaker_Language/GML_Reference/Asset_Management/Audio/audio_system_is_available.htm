audio_system_ist_verfügbar

Diese Funktion kann verwendet werden, um zu prüfen, ob das Audiosystem initialisiert wurde oder ob der Audiokontext läuft. Auf allen platforms gibt diese Funktion unmittelbar nach dem Spielstart true zurück, wenn die Audio-Engine initialisiert ist, außer auf dem HTML5-Ziel. Bei HTML5 kann sich der Status des Audiokontextes jederzeit ändern, abhängig von den Benutzereingaben, dem verwendeten Browser und dessen Version, so dass diese Funktion verwendet werden kann, um zu überprüfen, ob Audio abgespielt werden kann oder nicht. Wenn die Funktion false zurückgibt (d.h. der Audiokontextstatus läuft nicht), können nur nicht gestreamte Töne abgespielt werden (dies ist jedoch nicht garantiert, und Sie sollten davon ausgehen, dass keine Töne abgespielt werden können), während, wenn sie true zurückgibt, alle Töne abgespielt werden.

Syntax:

audio_system_is_available();

Rückgabe:

Beispiel:

if audio_system_is_available()
{
    if audio_is_paused(global.Music)
    {
        audio_resume_sound(global.Music)
    }
    else
    {
        if !audio_is_playing(global.Music)
        {
            global.Music = audio_play_sound(snd_Music, 0, true);
        }
    }
}
else
{
    if audio_is_playing(global.Music)
    {
        audio_pause_sound(global.Music);
    }
}

Die obige Seite code unterbricht einen Audiotitel, je nachdem, ob das Audiosystem initialisiert und verfügbar ist oder nicht.
