audio_sound_set_track_position

Diese Funktion setzt die Position (in Sekunden) für die angegebene Sound-ID oder das Asset.

Der gelieferte "Index" kann entweder eine einzelne Instanz eines Sounds sein (wie sie von audio_play_sound()) oder ein Sound-Asset (z.B. eines, das über den Asset-Browser hinzugefügt wurde). Das Verhalten dieser Funktion hängt davon ab, welche Art von Index Sie angegeben haben:

  • Wenn es sich um eine eindeutige Sound-ID handelt, die Sie verwenden und die bereits abgespielt wird, ändert sich ihre Position sofort zur angegebenen Zeit.
  • Wenn es sich um ein Tondokument handelt, beginnen alle weiteren Abspiele des betreffenden Tondokuments zur neuen Zeit.

Diese Funktion ändert nur die Spurposition für den aktuell abgespielten Sound oder dessen nächste Wiedergabe (falls ein Sound-Asset geliefert wird). Wenn der Ton mit aktivierter Schleifenfunktion abgespielt wird, beginnen die nachfolgenden Wiedergaben immer am Anfang (0,0 Sekunden) und nicht an der mit dieser Funktion festgelegten Spurposition. Wenn Sie z. B. einen Sound loop bei 5,7 Sekunden starten, wird der erste Sound ab diesem Punkt abgespielt, danach wird jedoch der gesamte Track von Anfang bis Ende wiederholt.

Syntax:

audio_sound_set_track_position(index, time);

Argument Typ Beschreibung
index Sound Asset oder Sound Instanz ID The index of the sound to change.
time Real The time (in seconds) to set the start point to. Values longer than the length of the given sound are ignored.

Rückgabe:

N/A

Beispiel:

var snd = audio_play_sound(snd_MainTrack, 0, false);
audio_sound_set_track_position(snd, 32);

Die obige code spielt einen Ton ab und verwendet dann den zurückgegebenen ID-Wert, um die Startposition für den Ton auf 32 Sekunden zu setzen.
