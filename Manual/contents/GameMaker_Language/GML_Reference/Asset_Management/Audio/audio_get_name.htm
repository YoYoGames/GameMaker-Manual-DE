audio_get_name

Diese Funktion gibt den Namen einer bestimmten Audioressource als String zurück. Der "Index"-Wert kann der der Ressource selbst sein (wie im Asset-Browser zu sehen) oder der eindeutige ID-Wert, der z. B. beim Abspielen des Sounds angegeben wird, audio_play_sound(). Beachten Sie, dass die zurückgegebene Zeichenkette nicht mit der ID der Ressource übereinstimmt und nicht für den Zugriff auf die Ressource selbst verwendet werden kann; sie sollte daher nur zur Anzeige oder zur Fehlerprüfung verwendet werden.

Syntax:

audio_get_name(index);

ArgumentTyp Beschreibung
index The index of the sound to check.

Rückgabe:

Beispiel:

var snd = audio_play_sound(choose(snd_One, snd_Two, snd_Three), 0, false);
var name = audio_get_name(snd);
show_debug_message("Sound = " + name);

Die obige code spielt einen zufälligen Sound aus drei verschiedenen Sound-Ressourcen ab und zeigt dann eine Debug-Meldung mit seinem Namen an.
