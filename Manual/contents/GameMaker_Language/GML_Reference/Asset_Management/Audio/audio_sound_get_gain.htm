audio_sound_get_gain

Diese Funktion gibt den aktuellen Verstärkungswert für den angegebenen Sound zurück. Der Sound kann entweder aus einem Index für einen individuellen Sound referenziert werden, der in einer Variablen gespeichert wurde, wenn die Funktion audio_play_sound() oder audio_play_sound_at() Funktionen gespeichert wurde, oder ein tatsächliches Sound-Asset aus dem Asset-Browser. Die Verstärkung wird in der Regel als ein Wert zwischen 0 und 1 berechnet, aber auf einigen platforms können Sie eine Verstärkung von mehr als 1 haben, obwohl ein Wert von 1 als "volle Lautstärke" angesehen wird und alles, was darüber liegt, zu Audio Clipping führen kann.

Syntax:

audio_sound_get_gain(index);

Argument Typ Beschreibung
index Sound Asset oder Sound Instanz ID The index of the sound to get the gain of.

Rückgabe:

Real

Beispiel:

if audio_sound_get_gain(snd_Music) != 1
{
    audio_sound_gain(snd_Music, 1, 0);
}

Die obige code ändert die Verstärkung des Tons, der von dem als "snd_Music" indizierten Ton abgespielt wird, wenn seine Verstärkung nicht gleich 1 ist.
