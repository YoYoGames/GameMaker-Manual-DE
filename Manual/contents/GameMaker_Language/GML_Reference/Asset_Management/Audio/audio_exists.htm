audio_exists

Diese Funktion gibt zurück, ob ein Ton vorhanden ist (true) oder nicht (false). Beachten Sie, dass diese Funktion einen Fehler verursacht, wenn der Index, nach dem Sie suchen, zuvor nicht initialisiert wurde, da sie nach nicht existierenden Asset-Indizes sucht. Der zu prüfende Sound kann entweder eine einzelne Instanz eines Sounds sein (der Index für einzelne abgespielte Sounds kann in einer Variablen gespeichert werden, wenn die Funktion audio_play_sound() oder audio_play_sound_at() Funktionen) oder ein Sound-Asset sein.

Syntax:

audio_exists(index);

Argument Typ Beschreibung
index Vermögen The index of the sound to check the existence of.

Rückgabe:

Boolean

Beispiel:

if audio_exists(global.Music)
{
    audio_play_sound(global.Music, 0, true);
}

Die obige code prüft, ob ein Ton vorhanden ist. Wenn ja, wird er in einem loop abgespielt.
