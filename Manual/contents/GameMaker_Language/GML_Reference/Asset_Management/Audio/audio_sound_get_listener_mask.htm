audio_sound_get_listener_mask

Diese Funktion gibt die Bit-Masken-Daten zurück, die definieren, von welchen Audio-Hörern ein Ton abgespielt werden soll. Weitere Informationen finden Sie im Abschnitt über Audiohörer.

Syntax:

audio_sound_get_listener_mask(soundID);

Argument Typ Beschreibung
soundID Sound-Instanz-ID The unique ID of the sound to get the mask of

Rückgabe:

Real

Beispiel:

var snd = audio_play_sound(snd_PlayerDead, 10, false);
if audio_sound_get_listener_mask(snd) != global.PlayerMask
{
    audio_sound_set_listener_mask(snd, global.PlayerMask);
}

Die obige code spielt einen Ton ab und prüft dann die Hörermaskendaten für den Ton, und wenn diese nicht mit den in einer globalen Variable gespeicherten Daten übereinstimmen, werden die Hörer zum Abspielen anhand der in der globalen Variable gespeicherten Maskendaten eingestellt.
