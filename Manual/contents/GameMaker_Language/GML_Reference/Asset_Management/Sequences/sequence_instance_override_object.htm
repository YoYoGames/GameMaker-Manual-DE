sequenz_instanz_ueberschreibendes_Objekt

Mit dieser Funktion können Sie alle Instanzen eines Objekts, das in einem sequence verwendet wird, durch ein anderes Objekt ersetzen (überschreiben). Sie geben die sequence Instanz-Struktur-ID (wie sie zurückgegeben wurde, als die sequence Instanz im Raum erstellt wurde, oder indem Sie eine der Raumschichtfunktionen verwenden - siehe hier), sowie den Objektindex (wie im Asset-Browser definiert) für das Objekt, das Sie überschreiben möchten. Schließlich geben Sie einen Objektindex oder eine Instanz-ID für das Objekt an, das die sequence überschreiben soll (bei Angabe einer Instanz-ID wird einfach das Objekt, von dem die Instanz erstellt wurde, als Überschreibungsobjekt verwendet). Beachten Sie, dass dies nur bei sequence Instanzen möglich ist (nicht bei sequence Objekten) und dass dies geschehen muss, bevor die Wiedergabe von sequence beginnt, sonst wird es nicht funktionieren.

Syntax:

sequence_instance_override_object(sequence_instance_struct, object_id, instance_or_object_id);

Argument Typ Beschreibung
sequence_instance_struct Sequenz-Instanz-Struktur The sequence instance struct to modify.
object_id Objekt Anlage The object index of the object within the sequence to override.
instance_or_object_id Objekt Asset- oder Instanz-ID The object index or instance ID to use to override the sequence objects.

Rückgabe:

N/A

Beispiel:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _seq_inst = layer_sequence_get_instance(_seq);
sequence_instance_override_object(_seq_inst, obj_Trees_Winter, obj_Trees_Summer);

Die obige code erstellt eine neue sequence Instanz auf der angegebenen Ebene und ändert sie dann so, dass alle Instanzen des Objekts "obj_Trees_Winter" durch Instanzen des Objekts "obj_Trees_Summer" ersetzt werden.
