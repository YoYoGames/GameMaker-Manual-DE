sequenz_instanz

Dies ist eine eingebaute Variable, die Teil der Instanzvariablen ist, die für jede Objektinstanz in Ihrem Spiel erstellt werden. Wenn die Instanz von einer sequence kontrolliert wird, enthält diese Variable die Sequenzinstanzstruktur für die Sequence, die die Instanz kontrolliert, andernfalls ist es undefined. Diese Variable ist schreibgeschützt und kann nicht geändert werden.

Beachten Sie, dass diese Variable zu undefined wird, nachdem das kontrollierende Sequence beendet wurde, selbst wenn das Sequence Element noch existiert, und dass sie erneut die Struktur erhält, wenn das Sequence Element erneut abgespielt wird.

Syntax:

sequence_instance

Rückgabe:

Sequence Instance Struct

Beispiel:

if (in_sequence)
{
    sequence_instance.speedScale = 2;
}

Die obige code prüft die in_sequence und wenn sie true ist (was bedeutet, dass die Instanz von einer sequence gesteuert wird), dann wird die Geschwindigkeitsskala dieser Sequence auf 2 geändert.
