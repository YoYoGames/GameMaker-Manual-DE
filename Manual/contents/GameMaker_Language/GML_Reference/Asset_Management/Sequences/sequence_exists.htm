sequenz_existiert

Mit dieser Funktion können Sie prüfen, ob ein sequence Objekt existiert oder nicht. Sie liefern entweder die sequence Objektstruktur (wie von der Funktion sequence_create() oder sequence_get()) oder die sequence ID (zurückgegeben von der Funktion layer_sequence_get_sequence() oder von der sequence Instanzstruktureigenschaft sequence, oder den Index aus dem Asset-Browser) und die Funktion gibt true zurück, wenn das sequence Objekt existiert, oder false, wenn es nicht existiert.

Syntax:

sequence_exists(sequence_struct_or_id);

Argument Typ Beschreibung
sequence_struct_or_id Sequenz Asset oder Sequenz Objekt Struktur The sequence to check for, can be the asset reference itself or its Sequence object struct.

Rückgabe:

Boolean

Beispiel:

if sequence_exists(my_seq)
{
    sequence_destroy(my_seq);
}

Die obige code prüft, ob das angegebene sequence Objekt existiert, und wenn ja, wird es zerstört.
