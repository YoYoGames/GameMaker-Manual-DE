sequenz_get_objects

Mit dieser Funktion können Sie ein Array mit allen Objektindizes abrufen, deren Instanzen innerhalb der angegebenen sequence erstellt wurden. Sie liefern entweder die sequence Objektstruktur (wie von der Funktion sequence_create() oder sequence_get()) oder die sequence ID (zurückgegeben von der Funktion layer_sequence_get_sequence() oder von der sequence Instanzstruktureigenschaft sequence) und die Funktion wird ein Array zurückgeben, wobei jedes Element im Array ein object_index für die verschiedenen Objekte ist, die von sequence verwendet werden, um Instanzen zu erstellen.

Syntax:

sequence_get_objects(sequence_struct_or_id);

Argument Typ Beschreibung
sequence_struct_or_id Sequenz Asset oder Sequenz Objekt Struktur The sequence object struct or ID to get the objects from.

Rückgabe:

Array

Beispiel:

var _seq = sequence_get(seq_Logo);
obj_array = sequence_get_objects(_seq);

Die obige code ruft die Struktur für ein sequence Objekt ab, ruft dann die verwendeten Objekte ab und speichert das Array in einer Variablen.
