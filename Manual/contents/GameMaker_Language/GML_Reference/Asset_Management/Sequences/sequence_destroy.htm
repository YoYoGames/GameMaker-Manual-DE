sequenz_zerstören

Mit dieser Funktion können Sie ein sequence Objekt zerstören, das dynamisch erstellt wurde. Sie geben entweder die sequence Objektstruktur (wie von der Funktion sequence_create()) oder die sequence ID (zurückgegeben von der Funktion layer_sequence_get_sequence() oder aus der sequence Instanzstruktureigenschaft sequence). Diese Funktion sollte immer dann verwendet werden, wenn ein dynamisch erstelltes sequence nicht mehr benötigt wird, um den damit verbundenen Speicher freizugeben.

Syntax:

sequence_destroy(sequence_struct_or_id);

Argument Typ Beschreibung
sequence_struct_or_id Sequenz Asset oder Sequenz Objekt Struktur The sequence object struct or ID to destroy

Rückgabe:

N/A

Beispiel:

if sequence_exists(my_seq)
{
    sequence_destroy(my_seq);
}

Die obige code prüft, ob das angegebene sequence Objekt existiert, und wenn ja, wird es zerstört.
