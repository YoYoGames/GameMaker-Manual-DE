sequence_create

Mit dieser Funktion können Sie ein neues sequence Objekt erstellen, dem Sie dann Spuren hinzufügen können. Die Funktion gibt eine Struktur zurück, auf die Sie dann zugreifen können, um die neue sequence, die Sie erstellt haben, einzurichten. Der Inhalt dieser Struktur wird auf dieser Seite beschrieben. Die sequence Objektstruktur kann dann verwendet werden, um Instanzen von sequence auf einer Raumebene mit der Funktion layer_sequence_create(). Beachten Sie, dass Sie beim Anlegen von sequence Objekten auf diese Weise diese wieder entfernen sollten, indem Sie die Funktion sequence_destroy() wieder entfernen, wenn sie nicht mehr benötigt werden, da es sonst zu einem Speicherleck kommt, das das Spiel verlangsamen und schließlich zum Absturz bringen kann.

Syntax:

sequence_create();

Rückgabe:

Sequence Object Struct

Beispiel:

myseq = sequence_create();
myseq.length = 120;
myseq.loopmode = seqplay_pingpong;

Die obige code erstellt eine neue sequence Objektstruktur und legt deren Wiedergabelänge und loop Modus fest.
