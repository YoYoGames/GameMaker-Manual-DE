Sequenz-Objekt-Struktur

A Sequenz-Objekt-Struktur ist der Name für die Struktur, die von einem Sequence Asset abgerufen wird. Alle Daten für ein Sequence sind in dieser Struktur unter runtime gespeichert und werden von allen Elementen der Struktur Sequence referenziert, die in einem Raum erstellt werden. Das bedeutet, dass, wenn irgendwelche Variablen in dieser Struktur (oder in einer ihrer verschachtelten Strukturen) geändert werden, sich die ursprüngliche Sequence ändern wird (bis das Spiel geschlossen und wieder geöffnet wird) und alle vorhandenen Sequence Elemente davon betroffen sind.

Diese Struktur hat die folgenden Eigenschaften, die geändert werden können:

Sequenz-Objekt-Struktur
Variabel Typ Beschreibung
name Zeichenfolge Dies ist der Name der sequence in Form einer Zeichenkette, und Sie können diesen Wert nach Bedarf abrufen oder festlegen. Beachten Sie, dass sequences, das mit der Funktion sequence_create() erstellt wird, keinen Namen hat und dies einfach eine leere Zeichenkette "" ist.
loopmode Sequenz Wiedergabemodus Konstant Dies ist der Wiedergabemodus des sequence Objekts und kann ausgelesen oder gesetzt werden. Dies kann eine der Konstanten sein, die in der Tabelle unter dieser Konstante aufgeführt sind.
playbackSpeed Real Hier wird die Wiedergabegeschwindigkeit von sequence angegeben, die je nach playbackSpeedType (siehe unten) entweder als Bilder pro Sekunde oder als Bilder pro Spiel-Frame interpretiert wird. Sie können diesen Wert abrufen oder setzen.
playbackSpeedType Sprite-Geschwindigkeitskonstante Dies gibt an, wie playbackSpeed interpretiert werden soll, und Sie können diesen Wert abrufen oder festlegen.
length Real Die Länge des sequence in Frames. Sie können diesen Wert abrufen oder festlegen, aber beachten Sie, dass eine Verkürzung von sequence zu Problemen führen kann, wenn eine Instanz von sequence, die auf dieses sequence verweist, ihren Abspielkopf auf eine Länge nach der neuen Länge eingestellt hat.
volume Real Dies ist ein skalarer Wert von 0 bis 1, der verwendet wird, um die Lautstärke aller Audiospuren in sequence zu skalieren. Sie können diesen Wert abrufen oder einstellen und er wird die globale Audioausgabe für alle Spuren ändern - wenn Sie beispielsweise eine Audiospur mit einer Lautstärke von 0,8 haben und dann die Eigenschaft sequence volume auf 0,5 einstellen, wird die Audiospur eine Endlautstärke von 0,4 haben.
xorigin Real Dies ist der Ursprung von sequence entlang der X-Achse.
yorigin Real Dies ist der Ursprung von sequence entlang der Y-Achse.
messageEventKeyframes Array von Sequenz Keyframe Strukturs Diese Eigenschaft ermöglicht den Zugriff auf die Keyframes der Nachrichtenereignisse für sequence. Sie können diese Nachrichtenereignisse abrufen oder setzen, und wenn Sie diese Eigenschaft abrufen, wird ein Array von Keyframe-Strukturen zurückgegeben, und zum Setzen der Eigenschaft sollten Sie ein Array von Keyframe-Strukturen angeben. Weitere Informationen finden Sie auf der Seite über Sequenzereignisse und Momente.
momentKeyframes Array von Sequenz Keyframe Strukturs Diese Eigenschaft ermöglicht den Zugriff auf die Keyframes der Moment-Ereignisse für sequence. Sie können diese Moment-Ereignisse abrufen oder setzen, und wenn Sie diese Eigenschaft abrufen, wird ein Array von Keyframe-Strukturen zurückgegeben, und zum Setzen der Eigenschaft sollten Sie ein Array von Keyframe-Strukturen angeben. Weitere Informationen finden Sie auf der Seite über Sequenzereignisse und Momente.
tracks Array von Sequenz Spur Strukturs Ermöglicht den Zugriff auf die Liste der Asset-Tracks auf der obersten Ebene von sequence. Sie können diese Eigenschaft abrufen oder setzen, und wenn Sie diese Eigenschaft abrufen, wird ein Array von Track-Strukturen zurückgegeben, und zum Setzen der Eigenschaft sollten Sie ein Array von Track-Strukturen angeben. Für weitere Informationen lesen Sie bitte den Abschnitt über Track-Strukturen.

loopMode kann eine der folgenden Konstanten sein:

Sequenz Wiedergabemodus Konstant
Konstante Beschreibung Wert
seqplay_oneshot Die sequence wird einmal abgespielt und stoppt, wenn sie fertig ist. 0
seqplay_loop Die sequence wird loop, wobei der Abspielkopf an den Anfang zurückkehrt, wenn er das Ende des Abspielbereichs erreicht. 1
seqplay_pingpong Die sequence wird loop, wobei der Abspielkopf die Richtung umkehrt, wenn er das Ende der Wiedergaberegion erreicht. 2


Bemerkungen

Beachten Sie, dass Sie, wenn Sie auf die Eigenschaften eines sequence zugreifen möchten, das im Asset-Browser erstellt wurde, zunächst die Funktion sequence_get() für den Asset-Index aufrufen, um die sequence Objektstruktur abzurufen. Beachten Sie auch, dass alle Änderungen, die an dieser sequence -Struktur vorgenommen werden, dazu führen, dass alle weiteren Instanzen dieses sequence -Assets ebenfalls diese Änderungen aufweisen, und dass sie beibehalten werden, solange das Spiel läuft, auch wenn Sie die Funktion game_restart() aufrufen.
