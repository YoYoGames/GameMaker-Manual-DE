Die Keyframe-Datenstruktur

Wenn Sie eine Keyframe-Struktur für eine Spur definieren, müssen Sie auch die Daten festlegen, die mit ihr verknüpft werden. Diese setzen sich aus verschiedenen Kanälen zusammen, wobei jeder Kanal einfach eine Keyframe-Datenstruktur ist. Einem Kanal kann eine bestimmte Art von Daten zugewiesen werden, je nachdem, für welche Art von Spur Sie die Keyframe-Daten festlegen.

Eine Keyframe-Datenstruktur enthält die folgenden Variablen:

Sequenz Keyframe Datenstruktur
Variabel Typ Beschreibung
channel Real Dies ist der Kanal, auf den die Keyframe-Daten angewendet werden sollen. Es handelt sich um einen positiven ganzzahligen Wert, der bei 0 beginnt. Es ist erwähnenswert, dass Sie beim Erstellen von Parameterspuren für "Positions-" oder "Skalierungs"-Keyframes ganz bestimmte Kanalwerte verwenden müssen. Diese sind: Kanal 0 ist die X-Position oder die X-Skala, Kanal 1 ist die Y-Position oder die Y-Skala.
spriteIndex Sprite-Asset Der Index des Assets sprite, das für die Spur verwendet werden soll. Diese Eigenschaft ist nur für Tracks des Typs seqtracktype_graphic verfügbar und Sie können sie abrufen oder festlegen.
soundIndex Sound Asset Der Index des für die Spur zu verwendenden Audio-Assets. Diese Eigenschaft ist nur für Tracks des Typs seqtracktype_audio verfügbar und Sie können sie abrufen oder festlegen.
playbackMode Sequenz Audio Key Constant Der Wiedergabemodus für den Ton. Diese Eigenschaft ist nur für Tracks des Typs seqtracktype_audio verfügbar und Sie können sie abrufen oder festlegen. Die Eigenschaft sollte eine der Konstanten sein, die in der Tabelle unter dieser Eigenschaft angegeben sind.
curve Animation Kurve Struktur Diese Eigenschaft erfordert eine Animationskurvenstruktur (siehe hier für weitere Informationen) und ist nur für Spuren des Typs seqtracktype_real verfügbar. Wenn keine Kurvenstruktur verwendet wird, ist der Wert für diese Eigenschaft -1.
value Real Diese Eigenschaft ist einfach ein Wert, der mit dem Keyframe-Datenkanal verknüpft ist, und ist nur für Spuren des Typs seqtracktype_real verfügbar, wenn keine curve struct bereitgestellt wird. Dies kann z. B. die X- oder Y-Position der Spur sein, wenn sie sich innerhalb einer Spur mit dem Parameter "Position" befindet.
colour Array von Reals Diese Eigenschaft gibt ein Array für den Farbwert des Keyframes im Format [A, R, G, B] zurück (oder erfordert es, wenn es gesetzt ist). Dies ist nur für Spuren des Typs seqtracktype_colour verfügbar. Beachten Sie, dass die Werte für jede Komponente als Werte zwischen 0 und 1 ausgedrückt werden sollten, wobei 0 dem HEX-Wert #00 und 1 dem HEX-Wert #FF entspricht (0 - 255, wie in der Farbauswahl für Farbspuren im Sequences Editor angezeigt).
sequence Sequenz-Objekt-Struktur Diese Eigenschaft gibt eine Sequenz-Objektstruktur zurück (oder erfordert, wenn sie gesetzt ist) und ist nur für Spuren des Typs seqtracktype_sequence verfügbar.
objectIndex Objekt Anlage Diese Eigenschaft gibt einen Objektindex zurück (oder erfordert, wenn sie gesetzt ist) und ist nur für Spuren des Typs seqtracktype_instance verfügbar.
events Array von Zeichenfolges Diese Eigenschaft ermöglicht den Zugriff auf die Ereignisse und Broadcast-Nachrichten, die mit der Keyframe-Datenstruktur verbunden sind. Sie können diese Eigenschaft abrufen oder setzen, und wenn Sie sie abrufen, wird ein Array von Strings zurückgegeben, und wenn Sie sie setzen, sollte ein Array von Strings angegeben werden. Weitere Informationen über Ereignisse finden Sie im Abschnitt Sequenzereignisse und Momente. Diese Eigenschaft ist nur für Tracks vom Typ seqtracktype_message verfügbar.
event Methode Diese Eigenschaft liefert (oder kann auf) die method die mit der Keyframe-Datenstruktur verbunden ist. Wenn keine Methode angegeben wurde oder Sie die Methode entfernen möchten, sollte die Eigenschaft -1 sein. Diese Eigenschaft ist nur für Spuren des Typs seqtracktype_moment verfügbar.
Alle zusätzlichen Variablen, die in der/den nachstehenden Tabelle(n) aufgeführt sind

Die Variable playbackMode kann eine der folgenden Konstanten sein:

Sequenz Audio Key Constant
Konstante Beschreibung Wert
seqaudiokey_loop Der Ton wird unter loop wiedergegeben. 0
seqaudiokey_oneshot Der Ton wird nur einmal abgespielt und hört dann auf. 1

Text Spurdaten

Die Struktur enthält die folgenden zusätzlichen Variablen, wenn sie einer Textspur zugewiesen ist (seqtracktype_text):

Sequenz Keyframe Datenstruktur
Variabel Typ Beschreibung
text Zeichenfolge Dies ist die Textzeichenfolge, die auf der Spur gezeichnet wird.
wrap Boolesche Dies ist ein boolescher Wert, der angibt, ob der Text umbrochen werden soll (true) oder nicht (false).
alignmentH Sequenz Textausrichtung Konstante Dies ist die horizontale Ausrichtung des Textes und wird eine der Sequenz Textausrichtung Konstantes unten angegeben.
alignmentV Sequenz Textausrichtung Konstante Dies ist die vertikale Ausrichtung des Textes und wird eine der Sequenz Textausrichtung Konstantes unten angegeben.
fontIndex Schriftart Asset Dies ist die Schriftart Asset die von der Textspur verwendet wird.

alignmentH kann eine der folgenden Konstanten sein:

Sequenz-Text-Abgleich Konstante (alignmentH)
Konstante Beschreibung
seqtextkey_left Der Text wird am linken Rand des Rahmens ausgerichtet.
seqtextkey_right Der Text wird am rechten Rand des Rahmens ausgerichtet.
seqtextkey_center Der Text wird horizontal innerhalb des Rahmens zentriert.
seqtextkey_justify Der Text wird innerhalb des Rahmens ausgerichtet.

alignmentV kann eine der folgenden Konstanten sein:

Sequenz-Text-Abgleich Konstante (alignmentV)
Konstante Beschreibung
seqtextkey_top Der Text wird am oberen Rand des Rahmens ausgerichtet.
seqtextkey_bottom Der Text wird am unteren Rand des Rahmens ausgerichtet.
seqtextkey_middle Der Text wird vertikal innerhalb des Rahmens zentriert.
