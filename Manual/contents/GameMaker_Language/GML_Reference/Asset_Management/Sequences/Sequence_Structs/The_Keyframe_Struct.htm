Die Keyframe-Struktur

Jedem Track (wie auf der Seite über Track-Strukturen definiert) werden eine oder mehrere Keyframe-Strukturen zugewiesen (die Sie über die Eigenschaft keyframes der Track-Struktur abrufen können), und jede davon hat die folgenden Eigenschaften:

Sequenz Keyframe Struktur
Variabel Typ Beschreibung
frame Real Die Position (in Frames) entlang der Zeitachse für den Keyframe. Der Standardwert ist 0.
length Real Die Länge des Keyframes. Der Standardwert ist 1, und wenn er auf größere Werte gesetzt wird, wird die Spureigenschaft, auf die sich der Keyframe bezieht, für die Dauer der angegebenen Länge auf dem Anfangswert gehalten. Beachten Sie, dass die Eigenschaft stretch dies außer Kraft setzt, wenn sie auf true gesetzt ist.
stretch Boolesche Wenn diese Eigenschaft auf true gesetzt ist, erstreckt sich der Keyframe entweder bis zum nächsten Keyframe des Tracks oder bis zum Ende des Tracks, wenn es der letzte Keyframe ist. Sie können diesen Wert abrufen oder festlegen; der Standardwert ist false.
channels Array von Sequenz Keyframe Datenstrukturs Diese Eigenschaft ermöglicht den Zugriff auf die Liste der Keyframe-Datenstrukturen für die Kanäle des Tracks. Beim Abrufen dieser Eigenschaft wird ein Array von Keyframe-Datenstrukturen zurückgegeben, und beim Setzen dieser Eigenschaft sollte ein Array von Keyframe-Datenstrukturen angegeben werden.
