sequenz_holen

Mit dieser Funktion können Sie die Sequence Object Struct von einem sequence Asset-Index abrufen. Sie geben den Asset-Index für sequence an, wie im Asset-Browser definiert, und die Funktion gibt die sequence Objektstruktur zurück, auf die dann zugegriffen werden kann, oder -1, wenn sequence nicht existiert oder der angegebene Asset-Index kein sequence ist.

Syntax:

sequence_get(sequence_index);

Argument Typ Beschreibung
sequence_index Sequenz Asset The sequence asset as defined in the Asset Browser.

Rückgabe:

Sequence Object Struct or -1

Beispiel:

var _seq = sequence_get(seq_Logo);
seq.loopmode = seqplay_pingpong;

Die obige code ruft die sequence Objektstruktur aus dem sequence Objekt "seq_Loop" ab und setzt dann den loop Modus für die sequence (alle nachfolgend erstellten sequence Instanzen werden nun diesen loop Modus verwenden).
