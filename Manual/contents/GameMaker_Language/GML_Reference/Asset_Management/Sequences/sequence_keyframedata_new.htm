sequenz_keyframedata_neu

Mit dieser Funktion können Sie eine neue Keyframe-Datenstruktur erstellen und dabei den Typ der Spur angeben, auf die die Keyframe-Daten angewendet werden sollen. Der Typ argument kann eine der auf dieser Seite aufgeführten Konstanten für den Sequenzspurtypsein.

Die Funktion gibt eine Track-Keyframe-Datenstruktur zurück, zu der dann Werte hinzugefügt werden können, bevor sie einer Keyframe-Struktur zugewiesen wird.

Syntax:

sequence_keyframedata_new(type);

Argument Typ Beschreibung
type Sequenz Spurart Konstante The type of keyframe data to create.

Rückgabe:

Sequence Keyframe Data Struct

Beispiel:

myseq = sequence_create();
var mytracks = array_create(1);
mytracks[0] = sequence_track_new(seqtracktype_graphic);
var graphickeys = array_create(1);
graphickeys[0] = sequence_keyframe_new(seqtracktype_graphic);
graphickeys[0].frame = 0;
graphickeys[0].length = 1;
graphickeys[0].stretch = true;
graphickeys[0].disabled = false;
var graphickeydata = array_create(1);
graphickeydata[0] = sequence_keyframedata_new(seqtracktype_graphic);
graphickeydata[0].spriteIndex = spr_Platform;
graphickeydata[0].channel = 0;
graphickeys[0].channels = graphickeydata;
mytracks[0].name = "TestGraphicTrack";
mytracks[0].keyframes = graphickeys;
myseq.tracks = mytracks;

Die obige code erstellt eine neue sequence und erstellt dann eine Grafik-Asset-Spur und setzt einige Keyframe-Daten auf die Spur. Diese Spur wird dann der Instanz zugewiesen.
