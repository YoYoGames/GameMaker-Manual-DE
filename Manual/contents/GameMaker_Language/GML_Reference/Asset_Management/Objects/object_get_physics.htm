objekt_get_physik

Diese Funktion teilt Ihnen mit, ob das zu prüfende Objekt als "physics enabled" gekennzeichnet wurde - in diesem Fall gibt sie true zurück - oder nicht - in diesem Fall gibt sie false zurück.

Syntax:

object_get_physics(obj);

ArgumentTyp Beschreibung
obj The index of the object to check.

Rückgabe:

Beispiel:

if object_get_physics(object_index)
{
    phy_active = true;
}

Die obige code prüft die Instanz, auf der sie läuft, um zu sehen, ob das Objekt, von dem sie erstellt wird, physikfähig ist, und wenn ja, aktiviert sie die Physiksimulation für die Instanz.
