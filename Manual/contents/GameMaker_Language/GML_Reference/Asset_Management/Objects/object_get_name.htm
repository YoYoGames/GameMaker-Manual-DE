objekt_get_name

Diese Funktion gibt den Namen des angegebenen Objekts als String zurück. Dieser Name ist derjenige, der für das Objekt im Asset-Browser des GameMaker-Hauptfensters angegeben wurde. Bitte beachten Sie, dass dies nur eine Zeichenkette ist ist und nicht verwendet werden kann, um das Objekt direkt zu referenzieren - dazu bräuchten Sie den Objektindex. Sie können diese Zeichenkette jedoch verwenden, um den Objektindex zu ermitteln, indem Sie die zurückgegebene Zeichenkette zusammen mit der Funktion asset_get_index().

Syntax:

object_get_name(obj);

ArgumentTyp Beschreibung
obj The index of the object to check.

Rückgabe:

Beispiel:

str = object_get_name(object_index);

Die obige code ermittelt den Namen des Objektindex für die Instanz, auf der die code läuft, und speichert den Rückgabewert in der Variablen "str".
