object_get_solid

Diese Funktion zeigt Ihnen an, ob das zu prüfende Objekt als "solid" gekennzeichnet wurde oder nicht. Ein solides Objekt erzeugt ein spezielles Kollisionsereignis, wenn das traditionelle Kollisionssystem verwendet wird (d.h.: die Physikwelt ist ausgeschaltet). Bitte beachten Sie, dass es sich hierbei nicht um eine Instanzfunktion handelt! Sie können also ein solides Objekt und eine normale Instanz desselben Objekts haben und umgekehrt. Du kannst das Solid-Flag einer einzelnen Instanz mit der solid Instanz-Variable setzen.

Syntax:

object_get_solid(obj);

ArgumentTyp Beschreibung
obj The index of the object to check.

Rückgabe:

Beispiel:

if (!solid) && (object_get_solid(object_index))
{
    solid = true;
}

Das obige code prüft die Instanz, die es ausführt, um festzustellen, ob sie solide ist oder nicht, und prüft den Objektindex der Instanz, um festzustellen, ob sie als solide gekennzeichnet ist oder nicht. Wenn die Instanz nicht solide ist, aber der Objektindex als solide gekennzeichnet ist, wird "solid" für diese Instanz auf true gesetzt.
