object_get_sprite

Diese Funktion sagt Ihnen, ob das zu prüfende Objekt eine sprite hat oder nicht, und wenn ja, dann gibt sie den Index dieser sprite zurück, oder -1, wenn es keine hat. Bitte beachten Sie, dass es sich hierbei nicht um eine Instanzfunktion handelt! Sie können ein Objekt ohne sprite haben, während eine Instanz desselben Objekts eine haben kann und umgekehrt, oder sie können sogar unterschiedliche sprites haben. Sie können eine einzelne Instanz sprite mit der sprite_index Instanz-Variable setzen.

Syntax:

object_get_sprite(obj);

ArgumentTyp Beschreibung
obj The index of the object to check

Rückgabe:

Sprite Asset

Beispiel:

var _spr = object_get_sprite(object_index);
if (sprite_index != _spr)
{
    sprite_index = _spr;
}

Im obigen Beispiel wird die sprite_index der Instanz mit der sprite der object_index der Instanz verglichen. Wenn sie nicht übereinstimmen, wird der Instanz dieselbe sprite wie die des Objektindex zugewiesen.
