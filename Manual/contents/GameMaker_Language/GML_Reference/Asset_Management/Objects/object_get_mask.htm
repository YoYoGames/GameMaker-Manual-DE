object_get_mask

Diese Funktion sagt Ihnen, ob das zu prüfende Objekt einen Maskenindex hat oder nicht, und wenn ja, dann gibt sie den Index dieser Maske zurück (das ist ein sprite asset), oder -1, wenn nicht. Bitte beachten Sie, dass es sich hierbei nicht um eine Instanzfunktion handelt! Sie können ein Objekt ohne Maske haben, während eine Instanz desselben Objekts eine haben kann und umgekehrt, oder sie können sogar unterschiedliche Masken haben. Sie können den Maskenindex einer einzelnen Instanz mit der mask_index Instanz-Variable setzen.

Syntax:

object_get_mask(obj);

ArgumentTyp Beschreibung
obj The index of the object to check

Rückgabe:

Sprite Asset

Beispiel:

if (mask_index != object_get_mask(object_index))
{
    mask_index = object_get_mask(object_index);
}

Im obigen Beispiel wird der Maskenindex der Instanz mit der Maske des object_index der Instanz verglichen. Wenn sie nicht übereinstimmen, wird der Instanz die gleiche Maske zugewiesen, die der Objektindex hat.
