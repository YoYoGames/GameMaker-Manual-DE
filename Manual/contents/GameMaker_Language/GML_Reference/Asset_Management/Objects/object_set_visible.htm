object_set_visible

Mit dieser Funktion können Sie ein bestimmtes Objekt als sichtbar oder nicht sichtbar kennzeichnen (weitere Informationen über das Kennzeichen "sichtbar" finden Sie im Abschnitt über den Objekteditor). Das bedeutet, dass alle Instanzen dieses Objekts, die nach der Änderung von "sichtbar" erstellt werden, mit diesem neuen Status erstellt werden, während Instanzen, die sich bereits im Raum befinden, nicht betroffen sind.

Bitte beachten Sie, dass es sich hierbei nicht um eine Instanzfunktion handelt! Sie können das visible flag einzelner Instanzen mit der visible setzen und so zehn Instanzen als sichtbar kennzeichnen, obwohl das betreffende Objekt in der IDE auf false gesetzt ist. Und selbst wenn du das visible flag des Objekts mit dieser Funktion auf true änderst, bleiben alle Instanzen, die sich derzeit im Raum befinden, so wie sie waren, und nur Instanzen, die nach dem Aufruf der Funktion erstellt werden, beginnen mit visible als true gekennzeichnet.

Syntax:

object_set_visible( index, vis );

Argument Typ Beschreibung
index Objekt Anlage The index of the object to change.
vis Boolesche The new visibility of the object (true=sichtbar, false=unsichtbar).

Rückgabe:

N/A

Beispiel:

if !object_get_visible(obj_Block)
{
    object_set_visible(obj_Block, true);
}

Die obige code prüft die Sichtbarkeitsmarkierung des Objekts "obj_Block", und wenn sie falsch ist, wird sie in "wahr" geändert.
