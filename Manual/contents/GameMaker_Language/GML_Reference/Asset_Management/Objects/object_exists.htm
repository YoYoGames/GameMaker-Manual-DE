objekt_existiert

Diese Funktion gibt zurück, ob der angegebene Wert ein Objekt-Asset ist (true) oder nicht (false). Beachten Sie, dass diese Funktion prüft, ob ein Objekt im Asset Browser vorhanden ist, und nicht im Spielraum. Hierfür sollten Sie die Funktion instance_exists().

Syntax:

object_exists(obj);

ArgumentTyp Beschreibung
obj The index of the object to check.

Rückgabe:

Beispiel:

var object_id = buffer_read(net_buffer, buffer_u16);
var pos_x = buffer_read(net_buffer, buffer_u16);
var pos_y = buffer_read(net_buffer, buffer_u16);

if (object_exists(object_id))
{
    instance_create_layer(pos_x, pos_y, "Instances", object_id);
}

Die obige code liest Daten von einer buffer, die über ein Netzwerk empfangen wurden. Da sie eine Objekt-ID enthält (die zum Erzeugen von Instanzen verwendet wird), müssen wir zunächst sicherstellen, dass der Wert nicht ungültig ist und dass das Objekt tatsächlich existiert. Zu diesem Zweck rufen wir object_exists() in einer Bedingung auf, und wenn diese wahr ist, erstellen wir eine Instanz dieses Objekts an der empfangenen Position.
