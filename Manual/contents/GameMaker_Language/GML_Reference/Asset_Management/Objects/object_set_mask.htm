object_set_mask

Mit dieser Funktion können Sie den mask_index eines bestimmten Objekts setzen (weitere Informationen zu Masken finden Sie unter Der Objekteditor). Das bedeutet, dass alle Instanzen dieses Objekts, die nach der Änderung der Maske erstellt werden, mit dieser neuen Maske erstellt werden, während Instanzen, die sich bereits im Raum befinden, möglicherweise nicht betroffen sind. Sie können diesen Wert auf -1 setzen, um eine Maske sprite zu entfernen und somit die für sprite des Objekts definierte Maske zu verwenden (oder keine Masken, wenn keine sprite gewählt wurde).

Bitte beachten Sie, dass es sich hierbei nicht um eine Instanzfunktion handelt! Sie können den Maskenindex der einzelnen Instanzen mit der Variable mask_index Variable setzen und so zehn Instanzen haben, die alle eine andere Maske haben als das Objekt, von dem sie erstellt wurden. Selbst wenn Sie den Maskenindex des Objekts mit dieser Funktion ändern, bleiben alle Instanzen, die sich derzeit im Raum befinden, so wie sie waren, und nur Instanzen, die nach dem Aufruf der Funktion erstellt werden, beginnen mit der neuen Maske.

Syntax:

object_set_mask(index, spr);

Argument Typ Beschreibung
index Objekt Anlage The index of the object to change.
spr Sprite-Asset The new sprite to use as the object's mask.

Rückgabe:

N/A

Beispiel:

if (global.level == 10)
{
    object_set_mask(obj_Platform, spr_mask_10);
}

Die obige code prüft den Wert der globalen Variablen und ändert dann den Maskenindex des Objekts "obj_Platform", wenn er gleich zehn ist.
