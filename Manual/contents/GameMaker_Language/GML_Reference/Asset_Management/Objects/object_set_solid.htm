object_set_solid

Mit dieser Funktion können Sie ein bestimmtes Objekt als solide oder nicht solide kennzeichnen (weitere Informationen über das "solid"-Flag finden Sie im Abschnitt über den Objekteditor). Das bedeutet, dass alle Instanzen dieses Objekts, die nach der Änderung von "solid" erstellt werden, mit diesem neuen Status erstellt werden, während Instanzen, die sich bereits im Raum befinden, davon nicht betroffen sind.

Bitte beachten Sie, dass es sich hierbei nicht um eine Instanzfunktion handelt! Sie können die Solid-Flagge einzelner Instanzen mit der Variable solid setzen und so zehn Instanzen als solid kennzeichnen, obwohl das betreffende Objekt die Flagge in der IDE auf false gesetzt hat. Und selbst wenn Sie die Solid-Flagge des Objekts mit dieser Funktion auf true ändern, bleiben alle Instanzen, die sich derzeit im Raum befinden, so wie sie waren, und nur Instanzen, die nach dem Aufruf der Funktion erstellt werden, beginnen mit solid als true.

Syntax:

object_set_solid( index, solid );

Argument Typ Beschreibung
index Objekt Anlage The index of the object to change.
solid Boolesche The new solidity of the object (true=fest, false=nicht fest).

Rückgabe:

N/A

Beispiel:

if !object_get_solid(obj_Block)
{
    object_set_solid(obj_Block, true);
}

Die obige code prüft das Solid-Flag des Objekts "obj_Block" und wenn es falsch ist, wird es als wahr gekennzeichnet.
