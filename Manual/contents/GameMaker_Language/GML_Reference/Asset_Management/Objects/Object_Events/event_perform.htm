event_perform

Diese Funktion führt das angegebene Ereignis mit dem angegebenen argument für die Instanz aus, auf der code läuft. Es gibt hier viele Optionen, die eine vollständige Simulation aller möglichen Ereignisse ermöglichen, aber beachten Sie, dass dies buchstäblich nur alle code in dem Ereignis ausführt und das Spiel nichts ändert, um es selbst manuell auszulösen, z. B. wenn Sie sich entscheiden, ein Tastaturdruck-Ereignis auszuführen, wird das Ereignis ausgelöst, aber die entsprechende Taste wird nicht als gedrückt erkannt. Oder wenn Sie ein Alarmereignis auslösen, wird der Alarmzähler nicht auf -1 gesetzt, sondern weiter heruntergezählt.

Asynchrone Ereignisse können aufgerufen werden mit event_perform_async().

In der folgenden Tabelle finden Sie alle Konstanten, die im GameMaker zur Referenzierung von Ereignistypen enthalten sind, sowie deren Zahlenwert. Die Ereignisse selbst sind fett gedruckt und entsprechen den Rückgabewerten der event_type Variablen (die in die erste argument dieser Funktion übergeben wird), während ihre Zahlenwerte kursiv gedruckt sind und den Rückgabewerten der event_number Variablen (die an die zweite argument dieser Funktion übergeben werden):

---- ev_gui_end Das End-Ereignis der Zeichen-GUI.

Syntax:

event_perform(type, numb);

Argument Typ Beschreibung
type Ereignistyp Konstante The type of event to perform (see the table above).
numb Real oder Ereignisnummer Konstante The specific event constant or value, if one is necessary (otherwise, just use 0).

Rückgabe:

N/A

Erweitertes Beispiel

Um zu verstehen, wie diese Funktion verwendet werden kann, sehen Sie sich bitte die folgenden Beispiele an:

event_perform(ev_keypress, ord("W"));

Dies würde das Ereignis ausführen, das mit der Tastaturprüfung Gedrückte Taste "W" verbunden ist (ohne tatsächlich einen Tastendruck zu erzeugen).

event_perform(ev_step, ev_step_begin);

Dadurch wird das Ereignis "Schritt beginnen" ausgeführt (wenn es von einem der Schritt-Ereignisse aufgerufen wird, wird das Ereignis "Schritt beginnen" code zweimal ausgeführt).

event_perform(ev_create, 0);

Dies würde das EreignisCreate der Instanzausführen .
