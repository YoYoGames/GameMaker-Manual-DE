Ereignis_Art

Diese schreibgeschützte Variable gibt den Typ des aktuell ausgeführten Ereignisses zurück, der eine der folgenden Konstanten sein kann:

  • ev_erstellen
  • ev_destroy
  • ev_step
  • ev_alarm
  • ev_keyboard
  • ev_keypress
  • ev_keyrelease
  • ev_maus
  • ev_kollision
  • ev_andere
  • ev_gesture
  • ev_draw


Eine vollständige Liste der Konstanten, die sich auf die verschiedenen Ereignisse beziehen, finden Sie unter event_perform()und wenn Sie die Ereignisnummer (d.h. das Unterereignis) benötigen, sollten Sie die event_number.

Syntax:

event_type;

Rückgabe:

Event Type Constant

Beispiel:

show_debug_message("Current Event = " + string(event_type));

Auf der obigen Seite code wird eine Debug-Meldung mit dem gerade ausgeführten Ereignistyp angezeigt.
