Erzeugen von Objektereignissen

GameMaker ist vollständig ereignisgesteuert, d. h. alle Aktionen werden durch bestimmte Ereignisse ausgelöst, die in einer bestimmten Reihenfolge oder durch ein bestimmtes Ereignis im Spiel ausgelöst werden. Es gibt eine Reihe verschiedener Ereignisse, jedes mit einer bestimmten Aufgabe oder einem bestimmten Zeitplan, und einige von ihnen sind sogar in separate "Unter"-Ereignisse aufgeteilt. Normalerweise werden diese Ereignisse von GameMaker ausgeführt, wenn er einen Auslöser im Spiel erkennt, oder bei jedem Spieltick (wie es bei den Ereignissen "Schritt" und "Zeichnen" der Fall ist), aber Sie können auch ein Ereignis auf eine Instanz im aktuellen Raum aus einem Teil von code anwenden.

Es gibt folgende Funktionen, die sich mit der Erzeugung von Ereignissen befassen:

  • event_inherited
  • event_perform
  • event_perform_async
  • event_perform_object
  • event_user

Es gibt auch eine Reihe von Nur-Lese-Variablen, die mit den oben aufgeführten Funktionen verwendet werden können und besonders für die Fehlersuche nützlich sind:

  • event_action
  • event_number
  • event_object
  • event_type

Weitere Informationen über Ereignisse finden Sie im Abschnitt Objektereignisse.
