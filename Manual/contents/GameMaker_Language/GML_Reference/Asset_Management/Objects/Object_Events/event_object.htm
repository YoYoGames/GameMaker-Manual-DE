Ereignis_Objekt

Diese schreibgeschützte Variable gibt den Objektindex der Instanz zurück, die das zu prüfende Ereignis ausführt.

Syntax:

event_object;

Rückgabe:

Object Asset

Beispiel:

global.obj = event_object;

Die obige code speichert den Objektindex der Instanz, die das Ereignis ausführt, in einer globalen Variablen.
