event_action DEPRECATED

Diese schreibgeschützte Variable gibt den Index der Aktion zurück, die gerade ausgeführt wird, beginnend bei 0 in früheren Versionen von GameMaker. Diese Variable ist in GameMaker jedoch veraltet . Sie wurde nur noch als Legacy-Support verwendet und wird immer 0 zurückgeben, da alle Aktionen miteinander verkettet werden, um die Ausführungsgeschwindigkeit zu erhöhen.

Syntax:

event_action;

Rückgabe:

Real

Beispiel:

num = event_action;

Die obige code speichert die aktuelle Aktionsnummer in der Variablen "num".
