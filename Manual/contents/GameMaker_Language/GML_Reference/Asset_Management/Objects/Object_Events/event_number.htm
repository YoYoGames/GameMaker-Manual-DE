event_number

Diese schreibgeschützte Variable gibt die Nummer des gerade aufgerufenen Ereignisses zurück, wobei sich die Nummer auf das "Unterereignis" des Ereignisses bezieht, d. h. für das Schritt-Ereignis könnte die Ereignisnummer eine der Konstanten ev_step_normal, ev_step_begin oder ev_step_end sein. Eine vollständige Liste der Konstanten, die für die einzelnen Unterereignisse verfügbar sind, finden Sie unter event_perform()und wenn Sie das Hauptereignis selbst wissen müssen, sollten Sie die Konstante event_type.

Syntax:

event_number

Rückgabe:

Event Number Constant

Beispiel:

switch (event_number)
{
    case ev_step_normal: show_debug_message("Step event!"); break;
    case ev_game_start: show_debug_message("Game Start""); break;
    case ev_room_start: show_debug_message("Room Start!"); break;
}

Das obige code könnte von script aus aufgerufen werden und dazu dienen, Debug-Meldungen anzuzeigen, die darüber informieren, welches Ereignis gerade ausgelöst wird.
