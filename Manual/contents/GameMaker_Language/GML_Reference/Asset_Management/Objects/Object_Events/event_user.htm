event_user

Mit dieser Funktion weisen Sie die Instanz an, die Aktionen oder code auszuführen, die in einem der 16 benutzerdefinierten Ereignisse platziert wurden. Diese Ereignisse können nur auf diese Weise oder mit der event_perform Funktion.

Syntax:

event_user(numb);

Argument Typ Beschreibung
numb Real The number of User Event to call, between 0 and 15.

Rückgabe:

N/A

Beispiel:

event_user(4);

Dies würde das benutzerdefinierte Ereignis 4 aufrufen.
