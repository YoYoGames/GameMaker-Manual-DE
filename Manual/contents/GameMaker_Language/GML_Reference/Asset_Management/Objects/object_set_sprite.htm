object_set_sprite

Mit dieser Funktion kannst du den sprite Index eines bestimmten Objekts setzen. Das bedeutet, dass alle Instanzen dieses Objekts, die erstellt werden , nachdem der sprite_index geändert wurde, mit diesem neuen sprite_index erstellt werden, während Instanzen, die sich bereits im Raum befinden, davon nicht betroffen sein dürfen.

Bitte beachten Sie, dass es sich hierbei nicht um eine Instanzfunktion handelt! Sie können den sprite Index der einzelnen Instanzen mit der sprite_index Variable setzen und so zehn Instanzen haben, die alle einen anderen sprite haben als das Objekt, von dem sie erstellt wurden. Selbst wenn Sie den sprite Index des Objekts mit dieser Funktion ändern, bleiben alle Instanzen, die sich derzeit im Raum befinden, so wie sie waren, und nur Instanzen, die nach dem Aufruf der Funktion erstellt werden, beginnen mit dem neuen sprite.

Syntax:

object_set_sprite( index, spr );

Argument Typ Beschreibung
index Objekt Anlage The index of the object to change.
spr Sprite-Asset The sprite to assign to the object.

Rückgabe:

N/A

Beispiel:

switch (room)
{
    case rm_start: object_set_sprite(obj_Player, spr_uniform); break;
    case rm_middle: object_set_sprite(obj_Player, spr_swimsuit); break;
    case rm_end: object_set_sprite(obj_Player, spr_casual); break;
}
instance_create_layer(32, 32, "Instances", obj_Player);

Die obige code setzt den Index des Objekts "obj_Player" sprite auf unterschiedliche Werte, je nachdem, in welchem Raum sich die Instanz, die code ausführt, gerade befindet. Anschließend wird eine Instanz von "obj_Player" erstellt.
