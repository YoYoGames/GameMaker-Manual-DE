object_get_visible

Diese Funktion teilt Ihnen mit, ob das zu prüfende Objekt als "sichtbar" gekennzeichnet wurde (sein Zeichenereignis wird ausgeführt) oder nicht (sein Zeichenereignis wird nicht ausgeführt). Bitte beachten Sie, dass es sich hierbei nicht um eine Instanzfunktion handelt! Sie können also ein sichtbares Objekt und eine unsichtbare Instanz desselben Objekts haben und andersherum. Sie können die Sichtbarkeit einer einzelnen Instanz mit der visible Instanz-Variable einstellen.

Syntax:

object_get_visible(obj);

ArgumentTyp Beschreibung
obj The index of the object to check.

Rückgabe:

Beispiel:

if (!visible) && (object_get_visible(object_index))
{
    visible = true;
}

Das obige code prüft die Instanz, die es ausführt, um zu sehen, ob sie sichtbar ist oder nicht, und prüft den Objektindex der Instanz, um zu sehen, ob sie als sichtbar gekennzeichnet ist oder nicht. Wenn die Instanz nicht sichtbar ist, aber der Objektindex als sichtbar gekennzeichnet ist, wird "visible" für diese Instanz auf true gesetzt.
