Objekt_Index

Diese schreibgeschützte Variable gibt den Index des Objekts zurück, von dem die Instanz erstellt wurde. Dies ist nicht dasselbe wie der Objektname, der eine Zeichenkette ist und mit object_get_name()gefunden werden kann, da diese Funktion die Indexnummer zurückgibt, die ein eindeutiger Wert ist, den GameMaker jedem Objekt zum Zeitpunkt der Erstellung zuweist.

Syntax:

object_index;

Rückgabe:

Object Asset

Beispiel:

obj_name = object_get_name(object_index);

Die obige code verwendet die object_index, um den Namen des Objekts zu finden, von dem die aktuelle Instanz erstellt wurde.
