object_get_persistent

Diese Funktion sagt dir, ob das Objekt, das du prüfst, als "persistent" gekennzeichnet wurde oder nicht. Ein persistentes Objekt ist eines, dessen Instanzen von Raum zu Raum mitgenommen werden, solange sie nicht explizit zerstört werden. Bitte beachte, dass dies keine Instanzfunktion ist! Du kannst also ein persistentes Objekt und eine nicht-persistente Instanz desselben Objekts haben und umgekehrt. Du kannst eine einzelne Instanz persistent machen, indem du die persistent Instanz-Variable setzen.

Syntax:

object_get_persistent(obj);

ArgumentTyp Beschreibung
obj The index of the object to check.

Rückgabe:

Beispiel:

if (!persistent) && (object_get_persistent(object_index))
{
    persistent = true;
}

Die obige code prüft die Instanz, die sie ausführt, um festzustellen, ob sie persistent ist oder nicht, und prüft den Objektindex der Instanz, um festzustellen, ob sie als persistent gekennzeichnet ist oder nicht. Wenn die Instanz nicht persistent ist, aber der Objektindex als persistent gekennzeichnet ist, wird "persistent" für diese Instanz auf true gesetzt.
