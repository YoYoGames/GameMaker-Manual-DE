tileset_get_name

Diese Funktion gibt den Namen des angegebenen Kachelsatz-Assets als String zurück. Dieser Name ist derjenige, der für das Kachelset im Asset-Browser des GameMaker-Hauptfensters angegeben wurde. Bitte beachten Sie, dass dies nur eine Zeichenkette ist und nicht verwendet werden kann, um das Kachelset direkt zu referenzieren - dazu bräuchten Sie den Kachelset-Index. Sie können diese Zeichenkette jedoch verwenden, um den Kachelsatzindex zu ermitteln, indem Sie die zurückgegebene Zeichenkette zusammen mit der Funktion asset_get_index().

Syntax:

tileset_get_name(index);

Argument Typ Beschreibung
index Kachelsatz Asset The index of the tile set to get the name of.

Rückgabe

String

Beispiel:

var _l = layer_get_id("tilemap_trees");
var _m = layer_tilemap_get_id(_l);
var _t = tilemap_get_tileset(_m);
tileset_name = tileset_get_name(_t);

Die obige code ermittelt den Namen des Kachelsatzindexes für die angegebene Ebene und speichert den Rückgabestring in der Variablen tileset_name.
