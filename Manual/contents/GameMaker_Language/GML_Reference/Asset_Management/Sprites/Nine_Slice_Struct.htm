Die Neun-Scheiben-Struktur

Eine Nine Slice-Struktur kann erstellt werden mit sprite_nineslice_create()erstellt oder von einer sprite abgerufen werden, indem sprite_get_nineslice(). Diese Struktur enthält Eigenschaften, die mit sprite geändert und auf eine angewendet werden können. sprite_set_nineslice().

Werden Werte aus einer sprite abgerufen , wirken sich Änderungen in dieser Struktur auf die ursprüngliche sprite aus.

Die folgenden Eigenschaften werden in dieser Struktur verfügbar sein:

Neun-Scheiben-Struktur
Variabel Typ Beschreibung
enabled Boolesche Dieser boolesche Wert ist true , wenn Nine Slicing für sprite aktiviert ist, und false , wenn es deaktiviert ist. Sie können dies ändern, um Nine Slicing für sprite zu aktivieren oder zu deaktivieren.
left Real Dies ist der Versatz für die linke Führung auf der X-Achse, bezogen auf den linken Rand der sprite.
right Real Dies ist der Versatz für die rechte Führung auf der X-Achse, bezogen auf den rechten Rand der sprite.
top Real Dies ist der Versatz für die obere Führung auf der y-Achse, bezogen auf die Oberkante der sprite.
bottom Real Dies ist der Versatz für die untere Führung auf der y-Achse, bezogen auf die Unterkante der sprite.
tilemode Array von Kachelmodus Konstantes Dies gibt ein Array zurück, das die Kachelmodi für die Rand- und Mittelscheiben enthält.

Kachel-Modi

Der Kachelmodus jedes Slice kann unter Verwendung dieser Konstanten als Array-Index gelesen oder geändert werden:

Neun-Scheiben-Kante Konstant
Konstante Beschreibung
nineslice_left Die linke Randscheibe
nineslice_top Die obere Randscheibe
nineslice_right Die rechte Randscheibe
nineslice_bottom Die untere Randscheibe
nineslice_centre Die mittlere Scheibe

Der Kachelmodus für ein Slice ist eine der folgenden Konstanten (oder kann auf eine solche eingestellt werden):

Kachelmodus Konstante
Konstante Beschreibung
nineslice_stretch Die Scheibe wird gestreckt
nineslice_repeat Die Scheibe wird wiederholt
nineslice_mirror Der Slice wird durch Spiegelung wiederholt
nineslice_blank Der Slice wird nicht gedehnt oder wiederholt, so dass nach ihm ein leerer Bereich entsteht.
nineslice_hide Die Scheibe wird nicht angezeigt

Da tilemode ein Array ist, müssen Sie das Array syntax verwenden, um den Kachelmodus eines Slice zu ändern. Das folgende code Beispiel setzt den Kachelmodus des obersten Slice auf " Repeat":

Beispiel:

var _nineslice = sprite_get_nineslice(spr_box);

_nineslice.tilemode[nineslice_top] = nineslice_repeat;
