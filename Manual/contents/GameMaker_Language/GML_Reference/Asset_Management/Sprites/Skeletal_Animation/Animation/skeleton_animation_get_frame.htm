skelett_animation_get_frame

Diese Funktion gibt die Bildnummer der Animation auf der angegebenen Animationsspur zurück. Die Funktion gibt -1 zurück, wenn der angegebenen Spur keine Animation zugeordnet ist.

Syntax:

skeleton_animation_get_frame(track)

Argument Typ Beschreibung
track Real The animation track to get the frame of.

Rückgabe:

Real

Beispiel:

var frame = skeleton_animation_get_frame(0);
if (frame > 0 && frame < 2)
{
    var box = skeleton_get_minmax();
    var ww = (box[2] - box[0]) / 2;
    var hh = (box[3] - box[1]) / 2;
    part_particles_create(global.p_sys, box[0] + ww, box[1] + hh, global.Stars, 10);
}

Die obige code prüft das aktuelle Bild der Animation, die der Spur 0 zugewiesen ist, und lässt dann einige Partikel von einem Punkt in der Mitte des gesamten Begrenzungsrahmens für die sprite platzen.
