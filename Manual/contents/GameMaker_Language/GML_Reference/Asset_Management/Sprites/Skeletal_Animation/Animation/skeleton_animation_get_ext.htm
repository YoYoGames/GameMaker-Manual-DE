skelett_animation_get_ext

Eine einzelne Skelettanimation sprite kann verschiedene Animations-Sets haben, und diese Sets können verschiedenen Spuren zugeordnet werden, so dass Sie Animations-Sets "mischen und anpassen" können. Diese Funktion gibt den Namen des Animationssets zurück, das derzeit von der angegebenen Spurnummer verwendet wird (wie von der Funktion skeleton_animation_set_ext).

Syntax:

skeleton_animation_get_ext(track);

Argument Typ Beschreibung
track Real The track number to get the animation set name of.

Rückgabe:

String

Beispiel:

if skeleton_animation_get_ext(1) != "bodyfight"
{
    skeleton_animation_set_ext("bodyfight", 1);
}

Die obige code ändert das von Spur 1 verwendete Animationsset in "bodyfight", falls es nicht bereits vorhanden ist.
