skeleton_animation_get_duration

Diese Funktion gibt die Zeit zurück, die die angegebene Animation benötigt, um zu laufen, bevor sie zum Anfang zurückkehrt. Der Rückgabewert wird in Sekunden angegeben.

Syntax:

skeleton_animation_get_duration(animname);

ArgumentTyp Beschreibung
animname The name (a string) of the animation set to use.

Rückgabe:

Beispiel 1:

time += delta_time / 1000000;
var duration = skeleton_animation_get_duration(skeleton_animation_get());
var frame = floor((image_number * (mTime / duration)) + 0.5) % image_number;
image_index = frame;
draw_self();

Die obige code setzt die image_index auf den richtigen Wert für die aktuell zugewiesene Skelettanimation sprite.

Beispiel2:

time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_white);

Die obige code zeichnet die angegebene Skelettanimation sprite und verwendet Delta-Time, um das zu zeichnende Bild festzulegen.
