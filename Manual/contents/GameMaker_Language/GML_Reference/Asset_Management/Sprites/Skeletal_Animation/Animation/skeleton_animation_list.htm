skelett_animation_liste

Mit dieser Funktion können Sie eine (vorab erstellte) DS-Liste mit allen Namen der Animationen füllen, die als Teil der Skelettanimation sprite enthalten sind. Die Namen sind Strings und können dann in den anderen Animationsfunktionen für diese Arten von sprite verwendet werden.

Syntax:

skeleton_animation_list(sprite, list);

Argument Typ Beschreibung
sprite Sprite-Asset The sprite index of the Spine skeletal animation to get the list from.
list DS-Liste ID The ID of the DS list to populate with the animation names.

Rückgabe:

N/A

Beispiel:

var list = ds_list_create();
skeleton_animation_list(sprite_index, list);

for (var i = 1; i < ds_list_size(list); i++)
{
    skeleton_animation_mix(list[| 0], list[| i], 0.5);
}

ds_list_destroy(list);

Die obige code erstellt eine DS-Liste mit allen Animationsnamen für die sprite, die von der Instanz verwendet werden. Dann geht loops durch diese und setzt den Mix-Wert für alle mit der ersten Animation auf 0,5.
