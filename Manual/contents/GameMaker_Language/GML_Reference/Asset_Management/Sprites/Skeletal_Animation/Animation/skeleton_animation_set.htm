skelett_animation_set

Sobald Sie Ihrer Instanz eine Skelettanimation sprite zugewiesen haben, können Sie mit dieser Funktion festlegen, welches Animationsset sie zu einem bestimmten Zeitpunkt verwenden soll. Bei der Erstellung deiner sprite (in Spine) hast du mehrere Animations-Sets für verschiedene Zustände definiert und sie entsprechend benannt. Diese Namen werden Sie verwenden, um GameMaker mitzuteilen, welches Animationsset zu einem bestimmten Zeitpunkt in Ihrem Spiel verwendet werden soll. Wenn dein sprite zum Beispiel für ein platform Spiel ist, dann kannst du ein "Sprung"-Set, ein "Lauf"-Set und ein "Leerlauf"-Set haben, die alle in dem einen sprite enthalten sind, und du kannst mit dieser Funktion switch zwischen ihnen in deinem Spiel wechseln.
Die Animation wird in einer Schleife abgespielt, was die Standardeinstellung ist.

Beachten Sie, dass das Setzen einer neuen Animation auf diese Weise den image_index auf 0 zurückgesetzt wird und die neue Animation am Anfang ihrer Animation loop beginnt. Wenn dies nicht das gewünschte Verhalten ist, dann können Sie einen "Mix"-Wert einstellen mit skeleton_animation_mix() einen "Mix"-Wert setzen, und wenn Sie eine neue Animation setzen, wird diese nahtlos mit der vorherigen interpoliert. Sie sollten auch darauf achten, dass am Anfang jeder Animation ein Keyframe steht, um zu verhindern, dass Daten aus früheren Animationen übernommen werden, es sei denn, Sie planen, separate Animationsspuren zu verwenden.

Syntax:

skeleton_animation_set(animname, [loop]);

Argument Typ Beschreibung
animname Zeichenfolge The name (a string) of the animation set to use.
loop Boolesche Ob die Animation loop sein soll oder nicht (Standard ist true)

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(vk_space)
{
    if skeleton_animation_get() != "jump"
    {
        skeleton_animation_set("jump");
    }
}

Die obige code ändert das verwendete Animations-Set in das "Sprung"-Set, wenn die Leertaste gedrückt wird, aber nur, wenn das aktuell verwendete Set nicht bereits "Sprung" ist.
