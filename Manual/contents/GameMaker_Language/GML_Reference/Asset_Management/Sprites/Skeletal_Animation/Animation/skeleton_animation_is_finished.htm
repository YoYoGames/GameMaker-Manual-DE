skelett_animation_ist_beendet

Mit dieser Funktion können Sie prüfen, ob die aktuelle Skelettanimation auf der angegebenen Spur beendet ist.

Syntax:

skeleton_animation_is_finished(track);

Argument Datenart Beschreibung
Spur Real Der Index der Animationsspur

Rückgabe:

Boolean

Beispiel:

var _finished = skeleton_animation_is_finished(1);
if (_finished)
{
    show_debug_message("The animation has finished playing");
}

Die obige code prüft zunächst, ob die Animation auf Spur 1 zu Ende abgespielt wurde und speichert das Ergebnis in einer lokalen Variablen _finished. Anschließend wird eine Fehlermeldung angezeigt, falls die Animation zu Ende abgespielt wurde.
