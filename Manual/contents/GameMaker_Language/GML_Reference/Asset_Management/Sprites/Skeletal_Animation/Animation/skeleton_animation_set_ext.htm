skelett_animation_set_ext

Wenn Sie ein Animationsset für Ihre Skelettanimation sprite einstellen, wird es intern der Spur 0 zugeordnet. Sie können jedoch weiteren Spuren (1 und höher) weitere Sets zuweisen und auf diese Weise Ihre sprite zwei (oder mehr) Animationen gleichzeitig abspielen lassen. Beachten Sie, dass die sprite in Ihrem Animationsprogramm so gestaltet sein sollte, dass die "Basis"-Animation in Spur 0 nur Keyframes für bestimmte Bones hat, während die nächsten Spuren Keyframes für die anderen Bones haben. Wenn Sie beispielsweise eine "Geh"-Animation haben, deren Keyframes sich nur auf die Beine des sprite auswirken, können Sie eine Reihe von Kopfanimationen und arm Animationen haben, die Sie in Spur 1 bzw. 2 einstellen können, so dass Sie eine Basisbewegung haben und diese in Echtzeit anpassen können, während das Spiel gespielt wird.
Die Animation wird in einer Schleife abgespielt, was die Standardeinstellung ist.

Syntax:

skeleton_animation_set_ext(animname, track, [loop]);

Argument Typ Beschreibung
animname Zeichenfolge The name (a string) of the animation set to use.
track Real The track for the animation set to use.
loop Boolesche Ob die Animation loop sein soll oder nicht (Standard ist true)

Rückgabe:

N/A

Beispiel:

if mouse_check_button(mb_left)
{
    skeleton_animation_set_ext("bodyfight", 1);
}
else
{
    skeleton_animation_set_ext("bodywalk", 1);
}

Die obige code ändert das Animations-Set, das von Spur 1 verwendet wird, in das "Bodyfight"-Set oder das "Bodywalk"-Set, wenn die linke Maustaste gedrückt wird.
