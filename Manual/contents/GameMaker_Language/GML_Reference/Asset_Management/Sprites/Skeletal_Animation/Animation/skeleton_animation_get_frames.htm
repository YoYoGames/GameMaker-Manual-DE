skelett_animation_get_frames

Diese Funktion kann verwendet werden, um die Anzahl der Frames einer bestimmten Skelettanimation zu ermitteln. Sie geben den Namen der Skelett-Animation an (als Zeichenkette, wie sie in dem Programm definiert ist, mit dem die Animation erstellt wurde, oder wie sie von der Funktion skeleton_animation_get() zurückgegeben wird) und die Funktion gibt die Anzahl der Frames, die sie hat, als Integer-Wert zurück. Die Funktion gibt 0 zurück, wenn die angegebene Animation nicht existiert.

Syntax:

skeleton_animation_get_frames(anim_name);

Argument Typ Beschreibung
anim_name Zeichenfolge The animation name to get the frames of.

Rückgabe:

Real

Beispiel:

var num = skeleton_animation_get_frames(skeleton_animation_get());
image_index = num -1;
image_speed = 0;

Die obige code ermittelt die Anzahl der Bilder in der Animation und setzt dann sprite auf das letzte Bild und stoppt die Animation.
