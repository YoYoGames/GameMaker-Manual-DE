skelett_animation_set_frame

Diese Funktion setzt die der angegebenen Spur zugewiesene Animation auf das von Ihnen angegebene Bild. Der Frame-Index sollte ein reeller Wert zwischen 0 und der Anzahl der Frames für die Animation sein (die Sie mit der Funktion skeleton_animation_get_frames). Wenn Sie einen Wert außerhalb dieses Bereichs angeben, wird die Animation in einer Schleife wiederholt, um die Differenz auszugleichen. Wenn die Animation z. B. 5 Bilder hat und Sie das Bild auf 8 setzen, wird das tatsächlich angezeigte Bild 3 sein.

Syntax:

skeleton_animation_set_frame(track, index);

Argument Typ Beschreibung
track Real The animation track to set the frame of.
index Real The new index for the animation frame.

Rückgabe:

N/A

Beispiel:

if global.pause
{
    skeleton_animation_set_frame(0, -image_speed);
}

Die obige code setzt den Frame-Index für die Animation in Spur 0 auf die negative Bildgeschwindigkeit, wenn die globale "pause"-Variable true ist (und hält damit die Animation an).
