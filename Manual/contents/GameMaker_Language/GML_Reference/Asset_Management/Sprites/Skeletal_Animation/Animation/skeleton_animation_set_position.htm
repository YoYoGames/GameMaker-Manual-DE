skelett_animation_setzen_position

Diese Funktion setzt die Animation, die der angegebenen Spur zugeordnet ist, auf die von Ihnen angegebene Position. Die Position sollte ein reeller Wert zwischen 0 und 1 sein. Wenn Sie einen Wert außerhalb dieses Bereichs angeben, wird die Spurposition auf den Bruchteil dieses Wertes gesetzt. Nehmen wir an, Sie verwenden einen Wert von 0,4, einen Wert von 1,4, einen Wert von 2,4, ... Alle diese Werte setzen die Position auf 0,4.

Syntax:

skeleton_animation_set_position(track, position);

Argument Typ Beschreibung
track Real The animation track to set the frame of.
position Real The new position for the animation frame.

Rückgabe:

N/A

Beispiel:

skeleton_animation_set_position(1, 0.25);

Die obige code setzt die Position für Spur 1 der Skelettanimation auf 0,25 oder, anders ausgedrückt, auf 25% vom Beginn der Animation.
