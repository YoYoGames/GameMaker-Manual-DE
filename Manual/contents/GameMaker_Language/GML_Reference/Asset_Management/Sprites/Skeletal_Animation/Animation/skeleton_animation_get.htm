skelett_animation_get

Mit dieser Funktion können Sie das aktuelle Animationsset ermitteln, das von Ihrer Skelettanimation verwendet wird sprite. Der Rückgabewert ist eine Zeichenkette, die den Namen des Sets enthält, wie Sie ihn in Ihrem Skelettanimationsprogramm definiert haben.

Syntax:

skeleton_animation_get();

Rückgabe:

Beispiel:

if keyboard_check_pressed(vk_space)
{
    if (skeleton_animation_get() != "jump")
    {
        skeleton_animation_set("jump");
    }
}

Die obige code ändert das verwendete Animations-Set in das "Sprung"-Set, wenn die Leertaste gedrückt wird, aber nur, wenn das aktuell verwendete Set nicht bereits "Sprung" ist.
