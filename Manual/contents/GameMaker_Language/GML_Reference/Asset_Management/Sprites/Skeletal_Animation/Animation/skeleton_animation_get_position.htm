skeleton_animation_get_position

Diese Funktion gibt die Position der Animation auf der angegebenen Animationsspur als Wert zwischen 0 und 1 zurück. Auf diese Weise können Sie die Spurposition als Prozentsatz erhalten, wobei der Wert 0 0% und der Wert 1 100% entspricht.

Es gibt -1 zurück, wenn der angegebenen Spur keine Animation zugeordnet ist oder wenn die Instanz keine sprite gesetzt hat.

Syntax:

skeleton_animation_get_position(track);

Argument Typ Beschreibung
track Real The animation track to get the position of.

Rückgabe:

Real

Beispiel:

var _position = skeleton_animation_get_position(0);
var _position_percent = _position * 100;
show_debug_message("The skeleton animation position on track 0 is at " + string(_position_percent) + "%.");

Die obige code prüft die aktuelle Position der Skelettanimation, die der Spur 0 zugewiesen ist, und speichert sie in einer temporären Variablen _position. Dann wird dieser Wert in einen Prozentsatz umgewandelt, indem er mit 100 multipliziert wird, und das Ergebnis wird in einer weiteren temporären Variablen _position_percent gespeichert.
Schließlich wird eine Debug-Meldung angezeigt, die den Prozentwert angibt.
