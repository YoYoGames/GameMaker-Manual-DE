skeleton_animation_get_event_frames

Diese Funktion kann verwendet werden, um alle Bilder für das angegebene Ereignis in der angegebenen Animation abzurufen. Sie geben den Namen der Skelett-Animation an (als String, wie er in dem Programm definiert ist, mit dem die Animation erstellt wurde, oder wie er von der Funktion skeleton_animation_get() zurückgegeben wird) und einen Ereignisnamen. Die Funktion gibt ein Array mit den Frames für dieses Ereignis zurück.

Wenn der angegebene Ereignisname nicht existiert, gibt die Funktion ein Array mit -1 als erstem (und einzigem) Element zurück.

Syntax:

skeleton_animation_get_event_frames(anim_name, event_name);

Argument Typ Beschreibung
anim_name Zeichenfolge The animation name to get the frames of.
event_name Zeichenfolge The event name to get the frames for.

Rückgabe:

Array of Reals

Beispiel:

var _frames = skeleton_animation_get_event_frames(skeleton_animation_get(), "Footstep");

if (_frames[0] != -1)
{
    var _count = array_length(_frames);
    var _current_frame = skeleton_animation_get_frame(0);

    for (var i = 0; i < _count; i ++)
    {
        if (_frames[i] == _current_frame)
        {
            audio_play_sound(snd_footstep, 1, false);
            break;
        }
    }
}

Die obige code ruft die Frames für das Ereignis "Footstep" ab. Wenn ein Array mit den gültigen Frames zurückgegeben wird, durchsucht es loops und prüft, ob das aktuelle Frame mit einem der Frames im Array übereinstimmt. In diesem Fall wird der Soundeffekt "Footstep" abgespielt und die loop unterbrochen (angehalten).
