skeleton_attachment_get

Zu einer Skelettanimation sprite können weitere sprites als Anhänge hinzugefügt werden, wobei diese sprites zu einem benannten Slot hinzugefügt werden (der Name wird beim Erstellen des Anhangs-Slots in Ihrem Animationsprogramm vergeben) und sie werden zusammen mit der Animation des aktuellen sprite gezeichnet. Mit dieser Funktion können Sie den Namen (als String) des Anhangs für den angegebenen Slot des aktuell zugewiesenen sprite abrufen. Beachten Sie, dass angehängte sprites durch ihren Namensstring referenziert werden, wie er in Spine zugewiesen wurde, oder wenn Sie skeleton_attachment_create().

Syntax:

skeleton_attachment_get(slot)

Argument Typ Beschreibung
slot Zeichenfolge The slot name (a string) to get the attachment of.

Rückgabe:

String

Beispiel:

if skeleton_attachment_get("slot_leftHand") == ""
{
    skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
}

Die obige code würde den aktuell zugewiesenen Namen der Anlage für den Steckplatz mit dem Namen "slot_leftHand" überprüfen, und wenn eine leere Zeichenkette zurückgegeben wird, wird eine neue sprite angelegt.
