Steckplätze

Die Skelettanimation sprites kann "Schlitze" haben, in denen Sie "Anhänge" platzieren können. Ein Aufsatz ist einfach ein Bild, das von sprite als Requisite verwendet werden kann und in einem Schlitz platziert wird, so dass es sich zusammen mit der Animation des Teils, in dem der Schlitz positioniert ist, bewegt. Stellen Sie sich die sprite eines Kriegers mit einem Aufsatz-Slot an seiner Hand vor - Sie können dann verschiedene Waffen sprites auf diesem Slot platzieren und den Krieger so aussehen lassen, als würde er ein Schwert oder eine Pike oder einen Schild oder was auch immer benutzen.

Dieser Abschnitt befasst sich speziell mit Slots und deren Einrichtung und Verwendung. Sie können jedoch auch Anhänge einrichten und Informationen über diese über die hier aufgeführten Funktionen erhalten.

Die verfügbaren Steckplatzfunktionen sind:

  • skeleton_slot_list
  • skeleton_slot_data
  • skeleton_slot_data_instance
  • skeleton_find_slot
  • skeleton_slot_colour_set
  • skeleton_slot_colour_get
  • skeleton_slot_alpha_get
