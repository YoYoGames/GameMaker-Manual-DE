skelett_schlitz_liste

Mit dieser Funktion können Sie eine (vorab erstellte) DS-Liste mit allen Namen der Slots füllen, die als Teil der Skelettanimation sprite erstellt wurden. Die Namen sind Strings und können dann in den anderen Skelettanimations-Slot-Funktionen für diese Arten von sprite verwendet werden.

Syntax:

skeleton_slot_list(sprite, list);

Argument Typ Beschreibung
sprite Sprite-Asset The sprite index of the Spine skeletal animation to get the list from.
list DS-Liste ID The ID of the DS list to populate with the bone names.

Rückgabe:

N/A

Beispiel:

bone_list = ds_list_create();
skeleton_slot_list(sprite_index, bone_list);

Die obige code erstellt eine DS-Liste und füllt sie mit den Namen der Steckplätze für die spätere Verwendung.
