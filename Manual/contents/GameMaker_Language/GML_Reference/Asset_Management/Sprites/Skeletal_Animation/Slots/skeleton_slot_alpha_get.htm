skeleton_slot_alpha_get

Mit dieser Funktion können Sie einen Alphawert für einen Slot im Anhang abrufen. Sie geben den Slotnamen (eine Zeichenkette) an und die Funktion gibt einen realen Wert für Alpha zwischen 0 und 1 zurück.

Syntax:

skeleton_slot_alpha_get(slot);

Argument Typ Beschreibung
slot Zeichenfolge The slot name to check, a string

Rückgabe:

Real

Beispiel:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if (!ds_list_empty(slot_list))
{
    for (var i = 0; i < ds_list_size(slot_list); ++i)
    {
        if skeleton_slot_alpha_get(slot_list[| i]) != 1
        {
            skeleton_slot_colour_set(slot_list[| i], c_white, 1);
        }
    }
}

Das obige code erstellt eine DS-Liste und füllt sie dann mit den Namen der Steckplätze, die an der Position der Maus im Raum gefunden wurden. Dann geht loops durch die Slot-Liste und setzt die Farbe für die gefundenen Slots auf Weiß mit einem Alpha von 1 zurück, wenn das Alpha nicht bereits 1 ist.
