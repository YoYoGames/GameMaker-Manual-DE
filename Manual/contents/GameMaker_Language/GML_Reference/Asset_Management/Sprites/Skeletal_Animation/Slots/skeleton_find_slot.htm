skelett_find_schlitz

Mit dieser Funktion können Sie herausfinden, welche Slots sich an einer bestimmten Raumposition in der Spine sprite befinden, die mit der aktuellen Instanz verbunden ist. Du erstellst eine DS-Liste und gibst ihre ID zusammen mit einer zu überprüfenden x/y-Position an. Die Liste wird dann mit Namensstrings für jeden der verfügbaren Anbringungssteckplätze für die sprite (einschließlich aller Anbringungsänderungen) gefüllt. Beachten Sie, dass die Liste immer in absteigender Reihenfolge sortiert ist, beginnend mit dem obersten Steckplatz.

Syntax:

skeleton_find_slot(x, y, list);

Argument Typ Beschreibung
x Real The x position in the room to check.
y Real The y position in the room to check.
list DS-Liste ID The ID of the DS list to populate with the DS maps.

Rückgabe:

N/A

Beispiel:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);

Die obige code erstellt eine DS-Liste und füllt sie mit den Namen der Steckplätze, die sich an der Position der Maus im Raum befinden.
