skelett_schlitz_farbe_set

Mit dieser Funktion können Sie die Farbe und den Alphawert eines angehängten Slots festlegen, so dass alle sprites, die mit diesem Slot verbunden sind, mit diesen Werten gezeichnet werden. Beachte, dass die Instanz sprite eine Überblendfarbe und eine Alpha-Einstellung haben kann (image_blend und image_alpha), ebenso wie der Anhang (siehe die Funktion skeleton_attachment_create_colour()), und so wird die endgültige Farbe und Alpha, die der zugewiesene Anhang sprite für den Slot haben wird, eine Zusammensetzung aus all diesen Werten sein.

Syntax:

skeleton_slot_colour_set(slot, colour, alpha);

Argument Typ Beschreibung
slot Zeichenfolge The slot name to set, a string
colour Farbe The colour to set, either an integer, a constant, or a hex value.
alpha Real The alpha to set from 0 to 1.

Rückgabe:

N/A

Beispiel:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if !ds_list_empty(slot_list)
{
    for (var i = 0; i < ds_list_size(slot_list); ++i)
    {
        if skeleton_slot_colour_get(slot_list[| i]) != c_white
        {
            skeleton_slot_colour_set(slot_list[| i], c_white, 1);
        }
    }
}

Das obige code erstellt eine DS-Liste und füllt sie mit den Namen der Steckplätze, die an der Position der Maus im Raum gefunden wurden. Dann geht loops durch die Slot-Liste und setzt die Farbe für die gefundenen Slots auf weiß mit einem Alpha-Wert von 1 zurück, wenn die Farbe nicht bereits weiß ist.
