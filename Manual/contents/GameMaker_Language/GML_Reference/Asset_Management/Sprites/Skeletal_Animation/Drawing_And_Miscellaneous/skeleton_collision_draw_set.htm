skeleton_collision_draw_set

Mit dieser Funktion können Sie das Zeichnen der Kollisionsdaten für die aktuelle Skelettanimation sprite, die von der Instanz verwendet wird, einschalten (true) oder ausschalten (false). Wenn dies eingeschaltet ist, werden die Bounding Box und die genaue Kollisionsmaske als Umrisse um die sprite gezeichnet.

Diese Funktion funktioniert nur, wenn das Zeichnen des Skeletts sprite vom Objekt übernommen wird. Um beim manuellen Zeichnen eines Skeletts sprite den Kollisionsbegrenzungsrahmen zu zeichnen, verwenden Sie draw_skeleton_collision().

Syntax:

skeleton_collision_draw_set(flag);

Argument Typ Beschreibung
flag Boolesche Set to true um das Zeichnen einzuschalten, und false, um es auszuschalten.

Rückgabe:

N/A

Beispiel:

if debug_mode == true
{
    skeleton_collision_draw_set(true);
}

Das obige code prüft, ob das Spiel im Debug-Modus läuft, und wenn ja, schaltet es die Skelettkollisionsdaten für die anzuzeigende Instanz um.
