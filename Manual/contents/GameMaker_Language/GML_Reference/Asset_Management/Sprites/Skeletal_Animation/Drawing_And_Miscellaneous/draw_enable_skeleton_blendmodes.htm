draw_enable_skeleton_blendmodes

Diese Funktion wird verwendet, um die Überblendungsmodi pro Slot für das Skelett zu aktivieren(true) oder zu deaktivieren(false) sprites.

Syntax:

draw_enable_skeleton_blendmodes(enable)

Argument Typ Beschreibung
enable Boolesche Whether to enable or disable blend modes

Rückgabe:

N/A

Beispiel:

draw_enable_skeleton_blendmodes(true);

Die obige Seite code ermöglicht Überblendungsmodi pro Slot für das Skelett sprites.
