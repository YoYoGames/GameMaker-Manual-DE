skelett_get_bounds

Diese Funktion gibt ein Array von Werten zurück, die mit einem bestimmten Begrenzungsrahmen für die aktuell zugewiesene Skelettanimation sprite verbunden sind. Sie geben die Indexnummer für die Boundingbox an, deren Details Sie erhalten möchten (Sie können die Gesamtzahl der Boundingboxen für sprite mit der Funktion skeleton_get_num_bounds()) und die Funktion gibt ein Array mit den folgenden Elementen zurück:

  • [0] - die Anzahl der Punkte auf dem Begrenzungsrahmen (ein Realwert)
  • [1] - der Name des Begrenzungsrahmens (eine Zeichenkette)
  • [2] - die x-Position des ersten Punktes
  • [3] - die y-Position des ersten Punktes
  • [usw...] - weitere x/y-Positionsdaten bis zur Anzahl der Punkte auf dem Begrenzungsrahmen

Syntax:

skeleton_get_bounds(index);

Argument Typ Beschreibung
index Real The bounding box index to get the values of.

Rückgabe:

Array (minimum 2 elements: numPoints, name [, xPos, yPos, etc...])

Beispiel:

var num = skeleton_get_num_bounds();
var yy = 60;
for(var i = 0; i < num; i++)
{
    var b_info = skeleton_get_bounds(i);
    if b_info[0] > 0
    {
        var data = b_info[1] + ":";
        for(var j = 0; j < b_info[0]; j++)
        {
            data += " (" + string(b_info[(j * 2) + 2]) + ", " + string(b_info[(j * 2) + 2 + 1]) + ")";
        }
        draw_text(20, yy, data);
        yy += 20;
    }
}

code Die obige Seite loop geht durch alle Bounding Boxes, die mit dem aktuell zugewiesenen sprite verbunden sind, und zeichnet diese Daten dann als String auf den Bildschirm.
