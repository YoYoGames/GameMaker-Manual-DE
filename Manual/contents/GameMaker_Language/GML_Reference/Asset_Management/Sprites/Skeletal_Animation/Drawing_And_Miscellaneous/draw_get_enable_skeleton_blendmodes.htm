draw_get_enable_skeleton_blendmodes

Diese Funktion gibt zurück, ob die Überblendmodi pro Slot für das Skelett sprites aktiviert(true) oder deaktiviert(false) sind.

Syntax:

draw_get_enable_skeleton_blendmodes()

Rückgabe:

Boolean

Beispiel:

if (!draw_get_enable_skeleton_blendmodes())
{
    draw_enable_skeleton_blendmodes(true);
}

Die obige code aktiviert die Überblendungsmodi pro Slot für das Skelett sprites, wenn sie deaktiviert sind.
