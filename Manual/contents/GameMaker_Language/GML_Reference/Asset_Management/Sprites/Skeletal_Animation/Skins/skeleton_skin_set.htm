skelett_haut_set

Eine Skelettanimation sprite kann mit einer Vielzahl von Texturen (so genannten "Skins") gezeichnet werden, so dass Sie eine einzige Animation für mehrere verschiedene Zwecke verwenden können. Sie können zum Beispiel eine allgemeine Laufanimation haben, die von Ihren NSC-Charakteren, Feinden usw. in einem Spiel verwendet werden kann, indem Sie einfach den Skin ändern, mit dem sie gezeichnet wird. Diese Skins müssen zuvor in Ihrem Animationsprogramm erstellt und benannt worden sein, und dieser Name wird an die Funktion (als String) übergeben, um sprite so einzustellen, dass dieser Skin beim Zeichnen verwendet wird.

Syntax:

skeleton_skin_set(skinname);

ArgumentTyp Beschreibung
skinname The name (a string) of the skin to use.

Rückgabe:

Beispiel:

if skeleton_skin_get() == "skin_Enemy1"
{
    skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
}

Die obige code würde den aktuell zugewiesenen Skin für die Animation überprüfen und, wenn es "skin_Enemy1" ist, einen neuen Skin aus einer der drei Optionen auswählen und einstellen.
