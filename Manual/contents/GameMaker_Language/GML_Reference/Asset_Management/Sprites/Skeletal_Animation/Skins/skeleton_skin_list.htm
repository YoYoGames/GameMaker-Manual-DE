Skelett_Haut_liste

Mit dieser Funktion können Sie eine (vorab erstellte) DS-Liste mit allen Namen der Skins füllen, die als Teil der Skelettanimation sprite enthalten sind. Die Namen sind Strings und können dann in den anderen Skelettanimations-Skin-Funktionen für diese Arten von sprite verwendet werden.

Syntax:

skeleton_skin_list(sprite, list);

Argument Typ Beschreibung
sprite Sprite-Asset The sprite index of the Spine skeletal animation to get the list from.
list DS-Liste ID The ID of the DS list to populate with the animation names.

Rückgabe:

N/A

Beispiel:

var list = ds_list_create();
skeleton_skin_list(sprite_index, list);
var num = ds_list_size(list);
skeleton_skin_set(list[| irandom(num - 1));
ds_list_destroy(list);

Die obige code erstellt eine DS-Liste und füllt sie mit den Skin-Namen. Dann wird ein zufälliger Name ausgewählt und auf die Instanz angewendet, bevor die Liste zerstört wird.
