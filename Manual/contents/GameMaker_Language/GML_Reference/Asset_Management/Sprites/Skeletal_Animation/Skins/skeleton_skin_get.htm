Skelett_Haut_holen

Mit der Skelettanimation sprites können Sie der Animation separate Texturen (sogenannte "Skins") zuweisen und so eine Animation für mehrere verschiedene Dinge verwenden. Diese Funktion gibt den Namen des Skins (als String) zurück, der der Skelettanimation sprite zugewiesen ist, die Ihre Instanz gerade verwendet. Der zurückgegebene Name ist der, den Sie bei der Erstellung der sprite in Ihrem Animationsprogramm festgelegt haben.

Syntax:

skeleton_skin_get();

Rückgabe:

Beispiel:

if skeleton_skin_get() == "skin_Enemy1"
{
    skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
}

Die obige code würde den aktuell zugewiesenen Skin für die Animation überprüfen und, wenn es "skin_Enemy1" ist, einen neuen Skin aus einer der drei Optionen auswählen und einstellen.
