sprite_prefetch

Diese Funktion kann verwendet werden, um eine Texturseite mit der angegebenen sprite im Voraus abzurufen (im Texturspeicher abzulegen). Sie geben den Index sprite an, wie er beim Erstellen des Assets sprite definiert wurde, und die Texturseite, auf der es sich befindet, wird in den Speicher geladen. Beachten Sie, dass die Funktion -1 zurückgibt, wenn Prefetch für die gewählte Ressource nicht unterstützt wird oder das Ziel platform HTML5 ist, oder sie gibt 0 zurück, wenn alles korrekt funktioniert.

HINWEIS: Auf den meisten Geräten kommt es zu Leistungseinbußen, wenn die Textur in den Texturspeicher hochgeladen wird, daher wird empfohlen, dass Sie sprite_prefetch auf zu Beginn eines Levels aufzurufen, um ein Abwürgen zu vermeiden.

Syntax:

sprite_prefetch(ind)

ArgumentTyp Beschreibung
ind The sprite index to fetch

Rückgabe:

(-1 or 0)

Beispiel:

sprite_prefetch(spr_Player_Aura);

Die obige code platziert die referenzierte sprite in den Texturspeicher, um sie zu verwenden.
