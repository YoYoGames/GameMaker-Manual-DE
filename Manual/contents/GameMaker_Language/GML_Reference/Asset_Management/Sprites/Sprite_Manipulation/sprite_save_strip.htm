sprite_save_strip

Mit dieser Funktion wird ein Streifenbild aus allen Teilbildern in sprite erstellt und unter dem angegebenen Dateinamen auf der Festplatte gespeichert. Das sprite muss unter runtime hinzugefügt worden sein (Sie können sprites nicht speichern, das über IDE hinzugefügt wurde) und die Datei muss mit der Erweiterung *.png gespeichert werden.

HINWEIS: Je nach gewähltem Ziel platform können Sie Dateien nur in begrenztem Umfang speichern und laden. Weitere Informationen finden Sie unter Dateiverwaltung.

Syntax:

sprite_save_strip(ind, filename);

Argument Typ Beschreibung
ind Sprite-Asset The index of the sprite to save.
filename Zeichenfolge The filename for the saved sprite strip.

Rückgabe

N/A

Beispiel:

var spr, i;
spr = sprite_create_from_surface(0, 0, 32, 32, true, true, 16, 16);

for (i = 1; i < 8; i +=1)
{
    sprite_add_from_surface(spr, i, 0, 32, 32, true, true, 16, 16);
}

sprite_save_strip(spr, "Custom_Player_Sprite.png");
sprite_delete(spr);

Die obige code erstellt eine sprite aus der Oberfläche, die in der Variable "surf" indiziert ist, wobei ihr Index der lokalen Variable "spr_Custom" zugewiesen wird, und verwendet dann eine for loop, um sich über die Oberfläche zu bewegen und verschiedene Abschnitte zu erfassen, die in die sprite als Unterbilder eingefügt werden. diese neue sprite wird dann als png-Streifen gespeichert, bevor sie aus dem Speicher entfernt wird.
