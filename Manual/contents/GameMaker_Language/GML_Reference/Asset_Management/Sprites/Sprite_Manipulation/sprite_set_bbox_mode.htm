sprite_set_bbox_mode

Mit dieser Funktion können Sie den Bounding-Box-Modus für eine sprite festlegen. Sie geben den Index sprite und den zu verwendenden Modus an, der eine der folgenden Konstanten sein sollte:

Bounding Box Modus Konstante
Konstante Beschreibung
bboxmode_automatic Automatisch - Der Begrenzungsrahmen wird automatisch berechnet, basierend auf der Toleranzeinstellung für das Sprite
bboxmode_fullimage Volles Bild - Der Begrenzungsrahmen wird so eingestellt, dass er die volle Breite und Höhe des sprite verwendet, unabhängig von der Toleranz und den "leeren" Pixeln.
bboxmode_manual Manuell - Der Begrenzungsrahmen wird manuell auf benutzerdefinierte Werte eingestellt, was die Verwendung der Funktion sprite_set_bbox()

HINWEIS Diese Funktion wirkt sich auf das Sprite-Asset aus, so dass alle weiteren Instanzen mit diesem sprite denselben Bounding-Box-Modus haben.

Syntax:

sprite_set_bbox_mode(ind, mode);

Argument Typ Beschreibung
ind Sprite-Asset The index of the sprite to change the mode of.
mode Bounding Box Modus Konstante The mode to set (a constant).

Rückgabe

N/A

Beispiel:

if sprite_get_bbox_mode(sprite_index) != bboxmode_automatic
{
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

Die obige code prüft den bbox-Modus für die aktuelle sprite und wenn er nicht automatic ist, wird er auf diesen Wert gesetzt.
