sprite_set_nineslice

Diese Funktion wird verwendet, um eine Nine-Slice-Struktur auf eine sprite anzuwenden. Sie geben den Index sprite und die anzuwendende Nine-Slice-Struktur an, die Sie mit sprite_nineslice_create() oder sprite_get_nineslice().

HINWEIS Diese Funktion wirkt sich auf das Sprite-Asset aus, so dass alle Änderungen, die Sie mit dieser Funktion vornehmen, sich auf alle Instanzen auswirken, die dieses sprite verwenden, und auch auf alle zukünftigen Instanzen.

Syntax:

sprite_set_nineslice(ind, nineslice);

Argument Typ Beschreibung
ind Sprite-Asset The index of the sprite to modify
nineslice Neun-Scheiben-Struktur The Nine Slice struct to apply

Rückgabe:

N/A

Beispiel:

var _nineslice = sprite_nineslice_create();

_nineslice.enabled = true;
_nineslice.left = 10;
_nineslice.right = 10;
_nineslice.top = 10;
_nineslice.bottom = 10;

sprite_set_nineslice(spr_box_0, _nineslice);

Die obige code erstellt eine neue Nine-Slice-Struktur, aktiviert sie und legt die Offsets für die Hilfslinien fest. Die Struktur wird dann auf eine sprite angewandt, wobei ihre Nine-Slice-Eigenschaften geändert werden.
