sprite_assign

Diese Funktion nimmt zwei zuvor erstellte (oder eingebundene) sprite -Indizes und kopiert das Bild von einem in den anderen. Auf diese Weise können Sie einen sprite in einen anderen Index kopieren (oder "klonen"). Beachten Sie, dass Sie nicht in eine Spielressource kopieren können. Sie müssen die sprite, in die kopiert werden soll, zuvor mit dem Befehl sprite_add() oder sprite_duplicate() Funktionen erstellt haben.

Syntax:

sprite_assign(index, sprite);

ArgumentTyp Beschreibung
index The index of the sprite to be copied to (ie: overwritten).
sprite The sprite to be copied from.

Rückgabe

Beispiel:

if sprite_exists(global.Player_Sprite)
{
    var t_spr = sprite_add("player.png", 16, true, true, 0, 0);
    sprite_assign(global.Player_Sprite, t_spr);
    sprite_delete(t_spr);
}
else
{
    global.Player_Sprite = sprite_add("player.png", 16, true, true, 0, 0);
}

Die obige code prüft, ob die globale Variable "Player_Sprite" ein sprite enthält, und wenn ja, verwendet sie sprite_assign, um es durch ein aus einer externen Datei geladenes zu ersetzen. Wenn sie kein sprite enthält, wird eines geladen und sein Index in dieser Variablen gespeichert.
