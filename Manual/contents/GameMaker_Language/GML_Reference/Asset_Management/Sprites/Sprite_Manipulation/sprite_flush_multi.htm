sprite_flush_multi

Mit dieser Funktion können Sie eine beliebige Anzahl von Texturseiten für die angegebene sprites aus dem Texturspeicher entfernen. Sie geben die sprite Indizes als Array an und die Texturseiten, denen sie zugeordnet sind, werden aus dem Texturspeicher entfernt. Beachten Sie, dass wenn eine der Textur-Seiten an anderer Stelle im Raum verwendet wird (von einer anderen Instanz sprite oder einem Hintergrund, etc...), kann es zu einem kleinen Leistungseinbruch kommen, da die Seite sofort wieder in den Speicher zurückgeladen wird, daher sollte man bei der Verwendung dieser Funktion vorsichtig sein. Beachten Sie, dass die Funktion -1 zurückgibt, wenn flush für die gewählten Ressourcen nicht unterstützt wird, oder sie gibt 0 zurück, wenn alles korrekt funktioniert hat.

Syntax:

sprite_flush_multi(array);

ArgumentTyp Beschreibung
array Array with the sprite indices to flush

Rückgabe:

(-1 or 0)

Beispiel:

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_flush_multi(spr_a);

Die obige code erstellt ein Array, in dem jedes Element einen sprite Index enthält. Dieses Array wird dann verwendet, um die sprite Texturen aus dem Speicher zu löschen.
