sprite_set_speed

Mit dieser Funktion können Sie die Grundgeschwindigkeit und den Typ der Animation für die Ressource sprite festlegen. Normalerweise wird dies im Sprite-Editor eingestellt, aber es kann vorkommen, dass Sie dies manuell für eine bestimmte sprite Ressource einstellen müssen. Sie müssen den sprite -Index, die Animationsgeschwindigkeit und die Art der Animationsgeschwindigkeit angeben, die Sie anwenden möchten. Die Art der Animationsgeschwindigkeit kann eine der folgenden Konstanten sein:

Konstante Beschreibung
spritespeed_framespersecond Die sprite wurde mit einer Animation in Bildern pro Sekunde definiert.
spritespeed_framespergameframe Die sprite wurde mit einer Animation in Frames pro Spielframe definiert.


Die Auswahl des Geschwindigkeitstyps, der für die Berechnung von Animationen verwendet werden soll, ist sehr wichtig, da es einen großen Unterschied zwischen den beiden Optionen gibt. Die folgenden zwei Beispiele veranschaulichen diesen Unterschied:

  • Wenn Sie ein sprite haben, das 1 Bild pro Sekunde zeichnet, und die Bildgeschwindigkeit auf 0,5 einstellen, wird es mit 0,5 Bildern pro Sekunde zeichnen. Wenn die Bildrate Ihres Spiels 60 Bilder pro Sekunde beträgt, zeichnet sprite 1 Bild pro 120 Spielbilder.
  • Wenn Sie ein sprite haben, das 1 Bild pro Spielbild zeichnet, und die Bildgeschwindigkeit auf 0,5 einstellen, wird es 0,5 Bilder pro Spielbild zeichnen. Wenn Ihre Spielbildrate 60 Bilder pro Sekunde beträgt, zeichnet sprite 30 Bilder pro 60 Spielbilder.

Sie können herausfinden, welche Geschwindigkeit für die Animation sprite verwendet wurde, indem Sie die Funktion sprite_get_speed()herausfinden, und die Art der Animationsgeschwindigkeit erhalten Sie mit sprite_get_speed_type().

WICHTIG! Diese Funktion wirkt sich auf die Ressource sprite aus, d.h. alle Änderungen, die Sie mit dieser Funktion vornehmen, wirken sich auf alle Instanzen aus, die die sprite verwenden, und auch auf alle zukünftigen Instanzen.

Syntax:

sprite_set_speed(index, speed, type)

Argument Typ Beschreibung
index Sprite-Asset The index of the sprite to set the animation speed of
speed Real The speed value to set
type Sprite-Geschwindigkeitskonstante The speed type to use

Rückgabe:

Constant (see above)

Beispiel:

sprite_set_speed(sprite_index, 10, spritespeed_framespersecond);

Die obige code setzt die Basis-Animationsgeschwindigkeit für die sprite, die der Instanz zugewiesen ist, auf 10 Bilder pro Sekunde.
