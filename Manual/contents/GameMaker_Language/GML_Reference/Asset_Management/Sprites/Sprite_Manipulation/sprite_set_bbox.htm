sprite_set_bbox

Mit dieser Funktion können Sie die Werte für die Bounding Box eines sprite festlegen. Sie geben den zu verwendenden Index sprite und dann die Werte für die Positionen der Bounding Box links, oben, rechts und unten an. Bei den Positionen handelt es sich um absolute Werte, wobei die Position (0, 0) der linken oberen Ecke von sprite entspricht, unabhängig vom Versatz für sprite, von "leeren" Pixeln, die sprite haben kann, oder davon, wo im Raum sie gezeichnet wird.

HINWEIS Diese Funktion wirkt sich auf das Asset sprite aus, so dass alle weiteren Instanzen mit diesem sprite denselben Begrenzungsrahmen haben.

Syntax:

sprite_set_bbox(ind, left, top, right, bottom);

ArgumentTyp Beschreibung
ind The index of the sprite to set the bounding box on.
left The left side of the bounding box
top The top of the bounding box.
right The right side of the bounding box
bottom The bottom of the bounding box.

Rückgabe

Beispiel:

if sprite_get_bbox_mode(sprite_index) == 2
{
    left = irandom(sprite_width / 2);
    right = irandom((sprite_width / 2) + irandom(sprite_width / 2));
    top = irandom(sprite_height / 2);
    bottom = irandom((sprite_height / 2) + irandom(sprite_height / 2));
    sprite_set_bbox(sprite_index, left, top, right, bottom);
}

Die obige code prüft den Modus des Begrenzungsrahmens des sprite, der dem sprite_index zugewiesen ist, und wenn dieser auf manual gesetzt ist, wird sein Begrenzungsrahmen geändert.
