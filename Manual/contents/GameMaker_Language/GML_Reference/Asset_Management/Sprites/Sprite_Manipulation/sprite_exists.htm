sprite_exists

Diese Funktion gibt zurück, ob eine sprite mit dem angegebenen Index im laufenden Projekt existiert oder nicht.

Syntax:

sprite_exists(index);

ArgumentTyp Beschreibung
index The index of the sprite to check.

Rückgabe

Beispiel:

if sprite_exists(spr_array[0])
{
    sprite_index = spr_array[0];
}
else
{
    sprite_index = spr_BaseSprite;
}

Das obige code prüft ein Array, um zu sehen, ob es einen gültigen sprite Index enthält und wenn ja, weist es der Instanz sprite zu, aber wenn nicht, weist es einen sprite aus den enthaltenen Ressourcen zu.
