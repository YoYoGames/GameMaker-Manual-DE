sprite_flush

Mit dieser Funktion können Sie die angegebene Textur-Seite für die angegebene sprite aus dem Textur-Speicher entfernen. Du gibst den sprite Index an (wie beim Erstellen der sprite Ressource definiert) und die Texturseite, der sie zugewiesen ist, wird aus dem Texturspeicher entfernt. Beachte, dass, wenn die Texturseite an anderer Stelle im Raum verwendet wird (durch eine andere Instanz sprite oder einen Hintergrund, etc...), du einen kleinen Performance-Schlag bekommen kannst, da die Seite sofort neu geladen wird, also solltest du vorsichtig sein, wenn du diese Funktion benutzt. Beachten Sie, dass die Funktion -1 zurückgibt, wenn flush für die gewählte Ressource nicht unterstützt wird, oder sie gibt 0 zurück, wenn alles korrekt funktioniert hat.

Syntax:

sprite_flush(ind)

ArgumentTyp Beschreibung
ind The index (resource name) of the sprite asset to flush

Rückgabe:

(-1 or 0)

Beispiel:

sprite_flush(spr_Player_Aura);

Die obige code löscht die sprite "spr_Player_Aura" aus dem Speicher.
