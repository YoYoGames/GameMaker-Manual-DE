sprite_add_from_surface

Diese Funktion funktioniert genau so wie sprite_create_from_surface() nur dass sie, anstatt ein neues sprite aus dem von Ihnen ausgewählten Bereich der indizierten Fläche zu erstellen, den definierten Bereich der Fläche als neues Teilbild zu einem zuvor erstellten sprite hinzufügt (das bedeutet, dass Sie es nicht direkt zu einem sprite aus dem Ressourcenbaum hinzufügen können, sondern nur zu einem, das zuvor aus einer Fläche erstellt wurde, oder zu einem, das aus einer Ressource sprite dupliziert wurde, indem sprite_duplicate()).

Es sollte beachtet werden, dass der Speicherbedarf beim Hinzufügen einer sprite auf diese Weise größer ist, als Sie vielleicht erwarten. Das liegt daran, dass GameMaker die sprite als Texturseite speichert und sie auch im GPU-Speicher abgelegt wird, so dass der Gesamtspeicher größer ist als bei einer Bilddatei der gleichen sprite zu erwarten wäre.

Syntax:

sprite_add_from_surface(index, surface, x, y, w, h, removeback, smooth);

Argument Typ Beschreibung
index Sprite-Asset The index of the sprite to add the new image to.
surface Oberfläche ID The index of the surface from which we get the image.
x Real The x position to copy from.
y Real The y position to copy from.
w Real The width of the area to be copied (from the x position).
h Real The height of the area to be copied (from the y position).
removeback Boolesche Indicates whether to make all pixels with the background colour (left-bottom pixel) transparent.
smooth Boolesche Indicates whether to smooth the edges.

Rückgabe:

N/A

Beispiel:

spr_custom = sprite_create_from_surface(surf, 0, 0, 32, 32, true, true, 16, 16);
var i;
for (i = 1; i < 8; i += 1)
{
    sprite_add_from_surface(spr_custom, surf, i, 0, 32, 32, false, false);
}

Die obige code erstellt eine sprite aus der in der Variablen "surf" indizierten Oberfläche, wobei ihr Index der Variablen "spr_Custom" zugewiesen wird, und verwendet dann eine for loop, um sich über die Oberfläche zu bewegen und verschiedene Abschnitte zu erfassen, die als Teilbilder in die sprite eingefügt werden.
