sprite_delete

Diese Funktion löscht ein sprite aus dem Spiel und gibt den dafür reservierten Speicher frei. Dies ist eine sehr wichtige Funktion für die Momente, in denen Sie sprites aus externen Quellen erstellen und ändern müssen (wie das Laden einer sprite aus einer Datei mit sprite_add() oder das Duplizieren eines sprite mit sprite_duplicate()) und sollte immer verwendet werden, um nicht mehr benötigte Assets zu entfernen nicht mehr benötigt werden, oder um ein indiziertes Asset aus einer Variablen zu löschen, bevor ein anderes Asset dieser Variablen neu zugewiesen wird. Dies kann nicht verwendet werden, um sprites zu löschen, die im Spiel als Teil der Assets im Asset-Browser enthalten sind.

Syntax:

sprite_delete(index);

ArgumentTyp Beschreibung
index The index of the sprite to be deleted.

Rückgabe

Beispiel:

var spr = sprite_create_from_surface(application_surface, 0, 0, 32, 32, false, false, 16, 16);
sprite_merge(spr_Player, spr);
sprite_delete(spr);

Die obige code erstellt eine lokale Variable und speichert dann den Index der sprite, die von der Anwendungsoberfläche erstellt wurde. Diese sprite wird dann mit dem in der Variablen "spr_Player" indizierten Asset zusammengeführt, bevor sie wieder aus dem Speicher entfernt wird.
