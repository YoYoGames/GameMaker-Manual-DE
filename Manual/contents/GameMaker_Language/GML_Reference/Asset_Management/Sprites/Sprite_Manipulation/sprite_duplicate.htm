sprite_duplicate

Diese Funktion gibt den Index einer neu erstellten sprite zurück, die ein Duplikat (Kopie) der als "index" argument eingegebenen Datei ist. Wenn die Funktion fehlschlägt, wird -1 zurückgegeben. Diese Funktion muss verwendet werden, um alle sprites aus den Original-Assets zu kopieren, bevor irgendwelche Transformationen daran vorgenommen werden können. Wenn Sie z. B. den Begrenzungsrahmen für ein sprite ändern oder dessen Alphawert von einem anderen sprite aus einstellen möchten, müssen Sie es zunächst duplizieren, dann die Operation an dem duplizierten sprite durchführen und dieses verwenden. Ein dupliziertes sprite wird bei der Erstellung auf seiner eigenen Texturseite platziert, was bedeutet, dass das Duplizieren mehrerer sprites mehrere Texturseiten erzeugt und sich auf die Leistung auswirkt, daher sollten Sie diese Funktion nur bei Bedarf verwenden.

HINWEIS: Wenn Sie eine sprite in GameMaker duplizieren, müssen Sie daran denken, sie wieder zu entfernen (mit sprite_delete()), wenn du sie nicht mehr brauchst, sonst besteht die Gefahr eines Speicherlecks, das dein Spiel verlangsamt und schließlich zum Absturz bringt.

Syntax:

sprite_duplicate(index);

ArgumentTyp Beschreibung
index The index of the sprite to duplicate.

Rückgabe

Sprite Asset or -1

Beispiel:

new_spr = sprite_duplicate(sprite_index)

Die obige code dupliziert die sprite, die derzeit als sprite Index der Instanz verwendet wird, und speichert den Index für diese neue sprite in der Variablen "new_spr".
