sprite_nineslice_create

Nine Slicing ist eine Technik zur Skalierung von rechteckigen Bildern, um Details zu erhalten, so dass sie nach der Skalierung ihre ursprüngliche Form beibehalten. Auf dieser Seite finden Sie Informationen zu Nine Slice.

Mit dieser Funktion wird eine Nine Slice-Struktur erstellt, die geändert und dann auf eine sprite angewendet werden kann. Der Inhalt dieser Struktur wird auf dieser Seite detailliert beschrieben.

Sie können diese Struktur in einer Variablen speichern, ihre Eigenschaften ändern und sie mit sprite_set_nineslice() auf jede sprites anwenden.

Syntax:

sprite_nineslice_create();

Rückgabe:

Nine Slice Struct

Beispiel:

var _nineslice = sprite_nineslice_create();

_nineslice.enabled = true;
_nineslice.left = 10;
_nineslice.right = 10;
_nineslice.top = 10;
_nineslice.bottom = 10;

sprite_set_nineslice(spr_box_0, _nineslice);

Die obige code erstellt eine neue Nine-Slice-Struktur, aktiviert sie und legt die Offsets für die Hilfslinien fest. Die Struktur wird dann auf eine sprite angewandt, wobei ihre Nine-Slice-Eigenschaften geändert werden.
