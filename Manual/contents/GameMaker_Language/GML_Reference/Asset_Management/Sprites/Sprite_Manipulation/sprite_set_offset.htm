sprite_set_offset

Diese Funktion kann verwendet werden, um den x- und y-Ursprung eines sprite festzulegen, und nimmt relative Werte an, die auf der Position (0,0) basieren, die die obere linke Ecke des sprite ist. Das folgende Bild veranschaulicht dies:

HINWEIS Diese Funktion wirkt sich auf das Sprite-Asset aus, so dass alle weiteren Instanzen mit diesem sprite denselben Versatz haben.

Syntax:

sprite_set_offset(ind, xoff, yoff);

ArgumentTyp Beschreibung
ind The index of the sprite to change the offset of.
xoff The x position of the origin.
yoff The y position of the origin.

Rückgabe

Beispiel:

sprite_assign(spr_Custom, spr_Base);
sprite_set_offset(spr_Custom, sprite_get_xoffset(spr_Base), sprite_get_yoffset(spr_Base));

Die obige code weist die sprite, die in "spr_Base" indiziert ist, der sprite, die in "spr_Custom" indiziert ist, zu und verwendet dann die x- und y-Versatzwerte von "spr_Base", um den Ursprung des neuen Sprites festzulegen.
