sprite_merge

Mit dieser Funktion wird das in sprite 1 ("ind2") indizierte argument mit dem in argument 0 ("ind1") indizierten Bild zusammengeführt. Die Bilder selbst werden NICHT zusammengeführt, sondern die Bildindizes werden zusammengeführt, wobei die Teilbilder von sprite "ind2" an die von sprite "ind1" angehängt werden, d. h. sie werden am Ende hinzugefügt. Wenn die sprites unterschiedlich groß sind, werden die angehängten sprites gestreckt, um sie an die Bildgröße für "ind1" anzupassen.

Diese Änderung ist dauerhaft, und von dem Moment an, in dem Sie diese Funktion verwenden, bis das Spiel geschlossen oder die sprite gelöscht wird, wird die sprite, in die sie zusammengeführt wird, geändert, aber die sprite, mit der sie zusammengeführt wird, bleibt dieselbe.

HINWEIS: Weder das Bild-Asset, in das zusammengeführt wird, noch das Bild-Asset, das zusammengeführt werden soll, darf eine der ursprünglichen Spielressourcen sein. Sie müssen sprite_duplicate() zuerst verwenden, um zwei neue Ressourcen zu erstellen (wenn Sie enthaltene Spielressourcen zusammenführen) und diese dann zusammenführen.

HINWEIS: Diese Funktion ist nur für Bitmaps nützlich sprites und funktioniert nicht mit SWF oder JSON (Spine) sprites.

Syntax:

sprite_merge(ind1, ind2);

ArgumentTyp Beschreibung
ind1 The index of the sprite to merge.
ind2 The index of the sprite that ind1 is to be merged with.

Rückgabe

Beispiel:

var tspr;
tspr[0] = spr_add(working_directory + "Sprite Assets\explode1.png", 16, true, true, 0, 0);
tspr[1] = sprite_duplicate(spr_Explosion);
sprite_merge(tspr[0], tspr[1]);
sprite_index = t_spr[0];
sprite_delete(tspr[1]);

Die obige code lädt eine sprite in eine lokale Variable und verschmilzt sie mit einer anderen sprite, die dupliziert wurde. Schließlich wird die zusammengeführte sprite der Instanz sprite_index zugewiesen, während die geladene sprite entfernt wird entfernt wird, um den von ihr verwendeten Speicher freizugeben. Beachte, dass du irgendwann nach dieser Operation, wie bei den Ereignissen Destroy oder Room End, auch die Funktion sprite_delete() verwenden, um das zusammengeführte Sprite aus dem Speicher zu entfernen.
