Bild_x-Skala

Dieser Wert legt die horizontale Skalierung fest, die auf die sprite angewendet wird, die der aktuellen Instanz zugewiesen wurde. Eine Skalierung von 1 bedeutet keine Skalierung (1:1), kleinere Werte skalieren nach unten (0,5 zum Beispiel halbiert die Breite von sprite), größere Werte skalieren nach oben und negative Werte spiegeln sprite und skalieren es, es sei denn, der verwendete Wert ist genau -1 (in diesem Fall wird sprite nur um seinen Ursprung gespiegelt und nicht skaliert).

Syntax:

image_xscale;

Rückgabe:

(single precision floating point value)

Beispiel:

if image_xscale < 5
{
    image_xscale += 0.2;
    image_yscale = image_xscale;
}
else
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

Die obige code skaliert die sprite und sobald sie auf das 5-fache ihrer ursprünglichen Größe skaliert ist, wird eine neue Instanz eines anderen Objekts erstellt und die Instanz zerstört.
