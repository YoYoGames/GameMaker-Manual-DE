sprite_index

Diese Variable gibt den Index der aktuellen sprite für die Instanz zurück, oder -1, wenn die Instanz nicht mit sprite verknüpft ist. Du kannst sie ändern, um der Instanz einen anderen sprite zu geben, indem du ihr den Namen einer sprite aus dem Ressourcenbaum gibst oder indem du eine Variable verwendest, in der ein extern geladenes sprite indiziert ist. Das Ändern von sprite ändert nicht den Index des aktuell sichtbaren Frames. Wenn du also sprite auf Frame Nummer 3 änderst, wird das neue sprite mit diesem sichtbaren Frame gezeichnet (vorausgesetzt, es hat die gleiche Anzahl von Frames).

Syntax:

sprite_index;

Rückgabe:

Sprite Asset

Beispiel:

with (obj_Check)
{
    if !collision_line(x, y, other.x, other.y, obj_Wall, false, true)
    {
        sprite_index = spr_spotted;
    }
    else
    {
        sprite_index = spr_clear;
    }
}

code Die obige loop durchläuft alle Instanzen von "obj_Check" und prüft, ob eine Kollisionslinie zwischen ihnen und der Instanz, die code ausführt, besteht. Die sprite dieser Instanzen wird je nach dem Rückgabewert (true oder false) für die Kollisionslinie geändert.
