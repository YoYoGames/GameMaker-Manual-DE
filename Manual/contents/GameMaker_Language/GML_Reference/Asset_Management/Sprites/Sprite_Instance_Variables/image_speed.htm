image_speed

Diese Variable bestimmt die Geschwindigkeit, mit der GameMaker die Teilbilder für die aktuelle Instanz sprite durchläuft. Der angegebene Geschwindigkeitswert ist ein Multiplikator, wobei 1 der Standardwert ist und die Einstellung 0,5 die Animationsgeschwindigkeit - wie im Sprite-Editor oder Bild-Editor eingestellt - halbiert, während die Einstellung 2 sie verdoppelt. Wenn das verwendete sprite keine Unterbilder hat, hat diese Variable keine Auswirkung.

Syntax:

image_speed;

Rückgabe:

(single precision floating point value)

Beispiel:

with (instance_create_layer(x, y, "Effects", obj_Explosion))
{
    image_speed = 0.5;
}

Die obige code erstellt eine Instanz des Objekts "obj_Explosion" und setzt dessen image_speed auf 0,5 (wodurch die Laufgeschwindigkeit effektiv halbiert wird).
