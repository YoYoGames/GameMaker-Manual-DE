mask_index

Wenn Sie ein Objekt in GameMaker definieren, können Sie ihm eine Maske zuweisen, die für Kollisionen verwendet wird und nicht die, die dem definierten sprite entspricht. Diese Variable kann verwendet werden, um die sprite_index dieser Maske zu finden (oder es wird -1 zurückgegeben, wenn keine sprite zugewiesen wurde) oder die Maske für eine Instanz auf die gewählte sprite zu setzen. Das Setzen des Maskenindexes bedeutet, dass man z.B. eine sprite für die Instanz mit einer unregelmäßigen Form haben kann, ihr aber eine kreisförmige Kollisionsmaske geben kann, die von einer anderen sprite bezogen wird.

Syntax:

mask_index

Rückgabe:

Sprite Asset

Beispiel:

mask_index = spr_Round;

Die obige code setzt die Maske der Instanz auf die der sprite "spr_Round".
