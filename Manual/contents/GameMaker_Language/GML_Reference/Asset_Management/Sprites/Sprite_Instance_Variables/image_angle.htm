Bild_Winkel

Dieser Wert legt den Winkel (Drehung) der sprite fest und wird in Grad gemessen, wobei rechts 0º, oben 90º, links 180º und unten 270º ist. Setzen Sie diese Variable auf 0, um die sprite so zu zeichnen, wie sie im sprite Editor definiert wurde. Bitte beachte, dass die Instanz entweder kein Draw-Ereignis haben sollte (und GameMaker daher standardmäßig die sprite zeichnet) oder mit einer der erweiterten Zeichenfunktionen gezeichnet werden sollte, damit Änderungen in dieser Variablen sichtbar werden. draw_self() oder draw_sprite_ext().

Syntax:

image_angle;

Rückgabe:

Real

Beispiel:

image_angle = point_direction(x, y, mouse_x, mouse_y);

Die obige code wird die sprite der Instanz so drehen, dass sie immer auf die Mausposition zeigt.
