image_blend

Diese Variable steuert die "Tönung" der Instanz sprite und der Standardwert ist -1 (kann aber auch c_white sein). Jeder andere Wert (einschließlich interner Farbkonstanten wie c_red oder c_aqua) wird die angegebene Farbe mit dem Original sprite mischen. Bitte beachte, dass die Instanz entweder kein Draw-Ereignis haben sollte (und GameMaker daher standardmäßig die sprite zeichnet) oder mit einer der erweiterten Zeichenfunktionen wie draw_self() oder draw_sprite_ext().

Bitte beachten Sie, dass Sie versuchen sollten, die Überblendung auf HTML5 platform zu begrenzen (es sei denn, Sie verwenden WebGL), da jede überblendete sprite separat zwischengespeichert werden muss und daher viele überblendete sprites die Leistung beeinträchtigen können (Sie können die Größe von cache auch mit der Funktion sprite_set_cache_size()).

Syntax:

image_blend

Rückgabe:

Colour Constant

Beispiel:

image_blend = make_colour_hsv(255, 255, random(255));

Die obige code setzt die image_blend der für die Instanz definierten sprite auf eine zufällige Farbe.
