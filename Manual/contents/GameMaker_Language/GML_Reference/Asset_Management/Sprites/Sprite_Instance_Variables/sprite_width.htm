sprite_width

Diese schreibgeschützte Variable gibt die Breite der sprite zurück, die der Instanz zugewiesen wurde. Diese Breite wird in Pixel zurückgegeben und ist abhängig von der image_xscale. Wenn Sie die nicht skalierte Breite benötigen, sollten Sie sprite_get_width().

Syntax:

sprite_width

Rückgabe:

Real

Beispiel:

if sprite_width != sprite_get_width(sprite_index)
{
    image_xscale = 1;
}

Die obige code prüft die Breite der sprite, die der Instanz zugewiesen ist, auf der code läuft, mit der Breite der sprite -Ressource, und wenn sie nicht übereinstimmt, setzt sie die image_xscale auf 1 zurück,
