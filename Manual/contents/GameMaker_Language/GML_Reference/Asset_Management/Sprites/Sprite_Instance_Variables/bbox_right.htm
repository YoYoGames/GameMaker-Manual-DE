bbox_right

Diese schreibgeschützte Variable gibt die Position innerhalb des Raums (entlang der x-Achse) der rechten Seite des Begrenzungsrahmens für die Instanz zurück, wobei der Begrenzungsrahmen durch die maximale Breite und Höhe der Maske für die Instanz definiert ist (wie von sprite_index oder mask_index festgelegt). Selbst wenn eine sprite eine genaue Kollisionsmaske hat, existiert der Begrenzungsrahmen und wird für bestimmte Dinge benutzt, und so kannst du diese Variable benutzen, um ihn zu finden. Bitte beachte, dass wenn der Instanz keine sprite zugewiesen ist, der zurückgegebene Wert derselbe ist wie die X-Position der Instanz.

Syntax:

bbox_right;

Rückgabe:

(integer)

Beispiel:

if bbox_right > room_width
{
    x = room_width - sprite_xoffset;
}

Die obige code prüft, ob der Begrenzungsrahmen der Instanz außerhalb des Raumes liegt, und wenn ja, setzt sie die Instanz an eine neue Position.
