sprite_xoffset

Diese schreibgeschützte Variable gibt den lokalen X-Offset (die X-Komponente des Ursprungs, wie im sprite Editor definiert) der sprite zurück, die der Instanz zugewiesen wurde. Dieser xOffset wird in Pixeln zurückgegeben und ist abhängig von der image_xscaleWenn also (zum Beispiel) ein quadratisches 32x32 sprite so definiert ist, dass der x-Ursprung bei 16 liegt, wird diese Variable 32 zurückgeben, wenn sprite um 2 skaliert wird. Wenn Sie den nicht skalierten x-Offset benötigen, sollten Sie sprite_get_xoffset().

Syntax:

sprite_xoffset;

Rückgabe:

Real

Beispiel:

if x - sprite_xoffset < 0
{
    x = sprite_xoffset;
}

Die obige code prüft, ob die linke Kante der sprite der Instanz außerhalb des Raumes liegt, und wenn dies der Fall ist, wird die Instanz so verschoben, dass die gesamte sprite sichtbar ist.
