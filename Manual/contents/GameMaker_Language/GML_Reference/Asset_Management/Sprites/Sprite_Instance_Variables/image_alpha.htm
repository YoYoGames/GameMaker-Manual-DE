image_alpha

Diese Variable wird verwendet, um den Alphawert für die sprite zu erhalten oder zu setzen. Alpha wird immer als Wert zwischen 0 und 1 berechnet, wobei 0 vollständig transparent und 1 vollständig undurchsichtig ist. Bitte beachte, dass Änderungen in dieser Variablen nur sichtbar werden, wenn die Instanz entweder kein Draw-Ereignis hat (und GameMaker daher standardmäßig die sprite zeichnet) oder mit einer der erweiterten Zeichenfunktionen gezeichnet wird wie draw_self() oder draw_sprite_ext().

Syntax:

image_alpha;

Rückgabe:

Real

Beispiel:

image_alpha = clamp(image_alpha - 0.01, 0, 1);

Mit der obigen code wird image_alpha langsam reduziert, bis es 0 erreicht.
