sprite_yoffset

Diese schreibgeschützte Variable gibt den lokalen yOffset (die y-Komponente des Ursprungs, wie im sprite Editor definiert) der sprite zurück, die der Instanz zugewiesen wurde. Dieser yOffset wird in Pixeln zurückgegeben und ist abhängig von der image_yscaleWenn also (zum Beispiel) ein quadratisches 32x32 sprite so definiert ist, dass der y-Ursprung bei 16 liegt, wird diese Variable 32 zurückgeben, wenn sprite um 2 skaliert wird. Wenn Sie den nicht skalierten y-Offset benötigen, sollten Sie sprite_get_yoffset().

Syntax:

sprite_yoffset;

Rückgabe:

Real

Beispiel:

if y - sprite_yoffset < 0
{
    y = sprite_yoffset;
}

Die obige code prüft, ob die Oberkante der sprite der Instanz außerhalb des Raumes liegt, und wenn ja, verschiebt sie die Instanz so, dass die gesamte sprite sichtbar ist.
