sprite_height

Diese schreibgeschützte Variable gibt die Höhe der sprite zurück, die der Instanz zugewiesen wurde. Diese Höhe wird in Pixel zurückgegeben und ist abhängig von der image_yscale. Wenn Sie die nicht skalierte Höhe benötigen, sollten Sie sprite_get_height().

Syntax:

sprite_height;

Rückgabe:

Real

Beispiel:

if sprite_height != sprite_get_height(sprite_index)
{
    image_yscale = 1;
}

Die obige code prüft die Höhe der sprite, die der Instanz zugewiesen ist, auf der code läuft, mit der Höhe der sprite -Ressource, und wenn sie nicht übereinstimmt, setzt sie die image_yscale auf 1 zurück,
