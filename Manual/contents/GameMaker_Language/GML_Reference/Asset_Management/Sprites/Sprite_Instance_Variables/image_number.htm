image_number

Diese schreibgeschützte Variable kann verwendet werden, um die Anzahl der Unterbilder in einer sprite zu ermitteln, die einer Instanz zugewiesen wurde (wenn Sie die Anzahl der Unterbilder für eine sprite benötigen, die nicht der Instanz zugewiesen ist, sollten Sie sprite_get_number()). Bitte beachte, dass diese Variable "1" zurückgibt, wenn es (zum Beispiel) 1 Unterbild gibt, aber der image_index dieses Unterbildes 0 ist.

Syntax:

image_number

Rückgabe:

(single precision floating point value)

Beispiel:

if image_speed > 0
{
    if image_index > image_number - 1 instance_destroy();
}

Die obige code prüft, ob die sprite animiert ist, und wenn ja, dann prüft sie, ob der aktuelle image_index größer ist als die Anzahl der Unterbilder, und wenn ja, zerstört sie die Instanz.
