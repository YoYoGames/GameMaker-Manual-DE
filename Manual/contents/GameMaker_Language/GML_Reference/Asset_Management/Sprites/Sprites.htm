Sprites

Sprites sind im Allgemeinen die visuellen Darstellungen von Objekten in dem von Ihnen erstellten Spiel, obwohl sie auch für viele andere Zwecke verwendet werden können. Ein sprite ist entweder ein einzelnes Bild oder eine Reihe von Bildern, die, wenn sie nacheinander abgespielt werden, wie eine Schleifenanimation aussehen. Sprites kann in vielen Formaten erstellt werden, wobei das "traditionelle" sprite aus Einzelbildern von Pixelbildern besteht, die in GameMaker als *.png Dateien importiert werden. Sie können aber auch Vektorgrafiken sprites importieren, die im Format *.swf gespeichert wurden, oder Skelettanimationen sprites, die als *.json Dateien mit begleitenden *.atlas und *.png Dateien gespeichert wurden (weitere Informationen zu diesen Arten von Bilddateien und wie man sie in GameMaker importiert, finden Sie im Abschnitt Importieren von Nicht-Bitmap-Sprites).

Da sprites ein integraler Bestandteil jedes Spiels ist, verfügt GameMaker über eine große Anzahl von Funktionen und Variablen, die dazu verwendet werden können, Informationen über bestimmte sprites abzurufen, die im Spiel vorhandenen zu ändern oder sogar eigene sprites unter runtime aus einer Quellbilddatei zu laden.
