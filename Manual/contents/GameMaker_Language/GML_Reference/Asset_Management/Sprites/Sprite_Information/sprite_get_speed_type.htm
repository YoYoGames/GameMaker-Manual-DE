sprite_get_speed_type

Diese Funktion kann verwendet werden, um den sprite Animationstyp abzurufen, wie er für die sprite Ressource im Sprite-Editor definiert ist. Der zurückgegebene Wert ist eine der folgenden Konstanten:

Sprite-Geschwindigkeitskonstante
Konstante Beschreibung
spritespeed_framespersecond Die sprite wurde mit einer Animation in Bildern pro Sekunde definiert.
spritespeed_framespergameframe Die sprite wurde mit einer Animation in Frames pro Spielframe definiert.

Die folgenden zwei Beispiele veranschaulichen den Unterschied, den dies bei der Berechnung der Animationsgeschwindigkeiten für das Sprite machen kann:

  • Wenn Sie ein sprite haben, das 1 Bild pro Sekunde zeichnet, und die Bildgeschwindigkeit auf 0,5 einstellen, wird es mit 0,5 Bildern pro Sekunde zeichnen. Wenn die Bildrate Ihres Spiels 60 Bilder pro Sekunde beträgt, zeichnet sprite 1 Bild pro 120 Spielbilder.
  • Wenn Sie ein sprite haben, das 1 Bild pro Spielbild zeichnet, und die Bildgeschwindigkeit auf 0,5 einstellen, wird es 0,5 Bilder pro Spielbild zeichnen. Wenn Ihre Spielbildrate 60 Bilder pro Sekunde beträgt, zeichnet sprite 30 Bilder pro 60 Spielbilder.

Den Geschwindigkeitswert für die Animation sprite können Sie mit der Funktion sprite_get_speed()herausfinden und die Geschwindigkeit und den Typ der Animation mit der Funktion sprite_set_speed().

Syntax:

sprite_get_speed_type(index)

Argument Typ Beschreibung
index Sprite-Asset The index of the sprite to get the animation type of

Rückgabe:

Sprite Speed Constant (see above)

Beispiel:

s_speed = sprite_get_speed(sprite_index);
s_type = sprite_get_speed_type(sprite_index);

Die obige code ruft die sprite Geschwindigkeit und den sprite Animationstyp ab und speichert sie in Instanzvariablen für die zukünftige Verwendung.
