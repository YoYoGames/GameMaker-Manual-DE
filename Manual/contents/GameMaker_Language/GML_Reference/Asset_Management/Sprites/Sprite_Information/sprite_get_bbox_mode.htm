sprite_get_bbox_mode

Diese Funktion gibt den aktuellen "Modus" für die Bounding-Box-Berechnungen zurück. Sie geben den sprite Index der zu prüfenden sprite an, und die Funktion gibt eine der unten aufgeführten Konstanten zurück.

Syntax:

sprite_get_bbox_mode(ind);

ArgumentTyp Beschreibung
ind The index of the sprite to check.

Rückgabe

Bounding Box Mode Constant

Konstante Beschreibung
bboxmode_automatic Automatisch - Der Begrenzungsrahmen wird automatisch berechnet, basierend auf der Toleranzeinstellung für das Sprite
bboxmode_fullimage Volles Bild - Der Begrenzungsrahmen wird so eingestellt, dass er die volle Breite und Höhe des sprite verwendet, unabhängig von der Toleranz und den "leeren" Pixeln.
bboxmode_manual Manuell - Der Begrenzungsrahmen wurde manuell auf benutzerdefinierte Werte gesetzt (entweder im sprite Editor oder mit der Funktion sprite_set_bbox())

Beispiel:

if sprite_get_bbox_mode(sprite_index) != 0
{
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

Die obige Funktion prüft den Bbox-Modus für die aktuelle sprite und wenn er nicht automatic ist, wird er auf diesen Wert gesetzt.
