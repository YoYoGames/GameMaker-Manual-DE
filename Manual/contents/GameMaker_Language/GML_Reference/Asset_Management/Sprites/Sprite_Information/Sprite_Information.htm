Sprite-Informationen

Die folgenden Funktionen können verwendet werden, um Informationen über ein sprite Asset zu erhalten. Bitte beachten Sie, dass diese Funktionen die Informationen von der Basis sprite (d.h. dem sprite Asset, wie es im Asset Browser erstellt wurde) abrufen und nicht von sprite, wie es in einer Instanz verwendet wird. Zum Beispiel können Sie die sprite Instanz-Variable sprite_width verwenden, um die Breite der sprite zu ermitteln, die von einer Instanz verwendet wird, und diese Breite berücksichtigt alle Skalierungen, die möglicherweise vorgenommen wurden. Wenn Sie jedoch die Breite von sprite wissen wollen, wie sie normalerweise wäre, ohne angewandte Transformationen, müssen Sie die folgende Funktion sprite_get_width() verwenden.

Weitere Informationen zu den sprite Instanzvariablen finden Sie im Abschnitt über Instanzvariablen.
