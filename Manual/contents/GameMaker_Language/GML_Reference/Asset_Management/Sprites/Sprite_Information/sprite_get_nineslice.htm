sprite_get_nineslice

Diese Funktion gibt die Nine-Slice-Struktur für ein sprite zurück, die alle Nine-Slice-Eigenschaften enthält, die im Sprite-Editor oder mit sprite_set_nineslice() gesetzt wurden. Der Inhalt dieser Struktur wird auf dieser Seite detailliert beschrieben.

Wenn der übergebenen sprite keine Nine-Slice-Struktur zugewiesen ist, wird eine neue Struktur mit Standard-Nine-Slice-Eigenschaften für die sprite erstellt und zurückgegeben.

Wenn Sie die Werte in dieser Struktur ändern , werden die Nine Slice-Eigenschaften des ursprünglichen sprite geändert, was sich auf alle zukünftigen Zeichenaufrufe mit diesem sprite auswirkt.

Syntax:

sprite_get_nineslice(ind);

ArgumentTyp Beschreibung
ind The index of the sprite from which the Nine Slice struct will be retrieved

Rückgabe:

Nine Slice Struct (or -1 if the sprite doesn't exist)

Beispiel:

var _box_nineslice = sprite_get_nineslice(spr_box_0);

_box_nineslice.enabled = true;
_box_nineslice.left = 10;
_box_nineslice.right = 10;
_box_nineslice.top = 10;
_box_nineslice.bottom = 10;

Die obige code ruft die Nine-Slice-Struktur von einer sprite ab, aktiviert Nine Slicing für sie und setzt die Führungsoffsets.
