sprite_get_yoffset

Wenn Sie ein sprite im sprite Editor definieren, müssen Sie den Ursprung für dieses sprite festlegen. Dies ist der Punkt, an dem die sprite "befestigt" oder "gezeichnet" wird, wenn sie mit einer Instanz verwendet wird. Diese Funktion gibt den relativen Versatz für die y-Achse des Ursprungs zurück, basierend auf der oberen linken Ecke, die die Position 0,0 ist, wobei ein +y nach unten und ein -y nach oben von diesem Punkt aus bedeutet. Das folgende Bild veranschaulicht dies:

Syntax:

sprite_get_yoffset(index);

ArgumentTyp Beschreibung
index The index of the sprite to find the yoffset of.

Rückgabe

Beispiel:

if y - sprite_get_yoffset(sprite_index) < 0
{
    y = sprite_get_yoffset(sprite_index);
}

Die obige code stellt sicher, dass eine Instanz innerhalb des Raumes auf der Grundlage der sprite beibehalten wird (so dass die sprite immer sichtbar ist).
