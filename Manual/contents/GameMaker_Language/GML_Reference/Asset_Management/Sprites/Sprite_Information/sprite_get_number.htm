sprite_get_number

Ein sprite muss mindestens ein Unterbild haben, und diese Funktion gibt zurück, wie viele es hat. Denken Sie daran, dass die Nummerierung eines sprite Unterbildes bei 0 beginnt. Wenn diese Funktion also 3 zurückgibt (das überprüfte sprite hat 3 Unterbilder), werden diese mit den Nummern 0, 1 und 2.

Syntax:

sprite_get_number(index);

ArgumentTyp Beschreibung
index The index of the sprite.

Rückgabe

Beispiel:

image_index = floor(random(sprite_get_number(sprite_index)));

Die obige code setzt den Bildindex auf ein zufälliges Bild, das auf der Anzahl der Unterbilder basiert, die die sprite hat.
