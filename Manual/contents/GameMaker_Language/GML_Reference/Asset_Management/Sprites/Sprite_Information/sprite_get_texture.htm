sprite_get_texture

Diese Funktion gibt einen speziellen Zeiger für die Texturseite sprite zurück. Dieser Wert kann dann in anderen Zeichenfunktionen verwendet werden, insbesondere in den 2D-Primitivfunktionen sowie in den Shader-Funktionen. Sie können weitere Informationen über die zurückgegebene Texturseite erhalten, indem Sie die verschiedenen texture_ Funktionen verwenden, die Sie hier finden.

Syntax:

sprite_get_texture(spr, subimg);

Argument Typ Beschreibung
spr Sprite-Asset The index of the sprite to use.
subimg Real The sub-image of the sprite to use.

Rückgabe

Texture

Beispiel:

var tex;
tex = sprite_get_texture(spr_Wall, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(480, 0, 1, 0);
draw_vertex_texture(480, 640, 1, 1);
draw_vertex_texture(0, 640, 0, 1);
draw_primitive_end();

Die obige code zeichnet einen dreieckigen Streifen mit 4 Eckpunkten, der mit der Textur in der Variable "tex" texturiert ist.
