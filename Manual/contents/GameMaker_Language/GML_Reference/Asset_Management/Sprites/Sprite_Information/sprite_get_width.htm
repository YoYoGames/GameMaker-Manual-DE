sprite_get_width

Mit dieser Funktion können Sie die Breite der Basisressource sprite ohne Transformationen in Pixel ermitteln.

Syntax:

sprite_get_width(index);

ArgumentTyp Beschreibung
index The index of the sprite to find the width of.

Rückgabe

Beispiel:

if sprite_width != sprite_get_width(sprite_index)
{
    image_xscale = 1;
}

Die obige code prüft die Breite der sprite, wie sie in der aktuellen Instanz ist, und wenn es einen Unterschied zwischen dieser und der ursprünglichen Basis sprite gibt, wird die Skala der x-Achse zurückgesetzt.
