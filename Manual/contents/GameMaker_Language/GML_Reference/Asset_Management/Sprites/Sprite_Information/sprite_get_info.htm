sprite_get_info

Diese Funktion wird verwendet, um Informationen für das angegebene sprite abzurufen. Sie geben einen Index sprite an (das kann ein Asset sein, das über den Asset-Browser hinzugefügt wurde, oder ein Sprite , das unter runtimehinzugefügt wurde ) und die Funktion gibt eine Struktur mit den folgenden Variablen zurück:

Sprite-Info-Struktur
Variabel Typ Beschreibung
Breite real Die Breite des Sprites (in Pixel)
Höhe real Die Höhe des Sprites (in Pixel)
xOffset real Der X-Offset/Ursprung des Sprites (in Pixeln)
yOffset real Y-Offset/Ursprung des Sprites (in Pixel)
transparent boolean true wenn die sprite als transparent markiert ist, ansonsten false
(Dies kann nur durch sprite_add() oder ähnliche Funktionen angegeben werden und wird false für sprites im IDE erstellt)
glatt boolean true wenn die sprite als glatt markiert ist, andernfalls false
(Dies kann nur über sprite_add() oder ähnliche Funktionen angegeben werden und wird false für sprites im IDE erstellt)
Typ real Der Typ des Sprites:
0 - Bitmap (Regelmäßig sprites)
1 - SWF
2 - Wirbelsäule
bbox_left real Position des linken Randes des Begrenzungsrahmens (in Pixel)
bbox_top real Position der oberen Kante des Begrenzungsrahmens (in Pixel)
bbox_right real Position des rechten Randes des Begrenzungsrahmens (in Pixel)
bbox_bottom real Position des unteren Randes des Begrenzungsrahmens (in Pixel)
Name String Der Name des Sprites
num_subimages real Die Anzahl der Teilbilder (oder Frames) im Sprite
verwenden_maske boolean true wenn diese sprite eine Kollisionsmaske verwendet, andernfalls false
num_masks real Die Anzahl der Masken in dieser sprite (ist größer als 1, wenn sprite Masken pro Bild verwendet)
Neuner-Scheibe Struktur Die Nine-Slice-Struktur für diese sprite, oder undefined, wenn sie keine Nine-Slice-Daten hat
Nachrichten Array Array von Broadcast-Nachrichten für diese sprite, wobei jede Broadcast-Nachricht eine Struktur ist, die Informationen über die Nachricht enthält (weitere Informationen unter "Allgemeine Sprite Daten")
rahmen_info Array Array von Frames für dieses sprite, wobei jedes Frame ein struct ist, das Informationen über sein Timing enthält (weitere Informationen unter "Allgemeine Sprite Daten")

Diese zusätzliche Variable ist nur für Bitmap (reguläre) Sprites verfügbar:

Sprite-Info-Struktur
Variabel Typ Beschreibung
Rahmen Array Array von Frames für diese sprite, wobei jeder Frame eine Struktur ist, die Informationen über seine Textur enthält (weitere Informationen unter "Bitmap Sprite Data")

Diese zusätzlichen Variablen sind nur für Spine sprites verfügbar:

Sprite-Info-Struktur
Variabel Typ Beschreibung
num_atlas real Die Anzahl der verwendeten Atlas-Texturen
atlas_textur Array Array von Textur-IDs, die für den Atlas verwendet werden
vormultipliziert boolean true wenn diese sprite als vormultipliziert gekennzeichnet ist, sonst false
animation_names Array Array mit den Namen der einzelnen Animationen in diesem Sprite
skin_names Array Array mit den Namen der einzelnen Skins in diesem Sprite
Knochen Array Array mit Strukturen für jeden Knochen in diesem Sprite (weitere Informationen unter "Spine Sprite Data")
Steckplätze Array Array mit Structs für jeden Slot in diesem Sprite (weitere Informationen unter "Spine Sprite Data")

Die Funktion gibt undefined zurück, wenn die angegebene sprite nicht existiert. Beachten Sie auch, dass die in dieser Struktur zurückgegebenen Informationen als schreibgeschützt betrachtet werden sollten, da die Änderung einer dieser Variablen keine Auswirkungen auf sprite hat.

Die folgenden Abschnitte enthalten Informationen über die Arrays und Strukturen, die in der zurückgegebenen Struktur enthalten sind, basierend auf dem Typ sprite:

Allgemeine Daten Sprite

Dieser Abschnitt enthält Informationen über Variablen, die in der Struktur für alle Arten von Sprites enthalten sind.

Die Variable messages ist ein Array, das Informationen über die Broadcast-Nachrichten enthält, die in der angegebenen sprite vorhanden sind. Jede Broadcast-Nachricht in diesem Array ist eine Struktur, die die folgenden Variablen enthält:

Variable Name Datenart Beschreibung
Rahmen real Das Timing dieser Broadcast-Nachricht ab dem Start der Animation (in Frames)
Nachricht String Die Zeichenkette der Broadcast-Nachricht

Die Variable frame_info ist ein Array, das Informationen über das Timing der Frames des Sprites enthält. Jedes Bild in diesem Array ist eine Struktur, die die folgenden Variablen enthält:

Variable Name Datenart Beschreibung
Rahmen real Die Zeit für den Beginn dieses Rahmens (in Rahmen)
Dauer real Die Dauer dieses Frames (in Frames)
bild_index real Der Bildindex dieses Bildes

Bitmap Sprite Daten

Dieser Abschnitt enthält Informationen über Variablen, die in der Struktur für Bitmap sprites (sprites ) enthalten sind, die nicht Spine oder SWF-Typ sprites sind.

Die Variable frames ist ein Array, das Informationen über die Texturen der Frames des Sprites enthält. Jedes Bild in diesem Array ist eine Struktur, die die folgenden Variablen enthält:

Variable Name Datenart Beschreibung
x real Die X-Position dieses Rahmens auf seiner Texturseite (in Pixel)
y real Die Y-Position dieses Rahmens auf seiner Texturseite (in Pixel)
w real Die logische Breite des Rahmens (in Pixeln), die intern verwendet wird
h real Die logische Höhe des Rahmens (in Pixeln), die intern verwendet wird
Textur real Die Texturseiten-ID für diesen Rahmen
ursprüngliche_Breite real Die ursprüngliche Breite des Rahmens (in Pixel)
ursprüngliche_Höhe real Die ursprüngliche Höhe des Rahmens (in Pixel)
beschneiden_breite real Die tatsächliche Breite des Rahmens auf der Texturseite nach dem Zuschneiden und Skalieren (da GameMaker automatisch den leeren Raum um ein Bild herum beschneidet und es auch verkleinert, wenn es nicht passt)
crop_height real Die tatsächliche Höhe des Rahmens auf der Texturseite nach dem Zuschneiden und Skalieren
x_Offset real Der X-Versatz von der linken Kante des Originalbildes zur linken Kante des zugeschnittenen Ausschnitts
y_Offset real Der Y-Versatz von der Oberkante des Originalbildes zur Oberkante des zugeschnittenen Ausschnitts

Wirbelsäule Sprite Daten

Dieser Abschnitt enthält Informationen zu den Variablen, die in der Struktur für Spine sprites enthalten sind.

Die Variable bones ist ein Array, das Informationen über alle Bones in der angegebenen Spine sprite enthält. Jeder Bone in diesem Array ist eine Struktur, die die folgenden Variablen enthält:

Variable Name Datenart Beschreibung
Elternteil String Der Name des übergeordneten Bones, oder undefined, wenn dieser Bone keinen übergeordneten Bone hat
Name String Der Name dieses Knochens
Index real Der Index dieses Knochens
Länge real Die Länge dieses Knochens
x real Die X-Position dieses Knochens
y real Die Y-Position dieses Knochens
Rotation real Die Drehung dieses Knochens
skala_x real (Intern auf Spine) Skalenwert auf X
skala_y real (Intern auf Spine) Skalenwert auf Y
scheren_x real (Intern auf Spine) Scherwert auf X
scheren_y real (Intern auf Spine) Scherwert auf Y
transform_mode real (Intern auf Spine) Der Transformationsmodus

HINWEIS: Weitere Informationen zu den internen Variablen von Spine entnehmen Sie bitte der Spine-Dokumentation.

Die Variable slots ist ein Array, das Informationen über alle Slots in der angegebenen Spine sprite enthält. Jeder Slot in diesem Array ist eine Struktur, die die folgenden Variablen enthält:

Variable Name Datenart Beschreibung
Name String Der Name des Steckplatzes
Index real Der Index des Steckplatzes
Knochen String Der Name des Bones des Steckplatzes, oder "(none)", wenn es keinen Bone für diesen Steckplatz gibt
Anhang String Name des Anhängers
rot real Rotanteil der Farbe des Steckplatzes (0-1)
grün real Grünanteil der Farbe des Steckplatzes (0-1)
blau real Blauanteil der Farbe des Steckplatzes (0-1)
alpha real Alphakomponente der Farbe des Steckplatzes (0-1)
blendMode real (Intern auf Spine) Überblendungsmodus für den Steckplatz
dunkel_rot* real Rotanteil der dunklen Farbe des Schlitzes (0-1)
dunkel_grün* real Grünanteil der dunklen Farbe des Schlitzes (0-1)
dunkel_blau* real Blauanteil der dunklen Farbe des Schlitzes (0-1)
dark_alpha* String Alphakomponente der dunklen Farbe des Schlitzes (0-1)

* HINWEIS: Die dunklen Farbvariablen sind nur verfügbar, wenn auf dem Steckplatz Tint Black aktiviert ist.

Syntax:

sprite_get_info(index);

Argument Typ Beschreibung
index Sprite-Asset The index of the sprite to get the information for.

Rückgabe

Sprite Info Struct (or undefined)

Beispiel:

var _info = sprite_get_info(sprite_index);

if (_info != undefined && _info.type == 0)
{
    var _messages = _info.messages;
    var _messageCount = array_length(_messages);
    
    for (var i = 0; i < _messageCount; i ++)
    {
        var _message = _messages[i];
        show_debug_message("Message at frame " + string(_message.frame) + ": " + _message.message);
    }
}

Die obige code ruft die Info-Struktur für die Instanz sprite ab und prüft dann, ob sie nicht undefined ist und ob ihr Typ 0 ist (was bedeutet, dass sie eine Bitmap sprite ist). In diesem Fall wird das Array mit den Broadcast-Nachrichten für sprite abgerufen und dann ein for loop ausgeführt, um jede Broadcast-Nachricht (zusammen mit ihrem Frame) in das Ausgabeprotokoll zu drucken.
