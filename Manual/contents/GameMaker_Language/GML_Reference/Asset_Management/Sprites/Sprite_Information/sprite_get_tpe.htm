sprite_get_tpe

Diese Funktion gibt einen speziellen Wert zurück, der sich auf die Position des angegebenen sprite und des Unterbildes auf der von Ihrem Spiel verwendeten Texturseite bezieht. Auf diese Weise können Sie den Wert an Erweiterungen übergeben, die außerhalb des kompilierten Spiels verwendet werden sollen, wobei die GameMaker-Zeichenfunktionen effektiv umgangen werden und die sprite in DOM-Inhalten verwendet werden kann, die dann an beliebiger Stelle innerhalb des Fensters gezeichnet werden können, das die Spieloberfläche enthält. Diese Funktion ist vor allem für diejenigen interessant, die Schaltflächen und andere interaktive Medien außerhalb des GameMaker-Canvas-Elements auf ihrer Host-Seite erstellen möchten, indem sie die Funktion clickable_add().

HINWEIS: Diese Funktion ist nur für HTML5 geeignet.

Syntax:

sprite_get_tpe(sprite, index);

Argument Typ Beschreibung
sprite Sprite-Asset The index of the sprite to find the texture page entry of.
index Real The image index of the sprite.

Rückgabe

Texture Page Entry

Beispiel:

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

Die obige code erstellt eine DOM-Schaltfläche an der Position (32, 32) der Seite, auf der das Game Canvas ausgeführt wird. Die Schaltfläche verwendet die sprite, die von der Textur-Seite als "spr_MS_Home" referenziert wird, und wenn sie angeklickt wird, öffnet sich ein neues Fenster für die angegebene URL und mit den definierten Eigenschaften für dieses Fenster.
