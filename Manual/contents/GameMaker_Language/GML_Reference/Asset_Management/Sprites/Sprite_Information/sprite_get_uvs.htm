sprite_get_uvs

Diese Funktion gibt ein Array mit den UV-Koordinaten und anderen Daten für die Textur des Unterbildes sprite auf der Texturseite zurück. Die Funktion gibt ein Array mit den folgenden 8 Elementen zurück:

  • [0] = links
  • [1] = oben
  • [2] = rechts
  • [3] = unten
  • [4] = Anzahl der Pixel, die das Asset compiler von der linken Seite von sprites abgeschnitten hat
  • [5] = Anzahl der Pixel, die das Asset compiler von der Oberseite von sprites abgeschnitten hat
  • [6] = normalisierter Prozentsatz der Pixeldaten der ursprünglichen Breite sprites, die auf der Texturseite gespeichert wurden
  • [7] = normalisierter Prozentsatz der Pixeldaten der ursprünglichen Höhe von sprites, die auf der Texturseite gespeichert wurden

Dieses Array kann dann in anderen Zeichenfunktionen verwendet werden, insbesondere bei der Arbeit in 3D oder mit den 2D-Primitivfunktionen sowie bei der Arbeit mit den Shader-Funktionen.

HINWEIS: Diese Funktion funktioniert nicht mit vector sprites oder Skelettanimation sprites.

Syntax:

sprite_get_uvs(sprite, subimage);

ArgumentTyp Beschreibung
sprite The index of the sprite to use.
subimage The sub-image of the sprite to use.

Rückgabe

(1D, 8 elements)

Beispiel:

var tex = sprite_get_uvs(sprite, 0);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_bottom = tex[3];

Die obige code speichert die UV-Koordinaten für die angegebene sprite in einem lokalen Array und weist die Werte dann den Instanzvariablen zu.
