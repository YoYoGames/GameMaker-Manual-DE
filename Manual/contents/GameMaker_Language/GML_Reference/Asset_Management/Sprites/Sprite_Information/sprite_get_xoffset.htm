sprite_get_xoffset

Wenn Sie ein sprite im sprite Editor definieren, müssen Sie den Ursprung für dieses sprite festlegen. Dies ist der Punkt, an dem die sprite "angehängt" oder "gezeichnet" wird, wenn sie mit einer Instanz verwendet wird. Diese Funktion gibt den relativen Versatz für die x-Achse des Ursprungs zurück, basierend auf der oberen linken Ecke, die die Position 0,0 ist, wobei ein +x rechts und ein -x links davon ist. Das folgende Bild veranschaulicht dies:

Syntax:

sprite_get_xoffset(index)

ArgumentTyp Beschreibung
index The index of the sprite to find the xoffset of.

Rückgabe

Beispiel:

if x - sprite_get_xoffset(sprite_index) < 0
{
    x = _xoffset;
}

Die obige code stellt sicher, dass eine Instanz innerhalb des Raumes auf der Grundlage der sprite beibehalten wird (so dass die sprite immer sichtbar ist).
