sprite_get_name

Diese Funktion gibt den Namen des angegebenen spriteals String zurück. Dieser Name ist derjenige, der für das sprite im Ressourcenbaum des GameMaker-Hauptfensters angegeben wurde. Bitte beachten Sie, dass dies nur eine Zeichenkette ist und nicht verwendet werden kann, um das sprite direkt zu referenzieren - dazu bräuchten Sie den Sprite-Index. Du kannst diese Zeichenkette jedoch verwenden, um den Sprite-Index zu erhalten, indem du die zurückgegebene Zeichenkette zusammen mit der Funktion asset_get_index().

Syntax:

sprite_get_name(index);

Argument Typ Beschreibung
index Vermögen The index of the sprite to get the name of.

Rückgabe

String

Beispiel:

str = sprite_get_name(sprite_index);

Die obige code ermittelt den Namen des sprite Index für die Instanz, auf der code läuft, und speichert den Rückgabewert in der Variablen "str".
