Instanz_am-weitesten

Diese Funktion prüft alle Instanzen des angegebenen Objekts, um zu sehen, welche am weitesten von dem angegebenen x/y-Punkt entfernt ist. Alle Prüfungen erfolgen von der angegebenen x/y-Position zum Ursprung (der x/y-Position) der Instanzen des angegebenen Objekts. Wenn keine Instanzen des Objekts existieren, gibt die Funktion das Schlüsselwort noonezurück, aber wenn es Instanzen gibt, dann gibt sie die id der gefundenen Instanz zurück. Bitte beachten Sie, dass die Instanz, die code ausführt, in die Prüfung einbezogen wird, wenn sie den gleichen Objektindex wie das zu prüfende Objekt hat (dies schließt Prüfungen für übergeordnete Objekte ein, wenn die aufrufende Instanz auch ein Kind des übergeordneten Objekts ist).

Syntax:

instance_furthest(x, y, obj);

ArgumentTyp Beschreibung
x The x position to check for instances far from.
y The y position to check for instances far from.
obj The object to check for instances of.

Rückgabe:

Instance ID or noone

Beispiel:

var inst;
inst = instance_furthest(x, y, object_index);
if (inst != id)
{
    draw_line(x, y, inst.x, inst.y);
}

Die obige code findet die am weitesten entfernte Instanz, die den gleichen Objektindex hat wie die Instanz, die die code ausführt, und speichert die ID in einer Variablen. Diese Variable wird dann daraufhin überprüft, ob sie mit der ID der aufrufenden Instanz übereinstimmt, und wenn dies nicht der Fall ist, wird eine Linie zwischen den beiden Instanzen gezogen.
