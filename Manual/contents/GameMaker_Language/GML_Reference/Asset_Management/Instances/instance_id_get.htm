instance_id_get

Mit dieser Funktion können Sie den eindeutigen ID-Wert einer beliebigen Instanz aus der derzeit aktiven Instanzliste abrufen. Sie geben den Index in der Instanzliste an, von dem Sie die ID abrufen möchten, und die Funktion gibt den Wert zum Speichern in einer Variablen zurück.

Syntax:

instance_id_get(index);

Argument Typ Beschreibung
index Real The index within the instance list from 0 auf Instanzanzahl - 1.

Rückgabe:

Instance ID

Beispiel:

for (var i = 0; i < instance_count; ++i;)
{
    var temp_id = instance_id_get(i);
    with (temp_id)
    {
        speed += 0.1;
    }
}

code Die obige loop wird durch alle Instanzen innerhalb des Raumes gehen und 0,1 zu ihrer Geschwindigkeit hinzufügen.
