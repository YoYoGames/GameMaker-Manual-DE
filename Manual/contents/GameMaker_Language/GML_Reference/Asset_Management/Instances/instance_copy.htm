Instanz_Kopie

Mit dieser Funktion können Sie eine Instanz "klonen", da dadurch eine neue Version der Instanz erstellt wird, die die code an derselben Position ausführt. Die Funktion "perf" argument wird verwendet, um diese neue Instanz anzuweisen, das Ereignis "create" auszuführen oder nicht. Diese Funktion gibt die id der neuen Instanz zurück, die dann in einer Variablen gespeichert oder für den Zugriff auf diese Instanz verwendet werden kann.

HINWEIS: Wenn Sie das Ereignis create nicht ausführen, können Fehler auftreten, wenn die Instanz von Variablen abhängt, die in diesem Ereignis initialisiert wurden.

Syntax:

instance_copy(perf);

Argument Typ Beschreibung
perf Boolesche Whether to perform the new instance's Create event (true) or not (false).

Rückgabe:

Instance ID

Beispiel:

var inst = instance_number(object_index);
if (inst < 10)
{
    instance_copy(true);
}

Die obige code erstellt eine lokale Variable und verwendet sie, um die Anzahl der Instanzen des Objekts zu speichern, das die code im Raum ausführt. Wenn die Zahl kleiner als 10 ist, erstellt die Instanz eine Kopie von sich selbst.
