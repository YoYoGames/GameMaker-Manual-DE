instance_id

Dieses schreibgeschützte Array enthält alle idaller aktiven Instanzen des Raums. Das bedeutet, dass wenn du eine der Instance Deactivate Funktionen benutzt hast, werden die Instanzen die deaktiviert wurden nicht in diesem Array enthalten sein (wenn du vorher einen Wert aus diesem Array benutzt hast, wird es jetzt das Schlüsselwort noone).

Syntax:

instance_id[num];

Rückgabe:

Instance ID

Beispiel:

for (var i = 0; i < instance_count; i ++;)
{
    with (instance_id[i]) speed += 0.1;
}

code Die obige loop wird durch alle Instanzen innerhalb des Raumes gehen und 0,1 zu ihrer Geschwindigkeit hinzufügen.
