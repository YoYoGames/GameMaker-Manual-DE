instanz_position

Mit dieser Funktion können Sie eine Position auf eine Kollision mit einer anderen Instanz oder allen Instanzen eines Objekts prüfen. Wenn Sie diese Funktion verwenden, prüfen Sie einen einzelnen Punkt im Raum auf eine Instanz oder ein Objekt. Diese Prüfung wird gegen die Bounding Box der Instanz oder gegen die Maske der Instanz durchgeführt, wenn diese Instanz präzise Kollisionen geprüft hat und gibt die eindeutige Instanz zurück id. Wenn du die ID der kollidierenden Instanz nicht benötigst, solltest du stattdessen die Funktion position_meeting() stattdessen verwenden. Diese Funktion akzeptiert auch die speziellen Schlüsselwörter all und other und gibt das Schlüsselwort noone zurück, wenn keine Kollision auftritt oder den eindeutigen ID-Wert der gefundenen Instanz, wenn eine Kollision auftritt.

Syntax:

instance_position( x, y, obj );

Argument Typ Beschreibung
x Real The x position to check for instances.
y Real The y position to check for instances.
obj Objekt Asset- oder Instanz-ID The object to check for instances of.

Rückgabe:

Instance ID or noone

Beispiel:

var inst;
inst = instance_position(mouse_x, mouse_y, obj_Pause_Button);
if (inst != noone)
{
    with (inst)
    {
        image_index = 1;
    }
    instance_create_layer(room_width / 2, 0, "Controllers", obj_Menu);
}

Die obige code prüft, ob eine Kollision mit einer Instanz von "obj_Pause_Button" an der Mausposition vorliegt, und wenn dies der Fall ist, wird die zurückgegebene ID verwendet, um ihren image_index auf einen neuen Wert zu setzen, bevor eine neue Instanz des Objekts "obj_Menu" erstellt wird.
