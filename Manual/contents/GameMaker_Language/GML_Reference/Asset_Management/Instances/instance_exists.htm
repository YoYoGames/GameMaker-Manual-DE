instance_exists

Diese Funktion kann auf zwei Arten verwendet werden, je nachdem, was Sie überprüfen möchten. In diesem Fall gibt die Funktion true zurück, wenn aktive Instanzen des angegebenen Objekts im aktuellen Raum vorhanden sind. Sie können ihr auch eine Instanz-ID übergeben, dann gibt die Funktion true zurück, wenn diese spezielle Instanz im aktuellen Raum vorhanden und aktiv ist.

Bitte beachten Sie, dass diese Funktion nicht die Instanzen berücksichtigt, die mit den Funktionen zum Deaktivieren von Instanzen deaktiviert wurden.

Syntax:

instance_exists(obj)

Argument Typ Beschreibung
obj Instanz-ID oder Objekt Asset The object or instance to check for the existence of.

Rückgabe:

Boolean

Beispiel:

if !instance_exists(obj_Enemy)
{
    score += 200;
    room_goto(rm_hiscores);
}

Die obige code prüft, ob es Instanzen des Objekts "obj_Enemy" gibt, und wenn nicht, fügt es der Variablen "score" hinzu und ändert den Raum.
