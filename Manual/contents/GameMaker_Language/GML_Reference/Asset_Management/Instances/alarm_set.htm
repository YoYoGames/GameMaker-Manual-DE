alarm_set

Mit dieser Funktion können Sie einen Alarm einstellen. Sie geben die Alarmnummer von 0 bis 11 und dann den Wert an, auf den der Alarm gesetzt werden soll. Der Wert muss eine ganze Zahl sein, und Sie können ihn auf -1 setzen, um den Alarm zu beenden (nicht ganzzahlige Werte werden auf die nächste ganze Zahl gerundet). Dies ist eine alternative Methode zum direkten Setzen des Alarmfeldes.

Syntax:

alarm_set(index, value);

Argument Typ Beschreibung
index Real The alarm index to set, from 0 to 11.
value Real The value (an integer) to set the alarm to.

Rückgabe:

N/A

Beispiel:

for (var i = 0; i < 12; i++)
{
    if alarm_get(i) > 0 alarm_set(i, -1);
}

Die obige code prüft alle Alarme in der aufrufenden Instanz, und wenn sie größer als 0 sind, werden sie auf -1 gesetzt, so dass sie nicht mehr weiter herunterzählen.
