alarm_get

Diese Funktion kann verwendet werden, um den aktuellen Wert des angegebenen Alarms zu ermitteln. Sie geben die Alarmnummer von 0 bis 11 an und diese Funktion gibt den Wert zurück, den der Alarm gerade hat. Dies ist eine alternative Methode, um den Wert des Alarm-Arrays direkt abzurufen.

Syntax:

alarm_get(index);

Argument Typ Beschreibung
index Real The alarm index to get, from 0 to 11.

Rückgabe:

Real (integer)

Beispiel:

for (var i = 0; i < 12; i++)
{
    if alarm_get(i) > 0 alarm_set(i, -1);
}

Die obige code prüft alle Alarme in der aufrufenden Instanz, und wenn sie größer als 0 sind, werden sie auf -1 gesetzt, so dass sie nicht mehr weiter herunterzählen.
