nächstgelegene Instanz

Diese Funktion prüft alle Instanzen des angegebenen Objekts, um zu sehen, welche dem angegebenen x/y-Punkt am nächsten ist. Alle Prüfungen erfolgen von der angegebenen x/y-Position zum Ursprung (der x/y-Position) der Instanzen des angegebenen Objekts. Wenn keine Instanzen des Objekts existieren, gibt die Funktion das Schlüsselwort noonezurück, aber wenn es Instanzen gibt, dann gibt sie die id der gefundenen Instanz zurück. Bitte beachten Sie, dass die Instanz, die code ausführt, in die Prüfung einbezogen wird, wenn sie den gleichen Objektindex wie das zu prüfende Objekt hat (dies schließt Prüfungen für übergeordnete Objekte ein, wenn die aufrufende Instanz auch ein Kind des übergeordneten Objekts ist).

Syntax:

instance_nearest(x, y, obj);

ArgumentTyp Beschreibung
x The x position to check from.
y The y position to check from.
obj The object to check for instances of.

Rückgabe:

Instance ID or noone

Beispiel:

var inst, xx;
xx = x;
x -= 10000;
inst = instance_nearest(xx, y, object_index);
if inst != id
{
    draw_line(x, y, inst.x, inst.y);
}
x += 10000;

Die obige code verschiebt die aktuelle Instanz um 10000 Pixel und überprüft dann ihre vorherige Position, um die nächstgelegene Instanz desselben Objekttyps zu finden. Wenn es sich dabei um die gleiche Instanz handelt, wird nichts weiter getan, als die ursprüngliche Position wieder einzunehmen, aber sollte die gefundene Instanz eine andere sein, wird eine Linie zwischen den beiden gezogen.
