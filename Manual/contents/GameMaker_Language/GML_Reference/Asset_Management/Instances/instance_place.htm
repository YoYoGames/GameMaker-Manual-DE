Instanz_Ort

Mit dieser Funktion können Sie eine Position auf eine Kollision mit einer anderen Instanz oder allen Instanzen eines Objekts prüfen, indem Sie die Kollisionsmaske der Instanz verwenden, die die code für die Prüfung ausführt. Wenn du diese Funktion verwendest, bittest du GameMaker, die Instanz an die neue Position zu bewegen, auf eine Kollision zu prüfen, zurückzufahren und dir zu sagen, ob eine Kollision gefunden wurde oder nicht.

Dies funktioniert bei präzisen Kollisionen, aber nur, wenn sowohl für die Instanz als auch für das zu prüfende Objekt präzise Kollisionsmasken ausgewählt sind, da sonst nur Bounding-Box-Kollisionen angewendet werden.

Diese Funktion gibt die eindeutige Instanz id des kollidierenden Objekts zurück, aber wenn das nicht benötigt wird, ist es etwas schneller, die Funktion place_meeting(). Diese Funktion akzeptiert auch die speziellen Schlüsselwörter all und other und gibt das Schlüsselwort noone zurück, wenn keine Kollision auftritt, oder den eindeutigen Instanz-ID-Wert der gefundenen Instanz, wenn eine Kollision auftritt.

Syntax:

instance_place(x, y, obj);

Argument Typ Beschreibung
x Real The x position to check for instances.
y Real The y position to check for instances.
obj Objekt Asset- oder Instanz-ID The object to check for instances of.

Rückgabe:

Instance ID or noone

Beispiel:

var _inst = instance_place(x, y, obj_Enemy);
if (_inst != noone)
{
    hp -= _inst.dmg;
    instance_destroy(_inst);
}

Die obige code prüft, ob eine Kollision mit Instanzen von "obj_Enemy" vorliegt, und wenn dies der Fall ist, wird die "hp"-Variable um den Betrag reduziert, der in der "dmg"-Variable der kollidierenden Instanz gespeichert ist, und dann die kollidierende Instanz zerstört.
