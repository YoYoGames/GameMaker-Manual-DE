instanz_platz_liste

Mit dieser Funktion können Sie eine Position auf eine Kollision mit allen Instanzen eines Objekts prüfen, indem Sie die Kollisionsmaske der Instanz verwenden, die die code für die Prüfung ausführt. Wenn du diese Funktion verwendest, bittest du GameMaker, die Instanz an die neue Position zu bewegen, auf eine Kollision zu prüfen, dann zurückzugehen und dir zu sagen, ob eine Kollision gefunden wurde oder nicht. Dies funktioniert bei präzisen Kollisionen, aber nur, wenn sowohl die Instanz als auch die Instanzen des zu prüfenden Objekts präzise Kollisionsmasken ausgewählt haben, ansonsten werden nur Bounding-Box-Kollisionen angewendet.

Sie geben auch eine DS-Liste an, so dass die id Werte aller Instanzen, die mit der aufrufenden Instanz kollidieren, an das Ende der angegebenen Liste angehängt werden. Sie können die Liste vor dem Aufruf dieser Funktion löschen, so dass sie nur die Ergebnisse dieses Funktionsaufrufs enthält. Du hast auch die Möglichkeit, die Liste nach dem Abstand zwischen dem Ursprung der Instanz, die die Prüfung durchführt, und dem Ursprung der Instanzen, bei denen eine Kollision festgestellt wurde, zu ordnen.

Beachten Sie, dass die Funktion auch das spezielle Schlüsselwort allakzeptiert. In diesem Fall werden alle Instanzen aufgelistet, die in Kollision gefunden wurden. Die Funktion gibt die Anzahl der gefundenen Instanzen zurück, oder 0, wenn keine gefunden wurden.

Syntax:

instance_place_list(x, y, obj, list, ordered);

Argument Typ Beschreibung
x Real The x position to check for instances.
y Real The y position to check for instances.
obj Objekt Asset- oder Instanz-ID The object to check for instances of.
list DS-Liste ID The DS list to use to store the IDs of colliding instances.
ordered Boolesche Whether the list should be ordered by distance (true) oder nicht (false).

Rückgabe:

Real (The number of instances found to be in collision)

Beispiel:

var _list = ds_list_create();
var _num = instance_place_list(x, y, obj_Enemy, _list, false);

if (_num > 0)
{
    for (var i = 0; i < _num; ++i)
    {
        instance_destroy(_list[| i]);
    }
}

ds_list_destroy(_list);

Die obige code prüft auf eine Kollision mit allen Instanzen, die an der Position der aufrufenden Instanz gefunden werden. Diese werden zu einer DS-Liste hinzugefügt, die dann in einer Schleife durchlaufen wird, um jede der kollidierenden Instanzen zu zerstören.
