Reibung

Friction ist eine der "eingebauten" Eigenschaften, die Instanzen haben können, und kann verwendet werden, um die Instanz mit der Zeit zu verlangsamen, wenn die speed anders als Null ist. Wenn die Reibung zum Beispiel auf 0,1 eingestellt ist und die Geschwindigkeit der Instanz 1 beträgt (1 Pixel pro Schritt), wird sie langsamer und bleibt nach 10 Schritten stehen. Beachten Sie auch, dass die Reibung auf positive und negative Geschwindigkeiten gleichermaßen angewandt wird, wobei das Ergebnis immer ist, dass das Objekt nach einer bestimmten Zeit eine Geschwindigkeit von 0 hat.

Syntax:

friction;

Rückgabe:

(single precision floating point value)

Beispiel:

if abs(speed) > 0
{
    friction = 0.05;
}
else
{
    friction = 0;
}

Die obige code wendet die Reibung nur an, wenn die absolute Geschwindigkeit der Instanz über 0 liegt.
