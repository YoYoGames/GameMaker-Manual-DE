solide

Eine Instanz kann über die Objekteigenschaften im Objekteditor oder durch Ändern des Werts dieser eingebauten Variablen als solide gekennzeichnet werden. Wenn solid auf true gesetzt ist, wird die kollidierende Instanz bei der Erkennung einer Kollision automatisch an die Position zurückgebracht, an der sie sich im Schritt vor der Kollision befand (und dann werden alle code oder Aktionen im Kollisionsereignis ausgeführt). Wenn es auf false gesetzt ist, müssen alle Positionierungen durch das Kollisionsereignis behandelt werden.

Syntax:

solid;

Rückgabe:

Beispiel:

if other.solid = true
{
    x = xprevious;
    y = yprevious;
}

Das obige code prüft das Solid-Flag der "anderen" Instanz bei einer Kollision und wenn es auf true gesetzt ist, wird diese Instanz zurück an ihre vorherige Position bewegt.
