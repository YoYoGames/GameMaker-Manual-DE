sichtbar

Eine Instanz kann als visible oder nicht gekennzeichnet werden, indem diese Variable auf true (sichtbar) oder false (unsichtbar) gesetzt wird. Dies funktioniert, indem GameMaker angewiesen wird, das Draw-Ereignis für diese Instanz zu überspringen. Daher ist Vorsicht geboten, wenn dies verwendet wird, da es bedeutet, dass keine code, die im Draw-Ereignis platziert ist, ausgeführt wird, während die Instanz nicht sichtbar ist. Beachte auch, dass wenn die layer der die Instanz zugewiesen ist, als unsichtbar gekennzeichnet ist, hat das Setzen dieser Variable auf true keine Wirkung, bis die Ebene selbst ebenfalls als sichtbar gekennzeichnet ist.

Syntax:

visible;

Rückgabe:

Beispiel:

if other.visible = true
{
    x = xprevious;
    y = yprevious;
}

Das obige code prüft das Sichtbarkeitsflag der "anderen" Instanz bei einem Kollisionsereignis und wenn es auf true gesetzt ist, verschiebt es diese Instanz zurück an ihre vorherige Position.
