x

Der x-Wert einer Instanz ist die horizontale Position im aktuellen Raum, gemessen in Pixeln. Dieser Wert kann entweder 0, positiv oder minus sein, wobei 0 die linke Seite des Raums ist und eine Bewegung nach rechts x erhöht, eine Bewegung nach links x verringert (ein negativer Wert für x bedeutet, dass die Instanz über die linke Seite des Raums hinausgegangen ist). Du kannst x sogar einen realen Wert wie 12.345 geben, aber da du nicht 0,345 Pixel bekommen kannst, scheint sich die Instanz nicht bewegt zu haben, obwohl diese Werte beibehalten werden und intern von GameMaker verwendet werden und vollkommen gültig sind.

Indem Sie die x- (und y-) Werte einstellen, können Sie das Objekt im Raum an die gewünschte Position springen lassen, oder Sie können kleinere Beträge zu den Werten addieren und subtrahieren, um die Illusion von Bewegung zu erzeugen, ohne die eingebauten speed und direction Funktionen. Dies sollte bei der Entwicklung Ihrer Spiele berücksichtigt werden, da sich eine bestimmte Art der Bewegung oft besser für einen bestimmten Spielstil eignet als eine andere.

Syntax:

x;

Rückgabe:

(single precision floating point value)

Beispiel:

if keyboard_check(vk_left) x += 5;
if keyboard_check(vk_right) x -= 5;

Die obige code verwendet die Pfeiltasten, um die x-Position zu addieren oder zu subtrahieren und so die Illusion einer Bewegung zu erzeugen.
