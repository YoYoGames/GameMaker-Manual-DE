id

Diese schreibgeschützte Variable enthält die eindeutige Identifikationsnummer für die Instanz. Jede Instanz, die Sie erstellen - sei es durch code oder durch Hinzufügen zu einem Raum im Raum-Editor - erhält eine Nummer, die intern verwendet wird, um diese Instanz zu identifizieren, und die Variable id ist das, was Sie verwenden können, um darauf zu verweisen. Die id wird auch zurückgegeben (und kann in einer Variablen gespeichert werden), wenn eine Instanz mit instance_create_layer() oder instance_create_depth() erstellt wird, ebenso wie andere Instanzfunktionen.

Diese Variable ist nun ein Verweis auf eine Instanz, anstatt einfach eine reale Zahl zu sein. Sie können die Instanz-ID-Nummer erhalten, indem Sie real() oder int64() für diese Variable aufrufen.

HINWEIS Der Wert dieser Variablen ist nicht derselbe wie der Bezeichner, den die Instanz im Raumeditor erhält, und unterscheidet sich auch von dem instance_id Array, das alle IDs aller derzeit aktiven Instanzen enthält.

Syntax:

id;

Rückgabe:

Instance ID

Beispiel:

for (var i = 0; i < instance_count; i++;)
{
    if (instance_id[i] != id)
    {
        instance_id[i].scr += 5;
    }
}

Die obige code fügt 5 zur "scr"-Variable für jede aktive Instanz im Raum hinzu, mit Ausnahme derjenigen, die die code ausführt. Sie tut dies, indem sie eine Schleife durch ALLE aktiven Instanzen durchläuft (unter Verwendung des "instance_id"-Arrays, um jede aktive Instanz-ID zurückzugeben) und sie mit der eingebauten "id"-Variable vergleicht, die die ID der ursprünglichen Instanz ist, die die code ausführt.
