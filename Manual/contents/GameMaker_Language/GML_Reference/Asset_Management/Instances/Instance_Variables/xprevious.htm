xprevious

Diese eingebaute Variable gibt die vorherige X-Position für die Instanz zurück. Diese Variable wird kurz vor dem Beginn des "begin step"-Ereignisses gesetzt, kann aber auch jederzeit über code gesetzt werden, d.h. Sie können ihr Ihren eigenen Wert geben, falls dies notwendig sein sollte.

Syntax:

xprevious;

Rückgabe:

(single precision floating point value)

Beispiel:

if x != xprevious || y != yprevious
{
    moved = true;
}

Die obige code prüft, ob es einen Unterschied zwischen den Werten xprevious und yprevious und den aktuellen x- und y-Werten gibt. Wenn dies der Fall ist, wird eine Variable auf true gesetzt.
