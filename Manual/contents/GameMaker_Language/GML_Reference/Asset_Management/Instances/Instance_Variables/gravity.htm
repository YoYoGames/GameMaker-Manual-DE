Schwerkraft

gravity ist eine der "eingebauten" Variablen, die alle Instanzen haben und die, wenn sie gesetzt ist, eine konstante Kraft auf die gravity_direction der Instanz und beeinflusst sowohl die Instanz speed als auch direction. Beachte, dass die Schwerkraft eine kumulative Kraft ist und das Objekt beschleunigt, wenn du die Endgeschwindigkeit nicht deckelst, und es ist üblich, dass du diese Variable auf kleine Dezimalwerte wie 0.01 setzt. Wenn du gravity auf 0 setzt, wird keine Schwerkraft auf die Instanz angewendet (dies ist der Standardwert).

Syntax:

gravity;

Rückgabe:

(single precision floating point value)

Beispiel:

if !place_meeting(x, y + 1, obj_Ground)
{
    gravity = 0.01;
}
else
{
    gravity = 0;
}

Die obige code wendet die Schwerkraft nur an, wenn die Instanz keine Instanzen von "obj_Ground" unter ihr findet.
