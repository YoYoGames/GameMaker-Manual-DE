Richtung

Alle Instanzen in GameMaker haben bestimmte "eingebaute" Eigenschaften, die Sie verwenden und einstellen können, um zu bestimmen, wie sie aussehen und sich verhalten. Direction ist eine dieser Eigenschaften und kann verwendet werden, um die Richtung der Bewegung der Instanz festzulegen, wenn die Instanz einen speed Beachten Sie, dass Richtungen in GameMaker in der Regel so berechnet werden, dass 0° rechts, 90° oben, 180° links und 270° unten ist, und dass die gravity und gravity_direction Variablen den Wert direction ändern können, wenn sie in deinen Spielen verwendet werden.

Syntax:

direction;

Rückgabe:

(single precision floating point value)

Beispiel:

if keyboard_check(vk_left) direction += 5;
if keyboard_check(vk_right) direction -= 5;

Die obige code ändert die Bewegungsrichtung der Instanz je nachdem, welche Taste (links oder rechts) gedrückt wird.
