Alarm

Dieses eindimensionale Array wird verwendet, um den aktuellen Wert für alle Alarme, die die Instanz haben kann, zu erhalten, oder es kann verwendet werden, um diese Alarme zu setzen. Es gibt zwölf Alarme, die in jede Instanz eines Objekts eingebaut sind, und jeder hat seine eigene event die ausgelöst wird, wenn diese Variable den Wert 0 erreicht.

Alarme werden zu Beginn jedes Schritts heruntergezählt, nach dem Ereignis "Schritt beginnen", aber vor dem Hauptereignis "Schritt", und bevor irgendetwas für das Bild gerendert wird. Das heißt, wenn Sie einen Alarm im Ereignis "Schritt beginnen" auf 1 setzen, wird sein Ereignis im selben Frame ausgeführt (da der Alarm nach dem Ereignis "Schritt beginnen" auf 0 heruntergezählt wird), aber wenn Sie einen Alarm im Ereignis "Schritt" auf 1 setzen, wird er im nächsten Frame ausgeführt.

Es sollte beachtet werden, dass der Alarm nicht beendet ist, wenn er 0 erreicht (obwohl das Ereignis ausgelöst wurde), da er im nächsten Schritt auf -1 zurückgeht. Wenn Sie also einen Alarm aus irgendeinem Grund stoppen müssen, sollten Sie seinen Array-Wert auf -1 und nicht 0 setzen. Die Alarmzeiten werden in Spielschritten berechnet, wobei ein Wert von 30 für 30 Schritte steht, 60 für 60 Schritte, usw..

HINWEIS Ein Alarm, der keine Aktionen oder code enthält, wird nicht ausgeführt. Aber auch wenn nur ein Kommentar und kein code angegeben ist, wird der Alarm heruntergezählt.

Sie können das Alarm-Array direkt setzen (und den aktuellen Alarmwert auch direkt über das Array abrufen), aber unter bestimmten Umständen ist dies nicht die geeignetste Methode, daher können Sie auch die Funktion alarm_set() verwenden, um einen Alarm zu setzen, und die Funktion alarm_get() um den Wert eines Alarms zu ermitteln.

Syntax:

alarm[val];

Rückgabe:

(-1 if the alarm isn't running)

Beispiel:

if canshoot
{
    if (keyboard_check_pressed(vk_space))
    {
        canshoot = false;
        alarm[0] = room_speed;
        instance_create_layer(x, y, "Bullets", obj_Bullet);
    }
}

Die obige code prüft eine Variable, und wenn sie wahr ist, prüft sie auf einen Tastendruck, und wenn dieser wahr ist, setzt sie die Variable schließlich auf false, setzt einen Alarm (wo sie wieder auf true gesetzt wird) und erzeugt eine Instanz.
