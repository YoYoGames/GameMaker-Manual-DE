gravity_direction

gravity_direction ist eine der "eingebauten" Eigenschaften aller Instanzen und kann verwendet werden, um die Richtung der Bewegung festzulegen, wenn die Instanz gravity Beachten Sie, dass Richtungen in GameMaker normalerweise so berechnet werden, dass 0° rechts, 90° oben, 180° links und 270° unten ist.

Syntax:

gravity_direction

Rückgabe:

(single precision floating point value)

Beispiel:

if place_meeting(x, y, obj_switch)
{
    gravity_direction += 180;
}

Die obige code wird die gravity_direction ändern, wenn das Player-Objekt auf eine Instanz von "obj_switch" trifft.
