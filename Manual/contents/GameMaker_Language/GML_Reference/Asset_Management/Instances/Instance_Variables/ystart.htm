ystart

Diese Variable speichert die anfängliche y-Position der Instanz, wenn sie zum ersten Mal im Raum angelegt wird. Dies ist keine schreibgeschützte Variable und kann sowohl gesetzt als auch gelesen werden.

Syntax:

ystart;

Rückgabe:

(single precision floating point value)

Beispiel:

if place_meeting(x, y, obj_spike)
{
    score -= 100;
    x = xstart;
    y = ystart;
}

Die obige code prüft, ob eine Kollision mit einer Instanz von "obj_spike" vorliegt, und wenn dies der Fall ist, werden 100 Punkte abgezogen und die Instanz wird zurück an ihre Ausgangsposition verschoben.
