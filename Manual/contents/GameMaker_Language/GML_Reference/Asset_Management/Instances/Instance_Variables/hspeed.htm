hspeed

hspeed ist eine der "eingebauten" Eigenschaften, die alle Instanzen haben und definiert die horizontale Bewegungsgeschwindigkeit (entlang der x-Achse) der Instanz in Pixeln pro Schritt. Also, ein hspeed von 3 bedeutet 3 Pixel Bewegung nach rechts (+x) pro Schritt, und ein hspeed von -3 würde 3 Pixel Bewegung nach links (-x) pro Schritt bedeuten.

Beachten Sie, dass, wenn Sie die Variablen speed und/oder direction setzen, der Wert hspeed automatisch aktualisiert wird, um diese Änderungen aktualisiert wird, um diese Änderungen widerzuspiegeln, und ebenso wirkt sich eine Änderung des Werts hspeed auch auf die Werte speed und direction aus. Beachten Sie auch, dass gravity, gravity_direction und friction den Wert dieser Variablen ändern können, wenn sie in Ihren Spielen verwendet werden.

Syntax:

hspeed;

Rückgabe:

(single precision floating point value)

Beispiel:

if keyboard_check(vk_left) hspeed = -5;
if keyboard_check(vk_right) hspeed = 5;

Die obige code ändert die horizontale Geschwindigkeit in Abhängigkeit davon, welche Tasten gedrückt werden.
