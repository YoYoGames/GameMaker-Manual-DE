instance_find

Alle Instanzen haben eine eindeutige Kennung (id), mit der sie während des Spiels verändert und manipuliert werden können. Da man aber nicht immer weiß, wie die ID einer bestimmten Instanz lautet, kann diese Funktion helfen, indem sie alle Instanzen durchläuft, um das zu finden, was man braucht. Du gibst das Objekt, dessen Instanz du finden willst, und eine Zahl an, und wenn es eine Instanz an dieser Position in der Instanzliste gibt, gibt die Funktion die ID dieser Instanz zurück, und wenn nicht, gibt sie das spezielle Schlüsselwort noone zurück. Du kannst auch das Schlüsselwort all verwenden, um durch alle Instanzen in einem Raum zu iterieren, sowie ein Elternobjekt, um durch alle Instanzen zu iterieren, die Teil dieser Eltern/Kind-Hierarchie sind, und du kannst sogar eine Instanz (wenn du ihre id hast) als Prüfung angeben, um zu sehen, ob sie tatsächlich im aktuellen Raum existiert. Bitte beachten Sie, dass es keine bestimmte Reihenfolge gibt, in der die Instanzen von dieser Funktion geprüft werden, da die Instanzen willkürlich sortiert werden und jede Instanz an jeder beliebigen Position stehen kann.

Der Höchstwert für "n" in dieser Funktion wäre

  • Für das Schlüsselwort all: instance_count - 1
  • Für einen Objektindex: instance_number(OBJ) - 1

Syntax:

instance_find(obj, n);

Argument Typ Beschreibung
obj Objekt Anlage The object to find the nth instance of
n Real The number of the instance to find.

Rückgabe:

Instance ID or noone

Beispiel:

for (var i = 0; i < instance_number(obj_Enemy); ++i;)
{
    enemy[i] = instance_find(obj_Enemy,i);
}

Die obige code verwendet eine for loop, um durch alle Instanzen von "obj_Enemy" zu iterieren und ihre ID im Array "enemy[]" zu speichern.
