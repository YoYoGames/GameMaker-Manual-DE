instance_deactivate_region

Mit dieser Funktion können Sie einen Bereich innerhalb des Raumes definieren, in dem Sie zuvor aktivierte Instanzen deaktivieren können. Dieser Bereich kann entweder als "innen" oder "außen" gekennzeichnet werden, wie in der folgenden Abbildung gezeigt:

In der obigen Abbildung ist zu sehen, dass die Instanz "Apfel" immer inaktiv ist, denn auch wenn die sprite selbst die Region nicht überlappt, so tut es doch der Begrenzungsrahmen. Instanzen werden also als innerhalb der angegebenen Region befindlich betrachtet, wenn sich ihr Begrenzungsrahmen mit ihr überschneidet, und der Zustand der Kollisionsmaske (präzise oder nicht) wird nicht berücksichtigt. Beachten Sie, dass die Deaktivierung nicht sofort erfolgt und eine Instanz, die auf diese Weise deaktiviert wurde, erst nach dem Ende des Ereignisses, in dem die Funktion aufgerufen wurde, als inaktiv betrachtet wird.

HINWEIS: Wenn Sie eine Instanz beim Start des Raums deaktivieren (d.h. aus der Raumerstellung code oder aus einem Instanzerstellungsereignis einer Instanz innerhalb des Raums), werden alle Instanzen, die aus dem Raumeditor im Raum platziert werden , weiterhin ihr Erstellungsereignis ausführen, bevor sie deaktiviert werden.

WARNUNG: Das Deaktivieren von Instanzen, bei denen die Physik aktiviert ist, verhindert NICHT, dass deren Einbauten innerhalb der Physiksimulation interagieren. Dafür solltest du ihre phy_active Variable auf true oder false setzen, wenn Sie die Instanzen aktivieren/deaktivieren.

Syntax:

instance_deactivate_region(left, top, width, height, inside, notme);

Argument Typ Beschreibung
left Real The x coordinate of the left of the rectangular region to deactivate.
top Real The y coordinate of the top of the rectangular region to deactivate.
width Real The width of the region to deactivate.
height Real The height of the region to deactivate.
inside Boolesche Whether to deactivate instances on the inside of the region (true) or the outside (false).
notme Boolesche Whether to exclude the calling instance from deactivation (true) or not (false).
collision_space Collision Space  The collision space where instances will be deactivated

Rückgabe:

N/A

Beispiel:

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_deactivate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false, false);

Die obige code aktiviert alle Instanzen und deaktiviert dann einen Bereich innerhalb des Raums.
