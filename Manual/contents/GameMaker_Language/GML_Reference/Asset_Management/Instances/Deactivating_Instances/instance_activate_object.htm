instance_activate_object

Mit dieser Funktion können Sie eine einzelne Instanz oder alle Instanzen eines bestimmten Objekts aus allen zuvor deaktivierten Instanzen aktivieren. Beachten Sie, dass Sie, wenn Sie eine Instanz oder ein Objekt, das als persistent gekennzeichnet ist, deaktiviert haben Wenn Sie eine Instanz oder ein Objekt, das als persistent gekennzeichnet ist, deaktiviert haben, müssen Sie es mit dieser Funktion erneut aktivieren, bevor Sie den Raum wechseln, da es sonst nicht übernommen und stattdessen verworfen wird. Beachte auch, dass die Aktivierung nicht sofort erfolgt, und eine Instanz, die auf diese Weise aktiviert wurde aktivierte Instanz wird erst nach dem Ende des Ereignisses, in dem die Funktion aufgerufen wurde, als aktiv betrachtet.

Syntax:

instance_activate_object(obj);

ArgumentTyp Beschreibung
obj The object or instance to activate (the keyword all can also be used).

Rückgabe:

Beispiel:

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_deactivate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false, false);
instance_activate_object(obj_Lights);

Die obige code aktiviert alle Instanzen innerhalb des Raums und deaktiviert dann diejenigen, die sich außerhalb der Grenzen der aktuellen Kameraansicht befinden, mit Ausnahme des Objekts "obj_Lights", das am Ende wieder aktiviert wird.
