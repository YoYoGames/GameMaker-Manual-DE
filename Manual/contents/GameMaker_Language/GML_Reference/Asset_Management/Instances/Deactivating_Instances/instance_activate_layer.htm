instance_activate_layer

Mit dieser Funktion können Sie eine Ebene aktivieren, die zuvor deaktiviert wurde. Sie müssen die Layer-ID angeben, die entweder der Name des Layers sein kann, wie er im code Editor geschrieben wurde (als String), oder der tatsächliche Wert der Layer-ID, wie er von der Funktion layer_create() und alle deaktivierten Instanzen auf dieser Ebene werden wieder aktiviert. Beachten Sie, dass Sie, wenn Sie eine Ebene deaktiviert haben, die persistente Instanzen hat, die Ebene mit dieser Funktion wieder aktivieren müssen, bevor Sie den Raum wechseln, da sonst alle persistenten Instanzen nicht übernommen werden und verworfen werden. Beachte auch, dass die Aktivierung nicht sofort erfolgt, und dass eine Instanz, die auf diese Weise aktiviert wurde, erst nach dem Ende des Ereignisses, in dem die Funktion aufgerufen wurde, als aktiv betrachtet wird.

Syntax:

instance_activate_layer(layer_id);

Argument Typ Beschreibung
layer_id Ebenen-ID oder String The layer name string (or ID value) to be used

Rückgabe:

N/A

Beispiel:

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
instance_deactivate_region(view_xview[0] - 64, view_yview - 64, _vx + 128, _vy + 128, false, false);
instance_activate_layer("Player_Layer");

Die obige code aktiviert alle Instanzen innerhalb des Raumes und deaktiviert dann die Instanzen, die sich außerhalb der Grenzen der aktuellen Kameraansicht befinden, mit Ausnahme der Instanzen auf der Ebene "Player_Layer", die am Ende wieder aktiviert werden.
