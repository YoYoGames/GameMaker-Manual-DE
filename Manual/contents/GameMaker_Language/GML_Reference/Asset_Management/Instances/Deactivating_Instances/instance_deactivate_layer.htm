instance_deactivate_layer

Mit dieser Funktion können Sie alle Instanzen deaktivieren, die einem bestimmten Layer zugeordnet sind. Sie müssen die Layer-ID angeben, die entweder der Name des Layers sein kann, wie er im code Editor geschrieben wurde (als String), oder der tatsächliche Layer-ID-Wert, wie er von der Funktion layer_create()Beachten Sie, dass Sie mit dieser Funktion nur Instanz-Ebenen deaktivieren können. Wenn Sie eine Ebene deaktiviert haben, die Instanzen von Objekten enthält, die als persistent gekennzeichnet sind, müssen Sie die Ebene vor dem Raumwechsel erneut mit der Funktion instance_activate_layer() reaktivieren, bevor Sie den Raum wechseln, da sonst alle persistenten Instanzen auf der Ebene nicht übernommen werden und verworfen werden. Beachten Sie auch, dass die Deaktivierung nicht sofort erfolgt und dass eine Instanz, die auf diese Weise deaktiviert wurde, erst nach dem Ende des Ereignisses, in dem die Funktion aufgerufen wurde, als inaktiv gilt.

HINWEIS: Wenn Sie eine Instanz beim Start des Raums deaktivieren (d.h. aus der Raumerstellung code oder aus einem Instanzerstellungsereignis einer Instanz innerhalb des Raums), werden alle Instanzen, die aus dem Raumeditor im Raum platziert werden , weiterhin ihr Erstellungsereignis ausführen, bevor sie deaktiviert werden.

WARNUNG: Das Deaktivieren von Instanzen, bei denen die Physik aktiviert ist, verhindert NICHT, dass deren Einbauten innerhalb der Physiksimulation interagieren. Dafür solltest du ihre phy_active Variable auf true oder false setzen, wenn Sie die Instanzen aktivieren/deaktivieren.

Syntax:

instance_deactivate_layer(obj);

Argument Typ Beschreibung
layer Ebenen-ID oder String The layer name string (or ID value) to be used

Rückgabe:

N/A

Beispiel:

instance_deactivate_layer("Enemy Layer");
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

Die obige code deaktiviert alle Instanzen, die dem Layer "Enemy_Layer" zugeordnet sind, und aktiviert dann eine Region innerhalb des Raumes.
