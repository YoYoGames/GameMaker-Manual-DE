Instanz_Nummer

Mit dieser Funktion können Sie herausfinden, wie viele aktive Instanzen des angegebenen Objekts im Raum vorhanden sind. Bei der Prüfung mit dieser Funktion werden, wenn das Objekt ein übergeordnetes Objekt ist, auch alle untergeordneten Objekte in den Rückgabewert einbezogen, und beachten Sie auch, dass die Instanzen, die mit den Instanzdeaktivierungsfunktionen deaktiviert wurden, nicht in diese Prüfung einbezogen werden.

Syntax:

instance_number(obj);

ArgumentTyp Beschreibung
obj The object to total the number of instances of.

Rückgabe:

Beispiel:

if instance_number(object_index) < 50
{
    instance_create_layer(random(room_width), random(room_height), "Instances", object_index);
}

Die obige code prüft die Anzahl der Instanzen, die aus demselben Objekt wie die aktuelle Instanz erstellt werden, und wenn es weniger als 50 sind, wird eine weitere Instanz an einer zufälligen Position im Raum erstellt.
