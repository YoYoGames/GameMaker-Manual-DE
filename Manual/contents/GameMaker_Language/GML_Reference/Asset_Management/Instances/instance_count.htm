instance_count

Mit dieser Nur-Lese-Variable können Sie eine Zählung aller aktiven Instanzen erhalten, die sich im Raum befinden. Dazu gehört die Instanz, auf der die code läuft, aber nicht die Instanzen, die mit den Funktionen zum Deaktivieren von Instanzen deaktiviert wurden. Beachte, dass diese Variable nur die Anzahl der Instanzen zu Beginn des Schritts angibt, so dass alle Änderungen an den Instanzen im Raum, die nach dem Start des Schritts vorgenommen wurden, nicht berücksichtigt werden.

Syntax:

instance_count

Rückgabe:

Beispiel:

if (instance_count < 100)
{
    var dif = 100 - instance_count;
    while (--dif > 0)
    {
        instance_create_layer(random(room_width), random(room_height), "Effects", obj_Star);
    }
}

Die obige code wird mehrere Instanzen des Objekts "obj_Star" erstellen, bis die Gesamtzahl der Instanzen 100 erreicht.
