script_exists

Diese Funktion gibt true oder false zurück, je nachdem, ob die Funktion script oder das Skript mit dem angegebenen Index existiert. Beachten Sie, dass es sich hierbei nicht um eine Zeichenkette handelt, sondern um den Asset-Namen, der den eindeutigen Index für jede script (wie er in IDE erscheinen würde) oder die benannte script Funktion enthält, wie sie im Asset script definiert ist (beachten Sie, dass dies nicht für Methodenvariablen funktioniert). Weitere Informationen zu scripts finden Sie unter Der Script Editor.

Syntax:

script_exists(scr);

Argument Typ Beschreibung
scr Skript Asset The script index that you want to check.

Rückgabe:

Boolean

Beispiel:

script[0] = -1;
script[1] = AI_Left;
script[2] = AI_Right;
var script_num = choose(0, 1, 2);
if script_exists(script[script_num])
{
    script_execute(script[script_num]);
}

Das obige Beispiel fügt zwei script Funktionen und einen Wert in ein Array ein, ermittelt dann eine Zufallszahl und verwendet diese, um eine script Funktion auszuwählen, die ausgeführt werden soll, es sei denn, die -1 wird gewählt, in diesem Fall passiert nichts.
