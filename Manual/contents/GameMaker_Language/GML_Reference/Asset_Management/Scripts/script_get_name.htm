script_get_name

Diese Funktion gibt den Namen des angegebenen scriptals String zurück. Dieser Name ist derjenige, der für das script im Asset-Browser des GameMaker-Hauptfensters angegeben wurde. Weitere Informationen zu scripts finden Sie unter Der Script Editor.

Syntax:

script_get_name(scr);

Argument Typ Beschreibung
scr Skript Asset The index of the script that you want to get the name of.

Rückgabe:

String

Beispiel:

scr_name = script_get_name(Help_File);

Im obigen Beispiel code wird der Name des angegebenen Index script in der Variablen "scr_name" gespeichert.
