skript_ausfuehren_ext

Diese Funktion funktioniert ähnlich wie die Funktion script_execute(), nur dass Sie ein Array übergeben können, das die für die aufgerufene Funktion/das aufgerufene Skript benötigten arguments enthält. Sie können auch zwei optionale arguments an die Funktion übergeben, um einen Offset im Array anzugeben, von dem die arguments abgerufen werden sollen, sowie die Anzahl der arguments, die aus dem Array verwendet werden sollen (dies darf maximal array_length - offset sein).

Syntax:

script_execute_ext(scr, array_args, [offset], [num_args]);

Argument Typ Beschreibung
scr Skript-Asset oder Skript-Funktion The function/script that you want to call
array_args Array The array containing the arguments for the function/script
[offset] Real OPTIONAL Der Offset im Array argument
[num_args] Real OPTIONAL Die Anzahl der zu verwendenden Argumente (ab dem Offset)

Rückgabe:

Variable (Will depend on the return value from the function/script being called)

Beispiel:

script_execute_ext(move_inst, move_array, floor(random(4)), 1);

Das obige Beispiel code verwendet script_execute_ext, um die angegebene Funktion aufzurufen, und liefert ein Array von vier arguments, wobei nur einer von ihnen zufällig verwendet wird.

HINWEIS Diese Funktion kann nicht mit integrierten Funktionen verwendet werden.
