asset_has_tags

Mit dieser Funktion können Sie prüfen, ob ein oder mehrere Tag-Strings einem Asset aus dem Asset-Browser zugewiesen sind. Sie geben entweder den Asset-Namen (als String) oder seinen Asset-Index an, sowie entweder einen einzelnen Tag-String oder ein Array, in dem jedes Element ein einzelner Tag-String ist. Wenn Sie einen Asset-Indexwert angeben, müssen Sie den optionalen Asset-Typ argument (eine Konstante) angeben, da Assets verschiedener Typen denselben Index haben können, auch wenn sie nicht denselben Namen haben. Die verfügbaren Asset-Typen sind in der nachstehenden Tabelle aufgeführt:

Konstante Beschreibung
asset_object Der angegebene Name bezieht sich auf ein Objekt.
asset_sprite Der angegebene Name bezieht sich auf ein Sprite.
asset_sound Der Vorname bezieht sich auf einen Klang.
asset_room Der Vorname bezieht sich auf einen room.
asset_tiles Der angegebene Name bezieht sich auf einen Kachelsatz.
asset_path Der angegebene Name verweist auf einen Pfad.
asset_script Der Vorname bezieht sich auf einen script.
asset_font Der angegebene Name bezieht sich auf eine Schriftart.
asset_timeline Der Vorname bezieht sich auf eine Zeitlinie.
asset_shader Der angegebene Name bezieht sich auf einen Shader.
asset_animationcurve Der angegebene Name bezieht sich auf eine Animationskurve.
asset_sequence Der angegebene Name bezieht sich auf eine Sequenz.

Wenn die Funktion erfolgreich ist und alle angegebenen Tags für das Asset vorhanden sind, gibt sie true zurück, andernfalls false. Wenn Sie nicht auf alle Tags, sondern nur auf eine Auswahl von Tags prüfen wollen, verwenden Sie die Funktion asset_has_any_tag().

Syntax:

asset_has_tags(name_or_index, tags, [asset_type]);

Argument Typ Beschreibung
name_or_index String oder Asset The name of the asset (a string) or its index value (an integer).
tags String oder Array von Strings A single asset tag string or an array with various asset tags.
[asset_type] Anlagenart Konstante OPTIONAL! The type of asset to check the tags for, only used when supplying an index for the first argument.

Rückgabe:

Boolean

Beispiel:

var _a = array_create(3);
_a[0] = "enemy";
_a[1] = "level_" + string(global.Level);
_a[2] = "boss";
if asset_has_tags(object_index, _a, asset_object)
{
    instance_create_layer(0, 0, "Overlay", obj_Boss_Text);
}

Die obige code erstellt ein Array von Tags und prüft dann, ob alle von ihnen auf das angegebene Objekt angewendet werden, und wenn sie es sind, wird eine Instanz eines anderen Objekts erstellt.
