tag_get_assets

Mit dieser Funktion können Sie die Namen aller Assets abrufen, denen das angegebene Tag oder die angegebenen Tags zugewiesen wurden. Sie geben entweder einen einzelnen Tag-String oder ein Array an, wobei jedes Element im Array ein Tag-String ist. Die Funktion gibt ein Array zurück, in dem jeder Eintrag der Name (als String) des Assets mit dem angegebenen Tag ist. Wenn Sie den eindeutigen Index für das Asset benötigen, dann können Sie die Funktion asset_get_index() zusammen mit dem zurückgegebenen Namen verwenden. Wenn es keine Assets mit dem/den angegebenen Tag(s) gibt oder ein Fehler auftritt (z.B.: die angegebenen Tags existieren nicht), wird ein leeres Array zurückgegeben.

Syntax:

tag_get_assets(tags);

Argument Typ Beschreibung
tags String oder Array von Strings A single asset tag string or an array with various asset tags.

Rückgabe:

Array

Beispiel:

backgrounds = ds_list_create();
var _assets = tag_get_assets("background");
for (var i = 0; i< array_length(_assets); ++i;)
{
    if asset_get_type(_assets[i]) == asset_sprite
    {
        ds_list_add(backgrounds, asset_get_index(_assets[i]));
    }
}

Die obige code erstellt eine Liste und ruft dann die Namen aller Assets mit dem Tag "background" ab. loops durchsucht das Array der zurückgegebenen Namen und prüft, ob es sich um sprite -Assets handelt. Ist dies der Fall, wird der eindeutige Indexwert für das Asset zur künftigen Verwendung in die Liste aufgenommen.
