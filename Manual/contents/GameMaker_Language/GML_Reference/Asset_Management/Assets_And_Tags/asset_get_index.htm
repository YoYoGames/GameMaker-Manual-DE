asset_get_index

Mit dieser Funktion können Sie den eindeutigen Identifikationsindex für ein Spiel-Asset anhand seines Namens ermitteln. Wenn das Asset nicht gefunden wird, gibt die Funktion den Wert -1 zurück, andernfalls wird die eindeutige Index-ID für das überprüfte Asset zurückgegeben. Diese ID kann dann in anderen Funktionen wie jeder andere Indexwert verwendet werden, z. B. die sprite_index oder die path_indexzum Beispiel. Bitte beachten Sie, dass Sie, obwohl diese Funktion verwendet werden kann, um Assets aus Strings zu referenzieren (siehe Beispiel unten), immer sicherstellen sollten, dass das Asset existiert, bevor Sie es verwenden, da sonst Fehler auftreten können, die Ihr Spiel zum Absturz bringen.

Syntax:

asset_get_index(name);

ArgumentTyp Beschreibung
name The name of the game asset to get the index of (a string).

Rückgabe:

Asset (any asset type)

Beispiel:

var obj = asset_get_index("obj_Enemy_" + string(global.Level));
if (obj > -1)
{
    instance_create_layer(random(room_width), random(room_height), obj, "Enemy_Layer");
}

Die obige code erhält einen Objektindex aus einer Zeichenkette, und wenn dieser Index existiert, wird eine Instanz des Objekts im Spiel erstellt.
