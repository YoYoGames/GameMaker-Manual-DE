zeitleiste_moment_klar

Mit dieser Funktion können Sie einen bestimmten Zeitpunkt in einer zuvor definierten Zeitleiste von allen codes und Aktionen löschen.

Syntax:

timeline_moment_clear(ind, step);

Argument Typ Beschreibung
ind Zeitleiste Asset The index of the timeline to clear.
step Real The moment to clear.

Rückgabe:

N/A

Beispiel:

timeline_moment_clear(global.tl, room_speed * 30);

Die obige code löscht den angegebenen Zeitpunkt in der Zeitleiste, die durch die Variable "global.tl" indiziert ist.
