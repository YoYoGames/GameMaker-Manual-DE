zeitleiste_index

Diese Variable enthält den Index der Zeitleiste, die derzeit mit der Instanz verbunden ist. Du kannst sie auf eine bestimmte Zeitleiste setzen, um diese zu verwenden, oder sie auf -1 setzen, um die Verwendung einer Zeitleiste für die Instanz zu beenden (wenn keine Zeitleiste für die Instanz definiert ist, wird auch -1 zurückgegeben). Beachte, dass dies nicht die Zeitleiste startet - verwende dafür die Variable timeline_running.

Syntax:

timeline_index;

Rückgabe:

Timeline Asset

Beispiel:

if timeline_exists(global.tl)
{
    timeline_index = global.tl;
}

Die obige code weist der Instanz, die code ausführt, eine Zeitleiste zu, die in der Variablen "global.tl" indiziert ist, wenn diese Zeitleiste existiert.
