zeitleiste_moment_add_script

Mit dieser Funktion können Sie dynamisch eine Skriptfunktion zu einem beliebigen "Moment" innerhalb der Zeitleiste hinzufügen, wobei ein "Moment" einem Spieltick (oder Schritt) entspricht. Auf diese Weise können Sie eine neue Zeitleiste mit der timeline_add() Funktion eine neue Zeitleiste erstellen und an jedem beliebigen Punkt verschiedene Verhaltensweisen hinzufügen oder einfach eine bereits erstellte Zeitleistenressource mit neuen Verhaltensweisen ändern. Beachten Sie, dass die Funktion keine zusätzliche arguments benötigt, wenn Sie diese Funktion verwenden, und wenn Sie sie verwenden, um ein Zeitleisten-Asset im Asset-Browser zu ändern, werden alle Instanzen, die diese Zeitleiste verwenden, davon betroffen sein, und die Änderung wird bis zum Ende des Spiels andauern (der Aufruf von game_restart() wird die Änderung auch nicht zurücksetzen).

Syntax:

timeline_moment_add_script(ind, step, script);

Argument Typ Beschreibung
ind Zeitleiste Asset The index of the timeline to add a moment to.
step Real The moment (step) to add to.
script Skript-Funktion The index of the script function to add into the moment.

Rückgabe:

N/A

Beispiel:

global.tl = timeline_add();
var i = room_speed * 60;
repeat(3)
{
    timeline_moment_add_script(global.tl, i, choose(Attack_1, Attack_2, Attack_3);
    i += room_speed * 60;
}

Die obige code erstellt eine neue Zeitleiste und speichert ihren Index in der Variablen "global.tl". Anschließend werden der Zeitleiste drei zufällig ausgewählte scripts im Abstand von einer Minute hinzugefügt.
