timeline_exists

Mit dieser Funktion können Sie prüfen, ob eine Zeitleiste vorhanden ist (gibt true zurück) oder nicht (gibt false zurück). Dies ist besonders nützlich bei der dynamischen Erstellung von Zeitleisten mit der timeline_add() Funktion dynamisch erstellt werden, aber Sie sollten beachten, dass bei der Suche nach einer nicht initialisierten Variablen (die sonst dem Index einer Zeitleiste zugewiesen wäre) ein Fehler ausgegeben wird.

Syntax:

timeline_exists(ind);

ArgumentTyp Beschreibung
ind The index of the time line to check for.

Rückgabe:

Beispiel:

if timeline_exists(global.tl)
{
    timeline_delete(global.tl);
}

Die obige code prüft, ob eine Zeitleiste existiert und in der globalen Variable "tl" indiziert ist, und wenn ja, löscht sie diese Zeitleiste.
