zeitleiste_position

Diese Variable enthält die aktuelle Position (Zeitpunkt), an der sich eine Zeitleiste gerade befindet. Sie können diesen Wert ändern, um Teile der Zeitleiste zu überspringen, Teile zu wiederholen oder die Zeitleiste wieder von vorne zu beginnen.

Syntax:

timeline_position;

Rückgabe:

(single precision floating point value)

Beispiel:

if !timeline_running
{
    timeline_position = 0;
    timeline_running = true;
}

Die obige code prüft, ob die Instanz eine Zeitleiste ausführt, und wenn nicht, setzt sie die zugewiesene Zeitleiste zurück, um sie im ersten Moment zu starten, und startet sie dann.
