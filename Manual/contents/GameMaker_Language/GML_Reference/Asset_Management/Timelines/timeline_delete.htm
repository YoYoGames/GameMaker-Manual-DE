timeline_delete

Mit dieser Funktion können Sie eine beliebige Zeitleiste aus Ihrem Spiel löschen. Wenn diese Zeitleiste im GameMaker Asset Browser erstellt wurde, beachten Sie bitte, dass sie vollständig aus dem Spiel entfernt wird und der Versuch, die indizierte Zeitleiste nach der Verwendung dieser Funktion erneut aufzurufen, einen Fehler verursacht. Wurde die Zeitleiste dynamisch mit der Funktion timeline_add() Funktion dynamisch erstellt wurde, sollte diese Funktion verwendet werden, sobald die Zeitleiste nicht mehr benötigt wird, um Speicherlecks zu vermeiden, die Ihr Spiel verlangsamen (und schließlich zum Absturz bringen) könnten.

Syntax:

timeline_delete(ind);

ArgumentTyp Beschreibung
ind The index of the time line to delete.

Rückgabe:

Beispiel:

if timeline_exists(global.tl)
{
    timeline_delete(global.tl);
}

Die obige code prüft, ob eine Zeitleiste existiert und in der globalen Variable "tl" indiziert ist, und wenn ja, löscht sie diese Zeitleiste.
