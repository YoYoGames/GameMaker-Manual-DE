timeline_get_name

Diese Funktion kann verwendet werden, um den Namen einer Zeitleiste als String zu erhalten. timeline_add() Funktion erstellt wurde, hat der zurückgegebene Name das Format "__newtimelineN" zurückgegeben, wobei "N" die Nummer der Zeitleiste ist (beginnend bei 0). Bitte beachten Sie, dass dies nur eine Zeichenkette ist und nicht verwendet werden kann, um die Zeitleiste direkt zu referenzieren - dafür Dazu benötigen Sie den Index der Zeitleiste. Sie können diese Zeichenkette jedoch verwenden, um den Zeitleistenindex zu erhalten, indem Sie die zurückgegebene Zeichenkette zusammen mit der Funktion asset_get_index().

Syntax:

timeline_get_name(ind);

ArgumentTyp Beschreibung
ind The index of the time line to check the name of.

Rückgabe:

Beispiel:

var _str = timeline_get_name(timeline_index);
show_debug_message("Current timeline = " + _str);

Die obige code ermittelt den Namen der aktuell zugewiesenen Zeitleiste und gibt ihn auf der Konsole aus.
