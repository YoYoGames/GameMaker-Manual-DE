timeline_clear

Mit dieser Funktion können Sie eine bestimmte Zeitleiste von "Momenten" löschen, indem Sie alle codes und Aktionen für diese Zeitleiste entfernen und sie leer lassen.

Syntax:

timeline_clear(ind);

Argument Typ Beschreibung
ind Zeitleiste Asset The index of the timeline to clear.

Rückgabe:

N/A

Beispiel:

if timeline_position > 200
{
    timeline_clear(global.tl);
    timeline_index = -1;
}

Die obige code löscht die angegebene Zeitleiste, die durch die Variable "global.tl" indiziert ist, von allen Zeitpunkten, an denen ein bestimmter Zeitpunkt überschritten wurde.
