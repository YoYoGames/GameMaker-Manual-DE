timeline_size

Mit dieser Funktion können Sie die Gesamtzahl der aktiven Momente für eine Zeitleiste ermitteln (ein "aktiver" Moment ist ein Moment, zu dem code oder GML Visual hinzugefügt wurde). Dies kann praktisch sein, wenn Sie dynamische Zeitleisten mit den Funktionen timeline_moment_add_script() und timeline_moment_clear() Funktionen zu erstellen, da Sie prüfen können, ob eine bestimmte Zeitleiste die richtige Anzahl aktiver Momente oder gar keine hat.

Syntax:

timeline_size(ind);

ArgumentTyp Beschreibung
ind The index of the timeline get the information from.

Rückgabe:

Beispiel:

if timeline_size(global.tl) == 0
{
    timeline_moment_add_script(global.tl, room_speed + irandom(room_speed), spawn_enemy);
}

Das obige code prüft die angegebene Größe der Zeitleiste, und wenn es 0 zurückgibt (d.h. die Zeitleiste hat keine aktiven Momente), fügt es eine script Funktion hinzu, die zu einem zufälligen Moment innerhalb der Zeitleiste verwendet wird, der in der globalen Variable "tl" indiziert ist.
