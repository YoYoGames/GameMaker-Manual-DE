Zeitleisten

Zeitleisten sind ein leistungsstarker Mechanismus zur Steuerung von Ereignissen in Ihrem Spiel und basieren auf "Momenten", wobei ein "Moment" einfach ein einzelner Spielschritt ist. Diese Zeitleisten werden in der Regel über den Asset-Browser mit dem Zeitleisten-Editor ähnlich wie ein Objekt erstellt, d. h. Sie erstellen eine Zeitleiste und fügen dann in jedem Moment, in dem sie eine Aktion ausführen soll, einige code oder ein GML visuelles Symbol hinzu, das sie ausführen soll. Nach der Erstellung können Sie Zeitleisten über code anhalten, starten und manipulieren. Sie können sogar einzelne Momente ändern, indem Sie eine Skriptfunktion definieren und diese dynamisch von einem Controller-Objekt aus in die Zeitleiste einfügen (zum Beispiel), was dies zu einem sehr leistungsfähigen und vielseitigen Werkzeug bei der Erstellung Ihrer Spiele macht.

Die folgenden Funktionen dienen zum Hinzufügen und Bearbeiten von Zeitleisten in Ihrem Spiel:

  • timeline_exists
  • timeline_get_name
  • timeline_add
  • timeline_delete
  • timeline_moment_add_script
  • timeline_moment_clear
  • timeline_clear
  • timeline_size
  • timeline_max_moment

Die folgenden Variablen sind in allen Instanzen "eingebaut", um die Arbeit mit Zeitleisten zu erleichtern und Ihnen die Kontrolle darüber zu geben:

  • timeline_index
  • timeline_running
  • timeline_speed
  • timeline_position
  • timeline_loop

Beachten Sie, dass Sie hier weitere Informationen über Instanzvariablen finden können: Instanz-Variablen.
