timeline_running

Diese Variable enthält den aktuellen Status der zugewiesenen Zeitleiste und gibt true zurück, wenn sie läuft und false, wenn sie nicht läuft. Sie können diese Variable auch auf true oder false setzen, um die Zeitleiste jederzeit zu starten und zu stoppen. Es ist zu beachten, dass eine gestoppte Zeitleiste nicht zurückgesetzt wird, so dass ein erneuter Start zu einem späteren Zeitpunkt die Zeitleiste genau ab dem Zeitpunkt startet, an dem sie gestoppt wurde.

Syntax:

timeline_running;

Rückgabe:

Beispiel:

if !timeline_running
{
    timeline_position = 0;
    timeline_running = true;
}

Die obige code prüft, ob die Instanz eine Zeitleiste ausführt, und wenn nicht, setzt sie die zugewiesene Zeitleiste zurück, um sie im ersten Moment zu starten, und startet sie dann.
