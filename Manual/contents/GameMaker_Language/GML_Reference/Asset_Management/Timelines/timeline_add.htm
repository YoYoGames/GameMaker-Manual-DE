zeitleiste_hinzufügen

Mit dieser Funktion können Sie eine neue (leere) Zeitleiste in Ihr Spiel einfügen. Die Funktion gibt den Index dieser neuen Zeitleiste zurück, der in einer Variablen gespeichert werden sollte, damit er in allen weiteren Funktionsaufrufen, die diese neue Zeitleiste betreffen, verwendet werden kann. Sie sollten auch sicherstellen, dass Sie die Funktion timeline_delete() immer dann verwenden, wenn Sie die Zeitleiste nicht mehr verwenden wollen (z. B. am Ende des Raums), um mögliche Speicherlecks zu vermeiden, die Ihr Spiel verlangsamen oder zum Absturz bringen könnten. Um Momente zu einer auf diese Weise erstellten Zeitleiste hinzuzufügen, siehe die Funktion timeline_moment_add_script().

Syntax:

timeline_add();

Rückgabe:

Timeline Asset

Beispiel:

global.tl = timeline_add();

Die obige code erstellt eine neue Zeitleiste und speichert ihren Index in der Variablen "global.tl".
