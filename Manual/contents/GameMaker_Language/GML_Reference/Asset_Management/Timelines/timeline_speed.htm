timeline_speed

Normalerweise wird die Position in der Zeitleiste bei jedem Schritt um 1 erhöht, Sie können diesen Wert jedoch ändern, indem Sie diese Variable auf einen anderen Wert setzen. Sie können reelle Zahlen verwenden (z. B. 0,5 oder 2,4), und wenn der Wert größer als 1 ist, können mehrere Momente innerhalb desselben Zeitschritts stattfinden (sie werden alle in der gleichen Reihenfolge ausgeführt, wie sie für die Zeitleiste definiert sind, so dass keine Aktionen übersprungen werden).

Syntax:

timeline_speed;

Rückgabe:

(single precision floating point value)

Beispiel:

if !timeline_running
{
    timeline_running = true;
    timeline_speed = 0.5;
}

Die obige code prüft, ob die Instanz ihre zugehörige Zeitleiste ausführt, und wenn nicht, wird die Zeitleiste mit halber Geschwindigkeit gestartet.
