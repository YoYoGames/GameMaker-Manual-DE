animcurve_create

Diese Funktion erstellt eine leere Animationskurvenstruktur, die Sie mit Kanaldaten füllen können. Die Funktion gibt einen Zeiger auf die Struktur zurück, der dann verwendet wird, um der Animationskurve Kanäle und andere Daten hinzuzufügen. Sie können diesen Zeiger auch an Funktionen wie animcurve_get() übergeben, um später die Daten aus der Kurve zu holen.

Wenn Sie diese Funktion verwenden, enthält die erstellte Struktur keine Daten. Um sie zu verwenden, sollten Sie mindestens einen Kanal hinzufügen, und der Kanal sollte mit Punkten versehen sein. Um einen Kanal hinzuzufügen, siehe die Funktion animcurve_channel_new()und zum Hinzufügen von Punkten zu einem Kanal siehe die Funktion animcurve_point_new(). Zusätzlich können Sie der Animationskurvenstruktur einen Namen geben, indem Sie die Variable "name" setzen (wie im Beispiel code unten gezeigt). Kanäle, die zu einer neuen Animationskurve hinzugefügt werden, sollten als Array hinzugefügt werden, wobei jedes Element im Array auf eine Kanalstruktur verweist, und die Punkte in einem Kanal sollten ebenfalls als Array hinzugefügt werden, wobei jedes Arrayelement auf eine Punktstruktur verweist.

Beachten Sie, dass auf diese Weise erstellte Animationskurven gelöscht werden müssen, wenn sie nicht mehr benötigt werden, da sie sonst Speicherplatz belegen, was zu einem Speicherleck führen kann, das Ihr Spiel verlangsamt und schließlich zum Absturz bringt. Sie können eine Animationskurve entfernen, wenn sie nicht mehr benötigt wird, indem Sie die Funktion animcurve_destroy(). Sie müssen keine Kanal- oder Punktdaten innerhalb der Kurve bereinigen, da dies vom Garbage Collector automatisch erledigt wird, wenn die Kurve selbst zerstört wird.

Syntax:

animcurve_create();

Rückgabe:

Animation Curve Struct

Beispiel:

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

Die obige code erstellt eine neue Animationskurvenstruktur und speichert sie in der Variablen "my_curve". Diese Struktur wird dann mit einem Namen und einem Kanal-Array gefüllt. Das Kanal-Array enthält einen einzelnen Kanal mit drei Punkten.
