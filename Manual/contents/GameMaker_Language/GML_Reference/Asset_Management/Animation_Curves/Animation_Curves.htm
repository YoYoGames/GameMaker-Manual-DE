Animation Kurven

Animationskurven können im Asset-Browser mit dem Animationskurven-Editor als Assets eingerichtet und dann im Sequenz-Editor verwendet werden, um Spuren in einem sequence zu steuern. Sie können aber auch in Ihrem code verwendet werden, um Werte im Laufe der Zeit in Ihren Spielen für alles Mögliche zu ändern, von Tweening-Effekten auf Schaltflächen über Bewegungen bis hin zu Farbänderungen und mehr. Sie können auch Ihre eigenen Animationskurven auf runtime erstellen, indem Sie die hier aufgelisteten Funktionen verwenden, was sie zu einem sehr flexiblen Werkzeug für Ihre Spiele macht.

Die verfügbaren Funktionen für Animationskurven sind wie folgt

  • animcurve_get
  • animcurve_get_channel_index
  • animcurve_get_channel
  • animcurve_channel_evaluate
  • animcurve_create
  • animcurve_exists
  • animcurve_channel_new
  • animcurve_point_new
  • animcurve_destroy
