animcurve_point_new

Diese Funktion kann verwendet werden, um eine neue Punktstruktur zu erstellen, die einem Animationskurvenkanal hinzugefügt wird. Wenn eine Punktstruktur erstellt wird, ist sie leer und Sie müssen die folgenden Variablen setzen, um die Punktdaten zu erzeugen:

Animation Kurvenpunkt Struktur
Variable Name Datenart Beschreibung
posx real Die zeitliche Position (normiert von 0 bis 1) des Punktes.
value real Der Wert des Punktes.

Jede Punktstruktur, die Sie erstellen, sollte zu einem Array hinzugefügt werden, und dieses Array wird der Animationskurven-Kanalstruktur-Variable "channels" hinzugefügt (wie im folgenden Beispiel gezeigt).

Syntax:

animcurve_point_new();

Rückgabe:

Animation Curve Point Struct

Beispiel:

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

Die obige code erstellt eine neue Animationskurvenstruktur und speichert sie in der Variablen "my_curve". Diese Struktur wird dann mit einem Namen und einem Kanal-Array gefüllt. Das Kanal-Array enthält einen einzelnen Kanal mit drei Punkten.
