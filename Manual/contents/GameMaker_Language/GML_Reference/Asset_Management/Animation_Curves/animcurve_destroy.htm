animcurve_destroy

Diese Funktion kann verwendet werden, um Animationskurven zu zerstören, die zuvor mit der Funktion animcurve_create(). Der Aufruf dieser Funktion entfernt die Animationskurve aus dem Speicher und bereinigt auch alle Kanäle oder Punkte, die sie enthält (diese müssen nie manuell bereinigt werden). Beachten Sie, dass Sie keine Animationskurven zerstören können, die im Asset-Browser erstellt wurden, sondern nur solche, die auf runtime erstellt wurden.

Syntax:

animcurve_destroy(curve_struct);

Argument Typ Beschreibung
curve_struct Animation Kurve Struktur The pointer to the curve struct to destroy.

Rückgabe:

N/A

Beispiel:

animcurve_destroy(my_curve);

Die obige code zerstört die (zuvor erstellte) Animationskurvenstruktur, die in der Variablen "my_curve" indiziert ist.
