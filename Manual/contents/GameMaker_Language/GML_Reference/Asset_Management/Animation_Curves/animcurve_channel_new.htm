animcurve_channel_new

Diese Funktion kann verwendet werden, um eine neue Animationskurven-Kanalstruktur zu erstellen. Eine Kanalstruktur hat die folgenden Variablen:

Animation Kurvenkanal Struktur
Variable Name Datenart Beschreibung
Name String Dies ist der Name des Kanals.
Typ Animation Kurve Interpolationstyp Konstant Dies ist eine der in der nachstehenden Tabelle angegebenen Konstanten.
Iterationen Ganzzahl Wenn der Kanal die Katmull-Rom-Interpolation ("glatte" Interpolation) verwendet, gibt dieser Wert an, wie viele Punkte für jedes Segment der Kurve erzeugt wurden (beachten Sie, dass diese zusätzlichen Punkte intern in der Funktion sind und nur für die Berechnungen von runtime verwendet werden). Wenn der Kanal eine lineare Interpolation verwendet, ist dieser Wert immer noch vorhanden, kann aber ignoriert werden, da er keinen Einfluss darauf hat, wie die Kurve interpoliert wird.
Punkte Array Dies ist ein Array, wobei jedes Element in dem Array ein Animation Kurvenpunkt Struktur.

Animation Kurve Interpolationstyp Konstant
Konstante Beschreibung
animcurvetype_linear Wird für die lineare Interpolation zwischen Punkten verwendet.
animcurvetype_catmullrom Wird für eine glatte Interpolation zwischen Punkten mittels Katmull-Rom-Interpolation verwendet.
animcurvetype_bezier Wird für die Bezier-Interpolation zwischen Punkten verwendet.


Wenn Sie eine neue Kanalstruktur erstellen, ist die Variable "Name" standardmäßig eine leere Zeichenkette, der "Typ" ist animcurvetype_linear und die "Iterationen" sind 16. Alle diese Variablen können auf die von Ihnen gewünschten Werte gesetzt werden (beachten Sie, dass der Wert "Iterationen" keine Auswirkungen auf lineare Kurventypen hat). Nach der Erstellung müssen Sie der Kurve Punkte hinzufügen, was durch Hinzufügen verschiedener Punktstrukturen zum Array"points" geschieht. Diese Punktstrukturen können mit der Funktion animcurve_point_new().

Syntax:

animcurve_channel_new();

Rückgabe:

Animation Curve Channel Struct

Beispiel:

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

Die obige code erstellt eine neue Animationskurvenstruktur und speichert sie in der Variablen "my_curve". Diese Struktur wird dann mit einem Namen und einem Kanal-Array gefüllt. Das Kanal-Array enthält einen einzelnen Kanal mit drei Punkten.
