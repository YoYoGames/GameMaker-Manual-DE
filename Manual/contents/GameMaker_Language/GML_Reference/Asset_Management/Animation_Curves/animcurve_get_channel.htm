animcurve_get_channel

Diese Funktion gibt die Struktur zurück, die die Kanaldaten für den in einem Animationskurven-Asset angegebenen Kanal enthält. Sie geben die Animationskurven-ID oder -Struktur sowie den Kanalnamen oder -index an, und die Funktion gibt eine Struktur mit dem folgenden Format zurück:

Variable Name Datenart Beschreibung
name String Dies ist der Name des Kanals.
type Animation Kurve Interpolationstyp Konstant Dabei handelt es sich um eine der Konstanten animcurvetype_linear für lineare Interpolation zwischen Punkten, animcurvetype_catmullrom für "glatte" Interpolation zwischen den Punkten mittels Katmull-Rom-Interpolation oder animcurvetype_bezier für Bezier-Interpolation.
iterations Ganzzahl Wenn der Kanal die Katmull-Rom-Interpolation ("glatte" Interpolation) verwendet, gibt dieser Wert an, wie viele Punkte für jedes Segment der Kurve erzeugt wurden (beachten Sie, dass diese zusätzlichen Punkte intern in der Funktion sind und nur für die Berechnungen von runtime verwendet werden). Wenn der Kanal eine lineare Interpolation verwendet, ist dieser Wert immer noch vorhanden, kann aber ignoriert werden, da er keinen Einfluss darauf hat, wie die Kurve interpoliert wird.
points Array-Zeiger Es handelt sich um ein Array, bei dem jedes Element des Arrays eine Punktstruktur ist.


Die Animationskurven-ID ist der Name der Animationskurve, wie er im Asset-Browser definiert wurde, oder der Strukturzeiger, wie er von der Funktion animcurve_create(). Der Kanalname ist eine Zeichenkette, die sich auf den Kanal bezieht, wie er im Animationskurven-Asset definiert wurde, oder Sie können einen Indexwert angeben, der von 0 bis n reicht, wobei n der letzte Kanal im Kurven-Asset ist (z. B.: Wenn eine Animationskurve 4 Kanäle hat, werden sie von 0 bis 3 indiziert). Beachten Sie, dass die Übergabe eines Indexwerts weniger Verarbeitung erfordert als die Übergabe eines Kanalnamens. Wenn die Funktion fehlschlägt (d. h. kein Kanal mit dem angegebenen Namen oder Index existiert), erhalten Sie eine Fehlermeldung.

Die Punkte eines einzelnen Kanals werden als Structs in einem Array gespeichert, wobei jedes Element im Array ein einzelnes Point-Struct ist. Die Punktstruktur hat die folgenden Variablen:

Variable Name Datenart Beschreibung
posx real Die zeitliche Position (normiert von 0 bis 1) des Punktes.
value real Der Wert des Punktes.

Syntax:

animcurve_get_channel(curve_struct_or_id, channel_name_or_index)

Argument Typ Beschreibung
curve_struct_or_id Animation Kurve Struktur The ID or struct pointer of the animation curve to target
channel_name_or_index Zeichenfolge oder Real The channel name (a string) or the channel index (an integer).

Rückgabe:

Animation Curve Channel Struct

Beispiel:

var _channeldata = animcurve_get_channel(ac_ButtonTween, 0);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i)
{
    _points[i].value += 1;
}

Die obige code ruft die Kanalstruktur für Kanal 0 im Kurven-Asset "ac_ButtonTween" ab, geht dann loops durch die Punkte auf der Kanalkurve und fügt ihrem Wert einen hinzu.
