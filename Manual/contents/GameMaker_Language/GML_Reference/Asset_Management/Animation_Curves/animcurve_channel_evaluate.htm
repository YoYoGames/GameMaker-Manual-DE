animcurve_channel_evaluate

Diese Funktion kann verwendet werden, um den Wert zu einem bestimmten Zeitpunkt von einer Kanalstruktur zu erhalten. Sie liefern den struct-Zeiger für den Kanal (wie von der Funktion animcurve_get_channel()zurückgegeben wird, oder wie er in der Animationskurven-Struktur von der Funktion animcurve_get()) und die "x"-Position (Zeit) entlang des auszuwertenden Kurvenkanals. Diese Position sollte zwischen 0 und 1 liegen, und die Funktion gibt den Kurvenwert an dieser Position zurück, oder sie gibt 0 zurück, wenn die übergebene Kanalstruktur ungültig ist.

Syntax:

animcurve_channel_evaluate(channel_struct, posx);

Argument Typ Beschreibung
channel_struct Animation Kurvenkanal Struktur The struct pointer for the channel to evaluate.
posx Real The position in time to check (from 0 to 1).

Rückgabe:

Real

Beispiel:

var _channel = animcurve_get_channel(ac_AlphaCurve, 0);
var _val = animcurve_channel_evaluate(_channel, sin(current_time/1000));
image_alpha = _val;

Die obige code ruft die Kanalstruktur für das Animationskurven-Asset "ac_AlphaCurve" ab. Anschließend wird der zurückgegebene Bewertungswert verwendet, um das Bild-Alpha für die Instanz einzustellen.
