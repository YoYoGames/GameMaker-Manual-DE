animcurve_get_channel_index

Diese Funktion gibt den Indexwert für einen bestimmten Animationskurvenkanal zurück. Sie geben die Animationskurven-ID oder die Struktur an, wobei die Kurven-ID der Name der Animationskurve ist, wie sie im Asset-Browser definiert wurde, oder der Strukturzeiger, wie er von der Funktion animcurve_create() zurückgegeben wird. Dann geben Sie den Namen des Kanals als Zeichenkette an, und die Funktion gibt den mit diesem Kanal verbundenen Indexwert zurück. Wenn die Kurve oder der Kanal nicht existiert, erhalten Sie eine Fehlermeldung.

Syntax:

animcurve_get_channel_index(curve_struct_or_id, channel_name);

Argument Typ Beschreibung
curve_struct_or_id Animation Kurve Struktur The ID or struct pointer of the animation curve to target
channel_name Zeichenfolge The channel name (a string).

Rückgabe:

Real

Beispiel:

var _channelindex = animcurve_get_channel_index(ac_ButtonTween, "x_pos")
var _channeldata = animcurve_get_channel(ac_ButtonTween, _channelindex);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i;)
{
    _points[i].value += 1;
}

Die obige code ruft die Kanalstruktur für den Kanal mit dem Namen "x_pos" im Kurven-Asset "ac_ButtonTween" ab, durchläuft dann loops die Punkte auf der Kanalkurve und fügt ihrem Wert einen hinzu.
