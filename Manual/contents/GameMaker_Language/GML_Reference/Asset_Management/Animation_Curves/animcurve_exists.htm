animcurve_exists

Diese Funktion gibt true zurück, wenn die angegebene Animation Kurve Asset oder Animationskurve Struktur existiert und eine gültige Animationskurve ist, oder false, wenn sie es nicht ist.

Syntax:

animcurve_exists(curve_struct_or_id);

Argument Typ Beschreibung
curve_struct_or_id Animation Kurve Struktur oder Animationskurve Asset The Animation Curve that will be checked.

Rückgabe:

Boolean

Beispiel:

if (animcurve_exists(spring_curve))
{
    apply_spring_animation(sprite_curve);
}

Die obige code prüft, ob die in der benutzerdefinierten Variable spring_curve gespeicherte Animationskurve existiert. Ist dies der Fall, wird eine benutzerdefinierte Methode zur Verwendung dieser Animationskurve ausgeführt.
