animcurve_get

Diese Funktion gibt eine Struktur zurück, die alle Daten für die angegebene Animationskurve enthält. Sie geben die Asset-ID der Animationskurve an (wie im Asset-Browser definiert), und die Funktion gibt eine Struktur mit den folgenden Variablen zurück:

Animation Kurve Struktur
Variable Name Datenart Beschreibung
name Zeichenfolge Dies ist der Name der Animationskurve.
channels Array von Animation Kurvenkanal Strukturs Es handelt sich um ein Array, wobei jedes Element des Arrays eine Kanalstruktur ist.


Die Variable channels in der Struktur ist ein Array, wobei jedes Array-Element ein Animation Kurvenkanal Struktur mit Daten, die sich auf einen Kanal innerhalb der Kurve beziehen. Die Kanalstruktur wird auf dieser Seite erklärt.

Wie bei den Kanälen werden die Punkte eines einzelnen Kanals als Structs in einem Array gespeichert, wobei jedes Element im Array ein einzelnes Point-Struct ist. Die Punktstruktur wird auf dieser Seite erklärt.

Wenn die Funktion fehlschlägt (z. B. wenn das angegebene Animationskurven-Asset nicht existiert), gibt die Funktion -1 zurück.

Syntax:

animcurve_get(curve_id)

Argument Typ Beschreibung
curve_id Animation Kurve Asset The asset browser ID (index) of the animation curve to get.

Rückgabe:

Animation Curve Struct or -1

Beispiel:

var _curve = animcurve_get(ac_ButtonTween);
var _channel = _curve.channels[0];
if (_channel.type != animcurvetype_linear)
{
    _channel.type = animcurvetype_linear;
}

Die obige code ruft die Animationskurvenstruktur für das Kurven-Asset "ac_ButtonTween" ab, prüft dann den Kurventyp für den ersten Kanal und setzt ihn, falls er nicht linear ist, auf linear.
