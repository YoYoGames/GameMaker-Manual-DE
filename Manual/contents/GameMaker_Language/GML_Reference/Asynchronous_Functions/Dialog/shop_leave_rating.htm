shop_leave_rating

Diese Funktion öffnet einen OS-abhängigen Dialog, in dem Sie den Benutzer auffordern können, eine Bewertung oder einen Kommentar zu einer bestimmten Seite abzugeben. Sie können den Text, der in dem Dialog erscheinen soll, sowie den Text, der auf den beiden Schaltflächen erscheinen soll, und die URL, unter der der Kommentar veröffentlicht werden soll, festlegen.

HINWEIS: Diese Funktion ist nur für die Ziele Android und iOS verfügbar.

WICHTIG! Unter iOS wird bei der Verwendung von TestFlight während der Entwicklung die Aufforderung angezeigt, aber Sie können nichts tun. Sobald Ihre App im App Store ist, wird diese Schaltfläche aktiv und der Benutzer hat die Möglichkeit, eine Rezension zu schreiben oder eine Bewertung zu hinterlassen. Wenn Sie das Spiel über TestFlight vertreiben, wird dieses Fenster möglicherweise gar nicht angezeigt.

Syntax:

shop_leave_rating(text, yes_string, no_string, url)

Argument Typ Beschreibung
text Zeichenfolge Text that appears on the dialog.
yes_string Zeichenfolge Text that appears on the "yes" button.
no_string Zeichenfolge Text that appears on the "no" button.
url Zeichenfolge The full URL where the comment has to be sent.

Rückgabe:

N/A

Beispiel:

if timer <= 0
{
    shop_leave_rating("Thanks for playing the game! Why not leave a comment?", "Okay!", "Not Today!", "http://MacSweeney/comments");
    timer = 10000000;
}
else
{
    timer -= 1;
}

Die obige code wird den Benutzer auffordern, einen Kommentar zu hinterlassen, wenn die Variable "timer" auf 0 heruntergezählt wurde.
