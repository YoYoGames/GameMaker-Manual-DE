Cloud-Sparen

Bei der Entwicklung von Spielen ist es oft notwendig, Informationen über den Spielzustand in einer Art Datei zu speichern. Die Speicherung dieser Informationen auf dem Gerät ist jedoch nicht immer die beste Option, da diese Informationen verloren gehen können, wenn der Spieler das Spiel löscht und es dann neu installiert. Um dies zu verhindern, können Sie verschiedene Cloud-Dienste nutzen, die eine Datenspeicherung über das Internet anbieten, so dass die Daten jederzeit abgerufen und geändert werden können. Sie sollten beachten, dass diese Funktion auf einen einzigen Daten-"Blob" pro Spiel beschränkt ist, so dass jedes Mal, wenn Sie einen neuen Speicherstand an den Cloud-Dienst senden, egal ob es sich um eine Zeichenkette oder eine Datei handelt, alle zuvor gespeicherten Daten überschrieben werden.

HINWEIS Derzeit nutzt nur das Android-Ziel diese Funktionen, und Sie müssen das Kontrollkästchen Google Cloud-Speicherung aktivieren im Abschnitt Soziales der Android Spieloptionen aktivieren. Dadurch werden Sie aufgefordert, die Google Services-Erweiterung herunterzuladen, die erforderlich ist, damit die Cloud-Speicherung auf Android funktioniert. Weitere Informationen zum Cloud Saving finden Sie in dem folgenden Helpdesk-Artikel: Android-Cloud-Sparen

GameMaker unterstützt dies mit einigen einfachen Funktionen, die in Verbindung mit dem Asynchronous Cloud Event arbeiten. Diese Funktionen werden auf den folgenden Seiten erklärt:

  • cloud_synchronise
  • cloud_string_save
  • cloud_file_save
