http_get_request_crossorigin

Diese Funktion kann verwendet werden, um den herkunftsübergreifenden Typ für HTML5-Spiele abzurufen und gibt einen String zurück (bei allen anderen platforms wird ein leerer String "" zurückgegeben).

Syntax:

http_get_request_crossorigin();

Rückgabe:

Beispiel:

if string_lower(http_get_request_crossorigin()) != "use-credentials"
{
    http_set_request_crossorigin("use-credentials");
}
sprite_add("sprites/player_outfit_1.png", 0, false, false, 0, 0);

Die obige code prüft zunächst den aktuell eingestellten Cross-Origin-Typ, und wenn dieser nicht "use-credentials" ist, wird er auf "use-credentials" gesetzt, und dann wird eine sprite aus einer Datei hinzugefügt.
