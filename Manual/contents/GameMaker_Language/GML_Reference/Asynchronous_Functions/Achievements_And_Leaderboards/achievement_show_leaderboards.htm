leistung_anzeigen_leaderboards

Mit dieser Funktion wird die Ranglistenseite für die gewählte platform geöffnet. Bitte beachten Sie, dass es sich hierbei um eine asynchrone Funktion handelt, d. h. Ihr Spiel läuft im Hintergrund weiter, während die Seite mit den Bestenlisten angezeigt wird. Daher sollten Sie Sie sollten daher vorsichtig sein, wo Sie diese Funktion verwenden, und sicherstellen, dass Sie das Spiel pausieren oder die Anzeige nur in Bereichen Ihres Spiels zulassen, in denen sie den Spielablauf nicht stört.

Syntax:

achievement_show_leaderboards()

Rückgabe:

N/A

Beispiel:

if achievement_available()
    {
    global.Pause = true;
    instance_create_layer(0, 0, "Controllers", obj_Pause);
    achievement_show_leaderboards();
    }

Die obige code prüft, ob der gewählte Leaderboard- und Achievement-Dienst verfügbar ist. Ist dies der Fall, wird eine globale Variable gesetzt und eine Controller-Instanz "pause" erstellt, bevor die Funktion zur Anzeige der Leaderboards auf dem Bildschirm aufgerufen wird.
