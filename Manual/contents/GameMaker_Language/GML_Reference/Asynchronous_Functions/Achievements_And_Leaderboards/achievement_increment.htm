Leistung_Zunahme

Spiele können Errungenschaften haben, die keinen Abschlusswert haben, und so können Sie diese Funktion verwenden, um diese Errungenschaft um einen bestimmten Betrag zu erhöhen. An Android senden Sie die ID der Errungenschaft als String (dies ist die eindeutige ID der Errungenschaft, die beim Einrichten der Errungenschaft zugewiesen wurde). zugewiesen wurde), während Sie auf allen anderen platforms den definierten Namen der Leistung als String übergeben und dann den tatsächlichen Wert der Leistung angeben, um den sie erhöht werden soll.

Die Funktion löst ein soziales asynchrones Ereignis aus, wobei die zurückgegebene async_load DS-Map die folgenden Schlüssel/Wert-Paare enthält:

  • "type" - Dies ist der Typ des Ereignisses, das ausgelöst wurde. Für diese Funktion ist dies die Zeichenkette "achievement_increment_result".
  • "id" - Dies ist die ID des Ereignisses, und für diese Funktion wird eine GML Konstante GooglePlayServices_EVENT_ID_INCREMENT_ACHIEVEMENT zurückgegeben.
  • "achievement_id" - Dies ist die eindeutige ID der Leistung, die inkrementiert wurde.
  • "Status" - Dieser Wert ist 1, wenn die Leistung erfolgreich erhöht wurde, oder 0, wenn die Anfrage fehlgeschlagen ist.
  • "complete" - Dieser Wert ist 1, wenn die Leistung bis zu diesem Inkrement abgeschlossen wurde, oder 0, wenn der Antrag fehlgeschlagen ist.

Syntax:

achievement_increment(name, value)

Argument Beschreibung
name Der Name/die ID der Leistung.
value Der Wert, der als Inkrement gesendet werden soll.

Rückgabe:

N/A

Beispiel:

achievement_increment(global.Achievement[0], score);

Die obige code fügt den aktuellen Punktestand des Spielers zum Erfolg mit der ID hinzu, die im globalen Array gespeichert ist.
