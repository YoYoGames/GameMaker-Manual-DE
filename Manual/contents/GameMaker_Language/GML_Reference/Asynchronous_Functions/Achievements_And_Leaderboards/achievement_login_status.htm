leistung_anmeldung_status

Diese Funktion ist derzeit nur für Google Play auf Android platform nützlich. Sie prüft, ob der Nutzer bereits bei den Google-Diensten angemeldet ist. In diesem Fall gibt sie true zurück, andernfalls false. Wenn sie true zurückgibt, muss die Funktion achievement_login() Funktion aufzurufen.

Syntax:

achievement_login_status();

Rückgabe:

Boolean

Beispiel:

if !achievement_login_status()
    {
    achievement_login();
    }

Über die obige Website code können Sie sich bei dem von Ihnen gewählten Dienst für Bestenlisten und Erfolge anmelden.
