leistung_post

Mit dieser Funktion können Sie Ihre Errungenschaften an den gewählten Leaderboard- und Achievement-Service senden. Auf Android benötigen Sie die ID der Errungenschaft als String (dies ist die eindeutige ID der Errungenschaft, die beim Einrichten der Errungenschaft zugewiesen wurde), auf allen anderen platforms geben Sie den Namen der Leistung als String an und dann den Prozentsatz, den Sie für die Erreichung der Leistung erreicht haben (0 - keine, 100 - erreicht). Die Funktion löst ein soziales asynchrones Ereignis aus, wobei die zurückgegebene async_load DS-Map die folgenden Schlüssel/Wert-Paare enthält:

  • "type" - Dies ist der Typ des Ereignisses, das ausgelöst wurde. Für diese Funktion ist dies die Zeichenfolge "achievement_post_result".
  • "id" - Dies ist die ID des Ereignisses, und für diese Funktion wird eine GML Konstante GooglePlayServices_EVENT_ID_POST_ACHIEVEMENT zurückgegeben.
  • "achievement_id" - Dies ist die eindeutige ID der Leistung, die aktualisiert wurde.
  • "Status" - Dieser Wert ist 1, wenn die Leistung erfolgreich aktualisiert wurde, oder 0, wenn die Anfrage fehlgeschlagen ist.

Syntax:

achievement_post(achievement, percent)

Argument Beschreibung
achievement Der Name/die ID der Leistung.
percent Der Prozentsatz der erbrachten Leistung.

Rückgabe:

N/A

Beispiel:

if achievement_available()
    {
    achievement_post(global.Achievement[0], 100);
}

Die obige code prüft, ob der gewählte Dienst für Bestenlisten und Erfolge verfügbar ist, und wenn ja, wird ein Erfolg veröffentlicht.
