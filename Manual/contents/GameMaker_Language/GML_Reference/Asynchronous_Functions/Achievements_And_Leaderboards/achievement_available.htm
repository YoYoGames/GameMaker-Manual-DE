leistung_verfügbar

Diese Funktion gibt true zurück, wenn der Benutzer gerade mit dem Internet verbunden ist und das gewählte Ranglisten- und Achievement-System verfügbar ist, andernfalls gibt sie false zurück.

Syntax:

achievement_available()

Rückgabe:

Boolean

Beispiel:

if achievement_available()
    {
    achievement_post("ach_Level1", 100);
    }

Die obige Seite code prüft, ob der Leaderboard-Dienst verfügbar ist, und wenn ja, wird ein Beitrag an ihn gesendet.
