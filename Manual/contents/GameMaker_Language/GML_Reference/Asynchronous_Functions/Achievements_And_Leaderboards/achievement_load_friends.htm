leistung_laden_freunde

Diese Funktion sendet eine Anfrage an server, um Informationen über alle Freunde des angemeldeten Benutzers zu erhalten und löst einen Callback Social Asynchronous Event aus, der Folgendes enthält die async_load Map mit den relevanten Schlüssel/Wert-Paaren enthält. Der id-Schlüssel dieser DS-Map wird verwendet, um den richtigen Callback zu identifizieren (es kann es mehr als eine Auslösefunktion für ein bestimmtes asynchrones Ereignis geben) und wird mit der Konstante achievement_friends_info sowie einer Reihe anderer Schlüssel/Wertpaare für jeden Freund gepaart. Der genaue Inhalt der Map ist unten dargestellt:

HINWEIS: Diese Funktion ist nur für iOS verfügbar.

  • "id" - Für diese Funktion sollte es achievement_friends_info sein
  • "FriendN" - Der Name des Freundes, wobei "N" ein ganzzahliger Wert ist, der seiner Position in der Freundesliste entspricht.
  • "FriendidN" - Die eindeutige Benutzerkennung des Freundes, "N".

Syntax:

achievement_load_friends()

Rückgabe:

N/A

Erweitertes Beispiel:

Die folgende code würde wahrscheinlich aufgerufen werden, nachdem sich der Spieler in sein Spielkonto eingeloggt hat, indem er achievement_login() um eine Liste aller Freunde des Spielers zu erhalten:

achievement_friends_info();

Dadurch wird eine Anfrage nach den Informationen über die Freunde des Benutzers gesendet und ein asynchroner Rückruf mit der speziellen async_load DS-Map erzeugt, die die folgenden Daten enthält:

var ident = ds_map_find_value(async_load, "id");
if ident == achievement_friends_info
    {
    var numfriends = ds_map_find_value(async_load, "numfriends");
    global.numfriends = numfriends;
    for(var i=0; i < numfriends; i++;)
        {
        global.friendname[i] = ds_map_find_value(async_load, "Friend" + string(i));
        global.friendid[i] = ds_map_find_value(async_load, "Friendid" + string(i));
        achievement_get_pic(global.friendid[i]);
        }
    }

Die obige code prüft die zurückgegebene DS Map im Social Asynchronous Event und wenn deren "id" mit der zu prüfenden Konstante übereinstimmt, dann loops durch die Map, die alle verschiedenen Werte in einer Reihe von Arrays speichert und fordert ein Bild für jeden der Einträge in der Karte an (siehe achievement_get_pic()).
