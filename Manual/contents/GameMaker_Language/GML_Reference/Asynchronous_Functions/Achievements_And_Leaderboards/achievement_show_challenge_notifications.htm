leistung_challenge_benachrichtigungen_anzeigen

Mit dieser Funktion können Sie die verschiedenen "Toast"-Popup-Benachrichtigungen zu Herausforderungen ein- oder ausblenden. Wenn arguments auf true eingestellt ist, erscheinen diese Meldungen und informieren den Spieler über alle lokalen oder entfernten Herausforderungen. sowie über abgeschlossene Herausforderungen, während die Einstellung false diese Benachrichtigungen unterdrückt.

Syntax:

achievement_show_challenge_notifications(receive, local, remote)

Argument Beschreibung
receive Pop-up-Fenster für empfangene Herausforderungen anzeigen(true) oder nicht(false).
local Pop-up-Fenster für abgeschlossene lokale Herausforderungen anzeigen(true) oder nicht(false).
remote Pop-up-Fenster für abgeschlossene Fernaufgaben anzeigen(true) oder nicht(false).

Rückgabe:

N/A

Beispiel:

achievement_show_challenge_notifications(true, false, false);

Mit der obigen code können Sie festlegen, dass Toast-Benachrichtigungen nur für empfangene Herausforderungen angezeigt werden.
