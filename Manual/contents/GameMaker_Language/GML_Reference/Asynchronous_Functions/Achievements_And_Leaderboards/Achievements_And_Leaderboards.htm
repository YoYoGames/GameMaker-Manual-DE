Errungenschaften und Bestenlisten

Spiele, die für Geräte entwickelt wurden, auf denen iOS oder Android läuft, können auch mit einer von zwei APIs für Bestenlisten und Erfolge verbunden werden. Für iOS gibt es iTunes Connect, und für Android können Sie Google Play verwenden, wobei beide beide nutzen die Funktionen auf dieser Seite. Diese Funktionen ermöglichen es Ihrem Spiel, Informationen über Spielstände und Erfolge an die Bestenlisten zu senden, die Sie im Entwicklerkonto für das von Ihnen entwickelte Spiel eingerichtet haben. Dies ist eine hervorragende Möglichkeit, die Spieler in die soziale Seite des Spiels einzubinden, da sie dann sehen können, was andere Personen, die Ihr Spiel gespielt haben, erreicht haben, und versuchen können, sie zu übertreffen.

Wenn Sie diese Funktionen verwenden, sollten Sie beachten, dass sie Erfolge und Punktestände nicht automatisch für Sie erfassen. Sie zeigen nur die Daten an, die du ihnen aus deinem Spiel sendest, also musst du code dein eigenes Achievement- und Hi-Score-System verwenden, um den Status der einzelnen Personen im Spiel zu verfolgen, und dann diese Funktionen verwenden, um die Daten zum gewählten Netzwerk hinzuzufügen (Sie können jedoch Fortschrittsinformationen für Erfolge mit der leistung_laden_fortschritt() Funktion).

Einige dieser Funktionen lösen ein soziales asynchrones Ereignis aus, das eine DS-Map mit Daten über die auslösende Funktion enthält. der Funktion, die es ausgelöst hat, enthält. Die Funktionen, die das Ereignis auslösen, geben auch die Schlüssel/Wert-Paare der Map an, die vom Callback zurückgegeben werden, aber ein Callback Social Event kann auch dadurch ausgelöst werden, dass Ihr Spiel externe Benachrichtigungen über Herausforderungen usw. erhält.

HINWEIS: Damit diese Funktionen mit iOS funktionieren, müssen Sie ein gültiges Game Center-Konto und eine iOS App-ID für Ihr Spiel haben.

HINWEIS: Damit diese Funktionen mit Android funktionieren, müssen Sie Ihr Spiel bei Google Play einreichen und die erforderlichen Abschnitte einrichten sowie die API in den Android-Spieloptionen für für Ihr Spiel aktiviert haben.

Die unten aufgeführten Funktionen sind die grundlegenden Funktionen für die Integration Ihres Spiels in die gewählte Plattform für soziale Spiele:

  • achievement_login_status
  • achievement_login
  • achievement_logout
  • achievement_available
  • achievement_post
  • achievement_post_score
  • achievement_increment
  • achievement_show_achievements
  • achievement_show_leaderboards
  • achievement_show_challenge_notifications
  • achievement_get_challenges
  • achievement_get_pic
  • achievement_load_friends
  • achievement_load_leaderboard
  • achievement_load_progress
  • achievement_send_challenge
  • achievement_reset
