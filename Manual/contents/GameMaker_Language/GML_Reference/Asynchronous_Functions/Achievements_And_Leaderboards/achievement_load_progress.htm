leistung_laden_fortschritt

Diese Funktion sendet eine Anfrage an server, um Informationen über alle verfügbaren Erfolge zu erhalten. Sie löst einen Callback Social Asynchronous Event aus, der Folgendes enthält die async_load Map mit den relevanten Schlüssel/Wert-Paaren enthält. Der id-Schlüssel dieser DS-Map wird verwendet, um den richtigen Callback zu identifizieren (es kann es mehr als eine Auslösefunktion für ein bestimmtes asynchrones Ereignis geben), und wird mit der Konstante achievement_achievement_info sowie einer Reihe anderer Schlüssel/Wertpaare für jeden Spieler gepaart. Der genaue Inhalt der Karte ist unten dargestellt:

HINWEIS: Diese Funktion ist nur für iOS verfügbar.

  • "id" - Für diese Funktion sollte es achievement_achievement_info sein.
  • "numentries" - Die Anzahl der Leistungen in der Liste.
  • "AchN" - Der Name der Leistung, wobei "N" ein ganzzahliger Wert ist, der seinem Platz in der Liste der Einträge entspricht.
  • "AchCompN" - Wie vollständig die Leistung "N" ist, als Prozentwert von 0 bis 100 (eine Zeichenkette).

Syntax:

achievement_load_progress()

Rückgabe:

N/A

Erweitertes Beispiel:

Die folgende code würde den Leistungsfortschritt abfragen, wenn sich der Benutzer auf einem iOS Gerät befindet:

if os_type == os_ios
    {
    achievement_load_progress();
    }

Dadurch wird eine Anfrage für die Informationen zu den Errungenschaften gesendet und ein asynchroner Rückruf mit der speziellen async_load DS-Map erzeugt, die die folgenden Daten enthält:

var ident = ds_map_find_value(async_load, "id");
if ident == achievement_achievement_info
    {
    var numentries = ds_map_find_value(async_load, "numentries");
    for(var i = 0; i < numentries; i++;)
        {
        ach_id[i] = ds_map_find_value(async_load, "Ach" + string(i));
        comp[i] = ds_map_find_value(async_load, "AchComp" + string(i));
        }
    }

Die obige code prüft die zurückgegebene DS-Map im sozialen asynchronen Ereignis und wenn ihre "id" mit der zu prüfenden Konstante übereinstimmt, dann loops durch die Einträge in der Map für jeden der Erfolge und speichert ihren Namen und den Fertigstellungsgrad.
