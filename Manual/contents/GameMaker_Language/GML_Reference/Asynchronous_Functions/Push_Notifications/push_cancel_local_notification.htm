push_cancel_local_notification

Diese Funktion bricht die angegebene Push-Benachrichtigung ab. Sie können die ID der abzubrechenden Benachrichtigung mit den Funktionen push_get_first_local_notification() und push_get_next_local_notification() und die Funktion gibt bei Erfolg true oder andernfalls false zurück (z. B. wenn die Benachrichtigung nicht existiert oder bereits ausgelöst wurde).

HINWEIS: Diese Funktion ist auf das iOS- und Android-Ziel modules beschränkt.

Syntax:

push_cancel_local_notification(ID);

Argument Typ Beschreibung
ID ID der Push-Benachrichtigung The ID of the notification to cancel.

Rückgabe:

Boolean

Beispiel:

var map = ds_map_create();
var ntf = push_get_first_local_notification(map);
while(ntf >= 0)
{
    var data = ds_map_find_value(map, "data");
    if data == "Daily_Reward"
    {
        push_cancel_local_notification(ntf);
    }
    ntf = push_get_next_local_notification(map);
}
ds_map_destroy(map);

Die obige code erstellt eine DS Map und fordert dann Daten für die erste verfügbare Meldung in der Warteschlange an. Wenn eine Meldung vorhanden ist, wird die Karte ausgefüllt, und code überprüft dann den Schlüssel "data" auf eine bestimmte Zeichenfolge. Wird die Zeichenfolge gefunden, wird die Meldung abgebrochen und die nächste Meldung geprüft, bis alle Meldungen in der Warteschlange geprüft wurden.
