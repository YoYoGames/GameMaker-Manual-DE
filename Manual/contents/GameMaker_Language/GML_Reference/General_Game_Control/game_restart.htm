game_restart

Mit dieser Funktion können Sie das Spiel neu starten. Dies ist im Wesentlichen dasselbe wie das erstmalige Starten des Spiels, so dass sowohl das Spielstart-Ereignis als auch das Spielende-Ereignis ausgelöst werden.

Es ist zu beachten, dass bestimmte Dinge nicht zurückgesetzt werden, wenn diese Funktion aufgerufen wird:

  • Globale Variablen werden nicht neu initialisiert, es sei denn, sie sind explizit als solche kodiert - zum Beispiel wird die eingebaute globale Variable score nach einem Neustart des Spiels nicht bei Null beginnen, wenn sie im Spiel bereits verändert wurde.
  • Der GPU-Zustand wird nicht geändert (wenn Sie also z. B. die Zeichenfarbe oder Alpha eingestellt haben, bleibt der geänderte Wert erhalten).
  • Die Spielgeschwindigkeit bleibt so, wie Sie sie in Ihrem Spiel code eingestellt haben (wenn Sie sie geändert haben, wird diese Änderung beibehalten).
  • Jedes Asset aus dem Asset-Browser, das zur Laufzeit im Spiel geändert wurde - zum Beispiel, wenn Sie den Ursprung für eine Ressource sprite ändern oder die Position einer Pfadressource verschieben - wird nicht zurückgesetzt.
  • Dynamische Ressourcen wie buffers, Oberflächen, Datenstrukturen oder importierte sprites werden ebenfalls nicht bereinigt oder entfernt (obwohl Sie möglicherweise Verweise auf sie verlieren, also achten Sie bei der Verwendung dieser Funktion darauf, entweder globale Verweise für die dynamische Ressource zu verwenden oder sie zu bereinigen, bevor die Funktion aufgerufen wird).

 

Syntax:

game_restart();

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(ord("R")) game_restart();

Dadurch würde das Spiel neu gestartet, wenn der Spieler die Taste "R" drückt.
