game_id

Diese schreibgeschützte Variable gibt den eindeutigen Bezeichner für das von Ihnen erstellte Spiel zurück. Sie können diese Variable verwenden, wenn Sie einen eindeutigen Dateinamen oder eine andere Kennung für Ihr Spiel benötigen. Dies kann in den Spieloptionen eingestellt werden.

Syntax:

game_id

Rückgabe:

Real

Beispiel:

ini_open("Score.ini");
ini_write_real("Scores", "0", score + game_id);
ini_close();

Die obige code verschlüsselt den Spielstand, indem sie vor dem Speichern die game_id hinzufügt. Beim erneuten Einlesen in das Spiel wird die game_id abgezogen, um den korrekten Wert wieder zu erhalten.
