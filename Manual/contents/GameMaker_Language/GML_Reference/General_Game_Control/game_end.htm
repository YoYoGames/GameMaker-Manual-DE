spiel_ende

Mit dieser Funktion können Sie das Spiel beenden (und das Game End Event wird ausgelöst). Dies geschieht nicht sofort, sondern erst am Ende des aktuellen Schrittes, so dass jeder Code der sich nach dem Aufruf dieser Funktion im selben Schritt befindet, wird weiterhin ausgeführt. Bitte beachten Sie, dass diese Funktion die folgenden Einschränkungen hat:

  • Auf Android -Geräten wird die App durch den Aufruf von game_end() in den Hintergrund geschoben, aber nicht geschlossen. Dies muss der Benutzer selbst tun.
  • Auf iOS wird es nichts tun und stillschweigend fehlschlagen.
  • Auf allen Konsolen (einschließlich UWP, wenn es auf einer Xbox läuft) kann das Spiel abstürzen oder bestenfalls stillschweigend fehlschlagen, und es kann auch ein Übermittlungsfehler sein, daher darf diese Funktion nicht verwendet werden.
  • Auf HTML5 wird das Spiel beendet, aber der Benutzer starrt auf eine leere Leinwand auf der Webseite, und als solche ist es nicht empfohlen, dass es auf diesem Ziel platform verwendet werden.
  • Auf Windows (einschließlich UWP auf einem PC), Linux und macOS beendet die Funktion einfach das Spiel und schließt das Spielfenster (das Game End Event wird ebenfalls ausgelöst).

Syntax:

game_end();

Rückgabe:

Beispiel:

if keyboard_check_pressed(vk_escape) game_end();

Dies würde das Spiel beenden, wenn der Spieler die "Escape"-Taste drückt.
