spiel_anzeige_name

Diese schreibgeschützte Variable gibt den Anzeigenamen Ihres Spiels für das Ziel platform zurück, wie er in den Spieloptionen festgelegt wurde.

Syntax:

game_display_name

Rückgabe:

Beispiel:

var name = game_display_name;
var ver = string(GM_version);
draw_text(32, 32, name + ":" + ver);

Die obige code ruft den Anzeigenamen und die Versionsnummer des Spiels ab und zeichnet sie ein.
