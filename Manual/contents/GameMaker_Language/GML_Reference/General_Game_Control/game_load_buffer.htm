spiel_laden_puffer

Mit dieser Funktion können Sie einen Spielzustand laden, der zuvor gespeichert wurde. Das Spiel wird aus einem zuvor erstellten "grow" buffer (siehe Buffers) geladen und der buffer muss einen Spielstatus haben, der mit game_save_buffer() Funktion.

Syntax:

game_load_buffer(buffer);

ArgumentTyp Beschreibung
buffer The buffer id to load from.

Rückgabe:

Beispiel:

if keyboard_check_pressed(ord("L"))
{
    if (global.Checkpoint) game_load_buffer(save_buff);
}

Die obige code lädt einen zuvor gespeicherten Spielstand von der buffer, die in der Variablen "save_buff" indiziert ist, nur dann, wenn die globale Variable true ist, wenn der Spieler die Taste "L" drückt.
