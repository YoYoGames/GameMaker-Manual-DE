game_get_speed

Diese Funktion kann verwendet werden, um die Spielgeschwindigkeit entweder als Anzahl der Spielframes pro Sekunde oder als Anzahl der Mikrosekunden pro Spielframe zu erhalten. Beachten Sie, dass dies nicht der tatsächliche Wert der Spielgeschwindigkeit (FPS) ist (verwenden Sie dazu die fps_real Variable), sondern die Anzahl der Spielframes (FPS), die das Spiel pro Sekunde zu halten versucht, oder die Länge jedes Spielframes in Mikrosekunden, die das Spiel zu halten versucht (MPF). Wenn Sie diese Funktion verwenden, müssen Sie eine der folgenden Konstanten angeben, die den Typ des Rückgabewerts bestimmen:

Spielgeschwindigkeit konstant
Konstante Beschreibung
gamespeed_fps Ermittelt die Spielgeschwindigkeit in Bildern pro Sekunde.
gamespeed_microseconds Ermittelt die Spielgeschwindigkeit in Mikrosekunden pro Frame.

Wenn zum Beispiel die Spielgeschwindigkeit in den Spieloptionen auf 30 eingestellt ist und Sie den FPS-Typ verwenden, gibt die Funktion 30 zurück, aber wenn Sie den MPF-Typ verwenden, gibt die Funktion 33333 zurück.

Syntax:

game_get_speed(type);

Argument Typ Beschreibung
type Spielgeschwindigkeit konstant The type of method used to get the game speed (see the constants above).

Rückgabe:

Real

Beispiel:

if game_get_speed(gamespeed_fps) != 60
{
    game_set_speed(60, gamespeed_fps);
}

Die obige code prüft, ob das Spiel mit einer Spielgeschwindigkeit von 60 FPS läuft und wenn nicht, wird es auf 60 FPS eingestellt.
