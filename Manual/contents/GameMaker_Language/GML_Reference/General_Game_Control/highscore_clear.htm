highscore_clear

Mit dieser Funktion können Sie die interne globale Highscore-Liste löschen, indem Sie alle Namen auf den Standardwert "Unbekannt" und alle Punktzahlen auf den Standardwert 0 zurücksetzen.

Syntax:

highscore_clear();

Rückgabe:

Beispiel:

highscore_clear();

Dies würde die Highscore-Liste löschen.
