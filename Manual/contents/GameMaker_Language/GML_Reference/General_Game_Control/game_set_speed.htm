Spiel_Geschwindigkeit_einstellen

Mit dieser Funktion können Sie die Spielgeschwindigkeit einstellen. Sie können diese auf zwei Arten einstellen - entweder als Spiel-Frames pro Sekunde (FPS) oder als Mikrosekunden pro Spiel-Frame (MPF) - unter Verwendung einer der beiden folgenden Konstanten:

Konstante Beschreibung
gamespeed_fps Legt die Spielgeschwindigkeit in Bildern pro Sekunde fest.
gamespeed_microseconds Legt die Spielgeschwindigkeit in Mikrosekunden pro Frame fest.

So würde zum Beispiel eine Spielgeschwindigkeit von 30 Bildern pro Sekunde 33333 Mikrosekunden pro Spielbild entsprechen, was dann durch diese Funktion wie folgt ausgedrückt werden würde:

game_set_speed(30, gamespeed_fps);

oder:

game_set_speed(33333, gamespeed_microseconds);

Syntax:

game_set_speed(speed, type);

Argument Typ Beschreibung
speed Real The new game speed (as either FPS or MPF).
type Spielgeschwindigkeit konstant The type of method used to set the game speed (see the constants above).

Rückgabe:

N/A

Beispiel:

if os_browser == browser_not_a_browser
{
    game_set_speed(60, gamespeed_fps);
}
else
{
    game_set_speed(30, gamespeed_fps);
}

Die obige code prüft, ob das Spiel in einem Browser läuft und setzt die Spielgeschwindigkeit entsprechend als FPS-Wert.
