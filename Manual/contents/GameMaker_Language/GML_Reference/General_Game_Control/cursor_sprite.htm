cursor_sprite

Das Setzen dieser Variable veranlasst GameMaker die angegebene sprite als Cursor zu verwenden (im Grunde wird sie bei jedem Schritt auf die aktuelle x/y-Position der Maus gesetzt). Der Standardwert ist -1, also kein sprite für den Cursor, aber Sie können einen beliebigen sprite -Index aus den Spiel-Assets oder aus einer externen Ressource importiert zuweisen. Bitte beachten Sie, dass es keine Möglichkeit gibt, die Animationsgeschwindigkeit oder den image_index zu steuern. Wenn sprite also Unterbilder hat, werden diese mit der gleichen Geschwindigkeit wie die Raumgeschwindigkeit durchlaufen. Um den Cursor sprite zu entfernen, können Sie diese Variable wieder auf -1 setzen.

Zu beachten ist auch, dass diese Variable nicht den Cursor des Spielfensters ersetzt, sondern dass dieser weiterhin wie gewohnt gezeichnet wird. Um dies zu vermeiden, können Sie die Funktion window_set_cursor() verwenden und sie auf die Konstante cr_none setzen, wodurch der Standardcursor unsichtbar wird.

Syntax:

cursor_sprite;

Rückgabe:

Sprite Asset

Beispiel:

cursor_sprite = spr_CustomCursor;

Die obige code setzt die sprite, die in der Variablen "spr_CustomCursor" indiziert ist, als Cursor sprite für das Spiel.
