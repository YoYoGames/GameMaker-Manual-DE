highscore_add

Mit dieser Funktion können Sie einen Namen und einen Punktestand in die interne globale Highscore-Liste eintragen. Es ist nicht nötig zu prüfen, ob der Wert hoch genug ist, um in die Highscore-Liste aufgenommen zu werden, da GameMaker nur die Werte speichert, die größer als die zehnte gespeicherte Position sind.

Syntax:

highscore_add(str, numb);

Argument Typ Beschreibung
str Zeichenfolge The string (name) to attribute the score to.
numb Real The number (score) to add.

Rückgabe:

N/A

Beispiel:

if (lives == 0)
{
    highscore_add(global.Name, score);
    score = 0;
    room_goto(rm_MainMenu);
}

Das obige code prüft die Leben, und wenn sie auf 0 gesetzt sind, fügt es den aktuellen Punktestand und die in der globalen Variable "Name" gespeicherte Zeichenkette in die Highscore-Liste ein, bevor es den Punktestand schließlich auf 0 setzt und den Spieler in den in "rm_MainMenu" indizierten Raum schickt.
