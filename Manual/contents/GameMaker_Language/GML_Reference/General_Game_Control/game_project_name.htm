spiel_projekt_name

Diese schreibgeschützte Variable gibt den Anzeigenamen Ihres Spiels für das Ziel platform in einem "speicherfreundlichen" Format für das Ziel platform zurück. Wenn der Anzeigename Zeichen enthält, die für einen Dateinamen nicht zulässig sind, werden diese werden diese automatisch durch "_" ersetzt. Der Anzeigename kann in den Spieloptionen eingestellt werden. Da es für HTML5 keine Einschränkungen für Dateinamen gibt, wird diese Zeichenfolge wahrscheinlich die gleiche sein wie die von wie die von game_display_name.

Syntax:

game_project_name

Rückgabe:

Beispiel:

var file = game_project_name + ".ini";
ini_open(file);
ini_write_real("Scores", "Highscore", score);
ini_close();

Die obige code ruft den Projektnamen ab und verwendet diesen, um eine Ini-Datei zu öffnen (oder zu erstellen), in die dann geschrieben wird.
