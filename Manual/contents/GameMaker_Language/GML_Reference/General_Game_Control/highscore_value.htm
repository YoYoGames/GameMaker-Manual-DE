highscore_wert

Mit dieser Funktion können Sie den Punktestand abrufen, der in der Highscore-Liste an der angegebenen Position gespeichert wurde. Wenn keine Punktzahl eingegeben wurde, gibt die Funktion 0 zurück.

Syntax:

highscore_value(place);

ArgumentTyp Beschreibung
place The place on the table (1-10).

Rückgabe:

Beispiel:

var i = 9;
repeat(10)
{
    scr[i] = highscore_value(i + 1);
    i -= 1;
}

code Die obige loop durchsucht die Highscore-Liste und speichert alle Ergebnisse in einem Array.
