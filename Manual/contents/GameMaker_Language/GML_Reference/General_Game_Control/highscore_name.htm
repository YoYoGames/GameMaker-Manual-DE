highscore_name

Mit dieser Funktion können Sie den Namensstring abrufen, der in der Highscore-Liste an der angegebenen Position gespeichert wurde. Wenn kein Name eingegeben wurde, wird die Zeichenkette "Unbekannt" zurückgegeben.

Syntax:

highscore_name(place);

ArgumentTyp Beschreibung
place The place on the table (1-10).

Rückgabe:

Beispiel:

var i = 9;
repeat(10)
{
    name[i] = highscore_name(i + 1);
    i -= 1;
}

code Die obige Seite loop durchsucht die Highscore-Liste und speichert alle Namen in einem Array.
