game_save_id

Diese schreibgeschützte Variable gibt die vollständige Pfad-ID des Verzeichnisses zurück, das von Ihrem Spiel zum Speichern von Dateien verwendet wird. Dieses Verzeichnis kann für andere Anwendungen sichtbar sein oder auch nicht, je nach platform.

Auf dem HTML5-Ziel wird ein leerer String zurückgegeben.

HINWEIS Diese Variable dient nur zur Information und sollte nicht verwendet werden, um einen Dateipfad oder etwas Ähnliches zu erhalten, da sie nicht in allen platforms korrekt funktioniert.

Syntax:

game_save_id

Rückgabe:

String

Beispiel:

save_dir = game_save_id;

Damit wird das Verzeichnis zum Speichern von Dateien in einer Variablen für die zukünftige Verwendung gespeichert.
