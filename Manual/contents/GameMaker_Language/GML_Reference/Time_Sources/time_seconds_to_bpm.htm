zeit_sekunden_bis_bpm

Diese Funktion nimmt die Länge eines Schlags in Sekunden und gibt einen Wert für die Schläge pro Minute zurück.

Syntax:

time_seconds_to_bpm(seconds);

Argument Typ Beschreibung
seconds Real The seconds-per-beat value to convert into beats-per-minute

Rückgabe:

Real

Beispiel:

var _period = time_source_get_period(time_source);
var _bpm = time_seconds_to_bpm(_period);

show_debug_message(_bpm);

Diese Funktion ermittelt die Periode einer Zeitquelle, wandelt sie in BPM um und druckt diesen Wert in das Ausgabeprotokoll.

Diese code folgt dem Beispiel für time_bpm_to_seconds().
