Anruf_später

Diese Funktion löst nach der angegebenen Zeit oder nach einem bestimmten Intervall eine Callback-Funktion aus. Sie tut dies, indem sie eine Zeitquelle erstellt (siehe Zeitquellen).

Um die Rückruffunktion abzubrechen, siehe call_cancel.

WICHTIG Während diese Funktion eine tatsächliche Zeitquellen-ID zurückgibt, kann diese ID nicht mit den Zeitquellenfunktionen verwendet werden, da die von call_later und call_cancel verwendeten Zeitquellen "unsichtbar" sind - sie erscheinen nicht als Kinder von eingebauten Zeitquellen und können nicht mit anderen Zeitquellenfunktionen verwendet werden. Sie sind jedoch in der von debug_event("ResourceCounts") zurückgegebenen Zeitquellenanzahl enthalten.

Argumente

Es folgt eine Aufschlüsselung der Argumente dieser Funktion:

Zeitraum

Dies ist eine Zeitspanne, die in Sekunden oder Frames ausgedrückt werden kann, je nachdem, welche Einheit auf der nächsten Seite argument angegeben ist.

Wenn Sie Frames als Einheit verwenden, muss die Periode eine ganze Zahl sein. Nicht ganzzahlige Werte werden abgerundet, mit Ausnahme von Werten kleiner als 1, die auf 1 aufgerundet werden.

Einheit

Dies ist die Einheit, in der der Zeitraum ausgedrückt wird, und kann time_source_units_seconds oder time_source_units_frames sein.

Siehe: Zeitquelleneinheiten

Sie können einen Wert für die Schlagzahl pro Minute verwenden, indem Sie time_bpm_to_seconds() verwenden.

Rückruf

Sie müssen einen "Callback" angeben method angeben, der aufgerufen wird, wenn die Zeitquelle abläuft.

Schleife

Ob die Zeitquelle loop sein soll.

Syntax:

call_later(period, unit, callback, [loop=false]);

Argument Typ Beschreibung
period Real The period that the Time Source läuft für, in den angegebenen Einheiten
units Zeitquelle Einheit Konstante The units that the given period is in
callback Methode oder Skript Funktion The method or script function to call after the specified time
loop Boolesche OPTIONAL Ob die Zeitquelle in einer Schleife laufen soll (Standard ist false)

Rückgabe:

Time Source ID

Beispiel 1:

var _callback = function()
{
    show_debug_message("Five seconds have passed!");
}
var _handle = call_later(5, time_source_units_seconds, _callback);

In diesem Beispiel erstellen wir eine Callback-Funktion, die nach Ablauf von fünf Sekunden eine Debug-Meldung anzeigt. Danach wird die zugrunde liegende Zeitquelle zerstört.

Beispiel 2:

var _create_enemy = function()
{
    var _x = random(room_width);
    var _y = random(room_height);
    instance_create_depth(_x, _y, depth, obj_enemy);
}

var _handle = call_later(10, time_source_units_seconds, _create_enemy, true);

In diesem Beispiel erstellen wir eine Callback-Funktion, die einen neuen Feind an einer zufälligen Position im Raum erzeugt. Diese Funktion wird dann alle 10 Sekunden ausgeführt, indem sie an call_later übergeben wird. Die letzte argument wird auf true gesetzt, so dass die Funktion weiterhin alle 10 Sekunden aufgerufen wird.

Verwenden Sie call_cancel mit dem von call_later zurückgegebenen Handle, um die Zeitquelle und die Aufrufe an den Callback zu beenden.
