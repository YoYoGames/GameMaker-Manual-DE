time_source_get_children

Für die angegebene Zeitquelle gibt diese Funktion ein Array zurück, das die Zeitquellen-IDs ihrer Kinder enthält.

Syntax:

time_source_get_children(id);

Argument Typ Beschreibung
id Zeitquellen-ID The Time Source um die Kinder von

Rückgabe:

Array of Time Source ID

Beispiel:

var _children = time_source_get_children(time_source);
var _count = array_length(_children);

for (var i = 0; i < _count; i ++)
{
    time_source_destroy(_children[i]);
}

time_source_destroy(time_source);

Diese code geht durch alle Kinder einer Zeitquelle und zerstört sie eines nach dem anderen.

Am Ende zerstört sie die übergeordnete Zeitquelle.
