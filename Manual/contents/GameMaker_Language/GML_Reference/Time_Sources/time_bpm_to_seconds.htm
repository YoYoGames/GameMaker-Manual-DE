time_bpm_to_seconds

Diese Funktion nimmt einen Wert für die Schläge pro Minute und gibt die Länge jedes Schlags in Sekunden zurück.

Dies kann beim Erstellen einer Zeitquelle verwendet werden, um einen BPM-Wert für die Zeitquelle zu verwenden. Es ist wichtig, dass eine solche Zeitquelle Sekunden als Einheit verwendet.

Syntax:

time_bpm_to_seconds(bpm);

Argument Typ Beschreibung
bpm Real The beats-per-minute value to convert into seconds-per-beat

Rückgabe:

Real

Beispiel:

var _bpm = 90;
var _seconds = time_bpm_to_seconds(_bpm);

time_source = time_source_create(time_source_game, _seconds, time_source_units_seconds, function()
{
    show_debug_message("BEAT!");
}, [], -1);

Diese code wandelt einen Wert von 90 BPM in Sekunden um und verwendet diesen, um eine Zeitquelle zu erstellen, die unbegrenzt läuft.

Bei jedem Beat wird die Meldung "BEAT!" in das Ausgabeprotokoll geschrieben.
