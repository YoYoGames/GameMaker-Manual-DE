time_source_get_state

Diese Funktion gibt den Status der angegebenen Zeitquelle zurück.

Syntax:

time_source_get_state(id);

Argument Typ Beschreibung
id Zeitquellen-ID The Time Source um den Zustand von

Rückgabe:

Time Source State Constant

Beispiel:

if (keyboard_check_pressed(vk_space))
{
    var _state = time_source_get_state(time_source);

    if (_state == time_source_state_active)
    {
        time_source_pause(time_source);
    }
    else if (_state == time_source_state_paused)
    {
        time_source_start(time_source);
    }
}

Wenn die Leertaste gedrückt wird, erhält diese code den Status einer Zeitquelle.

Wenn der Status aktiv ist, wird die Zeitquelle angehalten, und wenn sie angehalten wurde, wird sie wieder aufgenommen.
