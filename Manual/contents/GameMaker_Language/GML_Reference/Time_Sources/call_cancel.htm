Anruf_Abbrechen

Diese Funktion bricht eine Callback-Funktion ab, die zuvor durch call_later gestartet wurde.

Syntax:

call_cancel(handle);

Argument Typ Beschreibung
handle Zeitquellen-ID The handle to a Time Source von call_later zurückgegeben.

Rückgabe:

N/A

Beispiel:

call_cancel(handle);
