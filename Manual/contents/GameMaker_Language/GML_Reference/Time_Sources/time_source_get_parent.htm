time_source_get_parent

Diese Funktion gibt die Zeitquellen-ID der übergeordneten Zeitquelle zurück.

Der zurückgegebene Wert kann die ID einer benutzerdefinierten Zeitquelle oder einer der eingebauten Zeitquellen sein, je nachdem, wie die gegebene Zeitquelle erstellt wurde.

Syntax:

time_source_get_parent(id);

Argument Typ Beschreibung
id Zeitquellen-ID The Time Source um das Elternteil von

Rückgabe:

Time Source ID

Beispiel:

var _parent = time_source_get_parent(time_source);

Diese code ruft das Elternteil einer Zeitquelle ab und speichert es in einer lokalen Variablen.
