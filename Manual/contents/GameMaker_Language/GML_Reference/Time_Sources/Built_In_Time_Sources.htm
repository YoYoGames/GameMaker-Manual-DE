Eingebaute Zeitquellen

Zeitquelle Konstante
Konstante Beschreibung Wert
time_source_global Die globale Zeitquelle 0
time_source_game Die Spielzeitquelle 1

Sie können die globale Zeitquelle oder die Spielzeitquelle als übergeordnete Zeitquelle für Ihre benutzerdefinierte Zeitquelle verwenden. Beide Zeitquellensind global verfügbar.

Die globale Zeitquelle läuft außerhalb des Hauptspiels loop, während die Spielzeitquelle als Teil des Spiels loop läuft.

Die Wahl zwischen Global/Game hat keinen Einfluss darauf, ob die Zeitquellenvon der Framerate des Spiels beeinflusst werden; das hängt von der verwendeten Einheit ab.

Bestellung

Zeitquellen, die von der globalen Zeitquelle abstammen, werden vor der Spielzeitquelle verarbeitet.

Staaten

Die Spielzeitquelle hat Zustände, das heißt, sie kann paused. Die globale Zeitquelle hat keine Zustände und kann nicht pausiert werden.

Sie können die globale Zeitquelle für Timer verwenden, die unabhängig vom Spielstatus laufen sollen, und die Spielzeitquelle für Timer, die an Ihr Hauptspiel gebunden sind loop. Sie können dann alle Ihre spielbezogenen Zeitgeber anhalten, indem Sie die Spielzeitquelle pausieren.
