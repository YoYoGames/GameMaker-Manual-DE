time_source_get_reps_remaining

Diese Funktion gibt die Anzahl der Wiederholungen zurück, die die angegebene Zeitquelle noch hat, bis sie vollständig stoppt.

Die Funktion gibt einen undefinierten Wert zurück, wenn die angegebene Zeitquelle unendlich lange läuft oder wenn es sich um eine eingebaute Zeitquelle handelt.

Syntax:

time_source_get_reps_remaining(id);

Argument Typ Beschreibung
id Zeitquellen-ID The Time Source um Informationen zu erhalten für

Rückgabe:

Real
