zeit_quelle_get_units

Diese Funktion gibt die für die angegebene Zeitquelle eingestellten Einheiten zurück. Der Periodenwert für die Zeitquelle wird in diesen Einheiten ausgedrückt.

Syntax:

time_source_get_units(id);

Argument Typ Beschreibung
id Zeitquellen-ID The Time Source um die Einheiten für

Rückgabe:

Time Source Unit Constant
