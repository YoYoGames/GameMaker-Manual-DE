time_source_get_reps_completed

Diese Funktion gibt die Anzahl der Wiederholungen zurück, die die angegebene Zeitquelle seit ihrer letzten Rücksetzung durchgeführt hat. Das ist die Anzahl der Male, die sie abgelaufen ist und ihre Callback-Methode ausgeführt hat.

Syntax:

time_source_get_reps_completed(id);

Argument Typ Beschreibung
id Zeitquellen-ID The Time Source um Informationen zu erhalten für

Rückgabe:

Real
