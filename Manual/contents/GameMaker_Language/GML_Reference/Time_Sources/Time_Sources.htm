Zeit-Quellen

Eine "Zeitquelle" ist ein benutzerdefinierter Zeitgeber, den Sie erstellen. Er läuft für eine bestimmte Zeitspanne und läuft am Ende ab.

Eine Zeitquelle (TS) hat die Möglichkeit, bei Ablauf eine benutzerdefinierte Methode ("Callback") aufzurufen.

Sie ist auch in der Lage, sich eine bestimmte Anzahl von Malen oder für immer zu wiederholen. Jede Wiederholung einer Zeitquelle wird als "Wiederholung" bezeichnet.

Wie verwendet man Zeitquellen?

Erstellen und starten Sie eine Zeitquelle, die eine benutzerdefinierte Methode aufruft:

  • Rufen Sie auf. time_source_create() / Zeitquelle erstellen, um eine Zeitquellezu erstellen
    • Geben Sie an, ob sie von der globalen Zeitquelle, der Spielzeitquelle oder einer benutzerdefinierten Zeitquelle erbt.
    • Geben Sie die Zeitspanne und die verwendeten Einheiten an (Sekunden oder Frames)
    • Geben Sie eine benutzerdefinierte Methode oder script Funktion an, mit einem argument Array, das an die Methode übergeben wird.
    • Geben Sie optional an, wie oft der Vorgang wiederholt werden soll.
      • Standardmäßig wird es nur einmal ausgeführt
  • Rufen Sie auf. time_source_start() / Zeitquelle starten, um die Zeitquelle sofort zu starten oder sie später aufzurufen, wenn sie gestartet werden soll

Dadurch wird die Zeitquelle ausgeführt, und nach der angegebenen Zeitspanne läuft sie ab und ruft die von Ihnen angegebene Methode auf.

Eingebaute Zeitquellen

Eine Zeitquelle kann von einer der beiden eingebauten Zeitquellenoder von einer benutzerdefinierten Zeitquelle erben.

Siehe: Eingebaute Zeitquellen

Einfache Zeitschaltuhren

Die Einrichtung einer Zeitquelle erfordert zwar nicht viele Zeilen code, aber es kann bequem sein, eine Zeitschaltuhr mit einer einzigen Funktion schnell einzurichten und zu verwalten.

Die folgenden beiden Funktionen bieten einen einfacheren, kurzen Weg, um eine Zeitquelle zu erstellen und zu stoppen:

  • Anruf_später
  • Anruf_Abbrechen

HINWEIS Sie können die Zeitquellenfunktionen nicht mit den von diesen Funktionen verwendeten Zeitquellen-IDs verwenden. Siehe call_later für weitere Informationen.

Funktionsliste

Die folgenden Funktionen werden zum Erstellen und Ändern von Zeitquellenverwendet:

  • zeit_quelle_erstellen
  • zeit_quelle_zerstören
  • zeit_quelle_start
  • zeit_quelle_stop
  • zeit_quelle_pause
  • zeit_quelle_wiederaufnahme
  • zeit_quelle_rekonfigurieren
  • zeit_quelle_zuruecksetzen
  • time_source_get_children
  • time_source_get_parent
  • time_source_get_period
  • time_source_get_reps_completed
  • time_source_get_reps_remaining
  • time_source_get_state
  • time_source_get_time_remaining
  • zeit_quelle_get_units
  • zeit_quelle_existiert

Jede get_*-Funktion gibt undefiniert zurück, wenn die angegebene Zeitquelle nicht existiert.

Für die Konvertierung von Zeiträumen stehen die folgenden Hilfsfunktionen zur Verfügung:

  • zeit_sekunden_bis_bpm
  • zeit_bpm_zu_sekunden
