zeit_quelle_existiert

Diese Funktion gibt zurück, ob die angegebene ID eine gültige Zeitquelle ist.

Wenn die Zeitquelle zerstört wurde, gibt diese Funktion false zurück. Wurde die Zeitquelle jedoch angehalten oder ist sie lediglich abgelaufen, existiert sie weiter, so dass diese Funktion true zurückgibt.

Syntax:

time_source_exists(id);

Argument Typ Beschreibung
id Zeitquellen-ID The Time Source zur Kontrolle

Rückgabe:

Boolean

Beispiel:

if (time_source_exists(global.spawn_time_source))
{
    if (keyboard_check_pressed(vk_space))
    {
        time_source_destroy(global.spawn_time_source);
    }
}

Diese code prüft, ob eine Zeitquelle existiert. Ist dies der Fall, wird geprüft, ob die Leertaste gedrückt wurde.

In diesem Fall wird die Zeitquelle vernichtet.
