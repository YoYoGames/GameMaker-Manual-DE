time_source_start

Diese Funktion startet die angegebene Zeitquelle und ändert ihren Status auf time_source_state_active.

Bei der angegebenen Zeitquelle kann es sich um eine neue Zeitquelle handeln, die noch nie gestartet wurde, pausiert oder gestoppt wurde oder ohne Wiederholungen abgelaufen ist.

Mit dieser Funktion wird die angegebene Zeitquelle"sanft zurückgesetzt", d. h. die Ablaufzeit und die verbleibenden Wiederholungen werden auf die für sie konfigurierten Werte zurückgesetzt.

Syntax:

time_source_start(id);

Argument Typ Beschreibung
id Zeitquellen-ID The Time Source zum Start

Rückgabe:

N/A

Beispiel:

var _my_method = function()
{
    instance_destroy();
}

var _time_source = time_source_create(time_source_game, 300, time_source_units_frames, _my_method);

time_source_start(_time_source);

In diesem Beispiel soll sich die Instanz 300 Frames später selbst zerstören.

Die code erstellt zunächst eine Methode, die einfach aufruft instance_destroy().

Anschließend wird eine Zeitquelle erstellt, die von der Zeitquelle des Spiels geerbt wird. Sie setzt ihre Periode auf 300 Frames.

Schließlich wird die Zeitquelle gestartet.
