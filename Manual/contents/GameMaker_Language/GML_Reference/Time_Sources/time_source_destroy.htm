zeit_quelle_zerstören

Diese Funktion zerstört die angegebene Zeitquelle.

Sie können optional angeben, ob alle untergeordneten Zeitquellen ebenfalls gelöscht werden sollen.

Syntax:

time_source_destroy(id, [destroy_tree]);

Argument Typ Beschreibung
id Zeitquellen-ID The Time Source zu zerstören
destroy_tree Boolesche OPTIONAL Ob auch untergeordnete Zeitquellen zerstört werden sollen, ist standardmäßig falsch

Rückgabe:

N/A

Beispiel:

// Create event
time_source = time_source_create(time_source_game, 1, time_source_units_seconds, my_method, [], -1);

// Clean Up event
time_source_destroy(time_source);

Das Ereignis Erstellen code erstellt eine neue Zeitquelle, die einmal pro Sekunde abläuft und unbegrenzt wiederholt wird.

Das Ereignis Clean Up code zerstört diese Zeitquelle, da sie nach der Zerstörung der Instanz nicht mehr benötigt wird.
