time_source_stop

Diese Funktion hält die angegebene Zeitquelle an und setzt ihren Timer zurück. Sie können keine Zeitquelle stoppen, die entweder noch nicht gestartet ist oder sich in einem abgeschlossenen Zustand befindet.

Syntax:

time_source_stop(id);

Argument Typ Beschreibung
id Zeitquellen-ID The Time Source zu stoppen

Rückgabe:

N/A

Beispiel:

if (time_source_get_state(time_source) != time_source_state_stopped)
{
    time_source_stop(time_source);
}

Die obige code stoppt eine Zeitquelle, wenn sie nicht bereits gestoppt ist.
