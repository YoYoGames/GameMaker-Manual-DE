Zeitquellenstaaten

Zeit Quelle Zustand Konstante
Konstante Beschreibung Wert
time_source_state_initial Die Zeitquelle wurde noch nicht gestartet 0
time_source_state_active Die Zeitquelle wurde gestartet und zählt herunter 1
time_source_state_paused Die Zeitquelle ist paused 2
time_source_state_stopped Die Zeitquelle wurde gestoppt oder sie ist vollständig abgelaufen 3

Eine Zeitquelle kann einen Zustand haben, der eine der oben genannten Konstanten sein kann. Der Standardzustand für eine neu erstellte Zeitquelle ist time_source_state_initial.

Der Zustand kann geändert werden mit time_source_start(), time_source_stop(), und time_source_pause().

Der Status einer Zeitquelle wird zurückgegeben mit time_source_get_state().
