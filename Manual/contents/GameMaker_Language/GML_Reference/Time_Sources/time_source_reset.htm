zeit_quelle_zuruecksetzen

Diese Funktion setzt die angegebene Zeitquelle zurück, indem sie deren Zeitgeber und Wiederholungsinformationen zurücksetzt.

Nach dem Aufruf dieser Funktion müssen Sie die Zeitquelle erneut starten, damit sie abwärts zählt.

Syntax:

time_source_reset(id);

Argument Typ Beschreibung
id Zeitquellen-ID The Time Source zum Zurücksetzen

Rückgabe:

N/A

Beispiel:

// Room Start event
time_source_reset(global.spawn_time_source);
time_source_start(global.spawn_time_source);

Diese code setzt die angegebene Zeitquelle zurück, wenn ein neuer Raum beginnt, und startet dann die Zeitquelle erneut.
