code_ist_kompiliert

Diese Funktion gibt true zurück, wenn die platform außerhalb der virtual machine kompiliert wird, wie z.B. für die YYC und JS platforms.

Syntax:

code_is_compiled();

Rückgabe:

Beispiel:

if code_is_compiled()
{
    show_debug_message("Compiler okay!");
}

Das obige code prüft, ob das Spiel mit YoYo Compiler kompiliert wurde und zeigt eine Debug-Meldung an, wenn dies der Fall ist.
