get_string

Dadurch wird ein Pop-up-Fenster mit einer Standardmeldung und einer Schaltfläche "Ok" erstellt, die den Benutzer zur Eingabe einer Zeichenfolge auffordert. Die Funktion gibt die eingegebene Zeichenfolge zurück oder den Standardwert, wenn nichts eingegeben wurde.

HINWEIS Diese Funktion ist nur für die Fehlersuche auf dem Ziel Windows gedacht, ist aber auf allen anderen Zielen veraltet.

Syntax:

get_string(str, def);

Argument Typ Beschreibung
str Zeichenfolge The string to show in the pop-up message.
def Zeichenfolge The default string in the text box.

Rückgabe:

String

Beispiel:

global.test_name = get_string("Test highscore name:", "Anonymous");

Auf der obigen Seite code wird der Benutzer aufgefordert, einen Namen einzugeben, der dann in der globalen Variablen "test_name" gespeichert wird. Wird nichts eingegeben und drückt der Benutzer einfach auf "Ok", wird der Standardwert "Anonymous" zurückgegeben.
