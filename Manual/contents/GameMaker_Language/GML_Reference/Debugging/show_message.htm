show_message

Diese Funktion erstellt ein Popup-Fenster, das die angegebene Zeichenkette und eine Schaltfläche "Ok" zum Schließen anzeigt.

HINWEIS Diese Funktion ist nur für die Fehlersuche auf dem Ziel Windows gedacht, ist aber auf allen anderen Zielen veraltet.

Syntax:

show_message(str);

Argument Typ Beschreibung
str Zeichenfolge The string to show in the pop-up message.

Rückgabe:

N/A

Beispiel:

var tot = 0;
for (var i = 0; i < 10; i += 1)
{
    tot += inv[i];
}
show_message("Total = " + string(tot));

code Die obige Website loop durchsucht die im Array "inv" gespeicherten Werte und addiert sie zur Variablen "tot", bevor eine Meldung mit der Gesamtsumme angezeigt wird.
