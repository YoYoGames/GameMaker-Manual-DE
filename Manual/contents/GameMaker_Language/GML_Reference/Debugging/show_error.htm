show_error

Mit dieser Funktion wird eine benutzerdefinierte Zeichenfolge als Fehlermeldung angezeigt.

WICHTIG Das zweite argument existiert nur aus Gründen der Abwärtskompatibilität mit älteren Projekten und hat keinen Einfluss auf den angezeigten Fehlerdialog.

HINWEIS Diese Funktion ist nur zur Fehlersuche geeignet.

Syntax:

show_error(str, abort);

Argument Typ Beschreibung
str Zeichenfolge The string to show in the pop-up message.
abort Boolesche UNUSED Ob der Fehler das Spiel abbrechen (true) oder dem Spieler erlauben soll, ihn zu ignorieren (false).

Rückgabe:

N/A

Beispiel:

if (room != rm_Dungeon)
{
    show_error("Error: Went to wrong area. Aborting game.", true);
}

Die obige code prüft, ob der aktuelle Raum rm_Dungeon ist, und wenn nicht, wird eine Fehlermeldung angezeigt.
