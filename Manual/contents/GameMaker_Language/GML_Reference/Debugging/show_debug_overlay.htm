show_debug_overlay

Mit dieser Funktion switch können Sie das Standard-Debug-Overlay beim Testen Ihres Spiels ein- und ausschalten; es ist standardmäßig deaktiviert. Das Debug-Overlay zeigt einen grafischen CPU/GPU-Auslastungsbalken im eigentlichen Spielfenster selbst zusammen mit dem aktuellen realen fps-Wert, der Anzahl der Textur-Swaps und der Anzahl der Vertex-Batches (beachten Sie, dass Textur-Swaps und Vertex-Batches niemals Null sein werden und normalerweise Werte von 2 oder 3 anzeigen, da GameMaker auch bei einem leeren Raum und ohne Objekte noch Dinge zeichnen und stapeln muss).

Dieser Balken ist in Abschnitte unterteilt, wobei jeder Abschnitt 1/60 einer Sekunde entspricht. Wie Sie auf dem Bild oben sehen können, besteht der Balken aus verschiedenen Farben:

  • Grün - Eingabe-/Ausgabeverarbeitung (d.h.: Tastatur, Maus, Gamepad, Netzwerk usw...)
  • Rot - Die Aktualisierungsgeschwindigkeit des Schritt-Ereignisses
  • Gelb - Die für das Auslosungsereignis benötigte Zeit
  • Orange -Debug-Aktualisierungszeit, die normalerweise nur sichtbar ist, wenn Sie das Debug-Modul verwenden
  • Weiß - GPU-Restzeit, d. h. die Zeit, die damit verbracht wird, darauf zu warten, dass die GPU das Rendering des Bildes beendet, bevor das nächste beginnen kann
  • Cyan - Die Textwiedergabezeit
  • Grau - Die Zeit, die benötigt wird, um den Bildschirm für jeden Ziehungsschritt zu löschen
  • Blau - Die Zeit, die der Garbage Collector für die Ausführung benötigt
  • Dunkelrot - Der GPU-Flush, d. h. die Zeit, die der Grafikprozessor benötigt, um Bilder aus dem Speicher zu löschen

Mit dieser Funktion können Sie das Overlay hinzufügen, egal ob Sie sich im Debug-Modus befinden oder nicht. Auf diese Weise können Sie sehen, wie effizient Ihr Spiel läuft und erhalten einen visuellen Hinweis darauf, wie es die verfügbaren Ressourcen nutzt, ohne dass der Debugger nebenher läuft.

HINWEIS: Diese Funktion funktioniert nicht mit dem HTML5-Ziel, da es anders funktioniert als alle anderen Ziele.

Syntax:

show_debug_overlay(enable[, minimised, scale, alpha]);

ArgumentTyp Beschreibung
enable switch on (true) oder aus (false) the debug overlay.

Rückgabe:

Beispiel:

if global.debug
{
    show_debug_overlay(true);
}
else
{
    show_debug_overlay(false);
}

Die obige code schaltet das Debugging in Abhängigkeit vom Wert einer globalen Variablen ein oder aus.
