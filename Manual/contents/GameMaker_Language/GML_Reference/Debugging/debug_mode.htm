Debug_mode

Diese nur lesbare Variable gibt true zurück, wenn das Spiel im Debug-Modus gespielt wird, und false, wenn es normal gespielt wird.

Syntax:

debug_mode

Rückgabe:

Beispiel:

if debug_mode
{
    ini_open("Cheats.ini");
}
else
{
    ini_open("Game.ini");
}

Die obige code öffnet eine andere Ini-Datei, abhängig vom Wert der schreibgeschützten Variable debug_mode.
