show_question

Diese Funktion erstellt ein Pop-up-Fenster mit zwei Schaltflächen für "Ja" und "Nein". Sie gibt true oder false zurück, je nachdem, welche der beiden Schaltflächen der Benutzer drückt.

HINWEIS Diese Funktion ist nur für die Fehlersuche auf dem Ziel Windows gedacht, ist aber auf allen anderen Zielen veraltet.

Syntax:

show_question(str);

Argument Typ Beschreibung
str Zeichenfolge The string to show in the pop-up question.

Rückgabe:

Boolean

Beispiel:

if (score > 500) && debug_mode
{
    if show_question("Continue to next room?")
    {
        room_goto(rm_Level2);
    }
    else game_end();
}

Das obige code prüft den Punktestand, und wenn er über 500 liegt, wird der Benutzer gefragt, ob er das Spiel fortsetzen möchte, und wenn die Schaltfläche "Ja" angeklickt wird, geht es in einen anderen Raum, aber wenn die Schaltfläche "Nein" ausgewählt wird, wird das Spiel beendet.
