Ort_Versammlung

Mit dieser Funktion können Sie eine Position auf eine Kollision mit einer anderen Instanz oder allen Instanzen eines Objekts prüfen, indem Sie die Kollisionsmaske der Instanz verwenden, die die code für die Prüfung ausführt. Wenn du diese Funktion verwendest, bittest du GameMaker, die Instanz an die neue Position zu bewegen, auf eine Kollision zu prüfen, zurückzufahren und dir zu sagen, ob eine Kollision gefunden wurde oder nicht. Dies funktioniert bei präzisen Kollisionen, aber nur, wenn sowohl die Instanz als auch das zu prüfende Objekt präzise Kollisionsmasken ausgewählt haben, ansonsten werden nur Bounding-Box-Kollisionen angewendet. Diese Funktion akzeptiert auch die speziellen Schlüsselwörter all und other. Bitte beachte, dass du die eindeutige Instanz-ID des Objekts, mit dem kollidiert werden soll, mit instance_place().

Beachten Sie, dass die angegebenen x/y-Koordinaten vor der Prüfung auf die nächstliegende ganze Zahl heruntergerechnet werden.

Syntax:

place_meeting(x, y, obj);

Argument Typ Beschreibung
x Real The x position to check.
y Real The y position to check.
obj Objekt Asset- oder Instanz-ID The instance or object to check for.

Rückgabe:

Boolean

Beispiel:

if keyboard_check(vk_left)
{
    if !place_meeting(x - 5, y, obj_wall) x -=5;
}

Die obige code prüft, ob es links von der Instanz eine Kollision gibt und verschiebt die Instanz, wenn es keine gibt.
