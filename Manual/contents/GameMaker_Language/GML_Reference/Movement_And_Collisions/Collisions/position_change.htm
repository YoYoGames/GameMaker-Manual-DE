position_change

Diese Funktion prüft eine Position auf eine Kollision mit allen Instanzen am angegebenen Punkt, und wenn es eine gibt, ändert sie alle Instanzen in Kollision zu Instanzen des gewählten Objekts. Du kannst "perf" argument auf true setzen, was GameMaker zwingt, die Ereignisse Destroy und Clean Up der gefundenen Instanz sowie das Ereignis Create der neuen Instanz auszuführen, oder false, um keines dieser Ereignisse auszuführen. Bitte beachten Sie, dass, wenn Sie die Ereignisse Zerstören, Aufräumen und Erstellen nicht ausführen, jede erstellte Instanz, die eine Variable verwendet, die normalerweise im Ereignis Erstellen definiert ist, das Spiel zum Absturz bringt, da diese Variable nicht mehr existiert.

Syntax:

position_change(x, y, obj, perf);

Argument Typ Beschreibung
x Real The x coordinate of where to change colliding instances.
y Real The y coordinate of where to change colliding instances.
obj Objekt Anlage The new object the calling object will change into.
perf Boolesche Whether to perform that new object's Create event (true) or not (false).

Rückgabe:

N/A

Beispiel:

position_change(200, 200, obj_Bird, true);

Dadurch werden alle Instanzen, die am Punkt (200,200) kollidieren, in eine Instanz von "obj_Bird" umgewandelt, wobei das "obj_Bird"-Ereignis für jede dieser Instanzen ausgeführt wird.
