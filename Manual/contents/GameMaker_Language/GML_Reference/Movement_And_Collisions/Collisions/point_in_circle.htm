Punkt_im_Kreis

Wenn Sie diese Funktion verwenden, definieren Sie einen kreisförmigen Bereich und GameMaker ermittelt, ob der angegebene Punkt innerhalb seiner Grenzen liegt oder nicht. Wenn der Punkt innerhalb des definierten Kreises liegt, gibt die Funktion true zurück, andernfalls false.

Syntax:

point_in_circle(px, py, x1, y1, rad)

ArgumentTyp Beschreibung
px The x coordinate of the point to check.
py The y coordinate of the point to check.
x1 The x coordinate of the circle centre.
y1 The y coordinate of the circle centre.
rad The radius of the circle.

Rückgabe:

Beispiel:

if point_in_circle(mouse_x, mouse_y, x, y, 16)
{
    over = true;
}
else
{
    over = false;
}

Die obige code verwendet die Funktion point_in_circle, um zu prüfen, ob die Mausposition innerhalb des definierten kreisförmigen Bereichs liegt, und setzt die Variable "over" auf true, wenn dies der Fall ist, oder andernfalls auf false.
