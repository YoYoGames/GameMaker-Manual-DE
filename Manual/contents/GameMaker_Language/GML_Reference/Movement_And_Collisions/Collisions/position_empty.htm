position_leer

Diese Funktion prüft, ob eine gegebene Position mit einer Instanz kollidiert, die eine gültige Kollisionsmaske an der gegebenen Position hat.

Syntax:

position_empty(x, y);

ArgumentTyp Beschreibung
x The x position to check.
y The y position to check.

Rückgabe:

Beispiel:

var xx, yy;
xx = random(room_width);
yy = random(room_height);
if position_empty(xx, yy)
{
    instance_create_layer(xx, yy, "Bullets", obj_Bomb);
}

Dies wird eine zufällige Position irgendwo im Raum auf eine Kollision überprüfen und wenn es keine gibt, wird eine Instanz von obj_Bomb erstellt.
