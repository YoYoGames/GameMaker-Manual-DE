platz_frei

Du kannst diese Funktion verwenden, um zu überprüfen, ob die aufrufende Instanz mit einer Instanz kollidiert, die in deinem Spiel als solide gekennzeichnet ist. Es sollte beachtet werden, dass die Instanz, die code ausführt, eine gültige Kollisionsmaske haben muss (entweder für die sprite selbst oder durch die mask_index) und sie wird nur Kollisionen mit Instanzen registrieren, die ebenfalls eine gültige Maske haben.

Die Funktion selbst funktioniert im Wesentlichen so, dass sie die Instanz nimmt und auf Kollisionen mit nur soliden markierten Instanzen prüft, wenn sie an der durch die x/y arguments angegebenen Position platziert wird. Die Kollisionsprüfung kann entweder präzise sein oder auf dem Begrenzungsrahmen der Instanz basieren, je nachdem, welche Art von Kollisionsmaske ausgewählt wurde.

Beachten Sie, dass die angegebenen x/y-Koordinaten vor der Prüfung auf die nächstliegende ganze Zahl heruntergerechnet werden.

Syntax:

place_free(x, y);

ArgumentTyp Beschreibung
x The x position to check.
y The y position to check.

Rückgabe:

Beispiel:

if place_free(mouse_x, mouse_y)
{
    x = mouse_x;
    y = mouse_y;
}

Die obige code prüft, ob eine Kollision mit einer fest markierten Instanz vorliegt, wenn die aufrufende Instanz an der gleichen Position wie die Maus platziert wird. Wenn keine Kollision festgestellt wird, werden die x/y-Koordinaten der Instanz auf die der Maus gesetzt.
