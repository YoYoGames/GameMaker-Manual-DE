Punkt_im_Dreieck

Wenn Sie diese Funktion verwenden, definieren Sie eine dreieckige Fläche und GameMaker ermittelt, ob der angegebene Punkt innerhalb der Grenzen liegt oder nicht. Wenn der Punkt innerhalb des definierten Dreiecks liegt, gibt die Funktion true zurück, andernfalls false.

Syntax:

point_in_triangle(px, py, x1, y1, x2, y2, x3, y3)

ArgumentTyp Beschreibung
px The x coordinate of the point to check.
py The y coordinate of the point to check.
x1 The x coordinate of the first corner of the triangle to check.
y1 The y coordinate of the first corner of the triangle to check.
x2 The x coordinate of the second corner of the triangle to check.
y2 The y coordinate of the second corner of the triangle to check.
x3 The x coordinate of the third corner of the triangle to check.
y3 The y coordinate of the third corner of the triangle to check.

Rückgabe:

Beispiel:

var inst = instance_nearest(x, y, obj_Player);
if (instance_exists(inst))
{
    var _x1 = x + lengthdir_x(100, image_angle - 45);
    var _y1 = y + lengthdir_y(100, image_angle - 45);
    var _x2 = x + lengthdir_x(100, image_angle + 45);
    var _y2 = y + lengthdir_y(100, image_angle + 45);
    _can_see = point_in_triangle(_inst.x, _inst.y, x, y, _x1, _y1, _x2, _y2);
}

Die obige code verwendet die Funktion point_in_triangle als "Sichtkegel", um nach einer Instanz von "obj_player" zu suchen und setzt eine Variable auf true, wenn die x/y-Position des Objekts innerhalb des definierten Dreiecks liegt.
