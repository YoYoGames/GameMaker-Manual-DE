collision_line

Collision_line prüft entlang einer Linie vom Punkt x1,y1 zum Punkt x2,y2 auf eine Kollision mit einer beliebigen Instanz des durch argument "obj" angegebenen Objekts. Diese Prüfung kann entweder präzise oder nicht präzise sein, aber damit präzise Kollisionen aktiviert werden können, müssen für das Objekt oder die Instanz, auf die Sie prüfen, auch präzise Kollisionen für ihre sprite aktiviert sein. Die folgende Abbildung veranschaulicht, wie dies funktioniert:

Damit präzise Kollisionen berücksichtigt werden, müssen sowohl das Objekt sprite als auch die Kollisionsfunktion als präzise markiert sein. Es sollte auch beachtet werden, dass der Rückgabewert der Funktion die ID einer beliebigen Instanz sein kann, die als kollidierend entlang der Linie betrachtet wird. Wenn also drei Instanzen die definierte Linie überlappen, könnte jede ihrer IDs der Rückgabewert der Funktion sein.

Beachten Sie, dass Sie anstelle eines Objektindex eine Instanz angeben können id angeben können, um nach einer bestimmten Instanz zu suchen, oder die Instanz-Schlüsselwörter all oder other (je nach Ereignis und aktuellem Bereich code ).

Syntax:

collision_line(x1, y1, x2, y2, obj, prec, notme);

Argument Typ Beschreibung
x1 Real The x coordinate of the start of the line.
y1 Real The y coordinate of the start of the line.
x2 Real The x coordinate of the end of the line.
y2 Real The y coordinate of the end of the line.
obj Objekt Asset- oder Instanz-ID The object to check for instance collisions.
prec Boolesche Whether the check is based on precise collisions (true(was langsamer ist) oder seine Bounding Box im Allgemeinen (false, schneller).
notme Boolesche Whether the calling instance, if relevant, should be excluded (true) oder nicht (false).

Rückgabe:

Instance ID or noone

Beispiel:

with (obj_Enemy)
{
    if collision_line(100, 400, 100, 600, id, false, false) instance_destroy();
}

Dieses code veranlasst alle Instanzen von "obj_Enemy", entlang einer Linie von 100x, 400y bis 100x, 600y auf eine Kollision mit sich selbst zu prüfen, und wenn es eine gibt, werden sie zerstört.
