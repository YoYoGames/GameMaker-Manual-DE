kollision_kreis_liste

Diese Funktion ist die gleiche wie die collision_circle() Funktion, nur dass sie nicht nur eine Instanz in Kollision erkennt, sondern mehrere Instanzen. Sie geben die x/y-Position des Mittelpunkts des kreisförmigen Bereichs an, der zusammen mit dem Radius und dem zu prüfenden Objekt geprüft werden soll, und Sie können festlegen, dass die Prüfung präzise sein soll (in diesem Fall müssen alle geprüften Instanzen präzise Kollisionsmasken haben) und ob die Prüfung die aufrufende Instanz umfassen soll oder nicht.

Sie geben auch eine DS-Liste an, so dass die id Werte aller Instanzen, die mit der aufrufenden Instanz kollidieren, an das Ende der angegebenen Liste angehängt werden. Sie können die Liste vor dem Aufruf dieser Funktion löschen, so dass sie nur die Ergebnisse dieses Funktionsaufrufs enthält. Sie haben auch die Möglichkeit, die Instanzen nach ihren Abständen vom Ursprung der Kreisfläche zu ihren Ursprüngen zu ordnen. Die Funktion gibt die Anzahl der gefundenen Instanzen zurück, oder 0, wenn keine gefunden wurden.

Beachte, dass du anstelle eines Objektindexes das Instanzschlüsselwort all angeben kannst, um nach allen Instanzen im aktuellen Raum zu suchen, was auch die Instanz einschließen kann, die code ausführt (abhängig von notme argument).

Syntax:

collision_circle_list(x1, y1, rad, obj, prec, notme, list, ordered);

Argument Typ Beschreibung
x1 Real The x coordinate of the center of the circle to check.
y1 Real The y coordinate of the center of the circle to check.
rad Real The radius (distance in pixels from its center to its edge).
obj Objekt Asset- oder Instanz-ID The object to check for instance collisions.
prec Boolesche Whether the check is based on precise collisions (true(was langsamer ist) oder seine Bounding Box im Allgemeinen (false, schneller).
notme Boolesche Whether the calling instance, if relevant, should be excluded (true) oder nicht (false).
list DS-Liste ID The DS list to use to store the IDs of colliding instances.
ordered Boolesche Whether the list should be ordered by distance (true) oder nicht (false).

Rückgabe:

Real (The number of instances found to be in collision)

Beispiel:

var _list = ds_list_create();
var _num = collision_circle_list(x, y, 100, obj_Enemy, false, true, _list, false);
if (_num > 0)
{
    for (var i = 0; i < _num; ++i;)
    {
        instance_destroy(_list[| i]);
    }
}
ds_list_destroy(_list);

Die obige code prüft einen kreisförmigen Bereich mit einem Radius von 100 Pixeln um die Position der aufrufenden Instanz auf Kollisionen mit Instanzen von "obj_Enemy". Wenn es irgendwelche Kollisionen gibt, wird die zuvor erstellte Liste durchlaufen und jede Instanz, die an der Kollision beteiligt war, wird zerstört.
