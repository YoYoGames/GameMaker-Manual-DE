position_meeting

Mit dieser Funktion können Sie eine Position auf eine Kollision mit einer anderen Instanz oder allen Instanzen eines Objekts prüfen. Wenn du diese Funktion verwendest, überprüfst du einen einzelnen Punkt im Raum auf eine Instanz oder ein Objekt. Diese Prüfung wird gegen die Bounding Box der Instanz oder gegen die Maske der Instanz durchgeführt, wenn diese Instanz präzise Kollisionen geprüft hat. Wenn du die eindeutige Instanz ermitteln musst id des Objekts, mit dem kollidiert wird, solltest du verwenden instance_position(). Diese Funktion akzeptiert auch die speziellen Schlüsselwörter all und other.

Syntax:

position_meeting(x, y, obj);

Argument Typ Beschreibung
x Real The x position to check.
y Real The y position to check.
obj Objekt Asset- oder Instanz-ID The object (or instance id, or the keywords "all" or "other") to check for a collision with.

Rückgabe:

Boolean

Beispiel:

if mouse_check_button(mb_left)
{
    if !position_meeting(mouse_x, mouse_y, all)
    {
        instance_create_layer(mouse_x, mouse_y, "Walls", obj_Wall);
    }
}

Das obige code prüft, ob die linke Maustaste gedrückt ist, und wenn sie gedrückt ist, prüft es die x/y-Position der Maus auf eine Kollision mit einer beliebigen Instanz. Wenn es keine gibt, wird eine Instanz von "obj_Wall" erstellt.
