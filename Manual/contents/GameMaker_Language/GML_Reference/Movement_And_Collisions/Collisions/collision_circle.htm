collision_circle

Collision_circle erzeugt eine kreisförmige Zone mit dem vom Benutzer angegebenen Radius um die Punkte x1,y1 und prüft dann diese Zone auf eine Kollision mit einer beliebigen Instanz des durch argument "obj" angegebenen Objekts. Diese Prüfung kann entweder präzise oder nicht präzise sein, aber damit präzise Kollisionen aktiviert werden können, müssen für das Objekt oder die Instanz, auf die Sie prüfen, auch präzise Kollisionen für deren sprite aktiviert sein. Ist dies nicht der Fall, basiert die Standardprüfung auf Begrenzungsrahmen. Die folgende Abbildung veranschaulicht, wie dies funktioniert:

Hier verwendet die Instanz in der Mitte einen Kollisionskreis, um auf Ballobjekte zu prüfen. Die blauen Bälle haben keine präzise Bounding Box, und wie Sie sehen können, kann die Kollision auch dann stattfinden, wenn sprite den Kreis nicht berührt (selbst wenn Sie die Option precise in der Funktion auf true setzen), da die Bounding Box von sprite den durch collision_circle definierten Kreis überlappt. Andererseits werden die grünen Kugeln nur dann als kollidierend betrachtet, wenn die tatsächliche sprite mit dem definierten Kreis überlappt. Damit präzise Kollisionen berücksichtigt werden können, müssen sowohl das Objekt sprite als auch die Kollisionsfunktion als präzise markiert sein. Es sollte auch beachtet werden, dass der Rückgabewert der Funktion die ID einer der Instanzen sein kann, die als in Kollision befindlich betrachtet werden.

Beachten Sie, dass Sie anstelle eines Objektindex eine Instanz angeben können id angeben können, um nach einer bestimmten Instanz zu suchen, oder die Instanz-Schlüsselwörter all oder other (je nach Ereignis und aktuellem Bereich code ).

Syntax:

collision_circle(x1, y1, rad, obj, prec, notme);

Argument Typ Beschreibung
x1 Real The x coordinate of the center of the circle to check.
y1 Real The y coordinate of the center of the circle to check.
rad Real The radius (distance in pixels from its center to its edge).
obj Objekt Asset- oder Instanz-ID The object to check for instance collisions.
prec Boolesche Whether the check is based on pixel-perfect collisions (true = slow) or its bounding box in general (false = fast).
notme Boolesche Whether the calling instance, if relevant, should be excluded (true) oder nicht (false).

Rückgabe:

Instance ID or noone

Beispiel:

if collision_circle(x, y, 20, obj_Cursor, false, true)
{
    image_index = 1;
}
else image_index = 0;

Das obige code prüft einen kreisförmigen Bereich mit einem Radius von 20 Pixeln auf eine Kollision mit "obj_Cursor" und wenn es eine gibt, wird der image_index des Objekts, das die code ausführt, auf 1 gesetzt, aber wenn es keine gibt, wird der image_index des Objekts auf 0 gesetzt.
