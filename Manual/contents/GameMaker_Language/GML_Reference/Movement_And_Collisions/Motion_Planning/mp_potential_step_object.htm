mp_potential_step_object

Diese Funktion lässt die Instanz einen Schritt in Richtung einer bestimmten, durch xgoal/ygoal definierten Position machen, während sie gleichzeitig versucht, Hindernissen auszuweichen. Wenn die Instanz auf eine Instanz des durch "obj" argument angegebenen Objekts trifft, wird sie die Bewegungsrichtung ändern, um zu versuchen, diese Instanz zu umgehen und sich um sie herum zu bewegen. Es ist nicht garantiert, dass dieser Ansatz funktioniert, aber in den meisten einfachen Fällen wird er die Instanz effektiv in Richtung des Ziels bewegen. Die Funktion gibt zurück, ob das Ziel erreicht wurde oder nicht.

Syntax:

mp_potential_step_object(xgoal, ygoal, stepsize, obj)

Argument Typ Beschreibung
xgoal Real The target x position.
ygoal Real The target y position.
stepsize Real The speed the instance moves in pixels per step.
obj Objekt Asset- oder Instanz-ID The object that is to block the path of the instance running the function. Can be an object index, an instance id or the special keyword ,all.

Rückgabe:

Boolean

Beispiel:

if instance_exists(obj_Enemy)
{
    var inst;
    inst = instance_nearest(x, y, obj_Enemy);
    mp_potential_step_object(inst.x, inst.y, 5, obj_Wall);
}

Die obige code prüft zunächst, ob es Instanzen von "obj_Enemy" im Raum gibt. Wenn dies der Fall ist, wird die nächstgelegene Instanz gefunden und ihre ID in einer Variablen gespeichert. Diese Variable wird dann verwendet, um mp_potential_step_object anzuweisen, die Instanz mit der code in Richtung der x/y-Position des gefundenen Objekts mit einer Geschwindigkeit von 5 Pixeln pro Schritt zu bewegen, wobei nur Instanzen des Objekts "obj_Wall" vermieden werden.
