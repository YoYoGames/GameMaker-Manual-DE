mp_grid_to_ds_grid

Mit dieser Funktion können Sie das angegebene MP-Gitter in ein DS-Gitter kopieren. Das DS-Gitter sollte die gleiche Größe haben wie das MP-Gitter, muss es aber nicht (wenn es kleiner ist, gehen Daten verloren, und wenn es größer ist, sind alle zusätzlichen Gitterzellen 0). Eine Zelle im DS-Gitter enthält den Wert -1, wenn sie im MP-Gitter als belegt gekennzeichnet war, oder 0, wenn sie nicht belegt war.

Syntax:

mp_grid_to_ds_grid(source, destination);

Argument Typ Beschreibung
source MP Raster-ID Index of the mp_grid that is to be used
destination DS-Raster-ID Index of the ds_grid that is to be used to copy the grid data to.

Rückgabe:

Boolean

Beispiel:

motion_grid = ds_grid_create(room_width / 32, room_height / 32);
mp_grid_to_ds_grid(mp_grid, motion_grid);

Die obige code erstellt ein neues DS-Gitter und kopiert dann die in der Variablen "mp_grid" enthaltenen MP-Gitterdaten in das neue DS-Gitter.
