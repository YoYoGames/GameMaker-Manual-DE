mp_grid_create

Mit dieser Funktion können Sie ein mp_grid für die Bewegungsplanungsfunktionen erstellen. Sie gibt einen Index zurück, der in einer Variablen gespeichert werden sollte, damit er in allen anderen MP-Grid-Funktionsaufrufen verwendet werden kann. Die x- und y-Koordinaten geben die Position der linken oberen Ecke des Gitters an, hcells und vcells geben die Anzahl der horizontalen und vertikalen Zellen an, die innerhalb des Gitters erstellt werden sollen, und cell width und cell height geben die Größe der Zellen in Pixel an. Sie können mehrere Gitterstrukturen gleichzeitig erstellen und pflegen, wenn Sie sie für verschiedene Zwecke benötigen. Beachten Sie jedoch, dass jedes Gitter Speicherplatz benötigt, und je mehr Zellen es hat, desto mehr Speicherplatz wird benötigt. Beachten Sie auch, dass die MP-Funktionen ziemlich prozessorlastig sind. Je häufiger Sie sie aufrufen und je höher die Zellenauflösung ist, desto wahrscheinlicher ist ein Leistungsabfall.

Syntax:

mp_grid_create(left, top, hcells, vcells, cellwidth, cellheight);

Argument Typ Beschreibung
left Real Starting x coordinate of the mp_grid in the room
top Real Starting y coordinate of the mp_grid in the room
hcells Real Number of horizontal cells that the mp_grid will contain
vcells Real Number of vertical cells that the mp_grid will contain
cellwidth Real The width (in pixels) of each individual cell of the mp_grid
cellheight Real The height (in pixels) of each individual cell of the mp_grid

Rückgabe:

MP Grid ID

Beispiel:

global.grid = mp_grid_create(0, 0, room_width / 32, room_height / 32, 32, 32);

Die obige code erstellt eine globale Variable "grid", generiert dann ein mp_grid und weist dessen Index (id) dieser Variable zu, um ihn in allen weiteren mp_grid-Funktionsaufrufen zu verwenden. Das Raster deckt den Raum ab, mit einer Zellenauflösung von 32x32 Pixeln. Das bedeutet, dass zum Beispiel in einem Raum von 640x480 das Raster 300 Zellen enthalten würde: 20 horizontale Zellen (640 / 32) multipliziert mit 15 vertikalen Zellen (480 /32) ergibt 300 Zellen
