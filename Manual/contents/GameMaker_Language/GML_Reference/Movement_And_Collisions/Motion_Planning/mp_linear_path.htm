mp_linear_path

Diese Funktion berechnet einen geradlinigen Weg von der aktuellen Instanzposition zu dem durch die Werte "xgoal" und "ygoal" angegebenen Punkt. Sie verwendet die angegebene Schrittweite, die gleiche wie bei der Funktion mp_linear_step(). Der angegebene Pfad muss bereits existieren und wird durch den neuen Pfad überschrieben und die Funktion gibt zurück, ob ein vollständiger Pfad gefunden wurde (true) oder nicht (false). Wenn false zurückgegeben wird, wird zwar ein Pfad erzeugt, aber nur bis zu der Stelle, an der der Pfad blockiert war.

HINWEIS: Mit dieser Funktion wird die Instanz nicht bewegt. Sie legt nur einen Pfad fest, und Sie müssen die Pfadfunktionen für die Bewegung verwenden.

Syntax:

mp_linear_path(path, xgoal, ygoal, stepsize, checkall)

ArgumentTyp Beschreibung
path The index of the path to be used.
xgoal The target x position.
ygoal The target y position.
stepsize The speed the instance moves in pixels per step.
checkall Whether to check for collisions with all instances (true) or just solid ones (false).

Rückgabe:

Beispiel:

if mp_linear_path(path, obj_Player.x, obj_Player.y, 2, 0)
{
    path_start(path, 2, 0, 0);
}

Dies veranlasst das Objekt zu prüfen, ob es einen linearen Pfad von seiner aktuellen Position zur Position von "obj_Player" gibt. Ist dies der Fall, beginnt es den Pfad.
