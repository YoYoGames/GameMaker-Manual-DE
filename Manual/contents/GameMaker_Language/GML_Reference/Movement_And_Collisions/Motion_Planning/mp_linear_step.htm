mp_linear_step

Mit dieser Funktion weisen Sie eine Instanz an, einen "Schritt" in Richtung eines bestimmten Punktes zu machen, der durch die Werte xgoal und ygoal angegeben wird. Die Größe des Schrittes (d.h. wie viele Pixel sich die Instanz pro Schritt bewegen soll) wird durch die Schrittgröße angegeben, und wenn die Instanz bereits an der Position ist, wird sie anhalten und sich nicht weiter bewegen, im Gegensatz zu anderen, einfacheren Funktionen wie move_towards_point(). Die Schrittweite ist auch der Abstand "vor" dem das Objekt bei jedem Schritt auf eine Kollision prüft, und du kannst wählen, ob die Instanz bei einer Kollision mit jeder Instanz anhält oder nur mit denen, die als solide markiert sind. Die Funktion gibt zurück, ob sie die Zielposition erreicht hat (true) oder ob sie fehlgeschlagen ist (false).

HINWEIS Diese Funktion versucht nicht, einen Umweg zu machen, wenn sie auf ein Hindernis stößt, sie scheitert einfach und hält die Fahrt an.

Syntax:

mp_linear_step(xgoal, ygoal, stepsize, checkall);

ArgumentTyp Beschreibung
xgoal The target x position.
ygoal The target y position.
stepsize The speed the instance moves in pixels per step.
checkall Whether to check all instances (true) or just solid ones (false).

Rückgabe:

Beispiel:

if mp_linear_step(mouse_x, mouse_y, 5, 0)
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

Die obige code wird das Objekt mit einer Geschwindigkeit von 5 Pixeln pro Schritt auf die Maus zubewegen. Sobald es die Mausposition erreicht hat, wird es ein Objekt "obj_Explosion" erstellen und sich selbst zerstören.
