mp_grid_draw

Diese Funktion zeichnet das angegebene MP-Gitter (wie durch mp_grid_create()), wobei freie Zellen als grün und verbotene Zellen als rot markiert werden. Diese Funktion ist als Debug-Werkzeug unerlässlich, aber es sollte beachtet werden, dass sie sehr langsam ist und nur funktioniert, wenn sie im Draw-Ereignis der Instanz verwendet wird, und dass du das Draw-Alpha einstellen kannst, um die Deckkraft des Gitters zu ändern, was es dir erlaubt, es als Overlay zu zeichnen und gleichzeitig zu sehen, was tatsächlich im Raum ist.

Syntax:

mp_grid_draw(id);

Argument Typ Beschreibung
id MP Raster-ID Index of the mp_grid that is to be drawn

Rückgabe:

Boolean

Beispiel:

draw_set_alpha(0.3);
mp_grid_draw(grid);
draw_set_alpha(1);

Die obige code zeichnet das mp_grid, das in der Variable "grid" indiziert ist, als halbtransparentes Overlay (aber nur, wenn die Instanz, auf der die code läuft, eine geringere Tiefe als alle anderen hat).
