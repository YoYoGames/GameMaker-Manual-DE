mp_linearer_Weg_Objekt

Diese Funktion berechnet einen geradlinigen Weg von der aktuellen Instanzposition zu dem durch die Werte "xgoal" und "ygoal" angegebenen Punkt. Sie verwendet die angegebene Schrittweite, die gleiche wie bei der Funktion mp_linear_step(). Der angegebene Pfad muss bereits existieren und wird durch den neuen Pfad überschrieben und die Funktion gibt zurück, ob ein vollständiger Pfad gefunden wurde (true) oder nicht (false). Ein vollständiger Pfad wird nur gefunden, wenn es keine Kollision mit dem angegebenen Objekt oder der Instanz gab. Wenn false zurückgegeben wird, wird trotzdem ein Pfad generiert, der aber nur bis zu der Position läuft, an der der Pfad blockiert war.

HINWEIS: Mit dieser Funktion wird die Instanz nicht bewegt. Sie legt nur einen Pfad fest, und Sie müssen die Pfadfunktionen für die Bewegung verwenden.

Syntax:

mp_linear_path_object(path, xgoal, ygoal, stepsize, obj)

Argument Typ Beschreibung
path Pfad Anlage The index of the path to be used
xgoal Real The target x position.
ygoal Real The target y position.
stepsize Real The speed the instance moves in pixels per step.
obj Objekt Asset- oder Instanz-ID The object that is to block the path. Can be an object index, an instance id or the special keyword ,all

Rückgabe:

Boolean

Beispiel:

if mp_linear_path_object(path, mouse_x, mouse_y, 4, obj_Wall)
{
    path_start(path, 4, 0, 0);
}

Die obige code prüft auf eine Kollision mit "obj_Wall" entlang des Pfades zwischen dem Objekt, das die code ausführt, und der x/y-Position der Maus. Wenn es keine Kollision gibt und der komplette Pfad generiert ist, wird das Objekt entlang des generierten Pfades gestartet.
