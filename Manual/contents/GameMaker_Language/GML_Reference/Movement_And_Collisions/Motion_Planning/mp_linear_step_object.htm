mp_linear_step_object

Mit dieser Funktion weisen Sie eine Instanz an, einen "Schritt" in Richtung eines bestimmten Punktes zu machen, der durch die Werte xgoal und ygoal angegeben wird. Die Größe des Schrittes (d.h. wie viele Pixel sich die Instanz pro Schritt bewegen soll) wird durch die Schrittgröße angegeben, und wenn die Instanz bereits an der Position ist, wird sie anhalten und sich nicht weiter bewegen, im Gegensatz zu anderen, einfacheren Funktionen wie move_towards_point(). Die Schrittweite ist auch der Abstand "vor" dem das Objekt bei jedem Schritt auf eine Kollision prüft, und du kannst wählen, ob die Instanz bei einer Kollision mit einem Objekt oder einer Instanz deiner Wahl anhält.

Syntax:

mp_linear_step_object(xgoal, ygoal, stepsize, obj);

Argument Typ Beschreibung
xgoal Real The target x position.
ygoal Real The target y position.
stepsize Real The speed the instance moves in pixels per step.
obj Objekt Asset- oder Instanz-ID The object that is to block the path. Can be an object index, an instance id or the special keyword, all

Rückgabe:

Boolean

Beispiel:

if mp_linear_step_object(mouse_x, mouse_y, 5, obj_Wall)
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

Die obige code lässt das Objekt mit einer Geschwindigkeit von 5 Pixeln pro Schritt auf die Maus zugehen, wobei nur auf Kollisionen mit dem Objekt "obj_Wall" geprüft wird. Sobald es die Mausposition erreicht hat, wird es ein Objekt "obj_Explosion" erzeugen und sich selbst zerstören.
