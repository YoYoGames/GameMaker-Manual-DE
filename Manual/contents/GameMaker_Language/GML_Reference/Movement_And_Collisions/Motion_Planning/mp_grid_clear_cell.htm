mp_grid_clear_cell

Diese Funktion kann verwendet werden, um eine bestimmte "Zelle" eines MP-Gitters zu löschen. Zellen werden nicht als Raumkoordinaten, sondern als Gitterkoordinaten berechnet, wobei (0,0) die obere linke Ecke des Gitters ist. Das bedeutet, dass wir, um eine Zelle an einer bestimmten Position im Raum zu löschen, die x- und y-Koordinaten in Zellkoordinaten ändern müssen, indem wir sie durch die Auflösung des MP-Gitters teilen. Das folgende code Beispiel zeigt, wie das funktioniert.

Syntax:

mp_grid_clear_cell(id, h, v);

Argument Typ Beschreibung
id MP Raster-ID Index of the mp_grid that is to be used
h Real Horizontal position of the cell to clear
v Real vertical position of the cell to clear

Rückgabe:

Boolean

Beispiel:

with (obj_Box)
{
    mp_grid_clear_cell(grid, floor(x / 32), floor(y /32));
    instance_destroy();
}

Die obige code bewirkt, dass sich alle "obj_Box" selbst zerstören und die von ihnen belegten Zellen im mp_grid, die in der Variablen "grid" indiziert sind, als frei markieren. In diesem Beispiel finden wir die entsprechende Zelle, indem wir die x/y-Koordinaten des Objekts nehmen und sie durch die Auflösung des Gitters dividieren (mit floor, um die Werte als ganze Zahlen zu erhalten).
