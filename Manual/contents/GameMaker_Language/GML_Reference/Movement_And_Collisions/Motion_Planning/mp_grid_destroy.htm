mp_grid_destroy

Sie können diese Funktion verwenden, um das angegebene mp_grid (wie durch die Funktion mp_grid_create()) zu zerstören und den ihm zugewiesenen Speicher freizugeben. Es ist wichtig, dass du diese Funktion immer dann aufrufst, wenn das MP-Gitter fertig ist, da es sonst zu einem Speicherleck kommen kann, was bedeutet, dass dein Spiel mit der Zeit langsamer wird und schließlich abstürzt.

HINWEIS: Die Verwendung von mp_grid_* Funktionen auf einem Gitter, nachdem es zerstört wurde, führt zu einem Fehler.

Syntax:

mp_grid_destroy(id);

Argument Typ Beschreibung
id MP Raster-ID Index of the mp_grid that is to be destroyed

Rückgabe:

N/A

Beispiel:

if timer = 0
{
    mp_grid_destroy(grid);
    room_goto(rm_Menu);
}

Die obige code wird das in der Variable "grid" indizierte mp_grid zerstören, wenn die Variable "timer" gleich 0 ist, und dann in einen anderen Raum gehen.
