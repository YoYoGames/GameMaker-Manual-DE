verschieben_außen_alle

Mit dieser Funktion kannst du einer Instanz sagen, dass sie sich aus einer Kollision herausbewegen soll, und zwar in eine beliebige Richtung und um eine beliebige Anzahl von Pixeln pro Schritt, wobei ein Wert von -1 oder 0 für maxdist ein Standardwert von 1000px ist, d.h.: GameMaker wird die Instanz kontinuierlich um 1000 Pixel nach oben bewegen, bis sie aus der Kollision heraus ist.

Syntax:

move_outside_all(dir, maxdist);

ArgumentTyp Beschreibung
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Rückgabe:

Beispiel:

if place_meeting(x, y, all)
{
    move_outside_all(90, 1);
}

Die obige code verschiebt die Instanz bei jedem Schritt, bei dem sie mit einer anderen Instanz kollidiert, um 1 Pixel nach oben.
