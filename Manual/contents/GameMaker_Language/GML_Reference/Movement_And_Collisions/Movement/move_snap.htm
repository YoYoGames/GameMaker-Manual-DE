move_snap

Diese Funktion wird verwendet, um die Instanz an einem Raster einer bestimmten Größe "einzufangen". Sie wird an der nächstgelegenen entsprechenden Position auf dem "unsichtbaren" Raster, das durch die Werte hsnap und vsnap definiert ist, eingerastet.

Syntax:

move_snap(hsnap, vsnap);

ArgumentTyp Beschreibung
hsnap The horizontal snapping (the size in pixels between 'cells').
vsnap The vertical snapping (the size in pixels between 'cells').

Rückgabe:

Beispiel:

with (obj_Pieces)
{
    if (!place_snapped(32, 32))
    {
        move_snap(32, 32);
    }
}

Die obige code überprüft alle Instanzen von "obj_Pieces", um zu sehen, ob sie an einem Raster von 32x32 Pixeln gefangen sind, und wenn nicht, werden sie an der nächstgelegenen Position in diesem Raster gefangen.
