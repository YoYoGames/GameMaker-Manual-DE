move_random

Mit dieser Funktion kannst du die Instanz auf eine beliebige Position im Raum setzen, die jedoch an einem "unsichtbaren" Raster ausgerichtet ist. Ein Wert von 32 für hsnap und vsnap setzt die Instanz auf eine zufällige Position, die an einem Raster von 32x32 Quadraten ausgerichtet ist. Sie können diese Werte auf 1 setzen, um eine beliebige Position im Raum zu erhalten.

HINWEIS: Diese Funktion rastet jedes Mal an denselben Positionen ein, wenn das Spiel neu gestartet wird, da GameMaker jedes Mal denselben anfänglichen Zufallswert erzeugt, um debugging code zu vereinfachen. Um dieses Verhalten zu vermeiden, verwenden Sie randomise() beim Start des Spiels. Dies gilt nur für das Testen und debugging des Spiels, da das endgültige ausführbare Paket dieses Verhalten nicht zeigt und bei jedem Spiel zufällig ist.

Syntax:

move_random(hsnap, vsnap);

ArgumentTyp Beschreibung
hsnap The horizontal snapping (the size in pixels between 'cells').
vsnap The vertical snapping (the size in pixels between 'cells').

Rückgabe:

Beispiel:

if keyboard_check(vk_space)
{
    move_random(1, 1);
}

Dadurch wird die Instanz an eine zufällige Position irgendwo im Raum bewegt, wenn die Leertaste gedrückt wird.
