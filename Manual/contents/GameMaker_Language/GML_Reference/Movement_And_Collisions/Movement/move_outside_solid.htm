bewegen_außen_fest

Mit dieser Funktion kannst du einer Instanz sagen, dass sie sich aus einer Kollision mit einer beliebigen Instanz herausbewegen soll, die als solid in eine beliebige Richtung und um eine beliebige Anzahl von Pixeln pro Schritt bewegen, wobei ein Wert von -1 oder 0 für maxdist ein Standardwert von 1000px ist, d.h.: GameMaker wird die Instanz kontinuierlich um 1000 Pixel nach oben bewegen, bis sie außerhalb der Kollision ist.

Syntax:

move_outside_solid(dir, maxdist);

ArgumentTyp Beschreibung
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Rückgabe:

Beispiel:

if other.solid
{
    var pdir;
    pdir = point_direction(other.x, other.y, x, y);
    move_outside_solid(pdir, -1);
}

Die obige code würde in ein Kollisionsereignis gehen und prüfen, ob die "andere" Instanz in der Kollision als "solid" gekennzeichnet ist. Wenn ja, wird die Instanz aus der Kollision herausgezogen.
