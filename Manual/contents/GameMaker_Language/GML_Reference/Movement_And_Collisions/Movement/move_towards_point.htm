bewegung_zum_punkt

Diese Funktion weist die Instanz, die code ausführt, einfach an, sich mit einer bestimmten Geschwindigkeit auf einen bestimmten Punkt zuzubewegen. Beachte, dass es an dir liegt, ihr zu sagen, was sie tun soll, wenn sie dort angekommen ist, da sie nicht einfach von selbst aufhört. Das bedeutet, dass Sie eine "vibrierende" Instanz erhalten können, wenn sie sich (zum Beispiel) 3 Pixel vorwärts bewegt, die x/y-Zielposition um 2 überschreitet, sich dann wieder 3 Pixel zurückbewegt und den Zielpunkt um einen überschreitet und so weiter... Es gibt eine Reihe von Möglichkeiten, dies zu vermeiden, zum Beispiel mit distance_to_point() um zu sehen, wie weit die Instanz von der angegebenen Position entfernt ist, bevor sie sich bewegt.

HINWEIS: Diese Funktion legt die Geschwindigkeit der Instanz fest. Auch wenn Sie mit code aufhören, bewegt sich die Instanz weiter in die vorherige Richtung, daher ist es notwendig, die Instanz speed auf 0 zu setzen, wenn du willst, dass sie anhält. Siehe das code Beispiel unten.

Syntax:

move_towards_point( x, y, sp );

Argument Typ Beschreibung
x Real The x position of the point to move towards.
y Real The y position of the point to move towards.
sp Real The speed to move at in pixels per step.

Rückgabe:

N/A

Beispiel:

if point_distance(x, y, target.x, target.y) > 5
{
    move_towards_point(target.x, target.y, 5);
}
else speed = 0;

Die obige code bewegt die Instanz mit einer Geschwindigkeit von 5 Pixeln pro Schritt in Richtung der in der Variablen "target" indizierten Instanz. Sobald sie diese Position erreicht hat, wird die Geschwindigkeit auf 0 gesetzt.
