motion_set

Diese Funktion setzt eine neue Bewegungsrichtung und eine neue Geschwindigkeit für die Instanz, die die code ausführt. Beachten Sie, dass dies nicht zur aktuellen Geschwindigkeit und Richtung der Instanz hinzukommt (dafür würden Sie motion_add()), sondern sie wird auf die neuen Einstellungen gezwungen.

Syntax:

motion_set(dir, speed);

ArgumentTyp Beschreibung
dir The new direction.
speed The new speed.`

Rückgabe:

Beispiel:

if irandom(9) = 1
{
    motion_set(random(360), 1 + random(3));
}

Die oben genannte code lässt die Instanz in zufälligen Abständen Geschwindigkeit und Richtung ändern.
