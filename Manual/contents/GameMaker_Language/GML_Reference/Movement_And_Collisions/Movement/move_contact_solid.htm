bewegen_kontakt_fest

Diese Funktion verschiebt die Instanz, auf der code läuft, um eine bestimmte Anzahl von Pixeln in die angegebene Richtung, bis sie auf eine Instanz mit solid als wahr markiert ist. Du kannst -1 oder 0 für die maxdist verwenden, die standardmäßig 1000px beträgt, d.h.: GameMaker wird die Instanz kontinuierlich um 1000 Pixel nach oben bewegen, bis sie nicht mehr kollidiert.

Syntax:

move_contact_solid( dir, maxdist );

ArgumentTyp Beschreibung
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Rückgabe:

Beispiel:

if !place_meeting(x, y + 1, obj_Floor)
{
    move_contact_solid(270, -1);
}

Die obige code prüft unterhalb der Instanz auf eine Kollision mit "obj_Floor", und wenn es keine gibt, dann wird es nach unten bewegt, bis es eine Kollision gibt oder 1000 Pixel abgedeckt wurden.
