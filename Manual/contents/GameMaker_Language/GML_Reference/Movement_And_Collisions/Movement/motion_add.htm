motion_add

Diese Funktion ändert die aktuellen direction und speed der Instanz, auf der code läuft, und fügt die neuen Werte zu den aktuellen Werten hinzu. Wenn Sie diese Werte einfach nur ändern wollen, sollten Sie die Funktion motion_set() verwenden.

Syntax:

motion_add(dir, speed);

ArgumentTyp Beschreibung
dir The added direction.
speed The added speed.

Rückgabe:

N/A

Beispiel:

var pdir;
pdir = point_direction(other.x, other.y, x, y);
motion_add(pdir, other.speed);
if speed > 8 speed = 8;

würde das obige code im Kollisionsereignis mit einem anderen Objekt aufgerufen werden. Es fügt der Bewegungsrichtung und der Geschwindigkeit der Instanz eine vector hinzu, die auf der Position und der Geschwindigkeit der anderen an der Kollision beteiligten Instanz basiert. Es begrenzt dann die Geschwindigkeit, wenn sie über 8 (Pixel pro Schritt) geht.
