umziehen_kontakt_alle

Diese Funktion bewegt die Instanz, die auf code läuft, eine bestimmte Anzahl von Pixeln in die angegebene Richtung, bis sie auf eine andere Instanz mit einer gültigen Maske trifft. Du kannst -1 oder 0 für die maxdist verwenden, die standardmäßig 1000px beträgt, d.h.: GameMaker wird die Instanz kontinuierlich um 1000 Pixel nach oben bewegen, bis sie nicht mehr kollidiert.

Syntax:

move_contact_all(dir, maxdist);

ArgumentTyp Beschreibung
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Rückgabe:

Beispiel:

if !place_meeting(x, y + 1, all)
{
    move_contact_all(270, -1);
}

Die obige code prüft unterhalb der Instanz auf eine Kollision, und wenn es keine gibt, dann wird sie nach unten verschoben, bis es eine gibt oder bis 1000 Pixel abgedeckt wurden.
