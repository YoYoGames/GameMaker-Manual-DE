move_bounce_all

Mit dieser Funktion kannst du einer Instanz befehlen, an allen Instanzen innerhalb des Raumes abzuprallen, mit der einzigen Ausnahme der Instanzen, denen kein sprite oder Maskenindex zugewiesen ist. Du kannst ihr auch sagen, dass sie eine präzise Kollisionsprüfung verwenden soll, wenn sie aktiviert ist, aber sei dir bewusst, dass dies voraussetzt, dass alle Instanzen präzise Masken aktiviert haben und dein Spiel stark verlangsamen wird, wenn viele Instanzen involviert sind, aufgrund der Menge der Verarbeitung, die durchgeführt werden muss. Dies sollte normalerweise im Schritt-Ereignis einer Instanz geschehen, kann aber auch selektiv in einem Kollisions-Ereignis verwendet werden, wie das folgende Beispiel von code zeigt.

Syntax:

move_bounce_all( adv );

ArgumentTyp Beschreibung
adv Whether to enable advanced bouncing (true) or not (false).

Rückgabe:

Beispiel:

if other.visible
{
    move_bounce_all(false);
}

Die obige code würde in einem Kollisionsereignis mit einem anderen Objekt platziert werden. Zuerst wird geprüft, ob das Objekt sichtbar ist, und wenn ja, dann wird die Aktion move_bounce_all() ausgeführt. Beachten Sie, dass der Abprall in diesem Fall selektiv ist und nur für diese Kollision berechnet wird und nicht für jede Instanz, jeden Schritt.
