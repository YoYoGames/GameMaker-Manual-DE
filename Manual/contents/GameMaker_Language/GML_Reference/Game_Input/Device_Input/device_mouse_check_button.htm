device_mouse_check_button

Diese Funktion gibt true oder false zurück, je nachdem, ob die angegebene Maustaste auf dem angegebenen Gerät gedrückt gehalten wird. Für die Maustaste können Sie eine der auf dieser Seite aufgeführten Konstanten angeben.

Diese Funktion kann für Touchscreens verwendet werden, und das Argument device kann zwischen 0 und n liegen, wobei n die maximale Anzahl der "Finger" ist, die den Bildschirm gleichzeitig berühren können (die maximale Anzahl der Berührungen, die (die maximale Anzahl der Berührungen, die erkannt werden können, hängt stark von dem verwendeten Gerät und dem darauf laufenden Betriebssystem ab, aber die meisten Geräte sollten mindestens bis zu 4 Berührungen erkennen). Beachten Sie, dass mb_right nur erkannt wird, wenn eine doppelte Berührung erkannt wird (dieses Verhalten kann mit der Funktion die Funktion device_mouse_dbclick_enable()).

Syntax:

device_mouse_check_button(device, button)

ArgumentTyp Beschreibung
device The device (from 0 - n) that is being checked
button The button of the device that is being checked.

Rückgabe:

Beispiel:

if device_mouse_check_button(0, mb_left) && device_mouse_check_button(1, mb_left)
{
    room_goto(rm_Menu);
}

Die obige code prüft, ob die beiden Berührungen gleichzeitig gedrückt werden, und wenn ja, geht sie in einen anderen Raum.
