gerät_ist_tastatur_offen

Diese Funktion prüft das Gerät auf eine Tastatur, und wenn eine vorhanden ist, wird true zurückgegeben, andernfalls wird false zurückgegeben. Bitte beachten Sie, dass diese Funktion hauptsächlich für die Verwendung mit Android Geräten gedacht ist.

Für Benutzer mit einem Sony Xperia Play gibt es eine feste Tastenbelegung im GameMaker, so dass Sie die Tastaturkonstanten vk_up, vk_down, vk_left, vk_right für die Joypad-Tasten verwenden können und Dreieck ist ord("T"), Quadrat ist ord("S"), Kreis ist vk_alt + vk_backspace, während die Zurück-Taste einfach vk_backspace ist, Kreuz ist vk_space, Select ist vk_return und Start ist vk_rshift, der L-Trigger ist ord("L") und R-Trigger ist ord("R").

Syntax:

device_is_keypad_open

Rückgabe:

Beispiel:

if device_is_keypad_open()
{
     global.Setting = 2;
}
else
{
     global.Setting = 1;
}

Die obige code prüft, ob eine Tastatur vorhanden ist, und ändert dann eine globale Variable in Abhängigkeit von dem zurückgegebenen Wert.
