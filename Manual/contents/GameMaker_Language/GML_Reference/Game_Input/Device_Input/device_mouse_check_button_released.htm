device_mouse_check_button_released

Diese Funktion gibt true oder false zurück, je nachdem, ob die von Ihnen angegebene Geräteberührung ausgelöst wurde oder nicht. Das Gerät argument bezieht sich auf die Berührungsnummer des Geräts, die zwischen 0 und n liegen kann. Beachten Sie, dass die maximale Anzahl der Berührungen, die erkannt werden können, sehr stark von dem verwendeten Gerät und dem darauf laufenden Betriebssystem abhängt (die meisten Geräte unterstützen mindestens 4 gleichzeitige Berührungen). Diese Funktion wird nur einmal durch die eigentliche Loslass-Aktion ausgelöst, und die auf dieser Seite aufgeführten Konstanten können zur Überprüfung der Maustasten verwendet werden. Beachten Sie, dass mb_right nur dann erkannt wird, wenn eine doppelte Berührung erkannt und beim zweiten Antippen vor dem Loslassen gehalten wird (dieses Verhalten kann mit der Funktion device_mouse_dbclick_enable() deaktiviert werden).

Syntax:

device_mouse_check_button_released(device, button);

ArgumentTyp Beschreibung
device The device (from 0 - n) that is being checked.
button The button of the device that is being checked.

Rückgabe:

Beispiel:

if device_mouse_check_button_released(0, mb_left)
{
    press = false;
}

Die obige code prüft, ob das Gerät gedrückt wurde und setzt eine Variable auf false, wenn dies der Fall ist.
