device_mouse_raw_x

Diese Funktion gibt die rohe x-Position einer Berührung auf dem Gerät zurück. Das bedeutet, dass sie die tatsächliche Gerätedefinition der x-Position, die berührt wird, zurückgibt, nicht die GameMaker-Definition, und daher Dinge wie Ansichtsposition und Skalierung ignoriert. Beachten Sie, dass die maximale Anzahl von Berührungen, die erkannt werden können, sehr stark vom verwendeten Gerät und dem darauf laufenden Betriebssystem abhängt

HINWEIS: Diese Funktion ist stark geräteabhängig und Sie sollten zunächst mit dem gewünschten Ziel module und dem Gerät experimentieren, um zu sehen, was genau zurückgegeben wird.

Syntax:

device_mouse_raw_x(device);

ArgumentTyp Beschreibung
device The device (from 0 - n) that is being checked.

Rückgabe:

Beispiel:

if device_mouse_check_button(0, mb_left) && device_mouse_check_button(1, mb_left)
{
    x_av = mean(device_mouse_raw_x(0), device_mouse_raw_x(1));
    y_av = mean(device_mouse_raw_y(0), device_mouse_raw_y(1));
}

Die obige code prüft, ob Gerät1 und Gerät2 gedrückt werden, und wenn dies der Fall ist, wird die durchschnittliche Position der x/y-Koordinaten zwischen den einzelnen Druckpunkten berechnet.
