device_mouse_dbclick_enable

Mit dieser Funktion kann das Gerät so eingestellt werden, dass es ein doppeltes Antippen der mb_left (linke Maustaste) als mb_right (rechte Maustaste) Eingabe erkennt.

Standardmäßig ist dies auf true eingestellt, was bedeutet, dass jedes Mal, wenn der Benutzer zweimal schnell hintereinander auf den Bildschirm des Geräts tippt, der Rückgabewert derselbe ist, als ob mit der rechten Maustaste geklickt worden wäre. Wenn diese Funktion aktiviert ist, wird das erste Antippen als mb_left und das zweite als mb_right erkannt. Stellen Sie daher sicher, dass die von Ihnen verwendete code dies berücksichtigt.

Syntax:

device_mouse_dbclick_enable(bool);

Argument Typ Beschreibung
bool Boolesche Set double-click detection on (true) oder ausgeschaltet (false).

Rückgabe:

Boolean

Beispiel:

if os_type == os_windows || os_type == os_mac
{
    device_mouse_dbclick_enable(false);
}

Die obige code prüft, ob es sich bei dem Gerät, auf dem das Spiel läuft, um einen Windows PC oder einen Mac handelt, und deaktiviert in diesem Fall die Doppeltipp-Funktion.
