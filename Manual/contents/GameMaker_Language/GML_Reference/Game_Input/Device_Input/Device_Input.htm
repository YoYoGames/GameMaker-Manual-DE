Gerät eingeben

Wenn Ihr Spiel auf die platforms ausgerichtet ist, die Multi-Touch erlauben, sollten Sie diese Funktionen verwenden. Sie funktionieren genauso wie ein PC, an dem mehrere Mäuse angeschlossen sind, und die Konstanten, auf die geprüft wird, sind die gleichen wie die für die Standard-Mausfunktionen verwendeten:

Konstante Beschreibung
mb_left Die linke Maustaste
mb_middle Die mittlere Maustaste (dies gilt möglicherweise nicht für alle Ziele platforms)
mb_right Die rechte Maustaste
mb_side1* Seitliche Maustaste 1
mb_side2* Seitliche Maustaste 2
mb_any Eine der Maustasten
mb_none Keine Maustaste

HINWEIS * Die Schaltflächen mb_side1 und mb_side2 sind nur auf Windows, macOS, Ubuntu und HTML5 verfügbar.

Die Funktionen können alle einen "Berührungs"-Wert annehmen, der einer Berührung auf dem Gerätedisplay entspricht. Die maximale Anzahl der Berührungen, die erkannt werden können, hängt jedoch sehr stark vom Gerät und dem ausgeführten Betriebssystem ab. Zum Beispiel können einige Low-End-Geräte ( Android ) nur 2 oder 3 Berührungen gleichzeitig erkennen, während ein UWP-Gerät bis zu 10 Berührungen erkennen kann. Beachten Sie auch, dass auf Touchscreen-Geräten ein Doppeltipp als das Äquivalent eines Ereignisses der rechten Maustaste zurückgegeben wird. Wenn Sie also z. B. mehrere Berührungen kurz hintereinander erkennen müssen, müssen Sie sowohl mb_left als auch mb_right überprüfen. Diese Funktion kann mit der folgenden Funktion ein- oder ausgeschaltet werden device_mouse_dbclick_enable().

Nachfolgend finden Sie eine Liste aller gerätespezifischen Steuerfunktionen:

  • device_mouse_check_button
  • device_mouse_check_button_pressed
  • device_mouse_check_button_released
  • device_mouse_dbclick_enable
  • device_mouse_x
  • device_mouse_y
  • device_mouse_raw_x
  • device_mouse_raw_y
  • device_mouse_x_to_gui
  • device_mouse_y_to_gui
  • device_is_keypad_open
  • device_get_tilt_x
  • device_get_tilt_y
  • device_get_tilt_z

Bitte beachten Sie, dass Sie auf mobilen Geräten (Android, iOS und Windows Phone) jederzeit die Zurück-Taste verwenden können, da sie der Rücktaste der Tastatur zugeordnet ist, die die GameMaker-Tastaturkonstante vk_backspace ist. Sie können also Prüfungen für diese Taste durchführen und Aktionen auf diesen Geräten ausführen, als ob die Zurück-Taste verwendet worden wäre.
