gerät_gekippt_y

Diese Funktion gibt einen Wert zwischen -1 und 1 zurück, je nach dem "Neigungswinkel" des Geräts. Die tatsächliche Korrelation zwischen dem Neigungswinkel und dem zurückgegebenen Wert hängt vom Gerät und dem verwendeten Betriebssystem ab, aber im Allgemeinen ist ein Wert von 1 oder -1 dasselbe wie +/-90°. Die nachstehende Abbildung zeigt, wie sich jede der verfügbaren Funktionen auf das Gerät bezieht:

Syntax:

device_get_tilt_y()

Rückgabe:

Beispiel:

if display_get_orientation() = display_landscape
{
    x += sign(device_get_tilt_y());
}
else
{
    x += sign(device_get_tilt_x());
}

Die obige code überprüft die Ausrichtung des Displays und verwendet dann den entsprechenden Neigungswert, um den Player entlang der X-Achse zu bewegen.
