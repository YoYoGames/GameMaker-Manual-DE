gamepad_remove_mapping

Diese Funktion kann verwendet werden, um die aktuelle Gerätezuordnung aus dem angegebenen Gamepad-Slot-Index zu entfernen. Einmal aufgerufen, muss der Slot-Index mit der Funktion gamepad_test_mapping() Funktion neu zugeordnet werden, um die Gamepad-Konstanten zur korrekten Erkennung von Eingaben verwenden zu können (direkte Eingaben können immer mit den Funktionen gamepad_axis/button/hat_count() und gamepad_axis/button/hat_value() zusammen abgerufen werden).

Syntax:

gamepad_remove_mapping(index);

ArgumentTyp Beschreibung
index Which gamepad index "slot" to remove the mapping from.

Rückgabe:

Beispiel:

if remap == true
{
    gamepad_remove_mapping(global.PadIndex);
}

Die obige code entfernt die Zuordnung aus dem angegebenen Gamepad-Index-Slot basierend auf dem Wert einer Variablen.
