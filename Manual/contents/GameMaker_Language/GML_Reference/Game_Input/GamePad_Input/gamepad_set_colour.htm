gamepad_set_colour

Diese Funktion kann verwendet werden, um die Farbe der LEDs in einem PlayStation-Controller einzustellen. Sie geben den einzustellenden Gerätesteckplatz an und geben dann eine Farbe an, die eine der Farbkonstanten oder ein mit den spezifischen Farbfunktionen erstellter Farbwert oder ein HEX-Wert (wie $FFFFFFF) sein kann.

Syntax:

gamepad_set_colour(device, colour);

ArgumentTyp Beschreibung
device Which gamepad device "slot" to set.
colour The colour to use.

Rückgabe:

Beispiel:

if health < 10
{
    gamepad_set_colour(0, c_red);
}

Die obige code setzt die LEDs der PlayStation-Steuerung auf Rot, wenn die Variable health unter 10 fällt.
