gamepad_get_mapping

Diese Funktion kann verwendet werden, um die Zuordnungszeichenfolge für das Gamepad abzurufen. Diese Zeichenkette wird entweder sein:

  • Die aktuelle Kartenzeichenfolge für das Gamepad, zum Beispiel:

    "050000005e040000fd020000ffff3f00,Xbox Wireless Controller,a:b0,b:b1,start:b4,dpdown:h0.4,dpleft:h0.8,dpright:h0.2,dpup:h0.1,guide:b6,leftshoulder:b9,leftstick:b7,lefttrigger:a4,leftx:a0,lefty:a1,rightshoulder:b10,rightstick:b8,righttrigger:a5,rightx:a2,righty:a3,x:b2,y:b3,platform:android"

  • Eine Zeichenkette mit "no mapping", wenn es keine Zuordnung gibt (dies ist nicht dasselbe wie wenn keine Zuordnung vorhanden ist)
  • Ein String mit "Geräteindex außerhalb des Bereichs", wenn der Slot-Index kein gültiger Gamepad-Index ist
  • Eine leere Zeichenkette "", wenn keine Zuordnung für das angeschlossene Gamepad verfügbar ist

Weitere Informationen über die Formatierung des zurückgegebenen Map-Strings finden Sie in der Funktion gamepad_test_mapping().

Syntax:

gamepad_get_mapping(index)

ArgumentTyp Beschreibung
index Which gamepad index "slot" to get the mapping from.

Rückgabe:

Beispiel:

var _gpMap = gamepad_get_mapping(global.PadIndex);
show_debug_message("Gamepad Mapping = " + _gpMap);

Die obige code holt den Map-String für den angegebenen Gamepad-Slot und gibt ihn dann auf der Konsole für debugging aus.
