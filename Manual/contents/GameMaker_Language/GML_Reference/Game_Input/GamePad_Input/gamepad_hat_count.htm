gamepad_hat_count

Diese Funktion gibt die Gesamtzahl der Hats zurück, die für das Gamepad verfügbar sind, das mit dem angegebenen Geräte-"Slot" verbunden ist. Hüte beziehen sich im Allgemeinen auf die Auf-/Ab-/Links-/Rechts-Tasten, und beachten Sie, dass auf dem Ziel Windows Hüte nur auf DirectInput Controllern verfügbar sind (also ab Steckplatz 4 aufwärts).

Syntax:

gamepad_hat_count(device);

ArgumentTyp Beschreibung
device Which gamepad device "slot" to check.

Rückgabe:

Beispiel:

h_num = gamepad_hat_count(4);

Die obige code gibt die Anzahl der Hüte zurück, die auf dem Gamepad verfügbar sind, das an das Gerät "slot" 4 angeschlossen ist, und speichert den Wert in der Variablen "h_num".
