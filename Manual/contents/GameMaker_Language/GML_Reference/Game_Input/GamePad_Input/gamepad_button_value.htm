gamepad_button_value

Mit dieser Funktion können Sie den aktuellen Wert einer analogen Taste von 0 bis 1 ermitteln, wobei 0 für keinen Druck und 1 für vollen Druck steht. Sie geben den zu prüfenden Gamepad-Slot-Index an, zusammen mit einer Tastenkonstante (wie hier aufgeführt) oder einem Integer-Wert zwischen 0 und gamepad_button_count() - 1.

Beachten Sie, dass dies den Rohwert für die Schaltfläche zurückgibt und die Einstellung für den Schwellenwert nicht berücksichtigt wird (weitere Informationen zu Schwellenwerten finden Sie hier ).

Syntax:

gamepad_button_value(device, button);

Argument Typ Beschreibung
device Real Which gamepad device "slot" to check.
button Gamepad-Taste Konstante Which gamepad button constant to check for.

Rückgabe:

Real

Beispiel:

speed = gamepad_button_value(0, gp_shoulderrb) * 4;

Die obige code verwendet den analogen Triggerwert des Gamepads, das an Gerät "Slot" 0 angeschlossen ist, um die Geschwindigkeit der Instanz einzustellen.
