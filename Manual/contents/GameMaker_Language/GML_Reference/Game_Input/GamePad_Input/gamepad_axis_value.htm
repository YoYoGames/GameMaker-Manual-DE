gamepad_axis_value

Mit dieser Funktion können Sie den Wert der verschiedenen Achsen eines bestimmten Gamepad-Slots abfragen. Sie geben den zu prüfenden Gamepad-Slot-Index an, zusammen mit einer Achsenindex-Konstante (wie hier aufgeführt) oder einem ganzzahligen Wert zwischen 0 und gamepad_axis_count()-1. Der zurückgegebene Wert liegt zwischen -1 und 1 für jede der verfügbaren horizontalen und vertikalen Achsen.

Beachten Sie, dass der zurückgegebene Wert normalisiert wird, um den Wert zu berücksichtigen, der für die Totzone der Achse eingestellt wurde (mit der Funktion gamepad_set_axis_deadzone()), und daher immer zwischen -1 und 0 und 0 und 1 liegt.2 bedeutet zum Beispiel, dass beim Drücken des Knüppels nach rechts erst dann ein Wert zwischen 0 und 1 zurückgegeben wird, wenn der Rohwert der Achse über 0,2 liegt. Wenn der Rohwert also 0,2 beträgt, ist der Rückgabewert 0, wenn der Rohwert bei 0,5 liegt, ist der Rückgabewert 0,375, und wenn der Rohwert 0,9 beträgt, ist der Rückgabewert 0,875.

Syntax:

gamepad_axis_value(device, axisIndex);

Argument Typ Beschreibung
device Real Which gamepad device "slot" to check.
axisIndex Gamepad-Achse Konstante The axis index to check (see the constants list).

Rückgabe:

Real

Beispiel:

var haxis = gamepad_axis_value(0, gp_axislh);
var vaxis = gamepad_axis_value(0, gp_axislv);
direction = point_direction(0, 0, haxis, vaxis);
speed = point_distance(0 ,0, haxis, vaxis) * 5;

Die obige code legt die Geschwindigkeit und Richtung der Instanz auf der Grundlage der zurückgegebenen Werte des linken analogen Controllers des Gamepads fest, das in Gerät "Slot" 0 eingesteckt ist.
