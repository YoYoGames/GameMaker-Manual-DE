gamepad_ist_unterstützt

Mit dieser Funktion können Sie herausfinden, ob das Zielsystem platform Gamepads unterstützt (Rückgabe true) oder nicht (Rückgabe false).

Syntax:

gamepad_is_supported();

Rückgabe:

Beispiel:

global.GP = gamepad_is_supported();

Die obige code prüft, ob ein Gamepad unterstützt wird, und speichert den Rückgabewert in einer globalen Variablen für zukünftige Prüfungen.
