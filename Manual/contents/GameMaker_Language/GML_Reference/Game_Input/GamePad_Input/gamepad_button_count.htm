gamepad_button_count

Diese Funktion gibt die Gesamtzahl der Tasten zurück, die für das Gamepad verfügbar sind, das an das angegebene Gerät "slot" angeschlossen ist.

Syntax:

gamepad_button_count(device);

ArgumentTyp Beschreibung
device Which gamepad device "slot" to check.

Rückgabe:

Beispiel:

b_num = gamepad_button_count(0);

Die obige code gibt die Anzahl der verfügbaren Tasten auf dem Gamepad zurück, das in das Gerät "slot" 0 eingesteckt ist, und speichert den Wert in der Variablen "b_num".
