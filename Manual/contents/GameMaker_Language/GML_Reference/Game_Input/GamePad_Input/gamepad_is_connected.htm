gamepad_ist_verbunden

Diese Funktion gibt zurück, ob ein Gamepad an den angegebenen "Steckplatz" angeschlossen ist (gibt true zurück) oder nicht (gibt false zurück). Normalerweise verwenden Sie diese Funktion in Verbindung mit der Funktion gamepad_get_device_count() Funktion verwenden, um die korrekte Anzahl der verfügbaren Gamepads und/oder Gamepad-"Steckplätze" zu ermitteln. Beachten Sie, dass es zu einer leichten Verzögerung kommen kann, wenn der Benutzer das Gamepad anschließt und GameMaker es als verbunden erkennt (dies ist insbesondere bei über Bluetooth verbundenen Controllern der Fall).

HINWEIS: Auf einigen Plattformen - insbesondere Konsolen - kann diese Funktion unmittelbar nach dem Anschließen/Auswählen eines Gamepads einen falschen Wert zurückgeben und muss möglicherweise einige Frames (Schritte) später in einem Alarm überprüft werden, damit das Gamepad korrekt erkannt wird.

Syntax:

gamepad_is_connected(numb);

ArgumentTyp Beschreibung
device Which gamepad "slot" to check.

Rückgabe:

Beispiel:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
    if gamepad_is_connected(i) global.gp[i] = true else global.gp[i] = false;
}

Die obige code loops durchläuft die verfügbaren Gamepads (oder Gamepad-Slots) und prüft dann jedes einzelne auf ein angeschlossenes Gamepad. Der zurückgegebene Wert wird dann verwendet, um ein globales Array auf true oder false für die Verwendung in zukünftigen Prüfungen zu setzen.
