gamepad_get_button_threshold

Mit dieser Funktion kann die aktuelle Schwellenwerteinstellung der analogen Tasten für ein bestimmtes Gerät ermittelt werden. Der Standardschwellenwert für alle analogen Tasten ist 0,5, wobei der Bereich von 0 bis 1 reicht. Der Schwellenwert legt fest, ab welchem Punkt die Taste als "gedrückt" für Spiele gilt, bei denen sie als digitale Taste fungieren soll.

Syntax:

gamepad_get_button_threshold(device);

ArgumentTyp Beschreibung
device Which gamepad device "slot" to check.

Rückgabe:

Beispiel:

if gamepad_get_button_threshold(0) != 0.5
{
    gamepad_set_button_threshold(0, 0.5);
}

Die obige code prüft den Schwellenwert der analogen Tasten des Gamepads, das an das Gerät "Slot" 0 angeschlossen ist, und wenn dieser nicht der Standardwert 0,5 ist, wird er auf diesen Wert gesetzt.
