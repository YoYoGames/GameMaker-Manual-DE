gamepad_set_option

Mit dieser Funktion kann jede der verfügbaren Gamepad-Optionen eingestellt werden. Sie geben den "Slot" des Gamepads an, auf dem die Option gesetzt werden soll, sowie die zu setzende Optionszeichenfolge und den zu verwendenden Wert. Der verfügbare Optionsstring hängt von der platform ab, auf der das Projekt ausgeführt wird, wie unten aufgeführt:

Option Plattform Beschreibung Wert
"allow_rotation" tvOS Legen Sie fest, ob die Siri Remote drehgesperrt ist (d. h. nur vertikal verwendet werden kann) oder nicht. Wenn die Rotationssperre aufgehoben ist (ein Wert von true), ändert sich die Eingabe, so dass bei einer Drehung um 90º nach links die "rechte" Seite des Touchpads nach "oben" zurückkehrt usw., so dass alle Eingaben relativ zur Ausrichtung des Geräts erfolgen. Im gesperrten Zustand (ein Wert von false) wird die Eingabe nicht relativ zur Position des Controllers geändert und wird immer so behandelt, als ob der Controller in einer vertikalen Position wäre. Boolesche
"dpad_absolute" tvOS Legen Sie fest, ob das Siri Remote-Touchpad eine absolute oder eine relative Eingabe akzeptiert. Im absoluten Modus (ein Wert von true) wird die Mitte des Touchpads als (0,0)-Position betrachtet - d. h. dort, wo sich der Analogstick in Ruhe befindet oder wo die Mitte des D-Pads wäre - und eine Bewegung um diese Position erzeugt eine Eingabe. Bei der Einstellung "Relativ" (Wert false ) wird die Position (0,0) als der Ort betrachtet, an dem die erste Berührung des Pads erfolgt, und alle Eingaben werden von dieser Position aus generiert, nicht von der Mitte des Pads. Boolesche

Syntax:

gamepad_set_option(device, option_name, value);

Argument Typ Beschreibung
device Real Which gamepad device "slot" to set.
option_name Zeichenfolge The name of the option to set (a string, see the table above)
value Real, Boolesch oder String The value to set the option to (can be boolean, real or string - see the table above)

Rückgabe:

N/A

Beispiel:

for(var i = 0; i < 12; ++i;)
{
    if (gamepad_is_connected(i) && gamepad_get_description(i) == "tvOS Siri Remote")
    {
        if (gamepad_get_option(i, "allow_rotation") == false)
        {
            gamepad_set_option(i, "allow_rotation", true);
        }
    }
}

Die obige code loops durchläuft alle Gamepad-Slots und sucht auf tvOS platform nach der "Siri Remote". Wenn eine solche erkannt wird, wird die Fernbedienung so eingestellt, dass sie eine Eingabedrehung zulässt.
