gamepad_get_device_count

Diese Funktion zeigt entweder die Anzahl der angeschlossenen Gamepads oder die Anzahl der verfügbaren "Steckplätze" für Gamepads an.

Der tatsächliche Rückgabewert hängt von der platform und der internen Konfiguration dieser platform ab, und daher sollte diese Funktion in Verbindung mit der Funktion gamepad_is_connected() verwendet werden, um die genaue Anzahl der angeschlossenen Pads zu jedem Zeitpunkt zu ermitteln.

Syntax:

gamepad_get_device_count();

Rückgabe:

Real

Beispiel:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
    if (gamepad_is_connected(i))
    {
        global.gp[i] = true;
    }
    else
    {
        global.gp[i] = false;
    }
}

Die obige code loops durchläuft die verfügbaren Gamepads (oder Gamepad-Slots) und prüft dann jedes einzelne auf ein angeschlossenes Gamepad. Der zurückgegebene Wert wird dann verwendet, um ein globales Array auf true oder false für die Verwendung in zukünftigen Prüfungen zu setzen.
