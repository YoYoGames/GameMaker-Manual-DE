gamepad_get_option

Diese Funktion kann verwendet werden, um den angegebenen Optionswert abzurufen. Sie geben den zu prüfenden Gamepad-"Slot" sowie die Optionszeichenfolge an und die Funktion gibt den zugehörigen Wert zurück. Der verfügbare Optionsstring hängt von der platform ab, auf der das Projekt ausgeführt wird, wie unten aufgeführt:

Option Plattform Beschreibung Rückgabe
"allow_rotation" tvOS Gibt zurück, ob die Siri Remote drehgesperrt ist (d. h. nur vertikal verwendet werden kann) oder nicht. Wenn die Drehsperre aufgehoben ist, wird durch Drehen des Steuerelements die Eingabe geändert, d. h. eine Drehung um 90º nach links führt dazu, dass die "rechte" Seite des Touchpads nach "oben" zurückkehrt usw., so dass alle Eingaben relativ zur Ausrichtung des Geräts erfolgen Boolesche
"dpad_absolute" tvOS Gibt zurück, ob das Touchpad der Siri Remote absolute oder relative Eingaben akzeptiert. Im absoluten Modus wird die Mitte des Touchpads als (0,0)-Position betrachtet - d.h. dort, wo sich der Analogstick in Ruhe befindet oder wo die Mitte des Dpads wäre - und eine Bewegung um diese Position erzeugt eine Eingabe. Im relativen Modus wird die (0,0)-Position als der Ort betrachtet, an dem die erste Berührung des Pads erfolgt, und alle Eingaben werden von dieser Position aus generiert, nicht von der Mitte des Pads. Boolesche

Syntax:

gamepad_get_option(device, option_name);

ArgumentTyp Beschreibung
device Which gamepad device "slot" to check.
option_name The name of the option to get the value from (a string, see the table above)

Rückgabe:

Boolean, Real, String, or N/A (depends on platform and option)

Beispiel:

for(var i = 0; i < 12; ++i;)
{
    if (gamepad_is_connected(i) && gamepad_get_description(i) == "tvOS Siri Remote")
    {
        if (gamepad_get_option(i, "allow_rotation") == false)
        {
            gamepad_set_option(i, "allow_rotation", true);
        }
    }
}

Die obige code loops durchläuft alle Gamepad-Slots und sucht auf der tvOS platform nach der "Siri Remote". Wird eine solche erkannt, wird die Fernbedienung so eingestellt, dass sie eine Eingabedrehung zulässt.
