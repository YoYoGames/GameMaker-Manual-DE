gamepad_get_guid

Mit dieser Funktion können Sie die GUID für das Gamepad abrufen, das mit dem angegebenen Slot-Index verbunden ist. Sie geben den zu prüfenden Gamepad-Slot an (von 0 - 11), und die Funktion gibt eine der folgenden Zeichenketten zurück:

  • "keine", wenn keine GUID verfügbar ist oder kein Gamepad angeschlossen ist
  • "Geräteindex außerhalb des Bereichs", wenn der Index des Gamepad-Steckplatzes außerhalb des Bereichs liegt
  • Die GUID-Zeichenfolge, etwa "050000005e040000fd020000ffff3f00", wenn dem Steckplatz ein Gamepad zugewiesen ist und die GUID dafür abgerufen werden kann

Diese Funktion wird normalerweise in Verbindung mit gamepad_get_description() verwendet, um einen Teil des SDL Strings zu erzeugen, der für die Neuzuordnung von Controllern auf Windows, Android und macOS benötigt wird.

Plattformspezifische Hinweise
Plattform Hinweis
Oper GX Auf dem Opera GX-Ziel platform gibt diese Funktion für Nicht-DirectInput-Geräte immer "none" zurück.

Syntax:

gamepad_get_guid(index);

ArgumentTyp Beschreibung
index Which gamepad "slot" index to check

Rückgabe:

Beispiel:

var _guid = gamepad_get_guid(global.PadIndex);
var _desc = gamepad_get_description(global.PadIndex);
global.GamepadID = _guid + "," + _desc;

Die obige code ruft die GUID- und die Namensbeschreibungszeichenfolge ab, verknüpft sie und speichert die endgültige Zeichenfolge in einer globalen Variablen für zukünftige Referenzen.
