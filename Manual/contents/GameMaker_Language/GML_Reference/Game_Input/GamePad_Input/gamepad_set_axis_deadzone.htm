gamepad_set_axis_deadzone

Mit dieser Funktion kann die "tote Zone" der Joystick-Achse eingestellt werden. Sie geben den Geräte-Slot an, der eingestellt werden soll, und stellen dann einen Wert zwischen 0 und 1 ein. Wenn der eingegebene Betrag niedriger als der angegebene Wert ist, wird die Joystick-Achse als auf 0 stehend betrachtet. Beachten Sie, dass dies eine globale Einstellung ist, die sich auf alle Achsen aller Joysticks auswirkt, die an den angegebenen Geräteschacht angeschlossen sind.

Die Verwendung dieser Funktion bedeutet nicht, dass der Achsenwert mit dem Wert der Totzone beginnt, da die Funktion gamepad_axis_value() gibt immer einen normalisierten Wert zwischen -1 Wenn Sie beispielsweise die Totzone auf 0,2 einstellen, bedeutet dies, dass der Knüppel erst dann einen Wert zwischen 0 und 1 zurückgibt, wenn der Rohwert der Achse über 0,2 liegt - wenn der Rohwert also 0,2 beträgt, ist der Rückgabewert 0, Wenn der Rohwert bei 0,5 liegt, ist der Rückgabewert 0,375, oder wenn der Rohwert 0,9 ist, wäre der Rückgabewert 0,875.

Syntax:

gamepad_set_axis_deadzone(device, deadzone);

ArgumentTyp Beschreibung
device Which gamepad device "slot" to check.
deadzone The dead zone value from 0 to 1.

Rückgabe:

Beispiel:

if gamepad_is_connected(0) gamepad_set_axis_deadzone(0, 0.05);
if gamepad_is_connected(1) gamepad_set_axis_deadzone(1, 0.05);
if gamepad_is_connected(2) gamepad_set_axis_deadzone(2, 0.05);
if gamepad_is_connected(3) gamepad_set_axis_deadzone(3, 0.05);

Die obige code setzt die tote Zone der Joystick-Achse an jedem der 4 angeschlossenen Geräte auf 0,05.
