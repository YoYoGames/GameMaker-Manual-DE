gamepad_axis_count

Diese Funktion gibt die Anzahl der "Achsen"-Steuerungen auf dem geprüften Gerät zurück. Diese Steuerelemente sind die analogen "Daumensticks" der meisten Controller.

Syntax:

gamepad_axis_count(device);

ArgumentTyp Beschreibung
device Which gamepad device "slot" to check.

Rückgabe:

Beispiel:

axis = gamepad_axis_count(0))

Die obige code speichert die Anzahl der Achsen, die für das an Gerät "Slot" 0 angeschlossene Gamepad verfügbar sind, in der Variablen "axis".
