gamepad_set_button_threshold

Mit dieser Funktion kann die aktuelle Schwellenwerteinstellung der analogen Tasten für ein bestimmtes Gerät eingestellt werden. Der Standardschwellenwert für alle analogen Tasten ist 0,5, wobei der Bereich von 0 bis 1 reicht.

Der Schwellenwert legt fest, ab wann die Taste bei Spielen, bei denen sie als digitale Taste fungieren muss, als "gedrückt" gilt.

Diese Funktion wirkt sich auf die Zustände Prüfen, Drücken und Loslassen von analogen Tasten aus, nicht aber auf den Wert, der von der Funktion gamepad_button_value()zurückgegebenen Wert, der immer den Rohwert für die Taste darstellt.

Syntax:

gamepad_set_button_threshold(device, threshold);

Argument Typ Beschreibung
device Real Which gamepad device "slot" to check.
threshold Real The new threshold value (from 0 - 1, default 0.5).

Rückgabe:

N/A

Beispiel:

if gamepad_get_button_threshold(0) != 0.5
{
    gamepad_set_button_threshold(0, 0.5);
}

Die obige code prüft den Schwellenwert der analogen Tasten des Gamepads, das an das Gerät "Slot" 0 angeschlossen ist, und wenn dieser nicht der Standardwert 0,5 ist, wird er auf diesen Wert gesetzt.
