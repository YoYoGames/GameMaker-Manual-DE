gamepad_get_description

Diese Funktion gibt eine Zeichenkette mit der Beschreibung des gegebenen Gamepads zurück, z. B. kann ein PS3-Controller eine Zeichenkette ähnlich wie "PLAYSTATION(R)3 Controller" zurückgeben, oder ein XBox360-Controller könnte "Xbox 360 Controller (XInput STANDARD GAMEPAD)" zurückgeben. Diese Zeichenfolge ist hardwareabhängig und der zurückgegebene Wert hängt von dem Gamepad ab, das in den zu prüfenden Geräte-"Steckplatz" eingesteckt ist.

Syntax:

gamepad_get_description(device);

ArgumentTyp Beschreibung
device Which gamepad "slot" to get the name of.

Rückgabe:

Beispiel:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
    var _gamepad = gamepads[i];
    if (gamepad_is_connected(_gamepad))
    {
        draw_text(32, 32 + (i * 32), gamepad_get_description(_gamepad));
    }
    else
    {
        draw_text(32, 32 + (i * 32), "Gamepad not found");
    }
}

code Die obige Seite loop durchsucht alle Gamepad-Steckplätze, um zu prüfen, ob ein Gerät angeschlossen ist, und zeigt dann einen Text auf dem Bildschirm an, je nachdem, ob ein Gamepad an den Steckplatz angeschlossen ist oder nicht.
