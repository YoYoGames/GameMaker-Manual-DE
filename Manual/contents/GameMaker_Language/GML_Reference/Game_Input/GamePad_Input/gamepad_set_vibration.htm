gamepad_set_vibration

Mit dieser Funktion können Sie die Vibration der Gamepad-Motoren einstellen, wobei jeder Motor einen Wert zwischen 0 (keine Vibration) und 1 (volle Vibration) hat. switch Beachten Sie, dass es für diese Funktion keine zeitliche Begrenzung gibt. Sie müssen also eine Variable oder einen Alarm verwenden, um die Vibration nach einer bestimmten Zeit abzuschalten (Motoren auf 0 setzen), da das Gamepad sonst unbegrenzt weiter vibriert.

HINWEIS: Diese Funktion ist derzeit nur für die Standardziele Windows, PS4 und Xbox One verfügbar modules.

Syntax:

gamepad_set_vibration(device, left_motor, right_motor);

ArgumentTyp Beschreibung
device Which gamepad device "slot" to check.
left_motor The amount of vibration from the left motor from 0 to 1.
right_motor The amount of vibration from the right motor from 0 to 1.

Rückgabe:

Beispiel:

if lives = 0
{
    gamepad_set_vibration(0, 1, 1);
    alarm[0] = room_speed / 2;
}

Die obige code würde (zum Beispiel) in einem Kollisionsereignis verwendet werden, um das Gamepad, das in "Steckplatz" 0 eingesteckt ist, für eine halbe Sekunde vibrieren zu lassen, wobei der eingestellte Alarm verwendet wird, um switch wieder auszuschalten, sobald diese Zeit abgelaufen ist.
