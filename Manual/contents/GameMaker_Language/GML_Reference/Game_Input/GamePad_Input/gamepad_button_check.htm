gamepad_button_check

Diese Funktion gibt true oder false zurück, je nachdem, ob die angegebene Gamepad-Taste als gedrückt gehalten erkannt wird oder nicht. Wenn Sie eine analoge Taste prüfen, wird die Prüfung die Schwellenwerteinstellung berücksichtigen und nur dann true zurückgeben, wenn der rohe Eingabewert über dem angegebenen Schwellenwert liegt (Sie können diesen rohen Wert mit der Funktion gamepad_button_value() ermitteln).

Syntax:

gamepad_button_check(device, button);

Argument Typ Beschreibung
device Real Which gamepad device "slot" to check.
button Gamepad-Taste Konstante Which gamepad button constant to check for.

Rückgabe:

Boolean

Beispiel:

if gamepad_button_check(0, gp_face1)
{
    if (canshoot = true)
    {
        audio_play_sound(snd_Shoot, 0, false);
        instance_create_layer(x, y, "Bullets", obj_Bullet)
        canshoot = false;
        alarm[0] = room_speed / 2;
    }
}

Die obige code erkennt, ob die untere Taste der oberen Fläche (das "X" auf einem PS3-Controller) gedrückt wird, und wenn ja, schießt sie eine "Kugel"-Instanz und löst einen Alarm aus.
