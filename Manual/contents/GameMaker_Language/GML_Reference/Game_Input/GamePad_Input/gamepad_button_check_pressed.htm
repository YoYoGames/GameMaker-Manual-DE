gamepad_button_check_pressed

Diese Funktion gibt true oder false zurück, je nachdem, ob die angegebene Gamepad-Taste als gedrückt erkannt wurde oder nicht. Beachten Sie, dass diese Funktion nur einmal für die Taste ausgelöst wird, wenn sie zum ersten Mal gedrückt wird. Damit sie erneut ausgelöst wird, muss die Taste erst losgelassen und dann erneut gedrückt werden. Wenn Sie einen kontinuierlichen Tastendruck überprüfen müssen, sollten Sie die Funktion gamepad_button_check().

Wenn Sie eine analoge Taste prüfen, wird die Prüfung die Schwellenwerteinstellung berücksichtigen und nur dann true zurückgeben, wenn der rohe Eingangswert den angegebenen Schwellenwert überschreitet (Sie können diesen rohen Wert mit der Funktion gamepad_button_value() ermitteln).

Syntax:

gamepad_button_check_pressed(device, button);

Argument Typ Beschreibung
device Real Which gamepad device "slot" to check.
button Gamepad-Taste Konstante Which gamepad button constant to check for.

Rückgabe:

Boolean

Beispiel:

if gamepad_button_check_pressed(0, gp_start)
{
    audio_play_sound(snd_Start, 0, false);
    room_goto(rm_Level_1);
}

Die obige code erkennt, ob die "Start"-Taste des an Gerät "Slot" 0 angeschlossenen Gamepads gedrückt wurde oder nicht, und wechselt den Raum, wenn dies der Fall ist.
