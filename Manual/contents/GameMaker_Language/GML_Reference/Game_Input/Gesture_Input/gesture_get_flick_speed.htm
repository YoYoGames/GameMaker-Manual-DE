gesture_get_flick_speed

Mit dieser Funktion wird die Geschwindigkeit ermittelt, die erforderlich ist, damit ein Flick-Gesture-Ereignis ausgelöst wird, wenn eine Berührung oder ein Klick losgelassen wird. Die Geschwindigkeit wird in Zoll pro Sekunde gemessen und hat einen Standardwert von 2,0.

Syntax:

gesture_get_flick_speed();

Rückgabe:

(inches per second)

Beispiel:

if gesture_get_flick_speed() != 2
{
    gesture_flick_speed(2);
}

Die obige code prüft, ob die Bewegungsgeschwindigkeit für Gesten auf 2 Zoll pro Sekunde eingestellt ist, und wenn nicht, wird sie auf diesen Wert gesetzt.
