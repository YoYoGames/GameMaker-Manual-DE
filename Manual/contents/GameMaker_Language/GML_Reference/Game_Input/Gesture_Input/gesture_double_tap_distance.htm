gesture_double_tap_distance

Mit dieser Funktion können Sie den Abstand festlegen, innerhalb dessen Sie den Bildschirm nach einem einfachen Tippen erneut berühren/anklicken müssen, um eine Doppeltipp-Geste auszulösen. Der Abstand wird in Zoll gemessen und hat einen Standardwert von 0,1.

Syntax:

gesture_double_tap_distance(distance);

ArgumentTyp Beschreibung
distance The distance (in inches) to set for double tap gesture event detection.

Rückgabe:

Beispiel:

if gesture_get_double_tap_distance() != 0.1
{
    gesture_get_double_tap_distance(0.1);
}

Die obige code prüft, ob der Doppel-Tipp-Abstand für Gesten auf 0,1 Zoll eingestellt ist, und setzt ihn andernfalls auf diesen Wert.
