gesture_get_pinch_angle_towards

Diese Funktion wird verwendet, um den Winkel zu ermitteln, innerhalb dessen sich eine Berührung auf eine andere Berührung zubewegen muss, bevor eine Einklemmgeste ausgelöst werden kann. Der Winkel wird in Grad gemessen und hat einen Standardwert von 45°.

Syntax:

gesture_get_pinch_angle_towards();

Rückgabe:

(inches)

Beispiel:

if gesture_get_pinch_angle_towards() != 45
{
    gesture_pinch_angle_towards(45);
}

Die obige code prüft, ob der Einklemmwinkel für Gesten auf 45° eingestellt ist, und wenn nicht, wird er auf diesen Wert gesetzt.
