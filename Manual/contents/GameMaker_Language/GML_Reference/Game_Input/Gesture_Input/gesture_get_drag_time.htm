gesture_get_drag_time

Mit dieser Funktion wird die Zeit ermittelt, die für die Auslösung eines Drag-Start-Gesten-Ereignisses durch eine Berührung oder einen Klick benötigt wird. Diese Zeit wirkt sich auch darauf aus, wie das Tap-Ereignis ausgelöst wird, da ein Berühren/Klicken und Loslassen vor Ablauf dieser Zeit als Tap betrachtet wird. Die Zeit wird in Sekunden gemessen und hat einen Standardwert von 0,16.

Syntax:

gesture_get_drag_time();

Rückgabe:

(time in seconds)

Beispiel:

if gesture_get_drag_time() != 0.16
{
    gesture_drag_time(0.16);
}

Die obige code prüft, ob die Ziehzeit für Gesten auf 0,16 Sekunden eingestellt ist, und wenn dies nicht der Fall ist, wird sie auf diesen Wert gesetzt.
