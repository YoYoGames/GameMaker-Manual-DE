gesture_drag_distance

Mit dieser Funktion können Sie den Abstand festlegen, der erforderlich ist, damit ein Ereignis für eine Ziehgeste durch die Bewegung einer Berührung oder eines Klicks ausgelöst wird. Der Abstand wird in Zoll gemessen und hat einen Standardwert von 0,1.

Syntax:

gesture_drag_distance(distance);

ArgumentTyp Beschreibung
distance The distance (in inches) to set for drag gesture event detection.

Rückgabe:

Beispiel:

if gesture_get_drag_distance() != 0.1
{
    gesture_drag_distance(0.1);
}

Die obige code prüft, ob der Ziehabstand für Gesten auf 0,1 Zoll eingestellt ist, und wenn dies nicht der Fall ist, wird er auf diesen Wert gesetzt.
