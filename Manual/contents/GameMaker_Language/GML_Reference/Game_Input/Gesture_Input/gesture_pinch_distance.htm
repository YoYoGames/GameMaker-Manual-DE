gesten_zwicken_entfernung

Mit dieser Funktion können Sie den Abstand einstellen, innerhalb dessen Sie den Bildschirm berühren/anklicken und mit zwei Fingern bewegen müssen, bevor Sie eine Pinch-Geste auslösen. Der Abstand wird in Zoll gemessen und hat einen Standardwert von 0,1.

Syntax:

gesture_pinch_distance();

Rückgabe:

(inches)

Beispiel:

if gesture_get_pinch_distance() != 0.1
{
    gesture_pinch_distance(0.1);
}

Die obige code prüft, ob der Pinch-Abstand für Gesten auf 0,1 Zoll eingestellt ist, und wenn nicht, wird er auf diesen Wert gesetzt.
