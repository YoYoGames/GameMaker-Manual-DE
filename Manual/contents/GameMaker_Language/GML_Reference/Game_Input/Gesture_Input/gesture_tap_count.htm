gesten_tap_count

Mit dieser Funktion können Sie einstellen, ob das Zählen der Gewindebohrer aktiviert oder deaktiviert ist. Wenn sie auf true eingestellt ist, ist das Zählen der Gewindebohrer aktiviert, und wenn sie auf false eingestellt ist, ist es deaktiviert, obwohl Sie mit der Funktion gesture_get_tap_count(). Wenn sie aktiviert ist, hat jedes Tap-Ereignis einen zusätzlichen DS-Map-Eintrag "tapcount", der die Anzahl der Tap-Ereignisse registriert, und wenn sie aktiviert ist, bedeutet dies, dass alle Tap-Ereignisse ausgelöst werden, d.h. zwei Tap-Ereignisse lösen sowohl das Einzel-Tap-Ereignis als auch das Doppel-Tap-Ereignis aus, wobei der Tap-Count für das Einzel-Tap-Ereignis 1 und der Tap-Count für das Doppel-Tap-Ereignis 2 ist. Nach Ablauf der für die Erkennung eines Doppelklopfens eingestellten Zeit wird der Wert auf 0 zurückgesetzt (siehe die Funktion gesture_double_tap_time()). Wenn die Zapfwellenzählung deaktiviert ist, wird die erste Zapfung erst registriert, wenn die Zeit für die Doppelzapfung abgelaufen ist und keine zweite Zapfung erkannt wurde. Beachten Sie, dass diese Funktion standardmäßig aktiviert ist.

Syntax:

gesture_tap_count(enable);

ArgumentTyp Beschreibung
enable Enable (set to true) oder deaktivieren (auf false setzen) Sie die Tippzählung.

Rückgabe:

Beispiel:

if !gesture_get_tap_count()
{
    getsure_tap_count(true);
}

Die obige Website code prüft, ob das Zählen von Gewindeschneidern aktiviert ist, und wenn nicht, wird es eingeschaltet.
