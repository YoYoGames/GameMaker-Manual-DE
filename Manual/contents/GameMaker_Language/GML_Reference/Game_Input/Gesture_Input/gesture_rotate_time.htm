gesten_rotation_zeit

Mit dieser Funktion wird die Zeit festgelegt, innerhalb der sich ein Paar von Berührungen in eine einheitliche Richtung drehen muss, damit eine Drehstartgeste ausgelöst wird. Die Zeit wird in Sekunden gemessen und hat einen Standardwert von 0,16s.

WICHTIG! Derzeit können Sie für ein Spiel mit 60fps nur maximal eine Sekunde einstellen, da sonst keine Rotationen erkannt werden. Bei einer niedrigeren Bildrate erhöht sich dieser Wert, z. B. kann ein Spiel mit 30 Bildern pro Sekunde eine maximale Zeit von 2 Sekunden haben.

Syntax:

gesture_rotate_time();

Rückgabe:

(inches)

Beispiel:

if gesture_get_rotate_time() != 0.1
{
    gesture_rotate_time(0.1);
}

Die obige code prüft, ob die Rotationszeit für Gesten auf 0,1 Sekunden eingestellt ist, und setzt sie andernfalls auf diesen Wert.
