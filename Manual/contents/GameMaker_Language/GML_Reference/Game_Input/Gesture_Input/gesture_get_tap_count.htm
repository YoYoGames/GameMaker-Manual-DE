Geste_get_tap_count

Mit dieser Funktion wird geprüft, ob die Tippzählung aktiviert oder deaktiviert ist. Die Funktion gibt true zurück, wenn sie aktiviert ist, und andernfalls false. Sie können die Tippzählung mit der Funktion gesture_tap_count(). Wenn sie aktiviert ist, hat jedes Tap-Ereignis einen zusätzlichen DS-Map-Eintrag "tapcount", der die Anzahl der Tap-Ereignisse registriert, und wenn sie aktiviert ist, bedeutet dies, dass alle Tap-Ereignisse ausgelöst werden, d.h. zwei Tap-Ereignisse lösen sowohl das Einzel-Tap-Ereignis als auch das Doppel-Tap-Ereignis aus, wobei der Tap-Count des Einzel-Tap-Ereignisses 1 und der Tap-Count des Doppel-Tap-Ereignisses 2 ist. Nach Ablauf der Zeit, die für die Erkennung eines Doppelklopfens eingestellt wurde, wird der Wert auf 0 zurückgesetzt (siehe die Funktion gesture_double_tap_time()). Wenn die Zapfwellenzählung deaktiviert ist, wird die erste Zapfung erst registriert, wenn die Zeit für die Doppelzapfung abgelaufen ist und keine zweite Zapfung erkannt wurde. Beachten Sie, dass diese Funktion standardmäßig aktiviert ist.

Syntax:

gesture_get_tap_count();

Rückgabe:

Beispiel:

if !gesture_get_tap_count()
{
    getsure_tap_count(true);
}
