gesture_get_double_tap_distance

Diese Funktion wird verwendet, um den Abstand zu ermitteln, innerhalb dessen Sie den Bildschirm nach einem einfachen Tippen erneut berühren/anklicken müssen, um eine Doppeltippgeste auszulösen. Der Abstand wird in Zoll gemessen und hat einen Standardwert von 0,1.

Syntax:

gesture_get_double_tap_distance();

Rückgabe:

(inches)

Beispiel:

if gesture_get_double_tap_distance() != 0.1
{
    gesture_double_tap_distance(0.1);
}

Die obige code prüft, ob der Doppel-Tipp-Abstand für Gesten auf 0,1 Zoll eingestellt ist, und setzt ihn andernfalls auf diesen Wert.
