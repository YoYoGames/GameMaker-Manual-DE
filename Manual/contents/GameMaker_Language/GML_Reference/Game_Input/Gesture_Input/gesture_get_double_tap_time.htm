geste_get_double_tap_time

Mit dieser Funktion wird die Zeit ermittelt, die zwischen zwei Berührungen/Klicks vergeht, um ein Double-Tap-Gesture-Ereignis auszulösen. Die Zeit wird in Sekunden gemessen und hat einen Standardwert von 0,10.

Syntax:

gesture_get_double_tap_time();

Rückgabe:

Real (seconds)

Beispiel:

if gesture_get_double_tap_time() != 0.16
{
    gesture_double_tap_time(0.16);
}

Die obige code prüft, ob die Doppeltippzeit für Gesten auf 0,16 Sekunden eingestellt ist, und wenn nicht, wird sie auf diesen Wert gesetzt.
