geste_get_rotate_time

Mit dieser Funktion wird der Winkel ermittelt, den ein Paar von Berührungen überschreiten muss, um eine Drehstartgeste auszulösen. Der Winkel wird in Grad gemessen und hat einen Standardwert von 5°.

Syntax:

gesture_get_rotate_angle();

Rückgabe:

(inches)

Beispiel:

if gesture_get_rotate_angle() != 5
{
    gesture_rotate_angle(5);
}

Die obige code prüft, ob der Drehwinkel für Gesten auf 5° eingestellt ist, und wenn nicht, wird er auf diesen Wert gesetzt.
