gesten_rotieren_winkeln

Mit dieser Funktion wird der Winkel festgelegt, den ein Paar von Berührungen überschreiten muss, um eine Drehstartgeste auszulösen. Der Winkel wird in Grad gemessen und hat einen Standardwert von 5°.

Syntax:

gesture_rotate_angle();

Rückgabe:

(inches)

Beispiel:

if gesture_get_rotate_angle() != 5
{
    gesture_rotate_angle(5);
}

Die obige code prüft, ob der Rotationswinkel für Gesten auf 5° eingestellt ist, und wenn nicht, wird er auf diesen Wert gesetzt.
