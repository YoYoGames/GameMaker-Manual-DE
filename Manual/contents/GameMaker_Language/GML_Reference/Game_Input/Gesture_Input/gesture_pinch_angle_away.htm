Geste_Zwicken_Winkel_weg

Mit dieser Funktion wird der Winkel festgelegt, innerhalb dessen sich eine Berührung von einer anderen Berührung entfernen muss, bevor eine Pinch-Out-Geste ausgelöst werden kann. Der Winkel wird in Grad gemessen und hat einen Standardwert von 45°.

Syntax:

gesture_pinch_angle_away();

Rückgabe:

(inches)

Beispiel:

if gesture_get_pinch_angle_away() != 45
{
    gesture_pinch_angle_away(45);
}

Die obige code prüft, ob der Pinch-Out-Winkel für Gesten auf 45° eingestellt ist, und wenn nicht, wird er auf diesen Wert gesetzt
