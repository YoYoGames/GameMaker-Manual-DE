event_data

Diese Variable ist global und wird verwendet, um eine DS-Map zu speichern, wenn sie im entsprechenden Ereignis (z. B. Gestenereignisse und Broadcast-Nachrichten) verwendet wird, und -1 zu allen anderen Zeiten. Der tatsächliche Inhalt der DS-Map hängt von der Art des Ereignisses ab, das es ausgelöst hat, siehe also die einzelnen Abschnitte für diese Ereignisse.

Syntax:

event_data;

Rückgabe:

DS Map ID

Beispiel:

isFlick = event_data[?"isflick"];

if (isFlick)
{
    flickVelX = event_data[?"diffX"];
    flickVelY = event_data[?"diffY"];
}
else
{
    flickVelX = 0;
    flickVelY = 0;
}

Die obige code wird vom Ereignis " Drag End gesture" übernommen und prüft, ob es sich um ein "flick"-Ereignis handelt. Ist dies der Fall, werden die erforderlichen Daten aus der DS-Map event_data extrahiert und zum Setzen einiger Variablen verwendet. Wenn kein "Flick"-Ereignis erkannt wird, werden die gleichen Variablen auf 0 gesetzt.
