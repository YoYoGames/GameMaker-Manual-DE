mouse_clear

Diese Funktion löscht den aktuellen Zustand der angegebenen Maustaste. Das bedeutet, dass die Überprüfung, ob die Maustaste gedrückt ist, nicht true zurückgibt, bis der Spieler die Taste loslässt und sie erneut drückt (aber der Zustand des Loslassens wird immer noch erkannt, wenn der Löschvorgang durchgeführt wird, während die Maustaste gedrückt ist).

Sie können eine der auf dieser Seite aufgeführten Maustastenkonstanten angeben.

Syntax:

mouse_clear(button);

Argument Typ Beschreibung
button Maustaste Konstante Which mouse button constant to check for.

Rückgabe:

Boolean

Beispiel:

mouse_clear(mb_any);

Die obige code wird den Zustand aller Maustasten löschen.
