maus_rad_abwärts

Diese Funktion gibt true zurück, wenn das Mausrad nach unten gedreht wird, und andernfalls false.

Syntax:

mouse_wheel_down();

Rückgabe:

Beispiel:

if mouse_wheel_down()
{
    y += 10;
}

Dadurch wird die aktuelle Instanz auf dem Bildschirm nach unten verschoben, wenn das Mausrad nach oben gedreht wird.
