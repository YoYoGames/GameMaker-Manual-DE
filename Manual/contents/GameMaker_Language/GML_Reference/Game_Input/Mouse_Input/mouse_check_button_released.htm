mouse_check_button_released

Diese Funktion gibt true zurück, wenn die überprüfte Maustaste losgelassen wurde, oder false, wenn dies nicht der Fall ist. Diese Funktion wird für jede Maustaste nur einmal ausgelöst, wenn sie losgelassen wird. Um sie erneut auszulösen, muss die Taste gedrückt und wieder losgelassen werden. Sie geben die zu prüfende Maustaste über eine der folgenden Konstanten an:

Syntax:

mouse_check_button_released(numb);

Argument Typ Beschreibung
numb Maustaste Konstante Which mouse button constant to check for.

Rückgabe:

Boolean

Beispiel:

if mouse_check_button_released(mb_right)
{
    speed = point_distance(x, y, mouse_x, mouse_y) / 10;
}

Die obige code prüft, ob die rechte Maustaste losgelassen wurde, und wenn ja, wird die Geschwindigkeit der Instanz auf ein Zehntel des Abstands zwischen der aktuellen x/y-Position und der x/y-Position der Maus gesetzt.
