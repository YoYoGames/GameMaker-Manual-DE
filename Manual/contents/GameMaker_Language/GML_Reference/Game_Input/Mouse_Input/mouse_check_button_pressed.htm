mouse_check_button_pressed

Diese Funktion gibt true zurück, wenn die überprüfte Maustaste gedrückt wurde, oder false, wenn dies nicht der Fall ist. Diese Funktion wird für jede Maustaste nur einmal ausgelöst, wenn sie zum ersten Mal gedrückt wird. Um sie erneut auszulösen, muss die Taste losgelassen und erneut gedrückt werden. Beachten Sie, dass sie für die Dauer des Schritts und für alle Instanzen, die Mausereignisse haben oder dieselbe Funktion verwenden, als ausgelöst gilt.

Sie geben die zu prüfende Maustaste aus einer der folgenden Konstanten an:

Syntax:

mouse_check_button_pressed(numb);

Argument Typ Beschreibung
numb Maustaste Konstante Which mouse button constant to check for.

Rückgabe:

Boolean

Beispiel:

if mouse_check_button_pressed(mb_left)
{
    score += 50;
}

Die obige code prüft, ob die linke Maustaste gedrückt wurde, und fügt in diesem Fall 50 zum Ergebnis hinzu.
