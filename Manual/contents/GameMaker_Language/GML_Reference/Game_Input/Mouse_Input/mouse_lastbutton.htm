maus_lastbutton

Diese Variable gibt die zuletzt gedrückte Maustaste zurück und kann jede der speziellen Mauskonstanten außer mb_any zurückgeben (Sie können diese Variable auch auf eine der Konstanten setzen).

Syntax:

mouse_lastbutton;

Rückgabe:

Mouse Button Constant (except mb_any)

Beispiel:

if mouse_lastbutton = mb_left
{
    x -= 1;
}

Dies verschiebt die aktuelle Instanz nach links, wenn die letzte gedrückte Taste die linke Maustaste war.
