Maus_y

Diese schreibgeschützte Variable gibt die aktuelle y-Achsenposition der Maus im Raum zurück.

Syntax:

mouse_y;

Rückgabe:

Beispiel:

y = median(64, mouse_y, room_height - 64);

Die obige code wird die Instanz an der y-Position der Maus halten, solange sie sich innerhalb der Grenzen von 64 Pixeln vom oberen und unteren Rand des Raumes befindet.
