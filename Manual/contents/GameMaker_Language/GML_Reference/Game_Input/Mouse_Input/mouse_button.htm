mouse_button

Diese schreibgeschützte Variable gibt die Maustaste zurück, die gerade gedrückt wird (aktuell, wie in diesem Schritt) und kann jede der speziellen Mauskonstanten außer mb_any zurückgeben.

Syntax:

mouse_button

Rückgabe:

Mouse Button Constant (except mb_any)

Beispiel:

if mouse_button = mb_left
{
    x -= 1;
}

Dies verschiebt die aktuelle Instanz nach links, wenn die linke Maustaste gedrückt ist.
