maus_check_button

Diese Funktion gibt true zurück, wenn die zu prüfende Maustaste gedrückt ist, oder false, wenn sie nicht gedrückt ist. Sie können die zu prüfende Maustaste aus einer der folgenden Konstanten auswählen:

Syntax:

mouse_check_button(numb);

Argument Typ Beschreibung
numb Maustaste Konstante Which mouse button constant to check for.

Rückgabe:

Boolean

Beispiel:

if mouse_check_button(mb_left)
{
    instance_create_layer(mouse_x, mouse_y, "Effects", obj_Star);
}

Das obige code sucht nach der linken Maustaste und erstellt bei jedem Schritt, bei dem sie gedrückt wird, eine Instanz des in "obj_Star" indizierten Objekts.
