tastatur_lastkey

Diese Variable bezieht sich auf den Wert, der keyboard_key im vorherigen Frame war und gibt den Keycode dieser Taste zurück (alle Standard-Tastenkonstanten werden zurückgegeben). Diese Variable ist nicht schreibgeschützt und kann geändert werden, z. B. auf -1 gesetzt werden, wenn Sie sie bereits behandelt haben.

Syntax:

keyboard_lastkey;

Rückgabe:

Virtual Key Constant

Beispiel:

if (keyboard_lastkey != -1)
{
    str += keyboard_lastchar;
    keyboard_lastkey = -1;
}

Die obige code prüft, ob die Variable "lastkey" ungleich -1 ist, und wenn ja, fügt sie der Variablen "str" die letzte Taste als Zeichenkette hinzu und setzt dann die Variable keyboard_lastkey zurück, um weitere Eingaben zu akzeptieren.
