tastatur_lastchar

Diese Variable speichert eine Zeichenfolge der zuletzt gedrückten Taste. Diese Variable ist nicht schreibgeschützt und Sie können sie ändern, z. B. auf "" (eine leere Zeichenkette) setzen, wenn Sie sie bereits behandelt haben.

Syntax:

keyboard_lastchar;

Rückgabe:

Beispiel:

if keyboard_lastkey != -1
{
    str += keyboard_lastchar;
    keyboard_lastkey = -1;
}

Die obige code prüft, ob die Variable "lastkey" größer als -1 ist, und wenn ja, fügt sie der Variablen "str" den letzten Schlüssel als String hinzu.
