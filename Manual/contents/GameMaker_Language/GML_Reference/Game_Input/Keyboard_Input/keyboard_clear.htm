keyboard_clear

Mit dieser Funktion können Sie den aktuellen Tastaturstatus löschen, was im Wesentlichen bedeutet, dass eine gedrückte Taste nicht mehr erkannt wird, bis sie wieder losgelassen wird (was auch in diesem Fall nicht das Ereignis Keyboard Key Released auslöst) und erneut gedrückt wird. Die Funktion nimmt einen Keycode-Wert an, wie er von der Funktion ord() zurückgegeben wird (nur Großbuchstaben von A-Z oder Zahlen von 0-9), oder eine der vk_* Konstanten, die die auf der Hauptseite Tastatureingabe aufgeführt sind.

Syntax:

keyboard_clear(key);

ArgumentTyp Beschreibung
key The key to clear.

Rückgabe:

Beispiel:

keyboard_clear(vk_space);

Die obige code löscht den Status der Leertaste.
