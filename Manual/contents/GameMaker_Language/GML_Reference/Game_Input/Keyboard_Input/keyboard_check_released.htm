keyboard_check_released

Mit dieser Funktion können Sie prüfen, ob eine Taste freigegeben wurde oder nicht. Anders als die keyboard_check() Funktion wird diese Funktion nur einmal ausgeführt, wenn die Taste losgelassen wird. Damit sie erneut ausgelöst wird, muss die Taste also zuerst gedrückt und dann wieder losgelassen werden. Die Funktion nimmt einen Keycode-Wert an, wie er von der Funktion ord() zurückgegeben wird (nur Großbuchstaben von A-Z oder Zahlen von 0-9), oder eine der vk_* Konstanten, die auf der Hauptseite Tastatureingabe aufgeführt sind.

Syntax:

keyboard_check_released(key);

ArgumentTyp Beschreibung
key The key to check the released state of.

Rückgabe:

Beispiel:

if keyboard_check_released(ord("P"))
{
    instance_create_layer(0, 0, "Controllers", obj_Pause);
}

Die obige code prüft, ob die Taste "P" losgelassen wurde, und wenn ja, erstellt sie eine Instanz von "obj_Pause".
