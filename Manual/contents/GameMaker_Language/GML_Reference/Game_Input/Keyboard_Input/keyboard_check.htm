keyboard_check

Mit dieser Funktion können Sie prüfen, ob eine Taste gedrückt gehalten wird oder nicht. Anders als bei der keyboard_check_pressed() oder keyboard_check_released() Funktionen, die nur einmal ausgelöst werden, wenn die Taste gedrückt oder losgelassen wird, wird diese Funktion bei jedem Schritt ausgelöst, bei dem die Taste gedrückt gehalten wird. Die Funktion nimmt einen Keycode-Wert an, wie er von der Funktion ord() zurückgegeben wird (nur Großbuchstaben von A-Z oder Zahlen von 0-9), oder eine der vk_* Konstanten, die auf der Hauptseite Tastatureingabe aufgeführt sind.

Syntax:

keyboard_check(key);

Argument Typ Beschreibung
key Virtuelle Schlüsselkonstante (vk_*) The key to check the down state of.

Rückgabe:

Boolean

Beispiel:

if keyboard_check(vk_left)
{
    x -= 5;
}

Die obige code prüft, ob die Pfeiltaste gedrückt wird und verschiebt die Instanz 5 Pixel nach links bei jedem Schritt, der wahr zurückgibt.
