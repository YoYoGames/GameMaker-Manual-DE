keyboard_check_pressed

Mit dieser Funktion können Sie prüfen, ob eine Taste gedrückt wurde oder nicht. Anders als bei der keyboard_check() Funktion wird diese Funktion nur einmal für jede gedrückte Taste ausgeführt. Damit sie erneut ausgelöst wird, muss die Taste erst losgelassen und dann erneut gedrückt werden. Die Funktion nimmt einen Keycode-Wert an, wie er von der Funktion ord() zurückgegeben wird (nur Großbuchstaben von A-Z oder Zahlen von 0-9), oder eine der vk_* Konstanten, die auf der Hauptseite Tastatureingabe aufgeführt sind.

Syntax:

keyboard_check_pressed(key);

Argument Typ Beschreibung
key Virtuelle Schlüsselkonstante (vk_*) The key to check the pressed state of.

Rückgabe:

Boolean

Beispiel:

if keyboard_check_pressed(vk_anykey)
{
    room_goto_next();
}

Die obige code wird zum nächsten Raum weitergehen, wenn der Spieler eine beliebige Taste auf der Tastatur drückt (funktioniert wie eine Aufforderung "Drücken Sie eine beliebige Taste, um fortzufahren").
