keyboard_check_direct

Diese Funktion gibt true zurück, wenn die Taste mit dem bestimmten Keycode gedrückt wird, oder false, wenn dies nicht der Fall ist, indem sie die Hardware direkt überprüft. Sie erlaubt einige weitere Überprüfungen, insbesondere können Sie die Keycodes vk_lshift, vk_lcontrol, vk_lalt, vk_rshift, vk_rcontrol und vk_ralt verwenden, um zu prüfen, ob die linke oder rechte Shift-, Control- oder Alt-Taste gedrückt ist. Die Funktion nimmt einen Keycode-Wert, wie er von der Funktion ord() zurückgegeben wird (nur Großbuchstaben von A-Z oder Zahlen von 0-9), oder eine der vk_* Konstanten, die auf der Hauptseite Tastatureingabe aufgeführt sind.

HINWEIS Diese Funktion ist nur für das Standardziel Windows verfügbar und das Ergebnis ist unabhängig davon, welche Anwendung den Fokus hat.

Syntax:

keyboard_check_direct(key);

ArgumentTyp Beschreibung
key The key to check the down state of.

Rückgabe:

Beispiel:

if keyboard_check_direct(vk_ralt) || keyboard_check_direct(vk_lalt)
{
    crouch = true;
}

Die obige code prüft, ob die linke oder rechte Alt-Taste gedrückt wurde, und wenn ja, setzt sie die Variable "crouch" auf true.
