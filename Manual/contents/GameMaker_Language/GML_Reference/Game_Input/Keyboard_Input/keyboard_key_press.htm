keyboard_key_press

Mit dieser Funktion können Sie das Drücken einer beliebigen Taste auf der Tastatur simulieren. Wenn Sie diese Funktion verwenden, wird die Taste als gedrückt gekennzeichnet, bis die entsprechende Freigabefunktion aufgerufen wird (siehe keyboard_key_release()). Die Funktion nimmt einen Keycode-Wert an, wie er von der Funktion ord() zurückgegeben wird (nur Großbuchstaben von A-Z oder Zahlen von 0-9), oder eine der vk_* Konstanten, die auf der Hauptseite Tastatureingabe aufgeführt sind.

Beachten Sie, dass nach dem Aufruf dieser Funktion, wenn die Taste physisch auf der Tastatur gedrückt wird, dieses Druckereignis erst wieder registriert wird, wenn die Taste physisch losgelassen wird (wodurch das Freigabeereignis ausgelöst und diese Funktion beendet wird), oder die entsprechende Funktion "Taste loslassen" aufgerufen und die Taste erneut gedrückt wurde.

Syntax:

keyboard_key_press(key);

ArgumentTyp Beschreibung
key The key to simulate a press of.

Rückgabe:

Beispiel:

keyboard_key_press(vk_space);

Dadurch wird das Drücken der Leertaste simuliert.
