keyboard_get_map

Manchmal möchten Sie vielleicht den ascii code für eine zugeordnete Taste erhalten (um z.B. zu sehen, ob sie bereits zugeordnet ist), und genau das gibt diese Funktion zurück. Die Funktion nimmt einen Keycode-Wert, wie er von der Funktion ord() zurückgegeben wird (nur Großbuchstaben von A-Z oder Zahlen von 0-9), oder eine der vk_* Konstanten, die auf der Hauptseite Tastatureingabe aufgeführt sind.

Syntax:

keyboard_get_map(key);

ArgumentTyp Beschreibung
key This is the key that you wish to get the mapped value from

Rückgabe:

Beispiel:

if keyboard_get_map(ord("A")) = ord("A")
{
    keyboard_set_map(ord("A"), keyboard_lastkey);
}

Das obige Beispiel code prüft zunächst, ob "A" einer anderen Taste zugeordnet wurde, und wenn nicht, wird es der letzten Taste zugeordnet, die der Benutzer gedrückt hat.
