keyboard_get_numlock

Sie können diese Funktion verwenden, um den Status der Tastatursperre zu ermitteln, wobei true zurückgegeben wird, wenn sie eingeschaltet ist, und false, wenn sie ausgeschaltet ist.

HINWEIS Diese Funktion ist nur in den Windows exe-Builds verfügbar und funktioniert nicht auf anderen Geräten.

Syntax:

keyboard_get_numlock();

Rückgabe:

Beispiel:

if keyboard_get_numlock()
{
    keyboard_set_numlock(false);
}
else
{
    keyboard_set_numlock(true);
}

Das obige Beispiel code ermittelt den Status der Taste numberlock und setzt sie, wenn sie eingeschaltet ist (true), auf aus (false) und umgekehrt.
