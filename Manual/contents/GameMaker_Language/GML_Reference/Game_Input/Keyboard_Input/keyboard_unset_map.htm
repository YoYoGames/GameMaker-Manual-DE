keyboard_unset_map

Mit dieser Funktion können Sie alle neu zugewiesenen Tasten löschen, so dass sie in ihren Standardzustand zurückkehren, d. h. alle Tasten werden auf sich selbst zugewiesen.

Syntax:

keyboard_get_map();

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(vk_escape)
{
    keyboard_unset_map();
}

Das obige Beispiel code setzt alle zugewiesenen Tasten auf ihre Standardeinstellungen zurück, wenn der Benutzer die "Escape"-Taste drückt.
