keyboard_set_numlock

Mit dieser Funktion können Sie switch die Tastensperre ein- oder ausschalten ( true für ein und false für aus).

HINWEIS Diese Funktion ist nur in den Windows target builds verfügbar und funktioniert nicht auf anderen Geräten.

Syntax:

keyboard_set_numlock(value);

Argument Typ Beschreibung
value Boolesche Set this to true for "on" and false for "off"

Rückgabe:

N/A

Beispiel:

if keyboard_get_numlock()
{
    keyboard_set_numlock(false);
}
else
{
    keyboard_set_numlock(true);
}

Das obige Beispiel code ermittelt den Status der Taste numberlock und setzt sie, wenn sie eingeschaltet ist (true), auf aus (false) und umgekehrt.
