Tastatur-Zeichenfolge

Diese Variable enthält eine Zeichenkette mit den letzten (maximal) 1024 Zeichen, die auf der Tastatur eingegeben wurden. Diese Zeichenkette enthält nur druckbare Zeichen, aber sie reagiert korrekt auf das Drücken der Rücktaste, indem sie das letzte Zeichen löscht. Diese Variable ist nicht schreibgeschützt und Sie können sie ändern, z.B. auf "" (eine leere Zeichenkette) setzen, wenn Sie sie bereits behandelt haben, und Sie können die String-Funktionen verwenden, um sie zu manipulieren. Beachten Sie, dass bei Verwendung der virtuellen Bildschirmtastatur nur diese Variable mit der Tastatureingabe aktualisiert wird.

Syntax:

keyboard_string;

Rückgabe:

Beispiel:

if string_length(keyboard_string) > 15
{
    keyboard_string = string_copy(keyboard_string, 1, 15);
}

Die obige code begrenzt die Länge der Tastaturzeichenkette auf 15 Zeichen und entfernt die Zeichen, die darüber hinausgehen, indem die ersten fünfzehn Zeichen in die Variable zurückkopiert werden.
