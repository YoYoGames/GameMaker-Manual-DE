virtual_key_hide

Sobald Sie Ihre virtuelle Taste für Ihre Geräte erstellt haben, müssen Sie möglicherweise debuggen und testen, ob sie korrekt positioniert ist und ordnungsgemäß funktioniert, wofür Sie die Funktion virtual_key_show(). Sie können die virtuelle Taste mit dieser Funktion auch ausblenden, indem Sie den Index der virtuellen Taste verwenden, die Sie ausblenden möchten (zuvor erstellt und gespeichert mit virtual_key_add()). Wenn Sie damit zufrieden sind, empfiehlt es sich, eine eigene Grafik zu erstellen und in ihrem Bereich zu platzieren.

Syntax

virtual_key_hide(index);

Argument Typ Beschreibung
index Virtuelle Schlüssel-ID The index of the virtual key to hide.

Rückgabe:

N/A

Beispiel:

if global.Debug
{
    virtual_key_show(global.Left);
}
else
{
    virtual_key_hide(global.Left);
}

Die obige code prüft die globale Variable "global.Debug" und wenn sie wahr ist, dann wird die virtuelle Taste, die in der Variable "global.Left" indiziert ist, auf dem Bildschirm gezeichnet, und wenn sie falsch ist, dann wird die Taste versteckt.
