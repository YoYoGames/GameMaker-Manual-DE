tastatur_virtuell_status

Diese Funktion kann verwendet werden, um den Status der virtuellen Tastatur auf dem Gerät, auf dem das Spiel läuft, zu ermitteln. Die Funktion gibt true zurück, wenn die virtuelle Tastatur des Betriebssystems sichtbar ist/angezeigt wird, oder false, wenn sie verborgen/versteckt ist.

Syntax

keyboard_virtual_status();

Rückgabe:

Beispiel:

var _status = keyboard_virtual_status();
if (_status == false)
{
    keyboard_virtual_show(kbv_type_numbers, kbv_returnkey_default, kbv_autocapitalize_none, false);
}

Auf der obigen Seite code wird die virtuelle Tastatur des Betriebssystems angezeigt, wenn der aktuelle Status false ist.
