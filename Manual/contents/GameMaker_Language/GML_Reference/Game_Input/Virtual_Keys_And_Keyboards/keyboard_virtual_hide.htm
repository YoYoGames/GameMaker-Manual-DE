keyboard_virtual_hide

Diese Funktion kann verwendet werden, um die virtuelle Tastatur auf dem Gerät auszublenden, auf dem das Spiel läuft. Der Aufruf dieser Funktion erzeugt ein asynchrones Systemereignis, bei dem die async_load DS-Map mit den folgenden Schlüssel/Wert-Paaren gefüllt wird:

  • "event_type" - dies ist der "Status der virtuellen Tastatur", wenn er durch Funktionen der virtuellen Tastatur ausgelöst wird.
  • "screen_height" - die Höhe der virtuellen Tastatur (in Pixel). Dieser Wert ist 0, wenn die Tastatur unsichtbar ist.
  • "keyboard_status" - der aktuelle Status der Tastatur, zurückgegeben als eine der folgenden Zeichenketten:
    • "versteckt"
    • "versteckt"
    • "zeigend"
    • "sichtbar"

Syntax

keyboard_virtual_hide();

Rückgabe:

Beispiel:

if input == true
{
    input = false;
    keyboard_virtual_hide();
}

Die obige code wird die virtuelle Tastatur des Betriebssystems ausblenden, wenn die angegebene Variable nicht auf false gesetzt ist.
