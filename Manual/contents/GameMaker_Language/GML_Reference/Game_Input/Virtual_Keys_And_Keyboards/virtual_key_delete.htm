virtual_key_delete

Wenn Ihr Spiel verschiedene Räume oder Instanzen zur Steuerung von Menüs, Spielabläufen und anderen Dingen hat, ist es wahrscheinlich, dass Sie irgendwann die Position und die Tastenbelegung Ihrer virtuellen Tasten ändern müssen. Dazu können Sie diese Funktion verwenden, um die alten Tasten zu löschen, bevor Sie die neuen erstellen (falls erforderlich). Die Funktion erfordert, dass Sie die ID der zu löschenden virtuellen Taste angeben - wie sie von virtual_key_add().

Beachten Sie, dass alle virtuellen Schlüssel automatisch aus dem Raum entfernt werden, wenn der Raum gewechselt wird. Diese Funktion ist also nur dann notwendig, wenn Sie die Schlüssel manuell entfernen möchten, bevor der Raum selbst beendet ist.

Syntax

virtual_key_delete(index);

Argument Typ Beschreibung
index Virtuelle Schlüssel-ID The index of the virtual key to delete.

Rückgabe:

N/A

Beispiel:

virtual_key_delete(global.Left);

Die obige code löscht den virtuellen Schlüssel, der in der globalen Variable "global.Left" indiziert ist.
