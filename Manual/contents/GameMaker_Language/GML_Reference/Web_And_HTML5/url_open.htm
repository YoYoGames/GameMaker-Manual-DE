url_open

Dadurch wird die angegebene URL im Browser des gewählten Zielgeräts geöffnet, oder, wenn Sie HTML5 module verwenden, im aktuell geöffneten Browser.

Syntax:

url_open(url);

Argument Typ Beschreibung
url Zeichenfolge The URL (website address) to link to.

Rückgabe:

N/A

Beispiel:

url_open("http://gamemaker.io");

Dies würde die GameMaker-Homepage im aktuellen Fenster öffnen.
