browser_height

Diese Variable enthält die Höhe (in Pixel) des Browsers, in dem das Spiel ausgeführt wird. Wenn kein Browser vorhanden ist, wird die Fenstergröße zurückgegeben. Das folgende Bild veranschaulicht, wie diese Variable mit dem Rest der Positionierungsfunktionen zusammenhängt, die über die Fensterfunktionen verfügbar sind.

Syntax:

browser_height

Rückgabe:

Beispiel:

if (browser_height > window_get_height()) || (browser_width > window_get_width())
{
    var xx, yy;
    if (browser_width > window_get_width())
    {
        xx = (browser_width - window_get_width()) / 2;
    }
    else
    {
        xx = 0;
    }
    if (browser_height > window_get_height())
    {
        yy = (browser_height - window_get_height()) / 2;
    }
    else
    {
        yy = 0;
    }
    window_set_position(xx, yy);
}

Das obige code prüft die Breite und Höhe des Browsers, um festzustellen, ob diese größer sind als die aktuelle Fensterbreite und -höhe. Wenn dies der Fall ist, wird eine neue Position für das Spielfenster innerhalb des Browsers berechnet.
