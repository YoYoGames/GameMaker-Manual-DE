analyse_ereignis

Diese Funktion sendet den angegebenen Text an den Analyseanbieter, den Sie über die HTML5-Spieloptionen eingerichtet haben. Diese Funktion kann verwendet werden, um eine benutzerdefinierte Analyse zu erstellen, um etwas außerhalb des Anwendungsbereichs des verwendeten Anbieters zu verfolgen.

Syntax:

analytics_event(string);

ArgumentTyp Beschreibung
string A string to send to the provider

Rückgabe:

Beispiel:

if hs_new
{
    analytics_event("New hiscore of " + string (score));
}

Die obige code prüft eine Variable, um zu sehen, ob sie true ist, und wenn ja, wird ein spezielles Analyseereignis mit der angegebenen Zeichenfolge ausgelöst.
