anklickbar_ändern_ext

Mit dieser Funktion können Sie die sprite und die Position eines klickbaren Symbols ändern, das zuvor mit clickable_add(). Beachten Sie, dass die Position auf der Fenster- und nicht auf der Canvas-Position (0,0) basiert und dass sprite direkt von der Textur-Seite referenziert werden muss (siehe: sprite_get_tpe()). Mit dieser Funktion können Sie auch die Skalierung von sprite (als Multiplikator, so dass 1 der Standardwert ist, 0,5 die Hälfte und 2 das Doppelte) und den Alphawert von 0 (völlig transparent) bis 1 (völlig undurchsichtig) für das endgültige Symbol auf dem Bildschirm ändern.

Syntax:

clickable_change_ext(index, tpe, x, y, alpha, scale)

Argument Typ Beschreibung
index Anklickbare ID The index of the clickable icon to change.
tpe Textur Seiteneintrag The texture page entry for the sprite to be used
x Real The new x position within the window.
y Real The new y position within the window.
scale Real The scale of the icon (default 1).
alpha Real The image alpha of the icon (default 1).

Rückgabe:

N/A

Beispiel:

switch (room)
{
    case rm_Menu: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32, 2, 1); break;
    case rm_Game: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32, 1, 0.5); break;
}

Die obige code ändert den Bildindex und die Position des anklickbaren Symbols, das in der Variablen "global.Help" indiziert ist, je nach aktuellem Raum und ändert auch die Skalierung und den Alphawert des Symbols.
