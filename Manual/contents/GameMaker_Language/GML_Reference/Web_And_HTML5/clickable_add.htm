anklickbar_hinzufügen

Mit dieser Funktion können Sie ein benutzerdefiniertes, anklickbares Symbol erstellen, das als "Schaltfläche" außerhalb des Spiel-Canvas, aber innerhalb des Browserfensters selbst fungiert. Die Funktion gibt einen ID-Wert zurück, der in einer Variablen gespeichert werden muss, um in allen weiteren Schaltflächenfunktionen verwendet zu werden, die auf die neue klickbare Schaltfläche verweisen. Diese Funktion ist sehr nützlich, um zu verhindern, dass der Browser ein Popup-Fenster erzeugt, wenn man auf die Schaltfläche klickt, da dies alles von GameMaker selbst erledigt wird.

Sie müssen zuerst die Position festlegen, die relativ zur (0,0) des Browserfensters (linke obere Ecke) ist, und dann den Eintrag der Texturseite für die sprite angeben, die das Symbol der Schaltfläche haben soll (siehe: sprite_get_tpe()). Sie können dann entweder eine URL angeben, die geöffnet werden soll, oder eine script (aus dem Asset Browser von GameMaker), die geöffnet oder ausgeführt werden soll. Der "target"-Parameter, den Sie angeben, ist derselbe wie der Standard-JavaScript-"name"-Wert, wenn Sie die open() -Methode verwenden (beachten Sie, dass alle außer "_self" dazu führen können, dass der Browser den Aufruf blockiert oder den Benutzer fragt, ob er ihn zulassen möchte), und der "params"-Parameter argument ist derselbe wie der JavaScript-"specs"-Parameter zur Steuerung der Eigenschaften, die das neue Fenster/der neue Tab anzeigen soll (nicht alle Browser unterstützen möglicherweise alle Funktionen). Wenn Sie das Symbol verwenden, um ein internes script auszuführen, können die letzten beiden arguments als leere Strings "" belassen werden.

Gültige Ziele sind:

Ziel Beschreibung
_blank Öffnet das verknüpfte Dokument in einem neuen Fenster oder einer neuen Registerkarte.
_self Öffnet das verknüpfte Dokument in demselben Rahmen, in dem es angeklickt wurde (Standardeinstellung).
_Eltern Öffnet das verknüpfte Dokument im übergeordneten Rahmen.
_top Öffnet das verknüpfte Dokument im Volltext des Fensters.

Gültige Parameter sind:

Parameter Beschreibung
height=[px]' Die Höhe des Fensters, wobei der Mindestwert 100 beträgt.
width=[px]' Die Breite des Fensters, wobei der Mindestwert 100 beträgt.
left=[px]' Die linke Position des Fensters.
'top=[px]' Die obere Position des Fensters (nur IE).
location=[boolean]' Ob das Adressfeld angezeigt werden soll oder nicht (Standardwert ist 1).
menubar=[boolean]'. Ob die Menüleiste angezeigt werden soll oder nicht (Standard ist 1).
resizable=[boolean]' Ob das Fenster in der Größe veränderbar ist oder nicht (Standard ist 1).
scrollbars=[boolean]'. Ob Bildlaufleisten angezeigt werden sollen oder nicht (Standard ist 1).
status=[boolesch]' Ob eine Statusleiste hinzugefügt werden soll oder nicht (Standard ist 1).
titlebar=[boolean]' Ob die Titelleiste angezeigt werden soll oder nicht. Dies wird ignoriert, es sei denn, die aufrufende Anwendung ist eine HTML-Anwendung oder ein vertrauenswürdiges Dialogfeld (Standardwert ist 1);
toolbar=[boolean]' Ob die Browser-Symbolleiste angezeigt werden soll oder nicht (Standardeinstellung ist ja).


Es ist auch erwähnenswert, dass, wenn die URL argument, die Sie übergeben, der Name einer Script-Funktion ist (d.h. sie beginnt nicht mit "http://") UND die script heißt gmcallback_*, dann können Sie direkte Feeds in ein html5-Ereignis erhalten. GameMaker-Script-Funktionsnamen werden nicht verschleiert, wenn sie mit gmcallback_ beginnen.

Syntax:

clickable_add(x, y, tpe, url, target, params);

Argument Typ Beschreibung
x Real The x position within the window.
y Real The y position within the window.
tpe Textur Seiteneintrag The texture page entry for the sprite to be used.
url Zeichenfolge The URL (website address) to link to, or the script to be run.
target Zeichenfolge This is the target area to open the URL in (see description).
params Zeichenfolge The various parameters for the icon (see description).

Rückgabe:

Clickable ID

Beispiel:

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

Die obige code erstellt ein anklickbares DOM-Symbol an der Position (32, 32) der Seite, auf der das Game Canvas ausgeführt wird. Das Symbol verwendet die sprite, die von der Texturseite als "spr_MS_Home" referenziert wird, und wenn das Symbol angeklickt wird, öffnet sich ein neues Fenster für die angegebene URL und mit den definierten Eigenschaften für dieses Fenster.
