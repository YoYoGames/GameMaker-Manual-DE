anklickbar_löschen

Diese Funktion muss verwendet werden, um ein klickbares Symbol, das zuvor mit clickable_add() aus dem Spielfenster zu entfernen.

Syntax:

clickable_delete(index);

Argument Typ Beschreibung
index Anklickbare ID Index of the clickable icon to remove.

Rückgabe:

N/A

Beispiel:

if clickable_exists(global.Help_Icon)
{
    clickable_delete(global.Help_Icon);
}

Die obige code entfernt das anklickbare Symbol, das in der globalen Variable "Help_Icon" indiziert ist, aus dem Spielfenster, falls es existiert.
