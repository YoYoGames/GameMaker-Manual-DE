url_open_full

Dadurch wird die angegebene URL im Browser des gewählten Zielgeräts geöffnet, oder, wenn Sie HTML5 module verwenden, im aktuell geöffneten Browser. Der von Ihnen angegebene Parameter "target" entspricht dem JavaScript-Standardwert "name", wenn Sie die Methode open() verwenden (beachten Sie, dass alle Werte außer "_self" dazu führen können, dass der Browser den Aufruf blockiert oder den Benutzer fragt, ob er ihn zulassen möchte), und der Parameter "options" entspricht dem JavaScript-Parameter "specs" zur Steuerung der Eigenschaften, die das neue Fenster/der neue Tab anzeigen soll (nicht alle Browser unterstützen alle Funktionen).

Gültige Ziele sind:

Ziel Beschreibung
_blank Öffnet das verknüpfte Dokument in einem neuen Fenster oder einer neuen Registerkarte (dies funktioniert nicht, wenn Pop-ups vom Benutzer blockiert werden; in diesem Fall können Sie stattdessen die clickable_* Funktionen verwenden).
_self Öffnet das verknüpfte Dokument in demselben Rahmen, in dem es angeklickt wurde (Standardeinstellung).
_Eltern Öffnet das verknüpfte Dokument im übergeordneten Rahmen.
_top Öffnet das verknüpfte Dokument im Volltext des Fensters.

Gültige Optionen sind:

Parameter Beschreibung
height=[px]' Die Höhe des Fensters, wobei der Mindestwert 100 beträgt.
width=[px]' Die Breite des Fensters, wobei der Mindestwert 100 beträgt.
left=[px]' Die linke Position des Fensters.
'top=[px]' Die obere Position des Fensters (nur IE).
location=[boolean]' Ob das Adressfeld angezeigt werden soll oder nicht (Standardwert ist 1).
menubar=[boolean]'. Ob die Menüleiste angezeigt werden soll oder nicht (Standard ist 1).
resizable=[boolean]' Ob das Fenster in der Größe veränderbar ist oder nicht (Standard ist 1).
scrollbars=[boolean]'. Ob Bildlaufleisten angezeigt werden sollen oder nicht (Standard ist 1).
status=[boolesch]' Ob eine Statusleiste hinzugefügt werden soll oder nicht (Standard ist 1).
titlebar=[boolean]' Ob die Titelleiste angezeigt werden soll oder nicht. Dies wird ignoriert, es sei denn, die aufrufende Anwendung ist eine HTML-Anwendung oder ein vertrauenswürdiges Dialogfeld (Standardwert ist 1);
toolbar=[boolean]' Ob die Browser-Symbolleiste angezeigt werden soll oder nicht (Standardeinstellung ist ja).

Syntax:

url_open_full(url, target, options);

Argument Typ Beschreibung
url Zeichenfolge The URL (website address) to link to.
target Zeichenfolge This is the target area to open the URL in (see description).
options Zeichenfolge Standard browser options (see description).

Rückgabe:

N/A

Beispiel:

url_open_full("http://gamemaker.io", "_blank", "resizable=0, height=200, scrollbars=0");

Dies würde die GameMaker-Homepage in einem neuen Fenster öffnen, das nicht in der Größe verändert werden kann, eine Höhe von 200 Pixeln hat und keine Scrollbalken aufweist.
