browser_input_capture

Mit dieser Funktion können Sie festlegen, ob das Browserfenster alle Eingaben erfassen soll (setzen Sie sie auf false) oder ob das Spiel die Eingaben erfassen soll (setzen Sie sie auf true). Beachten Sie, dass diese Funktion nur für die Verwendung mit HTML5 module geeignet ist.

Syntax:

browser_input_capture(enable);

Argument Typ Beschreibung
enable Boolesche Enable or disable game input capture

Rückgabe:

N/A

Beispiel:

browser_input_capture(true);

Die obige code stellt das Spiel so ein, dass es alle Browser-Eingaben auf dem HTML5-Ziel platform erfasst.
