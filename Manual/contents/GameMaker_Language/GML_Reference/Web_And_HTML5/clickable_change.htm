clickable_change

Mit dieser Funktion können Sie die sprite und die Position eines klickbaren Symbols ändern, das zuvor mit clickable_add(). Bitte beachten Sie, dass die Position auf der Fenster- und nicht auf der Canvas-Position (0,0) basiert und dass die sprite direkt von der Textur-Seite referenziert werden muss (siehe: sprite_get_tpe()).

Syntax:

clickable_change(index, tpe, x, y)

Argument Typ Beschreibung
index Anklickbare ID The index of the clickable icon to change.
tpe Textur Seiteneintrag The texture page entry for the sprite to be used.
x Real The new x position within the window.
y Real The new y position within the window.

Rückgabe:

N/A

Beispiel:

switch (room)
{
    case rm_Menu: clickable_change(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32); break;
    case rm_Game: clickable_change(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32); break;
}

Die obige code ändert den Bildindex und die Position des anklickbaren Symbols, das in der Variablen "global.Help" indiziert ist, abhängig vom aktuellen Raum.
