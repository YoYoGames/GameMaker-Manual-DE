anklickbar_hinzufügen_ext

Mit dieser Funktion können Sie ein benutzerdefiniertes, anklickbares Symbol erstellen, das als "Schaltfläche" außerhalb des Spiel-Canvas, aber innerhalb des Browserfensters selbst fungiert. Die Funktion gibt einen ID-Wert zurück, der in einer Variablen gespeichert werden muss, um in allen weiteren Schaltflächenfunktionen verwendet zu werden, die auf die neue klickbare Schaltfläche verweisen. Diese Funktion ist sehr nützlich, um zu verhindern, dass der Browser ein Popup-Fenster erzeugt, wenn man auf die Schaltfläche klickt, da dies alles von GameMaker selbst erledigt wird.

Diese Funktion funktioniert genauso wie die clickable_add() Funktion, bei der Sie die Position relativ zur (0,0) des Browser-Fensters festlegen, den Textur-Seiteneintrag für die sprite angeben, die das Button-Icon haben soll (siehe: sprite_get_tpe()), entweder eine zu öffnende URL oder eine script (aus dem Asset-Browser von GameMaker) angeben, die geöffnet oder ausgeführt werden soll, und dann das Ziel und die zu verwendenden Parameter festlegen. Diese Funktion verfügt jedoch auch über zwei zusätzliche Optionen, mit denen Sie den Alphawert des Symbols von 0 (völlig transparent) bis 1 (völlig undurchsichtig) und die Skalierung des Symbols einstellen können, bei der es sich um einen Multiplikatorwert handelt, der auf die verwendete sprite angewendet wird. Ein Wert von 1 ist also der Standardmaßstab, während (zum Beispiel) 0,5 der halbe Maßstab und 2 das Doppelte wäre.

Es ist auch erwähnenswert, dass, wenn die URL argument, die Sie übergeben, der Name einer Script-Funktion ist (d.h. sie beginnt nicht mit "http://") UND die script heißt gmcallback_*, dann können Sie direkte Feeds in ein html5-Ereignis erhalten. GameMaker-Script-Funktionsnamen werden nicht verschleiert, wenn sie mit gmcallback_ beginnen.

Syntax:

clickable_add_ext(x, y, tpe, url, target, params, alpha, scale);

Argument Typ Beschreibung
x Real The x position within the window.
y Real The y position within the window.
tpe Textur Seiteneintrag The texture page entry for the sprite to be used.
url Zeichenfolge The URL (website address) to link to, or the script to be run.
target Zeichenfolge This is the target area to open the URL in (see description).
params Zeichenfolge The various parameters for the icon (see description).
alpha Real The image alpha of the icon (default 1).
scale Real The scale of the icon (default 1).

Rückgabe:

Clickable ID

Beispiel:

home_but = clickable_add_ext(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0", 1, 2);

Die obige code erstellt ein anklickbares DOM-Symbol an der Position (32, 32) der Seite, auf der das Game Canvas läuft. Das Symbol verwendet die sprite, die von der Texturseite als "spr_MS_Home" referenziert wird, und wenn es angeklickt wird, öffnet sich ein neues Fenster für die angegebene URL und mit den definierten Eigenschaften für dieses Fenster. Die Schaltfläche wird auch auf die doppelte Originalgröße von sprite und ein halbes Bild-Alpha von 1 skaliert.
