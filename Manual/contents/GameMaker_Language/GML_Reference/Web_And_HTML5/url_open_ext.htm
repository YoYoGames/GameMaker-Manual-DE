url_open_ext

Dadurch wird die angegebene URL im Browser des gewählten Zielgeräts geöffnet, oder, wenn Sie HTML5 module verwenden, im aktuell geöffneten Browser. Der Parameter "target", den Sie angeben, ist derselbe wie der Standard-JavaScript-Wert "name", wenn Sie die Methode open() verwenden, und Sie sollten sich darüber im Klaren sein, dass alle außer "_self" dazu führen können, dass der Browser blockiert oder den Benutzer fragt, ob er dies zulassen möchte.

Gültige Ziele sind:

Ziel Beschreibung
_blank Öffnet das verknüpfte Dokument in einem neuen Fenster oder einer neuen Registerkarte (dies funktioniert nicht, wenn Pop-ups vom Benutzer blockiert werden; in diesem Fall können Sie stattdessen die clickable_* Funktionen verwenden).
_self Öffnet das verknüpfte Dokument in demselben Rahmen, in dem es angeklickt wurde (Standardeinstellung).
_Eltern Öffnet das verknüpfte Dokument im übergeordneten Rahmen.
_top Öffnet das verknüpfte Dokument im Volltext des Fensters.

Syntax:

url_open_ext(url, target);

Argument Typ Beschreibung
url Zeichenfolge The URL (website address) to link to.
target Zeichenfolge This is the target area to open the URL in.

Rückgabe:

N/A

Beispiel:

url_open_ext("http://gamemaker.io", "_blank");

Dadurch wird die GameMaker-Homepage in einem neuen Fenster geöffnet.
