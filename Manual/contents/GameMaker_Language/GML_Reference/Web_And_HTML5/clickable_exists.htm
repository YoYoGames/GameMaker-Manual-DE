clickable_exists

Diese Funktion gibt zurück, ob ein klickbares DOM-Icon mit dem angegebenen Index existiert oder nicht. Bitte beachten Sie, dass der Wert, der für die Überprüfung verwendet wird, zuvor initialisiert worden sein muss, sonst erhalten Sie einen Fehler, der den GameMaker schließt.

Syntax:

clickable_exists(index);

Argument Typ Beschreibung
index Anklickbare ID The index of the clickable to check.

Rückgabe:

Boolean

Beispiel:

if !clickable_exists(home_but)
{
    home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");
}

Die obige code prüft die Variable "home_but", um festzustellen, ob sie bereits existiert, und wenn nicht, wird ein anklickbares DOM-Symbol an der Position (32, 32) der Seite erstellt, auf der die Spieloberfläche läuft. Das Icon verwendet die sprite, die von der Textur-Seite als "spr_MS_Home" referenziert wird, und wenn das Icon angeklickt wird, öffnet sich ein neues Fenster für die angegebene URL und mit den definierten Eigenschaften für dieses Fenster.
