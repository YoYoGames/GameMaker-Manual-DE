clickable_set_style

Mit dieser Funktion können Sie die CSS-Stileigenschaften für die angegebene Schaltfläche über die Schlüssel/Wertpaare in der bereitgestellten DS Map. Sie müssen zuvor sowohl das Button-Element (mit clickable_add()) und die DS Map erstellt haben und die gespeicherten Indizes für beide als arguments angeben.

Syntax:

clickable_set_style(index, map)

Argument Typ Beschreibung
index Anklickbare ID The index of the clickable icon to style.
map DS-Karten-ID The index of the DS Map um den Stil festzulegen.

Rückgabe:

N/A

Beispiel:

var map = ds_map_create();
ds_map_add(map, "opacity", "0.5");
clickable_set_style(button, map);
ds_map_destroy();

Die obige code wird den Stil der anklickbaren Schaltfläche mit dem in der Variablen "button" gespeicherten Index ändern.
