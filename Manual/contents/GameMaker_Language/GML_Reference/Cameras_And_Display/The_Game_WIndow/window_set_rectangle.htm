fenster_setzen_rechteck

Mit dieser Funktion können Sie die Position des Spielfensters innerhalb des Browsers (HTML5) oder der Anzeige (Windows, Ubuntu (Linux) oder macOS) und auch die Skalierung des Fensters festlegen. Weitere Informationen zu Fensterposition und Fenstergröße finden Sie unter window_set_position() und window_set_size().

Syntax:

window_set_rectangle(x, y, w, h);

ArgumentTyp Beschreibung
x The new x coordinate of the window.
y The new y coordinate of the window.
w The new width of the window.
h The new height of the window.

Rückgabe:

Beispiel:

window_set_rectangle(0, 0, display_get_width(), display_get_height());

Mit der obigen code wird das Spielfenster so eingestellt, dass es den gesamten Anzeigebereich einnimmt (entweder den Browser oder den Bildschirm, je nach Ziel platform).
