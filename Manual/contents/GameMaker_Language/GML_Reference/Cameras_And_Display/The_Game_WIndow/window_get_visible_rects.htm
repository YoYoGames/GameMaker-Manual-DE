window_get_visible_rects

Mit dieser Funktion können Sie den überlappenden Bereich des durch (x1,y1) bis (x2,y2) definierten Rechtecks auf jedem der angeschlossenen Displays finden. Die Funktion gibt ein Array mit 8 Werten pro Display zurück (d.h. wenn Sie zwei Displays haben, hat das Array eine Länge von 16 Indizes), wobei die Werte [0 ... 3] den Werten overlapx1, overlapy1, overlapx2, overlapy2 entsprechen - sie definieren den Bereich der Überlappung auf diesem Display und werden auf 0,0,0,0 gesetzt, wenn keine Überlappung vorliegt - und die Werte [4 ... 7] entsprechen den Werten monitorx1, monitory1, monitorx2, monitory2 - den Koordinaten der Anzeige im virtuellen Anzeigebereich. Dies kann verwendet werden, um zu testen, ob eine gespeicherte Fensterposition sichtbar sein wird oder nicht (der Benutzer könnte einen externen Monitor abgeschaltet oder das Fenster aus dem Bildschirm verschoben, so dass die gespeicherte Fensterposition nicht mehr gültig ist), zum Beispiel.

Syntax:

window_get_visible_rects(x1, y1, x2, y2);

ArgumentTyp Beschreibung
x1 The left edge of the rectangle to check
y1 The top edge of the rectangle to check.
x2 The right edge of the rectangle to check
y2 The bottom edge of the rectangle to check.

Rückgabe:

Beispiel:

var wx = window_get_x();
var wy = window_get_y();
var ww = window_get_width();
var wh = window_get_height();
display_data = window_get_visible_rects(wx, wy, wx + ww, wy + wh);
display_num = array_length(display_data) / 8;

Die obige code erzeugt ein 1D-Array in der Variablen display_data, das die Informationen über die Anzeigen enthält, und erstellt die Variable display_num, die die Anzahl der gefundenen aktiven Anzeigen enthält.
