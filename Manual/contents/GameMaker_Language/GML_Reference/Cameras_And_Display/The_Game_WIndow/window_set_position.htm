fenster_setzen_position

Mit dieser Funktion können Sie das Spielfenster auf eine bestimmte Position innerhalb des Bildschirms (auf macOS, Linux(Ubuntu) und Windows) oder innerhalb des Browsers (HTML5) setzen.

HINWEIS: Wenn Ihr HTML5-Spiel ein benutzerdefiniertes index.html verwendet, das die Leinwand auf eine feste Position setzt, hat diese Funktion keine Auswirkungen auf die Fensterposition.

Syntax:

window_set_position(x, y);

ArgumentTyp Beschreibung
x The x coordinate of where to position the window.
y The y coordinate of where to position the window.

Rückgabe:

Beispiel:

window_set_position(0, 0);

Die obige code positioniert das Spielfenster in der oberen linken Ecke des Browsers oder des Bildschirms (je nach verwendetem Ziel module ).
