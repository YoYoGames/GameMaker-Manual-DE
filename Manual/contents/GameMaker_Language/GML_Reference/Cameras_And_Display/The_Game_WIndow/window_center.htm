fenster_mitte

Mit dieser Funktion können Sie das Spielfenster auf dem Bildschirm zentrieren, wenn das Ziel module Windows , Ubuntu (Linux) oder macOS ist, oder Sie können es im Browser zentrieren, wenn das Ziel module HTML5 ist. Diese Funktion hat keine Auswirkungen auf andere Geräte.

HINWEIS Wenn Sie die Größe des Spielfensters im aktuellen Schritt geändert haben (z. B. durch Umschalten von Vollbild auf Fenster oder durch Verwendung von window_set_size()) und das neue Fenster zentrieren möchten, sollte diese Funktion mindestens einen Schritt später aufgerufen werden (z. B. durch Verwendung eines alarm), da sie sonst nicht korrekt funktioniert.

Syntax:

window_center();

Rückgabe:

Beispiel:

if !window_get_fullscreen()
{
    window_center();
}

Die obige code zentriert das Spielfenster auf dem Bildschirm, wenn es sich nicht im Vollbildmodus befindet.
