display_set_timing_method

Mit dieser Funktion können Sie die Timing-Methode festlegen, die für das Rendern Ihres Spiels verwendet werden soll. Die Methode kann eine der folgenden Konstanten sein:

Konstante Beschreibung
tm_sleep Verwenden Sie den Wert der Einschlafspanne als Hauptmethode für die Zeitmessung
tm_countvsyncs Vsync-Timing als Haupt-Timing-Methode verwenden (Standard für alle unterstützten platforms)

Die vsync-Timing-Methode verwendet die Unterstützung des Ziels platforms für die vertikale Synchronisation, um einen Anker für die Rendering-Timing-Berechnungen des Spiels zu liefern, während die Einstellung des Modus auf "Sleep Margin" nur versucht, sicherzustellen, dass jedes Bild die richtige Zeitspanne (z. B. 1/30 oder 1/60 Sekunde) dauert, indem es wartet oder schläft. Im Allgemeinen liefert das Standard-Vsync-Timing die besten Ergebnisse, aber beachten Sie, dass bei der Verwendung der Vsync-Methode die Werte für die Schlafspanne immer noch relevant sind, auch wenn sie eine geringere Auswirkung haben, und wir empfehlen, den Standardwert beizubehalten, wenn Sie diese Methode verwenden.

Standardmäßig verwendet GameMaker auf allen platforms außer PS4, Ubuntu und HTML5 die vsync-Timing-Methode, während auf den nicht unterstützten platformsnur Sleep-Margin-Timing verwendet werden kann.

Wenn Sie den Schlafabstand einstellen möchten, können Sie dies mit der Funktion display_set_sleep_margin() und Sie können herausfinden, welche Zeitmessungsmethode gerade verwendet wird, indem Sie die Funktion display_get_timing_method().

Syntax:

display_set_timing_method(method);

Argument Typ Beschreibung
method Anzeige Timing-Methode Konstant The timing method to use (see the list of constants, above)

Rückgabe:

N/A

Beispiel:

if display_get_timing_method() != tm_sleep
{
    display_set_timing_method(tm_sleep);
    if (display_get_sleep_margin() != 20)
    {
        display_set_sleep_margin(20);
    }
}

Die obige code prüft die Timing-Methode, und wenn sie nicht auf tm_sleep eingestellt ist, wird sie auf diese eingestellt und die Schlafspanne auf 20 gesetzt.
