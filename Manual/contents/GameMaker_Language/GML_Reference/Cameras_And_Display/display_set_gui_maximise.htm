display_set_gui_maximise

Mit dieser Funktion kann die GUI-Ebene maximiert und so eingestellt werden, dass sie in Bezug auf die Bildschirmabmessungen skaliert und verschoben wird, anstatt die Standardposition und -abmessungen der Anwendungsoberfläche zu verwenden. Standardmäßig ist die GUI-Ebene 1:1 mit der Auflösung der Anwendungsoberfläche und wird ebenfalls an der Position (0,0) der Oberfläche gezeichnet. Dies ist jedoch nicht immer das, was Sie wollen, und deshalb können Sie diese Funktion verwenden, um sie relativ zur absoluten (0,0)-Position des Anzeige- oder Spielfensters zu zeichnen. Welche arguments Sie dieser Funktion übergeben, hängt von dem Effekt ab, den Sie auf die GUI-Ebene haben möchten. Wenn Sie die Funktion ohne arguments aufrufen, wird die GUI-Ebene an der Position (0,0) des Bildschirms oder des Spielfensters gezeichnet, wobei die Breite und Höhe so skaliert wird, dass sie in den gesamten Bereich passt:

display_set_gui_maximise();

Sie können jedoch den Skalierungsfaktor für die GUI-Ebene festlegen, und die Breite und Höhe werden um diesen Wert skaliert. Denken Sie daran, dass die GUI-Ebene immer an die Größe des Bildschirms, des Spielfensters oder der Anwendungsoberfläche angepasst wird. Wenn Sie also einen anderen Wert als 1 einstellen, wird die Anzahl der Pixel in Breite und Höhe skaliert. Wenn Ihr Bildschirm eine Größe von 1024x768 hat und Sie den Skalierungsfaktor auf 0,5 setzen, dann wird Ihre GUI-Ebene die Hälfte der Größe des Bildschirms haben, was effektiv eine Verdoppelung der Pixelgröße bedeutet:

display_set_gui_maximise(0.5, 0.5);

Wenn Sie die Werte auf diese Weise einstellen, wird auch die Zeichenposition auf die (0,0) des Anzeige- oder Spielfensters gesetzt, so dass Sie Offset-Werte angeben können, um die (0,0)-Position zu "verschieben":

var pos = application_get_position();
display_set_gui_maximise(0.5, 0.5, pos[0], pos[1]);

Schließlich können Sie mit dieser Funktion die GUI-Ebene zurücksetzen, indem Sie die Skalierungsfaktoren auf -1 setzen. Dadurch wird die GUI-Ebene wieder auf einen Maßstab von 1:1 gesetzt und die Zeichenposition auf die (0,0)-Position der Anwendungsoberfläche und nicht auf die Anzeige oder das Fenster gesetzt.

display_set_gui_maximise(-1, -1);

Syntax:

display_set_gui_maximise(xscale, yscale, xoffset, yoffset);

Argument Typ Beschreibung
xscale Real OPTIONAL Der horizontale Skalierungsfaktor (verwenden Sie -1, um auf den Standardwert zurückzusetzen).
yscale Real OPTIONAL Der vertikale Skalierungsfaktor (verwenden Sie -1, um auf den Standardwert zurückzusetzen).
xoffset Real OPTIONAL Die x-Versatzposition für das Zeichnen.
yoffset Real OPTIONAL Die y-Versatzposition für das Zeichnen.

Rückgabe:

N/A

Beispiel:

display_set_gui_size(display_get_width() / 2, display_get_height() / 2);
display_set_gui_maximise(2, 2, 0, 0);

Die obige code sperrt das GUI-Ereignis auf die angegebene Breite und Höhe und skaliert alle Komponenten so, dass sie in das Display passen.
