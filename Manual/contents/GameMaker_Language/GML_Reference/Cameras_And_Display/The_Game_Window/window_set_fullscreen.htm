window_set_fullscreen

Mit dieser Funktion können Sie das Spielfenster auf Vollbild (true) oder nicht (false) einstellen. Bitte beachten Sie, dass Sie für das macOS-Ziel die Option "Im Vollbildmodus starten" deaktivieren und die Option "Dem Spieler erlauben, die Größe des Spielfensters zu ändern" in den Spieloptionen aktivieren müssen, sonst schlägt diese Funktion fehl. Beachten Sie auch, dass diese Funktion bei HTML5 nicht funktioniert, es sei denn, sie wird als "klickbarer" Callback hinzugefügt (siehe hier für weitere Details).

Syntax:

window_set_fullscreen(full);

ArgumentTyp Beschreibung
full Whether to set the screen to fullscreen (true) or not (false).

Rückgabe:

Beispiel:

if mouse_check_button_pressed(mb_left)
{
    if window_get_fullscreen()
    {
        window_set_fullscreen(false);
    }
    else
    {
        window_set_fullscreen(true);
    }
}

Das obige code prüft, ob eine Maustaste gedrückt wird, und setzt dann das Fenster auf Vollbild, wenn dies noch nicht der Fall ist, oder setzt es auf Fenster, wenn dies bereits der Fall ist.
