window_get_fullscreen

Diese Funktion gibt zurück, ob sich das Spielfenster im Vollbildmodus befindet (true) oder nicht (false).

Syntax:

window_get_fullscreen();

Rückgabe:

Beispiel:

if window_get_fullscreen()
{
    draw_text(32, 32, "Fullscreen is ON");
}
else
{
    draw_text(32, 32, "Fullscreen is OFF");
}

Die obige code prüft, ob das Fenster im Vollbildmodus ist oder nicht, und zeichnet je nach dem zurückgegebenen Wert eine Textnachricht.
