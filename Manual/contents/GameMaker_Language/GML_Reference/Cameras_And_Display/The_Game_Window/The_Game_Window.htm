Das Spielfenster

Das von Ihnen erstellte Spiel findet in einem Fenster statt (auch im Vollbildmodus), und dieses Fenster hat eine Reihe von Eigenschaften, wie Position, Größe, Vollbildmodus usw. Diese Details werden normalerweise automatisch für Sie eingestellt, basierend auf der Raumgröße und den aktivierten Ansichtsports in Kombination mit den Einstellungen in den Spieloptionen für das Ziel platform, aber Sie können sie während des Spiels mit den auf dieser Seite aufgeführten Funktionen ändern.

HINWEIS Diese Funktionen sind nur für Windows, Ubuntu (Linux), macOS und HTML5 Ziel modules und funktionieren möglicherweise nicht auf anderen Geräten.

Die folgende Abbildung veranschaulicht, wie einige allgemeine Fensterfunktionen zusammenhängen und miteinander interagieren:

Beachten Sie, dass das obige Bild das Spielfenster in einem Browser zeigt, aber für die anderen Ziele können Sie den Browser durch das Display ersetzen, so dass eine Funktion wie window_get_y() die Position des oberen Teils des Spielfensters relativ zum Display zurückgibt.
