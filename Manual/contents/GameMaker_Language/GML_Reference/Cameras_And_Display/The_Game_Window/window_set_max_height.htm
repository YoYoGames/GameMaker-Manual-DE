window_set_max_height

Mit dieser Funktion können Sie eine maximale Fensterhöhe für Ihr Spiel festlegen. Wenn Sie die Option zur Größenänderung des Fensters in den Spieloptionen für das Zielspiel platform aktivieren, kann der Spieler die Größe des Spielfensters beliebig ändern; mit dieser Funktion können Sie die maximale Höhe auf die von Ihnen angegebene Größe begrenzen. Wenn Sie zum Standardverhalten zurückkehren möchten (d.h. kein Minimum), dann verwenden Sie den Wert -1.

HINWEIS: Diese Funktion ist nur unter Windows, macOS und Linux platforms verfügbar.

Syntax:

window_set_max_height(height);

ArgumentTyp Beschreibung
height The maximum height in pixels for the game window.

Rückgabe:

Beispiel:

window_set_min_width(640);
window_set_min_height(480);
window_set_max_width(1280);
window_set_max_height(960);

Die obige code legt die minimale und maximale Breite und Höhe für das Spielfenster fest.
