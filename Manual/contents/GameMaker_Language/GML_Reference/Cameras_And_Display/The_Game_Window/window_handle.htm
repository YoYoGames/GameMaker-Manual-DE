window_handle

Mit dieser Funktion können Sie den internen Windows id-Wert (den HWND, einen Zeiger) erhalten. Diese Funktion ist wirklich nur für Autoren von Erweiterungen nützlich, die die window handle benötigen, um Windows APIs in der DLL code aufzurufen (der zurückgegebene Zeiger sollte in einen String umgewandelt werden und dann in C++ einfach in ein HWND umgewandelt werden). Die folgende Tabelle zeigt die unterstützten platforms und was sie zurückgeben:

Plattform Rückgabe
Windows Fenster HWND
macOS NSWindow-Klasse
Ubuntu (Linux) XWindow-Handle
HTML5 Segeltuch-ID

Syntax:

window_handle();

Rückgabe:

Pointer

Beispiel:

win_id = windows_handle();

Die obige code speichert die Spielfenster-ID in einer Variablen.
