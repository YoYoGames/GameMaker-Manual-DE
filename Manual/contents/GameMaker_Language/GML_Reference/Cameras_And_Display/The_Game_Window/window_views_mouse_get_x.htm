window_views_mouse_get_x

Diese Funktion liefert die x-Koordinate der Maus in Bezug auf alle aktiven Ansichten und gibt denselben Wert zurück mouse_x.

HINWEIS: Für reguläre Mausfunktionen siehe den Abschnitt über Mauseingabe.

Syntax:

window_views_mouse_get_x();

Rückgabe:

Beispiel:

myx = window_views_mouse_get_x();

Dies würde myx auf die x-Koordinate der Maus im Fenster relativ zu allen aktiven Ansichten setzen.
