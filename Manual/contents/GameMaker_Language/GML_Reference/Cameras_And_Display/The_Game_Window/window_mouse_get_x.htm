window_mouse_get_x

Mit dieser Funktion können Sie die x-Position des Mauszeigers (in Pixel) innerhalb des Browsers ermitteln, wenn es sich um ein HTML5-Spiel handelt, oder innerhalb der Anzeige, wenn es sich um ein Windows, Ubuntu (Linux) oder macOS Spiel handelt.

HINWEIS: Für die regulären Mausfunktionen siehe den Abschnitt über die Mauseingabe

Syntax:

window_mouse_get_x();

Rückgabe:

Beispiel:

wx = window_mouse_get_x();

Die obige code speichert die aktuelle x-Achsen-Fensterposition der Maus in der Variablen "wx".
