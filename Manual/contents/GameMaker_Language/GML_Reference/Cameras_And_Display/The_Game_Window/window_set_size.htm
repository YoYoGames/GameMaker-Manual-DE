window_set_size

Mit dieser Funktion können Sie die Größe des Spielfensters ändern. Auf dem Desktop platforms bezieht sich dies in der Regel auf den Client-Bereich des Fensters, ohne Titelleiste und Rahmen:

Wenn Sie diese Funktion verwenden, um die Größe des Spielfensters zu ändern, wird der Inhalt des Fensters so skaliert, dass er im Verhältnis 1:1 zur Hintergrundleinwand passt, wodurch das Bild scharf und klar bleibt (obwohl es skaliert wird, wenn das Fenster nicht der Raum- oder Ansichtsgröße entspricht). Wenn Sie z. B. ein Fenster mit 200 x 150 Pixeln haben und der Spieler das Fenster durch Ziehen der Ränder vergrößert, kann die endgültige Fenstergröße 400 x 300 Pixel betragen, aber die Zeichenfläche ist immer noch 200 x 150, was zu einem unscharfen Bild führt, da das Pixelverhältnis jetzt 1:2 ist. Wenn Sie jedoch diese Funktion verwenden, um die Fenstergröße zu ändern, wird das Verhältnis 1:1 beibehalten, auch wenn das Bild an die neue Fenstergröße angepasst wird.

Syntax:

window_set_size( w, h );

ArgumentTyp Beschreibung
w The new width of the window.
h The new height of the window.

Rückgabe:

N/A

Beispiel:

if mouse_check_button_pressed(mb_left)
{
    global.scale += 1;
    window_set_size(200 * global.scale, 150 * global.scale);
}

Die obige code wird die Größe des Fensters basierend auf dem Wert der Variable "global.scale" ändern, wenn die Maustaste gedrückt wird.
