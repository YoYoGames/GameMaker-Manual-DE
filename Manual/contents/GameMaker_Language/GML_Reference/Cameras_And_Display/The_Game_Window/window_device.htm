window_device DEPRECATED

Diese Funktion gibt den aktuellen d3d-Gerätezeiger zurück, den Sie dann (zum Beispiel) an eine DLL oder Dylib auf Windows und macOS weitergeben können.

HINWEIS Diese Funktion ist in GameMaker veraltet zugunsten von os_get_info().

Syntax:

window_device();

Rückgabe:

Pointer

Beispiel:

gfx_pointer = window_device();

Die obige code speichert den d3d-Gerätezeiger in einer Variablen.
