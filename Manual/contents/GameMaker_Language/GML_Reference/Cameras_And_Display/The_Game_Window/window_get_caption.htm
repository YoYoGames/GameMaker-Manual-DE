window_get_caption

Diese Funktion gibt die Beschriftung des Fensters zurück (das ist der Text, der oben im Fenster neben dem Symbol erscheint) und zeigt standardmäßig die Beschriftung des Raums an, in dem Sie sich gerade befinden.

Syntax:

window_get_caption();

Rückgabe:

Beispiel:

if window_get_caption() != ""
{
    window_set_caption("");
}

Die obige code prüft, ob die Fenstertitel einen Text enthalten oder nicht, und wenn ja, setzt sie ihn auf einen leeren String, so dass kein Titel angezeigt wird.
