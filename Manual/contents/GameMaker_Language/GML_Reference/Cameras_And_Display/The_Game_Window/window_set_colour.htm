fenster_setzen_farbe

Mit dieser Funktion kann die Hintergrundfarbe des Spielfensters eingestellt werden. Diese Farbe ist diejenige, die für die Bereiche des Spielfensters verwendet wird, die nicht von Ansichten belegt sind. Das folgende Bild veranschaulicht dies:

Das obige Bild hat zwei Ansichten mit zwei Ports, die jeweils an unterschiedlichen Positionen gezeichnet sind. Dadurch wird das Spielfenster so gestreckt, dass beide Ports Platz finden, und die Fensterfarbe wird verwendet, um den Hintergrund zu färben, wenn keine Ansicht angezeigt wird.

Syntax:

window_set_colour(colour);

ArgumentTyp Beschreibung
colour The colour to set the region.

Rückgabe:

Beispiel:

if window_get_colour() != c_black
{
    window_set_colour(c_black);
}

Die obige code prüft, ob die Farbe des Fensters schwarz ist oder nicht, und wenn nicht, wird sie auf schwarz gesetzt.
