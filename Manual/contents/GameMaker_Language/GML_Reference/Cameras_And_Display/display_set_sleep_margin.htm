display_set_sleep_margin

Diese Funktion kann verwendet werden, um den Wert für den Ruhezustand festzulegen, der für das Rendering-Timing Ihres Spiels verwendet wird, und erfordert einen Millisekundenwert. Die Standardwerte sind wie folgt:

Plattform Millisekunden
Windows 10
macOS 10
Ubuntu 10
HTML5 10
Android 4
iOS 4
Windows UWP 10
Xbox 10
PS4 10
Nintendo Switch 10

Beachten Sie, dass selbst wenn die Timing-Methode so eingestellt ist, dass sie das vsync-Timing verwendet, der Sleep-Margin eine gewisse Auswirkung auf das Rendering des Spiels hat, weshalb Sie bei der Einstellung dieses Wertes vorsichtig sein sollten. Für weitere Informationen zum Anzeige-Timing siehe display_set_timing_method().

HINWEIS: Zusätzlich zur Sleep Margin können Sie die Leistung und den Stromverbrauch auf Windows weiter kontrollieren, indem Sie die Auflösung des Thread Schedulers unter runtime anpassen.

Syntax:

display_set_sleep_margin(milliseconds);

Argument Typ Beschreibung
milliseconds Real The number of milliseconds for the sleep margin

Rückgabe:

N/A

Beispiel:

if display_get_timing_method() != tm_sleep
{
    display_set_timing_method(tm_sleep);
    if display_get_sleep_margin() != 20
    {
        display_set_sleep_margin(20);
    }
}

Die obige code prüft die Timing-Methode, und wenn sie nicht auf tm_sleep eingestellt ist, wird sie auf diese eingestellt und die Schlafspanne auf 20 gesetzt.
