gif_save

Mit dieser Funktion können Sie eine GIF-Animation abspeichern. Sie geben den GIF-Index (wie er von der Funktion gif_open()zurückgegeben wird) sowie einen Dateinamen, unter dem die Animation gespeichert werden soll. Beachten Sie, dass GameMaker die Dateierweiterung .gif nicht automatisch anhängt, so dass Sie diese als Teil des Dateinamens angeben sollten, wenn Sie möchten, dass die gespeicherte Datei als GIF erkannt wird. Das erstellte GIF wird mit der Universal 884 Palette palettiert (siehe hier für weitere Informationen).

Beachten Sie, dass die Funktion 0 zurückgibt, wenn sie erfolgreich ist und das GIF korrekt gespeichert wurde, andernfalls wird -1 zurückgegeben.

Syntax:

gif_save(gif_index, fname);

ArgumentTyp Beschreibung
gif_index The ID of gif to save
fname The filename to use for the gif

Rückgabe:

Beispiel:

if save_gif == true
{
    if (count == 0)
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if (count < 30)
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        gif_save(gif_image, "GameCapture.gif");
        count = 0;
        save_gif = false;
    }
    count++;
}

Mit der obigen code wird eine GIF-Bilddatei mit 30 Einzelbildern aus der Anwendungsoberfläche erstellt und anschließend gespeichert.
