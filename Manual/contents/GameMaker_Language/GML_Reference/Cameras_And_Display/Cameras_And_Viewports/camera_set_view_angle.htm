camera_set_view_angle

Mit dieser Funktion können Sie den Winkel der Fachkamera innerhalb des Raums aktualisieren. Sie geben den eindeutigen Kamera-ID-Wert an (wie er von den verschiedenen camera_create() Funktionen zurückgegeben wird) und dann den Winkel, den Sie den Sie die Kameraansicht einstellen möchten. Der Standardwert ist 0°, wobei positive Werte die Kamera gegen den Uhrzeigersinn drehen, d. h., wenn Sie den Wert auf 90 setzen, wird die Kamera um 90° nach links gedreht.

Syntax:

camera_set_view_angle(camera_id, angle)

ArgumentTyp Beschreibung
camera_id The unique camera ID value returned when you created the camera
angle The new angle to set the camera view to

Rückgabe:

Beispiel:

camera_set_view_angle(view_camera[0], point_direction(x, y, mouse_x, mouse_y));

Die obige code stellt den Kamerawinkel für die Kamera ein, die dem View Port[0] zugewiesen ist.
