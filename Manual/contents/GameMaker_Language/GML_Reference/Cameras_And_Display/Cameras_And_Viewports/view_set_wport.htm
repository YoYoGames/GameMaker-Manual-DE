view_set_wport

Diese Funktion kann verwendet werden, um die Breite des gegebenen View Ports einzustellen. Sie geben den Index des Ansichtsfensters (von 0 bis 7) und die neue Breite an, die verwendet werden soll.

Syntax:

view_set_wport(viewport, w)

ArgumentTyp Beschreibung
view_port The view port to target (0 - 7)
w The new width (in pixels)

Rückgabe:

Beispiel:

if view_get_wport(0) != (display_get_width () / 2)
{
    view_set_wport(0, display_get_width() / 2);
}

Die obige code prüft die Breite des View Ports[0], und wenn sie nicht die Hälfte der Anzeigebreite beträgt, wird sie auf diesen Wert gesetzt.
