camera_set_view_target

Mit dieser Funktion können Sie das Verfolgungsziel der Fachkamera innerhalb des Raums festlegen. Sie geben den eindeutigen Wert der Kamera-ID an (wie er von den verschiedenen camera_create() Funktionen zurückgegeben wird) und geben dann die Zielinstanz oder Objekt-ID an, auf die Sie die Kameraansicht einstellen möchten. Wenn du eine Objekt-ID auswählst und es mehr als eine Instanz dieses Objekts im Raum gibt, kann GameMaker nicht wissen, welcher Instanz du folgen willst. Es kann also jede beliebige sein.

Syntax:

camera_set_view_target(camera_id, id)

Argument Typ Beschreibung
camera_id Kamera-ID The unique camera ID value returned when you created the camera
instance_id/object_id Objekt Asset- oder Instanz-ID Instance or object to have the camera target for following

Rückgabe:

N/A

Beispiel:

camera_set_view_target(view_camera[0], id);

Die obige code setzt das Ziel der Ansichtskamera für die Kamera, die dem Ansichtsanschluss[0] zugewiesen ist, auf das der Instanz, auf der code läuft.
