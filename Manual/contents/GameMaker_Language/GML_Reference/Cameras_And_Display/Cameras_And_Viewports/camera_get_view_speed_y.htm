camera_get_view_speed_y

Diese Funktion kann verwendet werden, um die Bewegungsgeschwindigkeit der angegebenen Kamera entlang der y-Achse (vertikale Bewegung) zu ermitteln. Der Rückgabewert wird in Pixeln pro Spielframe angegeben.

Syntax:

camera_get_view_speed_y(camera_id)

ArgumentTyp Beschreibung
camera_id The unique camera ID value returned when you created the camera

Rückgabe:

Beispiel:

var xs = camera_get_view_speed_x(view_camera[0]);
var ys = camera_get_view_speed_y(view_camera[0]);
if xs != 5 || ys != 5
{
    camera_set_view_speed(view_camera[0], 5, 5);
}

Die obige code ruft die xspeed und yspeed der Kamera ab, die dem view port[0] zugewiesen ist, und überprüft dann, ob sie mit dem angegebenen Wert übereinstimmt. Ist dies nicht der Fall, wird die Fachkamera auf diese Geschwindigkeit eingestellt.
