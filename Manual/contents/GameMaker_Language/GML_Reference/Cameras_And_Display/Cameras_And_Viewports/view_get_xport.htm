view_get_xport

Diese Funktion kann verwendet werden, um die X-Position des gegebenen View Ports abzurufen.

Syntax:

view_get_xport(viewport)

ArgumentTyp Beschreibung
view_port The view port to target (0 - 7)

Rückgabe:

Real

Beispiel:

if view_get_xport(0) != (display_get_width() / 2) - (view_wport[0] / 2)
{
    view_set_xport(0, (display_get_width() / 2) - (view_wport[0] / 2));
}

Die obige code prüft die X-Position des View Port[0] und wenn sie nicht an der gewünschten Stelle ist, wird sie auf diese Position gesetzt.
