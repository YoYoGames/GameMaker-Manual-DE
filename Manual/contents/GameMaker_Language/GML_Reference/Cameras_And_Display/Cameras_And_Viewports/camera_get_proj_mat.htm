camera_get_proj_mat

Diese Funktion kann verwendet werden, um die Projektionsmatrix abzurufen. Die Funktion gibt das Matrix-Array zurück, das dann in anderen Matrixfunktionen verwendet werden kann oder um die Projektionsmatrix einer anderen Kamera zu setzen (mit camera_set_proj_mat()).

Syntax:

camera_get_proj_mat(camera_id)

ArgumentTyp Beschreibung
camera_id The unique camera ID value returned when you created the camera

Rückgabe:

Matrix Array

Beispiel:

mat = camera_get_proj_mat(camera_view[0]);

Die obige code liefert die Projektionsmatrix für die Kamera, die dem Viewport[0] zugeordnet ist.
