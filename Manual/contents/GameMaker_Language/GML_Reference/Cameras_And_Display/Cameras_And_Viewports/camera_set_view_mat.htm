camera_set_view_mat

Diese Funktion setzt die Ansichtsmatrix für eine bestimmte Kamera. Sie geben den eindeutigen Wert der Kamera-ID (wie er von den anderen camera_create() Funktionen zurückgegeben wird) und eine anzuwendende Ansichtsmatrix. Weitere Informationen zur Erstellung von Ansichtsmatrizen finden Sie im Abschnitt Matrixfunktionen, insbesondere matrix_build_lookat().

Beachte, dass wenn deine Kamera eine automatische Objektverfolgung durchführt - d.h.: sie wurde mit camera_create_view() mit einem Objektindex / einer Instanz-ID erstellt, die nicht -1 ist, oder du stellst eine im Raumeditor definierte Kamera ein, oder du stellst die Standardkamera ein - dann wird diese Matrix im nächsten Spielframe überschrieben.

Syntax:

camera_set_view_mat(camera_id, matrix)

ArgumentTyp Beschreibung
camera_id The unique camera ID value returned when you created the camera.
matrix The unique matrix ID returned when you created the matrix.

Rückgabe:

Beispiel:

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

Die obige code erstellt eine neue Kamera und weist sie dem View Port[0] zu. Für diese Kamera werden dann die Ansichts- und Projektionsmatrizen festgelegt.
