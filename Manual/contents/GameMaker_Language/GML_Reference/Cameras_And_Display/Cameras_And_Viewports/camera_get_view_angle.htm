camera_get_view_angle

Diese Funktion kann verwendet werden, um den Winkel der angegebenen Kamera abzurufen. Der Rückgabewert wird zwischen 0 und 360 liegen. Beachten Sie, dass diese Funktion nur für Kameras gültig ist, die mit camera_create_view() erstellt wurden, oder für solche, die im Raumeditor hinzugefügt wurden.

Syntax:

camera_get_view_angle(camera_id)

ArgumentTyp Beschreibung
camera_id The unique camera ID value returned when you created the camera

Rückgabe:

Beispiel:

var ang = camera_get_view_angle(view_camera[0]);
if (ang != 0)
{
    camera_set_view_angle(view_camera[0], 0);
}

Die obige code ruft den Winkel der Kamera ab, die dem Ansichtsport[0] zugewiesen ist, und prüft dann, ob er mit dem angegebenen Wert übereinstimmt. Ist dies nicht der Fall, wird die Fachkamera auf diesen Winkel eingestellt.
