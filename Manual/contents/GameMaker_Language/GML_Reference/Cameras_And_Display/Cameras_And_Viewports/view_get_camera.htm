view_get_camera

Diese Funktion kann verwendet werden, um den eindeutigen Kamera-ID-Wert für die Kamera abzurufen, die dem gegebenen Ansichtsanschluss zugewiesen ist (von 0 bis 7). Wenn keine Kamera zugewiesen ist, gibt die Funktion -1 zurück.

Syntax:

view_get_camera(view_port)

ArgumentTyp Beschreibung
view_port The view port to target (0 - 7)

Rückgabe:

Camera ID

Beispiel:

var cam = view_get_camera(0);
var cw = camera_get_view_width(cam);
var ch = camera_get_view_height(cam);
camera_set_view_pos(cam, mouse_x - (cw / 2), mouse_y - (ch / 2));

Die obige code ruft die Kamera-ID für die Kamera ab, die dem Viewport[0] zugewiesen ist, und ändert dann ihre Position.
