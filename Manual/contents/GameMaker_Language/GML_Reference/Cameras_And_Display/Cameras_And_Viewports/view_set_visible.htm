sicht_setzen_sichtbar

Diese Funktion kann verwendet werden, um die Sichtbarkeit des gegebenen View Ports einzustellen. Die Funktion nimmt den Index des View Ports (von 0 bis 7) und einen booleschen Wert true, wenn er sichtbar ist, und false, wenn er nicht sichtbar ist.

Syntax:

view_set_visible(viewport, visible)

ArgumentTyp Beschreibung
view_port The view port to target (0 - 7)
visible Visibility toggle (true ist sichtbar und false ist unsichtbar)

Rückgabe:

N/A

Beispiel:

if !view_get_visible(0)
{
    view_set_visible(0, true);
}

Die obige code prüft, ob der View Port[0] sichtbar ist, und wenn nicht, wird er auf sichtbar gesetzt.
