camera_get_view_border_y

Diese Funktion kann verwendet werden, um den Grenzwert für die Objekt-/Instanzverfolgung der angegebenen Kamera entlang der y-Achse (vertikaler Rand) zu ermitteln. Der Rückgabewert wird in Pixeln angegeben.

Syntax:

camera_get_view_border_y(camera_id)

ArgumentTyp Beschreibung
camera_id The unique camera ID value returned when you created the camera

Rückgabe:

Beispiel:

var xb = camera_get_view_border_x(view_camera[0]);
var yb = camera_get_view_border_y(view_camera[0]);
if xb != 200 || yb != 200
{
    camera_set_view_border(view_camera[0], 200, 200);
}

Die obige code ruft die xborder- und yborder-Werte der Kamera ab, die dem view port[0] zugewiesen ist, und überprüft dann, ob sie mit dem angegebenen Wert übereinstimmen. Wenn dies nicht der Fall ist, wird die Ansichtskamera auf den angegebenen Wert gesetzt.
