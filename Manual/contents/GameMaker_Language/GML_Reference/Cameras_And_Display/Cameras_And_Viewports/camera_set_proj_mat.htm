camera_set_proj_mat

Diese Funktion setzt die Projektionsmatrix für eine bestimmte Kamera. Sie geben den eindeutigen Wert der Kamera-ID (wie er von den verschiedenen camera_create() Funktionen zurückgegeben wird) und eine Projektionsmatrix, die angewendet werden soll. Weitere Informationen zur Erstellung von Projektionsmatrizen finden Sie im Abschnitt Matrixfunktionen, insbesondere unter matrix_build_projection_perspective() und matrix_build_projection_ortho().

Bei der Verwendung dieser Funktion sind einige Dinge zu beachten:

  • Wenn deine Kamera eine automatische Objektverfolgung durchführt - d.h.: sie wurde erstellt mit camera_create_view() mit einem Objektindex / einer Instanz-ID ungleich -1 erstellt wurde, oder du eine im Raumeditor definierte Kamera einstellst, oder du die Standardkamera einstellst - dann wird diese Matrix im nächsten Spielframe überschrieben.
  • Wenn Sie diese Funktion verwenden, um switch in einer perspektivischen Projektion (im Gegensatz zur standardmäßigen orthografischen Projektion) zu zeichnen, wird das automatische Zeichnen von gekachelten sprites nicht mehr funktionieren und muss manuell gehandhabt werden. Dies gilt auch für alle Hintergrundebenen, die horizontale oder vertikale Kacheln verwenden, sowie für die Funktionen draw_sprite_tiled() und draw_sprite_tiled_ext(), die sprite nur einmal zeichnen, anstatt es zu kacheln, wenn eine perspektivische Projektion verwendet wird.

Syntax:

camera_set_proj_mat(camera_id, matrix)

ArgumentTyp Beschreibung
camera_id The unique camera ID value returned when you created the camera.
matrix The unique matrix ID returned when you created the matrix.

Rückgabe:

Beispiel:

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

Die obige code erstellt eine neue Kamera und weist sie dem View Port[0] zu. Für diese Kamera werden dann die Ansichts- und Projektionsmatrizen festgelegt.
