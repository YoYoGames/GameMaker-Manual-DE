camera_set_end_script

Diese Funktion kann verwendet werden, um eine Skriptfunktion festzulegen, die am Ende jedes Spielframes aufgerufen wird, wenn die Kamera einem sichtbaren und aktiven Viewport zugewiesen ist, nachdem alles für diese Viewkamera gerendert wurde. Man gibt den eindeutigen Kamera-ID-Wert (wie er von den verschiedenen camera_create() Funktionen zurückgegeben wird) und die ID der aufzurufenden Funktion. Die Reihenfolge, in der die Funktionen den Kameras zugeordnet werden, und das eigentliche Rendering der Kameraansicht ist wie folgt:

  • Die Aktualisierungsfunktion der Kameras für alle sichtbaren und aktiven Viewports heißt
  • Dann für jede Ansicht:
    • Die Begin-Funktion für die Kamera für diese Ansicht wird aufgerufen
    • Die Zeichenereignisse werden für diese Ansicht ausgeführt
    • Die End-Funktion für die Kamera wird aufgerufen
  • Wechseln Sie zur nächsten aktiven sichtbaren Ansicht und wiederholen Sie den Vorgang...

Syntax:

camera_set_end_script(camera_id, script)

Argument Typ Beschreibung
camera_id Kamera-ID The unique camera ID value returned when you created the camera.
script Skript-Funktion The script to run each game frame

Rückgabe:

N/A

Beispiel:

camera_set_end_script(view_camera[0], endCamera);

Die obige code stellt das Ende script für die Kamera ein, die dem View Port [0] zugewiesen ist.
