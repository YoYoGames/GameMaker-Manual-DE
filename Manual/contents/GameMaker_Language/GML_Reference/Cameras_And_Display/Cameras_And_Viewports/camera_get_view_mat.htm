camera_get_view_mat

Diese Funktion kann verwendet werden, um die Ansichtsmatrix abzurufen. Die Funktion gibt das Matrix-Array zurück, das dann in anderen Matrix-Funktionen verwendet werden kann oder um die Ansichtsmatrix einer anderen Kamera zu setzen (mit camera_set_view_mat()). Beachten Sie, dass diese Funktion in der Regel für Kameras verwendet wird, die mit camera_create()erstellt wurden, aber sie kann auch für Kameras verwendet werden, die im Raumeditor (oder mit camera_create_view()) erstellt wurden - in diesem Fall ist die zurückgegebene Matrix nur dann gültig, wenn die angegebene Kamera zum ersten Mal in einem Viewport verwendet wird.

Syntax:

camera_get_view_mat(camera_id)

ArgumentTyp Beschreibung
camera_id The unique camera ID value returned when you created the camera

Rückgabe:

Matrix Array

Beispiel:

mat = camera_get_view_mat(camera_view[0]);

Die obige code liefert die Ansichtsmatrix für die Kamera, die dem Ansichtsanschluss[0] zugewiesen ist.
