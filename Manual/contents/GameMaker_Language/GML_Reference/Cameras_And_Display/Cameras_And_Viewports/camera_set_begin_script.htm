camera_set_begin_script

Diese Funktion kann verwendet werden, um eine Skriptfunktion festzulegen, die zu Beginn jedes Spielframes aufgerufen wird, wenn die Kamera einem sichtbaren und aktiven Viewport zugewiesen ist. Sie geben den eindeutigen Kamera-ID-Wert (wie von den verschiedenen camera_create() Funktionen zurückgegeben) und die ID der aufzurufenden Funktion an. Die Reihenfolge, in der scripts den Kameras zugewiesen wird, und das tatsächliche Rendering der Kameraansicht ist wie folgt:

  • Die Aktualisierungsfunktion der Kameras für alle sichtbaren und aktiven Viewports heißt
  • Dann für jede Ansicht:
    • Die Begin-Funktion für die Kamera für diese Ansicht wird aufgerufen
    • Die Zeichenereignisse werden für diese Ansicht ausgeführt
    • Die End-Funktion für die Kamera wird aufgerufen
  • Wechseln Sie zur nächsten aktiven sichtbaren Ansicht und wiederholen Sie den Vorgang...

Syntax:

camera_set_begin_script(camera_id, script)

Argument Typ Beschreibung
camera_id Kamera-ID The unique camera ID value returned when you created the camera.
script Skript-Funktion The script function to run each game frame

Rückgabe:

N/A

Beispiel:

camera_set_begin_script(view_camera[0], beginCamera);

Die obige code stellt die Funktion begin script für die Kamera ein, die dem View Port [0] zugeordnet ist.
