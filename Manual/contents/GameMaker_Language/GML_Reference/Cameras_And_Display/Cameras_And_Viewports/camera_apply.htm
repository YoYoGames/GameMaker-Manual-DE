Kamera_anwenden

Diese Funktion wendet die angegebenen Kameraeinstellungen sofort auf die aktuelle Kameraansicht an, die gerade gerendert wird. Während man mit den spezifischen Funktionen von camera_set_* jede beliebige Ansichtskamera einstellen kann, werden die neuen Einstellungen erst im nächsten Spielframe zum Rendern verwendet, aber mit dieser Funktion kann man diese Änderungen sofort vornehmen. Wenn Sie mehrere Ansichten verwenden, sollte die Funktion im Draw Event verwendet werden und Sie können die Variable view_current verwenden, um die angegebene Kamera selektiv nur auf die Kameraansicht anzuwenden, die gerade gerendert wird. Beachten Sie, dass dies NICHT den Wert der Kamera-ID ändert, sondern nur die Einstellungen für diese Kamera, und dass, wenn Sie diese Funktion auf die Standardkamera anwenden (d. h. die interne Kamera, die GameMaker verwendet, wenn keine Ansichtskameras aktiv sind), die Änderungen bereits im nächsten Spielframe durch die internen Einstellungen für das Zeichnen des Raums überschrieben werden.

Syntax:

camera_apply(camera_id)

ArgumentTyp Beschreibung
camera_id The unique camera ID value returned when you created the camera.

Rückgabe:

Beispiel:

if view_current == 0
{
    camera_apply(cutscene_cam);
}

Das obige code (in einem Draw Event) prüft, welche Kameraansicht gerade gerendert wird, und wenn es die Kameraansicht[0] ist, werden die Einstellungen für die Kamera, auf die die Variable "cutscene_cam" verweist, darauf angewendet.
