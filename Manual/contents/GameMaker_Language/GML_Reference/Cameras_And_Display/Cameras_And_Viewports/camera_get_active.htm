camera_get_active

Mit dieser Funktion können Sie den eindeutigen Kamera-ID-Wert der derzeit aktiven Kamera abrufen.

Syntax:

camera_get_active()

Rückgabe:

Camera ID

Beispiel:

var active = camera_get_active();
if active != view_camera[0]
{
    view_camera[0] = _active;
}

Die obige code ruft die Kamera-ID für die aktive Kamera ab und setzt die Fachkamera für port[0], um sie zu verwenden.
