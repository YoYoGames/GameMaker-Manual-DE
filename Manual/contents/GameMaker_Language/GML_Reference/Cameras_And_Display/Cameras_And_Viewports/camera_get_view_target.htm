camera_get_view_target

Diese Funktion kann verwendet werden, um das Verfolgungsziel der angegebenen Kamera abzurufen. Der Rückgabewert kann ein Objektindex, eine Instanz-ID oder -1 sein, wenn kein Verfolgungsziel gesetzt wurde.

Syntax:

camera_get_view_target(camera_id)

ArgumentTyp Beschreibung
camera_id The unique camera ID value returned when you created the camera

Rückgabe:

Object Asset, Instance ID or -1

Beispiel:

var target = camera_get_view_target(view_camera[0]);
if (target != obj_Player)
{
    camera_set_view_target(view_camera[0], obj_Player);
}

Die obige code ruft das Verfolgungsziel der Kamera ab, die dem Ansichtsport[0] zugewiesen ist, und überprüft dann, ob es mit dem Objektindex "obj_Player" übereinstimmt. Wenn dies nicht der Fall ist, wird die Ansichtskamera so eingestellt, dass sie einer Instanz dieses Objekts folgt.
