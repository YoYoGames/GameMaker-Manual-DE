kamera_zerstören

Mit dieser Funktion können Sie jede beliebige Kamera zerstören. Beim Aufruf der Funktion geben Sie den eindeutigen Kamera-ID-Wert an, den Sie aus den camera_create_* Funktionen oder aus dem view_camera Array, wenn Sie den Raum-Editor verwenden, um den Ansichtsanschluss und die Ansicht einzurichten. Du solltest niemals eine Kamera zerstören, die gerade einer sichtbaren Ansicht zugeordnet ist, es sei denn, du ordnest dieser Ansicht im selben Schritt eine neue Kamera zu, und du solltest immer alle Kameras zerstören, die du über code erstellt hast, wenn sie von deinem Spiel nicht mehr benötigt werden, um Speicherlecks zu vermeiden, und du kannst auch die Standardkameras zerstören, wenn du welche im Raumeditor zugewiesen hast, aber du solltest dem Ansichtsanschluss eine neue Kamera zuweisen (oder sie deaktivieren), sonst erhältst du seltsame Ergebnisse.

Syntax:

camera_destroy(camera_id)

ArgumentTyp Beschreibung
camera_id The unique camera ID value returned when you created the camera.

Rückgabe:

N/A

Beispiel:

camera_destroy(view_camera[0]);
view_camera[0] = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);

Die obige code zerstört die Kamera, die derzeit dem Ansichtsanschluss [0] zugewiesen ist, erstellt dann eine neue Kamera und weist ihre ID diesem Anschluss zu.
