camera_get_view_x

Diese Funktion kann verwendet werden, um die x-Position der Ansicht für eine bestimmte Kamera abzurufen. Beachten Sie, dass diese Funktion nur für Kameras gültig ist, die mit camera_create_view() erstellt wurden oder für Kameras, die im Raum-Editor.

Syntax:

camera_get_view_x(camera_id)

ArgumentTyp Beschreibung
camera_id The unique camera ID value returned when you created the camera

Rückgabe:

Beispiel:

var vx = camera_get_view_x(view_camera[0]);
var vy = camera_get_view_y(view_camera[0]);
draw_text(vx + 5, vy + 5, "SCORE" + string(score));

Die obige code ruft die Position der Kamera ab, die dem view port[0] zugewiesen ist, und zeichnet dann den Text relativ zu dieser Position.
