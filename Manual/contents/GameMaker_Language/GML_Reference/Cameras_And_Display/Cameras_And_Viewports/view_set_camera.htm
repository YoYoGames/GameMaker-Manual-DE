Ansicht_Kamera_setzen

Diese Funktion setzt eine Kamera auf einen bestimmten Viewport. Sie geben den zu setzenden Viewport an (von 0 bis 7) und liefern den eindeutigen Kamera-ID-Wert (wie er von den camera_create() Funktionen zurückgegeben wird oder wenn Sie die Funktion view_get_camera()). Wenn Sie als Kamera-ID den Wert -1 angeben, entfernen Sie eine Kamera aus dem Ansichtsfenster und beachten Sie, dass Sie, wenn das Ansichtsfenster aktiviert und sichtbar ist, einige unvorhersehbare Ergebnisse erhalten können.

Syntax:

view_set_camera(viewport, camera_id)

ArgumentTyp Beschreibung
view_port The view port to target (0 - 7)
camera_id The unique camera ID value returned when you created the camera

Rückgabe:

Beispiel:

var cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);
view_set_camera(0, cam);

Die obige code erstellt eine neue Kamera und weist ihr den Anschluss view[0] zu.
