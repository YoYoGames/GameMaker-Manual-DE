camera_get_begin_script

Mit dieser Funktion kann die ID der Skriptfunktion abgerufen werden, die als Beginn script für die angegebene Kamera zugewiesen wurde. Wenn keine script Funktion zugewiesen ist, gibt die Funktion -1 zurück.

Syntax:

camera_get_begin_script(camera_id)

ArgumentTyp Beschreibung
camera_id The unique camera ID value returned when you created the camera

Rückgabe:

Script Function or -1 if no function is assigned

Beispiel:

var scr = camera_get_begin_script(camera_view[0]);
if (scr != cutScene)
{
    camera_set_update_script(camera_view[0], cutScene);
}

Die obige code prüft die Funktion script, die der Kamera, die dem Viewport[0] zugewiesen ist, als begin script zugewiesen ist, und wenn es nicht "cutScene" ist, wird sie auf diese Funktion gesetzt.
