camera_get_default

Mit dieser Funktion kann der eindeutige Kamera-ID-Wert der Standardkamera abgerufen werden (die Kamera, die GameMaker verwendet, wenn in einem Spielraum keine Kameraansichten oder Ports aktiv sind).

Syntax:

camera_get_default()

Rückgabe:

Camera ID

Beispiel:

var def = camera_get_default();
view_camera[0] = def;

Die obige code ruft die Kamera-ID für die Standardkamera ab und stellt die Fachkamera für port[0] so ein, dass sie diese verwendet.
