view_enabled

Diese Variable steuert, ob die View Ports, die innerhalb des Raums sichtbar sind, aktiviert sind oder nicht. Wenn Sie die Ansichtsfenster auf sichtbar eingestellt haben und diese Option deaktivieren, wird der gesamte Raum auf den Bildschirm gezeichnet und auf die Fenstergröße skaliert, anstatt dass die verschiedenen Kameras durch die Ansichtsfenster gezeichnet werden.

Syntax:

view_enabled;

Rückgabe:

(true: enabled, false: disabled)

Beispiel:

if !view_enabled
{
    view_visible[0] = true;
    view_enabled = true;
}

Das obige code prüft, ob Ansichten für den Raum aktiviert sind, und wenn nicht, stellt es sicher, dass view port[0] sichtbar ist und aktiviert dann Ansichten für den Raum.
