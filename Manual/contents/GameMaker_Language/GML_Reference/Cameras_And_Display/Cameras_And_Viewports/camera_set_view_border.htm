camera_set_view_border

Mit dieser Funktion können Sie den Verfolgungsrahmen der Fachkamera innerhalb des Raums festlegen. Sie geben den eindeutigen Kamera-ID-Wert an (wie er von den verschiedenen camera_create Funktionen zurückgegeben wird) und geben dann die x-Randgröße und die y-Randgröße (horizontal und vertikal). Diese Werte werden nur verwendet, wenn der Fachkamera ein Instanzziel zum Verfolgen zugewiesen wurde (entweder im Raumeditor oder mit der Funktion camera_set_view_target()) und beziehen sich wie weit die Instanz vom Rand der Ansicht entfernt sein muss, bevor die Ansicht ihre Position zum Verfolgen aktualisiert.

Syntax:

camera_set_view_border(camera_id, x_border, y_border)

ArgumentTyp Beschreibung
camera_id The unique camera ID value returned when you created the camera.
x_border The border (in pixels) for the view camera along the horizontal axis
y_border The border (in pixels) for the view camera along the vertical axis

Rückgabe:

Beispiel:

camera_set_view_border(view_camera[0], 64, 64);

Mit der obigen code wird der Rahmen der Fachkamera für die Kamera festgelegt, die dem Fachport[0] zugewiesen ist.
