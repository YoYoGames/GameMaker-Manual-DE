anzeigen_sichtbar

Diese Variable kann verwendet werden, um herauszufinden, ob ein bestimmter Ansichtsanschluss derzeit sichtbar ist oder nicht. Sie können diese Variable auch so einstellen, dass sie eine Ansicht effektiv "ein-" oder "ausschaltet", indem Sie den Wert auf true (sichtbar) oder false (unsichtbar) setzen. Beachten Sie, dass, selbst wenn Sie einen Ansichtsanschluss auf sichtbar gesetzt haben, die Ansichtsanschlüsse nicht aktiviert sind (mit Hilfe der integrierten Variable view_enabled oder im Raum-Editor aktiviert), werden sie nicht auf den Bildschirm gezeichnet.

Syntax:

view_visible[0 ... 7];

Rückgabe:

(true: enabled, false: disabled)

Beispiel:

if !view_visible[0]
{
    view_visible[0] = true;
}

Die obige code prüft, ob view[0] sichtbar ist oder nicht, und wenn nicht, dann wird es sichtbar gemacht.
