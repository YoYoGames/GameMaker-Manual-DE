camera_set_default

Mit dieser Funktion können Sie die Standardkamera so einstellen, dass eine benutzerdefinierte Kamera verwendet wird, die Sie zuvor mit einer der folgenden Funktionen erstellt haben camera_create() Funktionen erstellt haben. Wenn Sie einen Raum ohne aktive Ansichtsfenster oder Ansichtskameras erstellen, verwendet GameMaker trotzdem eine Kamera, um das Geschehen im Spiel zu zeigen. Diese Kamera wird als Standardkamera bezeichnet und kann wie jede andere Kamera eingestellt und manipuliert (und sogar zerstört) werden.

Syntax:

camera_set_default(camera_id)

ArgumentTyp Beschreibung
camera_id The unique camera ID value returned when you created the camera

Rückgabe:

Beispiel:

var new_cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);
camera_set_default(new_cam);

Die obige code erstellt eine neue Kamera und weist sie dann als Standardkamera zu.
