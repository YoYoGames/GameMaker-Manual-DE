camera_get_view_height

Mit dieser Funktion können Sie die Höhe (in Pixeln) der angegebenen Kamera abfragen. Beachten Sie, dass diese Funktion nur für Kameras gültig ist, die mit camera_create_view() erstellt wurden oder die im Raum Editor erstellt wurden.

Syntax:

camera_get_view_height(camera_id)

ArgumentTyp Beschreibung
camera_id The unique camera ID value returned when you created the camera

Rückgabe:

Beispiel:

var vw = camera_get_view_width(view_camera[0]) / 2;
var vh = camera_get_view_height(view_camera[0]) / 2;
camera_set_view_pos(view_camera[0], x - vw, y - vh);

Die obige code ruft die Breite und Höhe der Kamera ab, die dem view port[0] zugewiesen ist, und setzt dann ihre Position relativ zum Zentrum.
