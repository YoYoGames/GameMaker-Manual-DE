view_camera

Dieses Array enthält die eindeutige Kamera-ID, die dem gegebenen Viewport zugewiesen ist, und kann auf eine neue Kamera gesetzt oder gelesen werden, um die aktuelle Kamera zu erhalten, wobei -1 zurückgegeben wird, wenn keine Kamera zugewiesen ist. Es können bis zu 8 View Ports in einem Raum aktiv sein (Array-Werte 0 bis 7), und man kann jedem von ihnen eine Kamera zuweisen, indem man diese Variable einfach auf den Wert der Kamera-ID setzt (wie von den Funktionen camera_create() oder camera_create_view()). Wenn Sie die Kamera über den Raumeditor einem Ansichtsanschluss zugewiesen haben, enthält diese Variable die Kamera-ID für diesen Anschluss.

Syntax:

view_camera[0 ... 7];

Rückgabe:

Camera ID or -1 (if no camera is assigned)

Beispiel:

if view_camera[0] == -1
{
    view_camera[0] = camera_create_view(0, 0, room_width, room_height);
}

Die obige code prüft, ob an den View Port [0] eine Kamera angeschlossen ist, und wenn nicht, wird eine erstellt und auf diesen gesetzt.
