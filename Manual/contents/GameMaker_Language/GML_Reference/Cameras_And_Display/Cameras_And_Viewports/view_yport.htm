view_yport

Mit diesem eingebauten Array können Sie die y-Position des gegebenen View Ports abrufen oder einstellen. Der Viewport ist der Bereich auf dem Bildschirm, in dem die Ansicht gezeichnet wird, und es können bis zu 8 gleichzeitig aktiv sein (das Array hat Werte von 0 bis einschließlich 7, so dass 8 Ports). Wenn Sie jedoch View Ports und Kameras aktivieren, ändert sich dieses Verhalten und es wird die Gesamtgröße der Bounding Box für alle View Ports verwendet. Wenn Sie also zwei verschiedene Ansichtsfenster an zwei verschiedenen Positionen haben, bestimmt die Gesamtfläche, die sie abdecken, die Größe des Spielfensters. Wenn Sie nur einen View Port haben, kann das Setzen des x- oder y-Wertes des Ports einige interessante Effekte haben, ist aber normalerweise nicht praktikabel (siehe das Bild unten). Daher ist es besser, die x- und y-Position als (0,0) beizubehalten.

Syntax:

view_yport[0 .. 7];

Rückgabe:

Beispiel:

view_xport[0] = 0;
view_yport[0] = 0;

Die obige code setzt die Position von view port[0] auf die Position (0,0) (linke obere Ecke) der Anzeige zurück.
