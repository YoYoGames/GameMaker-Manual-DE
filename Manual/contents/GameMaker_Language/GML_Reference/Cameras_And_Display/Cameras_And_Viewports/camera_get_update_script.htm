camera_get_update_script

Diese Funktion kann verwendet werden, um die ID der Skriptfunktion abzurufen, die als Update script für die angegebene Kamera zugewiesen wurde. Wenn keine script zugewiesen ist, gibt die Funktion -1 zurück.

Syntax:

camera_get_update_script(camera_id)

ArgumentTyp Beschreibung
camera_id The unique camera ID value returned when you created the camera

Rückgabe:

Script Function or -1 if no function is assigned

Beispiel:

var scr = camera_get_update_script(camera_view[0]);
if (scr != scr_cutscene)
{
    camera_set_update_script(camera_view[0], scr_cutscene);
}

Die obige code prüft die Funktion script, die als Aktualisierungsfunktion für die dem Viewport[0] zugewiesene Kamera zugewiesen ist, und wenn sie nicht "cutScene" ist, wird sie auf diese script gesetzt.
