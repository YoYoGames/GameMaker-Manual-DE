display_get_timing_method

Diese Funktion kann verwendet werden, um die Timing-Methode abzurufen, die für das Rendering Ihres Spiels verwendet werden soll. Die Methode kann eine der unten aufgeführten Konstanten sein.

Weitere Informationen zu den verschiedenen Zeitmessungsmethoden finden Sie unter display_set_timing_method().

Syntax:

display_get_timing_method();

Rückgabe:

Display Timing Method Constant

Anzeige Timing-Methode Konstant
Konstante Beschreibung
tm_sleep Der Wert der Schlafspanne ist die wichtigste Zeitmessmethode
tm_countvsyncs Vsync-Timing ist die wichtigste Timing-Methode (Standard für alle unterstützten platforms)

Beispiel:

if display_get_timing_method() != tm_sleep
{
    display_set_timing_method(tm_sleep);
    if display_get_sleep_margin() != 20
    {
        display_set_sleep_margin(20);
    }
}

Die obige code prüft die Timing-Methode, und wenn sie nicht auf tm_sleep eingestellt ist, wird sie auf diese eingestellt und die Schlafspanne auf 20 gesetzt.
