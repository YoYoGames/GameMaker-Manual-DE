gif_open

Mit dieser Funktion können Sie ein leeres Bild im GIF-Format erstellen, in das Sie Daten einfügen können. Wenn Sie die Funktion aufrufen, müssen Sie die Breite und Höhe des GIFs angeben (in Pixeln, und es gibt keine Obergrenze für die Größe, außer für den verfügbaren Speicher), und die Funktion gibt den eindeutigen ID-Wert zurück, der verwendet wird, um das Geschenk in nachfolgenden Funktionen zu identifizieren, oder sie gibt -1 zurück, wenn das GIF nicht initialisiert werden konnte (zum Beispiel, wenn die Breite/Höhe zu groß für den verfügbaren Speicher ist). Sie können auch eine optionale argument angeben, um die "klare Farbe" für das GIF festzulegen. Dabei handelt es sich um einen RGB-Wert (ohne Alphakomponente), der das GIF auf diese Farbe zurücksetzt, bevor ihm etwas hinzugefügt wird. Wenn Sie keine klare Farbe argument angeben, wird die Standardfarbe Schwarz verwendet.

Beachten Sie, dass Sie bei Verwendung dieser Funktion die Funktion gif_save() aufrufen müssen, um die Erstellung des GIFs zu beenden, bevor Sie eine andere Datei für die Aufnahme öffnen (daher muss für jede gif_open() eine begleitende gif_save() vorhanden sein).

Syntax:

gif_open(width, height, [clear_colour]);

Argument Typ Beschreibung
width Real The width of the gif to create
height Real The height of the gif to create
[clear_colour] Farbe OPTIONAL Die Farbe (RGB), auf die das Gif gelöscht werden soll

Rückgabe:

GIF ID

Beispiel:

if save_gif == true
{
    if count == 0
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if count < 30
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        gif_save(gif_image, "GameCapture.gif");
        count = 0;
        save_gif = false;
    }
    count++;
}

Mit der obigen code wird eine GIF-Bilddatei mit 30 Einzelbildern aus der Anwendungsoberfläche erstellt und anschließend gespeichert.
