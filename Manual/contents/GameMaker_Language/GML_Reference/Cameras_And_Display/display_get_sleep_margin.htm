display_get_sleep_margin

Diese Funktion kann verwendet werden, um den aktuellen Wert des Ruhezustands zu ermitteln, der für das Rendering-Timing Ihres Spiels verwendet wird, und gibt einen Wert im Millisekundenbereich zurück. Für weitere Informationen über das Anzeige-Timing, lesen Sie bitte display_set_timing_method().

Syntax:

display_get_sleep_margin();

Rückgabe:

Beispiel:

if display_get_timing_method() != tm_sleep
{
    display_set_timing_method(tm_sleep);
    if (display_get_sleep_margin() != 20)
    {
        display_set_sleep_margin(20);
    }
}

Die obige code prüft die Timing-Methode, und wenn sie nicht auf tm_sleep eingestellt ist, wird sie auf diese eingestellt und die Schlafspanne auf 20 gesetzt.
