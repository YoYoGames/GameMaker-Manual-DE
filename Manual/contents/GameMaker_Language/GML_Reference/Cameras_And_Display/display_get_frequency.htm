display_get_frequency

Diese Funktion gibt die Frequenz (oder Aktualisierungsrate) des Bildschirms zurück, auf dem das Spiel gespielt wird. Sie gibt einen realen Wert als Bilder pro Sekunde zurück, also zum Beispiel, wenn Ihr Monitor 60hz hat, erhalten Sie 60, wenn er mit 144hz läuft, erhalten Sie 144, und so weiter.

HINWEIS Diese Funktion wird von HTML5 nicht unterstützt.

Syntax:

display_get_frequency();

Rückgabe:

Beispiel:

var _freq = display_get_frequency();

if (_freq >= 240)
{
    game_set_speed(240, gamespeed_fps);
}
else if (_freq >= 120)
{
    game_set_speed(120, gamespeed_fps);
}
else if (_freq >= 60)
{
    game_set_speed(60, gamespeed_fps);
}
else
{
    game_set_speed(30, gamespeed_fps);
}

Das obige code ermittelt die Frequenz des Bildschirms und führt einige Bedingungen aus, um das Spiel mit 240, 120, 60 oder 30 fps laufen zu lassen. Das bedeutet, dass das Spiel mit 60 fps läuft, wenn der Bildschirm 90hz hat, wenn er 144hz hat, dann mit 120 fps usw. Natürlich können Sie die Frequenz des Bildschirms auch direkt in game_set_speed() eingeben, so dass sie als Framerate für das Spiel verwendet wird.
