display_set_ui_visibility

Diese Funktion kann nur unter Android verwendet werden, um die System-Benutzeroberfläche ein- oder auszublenden. Für die Funktion müssen Sie ein oder mehrere Systemflags als Ganzzahlwert angeben. Wenn Sie mehr als ein Flag verwenden, müssen diese mit dem bitweisen "oder" zusammengeführt werden, wie im folgenden Beispiel gezeigt. Eine Liste von Android system flags finden Sie hier.

Syntax:

display_set_ui_visibility(flags);

Argument Typ Flaggen
flags Real The Android SYSTEM flags to use (as an integer value).

Rückgabe:

N/A

Beispiel:

var flags = 1024|4096;
display_set_ui_visibility(flags);

Das obige code verwendet das Android System flags "SYSTEM_UI_FLAG_IMMERSIVE_STICKY" und "SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN", um die Sichtbarkeit der Anzeige einzustellen.
