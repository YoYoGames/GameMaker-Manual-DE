gif_save_buffer

Mit dieser Funktion können Sie eine GIF-Animation abspeichern. Sie geben den GIF-Index an (wie er von der Funktion gif_open()zurückgegeben wird) und die Funktion gibt ein 1-Byte-aligned grow buffer mit den GIF-Daten zurück. Beachten Sie, dass die endgültigen GIF-Daten mit der Universal 884 Palette palettiert wurden (siehe hier für weitere Informationen).

Syntax:

gif_save_buffer(gif_index);

ArgumentTyp Beschreibung
gif_index The ID of gif to save

Rückgabe:

Buffer ID

Beispiel:

if save_gif == true
{
    if count == 0
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if count < 30
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        global.capture_buff = gif_save_buffer(gif_image);
        count = 0;
        save_gif = false;
    }
    count++;
}

Die obige code erstellt eine GIF-Bilddatei mit 30 Bildern von der Anwendungsoberfläche und speichert sie dann auf buffer.
