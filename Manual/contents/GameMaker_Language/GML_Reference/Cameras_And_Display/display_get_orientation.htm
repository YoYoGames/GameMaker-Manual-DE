display_get_orientation

Diese Funktion gibt eine von vier Konstanten zurück, mit denen GameMaker feststellen kann, ob das Gerät, auf dem das Spiel läuft, im Hoch- oder Querformat gehalten wird (siehe Tabelle unten).

Beachten Sie, dass diese Funktion die Ausrichtung des Geräts möglicherweise nicht korrekt erkennt, wenn sie im HTML5-Ziel verwendet wird module. Dies lässt sich jedoch leicht durch die Verwendung des folgenden Skripts nachahmen:

return (browser_width < browser_height);

Eine solche Funktion würde true für Hochformat und false für Querformat zurückgeben.

Syntax:

display_get_orientation()

Rückgabe:

Display Orientation Constant

Display-Ausrichtung Konstant
Konstante Beschreibung
display_landscape Das Gerät wird waagerecht gehalten, d. h. die längste Kante ist von links nach rechts und die Menütaste befindet sich auf der rechten Seite.
display_landscape_flipped Wie oben, nur dass sich die Menütaste jetzt auf der linken Seite befindet.
display_portrait Das Gerät wird senkrecht gehalten, d. h. die längste Kante verläuft von oben nach unten, und die Menütaste befindet sich unten.
display_portrait_flipped Wie oben, nur dass sich jetzt die Menütaste oben befindet.

Beispiel:

if display_get_orientation() == display_landscape
{
    global.Config = 0;
}
else
{
    global.Config = 1;
}

Die obige code prüft die Ausrichtung des Geräts und setzt eine globale Variable in Abhängigkeit von dem zurückgegebenen Wert.
