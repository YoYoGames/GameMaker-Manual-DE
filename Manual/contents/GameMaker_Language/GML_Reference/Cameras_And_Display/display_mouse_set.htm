display_mouse_set

Mit dieser Funktion können Sie die Position der Maus innerhalb der Spielanzeige ändern oder festlegen, was z. B. für FPS-Spiele nützlich sein kann. Die Funktion funktioniert nur, wenn das Spiel fokussiert ist, und wenn Sie ALT + Tab verwenden, wird die Maus entsperrt.

Syntax:

display_mouse_set(x, y);

ArgumentTyp Beschreibung
x The x coordinate in the display.
y The y coordinate in the display.

Rückgabe:

Beispiel:

display_mouse_set(display_get_width() / 2, display_get_height() / 2);

Die obige code würde die Maus in der Spielanzeige zentrieren.
