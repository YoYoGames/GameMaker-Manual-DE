display_reset

Diese Funktion setzt die Anzeigeeinstellungen auf die beim Start des Spiels eingestellten zurück, ermöglicht es Ihnen aber auch, die aktuelle Stufe des verwendeten Vollbild-Anti-Aliasings zu ändern und festzulegen, ob die vertikale Synchronisation verwendet werden soll. Die verfügbaren Anti-Aliasing-Stufen sind 0, 2, 4 und 8, wobei der Standardwert beim Start auf 0 gesetzt ist und die Standardeinstellung für v-sync falsch (aus) ist. Das Einschalten von v-sync kann zu einem flüssigeren Spielerlebnis führen, benötigt aber auch mehr Rechenleistung, so dass die Auswirkungen vor der Verwendung sorgfältig abgewogen werden müssen; das Gleiche gilt für die Kantenglättung: je höher die Zahl, desto mehr Rechenleistung ist erforderlich.

Da nicht alle Zielgeräte gleich sind und manche zum Beispiel kein 8x oder 4x Anti-Aliasing unterstützen, gibt es eine schreibgeschützte Variable, um die verschiedenen AA-Stufen zu erhalten, die das Gerät, auf dem das Spiel läuft, anzeigen kann:

display_aa

Diese Variable gibt einen ganzzahligen Wert zurück, der auf der Einstellung der Bits für die verschiedenen Stufen basiert. Für 2xAA wird also 2 zurückgegeben, für 2x und 4x Verfügbarkeit wird 6 zurückgegeben. Für 8 und 4 wird sie 12 melden. Für alle 3 Stufen (2, 4 und 8) wird 14 gemeldet.

Syntax:

display_reset(aa, vsync)

Argument Typ Beschreibung
aa Real The level of anti-aliasing filtering (0, 2, 4 or 8).
vsync Boolesche Toggle vertical synchronisation to on (true) or off (false).

Rückgabe:

Real

Beispiel:

if (display_aa > 12)
{
    display_reset(8, true);
}

Die obige code setzt die Anti-Aliasing-Stufe auf 8, falls unterstützt, und switch v-sync auf on.
