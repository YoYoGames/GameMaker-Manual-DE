gc_freigeben

Mit dieser Funktion können Sie den Garbage Collector aktivieren oder deaktivieren. Der Aufruf der Funktion mit true als argument aktiviert ihn, mit false wird er deaktiviert (nicht empfohlen). Standardmäßig ist er aktiviert.

Syntax:

gc_enable(enable);

ArgumentTyp Beschreibung
enable enable (true) oder deaktivieren (false) Sie den Garbage Collector.

Rückgabe:

Beispiel:

if (global.debug == true)
{
    gc_enable(false);
}

Die obige code deaktiviert die Garbage Collection, wenn die angegebene globale Variable true ist.
