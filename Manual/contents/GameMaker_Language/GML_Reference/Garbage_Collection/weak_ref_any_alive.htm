weak_ref_any_alive

Mit dieser Funktion können Sie die Weak-Referenz auf verschiedene Structs überprüfen, um zu sehen, ob sie noch "lebendig" sind oder nicht. Sie geben ein Array von zu prüfenden Weak-Referenzen an (jede Weak-Referenz sollte mit der Funktion weak_ref_create()), und die Funktion gibt true zurück, wenn jede der Strukturen noch irgendwo referenziert werden oder false, wenn dies nicht der Fall ist und sie bereits gelöscht wurden. Beachten Sie, dass die Funktion undefined zurückgibt, wenn Sie ein Array übergeben, in dem keiner der Werte eine Weak-Referenz ist.

Beachten Sie, dass die Funktion auch zwei optionale Parameter benötigt: arguments. Mit dem ersten Parameter können Sie einen Anfangsindex für das zu prüfende Array angeben und mit argument die Anzahl der zu prüfenden Positionen (Länge) ab diesem Index festlegen. Falls angegeben, werden nur die Array-Indizes innerhalb dieser Parameter geprüft.

Syntax:

weak_ref_any_alive(array, [index], [length]);

Argument Typ Beschreibung
array Array von Struct Weak References Array containing weak references to the structs that you want to check.
[index] Real OPTIONAL Der Index im Array, ab dem die Prüfung beginnen soll
[length] Real OPTIONAL Die Anzahl der zu prüfenden Positionen, beginnend mit dem angegebenen Index, für

Rückgabe:

Boolean (or undefined)

Beispiel:

if weak_ref_any_alive(inventory_ref_array)
{
    instance_destroy(obj_Inventory_Control);
}

Die obige code prüft eine Reihe von schwachen Referenzen, und wenn eine davon noch aktiv ist, wird eine Instanz zerstört.
