weak_ref_alive

Mit dieser Funktion können Sie die schwache Referenz auf eine Struktur überprüfen, um zu sehen, ob sie noch "lebendig" ist oder nicht. Sie geben die zu prüfende Weak-Referenz an (wie von der Funktion weak_ref_create()), und die Funktion gibt true zurück, wenn die Struktur noch irgendwo referenziert wird, oder false, wenn dies nicht der Fall ist und sie bereits gelöscht wurde. Beachten Sie, dass die Funktion undefined zurückgibt, wenn Sie einen Wert angeben, der keine "weak reference" ist.

Syntax:

weak_ref_alive(weak_ref);

Argument Typ Beschreibung
weak_ref Struktur Schwache Referenz The weak reference to the struct you want to check.

Rückgabe:

Boolean (or undefined)

Beispiel:

if weak_ref_alive(inventory_ref)
{
    inventory = -1;
}

Die obige code prüft einen schwachen Verweis auf eine Struktur, und wenn er noch vorhanden ist, wird die Variable, die ihn referenziert, auf -1 gesetzt, wodurch der Verweis auf die Struktur aufgehoben wird und sie in die Müllabfuhr gelangt.
