gc_collect

Mit dieser Funktion können Sie den Garbage Collector auslösen und ihn zwingen, am Ende des aktuellen Frames (Schrittes) zu laufen. Es sei darauf hingewiesen, dass der Garbage Collector nicht aktiv sein muss, damit dies funktioniert. Wenn Sie diese Funktion nach dem Deaktivieren des Garbage Collectors (mit der Funktion gc_enable()) aufrufen, wird der Garbage Collector für einen Frame aktiviert, in dem alle Objekte, die zum Sammeln markiert wurden, aus dem Speicher entfernt werden, bevor der Garbage Collector wieder deaktiviert wird.

Syntax:

gc_collect();

Rückgabe:

Beispiel:

if (global.debug == true && keyboard_check_pressed(vk_f1))
{
    gc_collect();
}

Die obige code prüft eine globale Variable und eine gedrückte Taste, und wenn diese true sind, wird die Garbage Collection für das Ende des Frames (Schrittes) ausgelöst.
