gc_get_target_frame_time

Mit dieser Funktion können Sie den aktuellen Ziel-Frame-Wert für den Garbage Collector abrufen. Der zurückgegebene Wert ist in Mikrosekunden (wobei 1.000.000 Mikrosekunden einer Sekunde entsprechen) und die Standard-Zielbildzeit 100 Mikrosekunden beträgt. Wenn Sie diesen Wert ändern möchten, sollten Sie die Funktion gc_target_frame_time() verwenden.

Syntax:

gc_get_target_frame_time(time);

Rückgabe:

Beispiel:

if (gc_get_target_frame_time() != 50)
{
    gc_target_frame_time(50);
}

Die obige code prüft die aktuelle Rahmenzeitvorgabe für den Garbage Collector, und wenn sie nicht 50 Mikrosekunden beträgt, wird sie auf diesen Wert gesetzt.
