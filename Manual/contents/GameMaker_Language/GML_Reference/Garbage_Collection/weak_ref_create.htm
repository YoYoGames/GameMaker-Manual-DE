weak_ref_create

Mit dieser Funktion können Sie einen schwachen Verweis auf eine Struktur erstellen, der dann verwendet werden kann, um zu prüfen, ob die Struktur im Spiel noch "lebendig" (referenziert) ist oder nicht. Sie geben die Referenz auf die Struktur an, die Sie verfolgen wollen, und die Funktion gibt eine andere Struktur zurück, die eine schwache Referenz auf diese Struktur ist. Beachten Sie, dass Sie überprüfen können, ob eine Referenz "strong" oder "weak" ist, indem Sie die Funktion instanceof() verwenden, da eine "strong"-Referenz entweder "struct" oder den Namen der Konstruktorfunktion zurückgibt, die die Struktur erstellt hat, oder "weakref", wenn es eine "weak"-Referenz ist. Beachten Sie auch, dass die "weak reference"-Struktur eine Variable "ref" hat, auf die zugegriffen werden kann, um die "strong"-Referenz auf die betreffende Struktur zu erhalten, es sei denn, sie wurde garbage collected, in diesem Fall gibt sie undefined zurück.

Syntax:

weak_ref_create(struct_to_track);

Argument Typ Beschreibung
struct_to_track Struktur The struct that you want to create a weak reference for

Rückgabe:

Struct Weak Reference

Beispiel:

inventory_ref = weak_ref_create(inventory);

Die obige code erzeugt einen schwachen Verweis auf eine Struktur und speichert ihn in einer Instanzvariablen zur späteren Verwendung.
