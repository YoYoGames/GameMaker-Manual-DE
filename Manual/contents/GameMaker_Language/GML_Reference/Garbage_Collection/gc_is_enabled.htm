gc_is_enabled

Mit dieser Funktion können Sie prüfen, ob der Garbage Collector aktiviert ist oder nicht. Die Funktion gibt true zurück, wenn er aktiviert ist, und andernfalls false.

Syntax:

gc_is_enabled();

Rückgabe:

Beispiel:

if (!gc_is_enabled())
{
    gc_enable(true);
}

Die obige code prüft, ob der Garbage Collector aktiviert ist, und wenn nicht, wird er aktiviert.
