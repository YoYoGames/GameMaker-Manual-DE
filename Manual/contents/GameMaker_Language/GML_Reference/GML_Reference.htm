GML-Code-Referenz

Dieser Teil des Handbuchs ist ein Referenzhandbuch für den GameMaker Sprache (GML). Sie finden hier alle verfügbaren Funktionen dokumentiert, zusammen mit den erforderlichen arguments und Beispielen von code, um zu zeigen, wie sie verwendet werden können. Informationen zur Verwendung von GML finden Sie in der GML-Übersicht Abschnitt des Handbuchs.

HINWEIS Überall, wo im Handbuch "N/A" als Rückgabewert für eine Funktion angegeben ist, bedeutet dies, dass die Funktion keinen Wert als Teil ihrer Operation zurückgeben soll. Wenn Sie versuchen, einen Rückgabewert für diese Funktionen zu ermitteln, erhalten Sie den Wert undefined.
