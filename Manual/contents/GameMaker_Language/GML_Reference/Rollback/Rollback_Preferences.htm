Rollback-Einstellungen

Vielleicht möchten Sie jedem Spieler die Möglichkeit geben, einen Charakter auszuwählen oder seine Eigenschaften anzupassen. Solche Eigenschaften können als "Spielerpräferenzen" eingerichtet werden.

Wie werden die Einstellungen verwendet?

Aktivieren Sie zunächst die Einstellungen, indem Sie rollback_use_player_prefs() / Spielervoreinstellungen verwenden. Mit dieser Funktion können Sie optional die Standardeinstellungen für alle Spieler festlegen.

Sobald du diese Funktion aufrufst, wird "Autostart" deaktiviert, d.h. das Spiel wird nicht gestartet, auch wenn alle Spieler beigetreten sind, und du musst es manuell starten mit rollback_start_game() / Spiel starten.

Bevor das Spiel begonnen hat, können Sie die Einstellungen für einen Spieler festlegen, indem Sie rollback_set_player_prefs() / Set Player Prefs auf dem Client des Spielers aufrufen. Diese Einstellungen können später ausgelesen werden durch rollback_get_player_prefs() / Get Player Prefs oder die Variable player_prefs in einer Spielerinstanz gelesen werden.

Rollback-Ereignis

Wenn ein Spieler seine Einstellungen ändert, erhält jeder im Spiel ein rollback_player_prefs Rollback-Ereignis. Der Spieler muss vollständig verbunden sein, damit das Ereignis gesendet werden kann.

Wurden die Einstellungen vor dem Verbindungsaufbau festgelegt, so werden nach dem Verbindungsaufbau die zuletzt festgelegten Einstellungen an alle Spieler gesendet.

Startbedingung

Wenn Sie die Voreinstellungen aktiviert haben und es keine Standardeinstellungen gibt, wird das Spiel erst dann gestartet, wenn die Einstellungen für alle Spieler festgelegt wurden.

In einem solchen Fall können Sie entweder die Funktion rollback_start_game() wiederholt in einem Alarm aufrufen, um zu versuchen, das Spiel zu starten, oder es immer dann aufrufen, wenn das rollback_player_prefs Rollback-Ereignis empfangen wird.

Welche Typen kann ich verwenden?

Eine Präferenz kann von beliebigem Typ sein, z. B. eine Zahl, eine Zeichenkette, ein Array usw., aber es wird empfohlen, eine Struktur zu verwenden, da dies der einfachste Weg ist, eine Sammlung von Variablen zu speichern.
