rollback_verwendet_zufällige_Eingabe

Wenn Sie ein Spiel mit Sync Test spielen, erhalten alle "entfernten" Spieler zufällige Werte für ihre definierten Eingaben, als eine grundlegende Art des Testens. Sie können dieses Verhalten deaktivieren, indem Sie diese Funktion mit false als argument aufrufen, und es durch Angabe von true wieder aktivieren.

Syntax:

rollback_use_random_input(enabled);

Argument Typ Beschreibung
enabled Boolesche Specifies whether random input should be enabled (true) oder deaktiviert(false). Standardmäßig ist sie aktiviert.

Rückgabe:

N/A

Beispiel:

rollback_use_random_input(false);

Die obige code deaktiviert die Zufallseingabe während des Synchrontests.
