rollback_verwendung_player_prefs

Mit dieser Funktion können Sie die Spielereinstellungen aktivieren und die Standardeinstellungen für alle Spieler festlegen.

Die Standardeinstellungen können von beliebiger Art sein. Wenn Sie keine Standardeinstellungen angeben, müssen alle Spieler über Einstellungen verfügen, bevor das Spiel beginnen kann.

Wenn Sie diese Funktion aufrufen, wird der Autostart deaktiviert, so dass Sie das Spiel manuell starten müssen. Siehe: Manueller vs. automatischer Start

Wenn diese Funktion verwendet wird, muss sie vor der Funktion join/create ausgeführt werden.

Bitte lesen: Spieler-Einstellungen

Syntax:

rollback_use_player_prefs([default]);

Argument Typ Beschreibung
default Jede OPTIONAL Die Standardeinstellungen, die für alle Spieler festgelegt werden.

Rückgabe:

N/A

Beispiel:

rollback_use_player_prefs({
Zeichen: 0,
Farbe: c_white
});

Die obige code aktiviert die Spielereinstellungen und setzt die Standardeinstellungen in einer Struktur.
