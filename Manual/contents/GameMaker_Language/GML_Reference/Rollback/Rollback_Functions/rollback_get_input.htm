rollback_get_input

Diese Funktion gibt eine Struktur zurück, die die Eingabewerte für einen Spieler enthält. Sie benötigt ein optionales argument, das die ID des Spielers angibt, für den die Eingaben zurückgegeben werden.

Bei Verwendung in einer Spielerinstanz, die mit rollback_define_player()erstellt wurde, müssen Sie keine Spieler-ID angeben, so dass Sie diese Funktion ohne arguments aufrufen können.

Bei Verwendung in anderen Instanzen oder manuell erstellten Player-Instanzen müssen Sie eine Player-ID angeben.

Informationen darüber, welche Eingaben zurückgegeben werden und wie sie geändert werden können, finden Sie unter Definition von Eingaben.

Syntax:

rollback_get_input(player_id);

Argument Typ Beschreibung
player_id Real OPTIONAL Die ID des Spielers, für den Eingaben zurückgegeben werden sollen.

Rückgabe:

Struct

Beispiel:

var _input = rollback_get_input();

if (_input.left) x -= 2;
if (_input.right) x += 2;
if (_input.up) y -= 2;
if (_input.down) y += 2;

Die obige code ruft die Eingabestruktur in einer Player-Instanz ab, die automatisch vom Rollback-System erstellt wurde. Basierend auf den zurückgegebenen Werten wird die Instanz um 2 Pixel in jede Richtung verschoben.

Ein ausführliches Beispiel finden Sie unter Erstellen eines Multiplayer-Spiels.
