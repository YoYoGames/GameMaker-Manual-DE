rollback_manuell_starten

Standardmäßig beginnt ein Multiplayer-Spiel automatisch, wenn alle Spieler beigetreten sind.

Durch den Aufruf dieser Funktion wird das Autostartverhalten deaktiviert, so dass das Spiel darauf wartet, dass Sie es manuell starten.

Wenn diese Funktion verwendet wird, muss sie vor der Funktion join/create ausgeführt werden.

Syntax:

rollback_use_manual_start();

Rückgabe:

N/A

Beispiel:

rollback_use_manual_start();

if (!rollback_join_game())
{
    rollback_create_game(2);
}

Die obige code aktiviert den manuellen Start und tritt dann einem neuen Mehrspielerspiel bei oder erstellt ein solches.
