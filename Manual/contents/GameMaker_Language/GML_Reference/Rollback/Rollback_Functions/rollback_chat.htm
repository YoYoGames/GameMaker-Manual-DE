rollback_chat

Diese Funktion wird verwendet, um eine Nachricht an andere Spieler zu senden.

Sie geben eine zu sendende Zeichenfolge mit einer maximalen Größe von 128 Byte an.

Wenn das zweite argument nicht angegeben ist, wird die Nachricht an alle Spieler, einschließlich des Absenders, gesendet.

Sie können in der zweiten argument eine Spieler-ID angeben, die der Spieler ist, der die Nachricht erhalten soll.

Wie verwenden?

Diese Funktion ist für die "lokale" Verwendung gedacht, d.h. sie wird durch lokale Eingaben und nicht durch Rollback-Eingaben ausgelöst.

Siehe das Anwendungsbeispiel am Ende. Warten Sie auf empfangene Nachrichten im Ereignis Rollback Event.

Syntax:

rollback_chat(message, [to]);

Argument Typ Beschreibung
message Zeichenfolge The message to send, as a string.
to Real OPTIONAL Die ID des Spielers, an den die Nachricht gesendet werden soll. Wird argument nicht angegeben oder wird -1 angegeben, wird die Nachricht an alle Spieler gesendet.

Rückgabe:

N/A

Beispiel:

if (keyboard_check_pressed(vk_enter))
{
    rollback_chat(keyboard_string);
    keyboard_string = "";
}

Wenn die Eingabetaste lokal gedrückt wird, wird der keyboard_string Wert an alle Spieler gesendet. Diese Zeichenfolge wird dann gelöscht, damit eine neue Nachricht eingegeben werden kann.

Dadurch wird das Ereignis " Rollback Event " ausgelöst, wenn die Nachricht von allen Clients (einschließlich dem Absender) empfangen wird.

Rollback-Ereignis

if (rollback_event_id == rollback_chat_message)
{
    // Get received information
    var _message = rollback_event_param.message;
    var _from = rollback_event_param.from;

    // Find sender's name
    var _name = rollback_get_info(_from).player_name;

    // Print to output log
    show_debug_message(_name + " says: " + _message);
}
