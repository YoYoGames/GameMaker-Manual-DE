rollback_start_game

Diese Funktion erzwingt den Start des aktuellen Multiplayer-Spiels, bevor alle Spieler beigetreten sind. Sie müssen zuerst ein Mehrspielerspiel erstellen mit rollback_create_game().

Normalerweise wartet das System, bis alle Spieler beigetreten sind, bevor es das Spiel automatisch startet. Diese Funktion kann verwendet werden, um das Spiel frühzeitig zu starten, bevor alle Spieler beigetreten sind.

Sie können die Funktion rollback_use_manual_start() aufrufen, damit Rollback auf diese Funktion wartet, auch wenn alle Spieler beigetreten sind.

Durch den Aufruf dieser Funktion wird das Ereignis Rollback Start ausgelöst.

Syntax:

rollback_start_game();

Rückgabe:

N/A

Beispiel:

// Step event
if (!rollback_game_running && keyboard_check_pressed(vk_enter))
{
    rollback_start_game();
}

Wenn das Rollback-Spiel nicht läuft und die Eingabetaste gedrückt wird, wird das Spiel gestartet.

Dies ermöglicht es dem Spieler, das Spiel manuell zu starten, bevor alle Spieler teilnehmen.
