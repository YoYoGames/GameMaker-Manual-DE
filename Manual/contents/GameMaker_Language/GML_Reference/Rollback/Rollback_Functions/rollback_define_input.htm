rollback_define_input

Diese Funktion wird verwendet, um benutzerdefinierte Eingaben für alle Spieler zu definieren. Sie benötigt eine Struktur, die alle Eingänge mit den zugewiesenen Eingangskonstanten enthalten sollte.

Wie Sie benutzerdefinierte Eingänge zuweisen können, erfahren Sie unter Benutzerdefinierte Steuerelemente.

Wenn diese Funktion verwendet wird, muss sie vor der Funktion join/create ausgeführt werden.

Syntax:

rollback_define_input(input_struct);

Argument Typ Beschreibung
input_struct Struktur A struct containing input names as variables, and their assigned inputs as values.

Rückgabe:

N/A

Beispiel:

rollback_define_input({
    fire: mb_left,
    interact: vk_space,
    left: [ord("A"), vk_left],
    right: [ord("D"), vk_right]
});

Die obige code definiert benutzerdefinierte Eingaben für alle Spieler. Die letzten beiden Eingaben verwenden mehrere Steuerelemente, indem sie in einem Array aufgelistet werden.

Ein ausführliches Beispiel finden Sie unter Definition von Eingaben.
