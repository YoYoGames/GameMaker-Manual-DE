rollback_get_player_prefs

Diese Funktion gibt die Einstellungen für die angegebene Spieler-ID zurück.

Wird keine Spieler-ID angegeben, muss die Funktion innerhalb einer Spielerinstanz aufgerufen werden. In diesem Fall werden die Einstellungen für diesen Spieler zurückgegeben.

Sie können auch die Instanzvariable player_prefs verwenden.

Diese Funktion gibt die letzten Einstellungen zurück, die für den angegebenen Spieler synchronisiert wurden. Das Setzen der Einstellungen aktualisiert also nicht sofort die Einstellungen, die diese Funktion zurückgibt.

Bitte lesen: Spieler-Einstellungen

Syntax:

rollback_get_player_prefs([player_id]);

Argument Typ Beschreibung
player_id Real OPTIONAL Die ID des Spielers, für den die Einstellungen zurückgegeben werden.

Rückgabe:

Any

Beispiel:

var _my_prefs = rollback_get_player_prefs();

image_index = _my_prefs.character;

Die obige code ruft die für den aktuellen Spieler festgelegten Einstellungen ab und wendet eine Einstellung auf eine Instanzvariable an.
