rollback_display_events

Wenn Sie das Spiel starten, sehen Sie standardmäßig Debug-Meldungen in der oberen linken Ecke des Spiels, die Mehrspieler-Ereignisse anzeigen, während sie passieren. Du kannst diese Meldungen deaktivieren, indem du diese Funktion mit false als argument aufrufst, und sie wieder aktivieren, indem du true angibst.

Syntax:

rollback_display_events(enabled);

Argument Typ Beschreibung
enabled Boolesche Specifies whether event display should be enabled (true) oder deaktiviert(false). Standardmäßig ist sie aktiviert.

Rückgabe:

N/A

Beispiel:

rollback_display_events(false);

Die obige code hat die Anzeige von Ereignissen deaktiviert.
