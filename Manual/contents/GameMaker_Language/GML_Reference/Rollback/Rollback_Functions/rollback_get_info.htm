rollback_get_info

Diese Funktion gibt eine Struktur mit Informationen über einen Spieler zurück. Diese Funktion sollte nur verwendet werden, wenn man Spielerinstanzen manuell verwaltet. Wenn Sie die Funktion rollback_define_player()verwenden, erhält jede Spielerinstanz diese Informationen automatisch als Variablen.

Die Funktion nimmt die ID des Spielers als argument an und gibt eine Struktur mit den folgenden Variablen zurück:

Variabel Typ Beschreibung
player_avatar_url Zeichenfolge URL, die auf das Avatarbild des Spielers verweist
player_avatar_sprite Sprite-Asset Sprite, das den Avatar des Spielers enthält
player_name Zeichenfolge Der Name des Spielers
player_type Zeichenfolge Gibt an, ob der Spieler bei GX.games angemeldet ist oder nicht

Ist "Gast" für Gastbenutzer und "Benutzer" für angemeldete Benutzer
player_user_id Zeichenfolge Die ID des Spielers auf GX.games
player_prefs Jede Die für den Player festgelegten Einstellungen

Syntax:

rollback_get_info(player_id);

Argument Typ Beschreibung
player_id Real The ID of the player for which information will be returned.

Rückgabe:
