Spieler_Typ

Diese Variable gibt an, ob es sich bei der Player-Instanz um einen angemeldeten GX.games-Benutzer oder um einen Gast handelt. Sie kann eine der folgenden Zeichenketten zurückgeben:

Zeichenfolge Beschreibung
"Guest" Der Spieler ist nicht bei GX.games angemeldet
"User" Der Spieler ist bei GX.games angemeldet

Gastbenutzer erhalten Platzhalter für Avatare und Namen.

Diese Variable existiert nur in Spielerinstanzen, die von rollback_define_player()erstellt wurden, und ist für jeden Spieler unterschiedlich.

Syntax:

player_type;

Rückgabe:

String
