spieler_benutzer_id

Diese Variable speichert die GX.games-ID der aktuellen Spielerinstanz für die Verwendung bei direkten API-Aufrufen an den Server.

Diese Variable existiert nur in Spielerinstanzen, die von rollback_define_player()erstellt wurden, und ist für jeden Spieler unterschiedlich.

BeiGastspielern wird dieser Variablen eine leere Zeichenfolge zugewiesen.

Syntax:

player_user_id;

Rückgabe:

String
