player_avatar_url

Diese Variable speichert die URL zum Avatar des Spielers, der mit sprite_add() geladen werden kann.

Sie können player_avatar_sprite verwenden, das bereits die sprite geladen hat.

Diese Variable existiert nur in Spielerinstanzen, die von rollback_define_player()erstellt wurden, und ist für jeden Spieler unterschiedlich.

BeiGastspielern wird ein Platzhalter sprite angezeigt.

Syntax:

player_avatar_url

Rückgabe:

String
