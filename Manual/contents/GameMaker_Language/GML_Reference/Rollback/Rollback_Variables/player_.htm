spieler_name

Diese Variable speichert den Namen der aktuellen Spielerinstanz.

Diese Variable existiert nur in Spielerinstanzen, die von rollback_define_player()erstellt wurden, und ist für jeden Spieler unterschiedlich.

Gastspieler geben einen Platzhalternamen zurück.

Syntax:

player_name;

Rückgabe:

String

Beispiel:

// Draw event
draw_set_halign(fa_center);
draw_set_valign(fa_middle);

draw_text(x, y - 100, player_name);

draw_set_valign(fa_left);
draw_set_valign(fa_top);

Dies zeichnet den Namen des Spielers 100px über seiner Instanz, mit einer Mitte-Mitte-Ausrichtung.
