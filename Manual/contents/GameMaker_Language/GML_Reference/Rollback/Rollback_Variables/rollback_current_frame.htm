rollback_aktuell_frame

Dies ist eine globale Variable, die die Anzahl der Frames speichert, die seit dem Start des Multiplayer-Spiels vergangen sind.

Diese sollte anstelle von current_time und anderen ähnlichen Variablen verwendet werden, wie unter Zeitvariablen erläutert.

Syntax:

rollback_current_frame

Rückgabe:

Real

Beispiel:

image_angle = dsin(rollback_current_frame) * 90;

Die obige code verwendet rollback_current_frame mit dsin() um einen oszillierenden Wert zwischen 1 und -1 zu berechnen, der mit 90 multipliziert und auf den Winkel der Instanz angewendet wird.
