rollback_confirmed_frame

Diese globale Variable speichert die letzte Bildnummer, bei der die Eingaben für alle Spieler synchronisiert und bestätigt wurden.

Sie können dies mit rollback_current_frame vergleichen, um zu prüfen, ob der aktuelle Frame für alle Spieler bestätigte Eingaben hat.

Sie können auch die Funktion rollback_sync_on_frame() verwenden, um zu prüfen, ob der aktuelle Rahmen synchronisiert ist.

Syntax:

rollback_confirmed_frame

Rückgabe:

Real

Beispiel:

var _is_frame_confirmed = (rollback_current_frame == rollback_confirmed_frame);

if (_is_frame_confirmed)
{
    evaluate_game_over();
}

Diese code prüft, ob der aktuelle Frame von allen Spielern bestätigt wurde, indem sie prüft, ob der aktuelle Frame-Wert gleich dem bestätigten Frame-Wert ist.

Wenn dies der Fall ist, wird eine Funktion ausgeführt, die prüft, ob die Bedingung für das Ende des Spiels erfüllt ist, damit das Spiel beendet werden kann.

Dies geschieht, um zu verhindern, dass ein auf Vorhersagen basierender Zustand das Spiel beendet, und wartet auf einen vollständig synchronisierten Frame, bevor er versucht, die Bedingungen für das Ende des Spiels zu überprüfen.
