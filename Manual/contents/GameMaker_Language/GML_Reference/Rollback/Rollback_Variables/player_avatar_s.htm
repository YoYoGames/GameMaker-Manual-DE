spieler_avatar_sprite

Diese Variable speichert die sprite der aktuellen Spielerinstanz.

Diese Variable existiert nur in Spielerinstanzen, die von rollback_define_player()erstellt wurden, und ist für jeden Spieler unterschiedlich.

BeiGastspielern wird ein Platzhalter sprite angezeigt.

Syntax:

player_avatar_sprite;

Rückgabe:

Sprite Asset

Beispiel:

// Draw event
draw_sprite_stretched(player_avatar_sprite, 0, x, y - 256, 64, 64);

Dies zeigt den Avatar der Spielerinstanz, auf der die code läuft, in einem 64x64 Bereich, 256px über der Instanz.
