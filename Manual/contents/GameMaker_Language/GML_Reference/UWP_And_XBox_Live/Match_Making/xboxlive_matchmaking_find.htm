xboxlive_matchmaking_find

Diese Funktion sucht nach einer Multiplayer-Sitzung unter Verwendung der Sitzungsvorlage, des Matchmaking-Trichters und der Vorlage für die sichere Gerätezuordnung, die Sie zuvor im XDP-Dashboard erstellt haben sollten. Die Benutzer-ID können Sie für ein bestimmtes Gamepad abrufen, indem Sie die xboxlive_user_for_pad() Funktion. Die Funktion gibt einen eindeutigen Wert für die Request ID zurück, der dann zur Identifizierung des richtigen sozialen asynchronen Ereignisses für diese Funktion verwendet werden kann. Dieses Ereignis wird ausgelöst, wenn die Sitzung erstellt wurde, und enthält eine DS-Map in der Variablen async_load mit den folgenden Schlüssel/Wert-Paaren:

  • "requestid" - enthält die von der aufrufenden Funktion zurückgegebene Anfrage-ID
  • "status" - enthält die Zeichenkette "session_created", um Sie darüber zu informieren, welche Art von Ereignis ausgelöst wurde.
  • "num_results" - enthält 1 oder mehr, wenn eine Sitzung gefunden wurde, sonst 0.
  • "sessionid<index> " - enthält die Sitzungs-ID, oder -1 im Falle eines Fehlers(<index/gt; ist ein Wert von 0 bis "num_results" - 1).
  • "sessionOwner<index> " - enthält die ID des Sitzungsleiters, wenn eine Sitzung gefunden wurde(<index> ist ein Wert von 0 bis "num_results" - 1).
  • "error" - enthält 0 bei Erfolg oder -1 im Falle eines Fehlers

Syntax:

xboxlive_matchmaking_find(user_id, template, hopper, sdatemplate, [matchattributes]);

Argument Typ Beschreibung
user_id Xbox-Benutzer-ID The user ID pointer to use
template Zeichenfolge The name of the session template
hopper Zeichenfolge The name of the matchmaking hopper
sdatemplate Zeichenfolge The name of the secure device association template
matchattributes Real OPTIONAL Die Trefferattribute

Rückgabe:

Async Request ID

Beispiel:

var userID = xboxone_user_for_pad(global.PadIndex);
result = xboxone_matchmaking_find(userID, "MatchTicketSession", "MatchTicketHopper", "PeerServerTraffic");

Die obige Funktion ruft die Benutzer-ID für den Benutzer auf dem angegebenen Gamepad ab und versucht dann, eine Matchmaking-Sitzung für ihn zu finden.
