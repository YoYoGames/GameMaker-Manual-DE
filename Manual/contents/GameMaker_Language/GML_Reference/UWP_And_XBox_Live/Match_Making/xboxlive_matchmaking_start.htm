xboxlive_matchmaking_start

Diese Funktion muss aufgerufen werden, bevor Sie eine Multiplayer-Sitzung für den Benutzer starten können. Sie nimmt den Zeiger auf die Benutzer-ID, den Sie für ein bestimmtes Gamepad durch Aufruf der Funktion xboxlive_user_for_pad() Funktion abrufen können, und initialisiert die Multiplayer-API.

Syntax:

xboxlive_matchmaking_start(user_id);

ArgumentTyp Beschreibung
user_id The user ID pointer to use

Rückgabe:

Beispiel:

global.UserID = xboxlive_user_for_pad(global.PadIndex);
xboxlive_matchmaking_start(global.UserID);

Die obige code ruft die Benutzer-ID für den angegebenen Pad-Index ab und initialisiert dann die Multiplayer-Fähigkeiten.
