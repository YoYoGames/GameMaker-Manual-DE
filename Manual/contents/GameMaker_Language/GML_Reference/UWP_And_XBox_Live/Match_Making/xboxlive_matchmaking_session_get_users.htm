xboxlive_matchmaking_session_get_users

Diese Funktion erstellt und füllt eine DS-Map mit den Details der Benutzer in der angegebenen Sitzung oder -1, wenn ein Fehler auftritt. Sie können den Sitzungs-ID-Zeiger aus der async_load DS-Map entnehmen, die im Asynchronous Social Event zurückgegeben wird, wenn Sie eine Sitzung erstellt oder gefunden haben (siehe xboxlive_matchmaking_create() für weitere Details). Die zurückgegebene DS-Map enthält die folgenden Schlüssel/Wertpaare:

  • "num_results" - enthält die Anzahl der Nutzer in der Sitzung
  • "userId<index>" - enthält die ID des Benutzers (<index> ist ein Wert von 0 bis "num_results" - 1)
  • "userIsOwner<index>" - enthält 1, wenn der Benutzer der Gastgeber ist, sonst 0 (<index> ist ein Wert von 0 bis "num_results" - 1)

Syntax:

xboxlive_matchmaking_session_get_users(session_id);

ArgumentTyp Beschreibung
session_id The session ID pointer to use

Rückgabe:

Async Request ID

Beispiel:

var session_map = xboxlive_matchmaking_session_get_users(global.SessionID);
if (session_map != -1)
{
    for (var i = 0; i < session_map[? "num_results"]; i++)
    {
        if (session_map[? "userIsOwner" + string(i)] == 1)
        {
            global.OwnerID = session_map[? "userId" + string(i)];
        }
    }
}

Damit werden die Benutzerdaten aller an der Matchmaking-Sitzung teilnehmenden Benutzer abgerufen und eine globale Variable auf die ID des Sitzungsbesitzers gesetzt.
