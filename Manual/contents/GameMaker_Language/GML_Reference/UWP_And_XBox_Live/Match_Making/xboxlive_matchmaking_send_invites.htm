xboxlive_matchmaking_send_invites

Mit dieser Funktion wird der Xbox Live-Systemdialog aufgerufen, damit ein Benutzer Einladungen zu seinem Spiel senden kann. Sie geben den lokalen Benutzer an, der den Dialog steuern soll (und dessen Freundesliste vom Dialog verwendet wird), den eindeutigen Sitzungs-ID-Wert für die Sitzung, zu der Sie Leute einladen möchten, und einen Standardtext, der im Dialog angezeigt werden soll. Die Sitzungs-ID erhalten Sie aus der async_load DS-Map, die von der Funktion xboxlive_matchmaking_create() erzeugt wird.

Die Funktion gibt -1 zurück, wenn die arguments ungültig ist oder der Benutzer nicht bei Xbox Live angemeldet ist, oder andernfalls einen Wert größer als 0.

Syntax:

xboxlive_matchmaking_send_invites(from_user, session_to_invite_into, text_to_add_to_invite);

ArgumentTyp Beschreibung
from_user The local user ID for controlling the dialogue.
session_to_invite_into The unique session ID for the session to join.
text_to_add_to_invite A short text to add to the invitation.

Rückgabe:

Beispiel:

if gamepad_button_check_pressed(0, gp_face1)
{
    var _uid = xboxlive_user_for_pad(0);
    xboxlive_matchmaking_send_invites(_uid, global.session_ID, "Come and join my game!");
}

Die obige code prüft, ob eine Gamepad-Taste gedrückt wurde, und wenn dies der Fall ist, wird der Xbox Live-Dialog für Sitzungseinladungen geöffnet.
