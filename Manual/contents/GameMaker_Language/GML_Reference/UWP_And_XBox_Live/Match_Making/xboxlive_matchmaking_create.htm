xboxlive_matchmaking_create

Diese Funktion erstellt eine Multiplayer-Sitzung unter Verwendung der Sitzungsvorlage, des Matchmaking-Hoppers und der Vorlage für die sichere Gerätezuordnung, die Sie zuvor im XDP-Dashboard erstellt haben sollten. Die Benutzer-ID, die Sie für ein bestimmtes Gamepad durch Aufruf der xboxlive_user_for_pad() Funktion und Sichtbarkeit eine der folgenden Konstanten sein:

Xbox Match-Sichtbarkeit Konstante
Konstante Beschreibung
xboxlive_match_visibility_open gibt an, dass die anderen Teilnehmer der Sitzung beitreten können.
xboxlive_match_visibility_private legt fest, dass die Sitzung für Benutzer, die keine Sitzungsmitglieder sind, nicht sichtbar ist.
xboxlive_match_visibility_usetemplate gibt an, dass der in der Sitzungsvorlage verwendete Wert verwendet werden soll (dies sollte normalerweise verwendet werden, da Sie den Wert einer Vorlage nicht überschreiben können, wenn in XDP ein Wert festgelegt wurde).

Die Funktion gibt einen eindeutigen Request-ID-Wert zurück, der dann zur Identifizierung des richtigen sozialen asynchronen Ereignisses für diese Funktion verwendet werden kann. Dieses Ereignis wird ausgelöst, wenn die Sitzung erstellt wurde, und enthält eine DS-Map in der Variablen async_load mit den folgenden Schlüssel/Wertpaaren:

  • "requestid" - enthält die von der aufrufenden Funktion zurückgegebene Anfrage-ID
  • "status" - enthält die Zeichenkette "session_created", um Sie darüber zu informieren, welche Art von Ereignis ausgelöst wurde.
  • "sessionid" - enthält die Sitzungs-ID, oder -1 im Falle eines Fehlers
  • "error" - enthält 0 bei Erfolg oder -1 im Falle eines Fehlers
  • "correlationid" - das Sitzungshandle für die Korrelation mit der Funktion xboxlive_matchmaking_join_session()

Syntax:

xboxlive_matchmaking_create(user_id, visibility, template, hopper, sdatemplate, [matchattributes]);

Argument Typ Beschreibung
user_id Xbox-Benutzer-ID The user ID pointer to use
visibility Xbox Match-Sichtbarkeit Konstante One of the constants listed above
template Zeichenfolge The name of the session template
hopper Zeichenfolge The name of the matchmaking hopper
sdatemplate Zeichenfolge The name of the secure device association template
matchattributes Real OPTIONAL Die Trefferattribute

Rückgabe:

Async Request ID

Beispiel:

var userID = xboxlive_user_for_pad(global.PadIndex);
result = xboxlive_matchmaking_create(userID, xboxlive_match_visibility_usetemplate, "MatchTicketSession", "MatchTicketHopper", "PeerServerTraffic");

Die obige Funktion ruft die Benutzer-ID für den Benutzer auf dem angegebenen Gamepad ab und initialisiert dann eine Matchmaking-Sitzung für ihn.
