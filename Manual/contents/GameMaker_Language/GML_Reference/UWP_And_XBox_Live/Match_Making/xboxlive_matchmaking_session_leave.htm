xboxlive_matchmaking_session_leave

Diese Funktion bewirkt, dass der Benutzer, der die angegebene Sitzung erstellt oder gefunden hat, diese verlässt. Sie können den Sitzungs-ID-Zeiger aus der async_load DS-Map, die im Asynchronous Social Event zurückgegeben wird, wenn Sie eine Sitzung erstellt oder gefunden haben (siehe xboxlive_matchmaking_create() für weitere Details).

Syntax:

xboxlive_matchmaking_session_leave(session_id);

ArgumentTyp Beschreibung
session_id The Session ID pointer of the session to leave.

Rückgabe:

Beispiel:

if gamepad_button_check_pressed(0, gp_start)
{
    xboxlive_matchmaking_session_leave(global.SessionID);
}

Die obige code prüft, ob eine Taste des Gamepads gedrückt wurde, und wenn dies der Fall ist, wird der Benutzer von der jeweiligen Matchmaking-Sitzung abgemeldet.
