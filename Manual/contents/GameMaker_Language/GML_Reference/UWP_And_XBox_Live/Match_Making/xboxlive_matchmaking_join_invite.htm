xboxlive_matchmaking_join_invite

Diese Funktion kann aufgerufen werden, um eine Beitrittseinladung zu verarbeiten, nachdem Sie erfolgreich die Funktion xboxlive_matchmaking_start(). Sie geben die lokale UserID und die Werte der Einladung (ID und Host-ID) die beim Empfang der Einladung zurückgegeben wurden, zusammen mit dem Namen der von Ihnen erstellten Sitzungsvorlage (eine Zeichenkette). Der Benutzer wird dann der Sitzung beitreten, zu der er eingeladen wurde, und Sie erhalten asynchrone soziale Ereignisse wie als ob Sie aufgerufen hätten xboxlive_matchmaking_find().

Syntax:

xboxlive_matchmaking_join_invite(user_who_received_invite, invitation_id, invitation_host, sda_template_name);

ArgumentTyp Beschreibung
user_who_received_invite The local user ID pointer.
invitation_id The invitation ID.
invitation_host The invitation host ID.
sda_template_name The session template name.

Rückgabe:

Beispiel:

var userID = xboxlive_user_for_pad(global.PadIndex);
xboxlive_matchmaking_join_invite(userID, global.InviteID, global.InviteHost, global.SessionTemplate);

Die obige Funktion ruft die Benutzer-ID für das angegebene Gamepad ab und versucht dann, der Sitzung beizutreten, zu der sie eingeladen wurde.
