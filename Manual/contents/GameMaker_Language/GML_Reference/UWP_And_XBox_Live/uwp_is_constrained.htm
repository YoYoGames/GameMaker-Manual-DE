uwp_is_constrained

Diese Funktion gibt true zurück, wenn das Spiel in einem ressourcenbeschränkten Modus läuft, oder false, wenn es normal läuft. Der ressourcenbeschränkte Modus kann aktiviert werden, wenn das Spiel nicht fokussiert ist, und in diesem Modus verfügt das Spiel über reduzierte GPU-Ressourcen. Die Reaktion darauf sollte normalerweise darin bestehen, das Spiel anzuhalten (da der Benutzer keine Kontrolle mehr hat). Auch die Deaktivierung besonders belastender Effekte kann eine gute Idee sein, wenn in diesem Modus eine Verlangsamung beobachtet wird. Beachten Sie, dass im Gegensatz zur uwp_is_suspending() Funktion, diese Funktion kontinuierlich true zurückgibt, solange sich das Spiel im eingeschränkten Modus befindet (und nicht nur einmal, wenn es zum ersten Mal passiert).

Syntax:

uwp_is_constrained();

Rückgabe:

Beispiel:

if xboxliveis_constrained()
{
    if !instance_exists(obj_PauseMenu)
    {
        instance_create(0,0,obj_PauseMenu);
    }
}

Die obige code prüft, ob die Anwendung eingeschränkt ist, und wenn ja, erstellt sie eine Instanz eines Objekts (das zum Anhalten des Spiels verwendet wird).
