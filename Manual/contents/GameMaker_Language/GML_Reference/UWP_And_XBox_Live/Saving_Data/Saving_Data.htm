Speichern von Daten

Die folgenden Funktionen beziehen sich speziell auf das Speichern von Daten mit dem UWP-Export für Xbox One und die Nutzung der XBox Live-Funktionen.

Während die Speicherdaten, die sich auf den Spielerfortschritt beziehen, immer einem bestimmten Benutzer zugeordnet sein müssen, können Sie allgemeine Daten im Bereich "Maschinenspeicher" speichern, indem Sie den Wert -1 als Benutzer argument angeben. Dieser Bereich ist titelspezifisch, aber nicht mit einem bestimmten Benutzer verknüpft.

HINWEIS: Keine gespeicherte Datei darf größer als 16 Megabyte sein!

Die folgenden Funktionen sind für den Umgang mit Xbox Live-Speicherdaten gedacht:

  • xboxlive_set_savedata_user
  • xboxlive_get_savedata_user
  • xboxlive_get_file_error
