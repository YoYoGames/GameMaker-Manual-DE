xboxlive_get_file_error

Diese Funktion gibt eine Konstante zurück (siehe unten), die bei der Fehlersuche hilft, damit Sie in Ihrem Spiel code entsprechende Maßnahmen ergreifen können. Wenn eine der Dateifunktionen fehlschlägt, können Sie die Funktion aufrufen, um genau herauszufinden, was der Fehler war.

HINWEIS: Bei Verwendung der asynchronen Speicher-/Ladefunktionen hat die async_load map einen zusätzlichen Schlüssel, "error", der auch eine der Konstanten (siehe unten) enthält.

Syntax:

xboxlive_get_file_error();

Rückgabe:

Xbox File Error Constant

Xbox File Error Constant
Konstante Beschreibung
xboxlive_fileerror_outoflocalstorage Wenn dies während des Speicherns auftritt, bedeutet dies, dass der gesamte verfügbare Speicherplatz aufgebraucht ist und der Benutzer das (vom Betriebssystem unterbreitete) Angebot, Speicherplatz freizugeben, abgelehnt hat. Das Spiel sollte dem Benutzer bestätigen, dass er beabsichtigt, das Spiel ohne Speichern fortzusetzen. Wenn der Benutzer weiterhin speichern möchte, sollte das Spiel den Speichervorgang wiederholen.
xboxlive_fileerror_quotaexceeded In diesem Fall sollte das Spiel den Benutzer darauf hinweisen, dass er keine weiteren Daten speichern kann, bis er etwas Speicherplatz freigibt, und ihm dann eine Möglichkeit anbieten, dies zu tun.
xboxlive_fileerror_noerror Die Funktion zum Speichern oder Laden von Dateien wurde ohne Probleme ausgeführt.

Beispiel:

buffer_save(buff, "SaveGame.bin");
if (xboxlive_get_file_error() != xboxlive_fileerror_noerror)
{
    show_debug_message("Save failed. Error code = " + string(xboxlive_get_file_error()));
}

Die obige code prüft, ob ein Fehler beim Speichern aufgetreten ist, und wenn ja, wird der Fehler code auf der Konsole ausgegeben.
