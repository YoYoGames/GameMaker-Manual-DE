xboxlive_set_savedata_user

Diese Funktion legt fest, dass zukünftige Datei-Operationen, die im Speicherbereich des Spiels stattfinden (d.h. alle Schreibvorgänge und Lesevorgänge von Dateien, die sich nicht im Bundle-Bereich befinden), mit dem angegebenen Benutzer-ID-Zeiger verknüpft werden. Diese Funktion kann so oft wie nötig aufgerufen werden, um Speicherdaten an den entsprechenden Benutzer umzuleiten, oder Sie können die Konstante pointer_null verwenden, um in den allgemeinen Maschinenspeicherbereich zu speichern.

Syntax:

xboxlive_set_savedata_user(user_id);

Argument Typ Beschreibung
user_id Xbox-Benutzer-ID oder pointer_null The user ID (a pointer) to set for saving, or pointer_null

Rückgabe:

N/A

Beispiel:

if (xboxlive_get_savedata_user() != user_id[0])
{
    xboxlive_set_savedata_user(user_id[0]);
}

Die obige code prüft, ob ein Benutzer derzeit als Speicherziel zugewiesen ist, und wenn nicht, wird er zugewiesen.
