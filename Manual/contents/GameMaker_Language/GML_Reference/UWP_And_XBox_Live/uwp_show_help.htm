uwp_show_help

Mit dieser Funktion wird die Xbox One-Hilfe-App gestartet, die wiederum die Hilfedatei Ihres Spiels anzeigt (eine Datei im HTML-Format, die Sie zuvor in das XBox Developer Portal hochgeladen haben). Beachten Sie, dass beim Aufruf dieser Funktion eine Benutzer-ID angegeben wird, damit die Gebietsschema-Informationen des Benutzers bei der Anzeige verwendet werden können. Sie können die Benutzer-ID mit den folgenden Funktionen abrufen xboxlive_get_user(), xboxlive_get_activating_user()oder uwp_license_trial_user().

Syntax:

uwp_show_help(user_id);

ArgumentTyp Beschreibung
user_id The User ID pointer to open the helpfile for.

Rückgabe:

Beispiel:

if gamepad_button_check_pressed(0, gp_start)
{
    if uwp_license_trial_version()
    {
        var _uid = uwp_license_trial_user;
        uwp_show_help(_uid);
    }
}

Die obige code prüft, ob eine Schaltfläche gedrückt wird, und überprüft dann, ob die Anwendung mit einer Testlizenz ausgeführt wird. Ist dies der Fall, wird die Benutzer-ID für die Lizenz ermittelt und die Hilfedatei für diesen Benutzer geöffnet.
