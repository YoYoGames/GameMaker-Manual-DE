uwp_wurde_vom_Benutzer_geschlossen

Mit dieser Funktion können Sie herausfinden, ob das Spiel zuvor vom Benutzer während des Spielens beendet wurde (d.h. ob es korrekt und nicht vom System beendet wurde). Wenn dies der Fall ist, gibt die Funktion true zurück, andernfalls gibt sie false zurück.

Syntax:

uwp_was_closed_by_user();

Rückgabe:

Beispiel:

if uwp_was_closed_by_user()
{
    global.GameLoaded = false;
}

Die obige code prüft, ob die Anwendung zuvor vom Benutzer beendet wurde, und wenn ja, setzt sie eine globale Variable.
