xboxlive_stats_remove_user

Diese Funktion kann verwendet werden, um einen bestimmten Benutzer aus dem Statistik-Manager zu entfernen (abzumelden), wobei die Statistikdaten in die Live-Datenbank server übertragen werden. Laut der XBox-Dokumentation muss das Spiel den Benutzer nicht aus dem Statistik-Manager entfernen, da das XBox-Betriebssystem die Statistiken ohnehin in regelmäßigen Abständen leert. Sie geben die Benutzer-ID an, die (zum Beispiel) von der Funktion xboxlive_get_user()zurückgegeben wird, und die Funktion gibt -1 zurück, wenn ein Fehler auftrat oder die Benutzer-ID ungültig ist, oder einen beliebigen anderen Wert, wenn die Funktion erfolgreich aufgerufen wurde.

HINWEIS: Das Entfernen des Benutzers kann zu einem Fehler führen, wenn die für den Benutzer festgelegten Statistiken ungültig sind (z. B. wenn die Statistiknamen nicht alphanumerische Zeichen enthalten).

Die Funktion erzeugt einen Callback, der ein asynchrones Systemereignis auslöst. Dieses Ereignis hat die spezielle DS-Map async_load die dann nach den folgenden Schlüsseln geparst werden sollte:

  • "id" - wird die Konstante halten achievement_stat_event
  • "eventname" - enthält die Zeichenfolge "LocalUserRemoved".
  • "userid" - Die mit der Anfrage verbundene Benutzer-ID
  • "error" - 0 bei Erfolg, ein anderer Wert, wenn ein Fehler aufgetreten ist
  • "errormessage" - Eine Zeichenkette mit einer Fehlermeldung, falls eine vorhanden ist

Beachten Sie, dass Sie, wenn Sie die Statistikdaten jederzeit in die Live-Datenbank server flushen möchten, ohne den Benutzer zu entfernen, die folgende Funktion verwenden können xboxlive_stats_flush_user().

Syntax:

xboxlive_stats_remove_user(user_id);

ArgumentTyp Beschreibung
user_id The user ID (a pointer) to remove

Rückgabe:

Beispiel:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_remove_user(user_id[i]);
}

Die obige code loop durchläuft die gespeicherten Benutzerkonto-IDs und entfernt (hebt die Registrierung auf) jeden Benutzer aus dem Statistikmanager.
