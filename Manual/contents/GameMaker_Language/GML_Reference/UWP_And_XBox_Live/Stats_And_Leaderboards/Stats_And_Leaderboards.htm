Statistiken und Ranglisten

Xbox One Stats (Achievements und Leaderboards) sind bei der Verwendung von UWP funktionsfähig und haben einige spezielle Funktionen für den Umgang mit ihnen, obwohl sie auch einige der standardmäßig eingebauten Funktionen für Leaderboards verwenden. Das 2017er Xbox Live-Statistiksystem vereinfacht das gesamte Verfahren zur Erstellung von Achievements und Bestenlisten, indem es Ihnen ermöglicht, die verschiedenen Systeme innerhalb von GameMaker zu erstellen und zu verwenden. Der Hauptunterschied zwischen dem (titelverwalteten) System von 2017 und früheren Systemen besteht darin, dass Sie nicht mehr alle Statistiken, die Sie im Spiel verwenden, im Partner-Center definieren müssen, und dass Sie die Freischaltung von Errungenschaften auch nicht an ein Statistikereignis/eine Regel binden müssen (da sie stattdessen durch einen direkten Funktionsaufruf freigeschaltet werden).

HINWEIS Obwohl für dieses System keine Manifestdatei erforderlich ist, können Sie dennoch einige Statistiken auf XDP/Partner Center einrichten. Die dort eingerichteten Statistiken gelten als "Featured Stats ", d. h. Sie können globale Ranglisten einrichten und die Statistiken werden auf der Dashboard-Seite des Spiels angezeigt. Weitere Informationen finden Sie in diesem Artikel von Microsoft.

Bevor Sie eine der unten aufgeführten Statistikfunktionen aufrufen, müssen Sie sich vergewissern, dass der Benutzer angemeldet ist (siehe die oben aufgeführten Benutzerfunktionen), dann richten Sie das Statistiksystem ein und fügen den Benutzer mit diesen Funktionen dazu:

  • xboxlive_stats_setup (Nur für ereignisbasierte Statistiken (2013))
  • xboxlive_stats_add_user

Die folgenden Funktionen werden mit der titelverwalteten Statistik 2017 verwendet:

  • xboxlive_stats_remove_user
  • xboxlive_stats_flush_user
  • xboxlive_stats_set_stat_real
  • xboxlive_stats_set_stat_int
  • xboxlive_stats_set_stat_string
  • xboxlive_stats_delete_stat
  • xboxlive_stats_get_stat_names
  • xboxlive_stats_get_stat
  • xboxlive_stats_get_leaderboard
  • xboxlive_stats_get_social_leaderboard
  • xboxlive_erfolge_anzeigen_erfolge
  • xboxlive_Erfolg_laden_Freunde
  • xboxlive_erfolge_laden_leaderboard
  • xboxlive_achievements_set_progress

Die folgenden Funktionen werden mit dem (ereignisbasierten) Statistiksystem 2013 verwendet:

  • xboxlive_get_stats_for_user
  • xboxlive_read_player_leaderboard
  • xboxlive_fire_event
