xboxlive_stats_flush_user

Diese Funktion kann verwendet werden, um die Statistikdaten für einen bestimmten Benutzer aus dem Statistik-Manager in die Live-Anwendung server zu übertragen und so sicherzustellen, dass die server mit den aktuellen Werten auf dem neuesten Stand ist. Laut der XBox-Dokumentation sollten Entwickler darauf achten, diese Funktion nicht zu oft aufzurufen, da die XBox die Anfragen begrenzt und das XBox-Betriebssystem die Statistiken ohnehin alle 5 Minuten automatisch aktualisiert.

HINWEIS: Das Entfernen des Benutzers kann zu einem Fehler führen, wenn die für den Benutzer festgelegten Statistiken ungültig sind (z. B. wenn die Statistiknamen nicht alphanumerische Zeichen enthalten).

Um die Funktion zu verwenden, geben Sie die Benutzer-ID an, die z. B. von der Funktion xboxlive_get_user()zurückgegeben wird, und einen Prioritätswert (1 für hohe Priorität und 0 für niedrige Priorität). Die Funktion gibt -1 zurück, wenn ein Fehler auftrat oder die Benutzer-ID ungültig ist, oder einen beliebigen anderen Wert, wenn die Funktion erfolgreich aufgerufen wurde. Die Funktion erzeugt außerdem einen Callback, der ein asynchrones Systemereignis auslöst. Dieses Ereignis enthält die spezielle DS-Map async_load, die dann nach den folgenden Schlüsseln geparst werden sollte:

  • "id" - wird die Konstante halten achievement_stat_event
  • "eventname" - enthält die Zeichenfolge "StatisticUpdateComplete".
  • "userid" - Die mit der Anfrage verbundene Benutzer-ID
  • "error" - 0 bei Erfolg, ein anderer Wert, wenn ein Fehler aufgetreten ist
  • "errormessage" - Eine Zeichenkette mit einer Fehlermeldung, falls eine vorhanden ist

Beachten Sie, dass die Statistiken automatisch geleert werden, wenn Sie einen Benutzer aus dem Statistikmanager entfernen. Siehe xboxlive_stats_remove_user() für weitere Informationen.

Syntax:

xboxlive_stats_flush_user(user_id, priority);

Argument Typ Beschreibung
user_id Xbox-Benutzer-ID The user ID (a pointer) to use
priority Real The prority value (0 or 1)

Rückgabe:

Real

Beispiel:

for(var i = 0; i < array_length(user_id); ++i;)
{
    xboxlive_stats_flush_user(user_id[i], 0);
}

Die obige code loops durchläuft die gespeicherten Benutzerkonto-IDs und leert dann die Statistikmanagerdaten für jeden Benutzer.
