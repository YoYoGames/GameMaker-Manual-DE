xboxlive_stats_add_user

Diese Funktion kann verwendet werden, um einen bestimmten Benutzer-ID-Zeiger zum Statistik-Manager hinzuzufügen. Dies muss geschehen, bevor Sie eine der anderen Statistikfunktionen verwenden, um das Spiel automatisch mit XBox Live server zu synchronisieren und die neuesten Werte abzurufen. Sie geben die Benutzer-ID an, die (zum Beispiel) von der Funktion xboxlive_get_user()zurückgegeben wird, und die Funktion gibt -1 zurück, wenn ein Fehler aufgetreten ist oder die Benutzer-ID ungültig ist, oder einen beliebigen anderen Wert, wenn die Funktion erfolgreich aufgerufen wurde.

Die Funktion erzeugt einen Callback, der ein asynchrones Systemereignis auslöst. Dieses Ereignis hat die spezielle DS-Map async_load die dann nach den folgenden Schlüsseln geparst werden sollte:

  • "id" - wird die Konstante halten achievement_stat_event
  • "eventname" - Enthält die Zeichenfolge "LocalUserAdded".
  • "userid" - Die mit der Anfrage verbundene Benutzer-ID
  • "error" - 0 bei Erfolg, ein anderer Wert, wenn ein Fehler aufgetreten ist
  • "errormessage" - Eine Zeichenkette mit einer Fehlermeldung, falls eine vorhanden ist

Syntax:

xboxlive_stats_add_user(user_id);

ArgumentTyp Beschreibung
user_id The user ID (a pointer) to add

Rückgabe:

Beispiel:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_add_user(user_id[i]);
}

Die obige code wird die Anzahl der verfügbaren Benutzerkonten ermitteln, dann loop durchgehen und die Konto-ID einem Array zuweisen, sowie den Benutzer beim Statistikmanager registrieren.
