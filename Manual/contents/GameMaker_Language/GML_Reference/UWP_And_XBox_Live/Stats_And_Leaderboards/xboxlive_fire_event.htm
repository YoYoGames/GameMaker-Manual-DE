xboxlive_feuer_ereignis

Diese Funktion kann verwendet werden, um ein Statusereignis auszulösen. Der "event_name" argument ist der Name des auszulösenden Ereignisses, wie er in der XDP-Konsole für Ihr Spiel definiert ist, und die folgenden zusätzlichen Parameter hängen auch davon ab, was Sie für den Status eingerichtet haben. Die Funktion gibt 0 zurück, wenn das Ereignis gesendet wurde (und von server empfangen/verarbeitet werden sollte), oder -1, wenn ein Fehler aufgetreten ist (d.h. Ihr Ereignis wurde nicht eingerichtet, da die im Projekt enthaltene Ereignis-Manifestdatei eine andere Zahl angibt).

Syntax:

xboxlive_fire_event(event_name, [stat_parameters...]);

Argument Typ Beschreibung
event_name Zeichenfolge The name of the event to fire.
stat_parameters Variabel OPTIONAL Zusätzliche Parameter, die dem Ereignis hinzugefügt werden sollen.

Rückgabe:

Real

Beispiel:

var _user = xboxlive_get_user(0);
var _uid = xboxlive_user_id_for_user(_user);
result = xboxlive_fire_event("PlayerSessionStart", _uid, global.guid_str, 0, 42, 42);

Die obige code erhält einige Informationen über den Benutzer und feuert dann ein stat-Ereignis zusammen mit diesen (und zusätzlichen) Daten ab.
