xboxlive_stats_delete_stat

Mit dieser Funktion können Sie eine Statistik für die angegebene Benutzerkennung aus dem Statistikmanager löschen. Sie geben die Benutzer-ID an, wie sie z.B. von der Funktion xboxlive_get_user()zurückgegeben wird, und dann die zu löschende Statistikzeichenkette. Dies löscht den Wert der Statistik und entfernt ihn aus dem Statistik-Manager, d.h. er wird nicht mehr von den Funktionen xboxlive_stats_get_stat_names() oder zurückgegeben wird. xboxlive_stats_get_stat().

Die Funktion gibt -1 zurück, wenn ein Fehler auftrat oder die Benutzer-ID ungültig ist, oder einen anderen Wert, wenn die Funktion erfolgreich aufgerufen wurde

Syntax:

xboxlive_stats_delete_stat(user_id, stat);

ArgumentTyp Beschreibung
user_id The user ID pointer to delete the stat of
stat The statistic to delete (a string)

Rückgabe:

Beispiel:

for(var i = 0; i < xboxlive_get_user_count(); i++;)
{
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_delete_stat(user_id[i], "HighScore");
}

Die obige code loops durchläuft die verbundenen Benutzer und löscht für jeden die angegebene Statistik aus dem Statistikmanager.
