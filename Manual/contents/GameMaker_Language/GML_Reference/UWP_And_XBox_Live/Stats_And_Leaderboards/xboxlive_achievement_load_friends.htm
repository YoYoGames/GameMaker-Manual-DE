xboxlive_Erfolg_laden_Freunde

Diese Funktion sendet eine Anfrage an server, um Informationen über alle Freunde des angemeldeten Benutzers zu erhalten, und löst einen Callback Social Asynchronous Event aus, der die async_load Map mit den relevanten Schlüssel/Wert-Paaren enthält. Der id-Schlüssel dieses DS Map wird verwendet, um den richtigen Rückruf zu identifizieren (es kann mehr als eine Auslösefunktion für ein bestimmtes asynchrones Ereignis geben), und wird mit der Konstante xboxlive_achievement_friends_info sowie einer Reihe anderer Schlüssel/Wertpaare für jeden Freund gepaart. Der genaue Inhalt der Map ist unten dargestellt:

  • "id" - Für diese Funktion sollte es sein xboxlive_achievement_friends_info
  • "FriendN" - Der Name des Freundes, wobei "N" ein ganzzahliger Wert ist, der der Position des Freundes in der Freundesliste entspricht.
  • "FriendidN" - Die eindeutige Benutzerkennung des Freundes, "N".

Syntax:

xboxlive_achievement_load_friends();

Rückgabe:

N/A

Erweitertes Beispiel:

Das folgende code würde wahrscheinlich aufgerufen werden, nachdem sich der Spieler in sein Spielkonto eingeloggt hat, um eine Liste aller Freunde des Spielers zu erhalten:

xboxlive_achievement_load_friends();

Dies sendet eine Anfrage nach den Informationen über die Freunde des Benutzers und erzeugt einen asynchronen Rückruf mit der speziellen async_load DS-Map, die die folgenden Daten enthält:

var ident = ds_map_find_value(async_load, "id");
if (ident == xboxlive_achievement_friends_info)
{
    var numfriends = ds_map_find_value(async_load, "numfriends");
    global.numfriends = numfriends;
    for(var i=0; i < numfriends; i++)
    {
        global.friendname[i] = ds_map_find_value(async_load, "Friend" + string(i));
        global.friendid[i] = ds_map_find_value(async_load, "Friendid" + string(i));
    }
}

Die obige code prüft die zurückgegebene DS Map in der sozialen asynchronen Ereignis und wenn seine "id" mit der Konstante überprüft wird, es dann loops durch die Karte Speicherung aller verschiedenen Werte in einer Reihe von Arrays.
