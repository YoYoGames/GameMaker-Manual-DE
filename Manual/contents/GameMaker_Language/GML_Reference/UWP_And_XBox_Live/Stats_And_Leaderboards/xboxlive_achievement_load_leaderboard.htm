xboxlive_erfolge_laden_leaderboard

Diese Funktion sendet eine Anfrage an server, um Informationen über das angegebene Leaderboard zu erhalten. Das Leaderboard muss zuvor auf dem Entwickler-Dashboard für Ihr Spiel erstellt worden sein und die angegebene ID ist diejenige, die Sie als erste argument der Funktion angeben müssen. Es wird ein Callback Social Asynchronous Event ausgelöst, das die async_load Karte mit den relevanten Schlüssel/Wert-Paaren gefüllt wird. Der id-Schlüssel dieser DS Map wird verwendet, um den richtigen Rückruf zu identifizieren (es kann mehr als eine Auslösefunktion für ein bestimmtes asynchrones Ereignis geben), und wird mit der Konstante xboxlive_achievement_leaderboard_info sowie einer Reihe anderer Schlüssel/Wertpaare für jeden Spieler gepaart. Der genaue Inhalt der Karte ist unten dargestellt:

  • "id" - Für diese Funktion sollte es sein xboxlive_achievement_leaderboard_info
  • "leaderboardid" - Die eindeutige ID für das Leaderboard, wie sie auf dem Dashboard des Anbieters definiert ist.
  • "numentries" - Die Anzahl der Einträge in der Rangliste, die Sie erhalten haben.
  • "PlayerN" - Der Name des Spielers, wobei "N" ein ganzzahliger Wert ist, der seiner Position in der Liste der Ranglisteneinträge entspricht.
  • "PlayeridN" - Die eindeutige Benutzerkennung des Spielers, "N".
  • "RangN" - Der Rang des Spielers "N" innerhalb der Rangliste.
  • "ScoreN" - Der Punktestand des Spielers "N".

Sie können den genauen Teil der Rangliste angeben, der angezeigt werden soll, indem Sie einen minimalen und maximalen Indexwert angeben. Außerdem können Sie mit den folgenden Filterkonstanten festlegen, dass die Ergebnisse so gefiltert werden, dass nur die von Ihren Freunden oder die von allen Spielern geposteten Ergebnisse angezeigt werden:

Xbox Live Errungenschaftsfilter konstant
Konstante Beschreibung
xboxlive_achievement_filter_friends_only  Erhalten Sie nur die Spielstände der Freunde des Spielers.
xboxlive_achievement_filter_all_players  Alle Spielstände für alle Spieler abrufen.

Syntax:

xboxlive_achievement_load_leaderboard(ident, minindex, maxindex, filter)

Argument Typ Beschreibung
ident Zeichenfolge The unique ID of the leaderboard as shown on the developer dashboard.
minindex Real The starting index value to get the leaderboard data from.
maxindex Real The maximum index value to get the leaderboard data from.
filter Xbox Live Errungenschaftsfilter konstant Whether to filter the results to those that are on your friends list or to all players (see the Description above)

Rückgabe:

N/A

Erweitertes Beispiel:

Das folgende code würde wahrscheinlich aufgerufen werden, nachdem sich der Spieler in sein Spielkonto eingeloggt hat, um eine Liste der Ranglistenpositionen aller Freunde für das Spiel zu erhalten:

xboxlive_achievement_load_leaderboard("CgkIs9_51u0PEAIQBw", 1, 100, xboxlive_achievement_filter_friends_only);

Dies sendet eine Anfrage für die Informationen über das angegebene Leaderboard und erzeugt einen asynchronen Rückruf mit der speziellen async_load DS-Karte, die die folgenden Daten enthält:

var ident = ds_map_find_value(async_load, "id");
if (ident == xboxlive_achievement_leaderboard_info)
{
    var lbid = ds_map_find_value(async_load,"leaderboardid");
    if lbid == "CgkIs9_51u0PEAIQBw" || lbid == "leaderboard1id"
    {
        global.numentries = ds_map_find_value(async_load,"numentries");
        for(var i = 0; i < numentries; i++;)
        {
            global.playername[i] = ds_map_find_value(async_load, "Player" + string(i));
            global.playerid[i] = ds_map_find_value(async_load, "Playerid" + string(i));
            global.playerrank[i] = ds_map_find_value(async_load, "Rank" + string(i));
            global.playerscore[i] = ds_map_find_value(async_load, "Score" + string(i));
        }
    }
}

Die obige code prüft die zurückgegebene DS Map im sozialen asynchronen Ereignis, und wenn ihre "id" mit der zu prüfenden Konstante übereinstimmt, prüft sie dann, ob die gesendete Rangliste mit dem ursprünglichen Funktionsaufruf übereinstimmt, bevor sie die Karte in einer Schleife durchläuft und alle verschiedenen Werte in einer Reihe von Arrays speichert.
