xboxlive_stats_set_stat_string

Diese Funktion kann verwendet werden, um eine Statistik für die angegebene Benutzer-ID in eine Zeichenkette zu verwandeln. Sie geben die Benutzer-ID an, wie sie (zum Beispiel) von der Funktion xboxlive_get_user()zurückgegeben wird, dann die zu setzende Statistik(wenn die Statistik nicht bereits existiert, wird eine neue Statistik erstellt und auf den angegebenen Wert gesetzt) und dann die Zeichenkette, auf die die Statistik gesetzt werden soll. Beachten Sie, dass der Name der Statistik nur alphanumerisch sein darf und keine Symbole oder Leerzeichen enthalten darf, während der Wert der Zeichenkette, den Sie setzen, Leer- und Sonderzeichen enthalten darf.

Daher müssen Sie selbst entscheiden, ob die Zeichenfolge aktualisiert werden soll oder nicht, indem Sie die aktuelle Zeichenfolge mit der neuen Zeichenfolge vergleichen, bevor Sie sie setzen.

Die Funktion gibt -1 zurück, wenn ein Fehler auftrat oder die Benutzer-ID ungültig ist, oder einen anderen Wert, wenn die Funktion erfolgreich aufgerufen wurde.

Syntax:

xboxlive_stats_set_stat_string(user_id, stat, value);

Argument Typ Beschreibung
user_id Xbox-Benutzer-ID The user ID pointer to set the stat for
stat Zeichenfolge The statistic to set (a string)
value Zeichenfolge The value to set the stat to (a string)

Rückgabe:

Real

Beispiel:

xboxlive_stats_set_stat_string(p_user_id, "LevelName", global.LevelName);

Die obige code setzt den Status "LevelName" auf die Zeichenkette, die in der angegebenen globalen Variablen gespeichert ist.
