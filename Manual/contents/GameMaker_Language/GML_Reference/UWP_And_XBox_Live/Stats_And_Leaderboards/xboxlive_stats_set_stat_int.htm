xboxlive_stats_set_stat_int

Diese Funktion kann verwendet werden, um den Wert einer Statistik für die angegebene Benutzer-ID zu setzen. Sie geben die Benutzer-ID an, wie sie z.B. von der Funktion xboxlive_get_user()zurückgegeben wird, und dann die Zeichenfolge die gesetzt werden soll(wenn die Zeichenkette noch nicht existiert, wird eine neue Statistik erstellt und auf den angegebenen Wert gesetzt) und einen Wert (eine ganze Zahl), auf den die Statistik gesetzt werden soll. Beachten Sie, dass der Name der Statistik nur alphanumerisch sein darf, ohne Symbole oder Leerzeichen.

Beim Einstellen des Statistikwertes wird jeder vorherige Wert überschrieben, daher müssen Sie selbst bestimmen, ob der Statistikwert aktualisiert werden soll oder nicht (d. h. überprüfen, ob der Highscore tatsächlich der höchste ist), indem Sie den aktuellen Statistikwert mit dem neuen Wert, bevor Sie ihn setzen.

Die Funktion gibt -1 zurück, wenn ein Fehler auftrat oder die Benutzer-ID ungültig ist, oder einen anderen Wert, wenn die Funktion erfolgreich aufgerufen wurde.

Syntax:

xboxlive_stats_set_stat_int(user_id, stat, value);

ArgumentTyp Beschreibung
user_id The user ID pointer to set the stat for
stat The statistic to set (a string)
value The value to set the stat to (an integer)

Rückgabe:

Beispiel:

var _val = xboxlive_stats_get_stat(p_user_id, "GamesPlayed");
xboxlive_stats_set_stat_int(p_user_id, "GamesPlayed", ++_val);

Die obige code ruft den Wert für die Statistik "GamesPlayed" ab und setzt dann die gleiche Statistik auf diesen Wert plus 1.
