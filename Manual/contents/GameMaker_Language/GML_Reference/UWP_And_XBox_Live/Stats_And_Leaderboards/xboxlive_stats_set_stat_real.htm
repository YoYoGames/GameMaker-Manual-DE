xboxlive_stats_set_stat_real

Diese Funktion kann verwendet werden, um den Wert einer Statistik für die angegebene Benutzer-ID zu setzen. Sie geben die Benutzer-ID an, wie sie z.B. von der Funktion xboxlive_get_user()zurückgegeben wird, dann die zu setzende Statistik(wenn die Statistik nicht bereits existiert, wird eine neue Statistik erstellt und auf den angegebenen Wert gesetzt) und einen Wert (einen Realwert), auf den die Statistik gesetzt werden soll. Beachten Sie, dass der Name der Statistik nur alphanumerisch sein darf, ohne Symbole oder Leerzeichen.

Beim Einstellen des Wertes werden alle vorherigen Werte überschrieben. Daher müssen Sie selbst bestimmen, ob der Wert aktualisiert werden soll oder nicht (d. h. überprüfen, ob die höchste Punktzahl tatsächlich die höchste ist), indem Sie den aktuellen Wert mit dem neuen Wert vergleichen, bevor Sie ihn einstellen.

Die Funktion gibt -1 zurück, wenn ein Fehler auftrat oder die Benutzer-ID ungültig ist, oder einen anderen Wert, wenn die Funktion erfolgreich aufgerufen wurde.

Syntax:

xboxlive_stats_set_stat_real(user_id, stat, value);

ArgumentTyp Beschreibung
user_id The user ID pointer to set the stat for
stat The statistic to set (a string)
value The value to set the stat to (a real)

Rückgabe:

Beispiel:

if (game_over == true)
{
    if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
    {
        var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
        xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
    }
}

Die obige code prüft eine Variable, und wenn sie true ist, dann wird der Wert der Statistik "PercentDone" geprüft. Wenn dieser Wert kleiner als 100 ist, wird ein Wert generiert und die Statistik auf diesen Wert gesetzt.
