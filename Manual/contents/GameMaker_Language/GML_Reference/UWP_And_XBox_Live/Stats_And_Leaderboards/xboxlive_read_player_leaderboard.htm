xboxlive_lese_spieler_leaderboard

Die Funktion gibt die Leaderboard-Informationen für den angegebenen Benutzernamen zurück. Die Bestenliste muss zuvor im XDP-Dashboard für Ihr Spiel erstellt worden sein, und der Aufruf der Funktion löst ein soziales asynchrones Ereignis aus, das die async_load Map mit den relevanten Schlüssel/Wert-Paaren enthält. Der "id"-Schlüssel dieser DS-Map wird verwendet, um den richtigen Rückruf zu identifizieren (es kann mehr als eine Auslösefunktion für ein bestimmtes asynchrones Ereignis geben), und IT wird mit der Konstante achievement_leaderboard_info sowie einer Reihe von anderen Schlüssel/Wert-Paaren gepaart sein. Der genaue Inhalt der Map ist unten dargestellt:

  • "id" - Für diese Funktion sollte es sein achievement_leaderboard_info
  • "leaderboardid" - Die eindeutige ID für das Leaderboard, wie sie im Dashboard des Anbieters definiert ist.
  • "numentries" - Die Anzahl der Einträge in der Rangliste, die Sie erhalten haben.
  • "SpielerN" - Der Name des Spielers, wobei "N" ein ganzzahliger Wert ist, der seiner Position in der Liste der Ranglisteneinträge entspricht.
  • "SpielerIDN" - Die eindeutige Benutzerkennung des Spielers, "N".
  • "RankN" - Der Rang des Spielers "N" innerhalb der Rangliste.
  • "ScoreN" - Der Punktestand des Spielers "N".


Die Funktion erfordert die Angabe einer der folgenden Konstanten, um die Filtereigenschaften festzulegen:

Konstante Beschreibung
xboxlive_achievement_filter_all_players Erhalten Sie alle Punkte.
xboxlive_achievement_filter_friends_only Erhalten Sie nur die Bewertungen von Freunden in aufsteigender Reihenfolge.
xboxlive_achievement_filter_favorites_only Nur die Bewertungen der Favoriten in aufsteigender Reihenfolge abrufen
xboxlive_achievement_filter_friends_alt Erhalten Sie nur die Bewertungen von Freunden in absteigender Reihenfolge.
xboxlive_achievement_filter_favorites_alt Erhalten Sie nur die Bewertungen der Favoriten in absteigender Reihenfolge.

Syntax:

xboxlive_read_player_leaderboard(leaderboard_name, user_name, num_items, friend_filter);

Argument Typ Beschreibung
leaderboard_name Zeichenfolge The name of the leaderboard to read, as set up on XDP.
user_name Xbox-Benutzer-ID The ID of the user to read from.
num_items Real The number of items to retrieve.
friend_filter Xbox Live Errungenschaftsfilter konstant One of the filter constants (see the description, above).

Rückgabe:

Real

Beispiel:

var _user = xboxlive_user_for_pad(0);
var _uid = xboxlive_user_id_for_user(_user);
xboxlive_read_player_leaderboard("MyLeaderboard", _uid, 10, achievement_filter_all_players);

Die obige code ruft die Benutzer-ID ab und verwendet diese dann, um alle Spielerdaten für die ersten 10 Plätze der angegebenen Rangliste abzufragen.
