xboxlive_stats_get_stat_names

Diese Funktion kann verwendet werden, um alle definierten Statistiken aus dem Statistikmanager für einen bestimmten Benutzer abzurufen. Sie geben die Benutzer-ID an, wie sie z.B. von der Funktion xboxlive_get_user()zurückgegeben wird, und die Funktion gibt ein Array von Strings zurück, das die Statistiken für den Benutzer enthält. Wenn ein Fehler auftritt oder der Benutzer keine Statistiken hat, wird das Array trotzdem zurückgegeben, allerdings mit einer Elementanzahl von 0.

Syntax:

xboxlive_stats_get_stat_names(user_id);

ArgumentTyp Beschreibung
user_id The user ID pointer to get the stat names of

Rückgabe:

Beispiel:

var _stat_str = xboxlive_stats_get_stat_names(user_id);
for (var i = 0; i < array_length(_stat_str); ++i;)
{
    xboxlive_stats_delete_stat(user_id, _stat_str[i]);
}

Die obige code ruft alle definierten Statistiken für den Benutzer-ID-Zeiger ab, der in der angegebenen Variable gespeichert ist, und loops durchsucht dann das Array und löscht die angegebene Statistik aus dem Statistikmanager.
