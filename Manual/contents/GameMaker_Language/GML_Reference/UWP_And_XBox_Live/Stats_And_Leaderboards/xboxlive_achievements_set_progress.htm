xboxlive_erfolge_setzen_fortschritt

Diese Funktion kann verwendet werden, um den Fortschritt einer Leistung zu aktualisieren. Sie geben die Benutzer-ID an, wie sie z. B. von der Funktion xboxlive_get_user()zurückgegeben wird, und dann die Achievement-Zeichenkette (dies ist die Achievement-ID, wie sie im XDP/Windows Dev Center zugewiesen wurde, als es erstellt wurde). Schließlich setzen Sie den Fortschrittswert auf einen Wert zwischen 0 und 100. Beachten Sie, dass das Achievement-System Aktualisierungen ablehnt, die niedriger sind als der aktuelle Fortschrittswert Wert liegen.

Syntax:

xboxlive_achievements_set_progress(user_id, achievement, progress);

ArgumentTyp Beschreibung
user_id The user ID of the user to set the achievement for
achievement The achievement to set (a string)
progress The progress value to set (from 0 to 100)

Rückgabe:

Erweitertes Beispiel:

Im Folgenden finden Sie ein erweitertes Beispiel für die Verwendung dieser Funktion. Zunächst würden Sie sie in einem Ereignis wie Raum Start oder Erstellen aufrufen:

xboxlive_stats_get_social_leaderboard(user_id, "GlobalTime", 20, 1, false, false);

Die obige code würde aufgerufen werden, um eine Liste aller globalen Leaderboard-Positionen für das Spiel zu erhalten, und wird einen sozialen asynchronen Ereignisaufruf zurück erzeugen, den wir auf folgende Weise behandeln würden:

var _progress = (global.Level / global.LevelMax) * 100;
xboxlive_achievements_set_progress(user_id, "Game_Completed", _progress);

Die obige code erstellt einen Prozentwert auf der Grundlage einiger globaler Variablen und verwendet ihn dann, um den Leistungsfortschritt für die Leistung "Game_Completed" festzulegen.
