xboxlive_get_stats_for_user

Diese Funktion kann verwendet werden, um Daten über bestimmte Statistiken von Xbox Live servers abzurufen. Sie geben die Benutzer-ID ein, wie sie von der Funktion xboxlive_get_user()zurückgegeben wird, dann die Service-Konfigurations-ID Ihres Spiels (wie auf der XDP-Konsole angezeigt) und schließlich die gewünschten Statistiken. Sie können bis zu 14 Statistiken abfragen, was jedoch nicht garantiert, dass Sie 14 Ergebnisse erhalten, da die Gesamtlänge der Anfrage begrenzt ist und die maximale Anzahl der Statistiken daher von der Länge der Namen der Statistiken selbst abhängt.

Die Funktion gibt 0 zurück, wenn die Statistikanfrage gesendet wurde, oder -1, wenn ein Fehler aufgetreten ist, und der Callback löst ein asynchrones Systemereignis aus. Dieses Ereignis hat die spezielle DS-Map async_load die dann nach dem folgenden Schlüssel geparst werden sollte:

  • "event_type" - sollte die Zeichenkette "stat_result" enthalten, wenn das Ereignis durch diese Funktion ausgelöst wurde

Bezieht sich der Ereignistyp auf diese Funktion, so gibt es auch den folgenden zusätzlichen Schlüssel:

  • "user" - enthält die Benutzer-ID für die angeforderte Statistik.

Es kann auch einen zusätzlichen Satz von Schlüssel-Wert-Paaren geben, wobei der Schlüssel der Name der angeforderten Statistik ist und der Wert der Wert für diese Statistik auf server. Beachten Sie, dass, wenn eine Statistik nie für diesen Benutzer erstellt wurde (weil er die Ereignisse, die sie setzen, nicht ausgelöst hat), kein Wert für diese Statistik zurückgegeben werden kann (ds_map_exists() kann verwendet werden, um die Statistiken in der Karte async_load zu überprüfen).

Schlägt die Anfrage fehl, weil die Anfragelänge zu groß ist, sollte in der Konsolenausgabe eine Meldung mit dem Fehlercode erscheinen:

xboxlive_get_stats_for_user - exception occurred getting results - 0x80190190

In diesem Fall sollte die asynchrone DS-Ereigniskarte einen "succeeded"-Schlüssel mit dem Wert "0" haben, und in diesem Fall sollten Sie versuchen, weniger Statistiken mit einem einzigen Aufruf abzurufen.

Syntax:

xboxlive_get_stats_for_user(user_id, serviceconfig_id, statname1, …);

Argument Typ Beschreibung
user_id Xbox-Benutzer-ID The user ID pointer.
serviceconfig_id Zeichenfolge The service config file ID
statname1 (2, 3, etc...) Streicher The stat names to retrieve the information for.

Rückgabe:

Real

Beispiel:

var _uid = xboxlive_get_user(0);
var _configid = "00000000-0000-0000-0000-000000000000";
xboxlive_get_stats_for_user(xbuser, _configid, "GameProgress", "CurrentMode");

Die obige code ruft die Benutzer-ID ab und verwendet diese dann, um Informationen über bestimmte Statistiken für diesen Benutzer anzufordern.
