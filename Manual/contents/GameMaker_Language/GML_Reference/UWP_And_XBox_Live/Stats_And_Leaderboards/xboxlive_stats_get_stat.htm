xboxlive_stats_get_stat

Diese Funktion kann verwendet werden, um einen einzelnen Statistikwert aus dem Statistikmanager für einen bestimmten Benutzer abzurufen. Sie geben die Benutzer-ID an, wie sie z. B. von der Funktion xboxlive_get_user()zurückgegeben wird, und dann die Zeichenkette, die Sie bei der Erstellung des Benutzers mit einer der Funktionen xboxlive_stats_set_stat_* definiert haben. Der Rückgabewert kann entweder eine Zeichenkette oder ein Realwert sein (je nach der zu prüfenden Statistik) oder die Konstante GML undefined , wenn die Statistik nicht existiert oder ein Fehler aufgetreten ist.

Syntax:

xboxlive_stats_get_stat(user_id, stat);

ArgumentTyp Beschreibung
user_id The user ID pointer to get the stat names of
stat The statistic to set (a string)

Rückgabe:

Real, String or undefined

Beispiel:

if (game_over == true)
{
    if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
    {
        var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
        xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
    }
}

Die obige code prüft eine Variable, und wenn sie true ist, dann wird der Wert der Statistik "PercentDone" geprüft. Wenn dieser Wert kleiner als 100 ist, wird ein Wert generiert und die Statistik auf diesen Wert gesetzt.
