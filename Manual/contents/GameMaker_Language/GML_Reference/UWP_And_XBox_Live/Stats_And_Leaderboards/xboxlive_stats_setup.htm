xboxlive_stats_setup

Diese Funktion muss aufgerufen werden, bevor Sie eine der anderen Xbox Live-Statistikfunktionen verwenden können, und initialisiert einfach die erforderlichen Bibliotheken auf dem System. Die "user_id" argument ist die rohe Benutzer-ID, wie sie von der Funktion xboxlive_get_user()zurückgegeben wird, während die "service_config" und "title_id" die eindeutige ID für Ihr Spiel im Xbox Live Dev Center ist.

Diese Funktion ist nur für die Verwendung mit ereignisbasierten Statistiken (2013) vorgesehen.

Syntax:

xboxlive_stats_setup(user_id, service_config_id, title_id);

Argument Typ Beschreibung
user_id Xbox-Benutzer-ID The ID pointer of the user to check.
service_config_id Zeichenfolge This is the config_id string for the game.
title_id Zeichenfolge The unique ID for your game on the Xbox Dev Center.

Rückgabe:

N/A

Beispiel:

var user = xboxlive_get_user(0);
xboxlive_stats_setup(user, "4d61a1aa-61ac-4541-badd-31f91244fea6", $1244FEA6);

Die obige code initialisiert das Statistiksystem für die angegebene Benutzerkennung.
