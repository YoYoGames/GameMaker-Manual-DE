xboxlive_erfolge_anzeigen_erfolge

Diese Funktion öffnet die Seite mit den Erfolgen für die gewählte platform. Bitte beachten Sie, dass es sich hierbei um eine asynchrone Funktion handelt, d. h. Ihr Spiel läuft im Hintergrund weiter, während die Erfolgsseite angezeigt wird. Seien Sie daher vorsichtig, wo Sie diese Funktion verwenden, und stellen Sie sicher, dass Sie das Spiel anhalten oder die Funktion nur in Bereichen Ihres Spiels anzeigen lassen, in denen sie den Spielablauf nicht stört.

Syntax:

xboxlive_achievement_show_achievements()

Rückgabe:

Beispiel:

if achievements_enabled
{
    global.Pause = true;
    instance_create_layer(0, 0, "Controllers", obj_Pause);
    xboxlive_achievement_show_achievements();
}

Die obige code prüft eine benutzerdefinierte Variable, um festzustellen, ob die Achievements aktiviert sind. Ist dies der Fall, wird eine globale Variable gesetzt und eine Controller-Instanz "pause" erstellt, bevor die Funktion zur Anzeige der Achievements auf dem Bildschirm aufgerufen wird.
