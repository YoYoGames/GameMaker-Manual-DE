uwp_ist_suspendiert

Diese Funktion gibt true für einen einzelnen Frame zurück (und ansonsten false ), ähnlich wie bei os_is_paused(). Das Spiel hat dann eine Sekunde Zeit, den Spielzustand zu speichern, bevor es möglicherweise vom System beendet wird. Sobald das Spiel das Speichern der Daten beendet hat, sollte es die Funktion uwp_suspend() aufrufen, um es sauber zu beenden. Das Spiel sollte wahrscheinlich auch in einen Pausenzustand übergehen, damit der Spieler, wenn es vom System wieder aufgenommen wird (falls es nicht vollständig beendet wurde), nicht gleich wieder in eine Spielsituation zurückgeworfen wird.

Syntax:

uwp_is_suspending();

Rückgabe:

Beispiel:

if uwp_is_suspending()
{
    scr_Save_Game_Data();
    uwp_suspend();
}

Die obige code prüft, ob die App in die Unterbrechung geht, und wenn ja, ruft sie script auf, um die Spieldaten zu speichern, bevor sie das Spiel unterbricht.
