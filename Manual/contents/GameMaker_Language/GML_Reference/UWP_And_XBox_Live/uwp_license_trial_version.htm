uwp_lizenz_trial_version

Mit dieser Funktion kann überprüft werden, ob das Spiel unter einer Testlizenz steht oder nicht. Ist dies der Fall, gibt die Funktion true zurück, andernfalls false.

Syntax:

uwp_license_trial_version();

Rückgabe:

Beispiel:

if uwp_license_trial_version()
{
    global.LevelCap = 10;
}

Die obige code prüft, ob die Anwendung unter einer Testlizenz steht, und wenn ja, setzt sie eine globale Variable.
