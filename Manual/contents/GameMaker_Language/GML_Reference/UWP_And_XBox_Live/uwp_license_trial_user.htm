uwp_lizenz_trial_user

Diese Funktion gibt den Zeiger der Benutzer-ID zurück, deren Testlizenz gerade verwendet wird, oder sie gibt pointer_null zurück, wenn der Benutzer gerade nicht aktiv ist oder wenn sich das Spiel gerade nicht im Testmodus befindet.

Syntax:

uwp_license_trial_user();

Rückgabe:

Xbox User ID

Beispiel:

if gamepad_button_check_pressed(0, gp_start)
{
    if (uwp_license_trial_version())
    {
        var _uid = uwp_license_trial_user();
        uwp_show_help(_uid);
    }
}

Die obige code prüft, ob eine Schaltfläche gedrückt wird, und überprüft dann, ob die Anwendung mit einer Testlizenz ausgeführt wird. Ist dies der Fall, wird die Benutzer-ID für die Lizenz ermittelt und die Hilfedatei für diesen Benutzer geöffnet.
