xboxlive_get_user

Mit dieser Funktion können Sie den Zeiger auf die Benutzer-ID für den indizierten Benutzer abrufen. Wenn der Benutzer nicht existiert, gibt die Funktion stattdessen die Konstante pointer_null zurück. Die Anzahl der derzeit angemeldeten Benutzer können Sie mit der Funktion xboxlive_get_user_count().

WICHTIG: Diese Funktion sollte nur mit xboxlive_get_user_count() verwendet werden - verwenden Sie sie nicht mit Gamepads. Verwenden Sie stattdessen xboxlive_user_for_pad().

Syntax:

xboxlive_get_user(index);

Argument Typ Beschreibung
index Real The index (an integer) to get the User ID from.

Rückgabe:

Xbox User ID or pointer_null

Beispiel:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
}

Die obige loops durchläuft alle angemeldeten Benutzer und speichert ihre eindeutigen ID-Zeiger in einem Array.
