xboxlive_generate_player_session_id

Diese Funktion erzeugt eine eindeutige Zeichenfolge (mit einer GUID) für die aktuelle Spielersitzung. Diese kann dann mit der Funktion xboxlive_fire_event().

Syntax:

xboxlive_generate_player_session_id();

Rückgabe:

Beispiel:

if !xboxlive_user_is_signed_in()
{
    if (!xboxlive_user_is_signing_in())
    {
        xboxlive_show_account_picker();
    }
}
else
{
    global.GamerTag = xboxlive_gamertag_for_user();
}

Die obige code prüft, ob ein Benutzer bei XBox Live angemeldet ist. Wenn dies der Fall ist, ermittelt code den Gamertag des Benutzers und speichert ihn in einer globalen Variable für die zukünftige Verwendung. Wenn der Benutzer nicht angemeldet ist, prüft code, ob er gerade dabei ist, sich anzumelden. In diesem Fall geschieht nichts weiter, und wenn er es nicht ist, wird die Kontoauswahl geöffnet und der Benutzer wird aufgefordert, sich anzumelden.
