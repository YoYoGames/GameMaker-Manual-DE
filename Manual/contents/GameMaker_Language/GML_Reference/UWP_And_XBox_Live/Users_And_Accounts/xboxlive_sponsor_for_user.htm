xboxlive_sponsor_für_benutzer

Diese Funktion ruft die ID des Sponsors des angegebenen Benutzers ab.

Syntax:

xboxlive_sponsor_for_user(user_id);

Argument Typ Beschreibung
user_id Xbox-Benutzer-ID The ID pointer of the user to check.

Rückgabe:

Xbox Sponsor ID

Beispiel:

var _uid = xboxlive_user_for_pad(0);
sponsor = xboxlive_sponsor_for_user(_uid);

Die obige code ruft den Benutzer-ID-Zeiger für den Benutzer ab, der dem Gamepad [0]-Slot zugewiesen ist, und verwendet diese ID dann, um die Sponsor-ID des Benutzers abzurufen.
