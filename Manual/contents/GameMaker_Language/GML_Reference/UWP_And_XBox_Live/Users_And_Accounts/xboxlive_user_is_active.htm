xboxlive_benutzer_ist_aktiv

Mit dieser Funktion können Sie prüfen, ob die angegebene Benutzer-ID in der Liste der Benutzer enthalten ist, die derzeit die Konsole verwenden. Wenn ja, gibt die Funktion true zurück, andernfalls false. Sie können einen Zeiger auf die Benutzer-ID mit der Funktion xboxlive_get_user().

Syntax:

xboxlive_user_is_active(user_id);

ArgumentTyp Beschreibung
user_id The ID pointer of the user to check.

Rückgabe:

Beispiel:

global.PlayerNum = 0;
global.Player_ID = array_create();
for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    var _uid = xboxlive_get_user(i);
    if xboxlive_user_is_active(_uid)
    {
        global.Player_ID[global.PlayerNum++] = _uid;
    }
}

Das obige code loops durchsucht die Benutzerkonten und prüft dann, ob eines davon aktiv ist. Ist dies der Fall, wird ihre Benutzer-ID in ein globales Array aufgenommen.
