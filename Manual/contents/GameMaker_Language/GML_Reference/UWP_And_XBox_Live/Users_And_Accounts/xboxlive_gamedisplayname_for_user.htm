xboxlive_gamedisplayname_für_benutzer

Mit dieser Funktion können Sie den Anzeigenamen - als String - aus dem angegebenen Benutzer-ID-Zeiger abrufen. Wenn es sich um einen lokalen Benutzer handelt, ist der zurückgegebene Wert einfach eine Zeichenkette des Anzeigenamens, aber wenn es sich um einen entfernten Benutzer handelt, ist die Zeichenkette sein Gamertag, der dann z.B. zum Stummschalten im Voice-Chat verwendet werden kann.

Syntax:

xboxlive_gamedisplayname_for_user(user_id);

ArgumentTyp Beschreibung
user_id The ID pointer of the user to check.

Rückgabe:

Beispiel:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
    user_name[i] = xboxlive_gamedisplayname_for_user(user_id[i]);
}

Die obige code ruft den Benutzer-ID-Zeiger für jedes verfügbare Benutzerkonto ab und speichert den Zeiger dann zusammen mit dem Anzeigenamen in zwei Arrays.
