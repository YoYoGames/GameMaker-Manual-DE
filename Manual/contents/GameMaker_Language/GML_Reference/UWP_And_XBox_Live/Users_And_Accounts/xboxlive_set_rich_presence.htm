xboxlive_set_rich_presence

Diese Funktion setzt den Rich Presence String für den angegebenen Benutzer. Ein Rich Presence String zeigt die In-Game-Aktivitäten eines Benutzers nach dem Namen des Spiels, das der Benutzer gerade spielt, getrennt durch einen Bindestrich. Diese Zeichenfolge wird unter dem Gamertag eines Spielers in der Liste "Freunde & Clubs" sowie im Xbox Live-Benutzerprofil des Spielers angezeigt.

Wenn Sie diese Funktion verwenden, müssen Sie den Benutzer-ID-Zeiger für den Benutzer angeben, und dann können Sie den Benutzer als derzeit aktiv im Spiel kennzeichnen oder nicht (mit true/false). Wenn der Wert false angegeben wird, wird der Rich Presence String mit "/afk" oder etwas anderem angehängt. Das nächste argument ist die anzuzeigende Rich-Presence-String-ID, und schließlich können Sie (optional) einen service_config_id -String angeben. Diese argument ist optional, und wenn sie nicht angegeben wird, wird die in den Xbox-Spieloptionen angegebene SCID verwendet.

Syntax:

xboxlive_set_rich_presence(user_id, is_user_active, rich_presence_string, [service_config_id]);

Argument Typ Beschreibung
user_id Xbox-Benutzer-ID The ID pointer of the user to check.
is_user_active Boolesche Flag the user as active or not.
rich_presence_string Zeichenfolge The rich presence string ID to use (as defined in the Partner Center - max 50 characters)
service_config_id Zeichenfolge OPTIONAL Dies ist die config_id Zeichenfolge für das Spiel.

Rückgabe:

N/A

Beispiel:

var pad_uid = xboxlive_user_for_pad(0);
xboxlive_set_rich_presence(pad_uid, true, "Playing_Challenge");

Die obige code ruft den Benutzer-ID-Zeiger für den Benutzer ab, der dem Gamepad [0]-Steckplatz zugewiesen ist, und setzt dann den Rich Presence String für diesen Benutzer.
