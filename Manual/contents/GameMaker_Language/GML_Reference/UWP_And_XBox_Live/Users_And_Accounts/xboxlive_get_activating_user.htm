xboxlive_get_activating_user

Mit dieser Funktion können Sie den Benutzer-ID-Zeiger für den Benutzer abrufen, der das Spiel über die Konsole gestartet hat.

HINWEIS: Wie Sie diesen Zeiger verwenden, hängt davon ab, was Sie im Spiel tun wollen, aber Sie sollten sich nicht darauf verlassen, dass es die aktuell spielende Benutzer-ID ist, da der aktivierende Benutzer sich abmelden kann, während das Spiel läuft.

Syntax:

xboxlive_get_activating_user();

Rückgabe:

Xbox User ID

Beispiel:

global.activUser = xboxlive_get_activating_user();

Das obige Beispiel speichert den Zeiger der aktivierenden Benutzer-ID in einer globalen Variablen
