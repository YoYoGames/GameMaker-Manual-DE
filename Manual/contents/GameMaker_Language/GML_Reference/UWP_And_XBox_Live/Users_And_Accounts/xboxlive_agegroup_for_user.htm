xboxlive_agegroup_für_benutzer

Diese Funktion gibt eine von vier Konstanten (siehe unten) zurück, um den Altersbereich anzugeben, der dem angegebenen Benutzer-ID-Zeiger zugeordnet ist.

Syntax:

xboxlive_agegroup_for_user(user_id);

Argument Typ Beschreibung
user_id Xbox-Benutzer-ID The user ID (a pointer) to check

Rückgabe:

Xbox Age Group Constant

Xbox Altersgruppe Konstant
Konstante Beschreibung
xboxlive_agegroup_unknown Die Altersgruppe des Benutzers ist unbekannt
xboxlive_agegroup_child Die Altersgruppe der Nutzer liegt zwischen 8 und 12 Jahren (einschließlich)
xboxlive_agegroup_teen Die Altersgruppe der Nutzer liegt zwischen 13 und 17 Jahren (einschließlich, und zwischen 13 und 19 Jahren in Südkorea)
xboxlive_agegroup_adult Die Altersgruppe der Nutzer ist 18 oder älter (bzw. 20 und älter in Südkorea)

Beispiel:

var _age = xboxlive_agegroup_for_user(xboxlive_get_activating_user());
if (_age != xboxlive_agegroup_adult)
{
    global.ContentControl = true;
}
else global.ContentControl = false;

Die obige code prüft die Altersgruppe des aktivierenden Benutzers und setzt eine globale Variable je nach der zurückgegebenen Konstante auf true oder false.
