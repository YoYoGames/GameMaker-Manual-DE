xboxlive_get_user_count

Mit dieser Funktion können Sie die Gesamtzahl der derzeit im XBox-System angemeldeten Benutzer ermitteln. Der Rückgabewert ist ein ganzzahliger Wert von 0 - N, wobei N die Anzahl der angemeldeten Benutzer -1 ist.

Syntax:

xboxlive_get_user_count();

Rückgabe:

Beispiel:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
}

Die obige loops durchläuft alle angemeldeten Benutzer und speichert ihre eindeutigen ID-Zeiger in einem Array.
