xboxlive_profil_karte_für_benutzer_anzeigen

Mit dieser Funktion können Sie anfordern, dass die XBox die Profilkarte für den Ziel-Benutzer-ID-Zeiger anzeigt. Die Funktion erfordert sowohl den Benutzer-ID-Zeiger des Benutzers, der die Informationen anfordert, als auch den Benutzer-ID-Zeiger des Benutzers, dessen Profilkarte Sie abrufen möchten.

Syntax:

xboxlive_show_profile_card_for_user(requesting_user_id, target_user_id);

ArgumentTyp Beschreibung
requesting_user_id The user ID (a pointer) of the requesting user
target_user_id The user ID (a pointer) of the user to get the profile card of

Rückgabe:

Beispiel:

if (gamepad(0, gp_start))
{
    xboxlive_show_profile_card_for_user(user[0], user[1]);
}

Die obige code prüft, ob eine Gamepad-Taste gedrückt wurde, und zeigt, wenn dies der Fall ist, die Profilkarte für den jeweiligen Benutzer an.
