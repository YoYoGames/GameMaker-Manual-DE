xboxlive_user_is_guest

Diese Funktion gibt true zurück, wenn der angegebene Benutzer-ID-Zeiger ein Gastbenutzer ist, und false, wenn er es nicht ist. Sie können einen Zeiger auf die Benutzer-ID mit der Funktion xboxlive_get_user().

Syntax:

xboxlive_user_is_guest(user_id);

ArgumentTyp Beschreibung
user_id The ID pointer of the user to check.

Rückgabe:

Beispiel:

if xboxlive_user_is_guest(user_id[0])
{
    global.user_name[0] = "GUEST";
}
else
{
    global.user_name[0] = xboxlive_gamedisplayname_for_user(user_id[0]);
}

Der obige Befehl speichert den Zeiger auf die aktivierende Benutzer-ID in einer globalen Variablen.
