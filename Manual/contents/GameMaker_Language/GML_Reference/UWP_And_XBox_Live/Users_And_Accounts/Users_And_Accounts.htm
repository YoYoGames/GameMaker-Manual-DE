Benutzer und Konten

Dieser Abschnitt richtet sich an Benutzer der UWP oder Xbox One platforms und beschreibt die verfügbaren XBox Live-Funktionen für den Umgang mit Benutzern und Benutzerkonten.

Bei der Xbox One können sich mehrere Benutzer gleichzeitig an einer Konsole anmelden. Jedem Benutzer ist ein bestimmter Controller zugeordnet, und auch alle Spielstandsdaten sind benutzerspezifisch. Um dieses System optimal zu nutzen, führt GameMaker eine Reihe von Funktionen ( XBox One ) ein, die sich ausschließlich auf die Zuordnung von Controllern und Benutzern beziehen. So können Sie Informationen über den Benutzer abfragen und steuern, wie die Speicherdaten pro Benutzer behandelt werden. Weitere Informationen für Entwickler über dieses System finden Sie im offiziellen Whitepaper auf der Microsoft-Entwicklerseite.

HINWEIS: Beachten Sie, dass die Benutzer-ID für jede Person eindeutig ist und über verschiedene Konsolen hinweg beibehalten wird. Wenn sich also derselbe Benutzer auf zwei verschiedenen Geräten anmeldet, ist die Benutzer-ID dieselbe.

Die folgenden Funktionen können Sie mit XBox Live-Konten verwenden, sowohl auf XBox One als auch mit dem regulären UWP-Export (beachten Sie, dass dies die Mindestfunktionalität ist, die für UWP im Rahmen des Creators Program erlaubt ist, wie in diesem Helpdesk-Artikel beschrieben):

  • xboxlive_user_is_signed_in
  • xboxlive_user_is_signing_in
  • xboxlive_gamertag_for_user
  • xboxlive_show_account_picker

Die folgenden Funktionen sind nur gültig, wenn Sie die XBox One mit UWP und einem gültigen ID@XBox-Konto ansteuern:

  • xboxlive_get_user_count
  • xboxlive_get_user
  • xboxlive_get_activating_user
  • xboxlive_user_is_guest
  • xboxlive_user_is_active
  • xboxlive_user_is_remote
  • xboxlive_user_id_for_user
  • xboxlive_sponsor_for_user
  • xboxlive_set_rich_presence
  • xboxlive_gamedisplayname_for_user
  • xboxlive_user_for_pad
  • xboxlive_pad_for_user
  • xboxlive_pad_count_for_user
  • xboxlive_agegroup_for_user
  • xboxlive_gamerscore_for_user
  • xboxlive_show_profile_card_for_user
  • xboxlive_reputation_for_user
  • xboxlive_sprite_add_from_gamerpicture
  • xboxlive_generate_player_session_id
