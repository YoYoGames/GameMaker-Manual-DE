xboxlive_benutzer_id_für_benutzer

Diese Funktion wandelt den Zeiger auf die Benutzer-ID in eine Zeichenkette um.

Syntax:

xboxlive_user_id_for_user(user_id);

ArgumentTyp Beschreibung
user_id The ID pointer of the user to check.

Rückgabe:

Beispiel:

var _uid = xboxlive_user_for_pad(0);
uid_string = xboxlive_user_id_for_user(_uid);

Die obige code ruft den Benutzer-ID-Zeiger für den Benutzer ab, der dem Gamepad [0]-Slot zugewiesen ist, und speichert dann diese Benutzer-ID als String in einer Instanzvariablen.
