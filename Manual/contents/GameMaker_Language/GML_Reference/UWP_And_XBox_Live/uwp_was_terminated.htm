uwp_wurde_beendet

Mit dieser Funktion können Sie herausfinden, ob das Spiel zuvor vom System heruntergefahren wurde (d.h. es ist nicht der erste Durchlauf des Spiels). Wenn dies der Fall ist, gibt die Funktion true zurück, andernfalls gibt sie false zurück. Ein Beispiel für die Verwendung wäre, zu prüfen, ob das Spiel zuvor vom System beendet wurde, und dann (wenn sie true zurückgibt) die vorherige Phase zu laden, in der sich der Spieler im Spiel befand. Wenn man auf XBox One abzielt, dann ist das erforderliche Verhalten von Spielen, wenn dies erkannt wird, in den verschiedenen Versionen der Zertifizierungsdokumente unterschiedlich, so dass es sich lohnt, die neueste Version zu überprüfen, um zu sehen, was das Spiel tun sollte.

Syntax:

uwp_was_terminated();

Rückgabe:

Beispiel:

if xboxlivewas_terminated()
{
    buffer_load("Player_Save.sav");
    global.GameLoaded = true;
}

Die obige code prüft, ob die Anwendung zuvor vom System beendet wurde, und wenn ja, lädt sie eine buffer und setzt eine globale Variable.
