uwp_suspend

Diese Funktion zeigt dem System an, dass die App die Speicherung der Daten nach der Erkennung einer Aussetzungsanforderung abgeschlossen hat. Der Aufruf dieser Funktion ist eine Voraussetzung für die Übermittlung an Xbox One und sollte immer in Verbindung mit der Funktion uwp_is_suspending()verwendet werden, wie im folgenden Beispiel gezeigt.

Syntax:

uwp_suspend();

Rückgabe:

Beispiel:

if uwp_is_suspending()
{
    scr_Save_Game_Data();
    uwp_suspend();
}

Die obige code prüft, ob die App in die Unterbrechung geht, und wenn ja, ruft sie script auf, um die Spieldaten zu speichern, bevor sie das Spiel unterbricht.
