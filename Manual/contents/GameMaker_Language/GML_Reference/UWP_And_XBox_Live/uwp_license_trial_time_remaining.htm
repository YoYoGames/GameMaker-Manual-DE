uwp_license_trial_time_remaining

Diese Funktion gibt die ungefähre Anzahl von Sekunden zurück, bis eine zeitbasierte Prüfung abläuft. Der von dieser Funktion zurückgegebene Wert ist bedeutungslos, wenn die Prüfung nicht zeitbasiert ist oder wenn das Spiel nicht im Prüfungsmodus läuft.

Syntax:

uwp_license_trial_time_remaining();

Rückgabe:

Beispiel:

if uwp_license_trial_version()
{
    var secs = uwp_license_trial_time_remaining() mod 60;
    var mins = uwp_license_trial_time_remaining() div 60;
    var hours = mins div 60;
    if secs < 10 secs = "0" + string(secs) else secs = string(secs);
    if mins > 60 mins -= (hours * 60);
    if mins < 10 mins = "0" + string(mins) else mins = string(mins);
    if hours > 9 hours = "9" else hours = string(hours);
    time_string = hours + ":" + mins + ":" + secs;
}

Die obige code prüft, ob das Spiel mit einer Testlizenz betrieben wird, und wenn ja, erstellt sie eine Zeichenkette mit der Zeit bis zum Ablauf der Lizenz.
