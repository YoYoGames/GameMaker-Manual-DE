matrix_stack_is_empty

Diese Funktion kann verwendet werden, um zu prüfen, ob der Matrix-Stapel leer ist (gibt true zurück) oder nicht (gibt false zurück).

Syntax:

matrix_stack_is_empty();

Rückgabe:

Beispiel:

if !matrix_stack_is_empty()
{
    matrix_stack_clear();
}

Die obige code prüft, ob der Matrix-Stapel leer ist, und löscht ihn, wenn dies nicht der Fall ist.
