matrix_stack_push

Diese Funktion wird verwendet, um eine neue Matrix an die Spitze des aktuellen Matrix-Stapels zu schieben. Sie multipliziert zunächst die angegebene Matrix mit der Matrix, die sich derzeit oben auf dem Stapel befindet, und schiebt dann die resultierende Matrix auf den Stapel.

Diese Funktion ist nützlich, um mehrere Matrixtransformationen auf Ihr Bildmaterial anzuwenden, ohne dass Sie verschiedene Matrizen manuell miteinander multiplizieren müssen.

Sie können zum Beispiel eine Basismatrix auf den Stack schieben, die Grafiken in einen bestimmten rechteckigen Bereich (z. B. ein Fenster) zeichnet. Nachdem Sie einige Grafiken mit dieser Matrix gezeichnet haben, können Sie eine weitere Matrix auf den Stapel schieben, um Grafiken in einem Teilbereich (z. B. einer Schaltfläche im Fenster) zu zeichnen. Nachdem Sie etwas in diesem Unterbereich gezeichnet haben, können Sie matrix_stack_pop() aufrufen, um die Matrix vom Stapel zu entfernen und mit dem Zeichnen von Grafiken im Hauptfensterbereich fortzufahren.

Syntax:

matrix_stack_push(matrix);

ArgumentTyp Beschreibung
matrix The matrix to push to the stack

Rückgabe:

Beispiel:

var m1 = matrix_build(66, 145, 0, 0, 0, 0, 1, 1, 1);
var m2 = matrix_build(0, 0, 0, 0, 0, image_angle * 6, 1, 1, 1) ;
matrix_stack_push(m1);
matrix_stack_push(m2);
matrix_set(matrix_world, matrix_stack_top());
draw_sprite(spr_tyre, 0, 0, 0);
matrix_stack_pop();
matrix_stack_pop();

Die obige code erstellt zwei neue Matrizen und schiebt sie dann auf den Matrix-Stapel. Die Weltmatrix wird dann an das obere Ende des Stapels gesetzt, eine sprite wird gezeichnet und die Transformationen werden dann vom Stapel gepoppt.
