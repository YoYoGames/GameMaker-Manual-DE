matrix_build_projection_ortho

Diese Funktion erstellt eine orthografische Projektionsmatrix auf der Grundlage der unten aufgeführten Parameter (dies ist die Standardprojektionsmethode, die verwendet wird, wenn Sie einen Raum in GameMaker erstellen, ohne etwas zu ändern). Manchmal müssen Sie switch von einer perspektivischen Projektion in eine orthografische Projektion umwandeln, wozu diese Funktion dient. Sie wird typischerweise verwendet, um ein Overlay zu zeichnen, z. B. um den Spielstand oder andere Aspekte anzuzeigen, da dies eine "flache" Ansicht der gezeichneten Elemente (d. h. keine Perspektive) in einem 3D-Spiel ergibt. In der folgenden Abbildung sehen Sie den Unterschied zwischen orthografischer und perspektivischer Ansicht.

HINWEIS: Möglicherweise müssen Sie auch die Entfernung verdeckter Flächen vorübergehend deaktivieren ( switch ), wenn Sie möchten, dass die Informationen unabhängig vom aktuellen Tiefenwert gezeichnet werden.

Syntax:

matrix_build_projection_ortho(width, height, znear, zfar);

ArgumentTyp Beschreibung
w The width of the projection.
h The height of the projection.
znear The near clipping plane.
zfar The far clipping plane.

Rückgabe:

Matrix Array

Beispiel:

viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

Die obige code erstellt eine neue Look-at-Matrix und eine orthografische Matrix, speichert ihre IDs in lokalen Variablen und verwendet sie dann, um die Ansichts- und Projektionsmatrizen für die Kamera zu setzen, die dem Viewport[0] zugewiesen ist.
