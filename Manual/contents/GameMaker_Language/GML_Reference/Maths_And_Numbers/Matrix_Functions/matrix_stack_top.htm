matrix_stack_top

Diese Funktion gibt die aktuelle oberste Matrix des Stapels zurück, entfernt sie aber nicht (für diese Verwendung matrix_stack_pop()).

Syntax:

matrix_stack_top();

Rückgabe:

Matrix Array

Beispiel:

var m1 = matrix_build(66, 145, 0, 0, 0, 0, 1, 1, 1);
var m2 = matrix_build(0, 0, 0, 0, 0, image_angle * 6, 1, 1, 1) ;
matrix_stack_push(m1);
matrix_stack_push(m2);
matrix_set(matrix_world, matrix_stack_top());
draw_sprite(tyre, 0, 0, 0);
matrix_stack_pop();
matrix_stack_pop();

Die obige code erstellt zwei neue Matrizen und schiebt sie dann auf den Matrix-Stapel. Die Weltmatrix wird dann an das obere Ende des Stapels gesetzt, eine sprite wird gezeichnet und die Transformationen werden dann vom Stapel gepoppt.
