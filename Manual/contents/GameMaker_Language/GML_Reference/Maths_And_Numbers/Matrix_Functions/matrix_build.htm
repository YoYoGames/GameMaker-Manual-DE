matrix_build

Diese Funktion kann verwendet werden, um Ihre eigene benutzerdefinierte Matrix zu erstellen. Sie gibt ein Matrix-Array zurück, das in einer Variablen gespeichert werden sollte, um es später verwenden zu können. Sie akzeptiert 3-dimensionale(x, y, z) Translations-, Rotations- und Skalierungswerte und verwendet diese, um ein Matrix-Array zu erstellen.

Das Matrix-Array enthält insgesamt 16 Werte, wobei die ersten 4 Elemente Zeile/Spalte 1sind, die nächsten 4 Elemente sind Zeile/Spalte 2 und so weiter, als Teil einer 4x4-Matrix. Ob das Array nach Zeilen oder Spalten geordnet ist, hängt von der Zielseite platform ab, da der von platform verwendete Grafik-Renderer zeilen- oder spaltenbetonte Matrizen verwenden kann.

HINWEIS Wenn Sie eine neue Matrix auf diese Weise erstellen, ist die Reihenfolge der Operation YXZ.

Syntax:

matrix_build(x, y, z, xrotation, yrotation, zrotation, xscale, yscale, zscale)

ArgumentTyp Beschreibung
x The x component of the translation vector.
y The y component of the translation vector.
z The z component of the translation vector.
xrotation The angle to rotate around the x-axis (in degrees °).
yrotation The angle to rotate around the y-axis (in degrees °).
zrotation The angle to rotate around the z-axis (in degrees °).
xscale The x scale amount.
yscale The y scale amount.
zscale The z scale amount.

Rückgabe:

Matrix Array

Beispiel:

t_matrix = matrix_build(x, y, 0, 0, 90, 0, 1, 2, 1);

Die obige code erstellt eine neue Matrixtransformation und speichert den resultierenden Matrixindex in der Variablen "t_matrix".
