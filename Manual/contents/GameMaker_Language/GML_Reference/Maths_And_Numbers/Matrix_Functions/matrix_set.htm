matrix_set

Diese Funktion nimmt ein Array mit 16 Werten an, die einem bestimmten 4x4-Matrixtyp entsprechen, wobei die Elemente [0 - 3] die Zeile 1, die Elemente [4 -7] die Zeile 2 usw. sind... (siehe das Bild auf der Hauptseite). Sie können eine solche Matrix mit den Befehlen matrix_build() oder matrix_get() Funktionen erstellen oder einfach das Array selbst erstellen und es an die Funktion übergeben. Die verfügbaren Matrixtypen sind Ansicht, Projektion und Welt, für die Sie eine der folgenden Konstanten verwenden würden:

Konstante Beschreibung
matrix_view Die aktuelle Ansichtsmatrix
matrix_projection Die aktuelle Projektionsmatrix
matrix_world Die aktuelle Weltmatrix

Syntax:

matrix_set(type, matrix);

Argument Typ Beschreibung
type Matrix Typ Konstante The type of matrix to get the values of (see the constants listed above)
matrix Matrix-Array The matrix data as an array

Rückgabe:

N/A

Beispiel:

matrix_set(matrix_world, m_array);

Die obige code setzt die Werte der aktuellen Weltmatrix auf die Werte, die in der Array-Matrix "m_array" gespeichert sind.
