matrix_stack_set

Diese Funktion überschreibt die aktuelle Spitze des Matrix-Stapels mit der angegebenen Matrix.

Syntax:

matrix_stack_set(matrix);

ArgumentTyp Beschreibung
matrix The matrix index to use.

Rückgabe:

Beispiel:

var m = matrix_build(x, y, 0, 0, 0, 0, 1, 1, 1);
matrix_stack_set(m);

Die obige code baut eine neue Matrix auf und speichert den daraus resultierenden Matrixindex in der Variablen "m", bevor der oberste Punkt des Matrixstapels damit ersetzt wird.
