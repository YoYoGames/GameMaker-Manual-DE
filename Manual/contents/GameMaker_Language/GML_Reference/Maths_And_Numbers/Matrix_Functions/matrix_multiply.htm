matrix_multiplizieren

Mit dieser Funktion können Sie zwei Matrix-Arrays miteinander multiplizieren, um eine neue Transformationsmatrix zu erstellen. Die Funktion gibt den Index der neuen Matrix zurück, der zur späteren Verwendung in einer Variablen gespeichert werden sollte.

HINWEIS: Sie können mit dieser Funktion keine Matrixkonstante als argument verwenden. Wenn Sie also die (z. B.) Ansichtsmatrix mit einer von Ihnen erstellten benutzerdefinierten Matrix multiplizieren möchten, müssen Sie zuerst die Funktion matrix_get() aufrufen und die Werte der Ansichtsmatrix einer Array-Variablen zuweisen, diese dann mit Ihrer benutzerdefinierten Matrix multiplizieren und anschließend die gewählte Matrix (Ansicht, Projektion oder Welt) festlegen.

Syntax:

matrix_multiply(matrix1, matrix2);

ArgumentTyp Beschreibung
matrix1 The first matrix index to use.
matrix2 The second matrix index to use.

Rückgabe:

Matrix Array

Beispiel:

var v_matrix = matrix_get(matrix_view);
var new_matrix = matrix_multiply(v_matrix, my_matrix);
matrix_set(matrix_view, new_matrix);

Die obige code ermittelt die aktuelle Ansichtsmatrix, multipliziert diese mit einer benutzerdefinierten Matrix und verwendet dann den resultierenden Matrixindex, um die Ansichtsmatrix erneut festzulegen.
