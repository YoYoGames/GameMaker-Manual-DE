matrix_get

Diese Funktion gibt ein Array mit 16 Werten zurück, die dem angegebenen 4x4-Matrixtyp entsprechen, wobei Zeile 1 die Elemente [0 - 3], Zeile 2 die Elemente [4 -7] usw. sind... (siehe das Bild auf der Hauptseite). Die verfügbaren Matrizen sind view, projection und world, für die Sie eine der folgenden Konstanten verwenden:

Matrix Typ Konstante
Konstante Beschreibung
matrix_view Die aktuelle Ansichtsmatrix
matrix_projection Die aktuelle Projektionsmatrix
matrix_world Die aktuelle Weltmatrix

Syntax:

matrix_get(type);

Argument Typ Beschreibung
type Matrix Typ Konstante The type of matrix to get (see the constants listed above)

Rückgabe:

Matrix Array

Beispiel:

v_array = matrix_get(matrix_view);

Die obige code ermittelt die Werte der aktuellen Ansichtsmatrix und füllt das Array "v_array" mit ihnen.
