matrix_aufbau_projektion_perspektive

Diese Funktion erstellt eine perspektivische Projektionsmatrix auf der Grundlage der Dimensionen der nahen Clipping-Ebene unter Verwendung der unten aufgeführten Parameter.

Beachten Sie, dass das Sichtfeld der Kamera in Abhängigkeit von den angegebenen Werten width, height und znear variiert, da die Projektionsbreite und -höhe an der in znear argument angegebenen Z-Position platziert wird.

Bei einer konstanten Größe von 640x480 ist das Sichtfeld beispielsweise breiter, wenn sich znear näher an der Kamera befindet, aber schmaler, wenn znear weit von der Kamera entfernt ist. Dieses Verhalten wird in der folgenden Abbildung veranschaulicht:

Aus diesem Grund wird empfohlen, einen Wert für znear zu verwenden, der mit der Breite der Projektion übereinstimmt, so dass das Sichtfeld mit einer entsprechenden orthografischen Projektion übereinstimmt.

Syntax:

matrix_build_projection_perspective(width, height, znear, zfar);

ArgumentTyp Beschreibung
w The width of the projection at the near Z position.
h The height of the projection at the near Z position.
znear The near clipping plane.
zfar The far clipping plane.

Rückgabe:

Matrix Array

Beispiel:

var projmat = matrix_build_projection_perspective(640, 480, 640.0, 32000.0);
camera_set_proj_mat(view_camera[0], projmat);

Die obige code erstellt eine perspektivische Projektion und verwendet dann die zurückgegebene Matrix, um die dem Viewport[0] zugewiesene Kamera einzustellen.
