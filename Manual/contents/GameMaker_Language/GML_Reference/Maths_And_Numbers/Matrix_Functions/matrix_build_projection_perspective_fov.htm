matrix_build_projection_perspective_fov

Diese Funktion erstellt eine perspektivische Projektionsmatrix auf der Grundlage des Sichtfelds unter Verwendung der unten aufgeführten Parameter.

Syntax:

matrix_build_projection_perspective_fov(fov_y, aspect, znear, zfar);

ArgumentTyp Beschreibung
fov The angle of the field of view.
aspect The aspect ratio of the field of view.
znear The near clipping plane.
zfar The far clipping plane.

Rückgabe:

Matrix Array

Beispiel:

projmat = matrix_build_projection_perspective_fov(60, 320/240, 1.0, 32000.0);
camera_set_proj_mat(view_camera[0], projmat);

Die obige code erstellt eine Sichtfeld-Projektionsmatrix, die dann in einer Variablen gespeichert wird. Diese Matrix wird dann verwendet, um die Kamera einzurichten, die dem View Port[0] zugewiesen ist.
