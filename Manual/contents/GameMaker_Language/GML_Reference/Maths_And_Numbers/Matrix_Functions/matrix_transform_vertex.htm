matrix_transform_vertex

Diese Funktion transformiert eine vector durch eine Matrix. Sie geben die ID der Transformationsmatrix (wie von der Funktion matrix build()zurückgegeben wird), sowie die x-, y- und z-Werte für die zu transformierende vector. Die Funktion gibt zurück ein Array mit 3 Elementen zurück, wobei:

  • array[0] = x
  • array[1] = y
  • array[2] = z.

Syntax:

matrix_transform_vertex(matrix, x, y, z);

ArgumentTyp Beschreibung
matrix The matrix to use
x The x component of the transform vector
y The y component of the transform vector
z The z component of the transform vector

Rückgabe:

(3 elements)

Beispiel:

t_matrix = matrix_build(0, 0, 0, 0, 90, 0, 1, 2, 1);
verts = matrix_transform_vertex(t_matrix, x, y, z);

Die obige code transformiert die gegebenen Werte mit Hilfe der in der Variablen "t_matrix" gespeicherten Matrix und speichert sie in dem Array "verts".
