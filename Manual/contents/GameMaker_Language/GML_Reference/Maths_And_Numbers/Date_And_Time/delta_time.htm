delta_zeit

Diese Variable gibt die Frame-Delta-Zeit zurück, d. h. die Zeitdifferenz zwischen dem vorherigen Frame und dem aktuellen Frame. Dieser Wert wird in Mikrosekunden angegeben, wobei 1 Mikrosekunde der 1.000.000ste Teil einer Sekunde ist.

Der Zweck des Delta-Timings ist es, die Auswirkungen der Verzögerung oder Langsamkeit von Computern zu beseitigen, die versuchen, komplexe Grafiken oder viele code zu verarbeiten. Es ist ein Wert, der zur Geschwindigkeit von Objekten addiert werden kann, so dass sie sich schließlich mit der gleichen Geschwindigkeit bewegen, unabhängig von der Verzögerung. Dies wird in GameMaker erreicht, indem die Variable delta_time bei jedem Schritt verwendet wird, da sie die Zeit, die zwischen einem Schritt und dem nächsten vergangen ist, in Mikrosekunden misst. Daher kann die Variable delta_time verwendet werden, um zu berechnen, um wie viel schneller (zum Beispiel) sich eine Spielfigur bewegen muss, um eine Verzögerungsspitze im Spiel auszugleichen.

Syntax:

delta_time

Rückgabe:

Real (integer)

Beispiel:

var _dt = delta_time / 1000000;
speed = spd * _dt;

Mit der obigen code wird die Geschwindigkeit der Instanz unter Verwendung der Deltazeit zur Korrektur der Verzögerung festgelegt. Die Variable "_dt" wird verwendet, um den zuvor berechneten Deltazeitwert zu speichern, so dass das Verhältnis verwendet werden kann, um den Basisgeschwindigkeitswert (der in der Variablen "spd" gespeichert ist) zu multiplizieren und so eine einheitliche Geschwindigkeit für die Instanz zu erhalten.
