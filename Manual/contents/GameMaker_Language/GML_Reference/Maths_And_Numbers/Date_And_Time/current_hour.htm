aktuelle_Stunde

Diese schreibgeschützte Variable gibt die Stunde zurück, die dem aktuellen Zeitpunkt auf der Grundlage der Standardzeitzone des Systems (d. h. der Ortszeit) entspricht. Sie können die zu verwendende Basiszeitzone mit der Funktion date_set_timezone().

Syntax:

current_hour;

Rückgabe:

Beispiel:

draw_text(32, 32, "The time is " + string(current_hour) + ":" + string(current_minute) + "." + string(current_second));

Die obige code würde die aktuelle internationale Zeit auf dem Bildschirm anzeigen.
