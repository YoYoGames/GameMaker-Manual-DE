current_month

Diese schreibgeschützte Variable gibt den aktuellen Monat als numerischen Wert zurück, wobei 1 für Januar und 12 für Dezember steht.

Syntax:

current_month

Rückgabe:

Beispiel:

draw_text((32, 32, "Today is " + string(current_day) + "/" + string (current_month) + "/" + string(current_year) +".");

Die obige code zeigt den Tag, den Monat und das Jahr an.
