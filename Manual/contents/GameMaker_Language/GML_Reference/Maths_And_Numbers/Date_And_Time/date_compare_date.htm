datum_vergleich_datum

Mit dieser Funktion können Sie zwei Daten daraufhin überprüfen, welches der beiden Daten früher oder später ist als das andere. Die Funktion gibt -1 zurück, wenn Datum1 früher ist, 0, wenn beide Daten gleich sind, und 1, wenn Datum1 später ist.

Syntax:

date_compare_date( date1, date2 );

Argument Typ Beschreibung
date1 Datum und Uhrzeit The first date.
date2 Datum und Uhrzeit The date to compare it to.

Rückgabe:

Real

Beispiel:

d = date_compare_date(date_create_datetime(2011, 9, 15, 11, 4, 0), date_current_datetime());

Dadurch würde "d" auf den entsprechenden Wert gesetzt, je nachdem, welches der Daten das früheste ist, wahrscheinlich 1, da das aktuelle Datum weiter in der Zukunft liegt als der 15. September 2011.
