date_date_string

Mit dieser Funktion können Sie eine Zeichenkette erstellen, die das angegebene Datum enthält, formatiert als Tag/Monat/Jahr.

Syntax:

date_date_string(date);

ArgumentTyp Beschreibung
date The date to use.

Rückgabe:

Beispiel:

str = date_date_string(date_current_datetime());
draw_text(32, 32, str);

Dadurch wird "str" so eingestellt, dass es eine formatierte Zeichenkette mit dem aktuellen Datum und der aktuellen Uhrzeit enthält, wie sie vom System angezeigt wird.
