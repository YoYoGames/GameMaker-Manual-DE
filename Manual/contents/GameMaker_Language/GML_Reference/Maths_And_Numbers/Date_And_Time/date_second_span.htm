date_second_span

Mit dieser Funktion können Sie die Anzahl der Sekunden zwischen zwei Daten ermitteln. Der Rückgabewert ist immer positiv und wird eine ganze Zahl sein.

Syntax:

date_second_span(date1, date2);

ArgumentTyp Beschreibung
date1 The first datetime.
date2 The datetime to compare it to.

Rückgabe:

Beispiel:

diff = date_second_span(date_create_datetime(2011, 9, 15, 11, 4, 0 ), date_current_datetime());

Dies würde "diff" auf die Anzahl der Sekunden zwischen dem 15. September 2011, 11:04 und dem aktuellen Datum und der aktuellen Uhrzeit setzen.
