date_is_today

Diese Funktion gibt true zurück, wenn der angegebene Datumswert der Tag ist, an dem sie überprüft wird (z. B. heute), oder andernfalls false. Dies kann eine praktische Funktion für Dinge wie Easter Eggs in Ihren Spielen oder für die Freischaltung von saisonalen Inhalten sein. Beachten Sie, dass diese Funktion von der eingestellten Zeitzone beeinflusst wird (Standard ist die lokale Zeit), die Sie mit der Funktion date_set_timezone() Funktion ändern können.

Syntax:

date_is_today(date);

ArgumentTyp Beschreibung
date The datetime to use.

Rückgabe:

Beispiel:

if date_is_today(global.Halloween)
{
    global.Max_Levels = 200;
}

Die obige code prüft den in der globalen Variable "Halloween" gespeicherten Zeitpunkt, um zu sehen, ob es heute ist, und wenn ja, setzt sie eine andere globale Variable auf einen neuen Wert.
