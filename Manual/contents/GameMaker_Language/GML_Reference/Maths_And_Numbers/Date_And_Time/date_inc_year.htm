datum_ink_jahr

Mit dieser Funktion können Sie einen gegebenen Datumswert um eine bestimmte Anzahl von Jahren inkrementieren, und sie gibt den neuen Datumswert zurück.

Syntax:

date_inc_year(date, amount);

ArgumentTyp Beschreibung
date The datetime to add to
amount The number of years (must be an integer) to add.

Rückgabe:

Beispiel:

mynewdatetime = date_inc_year(date_current_datetime(), 1000);

Damit würde mynewdatetime auf das aktuelle Datum gesetzt, aber um 1000 Jahre verlängert.
