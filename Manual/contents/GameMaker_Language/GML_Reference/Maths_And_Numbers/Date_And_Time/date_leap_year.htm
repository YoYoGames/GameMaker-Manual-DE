Datum_Schaltjahr

Diese Funktion gibt true zurück, wenn die Jahreskomponente des angegebenen datetime-Wertes ein Schaltjahr ist, oder andernfalls false. Dies kann eine praktische Funktion für Dinge wie Easter Eggs in Ihren Spielen oder zum Freischalten spezieller Inhalte sein.

Syntax:

date_leap_year(date);

ArgumentTyp Beschreibung
date The datetime to use.

Rückgabe:

Beispiel:

if date_leap_year(date_current_datetime())
{
    if !global.ExtraContent
    {
        global.ExtraContent = true;
    }
}

Die obige code überprüft die aktuelle Datumszeit, um festzustellen, ob das Jahr ein Schaltjahr ist oder nicht. Ist dies der Fall, so wird eine globale Variable gesetzt.
