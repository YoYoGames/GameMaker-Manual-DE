Datum_Vergleich_Zeit

Mit dieser Funktion können Sie zwei Datums- und Zeitangaben daraufhin überprüfen, bei welcher die Zeitkomponente früher oder später liegt als bei der anderen. Die Funktion gibt -1 zurück, wenn datetime1 früher ist, 1, wenn datetime1 später ist und 0, wenn sie gleich sind, und sie ignoriert das Datum, also buchstäblich nur, welche der Zeiten weiter durch den gegebenen Tag geht.

Syntax:

date_compare_time(datetime1, datetime2);

ArgumentTyp Beschreibung
datetime1 The first datetime.
datetime2 The datetime to compare the first one to.

Rückgabe:

Beispiel:

d = date_compare_time(date_create_datetime( 2011, 9, 15, 11, 4, 0 ), date_current_datetime());

Dies würde "d" auf den entsprechenden Wert setzen, je nachdem, welche der Zeiten die früheste war. Wenn die aktuelle Zeit später als 11:04 Uhr ist, wird 1 zurückgegeben. Wenn sie früher ist, -1. Wenn die Zeit 11:04 Uhr ist, wird 0 zurückgegeben.
