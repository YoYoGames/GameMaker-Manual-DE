aktuelle_zeit

Diese schreibgeschützte Variable gibt die Anzahl der Millisekunden zurück, die seit dem Start des Spiels vergangen sind.

Syntax:

current_time;

Rückgabe:

Beispiel:

if (current_time > 600000)
{
    msg = show_question_async("Would you like to rate?");
}

Die obige code prüft, ob mehr als 10 Minuten vergangen sind, bevor sie dem Benutzer eine Frage stellt.
