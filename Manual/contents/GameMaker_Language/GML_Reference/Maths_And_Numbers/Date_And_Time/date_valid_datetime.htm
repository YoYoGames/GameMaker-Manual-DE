date_valid_datetime

Mit dieser Funktion können Sie einen Datumswert daraufhin überprüfen, ob er gültig ist ( true) oder nicht ( false). Beachten Sie, dass diese Funktion nur Daten nach dem 1.1.1970 als gültig ansieht und alles davor false zurückgibt, so dass Sie frühestens ab dem 1.1.1970 prüfen können:

date_valid_datetime(1970, 01, 01, 0, 0, 0);

Syntax:

date_valid_datetime(year, month, day, hour, minute, second);

ArgumentTyp Beschreibung
year The year to check.
month The month to check.
day The day to check.
hour The hour to check.
minute The minute to check.
second The second to check.

Rückgabe:

Boolean

Beispiel:

if date_valid_datetime(2011, 9, 15, 10, 3, 30)
{
    mydatetime = date_create_datetime(2011, 9, 15, 10, 3, 30);
}

Dadurch wird "mydatetime" auf den 15. September 2011, 10:03.30 Uhr, gesetzt, sofern es sich um einen gültigen Wert handelt (was der Fall ist).
