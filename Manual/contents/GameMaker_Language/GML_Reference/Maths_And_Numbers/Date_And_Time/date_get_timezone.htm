date_get_timezone

Diese Funktion ermittelt die Basiszeitzone, die für alle anderen Datums- und Zeitfunktionen verwendet wird. Diese Zeitzone kann entweder lokal (wie vom System eingestellt) oder UTC sein, und die Funktion gibt eine der folgenden Konstanten zurück:

Zeitzonen-Konstante
Konstante Beschreibung
timezone_local die vom System eingestellte lokale Zeitzone verwenden
timezone_utc Koordinierte Weltzeit verwenden

Syntax:

date_get_timezone();

Rückgabe:

Time Zone Constant

Beispiel:

if (date_get_timezone() != timezone_utc)
{
    date_set_timezone(timezone_utc);
}

code prüft die Basiszeitzoneneinstellung für das Spiel und ändert sie, wenn sie nicht UTC ist.
