Datum_ink_Minute

Mit dieser Funktion können Sie einen gegebenen Datumswert um eine bestimmte Anzahl von Minuten erhöhen, und sie wird den neuen Datumswert zurückgeben.

Syntax:

date_inc_minute( date, amount );

ArgumentTyp Beschreibung
date The datetime to add to.
amount The number of minutes (must be an integer) to add.

Rückgabe:

Beispiel:

mynewdatetime = date_inc_minute(date_current_datetime(), 60);

Damit wird "mynewdatetime" auf das aktuelle Datum gesetzt, aber um 60 Minuten verlängert.
