datum_zeit_zeichenkette

Mit dieser Funktion können Sie eine Zeichenkette erstellen, die die angegebene Zeit enthält, formatiert für das System oder Gerät, auf dem das Spiel läuft, wenn die Funktion aufgerufen wird.

Syntax:

date_time_string( date );

ArgumentTyp Beschreibung
date The datetime to use.

Rückgabe:

Beispiel:

str = date_time_string(date_current_datetime());

Dies würde die angegebene Variable auf etwas wie "11:36.00" setzen, abhängig von den Systemeinstellungen für die Anzeige von Datum und Uhrzeit und dem aktuellen Datum und der aktuellen Uhrzeit.
