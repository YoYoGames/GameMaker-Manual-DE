Abstand_zum_Objekt

Diese Funktion berechnet den Abstand vom Rand der Bounding Box der aufrufenden Instanz zum nächstgelegenen Rand der nächstgelegenen Instanz des angegebenen Objekts. Das Objekt kann ein Objektindex oder eine bestimmte Instanz-ID sein, sowie die Schlüsselwort sein, und der Abstand wird in Pixel zurückgegeben. Beachten Sie, dass die Ergebnisse falsch sind, wenn eines der Objekte keine sprite oder keine Maske definiert hat.

Syntax:

distance_to_object(obj);

Argument Typ Beschreibung
obj Objekt Asset- oder Instanz-ID The object to check for.

Rückgabe:

Real

Beispiel:

if (distance_to_object(obj_Player) < range)
{
    canshoot = true;
}

Die obige code prüft den Abstand zum Spielerobjekt, und wenn dieser kleiner ist als der in der Variable "range" gespeicherte Wert, wird die Variable "canshoot" auf true gesetzt.
