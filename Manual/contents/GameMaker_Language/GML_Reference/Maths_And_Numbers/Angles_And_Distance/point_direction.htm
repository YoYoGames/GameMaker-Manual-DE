Punkt_Richtung

Diese Funktion gibt die Richtung von vector zurück, die durch die angegebenen Komponenten [x1,y1] und [x2,y2] im Verhältnis zu den festen x/y-Koordinaten des Raums gebildet wird. Wenn wir z.B. im folgenden Bild die Richtung von der Position des Spielerschiffs zur Position des Feindes ermitteln wollen, um eine Rakete auf den Feind abzufeuern, dann würden wir diese Funktion verwenden (das genaue code ist im Beispiel unter dem Bild):

Syntax:

point_direction(x1, y1, x2, y2)

ArgumentTyp Beschreibung
x1 The x coordinate of the first component of the vector
y1 The y coordinate of the first component of the vector
x2 The x coordinate of the second component of the vector
y2 The y coordinate of the second component of the vector

Rückgabe:

Beispiel:

var ex, ey;
ex = instance_nearest(x, y, enemy).x;
ey = instance_nearest(x, y, enemy).y;
with (instance_create_layer(x, y, "Bullets", obj_Missile))
{
    direction = point_direction(x, y, ex, ey);
}

Die obige code ermittelt die x- und y-Koordinaten des nächstgelegenen Gegners und übergibt sie dann an ein Bullet-Objekt, das in der Funktion point_direction verwendet wird, um seine Bewegungsrichtung korrekt einzustellen.
