distance_to_point

Diese Funktion berechnet den Abstand vom Rand der Bounding Box der aufrufenden Instanz zur angegebenen x/y-Position im Raum, wobei der Rückgabewert in Pixeln angegeben wird. Beachten Sie, dass die Ergebnisse falsch sind, wenn das aufrufende Objekt keine sprite oder keine Maske definiert hat.

Syntax:

distance_to_point(x, y)

ArgumentTyp Beschreibung
x The x position to check.
y The y position to check.

Rückgabe:

Beispiel:

if (distance_to_point(obj_Player.x, obj_Player.y) < range)
{
    can_shoot = true;
}

Die obige code prüft den Abstand zur x/y-Position des Spielerobjekts, und wenn dieser kleiner ist als der in der Variable "range" gespeicherte Wert, wird die Variable "canshoot" auf true gesetzt.
