darcsin

Gibt den inversen Sinus von x zurück, d. h. wenn dsin(x)=n, darcsin(n)=x und die resultierende Zahl zwischen -90 und 90 liegt.

HINWEIS: Es wird nur eine Zahl zwischen -1 und 1 akzeptiert (alles andere führt zu einem Fehler).

HINWEIS: Diese Funktion liefert einen Wert in Grad, nicht im Bogenmaß.

Syntax:

darcsin(x);

ArgumentTyp Beschreibung
val The value to return the inverse sine of.

Rückgabe:

Beispiel:

val = darcsin(-1);

Dadurch wird "val" auf -90 gesetzt.
