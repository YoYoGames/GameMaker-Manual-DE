dtan

In einem rechtwinkligen Dreieck ist dtan(val) = Gegenüberliegend / Angrenzend, wobei "val" einer der drei Winkel ist.

HINWEIS: Eine große Anzahl von Werten (z. B. 90 oder -90) führt bei dieser Funktion zu Fehlern, da sie unendlich sind; eine grafische Darstellung würde bei diesen Werten Asymptoten erzeugen.

HINWEIS: Diese Funktion nimmt einen Wert in Grad und nicht im Bogenmaß an.

Syntax:

dtan(val)

Argument Typ Beschreibung
val Real The angle (in degrees) to return the tangent of.

Rückgabe:

Real

Beispiel:

val = dtan(45);

Dadurch wird "val" auf 1 gesetzt.
