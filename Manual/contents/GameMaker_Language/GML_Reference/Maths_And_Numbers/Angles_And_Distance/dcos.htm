dcos

In einem rechtwinkligen Dreieck ist cos(val) = Anliegend / Hypotenuse, wobei val einer der drei Winkel ist. Diese Funktion gibt immer eine Zahl zwischen 1 und -1 zurück.

HINWEIS: Diese Funktion nimmt einen Wert in Grad, nicht im Bogenmaß an.

Syntax:

dcos(val);

ArgumentTyp Beschreibung
val The angle (in degrees) to return the cosine of.

Rückgabe:

Beispiel:

val = dcos(45);

Dadurch wird "val" auf 0,71 gesetzt.
