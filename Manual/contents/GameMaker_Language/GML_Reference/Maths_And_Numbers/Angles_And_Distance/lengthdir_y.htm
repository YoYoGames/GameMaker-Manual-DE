lengthdir_y

Diese Funktion wird verwendet, um die y-Komponente einer Position "len" Pixel vom Startpunkt und in Richtung "dir" zu erhalten. Wenn du dir einen Kreis um deine Instanz vorstellst, und dir dann einen Punkt irgendwo auf diesem Kreis vorstellst, müssen wir, um uns zu diesem Punkt zu bewegen zu diesem Punkt zu bewegen, müssen wir das Objekt um so viele Pixel in diese Richtung verschieben... also liefert diese Funktion (wenn sie mit lengthdir_x()) die Position dieses Punktes auf dem Kreis, die von der Instanz in code verwendet wird. Siehe das Bild unten für Details:

Syntax:

lengthdir_y(len, dir);

ArgumentTyp Beschreibung
len The length away of the point to return.
dir The direction of the point to return.

Rückgabe:

Beispiel:

var _xx = x + lengthdir_x(64, image_angle);
var _yy = y + lengthdir_y(64, image_angle);
instance_create_layer(_xx, _yy, "Bullets", obj_bullet);

Dadurch wird eine Aufzählungszeicheninstanz bei (_xx, _yy) erstellt, die 64 Pixel von der übergeordneten Instanz in Richtung des Bildwinkels entfernt ist.
