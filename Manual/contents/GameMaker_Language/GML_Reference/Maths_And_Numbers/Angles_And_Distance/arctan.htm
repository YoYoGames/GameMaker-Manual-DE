arctan

Gibt den inversen Tangens von x zurück. Dies akzeptiert eine beliebige Zahl, da im Gegensatz zu tan() die Asymptoten von arctan() auf der y-Achse liegen, was bedeutet, dass Sie nie eine Zahl zurückbekommen größer als pi/2 oder kleiner als -pi/2.

HINWEIS: Diese Funktion nimmt einen Wert in Radiant, nicht in Grad an.

Syntax:

arctan(x)

ArgumentTyp Beschreibung
x The angle (in radians) to return the inverse tangent of.

Rückgabe:

Beispiel:

val = arctan( 0 );

Dadurch wird val auf 0 gesetzt.
