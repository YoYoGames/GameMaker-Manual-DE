Punkt_Entfernung

Diese Funktion gibt die Länge von vector zurück, die durch die angegebenen Komponenten [x1,y1] und [x2,y2] gebildet wird. Wenn wir z.B. im folgenden Bild die Entfernung zwischen der Position des Spielerschiffs und der Position des Feindes ermitteln wollen, um zu berechnen, ob der Feind nahe genug ist, um auf ihn zu schießen, dann würden wir diese Funktion verwenden (das genaue code ist im Beispiel unter dem Bild):

Syntax:

point_distance(x1, y1, x2, y2)

ArgumentTyp Beschreibung
x1 The x coordinate of the first component of the vector
y1 The y coordinate of the first component of the vector
x2 The x coordinate of the second component of the vector
y2 The y coordinate of the second component of the vector

Rückgabe:

Beispiel:

var ex, ey;
ex = instance_nearest(x, y, enemy).x;
ey = instance_nearest(x, y, enemy).y;
if (point_distance(x, y, ex, ey) < 200)
{
    instance_create_layer(x, y, "Bullets", obj_Missile)
}

Die obige code ermittelt die x- und y-Koordinaten des nächstgelegenen Gegners und verwendet diese, um den Abstand (Länge) zwischen vector und den Spielerkoordinaten zu überprüfen. Wenn der Wert kleiner als 200 ist, erstellt das Spielerobjekt eine Instanz von "obj_Missile".
