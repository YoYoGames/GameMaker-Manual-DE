tan

In einem rechtwinkligen Dreieck ist tan(val) = Gegenüberliegend / Angrenzend, wobei val einer der drei Winkel ist.

HINWEIS Diese Funktion nimmt einen Wert in Radiant und nicht in Grad an.

HINWEIS: Pi/2, (3 pi/2), -pi/2 und eine große Anzahl anderer Werte werden bei dieser Funktion einen Fehler aufweisen, da sie ins Unendliche führen; eine grafische Darstellung würde bei diesen Werten Asymptoten erzeugen.

Syntax:

tan(val)

ArgumentTyp Beschreibung
val The angle (in radians) to return the tangent of.

Rückgabe:

Beispiel:

val = tan( pi );

Dadurch wird val auf 0 gesetzt.
