darctan

Gibt den inversen Tangens von x zurück. Dies akzeptiert jede Zahl, da im Gegensatz zu dtan() darctan() Asymptoten auf der y-Achse liegen, was bedeutet, dass Sie niemals eine Zahl zurückbekommen größer als 90 oder kleiner als -90.

HINWEIS: Diese Funktion liefert einen Wert in Grad, nicht im Bogenmaß.

Syntax:

darctan(x);

ArgumentTyp Beschreibung
val The value to return the inverse tangent of.

Rückgabe:

Beispiel:

val = darctan(1);

Dadurch wird "val" auf -45 gesetzt.
