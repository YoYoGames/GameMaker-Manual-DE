cos

In einem rechtwinkligen Dreieck ist cos(val) = Anliegend / Hypotenuse, wobei val einer der drei Winkel ist. Diese Funktion gibt immer eine Zahl zwischen 1 und -1 zurück.

HINWEIS: Diese Funktion nimmt einen Wert in Bogenmaß und nicht in Grad an.

Syntax:

cos(val);

ArgumentTyp Beschreibung
val The angle (in radians) to return the cosine of.

Rückgabe:

Beispiel:

val = cos( 0 );

Dadurch wird val auf 1 gesetzt.
