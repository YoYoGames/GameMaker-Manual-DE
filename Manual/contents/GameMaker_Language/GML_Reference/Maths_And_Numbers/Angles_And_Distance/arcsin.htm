arcsin

Gibt den inversen Sinus von x zurück, d. h. wenn sin(x)=n, arcsin(n)=x und die resultierende Zahl zwischen -pi/2 und pi/2 liegt.

HINWEIS: Hier können nur Werte zwischen -1 und 1 eingegeben werden (alles andere führt zu einem Fehler).

HINWEIS: Diese Funktion liefert einen Wert im Bogenmaß, nicht im Gradmaß.

Syntax:

arcsin(x)

ArgumentTyp Beschreibung
x The value to return the inverse sine of.

Rückgabe:

Beispiel:

val = arcsin(0);

Dadurch wird val auf 0 gesetzt.
