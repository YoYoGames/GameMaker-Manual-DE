angle_difference

Diese Funktion gibt die kleinste Differenz zwischen den beiden angegebenen Winkeln zurück, wobei die Differenz vom Ausgangswinkel zum Zielwinkel berechnet wird. Der zurückgegebene Wert liegt zwischen -180 und 180 Grad (wobei ein positiver Winkel gegen den Uhrzeigersinn und ein negativer Winkel im Uhrzeigersinn ist).

Die Beziehung zwischen dem "Quell"- und dem "Ziel"-Winkel ist so, dass die Differenz, die Sie durch den Aufruf von angle_difference(dest, src) erhalten, wenn Sie sie zum src-Wert addieren, den Zielwinkel ergibt (obwohl die genauen numerischen Werte abweichen können). Sie können eine ähnliche Technik verwenden, um den Quellwinkel schrittweise an den Zielwinkel anzunähern, indem Sie z. B. 10 % der zurückgegebenen Differenz zu src addieren, um ihn ein wenig in Richtung dest zu verschieben (dies wird im folgenden Beispiel demonstriert).

Syntax:

angle_difference(dest, src)

ArgumentTyp Beschreibung
dest The destination or target angle
src The source angle

Rückgabe:

Beispiel:

var _dir = point_direction(x, y, mouse_x, mouse_y);
var _diff = angle_difference(_dir, image_angle);
image_angle += _diff * 0.1;

Die obige code ermittelt die Richtung von der Instanz zum Mauszeiger, dann die Differenz zwischen diesem Winkel und dem aktuellen image_angleund verwendet diesen Wert, um die Instanz schrittweise in Richtung der Maus zu drehen.
