Manchmal möchte man etwas anderes als Zahlen für eine Zufallsauswahl angeben, oder die gewünschten Zahlen liegen nicht in einer bestimmten Reihenfolge oder in einem bestimmten Bereich. In diesen Fällen würden Sie choose() verwenden, um ein zufälliges Ergebnis zu erzeugen. Wenn Sie z. B. ein Objekt mit einem zufälligen sprite am Anfang erstellen möchten, können Sie diese Funktion verwenden, um den Index sprite auf einen aus einer Reihe von gegebenen sprites zu setzen. Beachten Sie, dass Sie so viele haben können, wie Sie benötigen (beachten Sie, dass mehr arguments bedeutet, dass die Funktion langsamer analysiert werden kann).

HINWEIS Diese Funktion gibt bei jedem neuen Durchlauf des Spiels denselben Wert zurück, da GameMaker jedes Mal denselben anfänglichen Zufallswert erzeugt, um debugging code zu vereinfachen. Um dieses Verhalten zu vermeiden, verwenden Sie randomise() beim Start des Spiels.

Syntax:

choose(val0, val1, val2... max_val);

Argument Typ Beschreibung
val0... max_val Variabel Any type of value(s).

Rückgabe:

Variable (One of the given arguments)

Beispiel:

sprite_index = choose(spr_Cactus, spr_Flower, spr_Tree, spr_Shrub);
hp = choose(5, 8, 15, 32, 40);
name = choose("John", "Steven", "Graham", "Jack", "Emily", "Tina", "Jill", "Helen");

Die obige code verwendet choose, um eine Reihe von Eigenschaften für die Instanz festzulegen.
