lerp

Mit dieser Funktion können Sie den Wert finden, der der Position zwischen zwei anderen Werten für einen bestimmten Prozentsatz entspricht. Wenn Sie also zum Beispiel:

lerp(0, 10, 0.5)

würden Sie den Rückgabewert 5 erhalten, was 50 % der Eingabewerte entspricht. Sie können mit dieser Funktion auch extrapolieren, indem Sie einen positiven oder negativen Wert für den Interpolationsbetrag angeben, so dass Sie etwas tun wie:

lerp(0, 10, 2)

einen Wert von 20 zurück.

Syntax:

lerp(a, b, amt)

ArgumentTyp Beschreibung
a The first value.
b The second value.
amt The amount to interpolate.

Rückgabe:

Beispiel:

xx = lerp(x, x + hspeed, room_speed);
yy = lerp(y, y + vspeed, room_speed);

Die obige code verwendet die lineare Interpolationsfunktion, um vorherzusagen, wohin sich eine Instanz nach einer Sekunde Spielzeit bewegt haben würde.
