random_get_seed

Um eine Zufallszahl zu erzeugen, beginnt GameMaker mit einer zufälligen Seed-Zahl. Mit dieser Funktion können Sie diesen Seed abrufen, der dann gespeichert werden kann, um eine bestimmte Serie von Zufallsereignissen zu reproduzieren. Nehmen wir an, Sie haben ein prozedural generiertes Spiel, aber Sie möchten, dass der Spieler in zuvor entdeckte Gebiete zurückkehren kann. Nun, zu Beginn eines jeden Gebiets kann man den Zufallswert mit dieser Funktion speichern und jedes Mal, wenn man dorthin zurückkehren muss, kann man mit random_set_seed() verwenden, um den Seed auf den vorherigen Wert zu setzen und GameMaker so zu zwingen, dieselbe Reihe von "Zufallsfunktionen" zu reproduzieren, die er zuvor verwendet hat.

Syntax:

random_get_seed();

Rückgabe:

Beispiel:

ini_open("Save.ini")
ini_write_real("Levels", string(level), random_get_seed());
ini_close();

Die obige code ermittelt den Zufallswert und speichert ihn in einer Ini-Datei.
