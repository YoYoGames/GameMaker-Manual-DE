irregulär

Diese sehr nützliche Funktion gibt nur ganze Zahlen zurück. Wenn Sie also zum Beispiel eine Zufallszahl von 0 bis 9 erhalten möchten, können Sie irandom(9) verwenden und erhalten eine Zahl von 0 bis einschließlich 9. Beachten Sie, dass auch Gleitkommazahlen verwendet werden können, aber der obere Wert nach dem Punkt wird ausgeschlossen, so dass irandom(9.7) nur einen Wert von 0 bis 9 zurückgibt. Die Funktion hat eine Obergrenze von $7fffffffffffffffLL, daher ist bei der Verwendung sehr großer Zahlen Vorsicht geboten.

HINWEIS Diese Funktion gibt bei jedem neuen Durchlauf des Spiels denselben Wert zurück, da GameMaker jedes Mal denselben anfänglichen Zufallswert erzeugt, um debugging code zu vereinfachen. Um dieses Verhalten zu vermeiden, verwenden Sie randomise() beim Start des Spiels.

Syntax:

irandom(n);

ArgumentTyp Beschreibung
n The upper range from which the random number will be selected.

Rückgabe:

Beispiel:

if (irandom(9) == 1)
{
    score += 100;
}

Die Wahrscheinlichkeit, dass die Punktzahl um 100 erhöht wird, liegt bei eins zu zehn (da 0 enthalten ist).
