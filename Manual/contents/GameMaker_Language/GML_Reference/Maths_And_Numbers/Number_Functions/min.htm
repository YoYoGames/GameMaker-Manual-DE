min

Diese Funktion gibt das Minimum der Eingabewerte zurück, von denen sie so viele haben kann, wie Sie benötigen (beachten Sie, dass mehr arguments bedeutet, dass die Funktion beim Parsen langsamer ist). Zum Beispiel wird min(12, 96, 32, 75) 12 zurückgeben, da dies der der niedrigste aller Eingabewerte ist.

Syntax:

min(val1, val2, ... max_val);

ArgumentTyp Beschreibung
val0 ... max_val The values to compare.

Rückgabe:

Beispiel:

x = min(x, room_width);

Dadurch wird der Spieler daran gehindert, den Raum auf der rechten Seite zu verlassen. Das funktioniert, indem er ständig sein x entweder auf sich selbst oder auf die Variable room_width setzt, je nachdem, was kleiner ist. Wenn der Spieler den rechten Teil des Raumes verlässt, ist sein x größer als room_width, also wird er direkt zurückgesetzt werden.
