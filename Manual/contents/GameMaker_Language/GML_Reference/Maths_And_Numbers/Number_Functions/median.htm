median

Diese Funktion gibt den Median der Eingabewerte zurück, d. h. den mittleren Wert. Wenn die Anzahl von arguments gerade ist, wird der kleinere der beiden mittleren Werte zurückgegeben, und die Funktion kann so viele arguments haben wie erforderlich (beachten Sie, dass mehr Argumente bedeutet, dass die Funktion langsamer analysiert werden kann), die alle reelle Werte sein müssen. Das bedeutet, dass zum Beispiel median(43, 12, 25, 19, 6) 19 zurückgeben würde, da es der mittlere Wert zwischen allen anderen ist.

Syntax:

median(val1, val2, ... max_val)

ArgumentTyp Beschreibung
val0 ... max_val The values to compare.

Rückgabe:

Beispiel:

x = median( 0, x, room_width );
y = median( 0, y, room_height );

Dadurch wird der Spieler daran gehindert, eine beliebige Seite des Raums zu verlassen, indem der Median als Klammer verwendet wird. Bewegt sich der Spieler z.B. nach links von der Raumgrenze weg, ist sein x kleiner als 0. Das bedeutet, dass die mittlere Zahl des ersten Medians 0 sein, so dass der Spieler auf (0,y) gesprungen wird.
