randomisieren

Diese Funktion setzt den Seed auf einen Zufallswert. Wenn Sie einen konstanten Wert über mehrere Durchläufe eines Spiels benötigen, sollten Sie die Funktion random_set_seed(). Bitte beachten Sie, dass bei der Verwendung der Zufallszahlen Funktionen im GameMaker immer derselbe Anfangswert verwendet wird, da dies die Fehlersuche und debugging erheblich erleichtert. Wenn Sie mit echtem Zufall testen wollen, sollten Sie diese Funktion zu Beginn Ihres Spiels aufrufen. Die Funktion wird gibt den neuen Zufallswert zurück (eine 32-Bit-Ganzzahl ohne Vorzeichen).

Syntax:

randomise();
// or
randomize();

Rückgabe:

(unsigned 32 bit value)

Beispiel:

randomise();

Die obige code wird den Samen zufällig verteilen.
