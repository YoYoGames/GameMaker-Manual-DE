sqr

Multipliziert eine Zahl mit sich selbst und gibt somit das Quadrat dieser Zahl zurück. EG: sqr(5) würde 25 ergeben, da 5*5=25.

Syntax:

sqr(val);

ArgumentTyp Beschreibung
val The number to square.

Rückgabe:

Beispiel:

score += sqr(dmg);

Die obige code addiert das Quadrat der Variable "dmg" zur Punktzahl.
