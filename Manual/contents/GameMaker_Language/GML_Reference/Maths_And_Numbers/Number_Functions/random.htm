zufällig

Diese Funktion ist gut für Wahrscheinlichkeiten geeignet, bei denen die Rückgabe einer ganzen Zahl nicht erforderlich ist. Zum Beispiel gibt random(100) einen Wert zwischen 0 und 99 zurück, aber dieser Wert kann 22,56473 sein! Sie können auch reelle Zahlen und keine ganzen Zahlen in dieser Funktion verwenden, wie z. B. random(0.5), die einen Wert zwischen 0 und 0,4999999 zurückgibt.

HINWEIS: Diese Funktion gibt bei jedem neuen Durchlauf des Spiels denselben Wert zurück, da GameMaker jedes Mal denselben anfänglichen Zufallswert erzeugt, um debugging code zu vereinfachen. Um dieses Verhalten zu vermeiden, verwende randomise() beim Start des Spiels.

Syntax:

random(n);

ArgumentTyp Beschreibung
n The upper range from which the random number will be selected.

Rückgabe:

Beispiel:

if (random(10) >= 9)
{
    score += 100;
}

Die Chance, die Punktzahl um 100 zu erhöhen, liegt damit bei etwa eins zu zehn.
