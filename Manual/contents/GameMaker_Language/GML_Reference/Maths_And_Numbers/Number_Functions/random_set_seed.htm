random_set_seed

Um eine Zufallszahl zu erzeugen, startet GameMaker mit einer zufälligen Seed-Zahl. Mit dieser Funktion können Sie diesen Seed auf einen bekannten Wert setzen und so "erzwingen", dass der Ausgang aller Zufallsereignisse danach bei jedem Programmdurchlauf gleich ist. Diese Funktion kann zum Beispiel in Verbindung mit random_get_seed() verwendet werden, um prozedural generierte Inhalte zu erstellen und die Ergebnisse zu speichern, ohne große Savegames zu benötigen (man speichert nur den Seed, alles andere ist überflüssig). Sollten Sie wirklich zufällige Ergebnisse für alles benötigen, sollten Sie die randomise() Funktion verwenden.

HINWEIS Obwohl dieses Saatgut auf jedem Ziel platform konsistente Ergebnisse liefert, können die Ergebnisse zwischen platforms aufgrund der unterschiedlichen Funktionsweise jedes Ziels variieren.

Syntax:

random_set_seed(val);

ArgumentTyp Beschreibung
val The seed to set.

Rückgabe:

N/A

Beispiel:

if debug
{
    random_set_seed(1);
}

Die obige code setzt den Zufallswert nur dann auf 1, wenn die Variable "debug" wahr ist.
