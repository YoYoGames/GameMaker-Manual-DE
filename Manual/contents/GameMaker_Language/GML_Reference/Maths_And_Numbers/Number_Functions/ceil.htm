Decke

Diese Funktion nimmt eine beliebige reelle Zahl und rundet sie auf die nächste ganze Zahl auf. Bei dieser Funktion ist Vorsicht geboten, da ein häufiger Fehler darin besteht, einen Zufallswert aufzurunden und zu erwarten, dass er immer größer als 1 ist, d. h:

int = ceil(random(5));

Nun würde man erwarten, dass code immer eine ganze Zahl zwischen 1 und 5 ergibt, aber das ist nicht immer der Fall, denn es besteht eine sehr geringe Wahrscheinlichkeit, dass die Zufallsfunktion 0 zurückgibt, und das Aufrunden von 0 ergibt immer noch 0. Dies ist eine geringe Möglichkeit, sollte aber bei der Verwendung dieser Funktion in Betracht gezogen werden.

Syntax:

ceil(val);

ArgumentTyp Beschreibung
x The number to change.

Rückgabe:

Beispiel:

val = ceil( 3.4 );

Damit wird val auf 4 gesetzt.
