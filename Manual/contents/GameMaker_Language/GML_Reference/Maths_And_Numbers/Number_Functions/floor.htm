Boden

Gibt die Untergrenze von n zurück, d. h. n abgerundet auf eine ganze Zahl. Dies ist ähnlich wie die Funktion round() Funktion, aber sie rundet nur ab, egal wie hoch der Dezimalwert ist. floor(5.99999) gibt also 5 zurück, ebenso wie floor(5.2), floor(5.6457) usw...

Syntax:

floor(n);

ArgumentTyp Beschreibung
n The number to floor.

Rückgabe:

Beispiel:

val = floor( 3.9 );

Damit wird val auf 3 gesetzt.
