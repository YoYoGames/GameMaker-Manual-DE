random_range

Diese Funktion gibt eine Zufallszahl innerhalb des angegebenen Bereichs zurück, wobei dieser Rückgabewert keine ganze Zahl sein muss. Zum Beispiel gibt random_range(20,50) eine Zufallszahl zwischen 20 und 50 zurück, aber der Wert kann eine reelle Zahl wie 38,65265 sein. Als Eingabe können auch reelle Zahlen verwendet werden arguments.

HINWEIS: Diese Funktion gibt bei jedem neuen Durchlauf des Spiels denselben Wert zurück, da GameMaker jedes Mal denselben anfänglichen Zufallswert erzeugt, um debugging code zu vereinfachen. Um dieses Verhalten zu vermeiden, verwende randomise() beim Start des Spiels.

Syntax:

random_range(n1, n2);

ArgumentTyp Beschreibung
n1 The low end of the range from which the random number will be selected.
n2 The high end of the range from which the random number will be selected.

Rückgabe:

Beispiel:

score += random_range(500, 600);

Dadurch wird die Gesamtpunktzahl zwischen 500 und 600 Punkten erhöht.
