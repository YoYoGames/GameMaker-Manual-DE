Zeichen

Diese Funktion gibt zurück, ob eine Zahl positiv, negativ oder keines von beiden ist und gibt jeweils 1, -1 oder 0 zurück. Zum Beispiel: sign(458) gibt 1 zurück, sign(-5) gibt -1 zurück und sign(0) gibt 0 zurück.

Syntax:

sign(n);

ArgumentTyp Beschreibung
n The number to get the sign of.

Rückgabe:

Beispiel:

y += sign(y - mouse_y);

Die obige code addiert 1, -1 oder 0 zu y, je nach dem Ergebnis von y - mouse_y.
