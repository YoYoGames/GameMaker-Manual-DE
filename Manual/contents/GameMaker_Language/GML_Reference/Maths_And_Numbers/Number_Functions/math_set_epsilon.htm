math_set_epsilon

Epsilon ist ein Wert, der verwendet wird, um festzustellen, ob zwei Zahlen, die einem Rundungsfehler unterliegen, nahe genug beieinander liegen, um als "gleich" zu gelten. Er ist nützlich, wenn man mit Fließkommamathematik zu tun hat, da er die "Rundungsfehler" reduzieren kann, die dazu führen, dass bestimmte Operationen Werte zurückgeben, die falsch oder entgegen den Erwartungen erscheinen. Wir haben zum Beispiel einen Wert zum Bildindex einer sprite addiert und erwarten, dass das Ergebnis eine einzelne ganze Zahl ist, aber aufgrund der Natur der Fließkommamathematik endet der tatsächliche Endwert bei etwa 5,0000002, wenn wir also die folgende Prüfung durchführen:

if image_index == 5
{
    //do something
}

Der Wert code verhält sich nicht wie erwartet und gibt false zurück. Wenn wir jedoch den Epsilon-Wert auf 0,000001 setzen, wird der Wert image_index auf die nächste reelle Zahl gerundet, die +/- 0,000001 des ursprünglichen Wertes beträgt, so dass der obige Vergleich wahr ergibt.

Der Epsilon-Wert wird für alle folgenden ganzzahligen Operatoren verwendet:

  • < : Weniger als
  • > : Größer als
  • == : Gleichwertig mit
  • <= : Kleiner als oder gleich
  • >= : Größer als oder gleich
  • != : Nicht gleich

Beachten Sie, dass bei einem Epsilon-Wert von 0 alle Rundungen deaktiviert werden und bei einem Wert von 1 ein Fehler auftritt.

Syntax:

math_set_epsilon(epsilon);

Argument Typ Beschreibung
epsilon Real The new epsilon value (from 0 to 0.999999999).

Rückgabe:

Real

Beispiel:

math_set_epsilon(0.0001);

Damit wird der Epsilonwert für alle weiteren Gleitkommaoperationen festgelegt.
