Klammer

Mit dieser Funktion können Sie einen Eingangswert in einem bestimmten Bereich halten.

Syntax:

clamp(val, min, max)

ArgumentTyp Beschreibung
val The value to clamp.
min The minimum value to clamp between.
max The maximum value to clamp between.

Rückgabe:

Beispiel:

speed = clamp(speed, 1, 10);

Die obige code wird die Geschwindigkeit so einschränken, dass sie nie unter 1 fällt oder über 10 geht.
