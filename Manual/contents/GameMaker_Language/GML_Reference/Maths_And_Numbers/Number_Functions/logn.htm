logn

Diese Funktion ist ähnlich wie die Funktionen log2(n) und log10(n), nur dass Sie den logarithmischen Basiswert angeben. Zum Beispiel gibt logn(5,25) zurück, wie viele 5er wir multiplizieren müssen zu multiplizieren, um 25 zu erhalten (was 2 ist).

Syntax:

logn(n, val)

ArgumentTyp Beschreibung
n The log base.
val The input value.

Rückgabe:

Beispiel:

logval = logn(5, num);

Die obige code ruft die log5 des in "num" gespeicherten Wertes ab.
