math_get_epsilon

Diese Funktion gibt den aktuellen Epsilon-Wert für das Ziel platform zurück. Weitere Informationen zu epsilon finden Sie in der Funktion math_set_epsilon().

Syntax:

math_get_epsilon();

Rückgabe:

Beispiel:

var e = math_get_epsilon();
if e != 0.000001
{
    math_set_epsilon(0.000001);
}

Dadurch wird der aktuelle Epsilonwert abgerufen und in einer lokalen Variablen gespeichert, die dann überprüft und bei Bedarf ein neuer Epsilonwert festgelegt wird.
