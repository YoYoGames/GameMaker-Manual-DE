ln

Der natürliche Logarithmus ln(n) gibt die Zeit an, die benötigt wird, um ein bestimmtes Niveau des kontinuierlichen Wachstums zu erreichen, wobei n das erreichte Niveau ist. Wenn wir also herausfinden wollen, wie viele Zeiteinheiten wir brauchen, um ein Wachstum von 20 zu erreichen, würden wir ln(20) verwenden. 2,99 Zeiteinheiten zurückgibt, um dieses Wachstum zu erreichen.

Syntax:

ln(n);

ArgumentTyp Beschreibung
n The input value.

Rückgabe:

Beispiel:

alarm[0] = ln(age) * room_speed;

Die obige code verwendet den natürlichen Logarithmus des in der Variablen "age" gespeicherten Wertes, um einen Alarm auszulösen.
