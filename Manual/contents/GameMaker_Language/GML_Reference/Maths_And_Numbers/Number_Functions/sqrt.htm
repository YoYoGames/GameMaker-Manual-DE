sqrt

Wenn man eine Zahl mit sich selbst multipliziert, erhält man das Quadrat dieser Zahl, aber manchmal möchte man auch den umgekehrten Weg gehen und die Quadratwurzel aus einer Zahl erhalten. Um also herauszufinden, welche Zahl mit sich selbst multipliziert wurde, um einen bestimmten positiven Wert zu erhalten, verwenden wir diese Funktion. Zum Beispiel: sqrt(9) liefert 3, da 3*3=9.

Syntax:

sqrt(val);

ArgumentTyp Beschreibung
val The number to get the square root of.

Rückgabe:

Beispiel:

num = sqrt(exp);

Die obige code setzt die Variable "num" so, dass sie die Quadratwurzel des in "exp" enthaltenen Wertes enthält.
