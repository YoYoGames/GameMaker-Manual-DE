is_undefined

Diese Funktion gibt zurück, ob eine bestimmte Variable definiert ist oder nicht. In manchen Fällen möchten Sie überprüfen, welchen Datentyp eine Variable in GameMaker hat, und dann würden Sie diese Funktion verwenden. Sie gibt true oder false zurück, je nachdem, ob der Wert definiert ist oder nicht.

HINWEIS: Diese Funktion kann nicht verwendet werden, um das Vorhandensein einer Variablen zu überprüfen. Verwenden Sie variable_instance_exists() oder variable_global_exists() stattdessen.

Syntax:

is_undefined(n);

Argument Typ Beschreibung
n Variabel The argument to check.

Rückgabe:

Boolean

Beispiel:

var val = ds_map_find_value(map, 13);
if is_undefined(val)
{
    show_debug_message("Map entry does not exist!");
}

Die obige code prüft die Variable "val", ob sie undefiniert ist oder nicht, und zeigt eine Debug-Meldung an, wenn dies der Fall ist.
