array_length

Mit dieser Funktion können Sie die Länge (Anzahl der Einträge) einer Array-Dimension ermitteln. Sie geben den Array-Indexwert an und die Funktion gibt einen ganzzahligen Wert zurück, der die Anzahl der Einträge in dem Array angibt. Diese Funktion kann auch für mehrdimensionale Arrays verwendet werden, solange Sie bei der Angabe des Array-Indexes angeben, welche Dimension Sie abfragen möchten:

// Returns the first dimension of the array
val = array_length(my_array);

// Returns the second dimension of the array
val = array_length(my_array[0]);

// Returns the third dimension of the array
val = array_length(my_array[0][0]);

// etc.

Syntax:

array_length(array_index);

ArgumentTyp Beschreibung
array_index The index of the array to check.

Rückgabe:

Beispiel:

for (var i = 0; i < array_length(a); ++i)
{
    a[i] = -1;
}

code Die obige loop wird durch ein Array gehen und jeden Eintrag auf -1 setzen.
