variable_instance_get

Mit dieser Funktion können Sie den Wert einer bestimmten benannten Variablen abrufen. Sie geben den eindeutigen ID-Wert der Instanz an (der in den Instanzeigenschaften im Raumeditor zu finden ist oder zurückgegeben wird, wenn Sie die Funktion instance_create_layer()) sowie den Namen der Variable, deren Wert Sie abrufen möchten, als String (siehe Beispiel code unten). Die Funktion gibt den Wert zurück, den die Variable hat, oder undefined, wenn die Variable nicht existiert.

WICHTIG! Wenn die Variable, die Sie erhalten, nicht existiert, gibt die Funktion das Schlüsselwort undefined zurück, und Sie können Fehler erhalten, die das Spiel zum Stillstand bringen, also verwenden Sie im Zweifelsfall zuerst die Funktion variable_instance_exists zuerst.

Syntax:

variable_instance_get(instance_id, name);

Argument Typ Beschreibung
instance_id Instanz-ID The unique ID value of the instance to use
name Zeichenfolge The name of the variable to get (as a string)

Rückgabe:

Variable (any data type) or undefined (if the named variable does not exist)

Beispiel:

if variable_instance_exists(id, "shields")
{
    var ss = variable_instance_get(id, "shields");
}
else
{
    var ss = -1;
}

Die obige code prüft, ob eine Variable existiert, und wenn ja, wird der Wert, den sie enthält, abgerufen und in einer lokalen Variable gespeichert. Wenn sie nicht existiert, wird die lokale Variable auf -1 gesetzt.
