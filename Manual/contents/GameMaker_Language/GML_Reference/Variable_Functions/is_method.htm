is_method

Diese Funktion kann verwendet werden, um zu prüfen, ob eine Variable eine Methodenvariable ist (sie gibt true zurück) oder nicht (in diesem Fall gibt sie false zurück).

Syntax:

is_method(n);

Argument Typ Beschreibung
n Variabel The variable to check.

Rückgabe:

Boolean

Beispiel:

if is_method(get_vec)
{
    show_debug_message("Method variable!");
}

Die obige code prüft eine Variable, um zu sehen, ob sie eine Methode ist, und wenn die Funktion true zurückgibt, wird eine Debug-Meldung auf der Konsole ausgegeben.
