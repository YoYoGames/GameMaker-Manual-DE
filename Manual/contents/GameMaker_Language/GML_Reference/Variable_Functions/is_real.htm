ist_wirklich

Diese Funktion gibt zurück, ob eine gegebene Variable eine reelle Zahl (einfach, doppelt oder ganzzahlig) ist oder nicht. In manchen Fällen möchten Sie prüfen, ob eine Variable eine reelle Zahl enthält, und dann würden Sie diese Funktion verwenden. Sie gibt nicht die reelle Zahl zurück, sondern true oder false, d. h. ein Wert von z. B. "Fisch" würde false zurückgeben, ein Wert von 200 jedoch true.

HINWEIS: Diese Funktion gibt false für alle enum Werte zurück, da diese als int64s gespeichert sind. Die richtige Funktion zur Überprüfung dieses Datentyps ist is_int64().

Syntax:

is_real(n);

Argument Typ Beschreibung
n Variabel The argument to check.

Rückgabe:

Boolean

Beispiel:

if is_real(val)
{
    score += val;
}

Die obige code prüft, ob die Variable "val" eine reelle Zahl ist, und wenn ja, wird sie zur Punktzahl addiert.
