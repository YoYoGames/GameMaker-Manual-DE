array_insert

Mit dieser Funktion können Sie einen Wert (oder mehrere Werte) an einer beliebigen Position in ein Array einfügen. Für die Funktion müssen Sie eine Variable angeben, die das Array enthält, den Index (die Position) im Array, an der eingefügt werden soll, sowie mindestens einen einzufügenden Wert. Optional können Sie weitere arguments angeben, die dann alle ab dem angegebenen Index nacheinander in das Array eingefügt werden.

Beachten Sie, dass, wenn der angegebene Index außerhalb der Länge des Arrays liegt, die Werte am angegebenen Index hinzugefügt werden und alle leeren Slots im Array zwischen dem letzten Wert und den eingefügten Werten auf einen Standardwert von 0 gesetzt werden.

Syntax:

array_insert(variable, index, value, [value], [value], [etc...]);

Argument Typ Beschreibung
variable Array The variable that holds the array.
index Real The index (position) in the array to insert the value(s) 
value Variabel The value to insert
[value], [value], [etc...] Variabel OPTIONAL Weitere Werte zum Einfügen in das Array

Rückgabe:

N/A

Beispiel:

array_insert(score_array, score_rank, current_score);

Die obige code fügt die aktuelle Punktzahl in das Punktzahl-Array ein, und zwar an dem durch die Variable score_rank definierten Index.
