ist_ptr

Diese Funktion gibt zurück, ob eine gegebene Variable ein Zeiger ist oder nicht. In manchen Fällen möchten Sie überprüfen, welchen Datentyp eine Variable in GameMaker hat, und dann würden Sie diese Funktion verwenden. Sie gibt true oder false zurück, je nachdem, ob der Wert ein Zeiger ist oder nicht.

Syntax:

is_ptr(n);

Argument Typ Beschreibung
n Variabel The argument to check.

Rückgabe:

Boolean

Beispiel:

if !is_ptr(val)
{
    show_debug_message("Not a valid texture!");
}

Die obige code überprüft die Variable "val", um zu sehen, ob sie einen Zeiger enthält, und wenn nicht, wird eine Meldung in der Debug-Konsole angezeigt.
