array_length_1d DEPRECATED

WARNUNG! Diese Funktion ist veraltet (und wurde durch array_length() ersetzt), da Arrays nicht mehr auf 1 oder 2 Dimensionen beschränkt sind und diese Funktion daher nur zur Unterstützung älterer Projekte bereitgestellt wird. Alle neuen Projekte sollten die aktuelle Art der Erstellung und des Zugriffs auf mehrdimensionale Arrays verwenden (siehe hier für weitere Informationen).

Mit dieser Funktion können Sie die Länge (Anzahl der Einträge) eines 1D-Arrays ermitteln. Für 2D-Arrays sollten Sie die Funktionen array_height_2d und array_length_2d Funktionen verwenden.

WICHTIG!: Wenn das Array mehr als 32.000 Einträge hat, gibt diese Funktion einen fehlerhaften Wert zurück und sollte nicht verwendet werden.

Syntax:

array_length_1d(array_index);

Argument Typ Beschreibung
array_index Array The index of the array to check.

Rückgabe:

Real

Beispiel:

for (var i = array_length_1d(a) - 1; i > -1; i--;)
{
    a[i] = -1;
}

code Die obige loop wird durch ein Array gehen und jeden Eintrag auf -1 setzen.
