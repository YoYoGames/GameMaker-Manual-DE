is_struct

Diese Funktion prüft, ob der übergebene Wert eine Struktur ist. Ist dies der Fall, gibt sie true zurück, andernfalls false.

Beachten Sie, dass Methodenvariablen auch true und Objektinstanzen false zurückgeben.

Syntax:

is_struct(val);

Argument Typ Beschreibung
val Variabel The value to check.

Rückgabe:

Boolean

Beispiel:

if is_struct(a)
{
    delete(a);
}

Die obige code prüft eine Variable, um festzustellen, ob es sich um eine struct handelt, und wenn die Funktion true zurückgibt, wird die struct gelöscht.
