is_nan

Diese Funktion gibt zurück, ob eine gegebene Variable NaN (keine Zahl) ist oder nicht, und gibt true zurück, wenn sie es ist, und false, wenn sie es nicht ist.

Syntax:

is_nan(n);

Argument Typ Beschreibung
n Variabel The argument to check.

Rückgabe:

Boolean

Beispiel:

if is_nan(global.value)
{
    show_debug_message("Value is not a number");
}

Die obige code prüft die globale Variable "value", um zu sehen, ob sie eine Zahl ist oder nicht, und zeigt eine Debug-Meldung an, wenn sie es nicht ist.
