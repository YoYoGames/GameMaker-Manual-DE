array_sortieren

Mit dieser Funktion können Sie ein Array in aufsteigender oder absteigender Reihenfolge sortieren oder eine benutzerdefinierte Funktion verwenden, um die Sortierreihenfolge zu definieren. Die Funktion erfordert, dass Sie das zu sortierende Array angeben und dann eine der folgenden Optionen wählen:

  • Die Konstanten true oder false zur Angabe einer aufsteigenden (true) oder absteigenden (false) Sortierreihenfolge
  • Eine benutzerdefinierte Funktion zur Festlegung der Sortierreihenfolge

Diese Funktion ändert den Inhalt des ursprünglich übergebenen Arrays. Um eine sortierte Kopie zu erstellen, verwenden Sie array_copy(), um das Array zunächst zu duplizieren, und übergeben Sie es dann an diese Funktion.

Verwendung einer benutzerdefinierten Funktion

Wenn Sie eine benutzerdefinierte Funktion für die Sortierung verwenden, muss diese 2 Argumente annehmen, die jeweils die Werte des aktuellen Elements und des nächsten Elements erhalten. Die Funktion sollte einen der folgenden Werte zurückgeben, um die Sortierreihenfolge zu beeinflussen (wobei elm1 das aktuelle Element und elm2 das nächste Element ist):

  • 0Wenn elm1 und elm2 gleich sind
  • <= -1 (negative ganze Zahl): wenn elm1 vor elm2
  • >= 1 (positive ganze Zahl): wenn elm1 nach elm2

HINWEIS Der von Ihrer benutzerdefinierten Funktion zurückgegebene Wert muss eine ganze Zahl sein. Fließkommazahlen, die kleiner als 1 sind, werden als 0 gelesen, daher müssen Sie sign() oder round() verwenden, wenn Ihre Funktion mit Fließkommazahlen arbeitet.

Wenn das Array eine Reihe von Zeichenketten enthält, werden die Zeichenketten alphabetisch nach dem englischen Alphabet sortiert, wenn die Standard-Sortierart aufsteigend/absteigend verwendet wird. Alle anderen Datentypen werden auf der Grundlage ihres numerischen Wertes sortiert, wobei die genauen Werte vom Datentyp selbst abhängen (z. B. würde ein Array von buffers auf der Grundlage des numerischen Wertes ihrer Speicheradressen sortiert werden).

Syntax:

array_sort(variable, sorttype_or_function);

Argument Typ Beschreibung
variable Array The variable that holds the array.
sorttype_or_function  Boolesche oder Skript Funktion/Methode The sort type (true für aufsteigend oder false für absteigend) oder einen Funktionsverweis, der für die Sortierung verwendet wird.

Rückgabe:

N/A

Beispiel:

var _a = [10, 9, 8, 7, 6, 5];

array_sort(_a, function(elm1, elm2)
{
    return elm1 - elm2;
});

Die obige code verwendet eine benutzerdefinierte Sortierfunktion, um das Array in aufsteigender Reihenfolge zu sortieren. Die Sortierfunktion nimmt die beiden Array-Elemente elm1 und elm2 und führt eine Subtraktion an ihnen durch. Wenn elm1 größer als elm2 ist, ergibt die Subtraktion eine positive Zahl, die elm1 nach elm2 verschiebt und sie somit in eine aufsteigende Reihenfolge bringt.

Hier ist ein erweitertes Beispiel:

var _xx, _yy, _a;

for (var i = 0; i < 10; i++)
{
    _xx = irandom(room_width);
    _yy = irandom(room_height);
    _a[i] = instance_create_layer(_xx, _yy, layer, obj_Bullet);
}

show_debug_message(_a);

var _f = function(inst1, inst2)
{
    return inst1.x - inst2.x;
}

array_sort(_a, _f);
show_debug_message(_a);

Die obige code wird ein Array von 10 Instanzen erstellen, die an zufälligen Positionen im Raum platziert werden. Die Debug-Meldung zeigt etwas wie das Folgende:

[ 100003,100004,100005,100006,100007,100008,100009,100010,100011,100012 ]

code erstellt dann eine Methode, die in der Funktion array_sort() verwendet wird und die die Differenz zwischen den einzelnen X-Positionen der Instanzen im Raum zurückgibt. Dann wird die Funktion array_sort() unter Verwendung dieser Methode ausgeführt, wobei die Instanzen nach X vom niedrigsten zum höchsten Wert sortiert werden. Die Ausgabe für das Array würde dann etwa so aussehen:

[ 100011,100004,100009,100006,100012,100008,100010,100005,100007,100003 ]
