variable_instance_exists

Mit dieser Funktion können Sie prüfen, ob eine Instanzbereichsvariable existiert oder nicht. Sie geben den eindeutigen Wert der Instanz-ID (den Sie in den Instanzeigenschaften im Raumeditor finden können oder der zurückgegeben wird, wenn Sie die Funktion instance_create_layer()) sowie den Variablennamen, auf den geprüft werden soll , als String (siehe Beispiel code unten). Die Funktion gibt true zurück, wenn eine Variable mit dem angegebenen Namen für die Instanz existiert, und andernfalls false.

Syntax:

variable_instance_exists(instance_id, name);

Argument Typ Beschreibung
instance_id Instanz-ID The unique ID value of the instance to check
name Zeichenfolge The name of the variable to check for

Rückgabe:

Boolean

Beispiel:

if !variable_instance_exists(id, "shields")
{
    shields = 0;
}

Die obige code prüft, ob die Variable "shields" in der Instanz, auf der code läuft, vorhanden ist. Ist dies nicht der Fall, wird sie erstellt und mit 0 initialisiert.
