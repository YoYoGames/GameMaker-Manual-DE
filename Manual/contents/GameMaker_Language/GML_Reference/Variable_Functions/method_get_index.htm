methode_get_index

Mit dieser Funktion können Sie den Skriptindex für die script abrufen, in der die Methode definiert wurde. Wenn die Methode nicht in einer script definiert wurde, gibt die Funktion -1 zurück, andernfalls den Indexwert für die script.

Syntax:

method_get_index(method);

Argument Typ Beschreibung
method Methode The method variable to check

Rückgabe:

Script Asset or Real

Beispiel:

var _index = method_get_index(light_setup);
if _index != undefined
{
    show_debug_message(script_get_name(_index));
}

Die obige code prüft den Index script für die Methode "light_setup" und gibt den Namen script auf der Konsole aus, wenn er nicht -1 ist.
