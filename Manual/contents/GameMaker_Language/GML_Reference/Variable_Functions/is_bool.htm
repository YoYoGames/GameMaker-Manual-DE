is_bool

Diese Funktion gibt zurück, ob eine gegebene Variable ein boolescher Wert ist (true ioder false) oder nicht. In manchen Fällen möchten Sie prüfen, ob eine Variable in GameMaker einen booleschen Wert enthält, und dann würden Sie diese Funktion verwenden.

Syntax:

is_bool(n);

Argument Typ Beschreibung
n Variabel The argument to check.

Rückgabe:

Boolean

Beispiel:

if is_bool(val)
{
    global.Sound = val;
}
else
{
    global.Sound = true;
}

Die obige code prüft die Variable "val", um zu sehen, ob sie eine reelle Zahl ist, und wenn ja, verwendet sie sie, um eine globale Variable zu setzen.
