ptr

Diese Funktion versucht, einen gegebenen Wert in einen Zeigerdatentyp zu konvertieren, wobei der Wert entweder ein real, ein string, ein int64, ein int32 oder ein ptr sein muss. Alles andere führt zum Absturz des Spiels mit einer Fehlermeldung.

Syntax:

ptr(n);

Argument Typ Beschreibung
n Real, String, oder Zeiger The value to convert.

Rückgabe:

Pointer

Beispiel:

if !is_ptr(val)
{
    val = ptr(application_surface);
}

Die obige code prüft die Variable "val", um zu sehen, ob sie einen Zeiger enthält, und wenn nicht, wird ihr einer zugewiesen.
