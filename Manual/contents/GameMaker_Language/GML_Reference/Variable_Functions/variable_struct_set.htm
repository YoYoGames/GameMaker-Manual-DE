struct_set

Mit dieser Funktion können Sie den Wert einer bestimmten Variablen in einer Struktur setzen. Sie geben den Verweis auf die Struktur sowie den Namen der Variablen, deren Wert gesetzt werden soll, als Zeichenkette an (siehe Beispiel code unten), und schließlich den zu setzenden Wert (kann jeder gültige Datentyp sein). Wenn die Variable nicht bereits in der Struktur vorhanden ist, wird sie erstellt und dann der Wert zugewiesen.

Syntax:

struct_set(struct, name, val);

Argument Typ Beschreibung
struct Struktur The struct reference to set
name Zeichenfolge The name of the variable to set (as a string)
val Variabel The value to set the variable to

Rückgabe:

N/A

Beispiel:

if !variable_struct_exists(mystruct, "shields")
{
    variable_struct_set(mystruct, "shields", 0);
}

Die obige code prüft, ob die angegebene Variable in der angegebenen Struktur existiert, und wenn nicht, wird sie erstellt und auf 0 gesetzt.
