int64

Diese Funktion versucht, einen gegebenen Wert in einen vorzeichenbehafteten 64-Bit-Integer zu konvertieren, wobei der Wert entweder ein Real, ein String, ein int64, ein int32 oder ein ptr sein muss. Alles andere führt zum Absturz des Spiels mit einer Fehlermeldung. Sie können überprüfen, ob eine Variable einen int64 enthält, indem Sie die Funktion is_int64().

Syntax:

int64(val);

Argument Typ Beschreibung
val Real, Zeichenkette, oder Zeiger The value to convert.

Rückgabe:

int64 (signed 64-bit integer)

Beispiel:

steam_handle = int64(global.fileReadString);

Die obige code wandelt den Wert der globalen Variable in eine 64-Bit-Ganzzahl um.
