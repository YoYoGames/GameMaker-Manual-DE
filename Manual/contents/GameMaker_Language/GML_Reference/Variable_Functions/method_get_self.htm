methode_get_self

Mit dieser Funktion können Sie die Instanz-ID oder die Strukturreferenz abrufen, die den self Kontext, der verwendet wird, wenn die Methode aufgerufen wird. Wenn die Variable keine Methode ist, gibt die Funktion undefined zurück. Bitte beachten Sie, dass die Funktion auch die Konstante pointer_null zurückgeben kann. In diesem Fall wird das aktuelle Selbst zum Zeitpunkt des Aufrufs verwendet.

Syntax:

method_get_self(method);

ArgumentTyp Beschreibung
method The method variable to check.

Rückgabe:

Instance ID, Struct, undefined, or pointer_null

Beispiel:

var _self = method_get_self(light_properties);
show_debug_message(string(_self));

Die obige code ruft den Selbstkontext für die angegebene Methodenvariable ab und gibt ihn auf der Konsole aus.
