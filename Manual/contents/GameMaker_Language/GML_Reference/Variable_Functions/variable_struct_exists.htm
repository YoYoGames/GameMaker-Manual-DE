variable_struct_exists

Mit dieser Funktion können Sie prüfen, ob eine Variable innerhalb der angegebenen Struktur existiert oder nicht. Sie geben den zu verwendenden struct-Verweis sowie den zu prüfenden Variablennamen als String an (siehe Beispiel code unten). Die Funktion gibt true zurück, wenn eine Variable mit dem angegebenen Namen für die Struktur existiert, und andernfalls false.

Syntax:

variable_struct_exists(struct, name);

Argument Typ Beschreibung
struct Struktur The struct reference to check
name Zeichenfolge The name of the struct variable to check for (as a string)

Rückgabe:

Boolean

Beispiel:

if !variable_struct_exists(mystruct, "shields")
{
    mystruct.shields = 0;
}

Die obige code prüft, ob die Variable "shields" in der angegebenen Struktur vorhanden ist, und wenn nicht, wird sie erstellt und mit 0 initialisiert.
