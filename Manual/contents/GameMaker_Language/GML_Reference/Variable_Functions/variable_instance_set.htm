variable_instance_set

Mit dieser Funktion können Sie den Wert einer bestimmten Variable in einer Instanz setzen. Sie geben den eindeutigen ID-Wert der Instanz an (den Sie in den Instanzeigenschaften im Raumeditor finden können oder der zurückgegeben wird, wenn Sie die Funktion instance_create_layer()) sowie den Namen der Variablen, deren Wert gesetzt werden soll, als Zeichenkette (siehe Beispiel code unten), und schließlich den zu setzenden Wert (kann jeder gültige Datentyp sein). Wenn die Variable nicht bereits in der Instanz vorhanden ist, wird sie erstellt und dann der Wert zugewiesen.

Syntax:

variable_instance_set(instance_id, name, val);

Argument Typ Beschreibung
instance_id Instanz-ID The unique ID value of the instance to use
name Zeichenfolge The name of the variable to set (as a string)
val Variabel The value to set the variable to

Rückgabe:

N/A

Beispiel:

if !variable_instance_exists(id, "shields")
{
    variable_instance_set(id, "shields", 0);
}

Die obige code prüft, ob eine Instanzvariable in der aufrufenden Instanz existiert und wenn nicht, wird sie erstellt und auf 0 gesetzt.
