variable_struct_get_names

Mit dieser Funktion können Sie ein Array abrufen, das mit den Variablennamen einer Struktur gefüllt ist. Sie übergeben die zu prüfende Strukturreferenz, und jeder Eintrag im Array ist eine Zeichenkette mit den Variablennamen, die die Struktur enthält.

Syntax:

variable_struct_get_names(struct);

ArgumentTyp Beschreibung
struct The struct reference to check.

Rückgabe:

(each entry is a string)

Beispiel:

var str = "";
var array = variable_struct_get_names(mystruct);
show_debug_message("Variables for struct: " + string(array));
for (var i = 0; i < array_length(array); i++;)
{
    str = array[i] + ":" + string(variable_struct_get(mystruct, array[i]));
    show_debug_message(str);
}

Die obige code wird ein Array der Variablennamen für die angegebene Struktur abrufen und diese dann zusammen mit ihren Werten in der Debug-Ausgabe anzeigen.
