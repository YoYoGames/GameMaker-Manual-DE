array_delete

Mit dieser Funktion können Sie einen Wert (oder mehrere Werte) aus einem Array an einer beliebigen Position löschen. Für die Funktion müssen Sie eine Variable angeben, die das Array enthält, den Index (die Position) im Array, von dem aus gelöscht werden soll, sowie die Anzahl der zu löschenden Werte.

Syntax:

array_delete(variable, index, number);

ArgumentTyp Beschreibung
variable The variable that holds the array.
index The index (position) in the array to delete the value(s) from.
number The number of values to delete.

Rückgabe:

Beispiel:

array_delete(score_array, 0, 10);

Die obige code löscht die ersten 10 Werte in dem angegebenen Array, beginnend bei Index 0 (die Indizes 0 bis 9 werden also aus dem Array entfernt).
