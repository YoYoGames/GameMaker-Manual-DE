variable_global_set

Mit dieser Funktion können Sie den Wert einer bestimmten globalen Variablen setzen. Sie geben den Namen der globalen Variable, deren Wert gesetzt werden soll, als Zeichenkette an (siehe Beispiel code unten), und dann den zu setzenden Wert (kann jeder gültige Datentyp sein). Wenn die globale Variable noch nicht im Spiel vorhanden ist, wird sie erstellt und dann der Wert zugewiesen.

Syntax:

variable_global_set(name, val);

Argument Typ Beschreibung
name Zeichenfolge The name of the global variable to set (as a string)
val Variabel The value to set the global variable to

Rückgabe:

N/A

Beispiel:

if !variable_global_exists("enemy_num")
{
    variable_global_set("enemy_num", 0);
}

Die obige code prüft, ob eine globale Variable existiert, und wenn nicht, wird sie erstellt und auf 0 gesetzt.
