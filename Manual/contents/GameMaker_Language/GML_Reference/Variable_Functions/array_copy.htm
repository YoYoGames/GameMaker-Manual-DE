array_copy

Mit dieser Funktion können Sie ein Feld ganz oder teilweise an einer beliebigen Stelle in ein anderes Feld kopieren. Sie müssen sowohl das Quell- als auch das Ziel-Array angeben (beide müssen zuvor erstellt worden sein), sowie eine Position innerhalb des Quell-Arrays, von der aus kopiert werden soll, und eine Position innerhalb des Ziel-Arrays, in das kopiert werden soll. Schließlich müssen Sie die Länge des zu kopierenden Arrays (oder die Länge des Teils, den Sie kopieren wollen) angeben. Wenn die zu kopierenden Daten die Länge des Zielfeldes überschreiten, wird das Feld erweitert, um die Daten aufzunehmen.

Diese Funktion kann auch für mehrdimensionale Arrays verwendet werden, sofern Sie bei der Angabe des Array-Indexes angeben, welche Dimension Sie kopieren möchten, und zwar nach diesem Muster:

// Copy to the first dimension of an array
// from the second dimension of an array
array_copy(item_array, 0, inventory_array[0], 0, len);

// Copy to the third dimension of an array
// from the first dimension of an array
array_copy(item_array[0][0], 0, inventory_array, 0, len);

// etc...

Syntax:

array_copy(dest, dest_index, src, src_index, length);

Argument Typ Beschreibung
dest Array The ID of the array to copy to.
dest_index Real The index within the array to copy to.
src Array The ID of the array to copy from.
src_index Real The index with the array to start copying from.
length Real The length (number of array indices) to copy.

Rückgabe:

N/A

Beispiel:

if !array_equals(inventory_array, item_array)
{
    var len = array_length(inventory_array);
    array_copy(item_array, 0, inventory_array, 0, len);
}

Die obige code prüft zwei Arrays, um zu sehen, ob sie äquivalente Werte enthalten, und wenn dies nicht der Fall ist, kopiert code den gesamten Inhalt des Arrays "inventory_array" in das Array "item_array".
