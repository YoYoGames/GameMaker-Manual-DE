variable_global_get

Mit dieser Funktion können Sie den Wert einer bestimmten benannten globalen Variablen abrufen. Sie geben den Namen der globalen Variable, deren Wert Sie abrufen möchten , als String an (siehe Beispiel code unten) und die Funktion gibt den Wert der globalen Variable zurück oder undefined, wenn die Variable nicht existiert.

WICHTIG! Wenn die globale Variable, die Sie erhalten, nicht existiert, gibt die Funktion das Schlüsselwort undefined zurück, und Sie können Fehler erhalten, die das Spiel zum Stillstand bringen, also verwenden Sie im Zweifelsfall zuerst die Funktion variable_global_exists zuerst.

Syntax:

variable_global_get(name);

ArgumentTyp Beschreibung
name The name of the global variable to get (as a string)

Rückgabe:

Variable (any data type) or undefined (if the named variable does not exist)

Beispiel:

if variable_global_exists("enemy_num")
{
    show_debug_message("enemy_num = " + string(variable_global_get("enemy_num"));)
}

Die obige code prüft, ob eine globale Variable existiert, und wenn ja, wird sie auf der Konsole ausgegeben.
