typeof

Diese Funktion gibt den Datentyp einer gegebenen Variablen als String zurück. Die möglichen Rückgabewerte sind in der folgenden Tabelle aufgeführt:

Zeichenfolge Beschreibung
"number" Die Variable enthält eine reelle Zahl (Gleitkommazahl) - dies kann NaN und infinity
"string" Die Variable enthält eine Zeichenkette
"array" Die Variable verweist auf ein Array
"bool" Die Variable enthält einen booleschen Wert (true / false)
"int32" Die Variable enthält eine 32-Bit-Ganzzahl
"int64" Die Variable enthält eine 64-Bit-Ganzzahl
"ptr" Die Variable enthält einen Zeiger
"undefined" Die Variable ist undefiniert
"null" Die Variable enthält einen Nullwert (dies sollte normalerweise nicht zu sehen sein)
"vec3" Die Variable enthält einen Vektor mit 3 Werten
"vec4" Die Variable enthält einen Vektor mit 4 Werten
"method" Die Variable enthält eine Funktionsreferenz
"struct" Die Variable enthält eine Strukturreferenz
"ref" Die Variable enthält eine Instanzreferenz
"unknown" Wert ist unbekannt. Dieser Wert sollte niemals angezeigt werden und bedeutet, dass etwas auf der grundlegendsten Ebene schief gelaufen ist, z. B. ein Überschreiben des Speichers

Bitte beachten Sie, dass es Fälle gibt, in denen diese Funktion möglicherweise nicht den richtigen Wert für eine Methode zurückgibt. Betrachten Sie die folgenden zwei Funktionsdefinitionen:

a = function()
{
    // something
}

function b()
{
    // Something
}

Technisch gesehen werden beide als Methoden betrachtet, da sie eine Funktion an eine Instanzbereichsvariable binden, jedoch gibt der Aufruf von typeof() für die Funktion b "number" und nicht "method" zurück, während der Aufruf für a "method" zurückgibt. Dies liegt daran, dass Methoden, die wie die für b erstellt werden, script Indizes (die ganzzahlige Werte sind) zugewiesen werden, da dies die Art und Weise ist, wie compiler Skriptfunktionen erkennt, auch wenn die Funktion nicht in einer script definiert wurde.

Syntax:

typeof(variable);

Argument Typ Beschreibung
variable Jede The variable to get the data type of.

Rückgabe:

String (see table above)

Beispiel:

var _str = typeof(global.ExtensionInput);
show_debug_message(" global.ExtensionInput is a " + _str);

Die obige code ruft den Datentyp der angegebenen globalen Variablen ab und gibt die Zeichenkette an eine lokale Variable zurück, die dann zur Ausgabe einer Meldung an die Konsole verwendet wird.
