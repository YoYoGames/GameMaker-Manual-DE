array_pop

Diese Funktion entfernt das letzte Element im angegebenen Array und gibt dessen Wert zurück. Wenn das Array leer ist, wird undefined zurückgegeben.

Syntax:

array_pop(array);

ArgumentTyp Beschreibung
array The array to remove the last value from.

Rückgabe:

Variable or undefined

Beispiel:

var _lastscore = array_pop(score_array);
draw_text(32, 32, "Last Score = " + string(_lastscore));

Die obige code wird den letzten Wert aus dem gegebenen Array auslesen und ihn dann zusammen mit einem Text auf den Bildschirm zeichnen. Der abgefragte Eintrag wird auch aus dem Punkte-Array entfernt.
