variable_instanz_namens_anzahl

Mit dieser Funktion können Sie die Gesamtzahl der für eine Instanz definierten Variablen ermitteln. Sie geben die zu prüfende Instanz-ID an, und die Funktion gibt einen ganzzahligen Wert für die Anzahl der gefundenen Variablen oder (falls keine Instanz mit der angegebenen ID existiert) -1 zurück.

Syntax:

variable_instance_names_count(instance_id);

Argument Typ Beschreibung
instance_id Instanz-ID The unique ID value of the instance to check.

Rückgabe:

Real

Beispiel:

var _num = variable_instance_names_count(myinst);
show_debug_message("Instance Variables = " + string(_num));

Die obige code wird die Anzahl der Variablen in der gegebenen Instanz abrufen und eine Debug-Meldung in der Konsolenausgabe mit diesem Wert anzeigen.
