struct_names_count

Mit dieser Funktion können Sie die Gesamtzahl der für eine Struktur definierten Variablen ermitteln. Sie geben die zu prüfende Struktur-ID an, und die Funktion gibt einen ganzzahligen Wert für die Anzahl der gefundenen Variablen zurück, oder (falls keine Struktur mit der angegebenen ID existiert) -1.

 

Syntax:

struct_names_count(struct_id)

Argument Typ Beschreibung
struct_id Struct The unique ID value of the struct to check.

 

Rückgabe:

Real

 

Beispiel:

var _num = struct_names_count(mystruct);
show_debug_message("Struct Variables = " + string(_num));

Die obige code wird die Anzahl der Variablen in der gegebenen Struktur abrufen und eine Debug-Meldung in der Konsolenausgabe mit diesem Wert anzeigen.
