Variable Funktionen

Die Funktionen auf dieser Seite wurden entwickelt, um mit all den verschiedenen Variablen und Variablentypen umzugehen, wenn Sie die GameMaker Language in Ihren Spielen verwenden. Weitere Informationen über Variablen finden Sie auf der Seite über Variablen und Variablenumfang.

Die folgenden variablen Funktionen sind verfügbar:

  • variable_instance_exists
  • variable_instance_get_names
  • variable_instance_names_count
  • variable_instance_get
  • variable_instance_set
  • variable_global_exists
  • variable_global_get
  • variable_global_set
  • variable_struct_exists
  • variable_struct_get
  • variable_struct_set
  • variable_struct_remove
  • variable_struct_get_names
  • variable_struct_names_count
  • instanceof

 

Es gibt auch ein paar Funktionen, die mit der Verwendung von Arrays verbunden sind. Diese wurden entwickelt, um Ihnen mehr Flexibilität in Ihrem code zu geben, und ermöglichen es Ihnen, funktionellere und dynamischere Arrays zu erstellen, wenn Sie Ihre Spiele entwickeln. Diese Funktionen sind:

  • array_create
  • array_copy
  • array_equals
  • array_get
  • array_set
  • array_push
  • array_pop
  • array_insert
  • array_delete
  • array_sort
  • array_length
  • array_resize
  • array_length_1d(veraltet)
  • array_length_2d(veraltet)
  • array_height_2d(veraltet)

 

Die folgenden Funktionen beziehen sich alle auf die Verwendung von Methodenvariablen:

  • method
  • method_get_self
  • method_get_index

 

Schließlich gibt es noch die folgenden Funktionen zur Bestimmung des Variablentyps (weitere Informationen über Datentypen finden Sie hier, und Typentabellen finden Sie hier):

  • is_string
  • is_real
  • is_numeric
  • is_bool
  • is_array
  • is_struct
  • is_method
  • is_ptr
  • is_int32
  • is_int64
  • is_vec3
  • is_vec4
  • is_matrix
  • is_undefined
  • is_nan
  • is_infinity
  • typeof
  • bool
  • real
  • ptr
  • int64
