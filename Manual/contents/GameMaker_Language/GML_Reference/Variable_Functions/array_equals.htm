array_equals

Mit dieser Funktion können Sie prüfen, ob zwei Arrays gleich sind (gleichwertig oder gleich). Sie geben die beiden zu prüfenden Felder an, und die Funktion gibt true zurück, wenn sie gleich sind (entweder gleichwertig oder gleich), oder false, wenn sie es nicht sind. Beachten Sie, dass dies nicht dasselbe ist wie die Prüfung, ob zwei Arrays gleich sind, indem Sie == verwenden, was nicht prüft, ob die beiden Arrays gleichwertige Werte enthalten, sondern nur, ob die Arrays auf dasselbe Ausgangsarray verweisen. Zum Beispiel:

var a = [1,2,3,4];
var b = [1,2,3,4];

if (a == b)
{
    show_debug_message( "This will never fire, as a and b do not reference the SAME array" );
}

if (array_equals(a, b))
{
    show_debug_message( "This will fire as both arrays contain similar values" );
}

Beachten Sie, dass es einige Konstanten gibt, die nicht gleich sich selbst sein dürfen, wodurch diese Funktion fehlschlagen kann. Hier ist ein Beispiel:

if (array_equals([NaN], [NaN]))
{
    show_debug_message( "This will never fire as NaN cannot be equal to itself" );
}

Weitere Informationen finden Sie in der Gleichstellungstabelle.

Syntax:

array_equals(var1, var2);

Argument Typ Beschreibung
var1 Array The first array.
var2 Array The second array.

Rückgabe:

Boolean

Beispiel:

if !array_equals(inventory_array, item_array)
{
    var len = array_length(inventory_array);
    array_copy(item_array, 0, inventory_array, 0, len);
}

Die obige code prüft zwei Arrays, um zu sehen, ob sie äquivalente Werte enthalten, und wenn dies nicht der Fall ist, kopiert code den gesamten Inhalt des Arrays "inventory_array" in das Array "item_array".
