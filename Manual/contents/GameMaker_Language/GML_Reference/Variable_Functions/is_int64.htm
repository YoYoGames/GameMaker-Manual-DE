is_int64

Diese Funktion gibt zurück, ob eine gegebene Variable ein 64bit Integer ist oder nicht. In manchen Fällen möchten Sie überprüfen, welchen Datentyp eine Variable in GameMaker hat, und dann würden Sie diese Funktion verwenden. Sie gibt true oder false zurück, je nachdem, ob der Wert ein int64 ist oder nicht.

Syntax:

is_int64(n);

Argument Typ Beschreibung
n Variabel The argument to check.

Rückgabe:

Boolean

Beispiel:

if !is_int64(val)
{
    show_debug_message("Not a 64bit integer!");
}

Die obige code prüft die Variable "val", um zu sehen, ob sie einen int64 enthält, und wenn nicht, wird eine Meldung in der Debug-Konsole angezeigt.
