is_string

Diese Funktion gibt zurück, ob eine gegebene Variable eine String-Zahl ist oder nicht. In manchen Fällen möchten Sie überprüfen, ob eine Variable in GameMaker eine Zeichenkette und nicht eine reelle Zahl enthält, und dann würden Sie diese Funktion verwenden. Sie gibt nicht die Zeichenkette zurück, sondern true oder false, so dass ein Wert von z.B. "Fisch" für n true zurückgibt, aber ein Wert von 200 für n false.

Syntax:

is_string(n);

Argument Typ Beschreibung
n Variabel The argument to check.

Rückgabe:

Boolean

Beispiel:

if is_string(val)
{
    name = "Player: " + val;
}

Die obige code prüft, ob die Variable "val" eine Zeichenkette enthält, und wenn ja, fügt sie diese zu einer anderen Zeichenkette hinzu, die dann der Variablen "name" zugewiesen wird.
