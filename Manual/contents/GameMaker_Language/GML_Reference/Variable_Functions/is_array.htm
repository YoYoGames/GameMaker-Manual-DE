is_array

Diese Funktion kann verwendet werden, um zu prüfen, ob eine Variable ein Array enthält (sie gibt true zurück) oder nicht (in diesem Fall gibt sie false zurück).

Syntax:

is_array(n);

Argument Typ Beschreibung
n Variabel The variable to check.

Rückgabe:

Boolean

Beispiel:

if is_array(a)
{
    a = -1;
}

Die obige code prüft eine Variable, um festzustellen, ob es sich um ein Array handelt, und wenn die Funktion true zurückgibt, wird das Array gelöscht, indem die Variable auf -1 gesetzt wird.
