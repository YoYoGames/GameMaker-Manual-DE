bool

Diese Funktion versucht, einen gegebenen Wert in einen booleschen Datentyp umzuwandeln, wobei der Wert als true zurückgegeben wird, wenn er größer oder gleich 0,5 ist, und ansonsten als false.

Syntax:

bool(n);

Argument Typ Beschreibung
n Real The value to convert.

Rückgabe:

Boolean

Beispiel:

if !is_bool(val)
{
    val = bool(val);
}

Die obige code prüft, ob die Variable "val" ein boolescher Wert ist, und wenn nicht, wird sie in einen solchen umgewandelt.
