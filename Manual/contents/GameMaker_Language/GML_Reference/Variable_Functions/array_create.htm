array_erstellen

Mit dieser Funktion können Sie ein 1D-Array mit einer bestimmten Größe erstellen. Sie teilen der Funktion die Länge des zu erstellenden Arrays mit, und sie gibt das "Handle" für das Array zurück, das Sie dann einer Variablen zuweisen können. Bei Arrays, die auf diese Weise erstellt werden, wird jeder Eintrag mit 0 initialisiert, es sei denn, Sie geben einen (optionalen) Initialisierungswert an. Wenn Sie den zusätzlichen Wert für die Initialisierung des Arrays angeben, werden alle Indizes innerhalb des neuen Arrays auf diesen Wert anstelle von 0 gesetzt, aber beachten Sie, dass die Funktion in diesem Fall einen größeren Performance-Overhead hat.

Syntax:

array_create(size, [value]);

Argument Typ Beschreibung
size Real The size of the array to create.
value Variabel OPTIONAL Der Wert, der für die Initialisierung aller Array-Indizes verwendet wird.

Rückgabe:

Array

Beispiel:

instance_array = array_create(100, noone);

Die obige code erstellt ein neues Array mit 100 Einträgen, von denen jeder auf das Schlüsselwort noone gesetzt wird, und weist es dann der Variablen "instance_array" zu.
