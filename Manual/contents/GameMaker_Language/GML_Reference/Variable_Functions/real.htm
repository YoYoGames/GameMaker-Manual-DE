real

Diese Funktion kann verwendet werden, um eine gegebene Zeichenkette in eine reelle Zahl umzuwandeln. Bei der Verwendung dieser Funktion werden Zahlen, Minuszeichen, Dezimalpunkte und Exponentialteile in der Zeichenkette berücksichtigt, während andere Zeichen (wie Buchstaben) einen Fehler geworfen werden. Wenn Sie wissen oder vermuten, dass eine Zeichenkette andere Zeichen enthalten könnte, können Sie mit string_digits() verwenden, um alle nicht numerischen Zeichen zu entfernen, bevor Sie diese Funktion verwenden, um die resultierende Zeichenkette in eine reelle Zahl umwandelt.

 

Syntax:

real(string)

Argument Typ Beschreibung
string String The string to be converted to a real value.

 

Rückgabe:

Real

 

Beispiel:

var t_str = string_digits(input_str);
age = real(t_str);

Die obige code nimmt die Eingabezeichenkette, entfernt alle Zeichen, die keine Zahlen sind, und setzt dann die Variable "age" so, dass sie den realen Zahlenwert der Rückgabezeichenkette enthält.
