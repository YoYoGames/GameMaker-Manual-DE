array_push

Mit dieser Funktion können Sie einen Wert (oder mehrere Werte) an das Ende eines Arrays anfügen, ohne die Länge des Arrays kennen zu müssen. Die Funktion erfordert die Angabe einer Variablen, die das Array enthält, sowie mindestens einen zu verschiebenden Wert. Optional können Sie weitere arguments angeben, die dann alle in fortlaufender Reihenfolge in das Array verschoben werden.

Syntax:

array_push(variable, value, [value], [value], [etc...]);

Argument Typ Beschreibung
variable Array The variable that holds the array.
value Variabel The value to push onto the end of the array
[value], [value], [etc...] Variabel OPTIONAL Weitere Werte, die in das Array eingefügt werden sollen

Rückgabe:

N/A

Beispiel:

array_push(score_array, obj_Player1.scr, obj_Player2.scr, obj_Player3.scr, obj_Player4.scr);

Die obige code wird vier Werte an das Ende des angegebenen Arrays anfügen.
