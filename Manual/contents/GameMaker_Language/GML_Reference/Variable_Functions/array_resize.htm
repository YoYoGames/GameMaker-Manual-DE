array_resize

Mit dieser Funktion können Sie die Größe eines bestehenden Arrays auf eine neue Größe ändern. Sie geben das Array an, dessen Größe geändert werden soll, sowie die neue Anzahl der Indizes für das Array, und die Funktion ändert die Größe dieses Arrays. Beachten Sie, dass diese Funktion dafür gedacht ist, ein Array auf eine kleinere Länge zu verkleinern, da Sie die Größe durch einfaches Setzen eines neuen Indexes im Array erhöhen können. Wenn Sie die Funktion zum Vergrößern eines Arrays verwenden, werden alle neuen Indizes auf den Standardwert 0 gesetzt.

Diese Funktion kann auch für mehrdimensionale Arrays verwendet werden, sofern Sie bei der Angabe des Array-Indexes angeben, welche Dimension Sie ändern möchten, indem Sie diesem Muster folgen:

// Resize the first dimension of the array
array_resize(my_array, 10);

// Resize the second dimension of the array (only for the first slot)
array_resize(my_array[0], 10);

// Resize the third dimension of the array (only for the first slots)
array_resize(my_array[0][0], 10);

// ...and so on.

Laut code wird nur der erste Slot der zweiten Dimension in der Größe verändert, da die Slots in einer bestimmten Array-Dimension nicht miteinander verbunden sind und unterschiedliche Größen haben können; zum Beispiel:

array_resize(my_array[0], 10);
array_resize(my_array[1], 22);

In der obigen Abbildung code beträgt die Länge der zweiten Dimension des Arrays im ersten Steckplatz 10, im zweiten Steckplatz jedoch 22.

Syntax:

array_resize(array_index, new_size);

ArgumentTyp Beschreibung
array_index The index of the array to resize.
new_size The new size for the array (an integer, starting from 0).

Rückgabe:

Beispiel:

if array_length(a) > 10
{
    array_resize(a, 10);
}

Die obige code prüft die Länge eines Arrays, und wenn es mehr als 10 Indizes hat, wird es in der Größe angepasst.
