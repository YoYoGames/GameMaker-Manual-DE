variabel_global_existiert

Mit dieser Funktion können Sie prüfen, ob eine globale Bereichsvariable existiert oder nicht. Sie geben den Namen der globalen Variable, auf die geprüft werden soll , als Zeichenkette an (siehe Beispiel code unten), und die Funktion gibt true zurück, wenn eine globale Variable mit dem angegebenen Namen existiert, oder andernfalls false.

Syntax:

variable_global_exists(name);

Argument Typ Beschreibung
name Zeichenfolge The name of the global variable to check for (as a string)

Rückgabe:

Boolean

Beispiel:

if !variable_global_exists("enemy_num")
{
    global.enemy_num = instance_number(obj_Enemey_Parent);
}

Die obige code prüft, ob die globale Variable "enemy_num" existiert, und wenn nicht, wird sie erstellt und ein Wert gesetzt.
