variable_struct_get

Mit dieser Funktion können Sie den Wert einer bestimmten benannten Variablen innerhalb einer Struktur abrufen. Sie geben den Verweis auf die Struktur sowie den Namen der Variablen an, deren Wert Sie als Zeichenkette abrufen möchten (siehe Beispiel code unten). Die Funktion gibt den Wert der Variable zurück oder undefined, wenn die benannte Variable nicht existiert.

WICHTIG! Wenn die Variable, die Sie erhalten, nicht existiert, gibt die Funktion das Schlüsselwort undefined zurück, und Sie können Fehler erhalten, die das Spiel zum Stillstand bringen, also verwenden Sie im Zweifelsfall zuerst die Funktion variable_struct_exists() zuerst.

Syntax:

variable_struct_get(struct, name);

ArgumentTyp Beschreibung
struct The struct reference to use
name The name of the variable to get (as a string)

Rückgabe:

Variable (any data type) or undefined (if the named variable does not exist)

Beispiel:

if variable_struct_exists(mystruct, "shields")
{
    var ss = variable_struct_get(mystruct, "shields");
}
else
{
    var ss = -1;
}

Die obige code prüft, ob eine Variable in der gegebenen Struktur existiert, und wenn ja, dann wird der Wert, den sie enthält, abgerufen und in einer lokalen Variablen gespeichert. Wenn sie nicht existiert, wird die lokale Variable auf -1 gesetzt.
