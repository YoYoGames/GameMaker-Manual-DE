Opera GX und GXC

In diesem Abschnitt werden die Funktionen für das Opera GX-Ziel platform beschrieben. Die auf dieser Seite aufgeführten Funktionen ermöglichen es Ihnen, Ihren Bildschirm aufzuzeichnen und ihn zusammen mit Ihrer Punktzahl in GXC hochzuladen. Beachten Sie jedoch, dass Sie dazu die GXC-API-Bibliothek verwenden müssen. Eine Anleitung zur Implementierung der GXC-API-Bibliothek und zur Arbeit mit Challenges finden Sie im Abschnitt Opera GX auf dem YoYo Games Helpdesk.

Zum Hochladen von Videoaufnahmen mit Herausforderungen stehen folgende Funktionen zur Verfügung:

  • gxc_start_film_aufzeichnung
  • gxc_stop_film_aufzeichnung
  • gxc_pause_film_aufzeichnung
  • gxc_resume_movie_recording
