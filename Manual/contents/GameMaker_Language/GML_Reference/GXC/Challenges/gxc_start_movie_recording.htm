gxc_start_film_aufzeichnung

Mit dieser Funktion können Sie die Aufnahme Ihrer Challenge auf GXC starten. Sie legen die Breite, Höhe und Bildrate der Aufnahme fest und geben einen booleschen Wert an, der angibt, ob der Ton aufgenommen werden soll oder nicht.

Syntax:

gxc_start_movie_recording(width, height, fps, record_audio);

Argument Typ Beschreibung
width Real The width of the recording
height Real The height of the recording
fps Real The frame rate (frames-per-second) of the recording
record_audio Boolesche Whether audio should be recorded (true) oder nicht (false)

Rückgabe:

N/A

Beispiel:

gxc_start_movie_recording(320, 240, 4, false);

Damit wird eine Videoaufnahme mit einer Auflösung von 320x240 bei 4 Bildern pro Sekunde gestartet, ohne dass ein Ton aufgezeichnet wird.
