gxc_stop_film_aufzeichnung

Diese Funktion wird verwendet, um eine GXC-Aufnahme zu stoppen, die zuvor mit gxc_start_movie_recording() gestartet wurde. Sie sendet die Aufnahme auch an die in arguments angegebene URL. Das Verhalten ist jedoch je nach Art der angegebenen Zeichenkette unterschiedlich:

  • Wenn Sie einen Dateinamen angeben, wird die Aufnahme unter diesem Namen in Ihrem Download-Ordner gespeichert.
  • Wenn Sie eine URL angeben, wird die Aufnahme auf diese URL hochgeladen.

Die URL-Methode sollte verwendet werden, wenn die Aufzeichnung in GXC hochgeladen werden soll, und die Datei sollte nur lokal gespeichert werden, um zu überprüfen, ob die Aufzeichnung erfolgreich war.

Syntax:

gxc_stop_movie_recording(filename);

Argument Typ Beschreibung
filename Zeichenfolge The file name or URL where the recording is to be saved or uploaded

Rückgabe:

N/A

Beispiel:

gxc_submit_challenge_score(_score, function(_status, _result) {
    var _uploadUrl = _result.data.gameplayVideoUploadUrl;
    gxc_stop_movie_recording(_uploadUrl);
});

Die obige code verwendet die GXC-API-Bibliothek, um einen Spielstand auf die derzeit aktive Herausforderung hochzuladen. Wenn eine Antwort in der Callback-Methode empfangen wird, wird die Upload-URL für das Video abgerufen und die Aufnahme dorthin hochgeladen.
