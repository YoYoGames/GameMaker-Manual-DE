gxc_resume_movie_recording

Mit dieser Funktion können Sie eine GXC-Aufnahme fortsetzen, die zuvor mit gxc_pause_movie_recording() angehalten wurde.

Syntax:

gxc_resume_movie_recording();

Rückgabe:

Beispiel:

if (keyboard_check_pressed(vk_enter))
{
    gxc_resume_movie_recording();
}

Wenn Sie die Eingabetaste drücken, wird die derzeit angehaltene GXC-Filmaufnahme fortgesetzt.
