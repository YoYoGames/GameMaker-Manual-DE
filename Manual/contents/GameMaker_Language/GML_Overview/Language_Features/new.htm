neu

Der new Operator kann verwendet werden, um eine Struktur mit einer Funktion zu erstellen, die mit dem angegebenen arguments aufgerufen wird. Die Funktion kann dann die Struktur mit Variablen aus arguments füllen - ähnlich wie das Ereignis "create" einer Instanz die Anfangsvariablen für die Instanz setzt - und dann gibt der Operator new die Struktur zurück. Bevor Sie diesen Operator verwenden, ist es wichtig zu beachten, dass die angegebene Funktion als constructor Funktion gekennzeichnet sein muss, da sonst der new Operator die Struktur nicht erstellen kann (das code Beispiel unten zeigt dies, und für weitere Informationen, lesen Sie bitte die Seite über Strukturen).

HINWEIS Sie können eine Struktur überprüfen, um herauszufinden, mit welcher Funktion sie erstellt wurde, indem Sie die Funktion runtime verwenden instanceof().

Dieser Operator hat die folgende Syntax:

<variable> = new <function> (<argument>, <argument>, ...);

Wenn Sie den new-Operator zusammen mit einer Funktion verwenden, wird eine struct-Referenz zurückgegeben, die in der angegebenen Variable gespeichert wird, so dass Sie auf die struct und alle Variablen zugreifen können, die durch die Funktion darin erstellt wurden.

Das folgende Beispiel zeigt eine Funktion, die für die Verwendung als Konstruktor definiert wurde:

function init_struct(_a, _b, _c) constructor
{
    a = _a;
    b = _b;
    c = _c;
}

Diese Funktion kann dann zusammen mit dem new -Operator verwendet werden, um eine Struktur zu erstellen und sie mit den Variablen zu füllen, die auf die Werte der in der Funktion verwendeten arguments gesetzt sind, etwa so:

mystruct = new init_struct(10, 100, "Hello World");

Es ist wichtig zu beachten, dass der Aufruf von new für eine Funktion, die nicht als Konstruktor gekennzeichnet ist, eine Laufzeitausnahme verursacht.
