Die Anweisung continue hat die folgende grundlegende Syntax:

continue;

Wenn es innerhalb einer Anweisung verwendet wird, die eine loop bildet (for, repeat, while oder do / until), wird die aktuelle Iteration sofort beendet und an den Anfang von loop zurückgesprungen, wodurch eine neue Iteration beginnt und alle code, die nach continue innerhalb von loop folgen, ausgelassen werden. Er kann auch innerhalb der with Anweisung verwendet werden, wo er bewirkt, dass die code zur nächsten Instanz springt und erneut ausgeführt wird. Beachten Sie, dass die Verwendung von continue außerhalb eines dieser Kontexte zu einem Fehler führt.

Nachstehend finden Sie ein Beispiel für die Verwendung in einer for -Schleife:

var _val = 0;

for (var i = 0; i < 10; i += 1)
{
    if (val_array[i] <= 0)
    {
        continue;
    }
    _val += val_array[i];
}

draw_text(32, 32, "Positive Values Total = " + string(_val));

Nachstehend finden Sie ein Beispiel für die Verwendung in einer while -Schleife:

var file = file_text_open_read("Game_Data.txt");
var _num = 0;

while (!file_text_eof(file))
