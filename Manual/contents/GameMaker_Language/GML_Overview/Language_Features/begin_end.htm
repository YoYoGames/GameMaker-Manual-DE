Anfang/Ende

Mit GameMaker Language können Sie bei der Erstellung von code -Blöcken die Schlüsselwörter begin und end anstelle der sonst üblichen geschweiften Klammern {} verwenden. Das folgende Beispiel code zeigt, wie dies funktioniert:

if (!visible)
    begin
    exit;
    end
while (place_meeting(x, y))
    begin
    x -= lengthdir_x(1, direction - 180);
    y -= lengthdir_y(1, direction - 180);
    end

Beachten Sie, dass die Verwendung dieser Schlüsselwörter nicht typisch ist und als Teil der Sprache mehr zur Unterstützung von Altlasten als für irgendetwas anderes zur Verfügung gestellt wird, und dass sie in der Zukunft jederzeit veraltet sein können.
