Mit der Anweisung throw runtime können Sie Ihre eigenen Laufzeitausnahmen erzeugen, die das Spiel beenden und eine Fehlermeldung anzeigen, indem Sie die folgende Syntax verwenden:

throw (<expression>);

Der verwendete Ausdruck kann ein Wert, eine Zeichenkette oder ein beliebiger anderer Datentyp sein, der dann eine Fehlermeldung erzeugt, die standardmäßig auf dem Bildschirm angezeigt wird, und nach Schließen der Fehlermeldung wird das Spiel beendet. Zum Beispiel kann man dies aufrufen:

throw ("Hello World!");

Sie können jedoch diese Fehlermeldung übernehmen und Ihren eigenen Handler code verwenden, indem Sie die Funktion exception_unhandled_handler(). Diese Laufzeitfunktion erlaubt es Ihnen, einen eigenen method zu verwenden, die immer dann aufgerufen wird, wenn in Ihrem Spiel eine unbehandelte Ausnahme auftritt.
