return

Die Anweisung return hat die folgende Syntax:

return (<expression>)

Die Anweisung return wird nur in Skriptfunktionen und -methoden verwendet und dient dazu, einen Wert aus der Funktion zurückzugeben, der in weiteren code oder Funktionsaufrufen verwendet werden kann. Es ist zu beachten, dass die Ausführung der Funktion an der Anweisung return endet, was bedeutet, dass jede code, die nach dem Aufruf von return kommt, nicht ausgeführt wird. Hier ist ein kurzes Beispiel script mit dem Namen sqr_calc, das das Quadrat eines beliebigen Wertes berechnet, der ihm übergeben wird, und das eine Fehlerabfangfunktion für den Fall enthält, dass der übergebene Parameter keine reelle Zahl ist:

/// @function            sqr_calc(val);
/// @param {real}  val   The value to calculate the square of
/// @description         Calculate the square of the given value

function sqr_calc(value)
{
    if (!is_real(value))
    {
        return 0;
    }
    else
    {
        return (value * value);
    }
}

Um eine Funktion aus einem Teil von code heraus aufzurufen, verwenden Sie sie auf die gleiche Weise wie beim Aufruf von runtime Funktionen - das heißt, Sie schreiben den Funktionsnamen mit den Parameterwerten in Klammern. Die obige Funktion würde also wie folgt aufgerufen werden:

if keyboard_check_pressed(vk_enter)
{
    val = scr_sqr(amount);
}

Hier wird die Variable val entweder 0 sein (weil die Variable amount keine reelle Zahl ist) oder die Summe für value * value.
