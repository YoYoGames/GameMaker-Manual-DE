wiederholen

GameMaker Language bietet eine Reihe von Möglichkeiten, Schleifen auszuführen, d. h. eine Anweisung oder mehrere Anweisungen eine bestimmte Anzahl von Wiederholungen durchlaufen zu lassen. Die einfachste dieser Möglichkeiten ist die Anweisung repeat, die die Form hat:

repeat (<expression>)
{
    <statement>;
    <statement>;
    ...
}

Mit repeat wird die angegebene Anweisung so oft wiederholt, wie es der gerundete Wert des Ausdrucks angibt. Die folgende Anweisung erzeugt zum Beispiel fünf Bälle an zufälligen Positionen:

repeat (5)
{
    instance_create_layer(random(400), random(400), "Instances", obj_ball);
}

Dies kann sehr nützlich sein, um zu vermeiden, dass dieselbe code mehrfach eingegeben wird, oder um Arrays zu verwenden, oder um eine Reihe von Operationen durchzuzählen usw.. Sie sind auch nicht auf eine einzige Anweisung beschränkt, sondern können mehrere Anweisungen wiederholen, indem Sie sie in geschweifte Klammern {} einschließen. Zum Beispiel:

var _x = 32;
repeat (global.p_lives)
{
    draw_sprite(spr_heart, 0, _x, 32);
    _x += sprite_get_width(spr_heart);
}

Das obige Beispiel wiederholt die Anweisungen in den geschweiften Klammern für so viele Iterationen, wie die globale Variable "lives" hat, und bei jeder Iteration wird das Herz sprite an der Position _x gezeichnet und dann die Position entsprechend der Breite des Herzens sprite ein Stück weiter verschoben.

Wann sollten Sie eine repeat loop verwenden? Immer dann, wenn Sie eine oder mehrere Anweisungen eine feste Anzahl von Malen wiederholen wollen, ohne dass Sie eine Zählung der Iterationen vornehmen müssen.

Es ist erwähnenswert, dass Sie die speziellen Anweisungen break und continue auch innerhalb einer Schleife verwenden können. Wiederholung Schleife verwenden können. Wenn Sie break verwenden, wird loop sofort beendet und mit code fortgefahren, das sich in dem Ereignis oder der Funktion befindet, nachdem loop beendet sein sollte, z. B:

var i = 0;
var temp = 0;
repeat (10)
{
    temp += array[i];
    if (temp > max_total)
    {
        break;
    }
    else
    {
        i += 1;
    }
}

Die obige code loops durchläuft 10 Array-Werte und addiert sie zu einer lokalen Variablen. Wenn die Summe der lokalen Variablen größer ist als der angegebene Wert für max_total, dann wird loop mit break beendet, ansonsten wird loop fortgesetzt.

Ein Beispiel für die Verwendung von continue in einer repeat loop wäre:

repeat(10)
{   
    var _x = random(room_width);
    var _y = random(room_height);
    if (instance_position(_x, y, obj_Enemy)
    {
        continue;
    }
    instance_create_layer(_x, _y, "Instances", obj_Enemy);
}

code wird 10 Mal wiederholt, wobei eine zufällige Raumposition erzeugt und dann geprüft wird, ob eine Instanz des Objekts obj_Enemy an dieser Position existiert. Ist dies der Fall, wird die aktuelle loop Iteration mit continue beendet und eine neue Iteration gestartet. Ist dies nicht der Fall, wird eine Instanz des Objekts obj_Enemy an der zufälligen Position erstellt.

Weitere Beispiele für loop Funktionen finden Sie in den Abschnitten über while, do... until, und for.
