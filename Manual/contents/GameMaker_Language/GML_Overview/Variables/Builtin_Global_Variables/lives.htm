lebt

Diese Variable ist global und wird verwendet, um einen numerischen Wert zu speichern, der normalerweise für die Spielerleben verwendet wird. Diese Variable ist nur dazu gedacht, ältere Projekte aus früheren Versionen von GameMaker zu unterstützen und sollte darf nicht in neuen Projekten verwendet werden nicht mehr verwendet werden, da sie in Zukunft veraltet sein könnte.

Syntax:

lives;

Rückgabe:

Real (single precision floating point value)

Beispiel

if (lives < 5)
{
    lives += 1;
}

Die obige code prüft die Variable lives, und wenn sie kleiner als 5 ist, wird 1 zu ihr addiert.
