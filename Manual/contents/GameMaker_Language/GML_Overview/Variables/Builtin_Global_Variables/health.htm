health

Diese Variable ist global und wird verwendet, um einen numerischen Wert zu speichern, der normalerweise für die Gesundheit des Spielers verwendet wird. Diese Variable ist nur dazu gedacht, ältere Projekte aus früheren Versionen von GameMaker zu unterstützen und sollte darf nicht in neuen Projekten verwendet werden nicht mehr verwendet werden, da sie in Zukunft veraltet sein könnte.

Syntax:

health

Rückgabe:

Real (single precision floating point value)

Beispiel

if (health <= 0)
{
    global.state = "Game Over";
    instance_destroy();
}

Die obige code überprüft die Variable health und wenn sie kleiner oder gleich 0 ist, wird eine globale Variable gesetzt und die Instanz zerstört.
