Ergebnis

Diese Variable ist global und wird verwendet, um einen numerischen Wert zu speichern, der normalerweise für den Spielerstand verwendet wird. Diese Variable ist nur dazu gedacht, ältere Projekte aus früheren Versionen von GameMaker zu unterstützen und sollte darf nicht in neuen Projekten verwendet werden nicht mehr verwendet werden, da sie in Zukunft veraltet sein könnte.

Syntax:

score;

Rückgabe:

Real (single precision floating point value)

Beispiel

switch (object_index)
{
    case obj_Enemy_Fighter:
        score += 10;
    break;

    case obj_Enemy_Mage:
        score += 25;
    break;

    case obj_Enemy_Boss:
        score += 100;
    break;
}
instance_destroy();

Die obige code prüft den Objektindex der Instanz, auf der die code läuft, mit Hilfe einer switchund fügt dann unterschiedliche Beträge zur score Variablen hinzu, je nachdem, um welches Objekt es sich handelt.
