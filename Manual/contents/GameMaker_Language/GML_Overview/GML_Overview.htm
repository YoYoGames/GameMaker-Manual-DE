GML-Code-Übersicht

Dieser Teil des Handbuchs enthält alle Informationen, die zum Verständnis und zur Verwendung von GML-Code. Die runtime Funktionen in dieser Sprache können verwendet werden, um Ihre Spiele zu erstellen und werden in Objekte aus dem Objekt-Editorhinzugefügt, obwohl sie auch zusammen mit GML Visuell. Unten sehen Sie ein typisches Bild eines Objekts mit dem code Editor, der für ein Ereignis geöffnet ist:

Jedes Ereignis hat seine eigene Registerkarte im Editor, und Sie können jederzeit code hinzufügen, bearbeiten oder entfernen (weitere Informationen zu Ereignissen finden Sie unter Objekt-Ereignisse). Die code selbst muss eine Grundstruktur haben und kann Ressourcenindizes, Variablen, Funktionen, Ausdrücke, Schlüsselwörter usw. enthalten, die alle in den folgenden Abschnitten erläutert werden. Wenn Sie ein Neuling in der Programmierung oder der Erstellung von switch aus GML Visual sind, empfehlen wir Ihnen, mit der Seite zu beginnen Grundlegende Struktur von Code zu beginnen und dann alle anderen Seiten in diesem Abschnitt durchzulesen und code von jeder Seite aus in GameMaker selbst zu testen.

Sie können Feather für intelligente code Vervollständigung und Fehlerberichte in Ihrem GML Code scripts verwenden.
