Struktur Verbotene Variablen

Bei der Erstellung einer Struktur ist es zum Beispiel möglich, bestimmte eingebaute Variablen als Namen von Mitgliedsvariablen zu verwenden:

mystruct =
{
    object_index : obj_Player,
    speed : 3,
    image_blend : c_red
}

Allerdings können nur lokale und instanzielle eingebaute Variablen auf diese Weise verwendet und mit Werten belegt werden, als wären sie reguläre Strukturvariablen. Die Verwendung von eingebauten Variablen mit globalem Geltungsbereich ist verboten und führt zu Problemen in Ihrem Spiel. Im Folgenden finden Sie eine vollständige Liste dieser Variablen, damit Sie wissen, welche Sie vermeiden sollten.
