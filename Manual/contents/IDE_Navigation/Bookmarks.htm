Lesezeichen

Eine weitere Methode zum Navigieren in workspace ist die Verwendung von Lesezeichen. Sie können bis zu 9 "Lesezeichen" zu IDE Fenstern oder Abschnitten von Code/GML Visual in scripts oder Objektereignissen hinzufügen, zwischen denen Sie dann mit der richtigen Tastenkombination schnell hin- und herspringen können.

Um ein Lesezeichen hinzuzufügen, wählen Sie einfach das Fenster oder die Zeile von code, die Sie markieren möchten, und dann + + NumberKey[1 - 9] auf Windows oder + + NumberKey[1 - 9] auf Mac. Dadurch wird das Lesezeichen gesetzt und das Element in IDE wie folgt beschriftet:

In der obigen Abbildung haben wir die Lesezeichen mit den Nummern 1, 2 und 4 verschiedenen Bereichen von IDE zugewiesen, und wir können nun von diesen Fenstern weg navigieren oder sie sogar ganz schließen, da das Lesezeichen es uns erlaubt, später jederzeit wieder dorthin zurückzukehren. Dies geschieht durch Drücken von / + NumberKey[1 - 9], und wenn das mit einem Lesezeichen versehene Element geschlossen wurde, wird es geöffnet, andernfalls schwenkt workspace zu dem ausgewählten Lesezeichen.

WICHTIG Lesezeichen werden nicht mit dem Projekt gespeichert oder wenn IDE geschlossen wird. Sie sind nur für die Sitzung gültig, in der sie erstellt wurden.

Wenn Sie ein Lesezeichen entfernen möchten, wählen Sie einfach das mit einem Lesezeichen versehene Fenster oder die Zeile code aus und wiederholen Sie die Befehle + / + NumberKey[1 - 9], die Sie zum Erstellen des Lesezeichens verwendet haben.
