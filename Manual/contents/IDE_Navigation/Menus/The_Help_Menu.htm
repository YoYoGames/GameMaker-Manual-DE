Das Hilfe-Menü

Dieses Menü dient dazu, Hilfe bei der Verwendung von GameMaker zu erhalten. Es enthält die folgenden Menüpunkte:

  • Handbuch öffnen - Öffnet das Hilfehandbuch. Beachten Sie, dass das Anklicken der Hilfe-Schaltfläche in der Menüleiste ebenso zum Öffnen des Dokuments führt wie das Drücken von .
  • Wissensdatenbank - Hier gelangen Sie zur GameMaker-Wissensdatenbank, in der Sie Artikel über die Einrichtung von platform, bekannte Probleme, die Verwendung von GameMaker und eine Reihe anderer Themen finden.
  • Melde einen Fehler - Solltest du einen Fehler finden oder ein Problem bei der Benutzung von GameMaker haben, kannst du ihn hier melden.
  • Open Log In Explorer - Dies öffnet das Verzeichnis auf Ihrem System, in dem Sie die Datei ui.log finden können, die normalerweise für das Einreichen von IDE-bezogenen Fehlerberichten benötigt wird.
  • GameMaker.io - Öffnet die GameMaker-Website.
  • GameMaker Community - Dies öffnet die GameMaker Community Foren in Ihrem Browser. Beachten Sie, dass dies ein von der Community betriebenes Forum ist. Wenn Sie also einen Fehlerbericht haben oder technische Unterstützung benötigen, sollten Sie zuerst die offizielle Support-Seite besuchen (siehe oben).
  • Release Notes - Dies öffnet die Release Notes für GameMaker in Ihrem Browser. Sollten Sie ein Update zurücksetzen müssen, können Sie Installationsprogramme für frühere Versionen erhalten, indem Sie auf den Link mit der Versionsnummer auf dieser Seite klicken.
  • Aktuelle Runtime Versionshinweise - Da die Runtime unabhängig von IDE aktualisiert werden kann, verfügt sie über einen eigenen Satz von Runtime-Versionshinweisen. Wenn Sie diese Option wählen, werden sie in Ihrem Browser geöffnet, aber beachten Sie, dass die neueste runtime möglicherweise nicht die ist, die Sie verwenden. Um herauszufinden, auf welcher runtime Sie sich befinden, oder um Informationen darüber zu erhalten, wie Sie runtimes hinzufügen, entfernen oder aktivieren können, lesen Sie bitte den Abschnitt Laufzeitpräferenzen.
  • Erforderliche SDKs - Aufgrund des plattformübergreifenden Charakters von GameMaker werden eine Reihe von SDKs von Drittanbietern benötigt. Wenn Sie auf diese Schaltfläche klicken, gelangen Sie zu einem Support-Artikel, der detailliert beschreibt, was Sie für die einzelnen Plattformen benötigen.
  • Projekt im Explorer öffnen - Damit wird der Speicherort des aktuell geladenen Projekts im OS-Dateiexplorer geöffnet.
  • Systemschriftarten aktualisieren - Dies aktualisiert die Systemschriftarten, die GameMaker sehen kann, und ist nützlich, um neue Schriftarten zu erkennen, die Sie installiert haben, während IDE geöffnet war.
  • Lizenzen - Es öffnet sich ein Fenster mit den Lizenzen für alle in GameMaker verwendeten Bibliotheken von Drittanbietern.
  • Über - Öffnet das Fenster "Über GameMaker", in dem Sie Versionsinformationen zum Produkt erhalten (wichtig für die Meldung von Fehlern oder Problemen) und die Credits einsehen können.
