Das Menü Windows

Dieses Menü dient zum Öffnen der allgemeinen angedockten Fenster von IDE. Da Andockfenster neu angeordnet werden können und ihr Inhalt als eigenständige Fenster entfernt oder sogar ganz geschlossen werden kann, ermöglicht Ihnen dieses Fenster, sie wieder zu öffnen, wenn Sie sie für irgendetwas benötigen. Die Fenster, die von hier aus geöffnet werden können, sind das Asset-Browser-Fenster und alle verschiedenen Ausgabefenster (Ausgabe, Suche, Quellcodekontrolle, Haltepunkte und Kompilier-/Syntax-Fehler).

Als nächstes können Sie das Fenster Inspektor öffnen. In diesem Fenster können Sie die Eigenschaften des aktuell ausgewählten Elements IDE überprüfen und bei Bedarf bearbeiten.

Nach der Option Inspektor finden Sie die Liste der letzten Fenster, und es gibt auch die Option, das Fenster Umgebungsvariablen zu öffnen, in dem Sie die verschiedenen Pfade sehen können, auf die die verschiedenen Umgebungsvariablen in den Einstellungen Umgebungsvariablen, die in den Voreinstellungen verwendet werden, angezeigt werden. Diese Variablen können nicht bearbeitet werden (weder ihr Name noch ihr Pfad/ihre Einstellung), aber Sie können sie auswählen und kopieren/einfügen, um sie anderswo.

Schließlich haben Sie die Möglichkeit, Übungsfenster zu öffnen. Damit werden alle Fenster wieder geöffnet, die sich auf ein geöffnetes Lernprogramm-Projekt beziehen. die Sie geöffnet haben, aber nur, wenn es sich um ein Projekt handelt, das als Teil eines Tutorials erstellt wurde (andernfalls ist diese Option nicht verfügbar). Informationen zur Erstellung eigener Lernprogramme finden Sie im Abschnitt über den Marktplatz (da alle Lernprogramme dort erstellt, gehostet und heruntergeladen werden).
