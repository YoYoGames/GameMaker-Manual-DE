Das Menü Bearbeiten

Das Bearbeitungsmenü ist der Ort, an dem Sie die allgemeinen Bearbeitungswerkzeuge der GameMaker-Oberfläche verwenden können. Die verfügbaren Optionen sind:

  • Rückgängig - Macht eine Aktion rückgängig.
  • Wiederholen - Wiederholen Sie eine vorherige Aktion.
  • Suchen und Ersetzen - Es öffnet sich ein Suchfenster, mit dem Sie Ihr gesamtes code (einschließlich Raumerstellung code, scripts und Ereignisse) nach einer bestimmten Eingabe durchsuchen können - weitere Informationen finden Sie weiter unten.

Wenn Sie im Menü Bearbeiten die Option Suchen & Ersetzen wählen, wird das folgende Fenster geöffnet:

Mit dieser Funktion können Sie alle geskripteten Elemente in Ihrem Spiel durchsuchen, einschließlich Raum code, Ereignis code, scripts und Shader. Bei der Suche können Sie können Sie einen neuen Suchbegriff in das Feld oben eingeben oder auf das Pfeilsymbol rechts daneben klicken, um eine Liste der zuvor gesuchten Begriffe zu öffnen. Sie haben auch die Möglichkeit, einen Ersatzbegriff einzugeben, der den ursprünglichen Suchbegriff ersetzt. Sie können auch einen Ersatzbegriff eingeben, der den ursprünglichen Suchbegriff ersetzt, und auch hier haben Sie die Möglichkeit, über die Pfeilschaltfläche auf der rechten Seite eine zuvor verwendete Ersatzzeichenfolge zu verwenden.

Die Suche kann allgemein sein oder mit Hilfe der Optionen unterhalb der Suchbegriffeingabe verfeinert werden:

  • Sie können die Groß- und Kleinschreibung bei der Suche berücksichtigen. In diesem Fall werden nur Wörter einbezogen, die genau mit der Groß- und Kleinschreibung des Suchstrings übereinstimmen, d. h. wenn Sie ohne diese Option nach "My_Var" suchen, erhalten Sie Ergebnisse mit "my_var", "mY_VAR" oder "My_Var", aber wenn diese Option aktiviert ist, wird nur "My_Var" in die Suchergebnisse aufgenommen.
  • In diesem Fall werden alle Ergebnisse, die den Suchbegriff enthalten, aber keine exakte Übereinstimmung sind, ignoriert, d.h.: eine Suche nach "game" ohne diese Option zeigt Dinge wie die Funktion "game_end()" oder Variablen wie "game_state" an, während eine Suche nach "game" nur Ergebnisse zeigt, die "game" selbst enthalten.
  • Sie können wählen, ob die Suche Kommentare ignorieren soll. Wenn diese Option aktiviert ist, werden alle Kommentare in Ihrem code, die den Suchbegriff enthalten, einfach ignoriert und nicht in den Ergebnissen angezeigt.
  • Sie können einen Filter auf die Suchergebnisse anwenden, der auf den verschiedenen Asset-Typen im Asset-Browser basiert. Standardmäßig ist dieser Filter auf "Alles" eingestellt, aber wenn Sie darauf klicken, öffnet sich das folgende Menü, in dem Sie Folgendes an- oder abwählen können Assets, die in die Suche einbezogen werden sollen:

Sobald Sie einen Suchbegriff (und optional einen Ersetzungsbegriff) eingegeben und die Suchoptionen festgelegt haben, können Sie mit den verschiedenen Schaltflächen am unteren Rand auswählen, wie die Suche durchgeführt werden soll und welche Aktionen durchgeführt werden sollen. Diese Schaltflächen sind:

  • Alle suchen: Findet alle Suchbegriffe und zeigt sie im Ausgabefenster an.
  • Vorherige suchen: Sucht die vorherige Instanz des Suchbegriffs. Dadurch wird das Asset, das den Verweis enthält, in workspace geöffnet (falls es nicht bereits geöffnet ist) und der Schwenk dorthin ausgeführt.
  • Weiter suchen: Sucht die nächste Instanz des Suchbegriffs. Dadurch wird das Asset, das den Verweis enthält, in workspace geöffnet (falls es nicht bereits geöffnet ist) und der Schwenk dorthin ausgeführt.
  • Alles ersetzen: Diese Option führt eine Suche nach dem angegebenen Begriff mit den eingestellten Optionen durch und ersetzt ihn in den Assets durch den von Ihnen angegebenen neuen Ersetzungsbegriff.
  • Ersetzen: Ersetzt den aktuell ausgewählten Begriff im aktuellen Editor, wo er hervorgehoben ist, durch den ausgewählten Ersetzungsbegriff.
  • Suche beenden: Stoppt die aktuelle Suche, wenn eine solche gerade läuft.

Wenn Sie die Escape-Taste drücken, während dieses Fenster geöffnet ist, wird das ausgewählte Feld gelöscht; wenn das Feld jedoch bereits leer ist, wird das Fenster geschlossen.

Alle Ergebnisse des Fensters "Suchen & Ersetzen" werden im Fenster "Suchausgabe" am unteren Rand von IDE angezeigt. Jedes Ergebnis kann mit einem Doppelklick auf geöffnet werden, um das Asset mit dem Begriff im aktuellen workspace zu öffnen und zu diesem zu schwenken, um es zu bearbeiten.
