Das Menü Layouts

Dieses Menü bezieht sich auf das Speichern und Laden verschiedener Layouts von IDE. Sie können IDE so einrichten, wie Sie es wünschen, wobei bestimmte Elemente (wie der Asset-Browser, oder die Ausgabe Windows) anders angeordnet sind als in der Standardeinstellung. Mit den Optionen in diesem Menü können Sie die Layouts des aktuellen Projekts als *.xml Datei speichern, die dann in ein beliebiges Projekt geladen werden kann, um die gleichen Layouts zu verwenden. Die Optionen in diesem Menü sind wie folgt:

  • Layout laden - Es öffnet sich ein Dateiexplorer, in dem Sie eine zuvor gespeicherte Layoutdatei (*.xml) zum Laden auswählen können.
  • Layout speichern - Es öffnet sich ein Datei-Explorer, in dem Sie das aktuelle IDE Layout als *.xml-Datei benennen und speichern können.
  • Layout zurücksetzen - Hiermit wird das aktuelle Layout von IDE auf die Standardeinstellungen zurückgesetzt.
