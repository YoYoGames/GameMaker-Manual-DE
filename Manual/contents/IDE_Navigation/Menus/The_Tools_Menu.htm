Das Menü Werkzeuge

In diesem Menü können Sie die verschiedenen eigenständigen Werkzeuge öffnen, die Ihnen in GameMaker zur Verfügung stehen, mit den folgenden Optionen:

  • Texturgruppen - Zeigen Sie das Fenster Texturgruppen an.
  • Audiogruppen - Zeigen Sie das Fenster Audiogruppen an.
  • Sound Mixer - Anzeige des Sound Mixer-Fensters.
  • Projektbilder generieren - Öffnet das Fenster Projektbilder generieren.
  • Lokales Paket erstellen - Öffnet das Asset-Packager-Fenster, in dem Sie lokale Asset-Pakete zur späteren Verwendung erstellen und speichern können.
  • Lokales Paket importieren - Öffnet ein Datei-Explorer-Fenster, in dem Sie ein lokales Asset-Paket auswählen und importieren können, indem Sie es entweder zum aktuellen Projekt hinzufügen oder ein neues Projekt dafür erstellen. für es erstellen.
