Das Menü Datei

Im Dateimenü können Sie Projekte ändern, speichern oder importieren sowie die Einstellungen von GameMaker IDE festlegen. Die verfügbaren Optionen sind:

  • Neues Projekt - Hiermit können Sie ein neues Projekt von Grund auf in GameMaker erstellen. Es öffnet sich der "Speichern"-Dialog und Sie werden aufgefordert, einen Speicherort für das neue Projekt zu wählen. Sobald Sie dem Projekt einen Namen und einen Speicherort gegeben haben, werden Sie zu einem neuen, leeren Arbeitsbereich weitergeleitet, um mit der Arbeit daran zu beginnen. Die Projektdateien werden mit der Dateierweiterung *.yyp gespeichert.
  • Projekt öffnen - Es wird ein Datei-Explorer geöffnet, in dem Sie eine zuvor erstellte *.yyz Projektdatei zum Laden auswählen können. Bitte beachten Sie, dass Sie beim Import die folgende Warnmeldung erhalten, wenn das Projekt mit einer niedrigeren Version als 2.3.0 erstellt wurde: Wenn Sie fortfahren, wird ein Datei-Explorer geöffnet, in dem Sie dem Projekt einen aktualisierten Namen und einen Speicherort geben können, damit das ursprüngliche Projekt nicht überschrieben wird. Sobald das Projekt in das neue Format 2.3.0+ konvertiert wurde, wird es im IDE geöffnet und Sie erhalten eine Meldung, dass die Konvertierung erfolgreich war. Überprüfen Sie den Kompatibilitätsbericht, der durch den Konvertierungsprozess erstellt wurde, und sehen Sie nach, ob Sie etwas korrigieren müssen, bevor Sie versuchen, das Projekt auszuführen oder zu kompilieren. Wenn Sie beim Versuch, ein Projekt zu öffnen, das mit einer Version vor 2.3.0 erstellt wurde, wiederholt Fehler erhalten, wenden Sie sich bitte an YoYo Games (über die Option "Fehler melden" im Hilfemenü) und geben Sie die Originaldatei *.yyp an, die Sie zu konvertieren versuchen.
  • Projekt importieren - Hier können Sie Projekte importieren, die zuvor als *.yyz Archiv exportiert wurden (siehe "Projekt exportieren", unten), oder - wenn Sie ein Projekt haben, das mit dem älteren GameMaker: Studio 1.4 erstellt wurde, können Sie diese Option wählen, um entweder die *.gmx oder die *.gmz Datei aus dieser Produktlinie zu importieren. Wie beim Öffnen eines Projekts muss die Datei *.yyz, die mit einer GameMaker-Version vor dem Update 2.3.0 erstellt wurde, konvertiert werden, da sich das Format der Projektdatei mit diesem Update geändert hat.

HINWEIS: Um ein früheres GameMaker Projekt in GameMaker zu importieren, muss es mit der Version 1.4 erstellt worden sein, und selbst dann funktioniert es möglicherweise nicht direkt beim Import. Es gibt viele Unterschiede zwischen GameMaker und GameMaker: Studio 1.4, so dass Ihre älteren Projekte möglicherweise etwas angepasst werden müssen, damit sie funktionieren. Bitte lesen Sie den Artikel Portieren eines GM:S 1.4 Spiels nach GameMaker in der GameMaker Knowledge Base und den Abschnitt des Handbuchs, der veraltete Funktionen und Kompatibilitätsscripts behandelt.

WICHTIG! Beim Importieren eines *.yyz -Projekts, das Schriftarten-Ressourcen enthält, müssen Sie die Schriftarten möglicherweise "neu generieren", indem Sie sie entfernen/hinzufügen oder die Schriftart im Schriftarten-Editor ändern und dann wieder zurücksetzen. Dies liegt daran, dass die vorhandenen Schriftinformationen beim Export von *.yyz beibehalten werden und das Projekt beim späteren Import oder auf einem anderen Rechner möglicherweise nicht die erforderlichen Schriftarten bereits installiert hat (oder eine andere Schriftart mit demselben Namen enthält). Dies sollte nur dann gemacht werden, wenn Sie Probleme mit der Darstellung der Schrift nach dem Import haben.

  • Letzte Projekte - Mit dieser Option können Sie die zuletzt verwendeten Projekte anzeigen. Wenn Sie diese Option auswählen, wird die Startseite wieder geöffnet (falls Sie sie geschlossen haben) und Sie können ein kürzlich verwendetes Projekt aus der dortigen Liste auswählen.
  • Projekt speichern - Damit wird Ihr Spiel an dem Ort gespeichert, den Sie bei der Erstellung des Projekts festgelegt haben (oder von dem aus Sie es geöffnet haben).
  • Projekt speichern unter - Dasselbe wie oben, nur dass Sie den Speicherort und den neuen Namen des Projekts angeben.
  • Projekt exportieren - Hiermit können Sie Ihr Projekt entweder als Datei oder als Vorlage exportieren:
    • YYZ - Dies exportiert das Projekt als eine einzelne *.yyz Datei und ist sehr nützlich, um Kopien an andere Personen zu senden oder Sicherungskopien zu speichern.
    • Als Vorlage - Damit können Sie Ihr Projekt als Vorlage exportieren, die Sie später als Basis für andere Projekte auf der Startseite verwenden können. Wenn Sie diese Option auswählen, wird die folgende Aufforderung angezeigt:
      Klicken Sie zunächst auf "Vorlageninfo bearbeiten", um sicherzustellen, dass Ihre Vorlageninformationen korrekt sind, und wählen Sie dann "Exportieren", wenn Sie diese Aufforderung ein zweites Mal öffnen, um es als Vorlage zu speichern. Sie finden die Vorlage dann im Menü "Vorlage" auf dem Startbildschirm.
  • Einstellungen - Wenn Sie dies auswählen, gelangen Sie zum Fenster "Einstellungen", in dem Sie festlegen können, wie GameMaker aussieht und sich anfühlt.
  • Neue IDE - Wenn Sie diese Option wählen, wird eine neue Instanz der IDE geöffnet, die Sie verwenden können (sie wird auf der Startseite geöffnet).
  • Mein Konto - Diese Option öffnet Ihren Standard-Webbrowser und führt Sie zu Ihrer YoYo Kontoseite.
  • Abmelden - Wenn Sie dies wählen, werden Sie von GameMaker abgemeldet. Sie müssen sich mit der gleichen Lizenz (oder einer anderen) erneut anmelden, um das Produkt später verwenden zu können.

WICHTIG! Wenn Sie eine kostenlose Lizenz für GameMaker verwendet haben und dann eine der anderen Lizenzen kaufen, müssen Sie sich möglicherweise von IDE abmelden, GameMaker neu starten und sich erneut anmelden, um sicherzustellen, dass die neue Lizenz korrekt übernommen wird (dies sollte jedoch automatisch geschehen).

  • Beenden - Beendet GameMaker auf dem Desktop.
