Das Menü Marketplace

Neben der Möglichkeit, eigene Assets und code für Ihre Projekte zu erstellen, haben Sie auch Zugang zu GameMaker Marketplace, wo Sie sowohl kostenlose als auch kostenpflichtige Assets herunterladen können, um sie Ihren Projekten hinzuzufügen. Mehr über den Marketplace erfahren Sie unter dem Abschnitt über den GameMaker-Marktplatz, und die Optionen in diesem Menü sind unten aufgeführt:

  • Gehen Sie zum Marktplatz - Öffnen Sie das Fenster Marketplace, wo Sie nach Assets suchen können.
  • Meine Bibliothek - Zeigen Sie Ihre aktuelle Marketplace Inhaltsbibliothek an.
  • Meine Kaufhistorie - Diese Option öffnet einen Browser und bringt Sie zur Kontoseite des angemeldeten Benutzers, wo Sie die Kaufhistorie für dieses Konto einsehen können.
  • Vorhandenes Paket aktualisieren - Diese Option öffnet den Assistenten für die Paketaktualisierung, mit dem Sie ein bestehendes Marketplace Paket aktualisieren können, das Sie zuvor erstellt haben.
  • Paket erstellen - Mit dieser Option können Sie den Assistenten zum Erstellen von Paketen öffnen, um ein neues Asset-Paket oder ein Tutorial für die Marketplace zu erstellen.
