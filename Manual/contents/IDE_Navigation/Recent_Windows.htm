Neue Fenster

Die Liste " Zuletzt geöffnete Fenster " kann über das Menü " Fenster" geöffnet werden und zeigt eine kurze Liste (standardmäßig mit maximal 10 Einträgen) der Fenster an, die Sie zuvor geöffnet haben. Zu Beginn ist diese Liste leer, aber wenn Sie verschiedene Fenster in IDE öffnen (z. B. ein sprite oder ein script oder ein Objekt usw.), wird die Liste ergänzt. Wenn Sie zu einem dieser zuvor geöffneten Fenster zurückkehren möchten, klicken Sie einfach mit der linken Maustaste auf das Element in der Liste, und workspace springt zu diesem Fenster. Wenn Sie mit der rechten Maustaste auf einen Eintrag klicken, öffnet sich das folgende Menü: Hier können Sie einen Eintrag (oder mehrere, da Sie mit + oder / + mehrere auswählen können) aus der Liste löschen oder an den Anfang der Liste verschieben lassen. Das RMB-Menü verfügt auch über eine zusätzliche Option, mit der Sie die Liste vollständig löschen können. Es ist erwähnenswert, dass die Liste der letzten Windows an die Seiten von IDE angedockt werden kann und dass Sie bestimmte Einstellungen vornehmen können, um ihr Verhalten zu ändern.
