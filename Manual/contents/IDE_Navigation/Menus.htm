Menüs

Der obere Teil des GameMaker IDE wird von der Menüleiste eingenommen. Hier finden Sie eine Reihe von Dropdown-Menüs, mit denen Sie durch die IDE zu navigieren, Aufgaben auszuführen oder zusätzliche Fenster mit Werkzeugen zu öffnen, die Ihnen bei der Entwicklung Ihrer Spiele helfen. Beachten Sie, dass wir hier die standardmäßig verfügbaren Menüs auflisten, aber einige der Asset-Editoren fügen je nach Kontext zusätzliche Menüs hinzu. Diese zusätzlichen Menüs werden auf den Seiten des jeweiligen Editors erklärt, der sie erstellt hat.

Die verfügbaren Standardmenüs sind:

  • Das Menü Datei
  • Das Menü Bearbeiten
  • Das Menü Bauen
  • Das Menü Windows
  • Das Menü Werkzeuge
  • Das Menü Marketplace
  • Das Menü Layouts
  • Das Hilfe-Menü
