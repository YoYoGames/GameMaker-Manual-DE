IDE-Eingabe und Navigation

Die GameMaker-IDE akzeptiert Maus- und Tastatureingaben, und viele Operationen können mit der einen oder der anderen oder mit beiden durchgeführt werden. Im Allgemeinen können Sie mit der linken Maustaste irgendetwas auswählen, / + verwenden, um mehrere Elemente auszuwählen, gedrückt halten, um Elemente in verschiedene Docks oder auf die workspaces zu ziehen, und auch die , um kontextspezifische Menüs zu öffnen. Wenn Sie GameMaker auf einem macOS System betreiben und eine Ein-Tasten-Maus verwenden, benutzen Sie bitte + , um einen rechten Mausklick zu erhalten.

Es gibt auch eine große Anzahl von Tastenkombinationen, die verwendet werden können, um durch die verschiedenen workspace Elemente und Ressourcen-Editoren zu navigieren, und wir werden kurz einige der wichtigsten durchgehen (eine vollständige Liste finden Sie hier):

  • / + "Z" : Dies macht die vorherige Aktion rückgängig und funktioniert in den meisten Editoren. Es gibt auch mehrere Stufen der Rückgängigmachung, so dass Sie diese Taste mehrmals drücken können, um Änderungen rückgängig zu machen.
  • / + "Y" : Damit wird eine zuvor rückgängig gemachte Aktion wiederhergestellt, was in den meisten Editoren funktioniert. Wie beim Rückgängigmachen gibt es auch beim Wiederherstellen mehrere Stufen.
  • / + "S" : Damit wird GameMaker gezwungen, Ihr Projekt zu speichern.
  • / + "T" : Damit öffnen Sie das Fenster "Gehe zu", in dem Sie schnell nach einem beliebigen Objekt in Ihrem Spiel suchen und es aufrufen können. Wenn Sie den Suchbegriff eingeben, wird die Liste entsprechend angepasst. Wenn Sie also zum Beispiel alle Ihre Räume mit "rm_" beginnen, wird die Liste so angepasst, dass nur die Objekte angezeigt werden, die mit diesem Namen beginnen.

  • / + : Damit wird das Fenster Arbeitsbereich-Übersicht geöffnet, in dem Sie schnell zwischen den Elementen navigieren können, die in den verschiedenen workspaces geöffnet sind. Das Fenster bleibt geöffnet, solange Sie oder gedrückt halten, und wenn Sie einmal drücken, gelangen Sie zum nächsten Element im Fenster. Wenn Sie die Tastenkombination loslassen, gelangen Sie zu dem ausgewählten Element.

    HINWEIS Wenn Sie die Bindung für dieses Fenster so umstellen, dass entweder mehrere Modifikatoren (z. B. + + B) oder eine einzelne Taste (z. B. nur B ) verwendet werden, bleibt das Fenster nach dem Drücken der Tastenkombination geöffnet, unabhängig davon, ob Sie die Taste noch gedrückt halten oder nicht, bis Sie es manuell schließen oder ein Element aus dem Fenster auswählen.

  • : Dadurch wird das Handbuch geöffnet. Beachten Sie, dass Sie bei Verwendung von GML Visual oder GML code auch eine Aktion oder eine Funktion (oder ein Schlüsselwort oder was auch immer) auswählen und dann drücken oder die mittlere Maustaste drücken können, um das Handbuch auf der entsprechenden Seite zu öffnen.
  • : Mit dieser Funktion können Sie alle angedockten Fenster in IDE ein- und ausklappen.

Weitere Möglichkeiten zum Navigieren auf workspace sind die Tastenkombination / + + <Arrow Keys> um sich zwischen den geöffneten Fenstern in der gedrückten Richtung zu bewegen, und durch Drücken und Halten der mittleren Maustaste und anschließendes Ziehen der Maus können Sie sich auch auf workspace bewegen.

Laptop-Modus

Wenn Sie GameMaker auf einem Laptop verwenden, haben Sie oben auf der Seite IDE eine weitere Option für den Laptop-Modus:

Diese Funktion ist standardmäßig aktiviert, kann aber auch ausgeschaltet werden, wenn Sie dies wünschen, indem Sie die Schaltfläche. Der Laptop-Modus wird mit einigen Werkzeugen auf IDE kombiniert, um die Verwendung eines Touchpads zu verbessern und die drei wichtigsten Mausinteraktionen zu vereinfachen: Schwenken, Zoomen und Blättern. Hierfür werden zwei Modifikatortasten verwendet: Links / und Links . Wenn der Laptop-Modus aktiviert ist, Links / und Links für Mausinteraktionen wie das Verschieben des Raumeditors "reserviert", während Links / der Zoom ist, Links für Scrollen und Schwenken steht. Das Scrollen im Laptop-Modus verhält sich genauso wie das Schwenken, es sollte also intuitiv und präzise genug sein, damit Sie die Bildlaufleiste nicht verwenden müssen. Beachten Sie, dass Sie beim Halten dieser Tasten einfach die Maus bewegen müssen - nicht klicken oder ziehen, sondern nur bewegen - und dass, wenn es einen allgemeinen Shortcut gibt, der normalerweise die die Verwendung der / oder Tasten erfordert, verwenden Sie stattdessen die Rechte Tasten der Tastatur verwenden.



Touchscreen-Unterstützung

Abschließend sei noch erwähnt, dass die IDE für GameMaker auch eine minimale Unterstützung für Touchscreens bietet. Auf allen Betriebssystemen können Sie den Touchscreen zum Klicken und Ziehen von Objekten im Hauptfenster workspace verwenden, und wir unterstützen 2 gleichzeitige Zeiger, wobei ein zweites Antippen zu einem Rechtsklick führt. Beachten Sie, dass der GameMaker Windows 8 auf IDE und höher auch Stiftgeräte unterstützt.
